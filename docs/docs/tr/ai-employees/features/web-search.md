# Web Araması

Web araması, modelin eğitim verilerinin dışındaki güncel bilgileri tamamlamak için kullanılır.

## Nasıl Çalışır?

Web aramasının kullanılabilirliği, mevcut oturumda seçilen model servisinin Web Aramasını destekleyip desteklemediğine bağlıdır.

- **Destekleniyor:** Web araması anahtarı görüntülenir; ihtiyaca göre açılabilir veya kapatılabilir.
- **Desteklenmiyor:** Bu anahtar görüntülenmez ve arama durumu otomatik olarak kapatılır.

## Sohbet İçinde Kullanım

Sohbet giriş alanındaki web araması anahtarını kullanın:

- **Açık:** Yapay zeka önce arama yapabilir, ardından yanıt verebilir.

![web-search-switch-visible-on-supported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-visible-on-supported-model.png)

- **Kapalı:** Yapay zeka yalnızca mevcut bağlama dayalı olarak yanıt verir.

![web-search-switch-hidden-on-unsupported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-hidden-on-unsupported-model.png)

## Platform Farklılıkları

Farklı modellerin Web Araması desteği farklılık gösterir; lütfen mevcut duruma göre kullanın.
