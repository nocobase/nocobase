# AI Çalışanlarını Etkinleştirme

NocoBase, farklı senaryolar için önceden yapılandırılmış birden fazla yerleşik AI çalışanı sunar.

## Nasıl Etkinleştirilir

`AI employees` yönetim sayfasına girdikten sonra, listedeki `Enabled` anahtarını kullanarak çalışanları doğrudan etkinleştirebilirsiniz.

![ai-employee-list-enable-switch.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employee-list-enable-switch.png)

## Etkinleştirdikten Sonra Kullanım

Etkinleştirildikten sonra ilgili çalışan, kullanılabilir giriş noktalarında (sağ alt köşedeki kısayol girişi veya belirli senaryo girişleri gibi) görünür.
