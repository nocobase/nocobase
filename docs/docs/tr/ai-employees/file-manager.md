# Dosya Yönetimi

## Giriş

Yapay Zeka çalışanları ile yapılan görüşmeler sırasında yüklenen dosyaların depolama yöntemini yapılandırın.

## Ayarlar

Yapay Zeka çalışanı eklentisinin yapılandırma sayfasına gidin, `Settings` sekmesine tıklayarak depolama yöntemi ayarları sayfasına ulaşın.

![20251022002648](https://static-docs.nocobase.com/20251022002648.png)

`Files storage` için mevcut depolama seçeneklerini Dosya Yönetimi eklentisinde ayarlayabilirsiniz.

![20251022002853](https://static-docs.nocobase.com/20251022002853.png)
