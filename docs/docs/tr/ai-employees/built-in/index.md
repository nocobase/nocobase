# Yerleşik Yapay Zeka Çalışanları

## Tanıtım

NocoBase'de aşağıdaki yerleşik yapay zeka çalışanları bulunmaktadır. Bu çalışanlar, eksiksiz beceriler, araçlar ve bilgi tabanlarıyla donatılmıştır. Çalışmaya başlamaları için tek yapmanız gereken onlara bir LLM yapılandırmaktır.

- `Orin`: Veri modelleme uzmanı
- `Avery`: Form doldurucu
- `Viz`: İçgörü analisti
- `Lexi`: Çevirmen
- `Nathan`: Ön uç kod mühendisi
- `Cole`: NocoBase asistanı
- `Vera`: Araştırma analisti
- `Dex`: Veri düzenleyici
- `Ellis`: E-posta uzmanı

## Nasıl Etkinleştirilir

Yapay Zeka Çalışanları eklenti yapılandırma sayfasına gidin, `AI employees` sekmesine tıklayarak Yapay Zeka Çalışanları yönetim sayfasına ulaşın.

Sistemde birden fazla yerleşik yapay zeka çalışanı olduğunu göreceksiniz, ancak hiçbiri etkin durumda değildir. Bu nedenle, uygulama sayfasında bu yapay zeka çalışanlarıyla henüz iş birliği yapamazsınız.

![20251022121248](https://static-docs.nocobase.com/20251022121248.png)

Etkinleştirmek istediğiniz yerleşik yapay zeka çalışanını seçin, `Edit` (Düzenle) düğmesine tıklayarak yapay zeka çalışanı düzenleme sayfasına gidin.

Öncelikle, `Profile` (Profil) sekmesinde `Enable` (Etkinleştir) anahtarını açın.

![20251022121546](https://static-docs.nocobase.com/20251022121546.png)

Ardından, `Model settings` (Model Ayarları) sekmesinde, yerleşik yapay zeka çalışanı için modeli ayarlayın:

- LLM hizmet yönetimi bölümünde oluşturduğunuz LLM hizmetini seçin;
- Kullanmak istediğiniz büyük dil modelinin adını girin.

![20251022121729](https://static-docs.nocobase.com/20251022121729.png)

### Etkinleştirmeyi Tamamlama

Yerleşik yapay zeka çalışanı için modeli ayarladıktan sonra, değişiklikleri kaydetmek için `Submit` (Gönder) düğmesine tıklayın.

Ardından, sayfanın sağ alt köşesindeki yapay zeka çalışanı hızlı başlatma düğmesinde bu yerleşik yapay zeka çalışanını görebilirsiniz.

![20251022121951](https://static-docs.nocobase.com/20251022121951.png)

### Not

Bazı yerleşik yapay zeka çalışanları, etkinleştirildikten sonra bile sağ alt köşedeki yapay zeka çalışanı listesinde görünmeyebilir. Örneğin, Orin yalnızca ana veri yapılandırma sayfasında; Nathan ise yalnızca JS düzenleyicisinde görünecektir.
