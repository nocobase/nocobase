# Giriş

NocoBase, açık kaynaklı bir no-code (kodsuz) geliştirme platformudur.

Airtable ve Notion gibi araçların aksine, NocoBase verileriniz üzerinde size tam kontrol sağlar. Veriler, MySQL, PostgreSQL, SQL Server, SQLite gibi dilediğiniz bir veritabanında saklanabilir.

Ayrıca NocoBase, mevcut sistemlerle kolayca entegre edilebilen veya özel ihtiyaçlarınıza göre genişletilebilen zengin bir API ve esnek bir eklenti sistemi sunar.

## Temel Özellikler

### Verileri bir hesap tablosu gibi yönetin

NocoBase'in arayüzü sezgisel ve kullanımı kolaydır, tıpkı bir hesap tablosu kullanmak gibi. Hiç kod yazmadan verileri kolayca oluşturabilir, görüntüleyebilir, güncelleyebilir ve silebilirsiniz.

### Esnek izin kontrolü

NocoBase, her bir rolün verilere erişimini (görüntüleme, oluşturma, güncelleme ve silme işlemleri dahil) hassas bir şekilde kontrol etmenizi sağlayan rol tabanlı bir izin kontrol sistemi sunar.

### Güçlü API

NocoBase, verileriniz için otomatik olarak bir REST API oluşturarak diğer sistemlerle entegrasyonu kolaylaştırır.

### Genişletilebilir eklenti sistemi

NocoBase'in eklenti sistemi, platformu kendi ihtiyaçlarınıza göre genişletmenize olanak tanır. Mevcut eklentileri kurabilir veya belirli iş gereksinimlerini karşılamak için kendi eklentilerinizi geliştirebilirsiniz.
