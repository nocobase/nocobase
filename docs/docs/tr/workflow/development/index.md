# Genel Bakış

İş akışının yerleşik özellikleri tüm senaryoları kapsamayabilir. Örneğin, yerleşik düğüm türleri, tüm iş senaryolarındaki her işlemi kapsayamaz. Bu nedenle, iş akışını genişletmeye yönelik bir tasarım da sunuyoruz. Bu genişletme, tetikleyici ve düğüm türlerini kapsar. Yerleşik özelliklerin yetersiz kaldığı iş senaryolarında, düşük kodlu yöntemlerle genişletme yaparak çözümler üretebilirsiniz.

Genişletmeler başlıca iki bölümden oluşur:

- [Tetikleyici Türlerini Genişletme](./trigger.md)
- [Düğüm Türlerini Genişletme](./node.md)
