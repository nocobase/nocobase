# Çin Bölgesi Alanı

## Giriş

Çin Bölgesi Alanı eklentisi (`field-china-region`), NocoBase'e "Çin Bölgesi" alan türünü ekler.

## Kurulum

NocoBase ana arayüzünde "Eklenti Yöneticisi"ne gidin, "Çin Bölgesi" eklentisini bulun ve etkinleştirin.

## Kullanım

koleksiyon yapılandırmasında yeni bir alan ekleyin ve alan türü olarak "Çin Bölgesi"ni seçin.

![](/images/field-china-region-1.png)

### Alan Yapılandırması

Alan yapılandırmasında, alan için görüntüleme modunu ayarlayabilirsiniz:

- Eyalet
- Eyalet/Şehir
- Eyalet/Şehir/İlçe
- Eyalet/Şehir/İlçe/Kasaba

![](/images/field-china-region-2.png)
