# Dosya Önizleme

Dosya alanları (ek alanları dahil) içeren arayüzlerde, dosya küçük resmine veya simgesine tıklayarak dosyaları önizleyebilirsiniz. Yerleşik önizleme işlevi; resimler, PDF'ler ve tarayıcılar tarafından yerel olarak desteklenen çoğu dosya türü dahil olmak üzere çeşitli dosya türlerini destekler.

![20251129232307](https://static-docs.nocobase.com/20251129232307.png)

Yerel önizlemeyi desteklemeyen dosya türleri için, ilgili dosya önizleme eklentilerini yükleyerek veya genişleterek önizleme özelliğini etkinleştirebilirsiniz. Örneğin, Office Dosya Önizleme eklentisini yükledikten sonra Word, Excel ve PowerPoint dosyalarını önizleyebilirsiniz.

Şu anda NocoBase aşağıdaki dosya önizleme eklentilerini sunmaktadır:

* [Office Dosya Önizleme Eklentisi](../file-preview/ms-office.md)
