# Depolama Motoru: Amazon S3

Amazon S3 tabanlı bir depolama motorudur. Kullanmaya başlamadan önce ilgili hesap ve izinleri hazırlamanız gerekmektedir.

## Yapılandırma Parametreleri

![Amazon S3 Depolama Motoru Yapılandırma Örneği](https://static-docs.nocobase.com/20251031092524.png)

:::info{title=Not}
Bu bölümde yalnızca Amazon S3 depolama motoruna özel parametreler tanıtılmaktadır. Genel parametreler için lütfen [Motor Genel Parametreleri](./index#引擎通用参数) bölümüne bakın.
:::

### Bölge

S3 depolama bölgesini girin, örneğin: `us-west-1`.

:::info{title=Not}
Depolama alanınızın bölge bilgilerini [Amazon S3 konsolunda](https://console.aws.amazon.com/s3/) görüntüleyebilirsiniz. Yalnızca bölge ön ekini kullanmanız yeterlidir (tam alan adına gerek yoktur).
:::

### AccessKey ID

Amazon S3 yetkilendirilmiş erişim anahtarının ID'sini girin.

### AccessKey Secret

Amazon S3 yetkilendirilmiş erişim anahtarının Secret'ını girin.

### Kova

S3 kovasının adını girin.
