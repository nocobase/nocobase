# Küme Dağıtımı

NocoBase, uygulama performansını ve kullanılabilirliğini artırmak amacıyla uygulamaları küme modunda dağıtmayı destekler. Küme dağıtımı, uygulamanın farklı örneklerini birden çok sunucuda çalıştırmanıza veya tek bir sunucuda çok çekirdekli modda birden çok işlem çalıştırmanıza olanak tanır.

Detaylar için lütfen [Küme Modu](/cluster-mode/) belgelerine bakın.
