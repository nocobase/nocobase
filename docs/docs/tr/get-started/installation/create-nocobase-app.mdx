# create-nocobase-app Kurulumu

:::tip Ön Koşullar

- Node.js 20+, Yarn 1.22.x yüklü olmalıdır
- MySQL 8.0.17+, MariaDB 10.9+, PostgreSQL 10+ veritabanlarından birini yapılandırın ve başlatın
- Ağ ortamı nedenlerinden dolayı, yerel bir ayna (mirror) kullanmanız önerilir (zorunlu değildir)

```bash
$ yarn config set disable-self-update-check true
$ yarn config set registry https://registry.npmmirror.com/
```

:::

:::warning Ortam Değişkenleri Açıklaması

- `TZ`, uygulamanın saat dilimini ayarlamak için kullanılır, varsayılan olarak işletim sisteminin saat dilimidir
- `APP_KEY`, uygulamanın gizli anahtarıdır; kullanıcı token'ları vb. oluşturmak için kullanılır (APP_KEY değiştirilirse eski token'lar da geçersiz kalır). Herhangi bir rastgele dize olabilir. **Lütfen kendi gizli anahtarınızla değiştirin ve dışarıya sızmadığından emin olun**
- `DB_*`, veritabanı ile ilgili yapılandırmalardır; lütfen gerçek durumunuza göre veritabanı bağlantı bilgilerinizle değiştirin

:::

## 2. Dizini Değiştirin

```bash
cd my-nocobase-app
```

## 3. Bağımlılıkları Yükleyin

📢 Ağ ortamı, sistem yapılandırması vb. faktörlerin etkisiyle bu adım on dakikadan fazla sürebilir.

```bash
yarn install
```

:::tip Üretim Ortamı İpucu

Üretim ortamına dağıtım yaparken, boyutu azaltmak için yalnızca gerekli bağımlılıkları yükleyebilirsiniz:

```bash
yarn install --production
```

:::

## 4. NocoBase'i Kurun

```bash
yarn nocobase install --lang=tr-TR
```

Kurulum işlemi otomatik olarak veritabanı tablo yapısını oluşturacak ve verileri başlatacaktır.

## 5. NocoBase'i Başlatın

**Geliştirme ortamı**

```bash
yarn dev
```

**Üretim ortamı**

```bash
yarn start
```

## 6. NocoBase'e Giriş Yapın

Tarayıcıyı kullanarak [http://localhost:13000](http://localhost:13000) adresini açın; başlangıç hesabı ve şifresi `admin@nocobase.com` ve `admin123`'tür.

:::warning Hesap Güvenliği Uyarısı

İlk girişten sonra, sistem güvenliğini sağlamak için lütfen varsayılan şifreyi zamanında değiştirin.

:::
