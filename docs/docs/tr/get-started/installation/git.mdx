# Git Kaynak Kod Kurulumu

:::tip Ön Koşullar

- Git, Node.js 20+ ve Yarn 1.22.x yüklü olmalı
- Gerekli veritabanlarından biri (MySQL 8.0.17+, MariaDB 10.9+, PostgreSQL 10+) yapılandırılmış ve başlatılmış olmalı
- Ağ ortamı nedenlerinden dolayı, yerel bir yansıma kullanmanız önerilir (zorunlu değildir)

```bash
$ yarn config set disable-self-update-check true
$ yarn config set registry https://registry.npmmirror.com/
```

:::

## 1. NocoBase'i yerel makineye indirin

NocoBase sürümünü seçin ([Sürüm karşılaştırması](../quickstart.md)) ve ilgili komutu çalıştırın.

<Tabs>
<Tab label="Latest sürümü">

```bash
git clone https://github.com/nocobase/nocobase.git -b main --depth=1 my-nocobase
```

</Tab>
<Tab label="Beta sürümü">

```bash
git clone https://github.com/nocobase/nocobase.git -b next --depth=1 my-nocobase
```

</Tab>
<Tab label="Alpha sürümü">

```bash
git clone https://github.com/nocobase/nocobase.git -b develop --depth=1 my-nocobase
```

</Tab>
</Tabs>

## 2. Dizin değiştirin

```bash
cd my-nocobase
```

## 3. Bağımlılıkları yükleyin

📢 Ağ ortamı, sistem yapılandırması ve diğer faktörlerin etkisiyle, bu adım on dakikadan fazla sürebilir.

```bash
yarn install --frozen-lockfile
```

## 4. Ortam değişkenlerini ayarlayın

NocoBase için gerekli ortam değişkenleri kök dizindeki `.env` dosyasında saklanır, ortam değişkenlerini gerçek duruma göre düzenleyin.

```bash
TZ=Europe/Istanbul
APP_KEY=your-secret-key
DB_HOST=localhost
DB_PORT=5432
DB_DATABASE=postgres
DB_USER=nocobase
DB_PASSWORD=nocobase
```

:::warning Ortam değişkeni açıklamaları

- `TZ`, uygulamanın saat dilimini ayarlamak için kullanılır, varsayılan olarak işletim sisteminin saat dilimidir;
- `APP_KEY`, uygulamanın gizli anahtarıdır, kullanıcı token'ları vb. oluşturmak için kullanılır (eğer `APP_KEY` değiştirilirse, eski token'lar da geçersiz hale gelir). Herhangi bir rastgele dize olabilir. Lütfen kendi gizli anahtarınızla değiştirin ve dışarıya sızdırılmadığından emin olun;
- `DB_*` veritabanı ile ilgilidir, eğer örnekteki varsayılan veritabanı hizmeti değilse, lütfen gerçek duruma göre düzenleyin.

:::

## 5. NocoBase'i kurun

```bash
yarn nocobase install --lang=tr-TR
```

## 6. NocoBase'i başlatın


**Geliştirme ortamı**

```bash
yarn dev
```

**Üretim ortamı (önerilmez)**

Üretim ortamında doğrudan kaynak koddan dağıtım yapılması önerilmez, üretim ortamı için lütfen [Üretim ortamı dağıtımı](../deployment/production.md) belgesine bakın.

```bash
yarn build
yarn start
```

## 7. NocoBase'e giriş yapın

Tarayıcıyı kullanarak [http://localhost:13000](http://localhost:13000) adresini açın, başlangıç kullanıcı adı ve parolası `admin@nocobase.com` ve `admin123`'tür.

:::warning Hesap güvenliği uyarısı

İlk kez giriş yaptıktan sonra, sistem güvenliğini sağlamak için lütfen varsayılan parolayı zamanında değiştirin.

:::
