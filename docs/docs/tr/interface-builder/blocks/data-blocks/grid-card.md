# Izgara Kartı

## Giriş

Izgara Kartı bloğu, veri kayıtlarının özet bilgilerini kart formatında görüntüler. Farklı ekran boyutlarına göre sütun sayısını yapılandırmayı destekleyerek, çeşitli cihazlarda kullanıcı dostu bir görüntüleme sağlar.
