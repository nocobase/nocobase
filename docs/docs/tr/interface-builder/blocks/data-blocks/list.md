# Liste Bloğu

## Giriş

Liste bloğu, verileri liste şeklinde gösterir. Görev listeleri, haberler ve ürün bilgileri gibi veri görüntüleme senaryoları için oldukça uygundur.

## Blok Yapılandırması

![20251023202835](https://static-docs.nocobase.com/20251023202835.png)

### Veri Kapsamını Ayarlama

Resimde görüldüğü gibi: İptal edilmiş durumdaki siparişleri filtreleyin.

![20251023202927](https://static-docs.nocobase.com/20251023202927.png)

Daha fazla bilgi için [Veri Kapsamını Ayarlama](/interface-builder/blocks/block-settings/data-scope) bölümüne bakabilirsiniz.

### Sıralama Kurallarını Ayarlama

Resimde görüldüğü gibi: Sipariş tutarına göre azalan sırada sıralayın.

![20251023203022](https://static-docs.nocobase.com/20251023203022.png)

Daha fazla bilgi için [Sıralama Kurallarını Ayarlama](/interface-builder/blocks/block-settings/sorting-rule) bölümüne bakabilirsiniz.

## Alanları Yapılandırma

### Bu Koleksiyondaki Alanlar

> **Not**: Devralınan koleksiyonlardaki (yani üst koleksiyonlardaki) alanlar, mevcut alan listesinde otomatik olarak birleştirilir ve gösterilir.

![20251023203103](https://static-docs.nocobase.com/20251023203103.png)

### İlişkili Koleksiyonlardaki Alanlar

> **Not**: İlişkili koleksiyonlardaki alanlar görüntülenebilir (şu anda yalnızca bire bir ilişkiler desteklenmektedir).

![20251023203611](https://static-docs.nocobase.com/20251023203611.png)

Liste alanı yapılandırması için [Detay Alanı](/interface-builder/fields/generic/detail-form-item) bölümüne bakabilirsiniz.
