# Grafik

## Giriş

NocoBase'in veri görselleştirme eklentisi, görsel veri sorgulama yetenekleri ve zengin grafik bileşenleri sağlar. Kullanıcılar, basit yapılandırmalarla hızlıca görselleştirme panoları oluşturabilir, veri içgörülerini sergileyebilir ve çok boyutlu veri analizi ile sunumunu destekleyebilirler.
