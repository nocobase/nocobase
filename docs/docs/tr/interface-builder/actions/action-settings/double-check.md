# Onay

## Giriş

Onay yapılandırması, bir işlemin gerçekten yürütülüp yürütülmeyeceğini teyit etmek için kullanılır. Bir işleme tıklandığında bir onay penceresi açılır. Özellikle silme düğmelerinde, yanlışlıkla veri kaybını veya geri alınamaz sonuçları önlemek amacıyla onay varsayılan olarak etkindir.
