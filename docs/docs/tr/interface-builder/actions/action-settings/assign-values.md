# Alan Ataması

## Giriş

Alan ataması, bir işlem yürütülmeden önce bir alanın değerini önceden belirleme işlemidir. Bu sayede, kayıtları güncellerken veya oluştururken veriler manuel giriş gerektirmeden otomatik olarak doldurulur. Örneğin, sipariş durumlarını toplu olarak güncellemek veya bir sipariş teslim edildiğinde teslimat zamanını mevcut zamana ayarlamak gibi durumlarda bu özellikten faydalanabilirsiniz.
