# Düzenle

## Giriş

Düzenle işlemi, mevcut kaydı bağlam olarak alan bir satır düğmesidir. Kullanıcıların blokları özelleştirebileceği ve yapılandırabileceği bir açılır pencere olarak karşınıza çıkar.

![20251029165442](https://static-docs.nocobase.com/20251029165442.png)

İhtiyacınıza göre blokları yapılandırın.

## İşlem Ayarları

![20251029165620](https://static-docs.nocobase.com/20251029165620.png)
