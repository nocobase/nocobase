# Gönder

## Giriş
Gönder işlemi, form verilerini kaydetmek için kullanılır (bu işlem, form bloklarına özeldir). Ayrıca, veri otomasyon süreçlerini gerçekleştirmek için iş akışlarıyla birleştirilebilir.
