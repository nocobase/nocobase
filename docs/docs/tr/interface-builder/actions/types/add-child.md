# Alt Kayıt Ekle

## Giriş

Bu işlem, özellikle **ağaç tablo blokları** için tasarlanmıştır ve alt kayıtları hızlı bir şekilde eklemenizi sağlar. Bir alt kayıt eklerken, seçili satır üst düğüm olarak kabul edilir. Bu sayede, herhangi bir üst düğüm altına kolayca alt kayıtlar oluşturabilirsiniz. Bu özellik, hiyerarşik verilerin yönetim sürecini önemli ölçüde basitleştirerek, veri yapılarının genişletilmesini ve düzenlenmesini daha verimli ve sezgisel hale getirir.
