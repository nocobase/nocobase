# Filtreleme

## Giriş

Filtreleme işlemi genellikle veri bloklarında bulunur ve çeşitli koşullar yapılandırarak verileri filtrelemek için kullanılır. Unutmayın ki, eğer bir blok için zaten bir veri kapsamı yapılandırılmışsa, filtreleme işlemi veri kapsamı koşullarıyla birleşerek nihai filtreleme koşulunu oluşturacaktır.

![20251029173609](https://static-docs.nocobase.com/20251029173609.png)

## İşlem Ayarları

![20251029173708](https://static-docs.nocobase.com/20251029173708.png)

### Filtrelenebilir Alanlar

Varsayılan olarak, bu koleksiyonun tüm alanları etkinleştirilmiştir. Yalnızca etkinleştirilmiş alanlar, koşul alanı listesinde seçilebilir.

İlişkili koleksiyonlardaki alanlar da filtreleme alanı olarak kullanılabilir.
