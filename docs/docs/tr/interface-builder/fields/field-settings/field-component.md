# Alan Bileşeni

## Giriş

Alan bileşenleri, alan içeriklerini görüntülemek için kullanılan yapılardır.
Bazı alanlar, içinde birden fazla görüntüleme bileşeni barındırır. Kullanıcılar, alanın değerini farklı yöntemlerle sunmak için kendi ihtiyaçlarına en uygun bileşeni seçebilirler.

Örnek: Okuma modundayken, bir URL alanını Önizleme bileşenine veya URL bileşenine (varsayılan) geçirebilirsiniz.
