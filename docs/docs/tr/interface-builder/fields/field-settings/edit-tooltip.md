# İpucu Metnini Düzenleme

## Giriş

Alan ipucu, bir alanın üzerine fareyle geldiğinizde beliren ve o alan hakkında ek açıklama veya rehberlik sağlamak amacıyla tasarlanmış metindir. Bu ipucu metinleri genellikle alanın amacı, giriş formatı ve kısıtlamaları gibi bilgileri içerir. Bu sayede kullanıcılar alanı daha iyi anlar ve doğru şekilde kullanabilirler.
