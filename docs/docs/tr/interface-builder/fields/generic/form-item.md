# Form Alanları

## Giriş
Bir formdaki farklı alan türleri, farklı yapılandırma seçeneklerine sahiptir. Bu sayede kullanıcılar, eklentiler aracılığıyla alanın yapılandırma seçeneklerini genişleterek daha kişiselleştirilmiş ihtiyaçlarını karşılayabilirler.
