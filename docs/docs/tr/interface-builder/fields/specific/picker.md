# Veri Seçici

## Giriş

Veri Seçici, açılır bir tablo seçici şeklinde çalışır. Bu sayede, hedef koleksiyondaki mevcut verileri seçerek ilişkilendirebilir veya hedef koleksiyona yeni veriler ekleyip ardından ilişkilendirme yapabilirsiniz.

![20251027203343](https://static-docs.nocobase.com/20251027203343.png)

Seçilen veriler tablo seçicide gösterilmez.

![20251027203717](https://static-docs.nocobase.com/20251027203717.gif)

Tablo seçicide, ilişkilendirme hedef koleksiyonunu (ekleme, silme, içe/dışa aktarma vb. işlemlerle) daha detaylı yönetebilirsiniz.

![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
