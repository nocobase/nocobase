# Başlık Alanı Bileşeni

## Giriş

Salt okunur modda, ilişki alanları varsayılan olarak **Başlık Alanı Bileşeni** kullanılarak görüntülenir. Bu bileşen, başlık alanının değerini göstererek mevcut ilişkili kaydı tanımlar. Sistem, başlık alanının türüne göre ilgili başlık alanı bileşenini otomatik olarak eşleştirir ve oluşturur.
