# Alt Detaylar

## Giriş

Alt Detaylar, salt okunur modda kullanılan Alt Form bileşeninin karşılığıdır. Etiket ve Başlık bileşenleriyle kıyaslandığında, Alt Detaylar sadece mevcut koleksiyondaki daha fazla veriyi görüntülemenin yanı sıra, ilişkili koleksiyonlardan gelen veri bilgilerini de gösterecek şekilde yapılandırılabilir. Çok seviyeli ilişki verileri, iç içe geçmiş detaylar şeklinde net bir şekilde sunulur.
