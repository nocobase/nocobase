# Dosya Yöneticisi

## Giriş

Dosya Yöneticisi, hedef koleksiyonun dosyalar koleksiyonu olduğu ilişkileri yönetmek için kullanılan bir ilişkilendirme alanı bileşenidir.

![20251027194124](https://static-docs.nocobase.com/20251027194124.png)

## Alan Yapılandırması

- [Alan Bileşeni](/interface-builder/fields/association-field): Açılır Liste, Veri Seçici gibi diğer ilişkilendirme alanı bileşenlerine geçiş yapabilirsiniz.
- Mevcut Dosyaları Seçmeye İzin Ver: Varsayılan olarak etkinleştirilmiştir; yüklenmiş dosyalar arasından seçim yapıp ilişkilendirme yapmanızı sağlar.
