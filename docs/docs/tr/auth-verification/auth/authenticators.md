# Kimlik Doğrulayıcı Yönetimi

## Kullanıcı Kimlik Doğrulama Yönetimi

Kullanıcı kimlik doğrulama eklentisi yüklendiğinde, kullanıcının kullanıcı adı ve e-posta adresine dayalı bir `parola` kimlik doğrulama yöntemi başlatılır.

![](https://static-docs.nocobase.com/66eaa9d5421c9cb713b117366bd8a5d5.png)

## Kimlik Doğrulayıcıları Etkinleştirme

![](https://static-docs.nocobase.com/7f1fb8f8ca5de67ffc68eff0a65848f5.png)

Yalnızca etkinleştirilmiş kimlik doğrulama türleri giriş sayfasında gösterilir.

![](https://static-docs.nocobase.com/8375a36ef98417af0f0977f1e07345dd.png)

## Kullanıcı Kimlik Doğrulama Türleri

![](https://static-docs.nocobase.com/da4250c0cea343ebe470cbf7be4b12e4.png)

Farklı türde kimlik doğrulayıcılar ekleyerek sisteme ilgili kimlik doğrulama yöntemlerini kazandırabilirsiniz.

Mevcut eklentiler tarafından sağlanan kimlik doğrulama türlerine ek olarak, geliştiriciler kendi kullanıcı kimlik doğrulama türlerini de genişletebilirler. Ayrıntılar için [Geliştirici Rehberi](./dev/) bölümüne bakınız.
