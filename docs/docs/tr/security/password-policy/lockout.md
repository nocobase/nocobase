# Kullanıcı Kilitleme

## Giriş

Kilitli kullanıcıları yönetebilir ve kullanıcıları proaktif olarak kilitleyebilirsiniz.

![](https://static-docs.nocobase.com/202412281450083.png)

### Kullanıcı Kilidini Açma

Geçersiz parola giriş limitini aşma nedeniyle kilitlenen kullanıcıların kilidi, ilgili kaydı silerek açılabilir.

### Kullanıcı Kilitleme

Bir kullanıcı kaydı ekleyerek bir kullanıcıyı proaktif olarak kilitleyebilirsiniz. Bir kullanıcı kilitlendiğinde, API anahtarları da dahil olmak üzere hiçbir kimlik doğrulama yöntemiyle sisteme erişmesi engellenecektir.

![](https://static-docs.nocobase.com/202412281450512.png)
