# Yaşam Döngüsü

Bu bölümde, eklentilerin hem sunucu hem de istemci tarafındaki yaşam döngüsü kancalarını düzenleyerek, geliştiricilerin kaynakları doğru bir şekilde kaydetmelerine ve serbest bırakmalarına yardımcı oluyoruz. Ortak kavramları vurgulamak amacıyla FlowModel yaşam döngüsü ile karşılaştırma yapabilirsiniz.

## Önerilen İçerik

- Eklentiler yüklendiğinde, etkinleştirildiğinde, devre dışı bırakıldığında veya kaldırıldığında tetiklenen geri çağrılar.
- İstemci tarafı bileşenlerinin bağlanma, güncellenme ve yok edilme zamanlaması.
- Yaşam döngüsü içinde eşzamansız görevleri ve hataları ele almaya yönelik öneriler.
