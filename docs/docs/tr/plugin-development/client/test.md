# Test

:::tip

Bu bölümde, eklentilerin ön yüz (frontend) kısmı için testlerin nasıl yazılacağı ve çalıştırılacağı anlatılacaktır. İçerik şunları kapsayacaktır:
- Test ortamı yapılandırması ve bağımlılıkları;
- Vitest ve Testing Library kullanarak birim testleri yazma;
- NocoBase ön yüz çalışma zamanı ortamını nasıl simüle edeceğiniz;
- CI'da testleri otomatik olarak yürütme.

İlerleyen sürümlerde detaylı örnekler ve en iyi uygulamalar eklenecektir.

:::
