# ACL Yetki Kontrolü

:::tip

Bu bölümde, ön uç eklentilerinde ACL (Erişim Kontrol Listesi) kullanarak yetki yönetiminin nasıl yapılacağını tanıtacağız. İçerik şunları kapsayacaktır:

- Yetki denetimi;
- Yetkilere göre içeriği dinamik olarak nasıl oluşturacağınız;

İlerleyen sürümlerde detaylı örnekler ve en iyi uygulamalarla bu bölümü zenginleştireceğiz.

:::
