**Parametreler**

| Parametre Adı          | Tip           | Varsayılan Değer | Açıklama                                                                    |
| ---------------------- | ------------- | ---------------- | --------------------------------------------------------------------------- |
| `options.values`       | `M`           | `{}`             | Eklenecek veri nesnesi                                                      |
| `options.whitelist?`   | `string[]`    | -                | `values` alanları için beyaz liste. Yalnızca listedeki alanlar depolanır.   |
| `options.blacklist?`   | `string[]`    | -                | `values` alanları için kara liste. Listedeki alanlar depolanmaz.            |
| `options.transaction?` | `Transaction` | -                | İşlem                                                                       |
