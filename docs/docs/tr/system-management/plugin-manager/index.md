# Eklenti Yönetimi

## Eklenti Yöneticisi

Sayfanın sağ üst köşesindeki eklenti yöneticisi simgesine tıklayarak eklenti yöneticisi sayfasına ulaşabilirsiniz. Bu sayfada eklentileri görüntüleyebilir, kurabilir, etkinleştirebilir veya devre dışı bırakabilirsiniz.

![2024-04-10_08-47-45](https://static-docs.nocobase.com/2024-04-10_08-47-45.png)

## Eklenti Ayarları Merkezi

Eklentilerin genellikle kendi ayar sayfaları bulunur. Sayfanın sağ üst köşesindeki ayarlar simgesine tıklayarak tüm eklentilerin ayar menülerini görüntüleyebilirsiniz. İlgili menü öğesine tıklayarak eklentiyi yapılandırabilirsiniz.
