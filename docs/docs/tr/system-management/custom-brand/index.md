# Özel Marka

## Giriş

Sistemin varsayılan marka adı "NocoBase" olup, başlıca şu üç yerde görüntülenir:

![20240409111100](https://static-docs.nocobase.com/20240409111100.png)

Giriş sayfasının alt kısmında:

![20240409111254](https://static-docs.nocobase.com/20240409111254.png)

Konum 1, [Sistem Ayarları](/system-management/system-settings/) içindeki Logo seçeneği aracılığıyla yapılandırılabilir; Konum 2 ve 3 ise bu eklenti kullanılarak yapılandırılabilir.

## Kullanım Talimatları

Eklentiyi etkinleştirdikten sonra, yapılandırma sayfasına erişmek için sayfanın sağ üst köşesindeki "Özel Marka" menüsüne tıklayın.

![20240409113640](https://static-docs.nocobase.com/20240409113640.png)

- **Marka**: Giriş sayfasının altındaki metni (Konum 3) ayarlamak için kullanılır.
- **Hakkında**: Sayfanın sağ üst köşesindeki menü içeriğini yapılandırmak için kullanılır (aşağıdaki görselde gösterildiği gibi).

![20240409113915](https://static-docs.nocobase.com/20240409113915.png)

Yapılandırma, esnek içerik özelleştirmesi için HTML formatını destekler. Yalnızca basit metin değişiklikleri yapmanız gerekiyorsa, aşağıdaki görselde gösterildiği gibi mevcut içeriği doğrudan düzenleyebilirsiniz:
