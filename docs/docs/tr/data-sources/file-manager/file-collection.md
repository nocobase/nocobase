# Dosya Koleksiyonu

## Giriş

Dosyalar, dosya koleksiyonu olarak bilinen özel bir yapıya sahip koleksiyonda tutulan kayıtlardır. Bu koleksiyon, dosyaların meta verilerini depolamak ve Dosya Yöneticisi aracılığıyla yönetilmek üzere kullanılır.
