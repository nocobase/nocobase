# Aliyun OSS

Aliyun OSS tabanlı bir depolama motorudur. Kullanmadan önce ilgili hesapları ve izinleri hazırlamanız gerekmektedir.

## Yapılandırma Parametreleri

![Aliyun OSS depolama motoru yapılandırma örneği](https://static-docs.nocobase.com/20240712220011.png)

:::info{title=Not}
Burada sadece Aliyun OSS depolama motoruna özel parametreler tanıtılmaktadır. Genel parametreler için lütfen [Motor Genel Parametreleri](./index.md#引擎通用参数) bölümüne bakın.
:::

### Bölge

OSS depolamasının bölgesini girin, örneğin: `oss-cn-hangzhou`.

:::info{title=Not}
[Aliyun OSS konsolu](https://oss.console.aliyun.com/) üzerinden depolama alanının bölge bilgilerini görüntüleyebilirsiniz. Sadece bölgenin ön ek kısmını kullanmanız yeterlidir (tam alan adına gerek yoktur).
:::

### AccessKey ID

Aliyun yetkilendirilmiş erişim anahtarının ID'sini girin.

### AccessKey Secret

Aliyun yetkilendirilmiş erişim anahtarının Secret'ını girin.

### Depolama Kovası

OSS depolamasının depolama kovası adını girin.
