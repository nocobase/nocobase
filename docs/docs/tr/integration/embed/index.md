# NocoBase'i Gömme

## Giriş

Bu eklenti, NocoBase sayfalarını başka web sitelerine veya uygulamalara gömmek için kullanılır.

## Kurulum

Bu eklenti ticari bir eklentidir ve eklenti yöneticisi aracılığıyla yüklenip etkinleştirilmesi gerekir.

![20240323162741](https://static-docs.nocobase.com/20240323162741.png)

## Kullanım Talimatları

Arayüz yapılandırmasını etkinleştirdikten sonra, sayfanın ayarlar menüsünde (sağ üst köşe) "Gömme Bağlantısını Kopyala" seçeneğine tıklayın.

![](https://static-docs.nocobase.com/f11bd6d5e88d38731d7cd3cb149022c8.png)

Tıkladıktan sonra, örneğin `https://xxx.com/embed/qs087rz4o2b` gibi ayrı olarak açılabilen bir bağlantı elde edeceksiniz.

![](https://static-docs.nocobase.com/9d847805a00fd07372a474665ec0e968.png)

Eğer NocoBase sayfalarını başka web sitelerine veya uygulamalara gömmek isterseniz, kullanıcı entegrasyonu yapılması ve `token`'ın bağlantıya eklenmesi gerekir, örneğin `https://xxx.com/embed/qs087rz4o2b?token=xxx`.
