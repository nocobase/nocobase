# Toplu Gönderim

## Toplu Gönderime Genel Bakış
Birden fazla kişiye toplu e-posta gönderebilirsiniz (her alıcı ayrı bir e-posta alacaktır).

![](https://static-docs.nocobase.com/email-manager/Loading--12-31-2025_10_15_PM.png)

## Tablo Üzerinden Toplu Gönderim
E-postaları bir tablo üzerinden de toplu olarak gönderebilirsiniz:

1. Tabloya bir e-posta gönderme düğmesi ekleyin.
   ![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_04_PM.png)

2. Tablodaki e-posta alanını yapılandırın (tablo satırlarını seçtikten sonra e-posta adreslerini ayrıştırmak için kullanılır).
   ![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_17_PM%20(1)

3. E-posta göndermek için tablo verilerini işaretleyin.
   ![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_19_PM.png)

Eğer tablodaki tüm adreslere e-posta göndermeniz gerekiyorsa, **Varsayılan olarak mevcut tablodaki tüm kayıtları alıcı olarak seç** özelliğini etkinleştirebilirsiniz:
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_18_PM%20(1)

E-posta içeriğini düzenledikten sonra, göndermek için toplu gönder düğmesine tıklayın:
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_19_PM.png)
