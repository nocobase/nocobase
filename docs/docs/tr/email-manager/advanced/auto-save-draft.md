# Taslağı Otomatik Kaydet

**Taslağı Kaydet** düğmesinin menüsünden **Taslağı Otomatik Kaydet** özelliğini açabilirsiniz.
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_29_PM.png)

E-posta düzenleme sürecinde, sistem taslağı gerçek zamanlı olarak kaydederek içeriğinizin kaybolmamasını sağlar.
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_31_PM.png)

E-posta düzenleyiciyi tekrar açtığınızda (aynı gönderen ve alıcı için), sistem size kaydedilmiş taslağı kullanmak isteyip istemediğinizi soracaktır.
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_08_35_PM.png)
