# Kullanıcılar

## Kullanıcı Yönetimi

Bu eklenti, yapılandırma merkezinde bir kullanıcı yönetim arayüzü sunar. Uygulama başlatıldıktan sonra silinemeyen bir süper yönetici hesabı otomatik olarak oluşturulur. Süper yönetici, Root yetkilerine sahiptir ve tüm kaynaklara erişebilir.

![](https://static-docs.nocobase.com/44bf40f56b45d4dd96c424fb08082cf6.png)

Bunun yanı sıra, kullanıcıları yönetmek için Kullanıcılar koleksiyonundan çeşitli farklı bloklar da ekleyebilirsiniz, örneğin tablo blokları:
