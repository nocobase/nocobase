# Telemetry Dışa Aktarıcı: HTTP

## Ortam Değişkenleri

### TELEMETRY_METRIC_READER

Telemetry metrik dışa aktarıcı türü.

```bash
TELEMETRY_METRIC_READER=http
```

### TELEMETRY_HTTP_URL

Telemetry verilerini dışa aktarmak için HTTP URL'si.

```bash
TELEMETRY_HTTP_URL=
```
