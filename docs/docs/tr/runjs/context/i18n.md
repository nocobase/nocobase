# ctx.i18n

Dilleri okumak veya değiştirmek için kullanılan mevcut bağlamın i18n örneğidir. **Metinleri çevirmek için [ctx.t()](./t.md) kullanın**; `ctx.i18n.t` kullanmayın.

## Uygulama Senaryoları

Tüm RunJS yürütme ortamları `ctx.i18n` kullanabilir (örneğin; JSBlock, JSField, JSItem, JSColumn, iş akışı, bağlantı kuralları vb.).

## Tip Tanımı

```typescript
interface i18n: {
  language: string;
  changeLanguage(lng: string): Promise<any>;
}
```

## Yaygın Özellikler

| Özellik | Tip | Açıklama |
|------|------|------|
| `language` | `string` | Şu anda aktif olan dil kodu (örneğin; `zh-CN`, `en-US`) |

## Yaygın Yöntemler

### changeLanguage(lng)

Mevcut dili değiştirir.

| Parametre | Tip | Açıklama |
|------|------|------|
| `lng` | `string` | Hedef dil kodu (örneğin; `'en-US'`, `'zh-CN'`) |

**Dönüş değeri**: `Promise<any>`, dil değişimi tamamlandıktan sonra çözümlenir (resolve).

## Örnekler

### Mevcut dili okuma

```ts
const lang = ctx.i18n.language;
// 'zh-CN' | 'en-US' | ...
if (lang.startsWith('zh')) {
  ctx.render(ctx.t('Çince Arayüz'));
} else {
  ctx.render(ctx.t('English UI'));
}
```

### Dil değiştirme

```ts
// İngilizceye geç
await ctx.i18n.changeLanguage('en-US');

// Çinceye geç
await ctx.i18n.changeLanguage('zh-CN');
```

### Dil değiştirme butonu

```ts
const { Button } = ctx.libs.antd;
const isZh = ctx.i18n.language.startsWith('zh');
ctx.render(
  <Button onClick={async () => {
    await ctx.i18n.changeLanguage(isZh ? 'en-US' : 'zh-CN');
  }}>
    {ctx.t(isZh ? 'Switch to English' : 'Çinceye Geç')}
  </Button>,
);
```

## Notlar

- **Çeviri metni**: Tutarlı bir şekilde `ctx.t()` kullanın; `ctx.i18n.t` kullanmayın.
