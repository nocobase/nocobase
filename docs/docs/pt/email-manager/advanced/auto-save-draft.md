# Auto-salvar Rascunho

No menu do botão **Salvar Rascunho**, você pode ativar o **Auto-salvar rascunho**.

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_29_PM.png)

Durante o processo de edição do e-mail, o sistema salva o rascunho em tempo real, garantindo que nenhum conteúdo seja perdido.

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_31_PM.png)

Ao abrir o editor de e-mail novamente (para o mesmo remetente e destinatário), o sistema perguntará se você deseja usar o rascunho salvo.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_08_35_PM.png)
