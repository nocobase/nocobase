# Envio em Massa

## Visão Geral do Envio em Massa
Você pode enviar e-mails para várias pessoas em massa (cada destinatário receberá um e-mail separado).

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_06_10_PM.png)

## Envio em Massa via Tabela
Você também pode enviar e-mails em massa através de uma tabela:

1. Adicione um botão de envio de e-mail à tabela
   ![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_14_PM.png)

2. Configure o campo de e-mail na tabela (usado para analisar endereços de e-mail após selecionar as linhas da tabela)
   ![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_13_PM.png)

3. Marque os dados da tabela para enviar e-mails
   ![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_15_PM.png)

Se você precisar enviar e-mails para todos os endereços da tabela, pode habilitar a função **Selecionar todos os registros da tabela atual como destinatários por padrão**:
![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_16_PM.png)

Após editar o conteúdo do e-mail, clique no botão de envio em massa para enviar:
![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_17_PM.png)
