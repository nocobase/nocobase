# Assinatura de E-mail

## Configuração da Assinatura

Acesse a Central de Gerenciamento de E-mail e clique no botão de configurações:
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_08_PM.png)

Alterne para o painel de assinaturas:
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_08_PM%20(1)

Aqui você pode gerenciar as assinaturas. Marque a opção antes de uma assinatura para defini-la como padrão:
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_09_PM.png)

## Usando Assinaturas

No corpo do e-mail, clique no ícone de assinatura para selecionar a assinatura desejada:
![](https://static-docs.nocobase.com/email-manager/Loading--12-31-2025_10_11_PM.png)
