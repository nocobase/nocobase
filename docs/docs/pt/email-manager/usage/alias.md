# Alias de e-mail

## Sobre o recurso

O alias de e-mail permite que você envie mensagens a partir da mesma conta usando identidades de remetente diferentes.

Ao enviar, é possível escolher no seletor de remetente o e-mail principal ou qualquer endereço de alias já sincronizado. Em respostas, encaminhamentos e ao retomar rascunhos, o sistema preserva a identidade de remetente original.

![](https://static-docs.nocobase.com/%E5%8F%91%E9%80%81%E9%82%AE%E4%BB%B6-04-02-2026_06_02_PM.png)

> O Outlook não suporta este recurso.

## Sincronização de aliases

Após autorizar uma conta Gmail, o sistema sincroniza automaticamente os aliases disponíveis daquela caixa de e-mail.

Se você adicionar ou ajustar aliases no Gmail posteriormente, abra as configurações do e-mail e, em "Nome do remetente", clique em "Sincronizar aliases" para puxá-los novamente.

![](https://static-docs.nocobase.com/Email-settings-04-02-2026_06_04_PM.png)

## Selecionando um alias ao enviar

No editor de e-mail, clique no seletor de remetente para ver o e-mail principal e os aliases sincronizados daquela conta.

Se o mesmo endereço de alias estiver associado a várias contas, o seletor exibe a caixa principal correspondente, ajudando a distinguir o contexto de qual conta está em uso.

![](https://static-docs.nocobase.com/%E5%8F%91%E9%80%81%E9%82%AE%E4%BB%B6-04-02-2026_06_02_PM.png)
