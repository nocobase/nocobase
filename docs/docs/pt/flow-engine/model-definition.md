# Campo de Região da China

## Introdução

O plugin Campo de Região da China (`field-china-region`) adiciona o tipo de campo "Região da China" ao NocoBase.

## Instalação

Na interface principal do NocoBase, vá para o "Gerenciador de Plugins", encontre o plugin "Região da China" e habilite-o.

## Uso

Na configuração da coleção, adicione um novo campo e selecione "Região da China" como o tipo de campo.

![](/images/field-china-region-1.png)

### Configuração do Campo

Na configuração do campo, você pode definir o modo de exibição para o campo:

- Província
- Província/Cidade
- Província/Cidade/Distrito
- Província/Cidade/Distrito/Condado

![](/images/field-china-region-2.png)
