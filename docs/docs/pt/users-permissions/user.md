# Usuários

## Gerenciamento de Usuários

Este **plugin** oferece uma interface de gerenciamento de **usuários** no centro de configuração. Após a inicialização do aplicativo, uma conta de superadministrador, que não pode ser excluída, é criada automaticamente. O administrador possui permissões de Root e pode acessar todos os recursos.

![](https://static-docs.nocobase.com/44bf40f56b45d4dd96c424fb08082cf6.png)

Além disso, você também pode usar a **coleção** de **usuários** nas páginas para adicionar diversos blocos e gerenciar os **usuários**, como blocos de tabela:

![](https://static-docs.nocobase.com/76b5a4652f869541a9e8f18a4568a7c9.png)
