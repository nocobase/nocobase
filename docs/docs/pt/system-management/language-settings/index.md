# Configurações de Idioma

Em "Configurações do Sistema", você pode ajustar o ambiente de idioma global do sistema configurando os "Idiomas Ativados". O primeiro idioma da lista será usado como o idioma padrão.

![20240324192005](https://static-docs.nocobase.com/20240324192005.png)

Quando vários idiomas estão ativados, os usuários podem definir sua preferência de idioma pessoal na Central Pessoal (canto superior direito da página).

![20240324192509](https://static-docs.nocobase.com/20240324192509.png)

Se o ambiente de idioma que você está usando não tiver traduções, você pode usar o [plugin de Gerenciamento de Localização](/system-management/localization/) para traduzir e complementar.

![20240324193055](https://static-docs.nocobase.com/20240324193055.png)
