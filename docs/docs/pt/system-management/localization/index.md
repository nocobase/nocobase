# Gerenciamento de Localização

## Introdução

O **plugin** de Gerenciamento de Localização é usado para gerenciar e implementar os recursos de localização do NocoBase. Com ele, você pode traduzir menus do sistema, **coleções**, campos e todos os **plugins** para se adaptar à língua e cultura de regiões específicas.

## Instalação

Este **plugin** é integrado e não requer instalação adicional.

Atualmente, você pode sincronizar o seguinte conteúdo:

- Pacotes de idiomas locais para o sistema e **plugins**
- Títulos de **coleções**, títulos de campos e rótulos de opções de campo
- Títulos de menu

Após a sincronização, o sistema listará todos os itens traduzíveis para o idioma atual.
