# Personalizar Marca

## Introdução

O nome da marca padrão é "NocoBase", e ele é exibido principalmente em três locais:

![20240409111100](https://static-docs.nocobase.com/20240409111100.png)

Na parte inferior da página de login:

![20240409111254](https://static-docs.nocobase.com/20240409111254.png)

A Posição 1 pode ser configurada através da opção de Logo em [Configurações do Sistema](/system-management/system-settings/). As Posições 2 e 3 podem ser configuradas usando este plugin.

## Instruções de Uso

Após ativar o plugin, clique no menu "Personalizar Marca" no canto superior direito da página para acessar a página de configuração.

![20240409113640](https://static-docs.nocobase.com/20240409113640.png)

- **Marca**: Usado para definir o texto na parte inferior da página de login (posição 3)
- **Sobre**: Usado para configurar o conteúdo do menu no canto superior direito da página (conforme mostrado na imagem abaixo)

![20240409113915](https://static-docs.nocobase.com/20240409113915.png)

A configuração suporta o formato HTML, o que facilita a personalização flexível do conteúdo. Se você precisar apenas fazer alterações simples no texto, pode modificar diretamente o conteúdo existente, como mostrado abaixo:
