# Gerenciamento de Plugins

## Gerenciador de Plugins

Clique no ícone do gerenciador de plugins no canto superior direito da página para acessar a página do gerenciador de plugins. Nesta página, você pode visualizar, instalar, ativar ou desativar plugins.

![2024-04-10_08-47-45](https://static-docs.nocobase.com/2024-04-10_08-47-45.png)

## Central de Configurações de Plugins

Plugins geralmente vêm com suas próprias páginas de configuração. Clique no ícone de configurações no canto superior direito para visualizar os menus de configuração de todos os plugins. Clique no item de menu correspondente para configurar o plugin.
