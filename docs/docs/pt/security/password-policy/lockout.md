# Bloqueio de Usuário

## Introdução

Gerencie os usuários bloqueados e bloqueie usuários de forma proativa.

![](https://static-docs.nocobase.com/202412281450083.png)

### Desbloquear Usuário

Usuários que foram bloqueados por exceder o limite de tentativas de login com senha inválida podem ser desbloqueados ao excluir o registro.

### Bloquear Usuário

Adicionar um registro de usuário permite que você bloqueie um usuário de forma proativa. Uma vez que um usuário é bloqueado, ele será impedido de acessar o sistema por qualquer método de autenticação, incluindo chaves de API.

![](https://static-docs.nocobase.com/202412281450512.png)
