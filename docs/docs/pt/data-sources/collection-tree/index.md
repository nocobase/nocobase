# Coleção em Árvore

## Introdução

Uma coleção em estrutura de árvore é um padrão de design de coleção de dados usado para organizar dados hierárquicos. Essa estrutura de coleção se assemelha a uma árvore, onde cada item de dado pode ter um ou mais itens filhos, e esses itens filhos, por sua vez, podem ter seus próprios descendentes.
