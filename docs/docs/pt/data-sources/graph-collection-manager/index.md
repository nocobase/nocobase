# Gerenciamento visual de tabelas de dados

## Introdução

Uma ferramenta semelhante a um diagrama ER, que atualmente é compatível apenas com o banco de dados principal.

## Instalação

É um plug-in pré-instalado que precisa ser ativado antes do uso.
