# Seletor de coleção

O campo Seletor de coleção é um tipo de campo especial que permite aos usuários selecionar uma ou mais coleções de uma lista predefinida. Este tipo de campo é muito útil em cenários onde você precisa referenciar dinamicamente diferentes coleções, como na criação de relatórios configuráveis, painéis (dashboards) ou aplicações complexas que precisam alterar as fontes de dados com base na seleção do usuário.

## Casos de Uso

### Geração Dinâmica de Relatórios

Em um módulo de geração de relatórios, os usuários podem precisar selecionar uma coleção entre várias coleções como a fonte de dados para o relatório. Ao usar o Seletor de coleção, os usuários podem selecionar facilmente a coleção desejada, e o sistema gerará dinamicamente o relatório com base na seleção do usuário.

### Painéis Configuráveis

Em uma aplicação de painel, os usuários podem personalizar os dados exibidos no painel. O Seletor de coleção permite que os usuários escolham as coleções de seu interesse, exibindo assim gráficos e dados estatísticos de diferentes fontes de dados no painel.

### Ferramentas Flexíveis de Importação/Exportação de Dados

Ao desenvolver uma ferramenta genérica de importação ou exportação de dados, você pode usar o Seletor de coleção para permitir que os usuários especifiquem a coleção a ser operada. Dessa forma, a mesma ferramenta pode ser usada para processar dados de qualquer coleção no sistema, melhorando significativamente a reutilização da ferramenta.
