# Fonte de Dados Externa - PostgreSQL

## Introdução

Utilize um banco de dados PostgreSQL externo como uma fonte de dados. Versões atualmente suportadas: PostgreSQL >= 9.5

## Instalação

Este plugin é comercial e precisa ser carregado e ativado através do gerenciador de plugins.
