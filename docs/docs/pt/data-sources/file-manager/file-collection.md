# Coleção de Arquivos

## Introdução

Arquivos são registros em uma coleção com uma estrutura especial, conhecida como coleção de arquivos. Ela é usada para armazenar metadados de arquivos e pode ser gerenciada através do Gerenciador de Arquivos.
