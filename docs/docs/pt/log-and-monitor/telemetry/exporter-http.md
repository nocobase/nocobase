# Exportador de Telemetria: HTTP

## Variáveis de Ambiente

### TELEMETRY_METRIC_READER

Tipo de exportador de métricas de telemetria.

```bash
TELEMETRY_METRIC_READER=http
```

### TELEMETRY_HTTP_URL

URL HTTP para exportação de dados de telemetria.

```bash
TELEMETRY_HTTP_URL=
```
