# Funcionário AI · Nathan: Engenheiro Frontend

> Ajuda você a escrever e otimizar código frontend para implementar lógicas interativas complexas.

`Nathan` é o especialista em desenvolvimento frontend no NocoBase. Em cenários que exigem a escrita de JavaScript, como `JSBlock`, `JSField`, `JSItem`, `JSColumn`, `JSAction`, `Event Flow` e `Linkage`, você encontrará o avatar do `Nathan` no canto superior direito do editor de código, e poderá pedir para ele ajudar a escrever ou modificar o código no editor.

## Passos

No editor de código, clique em `Nathan` para abrir a caixa de diálogo do Funcionário AI. O código do editor será automaticamente anexado à caixa de entrada e enviado ao `Nathan` como contexto da aplicação.

![20251022163935](https://static-docs.nocobase.com/20251022163935.png)

Insira suas necessidades de codificação e envie para o `Nathan`, então espere pela resposta dele.

![20251022164041](https://static-docs.nocobase.com/20251022164041.png)

Clique no botão `Apply to editor` no bloco de código da resposta do `Nathan` para sobrescrever o código no editor de código.

![20251022164323](https://static-docs.nocobase.com/20251022164323.png)
