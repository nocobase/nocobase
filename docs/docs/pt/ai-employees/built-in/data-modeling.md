# Agente de IA · Orin: Modelador de Dados

> Projete coleções de forma inteligente e otimize a estrutura do banco de dados.

`Orin` é um especialista em modelagem de dados. Na página de configuração da **fonte de dados** principal, você pode pedir para o `Orin` ajudar você a criar ou modificar **coleções**.

![20251022160628](https://static-docs.nocobase.com/20251022160628.png)

## Passos

Acesse o **plugin** de Gerenciamento de **Fontes de Dados** e selecione para configurar a **fonte de dados** principal.

![20251022161146](https://static-docs.nocobase.com/20251022161146.png)

Clique no avatar do `Orin` no canto superior direito para abrir a caixa de diálogo do Agente de IA.

![20251022161641](https://static-docs.nocobase.com/20251022161641.png)

Descreva suas necessidades de modelagem para o `Orin`, envie e aguarde uma resposta.

![20251022161749](https://static-docs.nocobase.com/20251022161749.png)

Depois que o `Orin` confirmar suas necessidades, ele usará suas habilidades e responderá com uma prévia do modelo de dados.

Após revisar a prévia, clique no botão `Finish review and apply` para criar **coleções** de acordo com o modelo do `Orin`.

![20251022162142](https://static-docs.nocobase.com/20251022162142.png)
