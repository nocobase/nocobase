# Gerenciamento de Arquivos

## Introdução

Configure como os arquivos enviados durante as conversas com o funcionário de IA serão armazenados.

## Configurações

Acesse a página de configuração do plugin do funcionário de IA e clique na aba `Settings` para entrar na página de configurações do método de armazenamento.

![20251022002648](https://static-docs.nocobase.com/20251022002648.png)

As opções de armazenamento disponíveis para `Files storage` são configuradas no plugin de Gerenciamento de Arquivos.
