# Início Rápido

## Introdução
Após configurar o serviço LLM, você pode começar a criar funcionários de IA.

## Novo Funcionário de IA

### Configuração de Informações Básicas

Vá para a página de configuração do **plugin** de funcionários de IA, clique na aba `AI employees` para acessar a página de gerenciamento de funcionários de IA.

![20251021225643](https://static-docs.nocobase.com/20251021225643.png)

Clique no botão `New AI employee` para entrar na página de criação de funcionários de IA.

Primeiro, na aba `Profile`, você pode configurar as informações básicas do funcionário de IA:

- No campo `Username`, insira o identificador único para o funcionário de IA;
- No campo `Nickname`, insira o apelido do funcionário de IA;
- No campo `Position`, insira o cargo do funcionário de IA, usado para descrever brevemente suas responsabilidades;
- Em `Avatar`, selecione um avatar para o funcionário de IA.

![20251021232246](https://static-docs.nocobase.com/20251021232246.png)

- No campo `Bio`, insira o texto de introdução para o funcionário de IA, descrevendo suas habilidades e funções. Este texto será exibido no perfil do funcionário de IA;
- No campo `About me`, insira uma descrição detalhada das funções do funcionário de IA. Este texto será usado como um prompt de sistema ao conversar com o funcionário de IA.
- No campo `Greeting message`, insira a mensagem de saudação do funcionário de IA. Sempre que um usuário iniciar uma nova conversa com o funcionário de IA, o sistema enviará automaticamente esta mensagem ao usuário.

![20251021232712](https://static-docs.nocobase.com/20251021232712.png)

### Configuração do Modelo de Linguagem

Na aba `Model settings`, você pode configurar as definições do modelo para o funcionário de IA:

- Selecione o serviço LLM que você criou no gerenciamento de serviços LLM;
- Insira o nome do modelo de linguagem que você deseja usar.

![20251021231727](https://static-docs.nocobase.com/20251021231727.png)

### Concluir Criação

Após inserir todas as informações necessárias para criar o funcionário de IA, clique no botão `Submit` para concluir a criação.

Em seguida, você poderá ver o funcionário de IA recém-criado na lista de funcionários de IA. E no botão de acesso rápido do funcionário de IA, no canto inferior direito da página, você também poderá ver o funcionário de IA que acabou de criar.

![20251021232836](https://static-docs.nocobase.com/20251021232836.png)
