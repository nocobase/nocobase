# Pesquisa na Web

A Pesquisa na Web é usada para complementar o conhecimento do modelo com informações atualizadas além dos dados de treinamento.

## Como funciona

A disponibilidade da Pesquisa na Web depende se o serviço de modelo selecionado na sessão atual suporta Web Search.

- **Suportado:** o interruptor de pesquisa na web é exibido e pode ser ativado ou desativado conforme necessário.
- **Não suportado:** o interruptor não é exibido e o estado de pesquisa é desativado automaticamente.

## Uso na conversa

Use o interruptor de pesquisa na web na área de entrada da conversa:

- **Ativado:** a IA pode realizar uma busca antes de responder.

![web-search-switch-visible-on-supported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-visible-on-supported-model.png)

- **Desativado:** a IA responde apenas com base no contexto existente.

![web-search-switch-hidden-on-unsupported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-hidden-on-unsupported-model.png)

## Diferenças entre plataformas

A capacidade de Pesquisa na Web varia entre os diferentes modelos. Utilize-a de acordo com a situação real do serviço e modelo selecionados.
