# Gerar Relatório de Análise de Negócios

Tomando como exemplo o sistema CRM da DEMO online, vamos pedir ao Funcionário de IA que analise os dados de clientes e pedidos no sistema e gere um relatório de análise.

## Carregar Skills

Primeiro, o Funcionário de IA carrega os Skills relacionados e, em seguida, começa a trabalhar.

![](https://static-docs.nocobase.com/202604230924035.png)

## Analisar o Modelo de Dados

Obtém as fontes de dados e as informações de metadados das coleções.

![](https://static-docs.nocobase.com/202604230926297.png)

## Consultar Dados de Negócio

Inicia a consulta dos dados de negócio. Dependendo do conteúdo da análise, esse processo geralmente leva de algumas dezenas de segundos a alguns minutos.

![](https://static-docs.nocobase.com/202604230927861.png)

## Gerar Relatório

Após coletar dados de negócio suficientes, o Funcionário de IA começa a gerar o relatório de análise.

![](https://static-docs.nocobase.com/202604230920113.png)

Por fim, é gerado um relatório de análise de negócio com texto e gráficos, com suporte para download nos formatos Markdown, HTML e PDF.

![](https://static-docs.nocobase.com/202604230922030.png)

![](https://static-docs.nocobase.com/202604230922376.png)

![](https://static-docs.nocobase.com/202604230923542.png)

![](https://static-docs.nocobase.com/202604230923538.png)

![](https://static-docs.nocobase.com/202604230923049.png)

Atenção: se você baixar no formato Markdown, os gráficos precisam ser baixados separadamente na página da web.
