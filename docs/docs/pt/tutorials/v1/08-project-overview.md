## Roadmap futuro

### 1. Centro de base de conhecimento

Imagine sua equipe acumulando documentos, experiência e conhecimento — como gerenciar isso tudo de forma eficiente? Vamos construir um centro de gestão de conhecimento multifuncional:

- Estrutura em árvore para os documentos, organizando tudo nitidamente [(tabela em árvore)](https://docs.nocobase.com/handbook/collection-tree)
- Suporte a múltiplos tipos de documentos: documentação técnica, wikis da equipe, comunicados
- Controle de acesso flexível, garantindo segurança das informações (permissões, status, regras)

### 2. Dashboard personalizado

Cada um é único e merece seu espaço próprio:

- Área de foco em tarefas: destacar tarefas importantes e urgentes (filtros)
- Gráficos estatísticos: mostrar o progresso de forma clara [(gráficos)](https://docs-cn.nocobase.com/handbook/data-visualization/user/chart-block)
- Exibição personalizada: customize o que importa para você (biblioteca de documentos/comunicados)

### 3. Gerenciamento de subtarefas

Algumas tarefas são como um elefante: precisam ser comidas pedaço a pedaço:

- Decomposição de tarefas: divida tarefas complexas em pequenas
- Acompanhamento de progresso: veja o todo e cada parte com clareza
- Hierarquia: suporte a múltiplos níveis de subtarefas

### 4. Acompanhamento de horas e progresso (opcional)

Mantenha o andamento do projeto à vista:

- Apontamento diário de horas
- Registro de progresso
- Análise estatística de horas
- Visualização do progresso

### 5. Gerenciamento de salas de reunião (opcional)

Acabe com o peso das reuniões:

- Agenda das salas
- Reservas
- ...

### 6. Evoluir para uma plataforma completa de gerenciamento de projetos

Este é o objetivo final: integrar tudo em uma plataforma poderosa, atendendo equipes pequenas a grandes. Conforme avançarmos, podemos adicionar:

- Análise de custos
- Análise de projetos
- ...
