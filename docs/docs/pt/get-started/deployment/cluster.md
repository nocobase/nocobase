# Implantação em Cluster

O NocoBase permite implantar aplicações em modo de cluster para aumentar o desempenho e a disponibilidade da aplicação. A implantação em cluster permite executar diferentes instâncias da aplicação em múltiplos servidores, ou executar múltiplos processos em modo multi-core em um único servidor.

Para mais detalhes, consulte a documentação sobre [Modo de Cluster](/cluster-mode/).
