# Notificação: WeCom

## Introdução

O **plugin WeCom** permite que o aplicativo envie mensagens de notificação para usuários do WeCom.

## Adicionar e configurar o autenticador WeCom

Primeiro, você precisa adicionar e configurar um autenticador WeCom no NocoBase. Consulte [Autenticação de Usuário - WeCom](/auth-verification/auth-wecom). Apenas usuários do sistema que fizeram login via WeCom podem receber notificações do sistema através do WeCom.
