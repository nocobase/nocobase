# Filtro

## Introdução

A ação de filtro geralmente aparece em blocos de dados para filtrar dados configurando várias condições. Observe que, se um escopo de dados já estiver configurado para o bloco, a ação de filtro será combinada com as condições do escopo de dados como a condição de filtragem final.

### Campos Filtráveis

Por padrão, todos os campos desta coleção estão habilitados. Apenas os campos habilitados podem ser selecionados na lista de campos de condição.

Campos de coleções associadas também podem ser usados como campos de filtro.
