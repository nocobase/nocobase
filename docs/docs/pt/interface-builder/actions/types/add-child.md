# Adicionar Registro Filho

## Introdução

Esta operação foi projetada especificamente para **blocos de tabela em árvore** para adicionar registros filhos rapidamente. Ao adicionar um registro filho, a linha selecionada é tratada como o nó pai, permitindo que você crie registros filhos facilmente sob qualquer pai. Este recurso simplifica bastante o gerenciamento de dados hierárquicos, tornando a expansão e organização das estruturas de dados mais eficiente e intuitiva.
