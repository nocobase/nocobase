# Abrir Modal

## Introdução

A **ação Abrir Modal** é similar às ações de Visualizar e Editar. Você pode configurar um botão **Abrir Modal** nas linhas ou no cabeçalho de uma tabela. Ao clicar no botão, um modal correspondente será exibido, e os blocos dentro deste modal mostrarão dados com base no registro de contexto atual.

![20251029182351](https://static-docs.nocobase.com/20251029182351.png)

Configure os blocos no modal conforme necessário.
