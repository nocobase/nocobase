# Edição em Massa

## Introdução

A Edição em Massa é adequada para cenários que exigem atualizações flexíveis de dados em lote. Após clicar no botão de Edição em Massa, você pode configurar o formulário de edição em massa em uma janela pop-up e definir diferentes estratégias de atualização para cada campo.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM.png)

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM%20(1).png)


## Guia do Usuário

### Configuração do Formulário de Edição em Massa

1. Adicione um botão de Edição em Massa.

2. Defina o escopo da edição em massa: Selecionados / Todos, o padrão é Selecionados.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_14_AM.png)

3. Adicione um formulário de edição em massa.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_14_AM.png)

4. Configure os campos a serem editados e adicione um botão de envio.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM%20(1).png)

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM.png)

### Envio do Formulário

1. Selecione as linhas de dados que deseja editar.

2. Selecione o modo de edição para os campos e preencha os valores a serem enviados.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_10_33_AM.png)

:::info{title="Modos de Edição Disponíveis"}
* **Não atualizar**: O campo permanece inalterado.
* **Alterar para**: Atualiza o campo com o valor enviado.
* **Limpar**: Limpa os dados do campo.

:::

3. Envie o formulário.
