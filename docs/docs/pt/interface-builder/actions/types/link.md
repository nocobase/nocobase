# Link

## Introdução

A ação de link utiliza a navegação de rota, permitindo a passagem de variáveis de URL. Isso possibilita que a página de destino ajuste seu conteúdo dinamicamente com base nos dados recebidos. Você pode configurá-la em blocos de dados.
