# Confirmação

## Introdução

A configuração de confirmação é usada para verificar se uma ação deve ser executada. Um diálogo de confirmação aparece quando você clica na ação. O botão de exclusão, por exemplo, já vem com a confirmação ativada por padrão para evitar perda de dados ou resultados irreversíveis causados por operações acidentais.

Você pode personalizar o conteúdo e o título da confirmação, e também acessar variáveis.
