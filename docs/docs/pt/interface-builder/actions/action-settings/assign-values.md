# Atribuição de Campo

## Introdução

A atribuição de campo se refere à predefinição do valor de um campo antes de executar uma ação. Ao atualizar ou criar um registro, esses dados são preenchidos automaticamente, sem a necessidade de inserção manual. Isso é útil em cenários como a atualização em massa de status de pedidos ou a definição da hora de entrega para a hora atual quando um pedido é entregue.
