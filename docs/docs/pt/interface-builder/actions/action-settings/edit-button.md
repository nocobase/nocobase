# Editar Botão

## Introdução

O sistema permite personalizar o nome, ícone, cor de fundo e tipo do botão, oferecendo opções flexíveis de customização da interface do usuário (UI) para ajudar você a ajustar o estilo de exibição e a funcionalidade do botão de acordo com suas necessidades específicas.

- **Nome do botão**: Você pode personalizar o texto do botão de acordo com as necessidades do seu negócio, garantindo que o rótulo seja claro e intuitivo.
- **Ícone do botão**: É possível definir um ícone para o botão, o que melhora o reconhecimento visual e permite que você identifique rapidamente a função do botão.
- **Ação de Perigo (Danger Action)**: Quando um botão representa uma **ação de perigo** (como excluir, redefinir, etc.), ele pode ser configurado como do tipo **Danger Action**.
- **Tipo de Botão (Button Type)**: Os botões suportam diferentes configurações de tipo. Os tipos de botão comuns incluem:
  - **Padrão (Default)**: Um botão padrão, ideal para operações normais.
  - **Primário (Primary)**: Usado para ações importantes ou principais, geralmente com um efeito visual de destaque.
  - **Tracejado (Dashed)**: Um botão com borda tracejada, adequado para ações secundárias ou botões de dica.
  - **Texto (Text)**: Um botão que contém apenas texto e não possui cor de fundo, geralmente usado para ações leves ou links.
