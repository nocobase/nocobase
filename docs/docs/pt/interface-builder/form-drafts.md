# Rascunhos de Formulário

Quando os rascunhos de formulário estão ativados, o conteúdo que está sendo preenchido é salvo automaticamente localmente. Mesmo que você feche o formulário acidentalmente ou atualize a página, os dados preenchidos não serão perdidos.

![20251213093717](https://static-docs.nocobase.com/20251213093717.png)

Quando um rascunho existe, uma notificação será exibida no topo do formulário. O rascunho será excluído automaticamente após o envio bem-sucedido; se não for mais necessário, você também pode excluí-lo manualmente a qualquer momento.

![20251213092734](https://static-docs.nocobase.com/20251213092734.png)

::: tip Dica

Atualmente, os rascunhos de formulário suportam apenas o cache local. Versões futuras suportarão o cache no servidor para permitir a sincronização entre vários dispositivos.

:::
