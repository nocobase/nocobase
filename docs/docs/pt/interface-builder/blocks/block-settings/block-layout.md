# Layout

## Introdução

Blocos de formulário, como Blocos de Formulário, Blocos de Detalhes e Blocos de Lista, suportam configurações de layout. As opções de layout incluem o layout vertical (onde rótulos e campos de entrada são organizados linha por linha) e o layout horizontal (onde rótulos e campos de entrada são colocados lado a lado). Você pode ajustar o layout dos campos do bloco conforme suas necessidades específicas, garantindo que a entrada ou visualização de informações seja clara e eficiente.

### Vertical (Padrão)

O layout padrão é vertical.

![20251024120143](https://static-docs.nocobase.com/20251024120143.png)

### Horizontal

O layout horizontal exibe rótulos e campos de entrada lado a lado, o que economiza espaço de forma eficaz. Para campos com títulos longos, você pode optar por configurá-los para quebrar linha ou mostrar reticências, garantindo uma interface limpa e esteticamente agradável.

![20251024120237](https://static-docs.nocobase.com/20251024120237.png)

#### Alinhamento do Título do Campo

- **Alinhar à esquerda** (padrão)
- **Alinhar à direita**

#### Largura do Título do Campo

- A largura padrão é de **120px**, com uma largura mínima de **50px**.

#### Tratamento para Títulos de Campo que Excedem a Largura

- **Quebrar linha**
- **Reticências**

#### Exibir Dois Pontos

Você pode escolher se deseja exibir dois pontos após o título do campo.
