# Modo de carregamento de dados

## Introdução

O modo padrão de carregamento de dados é "Carregar todos os dados quando as condições de filtro estiverem vazias". Aqui, "condições de filtro" refere-se aos valores no "Bloco de filtro". Outra opção disponível é "Não carregar dados quando as condições de filtro estiverem vazias".
