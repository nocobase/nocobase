# Bloco

Blocos são os portadores de dados e conteúdo. Eles podem ser colocados em uma Página (Page), Modal (Modal) ou Gaveta (Drawer), e vários blocos podem ser arrastados e organizados livremente.

## Tipos de Bloco

![Tipos de Bloco](https://static-docs.nocobase.com/f71af45b5cd914ea0558f760ddbbba58.png)

- Bloco de Dados: Usado para exibir dados de uma **fonte de dados** na interface.
- Bloco de Filtro: Usado para usar dados de uma **fonte de dados** como condições de filtro para filtrar outros blocos de dados.
- Outro Bloco: Usado para conter conteúdo específico ou independente, como itens pendentes de **fluxo de trabalho**, logs de auditoria e Markdown.

## Adicionar Bloco

Blocos podem ser colocados em uma Página (Page), Modal (Modal) ou Gaveta (Drawer).

### Blocos em uma Página

Atualmente, os tipos de bloco em uma página incluem: Bloco de Dados, Bloco de Filtro e Outro Bloco.

![20251023222441](https://static-docs.nocobase.com/20251023222441.png)

### Blocos em um Popup (Modal ou Gaveta)

Popups vêm em dois tipos: modais e gavetas. Assim como as páginas, eles também podem ter blocos adicionados. A diferença é que os blocos de formulário em popups são geralmente para adicionar, editar ou visualizar um único registro. Os tipos de bloco incluem Bloco de Dados e Outro Bloco.

![20251023222613](https://static-docs.nocobase.com/20251023222613.png)

## Designer de Bloco

Cada bloco tem três pequenos ícones no canto superior direito. Da esquerda para a direita, eles são:

1.  Layout de Arrastar e Soltar
2.  [Fluxo de Eventos](/interface-builder/event-flow)
3.  Configurações do Bloco

![20251023224032](https://static-docs.nocobase.com/20251023224032.png)

Para blocos simples, todas as opções de configuração estão centralizadas em 'Configurações do Bloco', como o Bloco JS.

![20251023224903](https://static-docs.nocobase.com/20251023224903.png)

Blocos de dados complexos também fornecem 'Configurar campos' e 'Configurar ações' separados e incorporados.

![20251023225141](https://static-docs.nocobase.com/20251023225141.png)

Além disso, há flexibilidade para mais possibilidades de aninhamento, como visto no Bloco de Gráfico.

![](https://static-docs.nocobase.com/07588190b3f41ae3060e71d8b76b4447.png)
