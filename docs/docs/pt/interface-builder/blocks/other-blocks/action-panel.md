# Bloco: Painel de Ações

## Introdução

Usado para organizar diversas ações rápidas. Atualmente, ele oferece suporte para:

- [Link](/interface-builder/actions/types/link)
- [Escanear QR Code]
- [Ação de Pop-up](/interface-builder/actions/types/pop-up)

![20251023225754](https://static-docs.nocobase.com/20251023225754.png)

> 💡 Continuaremos adicionando mais tipos de ações nas próximas versões.
## Tipos de Ação

- **Link**: Navega para uma página específica ou URL externa.

![20251023230643](https://static-docs.nocobase.com/20251023230643.png)

- **Escanear QR Code**: Aciona uma ação correspondente ao escanear um QR Code.

- **Abrir Pop-up**: Abre um pop-up para exibir o conteúdo do bloco.
  ![20251023230522](https://static-docs.nocobase.com/20251023230522.png)
