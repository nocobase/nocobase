# Bloco de Lista

## Introdução

O Bloco de Lista exibe dados em formato de lista, ideal para cenários de exibição de dados como listas de tarefas, notícias e informações de produtos.

## Configuração do Bloco

![20251023202835](https://static-docs.nocobase.com/20251023202835.png)

### Definir Escopo de Dados

Como mostra a imagem: Filtre por pedidos com status "Cancelado".

![20251023202927](https://static-docs.nocobase.com/20251023202927.png)

Para saber mais, consulte [Definir Escopo de Dados](/interface-builder/blocks/block-settings/data-scope)

### Definir Regras de Ordenação

Como mostra a imagem: Ordene pelo valor do pedido em ordem decrescente.

![20251023203022](https://static-docs.nocobase.com/20251023203022.png)

Para saber mais, consulte [Definir Regras de Ordenação](/interface-builder/blocks/block-settings/sorting-rule)

## Configurar Campos

### Campos desta Coleção

> **Observação**: Campos de coleções herdadas (ou seja, campos da coleção pai) são automaticamente mesclados e exibidos na lista de campos atual.

![20251023203103](https://static-docs.nocobase.com/20251023203103.png)

### Campos de Coleções Relacionadas

> **Observação**: Campos de coleções relacionadas podem ser exibidos (atualmente, apenas relacionamentos um-para-um são suportados).

![20251023203611](https://static-docs.nocobase.com/20251023203611.png)

Para a configuração dos campos da lista, consulte [Campo de Detalhes](/interface-builder/fields/generic/detail-form-item)
