# Cartão em Grade

## Introdução

O bloco Cartão em Grade exibe informações resumidas de registros de dados em formato de cartão. Ele permite configurar o número de colunas para diferentes tamanhos de tela para garantir uma exibição amigável em diversos dispositivos.
