# Campos de Detalhes

## Introdução

As configurações de campo para blocos de Detalhes, Lista e Grade são basicamente as mesmas. Elas controlam principalmente como os campos são exibidos no modo de leitura.

![20251025172851](https://static-docs.nocobase.com/20251025172851.png)

## Opções de Configuração de Campo

### Formatação de Campo de Data

![20251025173005](https://static-docs.nocobase.com/20251025173005.png)

Para mais detalhes, consulte [Formatação de Data](/interface-builder/fields/specific/date-picker)

### Formatação de Campo Numérico

![20251025173242](https://static-docs.nocobase.com/20251025173242.png)

Suporta conversão de unidades simples, separadores de milhares, prefixos e sufixos, precisão e notação científica.

Para mais detalhes, consulte [Formatação Numérica](/interface-builder/fields/field-settings/number-format)

### Habilitar Clique para Abrir

Além dos campos de relacionamento, campos comuns também podem ser configurados para abrir um pop-up ao clicar. Você também pode definir como o pop-up será aberto (Gaveta, Diálogo ou Subpágina).

![20251025173549](https://static-docs.nocobase.com/20251025173549.gif)

### Modo de Exibição de Conteúdo Extravasado

Quando o conteúdo do campo extravasar sua largura, você pode definir o modo de extravasamento.

- Reticências (padrão)
- Quebra de linha

![20251025173917](https://static-docs.nocobase.com/20251025173917.png)

### Componente de Campo

Alguns campos suportam múltiplos formatos de exibição, o que pode ser alcançado alternando o componente de campo.

Por exemplo: O componente `URL` pode ser alternado para o componente `Preview`.

![20251025174042](https://static-docs.nocobase.com/20251025174042.png)

Por exemplo, campos de relacionamento podem ser exibidos de forma diferente, alternando do componente de campo de título para `Sub-detalhes` para mostrar mais conteúdo do relacionamento.

![20251025174311](https://static-docs.nocobase.com/20251025174311.gif)
