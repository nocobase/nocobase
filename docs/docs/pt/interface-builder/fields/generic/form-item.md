# Campos de Formulário

## Introdução

Os diferentes tipos de campos em um formulário possuem opções de configuração variadas. Isso permite que você, através de `plugins`, estenda essas opções para atender a necessidades mais personalizadas.

![20251025174555](https://static-docs.nocobase.com/20251025174555.png)

## Opções de Configuração do Campo

### Valor Padrão

Você pode configurar constantes ou variáveis como valores padrão para os campos (isso é suportado apenas para novos formulários).

![20251025174724](https://static-docs.nocobase.com/20251025174724.png)
