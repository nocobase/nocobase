# Campos da Tabela

## Introdução

Além de permitir ajustar a largura da coluna, o título do campo e a ordenação, os campos da tabela também oferecem configurações de exibição mais personalizadas para tipos específicos de campos, como campos de data, campos de relacionamento e campos numéricos.

![20251024174558](https://static-docs.nocobase.com/20251024174558.png)

## Opções de Configuração de Campo

### Formatação de Campo de Data

![20251024175303](https://static-docs.nocobase.com/20251024175303.png)

Para mais detalhes, consulte [Formatação de Data](/interface-builder/fields/specific/date-picker)

### Formatação de Campo Numérico

Suporta conversão simples de unidades, separadores de milhares, prefixos e sufixos, precisão e notação científica.

![20251024175445](https://static-docs.nocobase.com/20251024175445.png)

Para mais detalhes, consulte [Formatação Numérica](/interface-builder/fields/field-settings/number-format)

### Habilitar Edição Rápida

Ao habilitar a edição rápida, um botão de edição aparecerá quando você passar o mouse sobre a coluna. Clique nele para editar e salvar os dados rapidamente em uma janela pop-up.

![20251025171158](https://static-docs.nocobase.com/20251025171158.gif)

### Habilitar Clique para Abrir

Além dos campos de relacionamento que permitem abrir uma janela pop-up ao clicar, campos comuns também podem ter a opção "Clique para Abrir" habilitada para servir como um ponto de entrada para abrir uma janela pop-up.

![20251025172308](https://static-docs.nocobase.com/20251025172308.gif)

### Modo de Exibição de Conteúdo Extravasado

Quando o conteúdo da coluna extravasar a largura da coluna da tabela, você pode definir o modo de exibição:

- Reticências (padrão)
- Quebra de linha

### Componente de Campo

Alguns campos suportam múltiplos formatos de exibição. Você pode alternar os componentes de campo para obter diferentes efeitos visuais e atender às necessidades de diversos cenários. Por exemplo, um campo do tipo **URL** pode ser alterado para o componente **Preview** para exibir melhor o conteúdo do link ou pré-visualizações de imagens.

![20251025171658](https://static-docs.nocobase.com/20251025171658.png)

### Ordenável

Atualmente, a maioria dos tipos de campo suporta ordenação no lado do servidor. Quando a ordenação é habilitada, os dados podem ser ordenados em ordem crescente ou decrescente com base no campo de destino.
