# Campo

Na interface do usuário (UI), um campo é um componente de campo que serve como um portador para dados individuais. Diferentes tipos de dados são apresentados com diferentes componentes de campo. Os campos só podem ser anexados a blocos e não podem ser usados de forma independente.

## Campos em um Bloco

Os campos geralmente não são usados sozinhos, mas existem como elementos filhos de blocos de tipo de dados. Blocos de tipo de dados geralmente têm uma opção "Configurar campos", e a lista de campos é fornecida pela coleção atual.

![20251029204240](https://static-docs.nocobase.com/20251029204240.png)

## Designer de Campo (Barra de Ferramentas)

Assim como os blocos, o canto superior direito de um componente de campo também possui três ícones, que são:

- Arrastar para organizar o layout
- [Fluxo de eventos](/interface-builder/event-flow)
- Configuração

![20251025165432](https://static-docs.nocobase.com/20251025165432.png)

## Layout do Campo

Você pode personalizar o layout dos campos dentro de um bloco arrastando e soltando.
