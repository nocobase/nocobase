# Editar Dica de Ferramenta

## Introdução

A dica de ferramenta de um campo é o texto que aparece quando você passa o mouse sobre ele. Ela serve para fornecer explicações ou orientações adicionais sobre o campo. Geralmente, essas dicas incluem informações como a finalidade do campo, o formato de entrada e as restrições, ajudando você a entender e usar melhor o campo.
