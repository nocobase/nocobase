# Componente de Campo

## Introdução

Os componentes de campo servem para exibir o conteúdo de um campo.
Alguns campos já vêm com vários componentes de exibição integrados. Você pode escolher o componente mais adequado às suas necessidades para apresentar o valor do campo de diferentes maneiras.

Exemplo: No modo de visualização, um campo de URL pode ser alternado para um componente de Pré-visualização ou um componente de URL (padrão).
