# Mostrar Rótulo

## Introdução

Por padrão, os campos em um bloco mostram rótulos para identificar o que cada campo significa, o que ajuda os usuários a ler e entender o conteúdo dos dados.

Em alguns cenários (como em páginas de exibição ou quando o significado dos campos já está bem claro), você pode optar por ocultar os rótulos para ter uma interface mais limpa. Ao ocultar os rótulos, apenas os valores dos campos são exibidos, e os nomes dos campos não são mais mostrados.
