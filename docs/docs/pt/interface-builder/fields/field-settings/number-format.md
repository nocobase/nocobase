# Formatação de Números

## Introdução

A formatação de números se refere ao aprimoramento visual e à padronização de campos do tipo numérico (incluindo campos de fórmula) no modo de leitura, incluindo configurações para separadores de milhares, precisão decimal, conversão de unidades e notação científica. Ao configurar a formatação de números, você pode atender aos hábitos de expressão numérica de diferentes regiões e setores.

## Como Usar

- Suporta conversão simples de unidades (multiplicação/divisão)
- Separador de milhares
- Prefixo e sufixo
- Precisão
- Notação científica.

Exemplo: Formate o valor de um pedido adicionando um símbolo de moeda (por exemplo, R$), um separador de milhares (como vírgula ou ponto) e precisão decimal para tornar o valor claro e fácil de ler.
