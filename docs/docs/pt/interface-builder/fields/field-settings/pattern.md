# Modo de Exibição

## Introdução

Diferente dos blocos, os componentes de campo possuem três modos de exibição (suportados apenas para campos dentro de um formulário). Ao alternar entre os diferentes modos de exibição, você verá opções de configuração de campo correspondentes.

- Editável;
- Somente leitura (não editável);
- Somente leitura (modo de visualização);

### Modos de Exibição para Campos de Associação

O **modo de exibição de um campo de associação** determina como o campo é exibido na interface e quais tipos de componentes de campo estão disponíveis.

No **estado editável**, os campos de associação suportam vários tipos de componentes, permitindo que você escolha diferentes componentes de campo de associação para exibir ou selecionar dados relacionados, conforme suas necessidades de negócio.

#### Componentes de Campo de Associação no Estado Editável

![20251028220447](https://static-docs.nocobase.com/20251028220447.png)

Nesse estado, você pode escolher de forma flexível o método de exibição apropriado para lidar com os dados de maneira mais eficiente.

#### Componentes de Campo de Associação no Estado de Somente Leitura

Ao alternar para o **estado de somente leitura**, o sistema usa automaticamente o **componente de campo de título** por padrão para exibir os dados associados. Isso é ideal para cenários onde você precisa apenas visualizar os dados, sem fazer alterações.
