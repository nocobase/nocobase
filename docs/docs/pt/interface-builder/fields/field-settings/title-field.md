# Campo de Título

## Introdução

O **Campo de Título** é geralmente usado em componentes de campo de relacionamento como um identificador para os dados associados exibidos na interface. Ele define como as informações principais dos dados são apresentadas, geralmente para ajudar você a identificar e entender os dados rapidamente.

Diferentes tipos de campos de título correspondem a diferentes componentes de campo e suportam as opções de configuração correspondentes. Por exemplo:

- Quando o campo de título é um **Campo de Data**, o componente de campo usa um **componente de Data** para exibir os dados.
- Quando o campo de título é um **Campo de Texto**, o componente de campo usa um **componente de Texto** para exibir os dados.
- Quando o campo de título é um **Campo de Opção**, o componente de campo usa um **componente de Opção** para exibir os dados.
