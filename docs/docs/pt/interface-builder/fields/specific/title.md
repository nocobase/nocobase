# Título

## Introdução

No modo de leitura, os campos de associação são exibidos por padrão usando o **Componente de Campo de Título**. Ele identifica o registro associado atual mostrando o valor do campo de título. O sistema automaticamente seleciona e renderiza o componente de campo de título apropriado, de acordo com o tipo do campo de título.
