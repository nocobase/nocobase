# Sub-detalhes

## Introdução

Sub-detalhes é o componente correspondente ao Sub-formulário no modo de leitura. Comparado aos componentes de Rótulo e Título, os Sub-detalhes não só podem exibir mais dados da **coleção** atual, mas também podem ser configurados para exibir informações de dados de **coleções** relacionadas. Dados de relacionamento de vários níveis são exibidos claramente na forma de detalhes aninhados.
