# Subformulário

## Introdução

Subformulários são ideais para cenários onde você cria novos dados relacionados antes de vinculá-los. Dados relacionados de múltiplos níveis são exibidos de forma clara como um formulário aninhado. Em comparação com o Seletor de Dados e os componentes de Dropdown, um subformulário permite que você gerencie diretamente os campos da **coleção** relacionada dentro do bloco de página atual, e os dados relacionados são enviados junto com o formulário principal.
