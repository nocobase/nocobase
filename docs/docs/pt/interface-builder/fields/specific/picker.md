# Seletor de Dados

## Introdução

O Seletor de Dados, que aparece como um seletor de tabela em uma janela pop-up, permite que você selecione dados já existentes da **coleção** de destino para associá-los, ou adicione novos dados a essa **coleção** e os associe em seguida.

![20251027203343](https://static-docs.nocobase.com/20251027203343.png)

Os dados já selecionados não serão exibidos no seletor de tabela.

![20251027203717](https://static-docs.nocobase.com/20251027203717.gif)

No seletor de tabela, você também pode gerenciar a **coleção** de destino da associação (adicionar, excluir, importar/exportar, etc.).

![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
