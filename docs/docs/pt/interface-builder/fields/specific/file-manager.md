# Gerenciador de Arquivos

## Introdução

O Gerenciador de Arquivos é um componente de campo de associação usado para lidar com relacionamentos onde a **coleção** de destino é a **coleção** de arquivos.

![20251027194124](https://static-docs.nocobase.com/20251027194124.png)

## Configuração do Campo

- [Componente de Campo](/interface-builder/fields/association-field): Alterne para outros componentes de campo de associação, como Lista Suspensa, Seletor de Dados, etc.
- Permitir selecionar arquivos existentes: Ativado por padrão, permite selecionar e associar arquivos já enviados.
