O NocoBase oferece uma experiência de construção de interface no modelo WYSIWYG (What You See Is What You Get). Clique no botão "Construtor de Interface" para alternar entre o modo de edição e o modo de visualização.

## Layout da interface

O NocoBase oferece layouts para desktop e dispositivos móveis. O layout para desktop atende às operações comuns de administração e também adapta a navegação e o conteúdo das páginas a telas estreitas. O layout para dispositivos móveis oferece navegação e páginas configuradas de forma independente. Consulte [Layout da interface](./ui-layout/index.md) para obter mais informações.

## Opções de Configuração

Ao entrar no modo de edição da interface, você verá indicadores laranjas para os elementos configuráveis. Geralmente, o acesso às configurações de um elemento está localizado no canto superior direito dele. Quase todos os elementos podem ser configurados e ter seus efeitos visualizados em tempo real na interface. Veja alguns exemplos:
