# Construtor de Interface

## WYSIWYG

O NocoBase oferece uma experiência de construção de interface no modelo WYSIWYG (What You See Is What You Get). Clique no botão "Construtor de Interface" para alternar entre o modo de edição e o modo de visualização.

## Modelo de Layout

O NocoBase vem com um modelo de layout padrão: a navegação fica na parte superior e à esquerda, e a área de conteúdo fica à direita.

![未命名.002](https://static-docs.nocobase.com/未命名.002.jpeg)

## Opções de Configuração

Ao entrar no modo de edição da interface, você verá indicadores laranjas para os elementos configuráveis. Geralmente, o acesso às configurações de um elemento está localizado no canto superior direito dele.
