## BelongsToRepository

Sua interface é a mesma que [HasOneRepository](./has-one-repository.md).
`BelongsToRepository` é um `Repository` para lidar com relacionamentos `BelongsTo`. Ele oferece alguns métodos convenientes para gerenciar esses relacionamentos.
