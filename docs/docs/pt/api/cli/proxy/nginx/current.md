# nb proxy nginx current

Mostra o driver atual do provider Nginx.

## Uso

```bash
nb proxy nginx current
```

## Saída

O comando imprime diretamente um destes valores:

- `local`
- `docker`

## Exemplos

```bash
nb proxy nginx current
```
