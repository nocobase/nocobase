# nb proxy caddy start

Inicia o proxy Caddy com o driver atual.

## Uso

```bash
nb proxy caddy start
```

## Exemplos

```bash
nb proxy caddy start
```

## Notas

- Com o driver `local`, este comando inicia o processo local do Caddy
- Com o driver `docker`, este comando inicia ou cria o contêiner Docker
- Se o proxy já estiver em execução, o comando informará isso
