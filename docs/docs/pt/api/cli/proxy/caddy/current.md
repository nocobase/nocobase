# nb proxy caddy current

Mostra o driver atual do provider Caddy.

## Uso

```bash
nb proxy caddy current
```

## Saída

O comando imprime diretamente um destes valores:

- `local`
- `docker`

## Exemplos

```bash
nb proxy caddy current
```
