# nb config list

Lista as configurações do CLI definidas explicitamente. Valores padrão que nunca foram definidos explicitamente não são exibidos.

## Uso

```bash
nb config list
```

## Exemplos

```bash
nb config list
```

## Observações

Se nenhum valor de configuração estiver definido, o CLI exibirá `No CLI config values are set.`.
