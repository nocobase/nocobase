# nb session id

Mostra o id de sessão efetivo atual.

Se não houver um `NB_SESSION_ID` utilizável no shell ou runtime atual, o comando orienta você a executar [`nb session setup`](./setup.md) primeiro e depois abrir um novo shell ou runtime.

## Uso


nb session id

## Exemplos


nb session id

## Comandos relacionados

- [`nb session setup`](./setup.md)
- [`nb env current`](../env/current.md)
