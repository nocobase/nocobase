# nb skills check

Verifica os AI coding Skills globais do NocoBase, informando se eles são gerenciados pelo CLI e se há atualizações disponíveis.

## Uso

```bash
nb skills check [flags]
```

## Parâmetros

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| `--json` | boolean | Saída em JSON |

## Exemplos

```bash
nb skills check
nb skills check --json
```
