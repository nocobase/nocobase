# nb skills remove

Remove os AI coding Skills globais do NocoBase gerenciados pelo `nb`.

## Uso

```bash
nb skills remove [flags]
```

## Parâmetros

| Parâmetro | Tipo | Descrição |
| --- | --- | --- |
| `--yes`, `-y` | boolean | Pula a confirmação de remoção |
| `--json` | boolean | Saída em JSON |
| `--verbose` | boolean | Exibe a saída detalhada da remoção |

## Exemplos

```bash
nb skills remove
nb skills remove --yes
nb skills remove --json
```
