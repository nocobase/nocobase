# Autenticadores

## Gerenciamento de Autenticação de Usuários

Ao instalar o plugin de autenticação de usuários, ele inicializa um método de autenticação de `senha`, que utiliza o nome de usuário e o e-mail do usuário.

![](https://static-docs.nocobase.com/66eaa9d5421c9cb713b117366bd8a5d5.png)

## Ativar Autenticadores

![](https://static-docs.nocobase.com/7f1fb8f8ca5de67ffc68eff0a65848f5.png)

Apenas os tipos de autenticação ativados serão exibidos na página de login.

![](https://static-docs.nocobase.com/8375a36ef98417af0f0977f1e07345dd.png)

## Tipos de Autenticação de Usuários

![](https://static-docs.nocobase.com/da4250c0cea343ebe470cbf7be4b12e4.png)

Ao adicionar diferentes tipos de autenticadores, você pode habilitar os métodos de autenticação correspondentes no sistema.

Além dos tipos de autenticação fornecidos pelos plugins existentes, os desenvolvedores também podem estender os tipos de autenticação de usuários. Para mais detalhes, consulte o [Guia do Desenvolvedor](./dev/).
