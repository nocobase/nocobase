# Multi-space

<PluginInfo name="multi-space" licenseBundled="professional"></PluginInfo>

## Introdução

O plugin **Multi-space** permite múltiplos espaços de dados em uma única instância, com isolamento lógico.

## Instalação

Ative **Multi-Space** no gerenciador de plugins.

![](https://static-docs.nocobase.com/multi-space/Plugin-manager-NocoBase-10-15-2025_09_57_AM.png)

## Guia de uso

### Gestão de espaços

Após ativar, vá em **Users & Permissions** > **Spaces**.

![](https://static-docs.nocobase.com/multi-space/Users-Permissions-NocoBase-10-15-2025_09_58_AM.png)

### Criar espaço

Use **Add space**.

![](https://static-docs.nocobase.com/multi-space/Users-Permissions-NocoBase-10-15-2025_09_58_AM%20(1).png)

### Atribuir usuários

Selecione o espaço e atribua usuários à direita. Atualize a página para refletir no seletor de espaços.

![](https://static-docs.nocobase.com/multi-space/Users-Permissions-NocoBase-10-15-2025_09_59_AM.png)

### Alternar e visualizar

Use o seletor no canto superior direito. O ícone de olho permite visualizar múltiplos espaços.

![](https://static-docs.nocobase.com/multi-space/Goods-NocoBase-10-15-2025_10_26_AM.png)

### Campo Space em coleções

Coleções novas recebem campo **Space**. Só coleções com esse campo entram na lógica de espaço.

![](https://static-docs.nocobase.com/multi-space/data-source-manager-main-NocoBase-10-15-2025_10_01_AM.png)
![](https://static-docs.nocobase.com/multi-space/data-source-manager-main-NocoBase-10-15-2025_10_03_AM.png)

### Dados legados

1. Adicionar campo Space
2. Incluir usuários no **Unassigned Space**
3. Visualizar todos os espaços
4. Criar página de atribuição e tornar campo editável
5. Ajustar dados manualmente (ou por lote)
