# Visão

Uma visão é uma forma diferente de apresentar os dados de uma coleção. A mesma coleção pode ter várias visões, e cada uma pode ter configurações diferentes, como:

- **Tipo de visão**: Tabela, Formulário, Kanban, Calendário, etc.
- **Condições de filtro**: Exibe apenas os dados que atendem às condições.
- **Regras de ordenação**: Ordena os dados de acordo com os campos e a ordem especificados.
- **Campos exibidos**: Exibe apenas os campos necessários.

## Tipos de Visão

O NocoBase possui vários tipos de visão integrados para atender a diferentes cenários de negócios.

| Tipo de Visão | Descrição |
| --- | --- |
| Tabela | Exibe os dados em formato de tabela, com suporte a funções como filtragem, ordenação e agrupamento. |
| Formulário | Exibe os dados em formato de formulário, com suporte a funções como adicionar, editar e visualizar. |
| Kanban | Exibe os dados em formato de quadro Kanban, com suporte a funções como arrastar e soltar, e agrupamento. |
| Calendário | Exibe os dados em formato de calendário, permitindo a visualização por dia, semana, mês, etc. |
| Mapa | Exibe os dados em formato de mapa, permitindo marcar pontos de dados no mapa. |

## Criar uma Visão

Na página da coleção, clique no botão `+` no canto superior direito, selecione `Criar visão` e, em seguida, escolha o tipo de visão para criar uma nova.


![](https://static-docs.nocobase.com/2f534149-14a5-484c-843b-d2495b5c7774.png)


## Configurar uma Visão

Na página da visão, clique no botão `Configurar visão` no canto superior direito para configurar a visão.


![](https://static-docs.nocobase.com/a5732152-4467-4284-8178-00971775f320.png)
