# Campo de Región de China

## Introducción

El plugin Campo de Región de China (`field-china-region`) añade el tipo de campo "Región de China" a NocoBase.

## Instalación

En la interfaz principal de NocoBase, vaya a "Gestor de Plugins", busque el plugin "Región de China" y actívelo.

## Uso

En la configuración de la colección, añada un nuevo campo y seleccione "Región de China" como tipo de campo.

![](/images/field-china-region-1.png)

### Configuración del Campo

En la configuración del campo, puede establecer el modo de visualización para el campo:

- Provincia
- Provincia/Ciudad
- Provincia/Ciudad/Distrito
- Provincia/Ciudad/Distrito/Condado

![](/images/field-china-region-2.png)
