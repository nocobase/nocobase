# Marca personalizada

## Introducción

El nombre de marca predeterminado del sistema es "NocoBase" y se muestra principalmente en las siguientes tres ubicaciones:

![20240409111100](https://static-docs.nocobase.com/20240409111100.png)

En la parte inferior de la página de inicio de sesión:

![20240409111254](https://static-docs.nocobase.com/20240409111254.png)

La posición 1 se puede configurar a través de la opción "Logo" en [Configuración del sistema](/system-management/system-settings/). Las posiciones 2 y 3 se pueden configurar utilizando este **plugin**.

## Instrucciones de uso

Una vez activado el **plugin**, haga clic en el menú "Marca personalizada" en la esquina superior derecha de la página para acceder a la página de configuración.

![20240409113640](https://static-docs.nocobase.com/20240409113640.png)

- **Marca**: Se utiliza para establecer el texto en la parte inferior de la página de inicio de sesión (posición 3).
- **Acerca de**: Se utiliza para configurar el contenido del menú en la esquina superior derecha de la página (como se muestra en la imagen a continuación).

![20240409113915](https://static-docs.nocobase.com/20240409113915.png)

La configuración admite el formato HTML, lo que permite una personalización flexible del contenido. Si solo necesita realizar cambios de texto sencillos, puede modificar directamente el contenido existente, como se muestra a continuación:
