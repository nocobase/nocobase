# Gestión de plugins

## Administrador de plugins

Haga clic en el icono del administrador de plugins, ubicado en la esquina superior derecha de la página, para acceder a la página del administrador de plugins. Desde allí, usted podrá ver, instalar, habilitar o deshabilitar plugins.

![2024-04-10_08-47-45](https://static-docs.nocobase.com/2024-04-10_08-47-45.png)

## Centro de configuración de plugins

Los plugins suelen incluir sus propias páginas de configuración. Para acceder a los menús de configuración de todos los plugins, haga clic en el icono de configuración en la esquina superior derecha de la página. Luego, seleccione el elemento de menú correspondiente para configurar el plugin deseado.
