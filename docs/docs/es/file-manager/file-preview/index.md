# Vista previa de archivos

En las interfaces que contienen campos de archivo (incluidos los campos de adjuntos), usted puede previsualizar los archivos haciendo clic en la miniatura o en el icono del archivo. La función de vista previa integrada admite varios tipos de archivos, incluyendo imágenes, PDF y la mayoría de los formatos compatibles de forma nativa por los navegadores.

![20251129232307](https://static-docs.nocobase.com/20251129232307.png)

Para los tipos de archivos que no admiten la vista previa nativa, puede habilitar la funcionalidad de vista previa instalando o extendiendo los plugins de vista previa de archivos correspondientes. Por ejemplo, tras instalar el plugin de vista previa de archivos de Office, podrá previsualizar archivos de Word, Excel y PowerPoint.

Actualmente, NocoBase ofrece los siguientes plugins de vista previa de archivos:

* [Plugin de vista previa de archivos de Office](../file-preview/ms-office.md)
