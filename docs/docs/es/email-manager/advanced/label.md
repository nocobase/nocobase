# Etiquetas de correo electrónico

En el panel derecho de los detalles del correo electrónico, puede **seleccionar etiquetas** para el mensaje:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_06_05_PM.png)
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_06_06_PM.png)

Si crea una nueva etiqueta, esta se seleccionará automáticamente por defecto. También puede seleccionar otras etiquetas manualmente:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_06_06_PM%20(1).png)

En la lista de correos electrónicos, puede ver las etiquetas asignadas a cada uno:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_06_08_PM.png)
