# Envío Masivo

## Descripción General del Envío Masivo
Puede enviar correos electrónicos a varias personas de forma masiva (cada destinatario recibirá un correo electrónico individual).

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_06_10_PM.png)

## Envío Masivo a Través de una Tabla
También puede enviar correos electrónicos de forma masiva a través de una tabla:

1. Agregue un botón de envío de correo electrónico a la tabla
   ![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_14_PM.png)

2. Configure el campo de correo electrónico en la tabla (se utiliza para analizar las direcciones de correo electrónico después de seleccionar filas de la tabla)
   ![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_13_PM.png)

3. Seleccione los datos de la tabla para enviar correos electrónicos
   ![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_15_PM.png)

Si necesita enviar correos electrónicos a todas las direcciones de la tabla completa, puede habilitar la función **Seleccionar todos los registros de la tabla actual como destinatarios por defecto**:
![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_16_PM.png)

Después de editar el contenido del correo electrónico, haga clic en el botón de envío masivo para enviar:
![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_17_PM.png)
