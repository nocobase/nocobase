# Autoguardado de borradores

En el menú del botón **Guardar borrador**, puede activar el **autoguardado de borradores**.
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_29_PM.png)

Durante el proceso de edición del correo electrónico, el sistema guarda el borrador en tiempo real, lo que garantiza que no se pierda ningún contenido.
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_31_PM.png)

Cuando vuelva a abrir el editor de correo electrónico (para el mismo remitente y destinatario), el sistema le preguntará si desea usar el borrador guardado.
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_08_35_PM.png)
