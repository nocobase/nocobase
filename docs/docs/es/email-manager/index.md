# Gestor de Correo Electrónico

## Introducción
El plugin Gestor de Correo Electrónico es una herramienta eficiente y conveniente que permite la autorización y conexión con buzones de correo de Gmail y Outlook. Ayuda a los usuarios a integrar capacidades de gestión de correo electrónico, como el envío y la recepción de emails, en diferentes bloques y páginas. Mediante una configuración de autorización sencilla, usted podrá gestionar múltiples cuentas de forma unificada y disfrutar de una experiencia de comunicación por correo electrónico sin interrupciones.
