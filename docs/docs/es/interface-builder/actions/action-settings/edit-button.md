# Editar botón

## Introducción

El sistema le permite personalizar el nombre, el icono, el color de fondo y el tipo de un botón. Esto le ofrece opciones flexibles de personalización de la interfaz de usuario (UI) para que pueda ajustar el estilo de visualización y la funcionalidad del botón según sus necesidades específicas.

- **Nombre del botón**: Usted puede personalizar el texto del botón según las necesidades de su negocio, asegurando que la etiqueta sea clara e intuitiva.
- **Icono del botón**: Es posible configurar un icono para el botón, lo que mejora la identificación visual y le permite reconocer rápidamente su función.
- **Acción peligrosa**: Cuando un botón representa una **acción peligrosa** (como eliminar, restablecer, etc.), puede configurarlo como el tipo `Danger Action`.
- **Tipo de botón (`Button Type`)**: Los botones admiten diferentes configuraciones de tipo. Los tipos de botón comunes incluyen:
  - **`Default` (Predeterminado)**: Un botón estándar, adecuado para operaciones normales.
  - **`Primary` (Principal)**: Se utiliza para acciones importantes o principales, y suele tener un efecto visual destacado.
  - **`Dashed` (Con guiones)**: Un botón con un borde de guiones, adecuado para acciones secundarias o botones de sugerencia.
  - **`Text` (Texto)**: Un botón que solo contiene texto y no tiene color de fondo, utilizado normalmente para acciones ligeras o enlaces.
