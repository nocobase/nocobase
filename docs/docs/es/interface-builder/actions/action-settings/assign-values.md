# Asignación de Campos

## Introducción

La asignación de campos consiste en preestablecer el valor de un campo antes de ejecutar una operación. Esto permite que, al actualizar o crear un registro, estos datos se rellenen automáticamente, eliminando la necesidad de introducirlos manualmente. Es útil en situaciones como la actualización masiva del estado de pedidos o el establecimiento automático de la fecha y hora de entrega al momento actual cuando un pedido se marca como entregado.
