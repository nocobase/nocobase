# Confirmación

## Introducción

La configuración de confirmación se utiliza para verificar la ejecución de una acción. Al hacer clic en una acción, aparece un cuadro de diálogo de confirmación. El botón de eliminar tiene la confirmación habilitada por defecto para prevenir la pérdida de datos o resultados irreversibles debido a operaciones accidentales.

Permite personalizar el contenido y el título de la confirmación, y también acceder a variables.
