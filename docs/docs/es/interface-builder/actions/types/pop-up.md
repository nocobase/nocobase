# Abrir Ventana Modal

## Introducción

La acción **Abrir Ventana Modal** es similar a las acciones de Ver y Editar. Usted puede configurar un botón de **Abrir Ventana Modal** en las filas o en el encabezado de una tabla. Al hacer clic en el botón, se abrirá la ventana modal correspondiente, y los bloques dentro de esta ventana modal mostrarán datos basados en el registro de contexto actual.

![20251029182351](https://static-docs.nocobase.com/20251029182351.png)

Configure los bloques en la ventana modal según sea necesario.
