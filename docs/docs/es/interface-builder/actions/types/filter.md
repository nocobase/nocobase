# Filtrar

## Introducción

La acción de Filtrar suele aparecer en los bloques de datos para filtrar datos configurando diversas condiciones. Tenga en cuenta que, si un bloque ya tiene un alcance de datos configurado, la acción de filtrar se combinará con las condiciones de ese alcance de datos para formar la condición de filtrado final.

### Campos filtrables

Por defecto, todos los campos de esta colección están habilitados. Solo los campos habilitados pueden seleccionarse en la lista de campos de condición.

Los campos de las colecciones asociadas también pueden utilizarse como campos de filtro.
