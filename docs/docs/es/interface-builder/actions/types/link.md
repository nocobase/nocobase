# Enlace

## Introducción

La acción de enlace permite navegar a otras rutas, admitiendo el paso de variables URL. Esto posibilita que la página de destino ajuste su contenido dinámicamente según los datos recibidos. Puede configurarse y utilizarse dentro de los bloques de datos.
