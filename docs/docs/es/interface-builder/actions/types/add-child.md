# Agregar registro hijo

## Introducción

Esta operación está diseñada específicamente para **bloques de tabla de árbol** con el fin de agregar rápidamente registros hijos. Al agregar un registro hijo, la fila seleccionada se trata como el nodo padre, lo que permite a los usuarios crear fácilmente registros hijos bajo cualquier padre. Esta característica simplifica enormemente la gestión de datos jerárquicos, haciendo que la expansión y organización de las estructuras de datos sea más eficiente e intuitiva.
