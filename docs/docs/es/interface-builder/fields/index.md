# Campo

En la interfaz de usuario (UI), un campo es un componente que sirve como contenedor para datos individuales. Los distintos tipos de datos se muestran a través de diferentes componentes de campo. Los campos solo pueden adjuntarse a bloques y no pueden usarse de forma independiente.

## Campos en un Bloque

Los campos generalmente no se usan solos, sino que existen como elementos secundarios de los bloques de tipo de datos. Los bloques de tipo de datos suelen incluir una opción para "Configurar campos", y la lista de campos proviene de la colección actual.

![20251029204240](https://static-docs.nocobase.com/20251029204240.png)

## Diseñador de Campos (Barra de Herramientas)

Al igual que los bloques, la esquina superior derecha de un componente de campo también tiene tres iconos, que son:

- Arrastrar para organizar el diseño
- [Flujo de eventos](/interface-builder/event-flow)
- Configuración

![20251025165432](https://static-docs.nocobase.com/20251025165432.png)

## Diseño de Campos

Puede personalizar el diseño de los campos dentro de un bloque arrastrándolos y soltándolos.
