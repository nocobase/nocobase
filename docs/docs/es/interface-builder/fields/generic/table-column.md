# Campos de Tabla

## Introducción

Los campos de tabla no solo le permiten ajustar el ancho de las columnas, el título del campo y la ordenación, sino que también ofrecen configuraciones de visualización más personalizadas para campos específicos, como campos de fecha, campos de relación y campos numéricos.

![20251024174558](https://static-docs.nocobase.com/20251024174558.png)

## Opciones de Configuración de Campos

### Formato de Campos de Fecha

![20251024175303](https://static-docs.nocobase.com/20251024175303.png)

Para más detalles, consulte [Formato de Fecha](/interface-builder/fields/specific/date-picker)

### Formato de Campos Numéricos

Admite la conversión de unidades sencilla, separadores de miles, prefijos y sufijos, precisión y notación científica.

![20251024175445](https://static-docs.nocobase.com/20251024175445.png)

Para más detalles, consulte [Formato Numérico](/interface-builder/fields/field-settings/number-format)

### Habilitar Edición Rápida

Cuando habilita la edición rápida, aparecerá un botón de edición al pasar el ratón sobre la columna. Haga clic en él para editar y guardar datos rápidamente en una ventana emergente.

![20251025171158](https://static-docs.nocobase.com/20251025171158.gif)

### Habilitar Clic para Abrir

Además de los campos de relación que permiten abrir una ventana emergente al hacer clic, los campos normales también pueden configurarse con la opción "Hacer clic para abrir" para servir como punto de entrada a una ventana emergente.

![20251025172308](https://static-docs.nocobase.com/20251025172308.gif)

### Modo de Visualización de Contenido Desbordado

Cuando el contenido de una columna excede el ancho de la columna de la tabla, puede configurar el modo de desbordamiento:

- Elipsis (predeterminado)
- Ajuste de línea

### Componente de Campo

Algunos campos admiten múltiples formatos de visualización. Usted puede cambiar los componentes de campo para lograr diferentes efectos de visualización y satisfacer las necesidades de distintos escenarios. Por ejemplo, un campo de tipo **URL** puede cambiarse a un componente de **Vista Previa** para mostrar mejor el contenido del enlace o las vistas previas de imágenes.

![20251025171658](https://static-docs.nocobase.com/20251025171658.png)

### Ordenable

Actualmente, la mayoría de los tipos de campo admiten la ordenación en el lado del servidor. Cuando la ordenación está habilitada, los datos pueden ordenarse en orden ascendente o descendente según el campo objetivo.
