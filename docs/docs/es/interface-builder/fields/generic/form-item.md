# Campos de Formulario

## Introducción

Los diferentes tipos de campos en un formulario tienen distintas opciones de configuración. Esto permite a los usuarios extender las opciones de configuración de los campos a través de `plugins` para satisfacer necesidades más personalizadas.

![20251025174555](https://static-docs.nocobase.com/20251025174555.png)

## Opciones de Configuración de Campos

### Valor predeterminado

Puede configurar constantes o variables como valores predeterminados para los campos (esta función solo está disponible para formularios nuevos).

![20251025174724](https://static-docs.nocobase.com/20251025174724.png)

Para más detalles, consulte [Configurar valor predeterminado](/interface-builder/fields/field-settings/default-value)
