# Campos de Detalle

## Introducción

La configuración de los campos para los bloques de Detalle, Lista y Cuadrícula es prácticamente la misma. Su función principal es controlar cómo se muestran los campos en modo de lectura.

![20251025172851](https://static-docs.nocobase.com/20251025172851.png)

## Opciones de Configuración de Campos

### Formato de Campos de Fecha

![20251025173005](https://static-docs.nocobase.com/20251025173005.png)

Para más detalles, consulte [Formato de Fecha](/interface-builder/fields/specific/date-picker)

### Formato de Campos Numéricos

![20251025173242](https://static-docs.nocobase.com/20251025173242.png)

Admite conversiones de unidades sencillas, separadores de miles, prefijos y sufijos, precisión y notación científica.

Para más detalles, consulte [Formato Numérico](/interface-builder/fields/field-settings/number-format)

### Habilitar Clic para Abrir

Además de los campos de relación que permiten abrir una ventana emergente al hacer clic, los campos normales también se pueden configurar para que actúen como un punto de entrada para abrir una ventana emergente. También puede definir cómo se abre esta ventana (como un cajón, un diálogo o una subpágina).

![20251025173549](https://static-docs.nocobase.com/20251025173549.gif)

### Modo de Visualización de Desbordamiento de Contenido

Cuando el contenido de un campo excede su ancho, puede configurar cómo se maneja el desbordamiento:

- Elipsis (por defecto)
- Ajuste de línea

![20251025173917](https://static-docs.nocobase.com/20251025173917.png)

### Componente de Campo

Algunos campos admiten múltiples formatos de visualización, lo cual se logra cambiando el componente de campo.

Por ejemplo: El componente `URL` se puede cambiar al componente `Preview`.

![20251025174042](https://static-docs.nocobase.com/20251025174042.png)

Por ejemplo, los campos de relación se pueden mostrar de diferentes maneras, cambiando del componente de campo de título a `Sub-detalles` para mostrar más contenido de la relación.

![20251025174311](https://static-docs.nocobase.com/20251025174311.gif)
