# Selector de Datos

## Introducción

El Selector de Datos, presentado como un selector de tabla emergente, le permite seleccionar datos existentes de la colección de destino para asociarlos, o bien, añadir nuevos datos a la colección de destino y luego asociarlos.

![20251027203343](https://static-docs.nocobase.com/20251027203343.png)

Los datos ya seleccionados no se mostrarán en el selector de tabla.

![20251027203717](https://static-docs.nocobase.com/20251027203717.gif)

Desde el selector de tabla, también puede gestionar la colección de destino de la relación (añadir, eliminar, importar/exportar, etc.).

![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
