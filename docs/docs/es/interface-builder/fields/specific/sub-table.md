# Subtabla

## Introducción

La subtabla es ideal para gestionar campos de relación de uno a muchos. Permite la creación masiva y la asociación de nuevos datos en la colección de destino, o seleccionar y asociar datos ya existentes.

## Instrucciones de Uso

![20251027223350](https://static-docs.nocobase.com/20251027223350.png)

En la subtabla, los diferentes tipos de campos muestran distintos componentes de campo. Los campos de gran tamaño (como Texto Enriquecido, JSON, Texto Largo) se editan mediante una ventana emergente.

![20251027223426](https://static-docs.nocobase.com/20251027223426.png)

Campos de relación en la subtabla.

Pedidos (Uno a Muchos) > Productos del Pedido (Uno a Uno) > Oportunidad

![20251027223530](https://static-docs.nocobase.com/20251027223530.png)

El componente predeterminado para un campo de relación es el desplegable (compatible con desplegable/selector de datos).

![20251027223754](https://static-docs.nocobase.com/20251027223754.png)
