# Fecha y Hora

## Introducción

El campo de fecha admite formatos de fecha y hora personalizados.

## Dónde configurar el formato de fecha

### Campos de colección

![20240417114311](https://static-docs.nocobase.com/20240417114311.png)

> El formato de fecha que configure en el campo de la colección se utilizará como formato predeterminado del campo. Este formato puede ser sobrescrito por el formato de fecha establecido en la configuración del campo dentro de la configuración del bloque.
