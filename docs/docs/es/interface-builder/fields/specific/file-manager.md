# Gestor de Archivos

## Introducción

El Gestor de Archivos es un componente de campo de asociación que se utiliza para manejar relaciones donde la colección de destino es la colección de archivos.

![20251027194124](https://static-docs.nocobase.com/20251027194124.png)

## Configuración del Campo

- [Componente de Campo](/interface-builder/fields/association-field): Permite cambiar a otros componentes de campo de asociación, como Desplegable, Selector de Datos, etc.
- Permitir seleccionar archivos existentes: Activado por defecto, permite seleccionar y asociar archivos ya cargados.
