# Subformulario

## Introducción

Los subformularios son ideales para escenarios en los que necesita crear datos relacionados antes de vincularlos. Permiten mostrar datos relacionados de múltiples niveles de forma clara, anidados dentro del formulario principal. A diferencia de los selectores de datos o los desplegables, un subformulario le permite gestionar directamente los campos de la colección asociada dentro del mismo bloque de la página. Además, los datos relacionados se envían junto con el formulario principal.
