# Título

## Introducción

En modo de solo lectura, los campos de asociación se muestran por defecto utilizando el **Componente de Campo de Título**, que identifica el registro asociado actualmente mostrando el valor del campo de título. El sistema coincide y renderiza automáticamente el componente de campo de título correspondiente según el tipo de campo de título.
