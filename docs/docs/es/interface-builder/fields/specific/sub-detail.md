# Sub-detalles

## Introducción

Los Sub-detalles son el componente correspondiente a los Sub-formularios en modo de lectura. A diferencia de los componentes de Etiqueta y Título, los Sub-detalles no solo pueden mostrar más datos de la colección actual, sino que también se pueden configurar para mostrar información de datos de colecciones relacionadas. Los datos de relaciones multinivel se muestran claramente en forma de detalles anidados.
