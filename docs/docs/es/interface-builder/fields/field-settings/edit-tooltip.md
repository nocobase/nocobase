# Editar Tooltip

## Introducción

Un *tooltip* de campo es el texto que aparece cuando usted pasa el ratón sobre un campo, diseñado para proporcionar una explicación adicional o una guía sobre el mismo. Estos *tooltips* suelen incluir información como el propósito del campo, el formato de entrada y las restricciones, lo que ayuda a los usuarios a comprender y utilizar mejor el campo.
