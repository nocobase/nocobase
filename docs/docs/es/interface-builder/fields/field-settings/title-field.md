# Campo de Título

## Introducción

El **Campo de Título** se utiliza normalmente en los componentes de campo de relación como un identificador para los datos asociados que se muestran en la interfaz. Este campo determina cómo se presenta la información principal de los datos, lo que suele ayudar a los usuarios a identificarlos y comprenderlos rápidamente.

Diferentes tipos de campos de título corresponden a distintos componentes de campo y admiten sus respectivas opciones de configuración. Por ejemplo:

- Cuando el campo de título es un **Campo de Fecha**, el componente de campo utiliza un **componente de Fecha** para mostrar los datos.
- Cuando el campo de título es un **Campo de Texto**, el componente de campo utiliza un **componente de Texto** para mostrar los datos.
- Cuando el campo de título es un **Campo de Opción**, el componente de campo utiliza un **componente de Opción** para mostrar los datos.
