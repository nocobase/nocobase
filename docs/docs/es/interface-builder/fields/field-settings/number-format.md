# Formato de Números

## Introducción

El formato de números se refiere a la mejora visual y estandarización de los campos de tipo numérico (incluyendo campos de fórmula) en modo de lectura. Esto incluye configuraciones como separadores de miles, precisión decimal, conversión de unidades y notación científica. Al configurar el formato de números, usted puede adaptar la presentación de los valores a las costumbres de expresión numérica de diferentes regiones e industrias.

## Cómo Usar

- Soporta conversión de unidades simple (multiplicación/división)
- Separador de miles
- Prefijos y sufijos
- Precisión
- Notación científica.

Ejemplo: Formatee el monto de un pedido añadiendo un símbolo de moneda (como el símbolo de yuan ￥), un separador de miles (como una coma o un punto) y precisión decimal para que el monto sea claro y fácil de leer.
