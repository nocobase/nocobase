# Componentes de Campo

## Introducción

Los componentes de campo son los elementos que se utilizan para mostrar el contenido de un campo.
Algunos campos incluyen varios componentes de visualización integrados. Usted puede elegir el componente más adecuado según sus necesidades para presentar el valor del campo de diferentes maneras.

Ejemplo: En el modo de visualización, un campo URL se puede cambiar a un componente de Vista previa o a un componente URL (predeterminado).
