# Modo de Visualización

## Introducción
A diferencia de los bloques, los componentes de campo tienen tres modos de visualización (solo disponibles para campos dentro de un formulario). Al cambiar entre estos modos, se ajustan las opciones de configuración del campo.

*   Editable;
*   Solo lectura (no editable);
*   Solo lectura (modo de lectura);

### Modos de Visualización para Campos de Relación
El **modo de visualización de un campo de relación** define cómo se presenta el campo en la interfaz y qué tipos de componentes de campo están disponibles para su uso.

En el **estado editable**, los campos de relación admiten varios tipos de componentes. Esto le permite elegir diferentes componentes de campo de relación para mostrar o seleccionar datos asociados según sus necesidades de negocio.

#### Componentes de Campo de Relación en Estado de Lectura
Cuando se cambia al **estado de lectura**, el sistema utiliza automáticamente por defecto el **componente de campo de título** para mostrar los datos asociados. Esto es ideal para situaciones en las que solo necesita ver los datos sin realizar modificaciones.
