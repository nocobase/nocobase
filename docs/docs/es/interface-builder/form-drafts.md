# Borradores de formulario

Cuando se habilitan los borradores de formulario, el contenido que se está completando se guarda automáticamente de forma local. Incluso si el formulario se cierra accidentalmente o se actualiza la página, los datos ingresados no se perderán.

![20251213093717](https://static-docs.nocobase.com/20251213093717.png)

Cuando existe un borrador, se mostrará un mensaje de notificación en la parte superior del formulario. El borrador se eliminará automáticamente tras un envío exitoso; si ya no es necesario, también puede eliminarse manualmente en cualquier momento.

![20251213092734](https://static-docs.nocobase.com/20251213092734.png)

::: tip Sugerencia

Actualmente, los borradores de formulario solo admiten el almacenamiento en caché local. Las versiones futuras admitirán el almacenamiento en caché del lado del servidor para permitir la sincronización entre múltiples dispositivos.

:::
