# Tarjeta de Cuadrícula

## Introducción

El bloque Tarjeta de Cuadrícula muestra información resumida de los registros de datos en formato de tarjeta. Permite configurar el número de columnas según el tamaño de la pantalla, asegurando una visualización amigable en diferentes dispositivos.
