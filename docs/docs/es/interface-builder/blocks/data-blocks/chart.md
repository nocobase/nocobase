# Gráfico

## Introducción

El plugin de visualización de datos de NocoBase le permite consultar datos de forma visual y ofrece un amplio conjunto de componentes de gráficos. Con una configuración sencilla, usted puede crear rápidamente paneles de visualización para mostrar información clave de sus datos y facilitar el análisis y la presentación de datos multidimensionales.
