# Diseño

## Introducción

Los bloques de formulario, como los bloques de formulario, los bloques de detalles y los bloques de lista, permiten configurar su diseño. El diseño puede ser vertical (donde las etiquetas y los campos de entrada se organizan fila por fila) u horizontal (donde las etiquetas y los campos de entrada se colocan uno al lado del otro). Usted puede ajustar el diseño de los campos del bloque según sus necesidades específicas para asegurar que la entrada o visualización de la información sea clara y eficiente.

### Vertical (Predeterminado)

El diseño predeterminado es vertical.

![20251024120143](https://static-docs.nocobase.com/20251024120143.png)

### Horizontal

El diseño horizontal muestra las etiquetas y los campos de entrada uno al lado del otro, lo que permite ahorrar espacio de forma eficiente. Para campos con títulos largos, usted puede elegir si desea que el texto se ajuste (salto de línea) o que se muestre una elipsis (puntos suspensivos) para asegurar una interfaz limpia y estéticamente agradable.

![20251024120237](https://static-docs.nocobase.com/20251024120237.png)

#### Alineación del título del campo

- **Alineación izquierda** (predeterminado)
- **Alineación derecha**

#### Ancho del título del campo

- El ancho predeterminado es de **120px**, con un ancho mínimo de **50px**.

#### Manejo de títulos de campo que exceden el ancho

- **Ajustar texto**
- **Elipsis**

#### Mostrar dos puntos

Usted puede elegir si desea mostrar dos puntos después del título del campo.
