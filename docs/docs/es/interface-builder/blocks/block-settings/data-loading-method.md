# Modo de carga de datos

## Introducción

El modo de carga de datos predeterminado es "Cargar todos los datos cuando las condiciones de filtrado estén vacías". Aquí, las "condiciones de filtrado" se refieren a los valores en el "Bloque de filtrado". Otra opción disponible es "No cargar datos cuando las condiciones de filtrado estén vacías".
