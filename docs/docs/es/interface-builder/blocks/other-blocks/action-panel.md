# Bloque: Panel de Control

## Introducción

Este bloque permite colocar diversas acciones rápidas. Actualmente, soporta:

- [Enlace](/interface-builder/actions/types/link)
- [Escanear Código QR]
- [Acción de Ventana Emergente](/interface-builder/actions/types/pop-up)

![20251023225754](https://static-docs.nocobase.com/20251023225754.png)

> 💡 Continuaremos añadiendo más tipos de acción en futuras versiones.

## Tipos de Acción

- **Enlace**: Permite navegar a una página específica o a una URL externa.

![20251023230643](https://static-docs.nocobase.com/20251023230643.png)

- **Escanear Código QR**: Activa una acción correspondiente al escanear un código QR.

- **Abrir Ventana Emergente**: Abre una ventana emergente para mostrar el contenido del bloque.
  ![20251023230522](https://static-docs.nocobase.com/20251023230522.png)
