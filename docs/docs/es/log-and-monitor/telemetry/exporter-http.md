# Exportador de telemetría: HTTP

## Configuración de variables de entorno

### TELEMETRY_METRIC_READER

Tipo de exportador de métricas de telemetría.

```bash
TELEMETRY_METRIC_READER=http
```

### TELEMETRY_HTTP_URL

URL HTTP para la exportación de datos de telemetría.

```bash
TELEMETRY_HTTP_URL=
```
