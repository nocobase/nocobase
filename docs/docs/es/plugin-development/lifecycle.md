# Ciclo de Vida

Esta sección organiza los hooks del ciclo de vida para los plugins, tanto en el lado del servidor como en el lado del cliente, ayudando a los desarrolladores a registrar y liberar recursos de forma correcta.

Puede compararlo con el ciclo de vida de FlowModel para resaltar conceptos comunes.

## Contenido Sugerido

- Callbacks que se activan cuando los plugins se instalan, habilitan, deshabilitan o desinstalan.
- Momentos de montaje, actualización y destrucción de los componentes del lado del cliente.
- Recomendaciones para gestionar tareas asíncronas y errores dentro del ciclo de vida.
