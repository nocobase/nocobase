## API

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| `value` | `string` | Valor actual |
| `onChange` | `(event) => void` | Callback de cambio |
| `disabled` | `boolean` | Si está deshabilitado |
| `placeholder` | `string` | Texto de marcador |
