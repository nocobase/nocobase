# Usuarios

## Gestión de Usuarios

Este `plugin` le ofrece una interfaz de gestión de usuarios en el centro de configuración. Después de inicializar la aplicación, se crea automáticamente una cuenta de superadministrador que no se puede eliminar. Este administrador tiene permisos de `Root` y puede acceder a todos los recursos.

![](https://static-docs.nocobase.com/44bf40f56b45d4dd96c424fb08082cf6.png)

Además, también puede añadir diferentes bloques de la `colección` de Usuarios en las páginas para gestionar a los usuarios, como por ejemplo, bloques de tabla:

![](https://static-docs.nocobase.com/76b5a4652f869541a9e8f18a4568a7c9.png)
