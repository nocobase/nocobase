# Gestión de roles de departamento

Al asociar uno o más roles a un departamento, los miembros de dicho departamento pueden tener acceso a esos roles.

Existen dos formas de asociar roles a un departamento:

La primera es seleccionar un rol en el campo de rol del formulario de edición de departamento.

![](https://static-docs.nocobase.com/70f77bb89aa1fb415c152a51a51cc23b.png)

La segunda consiste en añadir departamentos al rol correspondiente desde la sección de gestión de roles.

![](https://static-docs.nocobase.com/f2a7bec937cf2f179ce868a92b98416d.png)

Los departamentos que ya tienen asignado este rol no se pueden seleccionar.

![](https://static-docs.nocobase.com/be10299893581e1f97a4e01ddd5c7e59.png)
