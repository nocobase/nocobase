# nb config list

Lista las configuraciones del CLI establecidas explícitamente. Los valores predeterminados que nunca se configuraron explícitamente no se muestran.

## Uso

```bash
nb config list
```

## Ejemplos

```bash
nb config list
```

## Notas

Si no hay valores de configuración establecidos, el CLI imprime `No CLI config values are set.`.
