# nb skills check

Comprueba las Skills de AI coding globales de NocoBase, e informa de si están gestionadas por la CLI y de si hay actualizaciones disponibles.

## Uso

```bash
nb skills check [flags]
```

## Parámetros

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| `--json` | boolean | Genera la salida en JSON |

## Ejemplos

```bash
nb skills check
nb skills check --json
```
