# nb skills remove

Elimina las Skills de AI coding globales de NocoBase gestionadas por `nb`.

## Uso

```bash
nb skills remove [flags]
```

## Parámetros

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| `--yes`, `-y` | boolean | Omite la confirmación de la eliminación |
| `--json` | boolean | Genera la salida en JSON |
| `--verbose` | boolean | Muestra la salida detallada de la eliminación |

## Ejemplos

```bash
nb skills remove
nb skills remove --yes
nb skills remove --json
```
