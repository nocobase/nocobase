# nb session id

Muestra el id de sesión efectivo actual.

Si no hay un `NB_SESSION_ID` utilizable en la shell o el runtime actual, el comando te indica que ejecutes primero [`nb session setup`](./setup.md) y luego abras una nueva shell o runtime.

## Uso


nb session id

## Ejemplos


nb session id

## Comandos relacionados

- [`nb session setup`](./setup.md)
- [`nb env current`](../env/current.md)
