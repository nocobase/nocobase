# nb proxy caddy current

Muestra el driver actual del proveedor Caddy.

## Uso

```bash
nb proxy caddy current
```

## Salida

El comando muestra directamente un valor:

- `local`
- `docker`

## Ejemplos

```bash
nb proxy caddy current
```
