# nb proxy nginx current

Muestra el driver actual del proveedor Nginx.

## Uso

```bash
nb proxy nginx current
```

## Salida

El comando muestra directamente un valor:

- `local`
- `docker`

## Ejemplos

```bash
nb proxy nginx current
```
