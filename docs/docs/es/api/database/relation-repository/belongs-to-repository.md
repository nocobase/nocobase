## BelongsToRepository

Su interfaz es la misma que la de [HasOneRepository](./has-one-repository.md).
`BelongsToRepository` es un `Repository` diseñado para gestionar relaciones `BelongsTo`. Este `Repository` le ofrece métodos convenientes para trabajar con este tipo de relaciones.
