# Instalación de create-nocobase-app

:::tip Prerrequisitos

- Instalado Node.js 20+, Yarn 1.22.x
- Configurada y en ejecución una de las siguientes bases de datos: MySQL 8.0.17+, MariaDB 10.9+ o PostgreSQL 10+ (elija una)
- Debido al entorno de red en China, se recomienda cambiar al espejo nacional (no es obligatorio)

```bash
$ yarn config set disable-self-update-check true
$ yarn config set registry https://registry.npmmirror.com/
```

:::

:::warning Descripción de las variables de entorno

- `TZ` se utiliza para configurar la zona horaria de la aplicación, por defecto es la zona horaria del sistema operativo.
- `APP_KEY` es la clave de la aplicación, utilizada para generar tokens de usuario, etc. (si se modifica `APP_KEY`, los tokens antiguos también quedarán invalidados). Puede ser cualquier cadena aleatoria. **Por favor, cámbiela por su propia clave secreta y asegúrese de no filtrarla al exterior.**
- `DB_*` son las configuraciones relacionadas con la base de datos, por favor modifíquelas según su situación real con la información de conexión de su base de datos.

:::

## 2. Cambiar de directorio

```bash
cd my-nocobase-app
```

## 3. Instalar dependencias

📢 Debido a la influencia del entorno de red, la configuración del sistema y otros factores, este paso puede tardar unos diez minutos.

```bash
yarn install
```

:::tip Sugerencia para el entorno de producción

Al implementar en un entorno de producción, para reducir el tamaño, puede instalar solo las dependencias necesarias:

```bash
yarn install --production
```

:::

## 4. Instalar NocoBase

```bash
yarn nocobase install --lang=es-ES
```

El proceso de instalación creará automáticamente la estructura de las tablas de la base de datos e inicializará los datos.

## 5. Iniciar NocoBase

**Entorno de desarrollo**

```bash
yarn dev
```

**Entorno de producción**

```bash
yarn start
```

## 6. Iniciar sesión en NocoBase

Utilice el navegador para abrir [http://localhost:13000](http://localhost:13000), la cuenta y contraseña iniciales son `admin@nocobase.com` y `admin123`.

:::warning Sugerencia de seguridad de la cuenta

Después del primer inicio de sesión, por favor cambie la contraseña predeterminada a tiempo para garantizar la seguridad del sistema.

:::
