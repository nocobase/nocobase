# Despliegue en Clúster

NocoBase permite desplegar aplicaciones en modo clúster para mejorar su rendimiento y disponibilidad. El despliegue en clúster permite ejecutar diferentes instancias de la aplicación en múltiples servidores, o bien, ejecutar varios procesos en modo multinúcleo en un único servidor.

Para más detalles, consulte la documentación sobre [Modo Clúster](/cluster-mode/).
