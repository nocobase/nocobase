# Colección en árbol

## Introducción

Una colección con estructura de árbol es un patrón de diseño de colección de datos utilizado para organizar información jerárquica. Esta estructura de colección se asemeja a un árbol, donde cada elemento de datos puede tener uno o más elementos secundarios, y estos, a su vez, pueden tener sus propios descendientes.
