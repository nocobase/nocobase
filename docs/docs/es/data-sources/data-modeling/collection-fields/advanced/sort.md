# Selector de colección

El campo Selector de colección es un tipo de campo especial que permite a los usuarios seleccionar una o más colecciones de una lista predefinida. Este tipo de campo es muy útil en escenarios donde necesita referenciar dinámicamente diferentes colecciones, como la creación de informes configurables, paneles de control (dashboards) o aplicaciones complejas que necesitan cambiar las fuentes de datos según la selección del usuario.

## Casos de Uso

### Generación Dinámica de Informes

En un módulo de generación de informes, los usuarios pueden necesitar seleccionar una colección de entre varias colecciones como fuente de datos para el informe. Al usar el Selector de colección, los usuarios pueden seleccionar fácilmente la colección deseada, y el sistema generará dinámicamente el informe basándose en la selección del usuario.

### Paneles de Control (Dashboards) Configurables

En una aplicación de panel de control (dashboard), los usuarios pueden personalizar los datos mostrados en el panel de control. El Selector de colección permite a los usuarios elegir las colecciones que les interesan, mostrando así gráficos y datos estadísticos de diferentes fuentes de datos en el panel de control.

### Herramientas Flexibles de Importación/Exportación de Datos

Al desarrollar una herramienta genérica de importación o exportación de datos, puede usar el Selector de colección para permitir a los usuarios especificar la colección sobre la que operar. De esta manera, la misma herramienta puede usarse para procesar datos de cualquier colección en el sistema, mejorando enormemente la reutilización de la herramienta.
