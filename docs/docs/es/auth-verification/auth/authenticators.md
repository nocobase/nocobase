# Autenticadores

## Gestión de Autenticación de Usuarios

Cuando se instala el plugin de autenticación de usuarios, se inicializa un método de autenticación de `contraseña` basado en el nombre de usuario y el correo electrónico del usuario.

![](https://static-docs.nocobase.com/66eaa9d5421c9cb713b117366bd8a5d5.png)

## Activar Autenticadores

![](https://static-docs.nocobase.com/7f1fb8f8ca5de67ffc68eff0a65848f5.png)

Solo los tipos de autenticación activados se mostrarán en la página de inicio de sesión.

![](https://static-docs.nocobase.com/8375a36ef98417af0f0977f1e07345dd.png)

## Tipos de Autenticación de Usuario

![](https://static-docs.nocobase.com/da4250c0cea343ebe470cbf7be4b12e4.png)

Al añadir diferentes tipos de autenticadores, puede habilitar los métodos de autenticación correspondientes en el sistema.

Además de los tipos de autenticación que ofrecen los plugins existentes, los desarrolladores también pueden extender los tipos de autenticación de usuario. Para más detalles, consulte la [Guía del desarrollador](./dev/).
