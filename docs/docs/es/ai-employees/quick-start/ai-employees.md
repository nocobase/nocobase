# Inicio Rápido

## Introducción
Después de configurar el servicio LLM, ya podemos empezar a crear empleados de IA.

## Nuevo empleado de IA

### Configuración de información básica

Vaya a la página de configuración del **plugin** de empleados de IA, haga clic en la pestaña `AI employees` para acceder a la página de gestión de empleados de IA.

![20251021225643](https://static-docs.nocobase.com/20251021225643.png)

Haga clic en el botón `New AI employee` para ir a la página de creación de empleados de IA.

Primero, en la pestaña `Profile`, puede configurar la información básica del empleado de IA:

- En el campo `Username`, introduzca el identificador único para el empleado de IA.
- En el campo `Nickname`, introduzca el apodo del empleado de IA.
- En el campo `Position`, introduzca el puesto del empleado de IA. Esto sirve para describir brevemente sus responsabilidades.
- En `Avatar`, seleccione un avatar para el empleado de IA.

![20251021232246](https://static-docs.nocobase.com/20251021232246.png)

- En el campo `Bio`, introduzca el texto de presentación del empleado de IA, describiendo sus habilidades y funciones. Este texto se mostrará en el perfil del empleado de IA.
- En el campo `About me`, introduzca una descripción detallada de las funciones del empleado de IA. Este texto se utilizará como *prompt* del sistema cuando converse con el empleado de IA.
- En el campo `Greeting message`, introduzca el mensaje de bienvenida del empleado de IA. Cada vez que un usuario inicie una nueva conversación con el empleado de IA, el sistema le enviará automáticamente este mensaje.

![20251021232712](https://static-docs.nocobase.com/20251021232712.png)

### Configuración del modelo grande

En la pestaña `Model settings`, puede configurar los ajustes del modelo para el empleado de IA:

- Seleccione el servicio LLM que creó en la gestión de servicios LLM.
- Introduzca el nombre del modelo grande que desea utilizar.

![20251021231727](https://static-docs.nocobase.com/20251021231727.png)

### Finalizar la creación

Después de introducir toda la información necesaria para crear el empleado de IA, haga clic en el botón `Submit` para finalizar la creación.

A continuación, podrá ver el empleado de IA recién creado en la lista de empleados de IA. También lo encontrará en el botón de acceso rápido a empleados de IA, situado en la parte inferior derecha de la página.

![20251021232836](https://static-docs.nocobase.com/20251021232836.png)
