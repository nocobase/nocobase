# Empleado de IA · Nathan: Ingeniero Frontend

> Le ayuda a escribir y optimizar código frontend para implementar lógica interactiva compleja.

`Nathan` es el experto en desarrollo frontend de NocoBase. En escenarios donde se requiere escribir JavaScript, como `JSBlock`, `JSField`, `JSItem`, `JSColumn`, `JSAction`, `Event Flow` y `Linkage`, encontrará el avatar de `Nathan` en la esquina superior derecha del editor de código, y podrá pedirle que le ayude a escribir o modificar el código en el editor.

![20251022163405](https://static-docs.nocobase.com/20251022163405.png)

## Pasos

En el editor de código, haga clic en `Nathan` para abrir el cuadro de diálogo del Empleado de IA. El código del editor se adjuntará automáticamente al cuadro de entrada y se enviará a `Nathan` como contexto de la aplicación.

![20251022163935](https://static-docs.nocobase.com/20251022163935.png)

Introduzca sus requisitos de codificación y envíelos a `Nathan`, luego espere su respuesta.

![20251022164041](https://static-docs.nocobase.com/20251022164041.png)

Haga clic en el botón `Apply to editor` del bloque de código en la respuesta de `Nathan` para sobrescribir el código en el editor de código.

![20251022164323](https://static-docs.nocobase.com/20251022164323.png)

Haga clic en el botón `Run` del editor de código para ver el efecto en tiempo real.

![20251022164436](https://static-docs.nocobase.com/20251022164436.png)
