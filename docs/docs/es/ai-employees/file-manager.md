# Gestión de Archivos

## Introducción

Aquí puede configurar cómo se almacenan los archivos que se suben durante las conversaciones con los empleados de IA.

## Configuración

Para configurar esto, vaya a la página de configuración del plugin de empleados de IA. Una vez allí, haga clic en la pestaña `Settings` para acceder a la página donde podrá definir el método de almacenamiento.

![20251022002648](https://static-docs.nocobase.com/20251022002648.png)

Las opciones de almacenamiento disponibles para `Files storage` se definen y gestionan directamente en el plugin de Gestión de Archivos.
