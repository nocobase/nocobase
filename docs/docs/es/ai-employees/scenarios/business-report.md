# Generación de informes de análisis de negocio

Tomando como ejemplo el sistema CRM de la DEMO en línea, vamos a pedir al AI Employee que analice los datos de clientes y pedidos del sistema y genere un informe de análisis.

## Carga de Skills

En primer lugar, el AI Employee carga las Skills correspondientes y, a continuación, comienza a trabajar.

![](https://static-docs.nocobase.com/202604230924035.png)

## Análisis del modelo de datos

Obtiene la información de las fuentes de datos y los metadatos de las tablas.

![](https://static-docs.nocobase.com/202604230926297.png)

## Consulta de los datos del negocio

Comienza a consultar los datos del negocio. Dependiendo del contenido a analizar, este proceso suele requerir desde varias decenas de segundos hasta varios minutos.

![](https://static-docs.nocobase.com/202604230927861.png)

## Generación del informe

Una vez recopilados suficientes datos del negocio, comienza a generar el informe de análisis.

![](https://static-docs.nocobase.com/202604230920113.png)

Finalmente se genera un informe de análisis de negocio con texto e imágenes, que se puede descargar en formato Markdown, HTML o PDF.

![](https://static-docs.nocobase.com/202604230922030.png)

![](https://static-docs.nocobase.com/202604230922376.png)

![](https://static-docs.nocobase.com/202604230923542.png)

![](https://static-docs.nocobase.com/202604230923538.png)

![](https://static-docs.nocobase.com/202604230923049.png)

Nota: si descarga el informe en formato Markdown, los gráficos deberán descargarse por separado desde la página web.
