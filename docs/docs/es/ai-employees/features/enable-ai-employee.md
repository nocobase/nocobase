# Habilitar empleados de IA

NocoBase ofrece varios empleados de IA integrados para diferentes escenarios.

## Cómo habilitar

Acceda a la página de administración de `AI employees` y active el interruptor `Enabled` directamente en la lista para habilitar al empleado.

![ai-employee-list-enable-switch.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employee-list-enable-switch.png)

## Uso después de la habilitación

Una vez habilitado, el empleado aparecerá en los puntos de acceso disponibles (como el acceso directo en la esquina inferior derecha o en entradas específicas de cada escenario).
