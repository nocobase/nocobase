# Habilitar empleados de IA

NocoBase incluye empleados de IA integrados para varios escenarios. Consulte la descripción completa en [Empleados de IA integrados](/ai-employees/built-in/).

## Cómo habilitar

Después de entrar en la página de gestión `AI employees`, active directamente el interruptor `Enabled` en la lista.

![](https://static-docs.nocobase.com/202605121048555.png)

## Uso después de habilitar

Después de habilitarlo, el empleado aparecerá en las entradas disponibles, como la entrada rápida inferior derecha o una entrada específica de escenario.
