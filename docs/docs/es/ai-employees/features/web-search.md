# Búsqueda en la web

La búsqueda en la web se utiliza para complementar los datos de entrenamiento del modelo con información externa actualizada.

## Cómo funciona

La disponibilidad de la búsqueda en la web depende de si el servicio de modelo seleccionado en la sesión actual admite la función Web Search.

- **Compatible:** se muestra el interruptor de búsqueda en la web, el cual puede activarse o desactivarse según sea necesario.
- **No compatible:** no se muestra el interruptor y el estado de búsqueda se desactivará automáticamente.

## Uso en la sesión

Utilice el interruptor de búsqueda en la web en el área de entrada de la conversación:

- **Activado:** la IA puede realizar una búsqueda de información antes de responder.

![web-search-switch-visible-on-supported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-visible-on-supported-model.png)

- **Desactivado:** la IA responderá basándose únicamente en el contexto existente.

![web-search-switch-hidden-on-unsupported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-hidden-on-unsupported-model.png)

## Diferencias entre plataformas

La capacidad de soporte para Web Search varía según el modelo. Por favor, utilice esta función de acuerdo con las capacidades reales del servicio de modelo seleccionado.
