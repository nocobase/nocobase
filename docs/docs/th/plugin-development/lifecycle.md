# ไลฟ์ไซเคิล

ส่วนนี้ได้รวบรวมไลฟ์ไซเคิลฮุก (lifecycle hooks) สำหรับปลั๊กอินทั้งฝั่งเซิร์ฟเวอร์และฝั่งไคลเอนต์ เพื่อช่วยให้นักพัฒนาสามารถลงทะเบียนและปล่อยทรัพยากรได้อย่างถูกต้องครับ/ค่ะ

สามารถนำไปเปรียบเทียบกับไลฟ์ไซเคิลของ FlowModel เพื่อเน้นย้ำแนวคิดทั่วไปได้ครับ/ค่ะ

## เนื้อหาที่แนะนำ

- คอลแบ็ก (callbacks) ที่ถูกเรียกใช้เมื่อปลั๊กอินถูกติดตั้ง, เปิดใช้งาน, ปิดใช้งาน หรือถอนการติดตั้ง
- ช่วงเวลาของการเมานต์ (mounting), อัปเดต (updating) และทำลาย (destruction) คอมโพเนนต์ฝั่งไคลเอนต์
- คำแนะนำสำหรับการจัดการงานแบบอะซิงโครนัส (asynchronous tasks) และข้อผิดพลาด (errors) ภายในไลฟ์ไซเคิล
