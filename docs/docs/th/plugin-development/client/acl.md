# การควบคุมสิทธิ์ ACL

:::tip

บทนี้จะแนะนำวิธีการใช้ ACL (Access Control List) สำหรับการจัดการสิทธิ์ในปลั๊กอินส่วนหน้าครับ/ค่ะ เนื้อหาจะประกอบด้วย:

- การตรวจสอบสิทธิ์
- วิธีการเรนเดอร์เนื้อหาแบบไดนามิกตามสิทธิ์

เราจะเพิ่มตัวอย่างโดยละเอียดและแนวทางปฏิบัติที่ดีที่สุดในเวอร์ชันถัดไปครับ/ค่ะ

:::
