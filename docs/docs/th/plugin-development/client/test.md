# การทดสอบ

:::tip

ในบทนี้ เราจะมาเรียนรู้วิธีการเขียนและรันการทดสอบสำหรับส่วน Frontend ของปลั๊กอินกันครับ/ค่ะ เนื้อหาจะครอบคลุมถึง:
- การตั้งค่าสภาพแวดล้อมการทดสอบและ Dependencies;
- การเขียน Unit Test โดยใช้ Vitest และ Testing Library;
- วิธีการจำลองสภาพแวดล้อมการรัน Frontend ของ NocoBase;
- การรันการทดสอบโดยอัตโนมัติใน CI ครับ/ค่ะ

เราจะเพิ่มตัวอย่างโดยละเอียดและแนวทางปฏิบัติที่ดีที่สุดในเวอร์ชันถัดไปครับ/ค่ะ

:::
