# ฟิลด์ภูมิภาคจีน

## บทนำ

ปลั๊กอินฟิลด์ภูมิภาคจีน (`field-china-region`) จะเพิ่มประเภทฟิลด์ "ภูมิภาคจีน" ให้กับ NocoBase ครับ

## การติดตั้ง

ในหน้าหลักของ NocoBase ให้ไปที่ "ตัวจัดการปลั๊กอิน" (Plugin Manager) จากนั้นค้นหาปลั๊กอิน "China Region" แล้วเปิดใช้งานได้เลยครับ

## การใช้งาน

ในการตั้งค่าคอลเลกชัน ให้เพิ่มฟิลด์ใหม่แล้วเลือก "China Region" เป็นประเภทฟิลด์ครับ

![](/images/field-china-region-1.png)

### การตั้งค่าฟิลด์

ในการตั้งค่าฟิลด์ คุณสามารถกำหนดโหมดการแสดงผลสำหรับฟิลด์ได้ดังนี้ครับ:

-   มณฑล (Province)
-   มณฑล/เมือง (Province/City)
-   มณฑล/เมือง/เขต (Province/City/District)
-   มณฑล/เมือง/เขต/อำเภอ (Province/City/District/County)

![](/images/field-china-region-2.png)

### การตั้งค่าอินเทอร์เฟซ

ในการตั้งค่าอินเทอร์เฟซ คุณสามารถกำหนดรูปแบบอินเทอร์เฟซต่างๆ สำหรับฟิลด์ "ภูมิภาคจีน" ได้ครับ
