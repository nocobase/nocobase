# การจัดการการแปลภาษา

## บทนำ

ปลั๊กอินการจัดการการแปลภาษาใช้สำหรับจัดการและนำทรัพยากรการแปลภาษาของ NocoBase ไปใช้งานครับ/ค่ะ โดยสามารถแปลเมนูของระบบ, ตารางข้อมูล, ฟิลด์ และปลั๊กอินทั้งหมด เพื่อให้เข้ากับภาษาและวัฒนธรรมของแต่ละภูมิภาคได้ครับ/ค่ะ

## การติดตั้ง

ปลั๊กอินนี้เป็นปลั๊กอินในตัว จึงไม่จำเป็นต้องติดตั้งเพิ่มเติมครับ/ค่ะ

### การซิงค์รายการคำแปล

<img src="https://static-docs.nocobase.com/202404202134850.png"/>

ปัจจุบัน รองรับการซิงค์เนื้อหาดังต่อไปนี้ครับ/ค่ะ:

- ชุดภาษาท้องถิ่นสำหรับระบบและปลั๊กอิน
- ชื่อคอลเลกชัน, ชื่อฟิลด์ และป้ายกำกับตัวเลือกฟิลด์
- ชื่อเมนู

หลังจากซิงค์ข้อมูลเรียบร้อยแล้ว ระบบจะแสดงรายการคำที่สามารถแปลได้ทั้งหมดสำหรับภาษาปัจจุบันครับ/ค่ะ

<img src="https://static-docs.nocobase.com/202404202136567.png"/>

:::info{title=ข้อแนะนำ}
โมดูลที่แตกต่างกันอาจมีรายการคำต้นฉบับที่เหมือนกัน ซึ่งจำเป็นต้องแปลแยกกันครับ/ค่ะ
:::
