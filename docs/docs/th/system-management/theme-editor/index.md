# ตัวแก้ไขธีม

> คุณสมบัติธีมในปัจจุบันถูกพัฒนาขึ้นโดยใช้ Ant Design เวอร์ชัน 5.x ครับ/ค่ะ ขอแนะนำให้ทำความเข้าใจแนวคิดเกี่ยวกับการ[ปรับแต่งธีม](https://ant.design/docs/react/customize-theme-cn#%E8%87%AA%E5%AE%9A%E4%B9%89%E4%B8%BB%E9%A2%98)ก่อนที่จะอ่านเอกสารนี้ครับ/ค่ะ

## บทนำ

ปลั๊กอินตัวแก้ไขธีมใช้สำหรับปรับแต่งสไตล์ของหน้าเว็บส่วนหน้า (frontend) ทั้งหมดครับ/ค่ะ ปัจจุบันรองรับการแก้ไขค่าแบบ Global สำหรับ [SeedToken](https://ant.design/docs/react/customize-theme-cn#seedtoken), [MapToken](https://ant.design/docs/react/customize-theme-cn#maptoken) และ [AliasToken](https://ant.design/docs/react/customize-theme-cn#aliastoken) รวมถึงรองรับการ[สลับ](https://ant.design/docs/react/customize-theme-cn#%E4%BD%BF%E7%94%A8%E9%A2%84%E8%AE%BE%E7%AE%97%E6%B3%95)ไปใช้ `โหมดมืด (Dark Mode)` และ `โหมดกะทัดรัด (Compact Mode)` ในอนาคต อาจจะรองรับการปรับแต่งธีมใน[ระดับคอมโพเนนต์](https://ant.design/docs/react/customize-theme-cn#%E4%BF%AE%E6%94%B9%E7%BB%84%E4%BB%B6%E5%8F%98%E9%87%8F-component-token)เพิ่มเติมครับ/ค่ะ

## วิธีใช้งาน

### การเปิดใช้งานปลั๊กอินตัวแก้ไขธีม

อันดับแรก อัปเดต NocoBase ให้เป็นเวอร์ชันล่าสุด (v0.11.1 หรือสูงกว่า) ก่อนครับ/ค่ะ จากนั้นไปที่หน้าจัดการปลั๊กอิน แล้วค้นหาการ์ด `ตัวแก้ไขธีม` คลิกปุ่ม `เปิดใช้งาน` ที่มุมขวาล่างของการ์ด แล้วรอให้หน้าเว็บรีเฟรชครับ/ค่ะ

![20240409132838](https://static-docs.nocobase.com/20240409132838.png)

### การเข้าสู่หน้าการตั้งค่าธีม

หลังจากเปิดใช้งานปลั๊กอินแล้ว ให้คลิกปุ่มตั้งค่าที่มุมซ้ายล่างของการ์ด เพื่อไปยังหน้าแก้ไขธีมครับ/ค่ะ โดยค่าเริ่มต้น จะมีตัวเลือกธีมให้ 4 แบบ ได้แก่ `ธีมเริ่มต้น (Default Theme)`, `ธีมมืด (Dark Theme)`, `ธีมกะทัดรัด (Compact Theme)` และ `ธีมมืดกะทัดรัด (Compact Dark Theme)` ครับ/ค่ะ

![20240409133020](https://static-docs.nocobase.com/20240409133020.png)

### การเพิ่มธีมใหม่

คลิกปุ่ม `เพิ่มธีมใหม่` แล้วเลือก `สร้างธีมใหม่ทั้งหมด` ครับ/ค่ะ ตัวแก้ไขธีมจะปรากฏขึ้นทางด้านขวาของหน้าจอ ซึ่งรองรับการแก้ไขตัวเลือกต่างๆ เช่น `สี`, `ขนาด`, `สไตล์` และอื่นๆ เมื่อแก้ไขเสร็จแล้ว ให้ป้อนชื่อธีมและคลิกบันทึก เพื่อสร้างธีมใหม่ครับ/ค่ะ

![20240409133147](https://static-docs.nocobase.com/20240409133147.png)

### การนำธีมใหม่ไปใช้งาน

เลื่อนเมาส์ไปที่มุมขวาบนของหน้าจอ คุณจะเห็นตัวเลือกสำหรับสลับธีมครับ/ค่ะ คลิกเพื่อสลับไปยังธีมอื่นๆ ได้เลย เช่น ธีมที่คุณเพิ่งสร้างไปเมื่อสักครู่ครับ/ค่ะ

![20240409133247](https://static-docs.nocobase.com/20240409133247.png)

### การแก้ไขธีมที่มีอยู่แล้ว

คลิกปุ่ม `แก้ไข` ที่มุมซ้ายล่างของการ์ดครับ/ค่ะ ตัวแก้ไขธีมจะปรากฏขึ้นทางด้านขวาของหน้าจอ (เหมือนกับการเพิ่มธีมใหม่) เมื่อแก้ไขเสร็จแล้ว ให้คลิกบันทึกเพื่อทำการเปลี่ยนแปลงธีมครับ/ค่ะ

![20240409134413](https://static-docs.nocobase.com/20240409134413.png)

### การตั้งค่าธีมที่ผู้ใช้เลือกได้

ธีมที่เพิ่มใหม่จะถูกตั้งค่าให้ผู้ใช้สามารถสลับไปใช้งานได้โดยค่าเริ่มต้นครับ/ค่ะ หากคุณไม่ต้องการให้ผู้ใช้สลับไปใช้ธีมใดธีมหนึ่ง คุณสามารถปิดสวิตช์ `ผู้ใช้เลือกได้` ที่มุมขวาล่างของการ์ดธีมได้ครับ/ค่ะ ซึ่งจะทำให้ผู้ใช้ไม่สามารถสลับไปใช้ธีมนั้นได้

![20240409133331](https://static-docs.nocobase.com/20240409133331.png)

### การตั้งค่าเป็นธีมเริ่มต้น

ในสถานะเริ่มต้น ธีมที่ถูกตั้งค่าไว้คือ `ธีมเริ่มต้น` ครับ/ค่ะ หากคุณต้องการตั้งค่าธีมใดธีมหนึ่งให้เป็นธีมเริ่มต้น คุณสามารถเปิดสวิตช์ `ธีมเริ่มต้น` ที่มุมขวาล่างของการ์ดธีมนั้นได้เลยครับ/ค่ะ ซึ่งจะทำให้ผู้ใช้เห็นธีมนี้เป็นอันดับแรกเมื่อเปิดหน้าเว็บ ข้อควรจำ: ไม่สามารถลบธีมเริ่มต้นได้ครับ/ค่ะ

![20240409133409](https://static-docs.nocobase.com/20240409133409.png)

### การลบธีม

คลิกปุ่ม `ลบ` ที่อยู่ใต้การ์ด จากนั้นยืนยันในกล่องโต้ตอบที่ปรากฏขึ้น เพื่อลบธีมได้เลยครับ/ค่ะ

![20240409133435](https://static-docs.nocobase.com/20240409133435.png)
