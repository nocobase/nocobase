# การตั้งค่าภาษา

ในส่วน "การตั้งค่าระบบ" คุณสามารถปรับเปลี่ยนสภาพแวดล้อมภาษาโดยรวมของระบบได้ โดยการตั้งค่า "ภาษาที่เปิดใช้งาน" ครับ/ค่ะ ภาษาแรกในรายการจะถูกใช้เป็นภาษาเริ่มต้นครับ/ค่ะ

![20240324192005](https://static-docs.nocobase.com/20240324192005.png)

เมื่อมีการเปิดใช้งานหลายภาษา ผู้ใช้สามารถตั้งค่าภาษาที่ต้องการได้ใน "ศูนย์ส่วนตัว" (มุมขวาบนของหน้าจอ) เพื่อกำหนดค่าภาษาที่ผู้ใช้ต้องการครับ/ค่ะ

![20240324192509](https://static-docs.nocobase.com/20240324192509.png)

หากสภาพแวดล้อมภาษาที่คุณใช้งานอยู่ยังขาดการแปล คุณสามารถใช้ปลั๊กอิน "[การจัดการการแปลภาษา](/system-management/localization/)" เพื่อเพิ่มหรือแก้ไขคำแปลได้ครับ/ค่ะ

![20240324193055](https://static-docs.nocobase.com/20240324193055.png)
