# การปรับแต่งแบรนด์

## บทนำ

ชื่อแบรนด์เริ่มต้นของระบบคือ "NocoBase" ซึ่งจะแสดงหลักๆ ใน 3 ตำแหน่งดังต่อไปนี้ครับ/ค่ะ

![20240409111100](https://static-docs.nocobase.com/20240409111100.png)

ที่ด้านล่างของหน้าเข้าสู่ระบบ:

![20240409111254](https://static-docs.nocobase.com/20240409111254.png)

ตำแหน่งที่ 1 สามารถตั้งค่าได้ผ่านตัวเลือก Logo ใน[การตั้งค่าระบบ](/system-management/system-settings/) ส่วนตำแหน่งที่ 2 และ 3 สามารถตั้งค่าได้โดยใช้ปลั๊กอินนี้ครับ/ค่ะ

## วิธีการใช้งาน

หลังจากเปิดใช้งานปลั๊กอินแล้ว ให้คลิกที่เมนู "การปรับแต่งแบรนด์" ที่มุมขวาบนของหน้า เพื่อเข้าสู่หน้าการตั้งค่าครับ/ค่ะ

![20240409113640](https://static-docs.nocobase.com/20240409113640.png)

- **แบรนด์ (Brand)**: ใช้สำหรับตั้งค่าข้อความที่อยู่ด้านล่างของหน้าเข้าสู่ระบบ (ตำแหน่งที่ 3)
- **เกี่ยวกับ (About)**: ใช้สำหรับตั้งค่าเนื้อหาของเมนูที่อยู่มุมขวาบนของหน้า (ดังที่แสดงในภาพด้านล่าง)

![20240409113915](https://static-docs.nocobase.com/20240409113915.png)

การตั้งค่ารองรับรูปแบบ HTML เพื่อให้คุณสามารถปรับแต่งเนื้อหาได้อย่างยืดหยุ่นครับ/ค่ะ หากต้องการแก้ไขข้อความง่ายๆ ก็สามารถแก้ไขจากเนื้อหาที่มีอยู่ได้โดยตรง ดังที่แสดงในภาพด้านล่างนี้ครับ/ค่ะ

![20240409114740](https://static-docs.nocobase.com/20240409114740.png)

การตั้งค่า Favicon (ไอคอนเว็บไซต์)
