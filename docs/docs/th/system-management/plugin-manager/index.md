# การจัดการปลั๊กอิน

## ตัวจัดการปลั๊กอิน

คลิกไอคอนตัวจัดการปลั๊กอินที่มุมขวาบนของหน้า เพื่อเข้าสู่หน้าตัวจัดการปลั๊กอินครับ/ค่ะ ในหน้านี้ คุณสามารถดู, ติดตั้ง, เปิดใช้งาน หรือปิดใช้งานปลั๊กอินได้ครับ/ค่ะ

![2024-04-10_08-47-45](https://static-docs.nocobase.com/2024-04-10_08-47-45.png)

## ศูนย์ตั้งค่าปลั๊กอิน

ปลั๊กอินมักจะมีหน้าตั้งค่าเป็นของตัวเองครับ/ค่ะ คลิกไอคอนการตั้งค่าที่มุมขวาบนของหน้า เพื่อดูเมนูการตั้งค่าของปลั๊กอินทั้งหมด จากนั้นคลิกรายการเมนูที่เกี่ยวข้องเพื่อตั้งค่าปลั๊กอินได้เลยครับ/ค่ะ
