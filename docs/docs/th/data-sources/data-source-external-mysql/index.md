# แหล่งข้อมูลภายนอก - MySQL

## บทนำ

ใช้ฐานข้อมูล MySQL ภายนอกเป็นแหล่งข้อมูล เวอร์ชันที่รองรับในปัจจุบัน: MySQL >= 5.7

## การติดตั้ง

ปลั๊กอินนี้เป็นปลั๊กอินเชิงพาณิชย์ ซึ่งจำเป็นต้องอัปโหลดและเปิดใช้งานผ่านตัวจัดการปลั๊กอิน
