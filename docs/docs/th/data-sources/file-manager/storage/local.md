# ที่เก็บข้อมูลภายในเครื่อง (Local Storage)

ไฟล์ที่อัปโหลดจะถูกบันทึกไว้ในไดเรกทอรีภายในเครื่องบนเซิร์ฟเวอร์ครับ/ค่ะ วิธีนี้เหมาะสำหรับสถานการณ์ที่ระบบมีการจัดการไฟล์ที่อัปโหลดในปริมาณไม่มาก หรือสำหรับการทดลองใช้งานครับ/ค่ะ

## ตัวเลือกการตั้งค่า

:::info{title=ข้อควรทราบ}
ส่วนนี้จะกล่าวถึงเฉพาะตัวเลือกเฉพาะสำหรับเอนจินจัดเก็บข้อมูลภายในเครื่องเท่านั้นครับ/ค่ะ สำหรับพารามิเตอร์ทั่วไป โปรดดูที่ [พารามิเตอร์เอนจินทั่วไป](./index.md#general-engine-parameters) ครับ/ค่ะ
:::

### พาธ (Path)

พาธนี้จะแสดงทั้งพาธสัมพัทธ์ของไฟล์ที่จัดเก็บอยู่บนเซิร์ฟเวอร์ และพาธสำหรับเข้าถึงด้วย URL ครับ/ค่ะ ตัวอย่างเช่น "`user/avatar`" (โดยไม่ต้องมีเครื่องหมาย "`/`" ทั้งที่จุดเริ่มต้นและจุดสิ้นสุด) จะหมายถึง:

1. พาธสัมพัทธ์ของไฟล์ที่อัปโหลดซึ่งจัดเก็บอยู่บนเซิร์ฟเวอร์: `/path/to/nocobase-app/storage/uploads/user/avatar`
2. คำนำหน้า URL สำหรับการเข้าถึงไฟล์: `http://localhost:13000/storage/uploads/user/avatar`
