# Tencent COS

Tencent COS เป็นเอนจินจัดเก็บข้อมูลที่ใช้ Tencent Cloud COS ครับ/ค่ะ ก่อนใช้งาน คุณต้องเตรียมบัญชีและสิทธิ์ที่เกี่ยวข้องให้พร้อมก่อนนะครับ/คะ

## ตัวเลือกการตั้งค่า

![ตัวอย่างการตั้งค่าเอนจินจัดเก็บข้อมูล Tencent COS](https://static-docs.nocobase.com/20240712222125.png)

:::info{title=เคล็ดลับ}
ส่วนนี้จะอธิบายเฉพาะพารามิเตอร์เฉพาะสำหรับเอนจินจัดเก็บข้อมูล Tencent Cloud COS เท่านั้นครับ/ค่ะ สำหรับพารามิเตอร์ทั่วไป โปรดดูที่ [พารามิเตอร์ทั่วไปของเอนจิน](./index.md#common-engine-parameters) ครับ/ค่ะ
:::

### ภูมิภาค

กรอกภูมิภาคของพื้นที่จัดเก็บ COS ครับ/ค่ะ เช่น `ap-chengdu`

:::info{title=เคล็ดลับ}
คุณสามารถดูข้อมูลภูมิภาคของ Bucket จัดเก็บข้อมูลได้ใน [Tencent Cloud COS Console](https://console.cloud.tencent.com/cos) ครับ/ค่ะ และจำเป็นต้องใช้เฉพาะส่วนที่เป็นคำนำหน้าของภูมิภาคเท่านั้น (ไม่ต้องใส่ชื่อโดเมนแบบเต็ม) ครับ/ค่ะ
:::

### SecretId

กรอก ID ของคีย์การเข้าถึงที่ได้รับอนุญาตจาก Tencent Cloud ครับ/ค่ะ

### SecretKey

กรอก Secret ของคีย์การเข้าถึงที่ได้รับอนุญาตจาก Tencent Cloud ครับ/ค่ะ

### Bucket

กรอกชื่อ Bucket ของพื้นที่จัดเก็บ COS ครับ/ค่ะ เช่น `qing-cdn-1234189398`
