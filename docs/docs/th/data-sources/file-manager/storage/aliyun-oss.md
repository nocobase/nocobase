# Aliyun OSS

เอนจินจัดเก็บข้อมูลที่ใช้ Aliyun OSS เป็นพื้นฐาน ก่อนใช้งาน คุณต้องเตรียมบัญชีผู้ใช้และสิทธิ์ที่เกี่ยวข้องให้พร้อม

## การตั้งค่า

![ตัวอย่างการตั้งค่าเอนจินจัดเก็บข้อมูล Aliyun OSS](https://static-docs.nocobase.com/20240712220011.png)

:::info{title=ข้อแนะนำ}
ในส่วนนี้จะแนะนำเฉพาะพารามิเตอร์ที่ใช้สำหรับเอนจินจัดเก็บข้อมูล Aliyun OSS โดยเฉพาะ สำหรับพารามิเตอร์ทั่วไป โปรดดูที่ [พารามิเตอร์ทั่วไปของเอนจิน](./index.md#common-engine-parameters)
:::

### ภูมิภาค

ระบุภูมิภาคของพื้นที่จัดเก็บ OSS เช่น `oss-cn-hangzhou`

:::info{title=ข้อแนะนำ}
คุณสามารถดูข้อมูลภูมิภาคของ Bucket จัดเก็บข้อมูลได้ที่ [คอนโซล Aliyun OSS](https://oss.console.aliyun.com/) และคุณเพียงแค่ใช้ส่วนที่เป็นคำนำหน้าของภูมิภาคเท่านั้น (ไม่จำเป็นต้องใช้ชื่อโดเมนแบบเต็ม)
:::

### AccessKey ID

กรอก ID ของคีย์การเข้าถึงที่ได้รับอนุญาตจาก Alibaba Cloud

### AccessKey Secret

กรอก Secret ของคีย์การเข้าถึงที่ได้รับอนุญาตจาก Alibaba Cloud

### Bucket

กรอกชื่อ Bucket ของ OSS
