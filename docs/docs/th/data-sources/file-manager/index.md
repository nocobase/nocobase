# ตัวจัดการไฟล์

## บทนำ

ปลั๊กอินตัวจัดการไฟล์นำเสนอคอลเลกชันไฟล์, ฟิลด์ไฟล์แนบ และเอนจินจัดเก็บไฟล์สำหรับการจัดการไฟล์อย่างมีประสิทธิภาพครับ/ค่ะ ไฟล์คือเรคคอร์ดในคอลเลกชันประเภทพิเศษที่เรียกว่าคอลเลกชันไฟล์ ซึ่งจัดเก็บข้อมูลเมตาของไฟล์และสามารถจัดการได้ผ่านตัวจัดการไฟล์ ฟิลด์ไฟล์แนบเป็นฟิลด์ความสัมพันธ์เฉพาะที่เชื่อมโยงกับคอลเลกชันไฟล์ ปลั๊กอินนี้รองรับวิธีการจัดเก็บหลายรูปแบบ เช่น การจัดเก็บในเครื่อง (local storage), Alibaba Cloud OSS, Amazon S3 และ Tencent Cloud COS ครับ/ค่ะ

## คู่มือการใช้งาน

### คอลเลกชันไฟล์

มีคอลเลกชัน `attachments` ในตัวสำหรับจัดเก็บไฟล์ทั้งหมดที่เชื่อมโยงกับฟิลด์ไฟล์แนบครับ/ค่ะ นอกจากนี้ ยังสามารถสร้างคอลเลกชันไฟล์ใหม่เพื่อจัดเก็บไฟล์เฉพาะได้อีกด้วยครับ/ค่ะ

[ดูรายละเอียดเพิ่มเติมได้ที่เอกสารคอลเลกชันไฟล์](/data-sources/file-manager/file-collection)

### ฟิลด์ไฟล์แนบ

ฟิลด์ไฟล์แนบเป็นฟิลด์ความสัมพันธ์เฉพาะที่เกี่ยวข้องกับคอลเลกชันไฟล์ ซึ่งสามารถสร้างได้ผ่าน "ประเภทฟิลด์ไฟล์แนบ" หรือกำหนดค่าผ่าน "ฟิลด์ความสัมพันธ์" ครับ/ค่ะ

[ดูรายละเอียดเพิ่มเติมได้ที่เอกสารฟิลด์ไฟล์แนบ](/data-sources/file-manager/field-attachment)

### เอนจินจัดเก็บไฟล์

เอนจินจัดเก็บไฟล์ใช้สำหรับบันทึกไฟล์ไปยังบริการเฉพาะ เช่น การจัดเก็บในเครื่อง (local storage) (บันทึกลงฮาร์ดไดรฟ์ของเซิร์ฟเวอร์), การจัดเก็บในคลาวด์ (cloud storage) เป็นต้นครับ/ค่ะ

[ดูรายละเอียดเพิ่มเติมได้ที่เอกสารเอนจินจัดเก็บไฟล์](./storage/index.md)

### HTTP API

การอัปโหลดไฟล์สามารถจัดการได้ผ่าน HTTP API โปรดดู [HTTP API](./http-api.md) ครับ/ค่ะ
