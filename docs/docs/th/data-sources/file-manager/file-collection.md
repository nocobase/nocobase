# คอลเลกชันไฟล์

## บทนำ

ไฟล์คือเรคคอร์ดในคอลเลกชันที่มีโครงสร้างเฉพาะ ซึ่งเราเรียกว่าคอลเลกชันไฟล์ครับ คอลเลกชันนี้ใช้สำหรับจัดเก็บข้อมูลเมตาของไฟล์ และสามารถจัดการได้ผ่านตัวจัดการไฟล์ (File Manager) ครับ
