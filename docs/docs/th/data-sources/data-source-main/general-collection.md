# คอลเลกชันทั่วไป

## บทนำ
คอลเลกชันประเภทนี้เหมาะสำหรับใช้งานในสถานการณ์ส่วนใหญ่ครับ/ค่ะ หากคุณไม่ได้ต้องการเทมเพลตคอลเลกชันแบบพิเศษ ก็สามารถใช้คอลเลกชันทั่วไปได้เลย
