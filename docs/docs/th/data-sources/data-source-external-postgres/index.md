# แหล่งข้อมูลภายนอก - PostgreSQL

## บทนำ

คุณสามารถใช้ฐานข้อมูล PostgreSQL ภายนอกเป็นแหล่งข้อมูลได้ครับ/ค่ะ เวอร์ชันที่รองรับในปัจจุบันคือ: PostgreSQL >= 9.5

## การติดตั้ง

ปลั๊กอินนี้เป็นปลั๊กอินเชิงพาณิชย์ ซึ่งจำเป็นต้องอัปโหลดและเปิดใช้งานผ่านตัวจัดการปลั๊กอินครับ/ค่ะ
