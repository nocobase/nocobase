# คอลเลกชันแบบต้นไม้

## บทนำ

คอลเลกชันโครงสร้างต้นไม้เป็นรูปแบบการออกแบบคอลเลกชันข้อมูลที่ใช้สำหรับจัดเก็บข้อมูลที่มีความสัมพันธ์แบบลำดับชั้นครับ/ค่ะ โครงสร้างคอลเลกชันนี้จะคล้ายกับโครงสร้างแบบต้นไม้ โดยที่ข้อมูลแต่ละรายการสามารถมีรายการย่อยได้หนึ่งรายการหรือมากกว่า และรายการย่อยเหล่านั้นก็สามารถมีรายการย่อยของตัวเองได้อีกทอดหนึ่งครับ/ค่ะ
