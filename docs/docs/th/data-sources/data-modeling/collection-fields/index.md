# ฟิลด์ของคอลเลกชัน

## ประเภท Interface ของฟิลด์

NocoBase แบ่งฟิลด์ออกเป็นประเภทต่างๆ โดยพิจารณาจากมุมมองของ Interface ดังนี้ครับ:

![20240512110352](https://static-docs.nocobase.com/20240512110352.png)

## ประเภทข้อมูลของฟิลด์

Field Interface แต่ละประเภทจะมีประเภทข้อมูล (Data Type) เริ่มต้นของตัวเองครับ ตัวอย่างเช่น ฟิลด์ที่มี Interface เป็นตัวเลข (Number) จะมีประเภทข้อมูลเริ่มต้นเป็น `double` แต่ก็สามารถเปลี่ยนเป็น `float`, `decimal` หรืออื่นๆ ได้ครับ ประเภทข้อมูลที่รองรับในปัจจุบันมีดังนี้ครับ:

![20240512103733](https://static-docs.nocobase.com/20240512103733.png)

## การแมปประเภทฟิลด์

ขั้นตอนการเพิ่มฟิลด์ใหม่ในฐานข้อมูลหลักมีดังนี้ครับ:

1. เลือกประเภท Interface
2. ตั้งค่าประเภทข้อมูลที่เลือกได้สำหรับ Interface ปัจจุบัน

![20240512172416](https://static-docs.nocobase.com/20240512172416.png)

ขั้นตอนการแมปฟิลด์จากแหล่งข้อมูลภายนอกมีดังนี้ครับ:

1. ระบบจะทำการแมปประเภทข้อมูล (Field type) และประเภท UI (Field Interface) ที่สอดคล้องกันโดยอัตโนมัติ โดยอ้างอิงจากประเภทฟิลด์ของฐานข้อมูลภายนอกครับ
2. สามารถแก้ไขเป็นประเภทข้อมูลและประเภท Interface ที่เหมาะสมยิ่งขึ้นได้ตามความต้องการครับ

![20240512172759](https://static-docs.nocobase.com/20240512172759.png)
