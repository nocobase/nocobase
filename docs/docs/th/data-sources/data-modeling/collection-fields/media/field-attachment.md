# ฟิลด์ไฟล์แนบ

## บทนำ

ระบบมีฟิลด์ประเภท "ไฟล์แนบ" ในตัว เพื่อรองรับการอัปโหลดไฟล์ของผู้ใช้ใน คอลเลกชัน ที่กำหนดเองครับ/ค่ะ

โดยพื้นฐานแล้ว ฟิลด์ไฟล์แนบเป็นฟิลด์ความสัมพันธ์แบบ Many-to-Many (หลายต่อหลาย) ที่เชื่อมโยงไปยัง คอลเลกชัน "ไฟล์แนบ" (`attachments`) ซึ่งเป็น คอลเลกชัน ไฟล์ในตัวของระบบครับ/ค่ะ เมื่อคุณสร้างฟิลด์ไฟล์แนบใน คอลเลกชัน ใดๆ ระบบจะสร้างตารางเชื่อมโยง (junction table) แบบ Many-to-Many ขึ้นมาโดยอัตโนมัติครับ/ค่ะ ข้อมูลเมตาของไฟล์ที่อัปโหลดจะถูกเก็บไว้ใน คอลเลกชัน "ไฟล์แนบ" และข้อมูลไฟล์ที่อ้างอิงใน คอลเลกชัน ของคุณจะเชื่อมโยงผ่านตารางเชื่อมโยงนี้ครับ/ค่ะ

### การจำกัดประเภท MIME

ใช้สำหรับจำกัดประเภทไฟล์ที่อนุญาตให้อัปโหลด โดยใช้ไวยากรณ์ [MIME](https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/MIME_types) ในการกำหนดรูปแบบครับ/ค่ะ ตัวอย่างเช่น `image/*` หมายถึงไฟล์รูปภาพครับ/ค่ะ สามารถระบุได้หลายประเภทโดยคั่นด้วยเครื่องหมายจุลภาค (comma) เช่น `image/*,application/pdf` ซึ่งหมายถึงอนุญาตทั้งไฟล์รูปภาพและไฟล์ PDF ครับ/ค่ะ

### เอนจินจัดเก็บข้อมูล

เลือกเอนจินจัดเก็บข้อมูลสำหรับไฟล์ที่อัปโหลดครับ/ค่ะ หากเว้นว่างไว้ ระบบจะใช้เอนจินจัดเก็บข้อมูลเริ่มต้นของระบบครับ/ค่ะ
