# แหล่งข้อมูลภายนอก - MariaDB

## บทนำ

ใช้ฐานข้อมูล MariaDB ภายนอกเป็นแหล่งข้อมูลครับ/ค่ะ เวอร์ชันที่รองรับในปัจจุบันคือ MariaDB >= 10.3

## การติดตั้ง

ปลั๊กอินนี้เป็นปลั๊กอินเชิงพาณิชย์ครับ/ค่ะ ซึ่งจำเป็นต้องอัปโหลดและเปิดใช้งานผ่านตัวจัดการปลั๊กอิน

## วิธีใช้งาน
