# บล็อกปฏิทิน

## แนะนำ

บล็อกปฏิทินช่วยให้คุณสามารถดูและจัดการข้อมูลที่เกี่ยวข้องกับกิจกรรมและวันที่ในรูปแบบปฏิทินได้อย่างง่ายดาย เหมาะสำหรับการจัดตารางการประชุม การวางแผนกิจกรรม และการจัดระเบียบเวลาของคุณให้มีประสิทธิภาพครับ/ค่ะ

## การติดตั้ง

ปลั๊กอินนี้เป็นแบบ Built-in (ติดตั้งมาให้ในตัว) ครับ/ค่ะ จึงไม่จำเป็นต้องติดตั้งเพิ่มเติม

## การเพิ่มบล็อก

![20250403220300](https://static-docs.nocobase.com/20250403220300.png)

1.  **ฟิลด์ชื่อเรื่อง (Title Field)**: ใช้สำหรับแสดงข้อมูลบนแถบปฏิทินครับ/ค่ะ ปัจจุบันรองรับฟิลด์ประเภท `input` (ข้อความบรรทัดเดียว), `select` (เลือกรายการเดียว), `phone` (เบอร์โทรศัพท์), `email` (อีเมล), `radioGroup` (กลุ่มปุ่มตัวเลือก) และ `sequence` (ลำดับ) ครับ/ค่ะ คุณสามารถขยายประเภทฟิลด์ชื่อเรื่องที่บล็อกปฏิทินรองรับได้ผ่านปลั๊กอินเพิ่มเติมครับ/ค่ะ
2.  **เวลาเริ่มต้น (Start Time)**: ระบุเวลาที่งานเริ่มต้นครับ/ค่ะ
3.  **เวลาสิ้นสุด (End Time)**: ระบุเวลาที่งานสิ้นสุดครับ/ค่ะ

### ฟิลด์สีพื้นหลัง

:::info{title=เคล็ดลับ}
NocoBase เวอร์ชันที่ใช้งานต้องเป็น v1.4.0-beta ขึ้นไปครับ/ค่ะ
:::

ตัวเลือกนี้ใช้สำหรับกำหนดสีพื้นหลังของกิจกรรมในปฏิทินครับ/ค่ะ มีวิธีการใช้งานดังนี้:

1.  ตารางข้อมูลปฏิทินของคุณต้องมีฟิลด์ประเภท **เลือกรายการเดียว (Single select)** หรือ **กลุ่มปุ่มตัวเลือก (Radio group)** ซึ่งฟิลด์นี้จะต้องมีการกำหนดค่าสีไว้ด้วยครับ/ค่ะ
2.  จากนั้น กลับไปที่หน้าจอการตั้งค่าบล็อกปฏิทิน และเลือกฟิลด์ที่คุณได้กำหนดค่าสีไว้เมื่อสักครู่ ในส่วนของ **ฟิลด์สีพื้นหลัง** ครับ/ค่ะ
3.  สุดท้าย คุณสามารถลองเลือกสีให้กับกิจกรรมในปฏิทิน แล้วคลิกส่ง (Submit) คุณก็จะเห็นว่าสีได้ถูกนำไปใช้งานแล้วครับ/ค่ะ

![20240914192017_rec_](https://static-docs.nocobase.com/20240914192017_rec_.gif)

### วันเริ่มต้นสัปดาห์

> รองรับในเวอร์ชัน v1.7.7 ขึ้นไป

บล็อกปฏิทินรองรับการตั้งค่าวันเริ่มต้นของสัปดาห์ครับ/ค่ะ โดยคุณสามารถเลือกให้ **วันอาทิตย์** หรือ **วันจันทร์** เป็นวันแรกของสัปดาห์ได้ วันเริ่มต้นค่าเริ่มต้นคือ **วันจันทร์** ซึ่งช่วยให้ผู้ใช้สามารถปรับการแสดงผลปฏิทินให้เข้ากับธรรมเนียมปฏิบัติของแต่ละภูมิภาค เพื่อประสบการณ์การใช้งานที่ดียิ่งขึ้นครับ/ค่ะ

### วันนี้

ปุ่ม "วันนี้" ในบล็อกปฏิทินช่วยให้คุณสามารถนำทางได้อย่างสะดวกสบายครับ/ค่ะ โดยผู้ใช้สามารถกลับไปยังหน้าปฏิทินของวันที่ปัจจุบันได้อย่างรวดเร็ว หลังจากเลื่อนดูวันที่อื่น ๆ ไปแล้ว

![20240419203514](https://static-docs.nocobase.com/20240419203514.png)
