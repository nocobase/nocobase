# การจัดการแหล่งข้อมูล

## บทนำ

NocoBase มีปลั๊กอินสำหรับจัดการแหล่งข้อมูล ซึ่งใช้สำหรับจัดการแหล่งข้อมูลและคอลเลกชันต่างๆ ครับ/ค่ะ ปลั๊กอินการจัดการแหล่งข้อมูลนี้มีหน้าที่เป็นเพียงส่วนติดต่อผู้ใช้ (interface) สำหรับการจัดการแหล่งข้อมูลทั้งหมดเท่านั้น ไม่ได้มีฟังก์ชันในการเชื่อมต่อกับแหล่งข้อมูลโดยตรง แต่จำเป็นต้องใช้งานร่วมกับปลั๊กอินแหล่งข้อมูลประเภทต่างๆ ครับ/ค่ะ แหล่งข้อมูลที่รองรับการเชื่อมต่อในปัจจุบันได้แก่:

- [ฐานข้อมูลหลัก](/data-sources/data-source-main): ฐานข้อมูลหลักของ NocoBase ที่รองรับฐานข้อมูลเชิงสัมพันธ์ (relational databases) เช่น MySQL, PostgreSQL และ MariaDB ครับ/ค่ะ
- [MySQL ภายนอก](/data-sources/data-source-external-mysql): ใช้ฐานข้อมูล MySQL ภายนอกเป็นแหล่งข้อมูลครับ/ค่ะ
- [MariaDB ภายนอก](/data-sources/data-source-external-mariadb): ใช้ฐานข้อมูล MariaDB ภายนอกเป็นแหล่งข้อมูลครับ/ค่ะ
- [PostgreSQL ภายนอก](/data-sources/data-source-external-postgres): ใช้ฐานข้อมูล PostgreSQL ภายนอกเป็นแหล่งข้อมูลครับ/ค่ะ
- [MSSQL ภายนอก](/data-sources/data-source-external-mssql): ใช้ฐานข้อมูล MSSQL (SQL Server) ภายนอกเป็นแหล่งข้อมูลครับ/ค่ะ
- [Oracle ภายนอก](/data-sources/data-source-external-oracle): ใช้ฐานข้อมูล Oracle ภายนอกเป็นแหล่งข้อมูลครับ/ค่ะ

นอกจากนี้ ยังสามารถขยายการรองรับแหล่งข้อมูลประเภทอื่นๆ เพิ่มเติมได้ผ่านปลั๊กอิน ซึ่งอาจเป็นฐานข้อมูลประเภทต่างๆ ที่นิยมใช้ หรือแพลตฟอร์มที่ให้บริการ API (SDK) ก็ได้ครับ/ค่ะ

## การติดตั้ง

เป็นปลั๊กอินที่มาพร้อมกับระบบ (built-in) จึงไม่จำเป็นต้องติดตั้งแยกต่างหากครับ/ค่ะ

## คำแนะนำการใช้งาน

เมื่อเริ่มต้นติดตั้งแอปพลิเคชัน ระบบจะจัดเตรียมแหล่งข้อมูลสำหรับจัดเก็บข้อมูลของ NocoBase ให้โดยอัตโนมัติ ซึ่งเราเรียกว่า "ฐานข้อมูลหลัก" ครับ/ค่ะ สำหรับข้อมูลเพิ่มเติม สามารถดูได้จากเอกสาร [ฐานข้อมูลหลัก](/data-sources/data-source-main/) ครับ/ค่ะ

### แหล่งข้อมูลภายนอก

NocoBase รองรับการใช้งานฐานข้อมูลภายนอกเป็นแหล่งข้อมูลครับ/ค่ะ สำหรับข้อมูลเพิ่มเติม สามารถดูได้จากเอกสาร [ฐานข้อมูลภายนอก / บทนำ](/data-sources/data-source-manager/external-database) ครับ/ค่ะ
