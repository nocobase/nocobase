# คอลเลกชันความคิดเห็น

## บทนำ

คอลเลกชันความคิดเห็นเป็นเทมเพลตตารางข้อมูลที่ออกแบบมาโดยเฉพาะสำหรับจัดเก็บความคิดเห็นและข้อเสนอแนะจากผู้ใช้ครับ/ค่ะ ด้วยฟังก์ชันความคิดเห็น คุณสามารถเพิ่มความสามารถในการแสดงความคิดเห็นให้กับตารางข้อมูลใดก็ได้ เพื่อให้ผู้ใช้สามารถพูดคุย ให้ข้อเสนอแนะ หรือใส่คำอธิบายประกอบสำหรับเรคคอร์ดที่เฉพาะเจาะจงได้ครับ/ค่ะ คอลเลกชันความคิดเห็นรองรับการแก้ไขแบบ Rich Text ซึ่งช่วยให้คุณสร้างเนื้อหาได้อย่างยืดหยุ่นครับ/ค่ะ

![comment-collection-2025-11-01-00-39-01](https://static-docs.nocobase.com/comment-collection-2025-11-01-00-39-01.png)

## คุณสมบัติเด่น

- **การแก้ไขแบบ Rich Text**: มีตัวแก้ไข Markdown (vditor) มาให้โดยค่าเริ่มต้น ซึ่งรองรับการสร้างเนื้อหาแบบ Rich Text ครับ/ค่ะ
- **เชื่อมโยงกับตารางข้อมูลใดก็ได้**: สามารถเชื่อมโยงความคิดเห็นกับเรคคอร์ดในตารางข้อมูลใดก็ได้ผ่านฟิลด์ความสัมพันธ์ครับ/ค่ะ
- **ความคิดเห็นหลายระดับ**: รองรับการตอบกลับความคิดเห็น เพื่อสร้างโครงสร้างความคิดเห็นแบบต้นไม้ครับ/ค่ะ
- **การติดตามผู้ใช้**: บันทึกผู้สร้างความคิดเห็นและเวลาที่สร้างโดยอัตโนมัติครับ/ค่ะ

## คู่มือการใช้งาน

### การสร้างคอลเลกชันความคิดเห็น

![comment-collection-2025-11-01-00-37-10](https://static-docs.nocobase.com/comment-collection-2025-11-01-00-37-10.png)

1.  ไปที่หน้าการจัดการตารางข้อมูลครับ/ค่ะ
2.  คลิกปุ่ม "สร้างคอลเลกชัน" ครับ/ค่ะ
3.  เลือกเทมเพลต "คอลเลกชันความคิดเห็น" ครับ/ค่ะ
4.  ป้อนชื่อตาราง (เช่น "ความคิดเห็นงาน", "ความคิดเห็นบทความ" เป็นต้น) ครับ/ค่ะ
5.  ระบบจะสร้างตารางความคิดเห็นพร้อมฟิลด์เริ่มต้นดังต่อไปนี้โดยอัตโนมัติครับ/ค่ะ:
    -   เนื้อหาความคิดเห็น (ประเภท Markdown vditor)
    -   สร้างโดย (เชื่อมโยงกับตารางผู้ใช้)
    -   สร้างเมื่อ (ประเภทวันที่และเวลา)

### การตั้งค่าความสัมพันธ์

เพื่อให้ความคิดเห็นสามารถเชื่อมโยงกับตารางข้อมูลเป้าหมายได้ คุณจำเป็นต้องตั้งค่าฟิลด์ความสัมพันธ์ครับ/ค่ะ:

![](https://static-docs.nocobase.com/Solution/demoE3v1-19N.gif)

1.  เพิ่มฟิลด์ความสัมพันธ์แบบ "Many-to-One" ในตารางความคิดเห็นครับ/ค่ะ
2.  เลือกตารางข้อมูลเป้าหมายที่ต้องการเชื่อมโยง (เช่น ตารางงาน, ตารางบทความ เป็นต้น) ครับ/ค่ะ
3.  กำหนดชื่อฟิลด์ (เช่น "สังกัดงาน", "สังกัดบทความ" เป็นต้น) ครับ/ค่ะ

### การใช้งานบล็อกความคิดเห็นบนหน้าเพจ

![Enable Comments Collection](https://static-docs.nocobase.com/Solution/demoE3v1-20.gif)

1.  ไปที่หน้าเพจที่คุณต้องการเพิ่มฟังก์ชันความคิดเห็นครับ/ค่ะ
2.  เพิ่มบล็อกในส่วนรายละเอียดหรือป๊อปอัปของเรคคอร์ดเป้าหมายครับ/ค่ะ
3.  เลือกประเภทบล็อก "ความคิดเห็น" ครับ/ค่ะ
4.  เลือกคอลเลกชันความคิดเห็นที่คุณเพิ่งสร้างขึ้นครับ/ค่ะ

### กรณีการใช้งานทั่วไป

-   **ระบบจัดการงาน**: สมาชิกในทีมสามารถพูดคุยและให้ข้อเสนอแนะเกี่ยวกับงานได้ครับ/ค่ะ
-   **ระบบจัดการเนื้อหา**: ผู้อ่านสามารถแสดงความคิดเห็นและโต้ตอบกับบทความได้ครับ/ค่ะ
-   **เวิร์กโฟลว์การอนุมัติ**: ผู้อนุมัติสามารถใส่คำอธิบายประกอบและให้ความเห็นเกี่ยวกับแบบฟอร์มการสมัครได้ครับ/ค่ะ
-   **ข้อเสนอแนะจากลูกค้า**: รวบรวมความคิดเห็นของลูกค้าเกี่ยวกับผลิตภัณฑ์หรือบริการครับ/ค่ะ

## ข้อควรทราบ

-   คอลเลกชันความคิดเห็นเป็นคุณสมบัติของปลั๊กอินเชิงพาณิชย์ ซึ่งต้องเปิดใช้งานปลั๊กอินความคิดเห็นก่อนจึงจะใช้งานได้ครับ/ค่ะ
-   ขอแนะนำให้ตั้งค่าสิทธิ์ที่เหมาะสมสำหรับตารางความคิดเห็น เพื่อควบคุมว่าใครสามารถดู สร้าง และลบความคิดเห็นได้บ้างครับ/ค่ะ
-   สำหรับสถานการณ์ที่มีความคิดเห็นจำนวนมาก ขอแนะนำให้เปิดใช้งานการแบ่งหน้า (Pagination) เพื่อเพิ่มประสิทธิภาพครับ/ค่ะ
