# แหล่งข้อมูลภายนอก - MSSQL

## บทนำ

ปลั๊กอินนี้ช่วยให้คุณสามารถใช้ฐานข้อมูล MSSQL ภายนอกเป็นแหล่งข้อมูลได้ครับ/ค่ะ เวอร์ชันที่รองรับในปัจจุบันคือ SQL Server 2014-2019

## การติดตั้ง

ปลั๊กอินนี้เป็นปลั๊กอินเชิงพาณิชย์ ซึ่งจำเป็นต้องอัปโหลดและเปิดใช้งานผ่านตัวจัดการปลั๊กอินครับ/ค่ะ
