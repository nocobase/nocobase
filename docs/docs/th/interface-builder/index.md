# การสร้าง UI

## WYSIWYG

NocoBase มีโหมดการสร้าง UI แบบ WYSIWYG (What You See Is What You Get) ครับ/ค่ะ คุณสามารถคลิกปุ่ม "สร้าง UI" เพื่อสลับระหว่างโหมดแก้ไข (Edit mode) และโหมดดู (View mode) ได้เลยครับ/ค่ะ

โหมดแก้ไข (Edit mode):

![20251023215951](https://static-docs.nocobase.com/20251023215951.png)

## เทมเพลตเค้าโครง

NocoBase มีเทมเพลตเค้าโครงเริ่มต้นมาให้ใช้งานครับ/ค่ะ โดยส่วนบนและด้านซ้ายจะเป็นพื้นที่นำทาง (navigation area) ส่วนด้านขวาจะเป็นพื้นที่เนื้อหา (content area) ครับ/ค่ะ

![未命名.002](https://static-docs.nocobase.com/未命名.002.jpeg)

## ตัวเลือกการตั้งค่า

เมื่อเข้าสู่โหมดแก้ไข (Edit mode) คุณจะเห็นเครื่องหมายสีส้มปรากฏขึ้นบนหน้าจอ ซึ่งบ่งบอกถึงองค์ประกอบที่สามารถตั้งค่าได้ครับ/ค่ะ โดยปกติแล้ว จุดเข้าถึงการตั้งค่าของแต่ละองค์ประกอบจะอยู่ที่มุมขวาบนขององค์ประกอบนั้นๆ ครับ/ค่ะ องค์ประกอบส่วนใหญ่รองรับการตั้งค่าแบบเรียลไทม์พร้อมการแสดงตัวอย่างผลลัพธ์ได้ทันทีครับ/ค่ะ นี่คือตัวอย่างบางส่วน:
