# ร่างแบบฟอร์ม (Form Drafts)

เมื่อเปิดใช้งานฟีเจอร์ร่างแบบฟอร์ม ข้อมูลที่กำลังกรอกจะถูกบันทึกไว้ในเครื่องโดยอัตโนมัติ แม้ว่าจะปิดแบบฟอร์มโดยไม่ตั้งใจหรือรีเฟรชหน้าเว็บ ข้อมูลที่กรอกไปแล้วก็จะไม่สูญหายครับ

![20251213093717](https://static-docs.nocobase.com/20251213093717.png)

เมื่อมีข้อมูลร่างอยู่ จะมีข้อความแจ้งเตือนแสดงที่ด้านบนของแบบฟอร์ม หลังจากส่งข้อมูลสำเร็จแล้ว ร่างจะถูกลบโดยอัตโนมัติ หรือหากไม่ต้องการใช้งานแล้ว ก็สามารถลบออกด้วยตนเองได้ทุกเมื่อครับ

![20251213092734](https://static-docs.nocobase.com/20251213092734.png)

::: tip คำแนะนำ

ในปัจจุบัน ร่างแบบฟอร์มรองรับเฉพาะการแคชในเครื่อง (Local Caching) เท่านั้น ในเวอร์ชันถัดไปจะมีการรองรับการแคชบนเซิร์ฟเวอร์ เพื่อให้สามารถซิงโครไนซ์ข้อมูลข้ามอุปกรณ์ได้ครับ

:::
