# ฟอร์มบล็อก

## บทนำ

ฟอร์มบล็อกเป็นบล็อกสำคัญที่ใช้สำหรับสร้างส่วนติดต่อผู้ใช้ในการป้อนและแก้ไขข้อมูลครับ/ค่ะ มีความยืดหยุ่นสูงในการปรับแต่ง โดยจะใช้คอมโพเนนต์ที่เหมาะสมเพื่อแสดงฟิลด์ที่ต้องการตามโมเดลข้อมูล ฟอร์มบล็อกสามารถแสดงฟิลด์แบบไดนามิกได้ผ่านการทำงานร่วมกันของกฎการเชื่อมโยง (linkage rules) และอีเวนต์ต่างๆ ครับ/ค่ะ นอกจากนี้ ยังสามารถทำงานร่วมกับ **เวิร์กโฟลว์** เพื่อเรียกใช้กระบวนการอัตโนมัติและจัดการข้อมูล ซึ่งจะช่วยเพิ่มประสิทธิภาพในการทำงาน หรือจัดลำดับการทำงานเชิงตรรกะได้ครับ/ค่ะ

## การเพิ่มฟอร์มบล็อก

- **แก้ไขฟอร์ม**: ใช้สำหรับแก้ไขข้อมูลที่มีอยู่แล้วครับ/ค่ะ
- **เพิ่มฟอร์ม**: ใช้สำหรับสร้างรายการข้อมูลใหม่ครับ/ค่ะ

## การตั้งค่าบล็อก

![20251023191448](https://static-docs.nocobase.com/20251023191448.png)

### กฎการเชื่อมโยงบล็อก

ใช้กฎการเชื่อมโยงเพื่อควบคุมพฤติกรรมของบล็อก (เช่น การแสดงผล หรือการรัน JavaScript) ครับ/ค่ะ

![20251023191703](https://static-docs.nocobase.com/20251023191703.png)

ดูรายละเอียดเพิ่มเติมได้ที่ [กฎการเชื่อมโยงบล็อก](/interface-builder/blocks/block-settings/block-linkage-rule)

### กฎการเชื่อมโยงฟิลด์

ใช้กฎการเชื่อมโยงเพื่อควบคุมพฤติกรรมของฟิลด์ในฟอร์มครับ/ค่ะ

![20251023191849](https://static-docs.nocobase.com/20251023191849.png)

ดูรายละเอียดเพิ่มเติมได้ที่ [กฎการเชื่อมโยงฟิลด์](/interface-builder/blocks/block-settings/field-linkage-rule)

### รูปแบบการจัดวาง (Layout)

ฟอร์มบล็อกรองรับรูปแบบการจัดวาง (layout) สองแบบ ซึ่งสามารถตั้งค่าได้ผ่านคุณสมบัติ `layout` ครับ/ค่ะ:

- **horizontal** (การจัดวางแนวนอน): รูปแบบนี้จะแสดงป้ายกำกับ (label) และเนื้อหาในบรรทัดเดียวกัน ช่วยประหยัดพื้นที่แนวตั้ง เหมาะสำหรับฟอร์มที่เรียบง่ายหรือมีข้อมูลไม่มากครับ/ค่ะ
- **vertical** (การจัดวางแนวตั้ง) (ค่าเริ่มต้น): ป้ายกำกับจะอยู่เหนือฟิลด์ รูปแบบนี้ช่วยให้ฟอร์มอ่านและกรอกได้ง่ายขึ้น โดยเฉพาะอย่างยิ่งสำหรับฟอร์มที่มีหลายฟิลด์หรือรายการป้อนข้อมูลที่ซับซ้อนครับ/ค่ะ

![20251023193638](https://static-docs.nocobase.com/20251023193638.png)

## การกำหนดค่าฟิลด์

### ฟิลด์ของ **คอลเลกชัน** นี้

> **หมายเหตุ**: ฟิลด์จาก **คอลเลกชัน** ที่สืบทอดมา (เช่น ฟิลด์จาก **คอลเลกชัน** หลัก) จะถูกรวมและแสดงในรายการฟิลด์ปัจจุบันโดยอัตโนมัติครับ/ค่ะ
