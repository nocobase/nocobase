# บล็อกตาราง

## บทนำ

บล็อกตารางเป็นหนึ่งในบล็อกข้อมูลหลักที่มาพร้อมกับ **NocoBase** ครับ/ค่ะ โดยมีวัตถุประสงค์หลักเพื่อแสดงและจัดการข้อมูลที่มีโครงสร้างในรูปแบบตาราง บล็อกนี้มีตัวเลือกการตั้งค่าที่ยืดหยุ่น ผู้ใช้สามารถปรับแต่งคอลัมน์ ความกว้างของคอลัมน์ กฎการเรียงลำดับ และขอบเขตข้อมูลของตารางได้ตามความต้องการ เพื่อให้แน่ใจว่าข้อมูลที่แสดงในตารางนั้นตรงตามความต้องการทางธุรกิจเฉพาะของคุณครับ/ค่ะ

#### คุณสมบัติหลัก:
- **การตั้งค่าคอลัมน์ที่ยืดหยุ่น**: คุณสามารถปรับแต่งคอลัมน์และความกว้างของคอลัมน์ในตารางได้ตามความต้องการในการแสดงข้อมูลที่แตกต่างกันครับ/ค่ะ
- **กฎการเรียงลำดับ**: รองรับการเรียงลำดับข้อมูลในตาราง ผู้ใช้สามารถจัดเรียงข้อมูลแบบจากน้อยไปมาก (ascending) หรือจากมากไปน้อย (descending) ตามฟิลด์ต่างๆ ได้ครับ/ค่ะ
- **การตั้งค่าขอบเขตข้อมูล**: ด้วยการตั้งค่าขอบเขตข้อมูล ผู้ใช้สามารถควบคุมช่วงของข้อมูลที่จะแสดง เพื่อหลีกเลี่ยงข้อมูลที่ไม่เกี่ยวข้องครับ/ค่ะ
- **การตั้งค่าการดำเนินการ**: บล็อกตารางมีตัวเลือกการดำเนินการในตัวที่หลากหลาย ผู้ใช้สามารถตั้งค่าการดำเนินการต่างๆ เช่น การกรอง (Filter), การเพิ่มใหม่ (Add New), การแก้ไข (Edit) และการลบ (Delete) ได้อย่างง่ายดาย เพื่อการจัดการข้อมูลที่รวดเร็วครับ/ค่ะ
- **แก้ไขด่วน**: รองรับการแก้ไขข้อมูลโดยตรงภายในตาราง ช่วยให้กระบวนการทำงานง่ายขึ้นและเพิ่มประสิทธิภาพในการทำงานครับ/ค่ะ

## การตั้งค่าบล็อก

![20251023150819](https://static-docs.nocobase.com/20251023150819.png)

### กฎการเชื่อมโยงบล็อก

ควบคุมพฤติกรรมของบล็อก (เช่น การแสดงผล หรือการรัน JavaScript) ผ่านกฎการเชื่อมโยงครับ/ค่ะ

![20251023194550](https://static-docs.nocobase.com/20251023194550.png)

ดูรายละเอียดเพิ่มเติมได้ที่ [กฎการเชื่อมโยง](/interface-builder/linkage-rule)

### การตั้งค่าขอบเขตข้อมูล

ตัวอย่าง: กรองคำสั่งซื้อที่ "สถานะ" เป็น "ชำระเงินแล้ว" โดยค่าเริ่มต้นครับ/ค่ะ

![20251023150936](https://static-docs.nocobase.com/20251023150936.png)

ดูรายละเอียดเพิ่มเติมได้ที่ [การตั้งค่าขอบเขตข้อมูล](/interface-builder/blocks/block-settings/data-scope)

### การตั้งค่ากฎการเรียงลำดับ

ตัวอย่าง: แสดงคำสั่งซื้อตามวันที่จากมากไปน้อยครับ/ค่ะ

![20251023155114](https://static-docs.nocobase.com/20251023155114.png)

ดูรายละเอียดเพิ่มเติมได้ที่ [การตั้งค่ากฎการเรียงลำดับ](/interface-builder/blocks/block-settings/sorting-rule)

### เปิดใช้งานแก้ไขด่วน

เปิดใช้งาน "แก้ไขด่วน" ในการตั้งค่าบล็อกและการตั้งค่าคอลัมน์ตาราง เพื่อกำหนดว่าคอลัมน์ใดบ้างที่สามารถแก้ไขได้อย่างรวดเร็วครับ/ค่ะ

![20251023190149](https://static-docs.nocobase.com/20251023190149.png)

![20251023190519](https://static-docs.nocobase.com/20251023190519.gif)

### เปิดใช้งานตารางแบบ Tree

เมื่อตารางข้อมูลเป็นตารางแบบลำดับชั้น (tree) บล็อกตารางสามารถเลือกเปิดใช้งานฟังก์ชัน **"เปิดใช้งานตารางแบบ Tree"** ได้ครับ/ค่ะ โดยค่าเริ่มต้นฟังก์ชันนี้จะปิดอยู่ เมื่อเปิดใช้งาน บล็อกจะแสดงข้อมูลในโครงสร้างแบบ Tree พร้อมรองรับตัวเลือกการตั้งค่าและการดำเนินการที่เกี่ยวข้องครับ/ค่ะ

![20251125205918](https://static-docs.nocobase.com/20251125205918.png)

### ขยายทุกแถวโดยค่าเริ่มต้น

เมื่อเปิดใช้งานตารางแบบ Tree บล็อกจะรองรับการขยายข้อมูลย่อยทั้งหมดโดยอัตโนมัติเมื่อโหลดบล็อกครับ/ค่ะ
