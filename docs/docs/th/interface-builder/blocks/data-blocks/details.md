# บล็อกรายละเอียด

## แนะนำ

บล็อกรายละเอียดใช้สำหรับแสดงค่าฟิลด์ต่างๆ ของข้อมูลแต่ละรายการครับ/ค่ะ บล็อกนี้รองรับการจัดวางฟิลด์ที่ยืดหยุ่น และมีฟังก์ชันการดำเนินการข้อมูลหลากหลายในตัว ช่วยให้ผู้ใช้ดูและจัดการข้อมูลได้อย่างสะดวกสบายครับ/ค่ะ

## การตั้งค่าบล็อก

![20251029202614](https://static-docs.nocobase.com/20251029202614.png)

### กฎการเชื่อมโยงบล็อก

คุณสามารถควบคุมพฤติกรรมของบล็อกได้ผ่านกฎการเชื่อมโยงครับ/ค่ะ (เช่น จะให้แสดงผลหรือไม่ หรือจะให้รัน JavaScript)

![20251023195004](https://static-docs.nocobase.com/20251023195004.png)
ดูรายละเอียดเพิ่มเติมได้ที่ [กฎการเชื่อมโยง](/interface-builder/linkage-rule)

### กำหนดขอบเขตข้อมูล

ตัวอย่าง: แสดงเฉพาะรายการสั่งซื้อที่ชำระเงินแล้ว

![20251023195051](https://static-docs.nocobase.com/20251023195051.png)

ดูรายละเอียดเพิ่มเติมได้ที่ [การกำหนดขอบเขตข้อมูล](/interface-builder/blocks/block-settings/data-scope)

### กฎการเชื่อมโยงฟิลด์

กฎการเชื่อมโยงในบล็อกรายละเอียดรองรับการตั้งค่าฟิลด์แบบไดนามิก เพื่อให้แสดงหรือซ่อนฟิลด์ได้ครับ/ค่ะ

ตัวอย่าง: ไม่แสดงจำนวนเงิน เมื่อสถานะการสั่งซื้อเป็น "ยกเลิก"

![20251023200739](https://static-docs.nocobase.com/20251023200739.png)

ดูรายละเอียดเพิ่มเติมได้ที่ [กฎการเชื่อมโยง](/interface-builder/linkage-rule)

## การกำหนดค่าฟิลด์

### ฟิลด์จากคอลเลกชันนี้

> **หมายเหตุ**: ฟิลด์จากคอลเลกชันที่สืบทอดมา (หรือฟิลด์จากคอลเลกชันแม่) จะถูกรวมและแสดงในรายการฟิลด์ปัจจุบันโดยอัตโนมัติครับ/ค่ะ

![20251023201012](https://static-docs.nocobase.com/20251023201012.png)

### ฟิลด์จากคอลเลกชันที่เกี่ยวข้อง

> **หมายเหตุ**: รองรับการแสดงฟิลด์จากคอลเลกชันที่เกี่ยวข้องครับ/ค่ะ (ปัจจุบันรองรับเฉพาะความสัมพันธ์แบบ One-to-One เท่านั้น)

![20251023201258](https://static-docs.nocobase.com/20251023201258.png)

### ฟิลด์อื่นๆ
- JS Field
- JS Item
- ตัวแบ่ง
- Markdown

![20251023201514](https://static-docs.nocobase.com/20251023201514.png)

> **เคล็ดลับ**: คุณสามารถเขียน JavaScript เพื่อสร้างเนื้อหาการแสดงผลแบบกำหนดเองได้ ซึ่งช่วยให้แสดงข้อมูลที่ซับซ้อนมากขึ้นได้ครับ/ค่ะ  
> ตัวอย่างเช่น คุณสามารถเรนเดอร์การแสดงผลที่แตกต่างกันได้ตามประเภทข้อมูล เงื่อนไข หรือตรรกะที่แตกต่างกันครับ/ค่ะ

![20251023202017](https://static-docs.nocobase.com/20251023202017.png)
