# แผนภูมิ

## บทนำ

ปลั๊กอิน Data Visualization ของ NocoBase มาพร้อมกับการเรียกดูข้อมูลแบบภาพ และชุดคอมโพเนนต์แผนภูมิที่หลากหลาย ผู้ใช้สามารถสร้างแดชบอร์ดการแสดงผลข้อมูลได้อย่างรวดเร็วด้วยการตั้งค่าที่ไม่ซับซ้อน เพื่อแสดงข้อมูลเชิงลึก และรองรับการวิเคราะห์และนำเสนอข้อมูลแบบหลายมิติครับ/ค่ะ
