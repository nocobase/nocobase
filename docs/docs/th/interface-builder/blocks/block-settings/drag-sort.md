# การจัดเรียงแบบลากและวาง

## บทนำ

การจัดเรียงแบบลากและวาง (Drag-and-drop sorting) จะขึ้นอยู่กับฟิลด์การจัดเรียง (Sort field) เพื่อใช้สำหรับการจัดเรียงลำดับของเรกคอร์ดในบล็อกด้วยตนเองครับ

:::info{title=คำแนะนำ}
* เมื่อใช้ฟิลด์การจัดเรียงเดียวกันในหลายบล็อกเพื่อจัดเรียงแบบลากและวาง อาจส่งผลกระทบต่อลำดับที่มีอยู่เดิมได้ครับ
* เมื่อใช้การจัดเรียงแบบลากและวางในตาราง ฟิลด์การจัดเรียงจะไม่สามารถตั้งค่ากฎการจัดกลุ่ม (Grouping rules) ได้ครับ
* ตารางแบบต้นไม้ (Tree table) รองรับการจัดเรียงเฉพาะโหนดที่อยู่ในระดับเดียวกันเท่านั้นครับ
:::

## การกำหนดค่า

เพิ่มฟิลด์ประเภท "Sort" (การจัดเรียง) โดยเมื่อสร้างคอลเลกชัน ระบบจะไม่สร้างฟิลด์การจัดเรียงให้โดยอัตโนมัติอีกต่อไป จำเป็นต้องสร้างขึ้นเองด้วยตนเองครับ

![](https://static-docs.nocobase.com/470891c7bb34c506328c1f3824a6cf20.png)

เมื่อเปิดใช้งานการจัดเรียงแบบลากและวางสำหรับตาราง คุณจำเป็นต้องเลือกฟิลด์การจัดเรียงที่ต้องการครับ

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_50_AM.png)

## คำอธิบายกฎการจัดเรียง

สมมติว่าลำดับปัจจุบันคือ:

```
[1,2,3,4,5,6,7,8,9]
```

เมื่อย้ายองค์ประกอบหนึ่ง (เช่น 5) ไปข้างหน้ายังตำแหน่งของ 3 ค่าการจัดเรียงของ 3, 4 และ 5 เท่านั้นที่จะเปลี่ยนแปลง: โดย 5 จะเข้าไปแทนที่ตำแหน่งของ 3 ส่วน 3 และ 4 จะขยับถอยหลังไปหนึ่งตำแหน่งครับ

```
[1,2,5,3,4,6,7,8,9]
```

ในตอนนี้ หากย้าย 6 ไปข้างหลังยังตำแหน่งของ 8 ตัวเลข 6 จะเข้าไปแทนที่ตำแหน่งของ 8 ส่วน 7 และ 8 จะขยับไปข้างหน้าหนึ่งตำแหน่งครับ

```
[1,2,5,3,4,7,8,6,9]
```
