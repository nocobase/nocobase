# ตั้งค่าขอบเขตข้อมูล

## บทนำ

การตั้งค่าขอบเขตข้อมูลคือการกำหนดเงื่อนไขการกรองเริ่มต้นสำหรับบล็อกข้อมูลครับ/ค่ะ ผู้ใช้สามารถปรับเปลี่ยนขอบเขตข้อมูลได้อย่างยืดหยุ่นตามความต้องการทางธุรกิจ แต่ไม่ว่าจะมีการกรองข้อมูลแบบใดก็ตาม ระบบจะนำเงื่อนไขการกรองเริ่มต้นนี้ไปใช้โดยอัตโนมัติ เพื่อให้มั่นใจว่าข้อมูลจะยังคงอยู่ในขอบเขตที่กำหนดไว้เสมอครับ/ค่ะ

## คู่มือการใช้งาน

![20251027110053](https://static-docs.nocobase.com/20251027110053.png)

ฟิลด์สำหรับกรองข้อมูลรองรับการเลือกฟิลด์จากคอลเลกชันปัจจุบันและคอลเลกชันที่เกี่ยวข้องครับ/ค่ะ

![20251027110140](https://static-docs.nocobase.com/20251027110140.png)

### ตัวดำเนินการ (Operators)

ฟิลด์ข้อมูลแต่ละประเภทจะรองรับตัวดำเนินการที่แตกต่างกันครับ/ค่ะ เช่น ฟิลด์ข้อความจะรองรับตัวดำเนินการอย่าง "เท่ากับ" (equals), "ไม่เท่ากับ" (not equals) และ "มี" (contains) ส่วนฟิลด์ตัวเลขจะรองรับตัวดำเนินการอย่าง "มากกว่า" (greater than) และ "น้อยกว่า" (less than) ในขณะที่ฟิลด์วันที่จะรองรับตัวดำเนินการอย่าง "อยู่ในช่วง" (is within) หรือ "ก่อนวันที่" (is before a specific date) เป็นต้นครับ/ค่ะ

![20251027111124](https://static-docs.nocobase.com/20251027111124.png)

### ค่าคงที่ (Static Value)

ตัวอย่าง: กรองข้อมูลตาม "สถานะ" ของคำสั่งซื้อครับ/ค่ะ

![20251027111229](https://static-docs.nocobase.com/20251027111229.png)

### ค่าตัวแปร (Variable Value)

ตัวอย่าง: กรองข้อมูลคำสั่งซื้อสำหรับผู้ใช้ปัจจุบันครับ/ค่ะ

![20251027113349](https://static-docs.nocobase.com/20251027113349.png)

สำหรับข้อมูลเพิ่มเติมเกี่ยวกับตัวแปร โปรดดูที่ [ตัวแปร](/interface-builder/variables) ครับ/ค่ะ
