# ลบบล็อก

## บทนำ

การลบบล็อกเป็นการนำบล็อกออกจากหน้าปัจจุบัน และเป็นการดำเนินการที่ไม่สามารถย้อนกลับได้ครับ/ค่ะ หากคุณได้สำรองข้อมูลไว้ก่อนลบ ก็สามารถกู้คืนได้จากข้อมูลสำรองนั้นครับ/ค่ะ
