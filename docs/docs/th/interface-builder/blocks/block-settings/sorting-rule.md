# ตั้งค่ากฎการเรียงลำดับ

## บทนำ

การตั้งค่ากฎการเรียงลำดับคือการกำหนดกฎการเรียงลำดับเริ่มต้นสำหรับบล็อกข้อมูล เพื่อให้ข้อมูลในบล็อกนั้นแสดงผลตามลำดับที่เราต้องการครับ/ค่ะ
