# ความสูงของบล็อก

## บทนำ

ความสูงของบล็อกรองรับ 3 รูปแบบ: **ความสูงเริ่มต้น**, **ระบุความสูง**, และ **ความสูงเต็มหน้าจอ** บล็อกส่วนใหญ่รองรับการตั้งค่าความสูงครับ

![20260211153947](https://static-docs.nocobase.com/20260211153947.png)

![20260211154020](https://static-docs.nocobase.com/20260211154020.png)

## รูปแบบความสูง

### ความสูงเริ่มต้น

กลยุทธ์ความสูงเริ่มต้นจะแตกต่างกันไปตามประเภทของบล็อก ตัวอย่างเช่น บล็อกตารางและแบบฟอร์มจะปรับความสูงตามเนื้อหาโดยอัตโนมัติ และจะไม่มีแถบเลื่อน (Scrollbar) ปรากฏขึ้นภายในบล็อกครับ

### ระบุความสูง

คุณสามารถระบุความสูงรวมของกรอบภายนอกบล็อกได้ด้วยตนเอง โดยภายในบล็อกจะคำนวณและจัดสรรความสูงที่ใช้งานได้โดยอัตโนมัติครับ

![20260211154043](https://static-docs.nocobase.com/20260211154043.gif)

### ความสูงเต็มหน้าจอ

รูปแบบความสูงเต็มหน้าจอมีความคล้ายคลึงกับการระบุความสูง แต่ความสูงของบล็อกจะถูกคำนวณตาม **พื้นที่ที่มองเห็น (Viewport)** ของเบราว์เซอร์ในปัจจุบันเพื่อให้ได้ความสูงสูงสุดเต็มหน้าจอ หน้าเว็บของเบราว์เซอร์จะไม่มีแถบเลื่อน โดยแถบเลื่อนจะปรากฏเฉพาะภายในบล็อกเท่านั้นครับ

การจัดการแถบเลื่อนภายในในรูปแบบความสูงเต็มหน้าจอจะมีความแตกต่างกันเล็กน้อยในแต่ละบล็อก:

- **ตาราง (Table)**: เลื่อนภายใน `tbody`
- **แบบฟอร์ม (Form) / รายละเอียด (Details)**: เลื่อนภายใน Grid (เลื่อนเฉพาะเนื้อหา ยกเว้นส่วนพื้นที่ดำเนินการ)
- **รายการ (List) / การ์ดกริด (Grid Card)**: เลื่อนภายใน Grid (เลื่อนเฉพาะเนื้อหา ยกเว้นส่วนพื้นที่ดำเนินการและแถบแบ่งหน้า)
- **แผนที่ (Map) / ปฏิทิน (Calendar)**: ปรับความสูงตามภาพรวมโดยอัตโนมัติ และไม่มีแถบเลื่อน
- **Iframe / Markdown**: จำกัดความสูงรวมของกรอบบล็อก และแถบเลื่อนจะปรากฏภายในบล็อก
