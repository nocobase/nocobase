# การจัดวางเลย์เอาต์

## บทนำ

บล็อกประเภทฟอร์ม เช่น บล็อกฟอร์ม, บล็อกรายละเอียด และบล็อกรายการ รองรับการตั้งค่าการจัดวางเลย์เอาต์ครับ/ค่ะ การจัดวางเลย์เอาต์แบ่งออกเป็นสองแบบ ได้แก่ การจัดวางแนวตั้ง (ที่แสดงป้ายกำกับและช่องป้อนข้อมูลเรียงกันทีละบรรทัด) และการจัดวางแนวนอน (ที่แสดงป้ายกำกับและช่องป้อนข้อมูลเคียงข้างกัน) ผู้ใช้สามารถปรับการจัดวางฟิลด์ของบล็อกได้ตามความต้องการ เพื่อให้การป้อนข้อมูลหรือการดูข้อมูลมีความชัดเจนและมีประสิทธิภาพครับ/ค่ะ

### แนวตั้ง (ค่าเริ่มต้น)

เลย์เอาต์เริ่มต้นคือแนวตั้งครับ/ค่ะ

![20251024120143](https://static-docs.nocobase.com/20251024120143.png)

### แนวนอน

การจัดวางเลย์เอาต์แนวนอนจะแสดงป้ายกำกับและช่องป้อนข้อมูลเคียงข้างกัน ซึ่งช่วยประหยัดพื้นที่ได้อย่างมีประสิทธิภาพครับ/ค่ะ สำหรับฟิลด์ที่มีชื่อยาว คุณสามารถเลือกตั้งค่าให้ข้อความขึ้นบรรทัดใหม่ (wrap) หรือแสดงเป็นจุดไข่ปลา (ellipsis) เพื่อให้มั่นใจว่าอินเทอร์เฟซดูเรียบร้อยและสวยงามครับ/ค่ะ

![20251024120237](https://static-docs.nocobase.com/20251024120237.png)

#### การจัดตำแหน่งชื่อฟิลด์

- **จัดชิดซ้าย** (ค่าเริ่มต้น)
- **จัดชิดขวา**

#### ความกว้างของชื่อฟิลด์

- ความกว้างเริ่มต้นคือ **120px** โดยมีความกว้างขั้นต่ำที่จำกัดไว้ที่ **50px** ครับ/ค่ะ

#### การจัดการเมื่อชื่อฟิลด์เกินความกว้างที่กำหนด

- **ขึ้นบรรทัดใหม่**
- **แสดงเป็นจุดไข่ปลา**

#### แสดงเครื่องหมายทวิภาค (Colon) หรือไม่

คุณสามารถเลือกได้ว่าจะให้แสดงเครื่องหมายทวิภาค (colon) หลังชื่อฟิลด์หรือไม่ครับ/ค่ะ
