# รูปแบบการโหลดข้อมูล

## บทนำ

รูปแบบการโหลดข้อมูลเริ่มต้นคือ "โหลดข้อมูลทั้งหมดเมื่อเงื่อนไขการกรองว่างเปล่า" โดย "เงื่อนไขการกรอง" ในที่นี้หมายถึงค่าใน "บล็อกตัวกรอง" (Filter block) ส่วนอีกตัวเลือกหนึ่งคือ "ไม่โหลดข้อมูลเมื่อเงื่อนไขการกรองว่างเปล่า" ครับ
