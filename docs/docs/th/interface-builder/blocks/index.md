# บล็อก

บล็อกเป็นส่วนประกอบที่ใช้แสดงข้อมูลและเนื้อหาครับ/ค่ะ สามารถวางไว้ในหน้า (Page), กล่องโต้ตอบ (Modal) หรือลิ้นชัก (Drawer) ได้ และสามารถลากจัดเรียงบล็อกหลายๆ บล็อกได้อย่างอิสระครับ/ค่ะ

## ประเภทของบล็อก

![区块类型](https://static-docs.nocobase.com/f71af45b5cd914ea0558f760ddbbba58.png)

- บล็อกข้อมูล: ใช้สำหรับแสดงข้อมูลจาก**แหล่งข้อมูล**บนหน้าจอ
- บล็อกตัวกรอง: ใช้ข้อมูลจาก**แหล่งข้อมูล**เป็นเงื่อนไขในการกรอง เพื่อกรองข้อมูลในบล็อกข้อมูลอื่นๆ
- บล็อกอื่นๆ: ใช้สำหรับแสดงเนื้อหาเฉพาะทางหรือเนื้อหาอิสระ เช่น รายการสิ่งที่ต้องทำของ**เวิร์กโฟลว์**, บันทึกการตรวจสอบ (Audit Log) หรือ Markdown เป็นต้น

## การเพิ่มบล็อก

บล็อกสามารถวางไว้ในหน้า (Page), กล่องโต้ตอบ (Modal) หรือลิ้นชัก (Drawer) ได้

### บล็อกในหน้า (Page)

ปัจจุบัน ประเภทของบล็อกที่สามารถใช้ในหน้า (Page) ได้แก่ บล็อกข้อมูล, บล็อกตัวกรอง และบล็อกอื่นๆ

![20251023222441](https://static-docs.nocobase.com/20251023222441.png)

### บล็อกในหน้าต่างป๊อปอัป (กล่องโต้ตอบหรือลิ้นชัก)

หน้าต่างป๊อปอัปมีสองประเภทคือ กล่องโต้ตอบและลิ้นชัก ซึ่งสามารถเพิ่มบล็อกได้เช่นเดียวกับหน้า (Page) ครับ/ค่ะ แต่ความแตกต่างคือ บล็อกฟอร์มในหน้าต่างป๊อปอัปมักจะใช้สำหรับการเพิ่ม, แก้ไข หรือดูข้อมูลทีละรายการ โดยประเภทของบล็อกที่ใช้ได้คือ บล็อกข้อมูลและบล็อกอื่นๆ ครับ/ค่ะ

![20251023222613](https://static-docs.nocobase.com/20251023222613.png)

## ตัวออกแบบบล็อก

บล็อกแต่ละบล็อกจะมีไอคอนเล็กๆ สามอันอยู่ที่มุมขวาบน โดยเรียงจากซ้ายไปขวา ดังนี้ครับ/ค่ะ

1. ลากเพื่อจัดเรียงเลย์เอาต์
2. [โฟลว์เหตุการณ์](/interface-builder/event-flow)
3. การตั้งค่าพารามิเตอร์บล็อก

![20251023224032](https://static-docs.nocobase.com/20251023224032.png)

สำหรับบล็อกแบบง่าย การตั้งค่าทั้งหมดจะรวมอยู่ในส่วน 'การตั้งค่าพารามิเตอร์บล็อก' เช่น บล็อก JS (JS Block) ครับ/ค่ะ

![20251023224903](https://static-docs.nocobase.com/20251023224903.png)

ส่วนบล็อกข้อมูลที่ซับซ้อน จะมีส่วน 'กำหนดค่าฟิลด์' (Configure fields) และ 'กำหนดค่าการทำงาน' (Configure actions) แบบฝังตัวแยกต่างหากให้ด้วยครับ/ค่ะ

![20251023225141](https://static-docs.nocobase.com/20251023225141.png)

นอกจากนี้ ยังสามารถปรับแต่งได้อย่างอิสระ เพื่อเพิ่มความเป็นไปได้ในการซ้อนกันของบล็อกได้มากขึ้น เช่น บล็อกแผนภูมิ (Chart block) ครับ/ค่ะ

![](https://static-docs.nocobase.com/07588190b3f41ae3060e71d8b76b4447.png)

## การจัดเรียงเลย์เอาต์บล็อก

สามารถปรับแต่งเลย์เอาต์ของบล็อกหลายๆ บล็อกได้ด้วยการลากและวางครับ/ค่ะ

![20251029201501](https://static-docs.nocobase.com/20251029201501.gif)
