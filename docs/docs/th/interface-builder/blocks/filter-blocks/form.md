# ฟอร์มตัวกรอง

## บทนำ

ฟอร์มตัวกรองช่วยให้ผู้ใช้สามารถกรองข้อมูลได้โดยการกรอกข้อมูลในช่องฟอร์มครับ/ค่ะ สามารถใช้กรองข้อมูลในบล็อกตาราง (Table block), บล็อกแผนภูมิ (Chart block), บล็อกรายการ (List block) และอื่นๆ ได้ครับ/ค่ะ

## วิธีใช้งาน

มาดูตัวอย่างง่ายๆ เพื่อทำความเข้าใจวิธีการใช้งานฟอร์มตัวกรองกันอย่างรวดเร็วครับ/ค่ะ สมมติว่าเรามีบล็อกตาราง (Table block) ที่แสดงข้อมูลผู้ใช้ และเราต้องการกรองข้อมูลเหล่านั้นโดยใช้ฟอร์มตัวกรอง ดังภาพตัวอย่างด้านล่างนี้ครับ/ค่ะ

![20251031163036_rec_](https://static-docs.nocobase.com/20251031163036_rec_.gif)

ขั้นตอนการตั้งค่ามีดังนี้ครับ/ค่ะ

1.  เปิดโหมดแก้ไข (Edit mode) แล้วเพิ่มบล็อก "ฟอร์มตัวกรอง" และบล็อก "ตาราง" (Table block) ลงในหน้าครับ/ค่ะ
    ![20251031163525_rec_](https://static-docs.nocobase.com/20251031163525_rec_.gif)
2.  เพิ่มฟิลด์ "ชื่อเล่น" (Nickname) ลงในบล็อกตารางและบล็อกฟอร์มตัวกรองครับ/ค่ะ
    ![20251031163932_rec_](https://static-docs.nocobase.com/20251031163932_rec_.gif)
3.  ตอนนี้ก็พร้อมใช้งานแล้วครับ/ค่ะ
    ![20251031163036_rec_](https://static-docs.nocobase.com/20251031163036_rec_.gif)

## การใช้งานขั้นสูง

บล็อกฟอร์มตัวกรองรองรับการตั้งค่าขั้นสูงได้หลากหลายมากขึ้นครับ/ค่ะ นี่คือตัวอย่างการใช้งานทั่วไปบางส่วนครับ/ค่ะ

### การเชื่อมโยงหลายบล็อก

ฟิลด์ในฟอร์มเพียงช่องเดียวสามารถใช้กรองข้อมูลจากหลายบล็อกพร้อมกันได้ครับ/ค่ะ ขั้นตอนการดำเนินการมีดังนี้:

1.  คลิกที่ตัวเลือกการตั้งค่า "Connect fields" ของฟิลด์นั้นๆ ครับ/ค่ะ
    ![20251031170300](https://static-docs.nocobase.com/20251031170300.png)
2.  เพิ่มบล็อกเป้าหมายที่ต้องการเชื่อมโยงครับ/ค่ะ ในตัวอย่างนี้ เราจะเลือกบล็อกรายการ (List block) ที่อยู่ในหน้าครับ/ค่ะ
    ![20251031170718](https://static-docs.nocobase.com/20251031170718.png)
3.  เลือกฟิลด์หนึ่งฟิลด์หรือมากกว่าจากบล็อกรายการเพื่อเชื่อมโยงครับ/ค่ะ ในที่นี้เราจะเลือกฟิลด์ "ชื่อเล่น" (Nickname) ครับ/ค่ะ
    ![20251031171039](https://static-docs.nocobase.com/20251031171039.png)
4.  คลิกปุ่มบันทึก (Save) เพื่อเสร็จสิ้นการตั้งค่าครับ/ค่ะ ผลลัพธ์ที่ได้จะเป็นดังภาพ:
    ![20251031171209_rec_](https://static-docs.nocobase.com/20251031171209_rec_.gif)

### ฟิลด์ที่กำหนดเอง

นอกจากการเลือกฟิลด์จากคอลเลกชันแล้ว คุณยังสามารถสร้างฟิลด์สำหรับฟอร์มได้โดยใช้ "ฟิลด์ที่กำหนดเอง" (Custom fields) ครับ/ค่ะ ตัวอย่างเช่น คุณสามารถสร้างฟิลด์แบบดรอปดาวน์สำหรับเลือกเพียงรายการเดียว และกำหนดตัวเลือกเองได้ ขั้นตอนการดำเนินการมีดังนี้:

1.  คลิกที่ตัวเลือก "ฟิลด์ที่กำหนดเอง" (Custom fields) เพื่อเปิดหน้าต่างการตั้งค่าครับ/ค่ะ
    ![20251031173833](https://static-docs.nocobase.com/20251031173833.png)
2.  กรอกชื่อฟิลด์ (Field title) เลือก "Select" เป็นโมเดลฟิลด์ (Field model) และตั้งค่าตัวเลือกต่างๆ ครับ/ค่ะ
    ![20251031174857_rec_](https://static-docs.nocobase.com/20251031174857_rec_.gif)
3.  ฟิลด์ที่กำหนดเองที่เพิ่มเข้ามาใหม่จะต้องเชื่อมโยงกับฟิลด์ในบล็อกเป้าหมายด้วยตนเองครับ/ค่ะ วิธีการดำเนินการมีดังนี้:
    ![20251031181957_rec_](https://static-docs.nocobase.com/20251031181957_rec_.gif)
4.  การตั้งค่าเสร็จสมบูรณ์ครับ/ค่ะ ผลลัพธ์ที่ได้จะเป็นดังภาพ:
    ![20251031182235_rec_](https://static-docs.nocobase.com/20251031182235_rec_.gif)

โมเดลฟิลด์ที่รองรับในปัจจุบันมีดังนี้:

-   Input: ช่องป้อนข้อความบรรทัดเดียว
-   Number: ช่องป้อนตัวเลข
-   Date: ตัวเลือกวันที่
-   Select: ดรอปดาวน์ (สามารถตั้งค่าให้เลือกได้ทั้งแบบเดี่ยวหรือหลายรายการ)
-   Radio group: ปุ่มตัวเลือก (Radio buttons)
-   Checkbox group: ช่องทำเครื่องหมาย (Checkboxes)

### การยุบ (Collapse)

การเพิ่มปุ่มยุบ (Collapse button) จะช่วยให้คุณสามารถยุบและขยายเนื้อหาของฟอร์มตัวกรองได้ ซึ่งจะช่วยประหยัดพื้นที่บนหน้าจอครับ/ค่ะ

![20251031182743](https://static-docs.nocobase.com/20251031182743.png)

การตั้งค่าที่รองรับมีดังนี้:

![20251031182849](https://static-docs.nocobase.com/20251031182849.png)

-   **จำนวนแถวที่แสดงเมื่อยุบ (Collapsed rows)**: กำหนดจำนวนแถวของฟิลด์ในฟอร์มที่จะแสดงเมื่ออยู่ในสถานะยุบครับ/ค่ะ
-   **ยุบโดยค่าเริ่มต้น (Default collapsed)**: เมื่อเปิดใช้งาน ฟอร์มตัวกรองจะแสดงผลในสถานะยุบโดยค่าเริ่มต้นครับ/ค่ะ
