# การแก้ไขแบบกลุ่ม (Bulk Edit)

## บทนำ

การแก้ไขแบบกลุ่มเหมาะสำหรับสถานการณ์ที่ต้องการอัปเดตข้อมูลจำนวนมากอย่างยืดหยุ่นครับ หลังจากคลิกปุ่มแก้ไขแบบกลุ่มแล้ว คุณสามารถกำหนดค่าฟอร์มการแก้ไขแบบกลุ่มในหน้าต่างป๊อปอัป และตั้งค่ากลยุทธ์การอัปเดตที่แตกต่างกันสำหรับแต่ละฟิลด์ได้ครับ

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM.png)


## การกำหนดค่าการดำเนินการ

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM%20(1).png)


## คู่มือการใช้งาน

### การกำหนดค่าฟอร์มการแก้ไขแบบกลุ่ม

1. เพิ่มปุ่มการแก้ไขแบบกลุ่ม (Bulk Edit) ครับ

2. ตั้งค่าขอบเขตของการแก้ไขแบบกลุ่ม: รายการที่เลือก (Selected) / ทั้งหมด (All) โดยค่าเริ่มต้นจะเป็นรายการที่เลือกครับ

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_14_AM.png)

3. เพิ่มฟอร์มการแก้ไขแบบกลุ่ม

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_14_AM.png)

4. กำหนดค่าฟิลด์ที่ต้องการแก้ไขและเพิ่มปุ่มส่งข้อมูล (Submit) ครับ

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM%20(1).png)

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM.png)

### การส่งฟอร์ม

1. เลือกแถวข้อมูลที่ต้องการแก้ไขครับ

2. เลือกโหมดการแก้ไขสำหรับฟิลด์และกรอกค่าที่ต้องการส่งครับ

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_10_33_AM.png)

:::info{title=โหมดการแก้ไขที่เลือกได้}
* **ไม่ต้องอัปเดต (Do not update)**: ฟิลด์นั้นจะยังคงเดิมไม่มีการเปลี่ยนแปลง
* **เปลี่ยนเป็น (Change to)**: อัปเดตฟิลด์นั้นให้เป็นค่าที่ส่งมา
* **ล้างข้อมูล (Clear)**: ล้างข้อมูลในฟิลด์นั้นให้ว่างเปล่า

:::

3. ส่งฟอร์มครับ
