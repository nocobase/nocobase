# ส่งออกไฟล์แนบ

## บทนำ

ฟังก์ชันการส่งออกไฟล์แนบช่วยให้คุณสามารถส่งออกข้อมูลในฟิลด์ที่เกี่ยวข้องกับไฟล์แนบออกมาในรูปแบบไฟล์บีบอัดได้ครับ/ค่ะ

#### การตั้งค่าการส่งออกไฟล์แนบ

![20251029173251](https://static-docs.nocobase.com/20251029173251.png)

![20251029173425](https://static-docs.nocobase.com/20251029173425.png)

![20251029173345](https://static-docs.nocobase.com/20251029173345.png)

- กำหนดฟิลด์ไฟล์แนบที่ต้องการส่งออก โดยรองรับการเลือกได้หลายรายการครับ/ค่ะ
- คุณสามารถเลือกได้ว่าจะให้สร้างโฟลเดอร์สำหรับแต่ละรายการข้อมูลหรือไม่ครับ/ค่ะ

กฎการตั้งชื่อไฟล์:

- หากคุณเลือกที่จะสร้างโฟลเดอร์สำหรับแต่ละรายการข้อมูล กฎการตั้งชื่อไฟล์จะเป็น: `{ค่าของฟิลด์ชื่อเรื่องของรายการข้อมูล}/{ชื่อฟิลด์ไฟล์แนบ}[-{ลำดับไฟล์}].{นามสกุลไฟล์}`
- หากคุณเลือกที่จะไม่สร้างโฟลเดอร์ กฎการตั้งชื่อไฟล์จะเป็น: `{ค่าของฟิลด์ชื่อเรื่องของรายการข้อมูล}-{ชื่อฟิลด์ไฟล์แนบ}[-{ลำดับไฟล์}].{นามสกุลไฟล์}`

ลำดับไฟล์จะถูกสร้างขึ้นโดยอัตโนมัติเมื่อฟิลด์ไฟล์แนบมีไฟล์แนบหลายไฟล์ครับ/ค่ะ

- [กฎการเชื่อมโยง](/interface-builder/actions/action-settings/linkage-rule): ใช้เพื่อแสดง/ซ่อนปุ่มแบบไดนามิกครับ/ค่ะ
- [แก้ไขปุ่ม](/interface-builder/actions/action-settings/edit-button): ใช้เพื่อแก้ไขชื่อ, ประเภท และไอคอนของปุ่มครับ/ค่ะ
