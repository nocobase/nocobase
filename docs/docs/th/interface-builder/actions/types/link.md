# ลิงก์

## แนะนำ

การดำเนินการลิงก์ (Link action) จะใช้การนำทางแบบ route ครับ/ค่ะ โดยรองรับการส่งตัวแปร URL ทำให้หน้าปลายทางสามารถปรับเนื้อหาได้แบบไดนามิกตามข้อมูลที่ส่งเข้ามา และสามารถตั้งค่าใช้งานได้ในบล็อกข้อมูลครับ/ค่ะ

### เปิดในหน้าต่างใหม่

เมื่อคุณเลือก 'เปิดในหน้าต่างใหม่' ลิงก์จะถูกเปิดขึ้นในหน้าต่างใหม่ครับ/ค่ะ

![20251029175327](https://static-docs.nocobase.com/20251029175327.png)

- [แก้ไขปุ่ม](/interface-builder/actions/action-settings/edit-button): แก้ไขชื่อ, ประเภท และไอคอนของปุ่มครับ/ค่ะ
- [กฎการเชื่อมโยง](/interface-builder/linkage-rule): ควบคุมสถานะของปุ่มแบบไดนามิกครับ/ค่ะ
