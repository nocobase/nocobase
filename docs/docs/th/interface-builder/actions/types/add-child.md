# เพิ่มรายการย่อย

## บทนำ

การดำเนินการนี้ออกแบบมาสำหรับ **บล็อกตารางแบบต้นไม้ (tree table blocks)** โดยเฉพาะ เพื่อเพิ่มรายการย่อยได้อย่างรวดเร็วครับ/ค่ะ เมื่อเพิ่มรายการย่อย แถวที่คุณเลือกจะถูกพิจารณาเป็นโหนดหลัก (parent node) ทำให้คุณสามารถสร้างรายการย่อยภายใต้โหนดหลักใดก็ได้ตามต้องการครับ/ค่ะ ฟังก์ชันนี้ช่วยให้การจัดการข้อมูลแบบลำดับชั้น (hierarchical data) ง่ายขึ้นอย่างมาก ทำให้การขยายและจัดระเบียบโครงสร้างข้อมูลมีประสิทธิภาพและใช้งานง่ายยิ่งขึ้นครับ/ค่ะ
