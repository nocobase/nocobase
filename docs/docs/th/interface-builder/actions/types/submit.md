# ส่งข้อมูล

## บทนำ

การดำเนินการส่งข้อมูลใช้สำหรับบันทึกข้อมูลฟอร์ม (ซึ่งเป็นคุณสมบัติเฉพาะของบล็อกฟอร์ม) และยังสามารถทำงานร่วมกับเวิร์กโฟลว์เพื่อสร้างกระบวนการจัดการข้อมูลแบบอัตโนมัติได้ครับ/ค่ะ

### เชื่อมโยงกับเวิร์กโฟลว์

เวิร์กโฟลว์ที่เชื่อมโยงจะถูกเรียกใช้งานก็ต่อเมื่อข้อมูลถูกส่งสำเร็จเท่านั้นครับ/ค่ะ

![20251029191045](https://static-docs.nocobase.com/20251029191045.png)

ดูรายละเอียดเพิ่มเติมได้ที่ [เชื่อมโยงกับเวิร์กโฟลว์](/interface-builder/actions/action-settings/bind-workflow) ครับ/ค่ะ

- [แก้ไขปุ่ม](/interface-builder/actions/action-settings/edit-button)
- [การยืนยัน](/interface-builder/actions/action-settings/double-check)
- [กฎการเชื่อมโยง](/interface-builder/linkage-rule): ควบคุมสถานะของปุ่มแบบไดนามิกครับ/ค่ะ
