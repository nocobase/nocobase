# แก้ไข

## บทนำ

การดำเนินการแก้ไขเป็นปุ่มที่อยู่บนแถว (row button) โดยมีข้อมูลเรคคอร์ดปัจจุบันเป็นบริบท (context) จะแสดงผลในรูปแบบป๊อปอัป (popup) ซึ่งผู้ใช้สามารถกำหนดค่าและตั้งค่าบล็อกได้เองภายในป๊อปอัปนั้นครับ/ค่ะ

## การตั้งค่าการดำเนินการ

![20251029165620](https://static-docs.nocobase.com/20251029165620.png)

- [กฎการเชื่อมโยง](/interface-builder/actions/action-settings/linkage-rule): แสดง/ซ่อนปุ่มแบบไดนามิกครับ/ค่ะ
- [ปุ่มแก้ไข](/interface-builder/actions/action-settings/edit-button): แก้ไขชื่อ (title), ประเภท (type) และไอคอน (icon) ของปุ่มครับ/ค่ะ
- [ป๊อปอัปแก้ไข](/interface-builder/actions/action-settings/edit-popup): กำหนดโหมดการเปิด (open mode) และขนาด (size) ครับ/ค่ะ
