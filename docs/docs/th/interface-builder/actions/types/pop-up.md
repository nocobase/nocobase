# เปิดโมดัล

## บทนำ

การดำเนินการ **เปิดโมดัล** คล้ายกับการดำเนินการดู (View) และแก้ไข (Edit) ครับ/ค่ะ ผู้ใช้สามารถตั้งค่าปุ่ม **เปิดโมดัล** ได้ที่แถวหรือส่วนหัวของตาราง เมื่อคลิกปุ่มแล้ว หน้าต่างโมดัลที่เกี่ยวข้องจะปรากฏขึ้นมา และบล็อกภายในโมดัลนั้นจะแสดงข้อมูลตามเรคคอร์ดบริบทปัจจุบันครับ/ค่ะ

![20251029182351](https://static-docs.nocobase.com/20251029182351.png)

คุณสามารถตั้งค่าบล็อกในโมดัลได้ตามต้องการครับ/ค่ะ

![20251029190208](https://static-docs.nocobase.com/20251029190208.png)

## การตั้งค่าการดำเนินการ

![20251029190352](https://static-docs.nocobase.com/20251029190352.png)

- [แก้ไขปุ่ม](/interface-builder/actions/action-settings/edit-button): แก้ไขชื่อ, ชนิด, และไอคอนของปุ่มครับ/ค่ะ
- [กฎการเชื่อมโยง](/interface-builder/linkage-rule): ควบคุมสถานะของปุ่มแบบไดนามิก (แสดง/ปิดใช้งาน) ครับ/ค่ะ
- [แก้ไขโมดัล](/interface-builder/actions/action-settings/edit-popup): ตั้งค่ารูปแบบการเปิด (โมดัลหรือลิ้นชัก) และขนาดครับ/ค่ะ
