# ส่งออก

## บทนำ

ฟีเจอร์การส่งออกช่วยให้คุณสามารถส่งออกเรคคอร์ดที่ถูกกรองแล้วในรูปแบบ **Excel** และรองรับการตั้งค่าฟิลด์ที่จะส่งออกได้ ผู้ใช้สามารถเลือกฟิลด์ที่ต้องการส่งออกได้ตามความต้องการ เพื่อนำไปวิเคราะห์ข้อมูล ประมวลผล หรือจัดเก็บในภายหลัง ฟีเจอร์นี้ช่วยเพิ่มความยืดหยุ่นในการจัดการข้อมูล โดยเฉพาะอย่างยิ่งในสถานการณ์ที่ต้องการย้ายข้อมูลไปยังแพลตฟอร์มอื่น หรือนำไปประมวลผลเพิ่มเติมครับ/ค่ะ

### จุดเด่นของฟีเจอร์:
- **การเลือกฟิลด์**: ผู้ใช้สามารถตั้งค่าและเลือกฟิลด์ที่ต้องการส่งออกได้ เพื่อให้มั่นใจว่าข้อมูลที่ส่งออกมีความถูกต้องและกระชับครับ/ค่ะ
- **รองรับรูปแบบ Excel**: ข้อมูลที่ส่งออกจะถูกบันทึกเป็นไฟล์ Excel มาตรฐาน ทำให้ง่ายต่อการนำไปรวมและวิเคราะห์กับข้อมูลอื่น ๆ ครับ/ค่ะ

ด้วยฟีเจอร์นี้ ผู้ใช้สามารถส่งออกข้อมูลสำคัญจากการทำงานได้อย่างง่ายดาย เพื่อนำไปใช้งานภายนอก ซึ่งช่วยเพิ่มประสิทธิภาพในการทำงานครับ/ค่ะ

### ฟิลด์ที่สามารถส่งออกได้

- ระดับที่ 1: ฟิลด์ทั้งหมดของคอลเลกชันปัจจุบันครับ/ค่ะ
- ระดับที่ 2: หากเป็นฟิลด์ความสัมพันธ์ คุณจะต้องเลือกฟิลด์จากคอลเลกชันที่เกี่ยวข้องครับ/ค่ะ
- ระดับที่ 3: จะประมวลผลเพียงสามระดับเท่านั้น ฟิลด์ความสัมพันธ์ในระดับสุดท้ายจะไม่แสดงครับ/ค่ะ

![20251029171557](https://static-docs.nocobase.com/20251029171557.png)

- [กฎการเชื่อมโยง](/interface-builder/actions/action-settings/linkage-rule): แสดง/ซ่อนปุ่มแบบไดนามิกครับ/ค่ะ
- [แก้ไขปุ่ม](/interface-builder/actions/action-settings/edit-button): แก้ไขชื่อ, ประเภท และไอคอนของปุ่มครับ/ค่ะ
