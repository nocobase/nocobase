# การอัปเดตเป็นชุด

## บทนำ

การดำเนินการอัปเดตเป็นชุดใช้เมื่อคุณต้องการอัปเดตเรคคอร์ดหลายรายการพร้อมกันด้วยการเปลี่ยนแปลงแบบเดียวกันครับ/ค่ะ ก่อนที่จะดำเนินการอัปเดตเป็นชุด ผู้ใช้จำเป็นต้องกำหนดตรรกะการกำหนดค่าฟิลด์สำหรับการอัปเดตไว้ล่วงหน้า ตรรกะนี้จะถูกนำไปใช้กับเรคคอร์ดทั้งหมดที่เลือกไว้เมื่อผู้ใช้คลิกปุ่มอัปเดตครับ/ค่ะ

![20251029195320](https://static-docs.nocobase.com/20251029195320.png)

## การกำหนดค่าการดำเนินการ

![20251029195729](https://static-docs.nocobase.com/20251029195729.png)

### ข้อมูลที่จะอัปเดต

เลือก/ทั้งหมด โดยค่าเริ่มต้นคือ 'เลือก' ครับ/ค่ะ

![20251029200034](https://static-docs.nocobase.com/20251029200034.png)

### การกำหนดค่าฟิลด์

กำหนดฟิลด์สำหรับการอัปเดตเป็นชุด เฉพาะฟิลด์ที่กำหนดไว้เท่านั้นที่จะถูกอัปเดตครับ/ค่ะ

ดังที่แสดงในรูปภาพ คุณสามารถกำหนดค่าการดำเนินการอัปเดตเป็นชุดในตารางคำสั่งซื้อ เพื่ออัปเดตข้อมูลที่เลือกให้เป็น 'รอการอนุมัติ' เป็นชุดได้ครับ/ค่ะ

![20251029200109](https://static-docs.nocobase.com/20251029200109.png)

- [แก้ไขปุ่ม](/interface-builder/actions/action-settings/edit-button): แก้ไขชื่อ, ประเภท และไอคอนของปุ่มครับ/ค่ะ
- [กฎการเชื่อมโยง](/interface-builder/actions/action-settings/linkage-rule): แสดง/ซ่อนปุ่มแบบไดนามิกครับ/ค่ะ
- [การยืนยันซ้ำ](/interface-builder/actions/action-settings/double-check)
