# ดู

## บทนำ

การดำเนินการ "ดู" เป็นปุ่มแถว (row button) ที่ใช้เรคคอร์ดปัจจุบันเป็นบริบทครับ/ค่ะ โดยจะแสดงผลในรูปแบบป๊อปอัป ซึ่งผู้ใช้สามารถปรับแต่งและกำหนดค่าบล็อกต่างๆ ภายในป๊อปอัปนั้นได้เลยครับ/ค่ะ
