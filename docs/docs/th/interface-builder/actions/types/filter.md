# การกรองข้อมูล

## บทนำ

การดำเนินการกรองข้อมูล (Filter) มักจะปรากฏอยู่ในบล็อกข้อมูล เพื่อช่วยให้คุณสามารถกรองข้อมูลได้ตามเงื่อนไขต่างๆ ที่กำหนดไว้ครับ/ค่ะ สิ่งสำคัญที่ควรทราบคือ หากบล็อกนั้นมีการกำหนดขอบเขตข้อมูล (data scope) ไว้แล้ว การกรองข้อมูลที่คุณตั้งค่าจะถูกนำไปรวมกับเงื่อนไขของขอบเขตข้อมูลนั้น เพื่อใช้เป็นเงื่อนไขการกรองขั้นสุดท้ายครับ/ค่ะ

### ฟิลด์ที่สามารถกรองได้

โดยค่าเริ่มต้น ฟิลด์ทั้งหมดของคอลเลกชันนี้จะถูกเปิดใช้งานครับ/ค่ะ และเฉพาะฟิลด์ที่เปิดใช้งานเท่านั้นที่จะสามารถเลือกได้ในรายการฟิลด์เงื่อนไขครับ/ค่ะ

ฟิลด์จากคอลเลกชันที่เกี่ยวข้องก็สามารถนำมาใช้เป็นฟิลด์สำหรับกรองข้อมูลได้เช่นกันครับ/ค่ะ
