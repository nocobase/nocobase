# การกำหนดค่าฟิลด์

## บทนำ

การกำหนดค่าฟิลด์คือการตั้งค่าล่วงหน้าสำหรับฟิลด์ต่างๆ ก่อนที่จะดำเนินการ ซึ่งข้อมูลเหล่านี้จะถูกกรอกโดยอัตโนมัติเมื่อมีการอัปเดตหรือสร้างเรคคอร์ดใหม่ โดยไม่จำเป็นต้องกรอกข้อมูลด้วยตนเองครับ/ค่ะ ตัวอย่างเช่น การอัปเดตสถานะคำสั่งซื้อจำนวนมาก หรือการตั้งเวลาจัดส่งเป็นเวลาปัจจุบันเมื่อคำสั่งซื้อถูกจัดส่งถึงมือลูกค้าครับ/ค่ะ
