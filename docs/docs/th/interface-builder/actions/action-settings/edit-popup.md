## Popup UID

"Popup UID" คือ UID ของคอมโพเนนต์ที่ใช้เปิดหน้าต่างป๊อปอัปนั้นๆ ซึ่งจะตรงกับส่วน `viewUid` ใน URL ปัจจุบันที่อยู่ในรูปแบบ `view/:viewUid` ครับ/ค่ะ คุณสามารถคัดลอก Popup UID ได้อย่างรวดเร็วโดยไปที่เมนูการตั้งค่าของฟิลด์หรือปุ่มที่เรียกหน้าต่างป๊อปอัป แล้วคลิก "คัดลอก Popup UID" ครับ/ค่ะ

![popup-copy-uid-20251102](https://static-docs.nocobase.com/popup-copy-uid-20251102.png)

การตั้งค่า Popup UID ช่วยให้คุณสามารถนำหน้าต่างป๊อปอัปกลับมาใช้ซ้ำได้ครับ/ค่ะ

### หน้าต่างป๊อปอัปภายใน (ค่าเริ่มต้น)
- "Popup UID" จะเท่ากับ UID ของปุ่มดำเนินการปัจจุบัน (โดยค่าเริ่มต้นจะใช้ UID ของปุ่มนี้ครับ/ค่ะ)

### หน้าต่างป๊อปอัปภายนอก (นำหน้าต่างป๊อปอัปที่มีอยู่แล้วกลับมาใช้ซ้ำ)
- ให้กรอก UID ของปุ่มที่เรียกใช้งานจากตำแหน่งอื่น (ซึ่งก็คือ Popup UID) ลงในช่อง "Popup UID" เพื่อนำหน้าต่างป๊อปอัปนั้นกลับมาใช้ซ้ำครับ/ค่ะ
- **การใช้งานทั่วไป:** ใช้สำหรับแชร์ UI และ Logic ของหน้าต่างป๊อปอัปเดียวกันในหลายๆ หน้า/บล็อก เพื่อหลีกเลี่ยงการตั้งค่าซ้ำซ้อนครับ/ค่ะ
- เมื่อใช้หน้าต่างป๊อปอัปภายนอก การตั้งค่าบางอย่างจะไม่สามารถแก้ไขได้ (ดูรายละเอียดด้านล่าง)

## การตั้งค่าอื่นๆ ที่เกี่ยวข้อง

- ``Data source / Collection``: แสดงผลแบบอ่านอย่างเดียว ใช้เพื่อระบุแหล่งข้อมูลและคอลเลกชันที่หน้าต่างป๊อปอัปนี้เชื่อมโยงอยู่ โดยค่าเริ่มต้นจะใช้คอลเลกชันของบล็อกปัจจุบันครับ/ค่ะ ในโหมดหน้าต่างป๊อปอัปภายนอก จะใช้การตั้งค่าของหน้าต่างป๊อปอัปเป้าหมาย และไม่สามารถแก้ไขการตั้งค่านี้ได้ครับ/ค่ะ
- ``Association name``: เป็นตัวเลือกเสริม ใช้สำหรับเปิดหน้าต่างป๊อปอัปจากฟิลด์ที่เชื่อมโยง (Association field) โดยจะแสดงเมื่อมีค่าเริ่มต้นเท่านั้นครับ/ค่ะ ในโหมดหน้าต่างป๊อปอัปภายนอก จะใช้การตั้งค่าของหน้าต่างป๊อปอัปเป้าหมาย และไม่สามารถแก้ไขการตั้งค่านี้ได้ครับ/ค่ะ
- ``Source ID``: จะปรากฏขึ้นเมื่อมีการตั้งค่า ``Association name`` เท่านั้น โดยค่าเริ่มต้นจะใช้ ``sourceId`` ของบริบทปัจจุบัน และสามารถเปลี่ยนเป็นตัวแปรหรือค่าคงที่ได้ตามต้องการครับ/ค่ะ
- ``filterByTk``: สามารถเว้นว่างไว้ได้ เป็นตัวแปรเสริม หรือเป็นค่าคงที่ก็ได้ครับ/ค่ะ ใช้สำหรับจำกัดข้อมูลบันทึกที่หน้าต่างป๊อปอัปจะโหลดครับ/ค่ะ
