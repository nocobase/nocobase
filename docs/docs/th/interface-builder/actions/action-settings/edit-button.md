# แก้ไขปุ่ม

## บทนำ

ระบบรองรับการปรับแต่งชื่อ, ไอคอน, สีพื้นหลัง และประเภทของปุ่ม เพื่อให้ผู้ใช้สามารถปรับแต่ง UI ได้อย่างยืดหยุ่น และช่วยให้ผู้ใช้สามารถปรับรูปแบบการแสดงผลและฟังก์ชันการทำงานของปุ่มได้ตามความต้องการเฉพาะได้เลยครับ/ค่ะ

- **ชื่อปุ่ม**: ผู้ใช้สามารถกำหนดข้อความบนปุ่มได้เองตามความต้องการทางธุรกิจ เพื่อให้ป้ายกำกับปุ่มชัดเจนและเข้าใจง่ายครับ/ค่ะ
- **ไอคอนปุ่ม**: รองรับการตั้งค่าไอคอนสำหรับปุ่ม เพื่อเพิ่มการจดจำทางสายตา และช่วยให้ผู้ใช้ระบุฟังก์ชันของปุ่มได้อย่างรวดเร็วครับ/ค่ะ
- **การดำเนินการอันตราย (Danger Action)**: เมื่อปุ่มแสดงถึง**การดำเนินการที่อาจเป็นอันตราย** (เช่น การลบ, การรีเซ็ต เป็นต้น) สามารถตั้งค่าเป็นประเภท **Danger Action** ได้ครับ/ค่ะ
- **ประเภทปุ่ม (Button Type)**: ปุ่มรองรับการตั้งค่าประเภทต่างๆ โดยประเภทปุ่มที่พบบ่อยได้แก่:
  - **ค่าเริ่มต้น (Default)**: ปุ่มมาตรฐาน เหมาะสำหรับการดำเนินการทั่วไปครับ/ค่ะ
  - **หลัก (Primary)**: ใช้สำหรับการดำเนินการที่สำคัญหรือหลัก มักจะมีเอฟเฟกต์ภาพที่โดดเด่นครับ/ค่ะ
  - **เส้นประ (Dashed)**: ปุ่มที่มีเส้นขอบเป็นเส้นประ เหมาะสำหรับการดำเนินการรอง หรือปุ่มที่ใช้เป็นคำแนะนำครับ/ค่ะ
  - **ข้อความ (Text)**: ปุ่มที่มีเพียงข้อความและไม่มีสีพื้นหลัง โดยทั่วไปจะใช้สำหรับการดำเนินการที่ไม่ซับซ้อน หรือใช้เป็นลิงก์ครับ/ค่ะ
