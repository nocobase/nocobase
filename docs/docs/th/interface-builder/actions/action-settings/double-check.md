# การยืนยัน

## บทนำ

การตั้งค่าการยืนยันนี้ใช้เพื่อยืนยันว่าคุณต้องการดำเนินการนั้นจริงหรือไม่ โดยเมื่อคุณคลิกที่การดำเนินการใด ๆ จะมีกล่องโต้ตอบยืนยันปรากฏขึ้นมาครับ/ค่ะ สำหรับปุ่ม 'ลบ' จะมีการเปิดใช้งานการยืนยันนี้โดยค่าเริ่มต้น เพื่อช่วยป้องกันการสูญหายของข้อมูลหรือผลลัพธ์ที่ไม่สามารถย้อนกลับได้จากการดำเนินการที่ผิดพลาดครับ/ค่ะ

![20251029125021](https://static-docs.nocobase.com/20251029125021.png)
