# การผูกเวิร์กโฟลว์

## บทนำ

บนปุ่มการทำงานบางปุ่ม คุณสามารถตั้งค่าการผูกเวิร์กโฟลว์ได้ครับ/ค่ะ เพื่อเชื่อมโยงการทำงานนั้นเข้ากับเวิร์กโฟลว์ ซึ่งจะช่วยให้การประมวลผลข้อมูลเป็นไปโดยอัตโนมัติครับ/ค่ะ

## การผูกเวิร์กโฟลว์หลายรายการพร้อมกัน

ปุ่มการทำงานหนึ่งปุ่มสามารถผูกกับเวิร์กโฟลว์ได้หลายรายการครับ/ค่ะ เมื่อมีการผูกเวิร์กโฟลว์หลายรายการ ลำดับการทำงานของเวิร์กโฟลว์จะเป็นไปตามกฎต่อไปนี้ครับ/ค่ะ:

1.  สำหรับเวิร์กโฟลว์ที่มีประเภทการเรียกใช้ (trigger type) เดียวกัน เวิร์กโฟลว์แบบซิงโครนัส (synchronous) จะทำงานก่อน ตามด้วยเวิร์กโฟลว์แบบอะซิงโครนัส (asynchronous) ครับ/ค่ะ
2.  เวิร์กโฟลว์ที่มีประเภทการเรียกใช้เดียวกันจะทำงานตามลำดับที่ตั้งค่าไว้ครับ/ค่ะ
3.  สำหรับเวิร์กโฟลว์ที่มีประเภทการเรียกใช้ต่างกัน:
    1.  เหตุการณ์ก่อนการทำงาน (Before action events) จะทำงานก่อนเหตุการณ์หลังการทำงาน (After action events) และเหตุการณ์การอนุมัติ (Approval events) เสมอครับ/ค่ะ
    2.  เหตุการณ์หลังการทำงานและเหตุการณ์การอนุมัติไม่มีลำดับที่แน่นอน การทำงานของระบบไม่ควรขึ้นอยู่กับลำดับการตั้งค่าครับ/ค่ะ
