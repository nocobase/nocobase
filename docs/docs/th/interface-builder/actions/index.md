# การดำเนินการ

ใน UI การดำเนินการคือปุ่มที่ใช้เพื่อทริกเกอร์ชุดคำสั่งครับ/ค่ะ สามารถวางได้โดยตรงบนหน้าเพจ, ในกล่องโต้ตอบ (dialog box) หรือในลิ้นชัก (drawer) และยังสามารถใช้ร่วมกับบล็อก (block) ได้อีกด้วย ปัจจุบัน การดำเนินการที่สามารถกำหนดค่าแบบไดนามิกได้ ส่วนใหญ่จะใช้ร่วมกับบล็อกครับ/ค่ะ
