# ตารางย่อย

## บทนำ

ตารางย่อยเหมาะสำหรับการจัดการกับฟิลด์ความสัมพันธ์แบบ "หนึ่งต่อหลาย" (to-many relationship) ครับ โดยรองรับการสร้างข้อมูลใหม่ใน **คอลเลกชัน** เป้าหมายแบบเป็นชุดแล้วเชื่อมโยง หรือจะเลือกเชื่อมโยงกับข้อมูลที่มีอยู่แล้วก็ได้ครับ

## วิธีใช้งาน

![20251027223350](https://static-docs.nocobase.com/20251027223350.png)

ฟิลด์ประเภทต่างๆ ในตารางย่อยจะแสดงคอมโพเนนต์ฟิลด์ที่แตกต่างกันไปครับ สำหรับฟิลด์ขนาดใหญ่ (เช่น Rich Text, JSON, Long Text) จะสามารถแก้ไขได้ผ่านป๊อปอัปที่แสดงขึ้นมาครับ

![20251027223426](https://static-docs.nocobase.com/20251027223426.png)

ฟิลด์ความสัมพันธ์ในตารางย่อย:

Orders (หนึ่งต่อหลาย) > Order Products (หนึ่งต่อหนึ่ง) > Opportunity

![20251027223530](https://static-docs.nocobase.com/20251027223530.png)

คอมโพเนนต์ฟิลด์ความสัมพันธ์ค่าเริ่มต้นคือ Dropdown (รองรับ Dropdown / Data Picker) ครับ

![20251027223754](https://static-docs.nocobase.com/20251027223754.png)

## ตัวเลือกการตั้งค่าฟิลด์

### อนุญาตให้เลือกข้อมูลที่มีอยู่แล้ว (ปิดใช้งานโดยค่าเริ่มต้น)

รองรับการเลือกและเชื่อมโยงข้อมูลที่มีอยู่แล้วครับ

![20251027224008](https://static-docs.nocobase.com/20251027224008.png)

![20251027224023](https://static-docs.nocobase.com/20251027224023.gif)

### คอมโพเนนต์ฟิลด์

[คอมโพเนนต์ฟิลด์](/interface-builder/fields/association-field): ใช้สำหรับสลับไปใช้คอมโพเนนต์ฟิลด์ความสัมพันธ์อื่นๆ เช่น Dropdown, Data Picker เป็นต้นครับ

### อนุญาตให้ยกเลิกการเชื่อมโยงข้อมูลที่มีอยู่แล้ว

> ตัวเลือกนี้จะกำหนดว่าฟิลด์ความสัมพันธ์ในฟอร์มแก้ไขจะอนุญาตให้ยกเลิกการเชื่อมโยงข้อมูลที่มีอยู่แล้วได้หรือไม่ครับ

![20251028153425](https://static-docs.nocobase.com/20251028153425.gif)
