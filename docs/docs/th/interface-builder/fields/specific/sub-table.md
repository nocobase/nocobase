# ตารางย่อย (การแก้ไขในแถว)

## บทนำ

ตารางย่อยเหมาะสำหรับจัดการฟิลด์ความสัมพันธ์แบบ To-many รองรับการสร้างข้อมูลใหม่ในตารางเป้าหมายแบบเป็นชุดแล้วเชื่อมโยง หรือเลือกเชื่อมโยงจากข้อมูลที่มีอยู่แล้วครับ

## วิธีใช้งาน

![20251027223350](https://static-docs.nocobase.com/20251027223350.png)

ฟิลด์ประเภทต่างๆ ในตารางย่อยจะแสดงคอมโพเนนต์ฟิลด์ที่แตกต่างกัน สำหรับฟิลด์ขนาดใหญ่ (Rich Text, JSON, ข้อความหลายบรรทัด ฯลฯ) จะแก้ไขผ่านป๊อปอัปแบบลอยครับ

![20251027223426](https://static-docs.nocobase.com/20251027223426.png)

ฟิลด์ความสัมพันธ์ในตารางย่อยครับ

คำสั่งซื้อ (หนึ่งต่อหลาย) > Order Products (หนึ่งต่อหนึ่ง) > Opportunity

![20251027223530](https://static-docs.nocobase.com/20251027223530.png)

คอมโพเนนต์ฟิลด์ความสัมพันธ์เริ่มต้นคือตัวเลือกแบบดรอปดาวน์ (รองรับตัวเลือกแบบดรอปดาวน์/ตัวเลือกข้อมูล) ครับ

![20251027223754](https://static-docs.nocobase.com/20251027223754.png)

## ตัวเลือกการตั้งค่าฟิลด์

### อนุญาตให้เลือกข้อมูลที่มีอยู่แล้ว (เปิดใช้งานโดยเริ่มต้น)

รองรับการเลือกเชื่อมโยงจากข้อมูลที่มีอยู่แล้วครับ
![20251027224008](https://static-docs.nocobase.com/20251027224008.png)

![20251027224023](https://static-docs.nocobase.com/20251027224023.gif)

### คอมโพเนนต์ฟิลด์

[คอมโพเนนต์ฟิลด์](/interface-builder/fields/association-field): สลับเป็นคอมโพเนนต์ฟิลด์ความสัมพันธ์อื่นๆ เช่น ตัวเลือกแบบดรอปดาวน์, ตัวเลือกข้อมูล เป็นต้นครับ

### อนุญาตให้ยกเลิกการเชื่อมโยงข้อมูลที่มีอยู่แล้ว

> ข้อมูลฟิลด์ความสัมพันธ์ในฟอร์มแก้ไขอนุญาตให้ยกเลิกการเชื่อมโยงกับข้อมูลที่มีอยู่แล้วหรือไม่ครับ

![20251028153425](https://static-docs.nocobase.com/20251028153425.gif)
