# รายละเอียดส่วนย่อย

## บทนำ ครับ/ค่ะ

รายละเอียดส่วนย่อย (Sub-details) เป็นคอมโพเนนต์ที่ใช้คู่กับฟอร์มย่อย (Sub-form) ในโหมดอ่านอย่างเดียวครับ/ค่ะ เมื่อเทียบกับคอมโพเนนต์ป้ายกำกับ (Label) และคอมโพเนนต์ชื่อเรื่อง (Title) แล้ว รายละเอียดส่วนย่อยไม่เพียงแต่สามารถแสดงข้อมูลจากคอลเลกชันปัจจุบันได้มากขึ้นเท่านั้น แต่ยังสามารถตั้งค่าให้แสดงข้อมูลจากคอลเลกชันที่เกี่ยวข้องได้อีกด้วยครับ/ค่ะ โดยข้อมูลความสัมพันธ์แบบหลายระดับชั้นจะถูกแสดงผลอย่างชัดเจนในรูปแบบของรายละเอียดที่ซ้อนกันครับ/ค่ะ
