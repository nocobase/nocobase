# ซับฟอร์ม

## บทนำ

ซับฟอร์มเหมาะสำหรับสถานการณ์ที่คุณต้องการสร้างข้อมูลที่เกี่ยวข้องขึ้นมาก่อน แล้วจึงค่อยเชื่อมโยงข้อมูลเหล่านั้นครับ/ค่ะ โดยข้อมูลที่มีความสัมพันธ์แบบหลายระดับจะแสดงผลอย่างชัดเจนในรูปแบบของฟอร์มที่ซ้อนกัน เมื่อเทียบกับตัวเลือกข้อมูล (Data Selector) และตัวเลือกแบบดรอปดาวน์ (Dropdown) ซับฟอร์มช่วยให้คุณสามารถจัดการฟิลด์ของคอลเลกชันที่เกี่ยวข้องได้โดยตรงภายในบล็อกหน้าปัจจุบัน และข้อมูลที่เกี่ยวข้องจะถูกส่งไปพร้อมกับฟอร์มหลักครับ/ค่ะ

## ตัวเลือกการตั้งค่าฟิลด์

### คอมโพเนนต์ฟิลด์

![20251030150601](https://static-docs.nocobase.com/20251030150601.png)

[คอมโพเนนต์ฟิลด์](/interface-builder/fields/association-field): ใช้เพื่อสลับไปยังคอมโพเนนต์ฟิลด์ความสัมพันธ์อื่น ๆ เช่น ดรอปดาวน์ (Dropdown) หรือตัวเลือกข้อมูล (Data Selector) เป็นต้นครับ/ค่ะ

### กฎการเชื่อมโยง

[กฎการเชื่อมโยง](/interface-builder/linkage-rule): ใช้เพื่อตั้งค่ากฎการเชื่อมโยงสำหรับฟิลด์ในซับฟอร์มครับ/ค่ะ
