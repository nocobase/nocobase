# หัวข้อ

## บทนำ

ในโหมดอ่านอย่างเดียว ฟิลด์ความสัมพันธ์จะแสดงผลโดยใช้ **คอมโพเนนต์ฟิลด์หัวข้อ** โดยค่าเริ่มต้น ซึ่งจะระบุเรคคอร์ดที่เชื่อมโยงอยู่ปัจจุบันด้วยการแสดงค่าของฟิลด์หัวข้อครับ/ค่ะ
ระบบจะจับคู่และเรนเดอร์คอมโพเนนต์ฟิลด์หัวข้อที่เหมาะสมโดยอัตโนมัติ ตามประเภทของฟิลด์หัวข้อนั้นๆ ครับ/ค่ะ

## ตัวเลือกการตั้งค่าฟิลด์

### เปิดใช้งานการคลิกเพื่อเปิด (เปิดใช้งานโดยค่าเริ่มต้น)

เมื่อคลิก จะมีหน้าต่างป๊อปอัปปรากฏขึ้น ซึ่งคุณสามารถปรับแต่งบล็อกเพื่อจัดการข้อมูลได้ครับ/ค่ะ

![20251028160742](https://static-docs.nocobase.com/20251028160742.png)

สำหรับรายละเอียดเพิ่มเติม โปรดดูที่ [คอมโพเนนต์ฟิลด์](/interface-builder/fields/association-field) ครับ/ค่ะ
