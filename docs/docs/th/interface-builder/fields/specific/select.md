# Dropdown

## บทนำ

Dropdown รองรับการเชื่อมโยงข้อมูลโดยเลือกจากข้อมูลที่มีอยู่แล้วในคอลเลกชันเป้าหมาย หรือเพิ่มข้อมูลใหม่เข้าไปในคอลเลกชันนั้นเพื่อเชื่อมโยงครับ/ค่ะ ตัวเลือกใน Dropdown ยังรองรับการค้นหาแบบ Fuzzy Search ด้วยครับ/ค่ะ

![20251029205901](https://static-docs.nocobase.com/20251029205901.png)

## การตั้งค่าฟิลด์

### กำหนดขอบเขตข้อมูล

ควบคุมขอบเขตข้อมูลของรายการ Dropdown ครับ/ค่ะ

![20251029210025](https://static-docs.nocobase.com/20251029210025.png)

ดูรายละเอียดเพิ่มเติมได้ที่ [กำหนดขอบเขตข้อมูล](/interface-builder/fields/field-settings/data-scope)

### กำหนดกฎการจัดเรียง

ควบคุมการจัดเรียงข้อมูลใน Dropdown ครับ/ค่ะ

ตัวอย่าง: จัดเรียงตามวันที่ให้บริการจากมากไปน้อย

![20251029210105](https://static-docs.nocobase.com/20251029210105.png)

### อนุญาตให้เพิ่ม/เชื่อมโยงหลายรายการ

จำกัดความสัมพันธ์แบบ One-to-Many ให้เชื่อมโยงได้เพียงรายการเดียวครับ/ค่ะ

![20251029210145](https://static-docs.nocobase.com/20251029210145.png)

### ฟิลด์ชื่อเรื่อง

ฟิลด์ชื่อเรื่องคือฟิลด์ป้ายกำกับที่แสดงในตัวเลือกครับ/ค่ะ

![20251029210507](https://static-docs.nocobase.com/20251029210507.gif)

> รองรับการค้นหาอย่างรวดเร็วโดยใช้ฟิลด์ชื่อเรื่อง

ดูรายละเอียดเพิ่มเติมได้ที่ [ฟิลด์ชื่อเรื่อง](/interface-builder/fields/field-settings/title-field)

### สร้างด่วน: เพิ่มข้อมูลก่อน แล้วจึงเลือก

![20251125220046](https://static-docs.nocobase.com/20251125220046.png)

#### เพิ่มผ่าน Dropdown

หลังจากสร้างข้อมูลใหม่ในคอลเลกชันเป้าหมาย ระบบจะเลือกข้อมูลนั้นโดยอัตโนมัติและเชื่อมโยงเมื่อส่งฟอร์มครับ/ค่ะ

ตัวอย่างเช่น คอลเลกชัน Orders มีฟิลด์ความสัมพันธ์แบบ Many-to-One ที่ชื่อว่า 'Account' ครับ/ค่ะ

![20251125220447](https://static-docs.nocobase.com/20251125220447.gif)

#### เพิ่มผ่าน Modal

การเพิ่มข้อมูลผ่าน Modal เหมาะสำหรับสถานการณ์การป้อนข้อมูลที่ซับซ้อนกว่าครับ/ค่ะ โดยสามารถกำหนดค่าฟอร์มสำหรับสร้างรายการใหม่ได้

![20251125220607](https://static-docs.nocobase.com/20251125220607.gif)
