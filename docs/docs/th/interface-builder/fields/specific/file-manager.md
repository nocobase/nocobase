# ตัวจัดการไฟล์

## แนะนำ

ตัวจัดการไฟล์เป็นคอมโพเนนต์ฟิลด์ความสัมพันธ์ที่ใช้สำหรับจัดการความสัมพันธ์ที่คอลเลกชันเป้าหมายคือคอลเลกชันไฟล์ครับ

![20251027194124](https://static-docs.nocobase.com/20251027194124.png)

## การตั้งค่าฟิลด์

- [คอมโพเนนต์ฟิลด์](/interface-builder/fields/association-field): สลับไปยังคอมโพเนนต์ฟิลด์ความสัมพันธ์อื่น ๆ เช่น ดรอปดาวน์, ตัวเลือกข้อมูล เป็นต้น ครับ
- อนุญาตให้เลือกไฟล์ที่มีอยู่แล้ว: เปิดใช้งานโดยค่าเริ่มต้น สามารถเลือกและเชื่อมโยงจากไฟล์ที่อัปโหลดไว้แล้วได้ครับ
