# วันที่และเวลา

## บทนำ

ฟิลด์วันที่รองรับการกำหนดรูปแบบวันที่และเวลาเองได้ครับ/ค่ะ

## ตั้งค่ารูปแบบวันที่ได้ที่ไหนบ้าง

### ฟิลด์ในคอลเลกชัน

![20240417114311](https://static-docs.nocobase.com/20240417114311.png)

> รูปแบบวันที่ที่ตั้งค่าไว้ในฟิลด์ของคอลเลกชันจะถูกใช้เป็นรูปแบบเริ่มต้นของฟิลด์นั้นๆ ครับ/ค่ะ และสามารถถูกแทนที่ได้ด้วยรูปแบบวันที่ที่ตั้งค่าไว้ในส่วนการตั้งค่าฟิลด์ภายใน Block Configuration ครับ/ค่ะ

### การตั้งค่าฟิลด์

![20251027191824](https://static-docs.nocobase.com/20251027191824.png)

นอกเหนือจากรูปแบบวันที่และเวลาที่มีให้เลือกหลากหลายแบบแล้ว ยังรองรับการกำหนดรูปแบบเองเพื่อตอบสนองความต้องการเฉพาะบุคคลได้ด้วยครับ/ค่ะ
