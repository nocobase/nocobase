# ตัวเลือกข้อมูล

## บทนำ

ตัวเลือกข้อมูล (Data Selector) จะปรากฏเป็นตารางเลือกข้อมูลแบบป๊อปอัปครับ/ค่ะ ซึ่งช่วยให้คุณสามารถเลือกข้อมูลที่มีอยู่จากคอลเลกชันเป้าหมายเพื่อเชื่อมโยง หรือจะเพิ่มข้อมูลใหม่เข้าไปในคอลเลกชันเป้าหมายแล้วจึงเชื่อมโยงก็ได้ครับ/ค่ะ

![20251027203343](https://static-docs.nocobase.com/20251027203343.png)

ข้อมูลที่เลือกไว้แล้วจะไม่แสดงในตารางเลือกข้อมูลครับ/ค่ะ

![20251027203717](https://static-docs.nocobase.com/20251027203717.gif)

ในตารางเลือกข้อมูล คุณยังสามารถจัดการคอลเลกชันเป้าหมายที่เชื่อมโยงได้เพิ่มเติมอีกด้วยครับ/ค่ะ (เช่น การเพิ่ม, ลบ, นำเข้า/ส่งออก เป็นต้น)

![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
