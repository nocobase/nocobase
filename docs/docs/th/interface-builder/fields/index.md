# ฟิลด์

ใน UI, ฟิลด์คือคอมโพเนนต์ฟิลด์ ซึ่งเป็นตัวกลางสำหรับข้อมูลแต่ละหน่วย ข้อมูลประเภทต่างๆ จะแสดงผลด้วยคอมโพเนนต์ฟิลด์ที่แตกต่างกันครับ/ค่ะ ฟิลด์สามารถแนบไปกับบล็อกเท่านั้น ไม่สามารถใช้งานได้อย่างอิสระครับ/ค่ะ

## ฟิลด์ในบล็อก

โดยทั่วไปแล้ว ฟิลด์จะไม่ถูกใช้งานเดี่ยวๆ ครับ/ค่ะ แต่จะอยู่เป็นองค์ประกอบย่อยของบล็อกประเภทข้อมูล บล็อกประเภทข้อมูลมักจะมีตัวเลือก "กำหนดค่าฟิลด์" (Configure fields) และรายการฟิลด์จะมาจากคอลเลกชันปัจจุบันครับ/ค่ะ

![20251029204240](https://static-docs.nocobase.com/20251029204240.png)

## ดีไซเนอร์ฟิลด์ (แถบเครื่องมือ)

เช่นเดียวกับบล็อก คอมโพเนนต์ฟิลด์ก็มีไอคอนสามอันที่มุมขวาบนเช่นกันครับ/ค่ะ ได้แก่:

- ลากเพื่อจัดวางเลย์เอาต์
- [โฟลว์เหตุการณ์](/interface-builder/event-flow)
- การตั้งค่า

![20251025165432](https://static-docs.nocobase.com/20251025165432.png)

## การจัดวางเลย์เอาต์ของฟิลด์

คุณสามารถปรับแต่งการจัดวางเลย์เอาต์ของฟิลด์ภายในบล็อกได้ด้วยการลากและวางครับ/ค่ะ

![20251025165605](https://static-docs.nocobase.com/20251025165605.gif)

## ตัวเลือกการตั้งค่าฟิลด์

นอกจากตัวเลือกการตั้งค่าฟิลด์ทั่วไปแล้ว ฟิลด์แต่ละประเภทก็ยังรองรับตัวเลือกการตั้งค่าเฉพาะของตนเองด้วยครับ/ค่ะ

![20251025170120](https://static-docs.nocobase.com/20251025170120.png)
