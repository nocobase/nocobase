# จำเป็น

## บทนำ

การกำหนดให้เป็นข้อมูลที่จำเป็น (Required) เป็นกฎที่ใช้บ่อยในการตรวจสอบความถูกต้องของฟอร์มครับ/ค่ะ คุณสามารถเปิดใช้งานได้โดยตรงในการตั้งค่าฟิลด์ หรือจะกำหนดให้ฟิลด์เป็นข้อมูลที่จำเป็นแบบไดนามิกผ่านกฎการเชื่อมโยงของฟอร์มก็ได้ครับ/ค่ะ

## ตั้งค่าฟิลด์ให้เป็นข้อมูลที่จำเป็นได้ที่ไหนบ้าง

### การตั้งค่าฟิลด์คอลเลกชัน

เมื่อฟิลด์ในคอลเลกชันถูกตั้งค่าให้เป็นข้อมูลที่จำเป็น ระบบจะเรียกใช้การตรวจสอบความถูกต้องจากฝั่ง Backend ครับ/ค่ะ และฝั่ง Frontend ก็จะแสดงผลเป็นข้อมูลที่จำเป็นโดยค่าเริ่มต้น (ไม่สามารถแก้ไขได้) ครับ/ค่ะ

![20251025175418](https://static-docs.nocobase.com/20251025175418.png)

### การตั้งค่าฟิลด์

คุณสามารถตั้งค่าฟิลด์ให้เป็นข้อมูลที่จำเป็นได้โดยตรงครับ/ค่ะ วิธีนี้เหมาะสำหรับฟิลด์ที่ผู้ใช้จำเป็นต้องกรอกเสมอ เช่น ชื่อผู้ใช้ รหัสผ่าน เป็นต้น ครับ/ค่ะ

![20251028222818](https://static-docs.nocobase.com/20251028222818.png)

### กฎการเชื่อมโยง

คุณสามารถกำหนดให้ฟิลด์เป็นข้อมูลที่จำเป็นตามเงื่อนไขได้ผ่านกฎการเชื่อมโยงฟิลด์ของบล็อกฟอร์มครับ/ค่ะ

ตัวอย่าง: หมายเลขคำสั่งซื้อจำเป็นต้องกรอก เมื่อวันที่สั่งซื้อไม่ว่างเปล่าครับ/ค่ะ
