# โหมดการแสดงผล

## บทนำ

ไม่เหมือนกับบล็อก คอมโพเนนต์ฟิลด์มีโหมดการแสดงผล 3 แบบ (ซึ่งรองรับเฉพาะฟิลด์ที่อยู่ในฟอร์มเท่านั้น) การสลับโหมดการแสดงผลจะส่งผลต่อตัวเลือกการกำหนดค่าฟิลด์ที่แตกต่างกันไปครับ/ค่ะ

- แก้ไขได้
- อ่านอย่างเดียว (ไม่สามารถแก้ไขได้)
- อ่านอย่างเดียว (โหมดอ่าน)

### โหมดการแสดงผลสำหรับฟิลด์ความสัมพันธ์

**โหมดการแสดงผลของฟิลด์ความสัมพันธ์** จะเป็นตัวกำหนดวิธีการแสดงฟิลด์นั้นๆ ในส่วนติดต่อผู้ใช้ (interface) รวมถึงประเภทของคอมโพเนนต์ฟิลด์ที่สามารถเลือกใช้งานได้ครับ/ค่ะ

ใน**สถานะแก้ไขได้** ฟิลด์ความสัมพันธ์จะรองรับคอมโพเนนต์ได้หลายรูปแบบ ผู้ใช้สามารถเลือกคอมโพเนนต์ฟิลด์ความสัมพันธ์ที่แตกต่างกันเพื่อแสดงหรือเลือกข้อมูลที่เกี่ยวข้องได้ตามความต้องการทางธุรกิจครับ/ค่ะ

#### คอมโพเนนต์ฟิลด์ความสัมพันธ์ในสถานะแก้ไขได้

![20251028220447](https://static-docs.nocobase.com/20251028220447.png)

ในสถานะนี้ ผู้ใช้สามารถเลือกวิธีการแสดงผลที่เหมาะสมได้อย่างยืดหยุ่น เพื่อให้การจัดการข้อมูลมีประสิทธิภาพมากยิ่งขึ้นครับ/ค่ะ

#### คอมโพเนนต์ฟิลด์ความสัมพันธ์ในสถานะอ่านอย่างเดียว

เมื่อสลับไปที่**สถานะอ่านอย่างเดียว** ระบบจะใช้**คอมโพเนนต์ฟิลด์ชื่อเรื่อง** (Title Field Component) เป็นค่าเริ่มต้นโดยอัตโนมัติเพื่อแสดงข้อมูลที่เกี่ยวข้องครับ/ค่ะ ซึ่งเหมาะสำหรับสถานการณ์ที่คุณต้องการเพียงแค่ดูข้อมูลโดยไม่ต้องแก้ไข
