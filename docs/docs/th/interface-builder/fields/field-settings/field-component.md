# ส่วนประกอบฟิลด์

## บทนำ

ส่วนประกอบฟิลด์ใช้สำหรับแสดงเนื้อหาของฟิลด์ครับ/ค่ะ

ฟิลด์บางประเภทมีส่วนประกอบสำหรับแสดงผลในตัวหลายแบบ ผู้ใช้สามารถเลือกส่วนประกอบที่เหมาะสมตามความต้องการ เพื่อแสดงค่าของฟิลด์ในรูปแบบที่แตกต่างกันได้ครับ/ค่ะ

ตัวอย่าง: ในโหมดการอ่าน ฟิลด์ URL สามารถสลับไปใช้ส่วนประกอบแบบพรีวิว (Preview component) หรือส่วนประกอบ URL (URL component) ซึ่งเป็นค่าเริ่มต้นได้ครับ/ค่ะ

![20251028214033](https://static-docs.nocobase.com/20251028214033.png)

![20251028214628](https://static-docs.nocobase.com/20251028214628.png)

ตัวอย่าง: ฟิลด์ความสัมพันธ์ (association field) สามารถสลับไปใช้เป็นฟอร์มย่อย (Sub-form), ตารางย่อย (Sub-table) (รองรับเฉพาะความสัมพันธ์แบบ "หนึ่งต่อหลาย" หรือ "หลายต่อหลาย"), ตัวเลือกข้อมูล (Data selector), ตัวเลือกแบบดรอปดาวน์ (Dropdown selector) ซึ่งเป็นค่าเริ่มต้น, หรือตัวจัดการไฟล์ (File manager) (รองรับเฉพาะเมื่อคอลเลกชันเป้าหมายของความสัมพันธ์เป็นคอลเลกชันไฟล์) ครับ/ค่ะ
