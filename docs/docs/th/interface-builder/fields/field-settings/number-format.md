# การจัดรูปแบบตัวเลข

## บทนำ

การจัดรูปแบบตัวเลข (Number Formatting) คือการปรับปรุงรูปลักษณ์และการจัดระเบียบฟิลด์ประเภทตัวเลข (รวมถึงฟิลด์สูตร) ในโหมดการอ่าน เพื่อให้ดูสวยงามและเป็นมาตรฐานครับ/ค่ะ ซึ่งรวมถึงการตั้งค่าตัวคั่นหลักพัน, ความแม่นยำของทศนิยม, การแปลงหน่วย และสัญกรณ์วิทยาศาสตร์ การตั้งค่าการจัดรูปแบบตัวเลขจะช่วยให้คุณสามารถแสดงผลตัวเลขได้ตามความต้องการและธรรมเนียมปฏิบัติของแต่ละภูมิภาคและอุตสาหกรรมที่แตกต่างกันครับ/ค่ะ

## วิธีใช้งาน

- รองรับการแปลงหน่วยอย่างง่าย (การคูณ/การหาร)
- ตัวคั่นหลักพัน
- คำนำหน้าและคำต่อท้าย
- ความแม่นยำ
- สัญกรณ์วิทยาศาสตร์

ตัวอย่าง: จัดรูปแบบยอดรวมคำสั่งซื้อ โดยเพิ่มสัญลักษณ์สกุลเงิน (เช่น สัญลักษณ์เงินหยวน ¥), ตัวคั่นหลักพัน (เช่น เครื่องหมายจุลภาค หรือจุด) และความแม่นยำของทศนิยม เพื่อให้ยอดเงินอ่านง่ายและชัดเจนครับ/ค่ะ
