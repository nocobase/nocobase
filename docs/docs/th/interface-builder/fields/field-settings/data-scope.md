# การตั้งค่าขอบเขตข้อมูล

## บทนำ

การตั้งค่าขอบเขตข้อมูลสำหรับฟิลด์ความสัมพันธ์จะคล้ายกับการตั้งค่าขอบเขตข้อมูลสำหรับบล็อกครับ โดยจะกำหนดเงื่อนไขการกรองเริ่มต้นสำหรับข้อมูลที่เกี่ยวข้อง

## วิธีใช้งาน

![20251028211328](https://static-docs.nocobase.com/20251028211328.png)

### ค่าคงที่

ตัวอย่าง: สามารถเลือกสินค้าที่ยังไม่ถูกลบเท่านั้นเพื่อเชื่อมโยงข้อมูลครับ

> รายการฟิลด์จะแสดงฟิลด์จากคอลเลกชันเป้าหมายของฟิลด์ความสัมพันธ์ครับ

![20251028211434](https://static-docs.nocobase.com/20251028211434.png)

### ค่าตัวแปร

ตัวอย่าง: สามารถเลือกสินค้าที่มีวันที่ให้บริการหลังจากวันที่สั่งซื้อเท่านั้นเพื่อเชื่อมโยงข้อมูลครับ

![20251028211727](https://static-docs.nocobase.com/20251028211727.png)

สำหรับข้อมูลเพิ่มเติมเกี่ยวกับตัวแปร โปรดดูที่ [ตัวแปร](/interface-builder/variables)

### การเชื่อมโยงฟิลด์ความสัมพันธ์

การเชื่อมโยงระหว่างฟิลด์ความสัมพันธ์สามารถทำได้โดยการตั้งค่าขอบเขตข้อมูลครับ

ตัวอย่าง: คอลเลกชันคำสั่งซื้อมีฟิลด์ความสัมพันธ์แบบ One-to-Many ชื่อ "Opportunity Product" และฟิลด์ความสัมพันธ์แบบ Many-to-One ชื่อ "Opportunity" ครับ และคอลเลกชัน Opportunity Product ก็มีฟิลด์ความสัมพันธ์แบบ Many-to-One ชื่อ "Opportunity" ด้วยเช่นกัน ในบล็อกฟอร์มคำสั่งซื้อ ข้อมูลที่เลือกได้สำหรับ "Opportunity Product" จะถูกกรองให้แสดงเฉพาะสินค้าโอกาสทางธุรกิจที่เชื่อมโยงกับ "Opportunity" ที่เลือกอยู่ในฟอร์มปัจจุบันเท่านั้นครับ
