# แสดงป้ายกำกับ

## บทนำ

โดยปกติแล้ว ฟิลด์ต่างๆ ในบล็อกจะแสดงป้ายกำกับฟิลด์ เพื่อช่วยระบุความหมายของแต่ละฟิลด์ ทำให้ผู้ใช้สามารถอ่านและทำความเข้าใจข้อมูลได้ง่ายขึ้นครับ/ค่ะ

ในบางสถานการณ์ เช่น ในหน้าแสดงผลข้อมูล หรือเมื่อความหมายของฟิลด์นั้นชัดเจนอยู่แล้ว คุณสามารถเลือกซ่อนป้ายกำกับได้ เพื่อให้ส่วนติดต่อผู้ใช้ (UI) ดูเรียบง่ายและสะอาดตามากขึ้นครับ/ค่ะ เมื่อซ่อนป้ายกำกับแล้ว ระบบจะแสดงเฉพาะค่าของฟิลด์เท่านั้น โดยจะไม่แสดงชื่อฟิลด์อีกต่อไปครับ/ค่ะ
