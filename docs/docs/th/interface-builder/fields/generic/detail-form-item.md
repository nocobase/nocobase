# ฟิลด์รายละเอียด

## บทนำ

การตั้งค่าฟิลด์สำหรับบล็อกรายละเอียด, บล็อกรายการ, และบล็อกตาราง จะคล้ายคลึงกันครับ/ค่ะ โดยมีวัตถุประสงค์หลักเพื่อควบคุมการแสดงผลของฟิลด์เมื่ออยู่ในโหมดอ่านอย่างเดียว

![20251025172851](https://static-docs.nocobase.com/20251025172851.png)

## ตัวเลือกการตั้งค่าฟิลด์

### การจัดรูปแบบฟิลด์วันที่

![20251025173005](https://static-docs.nocobase.com/20251025173005.png)

ดูรายละเอียดเพิ่มเติมได้ที่ [การจัดรูปแบบวันที่](/interface-builder/fields/specific/date-picker)

### การจัดรูปแบบฟิลด์ตัวเลข

![20251025173242](https://static-docs.nocobase.com/20251025173242.png)

รองรับการแปลงหน่วยอย่างง่าย, ตัวคั่นหลักพัน, คำนำหน้าและคำต่อท้าย, ความแม่นยำ, และสัญกรณ์วิทยาศาสตร์ครับ/ค่ะ

ดูรายละเอียดเพิ่มเติมได้ที่ [การจัดรูปแบบตัวเลข](/interface-builder/fields/field-settings/number-format)

### เปิดใช้งานการคลิกเพื่อเปิด

นอกเหนือจากฟิลด์ความสัมพันธ์ที่รองรับการคลิกเพื่อเปิดป๊อปอัปแล้ว ฟิลด์ทั่วไปก็สามารถเปิดใช้งานการคลิกเพื่อเปิด เพื่อใช้เป็นจุดเข้าในการเปิดป๊อปอัปได้เช่นกันครับ/ค่ะ และยังสามารถตั้งค่าวิธีการเปิดป๊อปอัปได้ด้วย (ลิ้นชัก, กล่องโต้ตอบ, หรือหน้าย่อย)

![20251025173549](https://static-docs.nocobase.com/20251025173549.gif)

### วิธีการแสดงผลเมื่อเนื้อหาเกิน

เมื่อเนื้อหาในฟิลด์เกินความกว้างที่กำหนด คุณสามารถตั้งค่าวิธีการจัดการการเกินได้ครับ/ค่ะ

- แสดงแบบมีจุดไข่ปลา (ค่าเริ่มต้น)
- ขึ้นบรรทัดใหม่

![20251025173917](https://static-docs.nocobase.com/20251025173917.png)

### คอมโพเนนต์ฟิลด์

ฟิลด์บางประเภทรองรับการแสดงผลได้หลายรูปแบบ ซึ่งสามารถทำได้โดยการสลับคอมโพเนนต์ฟิลด์ครับ/ค่ะ

ตัวอย่างเช่น: คอมโพเนนต์ `URL` สามารถสลับไปเป็นคอมโพเนนต์ `Preview` ได้ครับ/ค่ะ

![20251025174042](https://static-docs.nocobase.com/20251025174042.png)

ตัวอย่างเช่น ฟิลด์ความสัมพันธ์สามารถสลับการแสดงผลได้หลายแบบ โดยเปลี่ยนจากคอมโพเนนต์ฟิลด์ชื่อเรื่องเป็น `รายละเอียดรอง` เพื่อแสดงเนื้อหาของฟิลด์ความสัมพันธ์ได้มากขึ้นครับ/ค่ะ

![20251025174311](https://static-docs.nocobase.com/20251025174311.gif)
