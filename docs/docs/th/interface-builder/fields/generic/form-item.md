# ฟิลด์ในฟอร์ม

## บทนำ

ฟิลด์ประเภทต่างๆ ในฟอร์มจะมีตัวเลือกการตั้งค่าที่แตกต่างกันครับ ผู้ใช้สามารถขยายตัวเลือกการตั้งค่าของฟิลด์เหล่านี้ผ่าน **ปลั๊กอิน** เพื่อตอบสนองความต้องการเฉพาะบุคคลได้มากขึ้นครับ

![20251025174555](https://static-docs.nocobase.com/20251025174555.png)

## ตัวเลือกการตั้งค่าฟิลด์

### ค่าเริ่มต้น

คุณสามารถกำหนดค่าคงที่หรือตัวแปรเป็นค่าเริ่มต้นสำหรับฟิลด์ได้ครับ (รองรับเฉพาะฟอร์มที่สร้างใหม่เท่านั้น)

![20251025174724](https://static-docs.nocobase.com/20251025174724.png)

ดูรายละเอียดเพิ่มเติมได้ที่ [การตั้งค่าค่าเริ่มต้น](/interface-builder/fields/field-settings/default-value)
