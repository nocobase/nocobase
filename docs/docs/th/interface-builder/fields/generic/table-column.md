# ฟิลด์ตาราง

## บทนำ

ฟิลด์ตารางนั้น นอกจากจะรองรับการปรับความกว้างของคอลัมน์, ชื่อฟิลด์, การเรียงลำดับ และอื่นๆ แล้ว สำหรับฟิลด์เฉพาะบางประเภท (เช่น ฟิลด์วันที่, ฟิลด์ความสัมพันธ์, และฟิลด์ตัวเลข) ยังสามารถตั้งค่าการแสดงผลแบบเฉพาะตัวได้อีกด้วยครับ/ค่ะ

![20251024174558](https://static-docs.nocobase.com/20251024174558.png)

## ตัวเลือกการตั้งค่าฟิลด์

### การจัดรูปแบบฟิลด์วันที่

![20251024175303](https://static-docs.nocobase.com/20251024175303.png)

ดูรายละเอียดเพิ่มเติมได้ที่ [การจัดรูปแบบวันที่](/interface-builder/fields/specific/date-picker)

### การจัดรูปแบบฟิลด์ตัวเลข

รองรับการแปลงหน่วยอย่างง่าย, ตัวคั่นหลักพัน, คำนำหน้าและคำต่อท้าย, ความแม่นยำ, และสัญกรณ์วิทยาศาสตร์ครับ/ค่ะ

![20251024175445](https://static-docs.nocobase.com/20251024175445.png)

ดูรายละเอียดเพิ่มเติมได้ที่ [การจัดรูปแบบตัวเลข](/interface-builder/fields/field-settings/number-format)

### เปิดใช้งานการแก้ไขด่วน

เมื่อเปิดใช้งานการแก้ไขด่วน จะมีปุ่มแก้ไขปรากฏขึ้นเมื่อเลื่อนเมาส์ไปวางเหนือคอลัมน์นั้นๆ ครับ/ค่ะ คุณสามารถคลิกเพื่อแก้ไขและบันทึกข้อมูลได้อย่างรวดเร็วในหน้าต่างป๊อปอัปได้เลยครับ/ค่ะ

![20251025171158](https://static-docs.nocobase.com/20251025171158.gif)

### เปิดใช้งานการคลิกเพื่อเปิด

นอกเหนือจากฟิลด์ความสัมพันธ์ที่รองรับการคลิกเพื่อเปิดหน้าต่างป๊อปอัปแล้ว ฟิลด์ทั่วไปก็สามารถเปิดใช้งาน "คลิกเพื่อเปิด" เพื่อใช้เป็นจุดเข้าถึงในการเปิดหน้าต่างป๊อปอัปได้เช่นกันครับ/ค่ะ

![20251025172308](https://static-docs.nocobase.com/20251025172308.gif)

### โหมดการแสดงผลเมื่อเนื้อหาเกินขอบเขต

เมื่อเนื้อหาในคอลัมน์เกินความกว้างของคอลัมน์ตาราง คุณสามารถตั้งค่าโหมดการแสดงผลเมื่อเนื้อหาเกินขอบเขตได้ดังนี้ครับ/ค่ะ

- แสดงแบบย่อ (ค่าเริ่มต้น)
- ขึ้นบรรทัดใหม่

![20251025172549](https://static-docs.nocobase.com/20251025172549.png)

### คอลัมน์ตรึง

![20251025170858](https://static-docs.nocobase.com/20251025170858.gif)

### คอมโพเนนต์ฟิลด์

ฟิลด์บางประเภทรองรับรูปแบบการแสดงผลที่หลากหลายครับ/ค่ะ ผู้ใช้สามารถสลับคอมโพเนนต์ฟิลด์เพื่อสร้างเอฟเฟกต์การแสดงผลที่แตกต่างกัน เพื่อตอบสนองความต้องการในสถานการณ์ต่างๆ ได้ครับ/ค่ะ ตัวอย่างเช่น ฟิลด์ประเภท **URL** สามารถเปลี่ยนเป็นคอมโพเนนต์ **Preview** เพื่อแสดงเนื้อหาลิงก์หรือภาพตัวอย่างได้ดียิ่งขึ้นครับ/ค่ะ

![20251025171658](https://static-docs.nocobase.com/20251025171658.png)

- ข้อความ

### สามารถเรียงลำดับได้

ปัจจุบัน ฟิลด์ส่วนใหญ่รองรับการเรียงลำดับข้อมูลจากฝั่งเซิร์ฟเวอร์ครับ/ค่ะ เมื่อเปิดใช้งานการเรียงลำดับ คุณสามารถเรียงข้อมูลตามฟิลด์เป้าหมายได้ทั้งแบบจากน้อยไปมาก (Ascending) หรือจากมากไปน้อย (Descending) ครับ/ค่ะ

![20251125221247](https://static-docs.nocobase.com/20251125221247.png)
