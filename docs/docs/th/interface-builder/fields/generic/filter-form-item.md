# ฟิลด์สำหรับกรองข้อมูลในฟอร์ม

## บทนำ

ฟอร์มกรองข้อมูลรองรับการเลือกฟิลด์จากคอลเลกชันปัจจุบันและฟิลด์จากคอลเลกชันที่เกี่ยวข้อง (รวมถึงฟิลด์ของความสัมพันธ์ที่ซ้อนกัน) เพื่อใช้เป็นฟิลด์สำหรับกรองข้อมูลครับ/ค่ะ

### ตัวดำเนินการ

โปรดเลือกตัวดำเนินการที่เหมาะสมตามประเภทของฟิลด์เพื่อทำการกรองข้อมูลครับ/ค่ะ การทำเช่นนี้จะช่วยเพิ่มความแม่นยำและประสิทธิภาพในการกรองข้อมูล สำหรับฟิลด์ประเภทสตริง ค่าเริ่มต้นคือการจับคู่แบบคลุมเครือ (fuzzy matching) ครับ/ค่ะ
