# การแสดงตัวอย่างไฟล์

ในอินเทอร์เฟซที่มีฟิลด์ไฟล์ (รวมถึงฟิลด์ไฟล์แนบ) คุณสามารถแสดงตัวอย่างไฟล์ได้โดยการคลิกที่รูปภาพย่อ (thumbnail) หรือไอคอนของไฟล์ครับ ฟังก์ชันการแสดงตัวอย่างในตัวรองรับไฟล์หลากหลายประเภท รวมถึงรูปภาพ, PDF และประเภทไฟล์ส่วนใหญ่ที่เบราว์เซอร์รองรับโดยตรงครับ

![20251129232307](https://static-docs.nocobase.com/20251129232307.png)

สำหรับประเภทไฟล์ที่ไม่รองรับการแสดงตัวอย่างโดยตรง คุณสามารถเปิดใช้งานฟังก์ชันการแสดงตัวอย่างได้โดยการติดตั้งหรือขยายปลั๊กอินการแสดงตัวอย่างไฟล์ที่เกี่ยวข้องครับ ตัวอย่างเช่น หลังจากติดตั้งปลั๊กอิน Office File Preview แล้ว คุณจะสามารถแสดงตัวอย่างไฟล์ Word, Excel และ PowerPoint ได้ครับ

ปัจจุบัน NocoBase มีปลั๊กอินสำหรับการแสดงตัวอย่างไฟล์ดังนี้ครับ:

* [ปลั๊กอินแสดงตัวอย่างไฟล์ Office](../file-preview/ms-office.md)
