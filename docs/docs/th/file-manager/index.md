# ตัวจัดการไฟล์

## บทนำ

ปลั๊กอินตัวจัดการไฟล์มีให้ใช้งานทั้งคอลเลกชันไฟล์, ฟิลด์ไฟล์แนบ และเอนจินจัดเก็บไฟล์ เพื่อช่วยให้การจัดการไฟล์ที่ผู้ใช้ได้อัปโหลดทำได้ง่ายและสะดวกยิ่งขึ้นครับ/ค่ะ
