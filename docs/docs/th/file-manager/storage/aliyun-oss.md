# สตอเรจเอนจิน: Aliyun OSS

สตอเรจเอนจินที่ใช้ Aliyun OSS นี้ ก่อนใช้งาน คุณจำเป็นต้องเตรียมบัญชีผู้ใช้และสิทธิ์ที่เกี่ยวข้องให้พร้อมก่อนนะครับ/คะ

## พารามิเตอร์การตั้งค่า

![ตัวอย่างการตั้งค่าสตอเรจเอนจิน Aliyun OSS](https://static-docs.nocobase.com/20240712220011.png)

:::info{title=ข้อแนะนำ}
ส่วนนี้จะแนะนำเฉพาะพารามิเตอร์เฉพาะสำหรับสตอเรจเอนจิน Aliyun OSS เท่านั้นนะครับ/คะ สำหรับพารามิเตอร์ทั่วไป โปรดดูที่ [พารามิเตอร์ทั่วไปของเอนจิน](./index#引擎通用参数) ครับ/ค่ะ
:::

### ภูมิภาค

ระบุภูมิภาคของ OSS ที่ใช้จัดเก็บข้อมูล เช่น `oss-cn-hangzhou` ครับ/ค่ะ

:::info{title=ข้อแนะนำ}
คุณสามารถดูข้อมูลภูมิภาคของพื้นที่จัดเก็บ (Bucket) ได้จาก [คอนโซล Aliyun OSS](https://oss.console.aliyun.com/) นะครับ/คะ และคุณเพียงแค่ใช้ส่วนที่เป็นคำนำหน้าของภูมิภาคเท่านั้น (ไม่จำเป็นต้องใช้ชื่อโดเมนแบบเต็ม)
:::

### AccessKey ID

ระบุ ID ของ Access Key สำหรับการเข้าถึงที่ได้รับอนุญาตจาก Aliyun ครับ/ค่ะ

### AccessKey Secret

ระบุ Secret ของ Access Key สำหรับการเข้าถึงที่ได้รับอนุญาตจาก Aliyun ครับ/ค่ะ

### Bucket

ระบุชื่อ Bucket ของ OSS ที่ใช้จัดเก็บข้อมูลครับ/ค่ะ

### ระยะหมดเวลา

ระบุระยะหมดเวลาสำหรับการอัปโหลดไปยัง Aliyun OSS โดยมีหน่วยเป็นมิลลิวินาที ค่าเริ่มต้นคือ `60000` มิลลิวินาที (หรือ 60 วินาที) ครับ/ค่ะ
