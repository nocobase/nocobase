# เอนจินจัดเก็บข้อมูล: Amazon S3

เอนจินจัดเก็บข้อมูลที่ใช้ Amazon S3 เป็นพื้นฐาน ก่อนใช้งาน คุณต้องเตรียมบัญชีผู้ใช้และสิทธิ์ที่เกี่ยวข้องให้พร้อมครับ/ค่ะ

## พารามิเตอร์การตั้งค่า

![ตัวอย่างการตั้งค่าเอนจินจัดเก็บข้อมูล Amazon S3](https://static-docs.nocobase.com/20251031092524.png)

:::info{title=ข้อแนะนำ}
ส่วนนี้จะแนะนำเฉพาะพารามิเตอร์เฉพาะสำหรับเอนจินจัดเก็บข้อมูล Amazon S3 เท่านั้น สำหรับพารามิเตอร์ทั่วไป โปรดดูที่ [พารามิเตอร์ทั่วไปของเอนจิน](./index#引擎通用参数) ครับ/ค่ะ
:::

### รีเจียน

ระบุรีเจียนที่ใช้จัดเก็บข้อมูล S3 เช่น `us-west-1` ครับ/ค่ะ

:::info{title=ข้อแนะนำ}
คุณสามารถดูข้อมูลรีเจียนของ Bucket ได้ที่ [Amazon S3 console](https://console.aws.amazon.com/s3/) โดยใช้เพียงแค่ส่วนคำนำหน้าของรีเจียนเท่านั้น (ไม่จำเป็นต้องใช้ชื่อโดเมนแบบเต็ม) ครับ/ค่ะ
:::

### AccessKey ID

ระบุ AccessKey ID สำหรับการเข้าถึง Amazon S3 ครับ/ค่ะ

### AccessKey Secret

ระบุ AccessKey Secret สำหรับการเข้าถึง Amazon S3 ครับ/ค่ะ

### Bucket

ระบุชื่อ Bucket ของ S3 ที่ใช้จัดเก็บข้อมูลครับ/ค่ะ
