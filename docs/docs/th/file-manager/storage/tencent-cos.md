# Tencent Cloud COS

เป็นเอนจินจัดเก็บข้อมูลที่ทำงานบน Tencent Cloud COS ครับ/ค่ะ ก่อนใช้งาน คุณจะต้องเตรียมบัญชีผู้ใช้และสิทธิ์ที่เกี่ยวข้องให้พร้อมก่อนนะครับ/คะ

## พารามิเตอร์การตั้งค่า

![ตัวอย่างการตั้งค่าเอนจินจัดเก็บข้อมูล Tencent COS](https://static-docs.nocobase.com/20240712222125.png)

:::info{title=ข้อแนะนำ}
ส่วนนี้จะอธิบายเฉพาะพารามิเตอร์ที่ใช้สำหรับเอนจินจัดเก็บข้อมูล Tencent Cloud COS โดยเฉพาะครับ/ค่ะ สำหรับพารามิเตอร์ทั่วไป โปรดดูที่ [พารามิเตอร์ทั่วไปของเอนจิน](./index.md#general-engine-parameters) ครับ/ค่ะ
:::

### ภูมิภาค (Region)

กรอกภูมิภาคที่ใช้สำหรับจัดเก็บข้อมูล COS เช่น `ap-chengdu` ครับ/ค่ะ

:::info{title=ข้อแนะนำ}
คุณสามารถดูข้อมูลภูมิภาคของพื้นที่จัดเก็บ (Bucket) ได้ที่ [Tencent Cloud COS Console](https://console.cloud.tencent.com/cos) ครับ/ค่ะ โดยคุณจะต้องใช้เพียงแค่ส่วนนำหน้าของภูมิภาคเท่านั้น (ไม่จำเป็นต้องใช้ชื่อโดเมนแบบเต็ม)
:::

### SecretId

กรอก ID ของคีย์การเข้าถึง (Access Key) ที่ได้รับอนุญาตจาก Tencent Cloud ครับ/ค่ะ

### SecretKey

กรอก Secret ของคีย์การเข้าถึง (Access Key) ที่ได้รับอนุญาตจาก Tencent Cloud ครับ/ค่ะ

### บักเก็ต (Bucket)

กรอกชื่อบักเก็ต (Bucket) ที่ใช้สำหรับจัดเก็บข้อมูล COS เช่น `qing-cdn-1234189398` ครับ/ค่ะ
