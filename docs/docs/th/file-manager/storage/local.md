# เอนจินจัดเก็บข้อมูล: ที่จัดเก็บข้อมูลภายในเครื่อง

ไฟล์ที่อัปโหลดจะถูกบันทึกไว้ในไดเรกทอรีบนฮาร์ดไดรฟ์ภายในเครื่องของเซิร์ฟเวอร์ครับ/ค่ะ วิธีนี้เหมาะสำหรับสถานการณ์ที่ระบบมีการจัดการไฟล์ที่อัปโหลดในปริมาณไม่มาก หรือเพื่อวัตถุประสงค์ในการทดลองครับ/ค่ะ

## พารามิเตอร์การตั้งค่า

![ตัวอย่างการตั้งค่าเอนจินจัดเก็บไฟล์](https://static-docs.nocobase.com/20240529115151.png)

:::info{title=ข้อแนะนำ}
ส่วนนี้จะแนะนำเฉพาะพารามิเตอร์ที่ใช้สำหรับเอนจินจัดเก็บข้อมูลภายในเครื่องเท่านั้นครับ/ค่ะ สำหรับพารามิเตอร์ทั่วไป โปรดดูที่ [พารามิเตอร์เอนจินทั่วไป](./index.md#引擎通用参数) ครับ/ค่ะ
:::

### พาธ

พาธนี้จะแสดงทั้งพาธสัมพัทธ์ (relative path) สำหรับการจัดเก็บไฟล์บนเซิร์ฟเวอร์ และพาธ URL สำหรับการเข้าถึงครับ/ค่ะ ตัวอย่างเช่น "`user/avatar`" (โดยไม่ต้องมีเครื่องหมายสแลช "/" ทั้งที่จุดเริ่มต้นและจุดสิ้นสุด) จะหมายถึง:

1. พาธสัมพัทธ์บนเซิร์ฟเวอร์ที่ใช้จัดเก็บไฟล์ที่อัปโหลด: `/path/to/nocobase-app/storage/uploads/user/avatar` ครับ/ค่ะ
2. คำนำหน้า URL สำหรับการเข้าถึงไฟล์: `http://localhost:13000/storage/uploads/user/avatar` ครับ/ค่ะ
