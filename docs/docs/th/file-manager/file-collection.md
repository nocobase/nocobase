# คอลเลกชันไฟล์

## บทนำ

คอลเลกชันไฟล์คือคอลเลกชันที่ออกแบบมาโดยเฉพาะสำหรับจัดเก็บข้อมูลเมตาของไฟล์ โดยมีฟิลด์ในตัว เช่น ชื่อไฟล์ ขนาด ประเภท และพาธที่จัดเก็บครับ/ค่ะ
