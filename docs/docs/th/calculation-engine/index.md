# ภาพรวมของเอนจินการคำนวณ

เอนจินการคำนวณมีบทบาทสำคัญในฟิลด์สูตร (Formula fields), กฎการเชื่อมโยง (Linkage rules) และเวิร์กโฟลว์ โดยเป็นเครื่องมือคำนวณที่ใช้งานได้หลากหลายครับ

## Formula.js

**Formula.js** เป็นไลบรารี JavaScript น้ำหนักเบาที่ออกแบบมาเพื่อการคำนวณสูตรโดยเฉพาะ โดยรวบรวมฟังก์ชันทางคณิตศาสตร์และสถิติไว้อย่างครบถ้วน ช่วยให้ผู้ใช้สามารถคำนวณที่ซับซ้อนได้อย่างง่ายดายครับ คุณสมบัติหลัก ได้แก่:

- **ฟังก์ชันที่หลากหลาย**: มีชุดฟังก์ชันที่ครอบคลุมการคำนวณทางคณิตศาสตร์ประเภทต่างๆ
- **ใช้งานง่าย**: มี API ที่เรียบง่าย ช่วยให้ทั้งนักพัฒนาและผู้ที่ไม่ใช่นักพัฒนาสามารถสร้างสูตรได้โดยไม่ต้องเขียนโค้ดจำนวนมาก
- **การคำนวณแบบเรียลไทม์**: ปรับแต่งมาเพื่อการประมวลผลข้อมูลแบบเรียลไทม์ เหมาะสำหรับแอปพลิเคชันที่ต้องการผลลัพธ์การคำนวณในทันที

## Math.js

**Math.js** เป็นไลบรารีคณิตศาสตร์ที่มีฟีเจอร์ครบครันสำหรับ JavaScript และ Node.js รองรับการคำนวณทางคณิตศาสตร์ที่หลากหลาย ตั้งแต่เลขคณิตพื้นฐานไปจนถึงพีชคณิตขั้นสูง สถิติ และพีชคณิตเชิงเส้นครับ คุณสมบัติหลัก ได้แก่:

- **ชุดฟังก์ชันที่ครอบคลุม**: มีฟังก์ชันจำนวนมากสำหรับการคำนวณทางคณิตศาสตร์ทุกรูปแบบ
- **การแยกแยะนิพจน์ที่ยืดหยุ่น**: สามารถแยกแยะ (Parse) และประมวลผลนิพจน์ทางคณิตศาสตร์ในรูปแบบข้อความ (String) ได้
- **การคำนวณขั้นสูง**: รองรับการดำเนินการเมทริกซ์ (Matrix), การคำนวณเชิงสัญลักษณ์ (Symbolic computation) และการวิเคราะห์ทางสถิติ
- **ความสามารถในการขยาย**: สามารถเพิ่มฟังก์ชันและค่าคงที่ที่กำหนดเองได้ เพื่อตอบสนองความต้องการเฉพาะด้าน

## การประยุกต์ใช้ใน NocoBase

ด้วยการผสานการใช้งาน Formula.js และ Math.js เข้าด้วยกัน NocoBase ช่วยให้:

- **ดำเนินการคำนวณได้กว้างขวาง**: ตั้งแต่การคำนวณพื้นฐานไปจนถึงการดำเนินการทางคณิตศาสตร์ที่ซับซ้อน
- **สร้างแอปพลิเคชันที่แข็งแกร่งและมีประสิทธิภาพ**: สร้างแอปพลิเคชันที่ต้องการความแม่นยำและฟังก์ชันการคำนวณที่ทรงพลัง
- **เพิ่มประสิทธิภาพการทำงาน**: ใช้ประโยชน์จากฟังก์ชันสำเร็จรูปเพื่อหลีกเลี่ยงการเขียนโค้ดเองจำนวนมาก ซึ่งช่วยเพิ่มความเร็วในการพัฒนา

ไลบรารีทั้งสองนี้มีบทบาทสำคัญใน NocoBase ช่วยให้เราสามารถสร้างตรรกะการคำนวณที่ยืดหยุ่นและปรับแต่งได้โดยไม่ต้องมีความรู้ด้านการเขียนโปรแกรมเชิงลึก ซึ่งไม่เพียงแต่ช่วยให้กระบวนการพัฒนาง่ายขึ้น แต่ยังเพิ่มความสามารถและความยืดหยุ่นให้กับระบบอีกด้วยครับ
