| **ROMAN** | แปลงตัวเลขอาหรับเป็นตัวเลขโรมัน | `ROMAN(499)` | ตัวเลขอาหรับที่ต้องการแปลง | CDXCIX |
| **SEARCH** | ค้นหาตำแหน่งของข้อความย่อย (ไม่คำนึงถึงตัวพิมพ์เล็ก-ใหญ่) | `SEARCH('margin', 'Profit Margin')` | ข้อความที่ต้องการหา, ข้อความต้นฉบับ | 8 |
| **SUBSTITUTE** | แทนที่ข้อความเก่าด้วยข้อความใหม่ในตำแหน่งที่ระบุ | `SUBSTITUTE('Quarter 1, 2011', '1', '2', 3)` | ข้อความเดิม, ข้อความเก่า, ข้อความใหม่, ลำดับที่ต้องการแทนที่ (ไม่บังคับ) | Quarter 1, 2012 |
| **T** | คืนค่าข้อความหากค่านั้นเป็นข้อความ มิฉะนั้นจะคืนค่าว่าง | `T('Rainfall')` | อาร์กิวเมนต์สามารถเป็นข้อมูลประเภทใดก็ได้ | Rainfall |
| **TRIM** | ลบช่องว่างออกจากข้อความ ยกเว้นช่องว่างเดี่ยวระหว่างคำ | `TRIM(' First Quarter Earnings ')` | ข้อความที่ต้องการตัดช่องว่าง | First Quarter Earnings |
| **TEXTJOIN** | รวมข้อความหลายชุดเข้าด้วยกันโดยใช้ตัวคั่นที่ระบุ | `TEXTJOIN(' ', true, 'The', '', 'sun', 'will', 'come', 'up', 'tomorrow.')` | ตัวคั่น, แฟล็กข้ามค่าว่าง, รายการข้อความที่ต้องการรวม | The sun will come up tomorrow. |
| **UNICHAR** | คืนค่าอักขระสำหรับรหัส Unicode ที่กำหนด | `UNICHAR(66)` | รหัส Unicode | B |
| **UNICODE** | คืนค่ารหัส Unicode ของอักขระตัวแรกในข้อความ | `UNICODE('B')` | ข้อความที่มีอักขระตัวเดียว | 66 |
| **UPPER** | แปลงอักขระทั้งหมดเป็นตัวพิมพ์ใหญ่ | `UPPER('total')` | ข้อความที่ต้องการแปลง | TOTAL |
