# Mathjs

[Math.js](https://mathjs.org/) เป็นไลบรารีทางคณิตศาสตร์ที่มีฟีเจอร์มากมายสำหรับ JavaScript และ Node.js ครับ

## ฟังก์ชันอ้างอิง

### นิพจน์ (Expressions)

| ฟังก์ชัน | คำอธิบาย | ตัวอย่างการเรียกใช้ | พารามิเตอร์ | ผลลัพธ์ที่คาดหวัง |
| --- | --- | --- | --- | --- |
| **compile** | วิเคราะห์และคอมไพล์นิพจน์ (ทำหน้าที่วิเคราะห์เท่านั้น และไม่คืนค่าผลลัพธ์โดยตรง) | `compile('2 + 3')` | นิพจน์ (สตริง) | `{}` |
| **evaluate** | คำนวณนิพจน์และคืนค่าผลลัพธ์ | `evaluate('2 + 3')` | นิพจน์ (สตริง), ขอบเขต (ตัวเลือกเสริม) | `5` |
| **help** | เรียกดูเอกสารคู่มือการใช้งานสำหรับฟังก์ชันหรือประเภทข้อมูล | `help('evaluate')` | คำค้นหา (สตริง) | `{  "name": "evaluate",  "category": "Excodession",  "syntax": [    "evaluate(excodession)",    "evaluate(excodession, scope)",    "evaluate([expr1, expr2, expr3, ...])",    "evaluate([expr1, expr2, expr3, ...], scope)"  ],  "description": "Evaluate an excodession or an array with excodessions.",  "examples": [    "evaluate(\"2 + 3\")",    "evaluate(\"sqrt(16)\")",    "evaluate(\"2 inch to cm\")",    "evaluate(\"sin(x * pi)\", { \"x\": 1/2 })",    "evaluate([\"width=2\", \"height=4\",\"width*height\"])"  ],  "seealso": [],  "mathjs": "Help"}` |
| **parser** | สร้างตัววิเคราะห์ (parser) สำหรับการดำเนินการที่กำหนดเอง | `parser()` | ไม่มี | `{}` |

| **rationalize** | แปลงนิพจน์ที่สามารถทำให้เป็นตรรกยะได้ให้เป็นเศษส่วนตรรกยะ | `rationalize('1/(x+1)')` | นิพจน์ (สตริงหรือโหนด) | `{  "mathjs": "OperatorNode",  "op": "/",  "fn": "divide",  "args": [    {      "mathjs": "ConstantNode",      "value": 1    },    {      "mathjs": "OperatorNode",      "op": "+",      "fn": "add",      "args": [        {          "mathjs": "SymbolNode",          "name": "x"        },        {          "mathjs": "ConstantNode",          "value": 1        }      ],      "implicit": false,      "isPercentage": false    }  ],  "implicit": false,  "isPercentage": false}` |

| **symbolicEqual** | ตรวจสอบว่านิพจน์สองชุดมีความหมายทางสัญลักษณ์ที่เท่ากันหรือไม่ | `symbolicEqual('x+x', '2x')` | นิพจน์ 1 (สตริงหรือโหนด), นิพจน์ 2 (สตริงหรือโหนด) | `true` |

| **usolveAll** | ค้นหาคำตอบทั้งหมดของระบบสมการเชิงเส้นโดยใช้วิธีการแทนที่ย้อนกลับ | `usolveAll([[1,2],[0,1]], [3,4])` | U (อาร์เรย์หรือเมทริกซ์), b (อาร์เรย์หรือเมทริกซ์) | `[  [    [      -5    ],    [      4    ]  ]]` |

### เลขคณิต (Arithmetic)

| ฟังก์ชัน | คำอธิบาย | ตัวอย่างการเรียกใช้ | พารามิเตอร์ | ผลลัพธ์ที่คาดหวัง |
| --- | --- | --- | --- | --- |

| **ceil** | ปัดเศษขึ้นไปยังจำนวนเต็มที่ใกล้ที่สุด (สำหรับจำนวนเชิงซ้อน จะปัดเศษแต่ละส่วนแยกกัน) | `ceil(3.2)` | x (ตัวเลข, จำนวนเชิงซ้อน, อาร์เรย์ หรือเมทริกซ์) | `4` |

| **divide** | หารค่าสองค่า (x / y) | `divide(6, 2)` | x (ตัวเลข, อาร์เรย์ หรือเมทริกซ์), y (ตัวเลข, อาร์เรย์ หรือเมทริกซ์) | `3` |

| **dotPow** | คำนวณ x^y แบบทีละองค์ประกอบ | `dotPow([2,3],[2,3])` | x (อาร์เรย์หรือเมทริกซ์), y (อาร์เรย์หรือเมทริกซ์) | `[  4,  27]` |
| **exp** | คำนวณ e^x | `exp(1)` | x (ตัวเลข, จำนวนเชิงซ้อน, อาร์เรย์ หรือเมทริกซ์) | `2.718281828459045` |
| **expm1** | คำนวณ e^x - 1 | `expm1(1)` | x (ตัวเลขหรือจำนวนเชิงซ้อน) | `1.718281828459045` |
