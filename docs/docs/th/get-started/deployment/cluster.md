# การติดตั้งแบบคลัสเตอร์

NocoBase รองรับการติดตั้งแอปพลิเคชันในโหมดคลัสเตอร์ เพื่อเพิ่มประสิทธิภาพและความพร้อมใช้งานของแอปพลิเคชันครับ/ค่ะ การติดตั้งแบบคลัสเตอร์ช่วยให้คุณสามารถรันอินสแตนซ์ที่แตกต่างกันของแอปพลิเคชันบนเซิร์ฟเวอร์หลายเครื่อง หรือรันหลายโปรเซสในโหมดมัลติคอร์บนเซิร์ฟเวอร์เครื่องเดียวได้ครับ/ค่ะ

สำหรับรายละเอียดเพิ่มเติม โปรดดูเอกสาร [โหมดคลัสเตอร์](/cluster-mode/) ครับ/ค่ะ
