# ตัวจัดการเส้นทาง

## แนะนำ

ตัวจัดการเส้นทางเป็นเครื่องมือสำหรับจัดการเส้นทางของหน้าหลักในระบบ ซึ่งรองรับทั้งการใช้งานบน`เดสก์ท็อป`และ`มือถือ`ครับ/ค่ะ เส้นทางที่สร้างผ่านตัวจัดการเส้นทางจะถูกซิงค์และแสดงในเมนูโดยอัตโนมัติ (สามารถตั้งค่าไม่ให้แสดงในเมนูได้ครับ/ค่ะ) ในทางกลับกัน เมนูที่เพิ่มจากส่วนเมนูหน้าเพจก็จะถูกซิงค์มาแสดงในรายการของตัวจัดการเส้นทางด้วยเช่นกันครับ/ค่ะ

![20250107115449](https://static-docs.nocobase.com/20250107115449.png)

## คู่มือการใช้งาน

### ประเภทของเส้นทาง

ระบบรองรับเส้นทางอยู่ 4 ประเภทครับ/ค่ะ

- กลุ่ม (group): ใช้สำหรับจัดกลุ่มเส้นทางเพื่อการจัดการที่ง่ายขึ้น และสามารถมีเส้นทางย่อยได้ครับ/ค่ะ
- หน้าเพจ (page): หน้าภายในระบบ
- แท็บ (tab): ประเภทเส้นทางที่ใช้สำหรับสลับแท็บภายในหน้าเพจเดียวกันครับ/ค่ะ
- ลิงก์ (link): ลิงก์ภายในหรือภายนอก ที่สามารถนำทางไปยังที่อยู่ลิงก์ที่กำหนดค่าไว้ได้โดยตรงครับ/ค่ะ

### การเพิ่มเส้นทาง

คลิกปุ่ม “Add new” ที่มุมขวาบนเพื่อสร้างเส้นทางใหม่ได้เลยครับ/ค่ะ

1. เลือกประเภทเส้นทาง (Type)
2. กรอกชื่อเส้นทาง (Title)
3. เลือกไอคอนเส้นทาง (Icon)
4. ตั้งค่าว่าจะให้แสดงในเมนูหรือไม่ (Show in menu)
5. ตั้งค่าว่าจะเปิดใช้งานแท็บเพจหรือไม่ (Enable page tabs)
6. สำหรับเส้นทางประเภทหน้าเพจ ระบบจะสร้างพาธเส้นทาง (Path) ที่ไม่ซ้ำกันให้โดยอัตโนมัติครับ/ค่ะ

![20250124131803](https://static-docs.nocobase.com/20250124131803.png)

### การดำเนินการกับเส้นทาง

เส้นทางแต่ละรายการรองรับการดำเนินการดังต่อไปนี้ครับ/ค่ะ

- Add child: เพิ่มเส้นทางย่อย
- Edit: แก้ไขการตั้งค่าเส้นทาง
- View: ดูหน้าเพจของเส้นทาง
- Delete: ลบเส้นทาง

### การดำเนินการแบบกลุ่ม

แถบเครื่องมือด้านบนมีฟังก์ชันการดำเนินการแบบกลุ่มดังต่อไปนี้ครับ/ค่ะ

- Refresh: รีเฟรชรายการเส้นทาง
- Delete: ลบเส้นทางที่เลือก
- Hide in menu: ซ่อนเส้นทางที่เลือกจากเมนู
- Show in menu: แสดงเส้นทางที่เลือกในเมนู

### การกรองเส้นทาง

ใช้ฟังก์ชัน “Filter” ที่ด้านบนเพื่อกรองรายการเส้นทางตามที่คุณต้องการได้เลยครับ/ค่ะ

:::info{title=ข้อควรทราบ}
การแก้ไขการตั้งค่าเส้นทางจะส่งผลโดยตรงต่อโครงสร้างเมนูนำทางของระบบครับ/ค่ะ โปรดดำเนินการด้วยความระมัดระวังและตรวจสอบให้แน่ใจว่าการตั้งค่าเส้นทางถูกต้องสมบูรณ์นะครับ/คะ
:::
