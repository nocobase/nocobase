# ฝัง NocoBase

## บทนำ

ปลั๊กอินนี้ใช้สำหรับฝังหน้าของ NocoBase ลงในเว็บไซต์หรือแอปพลิเคชันอื่น ๆ ครับ/ค่ะ

## การติดตั้ง

ปลั๊กอินนี้เป็นปลั๊กอินเชิงพาณิชย์ครับ/ค่ะ และจำเป็นต้องอัปโหลดและเปิดใช้งานผ่านตัวจัดการปลั๊กอิน

![20240323162741](https://static-docs.nocobase.com/20240323162741.png)

## คู่มือการใช้งาน

หลังจากเปิดใช้งานการกำหนดค่าอินเทอร์เฟซแล้ว ให้คลิกที่ปุ่ม `คัดลอกลิงก์สำหรับฝัง` ในเมนูการตั้งค่าของหน้า (มุมขวาบน) ครับ/ค่ะ

![](https://static-docs.nocobase.com/f11bd6d5e88d38731d7cd3cb149022c8.png)

หลังจากคลิกแล้ว คุณจะได้รับลิงก์ที่สามารถเปิดแยกต่างหากได้ครับ/ค่ะ ตัวอย่างเช่น `https://xxx.com/embed/qs087rz4o2b`

![](https://static-docs.nocobase.com/9d847805a00fd07372a474665ec0e968.png)

หากคุณต้องการฝังหน้าของ NocoBase ลงในเว็บไซต์หรือแอปพลิเคชันอื่น ๆ จำเป็นต้องมีการยืนยันตัวตนผู้ใช้ครับ/ค่ะ และต้องเพิ่ม `token` เข้าไปในลิงก์ด้วย ตัวอย่างเช่น `https://xxx.com/embed/qs087rz4o2b?token=xxx`
