# พนักงาน AI · เริ่มต้นใช้งาน

พนักงาน AI ใน NocoBase ไม่ใช่แชทบอท หรือ Agent ที่ทำงานแยกกัน แต่เป็นเพื่อนร่วมงานอัจฉริยะที่ผสานรวมเข้ากับระบบธุรกิจของคุณได้อย่างราบรื่น สามารถเข้าใจบริบททางธุรกิจ และดำเนินการตามคำสั่งได้โดยตรงครับ/ค่ะ

พวกเขาจะปรากฏขึ้นในทุกที่ที่คุณต้องการ ไม่ว่าจะเป็นข้างตาราง หรือข้างฟอร์ม เพื่อให้คุณสามารถเรียกใช้งานได้ทันทีและง่ายดายครับ/ค่ะ

![20251102121036-2025-11-02-12-10-38](https://static-docs.nocobase.com/20251102121036-2025-11-02-12-10-38.png)

คุณยังสามารถเรียกใช้งานพวกเขาได้จากรายการพนักงานที่มุมล่างขวาของหน้าจอครับ/ค่ะ

![20251102121159-2025-11-02-12-12-01](https://static-docs.nocobase.com/20251102121159-2025-11-02-12-12-01.png)

พวกเขาจะดึงข้อมูลมาใช้เป็นบริบทโดยอัตโนมัติครับ/ค่ะ เช่น **Viz** ที่อยู่ข้างบล็อกตาราง จะดึงข้อมูลจากตารางมาใช้โดยอัตโนมัติ และเรียกใช้เครื่องมือที่เหมาะสมเพื่อประมวลผลข้อมูลนั้น ซึ่งหมายความว่าคุณไม่จำเป็นต้องคัดลอกข้อมูลเหล่านี้ไปวางในแชทบอทอีกต่อไปครับ/ค่ะ

<video controls class="rounded shadow"><source src="https://static-docs.nocobase.com/z-2025-11-02-12.13.35-2025-11-02-12-13-46.mp4" type="video/mp4"></video>

พวกเขายังสามารถเข้าใจโครงสร้างหน้าเว็บเป็นบริบทได้โดยอัตโนมัติครับ/ค่ะ เช่น **Dex** ที่อยู่ข้างบล็อกฟอร์ม จะดึงโครงสร้างฟิลด์ของฟอร์มมาใช้โดยอัตโนมัติ และเรียกใช้เครื่องมือที่เหมาะสมเพื่อดำเนินการบนหน้าเว็บ ซึ่งหมายความว่าคุณไม่จำเป็นต้องคัดลอกข้อมูลกลับมาจากแชทบอทอีกต่อไปครับ/ค่ะ

<video controls class="rounded shadow"><source src="https://static-docs.nocobase.com/z-2025-11-02-12.16.07-2025-11-02-12-16-21.mp4" type="video/mp4"></video>

คุณยังสามารถเลือกองค์ประกอบบนหน้าเว็บโดยตรง แล้วส่งให้พนักงาน AI ได้ ซึ่งพวกเขาจะดึงข้อมูลและโครงสร้างหน้าเว็บที่เกี่ยวข้องออกมาได้โดยอัตโนมัติครับ/ค่ะ

<video controls class="rounded shadow"><source src="https://static-docs.nocobase.com/z-2025-11-02-12.17.25-2025-11-02-12-17-44.mp4" type="video/mp4"></video>

คุณสามารถตั้งค่าภารกิจล่วงหน้าให้กับพนักงาน AI แต่ละคนในตำแหน่งปัจจุบันได้ครับ/ค่ะ เพื่อให้ผู้ใช้สามารถคลิกเพื่อเรียกใช้งานได้ทันที โดยไม่จำเป็นต้องพิมพ์อะไรเลยครับ/ค่ะ
