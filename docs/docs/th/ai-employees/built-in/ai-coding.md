# พนักงาน AI · Nathan: วิศวกรส่วนหน้า (Frontend Engineer)

> ช่วยคุณเขียนและปรับแต่งโค้ดส่วนหน้า (frontend code) เพื่อสร้างตรรกะการโต้ตอบที่ซับซ้อนครับ/ค่ะ

`Nathan` คือผู้เชี่ยวชาญด้านการพัฒนาส่วนหน้า (frontend) ใน NocoBase ครับ/ค่ะ ในสถานการณ์ที่ต้องเขียน JavaScript เช่น `JSBlock`, `JSField`, `JSItem`, `JSColumn`, `JSAction`, `Event Flow` และ `Linkage` คุณจะเห็นรูปโปรไฟล์ของ `Nathan` ที่มุมขวาบนของตัวแก้ไขโค้ด (code editor) ซึ่งคุณสามารถให้เขาช่วยเขียนหรือแก้ไขโค้ดในตัวแก้ไขได้เลยครับ/ค่ะ

![20251022163405](https://static-docs.nocobase.com/20251022163405.png)

## ขั้นตอนการใช้งาน

ในตัวแก้ไขโค้ด (code editor) ให้คลิกที่ `Nathan` เพื่อเปิดกล่องโต้ตอบของพนักงาน AI ครับ/ค่ะ โค้ดจากตัวแก้ไขจะถูกแนบไปยังช่องป้อนข้อมูลโดยอัตโนมัติ และส่งไปให้ `Nathan` ในฐานะบริบทของแอปพลิเคชัน (application context) ครับ/ค่ะ

![20251022163935](https://static-docs.nocobase.com/20251022163935.png)

ป้อนความต้องการในการเขียนโค้ดของคุณ แล้วส่งไปให้ `Nathan` จากนั้นรอการตอบกลับจากเขาได้เลยครับ/ค่ะ

![20251022164041](https://static-docs.nocobase.com/20251022164041.png)

คลิกปุ่ม `Apply to editor` ที่บล็อกโค้ดซึ่ง `Nathan` ตอบกลับมา เพื่อนำโค้ดนั้นไปเขียนทับในตัวแก้ไขโค้ดครับ/ค่ะ

![20251022164323](https://static-docs.nocobase.com/20251022164323.png)

คลิกปุ่ม `Run` ในตัวแก้ไขโค้ด เพื่อดูผลลัพธ์ได้แบบเรียลไทม์ครับ/ค่ะ

![20251022164436](https://static-docs.nocobase.com/20251022164436.png)

## ประวัติโค้ด

คลิกไอคอน 'command line' ที่มุมขวาบนของกล่องโต้ตอบของ `Nathan` เพื่อดูส่วนของโค้ด (code snippets) ที่คุณส่งไป และส่วนของโค้ดที่ `Nathan` ตอบกลับมาในเซสชันปัจจุบันได้เลยครับ/ค่ะ
