# AI Agent · Dex: ผู้ช่วยจัดระเบียบข้อมูล

> ดึงข้อมูลและกรอกฟอร์มได้อย่างรวดเร็ว เปลี่ยนข้อมูลที่กระจัดกระจายให้เป็นระเบียบ

`Dex` คือผู้เชี่ยวชาญด้านการจัดระเบียบข้อมูลครับ/ค่ะ โดยจะดึงข้อมูลที่จำเป็นจากข้อมูลหรือไฟล์ที่ไม่มีโครงสร้าง จัดระเบียบให้เป็นข้อมูลที่มีโครงสร้าง และสามารถใช้เครื่องมือเพื่อกรอกข้อมูลลงในฟอร์มได้ครับ/ค่ะ

![20251022155210](https://static-docs.nocobase.com/20251022155210.png)

## ขั้นตอนการใช้งาน

บนหน้าฟอร์ม ให้เรียกใช้งาน `Dex` เพื่อเปิดหน้าต่างสนทนาครับ/ค่ะ

![20251022155316](https://static-docs.nocobase.com/20251022155316.png)

คลิก `Add work context` ในช่องป้อนข้อมูล แล้วเลือก `Pick block` ครับ/ค่ะ หน้าเว็บจะเข้าสู่โหมดการเลือกบล็อก

![20251022155414](https://static-docs.nocobase.com/20251022155414.png)

เลือกบล็อกฟอร์มบนหน้าเว็บครับ/ค่ะ

![20251022155523](https://static-docs.nocobase.com/20251022155523.png)

ในกล่องสนทนา ให้ป้อนข้อมูลที่คุณต้องการให้ `Dex` จัดระเบียบครับ/ค่ะ

![20251022155555](https://static-docs.nocobase.com/20251022155555.png)

หลังจากส่งข้อมูลแล้ว `Dex` จะจัดโครงสร้างข้อมูล และใช้ความสามารถของมันเพื่ออัปเดตข้อมูลลงในฟอร์มที่เลือกครับ/ค่ะ

![20251022155733](https://static-docs.nocobase.com/20251022155733.png)
