# AI เอเจนต์ · Orin: นักออกแบบโมเดลข้อมูล

> ออกแบบคอลเลกชันและปรับโครงสร้างฐานข้อมูลให้เหมาะสมอย่างชาญฉลาดครับ/ค่ะ

`Orin` เป็นผู้เชี่ยวชาญด้านการสร้างโมเดลข้อมูลครับ/ค่ะ คุณสามารถให้ `Orin` ช่วยสร้างหรือแก้ไขคอลเลกชันได้ที่หน้าการตั้งค่าแหล่งข้อมูลหลักครับ/ค่ะ

![20251022160628](https://static-docs.nocobase.com/20251022160628.png)

## ขั้นตอนการใช้งาน

เข้าสู่ปลั๊กอินจัดการแหล่งข้อมูล จากนั้นเลือกตั้งค่าแหล่งข้อมูลหลักครับ/ค่ะ

![20251022161146](https://static-docs.nocobase.com/20251022161146.png)

คลิกที่รูปโปรไฟล์ `Orin` ที่มุมขวาบน เพื่อเปิดกล่องโต้ตอบ AI เอเจนต์ครับ/ค่ะ

![20251022161641](https://static-docs.nocobase.com/20251022161641.png)

อธิบายความต้องการในการสร้างโมเดลของคุณให้ `Orin` ทราบ จากนั้นส่งข้อความและรอการตอบกลับครับ/ค่ะ

![20251022161749](https://static-docs.nocobase.com/20251022161749.png)

เมื่อ `Orin` ยืนยันความต้องการของคุณแล้ว `Orin` จะใช้ความสามารถของมันและตอบกลับด้วยการแสดงตัวอย่างโมเดลข้อมูลให้คุณครับ/ค่ะ

หลังจากตรวจสอบตัวอย่างแล้ว ให้คลิกปุ่ม `Finish review and apply` เพื่อสร้างคอลเลกชันตามโมเดลที่ `Orin` แนะนำครับ/ค่ะ

![20251022162142](https://static-docs.nocobase.com/20251022162142.png)
