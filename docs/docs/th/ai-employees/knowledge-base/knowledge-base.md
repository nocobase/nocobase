# ฐานข้อมูลความรู้

## บทนำ

ฐานข้อมูลความรู้เป็นรากฐานสำคัญสำหรับการดึงข้อมูลแบบ RAG (Retrieval-Augmented Generation) ครับ/ค่ะ โดยจะจัดระเบียบเอกสารเป็นหมวดหมู่และสร้างดัชนีไว้ เมื่อ AI Employee ตอบคำถาม ระบบจะค้นหาคำตอบจากฐานข้อมูลความรู้เป็นอันดับแรกครับ/ค่ะ

## การจัดการฐานข้อมูลความรู้

ไปที่หน้าการตั้งค่าปลั๊กอิน AI Employee จากนั้นคลิกแท็บ `Knowledge base` เพื่อเข้าสู่หน้าการจัดการฐานข้อมูลความรู้ครับ/ค่ะ

![20251023095649](https://static-docs.nocobase.com/20251023095649.png)

คลิกปุ่ม `Add new` ที่มุมขวาบนเพื่อเพิ่มฐานข้อมูลความรู้ประเภท `Local` ครับ/ค่ะ

![20251023095826](https://static-docs.nocobase.com/20251023095826.png)

กรอกข้อมูลที่จำเป็นสำหรับฐานข้อมูลความรู้ใหม่ดังนี้ครับ/ค่ะ:

- ในช่อง `Name` ให้ป้อนชื่อฐานข้อมูลความรู้ครับ/ค่ะ
- ใน `File storage` ให้เลือกตำแหน่งจัดเก็บไฟล์ครับ/ค่ะ
- ใน `Vector store` ให้เลือกที่เก็บเวกเตอร์ (Vector Store) ครับ/ค่ะ สามารถดูรายละเอียดเพิ่มเติมได้ที่ [ที่เก็บเวกเตอร์](/ai-employees/knowledge-base/vector-store)
- ในช่อง `Description` ให้ป้อนคำอธิบายฐานข้อมูลความรู้ครับ/ค่ะ

คลิกปุ่ม `Submit` เพื่อสร้างฐานข้อมูลความรู้ครับ/ค่ะ

![20251023095909](https://static-docs.nocobase.com/20251023095909.png)

## การจัดการเอกสารในฐานข้อมูลความรู้

หลังจากสร้างฐานข้อมูลความรู้แล้ว บนหน้าแสดงรายการฐานข้อมูลความรู้ ให้คลิกที่ฐานข้อมูลความรู้ที่คุณเพิ่งสร้าง เพื่อเข้าสู่หน้าการจัดการเอกสารในฐานข้อมูลความรู้ครับ/ค่ะ

![20251023100458](https://static-docs.nocobase.com/20251023100458.png)

![20251023100527](https://static-docs.nocobase.com/20251023100527.png)

คลิกปุ่ม `Upload` เพื่ออัปโหลดเอกสารครับ/ค่ะ หลังจากอัปโหลดเอกสารแล้ว ระบบจะเริ่มกระบวนการแปลงเป็นเวกเตอร์ (Vectorization) โดยอัตโนมัติ ให้รอจนกระทั่ง `Status` เปลี่ยนจาก `Pending` เป็น `Success` ครับ/ค่ะ

ปัจจุบัน ฐานข้อมูลความรู้รองรับเอกสารประเภทต่างๆ ดังนี้ครับ/ค่ะ: txt, pdf, doc, docx, ppt, pptx; สำหรับไฟล์ PDF จะรองรับเฉพาะข้อความธรรมดา (plain text) เท่านั้นครับ/ค่ะ

![20251023100901](https://static-docs.nocobase.com/20251023100901.png)

## ประเภทของฐานข้อมูลความรู้

### ฐานข้อมูลความรู้แบบ Local

ฐานข้อมูลความรู้แบบ Local คือฐานข้อมูลความรู้ที่จัดเก็บใน NocoBase ครับ/ค่ะ โดยทั้งเอกสารและข้อมูลเวกเตอร์ของเอกสารจะถูกจัดเก็บไว้ใน NocoBase แบบ Local ทั้งหมด

![20251023101620](https://static-docs.nocobase.com/20251023101620.png)

### ฐานข้อมูลความรู้แบบ Readonly

ฐานข้อมูลความรู้แบบ Readonly เป็นฐานข้อมูลความรู้แบบอ่านอย่างเดียวครับ/ค่ะ โดยเอกสารและข้อมูลเวกเตอร์จะถูกดูแลจากภายนอก และ NocoBase จะสร้างเพียงการเชื่อมต่อกับฐานข้อมูลเวกเตอร์เท่านั้น (ปัจจุบันรองรับเฉพาะ PGVector ครับ/ค่ะ)

![20251023101743](https://static-docs.nocobase.com/20251023101743.png)

### ฐานข้อมูลความรู้แบบ External

ฐานข้อมูลความรู้แบบ External คือฐานข้อมูลความรู้ภายนอกที่เอกสารและข้อมูลเวกเตอร์ถูกดูแลจากภายนอกครับ/ค่ะ การดึงข้อมูลจากฐานข้อมูลเวกเตอร์ประเภทนี้จำเป็นต้องให้ผู้พัฒนาขยายความสามารถเพิ่มเติม ซึ่งจะช่วยให้สามารถใช้งานฐานข้อมูลเวกเตอร์ที่ NocoBase ยังไม่รองรับได้ครับ/ค่ะ

![20251023101949](https://static-docs.nocobase.com/20251023101949.png)
