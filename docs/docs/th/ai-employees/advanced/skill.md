# ขั้นสูง

## บทนำ

โมเดลภาษาขนาดใหญ่ (LLM) ทั่วไปมีความสามารถในการใช้เครื่องมือต่าง ๆ ครับ/ค่ะ โดยปลั๊กอิน AI employee ได้รวมเครื่องมือที่ใช้บ่อยบางส่วนไว้ให้โมเดลภาษาขนาดใหญ่ใช้งานได้เลย

ทักษะ (Skills) ที่ตั้งค่าไว้ในหน้าการตั้งค่า AI employee ก็คือเครื่องมือที่โมเดลภาษาขนาดใหญ่จะนำไปใช้นั่นเองครับ/ค่ะ

![20251022142348](https://static-docs.nocobase.com/20251022142348.png)

## การตั้งค่าทักษะ

ไปที่หน้าการตั้งค่าปลั๊กอิน AI employee ครับ/ค่ะ จากนั้นคลิกแท็บ `AI employees` เพื่อเข้าสู่หน้าจัดการ AI employee

เลือก AI employee ที่คุณต้องการตั้งค่าทักษะ แล้วคลิกปุ่ม `Edit` เพื่อเข้าสู่หน้าแก้ไข AI employee ครับ/ค่ะ

ในแท็บ `Skills` ให้คลิกปุ่ม `Add Skill` เพื่อเพิ่มทักษะให้กับ AI employee ที่เลือกอยู่ครับ/ค่ะ

![20251022145748](https://static-docs.nocobase.com/20251022145748.png)

## แนะนำทักษะ

### Frontend

กลุ่มทักษะ Frontend ช่วยให้ AI employee สามารถโต้ตอบกับส่วนประกอบ (component) ของส่วนหน้า (frontend) ได้ครับ/ค่ะ

- ทักษะ `Form filler` ช่วยให้ AI employee สามารถกรอกข้อมูลที่สร้างขึ้นกลับเข้าไปในฟอร์มที่ผู้ใช้ระบุได้ครับ/ค่ะ

![20251022145954](https://static-docs.nocobase.com/20251022145954.png)

### Data modeling

กลุ่มทักษะ Data modeling ช่วยให้ AI employee มีความสามารถในการเรียกใช้ API ภายในของ NocoBase เพื่อทำการสร้างแบบจำลองข้อมูล (data modeling) ครับ/ค่ะ

- `Intent Router` (เราเตอร์ความตั้งใจ) จะช่วยกำหนดว่าผู้ใช้ต้องการแก้ไขโครงสร้างของคอลเลกชัน หรือสร้างโครงสร้างคอลเลกชันใหม่ครับ/ค่ะ
- `Get collection names` ใช้สำหรับดึงชื่อคอลเลกชันทั้งหมดที่มีอยู่ในระบบครับ/ค่ะ
- `Get collection metadata` ใช้สำหรับดึงข้อมูลโครงสร้างของคอลเลกชันที่ระบุครับ/ค่ะ
- `Define collections` ช่วยให้ AI employee สามารถสร้างคอลเลกชันในระบบได้ครับ/ค่ะ

![20251022150441](https://static-docs.nocobase.com/20251022150441.png)

### Workflow caller

`Workflow caller` ช่วยให้ AI employee มีความสามารถในการเรียกใช้ เวิร์กโฟลว์ ครับ/ค่ะ โดย เวิร์กโฟลว์ ที่ตั้งค่า `Trigger type` เป็น `AI employee event` ในปลั๊กอิน เวิร์กโฟลว์ จะถูกนำมาใช้เป็นทักษะสำหรับ AI employee ที่นี่ครับ/ค่ะ

![20251022153320](https://static-docs.nocobase.com/20251022153320.png)

### Code Editor

ทักษะภายใต้กลุ่ม Code Editor ส่วนใหญ่จะช่วยให้ AI employee สามารถโต้ตอบกับตัวแก้ไขโค้ด (code editor) ได้ครับ/ค่ะ

- `Get code snippet list` ใช้สำหรับดึงรายการโค้ดสนิปเปต (code snippet) ที่ตั้งค่าไว้ล่วงหน้าครับ/ค่ะ
- `Get code snippet content` ใช้สำหรับดึงเนื้อหาของโค้ดสนิปเปตที่ระบุครับ/ค่ะ

![20251022153811](https://static-docs.nocobase.com/20251022153811.png)

### Others

- `Chart generator` ช่วยให้ AI employee มีความสามารถในการสร้างแผนภูมิ และแสดงผลแผนภูมิเหล่านั้นได้โดยตรงในการสนทนาครับ/ค่ะ

![20251022154141](https://static-docs.nocobase.com/20251022154141.png)
