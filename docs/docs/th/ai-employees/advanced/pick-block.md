# ขั้นสูง

## บทนำ

ในการสนทนากับ AI Agent นอกจากจะส่งไฟล์ได้แล้ว คุณยังสามารถส่งข้อมูลบริบทของแอปพลิเคชัน (application context information) ให้กับ AI Agent ได้ด้วยครับ/ค่ะ เพื่อให้ AI Agent ตอบกลับโดยอิงจากข้อมูลบริบทเหล่านั้น

### การส่งข้อมูลบล็อก

ในกล่องโต้ตอบของ AI Agent ให้คลิกปุ่ม `Add work context` ที่มุมล่างซ้าย แล้วเลือก `Pick block` ครับ/ค่ะ จากนั้นหน้าแอปพลิเคชันจะเข้าสู่โหมดการเลือกบล็อก

![20251022010148](https://static-docs.nocobase.com/20251022010148.png)

เมื่อคุณเลื่อนเมาส์ไปวางเหนือบล็อก สีของบล็อกที่สามารถส่งเป็นบริบทได้จะเปลี่ยนไปครับ/ค่ะ

![20251022010422](https://static-docs.nocobase.com/20251022010422.png)

หลังจากที่คุณเลือกบล็อกที่ต้องการส่งแล้ว ข้อมูลบริบทของบล็อกจะถูกแนบไปในกล่องโต้ตอบครับ/ค่ะ

![20251022010543](https://static-docs.nocobase.com/20251022010543.png)

หลังจากที่คุณพิมพ์คำถามแล้ว ให้คลิกปุ่มส่งเหมือนกับการส่งข้อความปกติครับ/ค่ะ แล้ว AI Agent จะตอบกลับโดยอิงจากข้อมูลบล็อกนั้นๆ

ข้อมูลบล็อกก็จะปรากฏขึ้นในรายการข้อความด้วยครับ/ค่ะ

![20251022010758](https://static-docs.nocobase.com/20251022010758.png)

### การสิ้นสุดการเลือกบล็อก

เมื่อหน้าแอปพลิเคชันอยู่ในโหมดการเลือกบล็อก แต่คุณไม่ต้องการเลือกบล็อกใดๆ คุณสามารถคลิกไอคอน "หยุด" (Stop) ที่ด้านล่าง เพื่อออกจากโหมดการเลือกบล็อกได้ครับ/ค่ะ

![20251022011043](https://static-docs.nocobase.com/20251022011043.png)
