# เริ่มต้นใช้งานอย่างรวดเร็ว

## บทนำ

ก่อนใช้งาน AI Employee (พนักงาน AI) คุณจำเป็นต้องเชื่อมต่อกับบริการ LLM ออนไลน์ก่อนครับ/ค่ะ NocoBase รองรับบริการ LLM ออนไลน์หลักๆ ที่ได้รับความนิยมในปัจจุบัน เช่น OpenAI, Gemini, Claude, DepSeek, Qwen และอื่นๆ ครับ/ค่ะ
นอกจากบริการ LLM ออนไลน์แล้ว NocoBase ยังรองรับการเชื่อมต่อกับโมเดล LLM แบบ Local (ภายในเครื่อง) ผ่าน Ollama ด้วยครับ/ค่ะ

## การตั้งค่าบริการ LLM

ไปที่หน้าการตั้งค่าปลั๊กอิน AI Employee (พนักงาน AI) จากนั้นคลิกที่แท็บ `LLM service` เพื่อเข้าสู่หน้าจัดการบริการ LLM ครับ/ค่ะ

![20251021213122](https://static-docs.nocobase.com/20251021213122.png)

เลื่อนเมาส์ไปที่ปุ่ม `Add New` ที่มุมขวาบนของรายการบริการ LLM แล้วเลือกบริการ LLM ที่คุณต้องการใช้งานครับ/ค่ะ

![20251021213358](https://static-docs.nocobase.com/20251021213358.png)

ในตัวอย่างนี้ เราจะใช้ OpenAI นะครับ/คะ ในหน้าต่างป๊อปอัป ให้คุณกรอก `title` ที่จำง่าย จากนั้นกรอก `API key` ที่ได้รับจาก OpenAI แล้วคลิก `Submit` เพื่อบันทึก เพียงเท่านี้ก็เรียบร้อยสำหรับการตั้งค่าบริการ LLM ครับ/ค่ะ

โดยปกติแล้ว `Base URL` สามารถเว้นว่างไว้ได้เลยครับ/ค่ะ แต่หากคุณกำลังใช้บริการ LLM ของบุคคลที่สามที่รองรับ API ของ OpenAI โปรดกรอก Base URL ที่เกี่ยวข้องลงไปครับ/ค่ะ

![20251021214549](https://static-docs.nocobase.com/20251021214549.png)

## การทดสอบความพร้อมใช้งาน

ที่หน้าการตั้งค่าบริการ LLM ให้คลิกปุ่ม `Test flight` จากนั้นกรอกชื่อโมเดลที่คุณต้องการใช้งาน แล้วคลิกปุ่ม `Run` เพื่อทดสอบว่าบริการ LLM และโมเดลนั้นพร้อมใช้งานหรือไม่ครับ/ค่ะ
