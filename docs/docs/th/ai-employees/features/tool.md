# การใช้งานทักษะ

ทักษะ (Skills หรือ Tools) คือสิ่งที่กำหนดว่าพนักงาน AI "สามารถทำอะไรได้บ้าง" ครับ

## โครงสร้างของทักษะ

หน้าทักษะจะแบ่งออกเป็น 3 ประเภทหลัก ดังนี้ครับ:

1. `General skills`: ทักษะส่วนกลางที่พนักงาน AI ทุกคนใช้งานร่วมกัน และเป็นแบบอ่านอย่างเดียว (Read-only)
2. `Employee-specific skills`: ทักษะเฉพาะสำหรับพนักงานคนปัจจุบัน โดยปกติจะเป็นแบบอ่านอย่างเดียว
3. `Custom skills`: ทักษะที่กำหนดเอง ซึ่งสามารถเพิ่ม ลบ และกำหนดสิทธิ์เริ่มต้นได้

![skills-three-sections-general-specific-custom.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/skills-three-sections-general-specific-custom.png)

## สิทธิ์การใช้งานทักษะ

สิทธิ์การใช้งานทักษะถูกกำหนดไว้เป็น 2 รูปแบบ คือ:

- `Ask`: สอบถามเพื่อยืนยันก่อนเรียกใช้งาน
- `Allow`: อนุญาตให้เรียกใช้งานได้ทันที

ข้อแนะนำ: สำหรับทักษะที่เกี่ยวข้องกับการแก้ไขข้อมูล ควรตั้งค่าเริ่มต้นเป็น `Ask` ครับ

![skills-permission-ask-allow-segmented.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/skills-permission-ask-allow-segmented.png)

## การเพิ่มและดูแลรักษา

คลิกที่ `Add skill` ในส่วนของ `Custom skills` เพื่อเพิ่มทักษะ และกำหนดสิทธิ์การใช้งานตามระดับความเสี่ยงของธุรกิจครับ
