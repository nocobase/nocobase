# ทำงานร่วมกับพนักงาน AI

หลังจากสร้างและเปิดใช้งานพนักงาน AI แล้ว คุณสามารถทำงานร่วมกับพวกเขาได้ในหน้าเพจต่าง ๆ ครับ

## ช่องทางการเข้าถึง

ช่องทางการเข้าถึงทั่วไปมี 2 ประเภท:

1. **ทางเข้าหลักที่มุมขวาล่าง**: เรียกเปิดแผงแชท AI จากมุมขวาล่างของหน้าเพจธุรกิจ เหมาะสำหรับการถามตอบทั่วไปและการทำงานร่วมกันข้ามบล็อกครับ
2. **ทางเข้าจาก Action ของบล็อก**: ในบล็อกที่รองรับ `Actions` สามารถเข้าถึงได้ผ่าน `Actions -> AI employees` เหมาะสำหรับการปฏิบัติงานในบล็อกปัจจุบัน (เช่น ในกรณีของ JSBlock เป็นต้น) ครับ

## การใช้งานแชทเบื้องต้น

แผงแชทรองรับการดำเนินการทั่วไป เช่น การส่งข้อความ, การอัปโหลดไฟล์แนบ, การดูประวัติการสนทนา, การสร้างการสนทนาใหม่ และการแก้ไข System Prompt ครับ

## การสลับภายในแชท

คุณสามารถสลับการตั้งค่าได้โดยตรงที่ส่วนท้ายของพื้นที่ป้อนข้อมูล:

- `AI Employee Switcher`: สลับพนักงานปัจจุบัน
- `Model Switcher`: สลับโมเดลของการสนทนาปัจจุบัน

ระบบจะบันทึกการตั้งค่าโมเดลตามพนักงานแต่ละคน และจะเรียกคืนค่าที่เลือกไว้เมื่อเข้าใช้งานในครั้งถัดไปครับ
