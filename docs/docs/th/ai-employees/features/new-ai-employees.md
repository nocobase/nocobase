# สร้างพนักงาน AI ใหม่

หากพนักงาน AI (AI Employee) ที่มีมาให้ในระบบยังไม่ตอบโจทย์ความต้องการของคุณ คุณสามารถสร้างและปรับแต่งพนักงาน AI ของคุณเองได้ครับ

## เริ่มต้นสร้าง

ไปที่หน้าการจัดการ `AI employees` แล้วคลิก `New AI employee`

## การตั้งค่าข้อมูลพื้นฐาน

ตั้งค่าในแท็บ `Profile` ดังนี้:

- `Username`: ตัวระบุที่ไม่ซ้ำกัน (Unique identifier)
- `Nickname`: ชื่อที่แสดง
- `Position`: คำอธิบายตำแหน่งงาน
- `Avatar`: รูปโปรไฟล์ของพนักงาน
- `Bio`: ข้อมูลแนะนำตัวสั้นๆ
- `About me`: System Prompt (คำสั่งระบบ)
- `Greeting message`: ข้อความทักทายในการสนทนา

![ai-employee-create-without-model-settings-tab.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employee-create-without-model-settings-tab.png)

## การตั้งค่าบทบาท (Role setting)

ในแท็บ `Role setting` ให้กำหนดค่า System Prompt ของพนักงาน ซึ่งเนื้อหาส่วนนี้จะเป็นการกำหนดตัวตน เป้าหมาย ขอบเขตการทำงาน และสไตล์การตอบกลับของพนักงานในการสนทนาครับ

ข้อแนะนำที่ควรระบุอย่างน้อย:

- การกำหนดบทบาทและขอบเขตความรับผิดชอบ
- หลักการจัดการงานและโครงสร้างการตอบกลับ
- ข้อห้าม ขอบเขตของข้อมูล และน้ำเสียงหรือสไตล์ที่ต้องการ

คุณสามารถใส่ตัวแปรได้ตามต้องการ (เช่น ผู้ใช้ปัจจุบัน, บทบาทปัจจุบัน, ภาษาปัจจุบัน และเวลา) เพื่อให้คำสั่ง (Prompt) ปรับเปลี่ยนตามบริบทของการสนทนาโดยอัตโนมัติครับ

![ai-employee-role-setting-system-prompt.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employee-role-setting-system-prompt.png)

## การตั้งค่าทักษะและความรู้

กำหนดสิทธิ์การใช้งานเครื่องมือในแท็บ `Skills` หากมีการเปิดใช้งานความสามารถของคลังความรู้ (Knowledge Base) คุณสามารถตั้งค่าเพิ่มเติมได้ในแท็บที่เกี่ยวข้องกับคลังความรู้ครับ

## เสร็จสิ้นการสร้าง

คลิก `Submit` เพื่อเสร็จสิ้นการสร้างครับ
