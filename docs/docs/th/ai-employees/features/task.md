# งานด่วน (Shortcut Tasks)

เพื่อให้พนักงาน AI เริ่มต้นทำงานได้อย่างมีประสิทธิภาพมากขึ้น เราสามารถผูกพนักงาน AI เข้ากับบล็อกในหน้าจอ และตั้งค่าล่วงหน้าสำหรับงานที่ใช้บ่อยได้ครับ

วิธีนี้ช่วยให้ผู้ใช้สามารถเริ่มประมวลผลงานได้ด้วยการคลิกเพียงครั้งเดียว โดยไม่จำเป็นต้อง **เลือกบล็อก** และ **ป้อนคำสั่ง** ใหม่ทุกครั้งครับ

## การผูกพนักงาน AI เข้ากับบล็อก

หลังจากเข้าสู่โหมดการแก้ไข UI สำหรับบล็อกที่รองรับ `Actions` ให้เลือกเมนู `AI employees` ภายใต้ `Actions` จากนั้นเลือกพนักงาน AI ที่ต้องการ พนักงาน AI คนนั้นจะถูกผูกเข้ากับบล็อกปัจจุบันครับ

![20251022135306](https://static-docs.nocobase.com/20251022135306.png)

เมื่อการผูกเสร็จสิ้น ทุกครั้งที่เข้าสู่หน้าเพจ พื้นที่ Actions ของบล็อกจะแสดงพนักงาน AI ที่ผูกไว้กับบล็อกนั้นครับ

![20251022135438](https://static-docs.nocobase.com/20251022135438.png)

## การตั้งค่างาน

หลังจากเข้าสู่โหมดการแก้ไข UI ให้วางเมาส์เหนือไอคอนพนักงาน AI ที่ผูกไว้กับบล็อก จะมีปุ่มเมนูปรากฏขึ้น ให้เลือก `Edit tasks` เพื่อเข้าสู่หน้าการตั้งค่างานครับ

ในหน้าการตั้งค่างาน คุณสามารถเพิ่มงานได้หลายรายการสำหรับพนักงาน AI คนปัจจุบัน

แต่ละแท็บจะแทนงานที่เป็นอิสระต่อกัน คุณสามารถคลิกเครื่องหมาย "+" ข้างๆ เพื่อเพิ่มงานใหม่ได้ครับ

![clipboard-image-1771913187](https://static-docs.nocobase.com/clipboard-image-1771913187.png)

แบบฟอร์มการตั้งค่างาน:

- ใส่ชื่อหัวข้องานในช่อง `Title` เพื่ออธิบายเนื้อหางานสั้นๆ โดยชื่อนี้จะปรากฏในรายการงานของพนักงาน AI ครับ
- ใส่เนื้อหาหลักของงานในช่อง `Background` ซึ่งเนื้อหานี้จะถูกใช้เป็น System Prompt เมื่อสนทนากับพนักงาน AI ครับ
- ใส่ข้อความเริ่มต้นของผู้ใช้ในช่อง `Default user message` ซึ่งจะถูกเติมลงในช่องป้อนข้อมูลโดยอัตโนมัติหลังจากเลือกงานครับ
- ในส่วนของ `Work context` ให้เลือกข้อมูลบริบทของแอปพลิเคชันที่จะส่งให้พนักงาน AI เป็นค่าเริ่มต้น ซึ่งส่วนนี้จะทำงานเหมือนกับการตั้งค่าในหน้าต่างแชทครับ
- ในส่วนของ `Skills` จะแสดงทักษะที่พนักงาน AI มีอยู่ คุณสามารถยกเลิกบางทักษะเพื่อให้พนักงาน AI ไม่ต้องใช้ทักษะนั้นเมื่อปฏิบัติงานนี้ได้ครับ
- ช่องติ๊ก `Send default user message automatically` ใช้สำหรับกำหนดว่าจะให้ส่งข้อความเริ่มต้นของผู้ใช้โดยอัตโนมัติทันทีหลังจากคลิกเลือกงานหรือไม่ครับ


## รายการงาน

หลังจากตั้งค่างานให้พนักงาน AI เรียบร้อยแล้ว งานเหล่านี้จะแสดงในหน้าต่างโปรไฟล์ของพนักงาน AI และในข้อความทักทายก่อนเริ่มการสนทนา คุณสามารถคลิกเพื่อเริ่มทำงานได้ทันทีครับ
