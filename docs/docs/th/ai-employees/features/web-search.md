# การค้นหาออนไลน์

การค้นหาออนไลน์ใช้สำหรับเพิ่มเติมข้อมูลล่าสุดที่อยู่นอกเหนือจากข้อมูลที่ใช้ในการฝึกฝนโมเดลครับ

## วิธีการทำงาน

ความสามารถในการใช้การค้นหาออนไลน์จะขึ้นอยู่กับว่าบริการของโมเดลที่เลือกในเซสชันปัจจุบันนั้นรองรับ Web Search หรือไม่ครับ

- รองรับ: จะแสดงสวิตช์เปิด/ปิดการค้นหาออนไลน์ ซึ่งสามารถเปิดหรือปิดได้ตามต้องการครับ
- ไม่รองรับ: จะไม่แสดงสวิตช์ดังกล่าว และสถานะการค้นหาจะถูกปิดโดยอัตโนมัติครับ

## การใช้งานในการสนทนา

ใช้สวิตช์เปิด/ปิดการค้นหาออนไลน์ในพื้นที่ป้อนข้อมูลการสนทนา:

- เมื่อเปิด: AI จะสามารถค้นหาข้อมูลก่อนแล้วจึงตอบคำถามครับ

![web-search-switch-visible-on-supported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-visible-on-supported-model.png)

- เมื่อปิด: AI จะตอบคำถามโดยอิงจากบริบทที่มีอยู่เท่านั้นครับ

![web-search-switch-hidden-on-unsupported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-hidden-on-unsupported-model.png)

## ความแตกต่างระหว่างแพลตฟอร์ม

ความสามารถในการรองรับ Web Search ของแต่ละโมเดลนั้นแตกต่างกัน โปรดตรวจสอบและใช้งานตามสถานการณ์จริงของบริการโมเดลที่คุณเลือกครับ
