# เริ่มต้นใช้งานอย่างรวดเร็ว

มาตั้งค่า AI Employee ขั้นพื้นฐานให้พร้อมใช้งานภายใน 5 นาทีกันครับ

## การติดตั้งปลั๊กอิน

AI Employee เป็นปลั๊กอินที่ติดตั้งมาพร้อมกับ NocoBase อยู่แล้ว (`@nocobase/plugin-ai`) จึงไม่จำเป็นต้องติดตั้งแยกต่างหากครับ

## การกำหนดค่าโมเดล

คุณสามารถกำหนดค่าบริการ LLM ได้จากช่องทางใดช่องทางหนึ่งดังนี้ครับ:

1. ทางเข้าจากหลังบ้าน: `การตั้งค่าระบบ (System Settings) -> AI Employee -> บริการ LLM (LLM service)`
2. ทางลัดจากหน้าบ้าน: ในแผงแชท AI เมื่อใช้ `Model Switcher` เพื่อเลือกโมเดล คุณสามารถคลิกที่ทางลัด "เพิ่มบริการ LLM" เพื่อไปยังหน้าตั้งค่าได้ทันทีครับ

![quick-start-model-switcher-add-llm-service.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/quick-start-model-switcher-add-llm-service.png)

โดยปกติคุณต้องตรวจสอบสิ่งต่อไปนี้ครับ:
1. เลือกผู้ให้บริการ (Provider)
2. กรอก API Key
3. กำหนดค่า `Enabled Models` โดยเริ่มต้นสามารถเลือกใช้ค่าแนะนำ (Recommend) ได้เลยครับ

## การเปิดใช้งานพนักงานที่ติดตั้งมาในตัว

AI Employee ที่ติดตั้งมาให้ในตัวจะถูกเปิดใช้งานไว้ทั้งหมดโดยค่าเริ่มต้น ปกติแล้วจึงไม่จำเป็นต้องเปิดใช้งานทีละรายการด้วยตนเองครับ

หากคุณต้องการปรับเปลี่ยนการใช้งาน (เปิด/ปิด พนักงานบางคน) สามารถแก้ไขได้ที่สวิตช์ `Enabled` ในหน้ารายการ `การตั้งค่าระบบ -> AI Employee` ครับ

![ai-employee-list-enable-switch.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employee-list-enable-switch.png)

## เริ่มต้นทำงานร่วมกัน

ในหน้าแอปพลิเคชัน ให้วางเมาส์เหนือทางลัดที่มุมขวาล่าง แล้วเลือก AI Employee ครับ
![ai-employees-entry-bottom-right.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employees-entry-bottom-right.png)

คลิกเพื่อเปิดหน้าต่างแชท AI:

![chat-footer-employee-switcher-and-model-switcher.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/chat-footer-employee-switcher-and-model-switcher.png)

นอกจากนี้คุณยังสามารถ:  
* เพิ่มบล็อก
* เพิ่มไฟล์แนบ
* เปิดใช้งานการค้นหาเว็บ
* เปลี่ยน AI Employee
* เลือกโมเดล

นอกจากนี้ AI Employee ยังสามารถดึงโครงสร้างหน้าเว็บมาเป็นบริบท (Context) ได้โดยอัตโนมัติ เช่น Dex ในบล็อกแบบฟอร์มจะสามารถอ่านโครงสร้างฟิลด์ของแบบฟอร์ม และเรียกใช้ทักษะที่เหมาะสมเพื่อดำเนินการบนหน้าเว็บได้ครับ

## งานทางลัด (Shortcut Tasks)

คุณสามารถตั้งค่างานที่ใช้บ่อยล่วงหน้าสำหรับ AI Employee แต่ละคนในตำแหน่งปัจจุบันได้ เพื่อให้สามารถเริ่มงานได้ด้วยการคลิกเพียงครั้งเดียว ซึ่งทั้งรวดเร็วและสะดวกสบายครับ

## รายชื่อพนักงานที่ติดตั้งมาในตัว

NocoBase ได้จัดเตรียม AI Employee ไว้หลายรูปแบบตามสถานการณ์การใช้งานที่แตกต่างกันครับ

คุณเพียงแค่ต้อง:

1. ตั้งค่าบริการ LLM
2. ปรับสถานะการเปิดใช้งานพนักงานตามต้องการ (เปิดใช้งานไว้แล้วโดยค่าเริ่มต้น)
3. เลือกโมเดลในหน้าแชทและเริ่มทำงานร่วมกันได้เลยครับ

| ชื่อพนักงาน | บทบาทหน้าที่ | ความสามารถหลัก |
| :--- | :--- | :--- |
| **Cole** | ผู้ช่วย NocoBase | ตอบคำถามการใช้งานผลิตภัณฑ์, ค้นหาข้อมูลจากเอกสาร |
| **Ellis** | ผู้เชี่ยวชาญด้านอีเมล | เขียนอีเมล, สรุปเนื้อหา, แนะนำการตอบกลับ |
| **Dex** | ผู้เชี่ยวชาญด้านการจัดการข้อมูล | แปลฟิลด์ข้อมูล, จัดรูปแบบ, สกัดข้อมูล |
| **Viz** | นักวิเคราะห์ข้อมูลเชิงลึก | วิเคราะห์ข้อมูลเชิงลึก, วิเคราะห์แนวโน้ม, ตีความตัวชี้วัดหลัก |
| **Lexi** | ผู้ช่วยแปลภาษา | แปลภาษาต่างๆ, ช่วยเหลือด้านการสื่อสาร |
| **Vera** | นักวิเคราะห์งานวิจัย | ค้นหาข้อมูลบนเว็บ, รวบรวมข้อมูล, วิจัยเชิงลึก |
| **Dara** | ผู้เชี่ยวชาญด้านการแสดงข้อมูลด้วยภาพ | กำหนดค่าแผนภูมิ, สร้างรายงานในรูปแบบภาพ |
| **Orin** | ผู้เชี่ยวชาญด้านการสร้างแบบจำลองข้อมูล | ช่วยออกแบบโครงสร้างคอลเลกชัน, แนะนำฟิลด์ข้อมูล |
| **Nathan** | วิศวกรฟรอนต์เอนด์ | ช่วยเขียนโค้ดฟรอนต์เอนด์, ปรับแต่งสไตล์ |

**หมายเหตุ**

AI Employee บางส่วนที่ติดตั้งมาในตัวจะไม่ปรากฏในรายการที่มุมขวาล่าง เนื่องจากมีสถานการณ์การทำงานเฉพาะทางครับ:

- Orin: หน้าการสร้างแบบจำลองข้อมูล
- Dara: บล็อกการกำหนดค่าแผนภูมิ
- Nathan: JS Block และเครื่องมือแก้ไขโค้ดอื่นๆ
