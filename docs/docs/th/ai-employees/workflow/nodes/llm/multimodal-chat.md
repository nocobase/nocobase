# การสนทนาแบบหลายโมดอล

## รูปภาพ

หากโมเดลรองรับ โหนด LLM สามารถส่งรูปภาพไปยังโมเดลได้ครับ/ค่ะ ในการใช้งาน คุณต้องเลือกฟิลด์ไฟล์แนบ หรือเรคคอร์ดของคอลเลกชันไฟล์ที่เกี่ยวข้องผ่านตัวแปรครับ/ค่ะ เมื่อเลือกเรคคอร์ดของคอลเลกชันไฟล์ คุณสามารถเลือกได้ในระดับออบเจกต์ หรือเลือกฟิลด์ URL ได้ครับ/ค่ะ

![](https://static-docs.nocobase.com/202503041034858.png)

มีสองตัวเลือกสำหรับรูปแบบการส่งรูปภาพดังนี้ครับ/ค่ะ:

- ส่งผ่าน URL - รูปภาพทั้งหมด ยกเว้นรูปภาพที่จัดเก็บในเครื่อง จะถูกส่งเป็น URL ส่วนรูปภาพที่จัดเก็บในเครื่องจะถูกแปลงเป็นรูปแบบ base64 ก่อนส่งครับ/ค่ะ
- ส่งผ่าน base64 - รูปภาพทั้งหมด ไม่ว่าจะจัดเก็บในเครื่องหรือบนคลาวด์ จะถูกส่งในรูปแบบ base64 วิธีนี้เหมาะสำหรับกรณีที่ URL ของรูปภาพไม่สามารถเข้าถึงได้โดยตรงจากบริการ LLM ออนไลน์ครับ/ค่ะ
