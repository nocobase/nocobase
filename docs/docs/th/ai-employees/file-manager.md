# การจัดการไฟล์

## บทนำ

กำหนดค่าวิธีการจัดเก็บไฟล์ที่อัปโหลดในระหว่างการสนทนาของ AI Employee ครับ/ค่ะ

## การตั้งค่า

ไปที่หน้าการตั้งค่าปลั๊กอิน AI Employee จากนั้นคลิกแท็บ `Settings` เพื่อเข้าสู่หน้าการตั้งค่าวิธีการจัดเก็บครับ/ค่ะ

![20251022002648](https://static-docs.nocobase.com/20251022002648.png)

ตัวเลือกการจัดเก็บสำหรับ `Files storage` สามารถกำหนดค่าได้ในปลั๊กอิน File Management ครับ/ค่ะ

![20251022002853](https://static-docs.nocobase.com/20251022002853.png)
