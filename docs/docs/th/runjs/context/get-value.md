# ctx.getValue()

ในสถานการณ์ของฟิลด์ที่แก้ไขได้ (editable fields) เช่น JSField และ JSItem ใช้สำหรับรับค่าล่าสุดของฟิลด์ปัจจุบัน เมื่อใช้งานร่วมกับ `ctx.setValue(v)` จะช่วยให้สามารถทำ Two-way binding กับฟอร์มได้ครับ

## สถานการณ์ที่ใช้งาน

| สถานการณ์ | คำอธิบาย |
|------|------|
| **JSField** | อ่านค่าที่ผู้ใช้กรอกหรือค่าปัจจุบันของฟอร์มในฟิลด์ที่กำหนดเองแบบแก้ไขได้ |
| **JSItem** | อ่านค่าของเซลล์ปัจจุบันในรายการที่แก้ไขได้ของตารางหรือตารางย่อย |
| **JSColumn** | อ่านค่าฟิลด์ของแถวที่เกี่ยวข้องในขณะที่เรนเดอร์คอลัมน์ของตาราง |

> **หมายเหตุ**: `ctx.getValue()` จะใช้งานได้เฉพาะในบริบทของ RunJS ที่มีการเชื่อมโยงกับฟอร์มเท่านั้น โดยจะไม่มีเมธอดนี้ในสถานการณ์ที่ไม่มีการเชื่อมโยงฟิลด์ เช่น เวิร์กโฟลว์ (Workflow) หรือกฎการเชื่อมโยง (Linkage rules) ครับ

## การกำหนดประเภท (Type Definition)

```ts
getValue<T = any>(): T | undefined;
```

- **ค่าที่ส่งกลับ**: ค่าของฟิลด์ปัจจุบัน ซึ่งประเภทของข้อมูลจะขึ้นอยู่กับประเภทของรายการฟอร์มนั้น ๆ หากฟิลด์ยังไม่ได้ลงทะเบียนหรือยังไม่ได้กรอกข้อมูล อาจส่งกลับเป็น `undefined` ครับ

## ลำดับการดึงค่า

`ctx.getValue()` จะดึงค่าตามลำดับดังนี้:

1. **สถานะฟอร์ม (Form State)**: ลำดับความสำคัญแรกคืออ่านจากสถานะปัจจุบันของ Ant Design Form
2. **ค่าสำรอง (Fallback Value)**: หากไม่มีฟิลด์นั้นในฟอร์ม จะกลับไปใช้ค่าเริ่มต้นของฟิลด์หรือ props

> หากฟอร์มยังเรนเดอร์ไม่เสร็จสิ้นหรือฟิลด์ยังไม่ได้ลงทะเบียน อาจส่งกลับเป็น `undefined` ครับ

## ตัวอย่าง

### เรนเดอร์ตามค่าปัจจุบัน

```ts
const current = ctx.getValue();
if (current == null || current === '') {
  ctx.render(<span>กรุณากรอกข้อมูลก่อน</span>);
} else {
  ctx.render(<span>ค่าปัจจุบัน: {current}</span>);
}
```

### ใช้งานร่วมกับ setValue เพื่อทำ Two-way binding

```tsx
const { Input } = ctx.libs.antd;

// อ่านค่าปัจจุบันเพื่อใช้เป็นค่าเริ่มต้น
const defaultValue = ctx.getValue() ?? '';

ctx.render(
  <Input
    defaultValue={defaultValue}
    onChange={(e) => ctx.setValue(e.target.value)}
  />
);
```

## สิ่งที่เกี่ยวข้อง

- [ctx.setValue()](./set-value.md) - ตั้งค่าฟิลด์ปัจจุบัน ใช้งานร่วมกับ `getValue` เพื่อทำ Two-way binding
- [ctx.form](./form.md) - อินสแตนซ์ของ Ant Design Form สำหรับอ่านและเขียนฟิลด์อื่น ๆ
