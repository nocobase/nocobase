# ctx.i18n

อินสแตนซ์ i18n ของบริบท (context) ปัจจุบัน ใช้สำหรับอ่านหรือเปลี่ยนภาษา **สำหรับการแปลข้อความให้ใช้ [ctx.t()](./t.md) เป็นมาตรฐานเดียวกัน** ห้ามใช้ `ctx.i18n.t` ครับ

## สภาพแวดล้อมที่ใช้งานได้

สภาพแวดล้อมการทำงานของ RunJS ทั้งหมดสามารถใช้ `ctx.i18n` ได้ (เช่น JSBlock, JSField, JSItem, JSColumn, เวิร์กโฟลว์, กฎการเชื่อมโยง เป็นต้น) ครับ

## การกำหนดประเภท (Type Definition)

```typescript
interface i18n: {
  language: string;
  changeLanguage(lng: string): Promise<any>;
}
```

## พร็อพเพอร์ตี้ที่ใช้บ่อย

| พร็อพเพอร์ตี้ | ประเภท | คำอธิบาย |
|------|------|------|
| `language` | `string` | รหัสภาษาที่ใช้งานอยู่ในปัจจุบัน (เช่น `zh-CN`, `en-US`) |

## เมธอดที่ใช้บ่อย

### changeLanguage(lng)

เปลี่ยนภาษาปัจจุบัน

| พารามิเตอร์ | ประเภท | คำอธิบาย |
|------|------|------|
| `lng` | `string` | รหัสภาษาเป้าหมาย (เช่น `'en-US'`, `'zh-CN'`) |

**ค่าที่ส่งกลับ**: `Promise<any>` ซึ่งจะ resolve หลังจากเปลี่ยนภาษาเสร็จสิ้นครับ

## ตัวอย่าง

### การอ่านภาษาปัจจุบัน

```ts
const lang = ctx.i18n.language;
// 'zh-CN' | 'en-US' | ...
if (lang.startsWith('zh')) {
  ctx.render(ctx.t('หน้าจอภาษาจีน'));
} else {
  ctx.render(ctx.t('English UI'));
}
```

### การเปลี่ยนภาษา

```ts
// เปลี่ยนเป็นภาษาอังกฤษ
await ctx.i18n.changeLanguage('en-US');

// เปลี่ยนเป็นภาษาจีน
await ctx.i18n.changeLanguage('zh-CN');
```

### ปุ่มเปลี่ยนภาษา

```ts
const { Button } = ctx.libs.antd;
const isZh = ctx.i18n.language.startsWith('zh');
ctx.render(
  <Button onClick={async () => {
    await ctx.i18n.changeLanguage(isZh ? 'en-US' : 'zh-CN');
  }}>
    {ctx.t(isZh ? 'Switch to English' : '切换到中文')}
  </Button>,
);
```

## ข้อควรระวัง

- **การแปลข้อความ**: ให้ใช้ `ctx.t()` เป็นมาตรฐานเดียวกัน ห้ามใช้ `ctx.i18n.t` ครับ

## หัวข้อที่เกี่ยวข้อง

- [ctx.t()](./t.md): การแปลข้อความ ให้ใช้วิธีนี้เป็นมาตรฐานเดียวกันครับ
