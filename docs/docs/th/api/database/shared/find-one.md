## ประเภท

```typescript
type FindOneOptions = Omit<FindOptions, 'limit'>;
```

## พารามิเตอร์

พารามิเตอร์ส่วนใหญ่เหมือนกับ `find()` ครับ/ค่ะ แต่ต่างกันตรงที่ `findOne()` จะคืนค่าข้อมูลเพียงรายการเดียว จึงไม่จำเป็นต้องใช้พารามิเตอร์ `limit` และในการคิวรีจะถูกกำหนดให้เป็น `1` เสมอ
