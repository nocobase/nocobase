## การเริ่มต้นใช้งาน NocoBase

NocoBase เป็นแพลตฟอร์มที่ยืดหยุ่นและทรงพลังสำหรับการจัดการข้อมูลของคุณ หัวใจสำคัญของ NocoBase คือแนวคิดเรื่อง **คอลเลกชัน** ครับ

### คอลเลกชัน (Collection)

**คอลเลกชัน** เป็นองค์ประกอบพื้นฐานที่กำหนดโครงสร้างข้อมูลของคุณใน NocoBase คุณสามารถคิดว่ามันเป็นตารางในฐานข้อมูล หรือเป็นประเภทของข้อมูลที่คุณต้องการจัดเก็บ เช่น "ลูกค้า" หรือ "สินค้า" ครับ

*   **การสร้างคอลเลกชัน:** คุณสามารถสร้างคอลเลกชันใหม่ได้อย่างง่ายดายจากหน้าแผงควบคุมผู้ดูแลระบบ (Admin Panel)
*   **การกำหนดฟิลด์:** แต่ละคอลเลกชันสามารถมีฟิลด์ (Fields) หรือคุณสมบัติที่หลากหลาย เพื่อเก็บข้อมูลประเภทต่างๆ ได้อย่างเหมาะสม

### เวิร์กโฟลว์ (Workflow)

นอกจากการจัดเก็บข้อมูลแล้ว NocoBase ยังช่วยให้คุณสามารถสร้างระบบอัตโนมัติได้ด้วย **เวิร์กโฟลว์** ครับ

*   **การทำงานอัตโนมัติ:** เวิร์กโฟลว์ช่วยให้คุณกำหนดชุดของขั้นตอนที่จะทำงานโดยอัตโนมัติ เมื่อมีเหตุการณ์บางอย่างเกิดขึ้นกับข้อมูลของคุณ เช่น การส่งอีเมลแจ้งเตือนเมื่อมีการสร้างรายการใหม่ในคอลเลกชัน

### แหล่งข้อมูล (Data Source)

NocoBase รองรับการเชื่อมต่อกับ **แหล่งข้อมูล** ที่หลากหลาย เพื่อให้คุณสามารถดึงข้อมูลจากที่ต่างๆ มาใช้งานร่วมกันได้ครับ

*   **การเชื่อมต่อ:** คุณสามารถกำหนดค่าแหล่งข้อมูลภายนอก เช่น ฐานข้อมูล SQL เพื่อให้ NocoBase สามารถอ่านและเขียนข้อมูลได้

### ปลั๊กอิน (Plugin)

หากคุณต้องการขยายขีดความสามารถของ NocoBase ให้มากยิ่งขึ้น เรามี **ปลั๊กอิน** หลากหลายให้เลือกใช้งานครับ

*   **เพิ่มฟังก์ชันการทำงาน:** ปลั๊กอินช่วยเพิ่มฟังก์ชันการทำงานใหม่ๆ หรือบูรณาการกับบริการภายนอก เพื่อให้ NocoBase ตอบโจทย์ความต้องการเฉพาะของคุณได้ดียิ่งขึ้น
*   **การติดตั้ง:** คุณสามารถติดตั้งและจัดการปลั๊กอินได้จากหน้าแผงควบคุมผู้ดูแลระบบครับ
