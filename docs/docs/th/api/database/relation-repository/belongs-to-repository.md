## BelongsToRepository

อินเทอร์เฟซของ `BelongsToRepository` เป็นแบบเดียวกับ [HasOneRepository](./has-one-repository.md) ครับ/ค่ะ

`BelongsToRepository` เป็น `Repository` สำหรับจัดการความสัมพันธ์แบบ `BelongsTo` โดยมีเมธอดที่ช่วยอำนวยความสะดวกในการจัดการความสัมพันธ์ประเภทนี้ครับ/ค่ะ
