## พารามิเตอร์

| ชื่อพารามิเตอร์        | ประเภท          | ค่าเริ่มต้น | คำอธิบาย                                                              |
| :--------------------- | :-------------- | :---------- | :--------------------------------------------------------------------- |
| `options.values`       | `M`             | `{}`        | อ็อบเจกต์ข้อมูลที่ใช้สำหรับแทรก                                        |
| `options.whitelist?`   | `string[]`      | -           | ไวต์ลิสต์ของฟิลด์สำหรับ `values` เฉพาะฟิลด์ที่อยู่ในรายการเท่านั้นที่จะถูกจัดเก็บ |
| `options.blacklist?`   | `string[]`      | -           | แบล็กลิสต์ของฟิลด์สำหรับ `values` ฟิลด์ที่อยู่ในรายการจะไม่ถูกจัดเก็บ   |
| `options.transaction?` | `Transaction`   | -           | ทรานแซกชัน                                                             |
