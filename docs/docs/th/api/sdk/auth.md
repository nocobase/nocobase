# Auth

## ภาพรวม

คลาส `Auth` ใช้เป็นหลักที่ฝั่งไคลเอนต์เพื่อเข้าถึงข้อมูลผู้ใช้และเรียกใช้ API ที่เกี่ยวข้องกับการยืนยันตัวตนของผู้ใช้ครับ/ค่ะ

## คุณสมบัติของอินสแตนซ์

### `locale`

ภาษาที่ผู้ใช้ปัจจุบันใช้งานอยู่ครับ/ค่ะ

### `role`

บทบาทที่ผู้ใช้ปัจจุบันใช้งานอยู่ครับ/ค่ะ

### `token`

โทเค็น API ครับ/ค่ะ

### `authenticator`

ตัวยืนยันตัวตนที่ใช้สำหรับการยืนยันตัวตนของผู้ใช้ปัจจุบันครับ/ค่ะ ดูเพิ่มเติมที่ [การยืนยันตัวตนของผู้ใช้](/auth-verification/auth/)

## เมธอดของคลาส

### `signIn()`

เข้าสู่ระบบของผู้ใช้ครับ/ค่ะ

#### รูปแบบการใช้งาน

- `async signIn(values: any, authenticator?: string): Promise<AxiosResponse<any>>`

#### รายละเอียด

| ชื่อพารามิเตอร์  | ประเภท     | คำอธิบาย                                    |
| --------------- | -------- | -------------------------------------------- |
| `values`        | `any`    | พารามิเตอร์สำหรับเรียกใช้ API การเข้าสู่ระบบ |
| `authenticator` | `string` | ตัวระบุของตัวยืนยันตัวตนที่ใช้ในการเข้าสู่ระบบ |

### `signUp()`

ลงทะเบียนผู้ใช้ครับ/ค่ะ

#### รูปแบบการใช้งาน

- `async signUp(values: any, authenticator?: string): Promise<AxiosResponse<any>>`

#### รายละเอียด

| ชื่อพารามิเตอร์  | ประเภท     | คำอธิบาย                                    |
| --------------- | -------- | -------------------------------------------- |
| `values`        | `any`    | พารามิเตอร์สำหรับเรียกใช้ API การลงทะเบียน |
| `authenticator` | `string` | ตัวระบุของตัวยืนยันตัวตนที่ใช้ในการลงทะเบียน |

### `signOut()`

ออกจากระบบครับ/ค่ะ

#### รูปแบบการใช้งาน

- `async signOut(values: any, authenticator?: string): Promise<AxiosResponse<any>>`

#### รายละเอียด

| ชื่อพารามิเตอร์  | ประเภท     | คำอธิบาย                                    |
| --------------- | -------- | -------------------------------------------- |
| `values`        | `any`    | พารามิเตอร์สำหรับเรียกใช้ API การออกจากระบบ |
| `authenticator` | `string` | ตัวระบุของตัวยืนยันตัวตนที่ใช้ในการออกจากระบบ |
