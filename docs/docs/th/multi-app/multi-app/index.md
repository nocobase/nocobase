# การจัดการหลายแอปพลิเคชัน

## ภาพรวม

NocoBase รองรับการจัดการหลายแอปที่แยกจากกันผ่านจุดเข้าเดียวด้วย AppSupervisor.


ช่วงเริ่มต้นมักใช้แอปเดียวได้ แต่เมื่อขยายระบบจะมีต้นทุนและความต้องการการแยกส่วนสูงขึ้น.


โหมดนี้รันหลายแอปใน NocoBase เดียวกัน แต่ละแอปมี DB แยกได้ แต่แชร์โปรเซสและหน่วยความจำ.

![](https://static-docs.nocobase.com/202512231055907.png)


เมื่อสเกลใหญ่ขึ้นควรใช้โหมดไฮบริด: Supervisor + หลาย Worker environment.
