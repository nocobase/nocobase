# เอาต์พุตของเวิร์กโฟลว์

## บทนำ

โหนด "เอาต์พุตของเวิร์กโฟลว์" ใช้สำหรับกำหนดค่าเอาต์พุตของเวิร์กโฟลว์ที่ถูกเรียกใช้ครับ/ค่ะ เมื่อเวิร์กโฟลว์หนึ่งถูกเรียกใช้โดยเวิร์กโฟลว์อื่น โหนด "เอาต์พุตของเวิร์กโฟลว์" จะช่วยให้คุณสามารถส่งค่ากลับไปยังตัวเรียกใช้ได้ครับ/ค่ะ

## สร้างโหนด

ในเวิร์กโฟลว์ที่ถูกเรียกใช้ ให้เพิ่มโหนด "เอาต์พุตของเวิร์กโฟลว์" ครับ/ค่ะ:

![20241231002033](https://static-docs.nocobase.com/20241231002033.png)

## กำหนดค่าโหนด

### ค่าเอาต์พุต

ป้อนหรือเลือกตัวแปรเพื่อใช้เป็นค่าเอาต์พุตครับ/ค่ะ ค่าเอาต์พุตสามารถเป็นได้ทุกประเภท ไม่ว่าจะเป็นค่าคงที่ เช่น สตริง, ตัวเลข, ค่าตรรกะ (boolean), วันที่ หรือ JSON ที่กำหนดเอง หรือจะเป็นตัวแปรอื่น ๆ ที่อยู่ในเวิร์กโฟลว์ก็ได้ครับ/ค่ะ

![20241231003059](https://static-docs.nocobase.com/20241231003059.png)

:::info{title=เคล็ดลับ}
หากมีการเพิ่มโหนด "เอาต์พุตของเวิร์กโฟลว์" หลายโหนดในเวิร์กโฟลว์ที่ถูกเรียกใช้ เมื่อมีการเรียกใช้เวิร์กโฟลว์นั้น ค่าเอาต์พุตจะมาจากโหนด "เอาต์พุตของเวิร์กโฟลว์" ตัวสุดท้ายที่ถูกประมวลผลครับ/ค่ะ
:::
