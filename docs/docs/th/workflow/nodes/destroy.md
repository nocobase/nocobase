# ลบข้อมูล

ใช้สำหรับลบข้อมูลจากคอลเลกชันที่ตรงตามเงื่อนไขที่กำหนดครับ/ค่ะ

การใช้งานพื้นฐานของโหนดลบข้อมูลจะคล้ายกับโหนดอัปเดตข้อมูลครับ/ค่ะ เพียงแต่โหนดลบข้อมูลไม่จำเป็นต้องกำหนดค่าฟิลด์ แค่เลือกคอลเลกชันและเงื่อนไขการกรองเท่านั้นครับ/ค่ะ ผลลัพธ์ของโหนดลบข้อมูลจะคืนค่าจำนวนแถวของข้อมูลที่ถูกลบสำเร็จ ซึ่งสามารถดูได้ในประวัติการทำงานเท่านั้น และไม่สามารถนำไปใช้เป็นตัวแปรในโหนดถัดไปได้ครับ/ค่ะ

:::info{title=ข้อควรทราบ}
ปัจจุบัน โหนดลบข้อมูลไม่รองรับการลบทีละรายการ แต่เป็นการลบแบบกลุ่ม (Batch Deletion) ทั้งหมดครับ/ค่ะ ดังนั้นจึงไม่ทริกเกอร์เหตุการณ์อื่น ๆ สำหรับการลบข้อมูลแต่ละรายการครับ/ค่ะ
:::

## สร้างโหนด

ในหน้าจอการตั้งค่าเวิร์กโฟลว์ ให้คลิกปุ่มเครื่องหมายบวก (“+”) ในโฟลว์ เพื่อเพิ่มโหนด “ลบข้อมูล” ครับ/ค่ะ

![สร้างโหนดลบข้อมูล](https://static-docs.nocobase.com/e1c6b8728251fcdbed6c7f50e5570da2.png)

## การตั้งค่าโหนด

![โหนดลบข้อมูล_การตั้งค่าโหนด](https://static-docs.nocobase.com/580600c2b13ef4e01dfa48b23539648e.png)

### คอลเลกชัน

เลือกคอลเลกชันที่ต้องการลบข้อมูลครับ/ค่ะ

### เงื่อนไขการกรอง

คล้ายกับเงื่อนไขการกรองในการค้นหาคอลเลกชันทั่วไป คุณสามารถใช้ตัวแปรบริบทของเวิร์กโฟลว์ได้ครับ/ค่ะ

## ตัวอย่าง

เช่น การล้างข้อมูลคำสั่งซื้อเก่าที่ไม่ถูกต้องและถูกยกเลิกแล้วตามกำหนดเวลา สามารถทำได้โดยใช้โหนดลบข้อมูลครับ/ค่ะ

![โหนดลบข้อมูล_ตัวอย่าง_การตั้งค่าโหนด](https://static-docs.nocobase.com/b94b75077a17252f8523c3f13ce5f320.png)

เวิร์กโฟลว์จะถูกทริกเกอร์ตามกำหนดเวลา และดำเนินการลบข้อมูลคำสั่งซื้อเก่าที่ไม่ถูกต้องและถูกยกเลิกทั้งหมดครับ/ค่ะ
