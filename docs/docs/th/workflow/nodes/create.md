# เพิ่มข้อมูล

ใช้สำหรับเพิ่มข้อมูลใหม่หนึ่งรายการในคอลเลกชัน

ค่าฟิลด์สำหรับข้อมูลใหม่สามารถใช้ตัวแปรจากบริบทของเวิร์กโฟลว์ได้ สำหรับการกำหนดค่าให้กับฟิลด์ความสัมพันธ์ คุณสามารถอ้างอิงตัวแปรข้อมูลที่เกี่ยวข้องในบริบทได้โดยตรง ซึ่งอาจเป็นอ็อบเจกต์หรือค่าของคีย์นอกก็ได้ หากไม่ใช้ตัวแปร คุณจะต้องกรอกค่าคีย์นอกด้วยตนเอง สำหรับค่าคีย์นอกหลายค่าในความสัมพันธ์แบบหนึ่งต่อหลาย จะต้องคั่นด้วยเครื่องหมายจุลภาค (คอมมา) ครับ/ค่ะ

## สร้างโหนด

ในหน้าจอการตั้งค่าเวิร์กโฟลว์ ให้คลิกปุ่มเครื่องหมายบวก (“+”) ในโฟลว์เพื่อเพิ่มโหนด “เพิ่มข้อมูล” ครับ/ค่ะ

![เพิ่มโหนด 'เพิ่มข้อมูล'](https://static-docs.nocobase.com/386c8c01c89b1eeab848510e77f4841a.png)

## การตั้งค่าโหนด

![โหนดเพิ่มข้อมูล_ตัวอย่าง_การตั้งค่าโหนด](https://static-docs.nocobase.com/5f7b97a51b64a1741cf82a4d4455b610.png)

### คอลเลกชัน

เลือกคอลเลกชันที่คุณต้องการเพิ่มข้อมูลใหม่ครับ/ค่ะ

### ค่าฟิลด์

กำหนดค่าให้กับฟิลด์ของคอลเลกชันครับ/ค่ะ คุณสามารถใช้ตัวแปรจากบริบทของเวิร์กโฟลว์ หรือจะกรอกค่าคงที่ด้วยตนเองก็ได้

:::info{title="ข้อแนะนำ"}
ข้อมูลที่สร้างโดยโหนด “เพิ่มข้อมูล” ในเวิร์กโฟลว์ จะไม่จัดการข้อมูลผู้ใช้ เช่น “ผู้สร้าง” และ “ผู้แก้ไขล่าสุด” โดยอัตโนมัตินะครับ/คะ คุณต้องกำหนดค่าสำหรับฟิลด์ทั้งสองนี้ด้วยตนเองตามความเหมาะสมครับ/ค่ะ
:::

### โหลดข้อมูลความสัมพันธ์ล่วงหน้า

หากฟิลด์ของข้อมูลใหม่มีฟิลด์ความสัมพันธ์ และคุณต้องการใช้ข้อมูลความสัมพันธ์ที่เกี่ยวข้องในขั้นตอนเวิร์กโฟลว์ถัดไป คุณสามารถเลือกฟิลด์ความสัมพันธ์ที่เกี่ยวข้องในการตั้งค่าการโหลดล่วงหน้าได้ครับ/ค่ะ ด้วยวิธีนี้ หลังจากสร้างข้อมูลใหม่แล้ว ข้อมูลความสัมพันธ์ที่เกี่ยวข้องจะถูกโหลดและจัดเก็บพร้อมกันในข้อมูลผลลัพธ์ของโหนดโดยอัตโนมัติครับ/ค่ะ

## ตัวอย่าง

ตัวอย่างเช่น เมื่อข้อมูลในคอลเลกชัน “บทความ” ถูกสร้างหรืออัปเดต จำเป็นต้องสร้างข้อมูล “เวอร์ชันบทความ” ใหม่โดยอัตโนมัติ เพื่อบันทึกประวัติการเปลี่ยนแปลงของบทความ คุณสามารถใช้โหนด “เพิ่มข้อมูล” เพื่อทำสิ่งนี้ได้ครับ/ค่ะ

![โหนดเพิ่มข้อมูล_ตัวอย่าง_การตั้งค่าเวิร์กโฟลว์](https://static-docs.nocobase.com/dfd4820d49c145fa331883fc09c9161f.png)

![โหนดเพิ่มข้อมูล_ตัวอย่าง_การตั้งค่าโหนด](https://static-docs.nocobase.com/1a0992e66170be12a068da6503298868.png)

หลังจากเปิดใช้งานเวิร์กโฟลว์ด้วยการตั้งค่านี้แล้ว เมื่อข้อมูลในคอลเลกชัน “บทความ” มีการเปลี่ยนแปลง ข้อมูล “เวอร์ชันบทความ” จะถูกสร้างขึ้นใหม่โดยอัตโนมัติ เพื่อบันทึกประวัติการเปลี่ยนแปลงของบทความครับ/ค่ะ
