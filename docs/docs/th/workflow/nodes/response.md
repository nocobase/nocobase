# การตอบกลับ HTTP

## บทนำ

โหนดนี้รองรับเฉพาะการใช้งานใน เวิร์กโฟลว์ Webhook ที่ทำงานแบบซิงโครนัสเท่านั้นครับ/ค่ะ โดยมีวัตถุประสงค์เพื่อส่งการตอบกลับไปยังระบบภายนอก ตัวอย่างเช่น ในขั้นตอนการประมวลผลการเรียกกลับการชำระเงิน (payment callback) หากกระบวนการทางธุรกิจเกิดผลลัพธ์ที่ไม่คาดคิด (เช่น มีข้อผิดพลาด หรือความล้มเหลว) คุณสามารถใช้โหนดการตอบกลับนี้เพื่อส่งข้อความแสดงข้อผิดพลาดกลับไปยังระบบภายนอกได้ครับ/ค่ะ เพื่อให้ระบบเหล่านั้นสามารถลองดำเนินการใหม่ได้ในภายหลัง โดยอ้างอิงจากสถานะที่ได้รับ

นอกจากนี้ การทำงานของโหนดการตอบกลับจะยุติการทำงานของ เวิร์กโฟลว์ ทันทีครับ/ค่ะ และโหนดที่ตามมาจะไม่ถูกดำเนินการต่อ หาก เวิร์กโฟลว์ ทั้งหมดไม่ได้กำหนดค่าโหนดการตอบกลับไว้ ระบบจะตอบกลับโดยอัตโนมัติตามสถานะการทำงานของโฟลว์ โดยจะส่ง `200` เมื่อดำเนินการสำเร็จ และ `500` เมื่อดำเนินการล้มเหลวครับ/ค่ะ

## การสร้างโหนดการตอบกลับ

ในหน้าจอการตั้งค่า เวิร์กโฟลว์ ให้คลิกปุ่มเครื่องหมายบวก ("+") ในโฟลว์ เพื่อเพิ่มโหนด "การตอบกลับ" ครับ/ค่ะ
