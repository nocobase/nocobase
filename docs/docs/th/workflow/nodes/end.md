# สิ้นสุดเวิร์กโฟลว์

เมื่อโหนดนี้ทำงาน จะเป็นการสิ้นสุดเวิร์กโฟลว์ที่กำลังทำงานอยู่ทันที ด้วยสถานะที่กำหนดไว้ในโหนดครับ/ค่ะ โดยทั่วไปแล้ว โหนดนี้จะใช้สำหรับการควบคุมการไหลของกระบวนการ (flow control) ตามตรรกะที่กำหนด เมื่อเงื่อนไขบางอย่างเป็นจริง ก็จะออกจากเวิร์กโฟลว์ปัจจุบันและไม่ดำเนินการขั้นตอนต่อไป สามารถเปรียบเทียบได้กับคำสั่ง `return` ในภาษาโปรแกรม ที่ใช้สำหรับออกจากฟังก์ชันที่กำลังทำงานอยู่ครับ/ค่ะ

## เพิ่มโหนด

ในหน้าจอการตั้งค่าเวิร์กโฟลว์ ให้คลิกปุ่มเครื่องหมายบวก ("+") ในโฟลว์ เพื่อเพิ่มโหนด "สิ้นสุดเวิร์กโฟลว์" ครับ/ค่ะ

### สถานะการสิ้นสุด

สถานะการสิ้นสุดจะส่งผลต่อสถานะสุดท้ายของการทำงานของเวิร์กโฟลว์ครับ/ค่ะ โดยสามารถตั้งค่าเป็น "สำเร็จ" หรือ "ล้มเหลว" ได้ เมื่อเวิร์กโฟลว์ทำงานมาถึงโหนดนี้ ระบบจะออกจากเวิร์กโฟลว์ทันทีด้วยสถานะที่กำหนดไว้ครับ/ค่ะ

:::info{title=ข้อแนะนำ}
เมื่อใช้โหนดนี้ในเวิร์กโฟลว์ประเภท "เหตุการณ์ก่อนดำเนินการ" (Before action event) จะทำให้มีการดักจับ (intercept) คำขอที่เริ่มต้นการดำเนินการนั้นครับ/ค่ะ สำหรับรายละเอียดเพิ่มเติม โปรดดูที่ [การใช้งาน "เหตุการณ์ก่อนดำเนินการ"](../triggers/pre-action)

นอกจากนี้ นอกจากการดักจับคำขอที่เริ่มต้นการดำเนินการแล้ว การตั้งค่าสถานะการสิ้นสุดยังส่งผลต่อสถานะของข้อมูลที่ตอบกลับใน "ข้อความตอบกลับ" (response message) ในเวิร์กโฟลว์ประเภทนี้ด้วยครับ/ค่ะ
:::
