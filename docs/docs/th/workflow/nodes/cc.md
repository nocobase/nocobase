# 抄送 <Badge>v1.8.2+</Badge>

## บทนำ ครับ

โหนด抄送 ใช้สำหรับส่งเนื้อหาบางส่วนที่เกี่ยวข้องกับบริบทในระหว่างการทำงานของเวิร์กโฟลว์ ไปยังผู้ใช้ที่ระบุ เพื่อให้ผู้ใช้รับทราบและตรวจสอบครับ ตัวอย่างเช่น ในกระบวนการอนุมัติหรือกระบวนการอื่น ๆ คุณสามารถส่งสำเนาข้อมูลที่เกี่ยวข้องไปยังผู้เข้าร่วมคนอื่น ๆ ได้ เพื่อให้พวกเขาทราบความคืบหน้าของงานได้ทันท่วงทีครับ

คุณสามารถตั้งค่าโหนด抄送 ได้หลายโหนดในเวิร์กโฟลว์ เพื่อให้เมื่อเวิร์กโฟลว์ทำงานมาถึงโหนดนี้ ข้อมูลที่เกี่ยวข้องจะถูกส่งไปยังผู้รับที่ระบุครับ

เนื้อหาที่ถูก抄送 จะแสดงอยู่ในเมนู "抄送ถึงฉัน" ในศูนย์รวมงาน (To-Do Center) ซึ่งผู้ใช้สามารถดูเนื้อหาทั้งหมดที่ถูก抄送ถึงตนเองได้จากที่นี่ครับ และระบบจะแจ้งเตือนเนื้อหาที่ยังไม่ได้อ่านตามสถานะที่ยังไม่ได้เปิดดู หลังจากผู้ใช้ดูแล้ว สามารถกดทำเครื่องหมายว่าอ่านแล้วได้เองครับ

## การสร้างโหนด ครับ

ในหน้าจอการตั้งค่าเวิร์กโฟลว์ ให้คลิกปุ่มเครื่องหมายบวก ("+") ในโฟลว์ เพื่อเพิ่มโหนด "抄送" ครับ

![เพิ่มโหนด抄送](https://static-docs.nocobase.com/20250710222842.png)

## การตั้งค่าโหนด ครับ

![การตั้งค่าโหนด](https://static-docs.nocobase.com/20250710224041.png)

ในหน้าจอการตั้งค่าโหนด คุณสามารถตั้งค่าพารามิเตอร์ต่อไปนี้ได้ครับ

### ผู้รับ ครับ

ผู้รับคือกลุ่มผู้ใช้ที่เป็นเป้าหมายของการ抄送 ซึ่งอาจเป็นผู้ใช้หนึ่งคนหรือหลายคนก็ได้ครับ แหล่งที่มาของการเลือกสามารถเป็นค่าคงที่ที่เลือกจากรายการผู้ใช้ หรือเป็นค่าไดนามิกที่ระบุโดยตัวแปร หรือเป็นผลลัพธ์จากการสืบค้น (query) บนคอลเลกชันผู้ใช้ครับ

![การตั้งค่าผู้รับ](https://static-docs.nocobase.com/20250710224421.png)

### ส่วนติดต่อผู้ใช้ ครับ

ผู้รับจำเป็นต้องดูเนื้อหาที่ถูก抄送 ในเมนู "抄送ถึงฉัน" ของศูนย์รวมงาน (To-Do Center) ครับ คุณสามารถตั้งค่าผลลัพธ์ของทริกเกอร์และโหนดใด ๆ ในบริบทของเวิร์กโฟลว์ให้เป็นบล็อกเนื้อหาได้ครับ

![ส่วนติดต่อผู้ใช้](https://static-docs.nocobase.com/20250710225400.png)

### หัวข้อของงาน ครับ

หัวข้อของงานคือชื่อเรื่องที่แสดงในศูนย์รวมงาน (To-Do Center) ครับ คุณสามารถใช้ตัวแปรจากบริบทของเวิร์กโฟลว์เพื่อสร้างหัวข้อแบบไดนามิกได้ครับ

![หัวข้อของงาน](https://static-docs.nocobase.com/20250710225603.png)

## ศูนย์รวมงาน ครับ

ผู้ใช้สามารถดูและจัดการเนื้อหาทั้งหมดที่ถูก抄送ถึงตนเองได้ในศูนย์รวมงาน (To-Do Center) และสามารถกรองหรือดูตามสถานะการอ่านได้ครับ

![20250710232932](https://static-docs.nocobase.com/20250710232932.png)

![20250710233032](https://static-docs.nocobase.com/20250710233032.png)

หลังจากดูแล้ว คุณสามารถทำเครื่องหมายว่าอ่านแล้วได้ และจำนวนรายการที่ยังไม่ได้อ่านก็จะลดลงตามไปด้วยครับ

![20250710233102](https://static-docs.nocobase.com/20250710233102.png)
