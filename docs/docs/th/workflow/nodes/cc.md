# สำเนา (CC) <Badge>v1.8.2+</Badge>

## บทนำ

โหนดสำเนา (CC) ใช้สำหรับส่งเนื้อหาบริบทบางอย่างในระหว่างกระบวนการทำงานของเวิร์กโฟลว์ไปยังผู้ใช้ที่ระบุ เพื่อให้รับทราบและตรวจสอบครับ ตัวอย่างเช่น ในกระบวนการอนุมัติหรือกระบวนการอื่น ๆ สามารถส่งสำเนาข้อมูลที่เกี่ยวข้องไปยังผู้เข้าร่วมคนอื่น ๆ เพื่อให้พวกเขาทราบความคืบหน้าของงานได้ทันท่วงทีครับ

สามารถตั้งค่าโหนดสำเนา (CC) ได้หลายโหนดในเวิร์กโฟลว์ เพื่อให้เมื่อเวิร์กโฟลว์ทำงานถึงโหนดนั้น ข้อมูลที่เกี่ยวข้องจะถูกส่งไปยังผู้รับที่ระบุครับ

เนื้อหาที่สำเนาจะแสดงในเมนู "สำเนาถึงฉัน" (CC to Me) ของศูนย์รวมงาน (To-Do Center) ซึ่งผู้ใช้สามารถดูเนื้อหาทั้งหมดที่สำเนาถึงตนเองได้ที่นี่ และจะมีการแจ้งเตือนเนื้อหาที่ยังไม่ได้อ่านตามสถานะการอ่าน หลังจากผู้ใช้ดูแล้วสามารถทำเครื่องหมายว่าอ่านแล้วได้ด้วยตนเองครับ

## การตั้งค่าโหนด

![节点配置](https://static-docs.nocobase.com/20250710224041.png)

ในหน้าจอการกำหนดค่าโหนด สามารถตั้งค่าพารามิเตอร์ต่อไปนี้ได้ครับ:

### ผู้รับ

ผู้รับคือคอลเลกชันของผู้ใช้เป้าหมายสำหรับการสำเนา ซึ่งอาจเป็นผู้ใช้หนึ่งคนหรือหลายคน แหล่งที่มาของการเลือกสามารถเป็นค่าคงที่ที่เลือกจากรายการผู้ใช้ หรือเป็นค่าไดนามิกที่ระบุโดยตัวแปร หรือเป็นผลลัพธ์จากการสืบค้นในคอลเลกชันผู้ใช้ครับ

![接收者配置](https://static-docs.nocobase.com/20250710224421.png)

### ส่วนติดต่อผู้ใช้

ผู้รับจำเป็นต้องดูเนื้อหาที่สำเนาในเมนู "สำเนาถึงฉัน" (CC to Me) ของศูนย์รวมงาน (To-Do Center) สามารถกำหนดค่าผลลัพธ์ของทริกเกอร์และโหนดใด ๆ ในบริบทของเวิร์กโฟลว์ให้เป็นบล็อกเนื้อหาได้ครับ

![用户界面](https://static-docs.nocobase.com/20250710225400.png)

### การ์ดงาน <Badge>2.0+</Badge>

สามารถใช้เพื่อกำหนดค่าการ์ดงานในรายการ "สำเนาถึงฉัน" (CC to Me) ในศูนย์รวมงาน (To-Do Center) ได้ครับ

![20260213010947](https://static-docs.nocobase.com/20260213010947.png)

ในการ์ดสามารถกำหนดค่าฟิลด์ธุรกิจที่ต้องการแสดงได้อย่างอิสระ (ยกเว้นฟิลด์ความสัมพันธ์) ครับ

หลังจากสร้างงานสำเนาเวิร์กโฟลว์แล้ว จะสามารถเห็นการ์ดงานที่กำหนดเองได้ในรายการของศูนย์รวมงานครับ

![20260214124325](https://static-docs.nocobase.com/20260214124325.png)

### หัวข้อของงาน

หัวข้อของงานคือชื่อเรื่องที่แสดงในศูนย์รวมงาน (To-Do Center) สามารถใช้ตัวแปรจากบริบทของเวิร์กโฟลว์เพื่อสร้างหัวข้อแบบไดนามิกได้ครับ

![任务标题](https://static-docs.nocobase.com/20250710225603.png)

## ศูนย์รวมงาน

ผู้ใช้สามารถดูและจัดการเนื้อหาทั้งหมดที่สำเนาถึงตนเองได้ในศูนย์รวมงาน (To-Do Center) และสามารถกรองหรือดูตามสถานะการอ่านได้ครับ

![20250710232932](https://static-docs.nocobase.com/20250710232932.png)

![20250710233032](https://static-docs.nocobase.com/20250710233032.png)

หลังจากดูแล้วสามารถทำเครื่องหมายว่าอ่านแล้วได้ และจำนวนรายการที่ยังไม่ได้อ่านจะลดลงตามไปด้วยครับ

![20250710233102](https://static-docs.nocobase.com/20250710233102.png)
