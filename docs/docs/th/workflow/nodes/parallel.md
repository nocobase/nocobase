# แขนงคู่ขนาน

โหนดแขนงคู่ขนานสามารถแบ่งเวิร์กโฟลว์ออกเป็นหลายแขนงได้ โดยแต่ละแขนงสามารถกำหนดค่าโหนดที่แตกต่างกัน และวิธีการทำงานของแต่ละแขนงก็จะแตกต่างกันไปตามโหมดที่เลือกครับ/ค่ะ โหนดนี้เหมาะสำหรับสถานการณ์ที่คุณต้องการให้มีการดำเนินการหลายอย่างพร้อมกันครับ/ค่ะ

## การติดตั้ง

ปลั๊กอินนี้เป็นแบบในตัว ไม่ต้องติดตั้งเพิ่มเติมครับ/ค่ะ

## การสร้างโหนด

ในหน้าจอการตั้งค่าเวิร์กโฟลว์ ให้คลิกปุ่มเครื่องหมายบวก (“+”) ในโฟลว์ เพื่อเพิ่มโหนด “แขนงคู่ขนาน” ครับ/ค่ะ

![เพิ่มแขนงคู่ขนาน](https://static-docs.nocobase.com/9e0f3faa0b9335270647a30477559eac.png)

หลังจากเพิ่มโหนดแขนงคู่ขนานในเวิร์กโฟลว์แล้ว ระบบจะเพิ่มแขนงย่อยสองแขนงให้โดยอัตโนมัติครับ/ค่ะ คุณสามารถคลิกปุ่มเพิ่มแขนงเพื่อเพิ่มแขนงได้อีกหลายแขนงตามต้องการ แต่ละแขนงสามารถเพิ่มโหนดได้ไม่จำกัดจำนวน ส่วนแขนงที่ไม่ต้องการ สามารถลบออกได้โดยคลิกปุ่มลบที่จุดเริ่มต้นของแขนงนั้นๆ ครับ/ค่ะ

![จัดการแขนงคู่ขนาน](https://static-docs.nocobase.com/36088a8b771eb3ee9bc2a757.png)

## การตั้งค่าโหนด

### โหมดแขนง

โหนดแขนงคู่ขนานมีโหมดการทำงานดังต่อไปนี้ 3 โหมดครับ/ค่ะ

-   **สำเร็จทั้งหมด**: เวิร์กโฟลว์จะดำเนินการต่อที่โหนดถัดไปหลังจากแขนงทั้งหมดทำงานสำเร็จเท่านั้นครับ/ค่ะ มิฉะนั้น หากแขนงใดแขนงหนึ่งหยุดทำงานก่อนกำหนด ไม่ว่าจะด้วยเหตุผลความล้มเหลว ข้อผิดพลาด หรือสถานะอื่นๆ ที่ไม่ใช่ความสำเร็จ โหนดแขนงคู่ขนานทั้งหมดจะหยุดทำงานก่อนกำหนดด้วยสถานะนั้นทันที หรือที่เรียกว่า “All mode” ครับ/ค่ะ
-   **สำเร็จแขนงใดแขนงหนึ่ง**: เวิร์กโฟลว์จะดำเนินการต่อที่โหนดถัดไปหลังจากแขนงใดแขนงหนึ่งทำงานสำเร็จครับ/ค่ะ โหนดแขนงคู่ขนานทั้งหมดจะหยุดทำงานก่อนกำหนดด้วยสถานะนั้น ก็ต่อเมื่อแขนงทั้งหมดหยุดทำงานก่อนกำหนด ไม่ว่าจะด้วยเหตุผลความล้มเหลว ข้อผิดพลาด หรือสถานะอื่นๆ ที่ไม่ใช่ความสำเร็จ หรือที่เรียกว่า “Any mode” ครับ/ค่ะ
-   **สำเร็จหรือล้มเหลวแขนงใดแขนงหนึ่ง**: เวิร์กโฟลว์จะดำเนินการต่อที่โหนดถัดไปหลังจากแขนงใดแขนงหนึ่งทำงานสำเร็จครับ/ค่ะ แต่หากโหนดใดโหนดหนึ่งล้มเหลว โหนดแขนงคู่ขนานทั้งหมดจะหยุดทำงานก่อนกำหนดด้วยสถานะนั้นทันที หรือที่เรียกว่า “Race mode” ครับ/ค่ะ

ไม่ว่าจะเป็นโหมดใดก็ตาม ระบบจะพยายามดำเนินการแต่ละแขนงจากซ้ายไปขวาตามลำดับ จนกว่าจะตรงตามเงื่อนไขของโหมดแขนงที่ตั้งไว้ จึงจะดำเนินการต่อที่โหนดถัดไป หรือหยุดทำงานก่อนกำหนดครับ/ค่ะ

## ตัวอย่าง

ดูตัวอย่างได้ที่ [โหนดหน่วงเวลา](./delay.md) ครับ/ค่ะ
