# การจัดการเวอร์ชัน

หลังจากที่เวิร์กโฟลว์ที่ตั้งค่าไว้ถูกทริกเกอร์อย่างน้อยหนึ่งครั้ง หากคุณต้องการแก้ไขการตั้งค่าของเวิร์กโฟลว์หรือโหนดภายใน คุณจะต้องสร้างเวอร์ชันใหม่ก่อนทำการแก้ไขครับ/ค่ะ วิธีนี้จะช่วยให้มั่นใจได้ว่าเมื่อคุณตรวจสอบประวัติการทำงานของเวิร์กโฟลว์ที่เคยถูกทริกเกอร์ไปแล้ว จะไม่ได้รับผลกระทบจากการแก้ไขในอนาคต

บนหน้าการตั้งค่าของเวิร์กโฟลว์ คุณสามารถดูเวอร์ชันเวิร์กโฟลว์ที่มีอยู่ได้จากเมนู "เวอร์ชัน" ที่มุมขวาบนครับ/ค่ะ

![ดูเวอร์ชันของเวิร์กโฟลว์](https://static-docs.nocobase.com/ad93d2c08166b0e3e643fb148713a63f.png)

ในเมนู "การดำเนินการเพิ่มเติม" ("...") ที่อยู่ด้านขวา คุณสามารถเลือกคัดลอกเวอร์ชันที่กำลังดูอยู่ไปยังเวอร์ชันใหม่ได้ครับ/ค่ะ

![คัดลอกเวิร์กโฟลว์เป็นเวอร์ชันใหม่](https://static-docs.nocobase.com/2805798e6caca2af004893390a744256.png)

หลังจากคัดลอกไปยังเวอร์ชันใหม่แล้ว ให้คลิกสวิตช์ "เปิดใช้งาน" / "ปิดใช้งาน" เพื่อเปลี่ยนเวอร์ชันที่ต้องการให้อยู่ในสถานะ "เปิดใช้งาน" ครับ/ค่ะ จากนั้นเวอร์ชันเวิร์กโฟลว์ใหม่ก็จะเริ่มมีผลใช้งาน

หากคุณต้องการกลับไปใช้เวอร์ชันเก่า ให้สลับไปที่เวอร์ชันนั้นจากเมนู "เวอร์ชัน" ครับ/ค่ะ จากนั้นคลิกสวิตช์ "เปิดใช้งาน" / "ปิดใช้งาน" อีกครั้งเพื่อเปลี่ยนเป็นสถานะ "เปิดใช้งาน" เวอร์ชันที่คุณกำลังดูอยู่ก็จะเริ่มมีผลใช้งาน และการทริกเกอร์ครั้งต่อไปจะดำเนินการตามกระบวนการของเวอร์ชันนั้นครับ/ค่ะ

เมื่อคุณต้องการปิดใช้งานเวิร์กโฟลว์ ให้คลิกสวิตช์ "เปิดใช้งาน" / "ปิดใช้งาน" เพื่อเปลี่ยนเป็นสถานะ "ปิดใช้งาน" ครับ/ค่ะ เวิร์กโฟลว์นั้นก็จะไม่ถูกทริกเกอร์อีกต่อไป

:::info{title=เคล็ดลับ}
แตกต่างจากการ "คัดลอก" เวิร์กโฟลว์จากรายการจัดการเวิร์กโฟลว์ครับ/ค่ะ เวิร์กโฟลว์ที่ "คัดลอกไปยังเวอร์ชันใหม่" จะยังคงจัดอยู่ในกลุ่มเวิร์กโฟลว์เดียวกัน เพียงแต่สามารถแยกแยะได้ด้วยเวอร์ชัน แต่การคัดลอกเวิร์กโฟลว์จะถือว่าเป็นเวิร์กโฟลว์ใหม่ทั้งหมด ไม่เกี่ยวข้องกับเวอร์ชันของเวิร์กโฟลว์ก่อนหน้า และจำนวนครั้งที่ดำเนินการก็จะถูกรีเซ็ตเป็นศูนย์ด้วยครับ/ค่ะ
:::
