# การตั้งค่าขั้นสูง

## โหมดการทำงาน

เวิร์กโฟลว์จะทำงานในโหมด "อะซิงโครนัส" (Asynchronous) หรือ "ซิงโครนัส" (Synchronous) โดยขึ้นอยู่กับประเภททริกเกอร์ที่เลือกในขณะสร้างครับ/ค่ะ โหมดอะซิงโครนัสหมายถึง เมื่อมีเหตุการณ์ที่กำหนดทริกเกอร์ เวิร์กโฟลว์จะเข้าสู่คิวและถูกจัดกำหนดการให้ทำงานทีละรายการโดยระบบเบื้องหลัง ส่วนโหมดซิงโครนัส เมื่อถูกทริกเกอร์แล้วจะไม่เข้าสู่คิวการจัดกำหนดการ แต่จะเริ่มทำงานทันทีและให้ผลตอบรับทันทีเมื่อทำงานเสร็จสิ้นครับ/ค่ะ

โดยค่าเริ่มต้น เหตุการณ์คอลเลกชัน, เหตุการณ์หลังการดำเนินการ, เหตุการณ์การดำเนินการที่กำหนดเอง, เหตุการณ์ตามกำหนดเวลา และเหตุการณ์การอนุมัติ จะทำงานแบบอะซิงโครนัส ส่วนเหตุการณ์ก่อนการดำเนินการจะทำงานแบบซิงโครนัสโดยค่าเริ่มต้นครับ/ค่ะ สำหรับเหตุการณ์คอลเลกชันและเหตุการณ์ฟอร์มนั้นรองรับทั้งสองโหมด ซึ่งคุณสามารถเลือกได้ในขณะสร้างเวิร์กโฟลว์ครับ/ค่ะ

![โหมดซิงโครนัส_สร้างเวิร์กโฟลว์แบบซิงโครนัส](https://static-docs.nocobase.com/39bc0821f50c1bde4729c531c6236795.png)

:::info{title=ข้อควรทราบ}
เวิร์กโฟลว์แบบซิงโครนัสมีข้อจำกัดตามลักษณะของโหมดนี้ จึงไม่สามารถใช้โหนดที่ทำให้เกิดสถานะ "รอ" ได้ เช่น "การประมวลผลด้วยตนเอง" เป็นต้นครับ/ค่ะ
:::

## การลบประวัติการทำงานอัตโนมัติ

เมื่อเวิร์กโฟลว์ถูกทริกเกอร์บ่อยครั้ง คุณสามารถตั้งค่าการลบประวัติการทำงานอัตโนมัติเพื่อลดความยุ่งเหยิง และช่วยลดภาระการจัดเก็บข้อมูลในฐานข้อมูลได้ครับ/ค่ะ

คุณสามารถตั้งค่าการลบประวัติการทำงานอัตโนมัติสำหรับเวิร์กโฟลว์ได้จากหน้าต่างสร้างและแก้ไขเวิร์กโฟลว์เช่นกันครับ/ค่ะ

![การตั้งค่าการลบประวัติการทำงานอัตโนมัติ](https://static-docs.nocobase.com/b2e4c08e7a01e213069912fe04baa7bd.png)

คุณสามารถตั้งค่าการลบอัตโนมัติได้ตามสถานะของผลการทำงานครับ/ค่ะ ในกรณีส่วนใหญ่ เราขอแนะนำให้เลือกเฉพาะสถานะ "เสร็จสมบูรณ์" เท่านั้น เพื่อให้ระบบยังคงเก็บรักษาบันทึกการทำงานที่ล้มเหลวไว้สำหรับการแก้ไขปัญหาในภายหลังครับ/ค่ะ

เราไม่แนะนำให้เปิดใช้งานการลบประวัติการทำงานอัตโนมัติในขณะที่คุณกำลังดีบักเวิร์กโฟลว์ครับ/ค่ะ ทั้งนี้ก็เพื่อให้คุณสามารถใช้ประวัติการทำงานเพื่อตรวจสอบว่าตรรกะการทำงานของเวิร์กโฟลว์เป็นไปตามที่คาดหวังไว้หรือไม่

:::info{title=ข้อควรทราบ}
การลบประวัติการทำงานของเวิร์กโฟลว์ไม่ได้ส่งผลให้จำนวนครั้งที่เวิร์กโฟลว์เคยทำงานลดลงครับ/ค่ะ
:::
