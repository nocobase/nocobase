# ภาพรวม

## บทนำ

ปลั๊กอินเวิร์กโฟลว์ช่วยให้คุณจัดการกระบวนการทางธุรกิจแบบอัตโนมัติใน NocoBase เช่น การอนุมัติประจำวัน, การซิงค์ข้อมูล, การแจ้งเตือน และงานอื่นๆ ในเวิร์กโฟลว์ คุณสามารถสร้างตรรกะทางธุรกิจที่ซับซ้อนได้ง่ายๆ เพียงแค่ตั้งค่าทริกเกอร์และโหนดที่เกี่ยวข้องผ่านอินเทอร์เฟซแบบกราฟิก โดยไม่ต้องเขียนโค้ดเลยครับ/ค่ะ

### ตัวอย่าง

เวิร์กโฟลว์แต่ละรายการประกอบด้วยทริกเกอร์หนึ่งตัวและโหนดหลายตัว ทริกเกอร์เป็นตัวแทนของเหตุการณ์ในระบบ ส่วนแต่ละโหนดเป็นตัวแทนของขั้นตอนการทำงาน ซึ่งทั้งหมดนี้จะอธิบายตรรกะทางธุรกิจที่ต้องดำเนินการหลังจากเกิดเหตุการณ์ขึ้น ภาพด้านล่างแสดงกระบวนการลดสต็อกสินค้าทั่วไปหลังจากมีการสั่งซื้อสินค้าครับ/ค่ะ

![ตัวอย่างเวิร์กโฟลว์](https://static-docs.nocobase.com/20251029222146.png)

เมื่อผู้ใช้ส่งคำสั่งซื้อ เวิร์กโฟลว์จะตรวจสอบสต็อกสินค้าโดยอัตโนมัติ หากสต็อกเพียงพอ ก็จะหักสต็อกและดำเนินการสร้างคำสั่งซื้อต่อไป มิฉะนั้น กระบวนการจะสิ้นสุดลงครับ/ค่ะ

### กรณีการใช้งาน

จากมุมมองที่กว้างขึ้น เวิร์กโฟลว์ในแอปพลิเคชัน NocoBase สามารถช่วยแก้ปัญหาในสถานการณ์ต่างๆ ได้ครับ/ค่ะ

- จัดการงานที่ทำซ้ำๆ โดยอัตโนมัติ: เช่น การตรวจสอบคำสั่งซื้อ, การซิงค์สต็อก, การล้างข้อมูล, การคำนวณคะแนน และอื่นๆ ที่ไม่จำเป็นต้องทำด้วยตนเองอีกต่อไป
- รองรับการทำงานร่วมกันระหว่างคนกับระบบ (Human-Machine Collaboration): จัดให้มีการอนุมัติหรือตรวจสอบที่โหนดสำคัญ และดำเนินการในขั้นตอนถัดไปตามผลลัพธ์ที่ได้ครับ/ค่ะ
- เชื่อมต่อกับระบบภายนอก: ส่งคำขอ HTTP, รับข้อมูลจากบริการภายนอก และทำให้เกิดระบบอัตโนมัติข้ามระบบครับ/ค่ะ
- ปรับตัวเข้ากับการเปลี่ยนแปลงทางธุรกิจได้อย่างรวดเร็ว: ปรับโครงสร้างกระบวนการ, เงื่อนไข หรือการตั้งค่าโหนดอื่นๆ และนำไปใช้งานได้ทันทีโดยไม่ต้องออกเวอร์ชันใหม่ครับ/ค่ะ

## การติดตั้ง

เวิร์กโฟลว์เป็นปลั๊กอินในตัวของ NocoBase จึงไม่จำเป็นต้องติดตั้งหรือตั้งค่าเพิ่มเติมครับ/ค่ะ
