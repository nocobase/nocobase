# บันทึกการตรวจสอบ (Audit Logger)

## บทนำ

บันทึกการตรวจสอบ (Audit Log) ใช้สำหรับบันทึกและติดตามกิจกรรมของผู้ใช้ รวมถึงประวัติการดำเนินการกับทรัพยากรภายในระบบครับ/ค่ะ

![](https://static-docs.nocobase.com/202501031627719.png)

![](https://static-docs.nocobase.com/202501031627922.png)

## คำอธิบายพารามิเตอร์

| พารามิเตอร์             | คำอธิบาย                                                                                                                               |
| --------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| **Resource**          | ประเภททรัพยากรเป้าหมายของการดำเนินการ                                                                                                 |
| **Action**            | ประเภทของการดำเนินการที่ทำ                                                                                                           |
| **User**              | ผู้ใช้งานที่ดำเนินการ                                                                                                         |
| **Role**              | บทบาทของผู้ใช้งานขณะดำเนินการ                                                                                                           |
| **Data source**       | แหล่งข้อมูล                                                                                                                           |
| **Target collection** | คอลเลกชันเป้าหมาย                                                                                                                     |
| **Target record UK**  | ตัวระบุเฉพาะของคอลเลกชันเป้าหมาย                                                                                            |
| **Source collection** | คอลเลกชันต้นทางของฟิลด์ความสัมพันธ์                                                                                            |
| **Source record UK**  | ตัวระบุเฉพาะของคอลเลกชันต้นทาง                                                                                            |
| **Status**            | รหัสสถานะ HTTP ของการตอบกลับคำขอการดำเนินการ                                                                                    |
| **Created at**        | เวลาที่ดำเนินการ                                                                                                           |
| **UUID**              | ตัวระบุเฉพาะของการดำเนินการ ซึ่งสอดคล้องกับ Request ID ของคำขอการดำเนินการ สามารถใช้เพื่อดึงบันทึกแอปพลิเคชันได้ |
| **IP**                | ที่อยู่ IP ของผู้ใช้งาน                                                                                                         |
| **UA**                | ข้อมูล UA ของผู้ใช้งาน                                                                                                            |
| **Metadata**          | เมตาดาต้า เช่น พารามิเตอร์, เนื้อหาคำขอ (request body) และเนื้อหาการตอบกลับ (response content) ของคำขอการดำเนินการ                                                  |

## คำอธิบายทรัพยากรที่ถูกตรวจสอบ

ปัจจุบัน การดำเนินการกับทรัพยากรต่อไปนี้จะถูกบันทึกในบันทึกการตรวจสอบครับ/ค่ะ:

### แอปพลิเคชันหลัก

| Operation        | Description             |
| ---------------- | ----------------------- |
| `app:resart`     | รีสตาร์ทแอปพลิเคชัน     |
| `app:clearCache` | ล้างแคชของแอปพลิเคชัน |

### ตัวจัดการปลั๊กอิน

| Operation    | Description    |
| ------------ | -------------- |
| `pm:add`     | เพิ่มปลั๊กอิน |
| `pm:update`  | อัปเดตปลั๊กอิน |
| `pm:enable`  | เปิดใช้งานปลั๊กอิน |
| `pm:disable` | ปิดใช้งานปลั๊กอิน |
| `pm:remove`  | ลบปลั๊กอิน |

### การยืนยันตัวตนผู้ใช้งาน

| Operation             | Description     |
| --------------------- | --------------- |
| `auth:signIn`         | เข้าสู่ระบบ         |
| `auth:signUp`         | ลงทะเบียน         |
| `auth:signOut`        | ออกจากระบบ        |
| `auth:changePassword` | เปลี่ยนรหัสผ่าน |

### ผู้ใช้งาน

| Operation             | Description    |
| --------------------- | -------------- |
| `users:updateProfile` | แก้ไขโปรไฟล์ส่วนตัว |

### การตั้งค่า UI

| Operation                  | Description      |
| -------------------------- | ---------------- |
| `uiSchemas:insertAdjacent` | แทรก UI Schema |
| `uiSchemas:patch`          | แก้ไข UI Schema |
| `uiSchemas:remove`         | ลบ UI Schema |

### การดำเนินการกับคอลเลกชัน

| Operation        | Description                       |
| ---------------- | --------------------------------- |
| `create`         | สร้างเรคคอร์ด                     |
| `update`         | อัปเดตเรคคอร์ด                     |
| `destroy`        | ลบเรคคอร์ด                     |
| `updateOrCreate` | อัปเดตหรือสร้างเรคคอร์ด           |
| `firstOrCreate`  | ค้นหาหรือสร้างเรคคอร์ด            |
| `move`           | ย้ายเรคคอร์ด                     |
| `set`            | ตั้งค่าเรคคอร์ดฟิลด์ความสัมพันธ์ |
| `add`            | เพิ่มเรคคอร์ดฟิลด์ความสัมพันธ์ |
| `remove`         | ลบเรคคอร์ดฟิลด์ความสัมพันธ์ |
| `export`         | ส่งออกเรคคอร์ด                     |
| `import`         | นำเข้าเรคคอร์ด                     |

## การเพิ่มทรัพยากรที่ถูกตรวจสอบอื่นๆ

หากคุณได้ขยายการดำเนินการกับทรัพยากรอื่นๆ ผ่านปลั๊กอิน และต้องการบันทึกพฤติกรรมการดำเนินการกับทรัพยากรเหล่านี้ลงในบันทึกการตรวจสอบ สามารถอ้างอิงได้จาก [API](/api/server/audit-manager.md) ครับ/ค่ะ
