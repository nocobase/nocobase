# การล็อกผู้ใช้

## บทนำ

คุณสามารถจัดการผู้ใช้ที่ถูกล็อกและล็อกผู้ใช้ได้ด้วยตนเองครับ/ค่ะ

![](https://static-docs.nocobase.com/202412281450083.png)

### ปลดล็อกผู้ใช้

ผู้ใช้ที่ถูกล็อกเนื่องจากพยายามเข้าสู่ระบบด้วยรหัสผ่านไม่ถูกต้องเกินจำนวนครั้งที่กำหนด สามารถปลดล็อกได้โดยการลบข้อมูลบันทึกนั้นออกครับ/ค่ะ

### ล็อกผู้ใช้

การเพิ่มข้อมูลบันทึกผู้ใช้จะช่วยให้คุณสามารถล็อกผู้ใช้ได้ด้วยตนเองครับ/ค่ะ เมื่อผู้ใช้ถูกล็อกแล้ว จะไม่สามารถเข้าถึงระบบได้ด้วยวิธีการยืนยันตัวตนใดๆ เลย รวมถึง API keys ด้วยครับ/ค่ะ

![](https://static-docs.nocobase.com/202412281450512.png)
