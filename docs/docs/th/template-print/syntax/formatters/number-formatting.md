### การจัดรูปแบบตัวเลข

#### 1. :formatN(precision)

##### คำอธิบายไวยากรณ์
จัดรูปแบบตัวเลขตามการตั้งค่าท้องถิ่น (Localization) ครับ/ค่ะ  
พารามิเตอร์:
- precision: จำนวนตำแหน่งทศนิยม  
  สำหรับรูปแบบ ODS/XLSX จำนวนตำแหน่งทศนิยมที่แสดงจะถูกกำหนดโดยโปรแกรมแก้ไขข้อความ ส่วนรูปแบบอื่น ๆ จะขึ้นอยู่กับพารามิเตอร์นี้ครับ/ค่ะ

##### ตัวอย่าง
```
'10':formatN()         // ผลลัพธ์ "10.000"
'1000.456':formatN()   // ผลลัพธ์ "1,000.456"
```

##### ผลลัพธ์
ตัวเลขจะถูกแสดงผลตามความแม่นยำและรูปแบบท้องถิ่นที่ระบุครับ/ค่ะ


#### 2. :round(precision)

##### คำอธิบายไวยากรณ์
ทำการปัดเศษตัวเลข (Round) โดยพารามิเตอร์จะระบุจำนวนตำแหน่งทศนิยมครับ/ค่ะ

##### ตัวอย่าง
```
10.05123:round(2)      // ผลลัพธ์ 10.05
1.05:round(1)          // ผลลัพธ์ 1.1
```

##### ผลลัพธ์
ผลลัพธ์คือค่าที่ผ่านการปัดเศษแล้วครับ/ค่ะ


#### 3. :add(value)

##### คำอธิบายไวยากรณ์
นำตัวเลขปัจจุบันมาบวกกับค่าที่ระบุครับ/ค่ะ  
พารามิเตอร์:
- value: ตัวบวก

##### ตัวอย่าง
```
1000.4:add(2)         // ผลลัพธ์ 1002.4
'1000.4':add('2')      // ผลลัพธ์ 1002.4
```

##### ผลลัพธ์
ผลลัพธ์คือค่าที่ได้จากการบวกครับ/ค่ะ


#### 4. :sub(value)

##### คำอธิบายไวยากรณ์
นำตัวเลขปัจจุบันมาลบด้วยค่าที่ระบุครับ/ค่ะ  
พารามิเตอร์:
- value: ตัวลบ

##### ตัวอย่าง
```
1000.4:sub(2)         // ผลลัพธ์ 998.4
'1000.4':sub('2')      // ผลลัพธ์ 998.4
```

##### ผลลัพธ์
ผลลัพธ์คือค่าที่ได้จากการลบครับ/ค่ะ


#### 5. :mul(value)

##### คำอธิบายไวยากรณ์
นำตัวเลขปัจจุบันมาคูณกับค่าที่ระบุครับ/ค่ะ  
พารามิเตอร์:
- value: ตัวคูณ

##### ตัวอย่าง
```
1000.4:mul(2)         // ผลลัพธ์ 2000.8
'1000.4':mul('2')      // ผลลัพธ์ 2000.8
```

##### ผลลัพธ์
ผลลัพธ์คือค่าที่ได้จากการคูณครับ/ค่ะ


#### 6. :div(value)

##### คำอธิบายไวยากรณ์
นำตัวเลขปัจจุบันมาหารด้วยค่าที่ระบุครับ/ค่ะ  
พารามิเตอร์:
- value: ตัวหาร

##### ตัวอย่าง
```
1000.4:div(2)         // ผลลัพธ์ 500.2
'1000.4':div('2')      // ผลลัพธ์ 500.2
```

##### ผลลัพธ์
ผลลัพธ์คือค่าที่ได้จากการหารครับ/ค่ะ


#### 7. :mod(value)

##### คำอธิบายไวยากรณ์
คำนวณค่ามอดุโล (หาเศษเหลือ) ของตัวเลขปัจจุบันเทียบกับค่าที่ระบุครับ/ค่ะ  
พารามิเตอร์:
- value: ตัวมอดุโล

##### ตัวอย่าง
```
4:mod(2)              // ผลลัพธ์ 0
3:mod(2)              // ผลลัพธ์ 1
```

##### ผลลัพธ์
ผลลัพธ์คือค่าที่ได้จากการดำเนินการมอดุโลครับ/ค่ะ


#### 8. :abs

##### คำอธิบายไวยากรณ์
ส่งคืนค่าสัมบูรณ์ของตัวเลขครับ/ค่ะ

##### ตัวอย่าง
```
-10:abs()             // ผลลัพธ์ 10
-10.54:abs()          // ผลลัพธ์ 10.54
10.54:abs()           // ผลลัพธ์ 10.54
'-200':abs()          // ผลลัพธ์ 200
```

##### ผลลัพธ์
ผลลัพธ์คือค่าสัมบูรณ์ครับ/ค่ะ


#### 9. :ceil

##### คำอธิบายไวยากรณ์
ปัดเศษขึ้น คือการส่งคืนจำนวนเต็มที่น้อยที่สุดที่มีค่ามากกว่าหรือเท่ากับตัวเลขปัจจุบันครับ/ค่ะ

##### ตัวอย่าง
```
10.05123:ceil()       // ผลลัพธ์ 11
1.05:ceil()           // ผลลัพธ์ 2
-1.05:ceil()          // ผลลัพธ์ -1
```

##### ผลลัพธ์
ผลลัพธ์คือจำนวนเต็มที่ได้จากการปัดเศษขึ้นครับ/ค่ะ


#### 10. :floor

##### คำอธิบายไวยากรณ์
ปัดเศษลง คือการส่งคืนจำนวนเต็มที่มากที่สุดที่มีค่าน้อยกว่าหรือเท่ากับตัวเลขปัจจุบันครับ/ค่ะ

##### ตัวอย่าง
```
10.05123:floor()      // ผลลัพธ์ 10
1.05:floor()          // ผลลัพธ์ 1
-1.05:floor()         // ผลลัพธ์ -2
```

##### ผลลัพธ์
ผลลัพธ์คือจำนวนเต็มที่ได้จากการปัดเศษลงครับ/ค่ะ


#### 11. :int

##### คำอธิบายไวยากรณ์
แปลงตัวเลขเป็นจำนวนเต็ม (ไม่แนะนำให้ใช้งานครับ/ค่ะ)

##### ตัวอย่างและผลลัพธ์
ขึ้นอยู่กับสถานการณ์การแปลงที่เฉพาะเจาะจงครับ/ค่ะ


#### 12. :toEN

##### คำอธิบายไวยากรณ์
แปลงตัวเลขเป็นรูปแบบภาษาอังกฤษ (จุดทศนิยมคือ '.') ไม่แนะนำให้ใช้งานครับ/ค่ะ

##### ตัวอย่างและผลลัพธ์
ขึ้นอยู่กับสถานการณ์การแปลงที่เฉพาะเจาะจงครับ/ค่ะ


#### 13. :toFixed

##### คำอธิบายไวยากรณ์
แปลงตัวเลขเป็นสตริง โดยคงไว้เพียงจำนวนตำแหน่งทศนิยมที่ระบุ ไม่แนะนำให้ใช้งานครับ/ค่ะ

##### ตัวอย่างและผลลัพธ์
ขึ้นอยู่กับสถานการณ์การแปลงที่เฉพาะเจาะจงครับ/ค่ะ


#### 14. :toFR

##### คำอธิบายไวยากรณ์
แปลงตัวเลขเป็นรูปแบบภาษาฝรั่งเศส (จุดทศนิยมคือ ',') ไม่แนะนำให้ใช้งานครับ/ค่ะ

##### ตัวอย่างและผลลัพธ์
ขึ้นอยู่กับสถานการณ์การแปลงที่เฉพาะเจาะจงครับ/ค่ะ
