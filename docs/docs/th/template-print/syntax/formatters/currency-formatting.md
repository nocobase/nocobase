### การจัดรูปแบบสกุลเงิน

#### 1. :formatC(precisionOrFormat, targetCurrency)

##### คำอธิบายไวยากรณ์
ฟังก์ชันนี้ใช้สำหรับจัดรูปแบบตัวเลขสกุลเงิน โดยคุณสามารถระบุจำนวนทศนิยมหรือรูปแบบการแสดงผลที่ต้องการได้ครับ/ค่ะ
พารามิเตอร์:
- `precisionOrFormat`: เป็นพารามิเตอร์เสริมที่สามารถเป็นได้ทั้งตัวเลข (เพื่อระบุจำนวนทศนิยม) หรือตัวระบุรูปแบบดังนี้ครับ/ค่ะ:
  - ตัวเลขจำนวนเต็ม: ใช้เพื่อเปลี่ยนความแม่นยำของทศนิยมเริ่มต้น
  - `'M'`: แสดงเฉพาะชื่อสกุลเงินหลักเท่านั้น
  - `'L'`: แสดงตัวเลขพร้อมสัญลักษณ์สกุลเงิน (ค่าเริ่มต้น)
  - `'LL'`: แสดงตัวเลขพร้อมชื่อสกุลเงินหลัก
- `targetCurrency`: เป็นพารามิเตอร์เสริมสำหรับรหัสสกุลเงินเป้าหมาย (ตัวพิมพ์ใหญ่ เช่น USD, EUR) ซึ่งจะใช้แทนที่การตั้งค่าส่วนกลางครับ/ค่ะ

##### ตัวอย่าง
```
// ตัวอย่างสภาพแวดล้อม: ตัวเลือก API { "lang": "en-us", "currency": { "source": "EUR", "target": "USD", "rates": { "EUR": 1, "USD": 2 } } }
'1000.456':formatC()      // ผลลัพธ์ "$2,000.91"
'1000.456':formatC('M')    // ผลลัพธ์ "dollars"
'1':formatC('M')           // ผลลัพธ์ "dollar"
'1000':formatC('L')        // ผลลัพธ์ "$2,000.00"
'1000':formatC('LL')       // ผลลัพธ์ "2,000.00 dollars"

// ตัวอย่างภาษาฝรั่งเศส (เมื่อการตั้งค่าสภาพแวดล้อมแตกต่างกัน):
'1000.456':formatC()      // ผลลัพธ์ "2 000,91 ..."  
'1000.456':formatC()      // เมื่อสกุลเงินต้นทางและปลายทางเหมือนกัน จะได้ผลลัพธ์ "1 000,46 €"
```

##### ผลลัพธ์
ผลลัพธ์ที่ได้จะขึ้นอยู่กับตัวเลือก API และการตั้งค่าอัตราแลกเปลี่ยนครับ/ค่ะ


#### 2. :convCurr(target, source)

##### คำอธิบายไวยากรณ์
ฟังก์ชันนี้ใช้สำหรับแปลงตัวเลขจากสกุลเงินหนึ่งไปยังอีกสกุลเงินหนึ่งครับ/ค่ะ โดยอัตราแลกเปลี่ยนสามารถส่งผ่านทางตัวเลือก API หรือตั้งค่าแบบส่วนกลางได้
หากคุณไม่ได้ระบุพารามิเตอร์ใดๆ การแปลงจะถูกดำเนินการโดยอัตโนมัติจาก `options.currencySource` ไปยัง `options.currencyTarget` ครับ/ค่ะ
พารามิเตอร์:
- `target`: เป็นพารามิเตอร์เสริมสำหรับรหัสสกุลเงินเป้าหมาย (ค่าเริ่มต้นคือ `options.currencyTarget`)
- `source`: เป็นพารามิเตอร์เสริมสำหรับรหัสสกุลเงินต้นทาง (ค่าเริ่มต้นคือ `options.currencySource`)

##### ตัวอย่าง
```
// ตัวอย่างสภาพแวดล้อม: ตัวเลือก API { "currency": { "source": "EUR", "target": "USD", "rates": { "EUR": 1, "USD": 2 } } }
10:convCurr()              // ผลลัพธ์ 20
1000:convCurr()            // ผลลัพธ์ 2000
1000:convCurr('EUR')        // ผลลัพธ์ 1000
1000:convCurr('USD')        // ผลลัพธ์ 2000
1000:convCurr('USD', 'USD') // ผลลัพธ์ 1000
```

##### ผลลัพธ์
ผลลัพธ์ที่ได้คือค่าสกุลเงินที่ถูกแปลงแล้วครับ/ค่ะ
