### การจัดรูปแบบสกุลเงิน

#### 1. :formatC(precisionOrFormat, targetCurrency)

##### คำอธิบายไวยากรณ์
จัดรูปแบบตัวเลขสกุลเงิน สามารถระบุจำนวนตำแหน่งทศนิยมหรือรูปแบบเอาต์พุตเฉพาะได้ครับ/ค่ะ  
พารามิเตอร์:
- precisionOrFormat：พารามิเตอร์เสริม สามารถเป็นได้ทั้งตัวเลข (ระบุจำนวนตำแหน่งทศนิยม) หรือตัวระบุรูปแบบเฉพาะ:
  - จำนวนเต็ม: เปลี่ยนความแม่นยำของทศนิยมเริ่มต้น
  - `'M'`：แสดงเฉพาะชื่อสกุลเงินหลักเท่านั้น
  - `'L'`：แสดงตัวเลขพร้อมสัญลักษณ์สกุลเงิน (ค่าเริ่มต้น)
  - `'LL'`：แสดงตัวเลขพร้อมชื่อสกุลเงินหลัก
- targetCurrency：พารามิเตอร์เสริม รหัสสกุลเงินเป้าหมาย (ตัวพิมพ์ใหญ่ เช่น USD, EUR) จะแทนที่การตั้งค่าส่วนกลางครับ/ค่ะ

##### ตัวอย่าง
```
'1000.456':formatC()      // เอาต์พุต "$2,000.91"
'1000.456':formatC('M')    // เอาต์พุต "dollars"
'1':formatC('M')           // เอาต์พุต "dollar"
'1000':formatC('L')        // เอาต์พุต "$2,000.00"
'1000':formatC('LL')       // เอาต์พุต "2,000.00 dollars"
```

##### ผลลัพธ์
ผลลัพธ์ขึ้นอยู่กับตัวเลือก API และการตั้งค่าอัตราแลกเปลี่ยนครับ/ค่ะ


#### 2. :convCurr(target, source)

##### คำอธิบายไวยากรณ์
แปลงตัวเลขจากสกุลเงินหนึ่งไปยังอีกสกุลเงินหนึ่ง อัตราแลกเปลี่ยนสามารถส่งผ่านตัวเลือก API หรือตั้งค่าแบบส่วนกลางได้ครับ/ค่ะ  
หากไม่ระบุพารามิเตอร์ จะทำการแปลงจาก `options.currencySource` ไปยัง `options.currencyTarget` โดยอัตโนมัติครับ/ค่ะ  
พารามิเตอร์:
- target：พารามิเตอร์เสริม รหัสสกุลเงินเป้าหมาย (ค่าเริ่มต้นเท่ากับ `options.currencyTarget`)
- source：พารามิเตอร์เสริม รหัสสกุลเงินต้นทาง (ค่าเริ่มต้นเท่ากับ `options.currencySource`)

##### ตัวอย่าง
```
10:convCurr()              // เอาต์พุต 20
1000:convCurr()            // เอาต์พุต 2000
1000:convCurr('EUR')        // เอาต์พุต 1000
1000:convCurr('USD')        // เอาต์พุต 2000
1000:convCurr('USD', 'USD') // เอาต์พุต 1000
```

##### ผลลัพธ์
ผลลัพธ์คือค่าสกุลเงินที่แปลงแล้วครับ/ค่ะ
