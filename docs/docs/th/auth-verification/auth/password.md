## ลืมรหัสผ่าน<Badge>v1.8.0+</Badge>

ฟังก์ชันลืมรหัสผ่านช่วยให้ผู้ใช้สามารถตั้งรหัสผ่านใหม่ได้โดยการยืนยันตัวตนผ่านอีเมล ในกรณีที่ลืมรหัสผ่านครับ/ค่ะ

### การตั้งค่าสำหรับผู้ดูแลระบบ

1.  **เปิดใช้งานฟังก์ชันลืมรหัสผ่าน**

    ในแท็บ "ตั้งค่า" > "การยืนยันตัวตนผู้ใช้" > "ลืมรหัสผ่าน" ให้เลือกช่องทำเครื่องหมาย "เปิดใช้งานฟังก์ชันลืมรหัสผ่าน" ครับ/ค่ะ

    ![20250423071957_rec_](https://static-docs.nocobase.com/20250423071957_rec_.gif)

2.  **ตั้งค่าช่องทางการแจ้งเตือน**

    เลือกช่องทางการแจ้งเตือนทางอีเมล (ปัจจุบันรองรับเฉพาะอีเมล) หากไม่มีช่องทางการแจ้งเตือนที่ใช้งานได้ คุณต้องเพิ่มช่องทางใหม่ก่อนครับ/ค่ะ

    ![20250423072225_rec_](https://static-docs.nocobase.com/20250423072225_rec_.gif)

3.  **ตั้งค่าอีเมลสำหรับรีเซ็ตรหัสผ่าน**

    คุณสามารถปรับแต่งหัวข้อและเนื้อหาอีเมลได้ โดยรองรับทั้งรูปแบบ HTML และข้อความธรรมดา (Plain Text) ครับ/ค่ะ สามารถใช้ตัวแปรต่อไปนี้ได้:
    - ผู้ใช้ปัจจุบัน (Current user)
    - การตั้งค่าระบบ (System settings)
    - ลิงก์สำหรับรีเซ็ตรหัสผ่าน (Reset password link)
    - ระยะเวลาที่ลิงก์รีเซ็ตมีผล (นาที) (Reset link expiration (minutes))

    ![20250427170047](https://static-docs.nocobase.com/20250427170047.png)

4.  **กำหนดระยะเวลาที่ลิงก์รีเซ็ตมีผล**

    กำหนดระยะเวลาที่ลิงก์รีเซ็ตมีผลเป็นนาที โดยค่าเริ่มต้นคือ 120 นาทีครับ/ค่ะ

    ![20250423073557](https://static-docs.nocobase.com/20250423073557.png)

### ขั้นตอนการใช้งานสำหรับผู้ใช้

1.  **เริ่มกระบวนการขอรีเซ็ตรหัสผ่าน**

    ที่หน้าเข้าสู่ระบบ ให้คลิกลิงก์ "ลืมรหัสผ่าน" (ผู้ดูแลระบบต้องเปิดใช้งานฟังก์ชันลืมรหัสผ่านก่อน) เพื่อไปยังหน้าลืมรหัสผ่านครับ/ค่ะ

    ![20250421103458_rec_](https://static-docs.nocobase.com/20250421103458_rec_.gif)

    กรอกอีเมลที่ลงทะเบียนไว้ แล้วคลิกปุ่ม "ส่งอีเมลรีเซ็ต" ครับ/ค่ะ

    ![20250421113442_rec_](https://static-docs.nocobase.com/20250421113442_rec_.gif)

2.  **รีเซ็ตรหัสผ่าน**

    ผู้ใช้จะได้รับอีเมลที่มีลิงก์สำหรับรีเซ็ตรหัสผ่าน เมื่อคลิกลิงก์แล้ว ระบบจะเปิดหน้าให้คุณตั้งรหัสผ่านใหม่ครับ/ค่ะ

    ![20250421113748](https://static-docs.nocobase.com/20250421113748.png)

    เมื่อตั้งค่าเรียบร้อยแล้ว ผู้ใช้สามารถเข้าสู่ระบบด้วยรหัสผ่านใหม่ได้ทันทีครับ/ค่ะ

### ข้อควรทราบ

- ลิงก์รีเซ็ตมีเวลาจำกัด โดยปกติจะมีอายุ 120 นาทีหลังจากสร้าง (ผู้ดูแลระบบสามารถกำหนดค่าได้)
- ลิงก์สามารถใช้ได้เพียงครั้งเดียว และจะหมดอายุทันทีหลังจากใช้งาน
- หากผู้ใช้ไม่ได้รับอีเมลรีเซ็ต โปรดตรวจสอบว่าที่อยู่อีเมลถูกต้องหรือไม่ หรือลองตรวจสอบในโฟลเดอร์สแปมครับ/ค่ะ
- ผู้ดูแลระบบควรตรวจสอบให้แน่ใจว่าการตั้งค่าเซิร์ฟเวอร์อีเมลถูกต้อง เพื่อให้มั่นใจว่าอีเมลรีเซ็ตสามารถส่งได้สำเร็จครับ/ค่ะ
