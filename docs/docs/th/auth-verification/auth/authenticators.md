# การจัดการ Authenticator

## การจัดการการยืนยันตัวตนผู้ใช้

เมื่อติดตั้งปลั๊กอินการยืนยันตัวตนผู้ใช้ ระบบจะเริ่มต้นวิธีการยืนยันตัวตนแบบ `รหัสผ่าน` โดยอ้างอิงจากชื่อผู้ใช้และอีเมลของผู้ใช้ครับ

![](https://static-docs.nocobase.com/66eaa9d5421c9cb713b117366bd8a5d5.png)

## การเปิดใช้งาน Authenticator

![](https://static-docs.nocobase.com/7f1b8f8ca5de67ffc68eff0a65848f5.png)

เฉพาะประเภทการยืนยันตัวตนที่เปิดใช้งานแล้วเท่านั้นที่จะแสดงบนหน้าเข้าสู่ระบบครับ

![](https://static-docs.nocobase.com/8375a36ef98417af0f0977f1e07345dd.png)

## ประเภทการยืนยันตัวตนผู้ใช้

![](https://static-docs.nocobase.com/da4250c0cea343ebe470cbf7be4b12e4.png)

ด้วยการเพิ่ม Authenticator ประเภทต่างๆ คุณสามารถเพิ่มวิธีการยืนยันตัวตนที่เกี่ยวข้องให้กับระบบได้ครับ

นอกเหนือจากประเภทการยืนยันตัวตนที่ปลั๊กอินที่มีอยู่ให้มาแล้ว นักพัฒนายังสามารถขยายประเภทการยืนยันตัวตนผู้ใช้ได้เองอีกด้วยครับ สำหรับรายละเอียด โปรดดูที่ [คู่มือสำหรับนักพัฒนา](./dev/)
