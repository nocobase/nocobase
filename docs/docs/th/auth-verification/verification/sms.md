# การยืนยัน: SMS

## บทนำ

รหัสยืนยัน SMS เป็นประเภทการยืนยันตัวตนที่มีมาให้ในระบบ ใช้สำหรับสร้างรหัสผ่านแบบใช้ครั้งเดียว (OTP) และส่งให้ผู้ใช้ทาง SMS ครับ/ค่ะ

## การเพิ่มตัวยืนยัน SMS

ไปที่หน้าการจัดการการยืนยันตัวตนครับ/ค่ะ

## การตั้งค่าสำหรับผู้ดูแลระบบ

![](https://static-docs.nocobase.com/202502271727711.png)

ปัจจุบัน ผู้ให้บริการ SMS ที่รองรับมีดังนี้ครับ/ค่ะ:

- <a href="https://www.aliyun.com/product/sms" target="_blank">Aliyun SMS</a>
- <a href="https://cloud.tencent.com/product/sms" target="_blank">Tencent Cloud SMS</a>

เมื่อตั้งค่าเทมเพลต SMS ในหน้าจัดการของผู้ให้บริการ คุณต้องสำรองพารามิเตอร์สำหรับรหัสยืนยันไว้ด้วยครับ/ค่ะ

- ตัวอย่างการตั้งค่า Aliyun: `รหัสยืนยันของคุณคือ: ${code}`

- ตัวอย่างการตั้งค่า Tencent Cloud: `รหัสยืนยันของคุณคือ: {1}`

นักพัฒนาสามารถขยายการรองรับผู้ให้บริการ SMS อื่นๆ ในรูปแบบของปลั๊กอินได้ครับ/ค่ะ ดูเพิ่มเติมที่: [การขยายการรองรับผู้ให้บริการ SMS](./dev/sms-type)

## การผูกเบอร์โทรศัพท์ของผู้ใช้

หลังจากเพิ่มตัวยืนยันแล้ว ผู้ใช้สามารถผูกเบอร์โทรศัพท์ในหน้าจัดการการยืนยันตัวตนส่วนตัวของตนเองได้ครับ/ค่ะ

![](https://static-docs.nocobase.com/202502271737016.png)

![](https://static-docs.nocobase.com/202502271737769.png)

![](https://static-docs.nocobase.com/202502271738515.png)

เมื่อผูกเบอร์โทรศัพท์สำเร็จแล้ว จะสามารถทำการยืนยันตัวตนในสถานการณ์ใดๆ ที่ใช้ตัวยืนยันนี้ได้ครับ/ค่ะ

![](https://static-docs.nocobase.com/202502271739607.png)

## การยกเลิกการผูกเบอร์โทรศัพท์ของผู้ใช้

การยกเลิกการผูกเบอร์โทรศัพท์จำเป็นต้องมีการยืนยันตัวตนผ่านวิธีการยืนยันตัวตนที่ผูกไว้แล้วครับ/ค่ะ

![](https://static-docs.nocobase.com/202502282103205.png)
