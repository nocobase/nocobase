# การยืนยันตัวตนด้วย SMS

## บทนำ

ปลั๊กอินการยืนยันตัวตนด้วย SMS ช่วยให้ผู้ใช้สามารถลงทะเบียนและเข้าสู่ระบบ NocoBase โดยใช้ SMS ได้ครับ/ค่ะ

> ต้องใช้งานร่วมกับฟังก์ชันรหัสยืนยัน SMS ที่มาจากปลั๊กอิน [`@nocobase/plugin-verification`](/auth-verification/verification/) ครับ/ค่ะ

## การตั้งค่าเวอร์ชันใหม่

:::info{title=ข้อควรทราบ}
การตั้งค่าเวอร์ชันใหม่นี้เริ่มนำมาใช้ตั้งแต่เวอร์ชัน `1.6.0-alpha.30` และมีแผนจะรองรับอย่างเสถียรตั้งแต่เวอร์ชัน `1.7.0` เป็นต้นไปครับ/ค่ะ
:::

![](https://static-docs.nocobase.com/202502282114821.png)

**Verificator:** ผูก Verificator SMS เพื่อใช้ส่งรหัสยืนยัน SMS ครับ/ค่ะ หากไม่มี Verificator ที่ใช้งานได้ คุณจะต้องไปที่หน้าจัดการการยืนยันเพื่อสร้าง Verificator SMS ก่อนครับ/ค่ะ
ดูเพิ่มเติม:

- [การยืนยัน](../verification/index.md)
- [การยืนยัน: SMS](../verification/sms/index.md)

**ลงทะเบียนอัตโนมัติเมื่อผู้ใช้ไม่มีอยู่ (Sign up automatically when the user does not exist):** เมื่อเลือกตัวเลือกนี้ หากเบอร์โทรศัพท์มือถือที่ผู้ใช้ใช้ไม่มีอยู่ในระบบ ระบบจะลงทะเบียนผู้ใช้ใหม่โดยใช้เบอร์โทรศัพท์มือถือนั้นเป็นชื่อเล่นครับ/ค่ะ

## การตั้งค่าเวอร์ชันเก่า

![](https://static-docs.nocobase.com/a4d35ec63ba22ae2ea9e3e8e1cbb783d.png)

ฟังก์ชันการยืนยันตัวตนเพื่อเข้าสู่ระบบด้วย SMS จะใช้ Provider รหัสยืนยัน SMS ที่ตั้งค่าไว้และถูกกำหนดให้เป็นค่าเริ่มต้นในการส่งข้อความ SMS ครับ/ค่ะ

**ลงทะเบียนอัตโนมัติเมื่อผู้ใช้ไม่มีอยู่ (Sign up automatically when the user does not exist):** เมื่อเลือกตัวเลือกนี้ หากเบอร์โทรศัพท์มือถือที่ผู้ใช้ใช้ไม่มีอยู่ในระบบ ระบบจะลงทะเบียนผู้ใช้ใหม่โดยใช้เบอร์โทรศัพท์มือถือนั้นเป็นชื่อเล่นครับ/ค่ะ
