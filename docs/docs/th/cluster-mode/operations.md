# ขั้นตอนการดูแลระบบ

## การเริ่มต้นใช้งานแอปพลิเคชันครั้งแรก

เมื่อเริ่มต้นใช้งานแอปพลิเคชันเป็นครั้งแรก ควรเริ่มจากเปิดใช้งานหนึ่งโหนดก่อนครับ/ค่ะ รอจนกว่าปลั๊กอินจะติดตั้งและเปิดใช้งานเรียบร้อยแล้ว จึงค่อยเปิดใช้งานโหนดอื่น ๆ ต่อไปครับ/ค่ะ

## การอัปเกรดเวอร์ชัน

เมื่อคุณต้องการอัปเกรดเวอร์ชันของ NocoBase โปรดดูขั้นตอนการดำเนินการนี้ครับ/ค่ะ

:::warning{title=ข้อควรระวัง}
ใน**สภาพแวดล้อมการทำงานจริง (Production Environment)** แบบคลัสเตอร์ ควรใช้ฟังก์ชันต่าง ๆ เช่น การจัดการปลั๊กอินและการอัปเกรดเวอร์ชันด้วยความระมัดระวัง หรือห้ามใช้งานโดยเด็ดขาดครับ/ค่ะ

NocoBase ยังไม่รองรับการอัปเกรดเวอร์ชันแบบออนไลน์สำหรับคลัสเตอร์ในขณะนี้ครับ/ค่ะ เพื่อให้มั่นใจในความสอดคล้องของข้อมูล จึงจำเป็นต้องระงับบริการภายนอกชั่วคราวในระหว่างกระบวนการอัปเกรดครับ/ค่ะ
:::

ขั้นตอนการดำเนินการ:

1.  หยุดบริการปัจจุบัน

    หยุดการทำงานของ NocoBase แอปพลิเคชันอินสแตนซ์ทั้งหมด และส่งต่อทราฟฟิกของ Load Balancer ไปยังหน้าสถานะ 503 ครับ/ค่ะ

2.  สำรองข้อมูล

    ก่อนการอัปเกรด ขอแนะนำอย่างยิ่งให้สำรองข้อมูลฐานข้อมูลไว้ เพื่อป้องกันปัญหาที่อาจเกิดขึ้นระหว่างกระบวนการอัปเกรดครับ/ค่ะ

3.  อัปเดตเวอร์ชัน

    โปรดดู [การอัปเกรด Docker](../get-started/upgrading/docker) เพื่ออัปเดตเวอร์ชันของอิมเมจแอปพลิเคชัน NocoBase ครับ/ค่ะ

4.  เริ่มบริการ

    1.  เริ่มต้นใช้งานหนึ่งโหนดในคลัสเตอร์ และรอจนกว่าการอัปเดตจะเสร็จสมบูรณ์และโหนดเริ่มทำงานได้สำเร็จครับ/ค่ะ
    2.  ตรวจสอบว่าฟังก์ชันการทำงานถูกต้อง หากพบความผิดปกติและไม่สามารถแก้ไขได้ คุณสามารถย้อนกลับไปยังเวอร์ชันก่อนหน้าได้ครับ/ค่ะ
    3.  เริ่มต้นใช้งานโหนดอื่น ๆ
    4.  โอนย้ายทราฟฟิกของ Load Balancer ไปยังคลัสเตอร์แอปพลิเคชัน

## การบำรุงรักษาภายในแอปพลิเคชัน

การบำรุงรักษาภายในแอปพลิเคชัน หมายถึง การดำเนินการที่เกี่ยวข้องกับการบำรุงรักษาในขณะที่แอปพลิเคชันกำลังทำงานอยู่ ซึ่งรวมถึง:

*   การจัดการปลั๊กอิน (การติดตั้ง, การเปิดใช้งาน, การปิดใช้งานปลั๊กอิน เป็นต้น)
*   การสำรองและกู้คืนข้อมูล
*   การจัดการการย้ายสภาพแวดล้อม

ขั้นตอนการดำเนินการ:

1.  ลดจำนวนโหนด

    ลดจำนวนโหนดที่กำลังรันแอปพลิเคชันในคลัสเตอร์ให้เหลือเพียง 1 โหนด และหยุดบริการของโหนดอื่น ๆ ครับ/ค่ะ

2.  ดำเนินการบำรุงรักษาภายในแอปพลิเคชัน เช่น การติดตั้งและเปิดใช้งานปลั๊กอิน, การสำรองข้อมูล เป็นต้น

3.  กู้คืนโหนด

    หลังจากดำเนินการบำรุงรักษาเสร็จสิ้นและตรวจสอบฟังก์ชันการทำงานว่าถูกต้องแล้ว ให้เริ่มต้นใช้งานโหนดอื่น ๆ ครับ/ค่ะ เมื่อโหนดเริ่มทำงานได้สำเร็จแล้ว ให้กู้คืนสถานะการทำงานของคลัสเตอร์ต่อไปครับ/ค่ะ
