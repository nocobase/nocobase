# คำถามที่พบบ่อย

## การเลือกแผนภูมิ
### ควรเลือกแผนภูมิแบบไหนดีครับ/คะ?
ตอบ: เลือกตามเป้าหมายของข้อมูลของคุณได้เลยครับ/ค่ะ:
- แนวโน้มและการเปลี่ยนแปลง: แผนภูมิเส้น หรือ แผนภูมิพื้นที่
- การเปรียบเทียบค่า: แผนภูมิแท่ง (แนวตั้ง) หรือ แผนภูมิแท่ง (แนวนอน)
- สัดส่วนองค์ประกอบ: แผนภูมิวงกลม หรือ แผนภูมิโดนัท
- ความสัมพันธ์และการกระจายตัว: แผนภูมิกระจาย
- โครงสร้างลำดับชั้นและความคืบหน้าของขั้นตอน: แผนภูมิกรวย

สำหรับประเภทแผนภูมิเพิ่มเติม สามารถดูได้ที่ [ตัวอย่าง ECharts](https://echarts.apache.org/examples) ครับ/ค่ะ

### NocoBase รองรับแผนภูมิประเภทใดบ้างครับ/คะ?
ตอบ: ในโหมดการตั้งค่าแบบแสดงผล (Visual mode) จะมีแผนภูมิที่ใช้บ่อยในตัว เช่น แผนภูมิเส้น, แผนภูมิพื้นที่, แผนภูมิแท่ง (แนวตั้ง), แผนภูมิแท่ง (แนวนอน), แผนภูมิวงกลม, แผนภูมิโดนัท, แผนภูมิกรวย, แผนภูมิกระจาย และอื่น ๆ ส่วนในโหมดการตั้งค่าแบบกำหนดเอง (Custom mode) คุณสามารถใช้แผนภูมิได้ทุกประเภทของ ECharts ครับ/ค่ะ

## ปัญหาเกี่ยวกับการสอบถามข้อมูล
### การตั้งค่าแบบแสดงผล (Visual mode) และ การตั้งค่า SQL (SQL mode) ใช้การตั้งค่าร่วมกันได้ไหมครับ/คะ?
ตอบ: ไม่สามารถใช้ร่วมกันได้ครับ/ค่ะ การตั้งค่าจะถูกจัดเก็บแยกกัน โหมดการตั้งค่าที่คุณบันทึกครั้งล่าสุดจะมีผลบังคับใช้ครับ/ค่ะ

## ปัญหาเกี่ยวกับตัวเลือกแผนภูมิ
### ควรตั้งค่าฟิลด์แผนภูมิอย่างไรครับ/คะ?
ตอบ: ในโหมดการตั้งค่าแบบแสดงผล (Visual mode) ให้เลือกฟิลด์ข้อมูลที่เหมาะสมกับประเภทแผนภูมิครับ/ค่ะ เช่น แผนภูมิเส้นหรือแผนภูมิแท่ง (แนวตั้ง) จะต้องตั้งค่าฟิลด์สำหรับแกน X และแกน Y ส่วนแผนภูมิวงกลมจะต้องตั้งค่าฟิลด์สำหรับหมวดหมู่และฟิลด์สำหรับค่า

แนะนำให้ลองคลิก `เรียกใช้การสอบถาม (Run query)` ดูก่อน เพื่อตรวจสอบว่าข้อมูลเป็นไปตามที่คุณต้องการหรือไม่ โดยปกติแล้ว ระบบจะจับคู่ฟิลด์แผนภูมิให้โดยอัตโนมัติครับ/ค่ะ

## ปัญหาเกี่ยวกับการแสดงตัวอย่างและการบันทึก
### หลังจากแก้ไขการตั้งค่าแล้ว ต้องแสดงตัวอย่างด้วยตัวเองไหมครับ/คะ?
ตอบ: ในโหมดการตั้งค่าแบบแสดงผล (Visual mode) เมื่อแก้ไขการตั้งค่าแล้ว ระบบจะแสดงตัวอย่างให้โดยอัตโนมัติครับ/ค่ะ แต่ในโหมด SQL และโหมดการตั้งค่าแบบกำหนดเอง (Custom mode) เพื่อหลีกเลี่ยงการรีเฟรชบ่อยครั้ง เมื่อเขียนเสร็จแล้ว ให้คลิก `แสดงตัวอย่าง (Preview)` ด้วยตัวเองครับ/ค่ะ

### ทำไมการแสดงตัวอย่างแผนภูมิถึงหายไปหลังจากปิดหน้าต่างป๊อปอัปครับ/คะ?
ตอบ: การแสดงตัวอย่างมีไว้สำหรับการดูชั่วคราวเท่านั้นครับ/ค่ะ หลังจากแก้ไขการตั้งค่าแล้ว โปรดบันทึกก่อนที่จะปิดหน้าต่าง การเปลี่ยนแปลงที่ยังไม่ได้บันทึกจะไม่ถูกเก็บไว้ครับ/ค่ะ
