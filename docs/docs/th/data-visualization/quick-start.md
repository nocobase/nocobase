# เริ่มต้นใช้งานอย่างรวดเร็ว

มาเริ่มต้นตั้งค่าสร้างแผนภูมิ (chart) ตั้งแต่ต้นจนจบกันครับ/ค่ะ โดยเราจะใช้ฟังก์ชันพื้นฐานที่จำเป็น และสำหรับความสามารถเสริมอื่นๆ จะกล่าวถึงในบทถัดไปครับ/ค่ะ

สิ่งที่ต้องเตรียม:
- ตั้งค่า แหล่งข้อมูล และ คอลเลกชัน (ตารางข้อมูล) ไว้เรียบร้อยแล้ว และมีสิทธิ์ในการอ่านข้อมูล

## เพิ่มบล็อกแผนภูมิ

ในตัวออกแบบหน้า (page designer) ให้คลิก “เพิ่มบล็อก” (Add block) จากนั้นเลือก “แผนภูมิ” (Chart) เพื่อเพิ่มบล็อกแผนภูมิครับ/ค่ะ

![clipboard-image-1761554593](https://static-docs.nocobase.com/clipboard-image-1761554593.png)

หลังจากเพิ่มบล็อกแล้ว ให้คลิก “ตั้งค่า” (Configure) ที่มุมขวาบนของบล็อกครับ/ค่ะ

![clipboard-image-1761554709](https://static-docs.nocobase.com/clipboard-image-1761554709.png)

แผงตั้งค่าแผนภูมิจะเปิดขึ้นทางด้านขวา ซึ่งประกอบด้วย 3 ส่วนหลัก ได้แก่ การสอบถามข้อมูล (Data Query), ตัวเลือกแผนภูมิ (Chart Options) และ เหตุการณ์ (Events) ครับ/ค่ะ

![clipboard-image-1761554848](https://static-docs.nocobase.com/clipboard-image-1761554848.png)

## ตั้งค่าการสอบถามข้อมูล (Data Query)
ในแผง “การสอบถามข้อมูล” (Data Query) คุณสามารถตั้งค่า แหล่งข้อมูล, เงื่อนไขการกรองข้อมูล และอื่นๆ ได้ครับ/ค่ะ

- อันดับแรก เลือก แหล่งข้อมูล และ คอลเลกชัน
  - ในแผง “การสอบถามข้อมูล” ให้เลือก แหล่งข้อมูล และ คอลเลกชัน ที่จะใช้เป็นฐานในการสอบถามข้อมูลครับ/ค่ะ
  - หากไม่สามารถเลือก คอลเลกชัน ได้ หรือ คอลเลกชัน ว่างเปล่า ให้ตรวจสอบก่อนว่ามีการสร้าง คอลเลกชัน นั้นแล้วหรือไม่ และผู้ใช้ปัจจุบันมีสิทธิ์เข้าถึงหรือไม่ครับ/ค่ะ

- ตั้งค่าตัววัด (Measures)
  - เลือกฟิลด์ที่เป็นตัวเลขหนึ่งฟิลด์หรือมากกว่า เพื่อใช้เป็นตัววัดครับ/ค่ะ
  - ตั้งค่าการรวมข้อมูล (aggregation) สำหรับแต่ละตัววัด เช่น Sum (ผลรวม), Count (จำนวน), Avg (ค่าเฉลี่ย), Max (ค่าสูงสุด), Min (ค่าต่ำสุด) ครับ/ค่ะ

- ตั้งค่ามิติข้อมูล (Dimensions)
  - เลือกหนึ่งฟิลด์หรือมากกว่า เพื่อใช้เป็นมิติข้อมูลสำหรับการจัดกลุ่ม (เช่น วันที่, หมวดหมู่, ภูมิภาค เป็นต้น) ครับ/ค่ะ
  - สำหรับฟิลด์วันที่/เวลา สามารถตั้งค่ารูปแบบได้ (เช่น `YYYY-MM`, `YYYY-MM-DD`) เพื่อให้แสดงผลได้อย่างสอดคล้องกันครับ/ค่ะ

![clipboard-image-1761555060](https://static-docs.nocobase.com/clipboard-image-1761555060.png)

เงื่อนไขอื่นๆ เช่น การกรอง (filter), การจัดเรียง (sort) และการแบ่งหน้า (pagination) เป็นตัวเลือกเสริมครับ/ค่ะ

## เรียกใช้การสอบถามและดูข้อมูล

- คลิก “เรียกใช้การสอบถาม” (Run query) เพื่อดึงข้อมูลและแสดงตัวอย่างแผนภูมิ (จะแสดงตัวอย่างโดยตรงในหน้าด้านซ้าย) ครับ/ค่ะ
- คุณสามารถคลิก “ดูข้อมูล” (View data) เพื่อดูตัวอย่างผลลัพธ์ข้อมูลที่ส่งกลับมา โดยรองรับการสลับรูปแบบระหว่าง Table และ JSON ครับ/ค่ะ หากคลิกอีกครั้งจะเป็นการซ่อนตัวอย่างข้อมูลครับ/ค่ะ
- หากผลลัพธ์ข้อมูลว่างเปล่าหรือไม่เป็นไปตามที่คาดไว้ ให้กลับไปที่แผงการสอบถามเพื่อตรวจสอบสิทธิ์ของ คอลเลกชัน, การแมปฟิลด์สำหรับตัววัด/มิติข้อมูล และชนิดข้อมูลครับ/ค่ะ

![clipboard-image-1761555228](https://static-docs.nocobase.com/clipboard-image-1761555228.png)

## ตั้งค่าตัวเลือกแผนภูมิ

ในแผง “ตัวเลือกแผนภูมิ” (Chart Options) คุณสามารถเลือกชนิดของแผนภูมิและตั้งค่าตัวเลือกต่างๆ ได้ครับ/ค่ะ

- อันดับแรก เลือกชนิดแผนภูมิ (เช่น แผนภูมิเส้น/พื้นที่, แผนภูมิแท่ง/บาร์, แผนภูมิวงกลม/โดนัท, แผนภูมิกระจาย เป็นต้น) ครับ/ค่ะ
- ทำการแมปฟิลด์หลักให้สมบูรณ์:
  - แผนภูมิเส้น/พื้นที่/แท่ง/บาร์: `xField` (มิติข้อมูล), `yField` (ตัววัด), `seriesField` (ชุดข้อมูล, เลือกได้)
  - แผนภูมิวงกลม/โดนัท: `Category` (มิติข้อมูลหมวดหมู่), `Value` (ตัววัด)
  - แผนภูมิกระจาย: `xField`, `yField` (ตัววัดหรือมิติข้อมูลสองตัว)
  - สำหรับการตั้งค่าแผนภูมิอื่นๆ เพิ่มเติม สามารถดูได้จากเอกสารของ ECharts: [Axis](https://echarts.apache.org/handbook/en/concepts/axis)
- หลังจากคลิก “เรียกใช้การสอบถาม” แล้ว การแมปฟิลด์จะถูกดำเนินการโดยอัตโนมัติครับ/ค่ะ หากมีการเปลี่ยนแปลงมิติข้อมูล/ตัววัด โปรดตรวจสอบการแมปอีกครั้งครับ/ค่ะ

![clipboard-image-1761555586](https://static-docs.nocobase.com/clipboard-image-1761555586.png)

## ดูตัวอย่างและบันทึก
การแก้ไขการตั้งค่าจะอัปเดตตัวอย่างแบบเรียลไทม์โดยอัตโนมัติ คุณสามารถดูแผนภูมิได้ในหน้าด้านซ้ายครับ/ค่ะ อย่างไรก็ตาม โปรดทราบว่าการแก้ไขทั้งหมดจะยังไม่ถูกบันทึกอย่างแท้จริงจนกว่าคุณจะคลิกปุ่ม “บันทึก” ครับ/ค่ะ

คุณยังสามารถใช้ปุ่มที่ด้านล่างได้อีกด้วย:

- **ดูตัวอย่าง (Preview)**: การเปลี่ยนแปลงการตั้งค่าจะรีเฟรชตัวอย่างโดยอัตโนมัติแบบเรียลไทม์ หรือคุณสามารถคลิกปุ่ม “ดูตัวอย่าง” ที่ด้านล่างเพื่อรีเฟรชด้วยตนเองได้ครับ/ค่ะ
- **ยกเลิก (Cancel)**: หากคุณไม่ต้องการการเปลี่ยนแปลงการตั้งค่าปัจจุบัน คุณสามารถคลิกปุ่ม “ยกเลิก” ที่ด้านล่าง หรือรีเฟรชหน้าเพื่อย้อนกลับไปยังสถานะที่บันทึกไว้ล่าสุดครับ/ค่ะ
- **บันทึก (Save)**: คลิก “บันทึก” เพื่อบันทึกการสอบถามและการตั้งค่าแผนภูมิทั้งหมดลงในฐานข้อมูลอย่างถาวร ซึ่งจะมีผลกับผู้ใช้ทุกคนครับ/ค่ะ

![clipboard-image-1761555803](https://static-docs.nocobase.com/clipboard-image-1761555803.png)

## ข้อควรทราบทั่วไป

- **การตั้งค่าขั้นต่ำที่ใช้งานได้**: เลือก คอลเลกชัน และตัววัดอย่างน้อยหนึ่งตัวครับ/ค่ะ แนะนำให้เพิ่มมิติข้อมูลเพื่อความสะดวกในการแสดงผลแบบจัดกลุ่มครับ/ค่ะ
- สำหรับมิติข้อมูลที่เป็นวันที่ แนะนำให้ตั้งค่ารูปแบบที่เหมาะสม (เช่น เลือก `YYYY-MM` สำหรับการนับตามเดือน) เพื่อหลีกเลี่ยงแกน X ที่ไม่ต่อเนื่องหรือผิดเพี้ยนครับ/ค่ะ
- **หากการสอบถามว่างเปล่าหรือไม่แสดงแผนภูมิ**:
  - ตรวจสอบ คอลเลกชัน/สิทธิ์ และการแมปฟิลด์ครับ/ค่ะ
  - ใช้ “ดูข้อมูล” เพื่อยืนยันว่าชื่อคอลัมน์และชนิดข้อมูลตรงกับการแมปแผนภูมิครับ/ค่ะ
- **การดูตัวอย่างเป็นเพียงชั่วคราว**: ใช้สำหรับการตรวจสอบและปรับแต่งเท่านั้นครับ/ค่ะ จะมีผลอย่างเป็นทางการหลังจากที่คุณคลิก “บันทึก” ครับ/ค่ะ
