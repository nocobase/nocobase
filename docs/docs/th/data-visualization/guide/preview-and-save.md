# ดูตัวอย่างและบันทึก

* ดูตัวอย่าง: แสดงผลการเปลี่ยนแปลงจากแผงการตั้งค่าบนแผนภูมิหน้าเพจชั่วคราว เพื่อให้คุณตรวจสอบผลลัพธ์
* บันทึก: บันทึกการเปลี่ยนแปลงจากแผงการตั้งค่าลงในฐานข้อมูลอย่างถาวร

## จุดเข้าใช้งาน

![clipboard-image-1761479218](https://static-docs.nocobase.com/clipboard-image-1761479218.png)

- ในโหมดการตั้งค่าแบบกราฟิก (Basic) การเปลี่ยนแปลงทั้งหมดจะถูกนำไปแสดงตัวอย่างโดยอัตโนมัติ
- ในโหมด SQL และโหมด Custom หลังจากทำการเปลี่ยนแปลง คุณสามารถคลิกปุ่ม `ดูตัวอย่าง` ทางด้านขวาเพื่อแสดงผลตัวอย่าง
- ที่ด้านล่างสุดของแผงการตั้งค่าทั้งหมด จะมีปุ่ม `ดูตัวอย่าง` ที่รวมไว้เป็นหนึ่งเดียว

## ลักษณะการทำงานของการดูตัวอย่าง
- แสดงการตั้งค่าบนหน้าเพจชั่วคราว แต่จะไม่บันทึกลงในฐานข้อมูล หลังจากรีเฟรชหน้าเพจหรือยกเลิกการทำงาน ผลลัพธ์การดูตัวอย่างจะไม่ถูกเก็บไว้
- มีระบบ Debounce ในตัว: หากมีการเรียกใช้การรีเฟรชหลายครั้งในเวลาอันสั้น ระบบจะดำเนินการเฉพาะครั้งสุดท้ายเท่านั้น เพื่อหลีกเลี่ยงการส่งคำขอที่บ่อยเกินไป
- การคลิก `ดูตัวอย่าง` อีกครั้งจะเขียนทับผลลัพธ์การดูตัวอย่างครั้งล่าสุด

## การแจ้งเตือนข้อผิดพลาด
- ข้อผิดพลาดในการสอบถามข้อมูลหรือการตรวจสอบไม่สำเร็จ: จะแสดงข้อความข้อผิดพลาดในส่วน `ดูข้อมูล`
- ข้อผิดพลาดในการตั้งค่าแผนภูมิ (เช่น การแมป Basic ขาดหายไป หรือเกิดข้อผิดพลาดจาก Custom JS): จะแสดงข้อผิดพลาดในพื้นที่แผนภูมิหรือคอนโซล โดยที่หน้าเพจยังคงใช้งานได้
- ควรตรวจสอบชื่อคอลัมน์และประเภทข้อมูลในส่วน `ดูข้อมูล` ก่อนที่จะทำการแมปฟิลด์หรือเขียนโค้ด Custom เพื่อช่วยลดข้อผิดพลาดได้อย่างมีประสิทธิภาพ

## บันทึกและยกเลิก
- บันทึก: บันทึกการเปลี่ยนแปลงในแผงปัจจุบันลงในการตั้งค่าบล็อก และมีผลกับหน้าเพจทันที
- ยกเลิก: ยกเลิกการเปลี่ยนแปลงที่ยังไม่ได้บันทึกในแผงปัจจุบัน และกลับสู่สถานะที่บันทึกไว้ล่าสุด
- ขอบเขตการบันทึก:
  - การสอบถามข้อมูล: พารามิเตอร์การสอบถามของ Builder; ในโหมด SQL จะบันทึกข้อความ SQL ด้วย
  - ตัวเลือกแผนภูมิ: ประเภทและการแมปฟิลด์ของ Basic รวมถึงคุณสมบัติ; ข้อความ JS ของ Custom
  - เหตุการณ์การโต้ตอบ: ข้อความ JS และตรรกะการผูกของเหตุการณ์
- หลังจากบันทึก บล็อกจะมีผลกับผู้เยี่ยมชมทุกคน (ขึ้นอยู่กับการตั้งค่าสิทธิ์ของหน้าเพจ)

## ขั้นตอนการทำงานที่แนะนำ
- ตั้งค่าการสอบถามข้อมูล → เรียกใช้การสอบถาม → `ดูข้อมูล` เพื่อยืนยันชื่อคอลัมน์และประเภท → ตั้งค่าตัวเลือกแผนภูมิเพื่อแมปฟิลด์หลัก → ดูตัวอย่างเพื่อตรวจสอบ → บันทึกเพื่อให้มีผล
