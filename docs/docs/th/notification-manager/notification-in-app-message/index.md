# การแจ้งเตือน: ข้อความในแอป

## บทนำ

ปลั๊กอินนี้ช่วยให้ผู้ใช้สามารถรับการแจ้งเตือนข้อความได้แบบเรียลไทม์ภายในแอปพลิเคชัน NocoBase ครับ/ค่ะ

## การติดตั้ง

ปลั๊กอินนี้เป็นปลั๊กอินที่มาพร้อมกับระบบอยู่แล้ว จึงไม่จำเป็นต้องติดตั้งเพิ่มเติมครับ/ค่ะ

## การเพิ่มช่องทางข้อความในแอป

ไปที่หน้าการจัดการการแจ้งเตือน (Notification Management) จากนั้นคลิกปุ่ม "เพิ่ม" (Add) และเลือก "ข้อความในแอป" (In-App Message) ครับ/ค่ะ
![2024-11-08-08-33-26-20241108083326](https://static-docs.nocobase.com/2024-11-08-08-33-26-20241108083326.png)

กรอกชื่อช่องทางและคำอธิบาย จากนั้นคลิก "ส่ง" (Submit) ครับ/ค่ะ
![2024-11-08-08-34-32-20241108083431](https://static-docs.nocobase.com/2024-11-08-08-34-32-20241108083431.png)

ช่องทางใหม่จะปรากฏในรายการครับ/ค่ะ

![2024-11-08-08-34-52-20241108083452](https://static-docs.nocobase.com/2024-11-08-08-34-52-20241108083452.png)

## ตัวอย่างการใช้งาน

เพื่อให้คุณเข้าใจวิธีการใช้งานข้อความในแอปได้ดียิ่งขึ้น เรามีตัวอย่างการใช้งาน "การติดตามลูกค้าเป้าหมายทางการตลาด" มาให้ชมครับ/ค่ะ

สมมติว่าทีมของคุณกำลังดำเนินแคมเปญการตลาดที่สำคัญ โดยมีเป้าหมายเพื่อติดตามผลตอบรับและความต้องการจากลูกค้าเป้าหมาย การใช้ข้อความในแอปจะช่วยให้คุณสามารถ:

**สร้างช่องทางการแจ้งเตือน**: อันดับแรก ให้คุณตั้งค่าช่องทางข้อความในแอปชื่อ "Marketing Clue" ในหน้าการจัดการช่องทางการแจ้งเตือน เพื่อให้สมาชิกในทีมสามารถระบุวัตถุประสงค์ของช่องทางนี้ได้อย่างชัดเจนครับ/ค่ะ

![2024-11-08-08-34-32-20241108083431](https://static-docs.nocobase.com/2024-11-08-08-34-32-20241108083431.png)

**ตั้งค่าเวิร์กโฟลว์**: สร้างเวิร์กโฟลว์ที่จะทริกเกอร์การแจ้งเตือนโดยอัตโนมัติเมื่อมีลูกค้าเป้าหมายทางการตลาดรายใหม่เข้ามา คุณสามารถเพิ่มโหนดการแจ้งเตือน (notification node) ในเวิร์กโฟลว์ เลือกช่องทาง "Marketing Clue" ที่สร้างไว้ก่อนหน้านี้ และตั้งค่าเนื้อหาข้อความตามความต้องการได้เลยครับ/ค่ะ ตัวอย่างเช่น:

![image-1-2024-10-27-14-07-17](https://static-docs.nocobase.com/image-1-2024-10-27-14-07-17.png)

**รับการแจ้งเตือนแบบเรียลไทม์**: เมื่อเวิร์กโฟลว์ถูกทริกเกอร์ บุคลากรที่เกี่ยวข้องทั้งหมดจะได้รับการแจ้งเตือนแบบเรียลไทม์ทันที ทำให้ทีมสามารถตอบสนองและดำเนินการได้อย่างรวดเร็วครับ/ค่ะ

![image-2-2024-10-27-14-07-22](https://static-docs.nocobase.com/image-2-2024-10-27-14-07-22.png)

**การจัดการและติดตามข้อความ**: ข้อความในแอปจะถูกจัดกลุ่มตามชื่อช่องทางที่ส่ง คุณสามารถกรองข้อความตามสถานะที่อ่านแล้วและยังไม่ได้อ่าน เพื่อให้ง่ายต่อการดูข้อมูลสำคัญได้อย่างรวดเร็ว การคลิกปุ่ม "ดู" (View) จะนำคุณไปยังหน้าลิงก์ที่ตั้งค่าไว้ เพื่อดำเนินการในเรื่องที่เกี่ยวข้องต่อไปครับ/ค่ะ
