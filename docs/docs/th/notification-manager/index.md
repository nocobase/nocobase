# การจัดการการแจ้งเตือน

## บทนำ

การจัดการการแจ้งเตือนเป็นบริการแบบรวมศูนย์ที่รวบรวมช่องทางการแจ้งเตือนที่หลากหลายเข้าไว้ด้วยกันครับ/ค่ะ โดยมีอินเทอร์เฟซที่เป็นหนึ่งเดียวสำหรับการตั้งค่าช่องทาง การจัดการการส่ง และการบันทึกข้อมูล พร้อมรองรับการขยายช่องทางเพิ่มเติมได้อย่างยืดหยุ่นครับ/ค่ะ

![20240928112556](https://static-docs.nocobase.com/20240928112556.png)

- **ส่วนสีม่วง**: การจัดการการแจ้งเตือน ให้บริการจัดการแบบครบวงจร ซึ่งครอบคลุมฟังก์ชันการตั้งค่าช่องทาง การบันทึกข้อมูล และอื่นๆ โดยสามารถขยายช่องทางการแจ้งเตือนเพิ่มเติมได้ครับ/ค่ะ
- **ส่วนสีเขียว**: ข้อความในแอป (In-App Message) เป็นช่องทางที่มาพร้อมกับระบบ ซึ่งช่วยให้ผู้ใช้สามารถรับการแจ้งเตือนข้อความได้โดยตรงภายในแอปพลิเคชันครับ/ค่ะ
- **ส่วนสีแดง**: อีเมล (Email) เป็นช่องทางที่สามารถขยายเพิ่มเติมได้ ซึ่งช่วยให้ผู้ใช้สามารถรับการแจ้งเตือนผ่านทางอีเมลได้ครับ/ค่ะ

## การจัดการช่องทาง

![20240928181752](https://static-docs.nocobase.com/20240928181752.png)

ช่องทางที่รองรับในปัจจุบันมีดังนี้ครับ/ค่ะ

- [ข้อความในแอป](/notification-manager/notification-in-app-message)
- [อีเมล](/notification-manager/notification-email) (ใช้การส่งผ่าน SMTP ที่มาพร้อมกับระบบ)

นอกจากนี้ คุณยังสามารถขยายช่องทางการแจ้งเตือนเพิ่มเติมได้อีกด้วยครับ/ค่ะ โดยอ้างอิงจากเอกสาร [การขยายช่องทาง](/notification-manager/development/extension) ครับ/ค่ะ

## บันทึกการแจ้งเตือน

ระบบจะบันทึกรายละเอียดและสถานะการส่งของการแจ้งเตือนแต่ละรายการอย่างละเอียด เพื่อช่วยในการวิเคราะห์และแก้ไขปัญหาครับ/ค่ะ
