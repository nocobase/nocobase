# การแจ้งเตือน: อีเมล

## บทนำ

สามารถส่งการแจ้งเตือนผ่านช่องทางอีเมลได้ครับ โดยปัจจุบันรองรับเฉพาะวิธีการส่งแบบ SMTP เท่านั้นครับ

## การติดตั้ง

ปลั๊กอินนี้เป็นปลั๊กอินที่ติดตั้งมาล่วงหน้าครับ ต้องเปิดใช้งานก่อนจึงจะสามารถใช้งานได้ครับ
