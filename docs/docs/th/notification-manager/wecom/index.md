# แจ้งเตือน: WeCom

## บทนำ

ปลั๊กอิน **WeCom** ช่วยให้แอปพลิเคชันสามารถส่งข้อความแจ้งเตือนไปยังผู้ใช้ WeCom ได้ครับ/ค่ะ

## เพิ่มและตั้งค่าตัวรับรองความถูกต้อง WeCom

อันดับแรก คุณต้องเพิ่มและตั้งค่าตัวรับรองความถูกต้อง WeCom บน NocoBase ครับ/ค่ะ โดยอ้างอิงจาก [การรับรองความถูกต้องของผู้ใช้ - WeCom](/auth-verification/auth-wecom) ทั้งนี้ เฉพาะผู้ใช้ระบบที่เข้าสู่ระบบด้วย WeCom เท่านั้น จึงจะสามารถรับการแจ้งเตือนจากระบบผ่าน WeCom ได้ครับ/ค่ะ
