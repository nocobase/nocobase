# บันทึกร่างอัตโนมัติ

ในเมนูของปุ่ม **บันทึกร่าง** คุณสามารถเปิดใช้งาน **บันทึกร่างอัตโนมัติ** ได้ครับ/ค่ะ

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_29_PM.png)

ในระหว่างการแก้ไขอีเมล ระบบจะบันทึกฉบับร่างแบบเรียลไทม์ เพื่อให้มั่นใจว่าเนื้อหาจะไม่สูญหายครับ/ค่ะ

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_31_PM.png)

เมื่อคุณเปิดหน้าต่างแก้ไขอีเมลอีกครั้ง (สำหรับผู้ส่งและผู้รับคนเดิม) ระบบจะถามว่าคุณต้องการใช้ฉบับร่างที่บันทึกไว้หรือไม่ครับ/ค่ะ

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_08_35_PM.png)
