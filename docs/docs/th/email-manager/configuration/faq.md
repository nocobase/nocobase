# คำถามที่พบบ่อย

## ไม่สามารถรับอีเมลได้หลังจากอนุญาตด้วยบัญชี Microsoft

A: ปัจจุบันรองรับเฉพาะการเข้าสู่ระบบด้วยบัญชีอีเมล Outlook และ Gmail เท่านั้นครับ/ค่ะ บัญชี Microsoft และ Google ยังไม่รองรับในขณะนี้ครับ/ค่ะ สามารถดูข้อมูลอ้างอิงได้ที่: [answers.microsoft.com](https://answers.microsoft.com/zh-hans/outlook_com/forum/all/%E7%8E%B0%E6%9C%89%E5%BE%AE%E8%BD%AF%E8%B4%A6/dba12dda-a7c7-4346-8263-53f4a6d9dc68)

**เคล็ดลับ**: หากคุณไม่แน่ใจว่าคุณมี "อีเมล Outlook.com จริง" หรือ "อีเมล Gmail" หรือไม่ คุณสามารถลองเข้าชม Outlook.com หรือ Gmail.com ผ่านเว็บเบราว์เซอร์ของคุณ เพื่อดูว่าสามารถเข้าสู่ระบบและส่งอีเมลได้ตามปกติหรือไม่ หากไม่สามารถทำได้ แสดงว่าคุณอาจไม่มีบริการอีเมลดังกล่าว และจำเป็นต้องเปิดใช้งานก่อน หรือใช้บริการอีเมลอื่นแทนครับ/ค่ะ

## ยกเลิกการอนุญาต

หากบัญชีอีเมลของคุณได้รับการอนุญาตแล้ว แต่คุณต้องการลบหรืออนุญาตใหม่ คุณสามารถทำตามขั้นตอนด้านล่างนี้ได้เลยครับ/ค่ะ

#### **Gmail**

1. เปิด https://myaccount.google.com/u/0/connections แล้วเข้าสู่ระบบครับ/ค่ะ

![](https://static-docs.nocobase.com/mail-1735634131347.png)

2. คลิกที่แอปพลิเคชันที่เกี่ยวข้อง จากนั้นคลิก "ลบ" ครับ/ค่ะ

![](https://static-docs.nocobase.com/mail-1735634131518.png)

![](https://static-docs.nocobase.com/mail-1735634131697.png)

#### **อีเมล Microsoft**

1. เปิด https://account.microsoft.com/ แล้วเข้าสู่ระบบครับ/ค่ะ
    
2. คลิกปุ่ม "Apps and services that can access your data" ครับ/ค่ะ

![](https://static-docs.nocobase.com/mail-1735634131870.png)

3. คลิก "แก้ไข" แล้ว "ลบ" ครับ/ค่ะ

![](https://static-docs.nocobase.com/mail-1735634132052.png)
