# ลายเซ็นอีเมล

## การตั้งค่าลายเซ็น

เข้าไปที่ศูนย์จัดการอีเมล แล้วคลิกปุ่มตั้งค่าครับ/ค่ะ:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_05_34_PM.png)

จากนั้นสลับไปที่แผงลายเซ็นครับ/ค่ะ:
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_05_51_PM.png)

ที่นี่คุณสามารถจัดการลายเซ็นได้ครับ/ค่ะ โดยติ๊กช่องตัวเลือกหน้าลายเซ็นเพื่อตั้งเป็นลายเซ็นเริ่มต้น:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_05_37_PM.png)

## การใช้งานลายเซ็น

ในช่องเนื้อหาอีเมล ให้คลิกไอคอนลายเซ็นเพื่อเลือกลายเซ็นที่คุณต้องการครับ/ค่ะ:
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_05_40_PM.png)
