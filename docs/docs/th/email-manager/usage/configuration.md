# การตั้งค่าบล็อก

## บล็อกข้อความอีเมล

### การเพิ่มบล็อก

บนหน้าการตั้งค่า ให้คลิกปุ่ม **สร้างบล็อก** จากนั้นเลือกบล็อก **ข้อความอีเมล (ทั้งหมด)** หรือ **ข้อความอีเมล (ส่วนตัว)** เพื่อเพิ่มบล็อกข้อความอีเมลครับ/ค่ะ

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_03_58_PM.png)

### การตั้งค่าฟิลด์

คลิกปุ่ม **ฟิลด์** ของบล็อกเพื่อเลือกฟิลด์ที่ต้องการแสดงผล สำหรับรายละเอียดการตั้งค่าเพิ่มเติม สามารถดูได้จากวิธีการตั้งค่าฟิลด์ของตารางครับ/ค่ะ

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_37_PM.png)

### การตั้งค่าการกรองข้อมูล

คลิกไอคอนการตั้งค่าทางด้านขวาของตาราง แล้วเลือก **ขอบเขตข้อมูล** เพื่อกำหนดช่วงข้อมูลสำหรับการกรองอีเมลครับ/ค่ะ

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_39_PM.png)

คุณสามารถกรองอีเมลที่มีนามสกุลเดียวกันได้โดยใช้ตัวแปรครับ/ค่ะ:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_41_PM.png)

## บล็อกรายละเอียดอีเมล

อันดับแรก ให้เปิดใช้งานฟังก์ชัน **เปิดเมื่อคลิก** บนฟิลด์ในบล็อกข้อความอีเมลครับ/ค่ะ:

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_45_PM.png)

เพิ่มบล็อก **รายละเอียดอีเมล** ในหน้าต่างป๊อปอัปครับ/ค่ะ:

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_46_PM.png)

คุณสามารถดูเนื้อหาโดยละเอียดของอีเมลได้ครับ/ค่ะ:

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_04_49_PM.png)

คุณสามารถตั้งค่าปุ่มที่ต้องการได้เองที่ด้านล่างครับ/ค่ะ

## บล็อกการส่งอีเมล

มีสองวิธีในการสร้างฟอร์มส่งอีเมลครับ/ค่ะ:

1. เพิ่มปุ่ม **ส่งอีเมล** ที่ด้านบนของตาราง:
   ![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_52_PM.png)

2. เพิ่มบล็อก **ส่งอีเมล**:
   ![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_53_PM.png)

ทั้งสองวิธีนี้สามารถสร้างฟอร์มส่งอีเมลที่สมบูรณ์ได้ครับ/ค่ะ:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_53_PM%20(1).png)

แต่ละฟิลด์ในฟอร์มอีเมลจะเหมือนกับฟอร์มทั่วไป และสามารถตั้งค่า **ค่าเริ่มต้น** หรือ **กฎการเชื่อมโยง** เป็นต้น ได้ครับ/ค่ะ

> ฟอร์มตอบกลับและฟอร์มส่งต่ออีเมลที่มาพร้อมกับส่วนท้ายของรายละเอียดอีเมล จะมีการประมวลผลข้อมูลบางส่วนเป็นค่าเริ่มต้น ซึ่งสามารถแก้ไขได้ผ่าน **FlowEngine** ครับ/ค่ะ
