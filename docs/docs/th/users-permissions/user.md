# ผู้ใช้งาน

## การจัดการผู้ใช้งาน

ปลั๊กอินนี้มีหน้าจอสำหรับจัดการผู้ใช้งานให้ใช้งานในศูนย์การตั้งค่าครับ/ค่ะ หลังจากเริ่มต้นแอปพลิเคชัน ระบบจะสร้างบัญชีผู้ดูแลระบบสูงสุด (Super Admin) ที่ไม่สามารถลบได้ขึ้นมาหนึ่งบัญชีครับ/ค่ะ ผู้ดูแลระบบสูงสุดนี้มีสิทธิ์ Root และสามารถเข้าถึงทรัพยากรทั้งหมดได้ครับ/ค่ะ

![](https://static-docs.nocobase.com/44bf40f56b45d4dd96c424fb08082cf6.png)

นอกจากนี้ คุณยังสามารถใช้คอลเลกชันผู้ใช้งานเพื่อเพิ่มบล็อกประเภทต่างๆ ในหน้าเพจเพื่อจัดการผู้ใช้งานได้ครับ/ค่ะ เช่น บล็อกตาราง (Table Block):

![](https://static-docs.nocobase.com/76b5a4652f869541a9e8f18a4568a7c9.png)

## การตั้งค่า

:::info{title=คำแนะนำ}
ต้องใช้ NocoBase เวอร์ชัน v1.3.34-beta หรือสูงกว่าครับ/ค่ะ
:::
