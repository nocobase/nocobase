# ซิงค์ข้อมูลผู้ใช้จาก WeChat Work

## บทนำ

ปลั๊กอิน **WeChat Work** รองรับการซิงค์ข้อมูลผู้ใช้และแผนกจาก WeChat Work ครับ/ค่ะ

## สร้างและตั้งค่าแอปพลิเคชัน WeChat Work แบบกำหนดเอง

ก่อนอื่น คุณต้องสร้างแอปพลิเคชัน WeChat Work แบบกำหนดเองในแผงควบคุมผู้ดูแลระบบ WeChat Work และรับ **Corp ID**, **AgentId** และ **Secret** ครับ/ค่ะ

โปรดดูที่ [การยืนยันตัวตนผู้ใช้ - WeChat Work](/auth-verification/auth-wecom/) ครับ/ค่ะ

## เพิ่มแหล่งข้อมูลสำหรับซิงค์ใน NocoBase

ไปที่ ผู้ใช้และสิทธิ์ - ซิงค์ - เพิ่ม แล้วกรอกข้อมูลที่ได้รับครับ/ค่ะ

![](https://static-docs.nocobase.com/202412041251867.png)

## ตั้งค่าการซิงค์รายชื่อติดต่อ

ไปที่แผงควบคุมผู้ดูแลระบบ WeChat Work - ความปลอดภัยและการจัดการ - เครื่องมือการจัดการ แล้วคลิกที่ ซิงค์รายชื่อติดต่อ ครับ/ค่ะ

![](https://static-docs.nocobase.com/202412041249958.png)

ตั้งค่าตามที่แสดงในรูปภาพ และกำหนด IP ที่เชื่อถือได้ขององค์กรครับ/ค่ะ

![](https://static-docs.nocobase.com/202412041250776.png)

หลังจากนี้ คุณก็สามารถดำเนินการซิงค์ข้อมูลผู้ใช้ได้เลยครับ/ค่ะ

## ตั้งค่าเซิร์ฟเวอร์รับเหตุการณ์

หากคุณต้องการให้การเปลี่ยนแปลงข้อมูลผู้ใช้และแผนกในฝั่ง WeChat Work สามารถซิงค์ไปยังแอปพลิเคชัน NocoBase ได้ทันที คุณสามารถตั้งค่าเพิ่มเติมได้ครับ/ค่ะ

หลังจากกรอกข้อมูลการตั้งค่าก่อนหน้านี้แล้ว คุณสามารถคัดลอก URL การแจ้งเตือน Callback ของรายชื่อติดต่อได้ครับ/ค่ะ

![](https://static-docs.nocobase.com/202412041256547.png)

นำไปกรอกในการตั้งค่า WeChat Work และรับ Token กับ EncodingAESKey เพื่อเสร็จสิ้นการตั้งค่าแหล่งข้อมูลสำหรับซิงค์ผู้ใช้ NocoBase ครับ/ค่ะ

![](https://static-docs.nocobase.com/202412041257947.png)
