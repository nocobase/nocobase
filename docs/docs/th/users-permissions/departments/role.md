# การจัดการบทบาทของแผนก

เมื่อคุณเชื่อมโยงบทบาทตั้งแต่หนึ่งบทบาทขึ้นไปเข้ากับแผนก สมาชิกในแผนกนั้นๆ ก็จะได้รับบทบาทเหล่านั้นครับ/ค่ะ

มีสองวิธีในการเชื่อมโยงบทบาทเข้ากับแผนกครับ/ค่ะ

วิธีแรกคือการเลือกบทบาทในช่องบทบาทของฟอร์มแก้ไขแผนกครับ/ค่ะ

![](https://static-docs.nocobase.com/70f77bb89aa1fb415c152a51a51cc23b.png)

วิธีที่สองคือการเพิ่มแผนกให้กับบทบาทที่เกี่ยวข้องในหน้าการจัดการบทบาทครับ/ค่ะ

![](https://static-docs.nocobase.com/f2a77bec937cf2f179ce868a92b98416d.png)

แผนกที่มีบทบาทนี้อยู่แล้วจะไม่สามารถเลือกได้ครับ/ค่ะ

![](https://static-docs.nocobase.com/be10299893581e1f97a4e01ddd5c7e59.png)
