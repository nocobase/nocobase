## การจัดการสมาชิกแผนก

### ดูรายการสมาชิกแผนก

![](https://static-docs.nocobase.com/2aaf4d9bf55da105b5fca4e9f7e23ca7.png)

### เพิ่มสมาชิกเข้าแผนก

ผู้ใช้หนึ่งคนสามารถเข้าร่วมได้หลายแผนกพร้อมกันครับ/ค่ะ โดยแผนกแรกที่เข้าร่วมจะถูกกำหนดให้เป็นแผนกหลักโดยอัตโนมัติ และผู้ใช้แต่ละคนจะมีแผนกหลักได้เพียงแผนกเดียวเท่านั้นครับ/ค่ะ

![](https://static-docs.nocobase.com/60afd282f33b555e6fe0662b9da544cc.png)

ผู้ใช้ที่เป็นสมาชิกของแผนกอยู่แล้ว จะไม่ปรากฏในรายการเลือกผู้ใช้ครับ/ค่ะ

![](https://static-docs.nocobase.com/6bcd93173c169973f970de35d2657993.png)

### กำหนดผู้รับผิดชอบแผนก

ในฟอร์มแก้ไขแผนก ให้คลิกที่ช่องข้อมูลผู้รับผิดชอบ เพื่อเลือกสมาชิกแผนกให้เป็นผู้รับผิดชอบครับ/ค่ะ รองรับการเลือกได้หลายคน และสมาชิกที่เป็นผู้รับผิดชอบแผนกอยู่แล้ว จะไม่ปรากฏในรายการเลือกสมาชิกครับ/ค่ะ

![](https://static-docs.nocobase.com/92970546cbd0aeb5a8b6a36da87583bd.png)

### กำหนดแผนกให้กับผู้ใช้

นอกจากการเพิ่มสมาชิกเข้าแผนกแล้ว คุณยังสามารถกำหนดแผนกให้กับผู้ใช้ได้จากมุมมองของผู้ใช้โดยตรงครับ/ค่ะ

![](https://static-docs.nocobase.com/ca82a802012572e225570e8be93a4094.png)

แผนกที่ผู้ใช้เข้าร่วมอยู่แล้ว จะไม่สามารถเลือกได้ครับ/ค่ะ

![](https://static-docs.nocobase.com/70e16d17ee9c4b5d43f8a5e1c633b177.png)

## ค้นหาผู้ใช้และแผนก

ค้นหาผู้ใช้ด้วยชื่อเล่น, ชื่อผู้ใช้, เบอร์โทรศัพท์, อีเมล และค้นหาแผนกด้วยชื่อแผนกครับ/ค่ะ

![](https://static-docs.nocobase.com/2d71346a5400205b22436b4db331a9b8.png)
