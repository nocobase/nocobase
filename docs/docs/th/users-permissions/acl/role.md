# บทบาท

## ศูนย์จัดการ

### การจัดการบทบาท

![](https://static-docs.nocobase.com/da7083c67d794e23dc6eb0f85b1de86c.png)

แอปพลิเคชันที่ติดตั้งมาตั้งแต่เริ่มต้นจะมีบทบาทในตัวสองบทบาท ได้แก่ "Admin" และ "Member" ซึ่งแต่ละบทบาทมีการตั้งค่าสิทธิ์เริ่มต้นที่แตกต่างกันครับ/ค่ะ

### การเพิ่ม ลบ และแก้ไขบทบาท

ตัวระบุบทบาท (ซึ่งเป็นตัวระบุเฉพาะของระบบ) ช่วยให้คุณสามารถกำหนดบทบาทเริ่มต้นเองได้ แต่จะไม่สามารถลบบทบาทเริ่มต้นที่มาพร้อมกับระบบได้ครับ/ค่ะ

![](https://static-docs.nocobase.com/35f323b346db4f9f12f9bee4dea63302.png)

### การตั้งค่าบทบาทเริ่มต้น

บทบาทเริ่มต้นในที่นี้หมายถึง บทบาทที่จะถูกกำหนดให้ผู้ใช้ใหม่โดยอัตโนมัติ หากไม่ได้ระบุบทบาทเฉพาะเจาะจงตอนสร้างผู้ใช้ครับ/ค่ะ

![](https://static-docs.nocobase.com/f41bba7ff55ca28715c486dc45bc1708.png)

## ศูนย์ข้อมูลส่วนตัว

### การสลับบทบาท

คุณสามารถกำหนดหลายบทบาทให้กับผู้ใช้หนึ่งคนได้ครับ/ค่ะ เมื่อผู้ใช้มีหลายบทบาท พวกเขาสามารถสลับบทบาทได้ที่ศูนย์ข้อมูลส่วนตัวครับ/ค่ะ

![](https://static-docs.nocobase.com/e31b11ec1ca3b8b7e0472105b167819.png)

ลำดับความสำคัญของบทบาทเริ่มต้นเมื่อผู้ใช้เข้าสู่ระบบจะเป็นดังนี้ครับ/ค่ะ: บทบาทที่สลับใช้งานล่าสุด (ค่านี้จะอัปเดตทุกครั้งที่มีการสลับบทบาท) > บทบาทแรก (บทบาทเริ่มต้นของระบบ)
