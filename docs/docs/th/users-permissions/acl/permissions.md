# การกำหนดสิทธิ์

## การตั้งค่าสิทธิ์ทั่วไป

![](https://static-docs.nocobase.com/119a9650259f9be71210121d0d3a435d.png)

### สิทธิ์การกำหนดค่า

1.  **อนุญาตให้กำหนดค่าส่วนติดต่อผู้ใช้ (UI)**: สิทธิ์นี้ควบคุมว่าผู้ใช้สามารถกำหนดค่าส่วนติดต่อผู้ใช้ได้หรือไม่ครับ/ค่ะ เมื่อเปิดใช้งานสิทธิ์นี้ จะมีปุ่มกำหนดค่า UI ปรากฏขึ้น บทบาท "admin" จะเปิดใช้งานสิทธิ์นี้โดยค่าเริ่มต้นครับ/ค่ะ
2.  **อนุญาตให้ติดตั้ง, เปิดใช้งาน, ปิดใช้งานปลั๊กอิน**: สิทธิ์นี้ควบคุมว่าผู้ใช้สามารถเปิดใช้งานหรือปิดใช้งานปลั๊กอินได้หรือไม่ครับ/ค่ะ เมื่อเปิดใช้งานสิทธิ์นี้ ผู้ใช้จะสามารถเข้าถึงหน้าจอตัวจัดการปลั๊กอินได้ บทบาท "admin" จะเปิดใช้งานสิทธิ์นี้โดยค่าเริ่มต้นครับ/ค่ะ
3.  **อนุญาตให้กำหนดค่าปลั๊กอิน**: สิทธิ์นี้ช่วยให้ผู้ใช้สามารถกำหนดค่าพารามิเตอร์ของปลั๊กอิน หรือจัดการข้อมูลแบ็กเอนด์ของปลั๊กอินได้ครับ/ค่ะ บทบาท "admin" จะเปิดใช้งานสิทธิ์นี้โดยค่าเริ่มต้นครับ/ค่ะ
4.  **อนุญาตให้ล้างแคช, รีสตาร์ทแอปพลิเคชัน**: สิทธิ์นี้เกี่ยวข้องกับงานบำรุงรักษาระบบของผู้ใช้ เช่น การล้างแคชและการรีสตาร์ทแอปพลิเคชันครับ/ค่ะ เมื่อเปิดใช้งาน ปุ่มการทำงานที่เกี่ยวข้องจะปรากฏขึ้นในศูนย์ส่วนตัว สิทธิ์นี้จะปิดใช้งานโดยค่าเริ่มต้นครับ/ค่ะ
5.  **รายการเมนูใหม่จะอนุญาตให้เข้าถึงได้โดยค่าเริ่มต้น**: เมนูที่สร้างขึ้นใหม่จะสามารถเข้าถึงได้โดยค่าเริ่มต้น และการตั้งค่านี้จะเปิดใช้งานโดยค่าเริ่มต้นครับ/ค่ะ

### สิทธิ์การดำเนินการแบบทั่วโลก

สิทธิ์การดำเนินการแบบทั่วโลกมีผลบังคับใช้กับทุกคอลเลกชัน และถูกจัดหมวดหมู่ตามประเภทการดำเนินการครับ/ค่ะ สิทธิ์เหล่านี้สามารถกำหนดค่าได้ตามขอบเขตข้อมูล ได้แก่ ข้อมูลทั้งหมด หรือข้อมูลของผู้ใช้เอง แบบแรกอนุญาตให้ดำเนินการกับคอลเลกชันทั้งหมด ในขณะที่แบบหลังจำกัดการดำเนินการเฉพาะข้อมูลที่เกี่ยวข้องกับผู้ใช้เท่านั้นครับ/ค่ะ

## สิทธิ์การดำเนินการของคอลเลกชัน

![](https://static-docs.nocobase.com/6a6e0281391cecdea5b5218e6173c5d7.png)

![](https://static-docs.nocobase.com/9814140434ff9e1bf028a6c282a5a165.png)

สิทธิ์การดำเนินการของคอลเลกชันช่วยให้สามารถปรับแต่งสิทธิ์การดำเนินการแบบทั่วโลกได้อย่างละเอียด โดยการกำหนดค่าการเข้าถึงทรัพยากรภายในแต่ละคอลเลกชันครับ/ค่ะ สิทธิ์เหล่านี้แบ่งออกเป็นสองส่วนครับ/ค่ะ

1.  **สิทธิ์การดำเนินการ**: ซึ่งรวมถึงการเพิ่ม, ดู, แก้ไข, ลบ, ส่งออก และนำเข้าข้อมูลครับ/ค่ะ สิทธิ์เหล่านี้จะถูกตั้งค่าตามขอบเขตข้อมูลครับ/ค่ะ
    -   **ข้อมูลทั้งหมด**: อนุญาตให้ผู้ใช้ดำเนินการกับระเบียนทั้งหมดภายในคอลเลกชันได้ครับ/ค่ะ
    -   **ข้อมูลของตนเอง**: จำกัดให้ผู้ใช้สามารถดำเนินการเฉพาะกับระเบียนที่ตนเองสร้างขึ้นเท่านั้นครับ/ค่ะ

2.  **สิทธิ์ฟิลด์**: สิทธิ์ฟิลด์ช่วยให้คุณสามารถตั้งค่าสิทธิ์เฉพาะสำหรับแต่ละฟิลด์ในการดำเนินการที่แตกต่างกันได้ครับ/ค่ะ ตัวอย่างเช่น บางฟิลด์สามารถกำหนดค่าให้เป็นแบบดูได้อย่างเดียว โดยไม่มีสิทธิ์ในการแก้ไขครับ/ค่ะ

## สิทธิ์การเข้าถึงเมนู

สิทธิ์การเข้าถึงเมนูจะควบคุมการเข้าถึงโดยอิงตามรายการเมนูครับ/ค่ะ

![](https://static-docs.nocobase.com/28eddfc843d27641162d9129e3b6e33f.png)

## สิทธิ์การกำหนดค่าปลั๊กอิน

สิทธิ์การกำหนดค่าปลั๊กอินใช้เพื่อควบคุมความสามารถในการกำหนดค่าพารามิเตอร์ของปลั๊กอินที่เฉพาะเจาะจงครับ/ค่ะ เมื่อเปิดใช้งาน หน้าจอการจัดการปลั๊กอินที่เกี่ยวข้องจะปรากฏขึ้นในศูนย์การดูแลระบบครับ/ค่ะ

![](https://static-docs.nocobase.com/5a742ae20a9de93dc2722468b9fd7475.png)
