# NocoBase Inbedden

## Introductie

Deze plugin gebruikt u om NocoBase-pagina's in andere websites of applicaties in te bedden.

## Installatie

Deze plugin is een commerciÃ«le plugin. U dient deze te uploaden en te activeren via de pluginbeheerder.

![20240323162741](https://static-docs.nocobase.com/20240323162741.png)

## Gebruik

Nadat u de interfaceconfiguratie hebt geactiveerd, klikt u op 'Kopieer insluitlink' in het instellingenmenu van de pagina (rechtsboven).

![](https://static-docs.nocobase.com/f11bd6d5e88d38731d7cd3cb149022c8.png)

Na het klikken ontvangt u een link die afzonderlijk kan worden geopend, bijvoorbeeld `https://xxx.com/embed/qs087rz4o2b`.

![](https://static-docs.nocobase.com/9d847805a00fd07372a474665ec0e968.png)

Als u NocoBase-pagina's in andere websites of applicaties wilt inbedden, is gebruikersauthenticatie vereist. De `token` moet dan aan de link worden toegevoegd, bijvoorbeeld `https://xxx.com/embed/qs087rz4o2b?token=xxx`.
