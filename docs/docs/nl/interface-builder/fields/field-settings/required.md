# Verplicht

## Introductie

"Verplicht" is een veelgebruikte regel voor formuliervalidatie. U kunt deze direct inschakelen in de veldconfiguratie, of een veld dynamisch als verplicht instellen via de koppelingsregels van het formulier.

## Waar kunt u een veld als verplicht instellen?

### Instellingen voor collectievelden

Wanneer een collectieveld als verplicht is ingesteld, activeert dit backend-validatie en wordt het aan de frontend standaard als verplicht weergegeven (niet te wijzigen).
![20251025175418](https://static-docs.nocobase.com/20251025175418.png)

### Veldinstellingen

Stel een veld direct in als verplicht. Dit is geschikt voor velden die de gebruiker altijd moet invullen, zoals gebruikersnaam, wachtwoord, enz.

![20251028222818](https://static-docs.nocobase.com/20251028222818.png)

### Koppelingsregels

Stel een veld als verplicht in op basis van voorwaarden via de koppelingsregels van het formulierblok.

Voorbeeld: Het ordernummer is verplicht wanneer de orderdatum niet leeg is.

![20251028223004](https://static-docs.nocobase.com/20251028223004.png)
