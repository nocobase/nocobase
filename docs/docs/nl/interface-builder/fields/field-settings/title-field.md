# Titelveld

## Introductie

Het **Titelveld** wordt doorgaans gebruikt in relatieveldcomponenten als een identificatie voor de gerelateerde gegevens die op de interface worden weergegeven. Het bepaalt hoe de belangrijkste informatie van de gegevens wordt gepresenteerd, en helpt gebruikers meestal om de gegevens snel te herkennen en te begrijpen.

Verschillende soorten titelvelden komen overeen met verschillende veldcomponenten en ondersteunen bijbehorende configuratieopties. Bijvoorbeeld:

- Wanneer het titelveld een **Datumveld** is, gebruikt de veldcomponent een **Datumcomponent** om de gegevens weer te geven.
- Wanneer het titelveld een **Tekstveld** is, gebruikt de veldcomponent een **Tekstcomponent** om de gegevens weer te geven.
- Wanneer het titelveld een **Optieveld** is, gebruikt de veldcomponent een **Optiecomponent** om de gegevens weer te geven.
