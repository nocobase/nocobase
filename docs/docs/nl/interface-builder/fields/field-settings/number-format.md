# Getalnotatie

## Introductie

Getalnotatie verwijst naar het visueel verbeteren en standaardiseren van numerieke velden (inclusief formulevelden) in de leesmodus. Dit omvat instellingen zoals duizendtalscheidingstekens, decimale precisie, eenheidsconversie en wetenschappelijke notatie. Door getalnotatie te configureren, kunt u voldoen aan de numerieke weergavegewoonten van verschillende regio's en sectoren.

## Hoe te gebruiken

- Ondersteunt eenvoudige eenheidsconversie (vermenigvuldigen/delen)
- Duizendtalscheidingsteken
- Voor- en achtervoegsel
- Precisie
- Wetenschappelijke notatie.
