# Validatieregels instellen

## Introductie

Validatieregels zorgen ervoor dat de gegevens die gebruikers invoeren, voldoen aan de verwachtingen.

## Waar u veldvalidatieregels kunt instellen

### Validatieregels configureren voor collectievelden

De meeste velden ondersteunen het configureren van validatieregels. Zodra een veld is geconfigureerd met validatieregels, wordt er bij het indienen van gegevens een backend-validatie geactiveerd. Verschillende veldtypen ondersteunen verschillende validatieregels.

- **Datumveld**

  ![20251028225946](https://static-docs.nocobase.com/20251028225946.png)

- **Numeriek veld**

  ![20251028230418](https://static-docs.nocobase.com/20251028230418.png)

- **Tekstveld**

  Naast het beperken van de tekstlengte, ondersteunen tekstvelden ook aangepaste reguliere expressies voor een verfijndere validatie.

  ![20251028230554](https://static-docs.nocobase.com/20251028230554.png)

### Frontend-validatie in veldconfiguratie

Validatieregels die u in de veldconfiguratie instelt, activeren frontend-validatie om ervoor te zorgen dat de gebruikersinvoer voldoet aan de voorschriften.

![20251028230105](https://static-docs.nocobase.com/20251028230105.png)

![20251028230255](https://static-docs.nocobase.com/20251028230255.png)

**Tekstvelden** ondersteunen ook aangepaste regex-validatie om te voldoen aan specifieke formaatvereisten.

![20251028230903](https://static-docs.nocobase.com/20251028230903.png)
