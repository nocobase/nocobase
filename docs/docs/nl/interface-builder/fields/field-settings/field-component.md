# Veldcomponent

## Introductie

Veldcomponenten worden gebruikt om de inhoud van een veld weer te geven.
Sommige velden hebben meerdere ingebouwde weergavecomponenten. U kunt de juiste component kiezen op basis van uw specifieke behoeften, om de waarde van het veld op verschillende manieren te presenteren.

Voorbeeld: In de weergavemodus kunt u een URL-veld schakelen naar een Voorbeeldcomponent of een URL-component (standaard).
