# Weergavemodus

## Introductie

In tegenstelling tot blokken hebben veldcomponenten drie weergavemodi (dit wordt alleen ondersteund voor velden binnen een formulier). Wanneer u wisselt tussen deze weergavemodi, corresponderen deze met verschillende configuratieopties voor het veld.

- Bewerkbaar;
- Alleen-lezen (niet bewerkbaar);
- Alleen-lezen (leesmodus);

### Weergavemodi voor relatievelden

De **weergavemodus van een relatieveld** bepaalt hoe het veld in de interface wordt weergegeven en welke typen veldcomponenten beschikbaar zijn.

In de **bewerkbare status** ondersteunen relatievelden meerdere componenttypen, zodat u verschillende relatieveldcomponenten kunt kiezen om gerelateerde gegevens weer te geven of te selecteren, afhankelijk van uw bedrijfsbehoeften.
