# Datum & Tijd

## Introductie

Het datumveld ondersteunt aangepaste datum- en tijdformaten.

## Waar u het datumformaat kunt instellen

### Collectievelden

![20240417114311](https://static-docs.nocobase.com/20240417114311.png)

> Het datumformaat dat is ingesteld in het collectieveld, dient als het standaardformaat voor dat veld. Dit kan worden overschreven door het datumformaat in de veldinstellingen binnen de blokconfiguratie.
