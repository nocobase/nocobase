# Subdetails

## Introductie

Subdetails is de tegenhanger van het subformulier in de leesmodus. Vergeleken met de label- en titelcomponenten, kunnen subdetails niet alleen meer gegevens van de huidige collectie weergeven, maar kunt u ze ook configureren om gegevens van gerelateerde collecties te tonen. Meerlaagse relatiegegevens worden duidelijk weergegeven in de vorm van geneste details.
