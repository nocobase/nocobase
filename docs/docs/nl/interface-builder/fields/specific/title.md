# Titel

## Introductie

In de leesmodus worden associatievelden standaard weergegeven met behulp van de **Titelveldcomponent**. Deze component identificeert de gekoppelde record door de waarde van het titelveld te tonen. Het systeem matcht en rendert automatisch de bijbehorende titelveldcomponent op basis van het type titelveld.
