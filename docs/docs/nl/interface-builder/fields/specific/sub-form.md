# Subformulier

## Introductie

Subformulieren zijn geschikt voor scenario's waarin u eerst nieuwe gerelateerde gegevens aanmaakt voordat u deze koppelt. Meerlaagse gerelateerde gegevens worden duidelijk weergegeven als een genest formulier. In vergelijking met de Data Selector en Dropdown-componenten, kunt u met een subformulier de velden van de gerelateerde collectie direct beheren binnen het huidige paginablok, en de gerelateerde gegevens worden samen met het hoofdformulier ingediend.
