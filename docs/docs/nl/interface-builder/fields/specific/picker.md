# Gegevenskiezer

## Introductie

De Gegevenskiezer, uitgevoerd als een pop-up tabelkiezer, stelt u in staat om bestaande gegevens uit de doelcollectie te selecteren voor koppeling, of om nieuwe gegevens aan de doelcollectie toe te voegen en deze vervolgens te koppelen.

![20251027203343](https://static-docs.nocobase.com/20251027203343.png)

Geselecteerde gegevens worden niet weergegeven in de tabelkiezer.

![20251027203717](https://static-docs.nocobase.com/20251027203717.gif)

Binnen de tabelkiezer kunt u de doelcollectie voor koppeling ook verder beheren (bijvoorbeeld gegevens toevoegen, verwijderen, importeren of exporteren).

![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
