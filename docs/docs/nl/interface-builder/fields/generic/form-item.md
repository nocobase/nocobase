# Formuliervelden

## Introductie
Verschillende typen velden in een formulier hebben elk hun eigen configuratieopties. Gebruikers kunnen deze opties uitbreiden via **plugins** om zo aan meer gepersonaliseerde behoeften te voldoen.

![20251025174555](https://static-docs.nocobase.com/20251025174555.png)
