# Bevestiging

## Introductie

De bevestigingsinstellingen worden gebruikt om te controleren of een actie moet worden uitgevoerd. Wanneer u op een actie klikt, verschijnt er een bevestigingsvenster. De verwijderknop heeft standaard een bevestiging ingeschakeld om gegevensverlies of onomkeerbare gevolgen door onbedoelde handelingen te voorkomen.

![20251029125021](https://static-docs.nocobase.com/20251029125021.png)

U kunt de inhoud en titel van de bevestiging aanpassen, en ook variabelen gebruiken.
