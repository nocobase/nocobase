# Veldtoewijzing

## Introductie

Veldtoewijzing houdt in dat u de waarde van een veld vooraf instelt voordat u een actie uitvoert. Wanneer u een record bijwerkt of aanmaakt, worden deze gegevens automatisch ingevuld, zonder dat u ze handmatig hoeft in te voeren. Denk hierbij aan scenario's zoals het in bulk bijwerken van orderstatussen, of het automatisch instellen van de levertijd op de huidige tijd wanneer een bestelling is afgeleverd.
