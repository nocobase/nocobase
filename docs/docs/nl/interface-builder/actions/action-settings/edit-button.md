# Knop bewerken

## Introductie

Het systeem ondersteunt het aanpassen van de naam, het pictogram, de achtergrondkleur en het type van een knop. Dit biedt flexibele UI-aanpassingsopties, zodat u de weergavestijl en functionaliteit van knoppen kunt aanpassen aan uw specifieke behoeften.

- **Knopnaam**: U kunt de tekst van de knop aanpassen aan uw bedrijfsbehoeften, om ervoor te zorgen dat het label van de knop duidelijk en intu√Øtief is.
- **Knopicoon**: U kunt een pictogram instellen voor de knop om de visuele herkenbaarheid te verbeteren, zodat u de functie van de knop snel kunt identificeren.
- **Gevaarlijke actie**: Wanneer een knop een **gevaarlijke actie** vertegenwoordigt (zoals verwijderen, resetten, enz.), kunt u deze instellen op het type `Danger Action`.
- **Knoptype (Button Type)**: Knoppen ondersteunen verschillende type-instellingen. Veelvoorkomende knoptypen zijn:
  - **Standaard (Default)**: Een standaardknop, geschikt voor normale acties.
  - **Primair (Primary)**: Wordt gebruikt voor belangrijke of primaire acties, meestal met een opvallend visueel effect.
  - **Gestippeld (Dashed)**: Een knop met een gestippelde rand, geschikt voor secundaire acties of hint-achtige knoppen.
  - **Tekst (Text)**: Een knop met alleen tekst en geen achtergrondkleur, doorgaans gebruikt voor lichte acties of links.
