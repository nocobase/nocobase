# Subrecord toevoegen

## Introductie

Deze bewerking is speciaal ontworpen voor **boomtabelblokken** om snel subrecords toe te voegen. Wanneer u een subrecord toevoegt, wordt de geselecteerde rij behandeld als het bovenliggende knooppunt, zodat u eenvoudig subrecords kunt aanmaken onder elk bovenliggend knooppunt. Deze functionaliteit vereenvoudigt het beheer van hiërarchische gegevens aanzienlijk, waardoor het uitbreiden en organiseren van datastructuren efficiënter en intuïtiever wordt.
