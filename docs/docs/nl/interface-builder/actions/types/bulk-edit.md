# Bulk-bewerken

## Inleiding

Bulk-bewerken is geschikt voor scenario's waarin gegevens flexibel in batch moeten worden bijgewerkt. Nadat u op de knop "Bulk-bewerken" heeft geklikt, kunt u het bulk-bewerkingsformulier configureren in een pop-upvenster en verschillende bijwerkstrategieën instellen voor elk veld.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM.png)


## Actieconfiguratie

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM%20(1).png)


## Gebruikershandleiding

### Configuratie van het bulk-bewerkingsformulier

1. Voeg een knop "Bulk-bewerken" toe.

2. Stel het bereik voor bulk-bewerken in: Geselecteerd / Alle, standaard is Geselecteerd.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_14_AM.png)

3. Voeg een bulk-bewerkingsformulier toe.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_14_AM.png)

4. Configureer de velden die bewerkt moeten worden en voeg een verzendknop toe.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM%20(1).png)

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM.png)

### Formulier indienen

1. Selecteer de rijen met gegevens die u wilt bewerken.

2. Selecteer de bewerkingsmodus voor de velden en vul de waarden in die moeten worden ingediend.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_10_33_AM.png)

:::info{title=Beschikbare bewerkingsmodi}
* **Niet bijwerken**: Het veld blijft ongewijzigd.
* **Wijzigen naar**: Werk het veld bij naar de ingediende waarde.
* **Wissen**: Wis de gegevens in het veld.

:::

3. Dien het formulier in.
