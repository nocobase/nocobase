# Filter

## Introductie

De filteractie verschijnt meestal in datablocks. Hiermee kunt u gegevens filteren door verschillende voorwaarden te configureren. Houd er rekening mee dat als er al een gegevensbereik is geconfigureerd voor het block, de filteractie wordt gecombineerd met de voorwaarden van dat gegevensbereik als de uiteindelijke filtervoorwaarde.

## Actie-instellingen

![20251029173708](https://static-docs.nocobase.com/20251029173708.png)

### Filterbare velden

Standaard zijn alle velden van deze collectie ingeschakeld. Alleen ingeschakelde velden kunnen worden geselecteerd in de lijst met voorwaardenvelden.

Velden van gerelateerde collecties kunnen ook als filtervelden worden gebruikt.
