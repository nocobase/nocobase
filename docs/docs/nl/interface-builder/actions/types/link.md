# Link

## Inleiding

De link-actie werkt via routenavigatie en ondersteunt het doorgeven van URL-variabelen. Hierdoor kan de doelpagina de inhoud dynamisch aanpassen op basis van de doorgegeven gegevens. U kunt deze actie configureren en gebruiken in datablokken.
