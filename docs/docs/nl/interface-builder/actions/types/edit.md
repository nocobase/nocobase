# Bewerken

## Introductie

De actie 'Bewerken' is een rijknop die de huidige record als context heeft. Deze verschijnt als een pop-up waarin u blokken kunt aanpassen en configureren.

![20251029165442](https://static-docs.nocobase.com/20251029165442.png)

Configureer blokken naar behoefte.

![20251029165514](https://static-docs.nocobase.com/20251029165514.png)

## Actie-instellingen

![20251029165620](https://static-docs.nocobase.com/20251029165620.png)

- [Koppelingsregels](/interface-builder/actions/action-settings/linkage-rule): Toon/verberg de knop dynamisch;
- [Bewerkingsknop](/interface-builder/actions/action-settings/edit-button): Bewerk de titel, het type en het pictogram van de knop;
- [Bewerkingspop-up](/interface-builder/actions/action-settings/edit-popup): Stel de openingsmodus en de grootte in.
