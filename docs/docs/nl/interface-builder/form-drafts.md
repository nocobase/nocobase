# Formulierconcepten

Wanneer formulierconcepten zijn ingeschakeld, wordt de ingevoerde inhoud automatisch lokaal opgeslagen. Zelfs als het formulier per ongeluk wordt gesloten of de pagina wordt vernieuwd, gaan de ingevulde gegevens niet verloren.

![20251213093717](https://static-docs.nocobase.com/20251213093717.png)

Wanneer er een concept bestaat, wordt er bovenaan het formulier een melding weergegeven. Na een succesvolle indiening wordt het concept automatisch verwijderd; als u het niet meer nodig heeft, kunt u het ook op elk gewenst moment handmatig verwijderen.

![20251213092734](https://static-docs.nocobase.com/20251213092734.png)

::: tip Tip

Momenteel ondersteunen formulierconcepten alleen lokale caching. Toekomstige versies zullen caching aan de serverzijde ondersteunen om synchronisatie tussen meerdere apparaten mogelijk te maken.

:::
