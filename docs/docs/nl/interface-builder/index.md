# Interface opbouwen

## Wat u ziet, is wat u krijgt (WYSIWYG)

NocoBase biedt een 'Wat u ziet, is wat u krijgt' (WYSIWYG) ervaring voor het opbouwen van de interface. Klik op de knop 'Interface opbouwen' om te schakelen tussen de configuratiemodus en de weergavemodus.

## Lay-outsjabloon

NocoBase wordt geleverd met een standaard lay-outsjabloon: navigatie bovenaan en links, en een inhoudsgebied rechts.

![未命名.002](https://static-docs.nocobase.com/未命名.002.jpeg)

## Configuratieopties

Zodra u de interfacebouwmodus activeert, verschijnen er oranje markeringen die configureerbare elementen aangeven. De toegang tot de instellingen van een element vindt u meestal in de rechterbovenhoek van dat element. Bijna alle elementen kunt u direct in de interface configureren en het effect ervan in realtime bekijken. Hier zijn enkele voorbeelden:
