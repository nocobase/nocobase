# Modus voor gegevensladen

## Inleiding

De standaardmodus voor het laden van gegevens is "Laad alle gegevens wanneer filtervoorwaarden leeg zijn". Hier verwijzen "filtervoorwaarden" naar de waarden in het "Filterblok". Een andere optie is "Geen gegevens laden wanneer filtervoorwaarden leeg zijn".
