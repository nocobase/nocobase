# Gegevensbereik instellen

## Introductie

Het instellen van een gegevensbereik betekent dat u standaard filtervoorwaarden definieert voor een gegevensblok. Gebruikers kunnen het gegevensbereik flexibel aanpassen aan hun zakelijke behoeften. Ongeacht de uitgevoerde filteracties, past het systeem deze standaard filtervoorwaarde automatisch toe, zodat de gegevens altijd binnen de grenzen van het opgegeven bereik blijven.

### Operatoren

Verschillende typen velden ondersteunen verschillende operatoren. Tekstvelden ondersteunen bijvoorbeeld operatoren zoals 'gelijk aan', 'niet gelijk aan' en 'bevat'; numerieke velden ondersteunen operatoren zoals 'groter dan' en 'kleiner dan'; terwijl datumvelden operatoren ondersteunen zoals 'binnen bereik' en 'vóór een specifieke datum'.
