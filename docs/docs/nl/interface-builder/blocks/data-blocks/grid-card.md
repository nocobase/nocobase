# Rasterkaart

## Introductie

Het Rasterkaart-blok toont samenvattende informatie van datarecords in een kaartformaat. U kunt het aantal kolommen configureren voor verschillende schermformaten, om zo een gebruiksvriendelijke weergave op diverse apparaten te garanderen.
