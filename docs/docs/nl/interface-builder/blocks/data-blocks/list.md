# Lijstblok

## Introductie

Het Lijstblok toont gegevens in een lijstvorm, ideaal voor het weergeven van data zoals takenlijsten, nieuwsberichten en productinformatie.

## Blokconfiguratie

![20251023202835](https://static-docs.nocobase.com/20251023202835.png)

### Gegevensbereik instellen

Zoals afgebeeld: Filter op bestellingen met de status 'Geannuleerd'.

![20251023202927](https://static-docs.nocobase.com/20251023202927.png)

Voor meer details, zie [Gegevensbereik instellen](/interface-builder/blocks/block-settings/data-scope)

### Sorteerregels instellen

Zoals afgebeeld: Sorteer op orderbedrag in aflopende volgorde.

![20251023203022](https://static-docs.nocobase.com/20251023203022.png)

Voor meer details, zie [Sorteerregels instellen](/interface-builder/blocks/block-settings/sorting-rule)

## Velden configureren

### Velden van deze collectie

> **Let op**: Velden van overgeërfde collecties (d.w.z. bovenliggende collecties) worden automatisch samengevoegd en weergegeven in de huidige lijst met velden.

![20251023203103](https://static-docs.nocobase.com/20251023203103.png)

### Velden van gerelateerde collecties

> **Let op**: Velden van gerelateerde collecties kunnen worden weergegeven (momenteel worden alleen één-op-één relaties ondersteund).

![20251023203611](https://static-docs.nocobase.com/20251023203611.png)

Voor de configuratie van lijstvelden, zie [Detailveld](/interface-builder/fields/generic/detail-form-item)
