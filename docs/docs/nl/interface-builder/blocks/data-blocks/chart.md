# Grafiek

## Introductie

De gegevensvisualisatie-plugin van NocoBase biedt visuele gegevensquery's en een uitgebreide set grafiekcomponenten. Gebruikers kunnen via een eenvoudige configuratie snel visualisatie-dashboards bouwen om gegevensinzichten weer te geven, en de plugin ondersteunt multidimensionale gegevensanalyse en -presentatie.
