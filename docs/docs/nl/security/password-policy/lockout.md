# Gebruikersvergrendeling

## Introductie

Beheer vergrendelde gebruikers en vergrendel gebruikers proactief.

![](https://static-docs.nocobase.com/202412281450083.png)

### Gebruiker ontgrendelen

Gebruikers die zijn vergrendeld omdat ze de limiet voor ongeldige wachtwoordpogingen hebben overschreden, kunt u ontgrendelen door de betreffende record te verwijderen.

### Gebruiker vergrendelen

U kunt een gebruiker proactief vergrendelen door een gebruikersrecord toe te voegen. Zodra een gebruiker is vergrendeld, wordt de toegang tot het systeem via elke authenticatiemethode, inclusief API-sleutels, verboden.

![](https://static-docs.nocobase.com/202412281450512.png)
