# Boomstructuur collectie

## Introductie

Een boomstructuur collectie is een ontwerppatroon voor een gegevenscollectie dat wordt gebruikt voor het organiseren van hiërarchische gegevens. Deze collectiestructuur weerspiegelt een boom, waarbij elk gegevenselement één of meer onderliggende elementen kan hebben, en deze onderliggende elementen op hun beurt weer hun eigen onderliggende elementen kunnen hebben.
