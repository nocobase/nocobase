# Aliyun OSS

Opslag-engine gebaseerd op Aliyun OSS. Voordat u deze gebruikt, moet u de benodigde accounts en machtigingen voorbereiden.

## Configuratie

![Voorbeeld van Aliyun OSS-configuratie](https://static-docs.nocobase.com/20240712220011.png)

:::info{title=Tip}
Hier worden alleen de specifieke parameters voor de Aliyun OSS opslag-engine behandeld. Voor algemene parameters verwijzen we u naar [Algemene engineparameters](./index.md#common-engine-parameters).
:::

### Regio

Vul de regio van de OSS-opslag in, bijvoorbeeld: `oss-cn-hangzhou`.

:::info{title=Tip}
U kunt de regiogegevens van de opslag-bucket bekijken in de [Aliyun OSS-console](https://oss.console.aliyun.com/). U hoeft alleen het voorvoegsel van de regio te gebruiken (zonder de volledige domeinnaam).
:::

### AccessKey ID

Vul de ID in van de geautoriseerde toegangssleutel van Alibaba Cloud.

### AccessKey Secret

Vul het secret in van de geautoriseerde toegangssleutel van Alibaba Cloud.

### Opslag-bucket

Vul de naam van de OSS-opslag-bucket in.
