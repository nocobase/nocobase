# Bestands-collectie

## Introductie

Bestanden zijn records in een speciaal gestructureerde collectie, die we een bestands-collectie noemen. Deze wordt gebruikt om metadata van bestanden op te slaan en kan worden beheerd via de Bestandsbeheerder.
