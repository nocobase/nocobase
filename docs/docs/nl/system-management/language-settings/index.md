# Taalinstellingen

Via de 'Systeeminstellingen' kunt u de globale taalinstellingen van het systeem aanpassen. Configureer hiervoor de 'Ingeschakelde talen'. De eerste taal in deze lijst wordt automatisch de standaardtaal.

![20240324192005](https://static-docs.nocobase.com/20240324192005.png)

Wanneer er meerdere talen zijn ingeschakeld, kunnen gebruikers hun persoonlijke taalvoorkeur instellen in het Persoonlijk Centrum (rechtsboven op de pagina).

![20240324192509](https://static-docs.nocobase.com/20240324192509.png)

Als de huidige taalomgeving geen vertalingen bevat, kunt u de plugin "[Lokalisatiebeheer](/system-management/localization/)" gebruiken om vertalingen toe te voegen of aan te vullen.

![20240324193055](https://static-docs.nocobase.com/20240324193055.png)
