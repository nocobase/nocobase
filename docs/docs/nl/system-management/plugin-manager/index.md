# Pluginbeheer

## Pluginbeheerder

Klik op het pluginbeheerder-icoon rechtsboven op de pagina om de pluginbeheerder-pagina te openen. Op deze pagina kunt u plugins bekijken, installeren, inschakelen of uitschakelen.

![2024-04-10_08-47-45](https://static-docs.nocobase.com/2024-04-10_08-47-45.png)

## Plugininstellingencentrum

Plugins hebben meestal hun eigen instellingenpagina's. Klik op het instellingen-icoon rechtsboven om de instellingenmenu's van alle plugins te bekijken. Klik op het betreffende menu-item om de plugin te configureren.

![2024-04-10_08-52-41](https://static-docs.nocobase.com/2024-04-10_08-52-41.png)
