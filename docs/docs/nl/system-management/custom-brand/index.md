# Merk aanpassen

## Introductie

Het standaard merk is "NocoBase" en wordt op de volgende drie plaatsen weergegeven:

![20240409111100](https://static-docs.nocobase.com/20240409111100.png)

Onderaan de inlogpagina:

![20240409111254](https://static-docs.nocobase.com/20240409111254.png)

Positie 1 configureert u via de optie 'Logo' in de [Systeeminstellingen](/system-management/system-settings/). Posities 2 en 3 kunt u met deze plugin instellen.

## Gebruiksaanwijzing

Nadat u de plugin heeft geactiveerd, klikt u op het menu 'Merk aanpassen' rechtsboven op de pagina om naar de configuratiepagina te gaan.

![20240409113640](https://static-docs.nocobase.com/20240409113640.png)

- **Merk**: Hiermee stelt u de tekst onderaan de inlogpagina in (positie 3).
- **Over**: Hiermee configureert u de inhoud van het menu rechtsboven (zoals weergegeven in de afbeelding hieronder).

![20240409113915](https://static-docs.nocobase.com/20240409113915.png)

De configuratie ondersteunt HTML-formaat voor flexibele aanpassing van de inhoud. Als u alleen eenvoudige tekstwijzigingen wilt aanbrengen, kunt u de bestaande inhoud direct aanpassen, zoals hieronder weergegeven:

![20240409114740](https://static-docs.nocobase.com/20240409114740.png)
