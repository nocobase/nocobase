# E-mailhandtekening

## Handtekening configureren

Ga naar het E-mailbeheercentrum en klik op de instellingenknop:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_05_34_PM.png)

Schakel over naar het handtekeningenpaneel:
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_05_51_PM.png)

Hier kunt u uw handtekeningen beheren. Vink de optie vóór een handtekening aan om deze als standaard in te stellen:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_05_37_PM.png)

## Handtekeningen gebruiken

Klik in het e-mailtekstvak op het handtekeningpictogram om de gewenste handtekening te selecteren:
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_05_40_PM.png)
