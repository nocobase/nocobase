# Bulkverzending

## Overzicht bulkverzending
U kunt e-mails in bulk naar meerdere personen verzenden (elke ontvanger krijgt een aparte e-mail).
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_06_10_PM.png)

## Bulkverzending via een tabel
U kunt e-mails ook in bulk verzenden via een tabel:

1. Voeg een 'E-mail verzenden'-knop toe aan de tabel
   ![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_14_PM.png)

2. Configureer het e-mailveld in de tabel (dit wordt gebruikt om e-mailadressen te parseren nadat u tabelrijen heeft geselecteerd)
   ![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_13_PM.png)

3. Vink de tabelgegevens aan om e-mails te verzenden
   ![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_15_PM.png)

Als u e-mails wilt verzenden naar alle adressen in de hele tabel, kunt u de functie **Standaard alle records in de huidige tabel als ontvangers selecteren** inschakelen:
![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_16_PM.png)

Nadat u de e-mailinhoud heeft bewerkt, klikt u op de knop 'Bulk verzenden' om de e-mails te versturen:
![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_17_PM.png)
