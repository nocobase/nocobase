# Opslag-engine: Amazon S3

Een opslag-engine gebaseerd op Amazon S3. Voordat u deze kunt gebruiken, moet u de relevante accounts en machtigingen voorbereiden.

## Configuratieparameters

![Voorbeeld van Amazon S3 Opslag-engine configuratie](https://static-docs.nocobase.com/20251031092524.png)

:::info{title=Opmerking}
Deze sectie behandelt alleen de specifieke parameters voor de Amazon S3 opslag-engine. Voor algemene parameters verwijzen we u naar [Algemene engineparameters](./index#引擎通用参数).
:::

### Regio

Vul de S3 opslagregio in, bijvoorbeeld: `us-west-1`.

:::info{title=Opmerking}
U kunt de regio-informatie van uw bucket bekijken in de [Amazon S3-console](https://console.aws.com/s3/). U hoeft alleen het regiocodedeel te gebruiken (niet de volledige domeinnaam).
:::

### AccessKey ID

Vul de AccessKey ID van Amazon S3 in.

### AccessKey Secret

Vul de AccessKey Secret van Amazon S3 in.

### Bucket

Vul de naam van de S3 bucket in.
