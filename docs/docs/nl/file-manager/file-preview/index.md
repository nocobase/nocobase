# Bestandsvoorvertoning

In interfaces die bestandsvelden bevatten (inclusief bijlagevelden), kunt u bestanden bekijken door op de miniatuur of het pictogram van het bestand te klikken. De ingebouwde voorvertoningsfunctie ondersteunt verschillende bestandstypen, waaronder afbeeldingen, PDF's en de meeste bestandstypen die standaard door browsers worden ondersteund.

![20251129232307](https://static-docs.nocobase.com/20251129232307.png)

Voor bestandstypen die geen standaard voorvertoning ondersteunen, kunt u de voorvertoningsfunctionaliteit inschakelen door de bijbehorende plugins voor bestandsvoorvertoning te installeren of uit te breiden. Na het installeren van de Office-bestandsvoorvertoning-plugin kunt u bijvoorbeeld Word-, Excel- en PowerPoint-bestanden bekijken.

Momenteel biedt NocoBase de volgende plugins voor bestandsvoorvertoning aan:
