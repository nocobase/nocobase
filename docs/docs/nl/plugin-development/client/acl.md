# ACL Toegangscontrole

:::tip

Dit hoofdstuk beschrijft hoe u ACL (Access Control List) gebruikt voor rechtenbeheer in frontend plugins. De inhoud omvat:

- Rechten controleren;
- Hoe u inhoud dynamisch rendert op basis van rechten;

In toekomstige versies vullen we dit aan met gedetailleerde voorbeelden en best practices.

:::
