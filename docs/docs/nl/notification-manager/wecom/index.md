# Notificatie: WeCom

## Introductie

De **WeCom** plugin stelt de applicatie in staat om notificatieberichten te versturen naar WeCom-gebruikers.

## WeCom-authenticatie toevoegen en configureren

Eerst moet u een WeCom-authenticatie toevoegen en configureren in NocoBase. Raadpleeg [Gebruikersauthenticatie - WeCom](/auth-verification/auth-wecom). Alleen systeemgebruikers die zijn ingelogd via WeCom kunnen systeemnotificaties ontvangen via WeCom.
