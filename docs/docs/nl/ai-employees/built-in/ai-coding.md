# AI Medewerker · Nathan: Frontend Engineer

> Helpt u bij het schrijven en optimaliseren van frontend code om complexe interactieve logica te implementeren.

`Nathan` is dé frontend ontwikkelingsspecialist binnen NocoBase. In situaties waar u JavaScript moet schrijven, zoals bij `JSBlock`, `JSField`, `JSItem`, `JSColumn`, `JSAction`, `Event Flow` en `Linkage`, ziet u `Nathan`'s avatar rechtsboven in de code-editor. U kunt hem dan vragen om de code in de editor voor u te schrijven of aan te passen.

![20251022163405](https://static-docs.nocobase.com/20251022163405.png)

## Stappen

Klik in de code-editor op `Nathan` om het AI Medewerker dialoogvenster te openen. De code uit de editor wordt automatisch aan het invoerveld toegevoegd en als applicatiecontext naar `Nathan` gestuurd.

![20251022163935](https://static-docs.nocobase.com/20251022163935.png)

Voer uw codeervereisten in en stuur deze naar `Nathan`. Wacht vervolgens op zijn antwoord.

![20251022164041](https://static-docs.nocobase.com/20251022164041.png)

Klik op de knop `Apply to editor` in het codeblok van `Nathan`'s antwoord om zijn code over te nemen in de code-editor.

![20251022164323](https://static-docs.nocobase.com/20251022164323.png)

Klik op de `Run`-knop in de code-editor om het resultaat direct te bekijken.

![20251022164436](https://static-docs.nocobase.com/20251022164436.png)

## Codegeschiedenis

Klik op het 'commandoregel'-icoon rechtsboven in het dialoogvenster van `Nathan` om de codefragmenten te bekijken die u hebt verzonden en de fragmenten die `Nathan` heeft beantwoord tijdens de huidige sessie.
