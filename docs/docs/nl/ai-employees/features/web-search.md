# Zoeken op het web

Zoeken op het web wordt gebruikt om de trainingsgegevens van het model aan te vullen met de meest recente informatie.

## Hoe het werkt

Of zoeken op het web beschikbaar is, hangt af van de vraag of de geselecteerde modelservice van de huidige sessie Web Search ondersteunt.

- **Ondersteund**: de schakelaar voor zoeken op het web wordt weergegeven en kan naar wens worden in- of uitgeschakeld.
- **Niet ondersteund**: de schakelaar wordt niet weergegeven en de zoekstatus wordt automatisch uitgeschakeld.

## Gebruik in chats

Gebruik de schakelaar voor zoeken op het web in het invoerveld van het gesprek:

- **Ingeschakeld**: de AI kan eerst informatie opzoeken en vervolgens antwoorden.

![web-search-switch-visible-on-supported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-visible-on-supported-model.png)

- **Uitgeschakeld**: de AI antwoordt uitsluitend op basis van de bestaande context.

![web-search-switch-hidden-on-unsupported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-hidden-on-unsupported-model.png)

## Verschillen per platform

De ondersteuning voor Web Search verschilt per model. Gebruik deze functie op basis van de feitelijke mogelijkheden van de door u geselecteerde modelservice en het model.
