:::warning Omgevingsvariabelen

- `TZ` wordt gebruikt om de tijdzone van de applicatie in te stellen. Deze is standaard de tijdzone van het besturingssysteem.
- `APP_KEY` is de geheime sleutel van de applicatie, gebruikt voor het genereren van gebruikerstokens, enz. (Als de `APP_KEY` wordt gewijzigd, worden oude tokens ongeldig). Het kan elke willekeurige tekenreeks zijn. **Wijzig deze in uw eigen geheime sleutel en zorg ervoor dat deze niet openbaar wordt gemaakt.**
- `DB_*` zijn databaserelateerde configuraties. Pas deze aan uw daadwerkelijke databaseverbindingsinformatie aan.

:::
