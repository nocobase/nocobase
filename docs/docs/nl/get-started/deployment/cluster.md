# Clusterimplementatie

NocoBase ondersteunt het implementeren van applicaties in clustermodus om de prestaties en beschikbaarheid van de applicatie te verbeteren. Clusterimplementatie stelt u in staat om verschillende instanties van de applicatie op meerdere servers te draaien, of om meerdere processen in multi-core modus op één server uit te voeren.

Voor meer details, raadpleeg alstublieft de [Clustermodus](/cluster-mode/) documentatie.
