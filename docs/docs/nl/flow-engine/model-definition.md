# China Regio Veld

## Introductie

De China Regio Veld plugin (field-china-region) voegt het veldtype "China Regio" toe aan NocoBase.

## Installatie

Ga in de hoofdinterface van NocoBase naar "Pluginbeheer", zoek de "China Regio" plugin en schakel deze in.

## Gebruik

Voeg in de collectieconfiguratie een nieuw veld toe en selecteer "China Regio" als veldtype.

![](/images/field-china-region-1.png)

### Veldconfiguratie

In de veldconfiguratie kunt u de weergavemodus voor het veld instellen:

- Provincie
- Provincie/Stad
- Provincie/Stad/District
- Provincie/Stad/District/Graafschap

![](/images/field-china-region-2.png)
