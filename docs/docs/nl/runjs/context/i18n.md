# ctx.i18n

De i18n-instantie van de huidige context, gebruikt voor het lezen of wijzigen van talen. **Gebruik [ctx.t()](./t.md) voor het vertalen van teksten**; gebruik `ctx.i18n.t` niet.

## Toepassingsscenario's

Alle RunJS-uitvoeringsomgevingen kunnen `ctx.i18n` gebruiken (zoals JSBlock, JSField, JSItem, JSColumn, workflow, koppelingsregels, enz.).

## Type-definitie

```typescript
interface i18n: {
  language: string;
  changeLanguage(lng: string): Promise<any>;
}
```

## Veelvoorkomende eigenschappen

| Eigenschap | Type | Beschrijving |
|------|------|------|
| `language` | `string` | De momenteel actieve taalcode (bijv. `zh-CN`, `en-US`) |

## Veelvoorkomende methoden

### changeLanguage(lng)

Wijzigt de huidige taal.

| Parameter | Type | Beschrijving |
|------|------|------|
| `lng` | `string` | Doeltaalcode (bijv. `'en-US'`, `'zh-CN'`) |

**Retourwaarde**: `Promise<any>`, wordt voltooid (resolve) nadat de taalwissel is afgerond.

## Voorbeelden

### De huidige taal lezen

```ts
const lang = ctx.i18n.language;
// 'zh-CN' | 'en-US' | ...
if (lang.startsWith('zh')) {
  ctx.render(ctx.t('中文界面'));
} else {
  ctx.render(ctx.t('English UI'));
}
```

### Van taal wisselen

```ts
// Overschakelen naar Engels
await ctx.i18n.changeLanguage('en-US');

// Overschakelen naar Chinees
await ctx.i18n.changeLanguage('zh-CN');
```

### Knop voor taalwissel

```ts
const { Button } = ctx.libs.antd;
const isZh = ctx.i18n.language.startsWith('zh');
ctx.render(
  <Button onClick={async () => {
    await ctx.i18n.changeLanguage(isZh ? 'en-US' : 'zh-CN');
  }}>
    {ctx.t(isZh ? 'Switch to English' : '切换到中文')}
  </Button>,
);
```

## Belangrijke opmerkingen

- **Vertaling van teksten**: Gebruik consequent `ctx.t()`, gebruik `ctx.i18n.t` niet.
