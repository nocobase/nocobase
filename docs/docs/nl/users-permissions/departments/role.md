# Afdelingsrolbeheer

Door één of meerdere rollen aan een afdeling te koppelen, kunnen de leden van die afdeling deze rollen krijgen.

Er zijn twee manieren om rollen aan een afdeling te koppelen.

De eerste manier is om een rol te selecteren in het rolveld van het afdeling-bewerkingsformulier.

![](https://static-docs.nocobase.com/70f77bb89aa1fb415c152a51a51cc23b.png)

De tweede manier is om in het rolbeheer afdelingen toe te voegen aan de betreffende rol.

![](https://static-docs.nocobase.com/f2a7bec37cf2f179ce868a92b98416d.png)

Afdelingen die deze rol al hebben, kunnen niet worden geselecteerd.

![](https://static-docs.nocobase.com/be10299893581e1f97a4e01ddd5c7e59.png)
