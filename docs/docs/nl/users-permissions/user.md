# Gebruikers

## Gebruikersbeheer

Deze plugin biedt een gebruikersbeheerinterface in het configuratiecentrum. Na het initialiseren van de applicatie wordt automatisch een onverwijderbaar superbeheerdersaccount aangemaakt. De superbeheerder heeft root-rechten en kan alle resources benaderen.

![](https://static-docs.nocobase.com/44bf40f56b45d4dd96c424fb08082cf6.png)

Daarnaast kunt u op de pagina verschillende blokken van de Gebruikers **collectie** toevoegen om gebruikers te beheren, zoals tabelblokken:

![](https://static-docs.nocobase.com/76c5a4652f869541a9e8f18a4568a7c9.png)
