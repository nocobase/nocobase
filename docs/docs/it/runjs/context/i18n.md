# ctx.i18n

L'istanza i18n del contesto corrente, utilizzata per leggere o cambiare lingua. **Utilizzi [ctx.t()](./t.md) per la traduzione dei testi**; non utilizzi `ctx.i18n.t`.

## Scenari di utilizzo

Tutti gli ambienti di esecuzione RunJS possono utilizzare `ctx.i18n` (ad esempio, JSBlock, JSField, JSItem, JSColumn, flussi di lavoro, regole di collegamento, ecc.).

## Definizione del tipo

```typescript
interface i18n: {
  language: string;
  changeLanguage(lng: string): Promise<any>;
}
```

## Proprietà comuni

| Proprietà | Tipo | Descrizione |
|------|------|------|
| `language` | `string` | Il codice della lingua attualmente attiva (ad es. `en-US`, `zh-CN`) |

## Metodi comuni

### changeLanguage(lng)

Cambia la lingua corrente.

| Parametro | Tipo | Descrizione |
|------|------|------|
| `lng` | `string` | Codice della lingua di destinazione (ad es. `'en-US'`, `'zh-CN'`) |

**Valore di ritorno**: `Promise<any>`, si risolve al completamento del cambio di lingua.

## Esempi

### Lettura della lingua corrente

```ts
const lang = ctx.i18n.language;
// 'en-US' | 'zh-CN' | ...
if (lang.startsWith('zh')) {
  ctx.render(ctx.t('Interfaccia cinese'));
} else {
  ctx.render(ctx.t('Interfaccia inglese'));
}
```

### Cambio della lingua

```ts
// Passa all'inglese
await ctx.i18n.changeLanguage('en-US');

// Passa al cinese
await ctx.i18n.changeLanguage('zh-CN');
```

### Pulsante per il cambio lingua

```ts
const { Button } = ctx.libs.antd;
const isZh = ctx.i18n.language.startsWith('zh');
ctx.render(
  <Button onClick={async () => {
    await ctx.i18n.changeLanguage(isZh ? 'en-US' : 'zh-CN');
  }}>
    {ctx.t(isZh ? 'Passa all\'inglese' : 'Passa al cinese')}
  </Button>,
);
```

## Note

- **Testi tradotti**: Utilizzi coerentemente `ctx.t()`; non utilizzi `ctx.i18n.t`.
