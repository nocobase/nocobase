# Campo Regione Cinese

## Introduzione

Il plugin Campo Regione Cinese (`field-china-region`) aggiunge il tipo di campo "Regione Cinese" a NocoBase.

## Installazione

Nell'interfaccia principale di NocoBase, vada su "Gestore Plugin", trovi il plugin "Regione Cinese" e lo abiliti.

## Utilizzo

Nella configurazione della collezione, aggiunga un nuovo campo e selezioni "Regione Cinese" come tipo di campo.

![](/images/field-china-region-1.png)

### Configurazione del Campo

Nella configurazione del campo, può impostare la modalità di visualizzazione per il campo:

- Provincia
- Provincia/Città
- Provincia/Città/Distretto
- Provincia/Città/Distretto/Contea

![](/images/field-china-region-2.png)
