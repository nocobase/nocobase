# Anteprima file

Nelle interfacce che contengono campi file (inclusi i campi allegato), è possibile visualizzare l'anteprima dei file cliccando sulla miniatura o sull'icona del file. La funzione di anteprima integrata supporta diversi tipi di file, tra cui immagini, PDF e la maggior parte dei formati supportati nativamente dai browser.

![20251129232307](https://static-docs.nocobase.com/20251129232307.png)

Per i tipi di file che non supportano l'anteprima nativa, può abilitare la funzionalità di anteprima installando o estendendo i relativi plugin di anteprima file. Ad esempio, dopo aver installato il plugin Office File Preview, potrà visualizzare l'anteprima di file Word, Excel e PowerPoint.

Attualmente, NocoBase fornisce i seguenti plugin per l'anteprima dei file:

* [Plugin Office File Preview](../file-preview/ms-office.md)
