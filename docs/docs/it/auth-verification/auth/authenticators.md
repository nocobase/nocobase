# Gestione Autenticatori

## Gestione dell'Autenticazione Utente

Quando il plugin di autenticazione utente viene installato, inizializza un metodo di autenticazione basato su `password`, utilizzando il nome utente e l'email dell'utente.

![](https://static-docs.nocobase.com/66eaa9d5421c9cb713b117366bd8a5d5.png)

## Attivare gli Autenticatori

![](https://static-docs.nocobase.com/7f1fb8f8ca5de67ffc68eff0a65848f5.png)

Solo i tipi di autenticazione attivati verranno visualizzati nella pagina di accesso.

![](https://static-docs.nocobase.com/8375a36ef98417af0f0977f1e07345dd.png)

## Tipi di Autenticazione Utente

![](https://static-docs.nocobase.com/da4250c0cea343ebe470cbf7be4b12e4.png)

Aggiungendo diversi tipi di autenticatori, può abilitare i metodi di autenticazione corrispondenti nel sistema.

Oltre ai tipi di autenticazione forniti dai plugin esistenti, gli sviluppatori possono anche estendere i tipi di autenticazione utente. Per maggiori dettagli, consulti la [Guida per gli Sviluppatori](./dev/).
