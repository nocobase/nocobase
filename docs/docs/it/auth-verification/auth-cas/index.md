# Autenticazione: CAS

## Introduzione

Il plugin Autenticazione: CAS segue lo standard del protocollo CAS (Central Authentication Service), consentendo agli utenti di accedere a NocoBase utilizzando gli account forniti da provider di servizi di autenticazione di terze parti (IdP).
