# Motori di calcolo

I motori di calcolo svolgono un ruolo fondamentale nei campi formula, nelle regole di collegamento e nei flussi di lavoro, fornendo strumenti di calcolo universali.

## Formula.js

**Formula.js** è una libreria JavaScript leggera, progettata specificamente per il calcolo di formule. Fornisce un set completo di funzioni matematiche e statistiche, consentendo agli utenti di eseguire calcoli complessi con facilità. Le sue caratteristiche principali includono:

- **Funzionalità ricche**: un ampio set di funzioni che copre varie operazioni matematiche.
- **Facilità d'uso**: offre un'API semplice che permette sia agli sviluppatori che ai non sviluppatori di creare formule senza dover scrivere grandi quantità di codice.
- **Calcolo in tempo reale**: ottimizzato per l'elaborazione dei dati in tempo reale, è adatto per applicazioni che richiedono un feedback di calcolo immediato.

## Math.js

**Math.js** è una libreria matematica ricca di funzionalità per JavaScript e Node.js. Supporta un'ampia gamma di operazioni matematiche, dall'aritmetica di base all'algebra avanzata, alla statistica e all'algebra lineare. Le sue caratteristiche principali includono:

- **Set di funzioni completo**: fornisce una vasta gamma di funzioni per tutti i tipi di operazioni matematiche.
- **Analisi flessibile delle espressioni**: in grado di analizzare e valutare espressioni matematiche in formato stringa.
- **Calcoli avanzati**: supporta operazioni su matrici, calcolo simbolico e analisi statistica.
- **Estensibilità**: può essere esteso con funzioni e costanti personalizzate per soddisfare esigenze specifiche.

## Applicazione in NocoBase

Combinando l'uso di Formula.js e Math.js, NocoBase:

- **Esegue un'ampia gamma di calcoli**: dalle operazioni semplici alle operazioni matematiche complesse.
- **Sviluppa applicazioni robuste ed efficienti**: crea applicazioni che richiedono funzioni di calcolo precise e potenti.
- **Aumenta la produttività**: utilizza funzioni predefinite per evitare di scrivere grandi quantità di codice personalizzato, migliorando così l'efficienza dello sviluppo.

Queste due librerie svolgono un ruolo cruciale in NocoBase, consentendo di creare logiche di calcolo flessibili e personalizzabili senza dover approfondire la programmazione. Ciò non solo semplifica il processo di sviluppo, ma ne migliora anche la funzionalità e la flessibilità.
