# Notifica: WeCom

## Introduzione

Il **plugin** WeCom permette all'applicazione di inviare messaggi di notifica agli utenti WeCom.

## Aggiungere e configurare un autenticatore WeCom

Per prima cosa, è necessario aggiungere e configurare un autenticatore WeCom su NocoBase. Faccia riferimento a [Autenticazione Utente - WeCom](/auth-verification/auth-wecom). Solo gli utenti di sistema che hanno effettuato l'accesso tramite WeCom potranno ricevere le notifiche di sistema tramite WeCom.
