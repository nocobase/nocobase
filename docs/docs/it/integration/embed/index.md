# Incorporare NocoBase

## Introduzione

Questo plugin serve per incorporare le pagine di NocoBase in altri siti web o applicazioni.

## Installazione

Questo plugin è un plugin commerciale e deve essere caricato e attivato tramite il gestore dei plugin.

![20240323162741](https://static-docs.nocobase.com/20240323162741.png)

## Guida all'uso

Dopo aver attivato la configurazione dell'interfaccia, clicchi su "Copia link di incorporamento" nel menu di configurazione della pagina (in alto a destra).

![](https://static-docs.nocobase.com/f11bd6d5e88d38731d7cd3cb149022c8.png)

Dopo aver cliccato, otterrà un link che può essere aperto separatamente, ad esempio, `https://xxx.com/embed/qs087rz4o2b`.

![](https://static-docs.nocobase.com/9d847805a00fd07372a474665ec0e968.png)

Se desidera incorporare le pagine di NocoBase in altri siti web o applicazioni, è necessaria l'autenticazione dell'utente e il `token` deve essere aggiunto al link, ad esempio, `https://xxx.com/embed/qs087rz4o2b?token=xxx`.
