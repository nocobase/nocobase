# Modalità di Visualizzazione

## Introduzione

A differenza dei blocchi, i componenti di campo offrono tre modalità di visualizzazione (disponibili solo per i campi all'interno di un modulo). Ogni modalità di visualizzazione presenta diverse opzioni di configurazione del campo.

- Modificabile;
- Sola lettura (non modificabile);
- Sola lettura (modalità di visualizzazione);

### Modalità di Visualizzazione per i Campi di Relazione

La **modalità di visualizzazione di un campo di relazione** determina come il campo viene visualizzato nell'interfaccia e quali tipi di componenti di campo sono disponibili.

Nello **stato modificabile**, i campi di relazione supportano diversi tipi di componenti, consentendo agli utenti di scegliere diversi componenti di campo di relazione per visualizzare o selezionare i dati associati in base alle esigenze aziendali.

In questo stato, gli utenti possono scegliere in modo flessibile il metodo di visualizzazione appropriato per gestire i dati in modo più efficiente.

#### Componenti dei Campi di Relazione nello Stato di Sola Lettura

Quando si passa allo **stato di sola lettura**, il sistema utilizza automaticamente per impostazione predefinita il **componente campo titolo** per visualizzare i dati associati. Questo è adatto per scenari in cui è necessario solo visualizzare i dati senza apportare modifiche.
