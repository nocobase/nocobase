# Campo Titolo

## Introduzione

Il **Campo Titolo** viene solitamente utilizzato nei componenti dei campi di relazione come identificatore per i dati associati visualizzati nell'interfaccia. Determina come vengono presentate le informazioni principali dei dati, aiutando gli utenti a identificarli e comprenderli rapidamente.

Diversi tipi di campi titolo corrispondono a diversi componenti di campo e supportano le relative opzioni di configurazione. Ad esempio:

- Quando il campo titolo è un **Campo Data**, il componente di campo utilizza un **componente Data** per visualizzare i dati.
- Quando il campo titolo è un **Campo Testo**, il componente di campo utilizza un **componente Testo** per visualizzare i dati.
- Quando il campo titolo è un **Campo Opzione**, il componente di campo utilizza un **componente Opzione** per visualizzare i dati.
