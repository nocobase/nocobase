# Mostra Etichetta

## Introduzione

Per impostazione predefinita, i campi all'interno di un blocco visualizzano le etichette. Queste etichette servono a identificare il significato di ciascun campo, facilitando la lettura e la comprensione dei dati da parte dell'utente.

In alcuni scenari (ad esempio, su pagine di sola visualizzazione o quando il significato dei campi è già molto chiaro), può scegliere di nascondere le etichette per ottenere un'interfaccia più pulita. Una volta nascoste le etichette, verranno visualizzati solo i valori dei campi e i nomi dei campi non verranno più mostrati.
