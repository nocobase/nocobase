# Componente Campo

## Introduzione

I componenti campo sono elementi utilizzati per visualizzare il contenuto di un campo.
Alcuni campi includono diversi componenti di visualizzazione integrati. Gli utenti possono scegliere il componente più adatto alle proprie esigenze per presentare il valore del campo in modi diversi.

Esempio: In modalità di visualizzazione, un campo URL può essere commutato in un componente Anteprima o in un componente URL (predefinito).
