# Campo

Nell'interfaccia utente (UI), un campo è un componente campo che funge da contenitore per i dati unitari. Dati di tipi diversi vengono presentati tramite componenti campo differenti. I campi possono essere allegati solo ai blocchi e non possono essere utilizzati in modo indipendente.

## Campi all'interno di un Blocco

I campi, di norma, non vengono utilizzati da soli, ma esistono come elementi figli dei blocchi di tipo dati. I blocchi di tipo dati solitamente includono un'opzione "Configura campi", e l'elenco dei campi è fornito dalla collezione attuale.

![20251029204240](https://static-docs.nocobase.com/20251029204240.png)

## Designer del Campo (Barra degli Strumenti)

Come per i blocchi, anche il componente campo presenta tre icone nell'angolo superiore destro, che sono:

- Trascina per il layout
- [Flusso di eventi](/interface-builder/event-flow)
- Configurazione

![20251025165432](https://static-docs.nocobase.com/20251025165432.png)

## Layout del Campo

È possibile personalizzare il layout dei campi all'interno di un blocco trascinandoli.
