# Campi Tabella

## Introduzione

Oltre a consentire la regolazione della larghezza delle colonne, del titolo del campo e dell'ordinamento, i campi tabella offrono anche configurazioni di visualizzazione più personalizzate per tipi di campo specifici (come campi data, campi relazione e campi numerici).

![20251024174558](https://static-docs.nocobase.com/20251024174558.png)

## Opzioni di Configurazione dei Campi

### Formattazione dei Campi Data

![20251024175303](https://static-docs.nocobase.com/20251024175303.png)

Per maggiori dettagli, consulti [Formattazione della Data](/interface-builder/fields/specific/date-picker)

### Formattazione dei Campi Numerici

Supporta la conversione semplice delle unità, i separatori delle migliaia, prefissi e suffissi, la precisione e la notazione scientifica.

![20251024175445](https://static-docs.nocobase.com/20251024175445.png)

Per maggiori dettagli, consulti [Formattazione Numerica](/interface-builder/fields/field-settings/number-format)

### Abilitare la Modifica Rapida

Quando la modifica rapida è abilitata, un pulsante di modifica apparirà al passaggio del mouse sulla colonna. Cliccandolo, potrà modificare e salvare rapidamente i dati in una finestra pop-up.

![20251025171158](https://static-docs.nocobase.com/20251025171158.gif)

### Abilitare il Clic per Aprire

Oltre ai campi relazione che supportano l'apertura di un pop-up tramite clic, anche i campi normali possono essere configurati con l'opzione "Clic per Aprire" per fungere da punto di accesso per l'apertura di un pop-up.

![20251025172308](https://static-docs.nocobase.com/20251025172308.gif)

### Modalità di Visualizzazione per Contenuto in Eccesso

Quando il contenuto di una colonna supera la larghezza della colonna della tabella, è possibile impostare la modalità di gestione dell'eccesso:

-   Punti di sospensione (predefinito)
-   Andare a capo

### Componente del Campo

Alcuni campi supportano diversi formati di visualizzazione. Gli utenti possono cambiare i componenti del campo per ottenere effetti di visualizzazione diversi e soddisfare le esigenze di vari scenari. Ad esempio, un campo di tipo **URL** può essere commutato in un componente **Preview** per visualizzare meglio il contenuto del link o le anteprime delle immagini.

![20251025171658](https://static-docs.nocobase.com/20251025171658.png)

### Ordinabile

Attualmente, la maggior parte dei tipi di campo supporta l'ordinamento lato server. Quando l'ordinamento è abilitato, i dati possono essere ordinati in ordine crescente o decrescente in base al campo di destinazione.
