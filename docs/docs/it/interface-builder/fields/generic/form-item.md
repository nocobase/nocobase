# Campi del Modulo

## Introduzione

I diversi tipi di campi in un modulo presentano diverse opzioni di configurazione. Questo permette agli utenti di estendere tali opzioni tramite i **plugin** per soddisfare esigenze più personalizzate.

![20251025174555](https://static-docs.nocobase.com/20251025174555.png)

## Opzioni di Configurazione del Campo

È possibile configurare costanti o variabili come valori predefiniti per i campi (questa funzionalità è supportata solo per i nuovi moduli).
