# Selettore Dati

## Introduzione

Il Selettore Dati, presentato come un selettore a tabella a comparsa (popup), le permette di selezionare dati esistenti dalla collezione di destinazione per associarli, oppure di aggiungere nuovi dati alla collezione di destinazione e poi associarli.

![20251027203343](https://static-docs.nocobase.com/20251027203343.png)

I dati già selezionati non verranno visualizzati nel selettore a tabella.

![20251027203717](https://static-docs.nocobase.com/20251027203717.gif)

All'interno del selettore a tabella, può anche gestire ulteriormente la collezione associata di destinazione (aggiungere, eliminare, importare/esportare, ecc.).

![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
