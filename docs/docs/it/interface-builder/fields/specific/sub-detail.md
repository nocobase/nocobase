# Sottodettagli

## Introduzione

I Sottodettagli sono il componente corrispondente al Sotto-modulo in modalità di sola lettura. Rispetto ai componenti Etichetta e Titolo, i Sottodettagli non solo possono visualizzare più dati dalla collezione corrente, ma possono anche essere configurati per mostrare informazioni da collezioni correlate. I dati relazionali a più livelli vengono visualizzati chiaramente sotto forma di dettagli annidati.
