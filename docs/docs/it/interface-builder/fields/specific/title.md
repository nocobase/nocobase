# Titolo

## Introduzione

In modalità di lettura, i campi di associazione vengono visualizzati per impostazione predefinita utilizzando il **Componente Campo Titolo**, che identifica il record attualmente associato mostrando il valore del campo titolo.
Il sistema abbina e renderizza automaticamente il componente campo titolo corrispondente in base al tipo di campo titolo.
