# Gestore File

## Introduzione

Il Gestore File è un componente di campo di associazione utilizzato per gestire le relazioni in cui la collezione di destinazione è la collezione dei file.

![20251027194124](https://static-docs.nocobase.com/20251027194124.png)

## Configurazione del Campo

- [Componente di Campo](/interface-builder/fields/association-field): Permette di passare ad altri componenti di campo di associazione, come il menu a discesa, il selettore di dati, ecc.
- Consenti la selezione di file esistenti: Abilitato per impostazione predefinita, permette di selezionare e associare file già caricati.
