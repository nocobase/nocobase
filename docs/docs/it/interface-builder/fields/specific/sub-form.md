# Sottomodulo

## Introduzione

I sottomoduli sono ideali per scenari in cui si creano nuovi dati correlati prima di collegarli. I dati correlati a più livelli vengono visualizzati chiaramente come un modulo nidificato. Rispetto ai componenti Selettore dati e Dropdown, un sottomodulo Le consente di gestire direttamente i campi della collezione associata all'interno del blocco pagina corrente, e i dati associati vengono inviati insieme al modulo principale.
