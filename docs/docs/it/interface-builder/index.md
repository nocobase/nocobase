# Costruzione Interfaccia

## WYSIWYG

NocoBase adotta un approccio WYSIWYG (What You See Is What You Get) per la costruzione dell'interfaccia. Cliccando sul pulsante "Costruzione Interfaccia", può passare dalla modalità di configurazione a quella di utilizzo.

## Modello di Layout

NocoBase include un modello di layout predefinito: l'area di navigazione si trova in alto e a sinistra, mentre l'area dei contenuti è a destra.

![未命名.002](https://static-docs.nocobase.com/未命名.002.jpeg)

## Opzioni di Configurazione

Una volta entrato nella modalità di configurazione dell'interfaccia, vedrà apparire degli indicatori arancioni che segnalano gli elementi configurabili. Solitamente, il punto di accesso alle opzioni di configurazione di ciascun elemento si trova nell'angolo in alto a destra dell'elemento stesso.
