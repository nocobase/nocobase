# Blocco: Pannello Operativo

## Introduzione

Utilizzato per posizionare varie azioni rapide. Attualmente supporta:

- [Link](/interface-builder/actions/types/link)
- [Scansione Codice QR]
- [Azione Popup](/interface-builder/actions/types/pop-up)

## Tipi di Azione

- **Link**: Consente di navigare a una pagina specificata o a un URL esterno.

![20251023230643](https://static-docs.nocobase.com/20251023230643.png)

- **Scansione Codice QR**: Attiva un'azione corrispondente tramite la scansione di un codice QR.
