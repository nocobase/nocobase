# Layout

## Introduzione

I blocchi di tipo "Form", come i blocchi "Form", "Dettagli" e "Lista", supportano le impostazioni di layout. Le opzioni di layout si dividono in layout verticale (etichette e campi di input sono disposti riga per riga) e layout orizzontale (etichette e campi di input sono posizionati affiancati). Gli utenti possono regolare il layout dei campi del blocco in base alle esigenze specifiche, garantendo che l'inserimento o la visualizzazione delle informazioni siano chiari ed efficienti.

### Verticale (Predefinito)

Il layout predefinito è verticale.

![20251024120143](https://static-docs.nocobase.com/20251024120143.png)

### Orizzontale

Il layout orizzontale visualizza etichette e campi di input affiancati, ottimizzando lo spazio. Per i campi con titoli lunghi, può scegliere di impostare l'omissione o l'andata a capo, per garantire un'interfaccia pulita ed esteticamente gradevole.

![20251024120237](https://static-docs.nocobase.com/20251024120237.png)

#### Allineamento del titolo del campo

- **A sinistra** (predefinito)
- **A destra**

#### Larghezza del titolo del campo

- La larghezza predefinita è di **120px**, con una larghezza minima di **50px**.

#### Gestione dei titoli di campo che superano la larghezza

- **Andata a capo**
- **Omissione**

#### Mostra due punti

Può scegliere se visualizzare i due punti dopo il titolo del campo.
