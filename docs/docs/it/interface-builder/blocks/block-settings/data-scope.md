# Impostare l'ambito dei dati

## Introduzione

Impostare l'ambito dei dati significa definire delle condizioni di filtro predefinite per un blocco di dati. Gli utenti possono adattare l'ambito dei dati in modo flessibile in base alle proprie esigenze aziendali. Tuttavia, indipendentemente dalle operazioni di filtro eseguite, il sistema applicherà automaticamente questa condizione di filtro predefinita, garantendo che i dati rimangano sempre entro i limiti dell'ambito specificato.

### Operatori

Campi di tipo diverso supportano operatori differenti. Ad esempio, i campi di testo supportano operatori come "uguale a", "diverso da" e "contiene"; i campi numerici supportano operatori come "maggiore di" e "minore di"; mentre i campi data supportano operatori come "è compreso tra" e "è precedente a una data specifica".
