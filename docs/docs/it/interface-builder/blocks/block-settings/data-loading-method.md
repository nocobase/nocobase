# Modalità di caricamento dei dati

## Introduzione

La modalità predefinita di caricamento dei dati è "Carica tutti i dati quando le condizioni di filtro sono vuote". In questo contesto, per "condizioni di filtro" si intendono i valori presenti nel "Blocco di filtraggio". Un'altra opzione disponibile è "Non caricare i dati quando le condizioni di filtro sono vuote".
