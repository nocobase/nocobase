# Blocco Lista

## Introduzione

Il blocco Lista visualizza i dati in formato elenco, ed è adatto per scenari di visualizzazione di dati come liste di attività, notizie e informazioni sui prodotti.

## Configurazione del Blocco

![20251023202835](https://static-docs.nocobase.com/20251023202835.png)

### Impostare l'Ambito dei Dati

Come mostrato: filtrare gli ordini con stato "Annullato".

![20251023202927](https://static-docs.nocobase.com/20251023202927.png)

Per maggiori dettagli, consulti [Impostare l'Ambito dei Dati](/interface-builder/blocks/block-settings/data-scope)

### Impostare le Regole di Ordinamento

Come mostrato: ordinare per importo dell'ordine in ordine decrescente.

![20251023203022](https://static-docs.nocobase.com/20251023203022.png)

Per maggiori dettagli, consulti [Impostare le Regole di Ordinamento](/interface-builder/blocks/block-settings/sorting-rule)

## Configurare i Campi

### Campi di questa Collezione

> **Nota**: i campi delle collezioni ereditate (ossia, i campi delle collezioni genitore) vengono automaticamente uniti e visualizzati nell'elenco dei campi attuale.

![20251023203103](https://static-docs.nocobase.com/20251023203103.png)

### Campi delle Collezioni Associate

> **Nota**: è possibile visualizzare i campi delle collezioni associate (attualmente sono supportate solo le relazioni uno-a-uno).

![20251023203611](https://static-docs.nocobase.com/20251023203611.png)

Per la configurazione dei campi della lista, consulti [Campo Dettagli](/interface-builder/fields/generic/detail-form-item)
