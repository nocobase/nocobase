# Grafico

## Introduzione

Il plugin di visualizzazione dati di NocoBase offre la possibilità di interrogare i dati in modo visuale e una ricca serie di componenti grafici. Gli utenti possono creare rapidamente dashboard di visualizzazione tramite una configurazione semplice, per mostrare insight sui dati e supportare l'analisi e la presentazione multidimensionale dei dati.
