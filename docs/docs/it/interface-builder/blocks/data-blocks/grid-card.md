# Griglia di Schede

## Introduzione

Il blocco Griglia di Schede visualizza le informazioni di riepilogo dei record di dati in formato scheda. Supporta la configurazione del numero di colonne in base alle diverse dimensioni dello schermo, per garantire una visualizzazione ottimale su vari dispositivi.
