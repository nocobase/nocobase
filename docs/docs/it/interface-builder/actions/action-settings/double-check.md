# Conferma

## Introduzione

La configurazione della conferma serve a verificare l'esecuzione di un'operazione. Quando si fa clic su un'azione, appare una finestra di dialogo di conferma. Per esempio, il pulsante di eliminazione ha la conferma abilitata di default per prevenire la perdita di dati o risultati irreversibili dovuti a operazioni accidentali.

![20251029125021](https://static-docs.nocobase.com/20251029125021.png)

È possibile personalizzare il contenuto e il titolo della conferma, oltre ad accedere alle variabili.
