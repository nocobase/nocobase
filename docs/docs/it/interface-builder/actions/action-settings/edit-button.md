# Modifica pulsante

## Introduzione

Il sistema Le permette di personalizzare il nome, l'icona, il colore di sfondo e il tipo di un pulsante. Questo Le offre opzioni flessibili per la personalizzazione dell'interfaccia utente, aiutandoLa ad adattare lo stile di visualizzazione e le funzionalità del pulsante in base alle Sue esigenze specifiche.

-   **Nome del pulsante**: Può personalizzare il testo del pulsante in base alle Sue esigenze aziendali, assicurando che l'etichetta sia chiara e intuitiva.
-   **Icona del pulsante**: È possibile impostare un'icona per il pulsante per migliorare il riconoscimento visivo, permettendo agli utenti di identificare rapidamente la funzione del pulsante.
-   **Azione pericolosa (Danger Action)**: Quando un pulsante rappresenta un'**azione pericolosa** (come eliminare, reimpostare, ecc.), può essere impostato come tipo **Danger Action**.
-   **Tipo di pulsante (Button Type)**: I pulsanti supportano diverse impostazioni di tipo. I tipi di pulsante più comuni includono:
    -   **Predefinito (Default)**: Un pulsante standard, adatto per operazioni normali.
    -   **Primario (Primary)**: Utilizzato per azioni importanti o principali, solitamente con un effetto visivo evidente.
    -   **Tratteggiato (Dashed)**: Un pulsante con un bordo tratteggiato, adatto per azioni secondarie o pulsanti di suggerimento.
    -   **Testo (Text)**: Un pulsante con solo testo e senza colore di sfondo, tipicamente usato per operazioni leggere o collegamenti.
