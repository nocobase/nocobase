# Assegnazione Campi

## Introduzione

L'assegnazione campi consiste nel preimpostare il valore di un campo prima di eseguire un'azione. Quando aggiorna o crea un record, questi dati vengono compilati automaticamente, senza bisogno di inserirli manualmente. Questo è utile in scenari come l'aggiornamento massivo dello stato degli ordini o l'impostazione dell'ora di consegna all'ora corrente quando un ordine viene consegnato.
