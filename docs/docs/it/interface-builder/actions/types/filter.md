# Filtro

## Introduzione

L'azione di filtro appare solitamente nei blocchi di dati per filtrare i dati configurando diverse condizioni. È importante notare che, se per il blocco è già stato configurato un ambito dati, l'azione di filtro verrà combinata con le condizioni dell'ambito dati come condizione di filtro finale.

## Impostazioni dell'azione

![20251029173708](https://static-docs.nocobase.com/20251029173708.png)

### Campi filtrabili

Per impostazione predefinita, tutti i campi di questa collezione sono abilitati. Solo i campi abilitati possono essere selezionati nell'elenco dei campi condizione.

Anche i campi delle collezioni associate possono essere utilizzati come campi di filtro.
