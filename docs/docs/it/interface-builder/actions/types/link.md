# Link

## Introduzione

L'azione "Link" consente di navigare tra le pagine, supporta il passaggio di variabili URL e permette alla pagina di destinazione di adattare dinamicamente il suo contenuto in base ai dati trasmessi. È possibile configurarla all'interno dei blocchi di dati.
