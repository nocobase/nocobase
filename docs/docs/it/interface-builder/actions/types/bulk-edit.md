# Modifica in blocco

## Introduzione

La modifica in blocco è adatta a scenari che richiedono l'aggiornamento flessibile di dati in batch. Dopo aver fatto clic sul pulsante di modifica in blocco, potrà configurare il modulo di modifica in blocco in una finestra pop-up e impostare diverse strategie di aggiornamento per ogni campo.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM.png)


## Configurazione dell'azione

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM%20(1).png)


## Guida all'uso

### Configurazione del modulo di modifica in blocco

1. Aggiunga un pulsante di modifica in blocco.

2. Imposti l'ambito della modifica in blocco: Selezionati / Tutti, l'impostazione predefinita è Selezionati.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_14_AM.png)

3. Aggiunga un modulo di modifica in blocco.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_14_AM.png)

4. Configuri i campi da modificare e aggiunga un pulsante di invio.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM%20(1).png)

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM.png)

### Invio del modulo

1. Selezioni le righe di dati da modificare.

2. Selezioni la modalità di modifica per i campi e inserisca i valori da inviare.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_10_33_AM.png)

:::info{title=Modalità di modifica disponibili}
* **Non aggiornare**: il campo rimane invariato.
* **Modifica in**: aggiorna il campo con il valore inviato.
* **Svuota**: cancella i dati nel campo.

:::

3. Invii il modulo.
