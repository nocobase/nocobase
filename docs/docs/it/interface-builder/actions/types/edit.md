# Modifica

## Introduzione

L'azione di Modifica è un pulsante di riga che ha come contesto il record corrente. Si presenta come una finestra pop-up dove gli utenti possono personalizzare e configurare i blocchi.

![20251029165442](https://static-docs.nocobase.com/20251029165442.png)

Configuri i blocchi secondo necessità.
