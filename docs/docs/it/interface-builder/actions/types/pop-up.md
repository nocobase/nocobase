# Apri Modale

## Introduzione

L'azione **Apri Modale** è simile alle azioni Visualizza e Modifica. Gli utenti possono configurare un pulsante **Apri Modale** nelle righe o nell'intestazione di una tabella. Facendo clic sul pulsante, si aprirà una modale corrispondente, e i blocchi all'interno di questa modale visualizzeranno i dati in base al record del contesto attuale.

![20251029182351](https://static-docs.nocobase.com/20251029182351.png)

Configuri i blocchi nella modale secondo necessità.

## Impostazioni dell'azione

![20251029190352](https://static-docs.nocobase.com/20251029190352.png)
