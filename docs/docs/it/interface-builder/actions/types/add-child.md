# Aggiungere un sotto-record

## Introduzione

Questa operazione è progettata specificamente per i **blocchi tabella ad albero** per aggiungere rapidamente sotto-record. Quando si aggiunge un sotto-record, la riga selezionata viene trattata come il nodo padre, consentendo agli utenti di creare facilmente sotto-record sotto qualsiasi genitore. Questa funzionalità semplifica notevolmente la gestione dei dati gerarchici, rendendo l'espansione e l'organizzazione delle strutture dati più efficiente e intuitiva.
