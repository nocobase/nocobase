# Bozze dei moduli

Quando le bozze dei moduli sono abilitate, il contenuto in fase di compilazione viene salvato automaticamente in locale. Anche se il modulo viene chiuso accidentalmente o la pagina viene aggiornata, i dati inseriti non andranno persi.

![20251213093717](https://static-docs.nocobase.com/20251213093717.png)

Quando esiste una bozza, verrà visualizzata una notifica nella parte superiore del modulo. La bozza verrà eliminata automaticamente dopo l'invio con successo; se non è più necessaria, può essere eliminata manualmente in qualsiasi momento.

![20251213092734](https://static-docs.nocobase.com/20251213092734.png)

::: tip Suggerimento

Attualmente, le bozze dei moduli supportano solo la cache locale. Le versioni future supporteranno la cache lato server per consentire la sincronizzazione tra più dispositivi.

:::
