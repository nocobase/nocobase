# Guida Rapida

## Introduzione
Dopo aver completato la configurazione del servizio LLM, possiamo iniziare a creare i dipendenti AI.

## Nuovo Dipendente AI

### Configurazione delle Informazioni di Base

Acceda alla pagina di configurazione del plugin dei dipendenti AI, quindi clicchi sulla scheda `AI employees` per accedere alla pagina di gestione dei dipendenti AI.

![20251021225643](https://static-docs.nocobase.com/20251021225643.png)

Clicchi sul pulsante `New AI employee` per accedere alla pagina di creazione del dipendente AI.

Innanzitutto, nella scheda `Profile`, può configurare le informazioni di base del dipendente AI:

- Nel campo `Username`, inserisca l'identificativo unico per il dipendente AI;
- Nel campo `Nickname`, inserisca il soprannome del dipendente AI;
- Nel campo `Position`, inserisca la posizione del dipendente AI, utile per descrivere brevemente le sue responsabilità;
- In `Avatar`, selezioni un avatar per il dipendente AI.

![20251021232246](https://static-docs.nocobase.com/20251021232246.png)

- Nel campo `Bio`, inserisca il testo introduttivo per il dipendente AI, descrivendone le competenze e le funzioni. Questo testo verrà visualizzato nel profilo del dipendente AI;
- Nel campo `About me`, inserisca una descrizione dettagliata delle funzioni del dipendente AI. Questo testo verrà utilizzato come prompt di sistema durante la conversazione con il dipendente AI.
- Nel campo `Greeting message`, inserisca il messaggio di benvenuto del dipendente AI. Ogni volta che un utente avvia una nuova conversazione con il dipendente AI, il sistema invierà automaticamente questo messaggio all'utente.

![20251021232712](https://static-docs.nocobase.com/20251021232712.png)

### Configurazione del Modello Linguistico

Nella scheda `Model settings`, può configurare le impostazioni del modello per il dipendente AI:

- Selezioni il servizio LLM che ha creato nella gestione dei servizi LLM;
- Inserisca il nome del modello linguistico che desidera utilizzare.

![20251021231727](https://static-docs.nocobase.com/20251021231727.png)

### Completare la Creazione

Dopo aver inserito tutte le informazioni necessarie per creare il dipendente AI, clicchi sul pulsante `Submit` per completare la creazione.

Potrà quindi visualizzare il dipendente AI appena creato nell'elenco dei dipendenti AI. Inoltre, il dipendente AI appena creato sarà visibile nel pulsante di avvio rapido dei dipendenti AI, in basso a destra nella pagina.
