# Gestione File

## Introduzione

Consente di configurare la modalità di archiviazione per i file caricati durante le conversazioni con l'AI dipendente.

## Impostazioni

Acceda alla pagina di configurazione del plugin AI dipendente, quindi clicchi sulla scheda `Settings` per accedere alla pagina delle impostazioni della modalità di archiviazione.

![20251022002648](https://static-docs.nocobase.com/20251022002648.png)

Le opzioni di archiviazione disponibili per `Files storage` vengono configurate nel plugin Gestione File.
