# Agente AI · Dex: Esperto di Organizzazione Dati

> Estrae e compila rapidamente i moduli, trasformando le informazioni disordinate in dati strutturati.

`Dex` è un esperto di organizzazione dati. Estrae le informazioni necessarie da dati o file non strutturati, le organizza in informazioni strutturate e può utilizzare strumenti per compilare i moduli.

![20251022155210](https://static-docs.nocobase.com/20251022155210.png)

## Passaggi

Nella pagina del modulo, richiami `Dex` per aprire la finestra di dialogo.

![20251022155316](https://static-docs.nocobase.com/20251022155316.png)

Clicchi su `Add work context` nel campo di input e selezioni `Pick block`. La pagina entrerà in modalità di selezione blocco.

![20251022155414](https://static-docs.nocobase.com/20251022155414.png)

Selezioni il blocco del modulo sulla pagina.

![20251022155523](https://static-docs.nocobase.com/20251022155523.png)

Nel campo di dialogo, inserisca i dati che desidera che `Dex` compili.

![20251022155555](https://static-docs.nocobase.com/20251022155555.png)

Dopo l'invio, `Dex` strutturerà i dati e utilizzerà le sue capacità per aggiornare i dati nel modulo selezionato.

![20251022155733](https://static-docs.nocobase.com/20251022155733.png)
