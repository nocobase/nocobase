# Abilitare i dipendenti AI

NocoBase offre diversi dipendenti AI predefiniti per vari scenari.

## Come abilitare

Dopo aver effettuato l'accesso alla pagina di gestione `AI employees`, è possibile attivare direttamente l'interruttore `Enabled` nell'elenco per abilitare il dipendente.

![ai-employee-list-enable-switch.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employee-list-enable-switch.png)

## Utilizzo dopo l'abilitazione

Una volta abilitato, il dipendente apparirà nei punti di accesso disponibili (come la scorciatoia in basso a destra o gli ingressi specifici per determinati scenari).
