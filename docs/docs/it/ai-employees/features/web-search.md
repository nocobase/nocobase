# Ricerca Web

La ricerca web viene utilizzata per integrare le informazioni più recenti al di fuori dei dati di addestramento del modello.

## Come funziona

La disponibilità della ricerca web dipende dal fatto che il servizio del modello selezionato per la sessione corrente supporti o meno la funzione Web Search.

- **Supportato**: viene visualizzato l'interruttore della ricerca web, che può essere attivato o disattivato secondo necessità.
- **Non supportato**: l'interruttore non viene visualizzato e lo stato di ricerca viene disattivato automaticamente.

## Utilizzo nella sessione

Utilizzi l'interruttore della ricerca web nell'area di inserimento del messaggio:

- **Attivato**: l'IA può effettuare una ricerca prima di rispondere.

![web-search-switch-visible-on-supported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-visible-on-supported-model.png)

- **Disattivato**: l'IA risponde basandosi esclusivamente sul contesto esistente.

![web-search-switch-hidden-on-unsupported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-hidden-on-unsupported-model.png)

## Differenze tra le piattaforme

La capacità di supporto per la ricerca web varia a seconda del modello; La invitiamo a utilizzarla in base alla situazione specifica del servizio e del modello selezionati.
