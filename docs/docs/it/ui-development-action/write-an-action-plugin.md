Le azioni per i blocchi forniscono tre tipi di operazioni: "Creazione in blocco", "Aggiornamento in blocco" ed "Eliminazione in blocco", che permettono di agire su più record di dati contemporaneamente.

## Creazione in blocco

Nel `Popup`, aggiungere un blocco `Form` e attivare l'opzione `Consenti creazione in blocco`.


![](https://oss-cn-beijing.aliyuncs.com/nocobase/1.0/bulk-create-2.png)


Successivamente, aggiungere un pulsante di azione `Aggiungi record` nel blocco `Tabella`.


![](https://oss-cn-beijing.aliyuncs.com/nocobase/1.0/bulk-create-1.png)


Cliccando sul pulsante `Aggiungi record`, sarà possibile creare dati in blocco nella finestra di popup.


![](https://oss-cn-beijing.aliyuncs.com/nocobase/1.0/bulk-create-3.png)


## Eliminazione in blocco

Nel blocco `Tabella`, attivare l'opzione `Consenti eliminazione in blocco`.


![](https://oss-cn-beijing.aliyuncs.com/nocobase/1.0/bulk-delete-1.png)


Successivamente, aggiungere un pulsante di azione `Eliminazione in blocco` nel blocco `Tabella`.


![](https://oss-cn-beijing.aliyuncs.com/nocobase/1.0/bulk-delete-2.png)


Selezionare i dati da eliminare, cliccare sul pulsante `Eliminazione in blocco` per procedere con l'eliminazione in blocco dei dati.


![](https://oss-cn-beijing.aliyuncs.com/nocobase/1.0/bulk-delete-3.png)


## Aggiornamento in blocco

Nel blocco `Tabella`, attivare l'opzione `Consenti aggiornamento in blocco`.


![](https://oss-cn-beijing.aliyuncs.com/nocobase/1.0/bulk-update-1.png)


Successivamente, aggiungere un pulsante di azione `Aggiornamento in blocco` nel blocco `Tabella`.


![](https://oss-cn-beijing.aliyuncs.com/nocobase/1.0/bulk-update-2.png)


Selezionare i dati da aggiornare, cliccare sul pulsante `Aggiornamento in blocco` per aggiornare i dati in blocco nella finestra di popup.


![](https://oss-cn-beijing.aliyuncs.com/nocobase/1.0/bulk-update-3.png)
