# Utenti

## Gestione Utenti

Questo plugin offre un'interfaccia di gestione utenti all'interno del centro di configurazione. Dopo l'inizializzazione dell'applicazione, viene creato automaticamente un account super amministratore non eliminabile. L'amministratore possiede i permessi di Root e può accedere a tutte le risorse.

![](https://static-docs.nocobase.com/44bf40f56b45d4dd96c424fb08082cf6.png)

Inoltre, può anche aggiungere diversi blocchi dalla collezione Utenti nelle pagine per gestire gli utenti, come ad esempio i blocchi tabella:

![](https://static-docs.nocobase.com/76b5a65d4659e8f18a4568a7c9.png)
