# Gestione dei Ruoli per Dipartimento

Associando uno o più ruoli a un dipartimento, i membri del dipartimento potranno avere questi ruoli.

Esistono due modi per associare ruoli a un dipartimento.

Il primo consiste nel selezionare un ruolo nel campo 'Ruolo' del modulo di modifica del dipartimento.

![](https://static-docs.nocobase.com/70f77bb89aa1fb415c152a51a51cc23b.png)

Il secondo consiste nell'aggiungere i dipartimenti al ruolo corrispondente all'interno della gestione dei ruoli.

![](https://static-docs.nocobase.com/f2a7bec937cf2f179ce868a92b98416d.png)

I dipartimenti che possiedono già questo ruolo non sono selezionabili.

![](https://static-docs.nocobase.com/be10299893581e1f97a4e01ddd5c7e59.png)
