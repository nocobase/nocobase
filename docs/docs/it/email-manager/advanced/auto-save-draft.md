# Salvataggio automatico bozza

Nel menu del pulsante **Salva bozza**, può abilitare la funzione **Salvataggio automatico bozza**.
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_29_PM.png)

Durante la modifica dell'email, il sistema salverà la bozza in tempo reale, assicurando che nessun contenuto vada perso.
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_31_PM.png)

Quando aprirà nuovamente l'editor email (per lo stesso mittente e destinatario), il sistema Le chiederà se desidera utilizzare la bozza salvata.
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_08_35_PM.png)
