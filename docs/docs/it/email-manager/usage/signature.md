# Firma Email

## Configurazione della Firma

Vada al Centro di Gestione Email e clicchi sul pulsante delle impostazioni:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_05_34_PM.png)

Passi al pannello della firma:
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_05_51_PM.png)

Qui può gestire le firme. Selezioni l'opzione prima di una firma per impostarla come predefinita:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_05_37_PM.png)

## Utilizzo delle Firme

Nel corpo dell'email, clicchi sull'icona della firma per selezionare la firma desiderata:
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_05_40_PM.png)
