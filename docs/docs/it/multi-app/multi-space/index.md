# Multi-space

<PluginInfo name="multi-space" licenseBundled="professional"></PluginInfo>

## Introduzione

Il plugin **Multi-space** consente di isolare logicamente piÃ¹ spazi dati nella stessa istanza.

## Installazione

Abilita **Multi-Space** dal plugin manager.

![](https://static-docs.nocobase.com/multi-space/Plugin-manager-NocoBase-10-15-2025_09_57_AM.png)

## Guida

- Gestione spazi in **Users & Permissions** > **Spaces**
- Creazione spazio con **Add space**
- Assegnazione utenti per spazio
- Switch spazio in alto a destra
- Campo **Space** necessario nelle collection

Per dati legacy:

1. Aggiungi campo Space
2. Includi utenti in **Unassigned Space**
3. Mostra tutti gli spazi
4. Configura pagina di assegnazione
5. Assegna manualmente (o batch)
