# Selettore di collezione

Il campo selettore di collezione è un tipo di campo speciale che consente agli utenti di selezionare una o più collezioni da un elenco predefinito. Questo tipo di campo è molto utile in scenari in cui è necessario fare riferimento dinamicamente a diverse collezioni, come la creazione di report configurabili, dashboard o applicazioni complesse che devono modificare le fonti dati in base alla selezione dell'utente.

## Casi d'uso

### Generazione dinamica di report

In un modulo di generazione report, gli utenti potrebbero aver bisogno di selezionare una collezione tra più collezioni come fonte dati per il report. Utilizzando il selettore di collezione, gli utenti possono facilmente selezionare la collezione desiderata e il sistema genererà dinamicamente il report in base alla selezione dell'utente.

### Dashboard configurabili

In un'applicazione dashboard, gli utenti possono personalizzare i dati visualizzati sulla dashboard. Il selettore di collezione consente agli utenti di scegliere le collezioni a cui sono interessati, visualizzando così grafici e dati statistici da diverse fonti dati sulla dashboard.

### Strumenti flessibili di importazione/esportazione dati

Quando si sviluppa uno strumento generico di importazione o esportazione dati, è possibile utilizzare il selettore di collezione per consentire agli utenti di specificare la collezione su cui operare. In questo modo, lo stesso strumento può essere utilizzato per elaborare dati da qualsiasi collezione nel sistema, migliorando notevolmente la riusabilità dello strumento.
