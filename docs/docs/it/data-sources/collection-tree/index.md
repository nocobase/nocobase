# Collezione ad albero

## Introduzione

Una collezione a struttura ad albero è un modello di progettazione per collezioni di dati, utilizzato per organizzare informazioni con relazioni gerarchiche. Questa struttura rispecchia quella di un albero, dove ogni elemento può avere uno o più sotto-elementi, i quali a loro volta possono avere i propri discendenti.
