# Aliyun OSS

Motore di archiviazione basato su Aliyun OSS. Prima di utilizzarlo, è necessario preparare gli account e le autorizzazioni pertinenti.

## Configurazione

![Esempio di configurazione del motore di archiviazione Aliyun OSS](https://static-docs.nocobase.com/20240712220011.png)

:::info{title=Nota}
Qui vengono presentati solo i parametri specifici per il motore di archiviazione Aliyun OSS. Per i parametri comuni, La preghiamo di consultare [Parametri comuni del motore](./index.md#common-engine-parameters).
:::

### Regione

Inserisca la regione di archiviazione OSS, ad esempio: `oss-cn-hangzhou`.

:::info{title=Nota}
Può visualizzare le informazioni sulla regione del bucket di archiviazione nella [console Aliyun OSS](https://oss.console.aliyun.com/). È sufficiente utilizzare solo la parte del prefisso della regione (senza il nome di dominio completo).
:::

### AccessKey ID

Inserisca l'ID della chiave di accesso autorizzata di Alibaba Cloud.

### AccessKey Secret

Inserisca il Secret della chiave di accesso autorizzata di Alibaba Cloud.

### Bucket

Inserisca il nome del bucket OSS.
