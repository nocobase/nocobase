# Esportatore di telemetria: HTTP

## Configurazione delle variabili d'ambiente

### TELEMETRY_METRIC_READER

Tipo di lettore delle metriche di telemetria.

```bash
TELEMETRY_METRIC_READER=http
```

### TELEMETRY_HTTP_URL

URL HTTP per l'esportazione dei dati di telemetria.

```bash
TELEMETRY_HTTP_URL=
```
