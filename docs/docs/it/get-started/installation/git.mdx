## 4. Impostare le variabili d'ambiente

Le variabili d'ambiente richieste da NocoBase sono memorizzate nel file `.env` nella directory principale. Le modifichi in base alla Sua situazione.

```bash
TZ=Asia/Shanghai
APP_KEY=your-secret-key
DB_HOST=localhost
DB_PORT=5432
DB_DATABASE=postgres
DB_USER=nocobase
DB_PASSWORD=nocobase
```

:::warning Descrizione delle variabili d'ambiente

- `TZ` serve per impostare il fuso orario dell'applicazione, che per impostazione predefinita Ã¨ quello del sistema operativo.
- `APP_KEY` Ã¨ la chiave segreta dell'applicazione, utilizzata per generare token utente e altro (se `APP_KEY` viene modificata, i vecchi token diventeranno invalidi). PuÃ² essere una stringa casuale qualsiasi. La modifichi con la Sua chiave segreta e si assicuri che non venga esposta.
- `DB_*` sono relative al database. Se non sta utilizzando il servizio di database predefinito dell'esempio, Le modifichi in base alla Sua situazione.

:::
