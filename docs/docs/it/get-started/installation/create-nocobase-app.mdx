# Installazione di create-nocobase-app

:::tip Prerequisiti

- Node.js 20+, Yarn 1.22.x installati
- Configurato e avviato uno dei database richiesti: MySQL 8.0.17+, MariaDB 10.9+, PostgreSQL 10+ (sceglierne uno)
- A causa dell'ambiente di rete in Cina, si consiglia di cambiare il mirror (non obbligatorio)

```bash
$ yarn config set disable-self-update-check true
$ yarn config set registry https://registry.npmmirror.com/
```

:::

:::warning Descrizione delle variabili d'ambiente

- `TZ` viene utilizzato per impostare il fuso orario dell'applicazione, il valore predefinito è il fuso orario del sistema operativo
- `APP_KEY` è la chiave segreta dell'applicazione, utilizzata per generare i token utente, ecc. (se `APP_KEY` viene modificata, anche i vecchi token scadranno). Può essere una stringa casuale qualsiasi. **La preghiamo di modificarla con la Sua chiave segreta e di assicurarsi che non venga divulgata all'esterno**
- `DB_*` sono configurazioni relative al database, La preghiamo di modificarle in base alle Sue effettive informazioni di connessione al database

:::

## 2. Cambiare directory

```bash
cd my-nocobase-app
```

## 3. Installare le dipendenze

📢 A causa dell'ambiente di rete, della configurazione del sistema e di altri fattori, questo passaggio potrebbe richiedere una decina di minuti.

```bash
yarn install
```

:::tip Suggerimento per l'ambiente di produzione

Durante il deployment in ambiente di produzione, per ridurre le dimensioni, è possibile installare solo le dipendenze necessarie:

```bash
yarn install --production
```

:::

## 4. Installare NocoBase

```bash
yarn nocobase install --lang=zh-CN
```

Il processo di installazione creerà automaticamente la struttura delle tabelle del database e inizializzerà i dati.

## 5. Avviare NocoBase

**Ambiente di sviluppo**

```bash
yarn dev
```

**Ambiente di produzione**

```bash
yarn start
```

## 6. Accedere a NocoBase

Utilizzi il browser per aprire [http://localhost:13000](http://localhost:13000), l'account e la password iniziali sono `admin@nocobase.com` e `admin123`.

:::warning Suggerimento per la sicurezza dell'account

Dopo il primo accesso, La preghiamo di modificare tempestivamente la password predefinita per garantire la sicurezza del sistema.

:::
