# Gestione dei plugin

## Gestore dei plugin

Per accedere alla pagina del Gestore dei plugin, clicchi sull'icona corrispondente nell'angolo in alto a destra della pagina. Qui potrà visualizzare, installare, abilitare o disabilitare i plugin.

![2024-04-10_08-47-45](https://static-docs.nocobase.com/2024-04-10_08-47-45.png)

## Centro impostazioni plugin

I plugin solitamente dispongono di proprie pagine di configurazione. Cliccando sull'icona delle impostazioni, situata nell'angolo in alto a destra della pagina, potrà visualizzare i menu di configurazione di tutti i plugin. Basterà poi cliccare sulla voce di menu desiderata per configurare il plugin specifico.
