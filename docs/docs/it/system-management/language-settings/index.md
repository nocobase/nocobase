# Impostazioni della lingua

Nelle "Impostazioni di sistema", può regolare l'ambiente linguistico globale del sistema configurando le "Lingue abilitate". La prima lingua nell'elenco verrà utilizzata come lingua predefinita.

![20240324192005](https://static-docs.nocobase.com/20240324192005.png)

Quando sono abilitate più lingue, gli utenti possono impostare la propria preferenza linguistica personale nel Centro personale (lo trova nell'angolo in alto a destra della pagina).

![20240324192509](https://static-docs.nocobase.com/20240324192509.png)

Se l'ambiente linguistico che sta utilizzando non dispone di traduzioni, può usare il [plugin](/system-management/localization/) "Gestione della localizzazione" per aggiungerle o modificarle.

![20240324193055](https://static-docs.nocobase.com/20240324193055.png)
