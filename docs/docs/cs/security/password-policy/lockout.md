# Uzamčení uživatelů

## Úvod

Spravujte uzamčené uživatele a aktivně uzamykejte uživatele.

![](https://static-docs.nocobase.com/202412281450083.png)

### Odemknutí uživatele

Uživatelé, kteří byli uzamčeni z důvodu překročení limitu neplatných pokusů o přihlášení, mohou být odemčeni smazáním záznamu.

### Uzamčení uživatele

Přidáním uživatelského záznamu můžete uživatele aktivně uzamknout. Jakmile je uživatel uzamčen, bude mu zakázán přístup do systému jakoukoli ověřovací metodou, včetně API klíčů.

![](https://static-docs.nocobase.com/202412281450512.png)
