# Náhled souborů

V rozhraních obsahujících pole pro soubory (včetně polí pro přílohy) můžete soubory zobrazit kliknutím na miniaturu nebo ikonu souboru. Vestavěná funkce náhledu podporuje různé typy souborů, včetně obrázků, PDF a většiny typů souborů nativně podporovaných prohlížeči.

![20251129232307](https://static-docs.nocobase.com/20251129232307.png)

Pro typy souborů, které nepodporují nativní náhled, můžete funkci náhledu povolit instalací nebo rozšířením odpovídajících pluginů pro náhled souborů. Například po instalaci pluginu Office File Preview můžete prohlížet soubory Word, Excel a PowerPoint.

Aktuálně NocoBase nabízí následující pluginy pro náhled souborů:

* [Plugin pro náhled souborů Office](../file-preview/ms-office.md)
