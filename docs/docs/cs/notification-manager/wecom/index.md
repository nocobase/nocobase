# Oznámení: WeCom

## Úvod

Plugin **WeCom** umožňuje aplikaci odesílat notifikační zprávy uživatelům WeCom.

## Přidání a konfigurace autentikátoru WeCom

Nejprve je třeba v NocoBase přidat a nakonfigurovat autentikátor WeCom. Podrobnosti naleznete v [Uživatelské ověřování - WeCom](/auth-verification/auth-wecom). Pouze systémoví uživatelé, kteří se přihlásili přes WeCom, mohou prostřednictvím WeCom přijímat systémová oznámení.

## Konfigurace notifikačního kanálu WeCom

Vyberte autentikátor, který jste právě nakonfigurovali.

![](https://static-docs.nocobase.com/202412041525284.png)

## Konfigurace uzlu oznámení v pracovním postupu

Vyberte nakonfigurovaný notifikační kanál WeCom. Podporuje tři typy zpráv: Textová karta, Markdown a Karta šablony.
