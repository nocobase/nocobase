# Nejčastější dotazy

## E-maily nelze přijímat po autorizaci pomocí účtu Microsoft

A: V současné době podporujeme autorizované přihlášení pouze pro e-mailové účty Outlook a Gmail. Účty Microsoft a Google prozatím nejsou podporovány. Pro více informací se podívejte na: [answers.microsoft.com](https://answers.microsoft.com/zh-hans/outlook_com/forum/all/%E7%8E%B0%E6%9C%89%E5%BE%AE%E8%BD%AF%E8%B4%A6/dba12dda-a7c7-4346-8263-53f4a6d9dc68)

**Tip**: Pokud si nejste jisti, zda máte „skutečný e-mail Outlook.com“ nebo „e-mail Gmail“, zkuste navštívit Outlook.com nebo Gmail.com ve svém webovém prohlížeči. Zjistěte, zda se můžete přímo přihlásit a normálně odesílat e-maily. Pokud ne, znamená to, že pravděpodobně nemáte odpovídající e-mailovou službu a budete ji muset nejprve aktivovat nebo použít jiný e-mail.

## Zrušení autorizace

Pokud je váš e-mailový účet již autorizován, ale přejete si jej odebrat nebo znovu autorizovat, postupujte podle těchto kroků:

#### **Gmail**

1. Otevřete https://myaccount.google.com/u/0/connections a přihlaste se.

![](https://static-docs.nocobase.com/mail-1735634131347.png)

2. Klikněte na příslušnou aplikaci a poté klikněte na „Odebrat“.

![](https://static-docs.nocobase.com/mail-1735634131518.png)

![](https://static-docs.nocobase.com/mail-1735634131697.png)

#### **E-mail Microsoft**

1. Otevřete https://account.microsoft.com/ a přihlaste se.
    
2. Klikněte na tlačítko „Aplikace a služby, které mají přístup k vašim datům“.

![](https://static-docs.nocobase.com/mail-1735634131870.png)

3. Klikněte na „Upravit“ a poté na „Odebrat“.

![](https://static-docs.nocobase.com/mail-1735634132052.png)
