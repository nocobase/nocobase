# Automatické ukládání konceptu

V menu tlačítka **Uložit koncept** můžete povolit **Automatické ukládání konceptu**.

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_29_PM.png)

Během úprav e-mailu systém ukládá koncept v reálném čase, čímž zajistí, že se žádný obsah neztratí.

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_31_PM.png)

Když znovu otevřete editor e-mailů (pro stejného odesílatele a příjemce), systém se vás zeptá, zda chcete použít uložený koncept.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_08_35_PM.png)
