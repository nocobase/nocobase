# Podpis e-mailu

## Nastavení podpisu

Přejděte do Centra správy e-mailů a klikněte na tlačítko nastavení:
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_08_PM.png)

Přepněte se na panel podpisu:
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_08_PM%20(1)

Zde můžete spravovat podpisy. Zaškrtnutím možnosti před podpisem jej nastavíte jako výchozí:
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_09_PM.png)

## Používání podpisů

V textovém poli e-mailu klikněte na ikonu podpisu a vyberte požadovaný podpis:
![](https://static-docs.nocobase.com/email-manager/Loading--12-31-2025_10_11_PM.png)
