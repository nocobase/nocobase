# Podpis e-mailu

## Nastavení podpisu

Přejděte do Centra správy e-mailů a klikněte na tlačítko nastavení:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_05_34_PM.png)

Přepněte se na panel podpisu:
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_05_51_PM.png)

Zde můžete spravovat podpisy. Zaškrtnutím možnosti před podpisem jej nastavíte jako výchozí:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_05_37_PM.png)

## Používání podpisů

V textovém poli e-mailu klikněte na ikonu podpisu a vyberte požadovaný podpis:
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_05_40_PM.png)
