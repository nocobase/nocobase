# Správa e-mailů

## Úvod
Plugin Správa e-mailů je efektivní a pohodlný nástroj, který podporuje autorizaci a připojení e-mailových schránek Gmail a Outlook. Pomáhá uživatelům integrovat funkce správy e-mailů, jako je odesílání a přijímání e-mailů, do různých bloků a stránek. Díky jednoduché konfiguraci autorizace můžete dosáhnout jednotné správy více účtů a užít si bezproblémovou e-mailovou komunikaci.
