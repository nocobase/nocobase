# Správa souborů

## Úvod

Konfigurujte způsob ukládání souborů nahraných během konverzací s AI zaměstnancem.

## Nastavení

Přejděte na konfigurační stránku pluginu AI zaměstnance, klikněte na záložku `Settings` a dostanete se na stránku nastavení způsobu ukládání.

![20251022002648](https://static-docs.nocobase.com/20251022002648.png)

Dostupné možnosti ukládání pro `Files storage` se konfigurují v pluginu pro správu souborů.
