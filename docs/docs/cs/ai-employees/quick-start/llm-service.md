# Rychlý start

## Úvod

Než začnete používat AI zaměstnance, je nutné připojit online službu LLM. NocoBase v současné době podporuje hlavní online služby LLM, jako jsou OpenAI, Gemini, Claude, DepSeek, Qwen a další.
Kromě online služeb LLM NocoBase také podporuje připojení k lokálním modelům Ollama.

## Konfigurace služby LLM

Přejděte na stránku konfigurace **pluginu** AI zaměstnance, klikněte na záložku `LLM service` a dostanete se na stránku správy služeb LLM.

![20251021213122](https://static-docs.nocobase.com/20251021213122.png)

Najeďte myší na tlačítko `Add New` v pravém horním rohu seznamu služeb LLM a vyberte službu LLM, kterou chcete použít.

![20251021213358](https://static-docs.nocobase.com/20251021213358.png)

Vezměme si jako příklad OpenAI: do vyskakovacího okna zadejte snadno zapamatovatelný `title`, poté zadejte `API key` získaný z OpenAI a kliknutím na `Submit` uložte. Tím je konfigurace služby LLM dokončena.

Pole `Base URL` můžete obvykle ponechat prázdné. Pokud používáte službu LLM třetí strany, která je kompatibilní s API OpenAI, vyplňte prosím odpovídající Base URL.

![20251021214549](https://static-docs.nocobase.com/20251021214549.png)

## Test dostupnosti

Na stránce konfigurace služby LLM klikněte na tlačítko `Test flight`, zadejte název modelu, který chcete použít, a klikněte na tlačítko `Run`. Tím otestujete, zda je služba LLM a model dostupný.

![20251021214903](https://static-docs.nocobase.com/20251021214903.png)
