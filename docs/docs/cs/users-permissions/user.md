# Uživatelé

## Správa uživatelů

Tento plugin poskytuje rozhraní pro správu uživatelů v centru konfigurace. Po inicializaci aplikace je automaticky vytvořen neodstranitelný účet superadministrátora. Superadministrátor má oprávnění Root a má přístup ke všem zdrojům.

![](https://static-docs.nocobase.com/44bf40f56b45d4dd96c424fb08082cf6.png)

Kromě toho můžete také přidat různé bloky kolekce Uživatelé pro správu uživatelů, například tabulkové bloky:

![](https://static-docs.nocobase.com/76b5a465f869541a9e8f18a4568a7c9.png)
