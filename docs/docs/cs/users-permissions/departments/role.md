# Správa rolí oddělení

Přiřazením jedné nebo více rolí k oddělení mohou členové daného oddělení získat tyto role.

Existují dva způsoby, jak přiřadit role k oddělení.

První způsob je vybrat roli v poli role ve formuláři pro úpravu oddělení.

![](https://static-docs.nocobase.com/70f77bb89aa1fb415c152a51a51cc23b.png)

Druhý způsob je přidat oddělení k příslušné roli ve správě rolí.

![](https://static-docs.nocobase.com/f2a7bec937cf2f179ce868a92b98416d.png)

Oddělení, která již tuto roli mají, nelze vybrat.

![](https://static-docs.nocobase.com/be10299893581e1f97a4e01ddd5c7e59.png)
