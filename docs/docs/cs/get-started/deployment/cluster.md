# Klastrové nasazení

NocoBase podporuje nasazení aplikací v klastrovém režimu za účelem zlepšení jejich výkonu a dostupnosti. Klastrové nasazení Vám umožňuje spouštět různé instance aplikace na více serverech, nebo více procesů v režimu více jader na jednom serveru.

Podrobnosti naleznete v související dokumentaci k [Klastrovému režimu](/cluster-mode/).
