# I18n internacionalizace

Pluginy NocoBase podporují vícejazyčnou internacionalizaci (i18n) pro frontend i backend. Díky jednotnému mechanismu můžete v pluginech snadno implementovat vícejazyčný obsah.

## Správa vícejazyčných souborů

Vícejazyčné soubory pluginů jsou jednotně uloženy v adresáři `src/locale`. Doporučujeme je pojmenovat podle jazyka, například:

```bash
|- /plugin-hello
  |- /src
    |- /locale
      |- en-US.json   # English language
      |- zh-CN.json   # Chinese language
```

Každý jazykový soubor exportuje JSON objekt, který obsahuje všechny překladové položky pro daný jazyk, například:

```json
// zh-CN.json
{
  "Hello": "你好",
  "World": "世界",
  "Enter your name": "请输入你的名字",
  "Your name is {{name}}": "你的名字是 {{name}}"
}
```

```json
// en-US.json
{
  "Hello": "Hello",
  "World": "World",
  "Enter your name": "Enter your name",
  "Your name is {{name}}": "Your name is {{name}}"
}
```

> Při prvním přidání jazykových souborů je nutné restartovat aplikaci, aby se změny projevily. Funkčnost překladových položek můžete ověřit pomocí API:  
> http://localhost:13000/api/app:getLang?locale=zh-CN
