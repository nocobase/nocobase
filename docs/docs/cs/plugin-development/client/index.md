# Přehled

Vývoj klientských pluginů pro NocoBase nabízí řadu funkcí a možností, které vám pomohou přizpůsobit a rozšířit frontendové funkce NocoBase. Níže naleznete přehled hlavních možností a souvisejících kapitol:

| Modul | Popis | Související kapitola |
|---|---|---|
| **Třída pluginu** | Vytváření a správa klientských pluginů, rozšíření funkcionality frontendu | [plugin.md](plugin.md) |
| **Správa routování** | Vlastní routování na frontendu, implementace navigace a přesměrování stránek | [router.md](router.md) |
| **Práce se zdroji** | Správa frontendových zdrojů, zpracování načítání a operací s daty | [resource.md](resource.md) |
| **Zpracování požadavků** | Vlastní HTTP požadavky, zpracování volání API a přenosu dat | [request.md](request.md) |
| **Správa kontextu** | Získání a použití kontextu aplikace, přístup k globálnímu stavu a službám | [context.md](context.md) |
| **Řízení přístupu (ACL)** | Implementace řízení přístupu na frontendu, kontrola oprávnění k stránkám a funkcím | [acl.md](acl.md) |
| **Správa zdrojů dat** | Správa a používání více zdrojů dat, implementace přepínání a přístupu ke zdrojům dat | [data-source-manager.md](data-source-manager.md) |
| **Styly a motivy** | Vlastní styly a motivy, přizpůsobení a vylepšení vzhledu uživatelského rozhraní | [styles-themes.md](styles-themes.md) |
| **Podpora více jazyků (I18n)** | Integrace podpory více jazyků, implementace internacionalizace a lokalizace | [i18n.md](i18n.md) |
| **Logování** | Vlastní formáty logů a metody výstupu, vylepšení možností ladění a monitorování | [logger.md](logger.md) |
| **Psaní testů** | Psaní a spouštění testovacích případů pro zajištění stability a funkční správnosti pluginu | [test.md](test.md) |
