:::tip

Tato kapitola představí, jak používat ACL (seznam řízení přístupu) pro správu oprávnění ve frontendových pluginech. Obsah zahrnuje:

- Detekce oprávnění;
- Jak dynamicky vykreslovat obsah na základě oprávnění;

V budoucích verzích doplníme podrobné příklady a osvědčené postupy.
