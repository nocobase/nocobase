# Pole Čínský region

## Úvod

Plugin Pole Čínský region (`field-china-region`) přidává do NocoBase typ pole „Čínský region“.

## Instalace

V hlavním rozhraní NocoBase přejděte na „Správce pluginů“, najděte plugin „Čínský region“ a povolte jej.

## Použití

V konfiguraci kolekce přidejte nové pole a jako typ pole vyberte „Čínský region“.

![](/images/field-china-region-1.png)

### Konfigurace pole

V konfiguraci pole můžete nastavit režim zobrazení pro pole:

- Provincie
- Provincie/Město
- Provincie/Město/Městský obvod
- Provincie/Město/Městský obvod/Okres

![](/images/field-china-region-2.png)
