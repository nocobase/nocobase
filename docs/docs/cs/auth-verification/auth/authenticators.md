# Autentizátory

## Správa uživatelské autentizace

Při instalaci pluginu pro uživatelskou autentizaci se automaticky inicializuje metoda autentizace `heslem`, která je založena na uživatelském jménu a e-mailu uživatele.

![](https://static-docs.nocobase.com/66eaa9d5421c9cb713b117366bd8a5d5.png)

## Aktivace autentizátorů

![](https://static-docs.nocobase.com/7f1bf8f8ca5de67ffc68eff0a65848f5.png)

Na přihlašovací stránce se zobrazí pouze aktivované typy autentizace.

![](https://static-docs.nocobase.com/8375a36ef98417af0f0977f1e07345dd.png)

## Typy uživatelské autentizace

![](https://static-docs.nocobase.com/da4250c0cea343ebe470cbf7be4b12e4.png)

Přidáním různých typů autentizátorů můžete do systému přidat odpovídající metody autentizace.

Kromě typů autentizace, které poskytují stávající pluginy, mohou vývojáři také sami rozšířit typy uživatelské autentizace. Podrobnosti naleznete v [Průvodci pro vývojáře](./dev/).
