# Exportér telemetrie: HTTP

## Konfigurace proměnných prostředí

### TELEMETRY_METRIC_READER

Typ exportéru metrik telemetrie.

```bash
TELEMETRY_METRIC_READER=http
```

### TELEMETRY_HTTP_URL

HTTP URL pro export telemetrických dat.

```bash
TELEMETRY_HTTP_URL=
```
