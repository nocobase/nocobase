# Vložení NocoBase

## Úvod

Tento plugin slouží k vkládání stránek NocoBase do jiných webových stránek nebo aplikací.

## Instalace

Tento plugin je komerční a je nutné jej nahrát a aktivovat prostřednictvím správce pluginů.

![20240323162741](https://static-docs.nocobase.com/20240323162741.png)

## Návod k použití

Po aktivaci konfigurace rozhraní klikněte v nastavení stránky (v pravém horním rohu) na „Kopírovat odkaz pro vložení“.

![](https://static-docs.nocobase.com/f11bd6d5e88d38731d7cd3cb149022c8.png)

Po kliknutí získáte odkaz, který lze otevřít samostatně, například `https://xxx.com/embed/qs087rz4o2b`.

![](https://static-docs.nocobase.com/9d847805a00fd07372a474665ec0e968.png)

Pokud chcete vložit stránky NocoBase do jiných webových stránek nebo aplikací, je nutné provést autentizaci uživatele a připojit `token` k odkazu, například `https://xxx.com/embed/qs087rz4o2b?token=xxx`.
