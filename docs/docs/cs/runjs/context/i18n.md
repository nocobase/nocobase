# ctx.i18n

Instance i18n aktuálního kontextu, která se používá ke čtení nebo přepínání jazyků. **Pro překlad textů používejte sjednoceně [ctx.t()](./t.md)**; nepoužívejte `ctx.i18n.t`.

## Vhodné scénáře

Všechna prostředí pro spouštění RunJS mohou používat `ctx.i18n` (např. JSBlock, JSField, JSItem, JSColumn, pracovní postup, pravidla propojení atd.).

## Definice typu

```typescript
interface i18n: {
  language: string;
  changeLanguage(lng: string): Promise<any>;
}
```

## Často používané vlastnosti

| Vlastnost | Typ | Popis |
|------|------|------|
| `language` | `string` | Kód aktuálně aktivního jazyka (např. `zh-CN`, `en-US`) |

## Často používané metody

### changeLanguage(lng)

Přepne aktuální jazyk.

| Parametr | Typ | Popis |
|------|------|------|
| `lng` | `string` | Kód cílového jazyka (např. `'en-US'`, `'zh-CN'`) |

**Návratová hodnota**: `Promise<any>`, vyřeší se (resolve) po dokončení přepnutí jazyka.

## Příklady

### Čtení aktuálního jazyka

```ts
const lang = ctx.i18n.language;
// 'zh-CN' | 'en-US' | ...
if (lang.startsWith('zh')) {
  ctx.render(ctx.t('中文界面'));
} else {
  ctx.render(ctx.t('English UI'));
}
```

### Přepínání jazyků

```ts
// Přepnout do angličtiny
await ctx.i18n.changeLanguage('en-US');

// Přepnout do čínštiny
await ctx.i18n.changeLanguage('zh-CN');
```

### Tlačítko pro přepnutí jazyka

```ts
const { Button } = ctx.libs.antd;
const isZh = ctx.i18n.language.startsWith('zh');
ctx.render(
  <Button onClick={async () => {
    await ctx.i18n.changeLanguage(isZh ? 'en-US' : 'zh-CN');
  }}>
    {ctx.t(isZh ? 'Switch to English' : '切换到中文')}
  </Button>,
);
```

## Poznatky

- **Překlad textů**: Používejte sjednoceně `ctx.t()`, nepoužívejte `ctx.i18n.t`.
