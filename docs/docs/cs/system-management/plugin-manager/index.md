# Správa pluginů

## Správce pluginů

Klikněte na ikonu správce pluginů v pravém horním rohu stránky a dostanete se na stránku správce pluginů. Na této stránce můžete pluginy prohlížet, instalovat, aktivovat nebo deaktivovat.

![2024-04-10_08-47-45](https://static-docs.nocobase.com/2024-04-10_08-47-45.png)

## Centrum nastavení pluginů

Pluginy mají obvykle své vlastní stránky nastavení. Klikněte na ikonu nastavení v pravém horním rohu stránky, abyste si zobrazili nabídky nastavení pro všechny pluginy. Kliknutím na příslušnou položku nabídky můžete plugin nakonfigurovat.
