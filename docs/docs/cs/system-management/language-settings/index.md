# Nastavení jazyka

V „Nastavení systému“ můžete nastavit globální jazykové prostředí systému konfigurací „Povolených jazyků“. První jazyk v seznamu bude sloužit jako výchozí jazyk.

![20240324192005](https://static-docs.nocobase.com/20240324192005.png)

Když je povoleno více jazyků, uživatelé si mohou nastavit své osobní jazykové preference v Osobním centru (v pravém horním rohu stránky).

![20240324192509](https://static-docs.nocobase.com/20240324192509.png)

Pokud aktuálně používané jazykové prostředí postrádá překlady, můžete použít plugin „[Správa lokalizace](/system-management/localization/)“ k jejich přidání nebo úpravě.

![20240324193055](https://static-docs.nocobase.com/20240324193055.png)
