# Potvrzení

## Úvod

Konfigurace potvrzení slouží k ověření, zda má být akce provedena. Po kliknutí na akci se zobrazí potvrzovací dialog. Tlačítko pro smazání má potvrzení ve výchozím nastavení zapnuto, aby se předešlo ztrátě dat nebo nevratným důsledkům způsobeným nechtěnými operacemi.

![20251029125021](https://static-docs.nocobase.com/20251029125021.png)

Můžete si přizpůsobit obsah a titulek potvrzení a také přistupovat k proměnným.
