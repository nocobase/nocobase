# Upravit tlačítko

## Úvod

Systém podporuje přizpůsobení názvu, ikony, barvy pozadí a typu tlačítka. Poskytuje tak flexibilní možnosti přizpůsobení uživatelského rozhraní, které uživatelům pomáhají upravit styl zobrazení a funkčnost tlačítka podle jejich specifických potřeb.

- **Název tlačítka**: Uživatelé si mohou přizpůsobit text tlačítka podle svých obchodních potřeb, čímž zajistí, že popisek tlačítka bude jasný a intuitivní.
- **Ikona tlačítka**: Podporuje nastavení ikony pro tlačítko, což zvyšuje vizuální rozpoznatelnost a umožňuje uživatelům rychle identifikovat funkci tlačítka.
- **Nebezpečná akce**: Pokud tlačítko představuje **nebezpečnou akci** (např. smazání, resetování apod.), lze jej nastavit na typ **Danger Action**.
- **Typ tlačítka (Button Type)**: Tlačítka podporují různá nastavení typů. Mezi běžné typy tlačítek patří:
  - **Výchozí typ (Default)**: Standardní tlačítko, vhodné pro běžné operace.
  - **Primární typ (Primary)**: Používá se pro důležité nebo primární operace, obvykle s výrazným vizuálním efektem.
  - **Přerušovaný typ (Dashed)**: Tlačítko s přerušovaným ohraničením, vhodné pro sekundární operace nebo tlačítka s informativním charakterem.
  - **Textový typ (Text)**: Tlačítko pouze s textem a bez barvy pozadí, obvykle se používá pro nenáročné operace nebo odkazy.
