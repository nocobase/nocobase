# Přiřazení polí

## Úvod

Přiřazení polí umožňuje přednastavit hodnotu pole ještě před spuštěním akce. Při aktualizaci nebo vytváření záznamu se tato data automaticky vyplní, aniž byste je museli zadávat ručně. To se hodí například při hromadné aktualizaci stavů objednávek nebo nastavení aktuálního času doručení, jakmile je objednávka doručena.
