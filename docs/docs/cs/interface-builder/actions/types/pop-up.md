# Otevřít vyskakovací okno

## Úvod

Akce **Otevřít vyskakovací okno** je podobná akcím Zobrazit a Upravit. Uživatelé si mohou nakonfigurovat tlačítko **Otevřít vyskakovací okno** v řádcích nebo záhlaví tabulky. Po kliknutí na tlačítko se otevře odpovídající vyskakovací okno a bloky v něm zobrazí data na základě záznamu aktuálního kontextu.

![20251029182351](https://static-docs.nocobase.com/20251029182351.png)

Nakonfigurujte bloky ve vyskakovacím okně podle potřeby.

![20251029190208](https://static-docs.nocobase.com/20251029190208.png)

## Nastavení akce

![20251029190352](https://static-docs.nocobase.com/20251029190352.png)

- [Upravit tlačítko](/interface-builder/actions/action-settings/edit-button): Upravte název, typ a ikonu tlačítka.
- [Pravidlo propojení](/interface-builder/linkage-rule): Dynamicky ovládejte stav tlačítka (viditelné/zakázané).
- [Upravit vyskakovací okno](/interface-builder/actions/action-settings/edit-popup): Nastavte způsob otevření (vyskakovací okno nebo postranní panel) a velikost.
