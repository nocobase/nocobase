# Hromadná úprava

## Úvod

Hromadná úprava je vhodná pro scénáře vyžadující flexibilní dávkové aktualizace dat. Po kliknutí na tlačítko hromadné úpravy můžete ve vyskakovacím okně nakonfigurovat formulář hromadné úpravy a nastavit různé strategie aktualizace pro jednotlivá pole.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM.png)


## Konfigurace akce

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM%20(1).png)


## Uživatelská příručka

### Konfigurace formuláře hromadné úpravy

1. Přidejte tlačítko hromadné úpravy.

2. Nastavte rozsah hromadné úpravy: Vybrané / Všechny, výchozí hodnota je Vybrané.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_14_AM.png)

3. Přidejte formulář hromadné úpravy.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_14_AM.png)

4. Nakonfigurujte pole, která mají být upravena, a přidejte tlačítko pro odeslání.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM%20(1).png)

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM.png)

### Odeslání formuláře

1. Zaškrtněte řádky dat, které chcete upravit.

2. Vyberte režim úprav pro pole a vyplňte hodnoty, které mají být odeslány.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_10_33_AM.png)

:::info{title=Dostupné režimy úprav}
* **Neaktualizovat**: Pole zůstane beze změny.
* **Změnit na**: Aktualizuje pole na odeslanou hodnotu.
* **Vymazat**: Vymaže data v poli.

:::

3. Odešlete formulář.
