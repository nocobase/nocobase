# Odkaz

## Úvod

Akce Odkaz funguje na principu přesměrování a podporuje předávání proměnných URL. Cílová stránka tak může dynamicky upravovat svůj obsah na základě předaných dat. Tuto akci můžete konfigurovat v datových blocích.
