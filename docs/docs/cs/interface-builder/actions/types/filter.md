# Filtrování

## Úvod

Operace filtrování se obvykle objevuje v datových blocích, kde slouží k filtrování dat nastavením různých podmínek. Mějte na paměti, že pokud je pro blok již nakonfigurován rozsah dat, operace filtrování se sloučí s podmínkami rozsahu dat a vytvoří tak konečnou podmínku filtrování.

## Nastavení akce

![20251029173708](https://static-docs.nocobase.com/20251029173708.png)

### Filtrovatelná pole

Ve výchozím nastavení jsou povolena všechna pole této kolekce. Pouze povolená pole lze vybrat v seznamu polí pro podmínky.

Pole z přidružených kolekcí lze také použít jako pole pro filtrování.
