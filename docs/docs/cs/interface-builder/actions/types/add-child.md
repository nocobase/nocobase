# Přidat podřízený záznam

## Úvod

Tato operace je speciálně navržena pro **bloky stromové tabulky** pro rychlé přidávání podřízených záznamů. Při přidávání podřízeného záznamu je vybraný řádek považován za rodičovský uzel, což Vám umožňuje snadno vytvářet podřízené záznamy pod libovolným rodičovským uzlem. Tato funkce výrazně zjednodušuje správu hierarchických dat a činí tak rozšiřování a organizaci datových struktur efektivnější a intuitivnější.
