# Upravit

## Úvod

Akce Upravit je tlačítko řádku, které má aktuální záznam jako svůj kontext. Zobrazuje se jako vyskakovací okno, kde si uživatelé mohou přizpůsobit a konfigurovat bloky.

![20251029165442](https://static-docs.nocobase.com/20251029165442.png)

Konfigurujte bloky podle potřeby.

![20251029165514](https://static-docs.nocobase.com/20251029165514.png)

## Nastavení akce

![20251029165620](https://static-docs.nocobase.com/20251029165620.png)

- [Pravidla propojení](/interface-builder/actions/action-settings/linkage-rule): Dynamicky zobrazit/skrýt tlačítko;
- [Tlačítko Upravit](/interface-builder/actions/action-settings/edit-button): Upravit název, typ a ikonu tlačítka;
- [Vyskakovací okno Upravit](/interface-builder/actions/action-settings/edit-popup): Nastavit způsob otevření a velikost.
