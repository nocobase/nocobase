# Graf

## Úvod

Plugin NocoBase pro vizualizaci dat nabízí vizuální dotazování dat a bohatou sadu komponent pro grafy. Pomocí jednoduché konfigurace můžete rychle vytvářet vizualizační panely, zobrazovat datové přehledy a podporovat vícerozměrnou analýzu a prezentaci dat.
