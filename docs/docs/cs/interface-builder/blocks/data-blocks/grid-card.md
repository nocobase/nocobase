# Mřížková karta

## Úvod

Blok Mřížková karta zobrazuje souhrnné informace o datových záznamech ve formě karet. Podporuje konfiguraci počtu sloupců pro různé velikosti obrazovek, aby zajistil uživatelsky přívětivé zobrazení na různých zařízeních.
