# Rozložení

## Úvod

Bloky formulářového typu, jako jsou bloky formuláře, bloky detailů a bloky seznamu, podporují nastavení rozložení. Možnosti rozložení zahrnují vertikální rozložení (popisky a vstupní pole jsou uspořádány po řádcích) a horizontální rozložení (popisky a vstupní pole jsou umístěny vedle sebe). Rozložení polí bloku můžete upravit podle svých konkrétních potřeb, abyste zajistili, že zadávání nebo prohlížení informací bude jasné a efektivní.

### Vertikální (výchozí)

Výchozí rozložení je vertikální.

![20251024120143](https://static-docs.nocobase.com/20251024120143.png)

### Horizontální

Horizontální rozložení zobrazuje popisky a vstupní pole vedle sebe, což efektivně šetří místo. U polí s dlouhými názvy můžete zvolit zalomení textu nebo zobrazení tří teček (elipsy), abyste zajistili čisté a esteticky příjemné rozhraní.

![20251024120237](https://static-docs.nocobase.com/20251024120237.png)

#### Zarovnání názvu pole

- **Zarovnání vlevo** (výchozí)
- **Zarovnání vpravo**

#### Šířka názvu pole

- Výchozí šířka je **120px**, s minimální šířkou **50px**.

#### Způsob zpracování názvů polí přesahujících šířku

- **Zalomení textu**
- **Tři tečky (elipsa)**

#### Zobrazit dvojtečku

Můžete zvolit, zda se má za názvem pole zobrazit dvojtečka.
