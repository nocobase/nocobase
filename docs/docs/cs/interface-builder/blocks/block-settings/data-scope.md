# Nastavení rozsahu dat

## Úvod

Nastavení rozsahu dat znamená definování výchozích podmínek filtrování pro datový blok. Uživatelé mohou rozsah dat flexibilně upravovat podle svých obchodních potřeb, ale bez ohledu na jakékoli provedené operace filtrování systém automaticky použije tuto výchozí podmínku filtru, čímž zajistí, že data vždy zůstanou v rámci specifikovaného rozsahu.

## Uživatelská příručka

![20251027110053](https://static-docs.nocobase.com/20251027110053.png)

Filtrační pole podporuje výběr polí z aktuální kolekce a souvisejících kolekcí.

![20251027110140](https://static-docs.nocobase.com/20251027110140.png)

### Operátory

Různé typy polí podporují různé operátory. Například textová pole podporují operátory jako „rovná se“, „nerovná se“ a „obsahuje“; číselná pole podporují operátory jako „větší než“ a „menší než“; zatímco datová pole podporují operátory jako „je v rozsahu“ a „je před konkrétním datem“.

![20251027111124](https://static-docs.nocobase.com/20251027111124.png)
