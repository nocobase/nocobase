# Režim načítání dat

## Úvod

Výchozím režimem načítání dat je „Načíst všechna data, pokud jsou filtrační podmínky prázdné“. „Filtračními podmínkami“ se zde rozumí hodnoty ve „Filtračním bloku“. Další možností je „Nenačítat data, pokud jsou filtrační podmínky prázdné“.
