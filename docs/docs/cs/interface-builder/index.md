# Tvorba uživatelského rozhraní

## WYSIWYG (Co vidíte, to dostanete)

NocoBase využívá režim tvorby uživatelského rozhraní na principu WYSIWYG (What You See Is What You Get). Kliknutím na tlačítko pro tvorbu UI můžete přepínat mezi režimem konfigurace a režimem zobrazení.

## Šablona rozvržení

NocoBase obsahuje výchozí šablonu rozvržení: navigační oblast nahoře a vlevo, a obsahová oblast vpravo.

![未命名.002](https://static-docs.nocobase.com/未命名.002.jpeg)

## Možnosti konfigurace

Jakmile vstoupíte do režimu tvorby uživatelského rozhraní, objeví se na obrazovce oranžové značky označující konfigurovatelné prvky.
