# Koncepty formulářů

Po povolení konceptů formulářů se vyplňovaný obsah automaticky ukládá lokálně. I když formulář nechtěně zavřete nebo stránku obnovíte, vyplněná data se neztratí.

![20251213093717](https://static-docs.nocobase.com/20251213093717.png)

Pokud existuje koncept, v horní části formuláře se zobrazí oznámení. Po úspěšném odeslání bude koncept automaticky smazán; pokud jej již nepotřebujete, můžete jej také kdykoli smazat ručně.

![20251213092734](https://static-docs.nocobase.com/20251213092734.png)

::: tip Tip

V současné době podporují koncepty formulářů pouze lokální mezipaměť. Budoucí verze budou podporovat ukládání na straně serveru, což umožní synchronizaci mezi více zařízeními.

:::
