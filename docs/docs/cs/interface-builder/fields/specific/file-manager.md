# Správce souborů

## Úvod

Správce souborů je komponenta asociačního pole určená pro práci se vztahy, kde cílovou kolekcí je kolekce souborů.

![20251027194124](https://static-docs.nocobase.com/20251027194124.png)

## Konfigurace pole

- [Komponenta pole](/interface-builder/fields/association-field): Přepněte na jiné komponenty asociačního pole, jako je rozevírací seznam, selektor dat apod.
- Povolit výběr existujících souborů: Ve výchozím nastavení je tato možnost zapnuta a umožňuje vybrat a asociovat soubory z již nahraných.
