# Název

## Úvod

V režimu čtení se asociační pole standardně zobrazují pomocí **komponenty titulního pole**, která identifikuje aktuálně asociovaný záznam zobrazením hodnoty titulního pole. Systém automaticky přiřadí a vykreslí odpovídající komponentu titulního pole na základě jeho typu.

## Možnosti konfigurace pole

### Titulní pole

Pokud je typ titulního pole „Číslo“:

![20251028155817](https://static-docs.nocobase.com/20251028155817.png)

Komponenta pole je číselná komponenta.

![20251028155837](https://static-docs.nocobase.com/20251028155837.png)

Pokud je typ titulního pole „Datum a čas“:

![20251029213244](https://static-docs.nocobase.com/20251029213244.png)

Komponenta pole se přepne na komponentu data.

![20251029213320](https://static-docs.nocobase.com/20251029213320.png)

![20251028155402](https://static-docs.nocobase.com/20251028155402.png)

![20251028155644](https://static-docs.nocobase.com/20251028155644.png)

### Povolit otevření kliknutím (standardně povoleno)

Kliknutím otevřete vyskakovací okno, kde můžete přizpůsobit bloky pro správu dat.

![20251028160742](https://static-docs.nocobase.com/20251028160742.png)

Více podrobností naleznete v [Komponenta pole](/interface-builder/fields/association-field).
