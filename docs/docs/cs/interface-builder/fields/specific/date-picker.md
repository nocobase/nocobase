# Datum a čas

## Úvod

Pole Datum podporuje vlastní formáty data a času.

## Kde nastavit formát data

### Pole kolekce

![20240417114311](https://static-docs.nocobase.com/20240417114311.png)

> Formát data nastavený v poli kolekce bude sloužit jako výchozí formát pole. Může být přepsán formátem data nastaveným v nastavení pole v rámci konfigurace bloku.

### Nastavení pole

![20251027191824](https://static-docs.nocobase.com/20251027191824.png)
