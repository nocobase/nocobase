# Datový selektor

## Úvod

Datový selektor, ve formě vyskakovacího tabulkového selektoru, umožňuje vybrat existující data z cílové kolekce pro asociaci, nebo přidat nová data do cílové kolekce a následně je asociovat.

![20251027203343](https://static-docs.nocobase.com/20251027203343.png)

Již vybraná data se v tabulkovém selektoru nezobrazují.

![20251027203717](https://static-docs.nocobase.com/20251027203717.gif)

V tabulkovém selektoru můžete také dále spravovat cílovou asociovanou kolekci (přidávat, mazat, importovat/exportovat atd.).

![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
