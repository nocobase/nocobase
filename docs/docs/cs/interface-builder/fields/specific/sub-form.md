# Podformulář

## Úvod

Podformuláře se hodí pro situace, kdy potřebujete vytvořit nová související data a následně je propojit. Víceúrovňová související data jsou přehledně zobrazena ve formě vnořeného formuláře. Oproti komponentám pro výběr dat (Data Selector) a rozbalovacímu seznamu (Dropdown) vám podformulář umožňuje přímo spravovat pole související kolekce v rámci aktuálního bloku stránky a související data se odesílají společně s hlavním formulářem.
