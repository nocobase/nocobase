# Kaskádový výběr

## Úvod

Kaskádový výběr je určen pro relační pole, jejichž cílová kolekce je stromová kolekce. Umožňuje uživatelům vybírat data podle hierarchické struktury stromové kolekce a podporuje částečné vyhledávání pro rychlé filtrování.

## Pokyny k použití

- Pro vazbu typu **jedna k jedné** je kaskádový výběr **jednoduchý**.

![20251125214656](https://static-docs.nocobase.com/20251125214656.png)

- Pro vazbu typu **jedna k mnoha** je kaskádový výběr **vícenásobný**.

![20251125215318](https://static-docs.nocobase.com/20251125215318.png)

## Možnosti konfigurace pole

### Pole názvu

Pole názvu určuje popisek zobrazený pro každou možnost.

![20251125214923](https://static-docs.nocobase.com/20251125214923.gif)

> Podporuje rychlé vyhledávání na základě pole názvu.

![20251125215026](https://static-docs.nocobase.com/20251125215026.gif)

Více informací naleznete zde: [Pole názvu](/interface-builder/fields/field-settings/title-field)

### Rozsah dat

Řídí rozsah dat stromového seznamu (pokud podřízený záznam splňuje podmínky, bude zahrnut i jeho nadřízený záznam).
