# Podrobné zobrazení

## Úvod

Podrobné zobrazení je komponenta odpovídající podřízenému formuláři v režimu čtení. Oproti komponentám Popisek a Název dokáže Podrobné zobrazení nejen zobrazit více dat z aktuální kolekce, ale lze jej také nakonfigurovat tak, aby zobrazovalo informace o datech ze souvisejících kolekcí. Data s víceúrovňovými vztahy jsou přehledně zobrazena ve formě vnořených podrobností.
