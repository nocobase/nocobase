# Filtrovací pole formuláře

## Úvod

Filtrovací formulář umožňuje vybrat pole z aktuální kolekce a souvisejících kolekcí (včetně polí z vnořených vztahů) jako pole pro filtrování.

Vyberte vhodný operátor na základě typu pole, abyste zlepšili přesnost a efektivitu filtrování. U textových polí je výchozí nastavení přibližná shoda.
