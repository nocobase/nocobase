# Pole formuláře

## Úvod

Různé typy polí ve formuláři mají odlišné možnosti konfigurace. Uživatelé mohou tyto možnosti rozšířit pomocí pluginů, aby vyhověli svým specifickým potřebám.

![20251025174555](https://static-docs.nocobase.com/20251025174555.png)

## Možnosti konfigurace polí

### Výchozí hodnota

Pro pole můžete nastavit konstanty/proměnné jako výchozí hodnoty. (Podporováno pouze pro nové formuláře.)

![20251025174724](https://static-docs.nocobase.com/20251025174724.png)

Více informací naleznete v části [Nastavení výchozí hodnoty](/interface-builder/fields/field-settings/default-value)
