# Pole

V uživatelském rozhraní (UI) je pole komponentou pole, která slouží jako nosič jednotlivých dat. Různé typy dat jsou prezentovány různými komponentami pole. Pole mohou být připojena pouze k blokům a nelze je používat samostatně.

## Pole v bloku

Pole se obvykle nepoužívají samostatně, ale existují jako podřízené prvky bloků datového typu. Bloky datového typu obvykle obsahují možnost „Konfigurovat pole“ a seznam polí je poskytován aktuální kolekcí.

![20251029204240](https://static-docs.nocobase.com/20251029204240.png)

## Návrhář polí (Panel nástrojů)

Stejně jako u bloků má i komponenta pole v pravém horním rohu tři ikony, a to:

- Přetažení pro rozvržení
- [Tok událostí](/interface-builder/event-flow)
- Konfigurace

![20251025165432](https://static-docs.nocobase.com/20251025165432.png)

## Rozvržení polí

Přetažením můžete přizpůsobit rozvržení polí v bloku.
