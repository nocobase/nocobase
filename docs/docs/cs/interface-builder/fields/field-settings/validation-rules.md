# Nastavení pravidel ověřování

## Úvod

Pravidla ověřování slouží k zajištění, že data zadaná uživatelem splňují očekávání.

## Kde můžete nastavit pravidla ověřování polí

### Konfigurace pravidel ověřování pro pole kolekce

Většina polí podporuje konfiguraci pravidel ověřování. Jakmile je pole nakonfigurováno s pravidly ověřování, spustí se ověřování na backendu při odeslání dat. Různé typy polí podporují různá pravidla ověřování.

- **Datumové pole**

  ![20251028225946](https://static-docs.nocobase.com/20251028225946.png)

- **Číselné pole**

  ![20251028230418](https://static-docs.nocobase.com/20251028230418.png)

- **Textové pole**

  Kromě omezení délky textu podporují textová pole také vlastní regulární výrazy pro detailnější ověřování.

  ![20251028230554](https://static-docs.nocobase.com/20251028230554.png)

### Ověřování na frontendu v konfiguraci pole

Pravidla ověřování nastavená v konfiguraci pole spustí ověřování na frontendu, aby se zajistilo, že zadaná data splňují požadavky.

![20251028230105](https://static-docs.nocobase.com/20251028230105.png)

![20251028230255](https://static-docs.nocobase.com/20251028230255.png)

**Textová pole** také podporují vlastní regulární výrazy pro ověřování, aby splňovala specifické požadavky na formát.

![20251028230903](https://static-docs.nocobase.com/20251028230903.png)
