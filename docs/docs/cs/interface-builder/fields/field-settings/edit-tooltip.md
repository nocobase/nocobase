# Úprava popisku

## Úvod

Popisek pole je text, který se zobrazí, když najedete myší na pole, a slouží k poskytnutí dalšího vysvětlení nebo pokynů k danému poli. Tyto popisky obvykle obsahují informace, jako je účel pole, formát vstupu a omezení, což uživatelům pomáhá lépe porozumět poli a používat jej.
