# Formátování čísel

## Úvod

Formátování čísel se týká vizuálního vylepšení a standardizace číselných polí (včetně polí s formulí) při jejich zobrazení. Zahrnuje nastavení oddělovače tisíců, přesnosti desetinných míst, převodu jednotek a vědecké notace. Konfigurací formátování čísel můžete vyhovět zvyklostem vyjádření čísel v různých regionech a odvětvích.

![20251028215021](https://static-docs.nocobase.com/20251028215021.png)

![20251028215059](https://static-docs.nocobase.com/20251028215059.png)

## Jak používat

- Podporuje jednoduchý převod jednotek (násobení/dělení)
- Oddělovač tisíců
- Předpona a přípona
- Přesnost
- Vědecká notace.

Příklad: Naformátujte částku objednávky přidáním symbolu měny (např. Kč, $), oddělovače tisíců (např. čárka nebo tečka) a přesnosti desetinných míst, aby byla částka jasná a snadno čitelná.

![20251028215249](https://static-docs.nocobase.com/20251028215249.gif)

- Použijte vědeckou notaci.
