# Komponenta pole

## Úvod

Komponenty polí slouží k zobrazení obsahu pole.
Některá pole mají vestavěno více zobrazovacích komponent. Uživatelé si mohou vybrat vhodnou komponentu podle svých aktuálních potřeb a prezentovat tak hodnotu pole různými způsoby.

Příklad: V režimu zobrazení lze pole typu URL přepnout na komponentu Náhled nebo komponentu URL (výchozí).

![20251028214033](https://static-docs.nocobase.com/20251028214033.png)

![20251028214628](https://static-docs.nocobase.com/20251028214628.png)
