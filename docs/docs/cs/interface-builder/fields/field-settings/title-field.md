# Pole názvu

## Úvod

**Pole názvu** se obvykle používá v komponentách relačních polí jako identifikátor pro asociovaná data zobrazená v uživatelském rozhraní. Určuje, jak se prezentují hlavní informace dat, a obvykle pomáhá uživatelům data rychle identifikovat a pochopit.

Různé typy polí názvu odpovídají různým komponentám polí a podporují odpovídající možnosti konfigurace. Například:

- Když je pole názvu **polem data**, komponenta pole použije **komponentu data** k zobrazení dat.
- Když je pole názvu **textovým polem**, komponenta pole použije **textovou komponentu** k zobrazení dat.
- Když je pole názvu **polem s volbami**, komponenta pole použije **komponentu s volbami** k zobrazení dat.
