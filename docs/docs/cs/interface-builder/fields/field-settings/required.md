# Povinné

## Úvod

Povinnost vyplnění je běžné pravidlo pro ověřování formulářů. Můžete ji aktivovat přímo v nastavení pole, nebo dynamicky nastavit pole jako povinné prostřednictvím pravidel propojení formuláře.

## Kde nastavit pole jako povinné

### Nastavení pole kolekce

Když je pole kolekce nastaveno jako povinné, spustí se ověřování na straně serveru a frontend jej také standardně zobrazuje jako povinné (nelze upravit).
![20251025175418](https://static-docs.nocobase.com/20251025175418.png)

### Nastavení pole

Nastavte pole přímo jako povinné. To je vhodné pro pole, která uživatel musí vždy vyplnit, jako je uživatelské jméno, heslo atd.

![20251028222818](https://static-docs.nocobase.com/20251028222818.png)

### Pravidla propojení

Nastavte pole jako povinné na základě podmínek prostřednictvím pravidel propojení polí bloku formuláře.

Příklad: Číslo objednávky je povinné, pokud datum objednávky není prázdné.

![20251028223004](https://static-docs.nocobase.com/20251028223004.png)
