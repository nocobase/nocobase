# Zobrazení štítků

## Úvod

Pole v rámci bloku standardně zobrazují své štítky, které identifikují význam každého pole. To uživatelům usnadňuje čtení a porozumění obsahu dat.

V některých scénářích (například na stránkách určených pouze k zobrazení, nebo když je význam polí již velmi jasný) se můžete rozhodnout štítky skrýt, abyste dosáhli čistšího rozhraní. Po skrytí štítků se zobrazí pouze hodnoty polí a názvy polí se již nezobrazují.
