# Stromová kolekce

## Úvod

Stromová kolekce je vzor návrhu datové kolekce, který se používá pro organizaci hierarchických dat. Tato struktura kolekce připomíná strom, kde každá datová položka může mít jednu nebo více podřízených položek a tyto podřízené položky mohou mít zase své vlastní potomky.
