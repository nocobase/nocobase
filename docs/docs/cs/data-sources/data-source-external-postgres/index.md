# Externí zdroj dat - PostgreSQL

## Úvod

Použijte externí databázi PostgreSQL jako zdroj dat. Aktuálně podporované verze: PostgreSQL >= 9.5

## Instalace

Tento plugin je komerční plugin, který je nutné nahrát a aktivovat prostřednictvím správce pluginů.
