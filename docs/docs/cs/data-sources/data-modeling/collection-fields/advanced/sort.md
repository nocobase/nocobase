# Selektor kolekcí

Pole Selektor kolekcí je speciální typ pole, který uživatelům umožňuje vybrat jednu nebo více kolekcí z předdefinovaného seznamu. Tento typ pole je velmi užitečný ve scénářích, kde potřebujete dynamicky odkazovat na různé kolekce, například při vytváření konfigurovatelných reportů, dashboardů nebo komplexních aplikací, které potřebují měnit zdroje dat na základě výběru uživatele.

## Případy použití

### Dynamické generování reportů

V modulu pro generování reportů mohou uživatelé potřebovat vybrat kolekci z více kolekcí jako zdroj dat pro report. Pomocí Selektoru kolekcí mohou uživatelé snadno vybrat požadovanou kolekci a systém dynamicky vygeneruje report na základě výběru uživatele.

### Konfigurovatelné dashboardy

V aplikaci dashboardu mohou uživatelé přizpůsobit data zobrazená na dashboardu. Selektor kolekcí umožňuje uživatelům vybrat kolekce, o které mají zájem, čímž se na dashboardu zobrazí grafy a statistická data z různých zdrojů dat.

### Flexibilní nástroje pro import/export dat

Při vývoji obecného nástroje pro import nebo export dat můžete použít Selektor kolekcí k tomu, aby uživatelé určili kolekci, se kterou se má pracovat. Tímto způsobem lze stejný nástroj použít ke zpracování dat z jakékoli kolekce v systému, čímž se výrazně zlepší znovupoužitelnost nástroje.
