# Externí zdroj dat - MariaDB

## Úvod

Použijte externí databázi MariaDB jako zdroj dat. Aktuálně podporované verze: MariaDB >= 10.3

## Instalace

Tento plugin je komerční plugin, který je třeba nahrát a aktivovat prostřednictvím správce pluginů.
