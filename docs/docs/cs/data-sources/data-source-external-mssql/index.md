# Externí zdroj dat - MSSQL

## Úvod

Použijte externí databázi MSSQL jako zdroj dat. Aktuálně podporované verze: SQL Server 2014-2019.

## Instalace

Tento plugin je komerční plugin, který je třeba nahrát a aktivovat prostřednictvím správce pluginů.
