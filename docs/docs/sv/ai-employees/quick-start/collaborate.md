# Snabbstart

## Introduktion

Efter att ni har skapat en AI-medarbetare kan ni samarbeta med den på vilken sida som helst i NocoBase.

Ni kan väcka dem genom att klicka på medarbetarens avatar i medarbetarlistan nere till höger på sidan, och sedan börja samarbeta.

![20251102121159-2025-11-02-12-12-01](https://static-docs.nocobase.com/20251102121159-2025-11-02-12-12-01.png)

Ni kan också placera AI-medarbetare där ni behöver dem – till exempel i tabeller eller formulär – och aktivera dem direkt med ett klick.

![20251102121036-2025-11-02-12-10-38](https://static-docs.nocobase.com/20251102121036-2025-11-02-12-10-38.png)

## Grundläggande funktioner i chattfönstret

När ni har aktiverat AI-medarbetaren, skriver ni in ert meddelande och klickar på skicka-knappen för att starta konversationen.

![20251021234541](https://static-docs.nocobase.com/20251021234541.png)

När ni håller muspekaren över ett användarmeddelande visas ikoner för snabbåtgärder:
- Redigera och skicka meddelandet igen
- Kopiera användarmeddelandets innehåll

![20251021235715](https://static-docs.nocobase.com/20251021235715.png)

När ni håller muspekaren över ett AI-meddelande visas ikoner för snabbåtgärder:
- Be AI:n svara igen
- Kopiera AI-meddelandets innehåll

![20251021235847](https://static-docs.nocobase.com/20251021235847.png)

Klicka på ikonen för "Ladda upp" för att ladda upp och skicka en fil till AI-medarbetaren.

![20251022000327](https://static-docs.nocobase.com/20251022000327.png)

I chattfönstrets övre vänstra hörn finns listan över tidigare konversationer. Klicka för att expandera listan och se gamla konversationer.

![20251021234640](https://static-docs.nocobase.com/20251021234640.png)

Klicka på "penn"-ikonen i det övre högra hörnet för att starta en ny konversation.

![20251021234841](https://static-docs.nocobase.com/20251021234841.png)

Klicka på "i"-ikonen i det övre högra hörnet för att redigera den anpassade systemprompten.

![20251021235314](https://static-docs.nocobase.com/20251021235314.png)

Klicka på "expandera"-ikonen i det övre högra hörnet för att visa AI-medarbetarens chattfönster i helskärmsläge.

![20251021235427](https://static-docs.nocobase.com/20251021235427.png)
