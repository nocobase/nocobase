# AI-agent · Dex: Expert på datahantering

> Extrahera snabbt information och fyll i formulär för att omvandla ostrukturerad data till strukturerad.

`Dex` är din expert på datahantering. Den extraherar önskad information från ostrukturerad data eller filer, organiserar den till strukturerad information och kan dessutom använda verktyg för att fylla i formulär.

![20251022155210](https://static-docs.nocobase.com/20251022155210.png)

## Steg

På formulärsidan kan ni öppna dialogfönstret genom att kalla på `Dex`.

![20251022155316](https://static-docs.nocobase.com/20251022155316.png)

Klicka på `Add work context` i inmatningsfältet och välj `Pick block`. Sidan går då in i ett läge för att välja block.

![20251022155414](https://static-docs.nocobase.com/20251022155414.png)

Välj formulärblocket på sidan.

![20251022155523](https://static-docs.nocobase.com/20251022155523.png)

I dialogrutan anger ni den data som ni vill att `Dex` ska fylla i.

![20251022155555](https://static-docs.nocobase.com/20251022155555.png)

Efter att ni har skickat kommer `Dex` att strukturera datan och använda sina färdigheter för att uppdatera det valda formuläret med informationen.

![20251022155733](https://static-docs.nocobase.com/20251022155733.png)
