# AI-agent · Orin: Datamodellerare

> Designa samlingar intelligent och optimera databasstrukturen.

`Orin` är en expert på datamodellering. På konfigurationssidan för den primära datakällan kan ni låta `Orin` hjälpa er att skapa eller ändra samlingar.

![20251022160628](https://static-docs.nocobase.com/20251022160628.png)

## Steg

Gå till datakällahanterings-`plugin`en och välj att konfigurera den primära datakällan.

![20251022161146](https://static-docs.nocobase.com/20251022161146.png)

Klicka på `Orin`-avataren i det övre högra hörnet för att öppna dialogrutan för AI-agenten.

![20251022161641](https://static-docs.nocobase.com/20251022161641.png)

Beskriv era modelleringsbehov för `Orin`, skicka det och vänta på svar.

![20251022161749](https://static-docs.nocobase.com/20251022161749.png)

När `Orin` har bekräftat era behov kommer den att använda sina färdigheter och svara med en förhandsgranskning av datamodellen.

Efter att ni har granskat förhandsgranskningen, klicka på knappen `Finish review and apply` för att skapa samlingar enligt `Orin`s modell.

![20251022162142](https://static-docs.nocobase.com/20251022162142.png)
