# AI-medarbetare · Nathan: Frontend-utvecklare

> Hjälper dig att skriva och optimera frontend-kod för att implementera komplex interaktiv logik.

`Nathan` är NocoBase expert på frontend-utveckling. I scenarier som kräver JavaScript-kod, som till exempel `JSBlock`, `JSField`, `JSItem`, `JSColumn`, `JSAction`, `Event Flow` och `Linkage`, ser ni `Nathan`s avatar i det övre högra hörnet av kodredigeraren. Där kan ni få hjälp av honom att skriva eller ändra koden i redigeraren.

![20251022163405](https://static-docs.nocobase.com/20251022163405.png)

## Så här gör ni

I kodredigeraren klickar ni på `Nathan` för att öppna dialogrutan för AI-medarbetaren. Koden från redigeraren bifogas automatiskt till inmatningsrutan och skickas till `Nathan` som applikationskontext.

![20251022163935](https://static-docs.nocobase.com/20251022163935.png)

Ange era kodningsbehov och skicka dem till `Nathan`, vänta sedan på hans svar.

![20251022164041](https://static-docs.nocobase.com/20251022164041.png)

Klicka på knappen `Apply to editor` på kodblocket i `Nathan`s svar för att skriva över koden i kodredigeraren.

![20251022164323](https://static-docs.nocobase.com/20251022164323.png)

Klicka på knappen `Run` i kodredigeraren för att se effekten i realtid.

![20251022164436](https://static-docs.nocobase.com/20251022164436.png)

## Kodhistorik

Klicka på 'kommandorads'-ikonen i det övre högra hörnet av `Nathan`s dialogruta för att se de kodsnuttar ni har skickat och de som `Nathan` har svarat med under den aktuella sessionen.
