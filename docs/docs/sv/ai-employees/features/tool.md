# Använda färdigheter

Färdigheter (Tools) definierar vad en AI-medarbetare kan göra.

## Färdighetsstruktur

Sidan för färdigheter är uppdelad i tre kategorier:

1. `General skills`: delas av alla AI-medarbetare, skrivskyddad.
2. `Employee-specific skills`: specifika för den aktuella medarbetaren, vanligtvis skrivskyddade.
3. `Custom skills`: anpassade färdigheter som kan läggas till, tas bort och konfigureras med standardbehörigheter.

![skills-three-sections-general-specific-custom.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/skills-three-sections-general-specific-custom.png)

## Behörigheter för färdigheter

Behörigheter för färdigheter är enhetliga:

- `Ask`: be om bekräftelse före anrop.
- `Allow`: tillåt direktanrop.

Rekommendation: använd `Ask` som standard för färdigheter som innebär ändring av data.

![skills-permission-ask-allow-segmented.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/skills-permission-ask-allow-segmented.png)

## Lägga till och underhålla

Klicka på `Add skill` under `Custom skills` för att lägga till färdigheter och konfigurera behörigheter baserat på verksamhetsrisk.
