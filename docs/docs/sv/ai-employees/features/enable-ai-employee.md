# Aktivera AI-medarbetare

NocoBase erbjuder flera inbyggda AI-medarbetare för olika scenarier.

## Hur ni aktiverar

Öppna hanteringssidan för `AI employees` och slå på strömbrytaren `Enabled` direkt i listan för att aktivera medarbetaren.

![ai-employee-list-enable-switch.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employee-list-enable-switch.png)

## Användning efter aktivering

Efter aktivering kommer medarbetaren att visas vid tillgängliga ingångspunkter (såsom snabbåtkomsten i det nedre högra hörnet eller specifika scenariogångar).
