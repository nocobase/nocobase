# Webbsökning

Webbsökning används för att komplettera modellens träningsdata med den senaste informationen.

## Hur det fungerar

Tillgängligheten för webbsökning beror på om den valda modelltjänsten i den aktuella sessionen stöder Web Search.

- **Stöds:** En brytare för webbsökning visas och kan aktiveras eller inaktiveras efter behov.
- **Stöds ej:** Brytaren visas inte och sökstatusen inaktiveras automatiskt.

## Användning i konversationer

Använd brytaren för webbsökning i inmatningsfältet:

- **På:** AI kan först söka efter information och sedan svara.

![web-search-switch-visible-on-supported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-visible-on-supported-model.png)

- **Av:** AI svarar endast baserat på befintlig kontext.

![web-search-switch-hidden-on-unsupported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-hidden-on-unsupported-model.png)

## Skillnader mellan plattformar

Olika modeller har olika stöd för Web Search. Vänligen använd funktionen baserat på den faktiska situationen för er valda modelltjänst och modell.
