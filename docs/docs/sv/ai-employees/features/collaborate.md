# Samarbeta med AI-medarbetare

Efter att ni har skapat och aktiverat AI-medarbetare kan ni samarbeta med dem på sidorna.

## Ingångar

Det finns två vanliga ingångar:

1. **Huvudingång i nedre högra hörnet**: Öppna AI-chattpanelen från det nedre högra hörnet på affärssidor. Lämplig för allmänna frågor och svar samt samarbete mellan olika block.
2. **Ingång via blockåtgärder (Actions)**: I block som stöder `Actions`, gå in via `Actions -> AI employees`. Lämplig för uppgifter som rör det aktuella blocket (till exempel JSBlock-scenarier).

## Grundläggande chattfunktioner

Chattpanelen stöder vanliga åtgärder som att skicka meddelanden, ladda upp bilagor, visa konversationshistorik, skapa nya chattar och redigera systemmeddelanden (prompts).

## Växling i chatten

Längst ner i inmatningsfältet kan ni växla direkt mellan:

- `AI Employee Switcher`: Växla aktuell medarbetare.
- `Model Switcher`: Växla modell för den aktuella sessionen.

Modellinställningar sparas per medarbetare och återställs nästa gång ni går in.
