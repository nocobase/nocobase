# Lägg till kontext - Block

När ni chattar med en AI-medarbetare kan ni, förutom att skicka filer, även skicka kontextinformation från applikationen till AI-medarbetaren. Detta gör att AI-medarbetaren kan svara baserat på applikationens kontext.

### Skicka blockinformation

I AI-medarbetarens chattfönster, klicka på knappen `Lägg till kontext` i det nedre vänstra hörnet och välj `Välj block`. Applikationssidan kommer då att gå in i ett läge för blockval.

![20251022010148](https://static-docs.nocobase.com/20251022010148.png)

När ni håller muspekaren över ett block kommer färgen på de block som kan skickas som kontext att ändras.

![20251022010422](https://static-docs.nocobase.com/20251022010422.png)

När ni har valt det block som ska skickas kommer blockets kontextinformation att bifogas i chattfönstret.

![20251022010543](https://static-docs.nocobase.com/20251022010543.png)

När ni har skrivit in er fråga klickar ni på skicka-knappen, precis som vid ett vanligt meddelande, så kommer AI-medarbetaren att svara baserat på blockinformationen.

Blockinformationen kommer även att visas i meddelandelistan.

![20251022010758](https://static-docs.nocobase.com/20251022010758.png)

### Avsluta blockval

Om applikationssidan är i läget för blockval men ni inte vill välja något block, klicka på "Stopp"-ikonen längst ner för att avsluta läget.

![20251022011043](https://static-docs.nocobase.com/20251022011043.png)
