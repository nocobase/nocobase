# Filhantering

## Introduktion

Här konfigurerar ni lagringsmetoden för filer som laddas upp under konversationer med AI-medarbetare.

## Inställningar

Gå till konfigurationssidan för AI-medarbetar-pluginen och klicka på fliken `Settings` för att komma till sidan för lagringsinställningar.

![20251022002648](https://static-docs.nocobase.com/20251022002648.png)

De tillgängliga lagringsalternativen för `Files storage` konfigureras i pluginen för filhantering.
