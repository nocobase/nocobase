# Avancerat

## Introduktion

När ni samtalar med en AI-medarbetare kan ni, förutom att skicka filer, även skicka applikationskontextinformation till AI-medarbetaren. Detta gör att AI-medarbetaren kan svara baserat på den informationen.

### Skicka blockinformation

I AI-medarbetarens dialogruta klickar ni på knappen `Add work context` i det nedre vänstra hörnet och väljer `Pick block`. Applikationssidan går då in i blockvals-läge.

![20251022010148](https://static-docs.nocobase.com/20251022010148.png)

När ni håller muspekaren över ett block ändras färgen på de block som kan skickas som kontext.

![20251022010422](https://static-docs.nocobase.com/20251022010422.png)

När ni har valt det block ni vill skicka, kommer blockets kontextinformation att bifogas i dialogrutan.

![20251022010543](https://static-docs.nocobase.com/20251022010543.png)

Efter att ni har skrivit in er fråga, klickar ni på skicka-knappen precis som när ni skickar ett vanligt meddelande, så kommer AI-medarbetaren att svara baserat på blockinformationen.

Blockinformationen kommer också att visas i meddelandelistan.

![20251022010758](https://static-docs.nocobase.com/20251022010758.png)

### Avsluta blockval

Om applikationssidan är i blockvals-läge men ni inte vill välja något block, kan ni klicka på "Stopp"-ikonen längst ner för att avsluta blockvals-läget.

![20251022011043](https://static-docs.nocobase.com/20251022011043.png)
