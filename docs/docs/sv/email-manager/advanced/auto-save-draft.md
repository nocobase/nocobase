# Autospara utkast

Under knappen **Spara utkast** kan ni aktivera **Autospara utkast**.

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_29_PM.png)

Under e-postredigeringen sparar systemet utkastet i realtid, vilket säkerställer att inget innehåll går förlorat.

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_31_PM.png)

När ni öppnar e-postredigeraren igen (för samma avsändare och mottagare) frågar systemet om ni vill använda det sparade utkastet.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_08_35_PM.png)
