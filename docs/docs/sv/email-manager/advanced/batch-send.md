# Massutskick

## Massutskick – översikt
Ni kan skicka e-postmeddelanden till flera personer som massutskick (varje mottagare får ett separat meddelande).
![](https://static-docs.nocobase.com/email-manager/Loading--12-31-2025_10_15_PM.png)

## Massutskick via tabell
Ni kan också skicka e-post som massutskick via en tabell:

1. Lägg till en knapp för att skicka e-post i tabellen
   ![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_04_PM.png)

2. Konfigurera e-postfältet i tabellen (används för att tolka e-postadresser efter att ni har valt tabellrader)
   ![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_17_PM%20(1)

3. Markera tabelldata för att skicka e-post
   ![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_19_PM.png)

Om ni behöver skicka e-post till alla adresser i hela tabellen kan ni aktivera funktionen **Välj alla poster i den aktuella tabellen som mottagare som standard**:
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_18_PM%20(1)

Efter att ni har redigerat e-postinnehållet klickar ni på knappen för massutskick för att skicka:
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_19_PM.png)
