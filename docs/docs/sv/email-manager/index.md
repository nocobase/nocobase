# E-posthantering

## Introduktion
Pluginet för e-posthantering är ett effektivt och smidigt verktyg som stöder auktorisering och anslutning för Gmail- och Outlook-brevlådor. Det hjälper användare att integrera funktioner för e-posthantering, såsom att skicka och ta emot e-post, i olika block och sidor. Genom enkel auktoriseringskonfiguration kan ni uppnå en enhetlig hantering av flera konton och få en sömlös e-postkommunikationsupplevelse.
