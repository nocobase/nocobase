# E-postsignatur

## Konfigurera signaturer

Gå till E-posthanteringscentret och klicka på inställningsknappen:
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_08_PM.png)

Växla till signaturpanelen:
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_08_PM%20(1)

Här kan ni hantera signaturer. Markera alternativet framför en signatur för att ställa in den som standardsignatur:
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_09_PM.png)

## Använda signaturer

I meddelandetexten klickar ni på signaturikonen för att välja önskad signatur:
![](https://static-docs.nocobase.com/email-manager/Loading--12-31-2025_10_11_PM.png)
