# E-postsignatur

## Konfigurera signaturer

Gå till E-posthanteringscentret och klicka på inställningsknappen:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_05_34_PM.png)

Växla till signaturpanelen:
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_05_51_PM.png)

Här kan ni hantera signaturer. Markera alternativet framför en signatur för att ställa in den som standardsignatur:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_05_37_PM.png)

## Använda signaturer

I meddelandetexten klickar ni på signaturikonen för att välja önskad signatur:
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_05_40_PM.png)
