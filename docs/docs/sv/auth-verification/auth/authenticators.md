# Autentiserare

## Hantera användarautentisering

När pluginet för användarautentisering installeras, initieras en autentiseringsmetod baserad på `lösenord`, som använder användarens användarnamn och e-postadress.

![](https://static-docs.nocobase.com/66eaa9d5421c9cb713b117366bd8a5d5.png)

## Aktivera autentiserare

![](https://static-docs.nocobase.com/7f1fb8f8ca5de67ffc68eff0a65848f5.png)

Endast aktiverade autentiseringstyper visas på inloggningssidan.

![](https://static-docs.nocobase.com/8375a36ef98417af0f0977f1e07345dd.png)

## Användarautentiseringstyper

![](https://static-docs.nocobase.com/da4250c0cea343ebe470cbf7be4b12e4.png)

Genom att lägga till olika typer av autentiserare kan ni aktivera motsvarande autentiseringsmetoder i systemet.

Utöver de autentiseringstyper som befintliga plugin tillhandahåller, kan utvecklare också utöka användarautentiseringstyperna. För mer information, se [Utvecklarguide](./dev/).
