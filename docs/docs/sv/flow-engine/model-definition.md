# Fält för Kina-region

## Introduktion

Pluginet för Kina-regionfältet (`field-china-region`) lägger till fälttypen "Kina-region" i NocoBase.

## Installation

I NocoBases huvudgränssnitt navigerar ni till "Pluginhanteraren", hittar pluginet "Kina-region" och aktiverar det.

## Användning

I samlingskonfigurationen lägger ni till ett nytt fält och väljer "Kina-region" som fälttyp.

![](/images/field-china-region-1.png)

### Fältkonfiguration

I fältkonfigurationen kan ni ställa in visningsläget för fältet:

- Provins
- Provins/Stad
- Provins/Stad/Distrikt
- Provins/Stad/Distrikt/Län

![](/images/field-china-region-2.png)
