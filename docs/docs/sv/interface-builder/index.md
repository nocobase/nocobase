# Gränssnittsbyggare

## WYSIWYG

NocoBase använder sig av en WYSIWYG-modell (What You See Is What You Get) för att bygga gränssnitt. Klicka på knappen för gränssnittsbyggaren för att växla mellan konfigurationsläge och visningsläge.

## Layoutmall

NocoBase levereras med en standardlayoutmall där navigering finns upptill och till vänster, och ett innehållsområde till höger.

![未命名.002](https://static-docs.nocobase.com/未命名.002.jpeg)

## Konfigurationsalternativ

När du går in i gränssnittsbyggarläget visas orange markeringar för konfigurerbara element på gränssnittet. Vanligtvis finns ingången till ett elements inställningar i dess övre högra hörn.
