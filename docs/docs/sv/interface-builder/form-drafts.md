# Formulärutkast

När formulärutkast är aktiverade sparas innehållet ni fyller i automatiskt lokalt. Även om ni råkar stänga formuläret eller uppdatera sidan kommer den ifyllda datan inte att gå förlorad.

![20251213093717](https://static-docs.nocobase.com/20251213093717.png)

När ett utkast finns visas ett meddelande högst upp i formuläret. Utkastet raderas automatiskt efter en lyckad inskickning; om det inte längre behövs kan ni även radera det manuellt när som helst.

![20251213092734](https://static-docs.nocobase.com/20251213092734.png)

::: tip Tips

För närvarande stöder formulärutkast endast lokal cachelagring. Framtida versioner kommer att stödja cachelagring på serversidan för att möjliggöra synkronisering mellan flera enheter.

:::
