# Fält

I användargränssnittet är ett fält en fältkomponent som fungerar som en bärare för enhetsdata. Olika typer av data presenteras med olika fältkomponenter. Fält kan endast kopplas till block och kan inte användas fristående.

## Fält i ett block

Fält används vanligtvis inte fristående, utan existerar som underordnade element till datatypsblock. Datatypsblock har oftast ett alternativ för att "Konfigurera fält", och listan över fält tillhandahålls av den aktuella samlingen.

![20251029204240](https://static-docs.nocobase.com/20251029204240.png)

## Fältets designer (verktygsfält)

Precis som block har även fältkomponentens övre högra hörn tre ikoner, nämligen:

- Dra för att justera layout
- [Händelseflöde](/interface-builder/event-flow)
- Parameterkonfiguration

![20251025165432](https://static-docs.nocobase.com/20251025165432.png)
