# Visa etikett

## Introduktion

Fält i ett block visar som standard fältetiketter. Dessa etiketter identifierar varje fälts betydelse och gör det enklare för dig att läsa och förstå datainnehållet.

I vissa scenarier (till exempel på sidor som endast visar information, eller när fältens betydelse redan är mycket tydlig) kan ni välja att dölja etiketterna för ett renare gränssnitt. När etiketterna har dolts visas endast fältvärdena, och fältnamnen visas inte längre.
