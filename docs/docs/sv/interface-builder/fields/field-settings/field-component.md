# Fältkomponent

## Introduktion

Fältkomponenter används för att visa innehållet i ett fält.
Vissa fält har flera inbyggda visningskomponenter. Ni kan välja lämplig komponent utifrån era faktiska behov för att presentera fältets värde på olika sätt.

Exempel: I visningsläge kan ett URL-fält växlas till en förhandsgranskningskomponent eller en URL-komponent (standard).
