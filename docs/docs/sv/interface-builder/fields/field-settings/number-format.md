# Numerisk formatering

## Introduktion

Numerisk formatering innebär visuell förbättring och standardisering av numeriska fält (inklusive formelfält) i läsläge. Detta inkluderar inställningar för tusentalsavgränsare, decimalprecision, enhetskonvertering och vetenskaplig notation. Genom att konfigurera numerisk formatering kan ni anpassa hur siffror presenteras för att möta de specifika uttryckssätten i olika regioner och branscher.

![20251028215021](https://static-docs.nocobase.com/20251028215021.png)

![20251028215059](https://static-docs.nocobase.com/20251028215059.png)

## Så här använder ni

- Stöder enkel enhetskonvertering (multiplikation/division)
- Tusentalsavgränsare
- Prefix och suffix
- Precision
- Vetenskaplig notation.

Exempel: Formatera ett orderbelopp genom att lägga till en valutasymbol (t.ex. ¥), en tusentalsavgränsare (som komma eller punkt) och decimalprecision för att göra beloppet tydligt och lättläst.

![20251028215249](https://static-docs.nocobase.com/20251028215249.gif)

- Använd vetenskaplig notation.
