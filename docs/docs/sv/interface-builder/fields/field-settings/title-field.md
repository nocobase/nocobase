# Titelfält

## Introduktion

**Titelfältet** används vanligtvis i relationsfältkomponenter som en identifierare för den relaterade data som visas i gränssnittet. Det avgör hur huvudinformationen för datan presenteras, och används oftast för att hjälpa användare att snabbt identifiera och förstå datan.

Olika typer av titelfält motsvarar olika fältkomponenter och stöder tillhörande konfigurationsalternativ. Till exempel:

- När titelfältet är ett **datumfält** använder fältkomponenten en **datumkomponent** för att visa datan.
- När titelfältet är ett **textfält** använder fältkomponenten en **textkomponent** för att visa datan.
- När titelfältet är ett **alternativfält** använder fältkomponenten en **alternativkomponent** för att visa datan.
