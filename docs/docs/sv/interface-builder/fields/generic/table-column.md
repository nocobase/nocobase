# Tabellfält

## Introduktion

Förutom att justera kolumnbredd, fälttitel och sortering med mera, erbjuder tabellfält även mer anpassade visningskonfigurationer för specifika fält (som datumfält, relationsfält och numeriska fält).

![20251024174558](https://static-docs.nocobase.com/20251024174558.png)

## Fältkonfigurationsalternativ

### Formatering av datumfält

![20251024175303](https://static-docs.nocobase.com/20251024175303.png)

För mer information, se [Datumformatering](/interface-builder/fields/specific/date-picker)

### Formatering av numeriska fält

Stöder enkel enhetskonvertering, tusentalsavgränsare, prefix och suffix, precision samt vetenskaplig notation.

![20251024175445](https://static-docs.nocobase.com/20251024175445.png)

För mer information, se [Numerisk formatering](/interface-builder/fields/field-settings/number-format)

### Aktivera snabbredigering

När snabbredigering är aktiverad visas en redigeringsknapp när muspekaren förs över kolumnen. Klicka på den för att snabbt redigera och spara data i ett popup-fönster.

![20251025171158](https://static-docs.nocobase.com/20251025171158.gif)

### Aktivera klicka för att öppna

Förutom att relationsfält stöder att klicka för att öppna ett popup-fönster, kan även vanliga fält aktiveras med "Klicka för att öppna" för att fungera som en ingång till ett popup-fönster.

![20251025172308](https://static-docs.nocobase.com/20251025172308.gif)

### Visningsläge för innehållsöverflöd

När kolumninnehållet överskrider tabellkolumnens bredd kan du ställa in hur överflödet ska hanteras:

- Visa med ellips (standard)
- Radbrytning

![20251025172549](https://static-docs.nocobase.com/20251025172549.png)

### Fasta kolumner

![20251025170858](https://static-docs.nocobase.com/20251025170858.gif)

### Fältkomponent

Vissa fält stöder flera visningsformat. Användare kan växla fältkomponenter för att uppnå olika visningseffekter och möta behoven i olika scenarier. Till exempel kan ett fält av typen **URL** växlas till komponenten **Preview** för att bättre visa länkinnehåll eller bildförhandsvisningar.

![20251025171658](https://static-docs.nocobase.com/20251025171658.png)

### Visningsstil

- Etikett
- Text

![20251025172723](https://static-docs.nocobase.com/20251025172723.png)

### Sorterbar

De flesta fälttyper stöder för närvarande sortering på serversidan. När sortering är aktiverad kan data sorteras i stigande eller fallande ordning baserat på det valda fältet.
