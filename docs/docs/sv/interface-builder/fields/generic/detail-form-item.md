# Detaljfält

## Introduktion

Fältkonfigurationerna för detalj-, lista- och rutnätsblock är i princip likadana. De styr främst hur fälten visas när de är i läsläge.

![20251025172851](https://static-docs.nocobase.com/20251025172851.png)

## Konfigurationsalternativ för fält

### Formatering av datumfält

![20251025173005](https://static-docs.nocobase.com/20251025173005.png)

För mer information, se [Datumformatering](/interface-builder/fields/specific/date-picker)

### Formatering av numeriska fält

![20251025173242](https://static-docs.nocobase.com/20251025173242.png)

Stöder enkel enhetskonvertering, tusentalsavgränsare, prefix och suffix, precision och vetenskaplig notation.

För mer information, se [Nummerformatering](/interface-builder/fields/field-settings/number-format)

### Aktivera klicka för att öppna

Förutom att relationsfält stöder att öppna en popup vid klick, kan även vanliga fält konfigureras för att öppna en popup som en ingång. Du kan också ställa in hur popupen öppnas (låda, dialogruta eller undersida).

![20251025173549](https://static-docs.nocobase.com/20251025173549.gif)

### Visningsläge för innehållsöverflöd

När fältinnehållet överskrider sin bredd kan du ställa in hur överflödet ska hanteras.

- Visa med ellips (standard)
- Radbrytning

![20251025173917](https://static-docs.nocobase.com/20251025173917.png)

### Fältkomponent

Vissa fält stöder flera visningsformat, vilket kan uppnås genom att växla fältkomponent.

Till exempel: `URL`-komponenten kan växlas till `Preview`-komponenten.

![20251025174042](https://static-docs.nocobase.com/20251025174042.png)

Till exempel kan relationsfält visas på olika sätt genom att växla från rubrikfältkomponenten till `Underdetaljer` för att visa mer innehåll från relationen.

![20251025174311](https://static-docs.nocobase.com/20251025174311.gif)
