# Formulärfält

## Introduktion

Olika typer av fält i ett formulär har olika konfigurationsalternativ. Du kan utöka fältens konfigurationsalternativ via **plugin** för att möta mer anpassade behov.

![20251025174555](https://static-docs.nocobase.com/20251025174555.png)

## Fältkonfigurationsalternativ

### Standardvärde

Du kan konfigurera konstanter/variabler som standardvärden för fält (stöds endast för nya formulär).

![20251025174724](https://static-docs.nocobase.com/20251025174724.png)

För mer information, se [Ställ in standardvärde](/interface-builder/fields/field-settings/default-value)
