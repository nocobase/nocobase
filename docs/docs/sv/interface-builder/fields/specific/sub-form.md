# Underformulär

## Introduktion

Underformulär är lämpliga när ni först skapar ny associerad data och sedan länkar den. Data med flera nivåer av associationer visas tydligt som ett kapslat formulär. Jämfört med komponenter som dataväljaren och rullgardinsväljaren, kan ni med ett underformulär direkt hantera fälten för den associerade samlingen inom det aktuella sidblocket. Den associerade datan skickas dessutom in tillsammans med huvudformuläret.
