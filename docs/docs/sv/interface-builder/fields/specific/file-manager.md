# Filhanterare

## Introduktion

Filhanteraren är en relationsfältkomponent som används för att hantera relationer där målsamlingen är filsammlingen.

![20251027194124](https://static-docs.nocobase.com/20251027194124.png)

## Fältkonfiguration

- [Fältkomponent](/interface-builder/fields/association-field): Växla till andra relationsfältkomponenter, som till exempel rullgardinsmenyer, dataväljare med mera.
- Tillåt val av befintliga filer: Aktiverad som standard, vilket gör det möjligt att välja och koppla från uppladdade filer.
