# Datum och tid

## Introduktion

Datumfältet stöder anpassade datum- och tidsformat.

## Var ni kan ställa in datumformatet

### Samlingsfält

![20240417114311](https://static-docs.nocobase.com/20240417114311.png)

> Datumformatet som ställs in i samlingsfältet kommer att fungera som fältets standardformat. Det kan åsidosättas av datumformatet i fältinställningarna inom blockkonfigurationen.

### Fältinställningar

![20251027191824](https://static-docs.nocobase.com/20251027191824.png)

Utöver de olika förinställda datum- och tidsformaten stöds även anpassade format för att tillgodose individuella behov.
