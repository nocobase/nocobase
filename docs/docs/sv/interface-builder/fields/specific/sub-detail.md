# Underdetaljer

## Introduktion

Underdetaljer är komponenten som motsvarar underformuläret i läsläge. Jämfört med komponenterna Etikett och Rubrik kan Underdetaljer inte bara visa mer data från den aktuella samlingen, utan kan också konfigureras för att visa data från relaterade samlingar. Data med relationer på flera nivåer visas tydligt i form av nästlade detaljer.
