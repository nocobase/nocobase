# Kaskaderande val

## Introduktion

Kaskadväljaren är utformad för relationsfält där målsamlingen är en trädtabell. Den låter dig välja data enligt trädtabellens hierarkiska struktur och stöder fritextsökning för snabb filtrering.

## Användningsinstruktioner

- För **en-till-en-relationer** är kaskadväljaren ett **enkelval**.

![20251125214656](https://static-docs.nocobase.com/20251125214656.png)

- För **en-till-många-relationer** är kaskadväljaren ett **flerval**.

![20251125215318](https://static-docs.nocobase.com/20251125215318.png)
