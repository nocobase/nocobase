# Dataväljare

## Introduktion

Dataväljaren, som presenteras som en tabellväljare i ett popup-fönster, gör det möjligt att välja befintlig data från målsamlingen för att koppla den, eller att lägga till data till målsamlingen och sedan koppla den.

![20251027203343](https://static-docs.nocobase.com/20251027203343.png)

Vald data visas inte i tabellväljaren.

![20251027203717](https://static-docs.nocobase.com/20251027203717.gif)

I tabellväljaren kan ni även hantera målsamlingen för koppling ytterligare (till exempel lägga till, ta bort, importera/exportera).

![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
