# Titel

## Introduktion

I läsläge visas associeringsfält som standard med hjälp av **Titelkomponenten**, vilken identifierar den aktuella associerade posten genom att visa värdet från titelfältet. Systemet matchar och renderar automatiskt den lämpliga titelkomponenten baserat på titelfältets typ.
