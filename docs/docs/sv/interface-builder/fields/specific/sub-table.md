# Undertabell

## Introduktion

Undertabellen är utformad för att hantera relationsfält där en post kan vara kopplad till flera andra. Den gör det möjligt att masskapa och koppla ny data i målsamlingen, eller att välja och koppla befintlig data.

## Användningsinstruktioner

![20251027223350](https://static-docs.nocobase.com/20251027223350.png)

I undertabellen visas olika fältkomponenter beroende på fälttyp. Stora fält (som Rich Text, JSON, flerradstext) redigeras via en flytande popup-ruta.

![20251027223426](https://static-docs.nocobase.com/20251027223426.png)

Relationsfält i undertabellen.

Beställningar (En-till-Många) > Beställningsprodukter (En-till-En) > Affärsmöjlighet

![20251027223530](https://static-docs.nocobase.com/20251027223530.png)

Standardkomponenten för ett relationsfält är en rullgardinslista (stöder rullgardinslista/dataväljare).

![20251027223754](https://static-docs.nocobase.com/20251027223754.png)

## Fältkonfigurationsalternativ

### Tillåt att välja befintlig data (inaktiverad som standard)

Stöder att välja och koppla befintlig data.
![20251027224008](https://static-docs.nocobase.com/20251027224008.png)

![20251027224023](https://static-docs.nocobase.com/20251027224023.gif)

### Fältkomponent

[Fältkomponent](/interface-builder/fields/association-field): Växla till andra relationsfältkomponenter, som till exempel rullgardinslista, dataväljare, med mera.

### Tillåt att koppla bort befintlig data

> Anger om det ska vara tillåtet att koppla bort befintlig data för relationsfältet i redigeringsformuläret.
