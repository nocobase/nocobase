# Undertabell (radredigering)

## Introduktion

Undertabellen är utformad för att hantera Association-fält där en post kan vara kopplad till flera andra. Den gör det möjligt att masskapa och koppla ny data i målsamlingen, eller att välja och koppla befintlig data.

## Användningsinstruktioner

![20251027223350](https://static-docs.nocobase.com/20251027223350.png)

I undertabellen visas olika fältkomponenter beroende på fälttyp. Stora fält (som Rich Text, JSON, Long Text) redigeras via en flytande popup-ruta.

![20251027223426](https://static-docs.nocobase.com/20251027223426.png)

Association-fält i undertabellen.

Beställningar (One-to-Many (O2M)) > Beställningsprodukter (One-to-One (O2O)) > Affärsmöjlighet

![20251027223530](https://static-docs.nocobase.com/20251027223530.png)

Standardkomponenten för ett Association-fält är en Single select (stöder Single select/Data selector).

![20251027223754](https://static-docs.nocobase.com/20251027223754.png)
