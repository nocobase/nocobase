# Rutnätskort

## Introduktion

Rutnätskort-blocket visar sammanfattande information om dataregister i kortformat. Det stöder konfiguration av antalet kolumner för olika skärmstorlekar för att säkerställa en användarvänlig visning på olika enheter.
