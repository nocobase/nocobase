# Diagram

## Introduktion

NocoBases datavisualiserings-plugin erbjuder visuell datafråga och ett brett utbud av diagramkomponenter. Genom enkel konfiguration kan ni snabbt bygga visualiseringspaneler för att visa datainsikter och stödja flerdimensionell dataanalys och presentation.
