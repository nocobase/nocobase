# Block

Block är bärare av data och innehåll. De kan placeras på en sida (Page), i en dialogruta (Modal) eller i en sidopanel (Drawer), och flera block kan fritt dras och arrangeras.

## Blocktyper

![Block Types](https://static-docs.nocobase.com/f71af45b5cd914ea0558f760ddbbba58.png)

- Datablock: Används för att visa data från en datakälla på gränssnittet.
- Filterblock: Används för att hämta data från en datakälla som filtervillkor för att filtrera andra datablock.
- Övriga block: Används för att bära specifikt eller oberoende innehåll som arbetsflödesuppgifter, granskningsloggar, Markdown, med mera.

## Lägg till block

Block kan placeras på en sida (Page), i en dialogruta (Modal) eller i en sidopanel (Drawer).

### Block på en sida

För närvarande inkluderar blocktyperna på en sida: Datablock, Filterblock och Övriga block.

![20251023222441](https://static-docs.nocobase.com/20251023222441.png)

### Block i en popup (dialogruta eller sidopanel)

Popups finns i två typer: dialogrutor och sidopaneler. Precis som på sidor kan ni också lägga till block. Skillnaden är att formulärblock i popups vanligtvis används för att lägga till, redigera eller visa en enskild post. Blocktyperna inkluderar Datablock och Övriga block.

![20251023222613](https://static-docs.nocobase.com/20251023222613.png)

## Blockdesignern

Varje block har tre små ikoner i det övre högra hörnet. Från vänster till höger är de:

1. Dra och släpp-layout
2. [Händelseflöde](/interface-builder/event-flow)
3. Blockinställningar

![20251023224032](https://static-docs.nocobase.com/20251023224032.png)

För enkla block är alla konfigurationsalternativ samlade under "Blockinställningar", till exempel JS-blocket.

![20251023224903](https://static-docs.nocobase.com/20251023224903.png)

Komplexa datablock erbjuder också separata, inbäddade "Konfigurera fält" och "Konfigurera åtgärder".

![20251023225141](https://static-docs.nocobase.com/20251023225141.png)

Utöver detta kan ni fritt utforska fler kapslingsmöjligheter, som till exempel i diagramblocket.

![](https://static-docs.nocobase.com/07588190b3f41ae3060e71d8b76b4447.png)
