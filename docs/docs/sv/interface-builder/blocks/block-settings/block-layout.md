# Layout

## Introduktion

Formulärliknande block, som formulärblock, detaljblock och listblock, stöder layoutinställningar. Layoutalternativen är vertikal layout (etiketter och inmatningsfält arrangeras rad för rad) och horisontell layout (etiketter och inmatningsfält placeras sida vid sida). Ni kan justera blockfältens layout efter era specifika behov för att säkerställa att informationsinmatning eller visning är både tydlig och effektiv.

### Vertikal (standard)

Standardlayouten är vertikal.

![20251024120143](https://static-docs.nocobase.com/20251024120143.png)

### Horisontell

Den horisontella layouten visar etiketter och inmatningsfält sida vid sida, vilket effektivt sparar utrymme. För fält med långa titlar kan ni välja att låta dem radbrytas eller visas med ellips för att säkerställa ett rent och estetiskt tilltalande gränssnitt.

![20251024120237](https://static-docs.nocobase.com/20251024120237.png)

#### Fälttiteljustering

- **Vänsterjusterad** (standard)
- **Högerjusterad**

#### Fälttitelbredd

- Standardbredden är **120px**, med en minimibredd på **50px**.

#### Hantering av fälttitlar som överskrider bredden

- **Radbryt**
- **Ellips**

#### Visa kolon

Ni kan välja om ett kolon ska visas efter fälttiteln.
