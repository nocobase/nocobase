# Filter

## Introduktion

Filterfunktionen visas vanligtvis i datablock för att filtrera data genom att konfigurera olika villkor. Observera att om ett datablock redan har konfigurerats med ett dataomfång, kommer filterfunktionen att kombineras med dataomfångsvillkoren som det slutgiltiga filtreringsvillkoret.

## Åtgärdsinställningar

![20251029173708](https://static-docs.nocobase.com/20251029173708.png)

### Filtrerbara fält

Som standard är alla fält i denna samling aktiverade. Endast aktiverade fält kan väljas i listan över villkorsfält.

Fält från relaterade samlingar kan också användas som filterfält.
