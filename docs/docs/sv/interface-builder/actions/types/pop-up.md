# Öppna modal

## Introduktion

Åtgärden **Öppna modal** liknar åtgärderna Visa och Redigera. Du kan konfigurera en **Öppna modal**-knapp i rader eller tabellhuvudet i en tabell. När du klickar på knappen öppnas en motsvarande modal, och blocken i denna modal visar data baserat på den aktuella kontextposten.

Konfigurera block i modalen vid behov.
