# Redigera

## Introduktion

Redigera-åtgärden är en radknapp som har den aktuella posten som kontext. Den presenteras som en popup-ruta där ni kan anpassa och konfigurera block.

![20251029165442](https://static-docs.nocobase.com/20251029165442.png)

Konfigurera block efter behov.

![20251029165514](https://static-docs.nocobase.com/20251029165514.png)
