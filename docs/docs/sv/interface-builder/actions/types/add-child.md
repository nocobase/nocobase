# Lägg till underordnad post

## Introduktion

Denna åtgärd är speciellt utformad för **trädtabellsblock** för att snabbt lägga till underordnade poster. När ni lägger till en underordnad post behandlas den markerade raden som en föräldranod, vilket gör att ni enkelt kan skapa underordnade poster under vilken förälder som helst. Denna funktion förenklar hanteringen av hierarkisk data avsevärt, och gör det mer effektivt och intuitivt att utöka och organisera datastrukturer.
