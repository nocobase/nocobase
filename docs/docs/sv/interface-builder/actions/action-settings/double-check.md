# Bekräftelse

## Introduktion

Bekräftelseinställningen används för att säkerställa att en åtgärd verkligen ska utföras. När ni klickar på en åtgärd visas en bekräftelseruta. Radera-knappen har bekräftelse aktiverad som standard för att förhindra dataförlust eller oåterkalleliga resultat på grund av oavsiktliga åtgärder.

![20251029125021](https://static-docs.nocobase.com/20251029125021.png)

Ni kan anpassa bekräftelsens innehåll och titel, och det går även att hämta variabler.
