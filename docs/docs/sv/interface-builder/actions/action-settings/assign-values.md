# Fältilldelning

## Introduktion

Fältilldelning innebär att du förinställer värdet på ett fält innan en åtgärd utförs. När du uppdaterar eller skapar en post fylls dessa data i automatiskt, utan att du behöver fylla i dem manuellt. Exempel på scenarier är massuppdatering av orderstatusar eller att ställa in leveranstiden till aktuell tid när en order levereras.
