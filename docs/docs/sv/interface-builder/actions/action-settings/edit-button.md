# Redigera knapp

## Introduktion

Systemet stöder anpassning av knappens namn, ikon, bakgrundsfärg och knapptyp, och erbjuder flexibla anpassningsalternativ för användargränssnittet för att hjälpa er att anpassa knappens utseende och funktionalitet efter era specifika behov.

- **Knappnamn**: Ni kan anpassa knappens text utifrån verksamhetens behov för att säkerställa att knappens etikett är tydlig och intuitiv.
- **Knappikon**: Det går att ställa in en ikon för knappen för att förbättra den visuella igenkänningen och göra det lättare för er att snabbt identifiera knappens funktion.
- **Farlig åtgärd**: När en knapp representerar en **farlig åtgärd** (som till exempel radera, återställa, etc.) kan den ställas in som typen **Danger Action**.
- **Knapptyp**: Knappar stöder olika typinställningar. Vanliga knapptyper inkluderar:
  - **Standard (Default)**: En standardknapp, lämplig för vanliga åtgärder.
  - **Primär (Primary)**: Används för viktiga eller primära åtgärder, vanligtvis med en framträdande visuell effekt.
  - **Streckad (Dashed)**: En knapp med streckad kantlinje, lämplig för sekundära åtgärder eller tips-liknande knappar.
  - **Text (Text)**: En knapp med endast text och ingen bakgrundsfärg, vanligtvis använd för lättviktiga åtgärder eller länkar.
