# Språkinställningar

I "Systeminställningar" kan ni justera systemets globala språkinställningar genom att konfigurera "Aktiverade språk". Det första språket i listan kommer att vara standardspråket.

![20240324192005](https://static-docs.nocobase.com/20240324192005.png)

När flera språk är aktiverade kan användare ställa in sina personliga språkpreferenser i Personligt center (uppe till höger på sidan).

![20240324192509](https://static-docs.nocobase.com/20240324192509.png)

Om den språkinställning ni använder saknar översättningar kan ni använda pluginet "[Lokaliseringshantering](/system-management/localization/)" för att lägga till eller redigera översättningar.

![20240324193055](https://static-docs.nocobase.com/20240324193055.png)
