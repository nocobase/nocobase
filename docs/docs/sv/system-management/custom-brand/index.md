# Anpassa varumärke

## Introduktion

Systemets standardvarumärke är "NocoBase", vilket huvudsakligen visas på följande tre platser:

![20240409111100](https://static-docs.nocobase.com/20240409111100.png)

Längst ner på inloggningssidan:

![20240409111254](https://static-docs.nocobase.com/20240409111254.png)

Position 1 kan konfigureras via alternativet "Logotyp" i [Systeminställningar](/system-management/system-settings/). Position 2 och 3 kan konfigureras med hjälp av denna **plugin**.

## Användningsinstruktioner

När **pluginen** har aktiverats klickar ni på menyn "Anpassa varumärke" i sidans övre högra hörn för att komma till konfigurationssidan.

![20240409113640](https://static-docs.nocobase.com/20240409113640.png)

- **Varumärke**: Används för att ställa in texten längst ner på inloggningssidan (position 3).
- **Om**: Används för att konfigurera innehållet i menyn i det övre högra hörnet (som visas i bilden nedan).

![20240409113915](https://static-docs.nocobase.com/20240409113915.png)

Konfigurationen stöder HTML-format för flexibel anpassning av innehåll. Om ni bara behöver göra enkla textändringar kan ni direkt ändra det befintliga innehållet, som visas nedan:

![20240409114740](https://static-docs.nocobase.com/20240409114740.png)
