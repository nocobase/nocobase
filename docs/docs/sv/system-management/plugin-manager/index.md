# Pluginhantering

## Pluginhanteraren

Klicka på ikonen för pluginhanteraren i sidans övre högra hörn för att komma till pluginhanterarens sida. På denna sida kan ni visa, installera, aktivera eller inaktivera plugin.

![2024-04-10_08-47-45](https://static-docs.nocobase.com/2024-04-10_08-47-45.png)

## Inställningscenter för plugin

Plugin har vanligtvis egna inställningssidor. Klicka på inställningsikonen i det övre högra hörnet för att se inställningsmenyerna för alla plugin. Klicka på motsvarande menyalternativ för att konfigurera pluginet.

![2024-04-10_08-52-41](https://static-docs.nocobase.com/2024-04-10_08-52-41.png)
