## BelongsToRepository

Dess gränssnitt är detsamma som [HasOneRepository](./has-one-repository.md).
`BelongsToRepository` är ett `Repository` för att hantera `BelongsTo`-relationer. Det tillhandahåller bekväma metoder för att hantera `BelongsTo`-relationer.
