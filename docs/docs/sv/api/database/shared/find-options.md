# Tomt läge

När det inte finns någon data i ett block, som till exempel en tabell, lista eller Kanban-tavla, visas ett tomt läge.

## Tabellblock

När ett tabellblock inte har någon data visas ett tomt läge.


![](https://static-docs.nocobase.com/22217d74110185125345706505436662.png)


## Listblock

När ett listblock inte har någon data visas ett tomt läge.


![](https://static-docs.nocobase.com/5688a25d292a835388046f50536d5951.png)


## Kanban-block

När ett Kanban-block inte har någon data visas ett tomt läge.


![](https://static-docs.nocobase.com/22217d74110185125345706505436662.png)


## Kalenderblock

När det inte finns någon data för ett specifikt datum visas ett tomt läge.


![](https://static-docs.nocobase.com/22217d74110185125345706505436662.png)


## Diagramblock

När ett diagramblock inte har någon data visas ett tomt läge.
