# ctx.i18n

i18n-instansen för den aktuella kontexten, som används för att läsa av eller byta språk. **Använd [ctx.t()](./t.md) för att översätta texter**; använd inte `ctx.i18n.t`.

## Användningsområden

Alla RunJS-exekveringsmiljöer kan använda `ctx.i18n` (t.ex. JSBlock, JSField, JSItem, JSColumn, arbetsflöde, länkregler, etc.).

## Typdefinition

```typescript
interface i18n: {
  language: string;
  changeLanguage(lng: string): Promise<any>;
}
```

## Vanliga egenskaper

| Egenskap | Typ | Beskrivning |
|------|------|------|
| `language` | `string` | Den för närvarande aktiva språkkoden (t.ex. `zh-CN`, `en-US`) |

## Vanliga metoder

### changeLanguage(lng)

Byter det aktuella språket.

| Parameter | Typ | Beskrivning |
|------|------|------|
| `lng` | `string` | Målspråkskod (t.ex. `'en-US'`, `'zh-CN'`) |

**Returvärde**: `Promise<any>`, som löses (resolve) när språkbytet är slutfört.

## Exempel

### Läsa av det aktuella språket

```ts
const lang = ctx.i18n.language;
// 'zh-CN' | 'en-US' | ...
if (lang.startsWith('zh')) {
  ctx.render(ctx.t('Kinesiskt gränssnitt'));
} else {
  ctx.render(ctx.t('Engelskt gränssnitt'));
}
```

### Byta språk

```ts
// Byt till engelska
await ctx.i18n.changeLanguage('en-US');

// Byt till kinesiska
await ctx.i18n.changeLanguage('zh-CN');
```

### Knapp för språkbyte

```ts
const { Button } = ctx.libs.antd;
const isZh = ctx.i18n.language.startsWith('zh');
ctx.render(
  <Button onClick={async () => {
    await ctx.i18n.changeLanguage(isZh ? 'en-US' : 'zh-CN');
  }}>
    {ctx.t(isZh ? 'Byt till engelska' : 'Byt till kinesiska')}
  </Button>,
);
```

## Observera

- **Översättningstext**: Använd `ctx.t()` konsekvent; använd inte `ctx.i18n.t`.
