# Användarlåsning

## Introduktion

Hantera låsta användare och lås användare proaktivt.

![](https://static-docs.nocobase.com/202412281450083.png)

### Låsa upp användare

Användare som har blivit låsta på grund av att de överskridit gränsen för ogiltiga lösenordsinloggningar kan låsas upp genom att radera posten.

### Låsa användare

Genom att lägga till en användarpost kan ni proaktivt låsa en användare. När en användare är låst kommer den att förbjudas från att komma åt systemet via någon autentiseringsmetod, inklusive API-nycklar.

![](https://static-docs.nocobase.com/202412281450512.png)
