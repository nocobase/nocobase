# Översikt

De inbyggda funktionerna i ett arbetsflöde kan inte täcka alla tänkbara scenarier. Till exempel kan de inbyggda nodtyperna inte omfatta varje enskild operation i alla affärsscenarier. Därför har vi också utformat en lösning för att utöka arbetsflödet, vilket inkluderar att utöka utlösare och nodtyper. I affärsscenarier där de inbyggda funktionerna inte räcker till kan ni lösa detta genom att utöka dem med hjälp av lågkodstekniker.

Utökningarna delas huvudsakligen in i två delar:

- [Utöka utlösartyper](./trigger.md)
- [Utöka nodtyper](./node.md)
