# Förhandsvisning av filer

I gränssnitt som innehåller filfält (inklusive bilagefält) kan ni förhandsvisa filer genom att klicka på filens miniatyrbild eller ikon. Den inbyggda funktionen för förhandsvisning stöder flera filtyper, inklusive bilder, PDF-filer och de flesta filtyper som webbläsare stöder nativt.

![20251129232307](https://static-docs.nocobase.com/20251129232307.png)

För filtyper som inte stöder nativ förhandsvisning kan ni aktivera funktionen genom att installera eller utöka med motsvarande plugin för filförhandsvisning. Till exempel kan ni förhandsvisa Word-, Excel- och PowerPoint-filer efter att ha installerat pluginet för förhandsvisning av Office-filer.

För närvarande erbjuder NocoBase följande plugin för filförhandsvisning:

* [Plugin för förhandsvisning av Office-filer](../file-preview/ms-office.md)
