# Bädda in NocoBase

## Introduktion

Detta plugin används för att bädda in NocoBase-sidor i andra webbplatser eller applikationer.

## Installation

Detta plugin är ett kommersiellt plugin och måste laddas upp och aktiveras via plugin-hanteraren.

![20240323162741](https://static-docs.nocobase.com/20240323162741.png)

## Användarhandbok

När ni har aktiverat gränssnittskonfigurationen klickar ni på "Kopiera inbäddningslänk" i sidans inställningsmeny (uppe till höger).

![](https://static-docs.nocobase.com/f11bd6d5e88d38731d7cd3cb149022c8.png)

Efter att ni har klickat får ni en länk som kan öppnas separat, till exempel `https://xxx.com/embed/qs087rz4o2b`.

![](https://static-docs.nocobase.com/9d847805a00fd07372a474665ec0e968.png)

Om ni vill bädda in NocoBase-sidor i andra webbplatser eller applikationer krävs användarautentisering, och `token` måste läggas till i länken, till exempel `https://xxx.com/embed/qs087rz4o2b?token=xxx`.
