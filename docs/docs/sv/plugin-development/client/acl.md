# ACL

:::tip

I det här kapitlet går vi igenom hur ni använder ACL (Access Control List) för behörighetshantering i frontend-plugins. Innehållet omfattar:

- Behörighetsdetektering;
- Hur ni dynamiskt renderar innehåll baserat på behörigheter;

Vi kommer att komplettera med detaljerade exempel och bästa praxis i kommande versioner.

:::
