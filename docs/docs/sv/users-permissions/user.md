# Användare

## Användarhantering

Denna plugin erbjuder ett gränssnitt för användarhantering i konfigurationscentret. Efter att applikationen har initierats skapas automatiskt ett superadministratörskonto som inte kan raderas. Superadministratören har Root-behörighet och kan komma åt alla resurser.

![](https://static-docs.nocobase.com/44bf40f56b45d4dd96c424fb08082cf6.png)

Dessutom kan ni också lägga till olika block från användar**samlingen** på sidan för att hantera användare, till exempel tabellblock:

![](https://static-docs.nocobase.com/76b5a4652f869541a9e8f18a4568a7c9.png)
