# Hantering av avdelningsroller

Genom att koppla en eller flera roller till en avdelning kan medlemmar i den avdelningen tilldelas dessa roller.

Det finns två sätt att koppla roller till en avdelning.

Det första sättet är att välja en roll i rollfältet på avdelningens redigeringsformulär.

![](https://static-docs.nocobase.com/70f77bb89aa1fb415c152a51a51cc23b.png)

Det andra sättet är att lägga till avdelningar till den aktuella rollen under rollhantering.

![](https://static-docs.nocobase.com/f2a7bec937cf2f179ce868a92b98416d.png)

Avdelningar som redan har denna roll kan inte väljas.

![](https://static-docs.nocobase.com/be10299893581e1f97a4e01ddd5c7e59.png)
