## Hantering av avdelningsmedlemmar

### Visa lista över avdelningsmedlemmar

![](https://static-docs.nocobase.com/2aaf4d9bf55da105b5fca4e9f7e23ca7.png)

### Lägg till medlemmar i avdelningen

En användare kan vara medlem i flera avdelningar samtidigt. Den första avdelningen som användaren går med i blir den primära avdelningen som standard. En användare kan endast ha en primär avdelning.

![](https://static-docs.nocobase.com/60afd282f33b555e6fe0662b9da544cc.png)

Användare som redan är medlemmar i avdelningen kommer inte att visas i listan för användarval.

![](https://static-docs.nocobase.com/6bcd93173c169973f970de35d2657993.png)

### Ange avdelningsansvarig

I formuläret för avdelningsredigering klickar ni på fältet för ansvarig för att välja avdelningsmedlemmar som ska bli ansvariga. Flera val stöds. Medlemmar som redan är avdelningsansvariga kommer inte att visas i listan för medlemsval.

![](https://static-docs.nocobase.com/92970546cbd0aeb5a8b6a36da87583bd.png)

### Konfigurera avdelningar för användare

Förutom att lägga till medlemmar i avdelningen kan ni också konfigurera avdelningar för användare utifrån användarens perspektiv.

![](https://static-docs.nocobase.com/ca82a802012572e225570e8be93a4094.png)
