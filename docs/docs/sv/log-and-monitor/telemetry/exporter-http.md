# Telemetri-exporterare: HTTP

## Miljövariabler

### TELEMETRY_METRIC_READER

Typ av telemetrimått-exporterare.

```bash
TELEMETRY_METRIC_READER=http
```

### TELEMETRY_HTTP_URL

HTTP-URL för export av telemetridata.

```bash
TELEMETRY_HTTP_URL=
```
