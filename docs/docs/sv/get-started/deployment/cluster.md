# Klusterdistribution

NocoBase stöder att distribuera applikationer i klusterläge för att förbättra applikationens prestanda och tillgänglighet. Klusterdistribution gör det möjligt att köra olika instanser av applikationen på flera servrar, eller att köra flera processer i flerkärnigt läge på en enskild server.

För mer information, se [Klusterläge](/cluster-mode/)-dokumentationen.
