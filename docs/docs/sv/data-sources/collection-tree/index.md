# Trädsamling

## Introduktion

En trädsamling är ett designmönster för datasamlingar som används för att organisera hierarkisk data. Denna samlingsstruktur liknar ett träd, där varje dataobjekt kan ha ett eller flera underordnade objekt, och dessa underordnade objekt kan i sin tur ha egna underordnade objekt.
