# Extern datakälla - MSSQL

## Introduktion

Använd en extern MSSQL-databas som datakälla. De versioner som stöds för närvarande är: SQL Server 2014-2019

## Installation

Detta plugin är ett kommersiellt plugin som behöver laddas upp och aktiveras via plugin-hanteraren.
