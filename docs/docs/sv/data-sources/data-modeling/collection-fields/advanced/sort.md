# Samlingsväljare

Fältet Samlingsväljare är en speciell fälttyp som låter användare välja en eller flera samlingar från en fördefinierad lista. Denna fälttyp är mycket användbar i scenarier där ni behöver dynamiskt referera till olika samlingar, till exempel när ni bygger konfigurerbara rapporter, instrumentpaneler eller komplexa applikationer som behöver ändra datakällor baserat på användarens val.

## Användningsfall

### Dynamisk rapportgenerering

I en modul för rapportgenerering kan användare behöva välja en samling från flera samlingar som datakälla för rapporten. Genom att använda Samlingsväljaren kan användare enkelt välja önskad samling, och systemet kommer dynamiskt att generera rapporten baserat på användarens val.

### Konfigurerbara instrumentpaneler

I en applikation för instrumentpaneler kan användare anpassa den data som visas på instrumentpanelen. Samlingsväljaren låter användare välja de samlingar de är intresserade av, och därmed visa diagram och statistisk data från olika datakällor på instrumentpanelen.

### Flexibla verktyg för dataimport/export

När ni utvecklar ett generiskt verktyg för dataimport eller export kan ni använda Samlingsväljaren för att låta användare ange vilken samling som ska bearbetas. På så sätt kan samma verktyg användas för att bearbeta data från vilken samling som helst i systemet, vilket avsevärt förbättrar verktygets återanvändbarhet.
