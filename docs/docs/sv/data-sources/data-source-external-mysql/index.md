# Extern datakälla - MySQL

## Introduktion

Använd en extern MySQL-databas som datakälla. Versioner som stöds för närvarande: MySQL >= 5.7

## Installation

Detta plugin är ett kommersiellt plugin som behöver laddas upp och aktiveras via pluginhanteraren.
