# Extern datakälla - MariaDB

## Introduktion

Använd en extern MariaDB-databas som datakälla. För närvarande stöds versionerna MariaDB >= 10.3.

## Installation

Detta plugin är ett kommersiellt plugin som behöver laddas upp och aktiveras via pluginhanteraren.
