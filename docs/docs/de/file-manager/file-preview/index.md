# Dateivorschau

In Oberflächen, die Dateifelder (einschließlich Anhangsfeldern) enthalten, können Sie Dateien durch Klicken auf das Vorschaubild oder das Symbol der Datei in der Vorschau anzeigen. Die integrierte Vorschaufunktion unterstützt verschiedene Dateitypen, darunter Bilder, PDFs und die meisten Dateitypen, die nativ von Browsern unterstützt werden.

![20251129232307](https://static-docs.nocobase.com/20251129232307.png)

Für Dateitypen, die keine native Vorschau unterstützen, können Sie die Vorschaufunktion durch die Installation oder Erweiterung entsprechender Dateivorschau-Plugins aktivieren. Beispielsweise können Sie nach der Installation des Office-Dateivorschau-Plugins Word-, Excel- und PowerPoint-Dateien in der Vorschau anzeigen.

Derzeit bietet NocoBase die folgenden Dateivorschau-Plugins an:

* [Office-Dateivorschau-Plugin](../file-preview/ms-office.md)
