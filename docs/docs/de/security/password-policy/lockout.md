# Benutzersperrung

## Einführung

Verwalten Sie gesperrte Benutzer und sperren Sie Benutzer proaktiv.

![](https://static-docs.nocobase.com/202412281450083.png)

### Benutzer entsperren

Benutzer, die aufgrund des Überschreitens der Anmeldeversuche mit ungültigem Passwort gesperrt wurden, können durch Löschen des Eintrags entsperrt werden.

### Benutzer sperren

Durch das Hinzufügen eines Benutzereintrags können Sie einen Benutzer proaktiv sperren. Sobald ein Benutzer gesperrt ist, wird ihm der Zugriff auf das System über jegliche Authentifizierungsmethode untersagt, einschließlich API-Schlüsseln.

![](https://static-docs.nocobase.com/202412281450512.png)
