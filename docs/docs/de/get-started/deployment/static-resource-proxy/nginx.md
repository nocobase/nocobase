# Nginx

Wenn Sie heute den Produktions-Proxy für eine NocoBase-Anwendung einrichten möchten, beginnen Sie am besten mit [Reverse Proxy in Produktion](../../../quickstart/production/reverse-proxy/index.md) und anschließend mit der Seite [Nginx](../../../quickstart/production/reverse-proxy/nginx.md).

Dieser ältere Abschnitt diente vor allem als Einstieg für den Proxy statischer Ressourcen. Die aktuelle Dokumentation wurde rund um `nb proxy nginx` neu organisiert und deckt die Konfigurationsgenerierung, den lokalen oder Docker-Betrieb sowie die Routen `uploads`, `dist`, `api`, `ws` und SPA einheitlich ab.
