# Cluster-Bereitstellung

NocoBase unterstützt die Bereitstellung von Anwendungen im Cluster-Modus, um deren Leistung und Verfügbarkeit zu verbessern. Bei der Cluster-Bereitstellung können Sie verschiedene Instanzen der Anwendung auf mehreren Servern ausführen oder mehrere Prozesse im Mehrkern-Modus auf einem einzigen Server betreiben.

Ausführliche Informationen finden Sie in der Dokumentation zum [Cluster-Modus](/cluster-mode/).
