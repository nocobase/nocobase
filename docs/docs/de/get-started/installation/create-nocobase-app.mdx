# create-nocobase-app Installation

:::tip Voraussetzungen

- Node.js 20+ und Yarn 1.22.x sind installiert.
- Eine der folgenden erforderlichen Datenbanken ist konfiguriert und läuft: MySQL 8.0.17+, MariaDB 10.9+ oder PostgreSQL 10+.

:::

## 1. NocoBase-Projekt erstellen

Wählen Sie die NocoBase-Version ([Versionsvergleich](../quickstart.md)) und den Datenbanktyp aus und führen Sie dann den entsprechenden Befehl aus.

<Tabs>
<Tab label="PostgreSQL">

```bash

   -e DB_HOST=localhost \
   -e DB_PORT=5432 \
   -e DB_DATABASE=nocobase \
   -e DB_USER=nocobase \
   -e DB_PASSWORD=nocobase \
   -e TZ=Asia/Shanghai
```

:::warning Umgebungsvariablen

- `TZ` wird verwendet, um die Zeitzone der Anwendung festzulegen. Standardmäßig wird die Zeitzone des Betriebssystems verwendet (z. B. `TZ=UTC`).
- `APP_KEY` ist der geheime Schlüssel der Anwendung, der zur Generierung von Benutzer-Tokens usw. verwendet wird (wenn der `APP_KEY` geändert wird, werden alte Tokens ungültig). Es kann eine beliebige zufällige Zeichenfolge sein. **Bitte ändern Sie ihn in Ihren eigenen geheimen Schlüssel und stellen Sie sicher, dass er nicht nach außen gelangt.**
- `DB_*` sind datenbankbezogene Konfigurationen. Bitte passen Sie diese entsprechend Ihrer tatsächlichen Einrichtung an Ihre Datenbankverbindungsinformationen an.

:::

## 2. Verzeichnis wechseln

```bash
cd my-nocobase-app
```

## 3. Abhängigkeiten installieren

📢 Abhängig von Ihrer Netzwerkumgebung, Systemkonfiguration und anderen Faktoren kann dieser Schritt über zehn Minuten dauern.

```bash
yarn install
```

:::tip Hinweis zur Produktionsumgebung

Beim Deployment in einer Produktionsumgebung können Sie zur Reduzierung der Größe nur die notwendigen Abhängigkeiten installieren:

```bash
yarn install --production
```

:::

## 4. NocoBase installieren

```bash
yarn nocobase install --lang=en-US
```

Der Installationsprozess erstellt automatisch die Datenbanktabellenstruktur und initialisiert die Daten.

## 5. NocoBase starten

**Entwicklungsumgebung**

```bash
yarn dev
```

**Produktionsumgebung**

```bash
yarn start
```

## 6. Bei NocoBase anmelden

Öffnen Sie [http://localhost:13000](http://localhost:13000) in Ihrem Browser. Der initiale Benutzername und das Passwort sind `admin@nocobase.com` und `admin123`.

:::warning Hinweis zur Kontosicherheit

Nach der ersten Anmeldung ändern Sie bitte umgehend das Standardpasswort, um die Systemsicherheit zu gewährleisten.

:::
