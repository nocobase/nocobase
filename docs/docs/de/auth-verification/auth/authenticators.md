# Authentifikatoren verwalten

## Benutzerauthentifizierung verwalten

Wenn das Benutzerauthentifizierungs-Plugin installiert wird, initialisiert es eine `Passwort`-Authentifizierungsmethode, die auf dem Benutzernamen und der E-Mail-Adresse des Benutzers basiert.

![](https://static-docs.nocobase.com/66eaa9d5421c9cb713b117366bd8a5d5.png)

## Authentifikatoren aktivieren

![](https://static-docs.nocobase.com/7f1fb8f8ca5de67ffc68eff0a65848f5.png)

Nur aktivierte Authentifizierungstypen werden auf der Anmeldeseite angezeigt.

![](https://static-docs.nocobase.com/8375a36ef98417af0f0977f1e07345dd.png)

## Benutzerauthentifizierungstypen

![](https://static-docs.nocobase.com/da4250c0cea343ebe470cbf7be4b12e4.png)

Indem Sie verschiedene Arten von Authentifikatoren hinzufügen, können Sie dem System entsprechende Authentifizierungsmethoden hinzufügen.

Neben den von bestehenden Plugins bereitgestellten Authentifizierungstypen können Entwickler auch eigene Benutzerauthentifizierungstypen erweitern. Weitere Informationen finden Sie im [Entwicklerhandbuch](./dev/).
