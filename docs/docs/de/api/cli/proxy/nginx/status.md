# nb proxy nginx status

Zeigt den aktuellen Laufzeitstatus des Nginx-Proxys an.

## Verwendung

```bash
nb proxy nginx status
```

## Ausgabe

Die Ausgabe enthält normalerweise:

- `driver`
- `status`
- `config`
- `nginx` oder `container`
