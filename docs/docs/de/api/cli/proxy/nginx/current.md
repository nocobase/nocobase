# nb proxy nginx current

Gibt den aktuellen Driver des Nginx-Providers aus.

## Verwendung

```bash
nb proxy nginx current
```

## Ausgabe

Der Befehl gibt direkt einen einzelnen Wert aus:

- `local`
- `docker`

## Beispiele

```bash
nb proxy nginx current
```
