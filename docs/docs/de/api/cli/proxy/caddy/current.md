# nb proxy caddy current

Gibt den aktuellen Driver des Caddy-Providers aus.

## Verwendung

```bash
nb proxy caddy current
```

## Ausgabe

Der Befehl gibt direkt einen einzelnen Wert aus:

- `local`
- `docker`

## Beispiele

```bash
nb proxy caddy current
```
