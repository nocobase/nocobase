# nb proxy caddy start

Startet den Caddy-Proxy mit dem aktuellen Driver.

## Verwendung

```bash
nb proxy caddy start
```

## Beispiele

```bash
nb proxy caddy start
```

## Hinweise

- Mit dem `local`-Driver startet der Befehl den lokalen Caddy-Prozess
- Mit dem `docker`-Driver startet oder erstellt er den Docker-Container
- Wenn der Proxy bereits läuft, meldet der Befehl das entsprechend
