# nb proxy caddy status

Zeigt den aktuellen Laufzeitstatus des Caddy-Proxys an.

## Verwendung

```bash
nb proxy caddy status
```

## Ausgabe

Die Ausgabe enthält normalerweise:

- `driver`
- `status`
- `config`
- `caddy` oder `container`
