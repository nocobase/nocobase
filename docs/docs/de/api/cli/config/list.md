# nb config list

Listet explizit gesetzte CLI-Einstellungen auf. Standardwerte, die nie explizit gesetzt wurden, werden nicht angezeigt.

## Verwendung

```bash
nb config list
```

## Beispiele

```bash
nb config list
```

## Hinweise

Wenn keine Konfigurationswerte gesetzt sind, gibt die CLI `No CLI config values are set.` aus.

## Verwandte Befehle

- [`nb config get`](./get.md)
- [`nb config set`](./set.md)
