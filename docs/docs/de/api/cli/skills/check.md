# nb skills check

Prüft die globalen NocoBase AI coding skills und meldet, ob sie von der CLI verwaltet werden und ob Aktualisierungen verfügbar sind.

## Verwendung

```bash
nb skills check [flags]
```

## Parameter

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| `--json` | boolean | Ausgabe als JSON |

## Beispiele

```bash
nb skills check
nb skills check --json
```

## Verwandte Befehle

- [`nb skills install`](./install.md)
- [`nb skills update`](./update.md)
