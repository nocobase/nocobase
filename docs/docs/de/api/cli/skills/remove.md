# nb skills remove

Entfernt die von `nb` verwalteten globalen NocoBase AI coding skills.

## Verwendung

```bash
nb skills remove [flags]
```

## Parameter

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| `--yes`, `-y` | boolean | Bestätigung der Entfernung überspringen |
| `--json` | boolean | Ausgabe als JSON |
| `--verbose` | boolean | Ausführliche Ausgabe der Entfernung anzeigen |

## Beispiele

```bash
nb skills remove
nb skills remove --yes
nb skills remove --json
```
