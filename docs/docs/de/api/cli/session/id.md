# nb session id

Zeigt die aktuell wirksame Sitzungs-ID an.

Wenn in der aktuellen Shell oder Runtime keine verwendbare `NB_SESSION_ID` verfügbar ist, fordert der Befehl Sie auf, zuerst [`nb session setup`](./setup.md) auszuführen und danach eine neue Shell-Sitzung oder Runtime zu öffnen.

## Verwendung


nb session id

## Beispiele


nb session id

## Verwandte Befehle

- [`nb session setup`](./setup.md)
- [`nb env current`](../env/current.md)
