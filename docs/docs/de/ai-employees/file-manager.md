# Dateiverwaltung

## Einführung

Konfigurieren Sie die Speichermethode für Dateien, die während der Gespräche mit KI-Mitarbeitern hochgeladen werden.

## Einstellungen

Navigieren Sie zur Konfigurationsseite des KI-Mitarbeiter-Plugins und klicken Sie auf den Tab `Settings`, um zur Seite für die Speichereinstellungen zu gelangen.

![20251022002648](https://static-docs.nocobase.com/20251022002648.png)

Die verfügbaren Speicheroptionen für `Files storage` legen Sie im Dateiverwaltungs-Plugin fest.
