# Erweitert

## Einführung

Beim Gespräch mit einem KI-Mitarbeiter können Sie nicht nur Dateien senden, sondern auch Anwendungskontextinformationen übermitteln. So kann der KI-Mitarbeiter auf Basis dieser Kontextinformationen antworten.

### Blockinformationen senden

Im Dialogfeld des KI-Mitarbeiters klicken Sie unten links auf die Schaltfläche `Add work context` und wählen `Pick block` aus. Die Anwendungsseite wechselt dann in den Blockauswahlmodus.

![20251022010148](https://static-docs.nocobase.com/20251022010148.png)

Wenn Sie den Mauszeiger über einen Block bewegen, ändert sich die Farbe der Blöcke, die als Kontext gesendet werden können.

![20251022010422](https://static-docs.nocobase.com/20251022010422.png)

Nachdem Sie den gewünschten Block ausgewählt haben, werden die Block-Kontextinformationen dem Dialogfeld hinzugefügt.

![20251022010543](https://static-docs.nocobase.com/20251022010543.png)

Nachdem Sie Ihre Frage eingegeben haben, klicken Sie wie beim Senden einer normalen Nachricht auf die Schaltfläche „Senden“. Der KI-Mitarbeiter wird dann basierend auf den Blockinformationen antworten.

Die Blockinformationen erscheinen auch in der Nachrichtenliste.

![20251022010758](https://static-docs.nocobase.com/20251022010758.png)

### Blockauswahl beenden

Wenn sich die Anwendungsseite im Blockauswahlmodus befindet, Sie aber keinen Block auswählen möchten, können Sie auf das „Stopp“-Symbol unten klicken, um den Blockauswahlmodus zu verlassen.

![20251022011043](https://static-docs.nocobase.com/20251022011043.png)
