# Schnellstart

## Einführung
Nachdem Sie die LLM-Dienstkonfiguration abgeschlossen haben, können Sie mit der Erstellung von KI-Mitarbeitern beginnen.

## Neuer KI-Mitarbeiter

### Grundlegende Informationen konfigurieren

Gehen Sie zur Konfigurationsseite des KI-Mitarbeiter-Plugins, klicken Sie auf den Tab `AI employees`, um die Verwaltungsseite für KI-Mitarbeiter aufzurufen.

![20251021225643](https://static-docs.nocobase.com/20251021225643.png)

Klicken Sie auf die Schaltfläche `New AI employee`, um zur Erstellungsseite für KI-Mitarbeiter zu gelangen.

Zuerst können Sie auf dem Tab `Profile` die grundlegenden Informationen des KI-Mitarbeiters konfigurieren:

- Geben Sie im Eingabefeld `Username` die eindeutige Kennung des KI-Mitarbeiters ein.
- Geben Sie im Eingabefeld `Nickname` den Spitznamen des KI-Mitarbeiters ein.
- Geben Sie im Eingabefeld `Position` die Position des KI-Mitarbeiters ein, um dessen Aufgaben kurz zu beschreiben.
- Wählen Sie unter `Avatar` ein Profilbild für den KI-Mitarbeiter aus.

![20251021232246](https://static-docs.nocobase.com/20251021232246.png)

- Geben Sie im Eingabefeld `Bio` einen Einführungstext für den KI-Mitarbeiter ein, der dessen Fähigkeiten und Funktionen beschreibt. Dieser Text wird im Profil des KI-Mitarbeiters angezeigt.
- Geben Sie im Eingabefeld `About me` eine detaillierte Beschreibung der Funktionen des KI-Mitarbeiters ein. Dieser Text wird als System-Prompt verwendet, wenn Sie mit dem KI-Mitarbeiter chatten.
- Geben Sie im Eingabefeld `Greeting message` die Begrüßungsnachricht des KI-Mitarbeiters ein. Jedes Mal, wenn ein Benutzer eine neue Konversation mit dem KI-Mitarbeiter beginnt, sendet das System diese Nachricht automatisch an den Benutzer.

![20251021232712](https://static-docs.nocobase.com/20251021232712.png)

### Modellkonfiguration

Auf dem Tab `Model settings` können Sie die Modelleinstellungen für den KI-Mitarbeiter konfigurieren:

- Wählen Sie den LLM-Dienst aus, den Sie in der LLM-Dienstverwaltung erstellt haben.
- Geben Sie den Namen des großen Modells ein, das Sie verwenden möchten.

![20251021231727](https://static-docs.nocobase.com/20251021231727.png)

### Erstellung abschließen

Nachdem Sie alle erforderlichen Informationen zur Erstellung des KI-Mitarbeiters eingegeben haben, klicken Sie auf die Schaltfläche `Submit`, um die Erstellung abzuschließen.

Anschließend sehen Sie den neu erstellten KI-Mitarbeiter in der Liste der KI-Mitarbeiter. Auch über die Schaltfläche zum schnellen Aufruf von KI-Mitarbeitern unten rechts auf der Seite können Sie den soeben erstellten KI-Mitarbeiter finden.

![20251021232836](https://static-docs.nocobase.com/20251021232836.png)
