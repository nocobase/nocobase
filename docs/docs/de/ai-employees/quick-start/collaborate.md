# Schnellstart

## Einführung

Nachdem Sie einen KI-Mitarbeiter erstellt haben, können Sie auf jeder beliebigen Seite in NocoBase mit ihm zusammenarbeiten.

Sie können den Avatar des Mitarbeiters in der Mitarbeiterliste unten rechts auf der Seite anklicken, um ihn zu aktivieren und die Zusammenarbeit zu beginnen.

![20251102121159-2025-11-02-12-12-01](https://static-docs.nocobase.com/20251102121159-2025-11-02-12-12-01.png)

Sie können KI-Mitarbeiter auch überall dort platzieren, wo Sie sie benötigen – zum Beispiel in Tabellen, Formularen oder anderen Komponenten – und sie direkt mit einem Klick aktivieren.

![20251102121036-2025-11-02-12-10-38](https://static-docs.nocobase.com/20251102121036-2025-11-02-12-10-38.png)

## Grundlegende Bedienung im Chatfenster

Nachdem Sie einen KI-Mitarbeiter aktiviert haben, geben Sie Ihre Nachricht ein und klicken Sie auf den Senden-Button, um die Unterhaltung zu starten.

![20251021234541](https://static-docs.nocobase.com/20251021234541.png)

Wenn Sie den Mauszeiger über eine Benutzernachricht bewegen, erscheinen Schnellaktionssymbole:
- Nachricht bearbeiten und erneut senden
- Inhalt der Benutzernachricht kopieren

![20251021235715](https://static-docs.nocobase.com/20251021235715.png)

Wenn Sie den Mauszeiger über eine KI-Nachricht bewegen, erscheinen Schnellaktionssymbole:
- KI erneut antworten lassen
- Inhalt der KI-Nachricht kopieren

![20251021235847](https://static-docs.nocobase.com/20251021235847.png)

Klicken Sie auf das „Hochladen“-Symbol, um eine Datei hochzuladen und an den KI-Mitarbeiter zu senden.

![20251022000327](https://static-docs.nocobase.com/20251022000327.png)

Oben links im Chatfenster befindet sich die Liste der bisherigen Unterhaltungen. Klicken Sie darauf, um die Liste zu erweitern und vergangene Unterhaltungen anzuzeigen.

![20251021234640](https://static-docs.nocobase.com/20251021234640.png)

Klicken Sie auf das „Bleistift“-Symbol oben rechts, um eine neue Unterhaltung zu beginnen.

![20251021234841](https://static-docs.nocobase.com/20251021234841.png)

Klicken Sie auf das „i“-Symbol oben rechts, um die benutzerdefinierte Systemaufforderung zu bearbeiten.

![20251021235314](https://static-docs.nocobase.com/20251021235314.png)

Klicken Sie auf das „Erweitern“-Symbol oben rechts, um das Chatfenster des KI-Mitarbeiters im Vollbildmodus anzuzeigen.

![20251021235427](https://static-docs.nocobase.com/20251021235427.png)
