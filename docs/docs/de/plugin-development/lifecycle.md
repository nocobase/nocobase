# Lebenszyklus

Dieser Abschnitt fasst die Lebenszyklus-Hooks für Plugins auf Server- und Clientseite zusammen. Er hilft Entwicklern dabei, Ressourcen korrekt zu registrieren und freizugeben.

Sie können dies mit dem FlowModel-Lebenszyklus vergleichen, um gemeinsame Konzepte hervorzuheben.

## Empfohlene Inhalte

- Rückrufe, die beim Installieren, Aktivieren, Deaktivieren oder Entfernen von Plugins ausgelöst werden.
- Der Zeitpunkt für das Mounten, Aktualisieren und Zerstören von clientseitigen Komponenten.
- Empfehlungen zur Behandlung von asynchronen Aufgaben und Fehlern innerhalb des Lebenszyklus.
