# Schaltfläche bearbeiten

## Einführung

Das System ermöglicht es Ihnen, den Namen, das Symbol, die Hintergrundfarbe und den Schaltflächentyp anzupassen. Dies bietet flexible UI-Anpassungsoptionen, damit Sie das Erscheinungsbild und die Funktionalität der Schaltflächen an Ihre spezifischen Anforderungen anpassen können.

- **Schaltflächenname**: Sie können den Text der Schaltfläche an Ihre geschäftlichen Anforderungen anpassen, um sicherzustellen, dass die Beschriftung klar und intuitiv ist.
- **Schaltflächensymbol**: Sie können ein Symbol für die Schaltfläche festlegen, um die visuelle Erkennung zu verbessern und die Funktion der Schaltfläche schnell zu identifizieren.
- **Gefährliche Aktion (Danger Action)**: Wenn eine Schaltfläche eine **gefährliche Aktion** darstellt (z. B. Löschen, Zurücksetzen usw.), können Sie sie als Typ **Danger Action** festlegen.
- **Schaltflächentyp (Button Type)**: Schaltflächen unterstützen verschiedene Typeneinstellungen. Gängige Schaltflächentypen sind:
  - **Standard (Default)**: Eine Standardschaltfläche, geeignet für normale Operationen.
  - **Primär (Primary)**: Wird für wichtige oder primäre Aktionen verwendet, oft mit einem auffälligen visuellen Effekt.
  - **Gestrichelt (Dashed)**: Eine Schaltfläche mit gestricheltem Rahmen, geeignet für sekundäre Aktionen oder Hinweisschaltflächen.
  - **Text (Text)**: Eine Schaltfläche nur mit Text und ohne Hintergrundfarbe, typischerweise für leichte Aktionen oder Links verwendet.
