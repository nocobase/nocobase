# Feldzuweisung

## Einführung

Feldzuweisung bedeutet, dass Sie die Werte eines Feldes voreinstellen, bevor eine Aktion ausgeführt wird. Beim Aktualisieren oder Erstellen eines Datensatzes werden diese Daten automatisch ausgefüllt, sodass keine manuelle Eingabe erforderlich ist. Beispiele hierfür sind das Massenaktualisieren von Bestellstatus oder das automatische Festlegen der Lieferzeit auf die aktuelle Zeit, wenn eine Bestellung zugestellt wird.
