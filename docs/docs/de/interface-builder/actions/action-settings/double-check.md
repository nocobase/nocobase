# Bestätigung

## Einführung

Die Bestätigungskonfiguration dient dazu, die Ausführung einer Aktion zu bestätigen. Beim Klicken auf eine Aktion wird ein Bestätigungsdialog angezeigt. Um Datenverlust oder irreversible Ergebnisse durch versehentliche Vorgänge zu vermeiden, ist die Bestätigungsabfrage für den Löschen-Button standardmäßig aktiviert.

![20251029125021](https://static-docs.nocobase.com/20251029125021.png)

Sie können den Inhalt und den Titel der Bestätigungsabfrage anpassen und Variablen abrufen.
