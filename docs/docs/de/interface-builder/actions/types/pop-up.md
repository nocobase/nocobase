# Modal öffnen

## Einführung

Die **Aktion 'Modal öffnen'** ähnelt den Aktionen zum Anzeigen und Bearbeiten. Sie können eine Schaltfläche **Modal öffnen** in den Zeilen oder im Kopfbereich einer Tabelle konfigurieren. Beim Klicken auf diese Schaltfläche öffnet sich ein entsprechendes Modal, und die Blöcke innerhalb dieses Modals zeigen Daten an, die auf dem aktuellen Kontextdatensatz basieren.

![20251029182351](https://static-docs.nocobase.com/20251029182351.png)

Konfigurieren Sie die Blöcke im Modal nach Bedarf.

![20251029190208](https://static-docs.nocobase.com/20251029190208.png)

## Aktionseinstellungen

![20251029190352](https://static-docs.nocobase.com/20251029190352.png)

- [Schaltfläche bearbeiten](/interface-builder/actions/action-settings/edit-button): Bearbeiten Sie den Titel, den Typ und das Symbol der Schaltfläche.
- [Verknüpfungsregel](/interface-builder/linkage-rule): Steuern Sie den Zustand der Schaltfläche dynamisch (sichtbar/deaktiviert).
- [Modal bearbeiten](/interface-builder/actions/action-settings/edit-popup): Legen Sie den Öffnungsmodus (Modal oder Drawer) und die Größe fest.
