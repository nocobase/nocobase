# Link

## Einführung

Die Link-Aktion ermöglicht die Navigation über Routen und unterstützt die Übergabe von URL-Variablen. Dadurch kann die Zielseite ihren Inhalt dynamisch an die übergebenen Daten anpassen. Sie lässt sich in Datenblöcken konfigurieren und verwenden.
