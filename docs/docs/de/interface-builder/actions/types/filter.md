# Filtern

## Einführung

Die Filter-Aktion finden Sie üblicherweise in Datenblöcken. Sie ermöglicht es Ihnen, Daten durch das Konfigurieren verschiedener Bedingungen zu filtern. Beachten Sie bitte, dass, falls für den Block bereits ein Datenbereich konfiguriert wurde, die Filter-Aktion mit den Bedingungen des Datenbereichs zu einer finalen Filterbedingung zusammengeführt wird.

### Filterbare Felder

Standardmäßig sind alle Felder dieser Sammlung aktiviert. Nur aktivierte Felder können in der Liste der Bedingungsfelder ausgewählt werden.

Felder aus verknüpften Sammlungen können ebenfalls als Filterfelder verwendet werden.
