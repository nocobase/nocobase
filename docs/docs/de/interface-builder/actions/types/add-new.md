# Hinzufügen
## Einführung

Die Aktion „Hinzufügen“ ermöglicht es Ihnen, einen Datensatz zur Sammlung des aktuellen Blocks hinzuzufügen. Durch die Konfiguration eines Formular-Blocks können Sie neue Datensätze für die Sammlung des aktuellen Blocks eingeben.
