# Einreichen

## Einführung

Die Aktion `Einreichen` dient dazu, Formulardaten zu speichern (dies ist spezifisch für Formularblöcke). Sie lässt sich auch mit Workflows kombinieren, um automatisierte Datenprozesse zu realisieren.
