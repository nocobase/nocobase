# Kinddatensatz hinzufügen

## Einführung

Diese Funktion ist speziell für **Baumtabellen-Blöcke** entwickelt, um schnell Kinddatensätze hinzuzufügen. Beim Hinzufügen eines Kinddatensatzes wird die ausgewählte Zeile als Elternknoten behandelt, sodass Sie bequem unter jedem beliebigen Elternknoten Kinddatensätze erstellen können. Diese Funktion vereinfacht die Verwaltung hierarchischer Daten erheblich und macht die Erweiterung und Organisation von Datenstrukturen effizienter und intuitiver.
