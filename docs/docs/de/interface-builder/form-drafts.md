# Formular-Entwürfe

Wenn Formular-Entwürfe aktiviert sind, werden die eingegebenen Inhalte automatisch lokal gespeichert. Selbst wenn das Formular versehentlich geschlossen oder die Seite aktualisiert wird, gehen die ausgefüllten Daten nicht verloren.

![20251213093717](https://static-docs.nocobase.com/20251213093717.png)

Wenn ein Entwurf existiert, wird am oberen Rand des Formulars eine Benachrichtigung angezeigt. Nach erfolgreichem Absenden wird der Entwurf automatisch gelöscht; falls er nicht mehr benötigt wird, kann er auch jederzeit manuell gelöscht werden.

![20251213092734](https://static-docs.nocobase.com/20251213092734.png)

::: tip Hinweis

Derzeit unterstützen Formular-Entwürfe nur das lokale Caching. Zukünftige Versionen werden serverseitiges Caching unterstützen, um eine Synchronisierung über mehrere Geräte hinweg zu ermöglichen.

:::
