# Tabellenfelder

## Einführung

Neben der Anpassung von Spaltenbreite, Feldtitel und Sortierung bieten Tabellenfelder für bestimmte Felder (wie Datumsfelder, Beziehungsfelder und Zahlenfelder) auch zusätzliche, personalisierte Anzeigeoptionen.

![20251024174558](https://static-docs.nocobase.com/20251024174558.png)

## Feldkonfigurationsoptionen

### Datumsfeldformatierung

![20251024175303](https://static-docs.nocobase.com/20251024175303.png)

Weitere Informationen finden Sie unter [Datumsformatierung](/interface-builder/fields/specific/date-picker)

### Zahlenfeldformatierung

Es werden einfache Einheitenumrechnungen, Tausendertrennzeichen, Präfixe und Suffixe, Genauigkeit und wissenschaftliche Notation unterstützt.

![20251024175445](https://static-docs.nocobase.com/20251024175445.png)

Weitere Informationen finden Sie unter [Zahlenformatierung](/interface-builder/fields/field-settings/number-format)

### Schnelle Bearbeitung aktivieren

Wenn die schnelle Bearbeitung aktiviert ist, erscheint beim Überfahren der Spalte mit der Maus ein Bearbeiten-Button. Klicken Sie darauf, um Daten schnell in einem Pop-up-Fenster zu bearbeiten und zu speichern.

![20251025171158](https://static-docs.nocobase.com/20251025171158.gif)

### Klicken zum Öffnen aktivieren

Neben Beziehungsfeldern, die das Öffnen eines Pop-ups durch Klicken unterstützen, können auch normale Felder mit der Option „Klicken zum Öffnen“ versehen werden, um als Einstiegspunkt für ein Pop-up zu dienen.

![20251025172308](https://static-docs.nocobase.com/20251025172308.gif)

### Anzeigemodus für Inhaltsüberlauf

Wenn der Spalteninhalt die Breite der Tabellenspalte überschreitet, können Sie den Überlaufmodus festlegen:

-   Auslassungspunkte (Standard)
-   Zeilenumbruch

### Feldkomponente

Einige Felder unterstützen mehrere Anzeigeformate. Benutzer können Feldkomponenten wechseln, um unterschiedliche Anzeigeeffekte zu erzielen und so den Anforderungen verschiedener Szenarien gerecht zu werden. Zum Beispiel kann ein Feld vom Typ **URL** auf die **Vorschau**-Komponente umgeschaltet werden, um Linkinhalte oder Bildvorschauen besser darzustellen.

![20251025171658](https://static-docs.nocobase.com/20251025171658.png)

### Sortierbar

Die meisten Feldtypen unterstützen derzeit die serverseitige Sortierung. Wenn die Sortierung aktiviert ist, können Daten basierend auf dem Zielfeld in aufsteigender oder absteigender Reihenfolge sortiert werden.
