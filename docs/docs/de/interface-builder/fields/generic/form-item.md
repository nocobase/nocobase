# Formularfelder

## Einführung

Verschiedene Feldtypen in einem Formular bieten unterschiedliche Konfigurationsoptionen. Sie können diese Optionen über Plugins erweitern, um spezifische und individuelle Anforderungen zu erfüllen.

![20251025174555](https://static-docs.nocobase.com/20251025174555.png)

## Konfigurationsoptionen für Felder

### Standardwert

Sie können Konstanten oder Variablen als Standardwerte für Felder festlegen. Beachten Sie, dass dies nur für neue Formulare unterstützt wird.
