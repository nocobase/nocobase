# Detailfelder

## Einführung

Die Feldkonfigurationen für Detail-, Listen- und Rasterblöcke sind im Wesentlichen identisch. Sie steuern hauptsächlich, wie Felder im Lesemodus angezeigt werden.

![20251025172851](https://static-docs.nocobase.com/20251025172851.png)

## Feldkonfigurationsoptionen

### Datumsfeldformatierung

![20251025173005](https://static-docs.nocobase.com/20251025173005.png)

Weitere Informationen finden Sie unter [Datumsformatierung](/interface-builder/fields/specific/date-picker)

### Zahlenfeldformatierung

![20251025173242](https://static-docs.nocobase.com/20251025173242.png)

Es unterstützt einfache Einheitenumrechnungen, Tausendertrennzeichen, Präfixe und Suffixe, Genauigkeit und wissenschaftliche Notation.

Weitere Informationen finden Sie unter [Zahlenformatierung](/interface-builder/fields/field-settings/number-format)

### Klick zum Öffnen aktivieren

Neben Beziehungsfeldern können auch normale Felder so konfiguriert werden, dass sie beim Anklicken ein Pop-up öffnen. Sie können auch festlegen, wie sich das Pop-up öffnet (als Schublade, Dialog oder Unterseite).

![20251025173549](https://static-docs.nocobase.com/20251025173549.gif)

### Anzeigemodus für Inhaltsüberlauf

Wenn der Feldinhalt die Breite überschreitet, können Sie den Überlaufmodus festlegen.

- Auslassungspunkte (Standard)
- Zeilenumbruch

![20251025173917](https://static-docs.nocobase.com/20251025173917.png)

### Feldkomponente

Einige Felder unterstützen mehrere Anzeigeformate, die Sie durch Wechseln der Feldkomponente realisieren können.

Zum Beispiel: Die `URL`-Komponente kann zur `Preview`-Komponente gewechselt werden.

![20251025174042](https://static-docs.nocobase.com/20251025174042.png)

Zum Beispiel können Beziehungsfelder unterschiedlich angezeigt werden, indem Sie von der Titelfeldkomponente zu `Unterdetails` wechseln, um mehr Inhalte aus der Beziehung anzuzeigen.

![20251025174311](https://static-docs.nocobase.com/20251025174311.gif)
