# Filterformularfelder

## Einführung

Das Filterformular ermöglicht es Ihnen, Felder aus der aktuellen Sammlung sowie aus verknüpften Sammlungen (auch Felder aus verschachtelten Verknüpfungen) als Filterfelder auszuwählen.

### Operator

Wählen Sie den passenden Operator basierend auf dem Feldtyp aus, um die Genauigkeit und Effizienz der Filterung zu verbessern. Bei Feldern vom Typ Zeichenkette ist die Standardeinstellung eine unscharfe Suche (Fuzzy Matching).
