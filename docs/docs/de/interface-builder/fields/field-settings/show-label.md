# Beschriftung anzeigen

## Einführung

Standardmäßig zeigen Felder in einem Block Feldbezeichnungen an. Diese kennzeichnen die Bedeutung jedes Feldes und erleichtern Ihnen das Lesen und Verstehen der Dateninhalte.

In bestimmten Szenarien (z. B. auf reinen Anzeigeseiten oder wenn die Bedeutung der Felder bereits sehr klar ist) können Sie die Beschriftungen ausblenden, um die Benutzeroberfläche übersichtlicher zu gestalten. Nach dem Ausblenden werden nur noch die Feldwerte angezeigt, die Feldnamen sind dann nicht mehr sichtbar.
