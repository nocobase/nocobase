# Zahlenformatierung

## Einführung

Die Zahlenformatierung ermöglicht die visuelle Aufbereitung und Standardisierung von Zahlenfeldern (einschließlich Formelfeldern) im Lesemodus. Dazu gehören Einstellungen für Tausendertrennzeichen, Dezimalstellen, Einheitenumrechnungen und die wissenschaftliche Notation. Durch die Konfiguration der Zahlenformatierung können Sie die Zahlenanzeige an die Gewohnheiten verschiedener Regionen und Branchen anpassen.

## Anwendung

- Unterstützt einfache Einheitenumrechnungen (Multiplikation/Division)
- Tausendertrennzeichen
- Präfix und Suffix
- Genauigkeit
- Wissenschaftliche Notation.

Beispiel: Formatieren Sie einen Bestellbetrag, indem Sie ein Währungssymbol (z.B. €), ein Tausendertrennzeichen (wie Komma oder Punkt) und Dezimalstellen hinzufügen, um den Betrag klar und leserlich darzustellen.
