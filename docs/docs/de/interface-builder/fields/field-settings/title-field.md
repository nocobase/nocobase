# Titelfeld

## Einführung

Das **Titelfeld** wird typischerweise in Beziehungsfeld-Komponenten verwendet, um die auf der Oberfläche angezeigten verknüpften Daten zu identifizieren. Es legt fest, wie die Hauptinformationen der Daten dargestellt werden, und hilft Benutzern in der Regel dabei, die Daten schnell zu erkennen und zu verstehen.

Verschiedene Arten von Titelfeldern entsprechen unterschiedlichen Feld-Komponenten und unterstützen entsprechende Konfigurationsoptionen. Zum Beispiel:

- Wenn das Titelfeld ein **Datumsfeld** ist, verwendet die Feld-Komponente eine **Datumskomponente**, um die Daten anzuzeigen.
- Wenn das Titelfeld ein **Textfeld** ist, verwendet die Feld-Komponente eine **Textkomponente**, um die Daten anzuzeigen.
- Wenn das Titelfeld ein **Optionsfeld** ist, verwendet die Feld-Komponente eine **Optionskomponente**, um die Daten anzuzeigen.
