# Anzeigemodus

## Einführung

Im Gegensatz zu Blöcken verfügen Feldkomponenten über drei Anzeigemodi (diese werden nur für Felder innerhalb eines Formulars unterstützt). Wenn Sie zwischen den verschiedenen Anzeigemodi wechseln, ändern sich die entsprechenden Feldkonfigurationsoptionen.

- Bearbeitbar;
- Nur lesbar (nicht bearbeitbar);
- Nur lesbar (Lesemodus);

### Anzeigemodi für Beziehungsfelder

Der **Anzeigemodus eines Beziehungsfeldes** bestimmt, wie das Feld in der Benutzeroberfläche dargestellt wird und welche Feldkomponententypen verfügbar sind.

Im **bearbeitbaren Zustand** unterstützen Beziehungsfelder verschiedene Komponententypen. Benutzer können je nach Geschäftsanforderungen unterschiedliche Beziehungsfeldkomponenten auswählen, um verknüpfte Daten anzuzeigen oder auszuwählen.

#### Beziehungsfeldkomponenten im bearbeitbaren Zustand

![20251028220447](https://static-docs.nocobase.com/20251028220447.png)

In diesem Zustand können Benutzer flexibel die passende Anzeigemethode wählen, um Daten effizienter zu verarbeiten.

#### Beziehungsfeldkomponenten im Lesemodus

Wenn Sie in den **Lesemodus** wechseln, verwendet das System automatisch standardmäßig die **Titel-Feldkomponente**, um die verknüpften Daten anzuzeigen. Dies eignet sich für Szenarien, in denen Sie Daten nur anzeigen, aber keine Änderungen vornehmen möchten.
