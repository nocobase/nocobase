# Tooltip bearbeiten

## Einführung

Ein Feld-Tooltip ist der Text, der angezeigt wird, wenn Sie den Mauszeiger über ein Feld bewegen. Er dient dazu, zusätzliche Erklärungen oder Anleitungen zu diesem Feld zu geben. Diese Tooltips enthalten in der Regel Informationen wie den Zweck des Feldes, das Eingabeformat und mögliche Einschränkungen, um Benutzern zu helfen, das Feld besser zu verstehen und zu nutzen.
