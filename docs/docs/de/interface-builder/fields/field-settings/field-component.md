# Feldkomponente

## Einführung

Feldkomponenten dienen dazu, den Inhalt eines Feldes anzuzeigen.
Einige Felder verfügen über mehrere integrierte Anzeigekomponenten. Sie können die passende Komponente je nach Ihren Anforderungen auswählen, um den Wert des Feldes auf unterschiedliche Weisen darzustellen.

Beispiel: Im Lesemodus kann ein URL-Feld auf eine Vorschau-Komponente oder eine URL-Komponente (Standard) umgeschaltet werden.
