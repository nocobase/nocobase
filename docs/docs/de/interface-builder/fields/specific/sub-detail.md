# Unterdetails

## Einführung

Unterdetails ist die Komponente, die einem Unterformular im Lesemodus entspricht. Im Vergleich zu den Komponenten 'Label' und 'Titel' können Unterdetails nicht nur mehr Daten aus der aktuellen Sammlung anzeigen, sondern auch so konfiguriert werden, dass sie Informationen aus verknüpften Sammlungen darstellen. Mehrstufige Beziehungsdaten werden dabei übersichtlich in Form von verschachtelten Details angezeigt.
