# Untertabelle

## Einführung

Die Untertabelle eignet sich für die Verwaltung von Beziehungsfeldern mit "Eins-zu-Viele"-Beziehungen. Sie ermöglicht es Ihnen, neue Daten in der Ziel-Sammlung in großen Mengen zu erstellen und zu verknüpfen oder bestehende Daten auszuwählen und zu verknüpfen.

## Verwendungshinweise

In der Untertabelle zeigen verschiedene Feldtypen unterschiedliche Feldkomponenten an. Große Felder (wie Rich Text, JSON, Mehrzeiliger Text) werden über ein schwebendes Pop-up-Fenster bearbeitet.

## Feldkonfigurationsoptionen

### Auswahl bestehender Daten erlauben (standardmäßig deaktiviert)

Ermöglicht die Auswahl und Verknüpfung bestehender Daten.
![20251027224008](https://static-docs.nocobase.com/20251027224008.png)

![20251027224023](https://static-docs.nocobase.com/20251027224023.gif)

### Feldkomponente

[Feldkomponente](/interface-builder/fields/association-field): Wechseln Sie zu anderen Beziehungsfeldkomponenten, wie z. B. Dropdown-Auswahl, Datenauswahl usw.

### Trennung bestehender Daten erlauben

> Legt fest, ob die Trennung bestehender Daten für das Beziehungsfeld im Bearbeitungsformular erlaubt ist.
