# Unterformular

## Einführung

Unterformulare eignen sich für Szenarien, in denen Sie zuerst neue Beziehungsdaten erstellen und diese dann verknüpfen. Mehrstufige Beziehungsdaten werden übersichtlich als verschachteltes Formular dargestellt. Im Vergleich zu Datenselektoren und Dropdown-Komponenten können Sie mit einem Unterformular die Felder der Beziehungssammlung direkt im aktuellen Seitenblock verwalten, und die Beziehungsdaten werden zusammen mit dem Hauptformular übermittelt.
