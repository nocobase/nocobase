# Kaskadierende Auswahl

## Einführung

Der kaskadierende Selektor ist für Beziehungsfelder konzipiert, deren Ziel-Sammlung eine Baumstruktur ist. Er ermöglicht es Ihnen, Daten entsprechend der hierarchischen Struktur der Baumstruktur-Sammlung auszuwählen und unterstützt die unscharfe Suche für eine schnelle Filterung.

## Verwendung

- Bei einer **Eins-zu-Eins-Beziehung** ist die kaskadierende Auswahl eine **Einzelauswahl**.

![20251125214656](https://static-docs.nocobase.com/20251125214656.png)

- Bei einer **Eins-zu-Viele-Beziehung** ist die kaskadierende Auswahl eine **Mehrfachauswahl**.

![20251125215318](https://static-docs.nocobase.com/20251125215318.png)
