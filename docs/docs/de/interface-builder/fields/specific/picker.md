# Daten-Selektor

## Einführung

Der Daten-Selektor ermöglicht es Ihnen, in Form eines Popup-Tabellen-Selektors, entweder vorhandene Daten aus der Ziel-Sammlung für eine Verknüpfung auszuwählen oder neue Daten zur Ziel-Sammlung hinzuzufügen und diese dann zu verknüpfen.

![20251027203343](https://static-docs.nocobase.com/20251027203343.png)

Bereits ausgewählte Daten werden im Tabellen-Selektor nicht angezeigt.

![20251027203717](https://static-docs.nocobase.com/2025103717.gif)

Im Tabellen-Selektor können Sie die Ziel-Sammlung der Verknüpfung auch weiter verwalten (z. B. hinzufügen, löschen, importieren/exportieren).

![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
