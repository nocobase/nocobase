# Dateimanager

## Einführung

Der Dateimanager ist eine Beziehungsfeld-Komponente, die Beziehungen verwaltet, bei denen die Ziel-Sammlung die Dateisammlung ist.

![20251027194124](https://static-docs.nocobase.com/20251027194124.png)

## Feldkonfiguration

- [Feldkomponente](/interface-builder/fields/association-field): Wechseln Sie zu anderen Beziehungsfeld-Komponenten, wie zum Beispiel Dropdown-Auswahl, Datenselektor usw.
- Auswahl bestehender Dateien erlauben: Standardmäßig aktiviert, ermöglicht die Auswahl und Verknüpfung von bereits hochgeladenen Dateien.
