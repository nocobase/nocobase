# Titel

## Einführung

Im schreibgeschützten Modus werden Beziehungsfelder standardmäßig mit der **Titelfeld-Komponente** angezeigt. Diese identifiziert den aktuell verknüpften Datensatz, indem sie den Wert des Titelfeldes darstellt. Das System passt die entsprechende Titelfeld-Komponente automatisch an und rendert sie, basierend auf dem Typ des Titelfeldes.
