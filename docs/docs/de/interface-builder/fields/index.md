# Felder

Im UI ist ein Feld eine Feldkomponente, die als Träger für einzelne Daten dient. Verschiedene Datentypen werden durch unterschiedliche Feldkomponenten dargestellt. Felder können nur Blöcken zugeordnet werden und nicht eigenständig verwendet werden.

## Felder in einem Block

Felder werden im Allgemeinen nicht einzeln verwendet, sondern existieren als Unterelemente von Datentyp-Blöcken. Datentyp-Blöcke verfügen normalerweise über die Option „Felder konfigurieren“, und die Liste der Felder wird von der aktuellen Sammlung bereitgestellt.

![20251029204240](https://static-docs.nocobase.com/20251029204240.png)

## Feld-Designer (Symbolleiste)

Ähnlich wie bei Blöcken hat auch die Feldkomponente oben rechts drei Symbole, und zwar:

- Layout per Drag & Drop
- [Ereignisfluss](/interface-builder/event-flow)
- Konfiguration
