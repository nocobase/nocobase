# Oberflächengestaltung

## WYSIWYG

NocoBase bietet eine WYSIWYG-Oberflächengestaltung. Klicken Sie auf die Schaltfläche 'UI-Gestaltung', um zwischen dem Konfigurationsmodus und dem Ansichtsmodus zu wechseln.

## Layout-Vorlage

NocoBase wird mit einer Standard-Layout-Vorlage geliefert: Oben und links befindet sich der Navigationsbereich, rechts der Inhaltsbereich.

![未命名.002](https://static-docs.nocobase.com/未命名.002.jpeg)

## Konfigurationsoptionen

Im Konfigurationsmodus erscheinen orangefarbene Markierungen für konfigurierbare Elemente auf der Oberfläche. Der Einstiegspunkt für die Einstellungen eines Elements befindet sich normalerweise in dessen rechter oberer Ecke. Fast alle Elemente können direkt auf der Oberfläche in Echtzeit konfiguriert und deren Auswirkungen sofort angesehen werden. Hier sind einige Beispiele:
