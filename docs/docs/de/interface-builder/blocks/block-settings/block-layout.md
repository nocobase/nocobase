# Layout

## Einführung

Formularähnliche Blöcke wie Formular-Blöcke, Detail-Blöcke und Listen-Blöcke unterstützen Layout-Einstellungen. Die Layout-Optionen umfassen ein vertikales Layout (Etiketten und Eingabefelder werden untereinander angeordnet) und ein horizontales Layout (Etiketten und Eingabefelder werden nebeneinander platziert). Sie können das Feld-Layout der Blöcke an Ihre spezifischen Anforderungen anpassen, um sicherzustellen, dass die Eingabe oder Anzeige von Informationen sowohl übersichtlich als auch effizient ist.

### Vertikal (Standard)

Das Standard-Layout ist vertikal.

![20251024120143](https://static-docs.nocobase.com/20251024120143.png)

### Horizontal

Das horizontale Layout zeigt Etiketten und Eingabefelder nebeneinander an, was effektiv Platz spart. Bei Feldern mit langen Titeln können Sie wählen, ob der Text umgebrochen oder gekürzt (mit Auslassungspunkten) werden soll, um eine saubere und ästhetisch ansprechende Oberfläche zu gewährleisten.

![20251024120237](https://static-docs.nocobase.com/20251024120237.png)

#### Ausrichtung des Feldtitels

- **Linksbündig** (Standard)
- **Rechtsbündig**

#### Breite des Feldtitels

- Die Standardbreite beträgt **120px**, die Mindestbreite liegt bei **50px**.

#### Umgang mit Feldtiteln, die die Breite überschreiten

- **Zeilenumbruch**
- **Kürzen**

#### Doppelpunkt anzeigen

Sie können wählen, ob nach dem Feldtitel ein Doppelpunkt angezeigt werden soll.
