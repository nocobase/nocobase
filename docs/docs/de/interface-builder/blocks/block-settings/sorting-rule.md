# Sortierregeln konfigurieren

## Einführung

Wenn Sie Sortierregeln konfigurieren, legen Sie fest, wie die Daten in einem Datenblock standardmäßig sortiert werden sollen. Die Daten werden dann in der von Ihnen definierten Reihenfolge angezeigt.
