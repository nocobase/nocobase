# Datenlademodus

## Einführung

Der Standard-Datenlademodus ist „Alle Daten laden, wenn die Filterbedingungen leer sind“. Hierbei beziehen sich die „Filterbedingungen“ auf die Werte im „Filter-Block“. Eine weitere Option ist „Keine Daten laden, wenn die Filterbedingungen leer sind“.
