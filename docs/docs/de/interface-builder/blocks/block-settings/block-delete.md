# Block löschen

## Einführung

Wenn Sie einen Block löschen, wird dieser von der aktuellen Seite entfernt. Bitte beachten Sie, dass dies ein irreversibler Vorgang ist. Falls Sie vor dem Löschen ein Backup erstellt haben, können Sie den Block über dieses Backup wiederherstellen.
