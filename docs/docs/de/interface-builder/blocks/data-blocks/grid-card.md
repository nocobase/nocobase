# Rasterkarte

## Einführung

Der Rasterkarten-Block zeigt zusammenfassende Informationen von Datensätzen in einem Kartenformat an. Sie können die Spaltenanzahl für verschiedene Bildschirmgrößen konfigurieren, um eine benutzerfreundliche Darstellung auf unterschiedlichen Geräten zu gewährleisten.
