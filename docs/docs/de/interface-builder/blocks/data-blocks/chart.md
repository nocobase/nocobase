# Diagramm

## Einführung

Das Datenvisualisierungs-Plugin von NocoBase bietet visuelle Datenabfragen und eine Vielzahl an Diagrammkomponenten. Sie können mit einer einfachen Konfiguration schnell Visualisierungs-Dashboards erstellen, um Dateneinblicke zu präsentieren und mehrdimensionale Datenanalysen sowie -darstellungen zu unterstützen.
