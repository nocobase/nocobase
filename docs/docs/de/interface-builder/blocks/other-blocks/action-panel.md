# Block: Werkbank

## Einführung

Hier können Sie verschiedene Schnellaktionen platzieren. Derzeit werden folgende Aktionen unterstützt:

- [Link](/interface-builder/actions/types/link)
- [QR-Code scannen]
- [Pop-up-Aktion](/interface-builder/actions/types/pop-up)

![20251023225754](https://static-docs.nocobase.com/20251023225754.png)

> 💡 In zukünftigen Versionen werden kontinuierlich weitere Aktionstypen hinzugefügt.
## Aktionstypen

- **Link**: Navigiert zu einer bestimmten Seite oder einer externen URL.

![20251023230643](https://static-docs.nocobase.com/20251023230643.png)

- **QR-Code scannen**: Löst eine entsprechende Aktion durch das Scannen eines QR-Codes aus.

- **Pop-up öffnen**: Öffnet ein Pop-up, um den Inhalt des Blocks anzuzeigen.
  ![20251023230522](https://static-docs.nocobase.com/20251023230522.png)
