# Markenanpassung

## Einführung

Der Standard-Markenname ist "NocoBase" und wird hauptsächlich an den folgenden drei Stellen angezeigt:

![20240409111100](https://static-docs.nocobase.com/20240409111100.png)

Am unteren Rand der Anmeldeseite:

![20240409111254](https://static-docs.nocobase.com/20240409111254.png)

Position 1 können Sie über die Option `Logo` in den [Systemeinstellungen](/system-management/system-settings/) konfigurieren. Die Positionen 2 und 3 lassen sich mit diesem `Plugin` anpassen.

## Anleitung zur Verwendung

Nachdem Sie das `Plugin` aktiviert haben, klicken Sie oben rechts auf der Seite auf das Menü „Markenanpassung“, um zur Konfigurationsseite zu gelangen.

![20240409113640](https://static-docs.nocobase.com/20240409113640.png)

- **Marke**: Hiermit legen Sie den Text am unteren Rand der Anmeldeseite fest (Position 3).
- **Über**: Hiermit konfigurieren Sie den Inhalt des Menüs oben rechts auf der Seite (siehe Abbildung unten).

![20240409113915](https://static-docs.nocobase.com/20240409113915.png)

Die Konfiguration unterstützt das HTML-Format, was eine flexible Anpassung der Inhalte ermöglicht. Wenn Sie nur einfache Textänderungen vornehmen möchten, können Sie den bestehenden Inhalt direkt bearbeiten, wie in der Abbildung unten gezeigt:

![20240409114740](https://static-docs.nocobase.com/20240409114740.png)
