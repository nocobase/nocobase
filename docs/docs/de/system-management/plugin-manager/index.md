# Plugin-Verwaltung

## Plugin-Manager

Klicken Sie auf das Plugin-Manager-Symbol oben rechts auf der Seite, um zur Plugin-Manager-Seite zu gelangen. Auf dieser Seite können Sie Plugins anzeigen, installieren, aktivieren oder deaktivieren.

![2024-04-10_08-47-45](https://static-docs.nocobase.com/2024-04-10_08-47-45.png)

## Plugin-Einstellungen

Plugins verfügen in der Regel über eigene Einstellungsseiten. Klicken Sie auf das Einstellungen-Symbol oben rechts auf der Seite, um die Einstellungsmenüs aller Plugins anzuzeigen. Klicken Sie auf den entsprechenden Menüpunkt, um das Plugin zu konfigurieren.
