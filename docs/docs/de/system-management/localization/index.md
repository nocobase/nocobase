# Lokalisierungsverwaltung

## Einführung

Das Plugin für die Lokalisierungsverwaltung dient dazu, die Lokalisierungsressourcen von NocoBase zu verwalten und bereitzustellen. Mit ihm können Sie Systemmenüs, Sammlungen, Felder und alle Plugins übersetzen, um sie an die Sprache und Kultur spezifischer Regionen anzupassen.

## Installation

Dieses Plugin ist ein integriertes Plugin und erfordert keine zusätzliche Installation.

### Übersetzungsbegriffe synchronisieren

<img src="https://static-docs.nocobase.com/202404202134850.png"/>

Derzeit können folgende Inhalte synchronisiert werden:

- Lokale Sprachpakete für das System und Plugins
- Titel von Sammlungen, Feldtitel und Beschriftungen von Feldoptionen
- Menütitel

Nach der Synchronisierung listet das System alle übersetzbaren Begriffe für die aktuelle Sprache auf.

<img src="https://static-docs.nocobase.com/202404202136567.png"/>

:::info{title=Hinweis}
Verschiedene Module können dieselben Originalbegriffe enthalten, die separat übersetzt werden müssen.
:::
