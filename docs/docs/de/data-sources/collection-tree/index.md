# Baumsammlung

## Einführung

Eine Baumstruktur-Sammlung ist ein Entwurfsmuster für Datensammlungen, das zum Organisieren hierarchischer Daten verwendet wird. Diese Sammlungsstruktur ähnelt einem Baum, wobei jedes Datenelement ein oder mehrere untergeordnete Elemente haben kann und diese untergeordneten Elemente wiederum eigene Nachfolger haben können.
