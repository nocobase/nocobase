# Dateisammlung

## Einführung

Dateien sind Datensätze in einer speziell strukturierten Sammlung, die als Dateisammlung bezeichnet wird. Sie dient zur Speicherung von Dateimetadaten und kann über den Dateimanager verwaltet werden.
