# Sammlungsselektor

Das Feld Sammlungsselektor ist ein spezieller Feldtyp, der es Benutzern ermöglicht, eine oder mehrere Sammlungen aus einer vordefinierten Liste auszuwählen. Dieser Feldtyp ist sehr nützlich in Szenarien, in denen Sie dynamisch auf verschiedene Sammlungen verweisen müssen, beispielsweise beim Erstellen konfigurierbarer Berichte, Dashboards oder komplexer Anwendungen, die ihre Datenquellen basierend auf der Benutzerauswahl ändern müssen.

## Anwendungsfälle

### Dynamische Berichtserstellung

In einem Modul zur Berichtserstellung müssen Benutzer möglicherweise eine Sammlung aus mehreren Sammlungen als Datenquelle für den Bericht auswählen. Durch die Verwendung des Sammlungsselektors können Benutzer die gewünschte Sammlung einfach auswählen, und das System generiert den Bericht dynamisch basierend auf der Auswahl des Benutzers.

### Konfigurierbare Dashboards

In einer Dashboard-Anwendung können Benutzer die auf dem Dashboard angezeigten Daten anpassen. Der Sammlungsselektor ermöglicht es Benutzern, die Sammlungen auszuwählen, an denen sie interessiert sind, wodurch Diagramme und statistische Daten aus verschiedenen Datenquellen auf dem Dashboard angezeigt werden.

### Flexible Datenimport-/Export-Tools

Beim Entwickeln eines generischen Tools für den Datenimport oder -export können Sie den Sammlungsselektor verwenden, um Benutzer die Sammlung auswählen zu lassen, mit der gearbeitet werden soll. Auf diese Weise kann dasselbe Tool verwendet werden, um Daten aus jeder Sammlung im System zu verarbeiten, was die Wiederverwendbarkeit des Tools erheblich verbessert.
