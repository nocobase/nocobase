# Externe Datenquelle - PostgreSQL

## Einführung

Verwenden Sie eine externe PostgreSQL-Datenbank als Datenquelle. Aktuell werden folgende Versionen unterstützt: PostgreSQL >= 9.5

## Installation

Dieses Plugin ist ein kommerzielles Plugin, das über den Plugin-Manager hochgeladen und aktiviert werden muss.
