# Benachrichtigung: WeCom

## Einführung

Das **WeCom** Plugin ermöglicht der Anwendung, Benachrichtigungen an WeCom-Benutzer zu senden.

## WeCom-Authentifikator hinzufügen und konfigurieren

Zuerst müssen Sie in NocoBase einen WeCom-Authentifikator hinzufügen und konfigurieren. Beachten Sie hierzu [Benutzerauthentifizierung - WeCom](/auth-verification/auth-wecom). Nur Systembenutzer, die sich über WeCom angemeldet haben, können Systembenachrichtigungen über WeCom empfangen.

## WeCom-Benachrichtigungskanal hinzufügen

![](https://static-docs.nocobase.com/202412041522365.png)

## WeCom-Benachrichtigungskanal konfigurieren

Wählen Sie den soeben konfigurierten Authentifikator aus.

![](https://static-docs.nocobase.com/202412041525284.png)

## Konfiguration des Workflow-Benachrichtigungsknotens

Wählen Sie den konfigurierten WeCom-Benachrichtigungskanal aus. Er unterstützt drei Nachrichtentypen: Textkarte, Markdown und Vorlagenkarte.

![](https://static-docs.nocobase.com/202412041529319.png)
