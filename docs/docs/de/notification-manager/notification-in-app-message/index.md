# Benachrichtigung: In-App-Nachricht

## Einführung

Es ermöglicht Ihnen, Nachrichtenbenachrichtigungen in Echtzeit direkt in der NocoBase-Anwendung zu empfangen.

## Installation

Dieses Plugin ist bereits integriert und erfordert keine zusätzliche Installation.

## Einen In-App-Nachrichtenkanal hinzufügen

Öffnen Sie die Benachrichtigungsverwaltung, klicken Sie auf die Schaltfläche „Neu hinzufügen“ und wählen Sie „In-App-Nachricht“.
![2024-11-08-08-33-26-20241108083326](https://static-docs.nocobase.com/2024-11-08-08-33-26-20241108083326.png)

Geben Sie den Kanalnamen und die Beschreibung ein und klicken Sie anschließend auf „Absenden“.
![2024-11-08-08-34-32-20241108083431](https://static-docs.nocobase.com/2024-11-08-08-34-32-20241108083431.png)

Ein entsprechender Kanal wird nun in der Liste hinzugefügt.

![2024-11-08-08-34-52-20241108083452](https://static-docs.nocobase.com/2024-11-08-08-34-52-20241108083452.png)

## Anwendungsbeispiel

Um Ihnen die Verwendung von In-App-Nachrichten besser zu veranschaulichen, finden Sie hier ein Beispiel zur „Nachverfolgung von Marketing-Leads“.

Stellen Sie sich vor, Ihr Team führt eine wichtige Marketingkampagne durch, um Rückmeldungen und Bedürfnisse potenzieller Kunden zu verfolgen. Mit In-App-Nachrichten können Sie:

**Benachrichtigungskanal erstellen**: Konfigurieren Sie zunächst in der Benachrichtigungsverwaltung einen In-App-Nachrichtenkanal mit dem Namen „Marketing Clue“, um sicherzustellen, dass die Teammitglieder dessen Zweck klar erkennen können.

![2024-11-08-08-34-32-20241108083431](https://static-docs.nocobase.com/2024-11-08-08-34-32-20241108083431.png)

**Workflow konfigurieren**: Erstellen Sie einen Workflow, der automatisch eine Benachrichtigung auslöst, wenn ein neuer Marketing-Lead generiert wird. Sie können dem Workflow einen Benachrichtigungsknoten hinzufügen, den zuvor erstellten Kanal „Marketing Clue“ auswählen und den Nachrichteninhalt nach Bedarf konfigurieren. Zum Beispiel:

![image-1-2024-10-27-14-07-17](https://static-docs.nocobase.com/image-1-2024-10-27-14-07-17.png)

**Echtzeit-Benachrichtigungen empfangen**: Sobald der Workflow ausgelöst wird, erhalten alle relevanten Personen Benachrichtigungen in Echtzeit, wodurch sichergestellt wird, dass das Team schnell reagieren und handeln kann.

![image-2-2024-10-27-14-07-22](https://static-docs.nocobase.com/image-2-2024-10-27-14-07-22.png)

**Nachrichtenverwaltung und -verfolgung**: In-App-Nachrichten werden nach ihrem Kanalnamen gruppiert. Sie können Nachrichten nach dem Status „Gelesen“ oder „Ungelesen“ filtern, um wichtige Informationen schnell anzuzeigen. Durch Klicken auf die Schaltfläche „Anzeigen“ werden Sie zur konfigurierten Linkseite weitergeleitet, um die entsprechenden Angelegenheiten weiter zu bearbeiten.

![20241027140648-2024-10-27-14-06-51-2024-10-29-13-26-41](https://static-docs.nocobase.com/20241027140648-2024-10-27-14-06-51-2024-10-29-13-26-41.png)
