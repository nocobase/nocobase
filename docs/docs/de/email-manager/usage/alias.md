# E-Mail-Alias

## Funktionsbeschreibung

Mit einem E-Mail-Alias können Sie unter einem einzigen E-Mail-Konto mit unterschiedlichen Absenderidentitäten E-Mails versenden.

Beim Verfassen einer E-Mail können Sie aus dem Absender-Auswahlfeld die Hauptadresse oder eine bereits synchronisierte Alias-Adresse auswählen. Bei Antworten, Weiterleitungen und der Wiederherstellung von Entwürfen behält das System die ursprüngliche Absenderidentität bei.

![](https://static-docs.nocobase.com/%E5%8F%91%E9%80%81%E9%82%AE%E4%BB%B6-04-02-2026_06_02_PM.png)

> Outlook unterstützt diese Funktion nicht.

## Alias-Synchronisierung

Nach erfolgreicher Autorisierung eines Gmail-Kontos synchronisiert das System automatisch die unter diesem Konto verfügbaren Alias-Adressen.

Wenn Sie später in Gmail Aliase hinzufügen oder anpassen, können Sie in den E-Mail-Einstellungen unter „Absendername" auf „Alias synchronisieren" klicken, um sie erneut abzurufen.

![](https://static-docs.nocobase.com/Email-settings-04-02-2026_06_04_PM.png)

## Alias beim Versand auswählen

Klicken Sie im E-Mail-Editor auf das Absender-Auswahlfeld, um die Hauptadresse sowie alle unter diesem Konto synchronisierten Aliase zu sehen.

Wenn dieselbe Alias-Adresse mehreren Konten zugeordnet ist, zeigt die Auswahl zusätzlich die jeweils zugehörige Hauptadresse an, damit Sie eindeutig erkennen, in welchem Kontokontext Sie sich befinden.

![](https://static-docs.nocobase.com/%E5%8F%91%E9%80%81%E9%82%AE%E4%BB%B6-04-02-2026_06_02_PM.png)
