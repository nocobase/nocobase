# E-Mail-Signatur

## Signatur konfigurieren

Gehen Sie zum E-Mail-Verwaltungszentrum und klicken Sie auf die Schaltfläche Einstellungen:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_05_34_PM.png)

Wechseln Sie zum Signatur-Panel:
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_05_51_PM.png)

Hier können Sie Ihre Signaturen verwalten. Aktivieren Sie die Option vor einer Signatur, um sie als Standardsignatur festzulegen:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_05_37_PM.png)

## Signaturen verwenden

Klicken Sie im E-Mail-Textfeld auf das Signatur-Symbol, um die gewünschte Signatur auszuwählen:
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_05_40_PM.png)
