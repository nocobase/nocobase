# E-Mail-Verwaltung

## Einführung
Das E-Mail-Verwaltungs-Plugin ist ein effizientes und praktisches Werkzeug. Es unterstützt die autorisierte Anbindung von Gmail- und Outlook-Postfächern und hilft Benutzern, Funktionen wie die E-Mail-Verwaltung sowie das Senden und Empfangen von E-Mails in verschiedene Blöcke und Seiten zu integrieren. Durch eine einfache Autorisierungskonfiguration können Sie mehrere Konten zentral verwalten und ein nahtloses E-Mail-Kommunikationserlebnis genießen.
