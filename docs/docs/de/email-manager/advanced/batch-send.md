# Massenversand

## Massenversand im Überblick
Sie können E-Mails an mehrere Personen gleichzeitig versenden. Dabei erhält jeder Empfänger eine separate E-Mail.  
![](https://static-docs.nocobase.com/email-manager/Loading--12-31-2025_10_15_PM.png)

## Massenversand über eine Tabelle
Sie können E-Mails auch im Massenversand über eine Tabelle versenden:

1. Fügen Sie der Tabelle einen „E-Mail senden“-Button hinzu.
   ![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_04_PM.png)

2. Konfigurieren Sie das E-Mail-Feld in der Tabelle (dieses wird verwendet, um E-Mail-Adressen nach der Auswahl von Tabellenzeilen zu ermitteln).  
   ![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_17_PM%20(1)

3. Wählen Sie die Tabellendaten aus, um E-Mails zu versenden.
   ![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_19_PM.png)

Wenn Sie E-Mails an alle Adressen der gesamten Tabelle senden möchten, können Sie die Funktion **Alle Datensätze der aktuellen Tabelle standardmäßig als Empfänger auswählen** aktivieren:  
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_18_PM%20(1)

Nachdem Sie den E-Mail-Inhalt bearbeitet haben, klicken Sie auf den „Massenversand“-Button, um die E-Mails zu versenden:  
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_19_PM.png)
