# Entwürfe automatisch speichern

Im Menü der Schaltfläche **Entwurf speichern** können Sie die Option **Entwurf automatisch speichern** aktivieren.

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_29_PM.png)

Während der E-Mail-Bearbeitung speichert das System den Entwurf in Echtzeit, sodass keine Inhalte verloren gehen.

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_31_PM.png)

Wenn Sie den E-Mail-Editor erneut öffnen (für denselben Absender und Empfänger), fragt das System, ob Sie den gespeicherten Entwurf verwenden möchten.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_08_35_PM.png)
