# Benutzer

## Benutzerverwaltung

Dieses Plugin bietet eine Benutzeroberfläche zur Benutzerverwaltung im Konfigurationszentrum. Nach der Initialisierung der Anwendung wird automatisch ein nicht löschbarer Super-Administrator-Account erstellt. Dieser Super-Administrator verfügt über Root-Berechtigungen und kann auf alle Ressourcen zugreifen.

![](https://static-docs.nocobase.com/44bf40f56b45d4dd96c424fb08082cf6.png)

Darüber hinaus können Sie auf der Seite verschiedene Blöcke der Benutzer-Sammlung hinzufügen, um Benutzer zu verwalten, zum Beispiel Tabellenblöcke:

![](https://static-docs.nocobase.com/76b5a4652f869541a9e8f18a4568a7c9.png)
