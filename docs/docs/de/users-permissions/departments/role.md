# Abteilungsrollenverwaltung

Indem Sie einer Abteilung eine oder mehrere Rollen zuweisen, erhalten die Mitglieder dieser Abteilung automatisch diese Rollen.

Es gibt zwei Möglichkeiten, Rollen mit einer Abteilung zu verknüpfen.

Die erste Möglichkeit besteht darin, im Bearbeitungsformular der Abteilung im Feld „Rolle“ die gewünschte Rolle auszuwählen.

![](https://static-docs.nocobase.com/70f77bb89aa1fb415c152a51a51cc23b.png)

Die zweite Möglichkeit ist, in der Rollenverwaltung die entsprechenden Abteilungen zu einer Rolle hinzuzufügen.

![](https://static-docs.nocobase.com/f2a7bec937cf2f179ce868a92b98416d.png)

Abteilungen, die diese Rolle bereits besitzen, können nicht ausgewählt werden.

![](https://static-docs.nocobase.com/be10299893581e1f97a4e01ddd5c7e59.png)
