# China Region Feld

## Einführung

Das China Region Feld Plugin (`field-china-region`) fügt den Feldtyp „China Region“ zu NocoBase hinzu.

## Installation

Gehen Sie in der Hauptoberfläche von NocoBase zum „Plugin Manager“, suchen Sie das Plugin „China Region“ und aktivieren Sie es.

## Verwendung

Fügen Sie in der Konfiguration der Sammlung eine neues Feld hinzu und wählen Sie „China Region“ als Feldtyp aus.

![](/images/field-china-region-1.png)

### Feldkonfiguration

In der Feldkonfiguration können Sie den Anzeigemodus für das Feld festlegen:

- Provinz
- Provinz/Stadt
- Provinz/Stadt/Bezirk
- Provinz/Stadt/Bezirk/Kreis

![](/images/field-china-region-2.png)
