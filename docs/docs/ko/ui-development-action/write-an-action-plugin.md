# NocoBase의 컬렉션과 워크플로우



## 컬렉션이란 무엇인가요?

NocoBase에서 **컬렉션**은 데이터를 저장하고 구성하는 기본적인 단위입니다. 데이터베이스의 테이블과 같다고 생각하시면 됩니다. 각 컬렉션은 데이터 구조를 정의하는 일련의 필드를 포함하고 있습니다.

-   **컬렉션 생성하기**:
    -   '데이터 관리' 페이지로 이동합니다.
    -   '새 컬렉션' 버튼을 클릭합니다.
    -   컬렉션 이름과 필드를 정의합니다.

## 워크플로우란 무엇인가요?

**워크플로우**는 NocoBase에서 비즈니스 로직을 자동화하는 강력한 도구입니다. 특정 이벤트가 발생했을 때 자동으로 실행될 일련의 작업을 정의할 수 있도록 해줍니다.

예를 들어, 특정 컬렉션에 새로운 데이터 항목이 추가될 때 알림 이메일을 보내도록 워크플로우를 구성할 수 있습니다.

-   **워크플로우 예시**:
    -   `사용자` 컬렉션에 새 사용자가 생성될 때.
    -   자동으로 환영 이메일을 보냅니다.
    -   `사용자 상태` 필드를 업데이트합니다.

## 컬렉션과 워크플로우의 결합

컬렉션과 워크플로우는 종종 함께 작동합니다. 컬렉션은 데이터를 제공하고, 워크플로우는 데이터를 처리하거나 데이터 변경에 응답합니다.
