# 인증기 관리

## 사용자 인증 관리

사용자 인증 플러그인을 설치하면, 사용자의 사용자 이름과 이메일을 기반으로 하는 `비밀번호` 인증 방식이 초기화됩니다.

![](https://static-docs.nocobase.com/66eaa9d5421c9cb713b117366bd8a5d5.png)

## 인증기 활성화

![](https://static-docs.nocobase.com/7f1b8f8ca5de67ffc68eff0a366bd8a5d5.png)

활성화된 인증 유형만 로그인 페이지에 표시됩니다.

![](https://static-docs.nocobase.com/8375a36ef98417af0f0977f1e07345dd.png)

## 사용자 인증 유형

![](https://static-docs.nocobase.com/da4250c0cea343ebe470cbf7be4b12e4.png)

다양한 유형의 인증기를 추가하여 시스템에 해당하는 인증 방식을 설정할 수 있습니다.

기존 플러그인에서 제공하는 인증 유형 외에도, 개발자는 사용자 인증 유형을 직접 확장할 수 있습니다. 자세한 내용은 [개발자 가이드](./dev/)를 참조하십시오.
