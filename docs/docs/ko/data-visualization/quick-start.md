# 빠른 시작

차트를 처음부터 끝까지 설정하는 방법을 안내합니다. 여기서는 필수적인 기본 기능만 다루며, 선택적인 고급 기능은 다음 장에서 자세히 설명합니다.

사전 준비:
- 데이터 소스와 컬렉션(데이터 테이블)이 설정되어 있어야 하며, 읽기 권한이 있어야 합니다.

## 차트 블록 추가

페이지 디자이너에서 '블록 추가'를 클릭하고 '차트'를 선택하여 차트 블록을 추가합니다.

![clipboard-image-1761554593](https://static-docs.nocobase.com/clipboard-image-1761554593.png)

블록을 추가한 후, 블록 오른쪽 상단에 있는 '설정'을 클릭합니다.

![clipboard-image-1761554709](https://static-docs.nocobase.com/clipboard-image-1761554709.png)

오른쪽에 차트 설정 패널이 열립니다. 이 패널은 데이터 쿼리, 차트 옵션, 이벤트의 세 부분으로 구성됩니다.

![clipboard-image-1761554848](https://static-docs.nocobase.com/clipboard-image-1761554848.png)

## 데이터 쿼리 설정
'데이터 쿼리' 패널에서 데이터 소스, 쿼리 필터 조건 등을 설정할 수 있습니다.

- 데이터 소스 및 컬렉션 선택
  - '데이터 쿼리' 패널에서 쿼리의 기본이 될 데이터 소스와 컬렉션을 선택합니다.
  - 컬렉션을 선택할 수 없거나 비어 있는 경우, 먼저 컬렉션이 생성되었는지와 현재 사용자에게 권한이 있는지 확인하세요.

- 측정값(Measures) 설정
  - 하나 이상의 숫자 필드를 측정값으로 선택합니다.
  - 각 측정값에 대해 합계(Sum) / 개수(Count) / 평균(Avg) / 최대값(Max) / 최소값(Min) 중 하나로 집계를 설정합니다.

- 차원(Dimensions) 설정
  - 하나 이상의 필드를 그룹화 차원(예: 날짜, 카테고리, 지역 등)으로 선택합니다.
  - 날짜/시간 필드는 `YYYY-MM`, `YYYY-MM-DD`와 같은 형식으로 설정하여 일관된 방식으로 표시할 수 있습니다.

![clipboard-image-1761555060](https://static-docs.nocobase.com/clipboard-image-1761555060.png)

필터, 정렬, 페이지네이션과 같은 다른 조건들은 선택 사항입니다.

## 쿼리 실행 및 데이터 보기

- '쿼리 실행'을 클릭하면 데이터를 요청하고 미리 보기 차트가 렌더링됩니다(왼쪽 페이지에서 직접 미리 볼 수 있습니다).
- '데이터 보기'를 클릭하여 반환된 데이터 결과를 미리 볼 수 있으며, 테이블/JSON 형식으로 전환할 수 있습니다. 다시 클릭하면 데이터 미리 보기가 닫힙니다.
- 데이터 결과가 비어 있거나 예상과 다른 경우, 쿼리 패널로 돌아가 컬렉션 권한, 측정값/차원의 필드 매핑 및 데이터 유형을 확인하세요.

![clipboard-image-1761555228](https://static-docs.nocobase.com/clipboard-image-1761555228.png)

## 차트 옵션 설정

'차트 옵션' 패널에서 차트 유형을 선택하고 차트 옵션을 설정할 수 있습니다.

- 먼저 차트 유형(꺾은선/영역, 막대/가로 막대, 원형/도넛, 분산형 등)을 선택합니다.
- 핵심 필드 매핑을 완료합니다:
  - 꺾은선/영역/막대/가로 막대: `xField`(차원), `yField`(측정값), `seriesField`(계열, 선택 사항)
  - 원형/도넛: `Category`(분류 차원), `Value`(측정값)
  - 분산형: `xField`, `yField`(두 개의 측정값 또는 차원)
  - 더 많은 차트 유형 및 설정은 ECharts 문서 [Axis](https://echarts.apache.com/handbook/en/concepts/axis)를 참조하세요.
- 이전에 '쿼리 실행'을 클릭하면 필드 매핑이 기본적으로 자동으로 완료됩니다. 차원/측정값을 변경한 후에는 매핑을 다시 확인해 주세요.

![clipboard-image-1761555586](https://static-docs.nocobase.com/clipboard-image-1761555586.png)

## 미리 보기 및 저장
설정 변경 사항은 기본적으로 실시간으로 미리 보기에 자동으로 업데이트되며, 왼쪽 페이지에서 차트를 확인할 수 있습니다. 하지만 '저장' 버튼을 클릭하기 전까지는 모든 변경 사항이 실제로 저장되지 않는다는 점에 유의하세요.

하단의 버튼을 클릭하여 다음 작업을 수행할 수도 있습니다:

- 미리 보기: 설정 변경 사항은 자동으로 실시간 미리 보기를 새로 고칩니다. 하단의 '미리 보기' 버튼을 클릭하여 수동으로 새로 고칠 수도 있습니다.
- 취소: 현재 변경된 설정을 원하지 않는다면, 하단의 '취소' 버튼을 클릭하거나 페이지를 새로 고쳐 이번 변경 사항을 취소하고 마지막으로 저장된 상태로 되돌릴 수 있습니다.
- 저장: '저장'을 클릭하면 현재의 모든 쿼리 및 차트 설정이 데이터베이스에 실제로 저장되어 모든 사용자에게 적용됩니다.

![clipboard-image-1761555803](https://static-docs.nocobase.com/clipboard-image-1761555803.png)

## 일반적인 유의 사항

- 최소 유효 설정: 컬렉션과 최소 하나 이상의 측정값을 선택해야 합니다. 그룹화하여 표시하려면 차원을 추가하는 것이 좋습니다.
- 날짜 차원의 경우, 가로축이 불연속적이거나 혼란스러워지는 것을 방지하기 위해 적절한 형식(예: 월별 통계는 `YYYY-MM`)으로 설정하는 것이 좋습니다.
- 쿼리가 비어 있거나 차트가 표시되지 않는 경우:
  - 컬렉션/권한 및 필드 매핑을 확인하세요.
  - '데이터 보기'에서 열 이름과 유형이 차트 매핑과 일치하는지 확인하세요.
- 미리 보기는 임시 상태입니다: 유효성 검사 및 조정에만 사용되며, '저장'을 클릭해야만 공식적으로 적용됩니다.
