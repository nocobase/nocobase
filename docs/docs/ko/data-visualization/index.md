# 개요

NocoBase의 데이터 시각화 플러그인은 시각적인 데이터 쿼리 기능과 다양한 차트 컴포넌트를 제공합니다. 사용자는 간단한 설정만으로 시각화 대시보드를 빠르게 구축하고, 데이터 인사이트를 효과적으로 보여주며, 다차원 데이터 분석 및 표시를 지원할 수 있습니다.

![clipboard-image-1761749573](https://static-docs.nocobase.com/clipboard-image-1761749573.png)

## 기본 개념
- 차트 블록: 페이지 내에서 설정 가능한 차트 컴포넌트로, 데이터 쿼리, 차트 옵션 및 인터랙션 이벤트를 지원합니다.
- 데이터 쿼리 (Builder / SQL): Builder를 통해 시각적으로 설정하거나, SQL을 작성하여 데이터를 가져옵니다.
- 측정값 (Measures) 및 차원 (Dimensions): 측정값은 숫자 집계에 사용되며, 차원은 데이터 그룹화(예: 날짜, 카테고리, 지역)에 사용됩니다.
- 필드 매핑: 쿼리 결과 열을 `xField`, `yField`, `seriesField` 또는 `Category / Value`와 같은 핵심 차트 필드에 매핑합니다.
- 차트 옵션 (Basic / Custom): Basic은 일반적인 속성을 시각적으로 설정하며, Custom은 JS를 통해 전체 ECharts `option`을 반환합니다.
- 쿼리 실행: 설정 패널에서 쿼리를 실행하여 데이터를 요청하고, Table / JSON으로 전환하여 반환된 데이터를 확인할 수 있습니다.
- 미리보기 및 저장: 미리보기는 임시 효과이며, "저장"을 클릭하면 설정이 데이터베이스에 기록되고 공식적으로 적용됩니다.
- 컨텍스트 변수: 페이지, 사용자, 필터 등 컨텍스트 정보(예: `{{ ctx.user.id }}`)를 쿼리 및 차트 설정에 재사용합니다.
- 페이지 필터 및 연동: 페이지 수준의 "필터 블록"은 통합된 입력 조건을 수집하고, 이를 차트 쿼리에 자동으로 병합하여 연동된 차트를 새로 고칩니다.
- 인터랙션 이벤트: `chart.on`을 통해 이벤트를 등록하여 하이라이트, 이동, 드릴다운 등의 동작을 구현합니다.

## 설치
데이터 시각화는 NocoBase의 내장 플러그인으로, 별도의 설치 없이 바로 사용할 수 있습니다.
