# 자주 묻는 질문

## 차트 선택 관련 질문
### 어떤 차트를 선택해야 할까요?
답변: 데이터 목표에 따라 선택하세요:
- 추세 및 변화: 꺾은선형 차트 또는 영역형 차트
- 값 비교: 세로 막대형 차트 또는 가로 막대형 차트
- 구성 비율: 원형 차트 또는 도넛형 차트
- 상관관계 및 분포: 분산형 차트
- 계층 구조 및 단계별 진행: 깔때기형 차트

더 많은 차트 유형은 [ECharts 예시](https://echarts.apache.org/examples)를 참고해 주세요.

### NocoBase는 어떤 차트 유형을 지원하나요?
답변: 시각화 설정 모드에서는 자주 사용되는 차트(꺾은선형, 영역형, 세로 막대형, 가로 막대형, 원형, 도넛형, 깔때기형, 분산형 등)를 기본으로 제공합니다. 사용자 정의 설정 모드에서는 ECharts의 모든 차트 유형을 사용할 수 있습니다.

## 데이터 쿼리 관련 질문
### 시각화 설정과 SQL 설정은 서로 연동되나요?
답변: 아니요, 서로 연동되지 않으며 설정이 독립적으로 저장됩니다. 마지막으로 저장된 설정 모드가 적용됩니다.

## 차트 옵션 관련 질문
### 차트 필드는 어떻게 설정해야 하나요?
답변: 시각화 설정 모드에서 차트 유형에 따라 해당 데이터 필드를 선택합니다. 예를 들어, 꺾은선형/세로 막대형 차트에는 X축 및 Y축 필드를 설정해야 하며, 원형 차트에는 분류 필드와 값 필드를 설정해야 합니다.
먼저 "쿼리 실행"을 클릭하여 데이터가 예상과 일치하는지 확인하는 것이 좋습니다. 기본적으로 차트 필드는 자동으로 매칭됩니다.

## 미리보기 및 저장 관련 질문
### 설정을 변경한 후 수동으로 미리보기를 해야 하나요?
답변: 시각화 설정 모드에서는 설정을 변경하면 자동으로 미리보기가 됩니다. SQL 및 사용자 정의 설정 모드에서는 잦은 새로고침을 방지하기 위해 작성을 완료한 후 "미리보기"를 수동으로 클릭해야 합니다.

### 팝업을 닫으면 차트 미리보기가 사라지나요?
답변: 미리보기는 임시 확인용입니다. 설정을 변경한 후에는 먼저 저장한 다음 닫아주세요. 저장되지 않은 변경 사항은 유지되지 않습니다.
