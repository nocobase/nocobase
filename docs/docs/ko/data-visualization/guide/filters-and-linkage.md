# 페이지 필터 및 연동

페이지 필터(필터 블록)는 페이지 수준에서 필터 조건을 통합적으로 입력하고, 이를 차트 쿼리에 병합하여 여러 차트에서 일관된 필터링 및 연동을 구현하는 데 사용됩니다.

## 기능 개요
- 페이지에 "필터 블록"을 추가하여 현재 페이지의 모든 차트에 통합된 필터링 진입점을 제공합니다.
- "필터", "초기화", "접기" 버튼을 통해 필터를 적용하고, 조건을 초기화하며, 블록을 접을 수 있습니다.
- 필터에서 차트와 연결된 필드를 선택하면, 해당 값은 자동으로 차트 쿼리 요청에 병합되어 차트 새로고침을 트리거합니다.
- 필터는 사용자 정의 필드를 생성하고 컨텍스트 변수에 등록할 수 있으며, 이를 차트, 테이블, 폼 등 다른 데이터 블록에서 참조할 수 있습니다.

## 차트 쿼리에서 페이지 필터 값 사용
- 빌더 모드 (권장)
  - 자동 병합: 동일한 데이터 소스 및 컬렉션인 경우, 차트 쿼리에 별도로 변수를 작성할 필요 없이 페이지 필터가 `$and`로 병합됩니다.
  - 수동 선택: 필터 조건에서 페이지 필터의 "사용자 정의 필드" 값을 직접 선택할 수도 있습니다.

- SQL 모드 (변수 주입을 통해)
  - SQL 문에서 "변수 선택"을 사용하여 페이지 필터의 "사용자 정의 필드" 값을 삽입합니다.
