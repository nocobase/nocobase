# 테마 편집기

> 현재 테마 기능은 Ant Design 5.x 버전을 기반으로 구현되었습니다. 이 문서를 읽기 전에 [테마 사용자 지정](https://ant.design/docs/react/customize-theme-cn#%E8%87%AA%E5%AE%9A%E4%B9%89%E4%B8%BB%E9%A2%98) 관련 개념을 먼저 이해하시는 것을 권장합니다.

## 소개

테마 편집기 플러그인은 전체 프런트엔드 페이지의 스타일을 수정하는 데 사용됩니다. 현재 전역 [SeedToken](https://ant.design/docs/react/customize-theme-cn#seedtoken), [MapToken](https://ant.design/docs/react/customize-theme-cn#maptoken), [AliasToken](https://ant.design/docs/react/customize-theme-cn#aliastoken) 편집을 지원하며, `다크 모드` 및 `컴팩트 모드`로 [전환](https://ant.design/docs/react/customize-theme-cn#%E4%BD%BF%E7%94%A8%E9%A2%84%E8%AE%BE%E7%AE%97%E6%B3%95)할 수 있습니다. 향후에는 [컴포넌트 수준](https://ant.design/docs/react/customize-theme-cn#%E4%BF%AE%E6%94%B9%E7%BB%84%E4%BB%B6%E5%8F%98%E9%87%8F-component-token)의 테마 사용자 지정도 지원될 수 있습니다.

## 사용 방법

### 테마 편집기 플러그인 활성화하기

먼저 NocoBase를 최신 버전(v0.11.1 이상)으로 업데이트합니다. 그런 다음 플러그인 관리 페이지에서 `테마 편집기` 카드를 검색하고, 카드의 오른쪽 하단에 있는 `활성화` 버튼을 클릭한 후 페이지가 새로 고쳐질 때까지 기다립니다.

![20240409132838](https://static-docs.nocobase.com/20240409132838.png)

### 테마 설정 페이지로 이동하기

플러그인을 활성화한 후, 카드의 왼쪽 하단에 있는 설정 버튼을 클릭하여 테마 편집 페이지로 이동합니다. 기본적으로 `기본 테마`, `다크 테마`, `컴팩트 테마`, `컴팩트 다크 테마`의 네 가지 테마 옵션이 제공됩니다.

![20240409133020](https://static-docs.nocobase.com/20240409133020.png)

### 새 테마 추가하기

`새 테마 추가` 버튼을 클릭하고 `완전히 새로운 테마 생성`을 선택합니다. 페이지 오른쪽에 테마 편집기가 나타나 `색상`, `크기`, `스타일` 등의 옵션을 편집할 수 있습니다. 편집을 마친 후 테마 이름을 입력하고 저장을 클릭하면 새 테마 추가가 완료됩니다.

![20240409133147](https://static-docs.nocobase.com/20240409133147.png)

### 새 테마 적용하기

페이지 오른쪽 상단으로 마우스를 이동하면 테마 전환 항목을 볼 수 있습니다. 이를 클릭하여 방금 추가한 테마와 같은 다른 테마로 전환할 수 있습니다.

![20240409133247](https://static-docs.nocobase.com/20240409133247.png)

### 기존 테마 편집하기

카드의 왼쪽 하단에 있는 `편집` 버튼을 클릭합니다. 페이지 오른쪽에 테마 편집기가 나타나며(새 테마 추가와 동일), 편집을 마친 후 저장을 클릭하면 테마 수정이 완료됩니다.

![20240409134413](https://static-docs.nocobase.com/20240409134413.png)

### 사용자가 선택할 수 있는 테마 설정하기

새로 추가된 테마는 기본적으로 사용자가 전환할 수 있습니다. 특정 테마로 사용자가 전환하는 것을 원하지 않는다면, 테마 카드 오른쪽 하단에 있는 `사용자 선택 가능` 스위치를 끄면 해당 테마로 전환할 수 없게 됩니다.

![20240409133331](https://static-docs.nocobase.com/20240409133331.png)

### 기본 테마로 설정하기

초기 상태에서는 `기본 테마`가 기본 테마입니다. 특정 테마를 기본 테마로 설정하려면 해당 테마 카드 오른쪽 하단에 있는 `기본 테마` 스위치를 켜십시오. 이렇게 하면 사용자가 페이지를 처음 열 때 해당 테마를 보게 됩니다. 참고: 기본 테마는 삭제할 수 없습니다.

![20240409133409](https://static-docs.nocobase.com/20240409133409.png)

### 테마 삭제하기

카드 하단에 있는 `삭제` 버튼을 클릭한 다음, 팝업되는 확인 대화 상자에서 확인을 클릭하면 테마가 삭제됩니다.

![20240409133435](https://static-docs.nocobase.com/20240409133435.png)
