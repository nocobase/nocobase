# 브랜드 사용자 지정

## 소개

시스템의 기본 브랜드 이름은 "NocoBase"이며, 주로 다음 세 곳에 표시됩니다:

![20240409111100](https://static-docs.nocobase.com/20240409111100.png)

로그인 페이지 하단:

![20240409111254](https://static-docs.nocobase.com/20240409111254.png)

위치 1은 [시스템 설정](/system-management/system-settings/)의 로고(Logo) 옵션을 통해 설정할 수 있습니다. 위치 2와 3은 이 플러그인을 사용하여 설정할 수 있습니다.

## 사용 방법

플러그인을 활성화한 후, 페이지 오른쪽 상단의 "브랜드 사용자 지정" 메뉴를 클릭하여 설정 페이지로 이동합니다.

![20240409113640](https://static-docs.nocobase.com/20240409113640.png)

- **브랜드**: 로그인 페이지 하단의 텍스트(위치 3)를 설정하는 데 사용됩니다.
- **정보**: 페이지 오른쪽 상단 메뉴의 내용(아래 이미지 참조)을 설정하는 데 사용됩니다.

![20240409113915](https://static-docs.nocobase.com/20240409113915.png)

설정은 HTML 형식을 지원하여 내용을 유연하게 사용자 지정할 수 있습니다. 간단한 텍스트 변경만 필요한 경우, 아래 이미지와 같이 기존 내용을 직접 수정할 수 있습니다:

![20240409114740](https://static-docs.nocobase.com/20240409114740.png)
