# 언어 설정

"시스템 설정"에서 "활성화된 언어"를 설정하여 시스템의 전역 언어 환경을 조정할 수 있습니다. 목록의 첫 번째 언어는 기본 언어로 사용됩니다.

![20240324192005](https://static-docs.nocobase.com/20240324192005.png)

여러 언어가 활성화된 경우, 사용자는 개인 센터(페이지 오른쪽 상단)에서 현재 사용자의 개인 언어 기본 설정을 지정할 수 있습니다.

![20240324192509](https://static-docs.nocobase.com/20240324192509.png)

현재 사용 중인 언어 환경에 번역이 누락된 경우, "[로컬라이제이션 관리](/system-management/localization/)" 플러그인을 사용하여 번역을 추가하거나 수정할 수 있습니다.

![20240324193055](https://static-docs.nocobase.com/20240324193055.png)
