# 플러그인 관리

## 플러그인 관리자

페이지 오른쪽 상단에 있는 플러그인 관리자 아이콘을 클릭하면 플러그인 관리자 페이지로 이동합니다. 이 페이지에서 플러그인을 확인하고, 설치하고, 활성화하거나 비활성화할 수 있습니다.

## 플러그인 설정 센터

플러그인에는 일반적으로 자체 설정 페이지가 있습니다. 페이지 오른쪽 상단에 있는 설정 아이콘을 클릭하면 모든 플러그인의 설정 메뉴를 확인할 수 있습니다. 해당 메뉴 항목을 클릭하여 플러그인을 구성할 수 있습니다.
