# 변수

## 소개

변수는 현재 컨텍스트에서 특정 값을 식별하는 데 사용되는 일련의 토큰입니다. 이 변수들은 블록 데이터 범위 설정, 필드 기본값, 연동 규칙, 워크플로우 등 다양한 상황에서 활용할 수 있습니다.

![20251030114458](https://static-docs.nocobase.com/20251030114458.png)

## 현재 지원되는 변수

### 현재 사용자

현재 로그인한 사용자의 데이터를 나타냅니다.

![20240416154950](https://static-docs.nocobase.com/20240416154950.png)

### 현재 역할

현재 로그인한 사용자의 역할 식별자(role name)를 나타냅니다.

![20240416155100](https://static-docs.nocobase.com/20240416155100.png)

### 현재 폼

현재 폼의 값으로, 폼 블록에서만 사용됩니다. 활용 사례는 다음과 같습니다.

- 현재 폼의 연동 규칙
- 폼 필드의 기본값 (새 데이터를 추가할 때만 유효합니다)
- 관계 필드의 데이터 범위 설정
- 제출 작업의 필드 값 할당 설정

#### 현재 폼의 연동 규칙

![20251027114920](https://static-docs.nocobase.com/20251027114920.png)

#### 폼 필드 기본값 (새 폼에만 해당)

![20251027115016](https://static-docs.nocobase.com/20251027115016.png)

#### 관계 필드의 데이터 범위 설정

상위 필드를 기반으로 하위 필드의 옵션을 동적으로 필터링하여 정확한 데이터 입력을 보장하는 데 사용됩니다.

**예시:**

1. 사용자가 **Owner** 필드의 값을 선택합니다.
2. 시스템은 선택된 Owner의 **userName**을 기반으로 **Account** 필드의 옵션을 자동으로 필터링합니다.

![20251030151928](https://static-docs.nocobase.com/20251030151928.png)

### 현재 레코드

레코드는 컬렉션의 행을 의미하며, 각 행은 하나의 레코드를 나타냅니다. 표시 유형 블록의 **행 작업 연동 규칙**에는 모두 '현재 레코드' 변수가 포함되어 있습니다.

예시: '결제 완료'된 문서의 삭제 버튼을 비활성화합니다.

![20251027120217](https://static-docs.nocobase.com/20251027120217.png)

### 현재 팝업 레코드

팝업 작업은 NocoBase 인터페이스 설정에서 매우 중요한 역할을 합니다.

- 행 작업의 팝업: 각 팝업에는 현재 행 레코드를 나타내는 '현재 팝업 레코드' 변수가 있습니다.
- 관계 필드의 팝업: 각 팝업에는 현재 클릭된 관계 레코드를 나타내는 '현재 팝업 레코드' 변수가 있습니다.

팝업 내의 블록은 모두 '현재 팝업 레코드' 변수를 사용할 수 있으며, 관련 활용 사례는 다음과 같습니다.

- 블록의 데이터 범위 설정
- 관계 필드의 데이터 범위 설정
- 필드의 기본값 설정 (새 데이터 추가용 폼)
- 작업의 연동 규칙 설정

### URL 쿼리 파라미터

이 변수는 현재 페이지 URL에 있는 쿼리 파라미터를 나타냅니다. 페이지 URL에 쿼리 문자열이 있을 때만 이 변수를 사용할 수 있습니다. [링크 작업](/interface-builder/actions/types/link)과 함께 사용하면 더욱 편리합니다.

![20251027173017](https://static-docs.nocobase.com/20251027173017.png)

![20251027173121](https://static-docs.nocobase.com/20251027173121.png)

### API 토큰

이 변수의 값은 NocoBase API에 접근하기 위한 자격 증명 문자열입니다. 사용자의 신원을 확인하는 데 사용할 수 있습니다.
