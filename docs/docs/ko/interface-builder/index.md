## WYSIWYG (위지윅)

NocoBase는 WYSIWYG(위지윅) 방식의 화면 구성 모드를 제공합니다. 화면 구성 버튼을 클릭하여 설정 모드와 사용 모드 사이를 전환할 수 있습니다.

## 레이아웃 템플릿

NocoBase는 기본 레이아웃 템플릿을 제공합니다. 이 템플릿은 상단과 왼쪽에 내비게이션 영역이 있고, 오른쪽에 콘텐츠 영역이 있습니다.

## 구성 옵션

화면 구성 모드에 진입하면, 화면에 주황색으로 표시된 구성 가능한 요소들이 나타납니다. 일반적으로 각 구성 가능한 요소의 설정 진입점은 해당 요소의 오른쪽 상단에 있습니다. 거의 모든 요소는 화면에서 실시간으로 구성하고 그 효과를 바로 확인할 수 있습니다. 다음은 몇 가지 예시입니다.
