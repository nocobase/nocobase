# 필드

UI에서 필드는 필드 컴포넌트이며, 단위 데이터를 담는 역할을 합니다. 데이터 유형에 따라 다양한 필드 컴포넌트로 표시됩니다. 필드는 블록에만 종속될 수 있으며, 독립적으로 사용할 수 없습니다.

## 블록 내 필드

필드는 일반적으로 단독으로 사용되지 않고, 데이터 유형 블록의 하위 요소로 존재합니다. 데이터 유형 블록에는 보통 "필드 구성" 옵션이 있으며, 필드 목록은 현재 컬렉션에서 제공됩니다.

![20251029204240](https://static-docs.nocobase.com/20251029204240.png)

## 필드 디자이너 (도구 모음)

블록과 마찬가지로, 필드 컴포넌트의 오른쪽 상단에도 세 가지 아이콘이 있습니다. 각 아이콘은 다음과 같습니다.

- 드래그하여 레이아웃 조정
- [이벤트 흐름](/interface-builder/event-flow)
- 구성

![20251025165432](https://static-docs.nocobase.com/20251025165432.png)

## 필드 레이아웃

드래그 앤 드롭으로 블록 내 필드의 레이아웃을 사용자 지정할 수 있습니다.

![20251025165605](https://static-docs.nocobase.com/20251025165605.gif)

## 필드 구성 옵션

일반적인 필드 구성 옵션 외에도, 필드 유형에 따라 특정 구성 옵션을 지원합니다.
