# 드롭다운 선택기

## 소개

드롭다운 선택기는 대상 컬렉션의 기존 데이터를 선택하여 연결하거나, 새 데이터를 추가한 후 연결하는 기능을 지원합니다. 드롭다운 옵션은 유사 검색을 지원합니다.

![20251029205901](https://static-docs.nocobase.com/20251029205901.png)

## 필드 설정

### 데이터 범위 설정

드롭다운 목록의 데이터 범위를 제어합니다.

![20251029210025](https://static-docs.nocobase.com/20251029210025.png)

더 자세한 내용은 [데이터 범위 설정](/interface-builder/fields/field-settings/data-scope)을 참고하세요.

### 정렬 규칙 설정

드롭다운 선택기 데이터의 정렬 방식을 제어합니다.

예시: 서비스 날짜를 기준으로 내림차순 정렬합니다.

![20251029210105](https://static-docs.nocobase.com/20251029210105.png)

### 여러 항목 추가/연결 허용

다중 관계 데이터에서 하나의 데이터만 연결하도록 제한합니다.

![20251029210145](https://static-docs.nocobase.com/20251029210145.png)

### 제목 필드

제목 필드는 옵션으로 표시되는 레이블 필드입니다.

![20251029210507](https://static-docs.nocobase.com/20251029210507.gif)

> 제목 필드를 기반으로 빠른 검색을 지원합니다.

더 자세한 내용은 [제목 필드](/interface-builder/fields/field-settings/title-field)를 참고하세요.

### 빠른 생성: 먼저 데이터를 추가한 후 선택

![20251125220046](https://static-docs.nocobase.com/20251125220046.png)

#### 드롭다운 메뉴를 통한 추가

대상 컬렉션에 새 데이터를 생성하면 자동으로 해당 데이터가 선택되고, 폼 제출 시 연결됩니다.

아래 예시에서 주문 컬렉션은 다대일 관계 필드 "Account"를 가지고 있습니다.

![20251125220447](https://static-docs.nocobase.com/20251125220447.gif)

#### 모달 창을 통한 추가

모달 창을 통한 추가는 더 복잡한 데이터 입력 시나리오에 적합하며, 새 레코드 생성을 위한 맞춤형 폼을 구성할 수 있습니다.

![20251125220607](https://static-docs.nocobase.com/20251125220607.gif)
