# 파일 관리자

## 소개

파일 관리자는 대상 컬렉션이 파일 컬렉션인 관계를 처리하는 데 사용되는 관계형 필드 컴포넌트입니다.

![20251027194124](https://static-docs.nocobase.com/20251027194124.png)

## 필드 설정

- [필드 컴포넌트](/interface-builder/fields/association-field): 드롭다운, 데이터 선택기 등 다른 관계형 필드 컴포넌트로 전환합니다.
- 기존 파일 선택 허용: 기본적으로 활성화되며, 이미 업로드된 파일 중에서 선택하여 연결할 수 있습니다.
