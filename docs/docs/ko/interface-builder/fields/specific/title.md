# 제목

## 소개

읽기 전용 상태에서 관계 필드는 기본적으로 **제목 필드 컴포넌트**를 사용하여 표시됩니다. 이 컴포넌트는 제목 필드의 값을 표시하여 현재 연결된 레코드를 식별합니다.
시스템은 제목 필드의 유형에 따라 해당 제목 필드 컴포넌트를 자동으로 매칭하고 렌더링합니다.

### 클릭하여 열기 활성화 (기본값: 활성화)

클릭하면 팝업 창이 열리며, 여기에서 데이터 관리를 위한 블록을 사용자 정의할 수 있습니다.

![20251028160742](https://static-docs.nocobase.com/20251028160742.png)
