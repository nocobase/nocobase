# 서브폼

## 소개

서브폼은 관련 데이터를 먼저 생성한 다음 연결하는 시나리오에 적합합니다. 다단계 관련 데이터는 중첩된 폼 형태로 명확하게 표시됩니다. 데이터 선택기 및 드롭다운 컴포넌트와 비교하여, 서브폼은 현재 페이지 블록 내에서 관련 컬렉션의 필드를 직접 관리할 수 있게 해주며, 관련 데이터는 메인 폼과 함께 제출됩니다.
