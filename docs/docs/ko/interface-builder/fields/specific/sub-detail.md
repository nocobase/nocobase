# 서브 디테일

## 소개

서브 디테일은 읽기 모드에서 서브 폼에 해당하는 컴포넌트입니다. 레이블 및 제목 컴포넌트와 비교하여, 서브 디테일은 현재 컬렉션의 데이터를 더 많이 표시할 수 있을 뿐만 아니라, 관련 컬렉션의 데이터 정보도 표시하도록 설정할 수 있습니다. 다단계 관계 데이터는 중첩된 디테일 형태로 명확하게 표시됩니다.
