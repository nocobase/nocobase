# 폼 필드

## 소개

폼의 다양한 필드 유형은 각기 다른 설정 옵션을 가집니다. 사용자는 플러그인을 통해 필드의 설정 옵션을 확장하여 더욱 개인화된 요구 사항을 충족할 수 있습니다.
