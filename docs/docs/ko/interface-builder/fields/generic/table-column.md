# 테이블 필드

## 소개
테이블 필드는 필드 너비, 필드 제목, 정렬 등을 조정하는 기능 외에도, 날짜 필드, 관계 필드, 숫자 필드와 같은 특정 필드의 경우 더욱 개인화된 표시 설정을 지원합니다.
![20251024174558](https://static-docs.nocobase.com/20251024174558.png)

## 필드 설정 옵션

### 날짜 필드 형식 지정
![20251024175303](https://static-docs.nocobase.com/20251024175303.png)
더 자세한 내용은 [날짜 형식 지정](/interface-builder/fields/specific/date-picker)을 참고해 주세요.

### 숫자 필드 형식 지정
간단한 단위 변환, 천 단위 구분 기호, 접두사 및 접미사, 정밀도, 과학적 표기법을 지원합니다.
![20251024175445](https://static-docs.nocobase.com/20251024175445.png)
더 자세한 내용은 [숫자 형식 지정](/interface-builder/fields/field-settings/number-format)을 참고해 주세요.

### 빠른 편집 활성화
빠른 편집을 활성화하면 해당 열에 마우스를 올렸을 때 편집 버튼이 나타납니다. 버튼을 클릭하여 팝업 창에서 데이터를 빠르게 편집하고 저장할 수 있습니다.
![20251025171158](https://static-docs.nocobase.com/20251025171158.gif)

### 클릭하여 열기 활성화
관계 필드에서 클릭하여 팝업을 여는 기능을 지원하는 것 외에도, 일반 필드에서도 '클릭하여 열기'를 활성화하여 팝업을 여는 진입점으로 사용할 수 있습니다.
![20251025172308](https://static-docs.nocobase.com/20251025172308.gif)

### 콘텐츠 오버플로 표시 방식
열 콘텐츠가 테이블 열 너비를 초과할 경우, 오버플로 표시 방식을 설정할 수 있습니다.
- 생략하여 표시 (기본값)
- 줄 바꿈
![20251025172549](https://static-docs.nocobase.com/20251025172549.png)

### 필드 컴포넌트
일부 필드는 다양한 표시 형태를 지원합니다. 사용자는 필드 컴포넌트를 전환하여 다양한 표시 효과를 구현하고 여러 시나리오의 요구 사항을 충족할 수 있습니다. 예를 들어, **URL** 타입 필드는 **Preview** 컴포넌트로 전환하여 링크 콘텐츠나 이미지 미리보기를 더 효과적으로 표시할 수 있습니다.
![20251025171658](https://static-docs.nocobase.com/20251025171658.png)

### 표시 스타일
- 태그
- 텍스트
![20251025172723](https://static-docs.nocobase.com/20251025172723.png)

### 정렬 가능
현재 대부분의 필드 타입은 서버 측 정렬을 지원합니다. 정렬을 활성화하면 대상 필드를 기준으로 오름차순/내림차순으로 데이터를 정렬할 수 있습니다.
