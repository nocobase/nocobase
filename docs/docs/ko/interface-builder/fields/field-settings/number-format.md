# 숫자 서식

## 소개

숫자 서식은 읽기 모드에서 숫자 유형 필드(수식 필드 포함)를 시각적으로 보기 좋게 만들고 표준화하는 작업을 말합니다. 여기에는 천 단위 구분 기호, 소수점 이하 자릿수 정밀도, 단위 변환, 과학적 표기법 등의 설정이 포함됩니다. 숫자 서식을 설정하면 다양한 지역 및 산업의 숫자 표현 방식을 충족시킬 수 있습니다.

## 사용 방법

- 간단한 단위 변환(곱셈/나눗셈)을 지원합니다.
- 천 단위 구분 기호
- 접두사 및 접미사
- 정밀도
- 과학적 표기법

예시: 주문 금액을 서식 지정하여 통화 기호(예: 위안화 기호 ￥)를 추가하고, 천 단위 구분 기호(예: 쉼표 또는 마침표)와 소수점 정밀도를 설정하여 금액을 명확하고 읽기 쉽게 만들 수 있습니다.
