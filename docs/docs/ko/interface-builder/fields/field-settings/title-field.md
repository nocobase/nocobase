# 제목 필드

## 소개

**제목 필드**는 일반적으로 관계 필드 컴포넌트에서 해당 필드와 연결된 데이터가 인터페이스에 표시될 때 식별자로 사용됩니다. 이 필드는 데이터의 주요 정보가 어떻게 표시될지 결정하며, 주로 사용자가 데이터를 빠르게 식별하고 이해하도록 돕는 역할을 합니다.

다양한 유형의 제목 필드는 각기 다른 필드 컴포넌트에 해당하며, 그에 맞는 설정 옵션을 지원합니다. 예를 들어:

- 제목 필드가 **날짜 필드**인 경우, 필드 컴포넌트는 **날짜 컴포넌트**를 사용하여 해당 데이터를 표시합니다.
- 제목 필드가 **텍스트 필드**인 경우, 필드 컴포넌트는 **텍스트 컴포넌트**를 사용하여 해당 데이터를 표시합니다.
- 제목 필드가 **옵션 필드**인 경우, 필드 컴포넌트는 **옵션 컴포넌트**를 사용하여 해당 데이터를 표시합니다.
