# 데이터 범위 설정

## 소개

관계 필드의 데이터 범위 설정은 블록의 데이터 범위 설정과 유사합니다. 관계 데이터에 대한 기본 필터 조건을 설정하는 데 사용됩니다.

## 사용 방법

![20251028211328](https://static-docs.nocobase.com/20251028211328.png)

### 정적 값

예시: 삭제되지 않은 상품만 연결할 수 있습니다.

> 필드 목록은 관계 필드의 대상 컬렉션 필드입니다.

![20251028211434](https://static-docs.nocobase.com/20251028211434.png)

### 변수 값

예시: 서비스 날짜가 주문 날짜보다 늦은 상품만 연결할 수 있습니다.

![20251028211727](https://static-docs.nocobase.com/20251028211727.png)

변수에 대한 더 자세한 내용은 [변수](/interface-builder/variables)를 참고하세요.

### 관계 필드 연동

관계 필드 간의 연동은 데이터 범위를 설정하여 구현됩니다.

예시: 주문 컬렉션에 일대다 관계 필드 '기회 제품'과 다대일 관계 필드 '기회'가 있습니다. 기회 제품 컬렉션에는 다대일 관계 필드 '기회'가 있습니다. 주문 폼 블록에서 '기회 제품'의 선택 가능한 데이터는 현재 폼에서 선택된 '기회'와 연결된 기회 제품으로 필터링됩니다.
