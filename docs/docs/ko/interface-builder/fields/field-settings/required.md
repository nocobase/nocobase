# 필수

## 소개

필수 항목은 폼(form) 유효성 검사에서 자주 사용되는 규칙입니다. 필드 설정에서 직접 필수 항목으로 지정할 수 있으며, 폼의 연동 규칙을 통해 필드를 동적으로 필수 항목으로 설정할 수도 있습니다.

## 필수 항목은 어디에서 설정할 수 있나요?

### 컬렉션 필드 설정

컬렉션 필드를 필수 항목으로 설정하면 백엔드 유효성 검사가 트리거되며, 프런트엔드에서도 기본적으로 필수 항목으로 표시됩니다 (수정 불가).
![20251025175418](https://static-docs.nocobase.com/20251025175418.png)

### 필드 설정

필드를 직접 필수 항목으로 설정합니다. 이 설정은 사용자 이름, 비밀번호와 같이 사용자가 항상 입력해야 하는 필드에 적합합니다.
![20251028222818](https://static-docs.nocobase.com/20251028222818.png)

### 연동 규칙

폼 블록의 필드 연동 규칙을 통해 조건에 따라 필수 항목을 설정할 수 있습니다.

예시: 주문 날짜가 비어 있지 않을 때 주문 번호는 필수 항목입니다.
