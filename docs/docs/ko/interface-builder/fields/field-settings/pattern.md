# 표시 모드

## 소개

- 편집 가능

### 관계 필드의 표시 모드

**관계 필드의 표시 모드**는 해당 필드가 인터페이스에서 어떻게 표시될지, 그리고 어떤 필드 컴포넌트 유형을 사용할 수 있을지를 결정합니다.

**편집 가능 상태**에서 관계 필드는 다양한 컴포넌트 유형을 지원하므로, 사용자는 비즈니스 요구사항에 따라 다른 관계 필드 컴포넌트를 선택하여 관련 데이터를 표시하거나 선택할 수 있습니다.

#### 편집 가능 상태의 관계 필드 컴포넌트

![20251028220447](https://static-docs.nocobase.com/20251028220447.png)

이 상태에서는 사용자가 데이터를 더욱 효율적으로 처리할 수 있도록 적절한 표시 방식을 유연하게 선택할 수 있습니다.

#### 읽기 전용 상태의 관계 필드 컴포넌트

**읽기 전용 상태**로 전환하면, 시스템은 관련 데이터를 표시하기 위해 자동으로 **제목 필드 컴포넌트**를 기본값으로 사용합니다. 이는 데이터를 수정하지 않고 단순히 보기만 하면 되는 상황에 적합합니다.
