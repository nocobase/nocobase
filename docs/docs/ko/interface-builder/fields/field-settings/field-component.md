# 필드 컴포넌트

## 소개

필드 컴포넌트는 필드 내용을 표시하는 데 사용됩니다.
일부 필드에는 여러 가지 표시 컴포넌트가 내장되어 있습니다. 사용자는 실제 필요에 따라 적절한 컴포넌트를 선택하여 필드 값을 다양한 방식으로 나타낼 수 있습니다.

예시: 읽기 모드에서 URL 필드는 미리보기 컴포넌트 또는 URL 컴포넌트(기본값)로 전환할 수 있습니다.
