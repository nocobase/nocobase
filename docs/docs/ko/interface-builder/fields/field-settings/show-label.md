# 레이블 표시

## 소개

블록 내 필드는 기본적으로 필드 레이블을 표시하여 각 필드의 의미를 식별하고, 사용자가 데이터 내용을 더 쉽게 읽고 이해할 수 있도록 돕습니다.

특정 상황에서는 (예를 들어, 정보 표시 전용 페이지이거나 필드의 의미가 이미 매우 명확한 경우) 레이블을 숨겨 인터페이스를 더 깔끔하게 만들 수 있습니다. 레이블을 숨기면 필드 값만 표시되며, 필드 이름은 더 이상 나타나지 않습니다.
