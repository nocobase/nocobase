# 작업 연동 규칙

## 소개

작업 연동 규칙을 사용하면 특정 조건에 따라 작업의 상태(예: 표시, 활성화, 숨기기, 비활성화 등)를 동적으로 제어할 수 있습니다. 이러한 규칙을 설정하여 작업 버튼의 동작을 현재 레코드, 사용자 역할 또는 컨텍스트 데이터와 연동할 수 있습니다.

![20251029150224](https://static-docs.nocobase.com/20251029150224.png)

## 사용 방법

조건이 충족되면 (조건이 없으면 기본적으로 통과), 속성 설정 또는 JavaScript 실행이 트리거됩니다. 조건 판단 시 상수와 변수를 사용할 수 있습니다.

![20251030224601](https://static-docs.nocobase.com/20251030224601.png)

버튼 속성 수정도 지원합니다.

![20251029150452](https://static-docs.nocobase.com/20251029150452.png)
