# 버튼 편집

## 소개

시스템에서 버튼의 이름, 아이콘, 배경색, 버튼 유형을 사용자 정의할 수 있습니다. 이는 유연한 UI 사용자 정의 옵션을 제공하여, 사용자가 특정 요구 사항에 맞춰 버튼의 표시 스타일과 기능을 조정할 수 있도록 지원합니다.

- **버튼 이름**: 사용자는 비즈니스 요구 사항에 따라 버튼 텍스트를 사용자 정의하여, 버튼 레이블을 명확하고 직관적으로 설정할 수 있습니다.
- **버튼 아이콘**: 버튼에 아이콘을 설정하여 시각적 식별성을 높이고, 사용자가 버튼의 기능을 빠르게 파악할 수 있도록 돕습니다.
- **위험 작업**: 버튼이 **위험 작업**(예: 삭제, 초기화 등)을 나타낼 때, **Danger Action** 유형으로 설정할 수 있습니다.
- **버튼 유형(Button Type)**: 버튼은 여러 가지 유형으로 설정할 수 있으며, 일반적인 버튼 유형은 다음과 같습니다.
  - **기본 유형(Default)**: 일반적인 작업에 적합한 표준 버튼입니다.
  - **주요 유형(Primary)**: 중요하거나 주요한 작업에 사용되며, 일반적으로 눈에 띄는 시각적 효과를 가집니다.
  - **점선 유형(Dashed)**: 점선 테두리가 있는 버튼으로, 보조 작업이나 힌트성 버튼에 적합합니다.
  - **텍스트 유형(Text)**: 배경색 없이 텍스트만 있는 버튼으로, 일반적으로 가벼운 작업이나 링크에 사용됩니다.
