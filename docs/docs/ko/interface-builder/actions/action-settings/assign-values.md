# 필드 할당

## 소개

필드 할당은 작업을 실행하기 전에 필드 값을 미리 설정하는 것을 의미합니다. 레코드를 업데이트하거나 생성할 때 이 데이터가 자동으로 채워지므로 수동으로 데이터를 입력할 필요가 없습니다. 예를 들어, 주문 상태를 일괄 업데이트하거나 주문이 배송될 때 배송 시간을 현재 시간으로 설정하는 등의 시나리오에서 활용할 수 있습니다.
