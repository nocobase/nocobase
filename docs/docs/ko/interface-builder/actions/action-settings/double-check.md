# 재확인

## 소개

재확인 설정은 작업 실행 여부를 확인하는 데 사용됩니다. 작업을 클릭하면 확인 대화 상자가 나타납니다. 삭제 버튼은 실수로 인한 데이터 손실이나 되돌릴 수 없는 결과를 방지하기 위해 기본적으로 재확인 기능이 활성화되어 있습니다.
