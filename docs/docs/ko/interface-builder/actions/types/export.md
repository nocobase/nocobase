# 내보내기

## 소개

내보내기 기능은 필터링된 레코드를 **Excel** 형식으로 내보낼 수 있으며, 내보낼 필드를 설정할 수 있도록 지원합니다. 사용자는 필요에 따라 내보낼 필드를 선택하여 이후 데이터 분석, 처리 또는 보관에 활용할 수 있습니다. 이 기능은 데이터 작업의 유연성을 높여주며, 특히 데이터를 다른 플랫폼으로 옮기거나 추가 처리가 필요한 경우에 유용합니다.

### 주요 기능:
- **필드 선택**: 사용자가 내보낼 필드를 설정하고 선택하여 내보내는 데이터의 정확성과 간결성을 보장합니다.
- **Excel 형식 지원**: 내보낸 데이터는 표준 Excel 파일로 저장되어 다른 데이터와 쉽게 통합 및 분석할 수 있습니다.

이 기능을 통해 사용자는 업무의 핵심 데이터를 쉽게 내보내 외부에서 활용할 수 있으며, 이를 통해 작업 효율성을 높일 수 있습니다.

### 내보내기 가능 필드

- 첫 번째 레벨: 현재 컬렉션의 모든 필드입니다.
- 두 번째 레벨: 관계 필드인 경우, 연결된 컬렉션의 필드를 선택해야 합니다.
- 세 번째 레벨: 총 세 개의 레벨까지만 처리하며, 마지막 레벨의 관계 필드는 표시되지 않습니다.

![20251029171557](https://static-docs.nocobase.com/20251029171557.png)

- [연동 규칙](/interface-builder/actions/action-settings/linkage-rule): 버튼을 동적으로 표시하거나 숨깁니다.
- [버튼 편집](/interface-builder/actions/action-settings/edit-button): 버튼의 제목, 유형, 아이콘을 편집합니다.
