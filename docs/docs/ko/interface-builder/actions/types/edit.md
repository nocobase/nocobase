# 편집

## 소개

편집 액션은 현재 레코드를 컨텍스트로 사용하는 행 버튼입니다. 팝업 형태로 나타나며, 사용자는 팝업 내에서 블록을 직접 설정하고 구성할 수 있습니다.

![20251029165442](https://static-docs.nocobase.com/20251029165442.png)

필요에 따라 블록을 구성합니다.

![20251029165514](https://static-docs.nocobase.com/20251029165514.png)

## 액션 설정

![20251029165620](https://static-docs.nocobase.com/20251029165620.png)

- [연동 규칙](/interface-builder/actions/action-settings/linkage-rule): 버튼을 동적으로 표시/숨깁니다.
- [편집 버튼](/interface-builder/actions/action-settings/edit-button): 버튼의 제목, 유형, 아이콘을 편집합니다.
- [편집 팝업](/interface-builder/actions/action-settings/edit-popup): 열기 방식, 크기를 설정합니다.
