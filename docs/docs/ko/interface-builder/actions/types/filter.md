# 필터

## 소개

"필터" 작업은 주로 데이터 블록에서 다양한 조건을 설정하여 데이터를 필터링하는 데 사용됩니다. 여기서 중요한 점은, 만약 해당 블록에 이미 데이터 범위가 설정되어 있다면, 필터 작업의 조건은 데이터 범위의 조건과 합쳐져 최종 필터링 조건으로 적용된다는 것입니다.

## 작업 설정

![20251029173708](https://static-docs.nocobase.com/20251029173708.png)

### 필터링 가능한 필드

기본적으로 해당 컬렉션의 모든 필드가 활성화됩니다. 활성화된 필드만 조건 필드 목록에서 선택할 수 있습니다.

관계형 컬렉션의 필드도 필터 필드로 사용할 수 있습니다.
