# 모달 열기

## 소개

**모달 열기** 작업은 보기 및 편집 작업과 유사합니다. 사용자는 테이블의 행 또는 헤더에 **모달 열기** 버튼을 구성할 수 있습니다. 버튼을 클릭하면 해당 모달이 나타나고, 이 모달 내의 블록은 현재 컨텍스트 레코드에 따라 데이터를 표시합니다.

![20251029182351](https://static-docs.nocobase.com/20251029182351.png)

모달에서 필요에 따라 블록을 구성합니다.

![20251029190208](https://static-docs.nocobase.com/20251029190208.png)

## 작업 설정

![20251029190352](https://static-docs.nocobase.com/20251029190352.png)

- [버튼 편집](/interface-builder/actions/action-settings/edit-button): 버튼의 제목, 유형, 아이콘을 편집합니다.
- [연동 규칙](/interface-builder/linkage-rule): 버튼의 상태(표시/비활성화)를 동적으로 제어합니다.
- [모달 편집](/interface-builder/actions/action-settings/edit-popup): 열기 방식, 크기를 설정합니다.
