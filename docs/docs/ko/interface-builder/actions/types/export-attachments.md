# 첨부 파일 내보내기

## 소개

첨부 파일 내보내기는 첨부 파일 관련 필드를 압축 파일 형식으로 내보내는 것을 지원합니다.

#### 첨부 파일 내보내기 설정

![20251029173251](https://static-docs.nocobase.com/20251029173251.png)

![20251029173425](https://static-docs.nocobase.com/20251029173425.png)

![20251029173345](https://static-docs.nocobase.com/20251029173345.png)

- 내보낼 첨부 파일 필드를 설정합니다. 다중 선택을 지원합니다.
- 각 레코드마다 폴더를 생성할지 여부를 선택할 수 있습니다.

파일 이름 지정 규칙:

- 각 레코드마다 폴더를 생성하도록 선택하면, 파일 이름 지정 규칙은 다음과 같습니다: `{레코드의 제목 필드 값}/{첨부 파일 필드 이름}[-{파일 순서 번호}].{파일 확장자}`.
- 폴더를 생성하지 않도록 선택하면, 파일 이름 지정 규칙은 다음과 같습니다: `{레코드의 제목 필드 값}-{첨부 파일 필드 이름}[-{파일 순서 번호}].{파일 확장자}`.

첨부 파일 필드에 여러 첨부 파일이 있을 때 파일 순서 번호는 자동으로 생성됩니다.

- [연동 규칙](/interface-builder/actions/action-settings/linkage-rule): 버튼을 동적으로 표시/숨김;
- [버튼 편집](/interface-builder/actions/action-settings/edit-button): 버튼의 제목, 유형, 아이콘을 편집합니다;
