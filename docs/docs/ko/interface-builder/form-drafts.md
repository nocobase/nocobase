# 폼 초안

폼 초안 기능을 활성화하면 작성 중인 내용이 로컬에 자동으로 저장됩니다. 실수로 폼을 닫거나 페이지를 새로고침하더라도 입력한 데이터가 손실되지 않습니다.

![20251213093717](https://static-docs.nocobase.com/20251213093717.png)

초안이 존재할 경우 폼 상단에 알림 메시지가 표시됩니다. 제출이 성공하면 초안은 자동으로 삭제되며, 더 이상 필요하지 않은 경우 언제든지 수동으로 삭제할 수 있습니다.

![20251213092734](https://static-docs.nocobase.com/20251213092734.png)

::: tip 팁

현재 폼 초안은 로컬 캐시만 지원합니다. 향후 버전에서는 서버 사이드 캐시를 지원하여 여러 기기 간의 동기화가 가능해질 예정입니다.

:::
