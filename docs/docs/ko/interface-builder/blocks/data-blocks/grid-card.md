# 그리드 카드

## 소개

그리드 카드 블록은 데이터 레코드의 요약 정보를 카드 형태로 표시합니다. 다양한 화면 크기에 따라 열 개수를 설정할 수 있어, 여러 기기에서 사용자 친화적인 화면을 제공합니다.
