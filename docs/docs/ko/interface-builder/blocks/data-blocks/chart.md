# 차트

## 소개

NocoBase의 데이터 시각화 플러그인은 시각적인 데이터 쿼리 기능과 다양한 차트 컴포넌트를 제공합니다. 사용자는 간단한 설정만으로 시각화 대시보드를 빠르게 구축하여 데이터 인사이트를 시각적으로 보여주고, 다차원적인 데이터 분석 및 표현을 지원합니다.
