# 드래그 앤 드롭 정렬

## 소개

드래그 앤 드롭 정렬은 정렬 필드를 기반으로 하며, 블록 내의 레코드를 수동으로 정렬하는 데 사용됩니다.


:::info{title=팁}
* 동일한 정렬 필드를 여러 블록에서 드래그 앤 드롭 정렬에 사용할 경우, 기존 정렬 순서가 흐트러질 수 있습니다.
* 테이블 드래그 앤 드롭 정렬 시, 정렬 필드에 그룹화 규칙을 설정할 수 없습니다.
* 트리 테이블은 동일한 계층의 노드 정렬만 지원합니다.

:::


## 드래그 앤 드롭 설정

"정렬" 유형의 필드를 추가합니다. 컬렉션을 생성할 때 정렬 필드가 더 이상 자동으로 생성되지 않으므로, 수동으로 생성해야 합니다.

## 정렬 규칙 설명

현재 정렬 순서가 다음과 같다고 가정합니다.

```
[1,2,3,4,5,6,7,8,9]
```

특정 요소(예: 5)를 3의 위치로 앞으로 이동하면 3, 4, 5의 정렬 값만 변경됩니다. 5가 3의 위치를 차지하고, 3과 4는 각각 한 칸씩 뒤로 밀려납니다.

```
[1,2,5,3,4,6,7,8,9]
```

이때 다시 6을 8의 위치로 뒤로 이동하면, 6이 8의 위치를 차지하고 7과 8은 각각 한 칸씩 앞으로 당겨집니다.

```
[1,2,5,3,4,7,8,6,9]
```
