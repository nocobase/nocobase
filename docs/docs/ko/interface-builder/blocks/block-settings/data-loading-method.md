# 데이터 로딩 방식

## 소개

기본 데이터 로딩 방식은 "필터 조건이 비어 있을 때 모든 데이터 로드"입니다. 여기서 "필터 조건"은 「필터 블록」의 값을 의미합니다. 또 다른 옵션으로는 "필터 조건이 비어 있을 때 데이터를 로드하지 않음"을 선택할 수 있습니다.
