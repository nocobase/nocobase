# 데이터 범위 설정

## 소개

데이터 범위 설정은 데이터 블록에 기본 필터 조건을 정의하는 것을 의미합니다. 사용자는 비즈니스 요구사항에 따라 데이터 범위를 유연하게 조정할 수 있지만, 어떤 필터링 작업을 수행하더라도 시스템은 이 기본 필터 조건을 자동으로 적용하여 데이터가 항상 지정된 범위의 제한을 따르도록 보장합니다.

## 사용 가이드

![20251027110053](https://static-docs.nocobase.com/20251027110053.png)

필터 필드는 현재 컬렉션의 필드와 관계형 컬렉션의 필드를 선택하는 것을 지원합니다.

![20251027110140](https://static-docs.nocobase.com/20251027110140.png)

### 연산자

다양한 유형의 필드는 각기 다른 연산자를 지원합니다. 예를 들어, 텍스트 필드는 '같음', '같지 않음', '포함' 등의 연산자를 지원하며, 숫자 필드는 '보다 큼', '보다 작음' 등의 연산자를 지원하고, 날짜 필드는 '범위 내', '특정 날짜 이전' 등의 연산자를 지원합니다.

![20251027111124](https://static-docs.nocobase.com/20251027111124.png)
