# 레이아웃

## 소개

폼 블록, 상세 블록, 목록 블록과 같은 폼 형태의 블록은 레이아웃 설정을 지원합니다. 레이아웃 방식은 수직 레이아웃(레이블과 입력 필드가 줄별로 배치됩니다)과 수평 레이아웃(레이블과 입력 필드가 나란히 배치됩니다)으로 나뉩니다. 사용자는 특정 요구 사항에 따라 블록 필드 레이아웃을 조정하여 정보 입력 또는 조회가 명확하고 효율적으로 이루어지도록 할 수 있습니다.

기본 레이아웃은 수직입니다.

### 수평

수평 레이아웃은 레이블과 입력 필드를 나란히 표시하여 공간을 효율적으로 절약합니다. 제목이 긴 필드의 경우, 인터페이스의 깔끔함과 미관을 위해 생략 또는 줄 바꿈을 설정할 수 있습니다.

![20251024120237](https://static-docs.nocobase.com/20251024120237.png)

#### 필드 제목 정렬 방식

- **왼쪽 정렬** (기본값)
- **오른쪽 정렬**

#### 필드 제목 너비

- 기본 너비는 **120px**이며, 최소 너비는 **50px**로 제한됩니다.

#### 필드 제목이 너비를 초과할 경우 처리 방식

- **줄 바꿈**
- **생략**

#### 콜론 표시 여부

필드 제목 뒤에 콜론을 표시할지 여부를 선택할 수 있습니다.
