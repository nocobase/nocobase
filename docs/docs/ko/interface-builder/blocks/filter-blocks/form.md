# 필터 폼

## 소개

필터 폼을 사용하면 폼 필드를 채워 데이터를 필터링할 수 있습니다. 이 기능은 테이블 블록, 차트 블록, 목록 블록 등 다양한 곳에서 데이터를 필터링하는 데 활용할 수 있습니다.

## 사용 방법

필터 폼 사용법을 빠르게 이해하기 위해 간단한 예시를 살펴보겠습니다. 사용자 정보가 담긴 테이블 블록이 있고, 필터 폼을 사용하여 데이터를 필터링하고 싶다고 가정해 봅시다. 아래와 같습니다.

![20251031163036_rec_](https://static-docs.nocobase.com/20251031163036_rec_.gif)

설정 단계는 다음과 같습니다.

1. 설정 모드를 켜고, 페이지에 "필터 폼" 블록과 "테이블" 블록을 추가합니다.
![20251031163525_rec_](https://static-docs.nocobase.com/20251031163525_rec_.gif)
2. 테이블 블록과 필터 폼 블록 모두에 "닉네임" 필드를 추가합니다.
![20251031163932_rec_](https://static-docs.nocobase.com/20251031163932_rec_.gif)
3. 이제 필터 폼을 사용할 수 있습니다.
![20251031163036_rec_](https://static-docs.nocobase.com/20251031163036_rec_.gif)

## 고급 사용법

필터 폼 블록은 더 다양한 고급 설정을 지원합니다. 다음은 몇 가지 일반적인 사용법입니다.

### 여러 블록 연결하기

하나의 폼 필드로 여러 블록의 데이터를 동시에 필터링할 수 있습니다. 자세한 방법은 다음과 같습니다.

1. 필드의 "Connect fields" 설정 항목을 클릭합니다.
![20251031170300](https://static-docs.nocobase.com/20251031170300.png)
2. 연결할 대상 블록을 추가합니다. 여기서는 페이지에 있는 목록 블록을 선택하겠습니다.
![20251031170718](https://static-docs.nocobase.com/20251031170718.png)
3. 목록 블록에서 하나 이상의 필드를 선택하여 연결합니다. 여기서는 "닉네임" 필드를 선택합니다.
![20251031171039](https://static-docs.nocobase.com/20251031171039.png)
4. 저장 버튼을 클릭하여 설정을 완료합니다. 결과는 다음과 같습니다.
![20251031171209_rec_](https://static-docs.nocobase.com/20251031171209_rec_.gif)

### 사용자 정의 필드

컬렉션에서 필드를 선택하는 것 외에도 "사용자 정의 필드"를 통해 폼 필드를 생성할 수 있습니다. 예를 들어, 드롭다운 단일 선택 필드를 만들고 옵션을 직접 정의할 수 있습니다. 자세한 방법은 다음과 같습니다.

1. "사용자 정의 필드" 옵션을 클릭하여 설정 패널을 엽니다.
![20251031173833](https://static-docs.nocobase.com/20251031173833.png)
2. 필드 제목을 입력하고, 필드 모델에서 "Select"를 선택한 다음 옵션을 설정합니다.
![20251031174857_rec_](https://static-docs.nocobase.com/20251031174857_rec_.gif)
3. 새로 추가된 사용자 정의 필드는 대상 블록의 필드와 수동으로 연결해야 합니다. 방법은 다음과 같습니다.
![20251031181957_rec_](https://static-docs.nocobase.com/20251031181957_rec_.gif)
4. 설정이 완료되었습니다. 결과는 다음과 같습니다.
![20251031182235_rec_](https://static-docs.nocobase.com/20251031182235_rec_.gif)

현재 지원되는 필드 모델은 다음과 같습니다.

- Input: 한 줄 텍스트 입력
- Number: 숫자 입력
- Date: 날짜 선택기
- Select: 드롭다운 (단일 또는 다중 선택으로 설정 가능)
- Radio group: 라디오 버튼
- Checkbox group: 체크박스

### 접기

접기 버튼을 추가하여 필터 폼 내용을 접거나 펼쳐 페이지 공간을 절약할 수 있습니다.

![20251031182743](https://static-docs.nocobase.com/20251031182743.png)

지원되는 설정은 다음과 같습니다.

![20251031182849](https://static-docs.nocobase.com/20251031182849.png)

- **접었을 때 표시되는 행 수**: 접힌 상태에서 표시될 폼 필드의 행 수를 설정합니다.
- **기본값으로 접기**: 이 옵션을 활성화하면 필터 폼이 기본적으로 접힌 상태로 표시됩니다.
