# 텔레메트리 익스포터: HTTP

## 환경 변수 설정

### TELEMETRY_METRIC_READER

텔레메트리 지표 익스포터 유형입니다.

```bash
TELEMETRY_METRIC_READER=http
```

### TELEMETRY_HTTP_URL

텔레메트리 데이터 내보내기를 위한 HTTP URL입니다.

```bash
TELEMETRY_HTTP_URL=
```
