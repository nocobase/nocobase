# 파일 미리보기

파일 필드(첨부 파일 필드 포함)가 포함된 인터페이스에서는 파일 썸네일이나 아이콘을 클릭하여 파일을 미리 볼 수 있습니다. 내장된 미리보기 기능은 이미지, PDF 및 대부분의 브라우저에서 기본적으로 지원하는 파일 형식을 포함하여 다양한 파일 형식을 지원합니다.

![20251129232307](https://static-docs.nocobase.com/20251129232307.png)

기본 미리보기를 지원하지 않는 파일 형식의 경우, 해당 파일 미리보기 플러그인을 설치하거나 확장하여 미리보기 기능을 구현할 수 있습니다. 예를 들어, Office 파일 미리보기 플러그인을 설치하면 Word, Excel 및 PowerPoint 파일을 미리 볼 수 있습니다.

현재 NocoBase에서 제공하는 파일 미리보기 플러그인은 다음과 같습니다:

* [Office 파일 미리보기 플러그인](../file-preview/ms-office.md)
