# 스토리지 엔진: 알리윈 OSS

알리윈 OSS 기반의 스토리지 엔진입니다. 사용하기 전에 관련 계정과 권한을 준비해야 합니다.

## 구성 파라미터

![알리윈 OSS 스토리지 엔진 구성 예시](https://static-docs.nocobase.com/20240712220011.png)

:::info{title=참고}
이 섹션에서는 알리윈 OSS 스토리지 엔진의 전용 파라미터만 설명합니다. 일반적인 파라미터는 [엔진 공통 파라미터](./index#引擎通用参数)를 참고해 주십시오.
:::

### 리전

OSS 스토리지의 리전을 입력합니다. 예: `oss-cn-hangzhou`.

:::info{title=참고}
[알리윈 OSS 콘솔](https://oss.console.aliyun.com/)에서 스토리지 버킷의 리전 정보를 확인할 수 있으며, 전체 도메인 없이 리전 접두사 부분만 사용하면 됩니다.
:::

### AccessKey ID

알리윈 인증 접근 키의 ID를 입력합니다.

### AccessKey Secret

알리윈 인증 접근 키의 Secret을 입력합니다.

### 버킷

OSS 스토리지 버킷의 이름을 입력합니다.

### 타임아웃

알리윈 OSS에 업로드할 타임아웃 시간을 입력합니다. 단위는 밀리초(ms)이며, 기본값은 `60000` 밀리초(즉, 60초)입니다.
