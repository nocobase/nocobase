# 텐센트 클라우드 COS

텐센트 클라우드 COS 기반의 스토리지 엔진입니다. 사용하기 전에 관련 계정 및 권한을 준비해야 합니다.

## 설정 파라미터

![텐센트 COS 스토리지 엔진 설정 예시](https://static-docs.nocobase.com/20240712222125.png)

:::info{title=참고}
이 섹션에서는 텐센트 클라우드 COS 스토리지 엔진의 전용 파라미터만 설명합니다. 일반적인 파라미터는 [엔진 공통 파라미터](./index.md#엔진-공통-파라미터)를 참고해 주세요.
:::

### 리전

COS 스토리지의 리전을 입력합니다. 예: `ap-chengdu`.

:::info{title=참고}
[텐센트 클라우드 COS 콘솔](https://console.cloud.tencent.com/cos)에서 스토리지 공간의 리전 정보를 확인할 수 있습니다. 리전 접두사 부분만 사용하면 되며, 전체 도메인 이름은 필요하지 않습니다.
:::

### SecretId

텐센트 클라우드 접근 키의 ID를 입력합니다.

### SecretKey

텐센트 클라우드 접근 키의 Secret을 입력합니다.

### 버킷

COS 스토리지 버킷 이름을 입력합니다. 예: `qing-cdn-1234189398`.
