# 이메일 관리

## 소개
이메일 관리 플러그인은 Gmail 및 Outlook 이메일 계정 인증 및 연결을 지원하는 효율적이고 편리한 도구입니다. 이 플러그인을 통해 사용자는 이메일 관리, 이메일 송수신 등의 기능을 다양한 블록과 페이지에 통합할 수 있습니다. 간단한 인증 설정만으로 여러 계정을 통합 관리하고 원활한 이메일 통신 경험을 누릴 수 있습니다.
