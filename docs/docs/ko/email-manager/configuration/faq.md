# 자주 묻는 질문

## Microsoft 계정으로 인증 로그인 시 이메일 수신 불가

현재 Outlook 이메일 계정과 Gmail 이메일 계정만 인증 로그인을 지원합니다. Microsoft 계정과 Google 계정은 현재 지원되지 않습니다. 자세한 내용은 다음을 참조하세요: [answers.microsoft.com](https://answers.microsoft.com/zh-hans/outlook_com/forum/all/%E7%8E%B0%E6%9C%89%E5%BE%AE%E8%BD%AF%E8%B4%A6/dba12dda-a7c7-4346-8263-53f4a6d9dc68)

**팁**: 만약 본인이 '진정한 Outlook.com 이메일' 또는 'Gmail 이메일'을 가지고 있는지 확실하지 않다면, 웹 브라우저를 통해 Outlook.com 또는 Gmail.com에 접속하여 직접 로그인하고 정상적으로 이메일을 보낼 수 있는지 확인해 보세요. 만약 로그인이나 이메일 발송이 되지 않는다면, 해당 이메일 서비스를 소유하고 있지 않거나 아직 활성화되지 않았을 수 있습니다. 먼저 서비스를 활성화하거나 다른 이메일을 사용해야 합니다.

## 인증 해제

이메일 계정이 이미 인증되었지만, 삭제하거나 다시 인증하고 싶다면 다음 단계를 따르세요.

#### **Gmail**

1. `https://myaccount.google.com/u/0/connections`에 접속하여 로그인합니다.

![](https://static-docs.nocobase.com/mail-1735634131347.png)

2. 해당 앱을 클릭한 다음 삭제를 클릭합니다.

![](https://static-docs.nocobase.com/mail-1735634131518.png)

![](https://static-docs.nocobase.com/mail-1735634131697.png)

#### **Microsoft 이메일**

1. `https://account.microsoft.com/`에 접속하여 로그인합니다.
    
2. "내 데이터에 액세스할 수 있는 앱 및 서비스" 버튼을 클릭합니다.

![](https://static-docs.nocobase.com/mail-1735634131870.png)

3. 편집을 클릭한 다음 제거를 클릭합니다.

![](https://static-docs.nocobase.com/mail-1735634132052.png)
