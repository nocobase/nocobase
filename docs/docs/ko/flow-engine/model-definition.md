# 중국 지역 필드

## 소개

중국 지역 필드 플러그인(`field-china-region`)은 NocoBase에 '중국 지역' 필드 유형을 추가합니다.

## 설치

NocoBase 메인 화면에서 '플러그인 관리'로 이동하여 '중국 지역' 플러그인을 찾아 활성화합니다.

## 사용법

컬렉션 설정에서 새 필드를 추가하고 필드 유형으로 '중국 지역'을 선택합니다.

![](/images/field-china-region-1.png)

### 필드 설정

필드 설정에서 필드의 표시 모드를 설정할 수 있습니다:

- 성(省)
- 성/시
- 성/시/구
- 성/시/구/현

![](/images/field-china-region-2.png)
