# 감사 로그

## 소개

감사 로그는 시스템 내 사용자 활동과 리소스 작업 이력을 기록하고 추적하는 데 사용됩니다.

![](https://static-docs.nocobase.com/202501031627719.png)

![](https://static-docs.nocobase.com/202501031627922.png)

## 파라미터 설명

| 파라미터                  | 설명                                                             |
| --------------------- | ---------------------------------------------------------------- |
| **Resource**          | 작업 대상 리소스 유형                                               |
| **Action**            | 수행된 작업 유형                                                     |
| **User**              | 작업을 수행한 사용자                                                         |
| **Role**              | 사용자 작업 시 역할                                                   |
| **Data source**       | 데이터 소스                                                           |
| **Target collection** | 대상 컬렉션                                                       |
| **Target record UK**  | 대상 컬렉션의 고유 식별자                                              |
| **Source collection** | 관계 필드의 소스 컬렉션                                                 |
| **Source record UK**  | 소스 컬렉션의 고유 식별자                                         |
| **Status**            | 작업 요청 응답의 HTTP 상태 코드                                       |
| **Created at**        | 작업 시간                                                              |
| **UUID**              | 작업의 고유 식별자로, 작업 요청의 Request ID와 일치하며 애플리케이션 로그 검색에 사용될 수 있습니다. |
| **IP**                | 사용자의 IP 주소                                                  |
| **UA**                | 사용자 UA 정보                                              |
| **Metadata**          | 작업 요청의 파라미터, 요청 본문, 응답 내용 등 메타데이터                         |

## 감사 리소스 설명

현재 다음 리소스 작업이 감사 로그에 기록됩니다:

### 메인 애플리케이션

| 작업             | 설명         |
| ---------------- | ------------ |
| `app:resart`     | 애플리케이션 재시작     |
| `app:clearCache` | 애플리케이션 캐시 지우기 |

### 플러그인 관리자

| 작업         | 설명     |
| ------------ | -------- |
| `pm:add`     | 플러그인 추가 |
| `pm:update`  | 플러그인 업데이트 |
| `pm:enable`  | 플러그인 활성화 |
| `pm:disable` | 플러그인 비활성화 |
| `pm:remove`  | 플러그인 제거 |

### 사용자 인증

| 작업                  | 설명     |
| --------------------- | -------- |
| `auth:signIn`         | 로그인     |
| `auth:signUp`         | 회원가입     |
| `auth:signOut`        | 로그아웃     |
| `auth:changePassword` | 비밀번호 변경 |

### 사용자

| 작업                  | 설명         |
| --------------------- | ------------ |
| `users:updateProfile` | 프로필 업데이트 |

### UI 설정

| 작업                       | 설명           |
| -------------------------- | -------------- |
| `uiSchemas:insertAdjacent` | UI 스키마 삽입 |
| `uiSchemas:patch`          | UI 스키마 수정 |
| `uiSchemas:remove`         | UI 스키마 제거 |

### 컬렉션 작업

| 작업             | 설명             |
| ---------------- | ---------------- |
| `create`         | 레코드 생성         |
| `update`         | 레코드 업데이트         |
| `destroy`        | 레코드 삭제         |
| `updateOrCreate` | 레코드 업데이트 또는 생성   |
| `firstOrCreate`  | 레코드 조회 또는 생성   |
| `move`           | 레코드 이동         |
| `set`            | 관계 필드 레코드 설정 |
| `add`            | 관계 필드 레코드 추가 |
| `remove`         | 관계 필드 레코드 제거 |
| `export`         | 레코드 내보내기         |
| `import`         | 레코드 가져오기         |

## 다른 감사 리소스 추가

플러그인을 통해 다른 리소스 작업을 확장했고 이러한 리소스 작업 동작을 감사 로그에 기록하고 싶다면, [API](/api/server/audit-manager.md)를 참조하십시오.
