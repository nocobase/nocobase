# 사용자 잠금

## 소개

잠긴 사용자를 관리하고, 필요에 따라 사용자를 직접 잠글 수 있습니다.

![](https://static-docs.nocobase.com/202412281450083.png)

### 사용자 잠금 해제

유효하지 않은 비밀번호 로그인 횟수 제한을 초과하여 잠긴 사용자는 해당 기록을 삭제하여 잠금을 해제할 수 있습니다.

### 사용자 잠금

사용자 기록을 추가하여 사용자를 직접 잠글 수 있습니다. 사용자가 잠기면 API 키를 포함한 어떤 인증 방식으로도 시스템에 접근할 수 없습니다.

![](https://static-docs.nocobase.com/202412281450512.png)
