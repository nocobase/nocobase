# 알림: 인앱 메시지

## 소개

NocoBase 애플리케이션 내에서 사용자가 실시간 메시지 알림을 받을 수 있도록 합니다.

## 설치

이 플러그인은 내장 플러그인이므로 별도로 설치할 필요가 없습니다.

## 인앱 메시지 채널 추가

알림 관리를 열고, '추가' 버튼을 클릭한 다음 인앱 메시지를 선택합니다.
![2024-11-08-08-33-26-20241108083326](https://static-docs.nocobase.com/2024-11-08-08-33-26-20241108083326.png)

채널 이름과 설명을 입력한 후, '제출'을 클릭합니다.
![2024-11-08-08-34-32-20241108083431](https://static-docs.nocobase.com/2024-11-08-08-34-32-20241108083431.png)

목록에 새로운 채널이 표시됩니다.

![2024-11-08-08-34-52-20241108083452](https://static-docs.nocobase.com/2024-11-08-08-34-52-20241108083452.png)

## 사용 사례 예시

인앱 메시지 사용 방법을 더 잘 이해하실 수 있도록, '마케팅 리드 추적'에 대한 예시를 소개합니다.

팀에서 중요한 마케팅 캠페인을 진행 중이며, 잠재 고객의 피드백과 요구 사항을 추적하는 것이 목표라고 가정해 봅시다. 인앱 메시지를 사용하면 다음을 수행할 수 있습니다.

**알림 채널 생성**: 먼저 알림 채널 관리에서 'Marketing Clue'라는 이름의 인앱 메시지 채널을 설정하여 팀원들이 이 채널의 용도를 명확하게 파악할 수 있도록 합니다.

![2024-11-08-08-34-32-20241108083431](https://static-docs.nocobase.com/2024-11-08-08-34-32-20241108083431.png)

**워크플로우 설정**: 새로운 마케팅 리드가 생성될 때 자동으로 알림을 트리거하는 워크플로우를 생성합니다. 워크플로우에 알림 노드를 추가하고, 이전에 생성한 'Marketing Clue' 채널을 선택한 다음, 실제 요구 사항에 따라 메시지 내용을 설정할 수 있습니다. 예를 들어:

![image-1-2024-10-27-14-07-17](https://static-docs.nocobase.com/image-1-2024-10-27-14-07-17.png)

**실시간 알림 수신**: 워크플로우가 트리거되면 모든 관련 담당자가 실시간으로 알림을 받게 되며, 이를 통해 팀이 신속하게 대응하고 조치할 수 있습니다.

![image-2-2024-10-27-14-07-22](https://static-docs.nocobase.com/image-2-2024-10-27-14-07-22.png)

**메시지 관리 및 추적**: 인앱 메시지는 발송 채널 이름별로 그룹화됩니다. 메시지의 읽음/읽지 않음 상태를 기준으로 필터링하여 중요한 정보를 빠르게 확인할 수 있습니다. '보기' 버튼을 클릭하면 설정된 링크 페이지로 이동하여 관련 사항을 추가로 처리할 수 있습니다.

![20241027140648-2024-10-27-14-06-51-2024-10-29-13-26-41](https://static-docs.nocobase.com/20241027140648-2024-10-27-14-06-51-2024-10-29-13-26-41.png)
