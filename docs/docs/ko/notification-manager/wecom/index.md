# 알림: WeCom

## 소개

**WeCom** 플러그인을 사용하면 애플리케이션에서 WeCom 사용자에게 알림 메시지를 보낼 수 있습니다.

## WeCom 인증기 추가 및 설정

먼저 NocoBase에 WeCom 인증기를 추가하고 설정해야 합니다. 자세한 내용은 [사용자 인증 - WeCom](/auth-verification/auth-wecom)을(를) 참조하세요. WeCom을 통해 로그인한 시스템 사용자만 WeCom을 통해 시스템 알림을 받을 수 있습니다.
