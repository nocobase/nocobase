# 라우트 관리자

## 소개

라우트 관리자는 시스템의 메인 페이지 라우트를 관리하는 도구이며, `데스크톱` 및 `모바일` 환경을 모두 지원합니다. 라우트 관리자를 사용하여 생성된 라우트는 메뉴에 동기화되어 표시됩니다(메뉴에 표시되지 않도록 설정할 수 있습니다). 반대로, 페이지 메뉴에서 추가된 메뉴도 라우트 관리자 목록에 동기화되어 표시됩니다.

![20250107115449](https://static-docs.nocobase.com/20250107115449.png)

## 사용 설명서

### 라우트 유형

시스템은 네 가지 유형의 라우트를 지원합니다:

- 그룹 (group): 라우트를 그룹으로 묶어 관리하며, 하위 라우트를 포함할 수 있습니다.
- 페이지 (page): 시스템 내부 페이지입니다.
- 탭 (tab): 페이지 내에서 탭을 전환하는 데 사용되는 라우트 유형입니다.
- 링크 (link): 내부 또는 외부 링크로, 설정된 링크 주소로 직접 이동할 수 있습니다.

### 라우트 추가

우측 상단의 "Add new" 버튼을 클릭하여 새로운 라우트를 생성할 수 있습니다:

1. 라우트 유형 (Type)을 선택합니다.
2. 라우트 제목 (Title)을 입력합니다.
3. 라우트 아이콘 (Icon)을 선택합니다.
4. 메뉴에 표시 여부 (Show in menu)를 설정합니다.
5. 탭 페이지 활성화 여부 (Enable page tabs)를 설정합니다.
6. 페이지 유형의 경우, 시스템이 고유한 라우트 경로 (Path)를 자동으로 생성합니다.

![20250124131803](https://static-docs.nocobase.com/20250124131803.png)

### 라우트 작업

각 라우트 항목은 다음 작업을 지원합니다:

- Add child: 하위 라우트를 추가합니다.
- Edit: 라우트 설정을 편집합니다.
- View: 라우트 페이지를 확인합니다.
- Delete: 라우트를 삭제합니다.

### 일괄 작업

상단 툴바는 다음 일괄 작업 기능을 제공합니다:

- Refresh: 라우트 목록을 새로 고칩니다.
- Delete: 선택된 라우트를 삭제합니다.
- Hide in menu: 메뉴에서 선택된 라우트를 숨깁니다.
- Show in menu: 메뉴에 선택된 라우트를 표시합니다.

### 라우트 필터링

상단의 "Filter" 기능을 사용하여 필요에 따라 라우트 목록을 필터링할 수 있습니다.

:::info{title=참고}
라우트 설정을 변경하면 시스템의 내비게이션 메뉴 구조에 직접적인 영향을 미칩니다. 신중하게 작업하고 라우트 설정의 정확성을 확인해 주십시오.
:::
