# AI 직원 · Viz: 인사이트 분석가

> 클릭 한 번으로 차트와 인사이트를 생성하여 데이터가 스스로 이야기하게 만드세요.

## 1. Viz는 누구인가요?

**Viz**는 내장된 **AI 인사이트 분석가**입니다.
Viz는 현재 페이지의 데이터(예: 리드, 기회, 계정)를 읽고, 추세 차트, 비교 차트, KPI 카드 및 간결한 결론을 자동으로 생성하여 비즈니스 분석을 쉽고 직관적으로 만듭니다.

Viz는 차가운 보고서 도구가 아니라, 질문을 이해하고 스토리를 들려주는 분석가입니다.

> 💡 "최근 매출이 왜 감소했는지" 알고 싶으신가요?
> Viz에게 한 문장만 말하면, Viz는 감소가 어느 단계에서 발생했는지, 가능한 원인은 무엇인지, 그리고 다음 단계에서 무엇을 할 수 있는지 알려드립니다.

## 2. Viz로 무엇을 할 수 있나요?

| 기능             | 설명                               | 예시                           |
| -------------- | ---------------------------------- | ------------------------------ |
| 📊 **차트 자동 생성** | 클릭 한 번으로 데이터 시각화, SQL 작성 불필요      | "이번 달 판매 추세 생성"             |
| 🔍 **변화 및 이상 감지** | 상승 또는 하락의 원인 분석                 | "이번 달은 지난달보다 무엇이 더 좋았나요?" |
| 🧭 **의사 결정 지원** | 데이터 기반의 실행 가능한 제안 제공            | "어떤 채널에 예산을 더 투입해야 할까요?"   |
| 🧩 **데이터 관점 집계** | 지역, 제품, 출처 등 여러 차원으로 비교 | "각 지역별 매출 비교 표시"           |

월별 경영 검토, 채널 ROI, 판매 퍼널 등 무엇이든 Viz는 몇 초 만에 차트와 해석 결과를 생성할 수 있습니다.

## 3. 사용 방법

### 3.1 페이지 진입점

* **오른쪽 상단 버튼 (권장)**
  리드, 기회, 계정 등 페이지의 오른쪽 상단에서 **Viz 아이콘**을 클릭하면 다음과 같은 사전 설정된 작업을 선택할 수 있습니다.

  * 단계별 전환 및 추세
  * 출처 채널 비교
  * 월별 검토 분석

  ![리드 페이지 예시](https://static-docs.nocobase.com/00_QuickStart_cn-2025-09-29-16-46-00.png)

* **오른쪽 하단 전역 패널**
  어떤 페이지에 있든 전역 AI 패널을 호출하여 Viz에게 직접 말할 수 있습니다.

  ```
  최근 90일간의 판매 변화 분석
  ```

  Viz는 현재 페이지의 데이터 컨텍스트를 자동으로 가져옵니다.

### 3.2 상호 작용 방식

Viz는 자연어 질문을 지원하며, 다단계 후속 질문도 이해할 수 있습니다.
예시:

```
Viz, 이번 달 리드 추세를 생성해 주세요.
```

```
타사 채널의 성과만 보여주세요.
```

```
그렇다면 어느 지역의 성장이 가장 빠른가요?
```

각 후속 질문은 이전 분석 결과를 기반으로 더 깊이 탐색하며, 데이터 조건을 반복해서 입력할 필요가 없습니다.

## 4. 일반적인 분석 시나리오

| 시나리오       | 알고 싶은 내용                             | Viz의 결과물                       |
| -------- | ---------------------------------- | ---------------------------------- |
| **월별 검토** | 이번 달은 지난달보다 무엇이 더 좋았나요?         | KPI 카드 + 추세 차트 + 세 가지 개선 제안 |
| **성장 분해** | 매출 상승은 수량 변화 때문인가요, 가격 변화 때문인가요? | 요인 분해 차트 + 비교표                |
| **채널 분석** | 어떤 채널에 계속 투자할 가치가 가장 높은가요?    | ROI 차트 + 유지율 곡선 + 제안          |
| **퍼널 분석** | 트래픽이 어느 단계에서 막히나요?             | 퍼널 차트 + 병목 현상 설명             |
| **고객 유지** | 어떤 고객이 가장 가치 있나요?              | RFM 세분화 차트 + 유지율 곡선          |
| **프로모션 평가** | 대규모 프로모션 효과는 어땠나요?             | 비교 차트 + 가격 탄력성 분석           |

> 📈 모든 차트는 유효한 ECharts 형식으로 생성되며, 차트당 하나의 핵심 포인트를 담고 간결한 결론이 함께 제공됩니다.
> 데이터가 부족할 경우, Viz는 결과를 조작하지 않고 직접 설명해 드립니다.

## 5. Viz와 대화하는 팁

| 방법                     | 효과                                     |
| ---------------------- | ---------------------------------------- |
| ✅ 시간 범위 지정          | "최근 30일", "지난달 vs 이번 달"과 같이 더 정확하게 |
| ✅ 차원 지정             | "지역/채널/제품별로 보기"를 통해 관점 정렬에 도움 |
| ✅ 세부 사항보다는 추세에 집중 | Viz는 변화 방향과 핵심 원인을 파악하는 데 능숙합니다 |
| ✅ 자연어 사용           | 명령형 문법 없이, 채팅하듯이 질문하세요        |

## 6. Viz는 누구에게 가장 적합한가요?

| 역할           | 용도                                   |
| ------------ | -------------------------------------- |
| **영업 관리자**  | 단계별 전환율, 채널 성과, 팀 실적 확인         |
| **마케터**     | 광고 투자 ROI, 프로모션 효과, 고객 유지 분석 |
| **운영 분석가**  | 데이터 신속 추출, 이상 감지, 가설 검증         |
| **경영진**     | 한눈에 경영 상태 파악, 의사 결정 신호 확보     |

## 7. 사용 제안

1.  **사전 설정된 작업으로 시작하세요**
    공식 데모에는 일반적인 작업이 내장되어 있어, 별도의 프롬프트 없이도 바로 효과를 경험할 수 있습니다.
    예시: 리드 페이지 → **Viz → 단계별 전환 및 추세** 클릭

2.  **출력 스타일을 관찰하세요**
    각 분석 지점에는 독립적인 차트와 간략한 설명이 있습니다.
    명확한 차트와 간결한 텍스트는 Viz의 표준 출력입니다.

3.  **점진적으로 질문을 심화하세요**
    분석 보고서를 읽은 후 "왜" "어떻게 개선해야 하는지"와 같이 계속해서 질문하면, Viz가 자동으로 후속 조치를 취합니다.

## 8. 요약

*   Viz = 당신의 데이터 인사이트 비서
*   SQL을 작성하거나 차트를 구성할 필요가 없습니다
*   자연어 한 문장으로 분석 보고서를 얻을 수 있습니다
*   모든 결론은 실제 데이터를 기반으로 하며, 명확하고 신뢰할 수 있습니다

> **리드 → Viz → 단계별 전환 및 추세**로 시작하여,
> 첫 번째 차트를 보는 것이 AI 직원을 이해하는 가장 좋은 시작점입니다.
