# AI 직원 · Nathan: 프런트엔드 엔지니어

> 프런트엔드 코드를 작성하고 최적화하여 복잡한 인터랙티브 로직을 구현하도록 돕습니다.

`Nathan`은 NocoBase의 프런트엔드 개발 전문가입니다. `JSBlock`, `JSField`, `JSItem`, `JSColumn`, `JSAction`, `Event Flow`, `Linkage` 등 JavaScript 작성이 필요한 상황에서 코드 편집기 우측 상단에 `Nathan`의 아바타가 나타납니다. 이 아바타를 클릭하여 코드 편집기 내 코드를 작성하거나 수정하도록 `Nathan`에게 요청할 수 있습니다.

![20251022163405](https://static-docs.nocobase.com/20251022163405.png)

## 사용 방법

코드 편집기에서 `Nathan`을 클릭하여 AI 직원 대화 상자를 엽니다. 코드 편집기의 코드는 자동으로 입력 상자에 첨부되며, 애플리케이션 컨텍스트로 `Nathan`에게 전송됩니다.

![20251022163935](https://static-docs.nocobase.com/20251022163935.png)

코딩 요구 사항을 입력하고 `Nathan`에게 전송한 후 답변을 기다립니다.

![20251022164041](https://static-docs.nocobase.com/20251022164041.png)

`Nathan`의 답변에 있는 코드 블록에서 `Apply to editor` 버튼을 클릭합니다. 그러면 해당 코드가 코드 편집기에 적용됩니다.

![20251022164323](https://static-docs.nocobase.com/20251022164323.png)

코드 편집기의 `Run` 버튼을 클릭하면 실시간으로 결과를 확인할 수 있습니다.

![20251022164436](https://static-docs.nocobase.com/20251022164436.png)

## 코드 기록

`Nathan` 대화 상자 우측 상단에 있는 '명령줄' 아이콘을 클릭하면 현재 세션에서 사용자가 보낸 코드 조각과 `Nathan`이 답변한 코드 조각을 확인할 수 있습니다.
