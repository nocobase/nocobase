# AI 에이전트 · Orin: 데이터 모델러

> 컬렉션을 지능적으로 설계하고 데이터베이스 구조를 최적화합니다.

`Orin`은 데이터 모델링 전문가입니다. 주 데이터 소스 설정 페이지에서 `Orin`의 도움을 받아 컬렉션을 생성하거나 수정할 수 있습니다.

![20251022160628](https://static-docs.nocobase.com/20251022160628.png)

## 사용 방법

데이터 소스 관리 플러그인으로 이동하여 주 데이터 소스를 설정합니다.

![20251022161146](https://static-docs.nocobase.com/20251022161146.png)

오른쪽 상단의 `Orin` 아바타를 클릭하여 AI 에이전트 대화 상자를 엽니다.

![20251022161641](https://static-docs.nocobase.com/20251022161641.png)

`Orin`에게 모델링 요구 사항을 설명하고, 전송 후 응답을 기다립니다.

![20251022161749](https://static-docs.nocobase.com/20251022161749.png)

`Orin`이 요구 사항을 확인하면, 자신의 기능을 사용하여 데이터 모델링 미리 보기를 회신합니다.

미리 보기를 확인한 후, `Finish review and apply` 버튼을 클릭하여 `Orin`의 모델링에 따라 컬렉션을 생성합니다.

![20251022162142](https://static-docs.nocobase.com/20251022162142.png)
