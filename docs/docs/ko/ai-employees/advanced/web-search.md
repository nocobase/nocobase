# 고급

## 소개

대규모 언어 모델(LLM)은 일반적으로 최신 데이터가 부족하여 정보의 시의성이 떨어집니다. 이러한 이유로 온라인 LLM 서비스 플랫폼에서는 AI가 답변하기 전에 도구를 사용하여 정보를 검색하고, 검색 결과를 바탕으로 답변을 생성할 수 있도록 '웹 검색' 기능을 제공합니다.

AI 직원 기능은 다양한 온라인 LLM 서비스 플랫폼의 웹 검색 기능을 지원하도록 개발되었습니다. 따라서 AI 직원 모델 설정과 대화 중에 웹 검색 기능을 활성화할 수 있습니다.

## 웹 검색 기능 활성화

AI 직원 **플러그인** 설정 페이지로 이동합니다. `AI employees` 탭을 클릭하여 AI 직원 관리 페이지로 들어갑니다.

![20251021225643](https://static-docs.nocobase.com/20251021225643.png)

웹 검색 기능을 활성화할 AI 직원을 선택하고, `Edit` 버튼을 클릭하여 AI 직원 편집 페이지로 이동합니다.

![20251022114043](https://static-docs.nocobase.com/20251022114043.png)

`Model settings` 탭에서 `Web Search` 스위치를 켜고, `Submit` 버튼을 클릭하여 변경 사항을 저장합니다.

![20251022114300](https://static-docs.nocobase.com/20251022114300.png)

## 대화에서 웹 검색 기능 사용하기

AI 직원의 웹 검색 기능을 활성화하면, 대화 입력창에 '웹' 아이콘이 나타납니다. 기본적으로 웹 검색이 활성화되어 있으며, 아이콘을 클릭하여 비활성화할 수 있습니다.

![20251022115110](https://static-docs.nocobase.com/20251022115110.png)

웹 검색이 활성화되면, AI 직원의 답변에 웹 검색 결과가 함께 표시됩니다.

![20251022115502](https://static-docs.nocobase.com/20251022115502.png)

## 플랫폼별 웹 검색 도구 차이점

현재 AI 직원의 웹 검색 기능은 온라인 LLM 서비스 플랫폼에 따라 달라집니다. 따라서 사용자 경험에 차이가 있을 수 있으며, 구체적인 차이점은 다음과 같습니다.
