# 고급

## 소개

AI 직원과 대화할 때, 파일 전송 외에도 애플리케이션 컨텍스트 정보를 함께 보내 AI 직원이 해당 정보를 기반으로 답변하도록 할 수 있습니다.

### 블록 정보 전송

AI 직원 대화 상자에서 왼쪽 하단의 `Add work context` 버튼을 클릭한 다음 `Pick block`을 선택합니다. 그러면 애플리케이션 페이지가 블록 선택 모드로 전환됩니다.

![20251022010148](https://static-docs.nocobase.com/20251022010148.png)

블록 위에 마우스를 올리면, 컨텍스트로 전송할 수 있는 블록의 색상이 바뀝니다.

![20251022010422](https://static-docs.nocobase.com/20251022010422.png)

전송할 블록을 선택하면, 해당 블록의 컨텍스트 정보가 대화 상자에 첨부됩니다.

![20251022010543](https://static-docs.nocobase.com/20251022010543.png)

질문 메시지를 입력한 후, 일반 메시지를 보낼 때와 마찬가지로 전송 버튼을 클릭하면 AI 직원이 블록 정보를 기반으로 답변합니다.

블록 정보는 메시지 목록에서도 확인할 수 있습니다.

![20251022010758](https://static-docs.nocobase.com/20251022010758.png)

### 블록 선택 종료

애플리케이션 페이지가 블록 선택 모드에 진입했지만, 특정 블록을 선택하고 싶지 않다면 하단의 "정지" 아이콘을 클릭하여 블록 선택 모드를 종료할 수 있습니다.

![20251022011043](https://static-docs.nocobase.com/20251022011043.png)
