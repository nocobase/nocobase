# 컨텍스트 추가 - 블록

AI 직원과 대화할 때 파일을 전송하는 것 외에도, 애플리케이션의 컨텍스트 정보를 AI 직원에게 전송하여 해당 정보를 바탕으로 답변을 받도록 할 수 있습니다.

### 블록 정보 전송하기

AI 직원 대화창의 왼쪽 하단에 있는 `컨텍스트 추가` 버튼을 클릭하고 `블록 선택`을 선택하면, 애플리케이션 페이지가 블록 선택 모드로 전환됩니다.

![20251022010148](https://static-docs.nocobase.com/20251022010148.png)

마우스를 블록 위에 올리면 컨텍스트로 전송 가능한 블록의 색상이 변경됩니다.

![20251022010422](https://static-docs.nocobase.com/20251022010422.png)

전송할 블록을 선택하면 해당 블록의 컨텍스트 정보가 대화창에 첨부됩니다.

![20251022010543](https://static-docs.nocobase.com/20251022010543.png)

질문 메시지를 입력한 후 일반적인 메시지 전송과 마찬가지로 전송 버튼을 클릭하면, AI 직원이 블록 정보를 기반으로 답변합니다.

블록 정보는 메시지 목록에도 표시됩니다.

![20251022010758](https://static-docs.nocobase.com/20251022010758.png)

### 블록 선택 종료하기

애플리케이션 페이지가 블록 선택 모드일 때 아무 블록도 선택하지 않고 종료하려면, 하단의 "중지" 아이콘을 클릭하여 블록 선택 모드에서 나갈 수 있습니다.

![20251022011043](https://static-docs.nocobase.com/20251022011043.png)
