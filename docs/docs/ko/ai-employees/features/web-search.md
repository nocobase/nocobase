# 웹 검색

웹 검색은 모델 학습 데이터 이외의 최신 정보를 보완하는 데 사용됩니다.

## 작동 방식

웹 검색의 사용 가능 여부는 현재 대화에서 선택한 모델 서비스가 웹 검색(Web Search)을 지원하는지에 따라 달라집니다.

- **지원함**: 웹 검색 스위치가 표시되며, 필요에 따라 켜거나 끌 수 있습니다.
- **지원하지 않음**: 스위치가 표시되지 않으며, 검색 상태가 자동으로 비활성화됩니다.

## 대화에서 사용하기

대화 입력창 영역에서 웹 검색 스위치를 사용합니다.

- **켜기**: AI가 먼저 정보를 검색한 후 답변합니다.

![web-search-switch-visible-on-supported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-visible-on-supported-model.png)

- **끄기**: AI가 기존 컨텍스트만을 바탕으로 답변합니다.

![web-search-switch-hidden-on-unsupported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-hidden-on-unsupported-model.png)

## 플랫폼별 차이

모델마다 웹 검색 지원 능력이 다르므로, 실제 상황에 맞춰 사용하시기 바랍니다.
