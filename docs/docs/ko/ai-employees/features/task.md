# 단축 작업

AI 직원이 더 효율적으로 업무를 시작할 수 있도록, 시나리오 블록에 AI 직원을 바인딩하고 자주 사용하는 몇 가지 작업을 미리 설정할 수 있습니다.

이를 통해 사용자는 매번 **블록 선택** 및 **명령어 입력**을 할 필요 없이 클릭 한 번으로 작업을 빠르게 시작할 수 있습니다.

## 블록에 AI 직원 바인딩

UI 편집 모드로 진입한 후, `Actions`를 지원하는 블록에서 `Actions` 하위의 `AI employees` 메뉴를 선택하고 AI 직원을 선택하면 해당 AI 직원이 현재 블록과 바인딩됩니다.

![20251022135306](https://static-docs.nocobase.com/20251022135306.png)

바인딩이 완료되면 페이지에 접속할 때마다 블록의 Actions 영역에 현재 블록과 바인딩된 AI 직원이 표시됩니다.

![20251022135438](https://static-docs.nocobase.com/20251022135438.png)

## 작업 설정

UI 편집 모드에서 블록에 바인딩된 AI 직원 아이콘 위에 마우스를 올리면 메뉴 버튼이 나타납니다. `Edit tasks`를 선택하여 작업 설정 페이지로 이동합니다.

작업 설정 페이지에서 현재 AI 직원에게 여러 개의 작업을 추가할 수 있습니다.

각 탭은 독립적인 작업을 나타내며, 옆의 "+" 기호를 클릭하여 새 작업을 추가할 수 있습니다.

![clipboard-image-1771913187](https://static-docs.nocobase.com/clipboard-image-1771913187.png)

작업 설정 양식:

- `Title` 입력란에 작업 제목을 입력합니다. 작업 내용을 짧게 기술하며, 이 제목은 AI 직원의 작업 목록에 표시됩니다.
- `Background` 입력란에 작업의 주요 내용을 입력합니다. 이 내용은 AI 직원과 대화할 때 시스템 프롬프트(System Prompt)로 사용됩니다.
- `Default user message` 입력란에 기본으로 전송할 사용자 메시지를 입력합니다. 작업을 선택하면 사용자 입력창에 자동으로 채워집니다.
- `Work context`에서 AI 직원에게 기본으로 전송할 애플리케이션 컨텍스트 정보를 선택합니다. 이 작업은 대화창에서의 조작과 동일합니다.
- `Skills` 선택창에는 현재 AI 직원이 보유한 기술이 표시됩니다. 특정 기술을 해제하여 AI 직원이 해당 작업을 수행할 때 그 기술을 사용하지 않도록 설정할 수 있습니다.
- `Send default user message automatically` 체크박스는 작업 실행 클릭 시 기본 사용자 메시지를 자동으로 전송할지 여부를 설정합니다.


## 작업 목록

AI 직원의 작업을 설정하면, 해당 작업들은 AI 직원의 프로필 팝업창과 대화 시작 전의 인사말 메시지에 표시됩니다. 클릭하면 바로 작업을 실행할 수 있습니다.
