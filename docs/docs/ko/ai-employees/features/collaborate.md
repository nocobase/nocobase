# AI 직원과 협업하기

AI 직원을 생성하고 활성화한 후, 페이지에서 해당 직원과 협업할 수 있습니다.

## 진입 경로

주요 진입 경로는 다음 두 가지가 있습니다:

1. **우측 하단 메인 진입로**: 비즈니스 페이지 우측 하단에서 AI 대화 패널을 불러옵니다. 일반적인 질의응답 및 블록 간 협업에 적합합니다.
2. **블록 Action 진입로**: `Actions`를 지원하는 블록에서 `Actions -> AI employees`를 통해 진입합니다. 현재 블록에 대한 작업(예: JSBlock 시나리오 등)을 수행하는 데 적합합니다.

## 대화 기본 조작

대화창에서는 메시지 전송, 첨부 파일 업로드, 대화 기록 확인, 새 대화 생성, 시스템 프롬프트 편집 등 일반적인 작업을 지원합니다.

## 대화 내 전환

입력 영역 하단에서 다음 항목을 직접 전환할 수 있습니다:

- `AI Employee Switcher`: 현재 직원을 전환합니다.
- `Model Switcher`: 현재 대화 모델을 전환합니다.

모델 설정은 직원별로 선호도가 저장되며, 다음 접속 시 우선적으로 복원됩니다.
