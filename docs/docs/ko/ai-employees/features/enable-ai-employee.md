# AI 직원 활성화

NocoBase는 다양한 시나리오에 최적화된 여러 내장 AI 직원을 제공합니다.

## 활성화 방법

`AI employees` 관리 페이지로 이동한 후, 목록에서 `Enabled` 스위치를 켜서 직원을 직접 활성화할 수 있습니다.

![ai-employee-list-enable-switch.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employee-list-enable-switch.png)

## 활성화 후 사용

활성화하면 해당 직원이 사용 가능한 진입점(예: 우측 하단 바로가기 또는 특정 시나리오 진입점)에 나타납니다.
