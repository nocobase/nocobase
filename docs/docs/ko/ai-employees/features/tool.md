# 스킬 사용하기

스킬(Tools)은 AI 직원이 '무엇을 할 수 있는지'를 정의합니다.

## 스킬 구조

스킬 페이지는 세 가지 카테고리로 나뉩니다:

1. `General skills`: 모든 AI 직원이 공유하며, 읽기 전용입니다.
2. `Employee-specific skills`: 현재 직원 전용이며, 일반적으로 읽기 전용입니다.
3. `Custom skills`: 사용자 정의 스킬로, 추가 및 삭제가 가능하며 기본 권한을 설정할 수 있습니다.

![skills-three-sections-general-specific-custom.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/skills-three-sections-general-specific-custom.png)

## 스킬 권한

스킬 권한은 다음과 같이 통합됩니다:

- `Ask`: 호출 전 확인을 요청합니다.
- `Allow`: 직접 호출을 허용합니다.

권장 사항: 데이터 수정과 관련된 스킬은 기본적으로 `Ask`를 사용하는 것이 좋습니다.

![skills-permission-ask-allow-segmented.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/skills-permission-ask-allow-segmented.png)

## 추가 및 유지 관리

`Custom skills`에서 `Add skill`을 클릭하여 스킬을 추가하고, 비즈니스 리스크에 따라 권한을 설정합니다.
