# 개요

## 소개

AI 지식 기반 플러그인은 AI 직원에게 RAG 검색 기능을 제공합니다.

RAG 검색 기능을 통해 AI 직원은 사용자 질문에 답변할 때 더욱 정확하고 전문적이며, 기업 내부 관련성이 높은 답변을 제공할 수 있습니다.

관리자가 유지 관리하는 지식 기반에서 제공하는 전문 분야 및 기업 내부 문서를 활용하여 AI 직원의 답변 정확성과 추적 가능성을 향상시킵니다.

### RAG란 무엇인가요?

RAG(Retrieval Augmented Generation)는 "검색-증강-생성"을 의미합니다.

- 검색: 사용자 질문은 임베딩 모델(예: BERT)을 통해 벡터로 변환됩니다. 벡터 라이브러리에서 밀집 검색(의미론적 유사성) 또는 희소 검색(키워드 매칭)을 통해 Top-K 관련 텍스트 청크를 불러옵니다.
- 증강: 검색 결과는 원본 질문과 결합되어 증강 프롬프트(Prompt)를 형성하며, 이는 LLM의 컨텍스트 창에 주입됩니다.
- 생성: LLM은 증강 프롬프트를 결합하여 최종 답변을 생성하며, 사실성과 추적 가능성을 보장합니다.
