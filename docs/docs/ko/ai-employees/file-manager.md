# 파일 관리

## 소개

AI 직원과의 대화 중에 업로드되는 파일의 저장 방식을 설정합니다.

## 설정

AI 직원 플러그인 설정 페이지로 이동하여, `Settings` 탭을 클릭하면 저장 방식 설정 페이지로 들어갈 수 있습니다.

![20251022002648](https://static-docs.nocobase.com/20251022002648.png)

`Files storage`에서 선택 가능한 저장 항목은 파일 관리 플러그인에서 설정합니다.
