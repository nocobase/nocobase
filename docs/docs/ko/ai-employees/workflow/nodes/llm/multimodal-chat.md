# 멀티모달 대화

## 이미지

모델이 지원하는 경우, LLM 노드는 모델에 이미지를 보낼 수 있습니다. 이미지를 사용할 때는 변수를 통해 첨부 파일 필드를 선택하거나, 연결된 파일 컬렉션 레코드를 선택해야 합니다. 파일 컬렉션 레코드를 선택할 때, 객체 레벨에서 선택할 수도 있고 URL 필드를 선택할 수도 있습니다.

![](https://static-docs.nocobase.com/202503041034858.png)

이미지 전송 형식에는 두 가지 옵션이 있습니다.

- URL을 통해 전송 - 로컬에 저장된 이미지를 제외한 모든 이미지는 URL 형식으로 전송됩니다. 로컬에 저장된 이미지는 전송 전에 base64 형식으로 변환됩니다.
- base64를 통해 전송 - 로컬 또는 클라우드에 저장된 모든 이미지는 base64 형식으로 전송됩니다. 이 옵션은 이미지 URL이 온라인 LLM 서비스에서 직접 접근할 수 없는 경우에 유용합니다.

![](https://static-docs.nocobase.com/202503041200638.png)
