# 첨부 파일 필드

## 소개

시스템에는 "첨부 파일" 타입의 필드가 내장되어 있어, 사용자 정의 컬렉션에서 파일 업로드를 지원합니다.

첨부 파일 필드는 내부적으로 시스템에 내장된 "Attachments" (`attachments`) 컬렉션을 가리키는 다대다 관계 필드입니다. 어떤 컬렉션에든 첨부 파일 필드를 생성하면, 다대다 관계를 위한 중간 테이블이 자동으로 생성됩니다. 업로드된 파일의 메타데이터는 "Attachments" 컬렉션에 저장되며, 해당 컬렉션에서 참조하는 파일 정보는 이 중간 테이블을 통해 연결됩니다.

### MIME 타입 제한

이 설정은 업로드 가능한 파일 타입을 제한하는 데 사용되며, [MIME](https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/MIME_types) 문법으로 형식을 지정합니다. 예를 들어, `image/*`는 이미지 파일을 나타냅니다. 여러 타입은 쉼표로 구분할 수 있습니다. 가령, `image/*,application/pdf`는 이미지 파일과 PDF 파일 모두를 허용합니다.

### 저장 엔진

업로드된 파일을 저장할 저장 엔진을 선택합니다. 비워두면 시스템의 기본 저장 엔진이 사용됩니다.
