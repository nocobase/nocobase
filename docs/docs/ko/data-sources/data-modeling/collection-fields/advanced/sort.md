# 컬렉션 선택기

컬렉션 선택기 필드는 사용자가 미리 정의된 목록에서 하나 이상의 컬렉션을 선택할 수 있도록 하는 특별한 필드 유형입니다. 이 필드 유형은 구성 가능한 보고서, 대시보드 또는 사용자 선택에 따라 데이터 소스를 변경해야 하는 복잡한 애플리케이션을 구축하는 등 다양한 컬렉션을 동적으로 참조해야 하는 시나리오에서 매우 유용합니다.

## 활용 사례

### 동적 보고서 생성

보고서 생성 모듈에서 사용자는 여러 컬렉션 중에서 보고서의 데이터 소스로 사용할 컬렉션을 선택해야 할 수 있습니다. 컬렉션 선택기를 사용하면 사용자는 원하는 컬렉션을 쉽게 선택할 수 있으며, 시스템은 사용자의 선택에 따라 보고서를 동적으로 생성합니다.

### 구성 가능한 대시보드

대시보드 애플리케이션에서 사용자는 대시보드에 표시되는 데이터를 사용자 정의할 수 있습니다. 컬렉션 선택기를 통해 사용자는 관심 있는 컬렉션을 선택하여 대시보드에 다양한 데이터 소스의 차트와 통계 데이터를 표시할 수 있습니다.

### 유연한 데이터 가져오기/내보내기 도구

일반적인 데이터 가져오기 또는 내보내기 도구를 개발할 때, 컬렉션 선택기를 사용하여 사용자가 작업할 컬렉션을 지정하도록 할 수 있습니다. 이렇게 하면 동일한 도구를 시스템의 모든 컬렉션에서 데이터를 처리하는 데 사용할 수 있어 도구의 재사용성을 크게 향상시킵니다.
