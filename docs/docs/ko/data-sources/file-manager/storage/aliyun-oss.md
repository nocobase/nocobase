# 알리윈 OSS

알리윈 OSS 기반의 스토리지 엔진입니다. 사용하시기 전에 관련 계정과 권한을 미리 준비해야 합니다.

## 설정 매개변수

![알리윈 OSS 스토리지 엔진 설정 예시](https://static-docs.nocobase.com/20240712220011.png)

:::info{title=참고}
여기서는 알리윈 OSS 스토리지 엔진의 전용 매개변수만 설명합니다. 공통 매개변수는 [엔진 공통 매개변수](./index.md#引擎通用参数)를 참고해 주세요.
:::

### 리전

OSS 스토리지의 리전을 입력해 주세요. 예시: `oss-cn-hangzhou`

:::info{title=참고}
[알리윈 OSS 콘솔](https://oss.console.aliyun.com/)에서 스토리지 버킷의 리전 정보를 확인할 수 있습니다. 전체 도메인 이름 없이 리전 접두사 부분만 사용하시면 됩니다.
:::

### AccessKey ID

알리바바 클라우드(Aliyun) 인증 접근 키의 ID를 입력해 주세요.

### AccessKey Secret

알리바바 클라우드(Aliyun) 인증 접근 키의 Secret을 입력해 주세요.

### 버킷

OSS 스토리지 버킷의 이름을 입력해 주세요.
