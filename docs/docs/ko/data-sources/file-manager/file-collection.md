# 파일 컬렉션

## 소개

파일은 특별한 구조를 가진 컬렉션의 레코드이며, 이를 파일 컬렉션이라고 부릅니다. 파일 컬렉션은 파일의 메타 정보를 저장하고 파일 관리자를 통해 관리할 수 있습니다.
