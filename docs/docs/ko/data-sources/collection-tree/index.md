# 트리 컬렉션

## 소개

트리 구조 컬렉션은 계층적 데이터를 저장하는 데 사용되는 데이터 컬렉션 설계 패턴입니다. 이 컬렉션 구조는 트리 형태와 유사하며, 각 데이터 항목은 하나 이상의 자식 항목을 가질 수 있고, 이 자식 항목들은 다시 자신만의 자식 항목을 가질 수 있습니다.
