# 외부 데이터 소스 - MSSQL

## 소개

외부 MSSQL 데이터베이스를 데이터 소스로 사용할 수 있습니다. 현재 지원되는 버전은 SQL Server 2014-2019입니다.

## 설치

이 플러그인은 상용 플러그인이며, 플러그인 관리자를 통해 업로드하고 활성화해야 합니다.
