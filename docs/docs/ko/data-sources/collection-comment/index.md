# 댓글 컬렉션

## 소개

댓글 컬렉션은 사용자 댓글과 피드백을 저장하기 위해 특별히 설계된 데이터 테이블 템플릿입니다. 댓글 기능을 활용하면 모든 데이터 테이블에 댓글 기능을 추가하여, 사용자가 특정 레코드에 대해 토론하고 피드백을 제공하거나 주석을 달 수 있습니다. 댓글 컬렉션은 리치 텍스트 편집을 지원하며, 유연한 콘텐츠 작성 기능을 제공합니다.

![comment-collection-2025-11-01-00-39-01](https://static-docs.nocobase.com/comment-collection-2025-11-01-00-39-01.png)

## 주요 기능

-   **리치 텍스트 편집**: 기본적으로 마크다운(vditor) 에디터를 포함하여 리치 텍스트 콘텐츠 작성을 지원합니다.
-   **모든 데이터 테이블과 연결**: 관계 필드를 통해 댓글을 모든 데이터 테이블의 레코드와 연결할 수 있습니다.
-   **다단계 댓글**: 댓글에 답글을 달아 댓글 트리 구조를 구축할 수 있습니다.
-   **사용자 추적**: 댓글 작성자와 작성 시간을 자동으로 기록합니다.

## 사용 가이드

### 댓글 컬렉션 생성

![comment-collection-2025-11-01-00-37-10](https://static-docs.nocobase.com/comment-collection-2025-11-01-00-37-10.png)

1.  데이터 테이블 관리 페이지로 이동합니다.
2.  "새 컬렉션" 버튼을 클릭합니다.
3.  "댓글 컬렉션" 템플릿을 선택합니다.
4.  컬렉션 이름을 입력합니다 (예: "작업 댓글", "게시글 댓글" 등).
5.  시스템이 다음 기본 필드를 포함하는 댓글 컬렉션을 자동으로 생성합니다.
    -   댓글 내용 (마크다운 vditor 유형)
    -   작성자 (사용자 컬렉션과 연결)
    -   작성 시간 (날짜/시간 유형)

### 관계 설정

댓글을 대상 데이터 테이블과 연결하려면 관계 필드를 설정해야 합니다.

![](https://static-docs.nocobase.com/Solution/demoE3v1-19N.gif)

1.  댓글 컬렉션에 "다대일" 관계 필드를 추가합니다.
2.  연결할 대상 데이터 테이블을 선택합니다 (예: 작업 컬렉션, 게시글 컬렉션 등).
3.  필드 이름을 설정합니다 (예: "소속 작업", "소속 게시글" 등).

### 페이지에서 댓글 블록 사용

![Enable Comments Collection](https://static-docs.nocobase.com/Solution/demoE3v1-20.gif)

1.  댓글 기능을 추가할 페이지로 이동합니다.
2.  대상 레코드의 상세 페이지 또는 팝업에 블록을 추가합니다.
3.  "댓글" 블록 유형을 선택합니다.
4.  방금 생성한 댓글 컬렉션을 선택합니다.

### 일반적인 활용 사례

-   **작업 관리 시스템**: 팀원들이 작업에 대해 토론하고 피드백을 제공합니다.
-   **콘텐츠 관리 시스템**: 독자들이 게시글에 댓글을 달고 상호작용합니다.
-   **승인 워크플로우**: 승인자가 신청서에 주석을 달고 의견을 제시합니다.
-   **고객 피드백**: 고객이 제품 또는 서비스에 대한 평가를 수집합니다.

## 참고 사항

-   댓글 컬렉션은 상업용 플러그인 기능이므로, 댓글 플러그인을 활성화해야 사용할 수 있습니다.
-   댓글 컬렉션에 적절한 권한을 설정하여 누가 댓글을 보고, 생성하고, 삭제할 수 있는지 제어하는 것이 좋습니다.
-   댓글이 많은 시나리오에서는 성능 향상을 위해 페이지네이션 로드를 활성화하는 것이 좋습니다.
