# 캘린더 블록

## 소개

캘린더 블록은 이벤트 및 날짜 관련 데이터를 캘린더 형식으로 표시하여 회의 일정, 활동 계획 등 다양한 상황에서 유용하게 활용할 수 있습니다.

## 설치

이 플러그인은 NocoBase에 내장되어 있어 별도로 설치할 필요가 없습니다.

## 블록 추가하기

![20250403220300](https://static-docs.nocobase.com/20250403220300.png)

1. 제목 필드: 캘린더 바에 표시될 정보를 설정합니다. 현재 `input`, `select`, `phone`, `email`, `radioGroup`, `sequence` 등의 필드 유형을 지원하며, 플러그인을 통해 캘린더 블록에서 지원하는 제목 필드 유형을 확장할 수 있습니다.
2. 시작 시간: 작업의 시작 시간을 나타냅니다.
3. 종료 시간: 작업의 종료 시간을 나타냅니다.

### 배경색 필드

:::info{title=팁}
NocoBase 버전이 v1.4.0-beta 이상이어야 합니다.
:::

이 옵션은 캘린더 이벤트의 배경색을 설정하는 데 사용됩니다. 사용 방법은 다음과 같습니다.

1. 캘린더 데이터 테이블에 **드롭다운 단일 선택(Single select)** 또는 **라디오 그룹(Radio group)** 유형의 필드가 있어야 하며, 이 필드에 색상이 설정되어 있어야 합니다.
2. 다음으로, 캘린더 블록 설정 화면으로 돌아가서 **배경색 필드**에서 방금 색상을 설정한 필드를 선택합니다.
3. 마지막으로, 캘린더 이벤트에 색상을 선택하고 제출을 클릭하면 색상이 적용된 것을 확인할 수 있습니다.

![20240914192017_rec_](https://static-docs.nocobase.com/20240914192017_rec_.gif)

### 주 시작 요일

> v1.7.7 이상 버전에서 지원됩니다.

캘린더 블록은 주 시작 요일을 설정할 수 있도록 지원하며, **일요일** 또는 **월요일**을 한 주의 첫날로 선택할 수 있습니다.
기본 시작 요일은 **월요일**이며, 사용자가 지역별 관습에 따라 캘린더 표시를 조정하여 실제 사용 요구 사항에 더 잘 맞출 수 있도록 합니다.

### 오늘

캘린더 블록의 "오늘" 버튼은 편리한 탐색 기능을 제공하여, 사용자가 다른 날짜로 이동한 후에도 현재 날짜가 있는 캘린더 페이지로 빠르게 돌아올 수 있도록 합니다.
