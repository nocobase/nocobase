# 역할

## 관리 센터

### 역할 관리

![](https://static-docs.nocobase.com/da7083c67d794e23dc6eb0f85b1de86c.png)

NocoBase를 처음 설치하면 'Admin'과 'Member' 두 가지 역할이 기본으로 제공됩니다. 각 역할은 기능에 따라 서로 다른 기본 권한 설정을 가지고 있습니다.

### 역할 추가, 삭제 및 수정

역할 식별자는 시스템에서 고유한 값으로, 이를 통해 기본 역할을 사용자 정의할 수 있습니다. 단, 시스템에 내장된 기본 역할은 삭제할 수 없습니다.

![](https://static-docs.nocobase.com/35f323b346db4f9f12f9bee4dea63302.png)

### 기본 역할 설정

여기서 '기본 역할'이란 새로운 사용자를 생성할 때 별도의 역할이 지정되지 않은 경우, 자동으로 할당되는 역할을 말합니다.

![](https://static-docs.nocobase.com/f41bba7ff55ca28715c486dc45bc1708.png)

## 개인 센터

### 역할 전환

한 사용자에게 여러 역할을 할당할 수 있으며, 사용자가 여러 역할을 가지고 있을 경우 개인 센터에서 역할을 전환할 수 있습니다.

![](https://static-docs.nocobase.com/e331d11ec1ca3b8b7e0472105b167819.png)

사용자가 시스템에 로그인할 때 적용되는 기본 역할의 우선순위는 다음과 같습니다.

1.  마지막으로 전환한 역할 (역할을 전환할 때마다 이 값이 업데이트됩니다.)
2.  첫 번째 역할 (시스템 기본 역할)
