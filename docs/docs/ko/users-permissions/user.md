# 사용자

## 사용자 관리

이 플러그인은 설정 센터에서 사용자 관리 인터페이스를 제공합니다. 애플리케이션 초기화 시 삭제할 수 없는 슈퍼 관리자 계정이 자동으로 생성됩니다. 이 관리자는 Root 권한을 가지며 모든 리소스에 접근할 수 있습니다.

![](https://static-docs.nocobase.com/44bf40f56b45d4dd96c424fb08082cf6.png)

이 외에도, 페이지에서 사용자 컬렉션(Users collection)의 다양한 블록을 추가하여 사용자를 관리할 수 있습니다. 예를 들어, 테이블 블록을 사용할 수 있습니다.

![](https://static-docs.nocobase.com/76b5a4652f869541a9e8f18a4568a7c9.png)
