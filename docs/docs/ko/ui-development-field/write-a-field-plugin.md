# 데이터 소스

데이터 소스는 NocoBase에서 매우 중요한 개념입니다.

NocoBase는 MySQL, PostgreSQL, SQLite와 같은 관계형 데이터베이스뿐만 아니라 외부 API 서비스 같은 비관계형 데이터 연결도 지원합니다.

NocoBase에서는 이러한 다양한 유형의 데이터 출처를 통칭하여 '데이터 소스'라고 부릅니다.

데이터 소스는 단순히 데이터의 출처일 뿐만 아니라, NocoBase의 다양한 기능의 기반이 됩니다. 예를 들어, 데이터 소스를 기반으로 컬렉션을 생성하고, 이 컬렉션을 통해 데이터를 관리할 수 있습니다. 또한 워크플로우에서 데이터 소스를 사용하여 데이터 처리 자동화를 구현할 수도 있습니다.

## 주요 기능

- **다양한 유형의 데이터 소스 지원**: NocoBase는 MySQL, PostgreSQL, SQLite와 같은 관계형 데이터베이스뿐만 아니라 외부 API 서비스 같은 비관계형 데이터 연결도 지원합니다.
- **데이터 소스 관리**: NocoBase에서 데이터 소스를 쉽게 추가, 편집, 삭제하고 그룹으로 묶어 관리할 수 있습니다.
- **데이터 소스 격리**: 각 데이터 소스는 독립적이며, 데이터 소스별로 다른 권한을 설정하여 데이터 격리 및 보안을 구현할 수 있습니다.
- **데이터 소스 확장**: NocoBase는 유연한 플러그인 메커니즘을 제공하여, 플러그인 개발을 통해 더 많은 유형의 데이터 소스를 지원하도록 확장할 수 있습니다.
