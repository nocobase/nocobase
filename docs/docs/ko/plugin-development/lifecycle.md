# 생명 주기

이 섹션에서는 플러그인의 서버 측과 클라이언트 측 생명 주기 훅을 정리하여, 개발자가 리소스를 올바르게 등록하고 해제할 수 있도록 돕습니다.

FlowModel의 생명 주기와 비교하여 공통적인 개념을 이해하는 데 활용할 수 있습니다.

## 권장 내용

- 플러그인이 설치, 활성화, 비활성화 또는 제거될 때 트리거되는 콜백.
- 클라이언트 측 컴포넌트의 마운트, 업데이트 및 소멸 시점.
- 생명 주기 내에서 비동기 작업 및 오류를 처리하기 위한 권장 사항.
