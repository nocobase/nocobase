# 계산 엔진

계산 엔진은 수식 필드, 연동 규칙, 워크플로우에서 핵심적인 역할을 하며, 범용적인 계산 도구를 제공합니다.

## Formula.js

**Formula.js**는 수식 계산을 위해 설계된 경량 JavaScript 라이브러리입니다. 광범위한 수학 및 통계 함수 세트를 제공하여 사용자가 복잡한 계산을 쉽게 수행할 수 있도록 돕습니다. 주요 특징은 다음과 같습니다:

- **풍부한 기능**: 다양한 수학 연산을 아우르는 광범위한 함수 세트를 제공합니다.
- **사용 용이성**: 단순한 API를 제공하여 개발자와 비개발자 모두 방대한 코드 작성 없이 수식을 만들 수 있습니다.
- **실시간 계산**: 실시간 데이터 처리에 최적화되어 즉각적인 계산 피드백이 필요한 애플리케이션에 적합합니다.

## Math.js

**Math.js**는 기능이 풍부한 JavaScript 및 Node.js용 수학 라이브러리입니다. 기본 산술부터 고급 대수학, 통계, 선형 대수학에 이르기까지 광범위한 수학 연산을 지원합니다. 주요 특징은 다음과 같습니다:

- **포괄적인 함수 세트**: 모든 유형의 수학 연산을 위한 방대한 함수를 제공합니다.
- **유연한 표현식 파싱**: 문자열 형식의 수학 표현식을 해석하고 계산할 수 있습니다.
- **고급 계산**: 행렬 연산, 기호 계산, 통계 분석 등의 기능을 지원합니다.
- **확장성**: 특정 요구 사항에 맞춰 사용자 정의 함수와 상수를 추가하여 확장할 수 있습니다.

## NocoBase에서의 응용

Formula.js와 Math.js를 결합하여 사용함으로써 NocoBase는 다음과 같은 이점을 제공합니다:

- **광범위한 계산 수행**: 단순한 연산부터 복잡한 수학적 작업까지 수행할 수 있습니다.
- **견고하고 효율적인 애플리케이션 구축**: 정밀하고 강력한 계산 기능이 필요한 애플리케이션을 제작할 수 있습니다.
- **생산성 향상**: 미리 빌드된 함수를 활용하여 방대한 사용자 정의 코드 작성을 피하고 개발 효율성을 높일 수 있습니다.

이 두 라이브러리는 NocoBase에서 핵심적인 역할을 하며, 심도 있는 프로그래밍 없이도 유연하고 사용자 정의 가능한 계산 로직을 만들 수 있게 해줍니다. 이는 개발 과정을 단순화할 뿐만 아니라 기능성과 유연성을 강화합니다.
