# 워크플로우 출력

## 소개

"워크플로우 출력" 노드는 호출된 워크플로우에서 해당 워크플로우의 출력 값을 정의하는 데 사용됩니다. 하나의 워크플로우가 다른 워크플로우에 의해 호출될 때, "워크플로우 출력" 노드를 통해 호출자에게 값을 다시 전달할 수 있습니다.

## 노드 생성

호출된 워크플로우에서 "워크플로우 출력" 노드를 추가합니다.

![20241231002033](https://static-docs.nocobase.com/20241231002033.png)

## 노드 설정

### 출력 값

출력 값으로 변수를 입력하거나 선택합니다. 출력 값은 모든 유형이 될 수 있으며, 문자열, 숫자, 논리값, 날짜 또는 사용자 정의 JSON과 같은 상수일 수도 있고, 워크플로우 내의 다른 변수일 수도 있습니다.

![20241231003059](https://static-docs.nocobase.com/20241231003059.png)

:::info{title=팁}
호출된 워크플로우에 여러 개의 "워크플로우 출력" 노드가 추가된 경우, 해당 워크플로우를 호출할 때 마지막으로 실행된 "워크플로우 출력" 노드의 값으로 출력됩니다.
:::
