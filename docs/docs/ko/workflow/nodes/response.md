# HTTP 응답

## 소개

이 노드는 동기 모드 Webhook 워크플로우에서만 지원되며, 제3자 시스템에 응답을 반환하는 데 사용됩니다. 예를 들어, 결제 콜백 처리 과정에서 비즈니스 처리에 예상치 못한 결과(예: 오류, 실패 등)가 발생한 경우, 응답 노드를 통해 제3자 시스템에 오류 응답을 반환하여, 제3자 시스템이 상태에 따라 나중에 재시도할 수 있도록 합니다.

또한, 응답 노드의 실행은 워크플로우의 실행을 종료시키며, 이후 노드는 더 이상 실행되지 않습니다. 전체 워크플로우에 응답 노드가 구성되지 않은 경우, 시스템은 워크플로우 실행 상태에 따라 자동으로 응답합니다. 실행에 성공하면 `200`을, 실패하면 `500`을 반환합니다.

## 응답 노드 생성하기

워크플로우 설정 화면에서, 워크플로우 내의 더하기("+"") 버튼을 클릭하여 “응답” 노드를 추가합니다.
