# Notification : WeCom

## Introduction

Le **plugin WeCom** permet à l'application d'envoyer des messages de notification aux utilisateurs de WeCom.

## Ajouter et configurer un authentificateur WeCom

Vous devez d'abord ajouter et configurer un authentificateur WeCom sur NocoBase. Consultez [Authentification utilisateur - WeCom](/auth-verification/auth-wecom). Seuls les utilisateurs système qui se sont connectés via WeCom peuvent recevoir des notifications système via WeCom.

## Configuration du nœud de notification du flux de travail

Sélectionnez le canal de notification WeCom configuré. Il prend en charge trois types de messages : Carte de texte, Markdown et Carte de modèle.
