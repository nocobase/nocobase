# Agent IA · Dex : Expert en organisation de données

> Extrayez et remplissez rapidement des formulaires, transformant les informations désordonnées en données structurées.

`Dex` est un expert en organisation de données. Il extrait les informations nécessaires à partir de données ou de fichiers non structurés, les organise en informations structurées et peut utiliser des outils pour remplir des formulaires.

## Étapes

Sur la page du formulaire, appelez `Dex` pour ouvrir la fenêtre de dialogue.

![20251022155316](https://static-docs.nocobase.com/20251022155316.png)

Cliquez sur `Add work context` dans le champ de saisie et sélectionnez `Pick block`. La page passe alors en mode de sélection de bloc.

![20251022155414](https://static-docs.nocobase.com/20251022155414.png)

Sélectionnez le bloc de formulaire sur la page.

![20251022155523](https://static-docs.nocobase.com/20251022155523.png)

Dans la boîte de dialogue, saisissez les données que vous souhaitez que `Dex` organise.

![20251022155555](https://static-docs.nocobase.com/20251022155555.png)

Après l'envoi, `Dex` structurera les données et utilisera ses compétences pour mettre à jour les données dans le formulaire sélectionné.
