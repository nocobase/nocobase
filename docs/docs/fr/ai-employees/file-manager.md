# Gestion des fichiers

## Introduction

Configurez la méthode de stockage pour les fichiers téléchargés lors des conversations avec les employés IA.

## Paramètres

Accédez à la page de configuration du plugin Employé IA, puis cliquez sur l'onglet `Settings` pour accéder à la page des paramètres de la méthode de stockage.
