# Ajouter du contexte - Bloc

Lorsque vous discutez avec un employé IA, en plus de l'envoi de fichiers, vous pouvez également envoyer des informations de contexte de l'application à l'employé IA, lui permettant de répondre en se basant sur ces informations.

### Envoyer des informations de bloc

Dans la boîte de dialogue de l'employé IA, cliquez sur le bouton `Ajouter du contexte` en bas à gauche et sélectionnez `Choisir un bloc`. La page de l'application passera en mode de sélection de bloc.

![20251022010148](https://static-docs.nocobase.com/20251022010148.png)

Lorsque vous survolez un bloc avec la souris, la couleur des blocs pouvant être envoyés comme contexte changera.

![20251022010422](https://static-docs.nocobase.com/20251022010422.png)

Après avoir sélectionné le bloc à envoyer, les informations de contexte du bloc seront jointes à la boîte de dialogue.

![20251022010543](https://static-docs.nocobase.com/20251022010543.png)

Après avoir saisi votre question, comme pour l'envoi d'un message classique, cliquez sur le bouton d'envoi. L'employé IA répondra en se basant sur les informations du bloc.

Les informations du bloc apparaîtront également dans la liste des messages.

![20251022010758](https://static-docs.nocobase.com/20251022010758.png)

### Quitter la sélection de bloc

Si la page de l'application est en mode de sélection de bloc mais que vous ne souhaitez finalement pas en sélectionner, cliquez sur l'icône « Arrêter » en bas pour quitter ce mode.

![20251022011043](https://static-docs.nocobase.com/20251022011043.png)
