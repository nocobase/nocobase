# Générer un rapport d'analyse métier

En prenant l'exemple du système CRM de la DEMO en ligne, vous allez demander à un AI Employee d'analyser les données clients et commandes du système pour générer un rapport d'analyse.

## Charger les compétences

Tout d'abord, l'AI Employee charge les compétences pertinentes, puis commence le travail.

![](https://static-docs.nocobase.com/202604230924035.png)

## Analyser le modèle de données

Récupérez la source de données et les métadonnées des collections.

![](https://static-docs.nocobase.com/202604230926297.png)

## Interroger les données métier

Le processus d'interrogation des données métier commence. Selon le contenu de l'analyse, ce processus peut prendre de quelques dizaines de secondes à plusieurs minutes.

![](https://static-docs.nocobase.com/202604230927861.png)

## Générer le rapport

Une fois suffisamment de données métier collectées, la génération du rapport d'analyse commence.

![](https://static-docs.nocobase.com/202604230920113.png)

Un rapport d'analyse métier illustré est finalement généré, téléchargeable aux formats Markdown, HTML et PDF.

![](https://static-docs.nocobase.com/202604230922030.png)

![](https://static-docs.nocobase.com/202604230922376.png)

![](https://static-docs.nocobase.com/202604230923542.png)

![](https://static-docs.nocobase.com/202604230923538.png)

![](https://static-docs.nocobase.com/202604230923049.png)

Note : si vous téléchargez au format Markdown, les graphiques doivent être téléchargés séparément depuis la page web.
