# Démarrage rapide

## Introduction

Après avoir créé un employé IA, vous pouvez collaborer avec lui sur n'importe quelle page de NocoBase.

Vous pouvez cliquer sur l'avatar de l'employé dans la liste des employés en bas à droite de la page pour le faire apparaître et commencer à collaborer.

![20251102121159-2025-11-02-12-12-01](https://static-docs.nocobase.com/20251102121159-2025-11-02-12-12-01.png)

Vous pouvez également positionner les employés IA là où vous en avez besoin — sur des tableaux, des formulaires ou d'autres composants — et les activer directement d'un simple clic.

![20251102121036-2025-11-02-12-10-38](https://static-docs.nocobase.com/20251102121036-2025-11-02-12-10-38.png)

## Opérations de base dans la boîte de dialogue

Une fois l'employé IA activé, saisissez votre message et cliquez sur le bouton d'envoi pour démarrer la conversation.

![20251021234541](https://static-docs.nocobase.com/20251021234541.png)

Lorsque vous survolez un message utilisateur, des icônes d'actions rapides apparaissent :
- Modifier et renvoyer le message
- Copier le contenu du message utilisateur

![20251021235715](https://static-docs.nocobase.com/20251021235715.png)

Lorsque vous survolez un message de l'IA, des icônes d'actions rapides apparaissent :
- Demander à l'IA de répondre à nouveau
- Copier le contenu du message de l'IA

![20251021235847](https://static-docs.nocobase.com/20251021235847.png)

Cliquez sur l'icône de téléversement pour envoyer un fichier à l'employé IA.

![20251022000327](https://static-docs.nocobase.com/20251022000327.png)

Dans le coin supérieur gauche de la boîte de dialogue se trouve la liste de l'historique des conversations. Cliquez pour développer la liste et consulter les conversations passées.

![20251021234640](https://static-docs.nocobase.com/20251021234640.png)

Cliquez sur l'icône « crayon » dans le coin supérieur droit pour démarrer une nouvelle conversation.

![20251021234841](https://static-docs.nocobase.com/20251021234841.png)

Cliquez sur l'icône « i » dans le coin supérieur droit pour modifier l'invite système personnalisée.

![20251021235314](https://static-docs.nocobase.com/20251021235314.png)

Cliquez sur l'icône « agrandir » dans le coin supérieur droit pour afficher la boîte de dialogue de l'employé IA en plein écran.
