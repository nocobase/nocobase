# Démarrage rapide

## Introduction
Une fois la configuration du service LLM terminée, vous pouvez commencer à créer des employés IA.

## Nouvel employé IA

### Configuration des informations de base

Accédez à la page de configuration du plugin `employé IA`, puis cliquez sur l'onglet `AI employees` pour accéder à la page de gestion des employés IA.

![20251021225643](https://static-docs.nocobase.com/20251021225643.png)

Cliquez sur le bouton `New AI employee` pour accéder à la page de création d'un employé IA.

Tout d'abord, dans l'onglet `Profile`, vous pouvez configurer les informations de base de l'employé IA :

- Dans le champ `Username`, saisissez l'identifiant unique de l'employé IA ;
- Dans le champ `Nickname`, saisissez le surnom de l'employé IA ;
- Dans le champ `Position`, saisissez le poste de l'employé IA pour décrire brièvement ses responsabilités ;
- Dans `Avatar`, sélectionnez un avatar pour l'employé IA.

![20251021232246](https://static-docs.nocobase.com/20251021232246.png)

- Dans le champ `Bio`, saisissez le texte de présentation de l'employé IA, décrivant ses compétences et ses fonctions. Ce texte sera affiché dans le profil de l'employé IA ;
- Dans le champ `About me`, saisissez une description détaillée des fonctions de l'employé IA. Ce texte sera utilisé comme invite système lors des conversations avec l'employé IA.
- Dans le champ `Greeting message`, saisissez le message d'accueil de l'employé IA. Chaque fois qu'un utilisateur entame une nouvelle conversation avec l'employé IA, le système lui enverra automatiquement ce message.

![20251021232712](https://static-docs.nocobase.com/20251021232712.png)

### Configuration du grand modèle

Dans l'onglet `Model settings`, vous pouvez configurer les paramètres du modèle de l'employé IA :

- Sélectionnez le service LLM que vous avez créé dans la gestion des services LLM ;
- Saisissez le nom du grand modèle que vous souhaitez utiliser.

![20251021231727](https://static-docs.nocobase.com/20251021231727.png)

### Finaliser la création

Après avoir saisi toutes les informations nécessaires à la création de l'employé IA, cliquez sur le bouton `Submit` pour finaliser la création.

Vous verrez ensuite l'employé IA que vous venez de créer apparaître dans la liste des employés IA. Il sera également visible dans le bouton de lancement rapide des employés IA, situé en bas à droite de la page.

![20251021232836](https://static-docs.nocobase.com/20251021232836.png)
