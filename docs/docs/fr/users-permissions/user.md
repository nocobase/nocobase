# Utilisateurs

## Gestion des utilisateurs

Ce plugin offre une interface de gestion des utilisateurs dans le centre de configuration. Après l'initialisation de l'application, un compte super administrateur non supprimable est automatiquement créé. Cet administrateur dispose des permissions Root et peut accéder à toutes les ressources.

![](https://static-docs.nocobase.com/44bf40f56b45d4dd96c424fb08082cf6.png)

De plus, vous pouvez également utiliser la collection d'utilisateurs sur une page pour ajouter différents blocs afin de gérer les utilisateurs, tels que des blocs de tableau :

![](https://static-docs.nocobase.com/76b5a4652f869541a9e8f18a4568a7c9.png)
