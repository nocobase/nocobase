# Intégrer NocoBase

## Introduction

Ce plugin permet d'intégrer les pages NocoBase dans d'autres sites web ou applications.

## Installation

Activez le plugin « Intégrer NocoBase » dans le gestionnaire de plugins pour l'utiliser.

## Guide d'utilisation

Après avoir activé le plugin, cliquez sur « Copier le lien d'intégration » dans le menu de configuration de la page (en haut à droite).

![](https://static-docs.nocobase.com/f11bd6d5e88d38731d7cd3cb149022c8.png)

Après avoir cliqué, vous obtiendrez un lien que vous pourrez ouvrir séparément, par exemple, `https://example.com/embed/qs087rz4o2b`.

![](https://static-docs.nocobase.com/9d847805a00fd07372a474665ec0e968.png)

Si vous souhaitez intégrer des pages NocoBase dans d'autres sites web ou applications, une authentification utilisateur est nécessaire. Vous devrez alors ajouter le `token` au lien, par exemple, `https://example.com/embed/qs087rz4o2b?token=xxx`.
