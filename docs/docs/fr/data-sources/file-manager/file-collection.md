# Collection de fichiers

## Introduction

Les fichiers sont des enregistrements au sein d'une collection structurée de manière spécifique, appelée collection de fichiers. Cette collection est utilisée pour stocker les métadonnées des fichiers et peut être gérée via le Gestionnaire de fichiers.
