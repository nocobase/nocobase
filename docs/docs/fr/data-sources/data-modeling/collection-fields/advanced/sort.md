# Sélecteur de collection

Le champ Sélecteur de collection est un type de champ spécial qui permet aux utilisateurs de sélectionner une ou plusieurs collections à partir d'une liste prédéfinie. Ce type de champ est très utile dans les scénarios où vous devez référencer dynamiquement différentes collections, comme la création de rapports configurables, de tableaux de bord ou d'applications complexes qui doivent modifier leurs sources de données en fonction de la sélection de l'utilisateur.

## Cas d'utilisation

### Génération de rapports dynamiques

Dans un module de génération de rapports, les utilisateurs peuvent avoir besoin de sélectionner une collection parmi plusieurs collections comme source de données pour le rapport. En utilisant le Sélecteur de collection, les utilisateurs peuvent facilement sélectionner la collection souhaitée, et le système générera dynamiquement le rapport en fonction de la sélection de l'utilisateur.

### Tableaux de bord configurables

Dans une application de tableau de bord, les utilisateurs peuvent personnaliser les données affichées sur le tableau de bord. Le Sélecteur de collection permet aux utilisateurs de choisir les collections qui les intéressent, affichant ainsi des graphiques et des données statistiques provenant de différentes sources de données sur le tableau de bord.

### Outils flexibles d'importation/exportation de données

Lors du développement d'un outil générique d'importation ou d'exportation de données, vous pouvez utiliser le Sélecteur de collection pour permettre aux utilisateurs de spécifier la collection sur laquelle opérer. De cette manière, le même outil peut être utilisé pour traiter les données de n'importe quelle collection du système, améliorant considérablement la réutilisabilité de l'outil.
