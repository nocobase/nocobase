# Collection arborescente

## Introduction

Une collection arborescente est un modèle de conception de collection de données utilisé pour organiser des données hiérarchiques. Cette structure de collection ressemble à une arborescence, où chaque élément de données peut avoir un ou plusieurs éléments enfants, et ces éléments enfants peuvent à leur tour avoir leurs propres descendants.
