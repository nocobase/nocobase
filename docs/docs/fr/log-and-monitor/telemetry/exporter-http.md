# Exportateur de télémétrie : HTTP

## Configuration des variables d'environnement

### TELEMETRY_METRIC_READER

Type d'exportateur de métriques de télémétrie.

```bash
TELEMETRY_METRIC_READER=http
```

### TELEMETRY_HTTP_URL

URL HTTP pour l'exportation des données de télémétrie.

```bash
TELEMETRY_HTTP_URL=
```
