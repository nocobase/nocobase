# Verrouillage des utilisateurs

## Introduction

Gérer les utilisateurs verrouillés et verrouiller proactivement des utilisateurs.

![](https://static-docs.nocobase.com/202412281450083.png)

### Déverrouiller un utilisateur

Les utilisateurs qui ont été verrouillés suite à un nombre excessif de tentatives de connexion avec un mot de passe invalide peuvent être déverrouillés en supprimant leur enregistrement.

### Verrouiller un utilisateur

Vous pouvez verrouiller proactivement un utilisateur en ajoutant un enregistrement utilisateur. Une fois qu'un utilisateur est verrouillé, il lui sera interdit d'accéder au système par quelque méthode d'authentification que ce soit, y compris les clés API.

![](https://static-docs.nocobase.com/202412281450512.png)
