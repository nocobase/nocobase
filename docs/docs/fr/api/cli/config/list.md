# nb config list

Liste les réglages CLI configurés explicitement. Les valeurs par défaut qui n'ont jamais été définies explicitement ne sont pas affichées.

## Utilisation

```bash
nb config list
```

## Exemples

```bash
nb config list
```

## Remarques

Si aucune valeur de configuration n'est définie, le CLI affiche `No CLI config values are set.`.
