# nb session id

Affiche l’identifiant de session effectivement utilisé.

Si aucun `NB_SESSION_ID` exploitable n’est disponible dans le shell ou le runtime courant, la commande vous indique d’exécuter d’abord [`nb session setup`](./setup.md), puis d’ouvrir un nouveau shell ou runtime.

## Utilisation


nb session id

## Exemples


nb session id

## Commandes connexes

- [`nb session setup`](./setup.md)
- [`nb env current`](../env/current.md)
