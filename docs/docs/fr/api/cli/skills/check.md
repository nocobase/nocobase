# nb skills check

Vérifier les NocoBase AI coding skills globaux et indiquer s'ils sont gérés par le CLI ainsi que la disponibilité d'une mise à jour.

## Utilisation

```bash
nb skills check [flags]
```

## Paramètres

| Paramètre | Type | Description |
| --- | --- | --- |
| `--json` | boolean | Sortie au format JSON |

## Exemples

```bash
nb skills check
nb skills check --json
```
