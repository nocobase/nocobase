# nb proxy caddy current

Affiche le driver actuel du provider Caddy.

## Utilisation

```bash
nb proxy caddy current
```

## Sortie

La commande affiche directement une seule valeur :

- `local`
- `docker`

## Exemples

```bash
nb proxy caddy current
```
