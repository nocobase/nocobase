# nb proxy caddy status

Affiche l'état d'exécution actuel du proxy Caddy.

## Utilisation

```bash
nb proxy caddy status
```

## Sortie

La sortie inclut généralement :

- `driver`
- `status`
- `config`
- `caddy` ou `container`

## Exemples

```bash
nb proxy caddy status
```
