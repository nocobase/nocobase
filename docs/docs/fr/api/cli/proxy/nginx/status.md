# nb proxy nginx status

Affiche l'état d'exécution actuel du proxy Nginx.

## Utilisation

```bash
nb proxy nginx status
```

## Sortie

La sortie inclut généralement :

- `driver`
- `status`
- `config`
- `nginx` ou `container`

## Exemples

```bash
nb proxy nginx status
```
