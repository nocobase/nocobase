# nb proxy nginx current

Affiche le driver actuel du provider Nginx.

## Utilisation

```bash
nb proxy nginx current
```

## Sortie

La commande affiche directement une seule valeur :

- `local`
- `docker`

## Exemples

```bash
nb proxy nginx current
```
