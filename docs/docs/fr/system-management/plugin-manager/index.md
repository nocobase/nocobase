# Gestion des plugins

## Gestionnaire de plugins

Cliquez sur l'icône du gestionnaire de plugins en haut à droite de la page pour accéder à la page du gestionnaire de plugins. Sur cette page, vous pouvez consulter, installer, activer ou désactiver des plugins.

![2024-04-10_08-47-45](https://static-docs.nocobase.com/2024-04-10_08-47-45.png)

## Centre de réglages des plugins

Les plugins disposent généralement de leurs propres pages de réglages. Cliquez sur l'icône des réglages en haut à droite de la page pour afficher les menus de réglages de tous les plugins. Cliquez sur l'élément de menu correspondant pour configurer le plugin.
