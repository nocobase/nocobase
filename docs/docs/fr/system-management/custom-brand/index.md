# Personnaliser la marque

## Introduction

Le nom de marque par défaut est "NocoBase". Il s'affiche principalement à trois endroits :

![20240409111100](https://static-docs.nocobase.com/20240409111100.png)

En bas de la page de connexion :

![20240409111254](https://static-docs.nocobase.com/20240409111254.png)

Vous pouvez configurer l'emplacement 1 via l'option Logo dans les [Paramètres système](/system-management/system-settings/). Les emplacements 2 et 3 peuvent être configurés à l'aide de ce plugin.

## Instructions d'utilisation

Une fois le plugin activé, cliquez sur le menu « Personnaliser la marque » en haut à droite de la page pour accéder à la page de configuration.

![20240409113640](https://static-docs.nocobase.com/20240409113640.png)

- **Marque** : Permet de définir le texte en bas de la page de connexion (emplacement 3).
- **À propos** : Permet de configurer le contenu du menu en haut à droite de la page (comme illustré ci-dessous).

![20240409113915](https://static-docs.nocobase.com/20240409113915.png)

La configuration prend en charge le format HTML pour une personnalisation flexible du contenu. Si vous avez uniquement besoin d'apporter de simples modifications textuelles, vous pouvez directement modifier le contenu existant, comme illustré ci-dessous :

![20240409114740](https://static-docs.nocobase.com/20240409114740.png)
