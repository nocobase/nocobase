# Nginx

Si vous cherchez aujourd'hui à configurer le proxy de production pour une application NocoBase, commencez de préférence par [Reverse proxy en production](../../../nocobase-cli/production/reverse-proxy/index.md) puis par la page [Nginx](../../../nocobase-cli/production/reverse-proxy/nginx.md).

Cette section plus ancienne servait surtout de point d'entrée pour le proxy des ressources statiques. La documentation actuelle a été réorganisée autour de `nb proxy nginx`, qui couvre de manière unifiée la génération de configuration, l'exécution locale ou Docker, ainsi que les routes `uploads`, `dist`, `api`, `ws` et SPA.
