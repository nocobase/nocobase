# Déploiement en Cluster

NocoBase prend en charge le déploiement d'applications en mode cluster afin d'améliorer leurs performances et leur disponibilité. Le déploiement en cluster vous permet d'exécuter différentes instances de l'application sur plusieurs serveurs, ou de faire fonctionner plusieurs processus en mode multi-cœur sur un seul serveur.

Pour plus de détails, veuillez consulter la documentation sur le [mode cluster](/cluster-mode/).
