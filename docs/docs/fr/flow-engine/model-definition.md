# Champ Région Chine

## Introduction

Le plugin Champ Région Chine (field-china-region) ajoute le type de champ "Région Chine" à NocoBase.

## Installation

Dans l'interface principale de NocoBase, accédez au "Gestionnaire de plugins", trouvez le plugin "Région Chine" et activez-le.

## Utilisation

Dans la configuration de la collection, ajoutez un nouveau champ et sélectionnez "Région Chine" comme type de champ.

![](/images/field-china-region-1.png)

### Configuration du champ

Dans la configuration du champ, vous pouvez définir le mode d'affichage pour ce champ :

- Province
- Province/Ville
- Province/Ville/District
- Province/Ville/District/Comté
