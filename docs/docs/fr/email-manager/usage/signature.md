# Signature d'e-mail

## Configuration de la signature

Rendez-vous dans le Centre de gestion des e-mails et cliquez sur le bouton des paramètres :
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_05_34_PM.png)

Basculez vers le panneau des signatures :
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_05_51_PM.png)

Vous pouvez gérer les signatures ici. Cochez l'option située devant une signature pour la définir comme signature par défaut :
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_05_37_PM.png)

## Utilisation des signatures

Dans le corps du message, cliquez sur l'icône de signature pour sélectionner la signature souhaitée :
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_05_40_PM.png)
