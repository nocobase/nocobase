# Foire aux questions

## Impossible de recevoir des e-mails après l'authentification via un compte Microsoft

R: Actuellement, seuls les comptes de messagerie Outlook et Gmail sont pris en charge pour l'authentification autorisée. Les comptes Microsoft et Google ne sont pas encore pris en charge. Pour plus de détails, consultez : [answers.microsoft.com](https://answers.microsoft.com/zh-hans/outlook_com/forum/all/%E7%8E%B0%E6%9C%89%E5%BE%AE%E8%BD%AF%E8%B4%A6/dba12dda-a7c7-4346-8263-53f4a6d9dc68)

**Conseil** : Si vous n'êtes pas certain(e) d'utiliser un « véritable compte Outlook.com » ou un « compte Gmail », essayez de vous rendre sur Outlook.com ou Gmail.com via votre navigateur web. Vérifiez si vous pouvez vous connecter directement et envoyer des e-mails normalement. Si ce n'est pas le cas, cela signifie que vous ne disposez peut-être pas du service de messagerie correspondant et que vous devrez l'activer ou utiliser une autre adresse e-mail.

## Annuler l'autorisation

Si votre compte de messagerie est déjà autorisé et que vous souhaitez le supprimer ou le ré-autoriser, voici la marche à suivre :

#### **Gmail**

1. Ouvrez https://myaccount.google.com/u/0/connections et connectez-vous.

![](https://static-docs.nocobase.com/mail-1735634131347.png)

2. Cliquez sur l'application correspondante, puis cliquez sur « Supprimer ».

![](https://static-docs.nocobase.com/mail-1735634131518.png)

![](https://static-docs.nocobase.com/mail-1735634131697.png)

#### **Messagerie Microsoft**

1. Ouvrez https://account.microsoft.com/ et connectez-vous.
    
2. Cliquez sur le bouton « Applications et services ayant accès à vos données ».

![](https://static-docs.nocobase.com/mail-1735634131870.png)

3. Cliquez sur « Modifier » puis sur « Supprimer ».

![](https://static-docs.nocobase.com/mail-1735634132052.png)
