# Enregistrement automatique du brouillon

Dans le menu du bouton **Enregistrer le brouillon**, vous pouvez activer l'option **Enregistrement automatique du brouillon**.
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_29_PM.png)

Lors de l'édition de l'e-mail, le système enregistre le brouillon en temps réel, afin d'éviter toute perte de contenu.
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_31_PM.png)

Lorsque vous rouvrez l'éditeur d'e-mail (pour le même expéditeur et destinataire), le système vous demandera si vous souhaitez utiliser le brouillon enregistré.
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_08_35_PM.png)
