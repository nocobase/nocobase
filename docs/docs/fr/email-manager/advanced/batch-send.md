# Envoi en masse

## Aperçu de l'envoi en masse
Vous pouvez envoyer des e-mails à plusieurs personnes en masse (chaque destinataire recevra un e-mail distinct).

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_06_10_PM.png)

## Envoi en masse via un tableau
Vous pouvez également envoyer des e-mails en masse via un tableau :

1. Ajoutez un bouton d'envoi d'e-mail au tableau
   ![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_14_PM.png)

2. Configurez le champ d'e-mail dans le tableau (utilisé pour analyser les adresses e-mail après avoir sélectionné des lignes du tableau)
   ![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_13_PM.png)

3. Cochez les données du tableau pour envoyer des e-mails
   ![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_15_PM.png)

Si vous devez envoyer des e-mails à toutes les adresses du tableau, vous pouvez activer la fonction **Sélectionner tous les enregistrements du tableau actuel comme destinataires par défaut** :
![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_16_PM.png)

Après avoir modifié le contenu de l'e-mail, cliquez sur le bouton d'envoi en masse pour l'envoyer :
![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_17_PM.png)
