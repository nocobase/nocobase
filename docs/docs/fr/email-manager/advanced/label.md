# Étiquettes d'e-mail

Dans les détails de l'e-mail, sur la droite, vous pouvez **sélectionner des étiquettes** pour un e-mail :
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_06_05_PM.png)
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_06_06_PM.png)

Si vous créez une nouvelle étiquette, elle sera sélectionnée par défaut. Vous pouvez également sélectionner manuellement d'autres étiquettes :
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_06_06_PM%20(1).png)

Dans la liste des e-mails, vous pouvez consulter les étiquettes associées à chaque e-mail :
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_06_08_PM.png)
