# Affectation de champ

## Introduction

L'affectation de champ consiste à prédéfinir la valeur d'un champ avant d'exécuter une opération. Lors de la mise à jour ou de la création d'un enregistrement, ces données sont automatiquement renseignées, vous évitant ainsi une saisie manuelle. C'est utile dans des scénarios comme la mise à jour en masse des statuts de commande, ou la définition automatique de l'heure de livraison à l'heure actuelle lorsqu'une commande est livrée.
