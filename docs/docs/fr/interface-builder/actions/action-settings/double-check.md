# Confirmation

## Introduction

La configuration de la confirmation est utilisée pour s'assurer qu'une action est bien exécutée. Un dialogue de confirmation apparaît lorsque vous cliquez sur l'action. Le bouton de suppression a la confirmation activée par défaut afin d'éviter toute perte de données ou des résultats irréversibles dus à des opérations accidentelles.

Vous pouvez personnaliser le contenu et le titre de la confirmation, et accéder à des variables.
