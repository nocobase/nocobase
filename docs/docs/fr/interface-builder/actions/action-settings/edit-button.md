# Modifier un bouton

## Introduction

Le système vous permet de personnaliser le nom, l'icône, la couleur d'arrière-plan et le type d'un bouton. Vous disposez ainsi d'options de personnalisation d'interface utilisateur flexibles pour ajuster le style d'affichage et les fonctionnalités du bouton selon vos besoins spécifiques.

- **Nom du bouton** : Vous pouvez personnaliser le texte du bouton en fonction de vos besoins métier, afin de garantir une étiquette claire et intuitive.
- **Icône du bouton** : Le système prend en charge l'ajout d'une icône au bouton pour améliorer la reconnaissance visuelle et permettre aux utilisateurs d'identifier rapidement sa fonction.
- **Action dangereuse** : Lorsqu'un bouton représente une **action dangereuse** (par exemple, supprimer, réinitialiser, etc.), vous pouvez le définir comme étant de type **Danger Action**.
- **Type de bouton (Button Type)** : Les boutons prennent en charge différents types de configuration. Les types de boutons courants incluent :
  - **Par défaut (Default)** : Un bouton standard, adapté aux opérations courantes.
  - **Principal (Primary)** : Utilisé pour les actions importantes ou primaires, généralement avec un effet visuel proéminent.
  - **Pointillé (Dashed)** : Un bouton avec une bordure en pointillés, adapté aux actions secondaires ou aux boutons d'indication.
  - **Texte (Text)** : Un bouton composé uniquement de texte et sans couleur d'arrière-plan, généralement utilisé pour les actions légères ou les liens.
