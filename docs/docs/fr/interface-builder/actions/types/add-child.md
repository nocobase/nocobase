# Ajouter un enregistrement enfant

## Introduction

Cette opération est spécialement conçue pour les **blocs de tableau arborescent** afin d'ajouter rapidement des enregistrements enfants. Lors de l'ajout d'un enregistrement enfant, la ligne sélectionnée est considérée comme le nœud parent, ce qui vous permet de créer facilement des enregistrements enfants sous n'importe quel parent. Cette fonctionnalité simplifie considérablement la gestion des données hiérarchiques, rendant l'extension et l'organisation des structures de données plus efficaces et intuitives.
