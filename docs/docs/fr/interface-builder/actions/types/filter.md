# Filtrer

## Introduction

L'action de filtrage se trouve généralement dans les blocs de données. Elle permet de filtrer les données en configurant diverses conditions. Veuillez noter que si une portée de données est déjà configurée pour le bloc, les conditions de l'action de filtrage seront combinées avec celles de la portée de données pour former la condition de filtrage finale.

### Champs filtrables

Par défaut, tous les champs de cette collection sont activés. Seuls les champs activés peuvent être sélectionnés dans la liste des champs de condition.

Les champs des collections associées peuvent également être utilisés comme champs de filtrage.
