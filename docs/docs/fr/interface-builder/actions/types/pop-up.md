# Ouvrir une fenêtre modale

## Introduction

L'action **Ouvrir une fenêtre modale** est similaire aux actions d'affichage et d'édition. Vous pouvez configurer un bouton **Ouvrir une fenêtre modale** dans les lignes ou l'en-tête d'un tableau. En cliquant sur ce bouton, une fenêtre modale correspondante s'ouvre, et les blocs qu'elle contient affichent les données en fonction de l'enregistrement du contexte actuel.

![20251029182351](https://static-docs.nocobase.com/20251029182351.png)

Configurez les blocs dans la fenêtre modale selon vos besoins.

![20251029190208](https://static-docs.nocobase.com/20251029190208.png)

## Paramètres de l'action

![20251029190352](https://static-docs.nocobase.com/20251029190352.png)

- [Modifier le bouton](/interface-builder/actions/action-settings/edit-button) : Modifiez le titre, le type et l'icône du bouton.
- [Règle de liaison](/interface-builder/linkage-rule) : Contrôlez dynamiquement l'état du bouton (visible/désactivé).
- [Modifier la fenêtre modale](/interface-builder/actions/action-settings/edit-popup) : Définissez le mode d'ouverture (fenêtre modale ou tiroir) et la taille.
