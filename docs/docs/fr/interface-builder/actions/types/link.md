# Lien

## Introduction

L'action de lien utilise la navigation par route pour prendre en charge le passage de variables d'URL. La page cible peut ainsi ajuster dynamiquement son contenu en fonction des données transmises. Vous pouvez la configurer et l'utiliser dans les blocs de données.
