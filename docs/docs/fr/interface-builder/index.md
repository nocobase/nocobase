# Construction de l'interface

## WYSIWYG

NocoBase vous offre une expérience de construction d'interface utilisateur en mode WYSIWYG (What You See Is What You Get). Cliquez sur le bouton 'Construction de l'interface' pour basculer entre le mode d'édition et le mode de visualisation.

## Mise en page de l'interface

NocoBase propose une mise en page bureau et une mise en page mobile. La mise en page bureau convient aux opérations courantes d'administration et adapte aussi la navigation et le contenu des pages sur les téléphones. La mise en page mobile offre une navigation et des pages propres aux appareils mobiles. Consultez [Mise en page de l'interface](./ui-layout/index.md) pour plus de détails.

## Options de configuration

En mode d'édition, des marqueurs orange indiquent les éléments configurables. Le point d'entrée pour les paramètres d'un élément se trouve généralement dans son coin supérieur droit.
