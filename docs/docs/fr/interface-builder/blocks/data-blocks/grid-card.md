# Grille de cartes

## Introduction

Le bloc Grille de cartes affiche les informations récapitulatives des enregistrements de données sous forme de cartes. Il permet de configurer le nombre de colonnes en fonction des différentes tailles d'écran, afin de garantir un affichage convivial sur divers appareils.
