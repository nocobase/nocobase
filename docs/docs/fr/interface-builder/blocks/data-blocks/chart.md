# Graphique

## Introduction

Le plugin de visualisation de données de NocoBase offre des requêtes de données visuelles et un ensemble riche de composants graphiques. Grâce à une configuration simple, vous pouvez rapidement créer des tableaux de bord de visualisation pour présenter des aperçus de données et prendre en charge l'analyse et la présentation de données multidimensionnelles.
