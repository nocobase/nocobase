# Mode de chargement des données

## Introduction

Le mode de chargement des données par défaut est « Charger toutes les données lorsque les conditions de filtrage sont vides ». Ici, les « conditions de filtrage » font référence aux valeurs du « Bloc de filtrage ». Une autre option est « Ne pas charger de données lorsque les conditions de filtrage sont vides ».
