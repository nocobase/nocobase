# Bloc : Plan de travail

## Introduction

Ce bloc vous permet de placer diverses actions rapides. Il prend actuellement en charge :

- [Lien](/interface-builder/actions/types/link)
- Scanner un code QR
- [Action de pop-up](/interface-builder/actions/types/pop-up)

![20251023225754](https://static-docs.nocobase.com/20251023225754.png)

> 💡 D'autres types d'actions seront progressivement ajoutés dans les futures versions.
## Types d'actions

- **Lien** : Redirige vers une page spécifique ou une URL externe.

![20251023230643](https://static-docs.nocobase.com/20251023230643.png)

- **Scanner un code QR** : Déclenche une action correspondante en scannant un code QR.

- **Ouvrir une pop-up** : Ouvre une pop-up pour afficher le contenu du bloc.
  ![20251023230522](https://static-docs.nocobase.com/20251023230522.png)
