# Brouillons de formulaire

Lorsque les brouillons de formulaire sont activés, le contenu en cours de saisie est automatiquement enregistré localement. Même si vous fermez le formulaire par inadvertance ou si vous actualisez la page, les données saisies ne seront pas perdues.

![20251213093717](https://static-docs.nocobase.com/20251213093717.png)

Lorsqu'un brouillon existe, une notification s'affiche en haut du formulaire. Le brouillon est automatiquement supprimé après une soumission réussie ; s'il n'est plus nécessaire, vous pouvez également le supprimer manuellement à tout moment.

![20251213092734](https://static-docs.nocobase.com/20251213092734.png)

::: tip Astuce

Actuellement, les brouillons de formulaire ne prennent en charge que le cache local. Les versions ultérieures prendront en charge le cache côté serveur pour permettre la synchronisation entre plusieurs appareils.

:::
