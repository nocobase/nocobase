# Champ de titre

## Introduction

Le **champ de titre** est généralement utilisé dans les composants de champ de relation comme identifiant pour les données associées affichées dans l'interface. Il détermine la manière dont les informations principales des données sont présentées, aidant généralement les utilisateurs à identifier et à comprendre rapidement les données.

Différents types de champs de titre correspondent à différents composants de champ et prennent en charge les options de configuration correspondantes. Par exemple :

- Lorsque le champ de titre est un **champ de date**, le composant de champ utilise un **composant de date** pour afficher les données.
- Lorsque le champ de titre est un **champ de texte**, le composant de champ utilise un **composant de texte** pour afficher les données.
- Lorsque le champ de titre est un **champ d'option**, le composant de champ utilise un **composant d'option** pour afficher les données.
