# Modifier l'info-bulle

## Introduction

Une info-bulle de champ est le texte qui apparaît lorsque vous survolez un champ. Elle est conçue pour fournir des explications supplémentaires ou des conseils concernant ce champ. Ces info-bulles incluent généralement des informations telles que l'objectif du champ, son format de saisie et ses contraintes, afin d'aider les utilisateurs à mieux comprendre et utiliser le champ.
