# Afficher les étiquettes

## Introduction

Par défaut, les champs d'un bloc affichent des étiquettes. Celles-ci servent à identifier la signification de chaque champ, facilitant ainsi la lecture et la compréhension du contenu des données pour les utilisateurs.

Dans certains scénarios (par exemple, sur des pages d'affichage uniquement ou lorsque la signification des champs est déjà très claire), vous pouvez choisir de masquer les étiquettes pour obtenir une interface plus épurée. Une fois les étiquettes masquées, seules les valeurs des champs sont affichées, et les noms des champs ne sont plus visibles.
