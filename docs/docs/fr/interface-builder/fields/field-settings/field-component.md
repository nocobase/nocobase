# Composant de champ

## Introduction

Les composants de champ servent à afficher le contenu d'un champ.
Certains champs intègrent plusieurs composants d'affichage. Vous pouvez choisir le composant approprié en fonction de vos besoins réels pour présenter la valeur du champ de différentes manières.

Exemple : En mode lecture, un champ URL peut être basculé vers un composant de prévisualisation ou un composant URL (par défaut).
