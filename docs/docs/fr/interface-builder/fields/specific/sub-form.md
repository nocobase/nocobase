# Sous-formulaire

## Introduction

Les sous-formulaires sont adaptés aux scénarios où vous créez d'abord de nouvelles données associées avant de les lier. Les données associées à plusieurs niveaux sont clairement affichées sous forme de formulaire imbriqué. Comparé aux sélecteurs de données et aux composants de liste déroulante, un sous-formulaire vous permet de gérer directement les champs de la collection associée au sein du bloc de page actuel, et les données associées sont soumises avec le formulaire principal.
