# Gestionnaire de fichiers

## Introduction

Le Gestionnaire de fichiers est un composant de champ d'association utilisé pour gérer les relations où la collection cible est la collection de fichiers.

![20251027194124](https://static-docs.nocobase.com/20251027194124.png)

## Configuration du champ

- [Composant de champ](/interface-builder/fields/association-field) : Permet de passer à d'autres composants de champ d'association, tels que la liste déroulante, le sélecteur de données, etc.
- Autoriser la sélection de fichiers existants : Activé par défaut, vous permet de sélectionner et d'associer des fichiers parmi ceux déjà téléchargés.
