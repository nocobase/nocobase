# Titre

## Introduction

En mode lecture, les champs d'association sont affichés par défaut via le **composant de champ de titre**. Ce composant identifie l'enregistrement associé en affichant la valeur du champ de titre. Le système sélectionne et affiche automatiquement le composant de champ de titre approprié en fonction du type de champ de titre.
