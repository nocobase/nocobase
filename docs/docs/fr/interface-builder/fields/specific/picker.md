# Sélecteur de données

## Introduction

Le sélecteur de données, présenté sous forme de sélecteur de tableau contextuel, vous permet de sélectionner des données existantes d'une **collection** cible pour les associer, ou d'ajouter de nouvelles données à cette **collection** cible avant de les associer.

![20251027203343](https://static-docs.nocobase.com/20251027203343.png)

Les données déjà sélectionnées n'apparaissent pas dans le sélecteur de tableau.

![20251027203717](https://static-docs.nocobase.com/20251027203717.gif)

Dans le sélecteur de tableau, vous pouvez également gérer plus en détail la **collection** cible d'association (ajouter, supprimer, importer/exporter, etc.).

![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
