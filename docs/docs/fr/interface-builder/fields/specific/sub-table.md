# Sous-tableau

## Introduction

Le sous-tableau est idéal pour gérer les champs de relation un-à-plusieurs. Il permet de créer et d'associer en masse de nouvelles données dans la collection cible, ou de sélectionner et d'associer des données existantes.

## Instructions d'utilisation

![20251027223350](https://static-docs.nocobase.com/20251027223350.png)

Dans le sous-tableau, les différents types de champs affichent des composants de champ variés. Les champs volumineux (tels que le texte enrichi, JSON, ou le texte long) s'éditent via une fenêtre modale flottante.

![20251027223426](https://static-docs.nocobase.com/20251027223426.png)

Les champs de relation dans le sous-tableau.

Commandes (Un-à-plusieurs) > Produits de commande (Un-à-un) > Opportunité

![20251027223530](https://static-docs.nocobase.com/20251027223530.png)

Le composant par défaut pour un champ de relation est le sélecteur déroulant (il prend en charge le sélecteur déroulant / le sélecteur de données).
