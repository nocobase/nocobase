# Champs de formulaire

## Introduction

Les différents types de champs dans un formulaire possèdent des options de configuration distinctes. Vous pouvez étendre ces options via des `plugin`s pour répondre à des besoins plus personnalisés.

![20251025174555](https://static-docs.nocobase.com/20251025174555.png)

## Options de configuration des champs

### Valeur par défaut

Vous pouvez configurer des constantes ou des variables comme valeurs par défaut pour les champs (cette fonctionnalité n'est disponible que pour les nouveaux formulaires).

![20251025174724](https://static-docs.nocobase.com/20251025174724.png)
