# Champs de tableau

## Introduction

Les champs de tableau vous permettent non seulement d'ajuster la largeur des colonnes, le titre des champs et le tri, mais ils offrent également des configurations d'affichage plus personnalisées pour certains types de champs, comme les champs de date, les champs de relation et les champs numériques.

![20251024174558](https://static-docs.nocobase.com/20251024174558.png)

## Options de configuration des champs

### Formatage des champs de date

![20251024175303](https://static-docs.nocobase.com/20251024175303.png)

Pour en savoir plus, consultez [Formatage des dates](/interface-builder/fields/specific/date-picker)

### Formatage des champs numériques

Vous pouvez configurer la conversion d'unités simple, les séparateurs de milliers, les préfixes et suffixes, la précision et la notation scientifique.

![20251024175445](https://static-docs.nocobase.com/20251024175445.png)

Pour en savoir plus, consultez [Formatage des nombres](/interface-builder/fields/field-settings/number-format)

### Activer l'édition rapide

Lorsque l'édition rapide est activée, un bouton d'édition apparaît lorsque vous survolez la colonne avec la souris. Cliquez dessus pour modifier et enregistrer rapidement les données dans une fenêtre contextuelle.

![20251025171158](https://static-docs.nocobase.com/20251025171158.gif)

### Activer le clic pour ouvrir

En plus des champs de relation qui permettent d'ouvrir une fenêtre contextuelle en cliquant, les champs ordinaires peuvent également être configurés avec l'option "Clic pour ouvrir" pour servir de point d'entrée à l'ouverture d'une fenêtre contextuelle.

![20251025172308](https://static-docs.nocobase.com/20251025172308.gif)

### Mode d'affichage du contenu en cas de dépassement

Lorsque le contenu d'une colonne dépasse la largeur de la colonne du tableau, vous pouvez définir le mode de dépassement :

-   Afficher des points de suspension (par défaut)
-   Retour à la ligne

### Composant de champ

Certains champs prennent en charge plusieurs formats d'affichage. Vous pouvez basculer entre différents composants de champ pour obtenir des effets d'affichage variés et répondre aux besoins de différents scénarios. Par exemple, un champ de type **URL** peut être converti en composant **Preview** pour mieux afficher le contenu du lien ou les aperçus d'images.

![20251025171658](https://static-docs.nocobase.com/20251025171658.png)

### Triable

La plupart des types de champs prennent actuellement en charge le tri côté serveur. Lorsque le tri est activé, vous pouvez trier les données par ordre croissant ou décroissant en fonction du champ cible.
