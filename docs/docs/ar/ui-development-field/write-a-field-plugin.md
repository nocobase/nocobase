# البدء باستخدام المجموعات



## البدء باستخدام المجموعات

تستخدم NocoBase مفهوم **المجموعات** لتنظيم بياناتك. تشبه المجموعة الجدول في قاعدة البيانات التقليدية، ولكنها تتمتع بمرونة أكبر وميزات قوية.

### ما هي المجموعة؟

**المجموعة** هي طريقة منظمة لتخزين بياناتك وإدارتها. تتكون كل مجموعة من حقول، تحدد نوع البيانات التي يمكنها الاحتفاظ بها (مثل: النصوص، الأرقام، التواريخ، العلاقات).

على سبيل المثال، يمكنك إنشاء مجموعة `Products` بحقول مثل `name` و`price` و`description`.

### إنشاء مجموعة جديدة

لإنشاء **مجموعة** جديدة:

1.  انتقل إلى قسم `Collections` في لوحة الإدارة.
2.  انقر على زر `Add New Collection`.
3.  أدخل `Collection Name` (مثال: `Tasks`) و`Display Name` (مثال: `My Tasks`).
4.  اختر `Collection Type` (مثال: `Database Table`).
5.  انقر على `Save`.

يمكنك أيضًا تحديد علاقات بين المجموعات. على سبيل المثال، قد يكون لمجموعة `Tasks` علاقة مع مجموعة `Users` لتعيين المهام لمستخدمين محددين.

### إدارة حقول المجموعة

بمجرد إنشاء مجموعة، يمكنك إضافة حقولها أو تعديلها أو حذفها. لكل حقل نوع وخيارات تهيئة متنوعة.

```javascript
// Example of a collection field definition
{
  "name": "title",
  "type": "string",
  "interface": "input",
  "defaultValue": "New Task"
}
```

يتيح لك هذا النهج بناء نماذج بيانات مرنة دون الحاجة لكتابة استعلامات قاعدة بيانات معقدة.

للحصول على ميزات أكثر تقدمًا مثل **سير العمل** أو الاتصال بـ **مصادر البيانات** الخارجية، يرجى الرجوع إلى أقسام التوثيق الخاصة بها.
