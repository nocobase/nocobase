# أدوات المصادقة

## إدارة مصادقة المستخدمين

عند تثبيت إضافة مصادقة المستخدمين، تُهيئ طريقة مصادقة ``كلمة المرور``، والتي تعتمد على اسم المستخدم والبريد الإلكتروني الخاص به.

![](https://static-docs.nocobase.com/66eaa9d5421c9cb713b117366bd8a5d5.png)

## تفعيل أدوات المصادقة

![](https://static-docs.nocobase.com/7f1bf8f8ca5de67ffc68eff0a65848f5.png)

فقط أنواع المصادقة المفعّلة هي التي ستظهر في صفحة تسجيل الدخول.

![](https://static-docs.nocobase.com/8375a36ef98417af0f0977f1e07345dd.png)

## أنواع مصادقة المستخدمين

![](https://static-docs.nocobase.com/da4250c0cea343ebe470cbf7be4b12e4.png)

بإضافة أنواع مختلفة من أدوات المصادقة، يمكنك تمكين طرق المصادقة المناسبة في النظام.

بالإضافة إلى أنواع المصادقة التي توفرها الإضافات الحالية، يمكن للمطورين أيضًا توسيع أنواع مصادقة المستخدمين. للتفاصيل، يُرجى الرجوع إلى [دليل المطورين](./dev/).
