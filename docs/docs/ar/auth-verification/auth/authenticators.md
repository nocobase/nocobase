# أدوات المصادقة

## إدارة مصادقة المستخدم

عند تثبيت إضافة مصادقة المستخدم، سيتم تهيئة طريقة مصادقة `password` استنادًا إلى اسم المستخدم والبريد الإلكتروني.

![](https://static-docs.nocobase.com/66eaa9d5421c9cb713b117366bd8a5d5.png)

## تفعيل أدوات المصادقة

![](https://static-docs.nocobase.com/7f1fb8f8ca5de67ffc68eff0a65848f5.png)

ستظهر فقط أنواع المصادقة المفعّلة في صفحة تسجيل الدخول.

![](https://static-docs.nocobase.com/8375a36ef98417af0f0977f1e07345dd.png)

## أنواع مصادقة المستخدم

![](https://static-docs.nocobase.com/da4250c0cea343ebe470cbf7be4b12e4.png)

من خلال إضافة أنواع مختلفة من أدوات المصادقة، يمكنك تفعيل طرق المصادقة المقابلة في النظام.

بالإضافة إلى أنواع المصادقة التي توفرها الإضافات الموجودة، يمكن للمطورين أيضًا توسيع أنواع مصادقة المستخدم. للتفاصيل، راجع [دليل المطور](./dev/).
