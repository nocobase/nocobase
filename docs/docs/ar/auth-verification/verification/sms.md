# التحقق: الرسائل النصية القصيرة (SMS)

## مقدمة

رمز التحقق عبر الرسائل النصية القصيرة (SMS) هو نوع تحقق مدمج يُستخدم لإنشاء كلمة مرور ديناميكية لمرة واحدة (OTP) وإرسالها إلى المستخدم عبر الرسائل النصية القصيرة.

## إضافة أداة التحقق عبر الرسائل النصية القصيرة (SMS)

## إعدادات المسؤول

![](https://static-docs.nocobase.com/202502271727711.png)

مقدمو خدمة الرسائل النصية القصيرة (SMS) المدعومون حاليًا هم:

- <a href="https://www.aliyun.com/product/sms" target="_blank">رسائل علي بابا السحابية (Aliyun SMS)</a>
- <a href="https://cloud.tencent.com/product/sms" target="_blank">رسائل تينسنت السحابية (Tencent Cloud SMS)</a>

عند تهيئة قالب الرسائل النصية القصيرة (SMS) في لوحة تحكم المسؤول لدى مقدم الخدمة، يجب عليك حجز معلمة (parameter) لرمز التحقق الخاص بالرسائل النصية القصيرة.

- مثال تهيئة علي بابا السحابية (Aliyun): `رمز التحقق الخاص بك هو: ${code}`

- مثال تهيئة تينسنت السحابية (Tencent Cloud): `رمز التحقق الخاص بك هو: {1}`

يمكن للمطورين أيضًا توسيع الدعم ليشمل مقدمي خدمة الرسائل النصية القصيرة (SMS) الآخرين في شكل إضافات (plugins). راجع: [توسيع مقدمي خدمة الرسائل النصية القصيرة (SMS)](./dev/sms-type)

## ربط المستخدم

بعد إضافة أداة التحقق، يمكن للمستخدمين ربط رقم هاتف للتحقق في إدارة التحقق الشخصية الخاصة بهم.

![](https://static-docs.nocobase.com/202502271737016.png)

![](https://static-docs.nocobase.com/202502271737769.png)

![](https://static-docs.nocobase.com/202502271738515.png)

بعد نجاح عملية الربط، يمكن إجراء التحقق من الهوية في أي سيناريو تحقق مرتبط بأداة التحقق هذه.

![](https://static-docs.nocobase.com/202502271739607.png)

## إلغاء ربط المستخدم

يتطلب إلغاء ربط رقم الهاتف التحقق عبر طريقة التحقق المرتبطة حاليًا.
