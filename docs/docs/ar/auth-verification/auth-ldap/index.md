# المصادقة: LDAP

## مقدمة

تتبع إضافة المصادقة: LDAP معيار بروتوكول LDAP (بروتوكول الوصول الخفيف للدليل)، مما يتيح للمستخدمين تسجيل الدخول إلى NocoBase باستخدام بيانات اعتماد خادم LDAP الخاص بهم.

## الإعدادات

### الإعدادات الأساسية

<img src="https://static-docs.nocobase.com/202405101605728.png"/>

- التسجيل التلقائي عند عدم وجود المستخدم - هل يتم إنشاء مستخدم جديد تلقائيًا عندما لا يتم العثور على مستخدم مطابق موجود.
- LDAP URL - عنوان URL الخاص بخادم LDAP
- DN الربط - DN المستخدم لاختبار اتصال الخادم والبحث عن المستخدمين.
- كلمة مرور الربط - كلمة مرور DN الربط.
- اختبار الاتصال - انقر على الزر لاختبار اتصال الخادم والتحقق من صلاحية DN الربط.

### إعدادات البحث

<img src="https://static-docs.nocobase.com/202405101609984.png"/>

- DN البحث - DN المستخدم للبحث عن المستخدمين.
- عامل تصفية البحث - شرط التصفية للبحث عن المستخدمين، باستخدام `{{account}}` لتمثيل حساب المستخدم المستخدم عند تسجيل الدخول.
- النطاق - `Base`, `One level`, `Subtree`, الافتراضي `Subtree`.
- حد الحجم - حجم صفحة البحث.

### تعيين السمات

<img src="https://static-docs.nocobase.com/202405101612814.png"/>

- استخدم هذا الحقل لربط المستخدم - الحقل المستخدم للربط بالمستخدمين الموجودين. اختر 'اسم المستخدم' إذا كان حساب تسجيل الدخول هو اسم مستخدم، أو 'البريد الإلكتروني' إذا كان بريدًا إلكترونيًا. الافتراضي هو اسم المستخدم.
- خريطة السمات - تعيين سمات المستخدم لحقول جدول مستخدمي NocoBase.

## تسجيل الدخول

قم بزيارة صفحة تسجيل الدخول، وأدخل اسم مستخدم وكلمة مرور LDAP في نموذج تسجيل الدخول.
