# المصادقة: LDAP

## مقدمة

تتبع إضافة المصادقة LDAP معيار بروتوكول LDAP (بروتوكول الوصول إلى الدليل الخفيف)، مما يتيح للمستخدمين تسجيل الدخول إلى NocoBase باستخدام بيانات اعتماد خادم LDAP الخاص بهم.

## التهيئة

### التهيئة الأساسية

<img src="https://static-docs.nocobase.com/202405101605728.png"/>

- التسجيل التلقائي عند عدم وجود المستخدم - ما إذا كان سيتم إنشاء مستخدم جديد تلقائيًا عند عدم العثور على مستخدم موجود مطابق.
- رابط LDAP - رابط خادم LDAP
- Bind DN - DN المستخدم لاختبار اتصال الخادم والبحث عن المستخدمين
- كلمة مرور Bind - كلمة مرور Bind DN
- اختبار الاتصال - انقر على الزر لاختبار اتصال الخادم والتحقق من Bind DN.

### تهيئة البحث

<img src="https://static-docs.nocobase.com/202405101609984.png"/>

- Search DN - DN المستخدم للبحث عن المستخدمين
- فلتر البحث - شرط التصفية للبحث عن المستخدمين، باستخدام `{{account}}` لتمثيل حساب المستخدم المستخدم لتسجيل الدخول
- النطاق - `Base`، `مستوى واحد`، `الشجرة الفرعية`، الافتراضي `الشجرة الفرعية`
- حد الحجم - حجم صفحة البحث

### تعيين السمات

<img src="https://static-docs.nocobase.com/202405101612814.png"/>

- استخدام هذا الحقل لربط المستخدم - الحقل المستخدم للربط بالمستخدمين الموجودين. اختر 'username' إذا كان حساب تسجيل الدخول هو اسم مستخدم، أو 'email' إذا كان عنوان بريد إلكتروني. الافتراضي هو username.
- خريطة السمات - تعيين سمات المستخدم إلى الحقول في جدول مستخدمي NocoBase.
