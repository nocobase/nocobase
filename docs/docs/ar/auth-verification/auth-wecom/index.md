# المصادقة: WeCom

## مقدمة

تتيح إضافة **WeCom** للمستخدمين تسجيل الدخول إلى NocoBase باستخدام حسابات WeCom الخاصة بهم.

## تفعيل الإضافة

![](https://static-docs.nocobase.com/202406272056962.png)

## إنشاء وتهيئة تطبيق WeCom مخصص

انتقل إلى لوحة تحكم WeCom لإنشاء تطبيق مخصص.

![](https://static-docs.nocobase.com/202406272101321.png)

![](https://static-docs.nocobase.com/202406272102087.png)

انقر على التطبيق للدخول إلى صفحة تفاصيله، ومرر للأسفل، وانقر على "تسجيل الدخول المعتمد عبر WeCom".

![](https://static-docs.nocobase.com/202406272104655.png)

اضبط نطاق رد الاتصال المعتمد على نطاق تطبيق NocoBase الخاص بك.

![](https://static-docs.nocobase.com/202406272105662.png)

ارجع إلى صفحة تفاصيل التطبيق وانقر على "تفويض الويب وJS-SDK".

![](https://static-docs.nocobase.com/202406272107063.png)

اضبط وتحقق من نطاق رد الاتصال لميزة تفويض الويب OAuth2.0 للتطبيق.

![](https://static-docs.nocobase.com/202406272107899.png)

في صفحة تفاصيل التطبيق، انقر على "عنوان IP الموثوق للشركة".

![](https://static-docs.nocobase.com/202406272108834.png)

قم بتهيئة عنوان IP لتطبيق NocoBase.

![](https://static-docs.nocobase.com/202406272109805.png)

## الحصول على بيانات الاعتماد من لوحة تحكم WeCom

في لوحة تحكم WeCom، تحت "شركتي"، انسخ "معرف الشركة".

![](https://static-docs.nocobase.com/202406272111637.png)

في لوحة تحكم WeCom، تحت "إدارة التطبيقات"، انتقل إلى صفحة تفاصيل التطبيق الذي أنشأته في الخطوة السابقة وانسخ AgentId و Secret.

![](https://static-docs.nocobase.com/202406272122322.png)

انتقل إلى صفحة إدارة إضافة مصادقة المستخدم.

### التهيئة

![](https://static-docs.nocobase.com/202412041459250.png)

| الخيار | الوصف | متطلبات الإصدار |
| ------ | ----- | --------------- |
| عند عدم تطابق رقم الهاتف مع مستخدم موجود، هل يجب إنشاء مستخدم جديد تلقائيًا | عند عدم تطابق رقم الهاتف مع مستخدم موجود، ما إذا كان سيتم إنشاء مستخدم جديد تلقائيًا. | - |
| معرف الشركة | معرف الشركة، يتم الحصول عليه من لوحة تحكم WeCom. | - |
| AgentId | يتم الحصول عليه من تهيئة التطبيق المخصص في لوحة تحكم WeCom. | - |
| Secret | يتم الحصول عليه من تهيئة التطبيق المخصص في لوحة تحكم WeCom. | - |
| الأصل | نطاق التطبيق الحالي. | - |
| رابط إعادة توجيه تطبيق لوحة العمل | مسار التطبيق للانتقال إليه بعد تسجيل الدخول بنجاح. | `v1.4.0` |
| تسجيل الدخول التلقائي | تسجيل الدخول تلقائيًا عند فتح رابط التطبيق في متصفح WeCom. | `v1.4.0` |
| رابط الصفحة الرئيسية لتطبيق لوحة العمل | رابط الصفحة الرئيسية لتطبيق لوحة العمل. | - |

## تهيئة الصفحة الرئيسية لتطبيق WeCom

:::info
للإصدارات `v1.4.0` وما فوق، عند تفعيل خيار "تسجيل الدخول التلقائي"، يمكن تبسيط رابط الصفحة الرئيسية للتطبيق إلى: `https://<url>/<path>`، على سبيل المثال، `https://example.nocobase.com/admin`.
:::

انتقل إلى لوحة تحكم WeCom والصق رابط الصفحة الرئيسية لتطبيق لوحة العمل المنسوخ في حقل عنوان الصفحة الرئيسية للتطبيق المقابل.

![](https://static-docs.nocobase.com/202406272123631.png)

![](https://static-docs.nocobase.com/202406272123048.png)

## تسجيل الدخول

قم بزيارة صفحة تسجيل الدخول وانقر على الزر أسفل نموذج تسجيل الدخول لبدء تسجيل الدخول عبر طرف ثالث.

![](https://static-docs.nocobase.com/202406272124608.png)

:::warning
نظرًا لقيود أذونات WeCom على المعلومات الحساسة مثل أرقام الهواتف، لا يمكن إتمام التفويض إلا داخل عميل WeCom.
:::

## تسجيل الدخول لأول مرة

من عميل WeCom، انتقل إلى لوحة العمل، ومرر للأسفل، وانقر على التطبيق للدخول إلى الصفحة الرئيسية التي هيأتها مسبقًا.

<img src="https://static-docs.nocobase.com/202406272131113.png" width="400" />
