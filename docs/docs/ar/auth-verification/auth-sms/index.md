# المصادقة: SMS

## مقدمة

تدعم إضافة مصادقة SMS المستخدمين للتسجيل عبر SMS وتسجيل الدخول إلى NocoBase.

> يحتاج إلى استخدامه بالتزامن مع وظيفة رمز التحقق عبر SMS التي توفرها [إضافة `@nocobase/plugin-verification`](../verification/index.md)

## إضافة مصادقة SMS

أدخل صفحة إدارة إضافة مصادقة المستخدم.

![](https://static-docs.nocobase.com/202502282112517.png)

إضافة - SMS

## تهيئة الإصدار الجديد

:::info{title=ملاحظة}
تم تقديم التهيئة الجديدة في `1.6.0-alpha.30` ومن المقرر دعمها بشكل مستقر بدءًا من `1.7.0`.
:::

![](https://static-docs.nocobase.com/202502282114821.png)

**أداة التحقق:** ربط أداة تحقق SMS لإرسال رموز التحقق عبر SMS. إذا لم تكن هناك أداة تحقق متاحة، تحتاج إلى الانتقال إلى صفحة إدارة التحقق لإنشاء أداة تحقق SMS أولاً.

راجع أيضًا:
- [التحقق](../verification/index.md)
- [التحقق: SMS](../verification/sms/index.md)

**التسجيل التلقائي عند عدم وجود المستخدم:** عند تحديد هذا الخيار، إذا لم يكن رقم هاتف المستخدم موجودًا، سيتم تسجيل مستخدم جديد باستخدام رقم الهاتف كاسم مستعار.

## تهيئة الإصدار القديم

![](https://static-docs.nocobase.com/a4d35ec63ba22ae2ea9e3e8e1cbb783d.png)

ستستخدم ميزة مصادقة تسجيل الدخول عبر SMS موفر رمز التحقق SMS المهيأ والافتراضي لإرسال الرسائل النصية.

**التسجيل التلقائي عند عدم وجود المستخدم:** عند تحديد هذا الخيار، إذا لم يكن رقم هاتف المستخدم موجودًا، سيتم تسجيل مستخدم جديد باستخدام رقم الهاتف كاسم مستعار.
