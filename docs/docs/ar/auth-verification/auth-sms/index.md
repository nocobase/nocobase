# المصادقة عبر الرسائل القصيرة

## مقدمة

تتيح إضافة المصادقة عبر الرسائل القصيرة للمستخدمين التسجيل وتسجيل الدخول إلى NocoBase باستخدام الرسائل القصيرة.

> ملاحظة: يتطلب هذا الاستخدام ميزة رمز التحقق عبر الرسائل القصيرة التي توفرها [`إضافة التحقق`](/auth-verification/verification/).

## إضافة المصادقة عبر الرسائل القصيرة

انتقل إلى صفحة إدارة إضافات مصادقة المستخدم.

![](https://static-docs.nocobase.com/202502282112517.png)

إضافة - رسائل قصيرة (SMS)

![](https://static-docs.nocobase.com/202502282113553.png)

## إعدادات الإصدار الجديد

:::info{title=ملاحظة}
تم تقديم هذا التكوين الجديد بدءًا من الإصدار `1.6.0-alpha.30`، ومن المخطط أن يتم دعمه بشكل مستقر اعتبارًا من الإصدار `1.7.0`.
:::

![](https://static-docs.nocobase.com/202502282114821.png)

**المُتحقِّق (Verificator):** اربط مُتحقِّق رسائل قصيرة لإرسال رموز التحقق عبر الرسائل القصيرة. إذا لم يكن هناك مُتحقِّق متاح، يجب عليك أولاً الانتقال إلى صفحة إدارة التحقق وإنشاء مُتحقِّق رسائل قصيرة.
راجع أيضًا:

- [التحقق](../verification/index.md)
- [التحقق: رسائل قصيرة](../verification/sms/index.md)

**التسجيل التلقائي عند عدم وجود المستخدم (Sign up automatically when the user does not exist):** عند تحديد هذا الخيار، إذا لم يكن رقم هاتف المستخدم موجودًا، فسيتم تسجيل مستخدم جديد باستخدام رقم الهاتف كاسم مستعار.

## إعدادات الإصدار القديم

![](https://static-docs.nocobase.com/a4d35ec63ba22ae2ea9e3e8e1cbb783d.png)

ستستخدم ميزة مصادقة تسجيل الدخول عبر الرسائل القصيرة مزود رمز التحقق الافتراضي والمُكوَّن لإرسال الرسائل النصية.

**التسجيل التلقائي عند عدم وجود المستخدم (Sign up automatically when the user does not exist):** عند تحديد هذا الخيار، إذا لم يكن رقم هاتف المستخدم موجودًا، فسيتم تسجيل مستخدم جديد باستخدام رقم الهاتف كاسم مستعار.
