# التحقق: أداة مصادقة TOTP

## مقدمة

تتيح أداة مصادقة TOTP للمستخدمين ربط أي أداة مصادقة تتوافق مع مواصفات TOTP (كلمة المرور لمرة واحدة المستندة إلى الوقت) (<a href="https://www.rfc-editor.org/rfc/rfc6238" target="_blank">RFC-6238</a>)، وإجراء التحقق من الهوية باستخدام كلمة مرور لمرة واحدة مستندة إلى الوقت (TOTP).

## تهيئة المسؤول

انتقل إلى صفحة إدارة التحقق.

![](https://static-docs.nocobase.com/202502271726791.png)

إضافة - أداة مصادقة TOTP

![](https://static-docs.nocobase.com/202502271745028.png)

بخلاف المعرف الفريد والعنوان، لا تتطلب أداة مصادقة TOTP أي تهيئة إضافية.

![](https://static-docs.nocobase.com/202502271746034.png)

## ربط المستخدم

بعد إضافة أداة المصادقة، يمكن للمستخدمين ربط أداة مصادقة TOTP في منطقة إدارة التحقق الشخصية.

![](https://static-docs.nocobase.com/202502272252324.png)

:::warning
لا توفر الإضافة حاليًا آلية رمز الاسترداد. بمجرد ربط أداة مصادقة TOTP، يُنصح المستخدمون بالحفاظ عليها بأمان. في حالة فقدان أداة المصادقة بشكل غير مقصود، يمكنهم استخدام طريقة تحقق بديلة للتحقق من هويتهم، وإلغاء ربط أداة المصادقة، ثم إعادة ربطها.
:::

## إلغاء ربط المستخدم

يتطلب إلغاء ربط أداة المصادقة التحقق باستخدام طريقة التحقق المرتبطة بالفعل.

![](https://static-docs.nocobase.com/202502282103205.png)
