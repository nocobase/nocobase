# التحقق: مُصادِق TOTP

## مقدمة

يتيح مُصادِق TOTP للمستخدمين ربط أي مُصادِق يتوافق مع مواصفات TOTP (كلمة المرور لمرة واحدة المستندة إلى الوقت) (<a href="https://www.rfc-editor.org/rfc/rfc6238" target="_blank">RFC-6238</a>)، وإجراء التحقق من الهوية باستخدام كلمة مرور لمرة واحدة مستندة إلى الوقت (TOTP).

## إعدادات المسؤول

انتقل إلى صفحة إدارة التحقق.

![](https://static-docs.nocobase.com/202502271726791.png)

إضافة - مُصادِق TOTP

![](https://static-docs.nocobase.com/202502271745028.png)

باستثناء مُعرّف فريد وعنوان، لا يتطلب مُصادِق TOTP أي إعدادات إضافية.

![](https://static-docs.nocobase.com/202502271746034.png)

## ربط المستخدم

بعد إضافة المُصادِق، يمكن للمستخدمين ربط مُصادِق TOTP في منطقة إدارة التحقق الشخصية الخاصة بهم.

![](https://static-docs.nocobase.com/202502272252324.png)

:::warning
لا توفر الإضافة حاليًا آلية لرموز الاسترداد. بمجرد ربط مُصادِق TOTP، يُنصح المستخدمون بالحفاظ عليه آمنًا. في حال فقدان المُصادِق عن طريق الخطأ، يمكنهم استخدام طريقة تحقق بديلة للتحقق من هويتهم، ثم إلغاء ربط المُصادِق وإعادة ربطه.
:::

## إلغاء ربط المستخدم

يتطلب إلغاء ربط المُصادِق التحقق باستخدام طريقة التحقق المربوطة مسبقًا.

![](https://static-docs.nocobase.com/202502282103205.png)
