# المصادقة: دينغ توك

## مقدمة

تتيح إضافة المصادقة: دينغ توك للمستخدمين تسجيل الدخول إلى NocoBase باستخدام حساباتهم على دينغ توك.

## تفعيل الإضافة

![](https://static-docs.nocobase.com/202406120929356.png)

## طلب أذونات واجهة برمجة التطبيقات في لوحة تحكم مطوري دينغ توك

لإنشاء تطبيق، ارجع إلى <a href="https://open.dingtalk.com/document/orgapp/tutorial-obtaining-user-personal-information" target="_blank">منصة دينغ توك المفتوحة - تنفيذ تسجيل الدخول لمواقع الطرف الثالث</a>.

انتقل إلى لوحة تحكم إدارة التطبيق وقم بتمكين "معلومات رقم الهاتف الشخصي" و"إذن قراءة معلومات جهات الاتصال الشخصية".

![](https://static-docs.nocobase.com/202406120006620.png)

## الحصول على بيانات الاعتماد من لوحة تحكم مطوري دينغ توك

انسخ Client ID و Client Secret.

![](https://static-docs.nocobase.com/202406120000595.png)

## إضافة مصادقة دينغ توك في NocoBase

انتقل إلى صفحة إدارة إضافات مصادقة المستخدم.

![](https://static-docs.nocobase.com/202406112348051.png)

إضافة - دينغ توك

![](https://static-docs.nocobase.com/202406112349664.png)

### الإعدادات

![](https://static-docs.nocobase.com/202406120016896.png)

- Sign up automatically when the user does not exist - هل يتم إنشاء مستخدم جديد تلقائيًا عندما لا يتطابق رقم الهاتف مع أي مستخدم موجود.
- Client ID و Client Secret - املأ المعلومات التي نسختها في الخطوة السابقة.
- Redirect URL - عنوان URL لإعادة التوجيه (Callback URL)، انسخه وانتقل إلى الخطوة التالية.

## تهيئة عنوان URL لإعادة التوجيه في لوحة تحكم مطوري دينغ توك

الصق عنوان URL لإعادة التوجيه الذي نسخته في لوحة تحكم مطوري دينغ توك.

![](https://static-docs.nocobase.com/202406120012221.png)

## تسجيل الدخول

زر صفحة تسجيل الدخول وانقر على الزر الموجود أسفل نموذج تسجيل الدخول لبدء تسجيل الدخول عبر طرف ثالث.

![](https://static-docs.nocobase.com/202406120014539.png)
