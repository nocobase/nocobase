# المصادقة: CAS

## مقدمة

تتبع إضافة المصادقة CAS معيار بروتوكول CAS (خدمة المصادقة المركزية)، مما يتيح للمستخدمين تسجيل الدخول إلى NocoBase باستخدام حسابات مقدمة من موفري خدمات مصادقة الهوية التابعين لجهات خارجية.
