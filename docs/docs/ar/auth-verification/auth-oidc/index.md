# المصادقة: OIDC

## مقدمة

تتبع إضافة المصادقة: OIDC معيار بروتوكول OIDC (Open ConnectID)، وتستخدم تدفق رمز التفويض (Authorization Code Flow) لتمكين المستخدمين من تسجيل الدخول إلى NocoBase باستخدام حسابات مقدمي خدمة المصادقة الخارجية (IdP).
