تمت ترجمة هذا المستند بواسطة الذكاء الاصطناعي. للحصول على معلومات دقيقة، يرجى الرجوع إلى [النسخة الإنجليزية](/template-print/syntax/formatters/currency-formatting).
:::

### تنسيق العملة

#### 1. :formatC(precisionOrFormat, targetCurrency)

##### شرح الصيغة
تنسيق أرقام العملات، مع إمكانية تحديد عدد المنازل العشرية أو تنسيق إخراج محدد.  
المعلمات:
- precisionOrFormat: معلمة اختيارية، يمكن أن تكون إما رقماً (يحدد عدد المنازل العشرية) أو معرف تنسيق محدد:
  - عدد صحيح: يغير دقة المنازل العشرية الافتراضية
  - `'M'`: يخرج اسم العملة الرئيسي فقط
  - `'L'`: يخرج الرقم مع رمز العملة (افتراضي)
  - `'LL'`: يخرج الرقم مع اسم العملة الرئيسي
- targetCurrency: اختياري، رمز العملة المستهدفة (بالأحرف الكبيرة، مثل USD، EUR)، يتجاوز الإعدادات العامة

##### مثال
```
'1000.456':formatC()      // المخرج "$2,000.91"
'1000.456':formatC('M')    // المخرج "dollars"
'1':formatC('M')           // المخرج "dollar"
'1000':formatC('L')        // المخرج "$2,000.00"
'1000':formatC('LL')       // المخرج "2,000.00 dollars"
```

##### النتيجة
تعتمد مخرجات النتيجة على خيارات API وإعدادات سعر الصرف.


#### 2. :convCurr(target, source)

##### شرح الصيغة
تحويل رقم من عملة إلى أخرى. يمكن تمرير سعر الصرف عبر خيارات API أو ضبطه عالمياً.  
إذا لم يتم تحديد معلمات، فسيتم التحويل تلقائياً من `options.currencySource` إلى `options.currencyTarget`.  
المعلمات:
- target: اختياري، رمز العملة المستهدفة (الافتراضي يساوي `options.currencyTarget`)
- source: اختياري، رمز عملة المصدر (الافتراضي يساوي `options.currencySource`)

##### مثال
```
10:convCurr()              // المخرج 20
1000:convCurr()            // المخرج 2000
1000:convCurr('EUR')        // المخرج 1000
1000:convCurr('USD')        // المخرج 2000
1000:convCurr('USD', 'USD') // المخرج 1000
```

##### النتيجة
المخرج هو قيمة العملة المحولة.
