:::tip{title="إشعار الترجمة بالذكاء الاصطناعي"}
تمت ترجمة هذا المستند بواسطة الذكاء الاصطناعي. للحصول على معلومات دقيقة، يرجى الرجوع إلى [النسخة الإنجليزية](/template-print/syntax/formatters/number-formatting).
:::

### تنسيق الأرقام

#### 1. :formatN(precision)

##### شرح الصيغة
تنسيق الرقم وفقاً لإعدادات التوطين.  
المعامل:
- precision: عدد المنازل العشرية  
  بالنسبة لتنسيقات ODS/XLSX، يتم تحديد عدد المنازل العشرية المعروضة بواسطة محرر النصوص؛ أما التنسيقات الأخرى فتعتمد على هذا المعامل.

##### مثال
```
'10':formatN()         // الناتج "10.000"
'1000.456':formatN()   // الناتج "1,000.456"
```

##### النتيجة
يتم إخراج الرقم وفقاً للدقة المحددة وتنسيق التوطين.


#### 2. :round(precision)

##### شرح الصيغة
إجراء عملية تقريب للرقم، حيث يحدد المعامل عدد المنازل العشرية.

##### مثال
```
10.05123:round(2)      // الناتج 10.05
1.05:round(1)          // الناتج 1.1
```

##### النتيجة
الناتج هو القيمة بعد التقريب.


#### 3. :add(value)

##### شرح الصيغة
جمع الرقم الحالي مع القيمة المحددة.  
المعامل:
- value: القيمة المراد جمعها

##### مثال
```
1000.4:add(2)         // الناتج 1002.4
'1000.4':add('2')      // الناتج 1002.4
```

##### النتيجة
الناتج هو القيمة بعد الجمع.


#### 4. :sub(value)

##### شرح الصيغة
طرح القيمة المحددة من الرقم الحالي.  
المعامل:
- value: المطروح

##### مثال
```
1000.4:sub(2)         // الناتج 998.4
'1000.4':sub('2')      // الناتج 998.4
```

##### النتيجة
الناتج هو القيمة بعد الطرح.


#### 5. :mul(value)

##### شرح الصيغة
ضرب الرقم الحالي في القيمة المحددة.  
المعامل:
- value: المضروب فيه

##### مثال
```
1000.4:mul(2)         // الناتج 2000.8
'1000.4':mul('2')      // الناتج 2000.8
```

##### النتيجة
الناتج هو القيمة بعد الضرب.


#### 6. :div(value)

##### شرح الصيغة
قسمة الرقم الحالي على القيمة المحددة.  
المعامل:
- value: المقسوم عليه

##### مثال
```
1000.4:div(2)         // الناتج 500.2
'1000.4':div('2')      // الناتج 500.2
```

##### النتيجة
الناتج هو القيمة بعد القسمة.


#### 7. :mod(value)

##### شرح الصيغة
حساب باقي القسمة (المودولو) للرقم الحالي على القيمة المحددة.  
المعامل:
- value: القاسم

##### مثال
```
4:mod(2)              // الناتج 0
3:mod(2)              // الناتج 1
```

##### النتيجة
الناتج هو نتيجة عملية باقي القسمة.


#### 8. :abs

##### شرح الصيغة
إرجاع القيمة المطلقة للرقم.

##### مثال
```
-10:abs()             // الناتج 10
-10.54:abs()          // الناتج 10.54
10.54:abs()           // الناتج 10.54
'-200':abs()          // الناتج 200
```

##### النتيجة
الناتج هو القيمة المطلقة.


#### 9. :ceil

##### شرح الصيغة
التقريب للأعلى، أي إرجاع أصغر عدد صحيح أكبر من أو يساوي الرقم الحالي.

##### مثال
```
10.05123:ceil()       // الناتج 11
1.05:ceil()           // الناتج 2
-1.05:ceil()          // الناتج -1
```

##### النتيجة
الناتج هو العدد الصحيح بعد التقريب.


#### 10. :floor

##### شرح الصيغة
التقريب للأسفل، أي إرجاع أكبر عدد صحيح أصغر من أو يساوي الرقم الحالي.

##### مثال
```
10.05123:floor()      // الناتج 10
1.05:floor()          // الناتج 1
-1.05:floor()         // الناتج -2
```

##### النتيجة
الناتج هو العدد الصحيح بعد التقريب.


#### 11. :int

##### شرح الصيغة
تحويل الرقم إلى عدد صحيح (لا يوصى باستخدامه).

##### مثال ونتيجة
بناءً على حالة التحويل المحددة.


#### 12. :toEN

##### شرح الصيغة
تحويل الرقم إلى التنسيق الإنجليزي (الفاصلة العشرية هي '.')، لا يوصى باستخدامه.

##### مثال ونتيجة
بناءً على حالة التحويل المحددة.


#### 13. :toFixed

##### شرح الصيغة
تحويل الرقم إلى سلسلة نصية، مع الاحتفاظ بعدد محدد من المنازل العشرية فقط، لا يوصى باستخدامه.

##### مثال ونتيجة
بناءً على حالة التحويل المحددة.


#### 14. :toFR

##### شرح الصيغة
تحويل الرقم إلى التنسيق الفرنسي (الفاصلة العشرية هي ',')، لا يوصى باستخدامه.

##### مثال ونتيجة
بناءً على حالة التحويل المحددة.
