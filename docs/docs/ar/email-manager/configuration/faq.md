# الأسئلة الشائعة

## عدم استقبال الرسائل بعد تفويض حساب Microsoft

**الإجابة:** حاليًا، يدعم تسجيل الدخول عبر التفويض فقط حسابات البريد الإلكتروني الخاصة بـ Outlook وGmail، ولا يتم دعم حسابات Microsoft أو Google بشكل عام. يُرجى الرجوع إلى: [answers.microsoft.com](https://answers.microsoft.com/zh-hans/outlook_com/forum/all/%E7%8E%B0%E6%9C%89%E5%BE%AE%E8%BD%AF%E8%B4%A6/dba12dda-a7c7-4346-8263-53f4a6d9dc68)

**ملاحظة:** إذا لم تكن متأكدًا مما إذا كان لديك "بريد Outlook.com فعلي" أو "بريد Gmail"، يمكنك محاولة الدخول إلى Outlook.com أو Gmail.com عبر المتصفح للتحقق مما إذا كان بإمكانك تسجيل الدخول وإرسال الرسائل بشكل طبيعي. إذا لم تتمكن من ذلك، فقد يعني هذا أنك لم تقم بتفعيل خدمة البريد الإلكتروني الخاصة بك، وستحتاج إلى تفعيلها أو استخدام مزود بريد آخر.

## إلغاء تفويض الحساب

إذا كنت بحاجة إلى حذف أو إعادة تفويض حساب بريد إلكتروني تم ربطه مسبقًا، يمكنك اتباع الخطوات التالية:

#### **Gmail**

1. افتح الرابط التالي وقم بتسجيل الدخول:  
   https://myaccount.google.com/u/0/connections

![](https://static-docs.nocobase.com/mail-1735634131347.png)

2. انقر على التطبيق المعني، ثم اختر **إزالة**

![](https://static-docs.nocobase.com/mail-1735634131518.png)

![](https://static-docs.nocobase.com/mail-1735634131697.png)

#### **Outlook**

1. افتح الرابط التالي وقم بتسجيل الدخول:  
   https://account.microsoft.com/

2. انقر على خيار **التطبيقات والخدمات التي يمكنها الوصول إلى بياناتك**

![](https://static-docs.nocobase.com/mail-1735634131870.png)

3. انقر على **تعديل** ثم **إزالة**

![](https://static-docs.nocobase.com/mail-1735634132052.png)
