# دليل الإعداد

## نظرة عامة

بعد تفعيل إضافة البريد الإلكتروني، يجب على المسؤولين إتمام الإعدادات اللازمة أولاً قبل أن يتمكن المستخدمون من ربط حساباتهم البريدية مع NocoBase (حاليًا، يتم دعم تسجيل الدخول عبر التفويض فقط لحسابات Outlook وGmail، بينما لا يتم دعم حسابات Microsoft وGoogle للدمج المباشر).

تكمن أهمية الإعداد في إعدادات المصادقة الخاصة باستدعاء واجهات API الخاصة بمزودي خدمة البريد الإلكتروني. يحتاج المسؤول إلى تنفيذ الخطوات التالية لضمان عمل الإضافة بشكل صحيح:

1. **الحصول على معلومات المصادقة من مزود الخدمة**
   - تسجيل الدخول إلى لوحة تحكم المطور الخاصة بمزود خدمة البريد الإلكتروني (مثل Google Cloud Console أو Microsoft Azure Portal).
   - إنشاء تطبيق أو مشروع جديد وتفعيل خدمة API الخاصة بـ Gmail أو Outlook.
   - الحصول على Client ID وClient Secret.
   - إعداد عنوان إعادة التوجيه (Redirect URI) ليتطابق مع عنوان الاستدعاء الخاص بإضافة NocoBase.

2. **إعداد مزود خدمة البريد الإلكتروني**
   - الانتقال إلى صفحة إعدادات إضافة البريد الإلكتروني.
   - إدخال معلومات المصادقة المطلوبة (Client ID وClient Secret) لضمان التفويض الصحيح مع مزود الخدمة.

3. **تسجيل الدخول عبر التفويض**
   - يقوم المستخدم بتسجيل الدخول إلى حساب البريد الإلكتروني باستخدام بروتوكول OAuth.
   - تقوم الإضافة بإنشاء وتخزين رمز التفويض (Token) تلقائيًا لاستخدامه في العمليات اللاحقة.

4. **ربط حسابات البريد الإلكتروني**
   - بعد نجاح عملية التفويض، يتم ربط حساب البريد الإلكتروني بـ NocoBase.
   - تقوم الإضافة بمزامنة بيانات البريد وتوفير ميزات إدارة وإرسال واستقبال الرسائل.

5. **استخدام ميزات البريد الإلكتروني**
   - يمكن للمستخدمين عرض الرسائل وإدارتها وإرسالها مباشرة من داخل النظام.
   - تتم جميع العمليات من خلال استدعاءات API الخاصة بمزود الخدمة، مما يضمن التزامن الفوري والكفاءة في الإرسال.

من خلال هذه الخطوات، توفر إضافة البريد الإلكتروني في NocoBase تجربة إدارة بريد إلكتروني فعالة وآمنة. في حال واجهت أي مشكلة أثناء الإعداد، يُرجى الرجوع إلى الوثائق ذات الصلة أو التواصل مع فريق الدعم الفني.

## إعداد الإضافة

### تفعيل إضافة البريد الإلكتروني

1. الانتقال إلى صفحة إدارة الإضافات
2. البحث عن إضافة "Email manager" وتفعيلها

### إعداد مزود خدمة البريد الإلكتروني

بعد تفعيل الإضافة، يمكنك إعداد مزودي خدمة البريد الإلكتروني. حاليًا يتم دعم Google وMicrosoft. انقر على "Settings" → "Email Settings" في الشريط العلوي للانتقال إلى صفحة الإعدادات.

![](https://static-docs.nocobase.com/mail-1733818617187.png)

![](https://static-docs.nocobase.com/mail-1733818617514.png)

لكل مزود خدمة، يجب إدخال Client ID وClient Secret. سيتم شرح كيفية الحصول على هاتين القيمتين في الأقسام التالية.
