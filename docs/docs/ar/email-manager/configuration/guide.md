# عملية الإعداد

## نظرة عامة
بعد تفعيل إضافة البريد الإلكتروني، يجب على المسؤولين أولاً إكمال الإعدادات اللازمة قبل أن يتمكن المستخدمون العاديون من ربط حسابات بريدهم الإلكتروني بـ NocoBase. (حاليًا، لا يدعم النظام سوى تسجيل الدخول المصرح به لحسابات Outlook و Gmail، ولا يدعم الربط المباشر لحسابات Microsoft و Google).

يكمن جوهر الإعداد في إعدادات المصادقة لاستدعاءات واجهة برمجة تطبيقات (API) لمزود خدمة البريد الإلكتروني. يحتاج المسؤولون إلى إكمال الخطوات التالية لضمان عمل الإضافة بشكل صحيح:

1.  **الحصول على معلومات المصادقة من مزود الخدمة**
    -   سجّل الدخول إلى لوحة تحكم المطور الخاصة بمزود خدمة البريد الإلكتروني (مثل Google Cloud Console أو Microsoft Azure Portal).
    -   أنشئ تطبيقًا أو مشروعًا جديدًا وقم بتمكين خدمة واجهة برمجة تطبيقات (API) لبريد Gmail أو Outlook.
    -   احصل على معرف العميل (Client ID) وسر العميل (Client Secret) المقابلين.
    -   قم بتكوين عنوان URI لإعادة التوجيه (Redirect URI) وتأكد من مطابقته لعنوان رد الاتصال (callback address) الخاص بإضافة NocoBase.

2.  **إعداد مزود خدمة البريد الإلكتروني**
    -   انتقل إلى صفحة إعدادات إضافة البريد الإلكتروني.
    -   قدّم معلومات مصادقة واجهة برمجة التطبيقات (API) المطلوبة، بما في ذلك معرف العميل (Client ID) وسر العميل (Client Secret) وغيرها، لضمان الربط الصحيح للمصادقة مع مزود خدمة البريد الإلكتروني.

3.  **تسجيل الدخول المصرح به**
    -   يسجل المستخدمون الدخول إلى حسابات بريدهم الإلكتروني عبر بروتوكول OAuth.
    -   ستقوم الإضافة تلقائيًا بإنشاء وتخزين رمز المصادقة (authorization token) الخاص بالمستخدم، لاستخدامه في استدعاءات واجهة برمجة التطبيقات (API) وعمليات البريد الإلكتروني اللاحقة.

4.  **ربط حسابات البريد الإلكتروني**
    -   بعد المصادقة الناجحة للمستخدم، سيتم ربط حساب بريده الإلكتروني بـ NocoBase.
    -   ستقوم الإضافة بمزامنة بيانات بريد المستخدم وتوفير ميزات لإدارة وإرسال واستقبال رسائل البريد الإلكتروني.

5.  **استخدام ميزات البريد الإلكتروني**
    -   يمكن للمستخدمين عرض رسائل البريد الإلكتروني وإدارتها وإرسالها مباشرة داخل المنصة.
    -   تتم جميع العمليات عبر استدعاءات واجهة برمجة تطبيقات (API) لمزود خدمة البريد الإلكتروني، مما يضمن المزامنة في الوقت الفعلي والنقل الفعال.

من خلال العملية الموضحة أعلاه، توفر إضافة البريد الإلكتروني في NocoBase للمستخدمين خدمات إدارة بريد إلكتروني فعالة وآمنة. إذا واجهت أي مشاكل أثناء الإعداد، يرجى الرجوع إلى الوثائق ذات الصلة أو الاتصال بفريق الدعم الفني للحصول على المساعدة.

## إعداد الإضافة

### تفعيل إضافة البريد الإلكتروني

1.  انتقل إلى صفحة إدارة الإضافات
2.  ابحث عن إضافة "Email manager" وقم بتفعيلها.

### إعداد مزود خدمة البريد الإلكتروني

بعد تفعيل إضافة البريد الإلكتروني، يمكنك إعداد مزودي خدمة البريد الإلكتروني. حاليًا، يتم دعم خدمات البريد الإلكتروني من Google و Microsoft. انقر على "الإعدادات" -> "إعدادات البريد الإلكتروني" في الشريط العلوي للانتقال إلى صفحة الإعدادات.

![](https://static-docs.nocobase.com/mail-1733818617187.png)

![](https://static-docs.nocobase.com/mail-1733818617514.png)

لكل مزود خدمة، ستحتاج إلى إدخال معرف العميل (Client ID) وسر العميل (Client Secret). ستوضح الأقسام التالية بالتفصيل كيفية الحصول على هذين المعاملين.
