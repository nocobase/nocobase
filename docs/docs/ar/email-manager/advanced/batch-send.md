# الإرسال الجماعي

## نظرة عامة على الإرسال الجماعي

تدعم هذه الميزة إرسال رسائل بريد إلكتروني إلى عدة مستلمين بشكل جماعي، مع إرسال رسالة مستقلة لكل مستلم.

![](https://static-docs.nocobase.com/email-manager/Loading--12-31-2025_10_15_PM.png)

## الإرسال الجماعي عبر الجدول

يمكنك إعداد وظيفة التحديد الجماعي وإرسال البريد الإلكتروني داخل جدول.

1. إضافة زر إرسال بريد إلكتروني إلى الجدول  
   ![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_04_PM.png)

2. إعداد حقل المستلم (سيقوم النظام باستخراج عناوين البريد الإلكتروني من بيانات الجدول المحددة).  
   ![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_17_PM%20(1).png)

3. تحديد الصفوف التي تريد إرسال البريد إليها.  
   ![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_19_PM.png)

إذا كنت ترغب في الإرسال إلى جميع السجلات في الجدول، يمكنك تعيين نطاق المستلمين إلى **All records**.

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_18_PM%20(1).png)

بعد تعديل محتوى البريد الإلكتروني، انقر على زر **Bulk send** لإتمام عملية الإرسال.

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_19_PM.png)

## تتبع إرسال البريد الإلكتروني

بعد الإرسال، يمكنك عرض حالة الإرسال ومعلومات الرسائل من خلال مركز إدارة البريد الإلكتروني.

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_20_PM.png)
