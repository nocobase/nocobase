# الحفظ التلقائي للمسودات

ضمن قائمة زر **حفظ المسودة**، يمكنك تفعيل خيار **الحفظ التلقائي للمسودة**.

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_29_PM.png)

أثناء عملية تحرير البريد الإلكتروني، يقوم النظام بحفظ المسودة في الوقت الفعلي، مما يضمن عدم فقدان أي محتوى.

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_31_PM.png)

عند إعادة فتح محرر البريد الإلكتروني (لنفس المرسل والمستلم)، سيسألك النظام عما إذا كنت ترغب في استخدام المسودة المحفوظة.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_08_35_PM.png)
