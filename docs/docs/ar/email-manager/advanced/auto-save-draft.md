# الحفظ التلقائي كمسودة

يمكنك تفعيل ميزة **الحفظ التلقائي كمسودة** من خلال زر **حفظ المسودة**.

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_29_PM.png)

بعد تفعيل هذه الميزة، سيقوم النظام بحفظ المسودات تلقائيًا وبشكل لحظي أثناء تحرير البريد الإلكتروني، مما يضمن عدم فقدان أي محتوى.

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_31_PM.png)

عند فتح محرر البريد الإلكتروني مرة أخرى باستخدام نفس المرسل والمستلم، سيقوم النظام بعرض خيار لاستخدام المسودة المحفوظة.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_08_35_PM.png)
