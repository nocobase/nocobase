# توقيع البريد الإلكتروني

## إعداد التوقيع

انتقل إلى مركز إدارة البريد الإلكتروني ثم انقر على زر **الإعدادات**.

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_08_PM.png)

قم بالانتقال إلى لوحة **التوقيع**.

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_08_PM%20(1).png)

يمكنك هنا إدارة التواقيع. قم بتحديد الخيار الموجود بجانب التوقيع لتعيينه كتوقيع افتراضي.

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_09_PM.png)

## استخدام التوقيع

في صندوق كتابة البريد الإلكتروني، انقر على أيقونة التوقيع لاختيار التوقيع الذي تريده.

![](https://static-docs.nocobase.com/email-manager/Loading--12-31-2025_10_11_PM.png)
