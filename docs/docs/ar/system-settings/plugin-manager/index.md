# إدارة الإضافات

## مدير الإضافات

من خلال النقر على أيقونة مدير الإضافات في الزاوية العلوية اليمنى، يمكنك الوصول إلى صفحة مدير الإضافات. هنا يمكنك عرض الإضافات وتثبيتها وتفعيلها أو تعطيلها.


![2024-04-10_08-47-45](https://static-docs.nocobase.com/2024-04-10_08-47-45.png)


## مركز إعدادات الإضافات

تحتوي الإضافات عادةً على صفحة إعدادات خاصة بها. من خلال النقر على أيقونة الإعدادات في الزاوية العلوية اليمنى، يمكنك عرض قائمة بجميع صفحات إعدادات الإضافات. انقر على العنصر المناسب في القائمة لتهيئة الإضافة المطلوبة.
