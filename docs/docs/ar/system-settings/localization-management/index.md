# إدارة الترجمة

## المقدمة

تم تصميم إضافة إدارة الترجمة للمساعدة في إدارة وتنفيذ عملية الترجمة في NocoBase. تقوم بترجمة قوائم NocoBase والمجموعات والحقول وجميع الإضافات لتتوافق مع لغة وثقافة منطقة معينة.

## التثبيت

هذه إضافة مدمجة، لذلك لا يتطلب الأمر أي تثبيت إضافي.

### مزامنة عناصر الترجمة

<img src="https://static-docs.nocobase.com/202404202134850.png"/>

تشمل العناصر التي يمكن مزامنتها حاليًا:

- حزم اللغة المحلية للنظام والإضافات
- عناوين المجموعات، عناوين الحقول، وتسميات خيارات الحقول
- عناوين القوائم

بعد المزامنة، سيتم عرض جميع العناصر القابلة للترجمة للغة الحالية.

<img src="https://static-docs.nocobase.com/202404202136567.png"/>

:::info{title=ملاحظة}
قد تحتوي وحدات مختلفة على نفس النصوص الأصلية، ويجب ترجمتها بشكل منفصل.
:::
