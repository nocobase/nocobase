# نشر عنقودي

تدعم NocoBase نشر التطبيقات في وضع المجموعات لتحسين أداء التطبيق وتوافره. يتيح النشر العنقودي تشغيل نُسخ مختلفة من التطبيق على خوادم متعددة، أو تشغيل عمليات متعددة في وضع النوى المتعددة على خادم واحد.

لمزيد من التفاصيل، يُرجى الرجوع إلى وثائق [وضع المجموعات](/cluster-mode/).
