# pm2

## الأوامر الشائعة

### تشغيل التطبيق

```bash
yarn start -d
```

### إيقاف التطبيق

```bash
yarn nocobase pm2 stop
```

### إعادة تشغيل التطبيق

```bash
yarn nocobase pm2 restart
```

### عرض السجلات

```bash
yarn nocobase pm2 logs
```

### عرض حالة العمليات

```bash
yarn nocobase pm2 ls
```

> **ملاحظة:** يستخدم NocoBase PM2 لإدارة عمليات التطبيق في بيئة الإنتاج. يمكنك استخدام `yarn start` مباشرةً، وإضافة المعامل `-d` لتشغيل التطبيق في الخلفية.
