# مدير المسارات

## المقدمة

مدير المسارات هو أداة تُستخدم لإدارة مسارات الصفحة الرئيسية للنظام، ويدعم نسختي `desktop` و `mobile`. سيتم مزامنة المسارات التي يتم إنشاؤها باستخدام مدير المسارات مع القائمة (يمكنك ضبطها بحيث لا تظهر في القائمة). وبالمقابل، فإن عناصر القائمة التي تتم إضافتها من خلال قائمة الصفحة سيتم أيضًا مزامنتها مع قائمة مدير المسارات.

![20250107115449](https://static-docs.nocobase.com/20250107115449.png)

## دليل الاستخدام

### أنواع المسارات

يدعم النظام أربعة أنواع من المسارات:

- مجموعة (group): تُستخدم لتنظيم المسارات ضمن مجموعات، ويمكن أن تحتوي على مسارات فرعية.
- صفحة (page): صفحة داخلية ضمن النظام.
- تبويب (tab): يُستخدم للتبديل بين التبويبات داخل الصفحة.
- رابط (link): رابط داخلي أو خارجي يسمح بالانتقال مباشرة إلى العنوان المحدد.

### إضافة مسار

اضغط على زر "Add new" في أعلى اليمين لإنشاء مسار جديد:

1. اختر نوع المسار (Type).
2. أدخل عنوان المسار (Title).
3. اختر أيقونة المسار (Icon).
4. حدد ما إذا كان المسار يجب أن يظهر في القائمة (Show in menu).
5. حدد ما إذا كان يجب تفعيل تبويبات الصفحة (Enable page tabs).
6. بالنسبة لنوع "Page"، سيقوم النظام تلقائيًا بإنشاء مسار فريد (Path).

![20250124131803](https://static-docs.nocobase.com/20250124131803.png)

### الإجراءات على المسارات

يدعم كل مسار الإجراءات التالية:

- Add child : إضافة مسار فرعي.
- Edit : تعديل إعدادات المسار.
- View : عرض صفحة المسار.
- Delete : حذف المسار.

### الإجراءات الجماعية

يوفر شريط الأدوات العلوي الإجراءات الجماعية التالية:

- Refresh : تحديث قائمة المسارات.
- Delete : حذف المسار المحدد.
- Hide in menu : إخفاء المسار المحدد من القائمة.
- Show in menu : إظهار المسار المحدد في القائمة.

### تصفية المسارات

استخدم خاصية "Filter" الموجودة أعلى الصفحة لتصفية قائمة المسارات حسب احتياجاتك.

:::info{title=ملاحظة}
تعديل إعدادات المسارات سيؤثر مباشرة على بنية قائمة التنقل في النظام. يرجى توخي الحذر والتأكد من صحة الإعدادات قبل الحفظ.
:::
