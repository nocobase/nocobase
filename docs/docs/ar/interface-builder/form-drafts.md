# مسودات النماذج

عند تفعيل مسودات النماذج، يتم حفظ المحتوى الذي يتم إدخاله تلقائيًا بشكل محلي. حتى في حال إغلاق النموذج عن طريق الخطأ أو تحديث الصفحة، لن تُفقد البيانات التي تم ملؤها.

![20251213093717](https://static-docs.nocobase.com/20251213093717.png)

عند وجود مسودة، سيظهر إشعار في أعلى النموذج. سيتم حذف المسودة تلقائيًا بعد نجاح عملية الإرسال؛ وإذا لم تعد هناك حاجة إليها، يمكن أيضًا حذفها يدويًا في أي وقت.

![20251213092734](https://static-docs.nocobase.com/20251213092734.png)

::: tip تلميح

تدعم مسودات النماذج حاليًا التخزين المؤقت المحلي فقط. ستدعم الإصدارات المستقبلية التخزين المؤقت من جهة الخادم لتمكين المزامنة عبر أجهزة متعددة.

:::
