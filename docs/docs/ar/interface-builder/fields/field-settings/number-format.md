# تنسيق الأرقام

## مقدمة

يشير تنسيق الأرقام إلى التحسين البصري وتوحيد الحقول الرقمية (بما في ذلك حقول الصيغ) في وضع القراءة، ويشمل ذلك إعدادات مثل فواصل الآلاف، ودقة المنازل العشرية، وتحويل الوحدات، والترميز العلمي. من خلال إعداد تنسيق الأرقام، يمكن تلبية عادات التعبير الرقمي لمختلف المناطق والصناعات.

![20251028215021](https://static-docs.nocobase.com/20251028215021.png)

![20251028215059](https://static-docs.nocobase.com/20251028215059.png)

## كيفية الاستخدام

- يدعم تحويل الوحدات البسيط (الضرب/القسمة)
- فواصل الآلاف
- البادئات واللاحقات
- الدقة
- الترميز العلمي.

مثال: لتنسيق مبلغ طلب، يمكنك إضافة رمز العملة (مثل رمز اليوان الصيني ￥)، وفاصل الآلاف (مثل الفاصلة أو النقطة)، ودقة المنازل العشرية، وغيرها، لجعل المبلغ واضحًا وسهل القراءة.

- استخدام الترميز العلمي.
