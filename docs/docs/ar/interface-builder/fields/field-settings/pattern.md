# وضع العرض

## مقدمة

على عكس الكتل، تحتوي مكونات الحقول على ثلاثة أوضاع عرض (مدعومة فقط للحقول داخل النموذج). عند التبديل بين أوضاع العرض المختلفة، تتغير خيارات تهيئة الحقل المقابلة.

- قابل للتحرير؛
- للقراءة فقط (غير قابل للتحرير)؛
- للقراءة فقط (وضع المشاهدة)؛

### أوضاع العرض لحقول الربط

يحدد **وضع عرض حقل الربط** كيفية عرض الحقل في الواجهة، ويحدد أنواع مكونات الحقول المتاحة.

في **الحالة القابلة للتحرير**، تدعم حقول الربط أنواعًا متعددة من المكونات، مما يتيح للمستخدمين اختيار مكونات حقول ربط مختلفة لعرض البيانات المرتبطة أو تحديدها بناءً على متطلبات العمل.

#### مكونات حقول الربط في الحالة القابلة للتحرير

![20251028220447](https://static-docs.nocobase.com/20251028220447.png)

في هذه الحالة، يمكن للمستخدمين اختيار طريقة العرض المناسبة بمرونة لمعالجة البيانات بكفاءة أكبر.

#### مكونات حقول الربط في حالة القراءة فقط

عند التبديل إلى **حالة القراءة فقط**، يستخدم النظام تلقائيًا **مكون حقل العنوان** لعرض البيانات المرتبطة. هذا مناسب للسيناريوهات التي تحتاج فيها فقط إلى عرض البيانات دون إجراء تعديلات.
