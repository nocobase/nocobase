# مكونات الحقول

## مقدمة

تُستخدم مكونات الحقول لعرض محتوى الحقل.
تحتوي بعض الحقول على مكونات عرض متعددة مدمجة. يمكن للمستخدمين اختيار المكون المناسب وفقًا لاحتياجاتهم الفعلية لتقديم قيمة الحقل بطرق مختلفة.

مثال: في وضع العرض، يمكن تبديل حقل URL إلى مكون معاينة أو مكون URL (افتراضي).

![20251028214033](https://static-docs.nocobase.com/20251028214033.png)

![20251028214628](https://static-docs.nocobase.com/20251028214628.png)
