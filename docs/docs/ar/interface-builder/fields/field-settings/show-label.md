# إظهار التسمية

## مقدمة

تُظهر الحقول في الكتلة تسميات الحقول بشكل افتراضي لتحديد معنى كل حقل، مما يسهل على المستخدمين قراءة محتوى البيانات وفهمه.

في بعض السيناريوهات (مثل صفحات العرض فقط أو عندما يكون معنى الحقول واضحًا جدًا بالفعل)، يمكنك اختيار إخفاء التسميات للحصول على واجهة أكثر بساطة. بعد إخفاء التسميات، يتم عرض قيم الحقول فقط، ولم تعد أسماء الحقول تظهر.
