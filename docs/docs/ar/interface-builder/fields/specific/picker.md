# محدد البيانات

## مقدمة

محدد البيانات، على شكل محدد جدول منبثق، يدعم اختيار البيانات الموجودة مسبقًا من المجموعة المستهدفة لربطها، أو إضافة بيانات إلى المجموعة المستهدفة ثم ربطها.

![20251027203343](https://static-docs.nocobase.com/20251027203343.png)

لن تظهر البيانات المحددة في محدد الجدول.

![20251027203717](https://static-docs.nocobase.com/20251027203717.gif)

في محدد الجدول، يمكنك أيضًا إدارة المجموعة المستهدفة للربط بشكل أكبر (إضافة، حذف، استيراد/تصدير، وما إلى ذلك).

![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
