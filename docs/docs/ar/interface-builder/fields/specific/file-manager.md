# مدير الملفات

## مقدمة

مدير الملفات هو مكون حقل ربط يُستخدم للتعامل مع العلاقات التي تكون فيها المجموعة المستهدفة هي مجموعة الملفات.

![20251027194124](https://static-docs.nocobase.com/20251027194124.png)

## إعدادات الحقل

- [مكون الحقل](/interface-builder/fields/association-field): التبديل إلى مكونات حقل ربط أخرى، مثل القائمة المنسدلة، ومحدد البيانات، وما إلى ذلك.
- السماح باختيار الملفات الموجودة: ممكّن افتراضيًا، يتيح الاختيار والربط من الملفات المرفوعة.
