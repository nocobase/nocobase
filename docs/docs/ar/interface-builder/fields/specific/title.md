# العنوان

## مقدمة

في وضع القراءة فقط، تُعرض حقول الارتباط افتراضيًا باستخدام **مكون حقل العنوان**، والذي يحدد السجل المرتبط حاليًا عن طريق عرض قيمة حقل العنوان. يقوم النظام تلقائيًا بمطابقة وعرض مكون حقل العنوان المناسب بناءً على نوع حقل العنوان.

## خيارات إعدادات الحقل

### حقل العنوان

عندما يكون نوع حقل العنوان "رقم":

يكون مكون الحقل هو مكون رقمي.

![20251028155837](https://static-docs.nocobase.com/20251028155837.png)

عندما يكون نوع حقل العنوان "تاريخ ووقت":

![20251029213244](https://static-docs.nocobase.com/20251029213244.png)

يتحول مكون الحقل إلى مكون تاريخ.

### تفعيل النقر للفتح (مفعل افتراضيًا)

بالنقر، ستُفتح نافذة منبثقة حيث يمكنك تخصيص الكتل لإدارة البيانات.

![20251028160742](https://static-docs.nocobase.com/20251028160742.png)

لمزيد من التفاصيل، راجع [مكون الحقل](/interface-builder/fields/association-field).
