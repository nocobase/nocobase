# التحديد المتسلسل

## مقدمة

أداة التحديد المتسلسل مصممة لحقول الربط التي تكون مجموعتها المستهدفة جدولاً شجريًا. تتيح للمستخدمين اختيار البيانات وفقًا للهيكل الهرمي للمجموعة الشجرية، وتدعم البحث التقريبي للتصفية السريعة.

## تعليمات الاستخدام

- عندما يكون الربط **واحدًا لواحد**، يكون التحديد المتسلسل **اختيارًا فرديًا**.

![20251125214656](https://static-docs.nocobase.com/20251125214656.png)

- عندما يكون الربط **واحدًا لمتعدد**، يكون التحديد المتسلسل **اختيارًا متعددًا**.

![20251125215318](https://static-docs.nocobase.com/20251125215318.png)

## خيارات تهيئة الحقل

### حقل العنوان

حقل العنوان هو الحقل الذي يحدد التسمية المعروضة لكل خيار.

![20251125214923](https://static-docs.nocobase.com/20251125214923.gif)

> يدعم البحث السريع بناءً على حقل العنوان.

![20251125215026](https://static-docs.nocobase.com/20251125215026.gif)

لمزيد من التفاصيل، راجع:
[حقل العنوان](/interface-builder/fields/field-settings/title-field)

يتحكم في نطاق بيانات القائمة الشجرية (إذا تطابق سجل فرعي مع الشروط، فسيتم تضمين سجله الأب أيضًا).

![20251125215111](https://static-docs.nocobase.com/20251125215111.png)

لمزيد من التفاصيل، راجع:
[نطاق البيانات](/interface-builder/fields/field-settings/data-scope)
