# التفاصيل الفرعية

## مقدمة

التفاصيل الفرعية هي المكون المقابل للنموذج الفرعي في وضع القراءة. مقارنةً بمكونات التسمية والعنوان، لا تعرض التفاصيل الفرعية المزيد من البيانات من المجموعة الحالية فحسب، بل يمكن تهيئتها أيضًا لعرض معلومات البيانات من المجموعات المرتبطة. يتم عرض بيانات العلاقات متعددة المستويات بوضوح على شكل تفاصيل متداخلة.
