# نموذج فرعي

## مقدمة

النماذج الفرعية مناسبة للسيناريوهات التي يتم فيها إنشاء بيانات علاقة جديدة قبل ربطها. يتم عرض بيانات العلاقة متعددة المستويات بوضوح في شكل نموذج متداخل. مقارنةً بمحدد البيانات ومكونات القائمة المنسدلة، يتيح لك النموذج الفرعي إدارة حقول المجموعة المرتبطة مباشرةً ضمن كتلة الصفحة الحالية، ويتم إرسال بيانات العلاقة مع النموذج الرئيسي.
