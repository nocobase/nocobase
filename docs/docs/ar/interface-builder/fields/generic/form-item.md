# حقول النموذج

## مقدمة

تتميز أنواع الحقول المختلفة في النموذج بخيارات إعدادات متنوعة، مما يتيح للمستخدمين توسيع هذه الخيارات عبر الإضافات (plugins) لتلبية احتياجاتهم المخصصة بشكل أكبر.

![20251025174555](https://static-docs.nocobase.com/20251025174555.png)

## خيارات إعدادات الحقول

### القيمة الافتراضية

يدعم إعداد الثوابت/المتغيرات كقيم افتراضية للحقول (مدعوم فقط للنماذج الجديدة).

![20251025174724](https://static-docs.nocobase.com/20251025174724.png)

لمزيد من التفاصيل، راجع [إعداد القيمة الافتراضية](/interface-builder/fields/field-settings/default-value)
