# الحقل

في واجهة المستخدم، الحقل هو مكون حقل، ويعمل كحامل للبيانات الفردية. تُعرض أنواع البيانات المختلفة باستخدام مكونات حقول مختلفة. لا يمكن ربط الحقول إلا بالكتل، ولا يمكن استخدامها بشكل مستقل.

## الحقول في الكتل

لا تُستخدم الحقول عادةً بمفردها، بل توجد كعناصر فرعية للكتل من نوع البيانات. تحتوي الكتل من نوع البيانات عادةً على خيار "تكوين الحقول"، وتُقدم قائمة الحقول من قِبل المجموعة الحالية.

![20251029204240](https://static-docs.nocobase.com/20251029204240.png)

## مصمم الحقول (شريط الأدوات)

مثل الكتل، يحتوي الركن العلوي الأيمن لمكون الحقل أيضًا على ثلاثة أيقونات، وهي:

- سحب وإفلات للتخطيط
- [تدفق الأحداث](/interface-builder/event-flow)
- التكوين

![20251025165432](https://static-docs.nocobase.com/20251025165432.png)

## تخطيط الحقول

يمكنك تخصيص تخطيط الحقول داخل الكتلة عن طريق السحب والإفلات.

![20251025165605](https://static-docs.nocobase.com/20251025165605.gif)

## خيارات تكوين الحقول

بالإضافة إلى خيارات تكوين الحقول العامة، تدعم أنواع الحقول المختلفة خيارات تكوين محددة.
