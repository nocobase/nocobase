# التخطيط

## مقدمة

تدعم الكتل الشبيهة بالنماذج، مثل كتل النماذج وكتل التفاصيل وكتل القوائم، إعدادات التخطيط. تنقسم خيارات التخطيط إلى تخطيط عمودي (حيث يتم ترتيب التسميات وحقول الإدخال سطرًا بسطر) وتخطيط أفقي (حيث يتم وضع التسميات وحقول الإدخال جنبًا إلى جنب). يمكن للمستخدمين تعديل تخطيط حقول الكتلة وفقًا لاحتياجاتهم الخاصة لضمان أن إدخال المعلومات أو عرضها واضح وفعال.

### عمودي (افتراضي)

التخطيط الافتراضي هو عمودي.

![20251024120143](https://static-docs.nocobase.com/20251024120143.png)

### أفقي

يعرض التخطيط الأفقي التسميات وحقول الإدخال جنبًا إلى جنب، مما يوفر المساحة بفعالية. بالنسبة للحقول ذات العناوين الطويلة، يمكنك اختيار تعيينها للالتفاف أو إظهار علامة حذف لضمان واجهة نظيفة وجمالية.

![20251024120237](https://static-docs.nocobase.com/20251024120237.png)

#### محاذاة عنوان الحقل

- **محاذاة لليسار** (افتراضي)
- **محاذاة لليمين**

#### عرض عنوان الحقل

- العرض الافتراضي هو **120px**، والحد الأدنى للعرض هو **50px**.

#### التعامل مع عناوين الحقول التي تتجاوز العرض

- **التفاف النص**
- **علامة الحذف**

#### إظهار النقطتين الرأسيتين

يمكنك اختيار ما إذا كنت تريد عرض نقطتين رأسيتين بعد عنوان الحقل.
