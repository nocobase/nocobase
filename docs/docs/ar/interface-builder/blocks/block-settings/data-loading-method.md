# وضع تحميل البيانات

## مقدمة

وضع تحميل البيانات الافتراضي هو "تحميل كافة البيانات عندما تكون شروط التصفية فارغة". تشير "شروط التصفية" هنا إلى القيم الموجودة في "كتلة التصفية". الخيار الآخر المتاح هو "عدم تحميل البيانات عندما تكون شروط التصفية فارغة".
