# تعيين نطاق البيانات

## مقدمة

يشير تعيين نطاق البيانات إلى تحديد شروط تصفية افتراضية لكتلة البيانات. يمكن للمستخدمين تعديل نطاق البيانات بمرونة حسب احتياجات العمل، ولكن بغض النظر عن أي عمليات تصفية يتم إجراؤها، سيطبق النظام تلقائيًا شرط التصفية الافتراضي هذا، مما يضمن بقاء البيانات دائمًا ضمن حدود النطاق المحدد.

## دليل المستخدم

![20251027110053](https://static-docs.nocobase.com/20251027110053.png)

يدعم حقل التصفية اختيار الحقول من المجموعة الحالية والمجموعات المرتبطة.

![20251027110140](https://static-docs.nocobase.com/20251027110140.png)

### عوامل التشغيل

تدعم أنواع الحقول المختلفة عوامل تشغيل متنوعة. على سبيل المثال، تدعم حقول النص عوامل تشغيل مثل يساوي، لا يساوي، ويحتوي؛ بينما تدعم حقول الأرقام عوامل تشغيل مثل أكبر من وأقل من؛ وتدعم حقول التاريخ عوامل تشغيل مثل ضمن نطاق وقبل تاريخ محدد.

![20251027111124](https://static-docs.nocobase.com/20251027111124.png)

### قيمة ثابتة

مثال: تصفية البيانات حسب "حالة" الطلب.

![20251027111229](https://static-docs.nocobase.com/20251027111229.png)

### قيمة متغيرة

مثال: تصفية بيانات الطلبات للمستخدم الحالي.

![20251027113349](https://static-docs.nocobase.com/20251027113349.png)
