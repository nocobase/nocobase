# بطاقة الشبكة

## مقدمة

تُعرض كتلة بطاقة الشبكة معلومات موجزة عن سجلات البيانات بتنسيق بطاقة. تدعم هذه الكتلة إمكانية تهيئة عدد الأعمدة لتناسب أحجام الشاشات المختلفة، مما يضمن عرضًا سهل الاستخدام على مجموعة متنوعة من الأجهزة.
