# مخطط بياني

## مقدمة

توفر إضافة NocoBase لتصور البيانات استعلامات بيانات مرئية ومجموعة غنية من مكونات المخططات البيانية. يمكن للمستخدمين، من خلال إعدادات بسيطة، إنشاء لوحات تحكم مرئية بسرعة لعرض رؤى البيانات، ودعم تحليل وعرض البيانات متعددة الأبعاد.
