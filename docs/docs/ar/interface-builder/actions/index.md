# الإجراء

في واجهة المستخدم (UI)، الإجراء هو زر يُستخدم لتشغيل مجموعة من التعليمات. يمكن وضعه مباشرة على صفحة، أو في مربع حوار، أو في درج، ويمكن استخدامه أيضًا بالاشتراك مع الكتل. حاليًا، تُستخدم الإجراءات القابلة للتكوين ديناميكيًا بشكل أساسي بالاشتراك مع الكتل.
