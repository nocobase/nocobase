# تأكيد الإجراءات

## مقدمة

تُستخدم إعدادات التأكيد للتأكد من تنفيذ الإجراء المطلوب. عند النقر على أي إجراء، يظهر مربع حوار تأكيد. يتم تفعيل هذه الميزة افتراضيًا لزر الحذف لمنع فقدان البيانات أو حدوث نتائج غير قابلة للإلغاء بسبب الأخطاء غير المقصودة.
