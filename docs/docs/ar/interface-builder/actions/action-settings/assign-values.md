# تعيين الحقول

## مقدمة

يشير تعيين الحقول إلى تحديد قيم الحقول مسبقًا قبل تنفيذ أي عملية. عند تحديث سجل أو إنشائه، يتم ملء هذه البيانات تلقائيًا دون الحاجة إلى إدخال يدوي. على سبيل المثال، في سيناريوهات مثل التحديث المجمع لحالات الطلبات، أو تعيين وقت التسليم إلى الوقت الحالي عند تسليم الطلب.
