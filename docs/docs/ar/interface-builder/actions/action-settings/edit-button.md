# تعديل الزر

## مقدمة

يدعم النظام تخصيص اسم الزر، أيقونته، لون خلفيته، ونوع الزر، مما يوفر خيارات مرنة لتخصيص واجهة المستخدم لمساعدة المستخدمين على تعديل نمط عرض الزر ووظيفته وفقًا لاحتياجاتهم الخاصة.

- **اسم الزر**: يمكن للمستخدمين تخصيص نص الزر وفقًا لاحتياجات العمل لضمان أن يكون وصف الزر واضحًا وبديهيًا.
- **أيقونة الزر**: يدعم تعيين أيقونة للزر لتعزيز التمييز البصري، مما يسمح للمستخدمين بتحديد وظيفة الزر بسرعة.
- **إجراء خطير (Danger Action)**: عندما يمثل الزر **إجراءً خطيرًا** (مثل الحذف، إعادة التعيين، إلخ)، يمكن تعيينه كنوع **إجراء خطير (Danger Action)**.
- **نوع الزر (Button Type)**: تدعم الأزرار إعدادات أنواع مختلفة. تشمل أنواع الأزرار الشائعة ما يلي:
  - **افتراضي (Default)**: زر قياسي، مناسب للعمليات العادية.
  - **أساسي (Primary)**: يُستخدم للعمليات الهامة أو الرئيسية، وعادةً ما يكون له تأثير بصري بارز.
  - **متقطع (Dashed)**: زر ذو حدود متقطعة، مناسب للعمليات الثانوية أو الأزرار التي تشبه التلميحات.
  - **نصي (Text)**: زر يحتوي على نص فقط وبدون لون خلفية، ويُستخدم عادةً للعمليات الخفيفة أو الروابط.
