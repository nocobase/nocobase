# تحرير

## مقدمة

إجراء التحرير هو زر صف يستخدم السجل الحالي كسياق له، ويظهر كنافذة منبثقة حيث يمكن للمستخدمين تخصيص الكتل وتكوينها.

![20251029165442](https://static-docs.nocobase.com/20251029165442.png)

تكوين الكتل حسب الحاجة.

![20251029165514](https://static-docs.nocobase.com/20251029165514.png)

## إعدادات الإجراء

![20251029165620](https://static-docs.nocobase.com/20251029165620.png)

- [قواعد الربط](/interface-builder/actions/action-settings/linkage-rule): لإظهار/إخفاء الزر ديناميكيًا؛
- [زر التحرير](/interface-builder/actions/action-settings/edit-button): لتحرير عنوان الزر ونوعه وأيقونته؛
- [النافذة المنبثقة للتحرير](/interface-builder/actions/action-settings/edit-popup): لتحديد طريقة الفتح والحجم.
