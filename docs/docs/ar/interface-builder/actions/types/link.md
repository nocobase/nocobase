# الرابط

## مقدمة

تستخدم عملية الرابط التنقل عبر المسار، وتدعم تمرير متغيرات URL، مما يسمح للصفحة المستهدفة بتعديل محتواها ديناميكيًا بناءً على البيانات المُمررة. يمكن تهيئتها واستخدامها في كتل البيانات.
