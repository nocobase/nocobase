# إضافة سجل فرعي

## مقدمة

صُممت هذه العملية خصيصًا لـ **مكونات الجدول الشجري** لإضافة السجلات الفرعية بسرعة. عند إضافة سجل فرعي، تُعامَل الصفوف المحددة كعقدة أب، مما يتيح للمستخدمين إنشاء سجلات فرعية بسهولة تحت أي عقدة أب. تُبسّط هذه الميزة إدارة البيانات الهرمية بشكل كبير، مما يجعل توسيع هياكل البيانات وتنظيمها أكثر كفاءة وبديهية.
