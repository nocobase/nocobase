# التعديل الجماعي

## مقدمة

يُعد التعديل الجماعي مناسباً للسيناريوهات التي تتطلب تحديثات مرنة للبيانات بكميات كبيرة. بعد النقر فوق زر التعديل الجماعي، يمكنك تهيئة نموذج التعديل الجماعي في نافذة منبثقة وتعيين استراتيجيات تحديث مختلفة لكل حقل.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM.png)


## تهيئة الإجراء

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM%20(1).png)


## دليل الاستخدام

### تهيئة نموذج التعديل الجماعي

1. أضف زر التعديل الجماعي.

2. حدد نطاق التعديل الجماعي: المحدد / الكل، الافتراضي هو "المحدد".

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_14_AM.png)

3. أضف نموذج التعديل الجماعي.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_14_AM.png)

4. قم بتهيئة الحقول المراد تعديلها وأضف زر الإرسال.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM%20(1).png)

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM.png)

### إرسال النموذج

1. حدد صفوف البيانات المراد تعديلها.

2. اختر وضع التعديل للحقول وقم بتعبئة القيم المراد إرسالها.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_10_33_AM.png)

:::info{title=أنماط التعديل المتاحة}
* **عدم التحديث**: يبقى الحقل دون تغيير.
* **تغيير إلى**: تحديث الحقل إلى القيمة المرسلة.
* **مسح**: مسح البيانات الموجودة في الحقل.

:::

3. أرسل النموذج.
