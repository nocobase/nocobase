# تصفية

## مقدمة

يظهر إجراء التصفية عادةً في كتل البيانات لتصفية البيانات عن طريق تكوين شروط متنوعة. تجدر الإشارة إلى أنه إذا كانت الكتلة قد تم تكوين نطاق بيانات لها بالفعل، فسيتم دمج إجراء التصفية مع شروط نطاق البيانات لتشكيل شرط التصفية النهائي.

### الحقول القابلة للتصفية

افتراضيًا، تكون جميع حقول هذه المجموعة ممكنة. فقط الحقول الممكنة هي التي يمكن تحديدها في قائمة حقول الشرط.

يمكن أيضًا استخدام حقول من المجموعات المرتبطة كحقول تصفية.
