# فتح نافذة منبثقة

## مقدمة

إجراء **فتح نافذة منبثقة** مشابه لإجراءي العرض والتحرير. يمكن للمستخدمين تهيئة زر **فتح نافذة منبثقة** في صفوف الجدول أو في رأسه. عند النقر على الزر، تظهر نافذة منبثقة مطابقة، وتعرض الكتل الموجودة داخل هذه النافذة البيانات بناءً على سجل السياق الحالي.

![20251029182351](https://static-docs.nocobase.com/20251029182351.png)

قم بتهيئة الكتل في النافذة المنبثقة حسب الحاجة.

![20251029190208](https://static-docs.nocobase.com/20251029190208.png)

## إعدادات الإجراء

![20251029190352](https://static-docs.nocobase.com/20251029190352.png)

- [تحرير الزر](/interface-builder/actions/action-settings/edit-button): قم بتحرير عنوان الزر ونوعه وأيقونته.
- [قاعدة الربط](/interface-builder/linkage-rule): التحكم ديناميكيًا في حالة الزر (مرئي/معطل).
- [تحرير النافذة المنبثقة](/interface-builder/actions/action-settings/edit-popup): تحديد وضع الفتح (نافذة منبثقة أو درج جانبي) وحجمها.
