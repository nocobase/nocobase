# عرض

## مقدمة

إجراء `العرض` هو زر يظهر على مستوى الصف، ويستخدم السجل الحالي كسياق له. يُعرض هذا الإجراء كنافذة منبثقة، حيث يمكن للمستخدمين تخصيص الكتل وتكوينها بداخلها.
