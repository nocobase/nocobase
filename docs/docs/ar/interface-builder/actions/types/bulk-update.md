# تحديث مجمع

## مقدمة

تُستخدم عملية التحديث المجمع عندما تحتاج إلى تطبيق نفس التحديث على مجموعة من السجلات. قبل تنفيذ التحديث المجمع، يحتاج المستخدم إلى تحديد منطق تعيين الحقول للتحديث مسبقًا. سيتم تطبيق هذا المنطق على جميع السجلات المحددة عند نقر المستخدم على زر التحديث.

![20251029195320](https://static-docs.nocobase.com/20251029195320.png)

## إعدادات الإجراء

![20251029195729](https://static-docs.nocobase.com/20251029195729.png)

### البيانات المراد تحديثها

محددة / الكل، الافتراضي هو محددة.

![20251029200034](https://static-docs.nocobase.com/20251029200034.png)

### تعيين الحقول

قم بتعيين الحقول للتحديث المجمع. سيتم تحديث الحقول التي تم تعيينها فقط.

كما هو موضح في الشكل، قم بتكوين إجراء التحديث المجمع في جدول الطلبات لتحديث البيانات المحددة بشكل مجمع إلى "قيد الانتظار للموافقة".

![20251029200109](https://static-docs.nocobase.com/20251029200109.png)

- [تعديل الزر](/interface-builder/actions/action-settings/edit-button): تعديل عنوان الزر ونوعه وأيقونته؛
- [قاعدة الربط](/interface-builder/actions/action-settings/linkage-rule): إظهار/إخفاء الزر ديناميكيًا؛
