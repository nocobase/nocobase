# تحديث سجل

## مقدمة

تتيح لك هذه الميزة إجراء تحديثات على البيانات من خلال إجراءات الصفوف المخصصة. يمكنك تحديد منطق تحديث الحقول عبر إعدادات تعيين قيم الحقول.

![20251029194309](https://static-docs.nocobase.com/20251029194309.png)

## إعدادات الإجراء

![20251029194330](https://static-docs.nocobase.com/20251029194330.png)

### قواعد الربط

مثال: إخفاء الزر عندما تكون حالة المستند "ملغاة".

![20251029194524](https://static-docs.nocobase.com/20251029194524.png)

لمزيد من التفاصيل، راجع [قواعد ربط الإجراءات](/interface-builder/linkage-rule).

### تعيين القيم

مثال: تحديث المسؤول في السجل الحالي ليكون المستخدم الحالي.

![20251029194652](https://static-docs.nocobase.com/20251029194652.png)

لمزيد من التفاصيل، راجع [تعيين القيم](/interface-builder/actions/action-settings/assign-values).
