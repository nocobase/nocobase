# تصدير

## مقدمة

تتيح ميزة التصدير تصدير السجلات المفلترة بتنسيق **Excel**، وتدعم تهيئة الحقول المراد تصديرها. يمكن للمستخدمين اختيار الحقول التي يحتاجون إلى تصديرها لتحليل البيانات أو معالجتها أو أرشفتها لاحقًا. تعزز هذه الميزة مرونة عمليات البيانات، خاصة في السيناريوهات التي تتطلب نقل البيانات إلى منصات أخرى أو معالجتها بشكل إضافي.

### أبرز الميزات:
- **اختيار الحقول**: يمكن للمستخدمين تهيئة واختيار الحقول المراد تصديرها، مما يضمن دقة البيانات المصدرة وإيجازها.
- **دعم تنسيق Excel**: سيتم حفظ البيانات المصدرة كملف Excel قياسي، مما يسهل دمجها وتحليلها مع البيانات الأخرى.

باستخدام هذه الميزة، يمكن للمستخدمين بسهولة تصدير البيانات الرئيسية من عملهم للاستخدام الخارجي، مما يحسن كفاءة العمل.

### الحقول القابلة للتصدير

- المستوى الأول: جميع حقول المجموعة الحالية؛
- المستوى الثاني: إذا كان حقل علاقة، يجب اختيار حقول من المجموعة المرتبطة؛
- المستوى الثالث: يتم معالجة ثلاثة مستويات فقط؛ لا يتم عرض حقول العلاقة في المستوى الأخير؛

![20251029171557](https://static-docs.nocobase.com/20251029171557.png)
