# تصدير المرفقات

## مقدمة

يدعم تصدير المرفقات تصدير الحقول المتعلقة بالمرفقات بصيغة حزمة مضغوطة.

#### إعدادات تصدير المرفقات

![20251029173251](https://static-docs.nocobase.com/20251029173251.png)

![20251029173425](https://static-docs.nocobase.com/20251029173425.png)

![20251029173345](https://static-docs.nocobase.com/20251029173345.png)

- قم بتكوين حقول المرفقات المراد تصديرها؛ يدعم الاختيار المتعدد.
- يمكنك اختيار ما إذا كنت تريد إنشاء مجلد لكل سجل.

قواعد تسمية الملفات:

- إذا اخترت إنشاء مجلد لكل سجل، فإن قاعدة تسمية الملفات تكون: `{قيمة حقل عنوان السجل}/{اسم حقل المرفق}[-{رقم تسلسل الملف}].{امتداد الملف}`.
- إذا اخترت عدم إنشاء مجلد، فإن قاعدة تسمية الملفات تكون: `{قيمة حقل عنوان السجل}-{اسم حقل المرفق}[-{رقم تسلسل الملف}].{امتداد الملف}`.

يتم إنشاء رقم تسلسل الملف تلقائيًا عندما يحتوي حقل المرفق على عدة مرفقات.

- [قاعدة الربط](/interface-builder/actions/action-settings/linkage-rule): إظهار/إخفاء الزر ديناميكيًا؛
- [تحرير الزر](/interface-builder/actions/action-settings/edit-button): تحرير عنوان الزر ونوعه وأيقونته؛
