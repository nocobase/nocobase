# بناء الواجهة

## ما تراه هو ما تحصل عليه (WYSIWYG)

تعتمد NocoBase نمط بناء الواجهة "ما تراه هو ما تحصل عليه" (WYSIWYG). يمكنك التبديل بين وضع التكوين ووضع الاستخدام بالنقر على زر بناء الواجهة.

## قالب التخطيط

يأتي NocoBase مع قالب تخطيط افتراضي، حيث توجد منطقة التنقل في الأعلى واليسار، ومنطقة المحتوى في اليمين.

![未命名.002](https://static-docs.nocobase.com/未命名.002.jpeg)

## خيارات التكوين
