# مجموعة الملفات

## المقدمة

مجموعة الملفات هي مجموعة مخصّصة لتخزين بيانات تعريف الملفات (metadata)، وتحتوي على حقول مدمجة مثل اسم الملف، الحجم، النوع، ومسار التخزين.
