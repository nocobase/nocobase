# محرك التخزين: Amazon S3

محرك تخزين يعتمد على Amazon S3. يجب تجهيز الحساب والصلاحيات اللازمة قبل الاستخدام.

## معلمات الإعداد

![مثال على إعداد محرك تخزين Amazon S3](https://static-docs.nocobase.com/20251031092524.png)

:::info{title=ملاحظة}
يوضح هذا القسم المعلمات الخاصة بمحرك تخزين Amazon S3 فقط. أما المعلمات العامة، فيُرجى الرجوع إلى [معلمات المحرك العامة](./index#引擎通用参数).
:::

### المنطقة (Region)

أدخل منطقة تخزين S3، على سبيل المثال: `us-west-1`.

:::info{title=ملاحظة}
يمكنك عرض معلومات المنطقة الخاصة بالحاوية (Bucket) من خلال [لوحة تحكم Amazon S3](https://console.aws.amazon.com/s3/)، ويكفي استخدام بادئة المنطقة فقط (وليس اسم النطاق الكامل).
:::

### AccessKey ID

أدخل معرّف AccessKey الخاص بـ Amazon S3.

### AccessKey Secret

أدخل المفتاح السري (AccessKey Secret) الخاص بـ Amazon S3.

### الحاوية (Bucket)

أدخل اسم حاوية S3.
