# محرك التخزين: Amazon S3

محرك تخزين يعتمد على Amazon S3. قبل الاستخدام، يجب إعداد الحسابات والأذونات اللازمة.

## معلمات التكوين

![مثال على تكوين محرك التخزين Amazon S3](https://static-docs.nocobase.com/20251031092524.png)

:::info{title=ملاحظة}
يقدم هذا القسم المعلمات الخاصة بمحرك التخزين Amazon S3 فقط. للاطلاع على المعلمات العامة، يرجى الرجوع إلى [معلمات المحرك العامة](./index#引擎通用参数).
:::

### المنطقة

أدخل منطقة تخزين S3، على سبيل المثال: `us-west-1`.

:::info{title=ملاحظة}
يمكنك عرض معلومات المنطقة الخاصة بمساحة التخزين (Bucket) في [وحدة تحكم Amazon S3](https://console.aws.amazon.com/s3/)، وتحتاج فقط إلى استخدام بادئة المنطقة (وليس اسم النطاق الكامل).
:::

### معرف مفتاح الوصول (AccessKey ID)

أدخل معرف مفتاح الوصول (AccessKey ID) الخاص بـ Amazon S3.

### سر مفتاح الوصول (AccessKey Secret)

أدخل سر مفتاح الوصول (AccessKey Secret) الخاص بـ Amazon S3.

### الحاوية (Bucket)

أدخل اسم حاوية S3.
