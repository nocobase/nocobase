# Tencent Cloud COS

محرك تخزين يعتمد على Tencent Cloud COS. يجب عليك إعداد الحسابات والأذونات ذات الصلة قبل الاستخدام.

## معلمات التكوين

![مثال على تكوين محرك تخزين Tencent COS](https://static-docs.nocobase.com/20240712222125.png)

:::info{title=ملاحظة}
يقدم هذا القسم المعلمات الخاصة بمحرك تخزين Tencent Cloud COS فقط. للاطلاع على المعلمات العامة، يرجى الرجوع إلى [معلمات المحرك العامة](./index.md#general-engine-parameters).
:::

### المنطقة

أدخل المنطقة الخاصة بتخزين COS، على سبيل المثال: `ap-chengdu`.

:::info{title=ملاحظة}
يمكنك عرض معلومات المنطقة الخاصة بالمجموعة في [وحدة تحكم Tencent Cloud COS](https://console.cloud.tencent.com/cos). ما عليك سوى استخدام بادئة المنطقة (وليس اسم النطاق الكامل).
:::

### SecretId

أدخل معرف (ID) مفتاح الوصول المصرح به لـ Tencent Cloud.

### SecretKey

أدخل سر (Secret) مفتاح الوصول المصرح به لـ Tencent Cloud.

### المجموعة

أدخل اسم مجموعة تخزين COS، على سبيل المثال: `qing-cdn-1234189398`.
