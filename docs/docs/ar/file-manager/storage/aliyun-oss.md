# محرك التخزين: Aliyun OSS

محرك تخزين يعتمد على خدمة Aliyun OSS. قبل الاستخدام، يجب تجهيز الحساب والصلاحيات اللازمة.

## معلمات الإعداد

![مثال على إعداد محرك تخزين Aliyun OSS](https://static-docs.nocobase.com/20240712220011.png)

:::info{title=ملاحظة}
يوضح هذا القسم المعلمات الخاصة بمحرك تخزين Aliyun OSS فقط. أما المعلمات العامة، فيُرجى الرجوع إلى [معلمات المحرك العامة](./index#引擎通用参数).
:::

### المنطقة (Region)

أدخل منطقة تخزين OSS، على سبيل المثال: `oss-cn-hangzhou`.

:::info{title=ملاحظة}
يمكنك عرض معلومات المنطقة الخاصة بالحاوية (Bucket) من خلال [لوحة تحكم Aliyun OSS](https://oss.console.aliyun.com/)، ويكفي استخدام بادئة المنطقة فقط (وليس اسم النطاق الكامل).
:::

### AccessKey ID

أدخل معرّف مفتاح الوصول الخاص بك في Aliyun.

### AccessKey Secret

أدخل المفتاح السري (Secret) لمفتاح الوصول الخاص بك في Aliyun.

### الحاوية (Bucket)

أدخل اسم حاوية OSS.

### المهلة الزمنية (Timeout)

أدخل المهلة الزمنية لرفع الملفات إلى Aliyun OSS بوحدة المللي ثانية. القيمة الافتراضية هي `60000` مللي ثانية (أي 60 ثانية).
