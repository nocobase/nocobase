# محرك التخزين: عليون OSS

محرك تخزين يعتمد على عليون OSS. قبل الاستخدام، يجب عليك إعداد الحسابات والأذونات اللازمة.

## معلمات التكوين

![مثال على تكوين محرك التخزين عليون OSS](https://static-docs.nocobase.com/20240712220011.png)

:::info{title=تلميح}
يقدم هذا القسم المعلمات الخاصة بمحرك التخزين عليون OSS فقط. للاطلاع على المعلمات العامة، يرجى الرجوع إلى [المعلمات العامة للمحرك](./index#引擎通用参数).
:::

### المنطقة

أدخل منطقة تخزين OSS، على سبيل المثال: `oss-cn-hangzhou`.

:::info{title=تلميح}
يمكنك عرض معلومات منطقة المخزن الخاص بك في [وحدة تحكم عليون OSS](https://oss.console.aliyun.com/)، وتحتاج فقط إلى استخدام بادئة المنطقة (وليس اسم النطاق الكامل).
:::

### AccessKey ID

أدخل معرف مفتاح الوصول الخاص بحساب عليون.

### AccessKey Secret

أدخل سر مفتاح الوصول الخاص بحساب عليون.

### المخزن (Bucket)

أدخل اسم مخزن OSS.

### مهلة

أدخل مهلة الرفع إلى عليون OSS، بالمللي ثانية. القيمة الافتراضية هي `60000` مللي ثانية (أي 60 ثانية).
