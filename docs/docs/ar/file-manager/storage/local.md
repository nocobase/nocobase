# محرك التخزين: التخزين المحلي

سيتم حفظ الملفات المرفوعة في دليل محلي على القرص الصلب للخادم. يناسب هذا السيناريوهات التي يكون فيها الحجم الإجمالي للملفات المرفوعة التي يديرها النظام صغيرًا، أو للأغراض التجريبية.

## معلمات التكوين

![مثال على تكوين محرك تخزين الملفات](https://static-docs.nocobase.com/20240529115151.png)

:::info{title=ملاحظة}
يُقدم هذا القسم المعلمات الخاصة بمحرك التخزين المحلي فقط. للمعلومات حول المعلمات العامة، يُرجى الرجوع إلى [معلمات المحرك العامة](./index.md#general-engine-parameters).
:::

### المسار

يمثل هذا كلاً من المسار النسبي لتخزين الملفات على الخادم ومسار الوصول عبر URL. على سبيل المثال، "`user/avatar`" (بدون شرطات مائلة في البداية أو النهاية) يمثل ما يلي:

1.  المسار النسبي على الخادم حيث يتم تخزين الملفات المرفوعة: `/path/to/nocobase-app/storage/uploads/user/avatar`.
2.  بادئة عنوان URL للوصول إلى الملفات: `http://localhost:13000/storage/uploads/user/avatar`.
