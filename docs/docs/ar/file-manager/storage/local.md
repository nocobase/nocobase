# محرك التخزين: التخزين المحلي

سيتم حفظ الملفات المرفوعة في مجلد محلي على القرص الصلب الخاص بالخادم. يُعد هذا مناسبًا للحالات التي يكون فيها الحجم الإجمالي للملفات المرفوعة صغيرًا أو لأغراض التجربة.

### المسار (Path)

يمثل كلًا من المسار النسبي لتخزين الملفات على الخادم ومسار الوصول عبر URL.  
على سبيل المثال: `user/avatar` (بدون "/" في البداية أو النهاية) يشير إلى:

1. المسار النسبي على الخادم حيث يتم حفظ الملفات المرفوعة:  
   `/path/to/nocobase-app/storage/uploads/user/avatar`

2. بادئة رابط الوصول إلى الملفات:  
   `http://localhost:13000/storage/uploads/user/avatar`
