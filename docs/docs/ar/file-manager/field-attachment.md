# حقل المرفقات

## مقدمة

يشتمل النظام على نوع حقل "المرفقات" المدمج لدعم تحميل الملفات من قبل المستخدمين في مجموعات البيانات المخصصة.

حقل المرفقات هو في الأساس حقل ارتباط من نوع علاقة متعدد لمتعدد، يشير إلى مجموعة "المرفقات" (`attachments`) المدمجة في النظام. عند إنشاء حقل مرفقات في أي مجموعة بيانات، يتم تلقائيًا إنشاء جدول وسيط لعلاقة متعدد لمتعدد. تُخزن البيانات الوصفية للملفات المحملة في مجموعة "المرفقات"، ويتم ربط معلومات الملفات المشار إليها في مجموعة البيانات عبر هذا الجدول الوسيط.

## إعدادات الحقل

![20240512180916](https://static-docs.nocobase.com/20251031000729.png)

### قيود نوع MIME

تُستخدم هذه القيود لتقييد أنواع الملفات المسموح بتحميلها، وذلك باستخدام تنسيق بناء جملة [MIME](https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/MIME_types). على سبيل المثال: يمثل `image/*` ملفات الصور. يمكن فصل أنواع متعددة بفاصلة، على سبيل المثال: يسمح `image/*,application/pdf` بأنواع ملفات الصور و PDF معًا.

### محرك التخزين

اختر محرك التخزين المستخدم لتخزين الملفات المحملة. إذا تُرك هذا الحقل فارغًا، فسيتم استخدام محرك التخزين الافتراضي للنظام.
