# معاينة الملفات

في الواجهات التي تحتوي على حقول ملفات (بما في ذلك حقول المرفقات)، يمكنك معاينة الملفات من خلال النقر على الصورة المصغّرة للملف أو الأيقونة الخاصة به. تدعم ميزة المعاينة المدمجة عدة أنواع من الملفات، بما في ذلك الصور وملفات PDF ومعظم أنواع الملفات التي يدعمها المتصفح بشكل أصلي.

![20251129232307](https://static-docs.nocobase.com/20251129232307.png)

بالنسبة لأنواع الملفات التي لا تدعم المعاينة بشكل أصلي، يمكنك تفعيل ميزة المعاينة عن طريق تثبيت أو توسيع إضافات معاينة الملفات المناسبة. على سبيل المثال، بعد تثبيت إضافة معاينة ملفات Office، يمكنك معاينة ملفات Word وExcel وPowerPoint.
