# معاينة الملفات

في الواجهات التي تحتوي على حقول الملفات (بما في ذلك حقول المرفقات)، يمكنك معاينة الملفات بالنقر على الصورة المصغرة للملف أو أيقونته. تدعم وظيفة المعاينة المدمجة أنواعًا متعددة من الملفات، بما في ذلك الصور وملفات PDF، ومعظم أنواع الملفات التي تدعمها المتصفحات بشكل أصلي.

![20251129232307](https://static-docs.nocobase.com/20251129232307.png)

بالنسبة لأنواع الملفات التي لا تدعم المعاينة الأصلية، يمكنك تمكين وظيفة المعاينة عن طريق تثبيت أو توسيع الإضافات المناسبة لمعاينة الملفات. على سبيل المثال، بعد تثبيت إضافة معاينة ملفات Office، يمكنك معاينة ملفات Word و Excel و PowerPoint.
