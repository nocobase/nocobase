# المساحات المتعددة

<PluginInfo name="multi-space" licenseBundled="professional"></PluginInfo>

## مقدمة

يوفر Multi-space عزلاً منطقياً للبيانات داخل نفس التطبيق عبر مساحات مستقلة.

## التثبيت

Enable **Multi-Space** in plugin manager.

![](https://static-docs.nocobase.com/multi-space/Plugin-manager-NocoBase-10-15-2025_09_57_AM.png)

## دليل الاستخدام

### إدارة المساحات المتعددة
