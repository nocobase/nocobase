# إدارة التطبيقات المتعددة

## نظرة عامة

تتيح إدارة التطبيقات المتعددة في NocoBase إنشاء وتشغيل عدة تطبيقات معزولة عبر نقطة دخول واحدة بواسطة AppSupervisor.


في البداية يكفي تطبيق واحد لسهولة النشر والإدارة، لكن مع نمو الأعمال تزداد كلفة الصيانة ويصعب عزل المجالات.


في هذا الوضع تعمل عدة تطبيقات داخل نفس عملية NocoBase مع استقلال قواعد البيانات ودورة الحياة لكل تطبيق، مع مشاركة الذاكرة والمعالج.

![](https://static-docs.nocobase.com/202512231055907.png)


عند التوسع الكبير يُنصح بالنشر الهجين متعدد البيئات: تطبيق دخول Supervisor وعدة بيئات Worker لتوزيع الحمل وتحسين العزل.

![](https://static-docs.nocobase.com/202512231215186.png)
