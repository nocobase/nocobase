# ctx.getModel()

جلب مثيل نموذج (مثل `BlockModel` أو `PageModel` أو `ActionModel` إلخ) من المحرك الحالي أو مكدس العرض بناءً على المعرف الفريد `uid` للنموذج. يُستخدم هذا في RunJS للوصول إلى نماذج أخرى عبر الكتل أو الصفحات أو النوافذ المنبثقة.

إذا كنت تحتاج فقط إلى النموذج أو الكتلة التي يتواجد فيها سياق التنفيذ الحالي، فمن الأفضل استخدام `ctx.model` أو `ctx.blockModel` بدلاً من `ctx.getModel`.

## حالات الاستخدام

| السيناريو | الوصف |
|------|------|
| **JSBlock / JSAction** | الحصول على نماذج الكتل الأخرى بناءً على `uid` معروف لقراءة أو كتابة `resource` أو `form` أو `setProps` وغيرها. |
| **RunJS في النوافذ المنبثقة** | عند الحاجة للوصول إلى نموذج في الصفحة التي فتحت النافذة المنبثقة، مرر `searchInPreviousEngines: true`. |
| **الإجراءات المخصصة** | تحديد موقع النماذج أو النماذج الفرعية في لوحة الإعدادات حسب الـ `uid` عبر مكدسات العرض لقراءة تكويناتها أو حالتها. |

## تعريف النوع (Type Definition)

```ts
getModel<T extends FlowModel = FlowModel>(
  uid: string,
  searchInPreviousEngines?: boolean
): T | undefined
```

## المعاملات (Parameters)

| المعامل | النوع | الوصف |
|------|------|------|
| `uid` | `string` | المعرف الفريد لمثيل النموذج المستهدف، يتم تحديده أثناء التكوين أو الإنشاء (مثل `ctx.model.uid`). |
| `searchInPreviousEngines` | `boolean` | اختياري، القيمة الافتراضية `false`. عند ضبطه على `true` يبحث في "مكدس العرض" بدءاً من المحرك الحالي وصولاً إلى الجذر، مما يسمح بالوصول إلى النماذج في المحركات العليا (مثل الصفحة التي فتحت نافذة منبثقة). |

## القيمة المرجعة

- يعيد مثيل الفئة الفرعية لـ `FlowModel` المقابل (مثل `BlockModel` أو `FormBlockModel` أو `ActionModel`) في حال العثور عليه.
- يعيد `undefined` في حال عدم العثور عليه.

## نطاق البحث

- **الافتراضي (`searchInPreviousEngines: false`)**: يبحث فقط داخل **المحرك الحالي** حسب الـ `uid`. في النوافذ المنبثقة أو العروض متعددة المستويات، يكون لكل عرض محرك مستقل؛ وبشكل افتراضي، يتم البحث فقط عن النماذج داخل العرض الحالي.
- **`searchInPreviousEngines: true`**: يبحث صعوداً عبر سلسلة `previousEngine` بدءاً من المحرك الحالي، ويعيد أول تطابق. هذا مفيد للوصول إلى نموذج في الصفحة التي فتحت النافذة المنبثقة الحالية.

## أمثلة

### الحصول على كتلة أخرى وتحديثها

```ts
const block = ctx.getModel('list-block-uid');
if (block?.resource) {
  await block.resource.refresh();
}
```

### الوصول إلى نموذج في الصفحة من نافذة منبثقة

```ts
// الوصول إلى كتلة في الصفحة التي فتحت النافذة المنبثقة الحالية
const pageBlock = ctx.getModel('page-block-uid', true);
if (pageBlock) {
  pageBlock.rerender?.();
}
```

### القراءة والكتابة عبر النماذج وتحفيز إعادة العرض (rerender)

```ts
const target = ctx.getModel('other-block-uid');
if (target) {
  target.setProps({ loading: true });
  target.rerender?.();
}
```

### التحقق من الأمان

```ts
const model = ctx.getModel(someUid);
if (!model) {
  ctx.message.warning('النموذج المستهدف غير موجود');
  return;
}
```
