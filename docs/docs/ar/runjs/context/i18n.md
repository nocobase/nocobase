# ctx.i18n

مثيل i18n للسياق الحالي، يُستخدم لقراءة أو تبديل اللغات. **استخدم [ctx.t()](./t.md) لترجمة النصوص**؛ لا تستخدم `ctx.i18n.t`.

## حالات الاستخدام

يمكن لجميع بيئات تنفيذ RunJS استخدام `ctx.i18n` (مثل JSBlock وJSField وJSItem وJSColumn وسير العمل وقواعد الارتباط، وما إلى ذلك).

## تعريف النوع (Type Definition)

```typescript
interface i18n: {
  language: string;
  changeLanguage(lng: string): Promise<any>;
}
```

## الخصائص الشائعة

| الخاصية | النوع | الوصف |
|------|------|------|
| `language` | `string` | رمز اللغة النشط حالياً (مثل `en-US` و `zh-CN`) |

## الطرق الشائعة (Methods)

### changeLanguage(lng)

تبديل اللغة الحالية.

| المعامل | النوع | الوصف |
|------|------|------|
| `lng` | `string` | رمز اللغة المستهدفة (مثل `'en-US'` أو `'zh-CN'`) |

**القيمة المرجعة**: `Promise<any>`، يتم حله (resolve) بعد اكتمال عملية تبديل اللغة.

## أمثلة

### قراءة اللغة الحالية

```ts
const lang = ctx.i18n.language;
// 'zh-CN' | 'en-US' | ...
if (lang.startsWith('zh')) {
  ctx.render(ctx.t('中文界面'));
} else {
  ctx.render(ctx.t('English UI'));
}
```

### تبديل اللغة

```ts
// التبديل إلى الإنجليزية
await ctx.i18n.changeLanguage('en-US');

// التبديل إلى الصينية
await ctx.i18n.changeLanguage('zh-CN');
```

### زر تبديل اللغة

```ts
const { Button } = ctx.libs.antd;
const isZh = ctx.i18n.language.startsWith('zh');
ctx.render(
  <Button onClick={async () => {
    await ctx.i18n.changeLanguage(isZh ? 'en-US' : 'zh-CN');
  }}>
    {ctx.t(isZh ? 'Switch to English' : '切换到中文')}
  </Button>,
);
```

## ملاحظات

- **نصوص الترجمة**: استخدم `ctx.t()` بشكل موحد، ولا تستخدم `ctx.i18n.t`.
