# نظرة عامة على المجموعات

توفر NocoBase لغة وصف خاصة (DSL) لوصف بنية البيانات، تُعرف باسم **المجموعة**. تعمل هذه اللغة على توحيد بنية البيانات من مصادر مختلفة، مما يوفر أساسًا موثوقًا لإدارة البيانات وتحليلها وتطبيقاتها اللاحقة.

![20240512161522](https://static-docs.nocobase.com/20240512161522.png)

لتسهيل استخدام نماذج البيانات المتنوعة، تدعم NocoBase إنشاء أنواع مختلفة من المجموعات:

- [المجموعة العامة](/data-sources/data-source-main/general-collection): تتضمن حقول النظام الشائعة المدمجة.
- [مجموعة الوراثة](/data-sources/data-source-main/inheritance-collection): تتيح لك إنشاء مجموعة رئيسية (أب) ثم اشتقاق مجموعة فرعية (ابن) منها. ترث المجموعة الفرعية بنية المجموعة الرئيسية ويمكنها أيضًا تعريف أعمدتها الخاصة.
- [مجموعة الشجرة](/data-sources/collection-tree): مجموعة ذات بنية شجرية، تدعم حاليًا تصميم قائمة التجاور فقط.
- [مجموعة التقويم](/data-sources/calendar/calendar-collection): تُستخدم لإنشاء مجموعات الأحداث المتعلقة بالتقويم.
- [مجموعة الملفات](/data-sources/file-manager/file-collection): تُستخدم لإدارة تخزين الملفات.
- []: تُستخدم لسيناريوهات التعبيرات الديناميكية في سير العمل.
- [مجموعة SQL](/data-sources/collection-sql): ليست مجموعة قاعدة بيانات فعلية، بل تعرض استعلامات SQL بسرعة وبشكل منظم.
- [مجموعة العرض](/data-sources/collection-view): تتصل بعروض قاعدة البيانات الموجودة مسبقًا.
- [المجموعة الخارجية](/data-sources/collection-fdw): تسمح لنظام قاعدة البيانات بالوصول المباشر إلى البيانات في مصادر البيانات الخارجية والاستعلام عنها، بالاعتماد على تقنية FDW.
