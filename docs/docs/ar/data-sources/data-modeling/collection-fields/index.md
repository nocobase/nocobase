## أنواع بيانات الحقول

لكل واجهة حقل (Field Interface) نوع بيانات افتراضي. على سبيل المثال، بالنسبة للحقول التي تكون واجهتها رقمية (Number)، يكون نوع البيانات الافتراضي هو `double`، ولكن يمكن أن يكون أيضًا `float` أو `decimal` وما إلى ذلك. أنواع البيانات المدعومة حاليًا هي:

![20240512103733](https://static-docs.nocobase.com/20240512103733.png)

## ربط أنواع الحقول

عملية إضافة حقول جديدة إلى قاعدة البيانات الرئيسية هي كالتالي:

1.  اختر نوع الواجهة (Interface).
2.  قم بتكوين نوع البيانات الاختياري للواجهة الحالية.

![20240512172416](https://static-docs.nocobase.com/20240512172416.png)

عملية ربط الحقول من مصادر البيانات الخارجية هي:

1.  قم بربط نوع البيانات المقابل (Field type) ونوع واجهة المستخدم (UI type) تلقائيًا بناءً على نوع الحقل في قاعدة البيانات الخارجية.
2.  عدّل إلى نوع بيانات ونوع واجهة (Interface) أكثر ملاءمة حسب الحاجة.

![20240512172759](https://static-docs.nocobase.com/20240512172759.png)
