# نظرة عامة

يُعد نمذجة البيانات خطوة أساسية في تصميم قواعد البيانات، وتتضمن عملية تحليل وتجريد عميق لأنواع البيانات المختلفة وعلاقاتها المتبادلة في العالم الحقيقي. في هذه العملية، نسعى للكشف عن الروابط الجوهرية بين البيانات وصياغتها رسميًا في نماذج بيانات، مما يضع الأساس لهيكل قاعدة بيانات نظام المعلومات. NocoBase هي منصة تعتمد على نماذج البيانات، وتتميز بما يلي:

## تدعم الوصول إلى البيانات من مصادر متنوعة

يمكن أن تكون مصادر بيانات NocoBase قواعد بيانات شائعة، ومنصات API (SDK)، وملفات.

![20240512085558](https://static-docs.nocobase.com/20240512085558.png)

توفر NocoBase [إضافة إدارة مصادر البيانات](/data-sources/data-source-manager) لإدارة مصادر البيانات المختلفة ومجموعاتها. توفر إضافة إدارة مصادر البيانات واجهة إدارة لجميع مصادر البيانات فقط، ولا توفر القدرة على الوصول المباشر إلى مصادر البيانات. يجب استخدامها بالاقتران مع إضافات مصادر البيانات المختلفة. تشمل مصادر البيانات المدعومة حاليًا ما يلي:

-   [قاعدة البيانات الرئيسية](/data-sources/data-source-main): قاعدة البيانات الرئيسية لـ NocoBase، تدعم قواعد البيانات العلائقية مثل MySQL و PostgreSQL و MariaDB.
-   [KingbaseES](/data-sources/data-source-kingbase): استخدام قاعدة بيانات KingbaseES كمصدر بيانات، ويمكن استخدامها كقاعدة بيانات رئيسية أو خارجية.
-   [MySQL خارجي](/data-sources/data-source-external-mysql): استخدام قاعدة بيانات MySQL خارجية كمصدر بيانات.
-   [MariaDB خارجي](/data-sources/data-source-external-mariadb): استخدام قاعدة بيانات MariaDB خارجية كمصدر بيانات.
-   [PostgreSQL خارجي](/data-sources/data-source-external-postgres): استخدام قاعدة بيانات PostgreSQL خارجية كمصدر بيانات.
-   [MSSQL خارجي](/data-sources/data-source-external-mssql): استخدام قاعدة بيانات MSSQL (SQL Server) خارجية كمصدر بيانات.
-   [Oracle خارجي](/data-sources/data-source-external-oracle): استخدام قاعدة بيانات Oracle خارجية كمصدر بيانات.

![20240512083651](https://static-docs.nocobase.com/20240512083651.png)

## توفر مجموعة متنوعة من أدوات نمذجة البيانات

**واجهة إدارة مجموعات بسيطة**: تُستخدم لإنشاء مجموعات مختلفة أو الاتصال بالمجموعات الموجودة.

![20240512090751](https://static-docs.nocobase.com/20240512090751.png)

**واجهة مرئية على غرار مخططات ER**: تُستخدم لاستخلاص الكيانات وعلاقاتها من متطلبات المستخدم والأعمال. توفر طريقة بديهية وسهلة الفهم لوصف نماذج البيانات. من خلال مخططات ER، يمكنك فهم كيانات البيانات الرئيسية وعلاقاتها في النظام بشكل أوضح.

![20240512091042](https://static-docs.nocobase.com/20240410075906.png)
