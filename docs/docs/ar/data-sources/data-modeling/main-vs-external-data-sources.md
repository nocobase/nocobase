# مقارنة بين قواعد البيانات الرئيسية والخارجية

تتجلى الفروقات بين قواعد البيانات الرئيسية وقواعد البيانات الخارجية في NocoBase بشكل أساسي في أربعة جوانب: دعم أنواع قواعد البيانات، دعم أنواع المجموعات، دعم أنواع الحقول، وقدرات النسخ الاحتياطي والترحيل.

## 1. دعم أنواع قواعد البيانات

لمزيد من التفاصيل، يرجى الاطلاع على: [إدارة مصادر البيانات](https://docs.nocobase.com/data-sources/data-source-manager)

### أنواع قواعد البيانات

| نوع قاعدة البيانات | دعم قاعدة البيانات الرئيسية | دعم قاعدة البيانات الخارجية |
|------------------|---------------------------|------------------------------|
| PostgreSQL | ✅ | ✅ |
| MySQL | ✅ | ✅ |
| MariaDB | ✅ | ✅ |
| KingbaseES | ✅ | ✅ |
| MSSQL | ❌ | ✅ |
| Oracle | ❌ | ✅ |

### إدارة المجموعات

| إدارة المجموعات | دعم قاعدة البيانات الرئيسية | دعم قاعدة البيانات الخارجية |

| الإدارة الأساسية | ✅ | ✅ |
| الإدارة المرئية | ✅ | ❌ |

## 4. مقارنة دعم النسخ الاحتياطي والترحيل

| الميزة | قاعدة البيانات الرئيسية | قاعدة البيانات الخارجية |
|---------|-------------------|---------------------|
| النسخ الاحتياطي والاستعادة | ✅ | ❌ (يتطلب إدارة المستخدم) |
| إدارة الترحيل | ✅ | ❌ (يتطلب إدارة المستخدم) |

:::info
يوفر NocoBase إمكانيات النسخ الاحتياطي والاستعادة وترحيل الهيكل لقواعد البيانات الرئيسية. أما بالنسبة لقواعد البيانات الخارجية، فيجب على المستخدمين إتمام هذه العمليات بشكل مستقل وفقًا لبيئة قاعدة بياناتهم الخاصة، ولا يقدم NocoBase دعمًا مدمجًا لذلك.
:::

## ملخص المقارنة

| عنصر المقارنة | قاعدة البيانات الرئيسية | قاعدة البيانات الخارجية |
|----------------|-------------------|---------------------|
| أنواع قواعد البيانات | PostgreSQL، MySQL، MariaDB، KingbaseES | PostgreSQL، MySQL، MariaDB، MSSQL، Oracle، KingbaseES |
| دعم أنواع المجموعات | جميع أنواع المجموعات | المجموعات العامة ومجموعات العرض فقط |
| دعم أنواع الحقول | جميع أنواع الحقول | جميع أنواع الحقول باستثناء حقول المرفقات |
| النسخ الاحتياطي والترحيل | دعم مدمج | يتطلب إدارة المستخدم |

## توصيات

- **إذا كنت تستخدم NocoBase لبناء نظام أعمال جديد بالكامل**، يرجى استخدام **قاعدة البيانات الرئيسية**، حيث سيمكنك ذلك من الاستفادة من وظائف NocoBase الكاملة.
- **إذا كنت تستخدم NocoBase للاتصال بقواعد بيانات أنظمة أخرى لإجراء عمليات CRUD الأساسية**، فاستخدم **قواعد البيانات الخارجية**.
