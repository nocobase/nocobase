# تنسنت COS

محرك تخزين يعتمد على تنسنت كلاود COS. قبل الاستخدام، يلزم إعداد الحسابات والأذونات اللازمة.

## خيارات التكوين

![مثال على خيارات تكوين محرك تخزين تنسنت COS](https://static-docs.nocobase.com/20240712222125.png)

:::info{title=ملاحظة}
يغطي هذا القسم فقط الخيارات المخصصة لمحرك تخزين تنسنت كلاود COS. للاطلاع على المعلمات العامة، يرجى الرجوع إلى [معلمات المحرك العامة](./index.md#common-engine-parameters).
:::

### المنطقة

املأ منطقة تخزين COS، على سبيل المثال: `ap-chengdu`.

:::info{title=ملاحظة}
يمكنك عرض معلومات المنطقة الخاصة بسلة التخزين في [وحدة تحكم تنسنت كلاود COS](https://console.cloud.tencent.com/cos)، وتحتاج فقط إلى أخذ الجزء البادئ من المنطقة (دون الحاجة إلى اسم النطاق الكامل).
:::

### SecretId

املأ معرف مفتاح الوصول المصرح به من تنسنت كلاود.

### SecretKey

املأ سر مفتاح الوصول المصرح به من تنسنت كلاود.

### سلة التخزين

املأ اسم سلة تخزين COS، على سبيل المثال: `qing-cdn-1234189398`.
