# التخزين المحلي

سيتم حفظ الملفات المرفوعة في دليل محلي على خادمك. هذا مناسب للسيناريوهات التجريبية أو ذات النطاق الصغير حيث يكون إجمالي عدد الملفات التي يديرها النظام قليلاً نسبيًا.

## الخيارات

![مثال على خيارات محرك تخزين الملفات](https://static-docs.nocobase.com/20240529115151.png)

:::info{title=ملاحظة}
يغطي هذا القسم الخيارات المحددة لمحرك التخزين المحلي فقط. للاطلاع على المعلمات المشتركة، يرجى الرجوع إلى [معلمات المحرك العامة](./index.md#general-engine-parameters).
:::

### المسار

يمثل المسار كلاً من المسار النسبي للملف المخزن على الخادم ومسار الوصول عبر URL. على سبيل المثال، يمثل "`user/avatar`" (بدون الشرطة المائلة الأمامية أو الخلفية "`/`") ما يلي:

1.  المسار النسبي للملف المرفوع المخزن على الخادم: `/path/to/nocobase-app/storage/uploads/user/avatar`.
2.  بادئة عنوان URL للوصول إلى الملف: `http://localhost:13000/storage/uploads/user/avatar`.
