# Aliyun OSS

محرك تخزين يعتمد على Aliyun OSS. يتطلب إعداد الحسابات والأذونات ذات الصلة قبل الاستخدام.

## الإعدادات

![مثال على إعداد محرك تخزين Aliyun OSS](https://static-docs.nocobase.com/20240712220011.png)

:::info{title=ملاحظة}
نقدم هنا فقط المعلمات الخاصة بمحرك تخزين Aliyun OSS. للاطلاع على المعلمات العامة، يرجى الرجوع إلى [معلمات المحرك العامة](./index.md#common-engine-parameters).
:::

### المنطقة

أدخل منطقة تخزين OSS، على سبيل المثال: `oss-cn-hangzhou`.

:::info{title=ملاحظة}
يمكنك عرض معلومات منطقة مساحة التخزين في [وحدة تحكم Aliyun OSS](https://oss.console.aliyun.com/). تحتاج فقط إلى استخدام الجزء البادئ للمنطقة (دون الحاجة إلى اسم النطاق الكامل).
:::

### معرف مفتاح الوصول (AccessKey ID)

أدخل معرف مفتاح الوصول المعتمد من Alibaba Cloud.

### سر مفتاح الوصول (AccessKey Secret)

أدخل سر مفتاح الوصول المعتمد من Alibaba Cloud.

### الحاوية (Bucket)

أدخل اسم حاوية تخزين OSS.
