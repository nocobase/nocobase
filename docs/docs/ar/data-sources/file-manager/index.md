# مدير الملفات

## مقدمة

توفر إضافة مدير الملفات مجموعة ملفات، وحقل مرفقات، ومحركات تخزين ملفات لإدارة الملفات بفعالية. الملفات هي سجلات في نوع خاص من المجموعات، تُعرف باسم مجموعة الملفات، والتي تخزن البيانات الوصفية للملفات ويمكن إدارتها عبر مدير الملفات. حقول المرفقات هي حقول ارتباط محددة مرتبطة بمجموعة الملفات. تدعم الإضافة طرق تخزين متعددة، وتشمل محركات تخزين الملفات المدعومة حاليًا التخزين المحلي، وAlibaba Cloud OSS، وAmazon S3، وTencent Cloud COS.

## دليل المستخدم

### مجموعة الملفات

توجد مجموعة مرفقات مدمجة لتخزين جميع الملفات المرتبطة بحقول المرفقات. بالإضافة إلى ذلك، يمكن إنشاء مجموعات ملفات جديدة لتخزين ملفات محددة.

[مزيد من المعلومات في وثائق مجموعة الملفات](/data-sources/file-manager/file-collection)

### حقل المرفقات

حقول المرفقات هي حقول ارتباط محددة مرتبطة بمجموعة الملفات، ويمكن إنشاؤها عبر نوع حقل "المرفقات" أو تكوينها عبر حقل "الارتباط".

[مزيد من المعلومات في وثائق حقل المرفقات](/data-sources/file-manager/field-attachment)

### محرك تخزين الملفات

يُستخدم محرك تخزين الملفات لحفظ الملفات في خدمات محددة، بما في ذلك التخزين المحلي (الحفظ على القرص الصلب للخادم)، والتخزين السحابي، وغير ذلك.

[مزيد من المعلومات في وثائق محرك تخزين الملفات](./storage/index.md)

### HTTP API

يمكن معالجة عمليات تحميل الملفات عبر HTTP API، راجع [HTTP API](./http-api.md).
