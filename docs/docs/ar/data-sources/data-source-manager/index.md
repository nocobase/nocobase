# إدارة مصادر البيانات

## مقدمة

توفر NocoBase إضافة إدارة مصادر البيانات، والتي تُستخدم لإدارة مصادر البيانات والمجموعات الخاصة بها. إن إضافة إدارة مصادر البيانات توفر فقط واجهة إدارة لجميع مصادر البيانات، ولا توفر القدرة على الوصول إلى مصادر البيانات مباشرةً. بل تحتاج إلى العمل جنبًا إلى جنب مع إضافات مصادر البيانات المختلفة. تشمل مصادر البيانات المدعومة حاليًا للوصول إليها ما يلي:

- [قاعدة البيانات الرئيسية](/data-sources/data-source-main): قاعدة بيانات NocoBase الرئيسية، تدعم قواعد البيانات العلائقية مثل MySQL وPostgreSQL وMariaDB.
- [MySQL خارجي](/data-sources/data-source-external-mysql): استخدام قاعدة بيانات MySQL خارجية كمصدر بيانات.
- [MariaDB خارجي](/data-sources/data-source-external-mariadb): استخدام قاعدة بيانات MariaDB خارجية كمصدر بيانات.
- [PostgreSQL خارجي](/data-sources/data-source-external-postgres): استخدام قاعدة بيانات PostgreSQL خارجية كمصدر بيانات.
- [MSSQL خارجي](/data-sources/data-source-external-mssql): استخدام قاعدة بيانات MSSQL (SQL Server) خارجية كمصدر بيانات.
- [Oracle خارجي](/data-sources/data-source-external-oracle): استخدام قاعدة بيانات Oracle خارجية كمصدر بيانات.

بالإضافة إلى ذلك، يمكن توسيع أنواع إضافية عبر الإضافات، والتي يمكن أن تكون أنواعًا شائعة من قواعد البيانات، أو منصات توفر واجهات برمجة التطبيقات (SDKs).

## التثبيت

إضافة مدمجة، لا تتطلب تثبيتًا منفصلاً.

## تعليمات الاستخدام

عند تهيئة التطبيق وتثبيته، يتم توفير مصدر بيانات افتراضي لتخزين بيانات NocoBase، يُعرف بقاعدة البيانات الرئيسية. لمزيد من المعلومات، راجع وثائق [قاعدة البيانات الرئيسية](/data-sources/data-source-main/).

### مصادر البيانات الخارجية

يتم دعم قواعد البيانات الخارجية كمصادر بيانات. لمزيد من المعلومات، راجع وثائق [قاعدة البيانات الخارجية / مقدمة](/data-sources/data-source-manager/external-database).

### دعم مزامنة جداول قواعد البيانات المخصصة
