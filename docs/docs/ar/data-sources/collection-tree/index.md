# المجموعة الشجرية

## مقدمة

المجموعة الشجرية هي نمط تصميم لمجموعات البيانات يُستخدم لتنظيم البيانات ذات العلاقات الهرمية. تشبه بنية هذه المجموعة الشجرة، حيث يمكن لكل عنصر بيانات أن يحتوي على عنصر فرعي واحد أو أكثر، ويمكن لهذه العناصر الفرعية بدورها أن تحتوي على عناصر فرعية خاصة بها.
