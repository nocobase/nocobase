# حقل المنطقة الصينية

## مقدمة

تضيف إضافة حقل المنطقة الصينية (`field-china-region`) نوع حقل "المنطقة الصينية" إلى NocoBase.

## التثبيت

في واجهة NocoBase الرئيسية، انتقل إلى "مدير الإضافات"، وابحث عن إضافة "المنطقة الصينية"، ثم قم بتمكينها.

## الاستخدام

في إعدادات المجموعة، أضف حقلاً جديداً واختر "المنطقة الصينية" كنوع للحقل.

![](/images/field-china-region-1.png)

### إعدادات الحقل

في إعدادات الحقل، يمكنك تعيين وضع العرض للحقل:

- المقاطعة
- المقاطعة/المدينة
- المقاطعة/المدينة/المنطقة
- المقاطعة/المدينة/المنطقة/المقاطعة الفرعية

![](/images/field-china-region-2.png)
