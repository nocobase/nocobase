## BelongsToRepository

واجهته مطابقة لواجهة `[HasOneRepository](./has-one-repository.md)`.
`BelongsToRepository` هو `Repository` يُستخدم للتعامل مع علاقات `BelongsTo`، ويوفر طرقًا مريحة لمعالجة هذه العلاقات.
