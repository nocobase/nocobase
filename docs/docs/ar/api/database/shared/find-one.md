## النوع

```typescript
type FindOneOptions = Omit<FindOptions, 'limit'>;
```

## المعلمات

معظم المعلمات هي نفسها المستخدمة في `find()`. الفرق هو أن `findOne()` تُرجع سجلًا واحدًا فقط، لذا لا حاجة لمعلمة `limit`، وتكون قيمتها دائمًا `1` أثناء الاستعلام.
