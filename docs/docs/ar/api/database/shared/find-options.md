# ما هو NocoBase؟

NocoBase هو منصة تطوير بدون كود (no-code) مفتوحة المصدر وعالية القابلية للتوسع. فهو يتيح للشركات والمطورين بناء أنظمة داخلية ومنصات إدارة مخصصة بسرعة وسهولة، مع الحفاظ على المرونة الكاملة للتوسع المستقبلي.

بدلاً من قضاء أسابيع أو أشهر في كتابة الأكواد، يتيح لك NocoBase نشر نظامك الخاص في غضون دقائق.

## المفاهيم الأساسية

### مجموعة (Collection)
تُعد الـ **مجموعة** الطريقة الأساسية لتنظيم البيانات في NocoBase. يمكنك التفكير فيها كجدول في قاعدة البيانات، حيث تحتوي كل مجموعة على حقول تحدد نوع البيانات المخزنة (مثل النصوص، الأرقام، أو العلاقات).

### سير العمل (Workflow)
يتيح لك **سير العمل** أتمتة العمليات التجارية والمهام الروتينية. يمكنك إعداد مشغلات (Triggers) وإجراءات (Actions) لتنفيذ العمليات تلقائياً بناءً على أحداث معينة داخل النظام، مما يقلل من التدخل البشري ويزيد الكفاءة.

### مصدر البيانات (Data Source)
يدعم NocoBase الاتصال بمختلف أنواع **مصدر البيانات**. سواء كنت تستخدم قاعدة بيانات MySQL أو PostgreSQL أو SQLite، يمكنك ربطها بالمنصة وإدارة بياناتك مباشرة دون الحاجة إلى ترحيلها.

### إضافة (Plugin)
يعتمد NocoBase على بنية قائمة على الإضافات. كل وظيفة في النظام هي عبارة عن **إضافة** مستقلة. يمكنك تفعيل الإضافات التي تحتاجها فقط، أو تطوير إضافات مخصصة لتوسيع وظائف المنصة بما يتناسب مع احتياجاتك الفريدة.

## لماذا تختار NocoBase؟

1.  **مفتوح المصدر**: تحكم كامل في بياناتك وكود المصدر الخاص بك.
2.  **قابلية توسع قوية**: بفضل نظام الإضافات، لا توجد حدود لما يمكنك بناؤه.
3.  **مرونة عالية**: تصميم واجهات المستخدم وهياكل البيانات بما يتناسب تماماً مع سير عملك.
4.  **سهولة الاستخدام**: واجهة رسومية بديهية تتيح لك بناء الأنظمة دون كتابة سطر برمج واحد.
