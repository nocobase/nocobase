:::tip إشعار الترجمة بالذكاء الاصطناعي
تمت ترجمة هذه الوثائق تلقائيًا بواسطة الذكاء الاصطناعي.
:::

## المعلمات

| اسم المعلمة             | النوع          | القيمة الافتراضية | الوصف                                                                     |
| :--------------------- | :-------------- | :---------------- | :------------------------------------------------------------------------ |
| `options.values`       | `M`             | `{}`              | كائن البيانات المراد إدراجه.                                              |
| `options.whitelist?`   | `string[]`      | -                 | قائمة بيضاء للحقول ضمن `values`. سيتم تخزين الحقول الموجودة في هذه القائمة فقط. |
| `options.blacklist?`   | `string[]`      | -                 | قائمة سوداء للحقول ضمن `values`. لن يتم تخزين الحقول الموجودة في هذه القائمة.  |
| `options.transaction?` | `Transaction`   | -                 | المعاملة.                                                                |
