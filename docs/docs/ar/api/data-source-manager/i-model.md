# IModel

تُعرّف واجهة `IModel` الخصائص والأساليب الأساسية لكائن النموذج.

```typescript
export interface IModel {
  toJSON: () => any;
}
```

## واجهة برمجة التطبيقات

### toJSON()

تحوّل كائن النموذج إلى صيغة JSON.
