# Auth

## نظرة عامة

تُستخدم فئة `Auth` بشكل أساسي على جانب العميل للوصول إلى معلومات المستخدم وطلب واجهات برمجة التطبيقات (APIs) المتعلقة بمصادقة المستخدم.

## خصائص الكائن

### `locale`

اللغة التي يستخدمها المستخدم الحالي.

### `role`

الدور الذي يستخدمه المستخدم الحالي.

### `token`

رمز الـ API.

### `authenticator`

المُصادِق المستخدم لمصادقة المستخدم الحالي. راجع [مصادقة المستخدم](/auth-verification/auth/).

## دوال الفئة

### `signIn()`

تسجيل دخول المستخدم.

#### التوقيع

- `async signIn(values: any, authenticator?: string): Promise<AxiosResponse<any>>`

#### التفاصيل

| اسم المعامل      | النوع     | الوصف                                          |
| --------------- | -------- | ---------------------------------------------- |
| `values`        | `any`    | معاملات الطلب لواجهة برمجة تطبيقات تسجيل الدخول. |
| `authenticator` | `string` | معرّف المُصادِق المستخدم لتسجيل الدخول.         |

### `signUp()`

تسجيل المستخدم.

#### التوقيع

- `async signUp(values: any, authenticator?: string): Promise<AxiosResponse<any>>`

#### التفاصيل

| اسم المعامل      | النوع     | الوصف                                          |
| --------------- | -------- | ---------------------------------------------- |
| `values`        | `any`    | معاملات الطلب لواجهة برمجة تطبيقات التسجيل.     |
| `authenticator` | `string` | معرّف المُصادِق المستخدم للتسجيل.               |

### `signOut()`

تسجيل الخروج.

#### التوقيع

- `async signOut(values: any, authenticator?: string): Promise<AxiosResponse<any>>`

#### التفاصيل

| اسم المعامل      | النوع     | الوصف                                          |
| --------------- | -------- | ---------------------------------------------- |
| `values`        | `any`    | معاملات الطلب لواجهة برمجة تطبيقات تسجيل الخروج. |
| `authenticator` | `string` | معرّف المُصادِق المستخدم لتسجيل الخروج.         |
