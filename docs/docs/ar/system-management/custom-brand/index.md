# تخصيص العلامة التجارية

## مقدمة

اسم العلامة التجارية الافتراضي للنظام هو "NocoBase"، ويظهر بشكل أساسي في ثلاثة مواضع:

![20240409111100](https://static-docs.nocobase.com/20240409111100.png)

في أسفل صفحة تسجيل الدخول:

![20240409111254](https://static-docs.nocobase.com/20240409111254.png)

يمكن تهيئة الموضع 1 عبر خيار الشعار (Logo) في [إعدادات النظام](/system-management/system-settings/). أما الموضعان 2 و 3 فيمكن تهيئتهما باستخدام هذه الإضافة.

## تعليمات الاستخدام

بعد تفعيل الإضافة، انقر على قائمة "تخصيص العلامة التجارية" في الزاوية العلوية اليمنى من الصفحة للوصول إلى صفحة التهيئة.

![20240409113640](https://static-docs.nocobase.com/20240409113640.png)

- **العلامة التجارية**: تُستخدم لتعيين النص في أسفل صفحة تسجيل الدخول (الموضع 3)
- **حول**: تُستخدم لتهيئة محتوى القائمة في الزاوية العلوية اليمنى (كما هو موضح في الصورة أدناه)

![20240409113915](https://static-docs.nocobase.com/20240409113915.png)

تدعم التهيئة استخدام تنسيق HTML لتخصيص المحتوى بمرونة. إذا كنت تحتاج فقط إلى إجراء تغييرات نصية بسيطة، فيمكنك تعديل المحتوى الحالي مباشرةً كما هو موضح أدناه:

![20240409114740](https://static-docs.nocobase.com/20240409114740.png)

تعيين أيقونة الموقع (favicon)
