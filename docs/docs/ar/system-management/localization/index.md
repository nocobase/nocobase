# إدارة التوطين

## مقدمة

تُستخدم إضافة إدارة التوطين لإدارة وتطبيق موارد التوطين في NocoBase. يمكنها ترجمة قوائم النظام، والمجموعات، والحقول، وجميع الإضافات، وذلك للتكيف مع لغة وثقافة مناطق معينة.

## التثبيت

هذه الإضافة مدمجة ولا تتطلب تثبيتًا إضافيًا.

### مزامنة عناصر الترجمة

<img src="https://static-docs.nocobase.com/202404202134850.png"/>

حاليًا، يمكن مزامنة المحتوى التالي:

- حزم اللغات المحلية للنظام والإضافات
- عناوين المجموعات، وعناوين الحقول، وتسميات خيارات الحقول
- عناوين القوائم

بعد اكتمال المزامنة، سيعرض النظام جميع عناصر الترجمة المتاحة للغة الحالية.

<img src="https://static-docs.nocobase.com/202404202136567.png"/>

:::info{title=ملاحظة}
قد تحتوي الوحدات المختلفة على نفس عناصر النص الأصلي، ويجب ترجمتها بشكل منفصل.
:::
