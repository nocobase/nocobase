# إعدادات اللغة

في "إعدادات النظام"، يمكنك تعديل بيئة اللغة العامة للنظام عن طريق تهيئة "اللغات المُمكّنة". ستكون اللغة الأولى في القائمة هي اللغة الافتراضية.

![20240324192005](https://static-docs.nocobase.com/20240324192005.png)

عند تمكين لغات متعددة، يمكن للمستخدمين تعيين تفضيلاتهم اللغوية الشخصية في المركز الشخصي (الزاوية العلوية اليمنى من الصفحة).

![20240324192509](https://static-docs.nocobase.com/20240324192509.png)

إذا كانت بيئة اللغة التي تستخدمها تفتقر إلى ترجمات، يمكنك استخدام إضافة "[إدارة التوطين](/system-management/localization/)" لإضافة الترجمات أو تعديلها.

![20240324193055](https://static-docs.nocobase.com/20240324193055.png)
