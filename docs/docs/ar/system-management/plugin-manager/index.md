# إدارة الإضافات

## مدير الإضافات

للوصول إلى صفحة مدير الإضافات، انقر على أيقونة مدير الإضافات في الزاوية العلوية اليمنى من الصفحة. في هذه الصفحة، يمكنك عرض الإضافات وتثبيتها وتفعيلها أو تعطيلها.

![2024-04-10_08-47-45](https://static-docs.nocobase.com/2024-04-10_08-47-45.png)

## مركز إعدادات الإضافات

عادةً ما تأتي الإضافات بصفحات إعدادات خاصة بها. لعرض قوائم إعدادات جميع الإضافات، انقر على أيقونة الإعدادات في الزاوية العلوية اليمنى. يمكنك بعد ذلك النقر على العنصر المناسب في القائمة لتكوين الإضافة.
