# الأدوار

## مركز الإدارة

### إدارة الأدوار

![](https://static-docs.nocobase.com/da7083c67d794e23dc6eb0f85b1de86c.png)

يأتي التطبيق مع دورين محددين مسبقًا: "المسؤول" و"العضو"، لكل منهما إعدادات أذونات افتراضية مختلفة تتناسب مع وظائفهما.

### إضافة الأدوار وحذفها وتعديلها

معرّف الدور هو معرّف نظام فريد يسمح بتخصيص الأدوار الافتراضية، لكن لا يمكن حذف الأدوار المحددة مسبقًا من النظام.

![](https://static-docs.nocobase.com/35f323b346db4f9f12f9bee4dea63302.png)

### تعيين الدور الافتراضي

الدور الافتراضي هو الدور الذي يُسند تلقائيًا للمستخدمين الجدد إذا لم يُحدد دور معين أثناء إنشائهم.

![](https://static-docs.nocobase.com/f41bba7ff55ca28715c486dc45bc1708.png)

## المركز الشخصي

### تبديل الأدوار

يمكن تعيين أدوار متعددة للمستخدمين والتبديل بينها في المركز الشخصي.

![](https://static-docs.nocobase.com/e331d11ec1ca3b8b7e0472105b167819.png)

الدور الافتراضي عند تسجيل الدخول يتحدد بناءً على آخر دور تم التبديل إليه (تتحدث هذه القيمة مع كل تبديل) أو، إذا لم ينطبق ذلك، الدور الأول (الدور الافتراضي للنظام).
