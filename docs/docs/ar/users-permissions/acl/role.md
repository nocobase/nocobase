# الأدوار

## مركز الإدارة

### إدارة الأدوار

![](https://static-docs.nocobase.com/da7083c67d794e23dc6eb0f85b1de86c.png)

يحتوي التطبيق على دورين محددين مسبقًا: "Admin" و "Member"، ولكل منهما إعدادات صلاحيات افتراضية مختلفة تتناسب مع وظائفهما.

### إضافة وحذف وتعديل الأدوار

معرّف الدور (Role Identifier) هو معرّف فريد داخل النظام، ويتيح تخصيص الأدوار الافتراضية، ولكن لا يمكن حذف الأدوار المعرّفة مسبقًا في النظام.

![](https://static-docs.nocobase.com/35f323b346db4f9f12f9bee4dea63302.png)

### تعيين الدور الافتراضي

الدور الافتراضي هو الدور الذي يتم تعيينه تلقائيًا للمستخدمين الجدد إذا لم يتم تحديد دور معين أثناء إنشائهم.

![](https://static-docs.nocobase.com/f41bba7ff55ca28715c486dc45bc1708.png)

## المركز الشخصي

### تبديل الأدوار

يمكن تعيين عدة أدوار للمستخدم، ويمكنه التبديل بينها من خلال المركز الشخصي.

![](https://static-docs.nocobase.com/e331d11ec1ca3b8b7e0472105b167819.png)

يتم تحديد الدور الافتراضي عند تسجيل الدخول بناءً على آخر دور تم التبديل إليه (حيث يتم تحديث هذه القيمة مع كل عملية تبديل)، أو في حال عدم توفر ذلك، يتم اختيار أول دور (الدور الافتراضي في النظام).
