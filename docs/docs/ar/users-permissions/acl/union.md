# اتحاد الأدوار

اتحاد الأدوار هو نمط لإدارة الصلاحيات. يمكن لمطوري النظام اختيار استخدام `أدوار مستقلة` أو `السماح باتحاد الأدوار` أو `اتحاد الأدوار فقط`.

![20250312184651](https://static-docs.nocobase.com/20250312184651.png)

## الأدوار المستقلة

بشكل افتراضي، يستخدم النظام الأدوار المستقلة. يجب على المستخدمين التبديل بين الأدوار التي يمتلكونها بشكل فردي.

![20250312184729](https://static-docs.nocobase.com/20250312184729.png)

![20250312184826](https://static-docs.nocobase.com/20250312184826.png)

## السماح باتحاد الأدوار

يمكن لمطوري النظام تفعيل `السماح باتحاد الأدوار`، مما يتيح للمستخدمين امتلاك صلاحيات جميع الأدوار المخصصة لهم في آنٍ واحد مع السماح لهم بالتبديل بين الأدوار بشكل فردي.

![20250312185006](https://static-docs.nocobase.com/20250312185006.png)

## اتحاد الأدوار فقط

يُجبر المستخدمون على استخدام اتحاد الأدوار فقط ولا يمكنهم التبديل بين الأدوار بشكل فردي.

![20250312185105](https://static-docs.nocobase.com/20250312185105.png)

## قواعد اتحاد الأدوار

يمنح اتحاد الأدوار الحد الأقصى من الصلاحيات عبر جميع الأدوار.

### دمج صلاحيات العمليات

مثال: الدور1 مهيأ لـ `السماح بتهيئة الواجهة` والدور2 مهيأ لـ `السماح بتثبيت الإضافات وتفعيلها وتعطيلها`

![20250312190133](https://static-docs.nocobase.com/20250312190133.png)

![20250312190352](https://static-docs.nocobase.com/20250312190352.png)

عند تسجيل الدخول بدور **الصلاحيات الكاملة**، سيمتلك المستخدم كلا الصلاحيتين في آنٍ واحد.

![20250312190621](https://static-docs.nocobase.com/20250312190621.png)

### دمج نطاق البيانات

#### صفوف البيانات

السيناريو 1: أدوار متعددة تضع شروطًا على نفس الحقل

فلتر الدور أ: العمر < 30

| معرف المستخدم | الاسم | العمر |
| ------------- | ----- | ----- |
| 1             | جاك   | 23    |
| 2             | ليلى  | 29    |

فلتر الدور ب: العمر > 25

| معرف المستخدم | الاسم | العمر |
| ------------- | ----- | ----- |
| 2             | ليلى  | 29    |
| 3             | سام   | 32    |

**بعد الدمج:**

| معرف المستخدم | الاسم | العمر |
| ------------- | ----- | ----- |
| 1             | جاك   | 23    |
| 2             | ليلى  | 29    |
| 3             | سام   | 32    |

#### أعمدة البيانات

أعمدة الدور أ المرئية: الاسم، العمر

أعمدة الدور ب المرئية: الاسم، الجنس

**بعد الدمج:** الاسم، العمر، الجنس

#### ملخص

قواعد دمج نطاق البيانات في اتحاد الأدوار:

1. بين الصفوف، إذا تحقق أي شرط، فإن الصف يمتلك الصلاحيات.
2. بين الأعمدة، يتم دمج الحقول.
3. عند تهيئة الصفوف والأعمدة معًا، يتم دمجهما بشكل منفصل.
