# المستخدمون

## إدارة المستخدمين

يوفّر هذا الإضافة واجهة لإدارة المستخدمين داخل مركز الإعدادات. بعد تهيئة التطبيق، يتم إنشاء حساب مدير عام (Super Admin) تلقائيًا ولا يمكن حذفه. يمتلك هذا المدير صلاحيات كاملة (Root) ويمكنه الوصول إلى جميع الموارد.

![](https://static-docs.nocobase.com/44bf40f56b45d4dd96c424fb08082cf6.png)

بالإضافة إلى ذلك، يمكنك أيضًا إضافة كتل مختلفة من مجموعة المستخدمين (Users) لإدارة المستخدمين، مثل كتل الجداول:

![](https://static-docs.nocobase.com/76b5a4652f869541a9e8f18a4568a7c9.png)
