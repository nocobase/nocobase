# المستخدمون

## إدارة المستخدمين

يوفر هذا الإضافة واجهة لإدارة المستخدمين في مركز الإعدادات. عند تهيئة التطبيق، يتم إنشاء حساب مسؤول أعلى غير قابل للحذف تلقائيًا. يمتلك المسؤول الأعلى صلاحيات كاملة ويمكنه الوصول إلى جميع الموارد.

![](https://static-docs.nocobase.com/44bf40f56b45d4dd96c424fb08082cf6.png)

بالإضافة إلى ذلك، يمكنك إضافة أنواع مختلفة من الكتل لمجموعة المستخدمين لإدارتهم، مثل كتل الجداول:

![](https://static-docs.nocobase.com/76b5a4652f869541a9e8f18a4568a7c9.png)
