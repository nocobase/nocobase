# المستخدمون

## إدارة المستخدمين

توفر هذه الإضافة واجهة لإدارة المستخدمين في مركز الإعدادات. بعد تهيئة التطبيق، يتم إنشاء حساب مشرف فائق لا يمكن حذفه تلقائيًا. يتمتع المشرف الفائق بصلاحيات الجذر (Root) ويمكنه الوصول إلى جميع الموارد.

![](https://static-docs.nocobase.com/44bf40f56b45d4dd96c424fb08082cf6.png)

بالإضافة إلى ذلك، يمكنك أيضًا إضافة كتل مختلفة من مجموعة المستخدمين لإدارة المستخدمين في الصفحات، مثل كتل الجدول:

![](https://static-docs.nocobase.com/76b5a4652f869541a9e8f18a4568a7c9.png)
