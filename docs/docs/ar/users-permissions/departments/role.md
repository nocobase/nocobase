# إدارة أدوار الأقسام

من خلال ربط دور واحد أو أكثر بقسم، يمكن لأعضاء هذا القسم الحصول على هذه الأدوار.

هناك طريقتان لربط الأدوار بقسم ما:

الطريقة الأولى هي تحديد الدور في حقل الدور ضمن نموذج تحرير القسم.

![](https://static-docs.nocobase.com/70f77bb89aa1fb415c152a51a51cc23b.png)

الطريقة الثانية هي إضافة أقسام إلى الدور المعني في إدارة الأدوار.

![](https://static-docs.nocobase.com/f2a7bec937cf2f179ce868a92b98416d.png)

لا يمكن تحديد الأقسام التي لديها هذا الدور بالفعل.

![](https://static-docs.nocobase.com/be10299893581e1f97a4e01ddd5c7e59.png)
