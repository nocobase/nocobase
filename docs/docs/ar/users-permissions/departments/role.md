# إدارة أدوار القسم

من خلال ربط دور واحد أو أكثر بقسم، يمكن لأعضاء القسم الحالي امتلاك هذه الأدوار.

هناك طريقتان لربط الأدوار بقسم.

الأولى هي تحديد دور في حقل الدور في نموذج تعديل القسم.

![](https://static-docs.nocobase.com/70f77bb89aa1fb415c152a51a51cc23b.png)

الثانية هي إضافة أقسام إلى الدور المقابل في إدارة الأدوار.

![](https://static-docs.nocobase.com/f2a7bec937cf2f179ce868a92b98416d.png)
