# قفل المستخدمين

## مقدمة

تتيح لك هذه الميزة إدارة المستخدمين المقفلين وقفل المستخدمين بشكل استباقي.

![](https://static-docs.nocobase.com/202412281450083.png)

### إلغاء قفل المستخدم

يمكن إلغاء قفل المستخدمين الذين تم قفلهم بسبب تجاوز حد محاولات تسجيل الدخول بكلمة مرور غير صالحة عن طريق حذف السجل الخاص بهم.

### قفل المستخدم

تتيح لك إضافة سجل مستخدم قفل المستخدم بشكل استباقي. بمجرد قفل المستخدم، سيُمنع من الوصول إلى النظام بأي طريقة مصادقة، بما في ذلك مفاتيح API.

![](https://static-docs.nocobase.com/202412281450512.png)
