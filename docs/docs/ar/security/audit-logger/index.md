# سجل التدقيق

## مقدمة

يُستخدم سجل التدقيق لتسجيل وتتبع أنشطة المستخدمين وسجل عمليات الموارد داخل النظام.

![](https://static-docs.nocobase.com/202501031627719.png)

![](https://static-docs.nocobase.com/202501031627922.png)

## وصف المعلمات

| المعلمة                  | الوصف                                                                                                                               |
| :--------------------- | :---------------------------------------------------------------------------------------------------------------------------------------- |
| **Resource**          | نوع المورد المستهدف للعملية                                                                                                 |
| **Action**            | نوع العملية التي تم تنفيذها                                                                                                           |
| **User**              | المستخدم الذي أجرى العملية                                                                                                         |
| **Role**              | دور المستخدم أثناء العملية                                                                                                 |
| **Data source**       | مصدر البيانات                                                                                                                           |
| **Target collection** | المجموعة المستهدفة                                                                                                                     |
| **Target record UK**  | المعرف الفريد للسجل المستهدف في المجموعة                                                                                            |
| **Source collection** | المجموعة المصدر لحقل العلاقة                                                                                            |
| **Source record UK**  | المعرف الفريد للسجل المصدر في المجموعة                                                                                            |
| **Status**            | رمز حالة HTTP لاستجابة طلب العملية                                                                                    |
| **Created at**        | وقت العملية                                                                                                                           |
| **UUID**              | المعرف الفريد للعملية، يتطابق مع معرف الطلب (Request ID) لطلب العملية، ويمكن استخدامه لاسترداد سجلات التطبيق. |
| **IP**                | عنوان IP للمستخدم                                                                                                                |
| **UA**                | معلومات وكيل المستخدم (UA)                                                                                                            |
| **Metadata**          | البيانات الوصفية مثل معلمات طلب العملية، وجسم الطلب، ومحتوى الاستجابة.                                                  |

## وصف موارد التدقيق

حاليًا، يتم تسجيل عمليات الموارد التالية في سجل التدقيق:

### التطبيق الرئيسي

| العملية             | الوصف                 |
| :---------------- | :-------------------- |
| `app:resart`     | إعادة تشغيل التطبيق     |
| `app:clearCache` | مسح ذاكرة التخزين المؤقت للتطبيق |

### مدير الإضافات

| العملية         | الوصف     |
| :------------ | :-------- |
| `pm:add`     | إضافة إضافة |
| `pm:update`  | تحديث إضافة |
| `pm:enable`  | تفعيل إضافة |
| `pm:disable` | تعطيل إضافة |
| `pm:remove`  | إزالة إضافة |

### مصادقة المستخدم

| العملية                  | الوصف         |
| :--------------------- | :------------ |
| `auth:signIn`         | تسجيل الدخول     |
| `auth:signUp`         | التسجيل     |
| `auth:signOut`        | تسجيل الخروج     |
| `auth:changePassword` | تغيير كلمة المرور |

### المستخدم

| العملية                  | الوصف         |
| :--------------------- | :------------ |
| `users:updateProfile` | تحديث الملف الشخصي |

### إعدادات واجهة المستخدم

| العملية                       | الوصف                       |
| :-------------------------- | :-------------------------- |
| `uiSchemas:insertAdjacent` | إدراج مخطط واجهة المستخدم (UI Schema) |
| `uiSchemas:patch`          | تعديل مخطط واجهة المستخدم (UI Schema) |
| `uiSchemas:remove`         | إزالة مخطط واجهة المستخدم (UI Schema) |

### عمليات المجموعة

| العملية             | الوصف                     |
| :---------------- | :------------------------ |
| `create`         | إنشاء سجل                 |
| `update`         | تحديث سجل                 |
| `destroy`        | حذف سجل                   |
| `updateOrCreate` | تحديث أو إنشاء سجل         |
| `firstOrCreate`  | استعلام أو إنشاء سجل       |
| `move`           | نقل سجل                   |
| `set`            | تعيين سجل حقل العلاقة     |
| `add`            | إضافة سجل حقل العلاقة     |
| `remove`         | إزالة سجل حقل العلاقة     |
| `export`         | تصدير سجل                 |
| `import`         | استيراد سجل               |

## إضافة موارد تدقيق أخرى

إذا قمت بتوسيع عمليات موارد أخرى عبر الإضافات وترغب في تسجيل سلوكيات عمليات الموارد هذه في سجل التدقيق، فيمكنك الرجوع إلى [واجهة برمجة التطبيقات (API)](/api/server/audit-manager.md).
