# مخرجات سير العمل

## مقدمة

تُستخدم عقدة "مخرجات سير العمل" لتحديد قيمة المخرجات لسير العمل الذي يتم استدعاؤه. عندما يتم استدعاء سير عمل بواسطة سير عمل آخر، يمكن استخدام عقدة "مخرجات سير العمل" لتمرير قيمة مرة أخرى إلى المستدعي.

## إنشاء العقدة

في سير العمل الذي يتم استدعاؤه، أضف عقدة "مخرجات سير العمل":

![20241231002033](https://static-docs.nocobase.com/20241231002033.png)

## تهيئة العقدة

### قيمة المخرجات

أدخل أو حدد متغيرًا كقيمة للمخرجات. يمكن أن تكون قيمة المخرجات من أي نوع، مثل قيمة ثابتة (نص، رقم، قيمة منطقية، تاريخ، أو JSON مخصص)، أو متغير آخر من سير العمل.

![20241231003059](https://static-docs.nocobase.com/20241231003059.png)

:::
