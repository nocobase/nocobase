# إنشاء سجل

يُستخدم لإضافة سجل جديد إلى مجموعة.

يمكن استخدام متغيرات من سياق سير العمل لقيم حقول السجل الجديد. لتعيين قيم لحقول العلاقة، يمكنك الإشارة مباشرة إلى متغيرات البيانات المقابلة في السياق، والتي يمكن أن تكون كائنًا أو قيمة مفتاح خارجي. إذا لم تستخدم المتغيرات، فستحتاج إلى إدخال قيم المفتاح الخارجي يدويًا. بالنسبة لقيم المفاتيح الخارجية المتعددة في علاقة متعددة الأطراف، يجب فصلها بفواصل إنجليزية.

## إنشاء عقدة

في واجهة إعدادات سير العمل، انقر على زر الزائد ("+") في سير العمل لإضافة عقدة "إنشاء سجل":

![إنشاء عقدة "إنشاء سجل"](https://static-docs.nocobase.com/386c8c01c89b1eeab848510e77f4841a.png)

## إعدادات العقدة

![عقدة إنشاء سجل_مثال_إعدادات العقدة](https://static-docs.nocobase.com/5f7b97a51b64a1741cf82a4d4455b610.png)

### المجموعة

اختر المجموعة التي تريد إضافة سجل جديد إليها.

### قيم الحقول

عيّن قيمًا لحقول المجموعة. يمكنك استخدام متغيرات من سياق سير العمل أو إدخال قيم ثابتة يدويًا.

:::info{title="ملاحظة"}
البيانات التي تُنشئها عقدة "إنشاء سجل" في سير العمل لا تتعامل تلقائيًا مع بيانات المستخدم مثل "المنشئ بواسطة" و "آخر تعديل بواسطة". تحتاج إلى إعداد قيم هذه الحقول بنفسك حسب الحاجة.
:::

### تحميل بيانات العلاقة مسبقًا

إذا كانت حقول السجل الجديد تتضمن حقول علاقة، وترغب في استخدام بيانات العلاقة المقابلة في خطوات سير العمل اللاحقة، فيمكنك تحديد حقول العلاقة المعنية في إعدادات التحميل المسبق. بهذه الطريقة، بعد إنشاء السجل الجديد، سيتم تحميل بيانات العلاقة المقابلة تلقائيًا وتخزينها معًا في بيانات نتائج العقدة.

## مثال

على سبيل المثال، عندما يتم إنشاء أو تحديث سجل في مجموعة "المقالات"، يلزم إنشاء سجل "إصدارات المقالات" تلقائيًا لتسجيل سجل التغييرات للمقالة. يمكنك استخدام عقدة "إنشاء سجل" لتحقيق ذلك:

![عقدة إنشاء سجل_مثال_إعدادات سير العمل](https://static-docs.nocobase.com/dfd4820d49c145fa331883fc09c9161f.png)

![عقدة إنشاء سجل_مثال_إعدادات العقدة](https://static-docs.nocobase.com/1a0992e66170be12a068da6503298868.png)

بعد تمكين سير العمل بهذا الإعداد، عند تغيير سجل في مجموعة "المقالات"، سيتم إنشاء سجل "إصدارات المقالات" تلقائيًا لتسجيل سجل التغييرات للمقالة.
