# نظرة عامة

لا يمكن للميزات المضمنة في سير العمل تغطية جميع السيناريوهات. على سبيل المثال، لا يمكن لأنواع العقد المضمنة تعداد كل عملية ممكنة في جميع سيناريوهات الأعمال. لذلك، قدمنا أيضًا تصميمًا لتوسيع سير العمل، بما في ذلك توسيع أنواع المشغلات والعقد. في سيناريوهات الأعمال التي لا تفي فيها الميزات المضمنة بالمتطلبات، يمكنك حلها من خلال التوسيع باستخدام طرق البرمجة المنخفضة.
