# البدء السريع

## مقدمة

قبل استخدام موظف الذكاء الاصطناعي، يجب عليك أولاً ربط خدمة LLM عبر الإنترنت. يدعم NocoBase حاليًا خدمات LLM الرئيسية عبر الإنترنت مثل OpenAI و Gemini و Claude و DepSeek و Qwen وغيرها.
بالإضافة إلى خدمات LLM عبر الإنترنت، يدعم NocoBase أيضًا ربط النماذج المحلية لـ Ollama.

## إعداد خدمة LLM

انتقل إلى صفحة إعداد إضافة موظف الذكاء الاصطناعي، ثم انقر على علامة التبويب `LLM service` للدخول إلى صفحة إدارة خدمة LLM.

![20251021213122](https://static-docs.nocobase.com/20251021213122.png)

مرر مؤشر الماوس فوق زر `Add New` في الزاوية العلوية اليمنى من قائمة خدمات LLM، ثم اختر خدمة LLM التي ترغب في استخدامها.

![20251021213358](https://static-docs.nocobase.com/20251021213358.png)

على سبيل المثال، باستخدام OpenAI، أدخل `title` يسهل تذكره في النافذة المنبثقة، ثم أدخل `API key` الذي حصلت عليه من OpenAI، وانقر على `Submit` للحفظ. بهذا تكون قد أكملت إعداد خدمة LLM.

يمكن عادةً ترك حقل `Base URL` فارغًا. إذا كنت تستخدم خدمة LLM خارجية متوافقة مع واجهة برمجة تطبيقات OpenAI، فيرجى ملء حقل Base URL المناسب.

![20251021214549](https://static-docs.nocobase.com/20251021214549.png)

## اختبار التوفر

في صفحة إعداد خدمة LLM، انقر على زر `Test flight`، ثم أدخل اسم النموذج الذي ترغب في استخدامه، وانقر على زر `Run` لاختبار ما إذا كانت خدمة LLM والنموذج متاحين.
