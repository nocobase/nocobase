# المحادثة متعددة الوسائط

## الصور

إذا كان النموذج يدعم ذلك، يمكن لعقدة LLM إرسال الصور إلى النموذج. عند الاستخدام، تحتاج إلى تحديد حقل مرفق أو سجل مجموعة ملفات مرتبط عبر متغير. عند تحديد سجل مجموعة ملفات، يمكنك تحديده على مستوى الكائن أو تحديد حقل URL.

![](https://static-docs.nocobase.com/202503041034858.png)

هناك خياران لتنسيق إرسال الصور:

- الإرسال عبر URL - سيتم إرسال جميع الصور، باستثناء تلك المخزنة محليًا، كروابط URL. سيتم تحويل الصور المخزنة محليًا إلى تنسيق base64 قبل الإرسال.
- الإرسال عبر base64 - سيتم إرسال جميع الصور، سواء كانت مخزنة محليًا أو في السحابة، بتنسيق base64. هذا مناسب للحالات التي لا يمكن فيها لخدمة LLM عبر الإنترنت الوصول مباشرة إلى رابط URL الخاص بالصورة.

![](https://static-docs.nocobase.com/202503041200638.png)
