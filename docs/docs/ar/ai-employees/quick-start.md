# البدء السريع

لنقم بإكمال الحد الأدنى من الإعداد القابل للاستخدام لموظفي الذكاء الاصطناعي (AI Employees) في غضون 5 دقائق.

## تثبيت الإضافة

موظفو الذكاء الاصطناعي مدمجون كإضافة في NocoBase (`@nocobase/plugin-ai`)، لذا لا يلزم تثبيت منفصل.

## تهيئة النماذج

يمكنك تهيئة خدمات نماذج اللغة الكبيرة (LLM) من خلال أي من المداخل التالية:

1. مدخل الإدارة: `إعدادات النظام -> موظفو الذكاء الاصطناعي -> خدمة LLM`.
2. اختصار الواجهة الأمامية: في لوحة دردشة الذكاء الاصطناعي، استخدم `Model Switcher` لاختيار نموذج، ثم انقر فوق الاختصار "إضافة خدمة LLM" للانتقال المباشر.

![quick-start-model-switcher-add-llm-service.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/quick-start-model-switcher-add-llm-service.png)

عادةً ما تحتاج إلى تأكيد ما يلي:
1. اختيار المزود (Provider).
2. إدخال مفتاح واجهة برمجة التطبيقات (API Key).
3. تهيئة `Enabled Models`؛ يمكنك ببساطة استخدام الخيار الموصى به (Recommend) افتراضيًا.

## تمكين الموظفين المدمجين

يتم تمكين موظفي الذكاء الاصطناعي المدمجين افتراضيًا، ولا يلزم عادةً تفعيلهم واحدًا تلو الآخر يدويًا.

إذا كنت بحاجة إلى ضبط نطاق التوفر (تمكين/تعطيل موظف معين)، يمكنك تعديل مفتاح `Enabled` في صفحة قائمة `إعدادات النظام -> موظفو الذكاء الاصطناعي`.

![ai-employee-list-enable-switch.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employee-list-enable-switch.png)

## بدء التعاون

في صفحة التطبيق، قم بتمرير الماوس فوق مدخل الاختصار في الزاوية اليمنى السفلية واختر موظف ذكاء اصطناعي.
![ai-employees-entry-bottom-right.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employees-entry-bottom-right.png)

انقر لفتح مربع حوار دردشة الذكاء الاصطناعي:

![chat-footer-employee-switcher-and-model-switcher.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/chat-footer-employee-switcher-and-model-switcher.png)

يمكنك أيضًا:  
* إضافة كتل (Blocks)
* إضافة مرفقات
* تمكين البحث عبر الويب
* تبديل موظفي الذكاء الاصطناعي
* اختيار النماذج

يمكنهم أيضًا الحصول تلقائيًا على هيكل الصفحة كـ "سياق" (Context). على سبيل المثال، يمكن للموظف Dex في كتلة النموذج قراءة هياكل حقول النموذج واستدعاء المهارات المناسبة لإجراء عمليات على الصفحة.

## المهام المختصرة 

يمكنك تعيين مهام شائعة مسبقًا لكل موظف ذكاء اصطناعي في الموقع الحالي، بحيث يمكنك بدء العمل بنقرة واحدة، مما يجعل الأمر سريعًا ومريحًا.

## نظرة عامة على الموظفين المدمجين

يوفر NocoBase العديد من موظفي الذكاء الاصطناعي المدمجين والمخصصين لسيناريوهات محددة.

كل ما تحتاجه هو:

1. تهيئة خدمات LLM.
2. ضبط حالة تمكين الموظفين عند الحاجة (ممكنون افتراضيًا).
3. اختيار النموذج في المحادثة وبدء التعاون.

| اسم الموظف | الدور الوظيفي | القدرات الجوهرية |
| :--- | :--- | :--- |
| **Cole** | مساعد NocoBase | أسئلة وأجوبة حول استخدام المنتج، استرجاع المستندات |
| **Ellis** | خبير البريد الإلكتروني | كتابة رسائل البريد الإلكتروني، توليد الملخصات، اقتراحات الردود |
| **Dex** | خبير تنظيم البيانات | ترجمة الحقول، التنسيق، استخراج المعلومات |
| **Viz** | محلل الرؤى | رؤى البيانات، تحليل الاتجاهات، تفسير المؤشرات الرئيسية |
| **Lexi** | مساعد الترجمة | الترجمة متعددة اللغات، المساعدة في التواصل |
| **Vera** | محلل أبحاث | البحث عبر الويب، تجميع المعلومات، الأبحاث المتعمقة |
| **Dara** | خبير تصور البيانات | تهيئة الرسوم البيانية، توليد التقارير المرئية |
| **Orin** | خبير نمذجة البيانات | المساعدة في تصميم هياكل المجموعات (Collections)، اقتراحات الحقول |
| **Nathan** | مهندس واجهات أمامية | المساعدة في كتابة مقتطفات برمجية للواجهة الأمامية، تعديلات الأنماط |

**ملاحظات**

بعض موظفي الذكاء الاصطناعي المدمجين لا يظهرون في القائمة الموجودة في الزاوية اليمنى السفلية لأن لديهم سيناريوهات عمل مخصصة:

- Orin: صفحات نمذجة البيانات.
- Dara: كتل تهيئة الرسوم البيانية.
- Nathan: كتلة JS (JS Block) ومحررات الأكواد المماثلة.
