# إدارة الملفات

## مقدمة

قم بتكوين طريقة تخزين الملفات التي يتم تحميلها أثناء محادثات موظف الذكاء الاصطناعي.

## الإعدادات

انتقل إلى صفحة تكوين إضافة موظف الذكاء الاصطناعي، ثم انقر على علامة التبويب `Settings` للدخول إلى صفحة إعدادات طريقة التخزين.

![20251022002648](https://static-docs.nocobase.com/20251022002648.png)

يتم تكوين خيارات التخزين المتاحة لـ `Files storage` في إضافة إدارة الملفات.
