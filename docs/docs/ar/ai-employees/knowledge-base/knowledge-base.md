# قاعدة المعرفة

## مقدمة

قاعدة المعرفة هي الأساس لاسترجاع RAG. تقوم بتنظيم المستندات وتصنيفها وبناء فهرس لها. عندما يجيب موظف الذكاء الاصطناعي على سؤال، فإنه يعطي الأولوية للبحث عن الإجابات في قاعدة المعرفة.

## إدارة قاعدة المعرفة

انتقل إلى صفحة إعدادات إضافة موظف الذكاء الاصطناعي، ثم انقر على علامة التبويب `Knowledge base` للدخول إلى صفحة إدارة قاعدة المعرفة.

![20251023095649](https://static-docs.nocobase.com/20251023095649.png)

انقر على زر `Add new` في الزاوية العلوية اليمنى لإضافة قاعدة معرفة `Local` جديدة.

![20251023095826](https://static-docs.nocobase.com/20251023095826.png)

أدخل المعلومات الضرورية لقاعدة المعرفة الجديدة:

- في حقل الإدخال `Name`، أدخل اسم قاعدة المعرفة؛
- في `File storage`، اختر موقع تخزين الملفات؛
- في `Vector store`، اختر مخزن المتجهات، راجع [مخزن المتجهات](/ai-employees/knowledge-base/vector-store)؛
- في حقل الإدخال `Description`، أدخل وصف قاعدة المعرفة؛

انقر على زر `Submit` لإنشاء قاعدة المعرفة.

![20251023095909](https://static-docs.nocobase.com/20251023095909.png)

## إدارة مستندات قاعدة المعرفة

بعد إنشاء قاعدة المعرفة، في صفحة قائمة قواعد المعرفة، انقر على قاعدة المعرفة التي أنشأتها للتو للدخول إلى صفحة إدارة مستندات قاعدة المعرفة.

![20251023100458](https://static-docs.nocobase.com/20251023100458.png)

![20251023100527](https://static-docs.nocobase.com/20251023100527.png)

انقر على زر `Upload` لرفع المستندات. بعد رفع المستندات، ستبدأ عملية التحويل إلى متجهات تلقائيًا. انتظر حتى يتغير `Status` من حالة `Pending` إلى حالة `Success`.

تدعم قاعدة المعرفة حاليًا أنواع المستندات التالية: txt, pdf, doc, docx, ppt, pptx؛ يدعم ملف PDF النص العادي فقط.

![20251023100901](https://static-docs.nocobase.com/20251023100901.png)

## أنواع قواعد المعرفة

### قاعدة المعرفة `Local`

قاعدة المعرفة `Local` هي قاعدة معرفة تخزن محليًا في NocoBase. يتم تخزين المستندات وبيانات المتجهات الخاصة بها محليًا بواسطة NocoBase.

![20251023101620](https://static-docs.nocobase.com/20251023101620.png)

### قاعدة المعرفة `Readonly`

قاعدة المعرفة `Readonly` هي قاعدة معرفة للقراءة فقط. يتم الاحتفاظ بالمستندات وبيانات المتجهات خارجيًا. يتم إنشاء اتصال بقاعدة بيانات المتجهات فقط في NocoBase (تدعم حاليًا PGVector فقط).

![20251023101743](https://static-docs.nocobase.com/20251023101743.png)

### قاعدة المعرفة `External`

قاعدة المعرفة `External` هي قاعدة معرفة خارجية حيث يتم الاحتفاظ بالمستندات وبيانات المتجهات خارجيًا. يتطلب استرجاع قاعدة بيانات المتجهات توسيعًا من المطورين، مما يسمح باستخدام قواعد بيانات متجهات لا تدعمها NocoBase حاليًا.

![20251023101949](https://static-docs.nocobase.com/20251023101949.png)
