# قاعدة بيانات المتجهات

## مقدمة

في قاعدة المعرفة، تخزن قاعدة بيانات المتجهات مستندات قاعدة المعرفة المحوّلة إلى متجهات. تعمل المستندات المحوّلة إلى متجهات كمؤشر لهذه المستندات.

عند تفعيل استرجاع RAG في محادثة وكيل الذكاء الاصطناعي، يتم تحويل رسالة المستخدم إلى متجهات، ثم يتم استرجاع أجزاء من مستندات قاعدة المعرفة من قاعدة بيانات المتجهات لمطابقة الفقرات ذات الصلة والنص الأصلي للمستند.

حاليًا، تدعم إضافة قاعدة المعرفة للذكاء الاصطناعي بشكل مدمج PGVector فقط، وهي إضافة لقاعدة بيانات PostgreSQL.

## إدارة قاعدة بيانات المتجهات

انتقل إلى صفحة إعدادات إضافة وكيل الذكاء الاصطناعي، ثم انقر على علامة التبويب `Vector store`، واختر `Vector database` للدخول إلى صفحة إدارة قاعدة بيانات المتجهات.

![20251022233704](https://static-docs.nocobase.com/20251022233704.png)

انقر على زر `Add new` في الزاوية العلوية اليمنى لإضافة اتصال جديد بقاعدة بيانات المتجهات `PGVector`:

- في حقل الإدخال `Name`، أدخل اسم الاتصال.
- في حقل الإدخال `Host`، أدخل عنوان IP لقاعدة بيانات المتجهات.
- في حقل الإدخال `Port`، أدخل رقم منفذ قاعدة بيانات المتجهات.
- في حقل الإدخال `Username`، أدخل اسم مستخدم قاعدة بيانات المتجهات.
- في حقل الإدخال `Password`، أدخل كلمة مرور قاعدة بيانات المتجهات.
- في حقل الإدخال `Database`، أدخل اسم قاعدة البيانات.
- في حقل الإدخال `Table name`، أدخل اسم الجدول، والذي يُستخدم عند إنشاء جدول جديد لتخزين بيانات المتجهات.

بعد إدخال جميع المعلومات الضرورية، انقر على زر `Test` لاختبار ما إذا كانت خدمة قاعدة بيانات المتجهات متاحة، ثم انقر على زر `Submit` لحفظ معلومات الاتصال.
