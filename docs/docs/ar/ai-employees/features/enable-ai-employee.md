# تفعيل موظفي الذكاء الاصطناعي

يوفر NocoBase العديد من موظفي الذكاء الاصطناعي المدمجين والمخصصين لمختلف السيناريوهات.

## كيفية التفعيل

افتح صفحة إدارة `AI employees` وقم بتفعيل مفتاح `Enabled` مباشرة من القائمة.

![ai-employee-list-enable-switch.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employee-list-enable-switch.png)

## الاستخدام بعد التفعيل

بعد التفعيل، سيظهر الموظف في نقاط الدخول المتاحة (مثل الاختصار الموجود في الزاوية السفلية اليمنى أو مداخل سيناريوهات محددة).
