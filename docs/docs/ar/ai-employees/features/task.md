# المهام المختصرة

لتمكين موظفي الذكاء الاصطناعي (AI Employees) من بدء العمل بكفاءة أكبر، يمكننا ربط موظفي الذكاء الاصطناعي بكتل السيناريوهات (scenario blocks) وتعيين عدة مهام شائعة مسبقاً.

يتيح ذلك للمستخدمين بدء معالجة المهام بنقرة واحدة، دون الحاجة إلى **اختيار الكتلة** و**إدخال الأمر** في كل مرة.

## ربط موظف الذكاء الاصطناعي بالكتلة

بعد الدخول إلى وضع تحرير واجهة المستخدم (UI editing mode)، وبالنسبة للكتل التي تدعم `Actions` (الإجراءات)، اختر قائمة `AI employees` ضمن `Actions` ثم حدد موظف ذكاء اصطناعي. سيتم ربط هذا الموظف بالكتلة الحالية.

![20251022135306](https://static-docs.nocobase.com/20251022135306.png)

بعد اكتمال الربط، في كل مرة تدخل فيها إلى الصفحة، ستعرض منطقة الإجراءات (Actions) في الكتلة موظف الذكاء الاصطناعي المرتبط بها.

![20251022135438](https://static-docs.nocobase.com/20251022135438.png)

## إعداد المهام

بعد الدخول إلى وضع تحرير واجهة المستخدم، قم بتمرير مؤشر الماوس فوق أيقونة موظف الذكاء الاصطناعي المرتبط بالكتلة. سيظهر زر قائمة، اختر منه `Edit tasks` للدخول إلى صفحة إعدادات المهام.

في صفحة إعدادات المهام، يمكنك إضافة مهام متعددة لموظف الذكاء الاصطناعي الحالي.

تمثل كل علامة تبويب مهمة مستقلة، انقر فوق علامة "+" المجاورة لإضافة مهمة جديدة.

![clipboard-image-1771913187](https://static-docs.nocobase.com/clipboard-image-1771913187.png)

نموذج إعدادات المهمة:

- أدخل عنوان المهمة في حقل `Title`. سيظهر هذا العنوان في قائمة مهام موظف الذكاء الاصطناعي.
- أدخل المحتوى الرئيسي للمهمة في حقل `Background`. سيُستخدم هذا المحتوى كرسالة توجيهية للنظام (system prompt) عند التحدث مع موظف الذكاء الاصطناعي.
- أدخل رسالة المستخدم الافتراضية في حقل `Default user message`. سيتم ملؤها تلقائياً في مربع إدخال المستخدم بعد اختيار المهمة.
- في `Work context` (سياق العمل)، اختر معلومات سياق التطبيق الافتراضية لإرسالها إلى موظف الذكاء الاصطناعي. تعمل هذه العملية بنفس الطريقة المتبعة في لوحة الدردشة.
- يعرض محدد `Skills` (المهارات) المهارات التي يمتلكها موظف الذكاء الاصطناعي الحالي. يمكنك إلغاء تحديد مهارة معينة ليتجاهلها الموظف عند تنفيذ هذه المهمة.
- يحدد خيار `Send default user message automatically` ما إذا كان سيتم إرسال رسالة المستخدم الافتراضية تلقائياً بعد النقر لتنفيذ المهمة.


## قائمة المهام

بعد إعداد المهام لموظف الذكاء الاصطناعي، ستظهر هذه المهام في النافذة المنبثقة لملف تعريف الموظف وفي رسالة الترحيب قبل بدء المحادثة. انقر فوق المهمة لتنفيذها.
