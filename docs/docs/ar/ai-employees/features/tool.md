# استخدام المهارات

تُحدد المهارات (Tools) ما يمكن لموظف الذكاء الاصطناعي القيام به.

## هيكل المهارات

تنقسم صفحة المهارات إلى ثلاث فئات:

1. `General skills`: مهارات مشتركة بين جميع موظفي الذكاء الاصطناعي، للقراءة فقط.
2. `Employee-specific skills`: مهارات خاصة بالموظف الحالي، وعادةً ما تكون للقراءة فقط.
3. `Custom skills`: مهارات مخصصة يمكن إضافتها أو حذفها وتكوين أذوناتها الافتراضية.

![skills-three-sections-general-specific-custom.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/skills-three-sections-general-specific-custom.png)

## أذونات المهارات

تم توحيد أذونات المهارات إلى:

- `Ask`: طلب التأكيد قبل الاستدعاء.
- `Allow`: السماح بالاستدعاء المباشر.

توصية: استخدم `Ask` بشكل افتراضي للمهارات التي تتضمن تعديل البيانات.

![skills-permission-ask-allow-segmented.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/skills-permission-ask-allow-segmented.png)

## الإضافة والصيانة

انقر فوق `Add skill` في قسم `Custom skills` لإضافة المهارات وتكوين الأذونات بناءً على مخاطر العمل.
