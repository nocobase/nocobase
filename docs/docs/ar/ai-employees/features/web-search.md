# البحث عبر الإنترنت

يُستخدم البحث عبر الإنترنت لتكملة بيانات تدريب النموذج بأحدث المعلومات الخارجية.

## آلية العمل

تعتمد إتاحة البحث عبر الإنترنت على ما إذا كانت خدمة النموذج المختارة في الجلسة الحالية تدعم ميزة البحث في الويب (Web Search).

- **مدعوم**: يظهر مفتاح تبديل البحث عبر الإنترنت، ويمكن تفعيله أو تعطيله حسب الحاجة.
- **غير مدعوم**: لا يظهر مفتاح التبديل، ويتم إيقاف حالة البحث تلقائيًا.

## الاستخدام في المحادثة

استخدم مفتاح تبديل البحث عبر الإنترنت في منطقة إدخال المحادثة:

- **عند التفعيل**: يمكن للذكاء الاصطناعي إجراء بحث أولاً ثم تقديم الإجابة.

![web-search-switch-visible-on-supported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-visible-on-supported-model.png)

- **عند التعطيل**: يجيب الذكاء الاصطناعي بناءً على السياق الحالي فقط.

![web-search-switch-hidden-on-unsupported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-hidden-on-unsupported-model.png)

## الاختلافات بين المنصات

تختلف قدرات دعم البحث في الويب باختلاف النماذج، يرجى الاستخدام وفقًا للوضع الفعلي لخدمة النموذج والمنصة المختارة.
