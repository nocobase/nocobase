# متقدم

## مقدمة

تمتلك نماذج اللغات الكبيرة الشائعة القدرة على استخدام الأدوات. تتضمن إضافة موظف الذكاء الاصطناعي (AI employee) بعض الأدوات الشائعة المدمجة لتستخدمها هذه النماذج.

المهارات التي تُحدد في صفحة إعدادات موظف الذكاء الاصطناعي هي الأدوات المتاحة لنموذج اللغة الكبير لاستخدامها.

![20251022142348](https://static-docs.nocobase.com/20251022142348.png)

## إعداد المهارات

انتقل إلى صفحة تهيئة إضافة موظف الذكاء الاصطناعي، ثم انقر على علامة التبويب `AI employees` للدخول إلى صفحة إدارة موظفي الذكاء الاصطناعي.

اختر موظف الذكاء الاصطناعي الذي ترغب في إعداد مهاراته، ثم انقر على زر `Edit` للدخول إلى صفحة تعديل موظف الذكاء الاصطناعي.

في علامة التبويب `Skills`، انقر على زر `Add Skill` لإضافة مهارة لموظف الذكاء الاصطناعي الحالي.

![20251022145748](https://static-docs.nocobase.com/20251022145748.png)

## مقدمة عن المهارات

### Frontend (الواجهة الأمامية)

تتيح مجموعة `Frontend` لموظف الذكاء الاصطناعي التفاعل مع مكونات الواجهة الأمامية.

- تُمكّن مهارة `Form filler` موظف الذكاء الاصطناعي من إعادة تعبئة بيانات النموذج المُنشأة في نموذج يحدده المستخدم.

![20251022145954](https://static-docs.nocobase.com/20251022145954.png)

### Data modeling (نمذجة البيانات)

تمنح مجموعة مهارات `Data modeling` موظف الذكاء الاصطناعي القدرة على استدعاء واجهات برمجة التطبيقات (APIs) الداخلية لـ NocoBase لإجراء نمذجة البيانات.

- `Intent Router`: موجّه النوايا، يحدد ما إذا كان المستخدم يرغب في تعديل هيكل مجموعة موجودة أو إنشاء هيكل مجموعة جديد.
- `Get collection names`: يحصل على أسماء جميع المجموعات الموجودة في النظام.
- `Get collection metadata`: يحصل على معلومات هيكل مجموعة محددة.
- `Define collections`: يسمح لموظف الذكاء الاصطناعي بإنشاء مجموعات في النظام.

![20251022150441](https://static-docs.nocobase.com/20251022150441.png)

### Workflow caller (مستدعي سير العمل)

يمنح `Workflow caller` موظف الذكاء الاصطناعي القدرة على تنفيذ سير العمل. سير العمل التي يتم تهيئتها في إضافة سير العمل (workflow plugin) بنوع مشغل (`Trigger type`) `AI employee event` ستكون متاحة هنا كمهارات ليستخدمها موظف الذكاء الاصطناعي.

![20251022153320](https://static-docs.nocobase.com/20251022153320.png)

### Code Editor (محرر الأكواد)

تُمكّن المهارات الموجودة ضمن مجموعة `Code Editor` موظف الذكاء الاصطناعي بشكل أساسي من التفاعل مع محرر الأكواد.

- `Get code snippet list`: يحصل على قائمة قصاصات الكود المعدة مسبقًا.
- `Get code snippet content`: يحصل على محتوى قصاصة كود محددة.

![20251022153811](https://static-docs.nocobase.com/20251022153811.png)

### أخرى

- `Chart generator`: يمنح موظف الذكاء الاصطناعي القدرة على إنشاء الرسوم البيانية وإخراجها مباشرة في المحادثة.

![20251022154141](https://static-docs.nocobase.com/20251022154141.png)
