# متقدم

## مقدمة

عادةً ما تفتقر نماذج اللغة الكبيرة إلى حداثة البيانات، مما يعني أنها لا تملك أحدث المعلومات. لهذا السبب، توفر منصات خدمات نماذج اللغة الكبيرة (LLM) عبر الإنترنت عادةً ميزة البحث عبر الويب، والتي تتيح للذكاء الاصطناعي البحث عن المعلومات باستخدام الأدوات قبل الرد، ثم بناء إجابته على نتائج البحث هذه.

لقد تم تكييف موظفي الذكاء الاصطناعي (AI employees) لدعم ميزة البحث عبر الويب في مختلف منصات خدمات نماذج اللغة الكبيرة عبر الإنترنت. يمكنك تفعيل ميزة البحث عبر الويب هذه ضمن إعدادات نموذج موظف الذكاء الاصطناعي وفي المحادثات.

## تفعيل ميزة البحث عبر الويب

انتقل إلى صفحة إعدادات إضافة موظف الذكاء الاصطناعي، ثم انقر على علامة التبويب `AI employees` للدخول إلى صفحة إدارة موظفي الذكاء الاصطناعي.

![20251021225643](https://static-docs.nocobase.com/20251021225643.png)

اختر موظف الذكاء الاصطناعي الذي ترغب في تفعيل ميزة البحث عبر الويب له، ثم انقر على زر `Edit` للدخول إلى صفحة تعديل موظف الذكاء الاصطناعي.

![20251022114043](https://static-docs.nocobase.com/20251022114043.png)

في علامة التبويب `Model settings`، قم بتشغيل مفتاح `Web Search`، ثم انقر على زر `Submit` لحفظ التغييرات.

![20251022114300](https://static-docs.nocobase.com/20251022114300.png)

## استخدام ميزة البحث عبر الويب في المحادثات

بعد تفعيل ميزة البحث عبر الويب لموظف الذكاء الاصطناعي، سيظهر رمز "الويب" (Web) إضافي في مربع إدخال المحادثة. تكون ميزة البحث عبر الويب مفعلة بشكل افتراضي، ويمكنك النقر عليها لتعطيلها.

![20251022115110](https://static-docs.nocobase.com/20251022115110.png)

عند تفعيل البحث عبر الويب، ستعرض إجابة موظف الذكاء الاصطناعي نتائج البحث التي تم الحصول عليها.

![20251022115502](https://static-docs.nocobase.com/20251022115502.png)
