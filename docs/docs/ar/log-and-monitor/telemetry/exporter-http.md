# مُصدِّر القياس عن بعد: HTTP

## متغيرات البيئة

### TELEMETRY_METRIC_READER

نوع مُصدِّر مقاييس القياس عن بعد.

```bash
TELEMETRY_METRIC_READER=http
```

### TELEMETRY_HTTP_URL

عنوان URL الخاص بـ HTTP لتصدير بيانات القياس عن بعد.

```bash
TELEMETRY_HTTP_URL=
```
