# التكامل

## نظرة عامة

يوفر NocoBase إمكانيات تكامل شاملة، مما يتيح الاتصال السلس بالأنظمة الخارجية وخدمات الطرف الثالث ومصادر البيانات المتنوعة. من خلال طرق التكامل المرنة، يمكنك توسيع وظائف NocoBase لتلبية احتياجات العمل المتنوعة.

## طرق التكامل

### تكامل API

يوفر NocoBase إمكانيات API قوية للتكامل مع التطبيقات والخدمات الخارجية:

![20250301003106](https://static-docs.nocobase.com/20250301003106.png)

- **[مفاتيح API](/integration/api-keys/)**: استخدم مفاتيح API للمصادقة الآمنة والوصول البرمجي إلى موارد NocoBase.
- **[وثائق API](/integration/api-doc/)**: وثائق API مدمجة لاستكشاف واختبار نقاط النهاية.

### تسجيل الدخول الموحد (SSO)

تكامل مع أنظمة الهوية المؤسسية لتحقيق مصادقة موحدة:

![auth_sso-2025-11-24-08-26-46](https://static-docs.nocobase.com/auth_sso-2025-11-24-08-26-46.png)

- **[تكامل SSO](/integration/sso/)**: يدعم مصادقة SAML و OIDC و CAS و LDAP ومنصات الطرف الثالث.
- إدارة مركزية للمستخدمين والتحكم في الوصول.
- تجربة مصادقة سلسة عبر الأنظمة.

### تكامل سير العمل

اربط سير عمل NocoBase بالأنظمة الخارجية:

![auth_sso-2025-11-24-12-02-01](https://static-docs.nocobase.com/auth_sso-2025-11-24-12-02-01.png)

- **[Webhook سير العمل](/integration/workflow-webhook/)**: استقبل الأحداث من الأنظمة الخارجية لتشغيل سير العمل.
- **[طلب HTTP لسير العمل](/integration/workflow-http-request/)**: أرسل طلبات HTTP إلى واجهات API الخارجية من سير العمل.
- أتمتة عمليات الأعمال عبر المنصات.

### مصادر البيانات الخارجية

اتصل بقواعد البيانات وأنظمة البيانات الخارجية:

![20240512085558](https://static-docs.nocobase.com/20240512085558.png)

- **[قواعد البيانات الخارجية](/data-sources/)**: اتصل مباشرة بقواعد بيانات MySQL و PostgreSQL و MariaDB و MSSQL و Oracle و KingbaseES.
- تعرف على هياكل جداول قواعد البيانات الخارجية وقم بإجراء عمليات الإنشاء والقراءة والتحديث والحذف (CRUD) على البيانات الخارجية مباشرة داخل NocoBase.
- واجهة موحدة لإدارة البيانات.

### المحتوى المضمن

ادمج محتوى خارجيًا داخل NocoBase:

![20240408220322](https://static-docs.nocobase.com/20240408220322.png)

- **[مكون Iframe](/integration/block-iframe/)**: ادمج صفحات الويب والتطبيقات الخارجية.
- **مكونات JS**: نفّذ تعليمات JavaScript برمجية مخصصة لتحقيق تكاملات متقدمة.

## سيناريوهات التكامل الشائعة

### تكامل أنظمة المؤسسات

- اربط NocoBase بأنظمة ERP أو CRM أو غيرها من أنظمة المؤسسات.
- مزامنة البيانات ثنائية الاتجاه.
- أتمتة سير العمل عبر الأنظمة.

### تكامل خدمات الطرف الثالث

- استعلم عن حالة الدفع من بوابات الدفع، أو ادمج خدمات الرسائل، أو منصات السحابة.
- استغل واجهات API الخارجية لتوسيع الوظائف.
- أنشئ تكاملات مخصصة باستخدام webhooks وطلبات HTTP.

### تكامل البيانات

- اتصل بمصادر بيانات متعددة.
- اجمع البيانات من أنظمة مختلفة.
- أنشئ لوحات معلومات وتقارير موحدة.

## اعتبارات الأمان

عند دمج NocoBase مع الأنظمة الخارجية، يرجى مراعاة أفضل ممارسات الأمان التالية:

1.  **استخدم HTTPS**: استخدم دائمًا اتصالات مشفرة لنقل البيانات.
2.  **حماية مفاتيح API**: قم بتخزين مفاتيح API بشكل آمن وقم بتدويرها بانتظام.
3.  **مبدأ الحد الأدنى من الامتيازات**: امنح فقط الأذونات الضرورية للتكامل.
4.  **سجلات التدقيق**: راقب وسجل أنشطة التكامل.
5.  **التحقق من صحة البيانات**: تحقق من صحة جميع البيانات الواردة من المصادر الخارجية.
