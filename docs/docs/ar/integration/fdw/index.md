# ربط جداول البيانات الخارجية (FDW)

## مقدمة

ميزة تتيح ربط جداول البيانات البعيدة بناءً على تطبيق Foreign Data Wrapper (FDW) الخاص بقاعدة البيانات. تدعم حالياً قواعد بيانات MySQL و PostgreSQL.

:::info{title="ربط مصادر البيانات مقابل ربط جداول البيانات الخارجية"}
- **ربط مصادر البيانات** يشير إلى إنشاء اتصال مع قاعدة بيانات محددة أو خدمة API، مما يتيح لك استخدام ميزات قاعدة البيانات أو الخدمات التي يوفرها الـ API بشكل كامل.
- **ربط جداول البيانات الخارجية** يشير إلى جلب البيانات من الخارج وربطها للاستخدام المحلي. تُعرف في قواعد البيانات باسم FDW (Foreign Data Wrapper)، وهي تقنية تركز على استخدام الجداول البعيدة كأنها جداول محلية، ويمكن ربط جدول واحد فقط في كل مرة. ونظراً لأن الوصول يتم عن بُعد، فهناك قيود وحدود متنوعة عند الاستخدام.

يمكن أيضاً استخدامهما معاً؛ حيث يُستخدم الأول لإنشاء اتصال مع مصدر البيانات، بينما يُستخدم الثاني للوصول عبر مصادر البيانات المختلفة. على سبيل المثال، عند الاتصال بمصدر بيانات PostgreSQL معين، قد يحتوي هذا المصدر على جدول خارجي تم إنشاؤه بناءً على FDW.
:::

### MySQL

تستخدم MySQL محرك `federated` الذي يتطلب التنشيط، ويدعم الاتصال بقواعد بيانات MySQL البعيدة والقواعد المتوافقة مع بروتوكولها مثل MariaDB. لمزيد من التفاصيل، راجع وثائق [Federated Storage Engine](https://dev.mysql.com/doc/refman/8.0/en/federated-storage-engine.html).

### PostgreSQL

في PostgreSQL، يمكن استخدام أنواع مختلفة من إضافات `fdw` لدعم أنواع مختلفة من البيانات البعيدة. الإضافات المدعومة حالياً هي:

- [postgres_fdw](https://www.postgresql.org/docs/current/postgres-fdw.html): للاتصال بقاعدة بيانات PostgreSQL بعيدة داخل PostgreSQL.
- [mysql_fdw](https://github.com/EnterpriseDB/mysql_fdw): للاتصال بقاعدة بيانات MySQL بعيدة داخل PostgreSQL.
- بالنسبة للأنواع الأخرى من إضافات fdw، راجع [PostgreSQL Foreign Data Wrappers](https://wiki.postgresql.org/wiki/Foreign_data_wrappers). يتطلب دمجها في NocoBase تنفيذ واجهة التكيف (adaptation interface) المقابلة في الكود.

## المتطلبات المسبقة

- إذا كانت قاعدة البيانات الرئيسية لـ NocoBase هي MySQL، فيجب تنشيط `federated`. راجع [كيفية تمكين محرك federated في MySQL](./enable-federated)

ثم قم بتثبيت وتنشيط الإضافة من خلال مدير الإضافات.

## دليل المستخدم

ضمن "إدارة المجموعات > إنشاء مجموعة"، اختر "ربط البيانات الخارجية" من القائمة المنسدلة.

![ربط البيانات الخارجية](https://static-docs.nocobase.com/029d946a6d067d1c35a39755219d623c.png)

في القائمة المنسدلة "خادم قاعدة البيانات"، اختر خدمة قاعدة بيانات موجودة، أو "إنشاء خادم قاعدة بيانات".

![خدمة قاعدة البيانات](https://static-docs.nocobase.com/766271708a911950a5599d60d6be4a4d.png)

إنشاء خادم قاعدة بيانات

![إنشاء خادم قاعدة بيانات](https://static-docs.nocobase.com/1e357216e04cc4f200bd6212827281c8.png)

بعد اختيار خادم قاعدة البيانات، اختر جدول البيانات الذي تريد ربطه من القائمة المنسدلة "الجدول البعيد".

![اختر جدول البيانات الذي تريد ربطه](https://static-docs.nocobase.com/e91fd6152b52b4fc01b3808053cc8dc4.png)

تكوين معلومات الحقول

![تكوين معلومات الحقول](https://static-docs.nocobase.com/e618fecc5fe327f6a495e61405e5f040.png)

![المزامنة من الجدول البعيد](https://static-docs.nocobase.com/3751a9a39f933889fb3fcc4d85a6f4ad.png)
