# تضمين NocoBase

## مقدمة

تُستخدم هذه الإضافة لتضمين صفحات NocoBase في مواقع ويب أو تطبيقات أخرى.

## التثبيت

هذه الإضافة تجارية، ويجب رفعها وتفعيلها عبر مدير الإضافات.

![20240323162741](https://static-docs.nocobase.com/20240323162741.png)

## دليل الاستخدام

بعد تفعيل إعدادات الواجهة، انقر على "نسخ رابط التضمين" في قائمة إعدادات الصفحة (الزاوية العلوية اليمنى).

![](https://static-docs.nocobase.com/f11bd6d5e88d38731d7cd3cb149022c8.png)

بعد النقر، ستحصل على رابط يمكن فتحه بشكل منفصل، على سبيل المثال: `https://xxx.com/embed/qs087rz4o2b`.

![](https://static-docs.nocobase.com/9d847805a00fd07372a474665ec0e968.png)

إذا كنت ترغب في تضمين صفحات NocoBase في مواقع ويب أو تطبيقات أخرى، فيلزم مصادقة المستخدم، ويجب إضافة `token` إلى الرابط، على سبيل المثال: `https://xxx.com/embed/qs087rz4o2b?token=xxx`.
