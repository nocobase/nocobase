# نظرة عامة

يوفر تطوير إضافات NocoBase من جانب العميل مجموعة متنوعة من الوظائف والإمكانيات لمساعدة المطورين على تخصيص وتوسيع ميزات الواجهة الأمامية لـ NocoBase. فيما يلي القدرات الرئيسية والفصول ذات الصلة لتطوير إضافات NocoBase من جانب العميل:

| الوحدة | الوصف | الفصل ذو الصلة |
|---|---|---|
| **فئة الإضافة** | إنشاء وإدارة إضافات جانب العميل، وتوسيع وظائف الواجهة الأمامية | [plugin.md](plugin.md) |
| **إدارة التوجيه (Router)** | تخصيص توجيه الواجهة الأمامية، وتنفيذ التنقل بين الصفحات وإعادة التوجيه | [router.md](router.md) |
| **التعامل مع الموارد** | إدارة موارد الواجهة الأمامية، ومعالجة جلب البيانات والعمليات | [resource.md](resource.md) |
| **معالجة الطلبات** | تخصيص طلبات HTTP، ومعالجة استدعاءات API ونقل البيانات | [request.md](request.md) |
| **إدارة السياق** | الحصول على سياق التطبيق واستخدامه، والوصول إلى الحالة والخدمات العامة | [context.md](context.md) |
| **التحكم بالوصول (ACL)** | تنفيذ التحكم بالوصول للواجهة الأمامية، والتحكم في أذونات الوصول للصفحات والميزات | [acl.md](acl.md) |
| **إدارة مصادر البيانات** | إدارة واستخدام مصادر بيانات متعددة، وتنفيذ تبديل مصادر البيانات والوصول إليها | [data-source-manager.md](data-source-manager.md) |
| **الأنماط والمظاهر** | تخصيص الأنماط والمظاهر، وتنفيذ تخصيص الواجهة وتحسينها | [styles-themes.md](styles-themes.md) |
| **دعم اللغات المتعددة (I18n)** | دمج دعم اللغات المتعددة، وتنفيذ التدويل والتعريب | [i18n.md](i18n.md) |
| **إخراج السجلات** | تخصيص تنسيقات السجلات وطرق الإخراج، وتحسين قدرات التصحيح والمراقبة | [logger.md](logger.md) |
| **كتابة حالات الاختبار** | كتابة وتشغيل حالات الاختبار، لضمان استقرار الإضافة ودقة الوظائف | [test.md](test.md) |

توسيعات واجهة المستخدم (UI)

| الوحدة | الوصف | الفصل ذو الصلة |
|---|---|---|
| **تكوين واجهة المستخدم (UI)** | استخدام محرك السير (FlowEngine) ونماذج السير (flow models) لتنفيذ التكوين الديناميكي وتنسيق خصائص المكونات، ودعم التخصيص المرئي للصفحات والتفاعلات المعقدة | [محرك السير](../../flow-engine/index.md) و [نماذج السير](../../flow-engine/flow-model.md) |
| **توسيعات الكتل** | تخصيص كتل الصفحات، وإنشاء وحدات وتخطيطات واجهة مستخدم قابلة لإعادة الاستخدام | [الكتل](../../ui-development-block/index.md) |
| **توسيعات الحقول** | تخصيص أنواع الحقول، وتنفيذ عرض وتحرير البيانات المعقدة | [الحقول](../../ui-development-field/index.md) |
| **توسيعات الإجراءات** | تخصيص أنواع الإجراءات، وتنفيذ المنطق المعقد ومعالجة التفاعل | [الإجراءات](../../ui-development-action/index.md) |
