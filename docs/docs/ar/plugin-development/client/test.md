# الاختبار

:::tip

سيقدم هذا الفصل كيفية كتابة وتشغيل الاختبارات للجزء الأمامي من الإضافات. سيتضمن المحتوى:
- إعداد بيئة الاختبار والتبعيات؛
- كتابة اختبارات الوحدة باستخدام Vitest و Testing Library؛
- كيفية محاكاة بيئة تشغيل NocoBase الأمامية؛
- التنفيذ التلقائي للاختبارات في CI.

سنقوم بإضافة أمثلة مفصلة وأفضل الممارسات في الإصدارات المستقبلية.
