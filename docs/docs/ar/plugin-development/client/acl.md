# التحكم بالصلاحيات باستخدام ACL

:::tip

سيتناول هذا الفصل كيفية استخدام ACL (قائمة التحكم بالوصول) لإدارة الصلاحيات في إضافات الواجهة الأمامية. سيتضمن المحتوى:

- الكشف عن الصلاحيات؛
- كيفية عرض المحتوى ديناميكيًا بناءً على الصلاحيات؛

سنقوم بإضافة أمثلة تفصيلية وأفضل الممارسات في الإصدارات اللاحقة.

:::
