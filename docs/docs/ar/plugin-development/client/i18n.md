# I18n تدويل التطبيقات

تدعم إضافات NocoBase تدويل التطبيقات (i18n) متعدد اللغات لكل من الواجهة الأمامية والخلفية. من خلال آلية موحدة، يمكنك بسهولة تنفيذ محتوى متعدد اللغات في الإضافات.

## إدارة ملفات اللغات المتعددة

تُخزّن ملفات اللغات المتعددة للإضافات بشكل موحد في المجلد `src/locale`. يُنصح بتسميتها حسب اللغة، على سبيل المثال:

```bash
|- /plugin-hello
  |- /src
    |- /locale
      |- en-US.json   # اللغة الإنجليزية
      |- zh-CN.json   # اللغة الصينية
```

يُصدّر كل ملف لغة كائن JSON يحتوي على جميع إدخالات الترجمة لتلك اللغة، على سبيل المثال:

```json
// zh-CN.json
{
  "Hello": "你好",
  "World": "世界",
  "Enter your name": "请输入你的名字",
  "Your name is {{name}}": "你的名字是 {{name}}"
}
```

```json
// en-US.json
{
  "Hello": "Hello",
  "World": "World",
  "Enter your name": "Enter your name",
  "Your name is {{name}}": "Your name is {{name}}"
}
```

> عند إضافة ملفات لغة لأول مرة، تحتاج إلى إعادة تشغيل التطبيق لتصبح سارية المفعول. يمكنك التحقق من سريان مفعول إدخالات الترجمة عبر واجهة برمجة التطبيقات (API):  
> http://localhost:13000/api/app:getLang?locale=zh-CN
