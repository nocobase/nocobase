# استخدام متغيرات السياق

باستخدام متغيرات السياق، يمكنك إعادة استخدام المعلومات من الصفحة الحالية، المستخدم، الوقت، ومدخلات التصفية، مما يتيح لك عرض الرسوم البيانية وتمكين الربط بناءً على السياق.

## نطاق التطبيق
- في استعلام البيانات بوضع Builder، اختر المتغيرات لشروط التصفية.
![clipboard-image-1761486073](https://static-docs.nocobase.com/clipboard-image-1761486073.png)

- في استعلام البيانات بوضع SQL، اختر المتغيرات وأدخل التعبيرات (على سبيل المثال، `{{ ctx.user.id }}`).
![clipboard-image-1761486145](https://static-docs.nocobase.com/clipboard-image-1761486145.png)

- في خيارات الرسوم البيانية بوضع Custom، اكتب تعبيرات JS مباشرةً.
![clipboard-image-1761486604](https://static-docs.nocobase.com/clipboard-image-1761486604.png)

- في أحداث التفاعل (على سبيل المثال، النقر لفتح نافذة منبثقة (drill-down) وتمرير البيانات)، اكتب تعبيرات JS مباشرةً.
![clipboard-image-1761486683](https://static-docs.nocobase.com/clipboard-image-1761486683.png)

**ملاحظة:**
- لا تضع `{{ ... }}` بين علامات اقتباس مفردة أو مزدوجة؛ يتم التعامل مع الربط بأمان بناءً على نوع المتغير (نص، رقم، وقت، NULL).
- عندما يكون المتغير `NULL` أو غير معرّف، تعامل مع القيم الفارغة بشكل صريح في SQL باستخدام `COALESCE(...)` أو `IS NULL`.
