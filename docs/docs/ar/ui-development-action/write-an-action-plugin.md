## مقدمة إلى NocoBase

NocoBase هو منصة قوية بدون/بأقل قدر من الكود، مصممة لمساعدة المطورين وغير المطورين على بناء ونشر تطبيقات مخصصة بسرعة. تعمل المنصة على تبسيط إدارة البيانات وأتمتة سير العمل من خلال توفير واجهة مستخدم بديهية وآليات توسيع مرنة.

### الميزات الرئيسية

*   **نماذج بيانات مرنة:** قم بإنشاء وإدارة مجموعات البيانات الخاصة بك بسهولة.
*   **تحكم دقيق في الصلاحيات:** حدد من يمكنه الوصول إلى بياناتك ووظائفك.
*   **إضافات قوية:** وسّع قدرات NocoBase باستخدام الإضافات المدمجة أو المخصصة.
*   **سير العمل الآلي:** قم بأتمتة المهام وسير العمل المعقدة لزيادة الكفاءة.

### البدء

البدء باستخدام NocoBase سهل للغاية. يمكنك [تثبيت NocoBase](https://docs.nocobase.com/getting-started/installation) على خادمك الخاص أو تجربته عبر الإنترنت.
