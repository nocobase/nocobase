# الإشعارات: WeCom

## المقدمة

تتيح إضافة **WeCom** للتطبيق إرسال إشعارات إلى مستخدمي WeCom.

## إضافة وإعداد موثّق WeCom

أولاً، يجب عليك إضافة وإعداد موثّق WeCom داخل NocoBase. راجع: [مصادقة المستخدم - WeCom](/auth-verification/auth-wecom).  
يمكن فقط للمستخدمين الذين قاموا بتسجيل الدخول عبر WeCom تلقي إشعارات النظام من خلاله.
