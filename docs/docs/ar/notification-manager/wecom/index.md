# الإشعارات: WeCom

## مقدمة

تتيح **إضافة WeCom** للتطبيق إرسال رسائل إشعارات إلى مستخدمي WeCom.

## إضافة وتكوين أداة مصادقة WeCom

أولاً، تحتاج إلى إضافة وتكوين أداة مصادقة WeCom على NocoBase. راجع [مصادقة المستخدم - WeCom](/auth-verification/auth-wecom). فقط مستخدمو النظام الذين قاموا بتسجيل الدخول عبر WeCom هم من يمكنهم تلقي إشعارات النظام من خلال WeCom.
