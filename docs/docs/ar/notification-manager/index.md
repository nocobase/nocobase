# إدارة الإشعارات

## مقدمة

إدارة الإشعارات هي خدمة مركزية تدمج قنوات إشعارات متعددة. توفر هذه الخدمة واجهة موحدة لتكوين القنوات، وإدارة عمليات الإرسال، وتسجيل السجلات، كما تدعم التوسع المرن لإضافة قنوات جديدة.

![20240928112556](https://static-docs.nocobase.com/20240928112556.png)

-   **الجزء الأرجواني**: إدارة الإشعارات، توفر خدمة شاملة تتضمن تكوين القنوات وتسجيل السجلات، مع إمكانية التوسع لإضافة قنوات إشعارات إضافية.
-   **الجزء الأخضر**: الرسائل داخل التطبيق (In-App Message)، وهي قناة مدمجة تمكن المستخدمين من تلقي الإشعارات مباشرة داخل التطبيق.
-   **الجزء الأحمر**: البريد الإلكتروني (Email)، وهي قناة قابلة للتوسيع، تتيح للمستخدمين تلقي الإشعارات عبر البريد الإلكتروني.

## إدارة القنوات

![20240928181752](https://static-docs.nocobase.com/20240928181752.png)

القنوات المدعومة حاليًا هي:

-   [الرسائل داخل التطبيق](/notification-manager/notification-in-app-message)
-   [البريد الإلكتروني](/notification-manager/notification-email) (باستخدام طريقة نقل SMTP المدمجة)

يمكنك أيضًا التوسع لإضافة المزيد من قنوات الإشعارات، يُرجى الرجوع إلى وثائق [توسيع القنوات](/notification-manager/development/extension).

## سجلات الإشعارات

يسجل النظام معلومات مفصلة وحالة كل إشعار، مما يسهل عملية التحليل واستكشاف الأخطاء وإصلاحها.
