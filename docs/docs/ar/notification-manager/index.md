# مدير الإشعارات

## المقدمة

مدير الإشعارات هو خدمة مركزية تدمج عدة قنوات للإشعارات. يوفر واجهة موحدة لإعداد القنوات وإدارة الإرسال وتسجيل العمليات، كما يدعم التوسّع المرن لإضافة قنوات إشعارات جديدة.

![20240928112556](https://static-docs.nocobase.com/20240928112556.png)

- **القسم البنفسجي**: يوفر مدير الإشعارات خدمة شاملة تشمل إعداد القنوات وتسجيل العمليات، مع إمكانية التوسّع لإضافة قنوات إشعارات إضافية.
- **القسم الأخضر**: الرسائل داخل التطبيق (In-App Message)، وهي قناة مدمجة تتيح للمستخدمين تلقي الإشعارات مباشرة داخل التطبيق.
- **القسم الأحمر**: البريد الإلكتروني (Email)، وهي قناة قابلة للتوسّع تتيح للمستخدمين تلقي الإشعارات عبر البريد الإلكتروني.

## إدارة القنوات

![20240928181752](https://static-docs.nocobase.com/20240928181752.png)

القنوات المدعومة حاليًا:

- [رسائل داخل التطبيق](/notification-manager/notification-in-app-message)
- [البريد الإلكتروني](/notification-manager/notification-email) (باستخدام بروتوكول SMTP المدمج)

يمكنك أيضًا التوسّع لإضافة المزيد من القنوات، راجع وثائق [توسيع القنوات](/notification-manager/development/extension).
