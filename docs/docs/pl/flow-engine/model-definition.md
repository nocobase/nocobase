# Pole Regionu Chin

## Wprowadzenie

Wtyczka Pola Regionu Chin (`field-china-region`) dodaje typ pola „Region Chin” do NocoBase.

## Instalacja

W głównym interfejsie NocoBase przejdź do „Menedżera wtyczek”, znajdź wtyczkę „Region Chin” i ją włącz.

## Użycie

W konfiguracji kolekcji dodaj nowe pole i wybierz „Region Chin” jako typ pola.

![](/images/field-china-region-1.png)

### Konfiguracja pola

W konfiguracji pola można ustawić tryb wyświetlania dla pola:

- Prowincja
- Prowincja/Miasto
- Prowincja/Miasto/Dzielnica
- Prowincja/Miasto/Dzielnica/Powiat

![](/images/field-china-region-2.png)
