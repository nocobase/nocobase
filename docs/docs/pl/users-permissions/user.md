# Użytkownicy

## Zarządzanie użytkownikami

Ta wtyczka udostępnia interfejs do zarządzania użytkownikami w centrum konfiguracji. Po inicjalizacji aplikacji automatycznie tworzone jest konto superadministratora, którego nie można usunąć. Superadministrator posiada uprawnienia Root i ma dostęp do wszystkich zasobów.

![](https://static-docs.nocobase.com/44bf40f56b45d4dd96c424fb08082cf6.png)

Ponadto, mogą Państwo również dodawać różne bloki z kolekcji Użytkownicy, aby zarządzać użytkownikami, na przykład bloki tabeli:

![](https://static-docs.nocobase.com/76b5a4652f869541a9e8f18a4568a7c9.png)
