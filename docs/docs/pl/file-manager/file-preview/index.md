# Podgląd plików

W interfejsach zawierających pola plików (w tym pola załączników) mogą Państwo wyświetlić podgląd plików, klikając miniaturę lub ikonę pliku. Wbudowana funkcja podglądu obsługuje różne typy plików, w tym obrazy, pliki PDF oraz większość formatów obsługiwanych natywnie przez przeglądarki.

![20251129232307](https://static-docs.nocobase.com/20251129232307.png)

W przypadku typów plików, które nie obsługują natywnego podglądu, mogą Państwo włączyć tę funkcjonalność, instalując lub rozszerzając odpowiednie wtyczki podglądu plików. Na przykład, po zainstalowaniu wtyczki Office File Preview, możliwe jest przeglądanie plików Word, Excel oraz PowerPoint.

Obecnie NocoBase oferuje następujące wtyczki podglądu plików:

* [Wtyczka Office File Preview](../file-preview/ms-office.md)
