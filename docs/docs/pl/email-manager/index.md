# Menedżer poczty e-mail

## Wprowadzenie
Wtyczka Menedżer poczty e-mail to wydajne i wygodne narzędzie, które umożliwia autoryzację i integrację ze skrzynkami pocztowymi Gmail i Outlook. Pomaga użytkownikom włączyć funkcje zarządzania pocztą e-mail, takie jak wysyłanie i odbieranie wiadomości, do różnych bloków i stron. Dzięki prostej konfiguracji autoryzacji, mogą Państwo zarządzać wieloma kontami w jednym miejscu i cieszyć się płynną komunikacją e-mailową.
