# Automatyczny zapis wersji roboczej

W menu przycisku **Zapisz wersję roboczą** mogą Państwo włączyć opcję **Automatyczne zapisywanie wersji roboczej**.

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_29_PM.png)

Podczas edycji wiadomości e-mail system zapisuje wersję roboczą w czasie rzeczywistym, zapewniając, że żadna treść nie zostanie utracona.

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_31_PM.png)

Po ponownym otwarciu edytora wiadomości e-mail (dla tego samego nadawcy i odbiorcy) system zapyta, czy chcą Państwo użyć zapisanej wersji roboczej.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_08_35_PM.png)
