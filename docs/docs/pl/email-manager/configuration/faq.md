# Często zadawane pytania

## Nie można odbierać wiadomości e-mail po autoryzacji kontem Microsoft

Obecnie autoryzacja logowania jest obsługiwana tylko dla kont e-mail Outlook i Gmail. Konta Microsoft i Google nie są obecnie obsługiwane. Więcej informacji znajdą Państwo pod adresem: [answers.microsoft.com](https://answers.microsoft.com/zh-hans/outlook_com/forum/all/%E7%8E%B0%E6%9C%89%E5%BE%AE%E8%BD%AF%E8%B4%A6/dba12dda-a7c7-4346-8263-53f4a6d9dc68)

**Wskazówka**: Jeśli nie są Państwo pewni, czy posiadają "prawdziwe konto e-mail Outlook.com" lub "konto Gmail", mogą Państwo spróbować odwiedzić Outlook.com lub Gmail.com w przeglądarce internetowej, aby sprawdzić, czy można się zalogować bezpośrednio i normalnie wysyłać wiadomości e-mail. Jeśli to się nie uda, oznacza to, że prawdopodobnie nie posiadają Państwo odpowiedniej usługi pocztowej i należy ją najpierw aktywować lub użyć innego konta e-mail.

## Cofanie autoryzacji

Jeśli Państwa konto e-mail zostało już autoryzowane, ale chcą Państwo usunąć lub ponownie autoryzować dostęp, proszę postępować zgodnie z poniższymi krokami:

#### **Gmail**

1. Proszę otworzyć https://myaccount.google.com/u/0/connections i zalogować się.

![](https://static-docs.nocobase.com/mail-1735634131347.png)

2. Proszę kliknąć odpowiednią aplikację, a następnie kliknąć "Usuń".

![](https://static-docs.nocobase.com/mail-1735634131518.png)

![](https://static-docs.nocobase.com/mail-1735634131697.png)

#### **Poczta Microsoft**

1. Proszę otworzyć https://account.microsoft.com/ i zalogować się.
    
2. Proszę kliknąć przycisk "Aplikacje i usługi, które mają dostęp do Twoich danych".

![](https://static-docs.nocobase.com/mail-1735634131870.png)

3. Proszę kliknąć "Edytuj", a następnie "Usuń".

![](https://static-docs.nocobase.com/mail-1735634132052.png)
