# Podpis e-mail

## Konfiguracja podpisu

Proszę przejść do Centrum Zarządzania E-mailami i kliknąć przycisk ustawień:

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_05_34_PM.png)

Następnie proszę przełączyć się na panel podpisu:

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_05_51_PM.png)

Tutaj mogą Państwo zarządzać podpisami. Zaznaczenie opcji przed podpisem pozwala ustawić go jako domyślny:

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_05_37_PM.png)

## Używanie podpisów

W treści wiadomości e-mail proszę kliknąć ikonę podpisu, aby wybrać żądany podpis:

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_05_40_PM.png)
