# Podpis e-mail

## Konfiguracja podpisu

Proszę przejść do Centrum Zarządzania E-mailami i kliknąć przycisk ustawień:

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_08_PM.png)

Następnie proszę przełączyć się na panel podpisu:

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_08_PM%20(1)

Tutaj mogą Państwo zarządzać podpisami. Zaznaczenie opcji przed podpisem pozwala ustawić go jako domyślny:

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_09_PM.png)

## Używanie podpisów

W treści wiadomości e-mail proszę kliknąć ikonę podpisu, aby wybrać żądany podpis:

![](https://static-docs.nocobase.com/email-manager/Loading--12-31-2025_10_11_PM.png)
