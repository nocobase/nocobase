# ctx.i18n

Instancja i18n bieżącego kontekstu, używana do odczytu lub zmiany języka. **Do tłumaczenia tekstów należy używać [ctx.t()](./t.md)**; nie należy używać `ctx.i18n.t`.

## Scenariusze użycia

Wszystkie środowiska wykonawcze RunJS mogą korzystać z `ctx.i18n` (np. JSBlock, JSField, JSItem, JSColumn, przepływ pracy, reguły powiązań itp.).

## Definicja typu

```typescript
interface i18n: {
  language: string;
  changeLanguage(lng: string): Promise<any>;
}
```

## Często używane właściwości

| Właściwość | Typ | Opis |
|------|------|------|
| `language` | `string` | Kod aktualnie aktywnego języka (np. `pl-PL`, `en-US`, `zh-CN`) |

## Często używane metody

### changeLanguage(lng)

Przełącza bieżący język.

| Parametr | Typ | Opis |
|------|------|------|
| `lng` | `string` | Kod języka docelowego (np. `'en-US'`, `'zh-CN'`) |

**Zwracana wartość**: `Promise<any>`, rozwiązuje się po zakończeniu zmiany języka.

## Przykłady

### Odczyt bieżącego języka

```ts
const lang = ctx.i18n.language;
// 'zh-CN' | 'en-US' | ...
if (lang.startsWith('zh')) {
  ctx.render(ctx.t('Interfejs chiński'));
} else {
  ctx.render(ctx.t('English UI'));
}
```

### Przełączanie języków

```ts
// Przełącz na angielski
await ctx.i18n.changeLanguage('en-US');

// Przełącz na chiński
await ctx.i18n.changeLanguage('zh-CN');
```

### Przycisk zmiany języka

```ts
const { Button } = ctx.libs.antd;
const isZh = ctx.i18n.language.startsWith('zh');
ctx.render(
  <Button onClick={async () => {
    await ctx.i18n.changeLanguage(isZh ? 'en-US' : 'zh-CN');
  }}>
    {ctx.t(isZh ? 'Switch to English' : 'Przełącz na chiński')}
  </Button>,
);
```

## Uwagi

- **Teksty tłumaczeń**: Należy konsekwentnie używać `ctx.t()`, nie należy używać `ctx.i18n.t`.
