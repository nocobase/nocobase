# Osadzanie NocoBase

## Wprowadzenie

Ta wtyczka służy do osadzania stron NocoBase w innych witrynach internetowych lub aplikacjach.

## Instalacja

Ta wtyczka jest wtyczką komercyjną i należy ją przesłać oraz aktywować za pośrednictwem menedżera wtyczek.

![20240323162741](https://static-docs.nocobase.com/20240323162741.png)

## Instrukcja użytkowania

Po aktywacji konfiguracji interfejsu, proszę kliknąć „Kopiuj link do osadzenia” w menu konfiguracji strony (w prawym górnym rogu).

![](https://static-docs.nocobase.com/f11bd6d5e88d38731d7cd3cb149022c8.png)

Po kliknięciu otrzymają Państwo link, który można otworzyć osobno, na przykład `https://xxx.com/embed/qs087rz4o2b`.

![](https://static-docs.nocobase.com/9d847805e88d38731d7cd3cb149022c8.png)

Jeśli chcą Państwo osadzić strony NocoBase w innych witrynach internetowych lub aplikacjach, wymagana jest autoryzacja użytkownika, a `token` należy dołączyć do linku, na przykład `https://xxx.com/embed/qs087rz4o2b?token=xxx`.
