# Eksporter telemetrii: HTTP

## Konfiguracja zmiennych środowiskowych

### TELEMETRY_METRIC_READER

Typ eksportera metryk telemetrii.

```bash
TELEMETRY_METRIC_READER=http
```

### TELEMETRY_HTTP_URL

Adres URL HTTP do eksportowania danych telemetrycznych.

```bash
TELEMETRY_HTTP_URL=
```
