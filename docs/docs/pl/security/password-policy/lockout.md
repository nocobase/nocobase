# Blokada użytkownika

## Wprowadzenie

Umożliwia zarządzanie zablokowanymi użytkownikami oraz aktywne blokowanie dostępu.

![](https://static-docs.nocobase.com/202412281450083.png)

### Odblokowywanie użytkownika

Użytkowników, którzy zostali zablokowani z powodu przekroczenia limitu nieprawidłowych prób logowania, można odblokować, usuwając odpowiedni rekord.

### Blokowanie użytkownika

Aktywnie zablokować użytkownika można, dodając jego rekord. Po zablokowaniu użytkownikowi zostanie uniemożliwiony dostęp do systemu za pomocą dowolnej metody uwierzytelniania, włączając w to klucze API.

![](https://static-docs.nocobase.com/202412281450512.png)
