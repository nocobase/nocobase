# Tryb ładowania danych

## Wprowadzenie

Domyślnym trybem ładowania danych jest „Ładuj wszystkie dane, gdy warunki filtrowania są puste”. W tym przypadku „warunki filtrowania” odnoszą się do wartości w „Bloku filtrowania”. Inną dostępną opcją jest „Nie ładuj danych, gdy warunki filtrowania są puste”.
