# Układ

## Wprowadzenie

Bloki formularzy, takie jak blok formularza, blok szczegółów czy blok listy, obsługują ustawienia układu. Dostępne opcje układu to układ pionowy (etykiety i pola wprowadzania są ułożone w wierszach) oraz układ poziomy (etykiety i pola wprowadzania są umieszczone obok siebie). Mogą Państwo dostosować układ pól bloku do swoich konkretnych potrzeb, aby zapewnić przejrzyste i efektywne wprowadzanie oraz przeglądanie informacji.

### Pionowy (domyślny)

Domyślny układ jest pionowy.

![20251024120143](https://static-docs.nocobase.com/20251024120143.png)

### Poziomy

Układ poziomy wyświetla etykiety i pola wprowadzania obok siebie, co skutecznie oszczędza miejsce. W przypadku pól z długimi tytułami mogą Państwo wybrać opcję skracania (wielokropek) lub zawijania tekstu, aby zapewnić schludny i estetyczny wygląd interfejsu.

![20251024120237](https://static-docs.nocobase.com/20251024120237.png)

#### Wyrównanie tytułu pola

- **Do lewej** (domyślnie)
- **Do prawej**

#### Szerokość tytułu pola

- Domyślna szerokość to **120px**, z minimalną szerokością **50px**.

#### Obsługa tytułów pól przekraczających szerokość

- **Zawijanie tekstu**
- **Wielokropek**

#### Wyświetlanie dwukropka

Mogą Państwo wybrać, czy po tytule pola ma być wyświetlany dwukropek.
