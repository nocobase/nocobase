# Usuwanie bloku

## Wprowadzenie

Usunięcie bloku powoduje jego usunięcie z bieżącej strony i jest to operacja nieodwracalna. Jeśli przed usunięciem wykonano kopię zapasową, mogą Państwo ją przywrócić.
