# Ustawianie zakresu danych

## Wprowadzenie

Ustawienie zakresu danych oznacza zdefiniowanie domyślnych warunków filtrowania dla bloku danych. Użytkownicy mogą elastycznie dostosowywać zakres danych do swoich potrzeb biznesowych, jednak niezależnie od przeprowadzanych operacji filtrowania, system automatycznie zastosuje ten domyślny warunek, zapewniając, że dane zawsze pozostaną w granicach określonego zakresu.

## Przewodnik użytkownika

![20251027110053](https://static-docs.nocobase.com/20251027110053.png)

Pole filtrowania umożliwia wybór pól z bieżącej kolekcji oraz z powiązanych kolekcji.

![20251027110140](https://static-docs.nocobase.com/20251027110140.png)

### Operatory

Różne typy pól obsługują różne operatory. Na przykład, pola tekstowe obsługują operatory takie jak 'równa się', 'nie równa się' i 'zawiera'; pola numeryczne obsługują operatory takie jak 'większe niż' i 'mniejsze niż'; natomiast pola daty obsługują operatory takie jak 'jest w zakresie' i 'jest przed określoną datą'.

![20251027111124](https://static-docs.nocobase.com/20251027111124.png)
