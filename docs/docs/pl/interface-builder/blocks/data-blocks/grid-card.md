# Siatka kart

## Wprowadzenie

Blok Siatka kart wyświetla podsumowanie rekordów danych w formie kart. Obsługuje konfigurację liczby kolumn dla różnych rozmiarów ekranu, aby zapewnić przyjazne dla użytkownika wyświetlanie na różnych urządzeniach.
