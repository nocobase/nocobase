# Wykres

## Wprowadzenie

Wtyczka do wizualizacji danych NocoBase oferuje wizualne zapytania danych oraz bogaty zestaw komponentów wykresów. Użytkownicy mogą szybko tworzyć panele wizualizacyjne za pomocą prostej konfiguracji, aby prezentować wnioski z danych oraz wspierać wielowymiarową analizę i prezentację danych.
