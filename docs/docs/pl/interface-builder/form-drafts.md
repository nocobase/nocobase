# Wersje robocze formularzy

Po włączeniu wersji roboczych formularzy, wprowadzana treść jest automatycznie zapisywana lokalnie. Nawet jeśli formularz zostanie przypadkowo zamknięty lub strona zostanie odświeżona, wypełnione dane nie zostaną utracone.

![20251213093717](https://static-docs.nocobase.com/20251213093717.png)

Gdy istnieje wersja robocza, u góry formularza wyświetli się powiadomienie. Po pomyślnym przesłaniu formularza wersja robocza zostanie automatycznie usunięta; jeśli nie jest już potrzebna, można ją również usunąć ręcznie w dowolnym momencie.

![20251213092734](https://static-docs.nocobase.com/20251213092734.png)

::: tip Wskazówka

Obecnie wersje robocze formularzy obsługują wyłącznie buforowanie lokalne. Przyszłe wersje będą wspierać buforowanie po stronie serwera, co umożliwi synchronizację między wieloma urządzeniami.

:::
