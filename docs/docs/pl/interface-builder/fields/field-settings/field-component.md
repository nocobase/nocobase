# Komponent pola

## Wprowadzenie

Komponenty pola służą do wyświetlania zawartości pola.
Niektóre pola posiadają wbudowane różne komponenty wyświetlania. Mogą Państwo wybrać odpowiedni komponent zgodnie z Państwa potrzebami, aby prezentować wartość pola na różne sposoby.

Przykład: W trybie podglądu, pole URL można przełączyć na komponent Podgląd lub komponent URL (domyślny).
