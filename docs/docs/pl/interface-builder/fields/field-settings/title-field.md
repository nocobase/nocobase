# Pole tytułowe

## Wprowadzenie

**Pole tytułowe** jest zazwyczaj używane w komponentach pól relacyjnych jako identyfikator dla powiązanych danych wyświetlanych w interfejsie. Określa ono sposób prezentacji głównych informacji o danych, zazwyczaj w celu pomocy użytkownikom w szybkim ich zidentyfikowaniu i zrozumieniu.

Różne typy pól tytułowych odpowiadają różnym komponentom pól i obsługują odpowiednie opcje konfiguracji. Na przykład:

- Gdy pole tytułowe jest **polem daty**, komponent pola używa **komponentu daty** do wyświetlenia danych.
- Gdy pole tytułowe jest **polem tekstowym**, komponent pola używa **komponentu tekstowego** do wyświetlenia danych.
- Gdy pole tytułowe jest **polem wyboru**, komponent pola używa **komponentu wyboru** do wyświetlenia danych.
