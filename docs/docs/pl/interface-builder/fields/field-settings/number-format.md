# Formatowanie Liczb

## Wprowadzenie

Formatowanie liczb to wizualne ulepszenie i standaryzacja pól typu liczbowego (w tym pól formuł) w trybie podglądu. Obejmuje ono ustawienia takie jak separatory tysięcy, precyzja miejsc dziesiętnych, konwersja jednostek oraz notacja naukowa. Konfigurując formatowanie liczb, mogą Państwo dostosować sposób wyświetlania wartości liczbowych do zwyczajów panujących w różnych regionach i branżach.

## Jak używać

- Obsługuje proste konwersje jednostek (mnożenie/dzielenie)
- Separator tysięcy
- Prefiks i sufiks
- Precyzja
- Notacja naukowa.

Przykład: Sformatujmy kwotę zamówienia, dodając symbol waluty (np. symbol RMB ￥), separator tysięcy (np. przecinek lub kropka) oraz precyzję miejsc dziesiętnych, aby kwota była czytelna i łatwa do zrozumienia.
