# Wyświetlanie etykiet

## Wprowadzenie

Domyślnie, pola w bloku wyświetlają etykiety, które identyfikują ich znaczenie. Ułatwia to Państwu czytanie i rozumienie zawartości danych.

W niektórych scenariuszach (np. na stronach przeznaczonych tylko do wyświetlania lub gdy znaczenie pól jest już bardzo jasne), mogą Państwo ukryć etykiety, aby interfejs był bardziej przejrzysty. Po ukryciu etykiet wyświetlane są tylko wartości pól, a ich nazwy nie są już widoczne.
