## Wprowadzenie

### Operator

Proszę wybrać odpowiedni operator na podstawie typu pola, aby zwiększyć dokładność i efektywność filtrowania. Dla pól tekstowych domyślnie używane jest dopasowanie rozmyte (fuzzy matching).
