# Pola formularza

## Wprowadzenie

Różne typy pól w formularzu posiadają odmienne opcje konfiguracji. Dzięki wtyczkom użytkownicy mogą rozszerzać te opcje, aby sprostać bardziej spersonalizowanym potrzebom.

![20251025174555](https://static-docs.nocobase.com/20251025174555.png)

## Opcje konfiguracji pól

### Wartość domyślna

Mogą Państwo skonfigurować stałe lub zmienne jako wartości domyślne dla pól (funkcja dostępna tylko dla nowych formularzy).

![20251025174724](https://static-docs.nocobase.com/20251025174724.png)
