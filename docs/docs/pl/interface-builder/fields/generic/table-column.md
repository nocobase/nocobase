# Pola tabeli

## Wprowadzenie

Pola tabeli, oprócz możliwości dostosowania szerokości kolumn, tytułów pól, sortowania itp., dla specyficznych pól (takich jak pola daty, pola relacji i pola liczbowe) obsługują również bardziej spersonalizowane konfiguracje wyświetlania.

![20251024174558](https://static-docs.nocobase.com/20251024174558.png)

## Opcje konfiguracji pól

### Formatowanie pól daty

![20251024175303](https://static-docs.nocobase.com/20251024175303.png)

Więcej szczegółów znajdą Państwo w [Formatowanie daty](/interface-builder/fields/specific/date-picker)

### Formatowanie pól liczbowych

Obsługuje prostą konwersję jednostek, separatory tysięcy, prefiksy i sufiksy, precyzję oraz notację naukową.

![20251024175445](https://static-docs.nocobase.com/20251024175445.png)

Więcej szczegółów znajdą Państwo w [Formatowanie liczb](/interface-builder/fields/field-settings/number-format)

### Włącz szybką edycję

Po włączeniu szybkiej edycji, przycisk edycji pojawi się po najechaniu myszą na kolumnę. Kliknięcie go pozwoli szybko edytować i zapisać dane w wyskakującym oknie.

![20251025171158](https://static-docs.nocobase.com/20251025171158.gif)

### Włącz kliknięcie, aby otworzyć

Oprócz pól relacji, które obsługują otwieranie wyskakującego okna po kliknięciu, zwykłe pola również mogą mieć włączoną opcję "Kliknij, aby otworzyć", aby służyć jako punkt wejścia do otwierania wyskakującego okna.

![20251025172308](https://static-docs.nocobase.com/20251025172308.gif)

### Tryb wyświetlania przepełnienia treści

Gdy zawartość kolumny przekracza szerokość kolumny tabeli, mogą Państwo ustawić tryb przepełnienia:

-   Wielokropek (domyślnie)
-   Zawijanie tekstu

![20251025172549](https://static-docs.nocobase.com/20251025172549.png)

### Stała kolumna

![20251025170858](https://static-docs.nocobase.com/20251025170858.gif)

### Komponent pola

Niektóre pola obsługują wiele formatów wyświetlania. Użytkownicy mogą przełączać komponenty pól, aby uzyskać różne efekty wyświetlania, spełniając potrzeby różnych scenariuszy. Na przykład, pole typu **URL** można przełączyć na komponent **Podgląd** (Preview), aby lepiej wyświetlać zawartość linku lub podgląd obrazu.

![20251025171658](https://static-docs.nocobase.com/20251025171658.png)

### Styl wyświetlania

-   Etykieta
-   Tekst

![20251025172723](https://static-docs.nocobase.com/20251025172723.png)

### Możliwość sortowania

Obecnie większość typów pól obsługuje sortowanie po stronie serwera. Po włączeniu sortowania, dane mogą być sortowane rosnąco/malejąco na podstawie wybranego pola.
