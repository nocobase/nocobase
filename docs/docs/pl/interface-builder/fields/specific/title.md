# Tytuł

## Wprowadzenie

W trybie odczytu pola powiązań domyślnie są wyświetlane za pomocą **komponentu pola tytułowego**, który identyfikuje aktualnie powiązany rekord, wyświetlając wartość pola tytułowego. System automatycznie dopasowuje i renderuje odpowiedni komponent pola tytułowego w zależności od jego typu.
