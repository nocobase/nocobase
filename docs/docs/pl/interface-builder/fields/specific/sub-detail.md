# Szczegóły podrzędne

## Wprowadzenie

Szczegóły podrzędne to komponent, który odpowiada formularzowi podrzędnemu w trybie odczytu. W przeciwieństwie do komponentów Etykieta i Tytuł, Szczegóły podrzędne potrafią wyświetlać nie tylko więcej danych z bieżącej kolekcji, ale również mogą być skonfigurowane do prezentowania informacji z powiązanych kolekcji. Wielopoziomowe dane relacyjne są czytelnie przedstawiane w postaci zagnieżdżonych szczegółów.
