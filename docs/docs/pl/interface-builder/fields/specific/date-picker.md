## Wprowadzenie

## Gdzie można ustawić format daty

### Pola kolekcji

![20240417114311](https://static-docs.nocobase.com/20240417114311.png)

> Format daty ustawiony w polu kolekcji będzie służył jako domyślny format dla tego pola. Może on zostać nadpisany przez format daty z ustawień pola w konfiguracji bloku.
