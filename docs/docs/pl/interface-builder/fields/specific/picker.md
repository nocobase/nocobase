# Selektor danych

## Wprowadzenie

Selektor danych, dostępny w formie wyskakującego selektora tabeli, umożliwia powiązanie istniejących danych z docelowej kolekcji lub dodanie nowych danych do docelowej kolekcji, a następnie ich powiązanie.

![20251027203343](https://static-docs.nocobase.com/20251027203343.png)

Wybrane dane nie będą wyświetlane w selektorze tabeli.

![20251027203717](https://static-docs.nocobase.com/20251027203717.gif)

W selektorze tabeli mogą Państwo również zarządzać docelową kolekcją powiązań (dodawać, usuwać, importować/eksportować dane itp.).

![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
