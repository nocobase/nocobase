# Menedżer plików

## Wprowadzenie

Menedżer plików to komponent pola relacji, który umożliwia zarządzanie powiązaniami, w których docelową kolekcją jest kolekcja plików.

![20251027194124](https://static-docs.nocobase.com/20251027194124.png)

## Konfiguracja pola

- [Komponent pola](/interface-builder/fields/association-field): Przełącz na inne komponenty pola relacji, takie jak lista rozwijana czy selektor danych.
- Zezwalaj na wybór istniejących plików: Domyślnie włączone. Umożliwia powiązanie z już przesłanymi plikami.
