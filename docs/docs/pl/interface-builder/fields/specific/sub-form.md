# Podformularz

## Wprowadzenie

Podformularze są idealne w scenariuszach, gdzie najpierw tworzy się powiązane dane, a następnie je łączy. Wielopoziomowe dane powiązane są wyraźnie prezentowane w postaci zagnieżdżonych formularzy. W porównaniu do selektora danych i listy rozwijanej, podformularz pozwala na bezpośrednie zarządzanie polami powiązanej kolekcji w bieżącym bloku strony, a powiązane dane są przesyłane razem z formularzem głównym.
