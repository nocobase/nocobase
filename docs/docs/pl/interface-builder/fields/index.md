# Pole

W interfejsie użytkownika (UI) pole jest komponentem, który służy do przechowywania pojedynczych danych. Różne typy danych są prezentowane za pomocą różnych komponentów pól. Pola mogą być przypisane wyłącznie do bloków i nie mogą być używane samodzielnie.

## Pola w bloku

Pola zazwyczaj nie są używane samodzielnie, lecz istnieją jako elementy podrzędne bloków typu danych. Bloki typu danych zazwyczaj posiadają opcję „Konfiguruj pola”, a lista pól jest dostarczana przez bieżącą kolekcję.

![20251029204240](https://static-docs.nocobase.com/20251029204240.png)

## Projektant pól (Pasek narzędzi)

Podobnie jak bloki, komponent pola również posiada trzy ikony w prawym górnym rogu, są to:

- Przeciągnij, aby zmienić układ
- [Przepływ zdarzeń](/interface-builder/event-flow)
- Konfiguracja

![20251025165432](https://static-docs.nocobase.com/20251025165432.png)

## Układ pól

Mogą Państwo dostosować układ pól w bloku, przeciągając je i upuszczając.

![20251025165605](https://static-docs.nocobase.com/20251025165605.gif)

## Opcje konfiguracji pól

Oprócz ogólnych opcji konfiguracji pól, różne typy pól obsługują specyficzne opcje konfiguracji.
