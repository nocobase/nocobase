# Edytuj przycisk

## Wprowadzenie

System umożliwia dostosowanie nazwy, ikony, koloru tła i typu przycisku, oferując elastyczne opcje personalizacji interfejsu użytkownika, aby pomóc użytkownikom dostosować styl wyświetlania i funkcjonalność przycisku do ich konkretnych potrzeb.

-   **Nazwa przycisku**: Użytkownicy mogą dostosować tekst przycisku do swoich potrzeb biznesowych, aby etykieta przycisku była jasna i intuicyjna.
-   **Ikona przycisku**: Można ustawić ikonę dla przycisku, co zwiększa jego rozpoznawalność wizualną i pozwala użytkownikom szybko zidentyfikować jego funkcję.
-   **Akcja niebezpieczna (Danger Action)**: Gdy przycisk reprezentuje **akcję niebezpieczną** (np. usunięcie, resetowanie itp.), można ustawić jego typ na `Danger Action`.
-   **Typ przycisku (Button Type)**: Przyciski obsługują różne ustawienia typów. Typowe typy przycisków to:
    -   **Domyślny (Default)**: Standardowy przycisk, odpowiedni do zwykłych operacji.
    -   **Główny (Primary)**: Używany do ważnych lub głównych operacji, zazwyczaj z wyraźnym efektem wizualnym.
    -   **Przerywany (Dashed)**: Przycisk z przerywaną ramką, odpowiedni do operacji drugorzędnych lub przycisków informacyjnych.
    -   **Tekstowy (Text)**: Przycisk zawierający tylko tekst, bez koloru tła, zazwyczaj używany do lekkich operacji lub linków.
