# Dodaj rekord podrzędny

## Wprowadzenie

Ta operacja jest przeznaczona specjalnie dla **bloków tabel drzewiastych**, aby szybko dodawać rekordy podrzędne. Podczas dodawania rekordu podrzędnego, wybrany wiersz jest traktowany jako węzeł nadrzędny, co pozwala użytkownikom na łatwe tworzenie rekordów podrzędnych pod dowolnym węzłem nadrzędnym. Ta funkcja znacznie upraszcza proces zarządzania danymi hierarchicznymi, sprawiając, że rozszerzanie i organizowanie struktur danych staje się bardziej wydajne i intuicyjne.
