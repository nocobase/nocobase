# Edycja masowa

## Wprowadzenie

Edycja masowa jest odpowiednia dla scenariuszy wymagających elastycznych aktualizacji zbiorczych danych. Po kliknięciu przycisku edycji masowej mogą Państwo skonfigurować formularz edycji masowej w oknie wyskakującym i ustawić różne strategie aktualizacji dla każdego pola.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM.png)


## Konfiguracja akcji

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM%20(1).png)


## Instrukcja obsługi

### Konfiguracja formularza edycji masowej

1. Proszę dodać przycisk edycji masowej.

2. Proszę ustawić zakres edycji masowej: Wybrane / Wszystkie, domyślnie Wybrane.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_14_AM.png)

3. Proszę dodać formularz edycji masowej.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_14_AM.png)

4. Proszę skonfigurować pola wymagające edycji i dodać przycisk zatwierdzania.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM%20(1).png)

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM.png)

### Przesyłanie formularza

1. Proszę zaznaczyć wiersze danych, które mają zostać poddane edycji.

2. Proszę wybrać tryb edycji dla poszczególnych pól i wprowadzić wartości do przesłania.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_10_33_AM.png)

:::info{title=Dostępne tryby edycji}
* **Nie aktualizuj**: Pole pozostaje bez zmian.
* **Zmień na**: Aktualizuje pole do przesłanej wartości.
* **Wyczyść**: Usuwa dane z pola.

:::

3. Proszę przesłać formularz.
