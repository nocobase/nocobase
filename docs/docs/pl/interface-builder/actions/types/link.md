# Link

## Wprowadzenie

Akcja linku umożliwia nawigację po trasach (routing), wspiera przekazywanie zmiennych URL, dzięki czemu strona docelowa może dynamicznie dostosowywać swoją zawartość w oparciu o przekazane dane. Można ją skonfigurować w blokach danych.
