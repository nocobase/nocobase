# Filtrowanie

## Wprowadzenie

Operacja filtrowania zazwyczaj pojawia się w blokach danych. Służy do filtrowania danych poprzez konfigurację różnorodnych warunków. Należy pamiętać, że jeśli dla danego bloku skonfigurowano już zakres danych, operacja filtrowania zostanie połączona z warunkami zakresu danych, tworząc ostateczny warunek filtrowania.

## Ustawienia operacji

![20251029173708](https://static-docs.nocobase.com/20251029173708.png)

### Pola do filtrowania

Domyślnie wszystkie pola tej kolekcji są włączone. Tylko włączone pola mogą być wybrane z listy pól warunków.

Pola z powiązanych kolekcji również mogą być używane jako pola do filtrowania.
