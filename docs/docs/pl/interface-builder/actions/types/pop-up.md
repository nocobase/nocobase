# Otwórz okno modalne

## Wprowadzenie

Operacja **Otwórz okno modalne** jest podobna do operacji Podgląd i Edycja. Mogą Państwo skonfigurować przycisk **Otwórz okno modalne** w wierszach lub nagłówku tabeli. Kliknięcie przycisku spowoduje otwarcie odpowiedniego okna modalnego, a bloki w tym oknie wyświetlą dane na podstawie bieżącego rekordu kontekstowego.

![20251029182351](https://static-docs.nocobase.com/20251029182351.png)

Bloki w oknie modalnym konfiguruje się zgodnie z potrzebami.

![20251029190208](https://static-docs.nocobase.com/20251029190208.png)

## Ustawienia operacji

![20251029190352](https://static-docs.nocobase.com/20251029190352.png)

- [Edytuj przycisk](/interface-builder/actions/action-settings/edit-button): Edycja tytułu, typu i ikony przycisku.
- [Reguła powiązania](/interface-builder/linkage-rule): Dynamiczne sterowanie stanem przycisku (widoczny/wyłączony).
- [Edytuj okno modalne](/interface-builder/actions/action-settings/edit-popup): Ustawienie trybu otwierania (okno modalne lub wysuwany panel) i rozmiaru.
