# Prześlij

## Wprowadzenie

Akcja `Prześlij` służy do zapisywania danych formularza (jest to funkcja specyficzna dla bloków formularzy). Można ją również połączyć z przepływami pracy, aby zaimplementować zautomatyzowane procesy danych.

### Powiąż z przepływem pracy
