# Zarządzanie autentykatorami

## Zarządzanie uwierzytelnianiem użytkowników

Po zainstalowaniu wtyczki do uwierzytelniania użytkowników, system automatycznie zainicjuje metodę uwierzytelniania `hasłem`, opartą na nazwie użytkownika i adresie e-mail.

![](https://static-docs.nocobase.com/66eaa9d5421c9cb713b117366bd8a5d5.png)

## Aktywacja autentykatorów

![](https://static-docs.nocobase.com/7f1fb8f8ca5de67ffc68eff0a65848f5.png)

Tylko aktywne typy uwierzytelniania będą widoczne na stronie logowania.

![](https://static-docs.nocobase.com/8375a36ef98417af0f0977f1e07345dd.png)

## Typy uwierzytelniania użytkowników

![](https://static-docs.nocobase.com/da4250c0cea343ebe470cbf7be4b12e4.png)

Dodając różne typy autentykatorów, mogą Państwo włączyć w systemie odpowiednie metody uwierzytelniania.

Oprócz typów uwierzytelniania dostarczanych przez istniejące wtyczki, deweloperzy mogą również samodzielnie rozszerzać typy uwierzytelniania użytkowników. Szczegóły znajdą Państwo w [Przewodniku dla deweloperów](./dev/).
