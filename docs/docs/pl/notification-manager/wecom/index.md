# Powiadomienia: WeCom

## Wprowadzenie

Wtyczka **WeCom** umożliwia aplikacji wysyłanie powiadomień do użytkowników WeCom.

## Dodawanie i konfiguracja uwierzytelniacza WeCom

Najpierw należy dodać i skonfigurować uwierzytelniacz WeCom w NocoBase. Proszę zapoznać się z sekcją [Uwierzytelnianie użytkowników - WeCom](/auth-verification/auth-wecom). Tylko użytkownicy systemu, którzy zalogowali się za pośrednictwem WeCom, mogą otrzymywać powiadomienia systemowe przez WeCom.

## Dodawanie kanału powiadomień WeCom

![](https://static-docs.nocobase.com/202412041522365.png)

## Konfiguracja kanału powiadomień WeCom

Proszę wybrać uwierzytelniacz, który Państwo właśnie skonfigurowali.

![](https://static-docs.nocobase.com/202412041525284.png)

## Konfiguracja węzła powiadomień przepływu pracy

Proszę wybrać skonfigurowany kanał powiadomień WeCom. Obsługuje on trzy typy wiadomości: kartę tekstową, Markdown oraz kartę szablonu.

![](https://static-docs.nocobase.com/202412041529319.png)
