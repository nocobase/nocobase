# Zarządzanie plikami

## Wprowadzenie

Skonfiguruj sposób przechowywania plików przesyłanych podczas rozmów z pracownikami AI.

## Ustawienia

Proszę przejść do strony konfiguracji wtyczki pracownika AI, kliknąć zakładkę `Settings`, aby przejść do strony ustawień sposobu przechowywania.

![20251022002648](https://static-docs.nocobase.com/20251022002648.png)

Dostępne opcje przechowywania dla `Files storage` konfiguruje się w wtyczce Zarządzanie plikami.

![20251022002853](https://static-docs.nocobase.com/20251022002853.png)
