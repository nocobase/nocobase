# Wyszukiwanie w sieci

Wyszukiwanie w sieci służy do uzupełniania wiedzy modelu o najnowsze informacje spoza danych treningowych.

## Jak to działa

Dostępność wyszukiwania w sieci zależy od tego, czy usługa modelu wybrana dla bieżącej sesji obsługuje funkcję Web Search.

- **Obsługiwane:** wyświetlany jest przełącznik wyszukiwania w sieci, który można włączyć lub wyłączyć według potrzeb.
- **Nieobsługiwane:** przełącznik nie jest wyświetlany, a stan wyszukiwania zostaje automatycznie wyłączony.

## Użycie w czacie

Należy użyć przełącznika wyszukiwania w sieci w obszarze wprowadzania wiadomości:

- **Włączone:** AI może najpierw przeszukać sieć, a następnie udzielić odpowiedzi.

![web-search-switch-visible-on-supported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-visible-on-supported-model.png)

- **Wyłączone:** AI odpowiada wyłącznie na podstawie dostępnego kontekstu.

![web-search-switch-hidden-on-unsupported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-hidden-on-unsupported-model.png)

## Różnice między dostawcami

Możliwości obsługi wyszukiwania w sieci różnią się w zależności od modelu. Prosimy o korzystanie z funkcji zgodnie z rzeczywistymi możliwościami wybranej usługi i modelu.
