# Korzystanie z umiejętności

Umiejętności (Narzędzia) definiują to, co pracownik AI „może zrobić”.

## Struktura umiejętności

Strona umiejętności jest podzielona na trzy sekcje:

1. `General skills`: udostępniane wszystkim pracownikom AI, tylko do odczytu.
2. `Employee-specific skills`: przypisane do bieżącego pracownika, zazwyczaj tylko do odczytu.
3. `Custom skills`: umiejętności niestandardowe, które można dodawać/usuwać oraz konfigurować dla nich domyślne uprawnienia.

![skills-three-sections-general-specific-custom.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/skills-three-sections-general-specific-custom.png)

## Uprawnienia umiejętności

Uprawnienia umiejętności są ujednolicone jako:

- `Ask`: prośba o potwierdzenie przed wywołaniem.
- `Allow`: zezwolenie na bezpośrednie wywołanie.

Zalecenie: w przypadku umiejętności modyfikujących dane należy domyślnie używać opcji `Ask`.

![skills-permission-ask-allow-segmented.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/skills-permission-ask-allow-segmented.png)

## Dodawanie i zarządzanie

Aby dodać umiejętności, należy kliknąć przycisk `Add skill` w sekcji `Custom skills` i skonfigurować uprawnienia w zależności od ryzyka biznesowego.
