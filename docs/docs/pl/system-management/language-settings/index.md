# Ustawienia języka

W `Ustawieniach systemu` mogą Państwo dostosować globalne środowisko językowe systemu, konfigurując `Włączone języki`. Pierwszy język na liście będzie językiem domyślnym.

![20240324192005](https://static-docs.nocobase.com/20240324192005.png)

Gdy włączonych jest wiele języków, użytkownicy mogą ustawić swoje osobiste preferencje językowe w `Centrum osobistym` (w prawym górnym rogu strony).

![20240324192509](https://static-docs.nocobase.com/20240324192509.png)

Jeśli w używanym środowisku językowym brakuje tłumaczeń, mogą Państwo użyć wtyczki `[Zarządzanie lokalizacją](/system-management/localization/)`, aby je dodać lub uzupełnić.

![20240324193055](https://static-docs.nocobase.com/20240324193055.png)
