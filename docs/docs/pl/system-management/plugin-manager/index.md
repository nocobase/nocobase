# Zarządzanie wtyczkami

## Menedżer wtyczek

Aby przejść do strony menedżera wtyczek, proszę kliknąć ikonę menedżera wtyczek w prawym górnym rogu strony. Na tej stronie mogą Państwo przeglądać, instalować, włączać lub wyłączać wtyczki.

![2024-04-10_08-47-45](https://static-docs.nocobase.com/2024-04-10_08-47-45.png)

## Centrum ustawień wtyczek

Wtyczki zazwyczaj posiadają własne strony ustawień. Proszę kliknąć ikonę ustawień w prawym górnym rogu strony, aby wyświetlić menu ustawień dla wszystkich wtyczek. Kliknięcie odpowiedniej pozycji menu pozwoli Państwu skonfigurować daną wtyczkę.
