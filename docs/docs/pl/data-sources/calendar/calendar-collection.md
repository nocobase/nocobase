## Wprowadzenie

Kolekcja kalendarza to wyspecjalizowana tabela danych, przeznaczona do przechowywania dat i powiązanych z nimi informacji. Jest powszechnie wykorzystywana do zarządzania i śledzenia czasu w aplikacjach lub systemach.
