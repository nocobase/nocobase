# Selektor kolekcji

Pole Selektor kolekcji to specjalny typ pola, który umożliwia użytkownikom wybór jednej lub wielu kolekcji z predefiniowanej listy. Ten typ pola jest niezwykle przydatny w scenariuszach, gdzie potrzebne jest dynamiczne odwoływanie się do różnych kolekcji, na przykład podczas tworzenia konfigurowalnych raportów, pulpitów nawigacyjnych (dashboardów) lub złożonych aplikacji, które muszą zmieniać źródła danych w zależności od wyboru użytkownika.

## Przykłady użycia

### Dynamiczne generowanie raportów

W module generowania raportów użytkownicy mogą potrzebować wybrać kolekcję spośród wielu dostępnych jako źródło danych dla raportu. Używając Selektora kolekcji, użytkownicy mogą łatwo wybrać pożądaną kolekcję, a system dynamicznie wygeneruje raport w oparciu o ich wybór.

### Konfigurowalne pulpity nawigacyjne (dashboardy)

W aplikacji typu pulpit nawigacyjny (dashboard) użytkownicy mogą dostosowywać wyświetlane dane. Selektor kolekcji umożliwia użytkownikom wybór kolekcji, które ich interesują, co pozwala na wyświetlanie wykresów i danych statystycznych z różnych źródeł danych na pulpicie nawigacyjnym.

### Elastyczne narzędzia do importu/eksportu danych

Podczas tworzenia ogólnego narzędzia do importu lub eksportu danych, można użyć Selektora kolekcji, aby umożliwić użytkownikom określenie kolekcji, na której ma operować narzędzie. W ten sposób to samo narzędzie może być używane do przetwarzania danych z dowolnej kolekcji w systemie, co znacznie zwiększa jego użyteczność i możliwość ponownego wykorzystania.
