# Kolekcja plików

## Wprowadzenie

Pliki to rekordy w specjalnie ustrukturyzowanej kolekcji, znanej jako kolekcja plików. Służy ona do przechowywania metadanych plików i może być zarządzana za pomocą Menedżera plików.
