# Zewnętrzne źródło danych - PostgreSQL

## Wprowadzenie

Ten plugin pozwala na wykorzystanie zewnętrznej bazy danych PostgreSQL jako źródła danych. Obecnie wspierane wersje: PostgreSQL >= 9.5
