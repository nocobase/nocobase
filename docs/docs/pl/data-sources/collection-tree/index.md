# Kolekcja drzewiasta

## Wprowadzenie

Kolekcja drzewiasta to wzorzec projektowy kolekcji danych, który służy do organizacji danych o strukturze hierarchicznej. Jej struktura przypomina drzewo, gdzie każdy element danych może mieć jeden lub więcej elementów podrzędnych, a te z kolei mogą mieć własne elementy podrzędne.
