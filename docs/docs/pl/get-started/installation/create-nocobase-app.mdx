# Instalacja create-nocobase-app

:::tip Wymagania wstępne

- Zainstalowany Node.js 20+ oraz Yarn 1.22.x
- Skonfigurowana i uruchomiona jedna z wymaganych baz danych: MySQL 8.0.17+, MariaDB 10.9+ lub PostgreSQL 10+


:::

:::warning Zmienne środowiskowe

- Zmienna `TZ` służy do ustawienia strefy czasowej aplikacji, która domyślnie przyjmuje strefę czasową systemu operacyjnego.
- Zmienna `APP_KEY` to klucz tajny aplikacji, używany do generowania tokenów użytkownika itp. (Jeśli `APP_KEY` zostanie zmieniony, stare tokeny staną się nieważne). Może to być dowolny losowy ciąg znaków. **Proszę zmienić go na własny klucz tajny i upewnić się, że nie zostanie on ujawniony.**
- Zmienne `DB_*` to konfiguracje związane z bazą danych. Proszę je zmodyfikować, aby odpowiadały Państwa rzeczywistym informacjom połączeniowym z bazą danych.

:::

## 2. Zmiana katalogu

```bash
cd my-nocobase-app
```

## 3. Instalacja zależności

📢 W zależności od środowiska sieciowego, konfiguracji systemu i innych czynników, ten krok może zająć kilkanaście minut.

```bash
yarn install
```

:::tip Wskazówka dla środowiska produkcyjnego

Podczas wdrażania w środowisku produkcyjnym, aby zmniejszyć rozmiar, można zainstalować tylko niezbędne zależności:

```bash
yarn install --production
```

:::

## 4. Instalacja NocoBase

```bash
yarn nocobase install --lang=en-US
```

Proces instalacji automatycznie utworzy strukturę tabel bazy danych i zainicjuje dane.

## 5. Uruchamianie NocoBase

**Środowisko deweloperskie**

```bash
yarn dev
```

**Środowisko produkcyjne**

```bash
yarn start
```

## 6. Logowanie do NocoBase

Proszę otworzyć [http://localhost:13000](http://localhost:13000) w przeglądarce. Początkowe dane logowania to `admin@nocobase.com` i `admin123`.

:::warning Wskazówka dotycząca bezpieczeństwa konta

Po pierwszym zalogowaniu, proszę natychmiast zmienić domyślne hasło, aby zapewnić bezpieczeństwo systemu.

:::
