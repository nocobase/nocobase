# 概述

AI 开发是 NocoBase 面向开发者的 AI 辅助能力——你可以借助 AI 来生成插件代码、排查问题、编写配置，把重复性的开发工作交给 AI 处理。

如果你有一定的开发经验，AI 开发可以帮你加速这些场景：

- **插件脚手架**：描述插件功能，AI 生成基础代码结构
- **代码生成**：根据需求生成 Collection 定义、Action 处理逻辑、前端组件等
- **调试辅助**：把报错信息和上下文给 AI，快速定位问题

:::tip 提示

AI 开发生成的代码遵循 NocoBase 插件规范，可以直接放到插件目录中使用。不过生成的代码建议先review 一遍再上线。

:::
