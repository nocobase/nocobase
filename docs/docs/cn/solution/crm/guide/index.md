# 操作指南

> 本指南带你从零走完 CRM 的核心销售流程。建议按顺序阅读，每一章都会衔接前一章的操作结果。

## 阅读顺序

CRM 的核心业务链路是 **线索 → 客户/联系人 → 商机 → 报价 → 订单**。本指南围绕这条主线展开：

| 章节 | 内容 | 你会学到 |
|------|------|---------|
| [系统介绍与仪表盘](./guide-overview) | 系统概览、Analytics 数据分析、Overview 工作台 | 菜单结构、KPI 穿透、图表联动 |
| [线索管理](./guide-leads) | 线索的创建、筛选、AI 评分、跟进与转化 | 如何高效筛选高质量线索，一键转化为客户+商机 |
| [商机管理](./guide-opportunities) | 商机看板、阶段推进、报价创建与审批 | 如何推进销售漏斗，走完报价审批 |
| [产品、报价与订单](./guide-products-orders) | 产品目录、阶梯定价、报价明细、订单交付与回款 | 从产品维护到订单闭环 |
| [客户、联系人与邮件](./guide-customers-emails) | 客户 360 视图、健康评分、合并、联系人、邮件、AI 辅助 | 客户全生命周期 + 沟通管理 |
| [角色与权限](./guide-roles) | 角色体系、菜单可见性、数据操作范围 | 各岗位能看到什么、能操作什么 |
| AI 员工 | AI 按钮使用、场景示例 | 如何让 AI 辅助日常销售工作 |

## 前置条件

- 已安装并启动 CRM 2.0（参见[如何安装](../installation)）
- 使用管理员账号登录系统
