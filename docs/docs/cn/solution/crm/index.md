# NocoBase CRM 2.0 解决方案

> 基于 NocoBase 低代码平台的模块化销售管理系统，AI 员工辅助决策

## 1. 背景

### 销售团队面临的挑战

企业销售团队在日常运营中常遇到这些问题：线索质量参差不齐难以快速筛选、商机跟进容易遗漏、客户信息散落在邮件和各个系统中、销售预测全凭经验、报价审批流程不规范。

**典型场景：** 线索快速评估与分配、商机健康度监控、客户流失预警、报价多级审批、邮件与客户/商机关联。

### 目标用户

面向中小型至中大型企业的 B2B 销售、项目型销售、外贸销售团队。这些企业正在从 Excel/邮件管理升级到系统化管理，同时对客户数据安全有较高要求。

### 现有方案的不足

- **成本高**：Salesforce/HubSpot 按人头收费，中小企业难以承担
- **功能过载**：大型 CRM 功能繁多，学习成本高，实际用到的不足 20%
- **定制困难**：SaaS 系统难以适配企业自身业务流程，改个字段都要走流程
- **数据安全**：客户数据存储在第三方服务器，合规和安全风险高
- **自研成本高**：传统自研周期长、维护成本高，业务变化时难以快速调整

---

## 2. 差异化优势

**市面主流产品：** Salesforce、HubSpot、Zoho CRM、纷享销客、Odoo CRM、SuiteCRM 等。

**平台级优势：**

- **配置化优先**：数据模型、页面布局、业务流程均可通过 UI 配置，无需写代码
- **低代码快速搭建**：比自研更快，比 SaaS 更灵活
- **模块可拆解**：各模块独立设计，可按需裁剪，最小可用仅需客户+商机两个模块

**传统 CRM 做不到或成本极高的：**

- **数据主权**：自托管部署，客户数据存储在自有服务器，满足合规要求
- **AI 员工原生集成**：AI 员工深度嵌入业务页面，自动感知数据上下文，非"加个 AI 按钮"
- **所有设计可被复制**：用户可基于方案模板自行扩展，不依赖供应商

---

## 3. 设计原则

- **低认知成本**：界面简洁，核心功能一目了然
- **业务先于技术**：聚焦销售场景，而非技术炫耀
- **可演进**：支持分阶段上线，逐步完善
- **配置优先**：能配置的不写代码
- **人与 AI 协同**：AI 员工辅助决策，而非替代销售人员的判断

---

## 4. 解决方案概览

### 核心能力

- **全流程管理**：线索 → 商机 → 报价 → 订单 → 客户成功
- **模块可裁剪**：完整版 7 个模块，最小可用仅需 2 个核心模块
- **多币种支持**：CNY/USD/EUR/GBP/JPY 自动换算
- **AI 辅助**：线索评分、赢率预测、下一步行动建议

### 核心模块

| 模块 | 必需 | 说明 | AI 辅助 |
|------|:----:|------|--------|
| 客户管理 | ✅ | 客户档案、联系人、客户层级 | 健康度评估、流失预警 |
| 商机管理 | ✅ | 销售漏斗、阶段推进、活动记录 | 赢率预测、下一步建议 |
| 线索管理 | - | 线索录入、状态流转、转化跟踪 | 智能评分 |
| 报价管理 | - | 多币种、版本管理、审批流程 | - |
| 订单管理 | - | 订单生成、回款跟踪 | - |
| 产品管理 | - | 产品目录、分类、阶梯价格 | - |
| 邮件集成 | - | 邮件收发、CRM 关联 | 情感分析、摘要生成 |

### 方案裁剪

- **完整版**（全部 7 个模块）：流程完整的 B2B 销售团队
- **标准版**（客户 + 商机 + 报价 + 订单 + 产品）：中小企业销售管理
- **轻量版**（客户 + 商机）：简单的客户和商机跟踪
- **外贸版**（客户 + 商机 + 报价 + 邮件）：外贸型企业

## 5. AI 员工

CRM 系统预置了 5 名 AI 员工，深度嵌入业务页面。与普通 AI 聊天工具不同，他们能自动识别你当前正在查看的数据——无论是线索列表、商机详情还是邮件记录——无需手动复制粘贴，直接开始工作。

**使用方式**：点击页面右下角的 AI 悬浮球，或直接点击区块旁的 AI 图标，唤起对应员工。还可以为每名员工预设常用任务，下次只需一键点击即可触发。

| 员工 | 职责 | 在 CRM 中的典型用途 |
|------|------|-----------------|
| **Viz** | 洞察分析师 | 线索渠道分析、销售趋势、管线健康度 |
| **Ellis** | 邮件专家 | 起草跟进邮件、生成沟通摘要 |
| **Lexi** | 翻译助理 | 多语言邮件、外贸客户沟通 |
| **Dara** | 可视化专家 | 报表图表配置、仪表盘搭建 |
| **Orin** | 任务规划 | 每日优先事项、下一步行动建议 |

### AI 员工的业务价值

| 价值维度 | 具体效果 |
|----------|----------|
| 提升效率 | 线索评分自动完成，节省人工筛选；跟进邮件一键起草 |
| 赋能员工 | 销售数据分析随时触手可及，无需等待数据团队出报告 |
| 沟通提质 | 专业邮件 + AI 润色，外贸团队多语言无障碍沟通 |
| 决策支持 | 实时赢率判断和下一步建议，减少商机因遗漏跟进而流失 |

## 6. 亮点

**模块可拆解** — 各模块独立设计，可单独开关。最小配置仅需客户+商机两个核心模块，够用就好，不强制全上。

**完整销售闭环** — 线索→商机→报价→订单→回款→客户成功，全链路数据贯通，无需在多系统间切换。

**AI 员工原生集成** — 不是"加个 AI 按钮"，而是 5 名 AI 员工融入每个业务页面，自动获取当前数据上下文，一键触发分析与建议。

**邮件深度集成** — 邮件自动关联客户、联系人、商机，支持 Gmail、Outlook，多个英文邮件模板覆盖常见销售场景。

**多币种外贸支持** — 支持 CNY/USD/EUR/GBP/JPY，配置汇率换算，适合外贸和跨国销售团队。

## 7. 安装使用

使用 NocoBase 迁移管理功能，将 CRM 应用包一键迁移到目标环境。

**开箱即用：** 预置数据表、工作流、仪表盘，多角色视图（销售经理/销售代表/高管），37 个邮件模板覆盖常见销售场景。

## 8. 后续规划

- **商机自动化**：阶段推进触发通知，停滞商机自动预警，减少人工盯盘
- **审批流程**：报价多级审批工作流，支持移动端审批
- **AI 自动化**：AI 员工嵌入工作流，支持后台自动运行，实现无人值守的线索评分和商机分析
- **移动端适配**：移动端友好界面，随时随地跟进客户
- **多租户支持**：多空间/多应用横向扩展，分发给不同销售团队独立运营
