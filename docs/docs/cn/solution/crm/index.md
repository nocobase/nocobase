# CRM 解决方案

> **版本说明**：当前版本运行于 NocoBase 2.0 beta，但业务逻辑和数据表设计仍沿用 1.x 版本，仅对部分前端界面进行了改造，功能尚不完善，**仅供预览**。后续版本中，我们将对 CRM 方案进行彻底重构，届时将提供更完整的功能和更好的使用体验。

## 方案简介

NocoBase CRM 是一个无代码、灵活且强大的客户关系管理基座。开箱即用，提供极致的定制能力，且完全保障数据主权。你可以在此基础上任意扩展字段、区块、功能及页面，使其完全符合你的业务需求。

## 核心特点

### 完全无代码构建

无需编程技能，也能对 CRM 进行个性化改造。

### 涵盖 CRM 常用功能

- **客户管理**：客户信息、联系人、客户分级
- **销售管道**：线索管理、商机跟进、销售漏斗
- **订单管理**：报价单、订单、合同
- **沟通协作**：邮件沟通、评论、审批流程
- **数据分析**：仪表板、报表、数据可视化

### 可以任意修改且极易扩展

支持自定义字段、流程和业务逻辑，灵活的架构设计让"二次开发"变得简单。

### 开源，数据完全掌控

对系统代码和系统数据完全掌控，确保企业数据安全。

## 核心模块

| 模块 | 说明 |
|------|------|
| 线索管理 | 线索录入、线索分配、线索转化 |
| 客户管理 | 客户档案、联系人管理、客户分级 |
| 商机管理 | 商机跟进、销售阶段、赢单/丢单 |
| 订单管理 | 报价单、订单创建、合同管理 |
| 数据看板 | 销售漏斗、业绩统计、趋势分析 |
