# 设置验证规则

## 介绍

设置验证规则是为了确保用户输入的数据符合预期，除了预置的格式校验，支持用户自定义规则。


![20240411112215](https://static-docs.nocobase.com/20240411112215.png)

预置了多种常用格式校验（如数值、身份证号码、邮箱、手机号等）

![20240411112413](https://static-docs.nocobase.com/20240411112413.png)
