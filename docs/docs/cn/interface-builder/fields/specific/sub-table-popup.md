# 子表格（弹窗编辑）

## 介绍

子表格（弹窗编辑）用于在表单内管理对多关联数据（如一对多、多对多）。表格仅展示已关联记录，新增/编辑在弹窗中完成，数据会随主表单提交时统一入库。

## 使用说明

![20260212152204](https://static-docs.nocobase.com/20260212152204.png)

适用场景：

- 关联字段：o2m / m2m / mbm
- 典型用途：订单明细、子项列表、关联标签/成员等

## 字段配置项

### 允许选择已有数据（默认：开启）

支持从已有数据选择关联。

![20260212152312](https://static-docs.nocobase.com/20260212152312.png)

![20260212152343](https://static-docs.nocobase.com/20260212152343.gif)

### 字段组件

[字段组件](/interface-builder/fields/association-field)：切换为其他关系字段组件，如下拉选择、数据选择器等。

### 允许解除已有数据关联（默认：开启）

> 控制编辑表单中的已关联的数据是否允许解除已有数据的关联，新增的数据总是允许移除。

![20260212165752](https://static-docs.nocobase.com/20260212165752.gif)

### 允许新增（默认：开启）

控制是否显示添加按钮。当用户无当前目标表的 create 权限时，按钮会禁用并提示无权限。

### 允许快速编辑（默认：关闭）

开启后，鼠标移入单元格会出现编辑图标，可快速编辑单元格内容。

支持在关系字段组件上为所有字段开启快捷编辑。

![20260212165102](https://static-docs.nocobase.com/20260212165102.png)

也支持为单个列字段开启快捷编辑。

![20260212165025](https://static-docs.nocobase.com/20260212165025.png)

![20260212165201](https://static-docs.nocobase.com/20260212165201.gif)

### 分页大小（默认：10）

设置子表格每页显示的条数。

## 行为说明

- 选择已有记录时会按主键去重，避免同一记录重复关联
- 新增记录会直接回填到子表格，默认跳转到包含新增记录的分页
- 行内编辑仅修改当前行数据
- 移除只会解除当前表单中的关联，不删除源数据
- 数据随主表单提交时统一入库
