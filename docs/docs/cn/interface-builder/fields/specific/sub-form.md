# 子表单

## 介绍

子表单适用于先新建关系数据后关联数据的场景，多层级的关系数据以嵌套表单的形式清晰展示，与数据选择器和下拉选择器相比，子表单在当前页面区块上直接维护关系表字段，且关系数据随主表一起提交。
