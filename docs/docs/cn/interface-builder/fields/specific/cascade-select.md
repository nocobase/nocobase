# 级联选择

## 介绍

级联选择器适用于目标表为树表的关系字段，用户可以按照树表的层级结构选取数据，支持对数据的模糊搜索。

## 使用说明

- 关系为对一，级联为单选。

![20251125214656](https://static-docs.nocobase.com/20251125214656.png)

- 关系为对多，级联为多选.

![20251125215318](https://static-docs.nocobase.com/20251125215318.png)
