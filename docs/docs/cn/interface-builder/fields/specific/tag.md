# 标签

## 介绍

标签是关系字段阅读模式下的一种展示组件，需要配置标题字段和颜色字段。
## 使用说明
### 在表格中使用

订单表中有多对一关系字段「标签」。

标签表有「标签名称」和「标签颜色」两个字段。
