# 筛选

## 介绍

筛选操作通常出现在数据区块中，通过配置各种条件来过滤数据，需要注意的是如果区块已经配置了数据范围，筛选操作将与数据范围的条件合并作为最终过滤条件。

### 可筛选的字段

默认本表所有字段启用，只有启用的字段才能在条件字段列表中选到。

关系表的字段也可以作为筛选字段。
