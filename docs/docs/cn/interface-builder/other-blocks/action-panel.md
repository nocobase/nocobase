# 区块：操作面板

:::tip 提示
该功能由插件 `plugin-block-workbench` 提供。
:::

## 介绍

用于放置各种快捷操作，目前支持

- [链接](/handbook/ui/actions/types/link)
- [扫二维码](/handbook/action-qr-scan)
- [弹窗操作](/handbook/action-popup)
- [自定义请求](/handbook/action-custom-request)
