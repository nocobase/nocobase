# 树筛选

## 介绍

树筛选区块通过树形结构提供数据筛选能力，适用于具有层级关系的数据场景，例如商品类目、组织架构等。  
用户可以通过选择不同层级的节点，对关联的数据区块进行联动筛选。

## 如何使用

树筛选区块需要与数据区块配合使用，为其提供筛选能力。

配置步骤如下：

1. 开启配置模式，在页面中添加"树筛选"区块和一个数据区块（如"表格区块"）。
2. 配置树筛选区块，选择树形数据表（如商品类目表）。
3. 设置树筛选区块与数据区块的连接关系。
4. 配置完成后，点击树节点即可对数据区块进行筛选。

## 添加区块

在配置模式下，点击页面的"添加区块"按钮，在"筛选区块"分类中选择"树"即可完成添加。

## 区块配置项

![](https://static-docs.nocobase.com/Tree-filter-04-07-2026_03_12_PM%20(1).png)

### 连接数据区块

树筛选区块必须连接到数据区块才能生效。  
通过"连接数据区块"配置项，可以将树筛选与页面中的表格、列表、图表等区块建立联动关系，从而实现筛选功能。

![](https://static-docs.nocobase.com/Tree-filter-04-07-2026_03_14_PM.png)

### 标题字段

用于指定树节点显示的字段（即节点名称）。

### 搜索

开启后，支持通过关键词快速搜索并定位树节点，提高查找效率。

### 展开所有

控制页面初次加载时，是否默认展开所有树节点。

### 筛选子节点

开启后，选择某个节点时，将同时包含其所有子节点的数据进行筛选。  
适用于需要按父级分类查看全部子级数据的场景。
