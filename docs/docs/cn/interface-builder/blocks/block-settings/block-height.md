# 区块高度

## 介绍

区块高度支持三种模式：**默认高度**、**指定高度**、**全高**。大多数区块都支持高度设置。

![20260211154020](https://static-docs.nocobase.com/20260211154020.png)

## 高度模式

### 默认高度

不同类型区块的默认高度策略不同。例如表格与表单区块会根据内容自适应高度，
区块内部不会出现滚动条。

### 指定高度

可手动指定区块外框的总高度，区块内部会自动计算并分配可用高度。

![20260211154043](https://static-docs.nocobase.com/20260211154043.gif)

### 全高

全高模式与指定高度类似，区块高度会基于当前浏览器**可视区域**计算满屏的最大高度。
浏览器页面不出现滚动条，滚动条仅出现在区块内部。

不同区块在全高模式下的内部滚动处理略有差异：

- **表格**：`tbody` 内部滚动；
- **表单 / 详情**：Grid 内滚动（除操作区外的内容滚动）；
- **列表 / 网格卡片**：Grid 内滚动（除操作区和分页栏外的内容滚动）；
- **地图 / 日历**：整体自适应高度，无滚动条；
- **Iframe / Markdown**：限制区块外框总高度，滚动条出现在区块内部。
