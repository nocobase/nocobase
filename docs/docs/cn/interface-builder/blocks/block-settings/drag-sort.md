# 拖拽排序

## 介绍

拖拽排序依赖排序字段，用于对区块记录进行手动排序。


:::info{title=提示}
* 同一个排序字段用于拖拽排序时，多区块混用可能会破坏既有排序。
* 表格拖拽排序时，排序字段不能设置分组规则。
* 树表仅支持同层级节点排序。

:::

## 排序规则说明

假设当前排序为

```
[1,2,3,4,5,6,7,8,9]
```

当某个元素（例如 5）向前移动到 3 的位置时，只有 3、4、5 的排序值会变化：5 占用 3 的位置，3、4 各向后移动一个位置。

```
[1,2,5,3,4,6,7,8,9]
```

此时再将 6 向后移动到 8 的位置，6 占用 8 的位置，7、8 各向前移动一个位置。

```
[1,2,5,3,4,7,8,6,9]
```
