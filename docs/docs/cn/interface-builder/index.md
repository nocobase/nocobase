# 界面搭建

## 所见即所得

NocoBase 采用所见即所得的界面搭建模式。点击界面搭建按钮，可以在配置模式和使用模式之间切换。

## UI 布局

NocoBase 提供桌面端布局和移动端布局。桌面端布局适合常规管理操作，也能在手机中自动调整导航和页面内容；移动端布局提供独立的移动端导航和页面配置。详细介绍见 [UI 布局](./ui-layout/index.md)。
