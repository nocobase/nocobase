# 界面设计器

## 所见即所得

NocoBase 采用所见即所得的界面配置模式。点击界面配置按钮，可以在配置模式和使用模式之间切换。

## 布局模板

NocoBase 内置一个布局模板，顶部和左侧是导航区域，右侧是内容区域。

![2024-01-20_11-36-38](https://static-docs.nocobase.com/2024-01-20_11-36-38.jpg)
