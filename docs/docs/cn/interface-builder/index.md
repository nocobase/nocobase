# 界面构建器

## 所见即所得

NocoBase 采用所见即所得的界面配置模式。点击界面配置按钮，可以在配置模式和使用模式之间切换。

## 布局模板

NocoBase 内置一个布局模板，顶部和左侧是导航区域，右侧是内容区域。

![未命名.002](https://static-docs.nocobase.com/未命名.002.jpeg)
