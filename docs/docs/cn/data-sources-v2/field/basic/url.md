# URL

## 介绍

URL 字段，存储网页链接，支持格式校验。 创建字段时，建议根据业务含义、数据来源和页面交互方式选择该字段类型。

## 示例

比如，在业务数据表中使用「URL」字段记录对应信息；在页面区块中可以用于展示、筛选或表单录入，具体交互方式以字段配置和区块配置为准。
