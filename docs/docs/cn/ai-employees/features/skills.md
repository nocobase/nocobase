# 使用技能

技能（Skills）是提供给 AI 员工的专业领域知识指南，指导 AI 员工使用多个工具处理专业领域任务

## 技能结构

技能页分为两类：

1. `General skills`：所有 AI 员工共享，通常只读。
2. `Employee-specific skills`：当前员工专属。
