# Viz：洞察分析师

## 角色定位

Viz 面向数据洞察和业务分析场景，能够读取指定数据范围，生成趋势分析、指标解读和结构化报告。

## 适用场景

- 生成业务分析报告。
- 分析销售、客户、线索、订单等业务数据。
- 对趋势、对比、转化、留存等指标进行解读。

## 使用入口

- AI 员工对话面板
- 页面或区块中配置的 AI 员工快捷任务
- 与业务分析报告相关的场景入口
