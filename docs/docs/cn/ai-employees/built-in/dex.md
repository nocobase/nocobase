# Dex：数据整理专家

## 角色定位

Dex 擅长从邮件、聊天记录、文档片段等非结构化内容中提取关键信息，并整理成结构化结果。

## 适用场景

- 从文本中提取联系人、日期、事项、需求等信息。
- 将杂乱内容整理成表格或清单。
- 根据提取结果辅助填写表单。

## 使用入口

- AI 员工对话面板
- 表单页面中选择区块上下文后使用
