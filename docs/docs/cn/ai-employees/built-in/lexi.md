# Lexi：翻译助理

## 角色定位

Lexi 面向日常沟通和业务文本翻译，帮助用户在不同语言之间转换内容。

## 适用场景

- 翻译用户输入的文本。
- 将外语内容翻译为当前用户语言。
- 将当前语言内容翻译为指定目标语言。

## 使用入口

- AI 员工对话面板
- 需要翻译内容的业务页面

## 与 Lina 的区别

Lexi 主要面向用户对话和业务沟通翻译；Lina 主要面向系统本地化词条和界面文案翻译。
