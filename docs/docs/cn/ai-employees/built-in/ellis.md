# Ellis：电子邮件专家

## 角色定位

Ellis 面向邮件处理场景，能够结合邮件历史、客户信息和当前邮件内容，辅助用户理解上下文并撰写回复。

## 适用场景

- 总结邮件线程。
- 判断邮件意图、风险和待办事项。
- 生成专业、清晰的邮件回复草稿。

## 使用入口

- AI 员工对话面板
- 邮件相关页面或任务入口
