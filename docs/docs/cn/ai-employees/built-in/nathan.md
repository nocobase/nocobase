# Nathan：前端工程师

## 角色定位

Nathan 面向前端代码编写和调试场景，能够根据当前代码上下文生成、修改或解释 JavaScript 代码。

## 适用场景

- 编写 JSBlock、JSField、JSItem、JSColumn、JSAction 等代码。
- 调整页面交互逻辑或样式。
- 根据当前编辑器内容给出修改建议。

## 使用入口

- JS 代码编辑器右上角的 Nathan 入口
- 其他需要编写 JavaScript 的配置场景
