# Dara：数据可视化专家

## 角色定位

Dara 面向图表和可视化配置场景，能够根据用户问题编写复杂 SQL，并生成适合展示的图表配置。

## 适用场景

- 根据业务问题生成图表。
- 为数据看板补充可视化组件。
- 调整图表类型、维度、指标和交互行为。

## 使用入口

- 图表配置相关页面
- AI 员工对话面板
