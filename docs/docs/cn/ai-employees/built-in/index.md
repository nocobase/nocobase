# 内置 AI 员工

NocoBase 预置了一组面向不同场景的 AI 员工。它们可以协助完成数据分析、数据整理、邮件处理、翻译、本地化、前端开发等任务。

![](https://static-docs.nocobase.com/202605121048555.png)

## 员工列表

| 员工                  | 角色定位       | 典型场景                                 | 类型     |
| --------------------- | -------------- | ---------------------------------------- | -------- |
| [Atlas](./atlas.md)   | 团队领导       | 理解用户意图，协调合适的 AI 员工处理任务 | 主 Agent |
| [Viz](./viz.md)       | 洞察分析师     | 查询业务数据，生成分析结论和报告         | 业务员工 |
| [Dex](./dex.md)       | 数据整理专家   | 从文本中提取结构化信息，辅助填写表单     | 业务员工 |
| [Ellis](./ellis.md)   | 电子邮件专家   | 总结邮件上下文，辅助撰写回复             | 业务员工 |
| [Lexi](./lexi.md)     | 翻译助理       | 面向用户对话的多语言翻译                 | 业务员工 |
| [Vera](./vera.md)     | 研究分析师     | 联网搜索、信息核验和资料汇总             | 业务员工 |
| [Nathan](./nathan.md) | 前端工程师     | 在代码编辑器中辅助编写或修改前端代码     | 开发员工 |
| [Lina](./lina.md)     | 本地化工程师   | 翻译系统本地化词条和插件界面文案         | 开发员工 |
| [Dara](./dara.md)     | 数据可视化专家 | 根据数据生成图表和可视化配置             | 开发员工 |
