# 用户和权限

## 介绍

用户和权限模块是 NocoBase 的核心功能模块，提供了完整的用户管理、角色权限控制、部门管理以及用户数据同步等功能。通过这个模块，您可以构建一个完善的用户权限体系，确保系统的安全性和数据的合理访问控制。

## 主要功能模块

### [用户管理](/users-permissions/users/)

提供基础的用户模型和用户管理界面，包括：
- 用户的创建、编辑、删除
- 用户资料管理
- 密码管理和重置
- 用户状态管理

### [角色和权限](/users-permissions/acl/)

灵活强大的角色权限管理系统：
- 角色的创建和管理
- 基于角色的访问控制（RBAC）
- 数据权限控制
- 菜单权限配置
- API 权限管理

### [部门管理](/users-permissions/departments/)

组织架构和部门管理功能：
- 部门层级结构管理
- 部门成员管理
- 基于部门的权限控制
- 部门数据隔离

### [用户数据同步](/users-permissions/user-data-sync/)

支持从外部系统同步用户数据：
- 多种数据源支持（HTTP API、企业微信等）
- 定时同步配置
- 字段映射
- 同步日志和监控

## 快速开始

1. **配置用户管理**：进入「配置中心」-「用户管理」，添加和管理系统用户
2. **设置角色权限**：在「角色和权限」中创建角色并分配相应权限
3. **组织部门架构**：如需要，可以在「部门管理」中构建组织架构
4. **配置数据同步**：如有外部用户系统，可配置用户数据同步

## 扩展开发

用户和权限模块支持丰富的扩展能力：
- [扩展同步数据源](/users-permissions/user-data-sync/dev/source)
- [扩展同步目标资源](/users-permissions/user-data-sync/dev/resource)
- 自定义认证方式
- 自定义权限规则
