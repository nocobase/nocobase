# 多工作区

## 工作区概念

不同工作区有自己的 

- 布局：移动端、桌面端

- 菜单：可以自己组织自己的菜单页面

- 访问：设置不同的访问权限

## 多工作区插件

- 提供工作区管理

- 提供工作区切换

注：多应用只提供 sso
