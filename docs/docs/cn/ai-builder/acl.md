# 权限配置

:::tip 前置条件

阅读本页前，请确保你已按照 [AI 搭建快速开始](./index.md) 安装了 NocoBase CLI 并完成了初始化。

:::

## 简介

权限配置 Skill 用于通过自然语言管理 NocoBase 的角色、权限策略、用户绑定和 ACL 风险评估——你描述业务目标，它来选择命令和参数。

使用前需要确保 [NocoBase CLI](../get-started/nocobase-cli.md) 环境已就绪。


## 能力范围

- 创建角色并设置初始权限基线
- 切换全局角色模式（独立模式 / 联合模式）
- 给数据表配置动作权限和数据范围
- 批量绑定或解绑用户与角色的关系
- 输出角色级、用户级、系统级风险评估报告

### 场景 A：创建角色

```
帮我创建一个 sales 角色
```

## 常见问题

**配置了权限但不生效怎么办？**

先确认全局角色模式是否正确——如果用户同时拥有多个角色，联合模式和独立模式的行为差异很大，可以查看当前模式确认问题。
