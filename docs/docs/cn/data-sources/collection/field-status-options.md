# 状态和选项字段

## 介绍

状态和选项字段适合保存固定范围内的值。

## 章节目录

- [适用场景](#适用场景)
- [使用建议](#使用建议)
- [相关链接](#相关链接)

## 适用场景

| 你要保存 | 建议字段 | 适合场景 |
| --- | --- | --- |
| 是 / 否 | 勾选、开关 | 是否启用、是否归档、是否已付款、是否需要回访。 |
| 单个固定选项 | 下拉菜单、单选框 | 订单状态、客户等级、工单优先级、合同类型。 |
| 多个固定选项 | 多选框 | 客户标签、适用渠道、技能标签、产品特性。 |

## 使用建议

选项字段适合值集合稳定的场景。比如「订单状态」通常是固定流程，用单选更合适；「客户标签」可能同时有多个值，用多选更合适。

如果选项会频繁增删，或者每个选项还需要保存更多属性，建议单独建一张数据表，再用关系字段关联。
