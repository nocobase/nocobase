# 数字字段

## 介绍

数字字段适合保存数量、金额、评分、比例等数值。

## 章节目录

- [适用场景](#适用场景)
- [使用建议](#使用建议)
- [相关链接](#相关链接)

## 适用场景

| 你要保存 | 建议字段 | 适合场景 |
| --- | --- | --- |
| 整数 | 整数 | 数量、人数、库存、次数。 |
| 小数 | 数字 | 重量、面积、评分、用量、单价。 |
| 百分比 | 百分比 | 完成率、折扣率、毛利率、进度。 |
| 金额 | 金额 | 合同金额、订单金额、预算、费用。 |

## 使用建议

数字字段要提前想清楚精度。比如金额、比例、面积这类字段，通常不要用整数；库存数量、人员数量这类字段，通常用整数更直观。
