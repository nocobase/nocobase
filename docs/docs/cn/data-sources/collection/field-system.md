# 系统信息字段

## 介绍

系统信息字段由 NocoBase 或数据库维护，通常不作为普通业务字段手动创建。

## 章节目录

- [常见字段](#常见字段)
- [使用建议](#使用建议)
- [相关链接](#相关链接)

## 常见字段

| 字段 | 说明 | 常见用途 |
| --- | --- | --- |
| ID | 默认主键字段，用于唯一标识一条记录。 | 关系字段、API、权限、工作流。 |
| 创建时间 | 自动记录记录创建时间。 | 排序、筛选、审计。 |
| 创建人 | 自动记录创建记录的用户。 | 责任追踪、权限控制。 |
| 更新时间 | 自动记录最后更新时间。 | 审计、同步判断。 |
| 更新人 | 自动记录最后更新记录的用户。 | 协作追踪、审计。 |
| Table OID | PostgreSQL 相关系统字段。 | 继承表、底层表识别等特殊场景。 |

## 使用建议

普通业务表通常建议保留 ID、创建时间、创建人、更新时间、更新人这些预设字段。它们对后续权限、审计、工作流和数据排查都有帮助。
