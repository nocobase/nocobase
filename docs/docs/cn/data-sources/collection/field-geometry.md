# 几何图形字段

## 介绍

几何字段适合保存空间或地理信息。

## 章节目录

- [适用场景](#适用场景)
- [使用建议](#使用建议)
- [相关链接](#相关链接)

## 适用场景

| 你要保存 | 建议字段 | 适合场景 |
| --- | --- | --- |
| 点 | 点 | 门店位置、用户位置。 |
| 线 | 线 | 配送路线、巡检路线。 |
| 面 | 面 | 配送范围、地理围栏。 |
| 圆 | 圆 | 覆盖范围、影响范围。 |

## 使用建议

如果数据源没有可用的空间字段，或者只是从外部系统同步一段地理数据，也可以用 JSON 保存坐标、路径或范围数据。不过如果这些数据需要经常筛选、排序或在地图上展示，优先使用对应的几何图形字段。
