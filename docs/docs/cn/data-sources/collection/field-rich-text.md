# 富文本字段

## 介绍

富文本字段适合保存正文、说明文档、处理方案、代码片段等较复杂内容。

## 章节目录

- [适用场景](#适用场景)
- [使用建议](#使用建议)
- [相关链接](#相关链接)

## 适用场景

| 你要保存 | 建议字段 | 适合场景 |
| --- | --- | --- |
| Markdown 内容 | Markdown | 知识库正文、需求说明、技术文档、处理方案。 |
| 所见即所得 Markdown | Markdown(Vditor) | 需要实时预览、所见即所得编辑、上传图片或附件到对象存储的说明文档、文章正文。 |
| 富文本内容 | 富文本 | 公告、产品说明、带格式的处理记录。 |
| 代码片段 | Code | 脚本、SQL 片段、配置片段、技术备注。 |

## 使用建议

Markdown 字段适合保存普通 Markdown 文本。Markdown(Vditor) 提供更完整的编辑器体验，支持所见即所得、即时渲染、源码编辑等模式，也可以把编辑器中上传的文件保存到对象存储。

`Markdown(Vditor)` 和 `Code` 属于插件字段，启用对应插件后才会出现在字段类型中。如果只是普通备注，不需要一开始就用富文本或 Markdown，多行文本通常更简单。
