# 附件字段

## 介绍

附件字段适合上传文件，或者保存外部文件地址。

## 章节目录

- [适用场景](#适用场景)
- [使用建议](#使用建议)
- [相关链接](#相关链接)

## 适用场景

| 你要保存 | 建议字段 | 适合场景 |
| --- | --- | --- |
| 上传到 NocoBase 管理的文件 | 附件 | 合同附件、发票图片、员工证件、项目资料。 |
| 已经在外部系统中的文件地址 | 附件 URL | 对象存储、网盘、第三方系统中的文件链接。 |

## 使用建议

附件字段依赖文件表和文件存储引擎。用户上传文件后，文件本体由文件存储引擎保存，文件名、大小、MIME 类型、URL、存储位置等元信息会保存到文件表中。

如果文件本体不由 NocoBase 管理，只保存外部地址时，优先考虑附件 URL 或 URL 字段。
