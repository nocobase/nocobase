# 数字

## 介绍

在 NocoBase 中，**数字（Number）** 用于保存可以带小数的数值。

数字字段适合金额、重量、面积、评分、单价等业务数据。它可以参与筛选、排序、统计、公式和工作流条件。

如果值必须是整数，选择[整数](./integer.md)更直接。如果要展示为比例或百分比，选择[百分比](./percent.md)。

## 适用场景

数字适合这些业务场景：

- 订单金额、合同金额、单价
- 重量、面积、体积、距离
- 评分、系数、折扣前数值
- 需要参与统计或公式计算的小数

## 创建配置

在数据表的「Configure fields」页面中，点击「Add field」，选择「数字」可以创建数字字段。

![20240512175752](https://static-docs.nocobase.com/20240512175752.png)

| 配置 | 说明 |
| --- | --- |
| Field interface | 字段的界面类型。数字对应 `number`，决定页面中如何录入和展示。 |
| Field display name | 字段在界面中显示的名称，比如「订单金额」「评分」「重量」。建议使用业务人员能直接理解的名称。 |
| Field name | 字段标识名称，用于 API、关系字段、权限、工作流等内部引用。创建后通常不再修改，只支持字母、数字和下划线，并且必须以字母开头。 |
| Field type | 字段在数据层的类型。数字字段默认是 `double`，金额等需要精确小数的场景可以选择 `decimal`。 |
| Default value | 默认值。新增记录时，如果用户没有填写，可以自动带出默认值。 |
| Validation rules | 校验规则。可以限制最小值、最大值、精度或是否必填。 |
| Description | 字段说明。适合写字段含义、填写要求、数据来源或维护人。 |

:::warning 注意

字段名创建后会被页面区块、权限、工作流和 API 引用。创建前先确认命名，避免后续修改带来配置调整成本。

:::

## 字段特性

数字字段的默认行为如下：

| 特性 | 说明 |
| --- | --- |
| 默认 Field interface | `number`。 |
| 默认 Field type | `double`。 |
| 可选 Field type | `float`、`double`、`decimal`。 |
| 页面组件 | 编辑模式使用数字输入框。 |
| 筛选 | 支持等于、不等于、大于、小于、区间、为空、不为空等数值筛选。 |
| 排序 | 支持在表格区块中排序。 |
| 校验 | 支持数值范围、必填等校验。 |

## 编辑配置

创建后，点击字段右侧的「Edit」可以编辑数字字段配置。编辑字段主要用于调整字段在 NocoBase 中的展示和使用方式，比如修改显示名称、说明、默认值、校验规则或字段专属配置。

如果字段来自主数据库中已经同步的表，编辑时通常是在做字段映射——把数据库字段映射为 NocoBase 的 Field type 和 Field interface。

| 配置 | 允许编辑 | 说明 |
| --- | --- | --- |
| Field display name | 是 | 修改字段在界面中的显示名称，不改变字段标识名称。 |
| Field name | 否 | 字段标识名称创建后通常不能在编辑表单中修改。 |
| Field interface | 条件支持 | 主数据库字段或同步字段在字段映射时可以调整。调整后会影响页面输入、展示和校验方式。 |
| Field type | 条件支持 | 主数据库字段或同步字段在字段映射时可以调整。调整前需要确认已有数据能否按新类型使用。 |
| Default value | 是 | 调整新增记录时的默认值。 |
| Validation rules | 是 | 调整字段校验规则。 |
| Description | 是 | 补充字段含义、填写要求、数据来源或维护人。 |

:::warning 注意

切换 Field type 或 Field interface 不等于简单改一个显示名称。它会影响字段的存储方式、输入组件、校验规则、筛选条件和工作流变量使用方式。已有数据较多时，先确认数据格式是否匹配。

:::

## 删除字段

点击字段右侧的「Delete」可以删除数字字段。主数据库中还可以勾选多个字段后批量删除。

删除主数据库中新建的数字字段时，通常会同时删除数据库中的真实列及该列已有数据。删除从数据库同步或外部数据源映射出的字段时，影响范围取决于对应数据源和字段来源。

:::danger 警告

删除字段可能影响页面区块、表单、筛选、权限、工作流、API、导入导出和已有数据。删除前先确认字段是否仍被业务配置引用。

:::

## 页面配置使用

数字字段适合在录入、统计、图表和工作流判断中使用。
![20260709225103](https://static-docs.nocobase.com/20260709225103.png)

| 场景 | 用途 |
| --- | --- |
| 表单区块 | 录入金额、评分、重量等数值。 |
| 表格区块 | 展示、排序和筛选数值。 |
| 图表区块 | 按数值字段汇总、求和或平均。 |
| 公式字段 | 作为公式计算的输入字段。 |
