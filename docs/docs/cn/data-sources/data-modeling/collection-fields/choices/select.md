# 下拉单选

## 介绍

在 NocoBase 中，**下拉单选（Select）** 用于从一组选项中选择一个值。

下拉单选适合状态、等级、类型、来源等固定范围的业务字段。选项可以配置显示名称、选项值和颜色。

如果需要选择多个值，选择[下拉多选](./multiple-select.md)。如果只有是或否，选择[勾选](./checkbox.md)。

## 适用场景

下拉单选适合这些业务场景：

- 订单状态、工单状态、审批状态
- 客户等级、线索来源、优先级
- 项目类型、资产类别、合同类型
- 固定范围内只能选一个值的字段

## 创建配置

在数据表的「Configure fields」页面中，点击「Add field」，选择「下拉单选」可以创建下拉单选字段。

![20240512180203](https://static-docs.nocobase.com/20240512180203.png)

| 配置 | 说明 |
| --- | --- |
| Field interface | 字段的界面类型。下拉单选对应 `select`，决定页面中如何录入和展示。 |
| Field display name | 字段在界面中显示的名称，比如「订单状态」「客户等级」「优先级」。建议使用业务人员能直接理解的名称。 |
| Field name | 字段标识名称，用于 API、关系字段、权限、工作流等内部引用。创建后通常不再修改，只支持字母、数字和下划线，并且必须以字母开头。 |
| Field type | 字段在数据层的类型。下拉单选默认是 `string`，保存选中的选项值。 |
| Default value | 默认值。新增记录时，如果用户没有填写，可以自动带出默认值。 |
| Validation rules | 校验规则。通常配置必填，并维护选项值。 |
| Description | 字段说明。适合写字段含义、填写要求、数据来源或维护人。 |

:::warning 注意

字段名创建后会被页面区块、权限、工作流和 API 引用。创建前先确认命名，避免后续修改带来配置调整成本。

:::

## 字段特性

下拉单选字段的默认行为如下：

| 特性 | 说明 |
| --- | --- |
| 默认 Field interface | `select`。 |
| 默认 Field type | `string`。 |
| 可选 Field type | `string`。 |
| 页面组件 | 编辑模式使用下拉选择器。 |
| 筛选 | 支持按选项筛选。 |
| 排序 | 支持按选项值排序。 |
| 校验 | 支持必填和选项范围约束。 |

## 编辑配置

创建后，点击字段右侧的「Edit」可以编辑下拉单选字段配置。编辑字段主要用于调整字段在 NocoBase 中的展示和使用方式，比如修改显示名称、说明、默认值、校验规则或字段专属配置。

如果字段来自主数据库中已经同步的表，编辑时通常是在做字段映射——把数据库字段映射为 NocoBase 的 Field type 和 Field interface。

| 配置 | 允许编辑 | 说明 |
| --- | --- | --- |
| Field display name | 是 | 修改字段在界面中的显示名称，不改变字段标识名称。 |
| Field name | 否 | 字段标识名称创建后通常不能在编辑表单中修改。 |
| Field interface | 条件支持 | 主数据库字段或同步字段在字段映射时可以调整。调整后会影响页面输入、展示和校验方式。 |
| Field type | 条件支持 | 主数据库字段或同步字段在字段映射时可以调整。调整前需要确认已有数据能否按新类型使用。 |
| Default value | 是 | 调整新增记录时的默认值。 |
| Validation rules | 是 | 调整字段校验规则。 |
| Description | 是 | 补充字段含义、填写要求、数据来源或维护人。 |

:::warning 注意

切换 Field type 或 Field interface 不等于简单改一个显示名称。它会影响字段的存储方式、输入组件、校验规则、筛选条件和工作流变量使用方式。已有数据较多时，先确认数据格式是否匹配。

:::

## 删除字段

点击字段右侧的「Delete」可以删除下拉单选字段。主数据库中还可以勾选多个字段后批量删除。

删除主数据库中新建的下拉单选字段时，通常会同时删除数据库中的真实列及该列已有数据。删除从数据库同步或外部数据源映射出的字段时，影响范围取决于对应数据源和字段来源。

:::danger 警告

删除字段可能影响页面区块、表单、筛选、权限、工作流、API、导入导出和已有数据。删除前先确认字段是否仍被业务配置引用。

:::

## 页面配置使用

下拉单选字段适合在表单、表格、看板和筛选中使用。
![20260709225912](https://static-docs.nocobase.com/20260709225912.png)

| 场景 | 用途 |
| --- | --- |
| 表单区块 | 从下拉选项中选择一个值。 |
| 表格区块 | 以标签或文本展示选项。 |
| 看板区块 | 按状态、阶段等选项分组。 |
| 筛选区块 | 按选项快速筛选记录。 |
