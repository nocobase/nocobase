# 附件字段

## 介绍

系统内置"附件"类型的字段，用于在自定义数据表中支持用户上传文件。

附件字段底层是一个多对多关系字段，指向系统内置的一个文件表"附件"（`attachments`）。任意数据表创建附件字段后，会自动生成一个附件表的多对多关系中间表，上传的文件元数据会存储在"附件"表中，数据表中引用的文件信息会通过这个中间表进行关联。

### MIME 类型限制

用于限制允许上传的文件类型，使用 [MIME](https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/MIME_types) 语法描述格式。例如：`image/*` 代表图片类文件。多个类型可以用英文逗号分隔，如：`image/*,application/pdf` 代表允许图片类型和 PDF 类型的文件。

### 存储引擎

选择用于存储上传文件的存储引擎，不填则使用系统的默认存储引擎。
