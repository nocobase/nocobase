<PluginInfo commercial="true" name="data-source-external-mssql"></PluginInfo>

## 介绍

使用外部的 MSSQL 数据库作为数据源。目前支持的版本：SQL Server 2014-2019 

## 安装

该插件为商业插件，需要通过插件管理器上传并激活插件
