# 数据表字段：手写签名

## 介绍

手写签名字段让用户在画布上用鼠标或触摸屏书写签名。保存后，签名图片会写入所选的 **文件数据表**，并复用 **文件管理器** 提供的文件上传与存储流程。

## 安装

1. 确认当前环境为 **专业版及以上**，且授权有效。
2. 打开 **插件管理器**，找到 **数据表字段：手写签名**（`@nocobase/plugin-field-signature`）并启用。
3. 确保已启用 **文件管理器**（`@nocobase/plugin-file-manager`）。手写签名字段依赖它提供文件数据表、上传与存储能力；未启用时将无法保存签名图片。

## 使用说明

### 添加字段

在 **数据源** → 选择数据表 → **配置字段** → **添加字段** → 在多媒体分组中选择 **手写签名**。

### 字段配置

- **文件数据表**：必填；请选择一个用于保存文件的文件数据表（例如 `attachments`），签名图片会保存到这里。
- 签名图片实际使用的存储配置和上传规则，由所选文件数据表自身决定。

### 界面配置

- 将手写签名字段添加到表单后，可在字段的界面配置中调整 **签名设置**，包括笔触颜色、背景颜色、签名画布宽度、签名画布高度，以及缩略图宽度和缩略图高度。
- 在只读展示场景中，还可以调整签名缩略图的宽度和高度，以控制签名图片的展示尺寸。

### 界面操作

- 点击字段区域打开签名画布，书写完成后确认即可上传并关联对应的文件记录。
- 在小屏设备上可使用横屏/全屏类签名界面，便于书写。
