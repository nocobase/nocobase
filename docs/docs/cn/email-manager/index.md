# 邮件管理

## 介绍
邮件管理插件是一款高效、便捷的工具，支持 Gmail 和 Outlook 邮箱授权接入，帮助用户将邮件管理、邮件收发等能力集成到各区块和页面中。通过简单的授权配置，用户即可实现多账户统一管理，享受无缝的邮件通信体验。
