# nb proxy caddy status

查看当前 Caddy 代理的运行状态。

## 用法

```bash
nb proxy caddy status
```

## 输出

通常会输出这些信息：

- `driver`
- `status`
- `config`
- `caddy` 或 `container`

## 示例

```bash
nb proxy caddy status
```
