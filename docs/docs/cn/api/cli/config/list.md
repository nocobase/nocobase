# nb config list

列出当前显式设置过的 CLI 配置项。未显式设置的默认值不会显示。

## 用法

```bash
nb config list
```

## 示例

```bash
nb config list
```

## 说明

如果没有设置任何配置项，CLI 会输出 `No CLI config values are set.`。
