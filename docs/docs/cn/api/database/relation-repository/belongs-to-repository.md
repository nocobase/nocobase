## BelongsToRepository

其接口与 [HasOneRepository](./has-one-repository.md) 一致。
`BelongsToRepository` 是用于处理 `BelongsTo` 关系的 `Repository`，它提供了一些便捷的方法来处理 `BelongsTo` 关系。
