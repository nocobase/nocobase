# 操作ガイド

> 本ガイドでは、CRM のコア販売プロセスをゼロから順に解説します。各章は前章の操作結果を引き継いでいるため、順番に読み進めることをお勧めします。

## 読む順序

CRM のコアビジネスフローは **リード → 顧客/連絡先 → 商談 → 見積 → 注文** です。本ガイドはこのメインラインに沿って展開されます：

| 章 | 内容 | 学べること |
|------|------|---------|
| [システム紹介とダッシュボード](./guide-overview) | システム概要、Analytics データ分析、Overview ワークスペース | メニュー構造、KPI ドリルスルー、チャート連動 |
| [リード管理](./guide-leads) | リードの作成、フィルタリング、AI スコアリング、フォローアップとコンバージョン | 高品質なリードを効率的にフィルタリングし、ワンクリックで顧客 + 商談に変換する方法 |
| [商談管理](./guide-opportunities) | 商談カンバン、ステージ推進、見積作成と承認 | 販売ファネルの推進方法、見積承認の完了方法 |
| [製品、見積と注文](./guide-products-orders) | 製品カタログ、階段型価格設定、見積明細、注文の納品と回収 | 製品管理から注文クローズまで |
| [顧客、連絡先とメール](./guide-customers-emails) | 顧客 360 ビュー、ヘルススコア、マージ、連絡先、メール、AI アシスタント | 顧客の全ライフサイクル + コミュニケーション管理 |
| [ロールと権限](./guide-roles) | ロール体系、メニューの可視性、データ操作範囲 | 各ポジションが見られる範囲と操作できる範囲 |
| AI スタッフ | AI ボタンの使い方、シナリオ例 | AI を日常の営業業務に活用する方法 |

## 前提条件

- CRM 2.0 がインストール済みで起動していること（[インストール方法](../installation)を参照）
- 管理者アカウントでシステムにログインしていること
