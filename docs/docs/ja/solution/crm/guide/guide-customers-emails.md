# 顧客、連絡先とメール

> 顧客、連絡先、メールは密接に関連する 3 つのモジュールです。顧客は主体であり、連絡先はコミュニケーションの対象、メールはコミュニケーションの記録です。本章ではこれらを統合して紹介します。

![cn_04-customers-emails](https://yifan2-500g.oss-cn-hangzhou.aliyuncs.com/crm-guide-mermaid/cn_04-customers-emails.png)

## 顧客管理

上部メニューから **顧客** ページに入ります。2 つのタブがあります：顧客リスト と 顧客マージツール。

### 顧客リスト

リスト上部にフィルターボタンがあります：

| フィルター条件 | 説明 |
|---------|------|
| **All** | すべての顧客 |
| **Active** | アクティブな顧客 |
| **Potential** | 見込み顧客、まだ成約していない |
| **Dormant** | 休眠顧客、長期間やり取りがない |
| **Key Accounts** | 大口顧客/重点顧客 |
| **New This Month** | 今月新規追加 |


![04-customers-emails-2026-04-07-01-32-03](https://static-docs.nocobase.com/04-customers-emails-2026-04-07-01-32-03.png)


**主要カラム**：

- **AI ヘルススコア**：リングプログレスバー 0-100 点（🟢 70-100 健全 / 🟡 40-69 警告 / 🔴 0-39 危険）
- **最終活動**：相対時間 + カラーコーディング。連絡が途絶えている期間が長いほど色が濃くなります

### 顧客詳細

顧客名をクリックすると詳細ポップアップが開きます。**3 つのタブ**があります：

| タブ | 内容 |
|-------|------|
| **詳細** | 顧客プロフィール、統計カード、連絡先、商談、コメント |
| **メール** | この顧客のすべての連絡先とのメールのやり取り、5 つの AI ボタン |
| **変更履歴** | フィールドレベルの監査ログ |

**詳細タブ**は左 2/3 + 右 1/3 の 2 カラムレイアウトです：

- **左カラム**：顧客アバター（レベルに応じて色分け：Normal = グレー、Important = アンバー、VIP = ゴールド）、4 カラムサマリー（レベル/規模/地域/タイプ）、統計カード（累計成約金額/アクティブ商談数/今月のやり取り回数、リアルタイム API クエリ）、連絡先リスト、商談リスト、コメントエリア
- **右カラム**：AI スマートプロフィール（AI タグ、ヘルススコアリングチャート、離脱リスク、最適な連絡時間帯、コミュニケーション戦略）

![04-customers-emails-2026-04-07-01-33-47](https://static-docs.nocobase.com/04-customers-emails-2026-04-07-01-33-47.png)

### AI ヘルススコア

ヘルススコアは以下の次元を総合して自動計算されます：やり取り頻度、商談の活動状況、注文状況、連絡先カバレッジ。

活用のアドバイス：

1. 毎日顧客リストを開き、ヘルススコアでソートします
2. 赤色（Critical）の顧客を優先的にフォロー――離脱が進行中の可能性があります
3. 黄色（Warning）の顧客――軽度のフォローを手配します
4. 緑色（Healthy）の顧客――通常のペースでメンテナンスします

### 顧客マージ

重複する顧客レコードが発生した場合、マージツールでクリーンアップします：

1. **マージを開始**：顧客リストで複数の顧客を選択 → 「Customer Merge」ボタンをクリックします
2. **マージツールに入る**：2 番目のタブに切り替えて、マージリクエストリスト（Pending / Merged / Cancelled）を確認します
3. **マージを実行**：マスターレコードを選択 → フィールドごとに差異を比較 → プレビュー → 確認。バックエンドのワークフローがすべての関連データ（商談、連絡先、活動、コメント、注文、見積、共有）を自動的に移行し、マージされた顧客を無効化します

![04-customers-emails-2026-04-07-01-35-37](https://static-docs.nocobase.com/04-customers-emails-2026-04-07-01-35-37.png)

![04-customers-emails-2026-04-07-01-38-07](https://static-docs.nocobase.com/04-customers-emails-2026-04-07-01-38-07.png)

:::tip[マージ前に慎重に確認してください]
顧客マージは不可逆の操作です。実行前に、マスターレコードの選択とフィールド値の取捨選択を慎重に確認してください。
:::

## 連絡先管理

上部メニューから **設定 → 連絡先** ページに入ります。

### 連絡先情報

| フィールド | 説明 |
|------|------|
| Name | 連絡先の氏名 |
| Company | 所属企業（顧客レコードに関連付け） |
| Email | メールアドレス（メールの自動関連付けに使用） |
| Phone | 電話番号 |
| Role | ロールラベル |
| Level | 連絡先レベル |
| Primary Contact | この顧客の主要連絡先かどうか |

### ロールラベル

| ロール | 意味 |
|------|------|
| Decision Maker | 意思決定者 |
| Influencer | 影響者 |
| Technical | 技術責任者 |
| Procurement | 調達責任者 |

![04-customers-emails-2026-04-07-01-38-26](https://static-docs.nocobase.com/04-customers-emails-2026-04-07-01-38-26.png)

### 連絡先からメールを送信

連絡先の詳細ページを開くと、他のデータ管理と同様に、詳細、メール、フィールド記録などのタブが含まれています。

![04-customers-emails-2026-04-07-01-38-52](https://static-docs.nocobase.com/04-customers-emails-2026-04-07-01-38-52.png)

### メールと CRM の関連付け

メールは顧客、連絡先、商談に自動的に関連付けられます：

- 顧客詳細の「メール」タブ → その顧客のすべての連絡先とのメールのやり取り
- 連絡先詳細 → その連絡先の完全なメール履歴
- 商談詳細 → 関連するコミュニケーション記録

関連付けはビューを通じて、連絡先のメールアドレスに基づいて自動的にマッチングされます。

![04-customers-emails-2026-04-07-01-40-26](https://static-docs.nocobase.com/04-customers-emails-2026-04-07-01-40-26.png)

![04-customers-emails-2026-04-07-01-41-13](https://static-docs.nocobase.com/04-customers-emails-2026-04-07-01-41-13.png)

### AI メールアシスタント

メールページには 6 つの AI アシストシーンが用意されています：

| シーン | 機能 |
|------|------|
| **提案書ドラフト** | AI が顧客と商談のコンテキストに基づいて提案メールを生成します |
| **フォローアップメール** | AI が適切なトーンのフォローアップメールを生成します |
| **メール分析** | AI がメールの感情傾向と重要ポイントを分析します |
| **メール要約** | AI がメールスレッドの要約を生成します |
| **顧客コンテキスト** | AI が顧客の背景情報を集約します |
| **エグゼクティブブリーフィング** | AI がメールスレッドから重要情報を抽出してブリーフィングを生成します |

![04-customers-emails-2026-04-07-01-41-46](https://static-docs.nocobase.com/04-customers-emails-2026-04-07-01-41-46.png)

## 活動記録

上部メニューから **設定 → 活動** ページに入ります。これはすべての顧客とのやり取りの中央ログです。

| 活動タイプ | 説明 |
|---------|------|
| Meeting | 会議 |
| Call | 電話 |
| Email | メール |
| Visit | 訪問 |
| Note | メモ |
| Task | タスク |

![04-customers-emails-2026-04-07-01-42-20](https://static-docs.nocobase.com/04-customers-emails-2026-04-07-01-42-20.png)

活動記録は Overview ダッシュボードのカレンダービューにも表示されます。
