# 更新履歴

このページでは、CRM ソリューションの各バージョンの更新内容を記録しています。新しいバックアップがリリースされるたびに、このページの先頭にレコードが追加されます。

---

## 2026-04-05

**バックアップファイル**：
- [nocobase_crm_v2_backup_260406.nbdata](https://static-docs.nocobase.com/nocobase_crm_v2_backup_260406.nbdata)
- [nocobase_crm_v2_sql_260406.zip](https://static-docs.nocobase.com/nocobase_crm_v2_sql_260406.zip)

**更新内容**：
- テーブルの Summary 集計表示の異常を修正しました
- ロールと権限の設定を完了しました
- グローバル検索ページ（Lookup）を追加しました。1 つのページで複数テーブルのレコードを同時に検索できます
- インストール時に `plugin-disable-pm-add` が見つからないスクリプトエラーを修正しました
- リードのアサインボタンを追加しました
- 商談のフィルタークリック時のエラーを修正しました
