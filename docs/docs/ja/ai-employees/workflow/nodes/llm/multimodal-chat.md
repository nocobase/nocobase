# マルチモーダル会話

## 画像

モデルがサポートしている場合、LLMノードはモデルに画像を送信できます。使用する際は、変数を通じて添付ファイルフィールドを選択するか、関連するファイルコレクションのレコードを選択する必要があります。ファイルコレクションのレコードを選択する際、オブジェクトレベルで選択することも、URLフィールドを選択することも可能です。

![](https://static-docs.nocobase.com/202503041034858.png)

画像の送信形式には、2つのオプションがあります。

- URLで送信 - ローカルに保存されている画像を除き、すべての画像はURL形式で送信されます。ローカルに保存されている画像は、base64形式に変換されて送信されます。
- base64で送信 - ローカルに保存されているかクラウドに保存されているかに関わらず、すべての画像はbase64形式で送信されます。これは、画像のURLがオンラインLLMサービスから直接アクセスできない場合に適しています。

![](https://static-docs.nocobase.com/202503041200638.png)
