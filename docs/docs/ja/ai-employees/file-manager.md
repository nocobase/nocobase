# ファイル管理

## 概要

AI従業員との対話中にアップロードされたファイルのストレージ方式を設定します。

## 設定

AI従業員プラグインの設定ページに移動し、`Settings`タブをクリックしてストレージ方式の設定ページに入ります。


![20251022002648](https://static-docs.nocobase.com/20251022002648.png)


`Files storage`で選択可能なストレージは、ファイル管理プラグインで設定します。
