# 高度な機能

## はじめに

AI エージェントと会話する際、ファイルを送信するだけでなく、アプリケーションのコンテキスト情報を送ることもできます。これにより、AI エージェントはそのコンテキスト情報に基づいて返答できるようになります。

### ブロック情報の送信

AI エージェントのダイアログボックスで、左下隅にある `Add work context` ボタンをクリックし、`Pick block` を選択すると、アプリケーションページがブロック選択モードになります。

![20251022010148](https://static-docs.nocobase.com/20251022010148.png)

ブロックにマウスカーソルを合わせると、コンテキストとして送信できるブロックの色が変化します。

![20251022010422](https://static-docs.nocobase.com/20251022010422.png)

送信したいブロックを選択すると、そのブロックのコンテキスト情報がダイアログボックスに添付されます。

![20251022010543](https://static-docs.nocobase.com/20251022010543.png)

質問メッセージを入力したら、通常のメッセージ送信と同様に送信ボタンをクリックしてください。AI エージェントがブロック情報に基づいて返答します。

ブロック情報はメッセージリストにも表示されます。

![20251022010758](https://static-docs.nocobase.com/20251022010758.png)

### ブロック選択の終了

アプリケーションページがブロック選択モードになっているが、どのブロックも選択したくない場合は、下部にある「停止」アイコンをクリックするとブロック選択モードを終了できます。

![20251022011043](https://static-docs.nocobase.com/20251022011043.png)
