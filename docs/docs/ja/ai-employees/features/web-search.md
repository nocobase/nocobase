# Web 検索

Web 検索は、モデルの学習データに含まれていない最新情報を補完するために使用されます。

## 仕組み

Web 検索が利用可能かどうかは、現在のセッションで選択されているモデルサービスが Web Search をサポートしているかどうかに依存します。

- **サポートされている場合**: Web 検索のスイッチが表示され、必要に応じてオン/オフを切り替えることができます。
- **サポートされていない場合**: スイッチは表示されず、検索状態は自動的にオフになります。

## チャットでの利用

メッセージ入力エリアにある Web 検索スイッチを使用します。

- **オン**: AI は回答の前に情報を検索します。

![web-search-switch-visible-on-supported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-visible-on-supported-model.png)

- **オフ**: AI は既存のコンテキストのみに基づいて回答します。

![web-search-switch-hidden-on-unsupported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-hidden-on-unsupported-model.png)

## プラットフォームによる違い

モデルによって Web Search のサポート能力が異なります。実際の状況に合わせてご利用ください。
