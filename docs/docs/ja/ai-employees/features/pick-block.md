# コンテキストの追加 - ブロック

AIアシスタント（AI Employee）と対話する際、ファイルの送信に加えて、アプリケーションのコンテキスト情報を送信することができます。これにより、AIアシスタントはアプリケーションのコンテキスト情報に基づいて回答できるようになります。

### ブロック情報の送信

AIアシスタントのチャットボックスで、左下の `コンテキストを追加` ボタンをクリックし、`ブロックを選択` を選択します。すると、アプリケーションのページがブロック選択モードに切り替わります。

![20251022010148](https://static-docs.nocobase.com/20251022010148.png)

ブロックの上にマウスをホバーすると、コンテキストとして送信可能なブロックの色が変わります。

![20251022010422](https://static-docs.nocobase.com/20251022010422.png)

送信したいブロックを選択すると、そのブロックのコンテキスト情報がチャットボックスに添付されます。

![20251022010543](https://static-docs.nocobase.com/20251022010543.png)

質問を入力した後、通常のメッセージ送信と同様に送信ボタンをクリックすると、AIアシスタントはブロック情報に基づいて回答します。

ブロック情報はメッセージ履歴にも表示されます。

![20251022010758](https://static-docs.nocobase.com/20251022010758.png)

### ブロック選択の終了

アプリケーションがブロック選択モードのときに、ブロックを選択せずに終了したい場合は、画面下部の「停止」アイコンをクリックすることで、ブロック選択モードを解除できます。

![20251022011043](https://static-docs.nocobase.com/20251022011043.png)
