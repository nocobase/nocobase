# AI 従業員とのコラボレーション

AI 従業員を作成して有効化すると、ページ内で彼らとコラボレーションできるようになります。

## エントリポイント

主なエントリポイントは以下の 2 つです：

1. **右下のメインエントリ**：業務ページの右下から AI チャットパネルを呼び出します。一般的な質疑応答やブロックをまたいだコラボレーションに適しています。
2. **ブロックのアクションエントリ**：`Actions` をサポートするブロックで、`Actions -> AI employees` からアクセスします。現在のブロックに対するタスクの実行（例：JSBlock などのシナリオ）に適しています。

## チャットの基本操作

チャットパネルでは、メッセージの送信、添付ファイルのアップロード、履歴の表示、新しいチャットの作成、システムプロンプトの編集などの一般的な操作が可能です。

## チャット内での切り替え

入力エリアの下部で、以下の項目を直接切り替えることができます：

- `AI Employee Switcher`: 現在の従業員を切り替えます。
- `Model Switcher`: 現在のセッションモデルを切り替えます。

モデルの構成は従業員ごとに保存され、次回アクセス時に優先的に復元されます。
