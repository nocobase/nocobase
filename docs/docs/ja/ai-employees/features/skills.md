# スキルの使用

スキル（Skills）は AI 従業員に提供される専門領域のナレッジガイドで、AI 従業員が複数のツールを使用して専門分野のタスクを処理するよう導きます。

現在、スキルはカスタマイズに対応しておらず、システムにプリセットされたもののみが利用可能です。

## スキル構成

スキルページは 2 つのカテゴリに分かれています：

1. `General skills`：すべての AI 従業員で共有され、通常は読み取り専用です。
2. `Employee-specific skills`：現在の従業員専用のスキルです。

## スキル紹介

### 汎用スキル

| スキル名                 | 機能説明                                                                           |
| ------------------------ | ---------------------------------------------------------------------------------- |
| Data metadata            | システムのデータモデル、データテーブル、フィールド情報などのメタデータを取得し、AI 従業員がビジネスコンテキストを理解するのに役立ちます。           |
| Data query               | データテーブルのデータをクエリし、条件フィルタリングや集計クエリなどの機能をサポートして、AI 従業員がビジネスデータを取得するのに役立ちます。       |
| Business analysis report | ビジネスデータに基づいて分析レポートを生成し、多次元分析と可視化をサポートして、AI 従業員のビジネスインサイトに役立ちます。       |
| Document search          | プリセットされたドキュメントコンテンツを検索・閲覧し、AI 従業員がドキュメントベースの作業を完了するのに役立ちます。現在は主に JS コードの記述に使用されます。 |

### 専属スキル

| スキル名           | 機能説明                             | 所属従業員 |
| ------------------ | ------------------------------------ | -------- |
| Data modeling      | データモデリングスキル。ビジネスデータモデルの理解と構築 | Orin     |
| Frontend developer | フロントエンドブロックの JS コードの記述とテスト         | Nathan   |
