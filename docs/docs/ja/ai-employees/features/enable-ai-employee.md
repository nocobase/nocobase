# AI 従業員を有効化する

NocoBase には複数の場面向けの組み込み AI 従業員が用意されています。詳細は [組み込み AI 従業員](/ai-employees/built-in/) を参照してください。

## 有効化方法

`AI employees` 管理ページに入り、一覧で `Enabled` スイッチをオンにすると従業員を有効化できます。

![](https://static-docs.nocobase.com/202605121048555.png)

## 有効化後の使用

有効化後、その従業員は右下のショートカット入口や特定場面の入口など、利用可能な入口に表示されます。
