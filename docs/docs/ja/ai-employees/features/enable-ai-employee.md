# AI従業員の有効化

NocoBaseには、さまざまなシナリオに対応した複数のAI従業員がプリセットされています。

## 有効化の方法

`AI employees` 管理ページに移動し、リスト内の `Enabled` スイッチをオンにすることで、従業員を有効にできます。

![ai-employee-list-enable-switch.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employee-list-enable-switch.png)

## 有効化後の利用

有効化すると、その従業員は利用可能なエントリポイント（画面右下のショートカットや特定のシナリオ用エントリなど）に表示されるようになります。
