# MCP 接続

AI 従業員は Model Context Protocol (MCP) プロトコルに準拠した MCP サービスに接続できます。MCP サービスに接続すると、AI 従業員は MCP サービスが提供するツールを使用してタスクを完了できます。


## MCP 設定

MCP 設定モジュールに移動すると、新しい MCP サービスの追加や、接続済みの MCP サービスの管理ができます。

![20260323095943](https://static-docs.nocobase.com/20260323095943.png)


## MCP サービスの追加

MCP サービス一覧の右上にある`追加`ボタンをクリックし、ポップアップで MCP サービスの接続情報を入力して MCP サービスの追加を完了します。

Stdio と HTTP（Streamable / SSE）の 2 種類の MCP サービス転送プロトコルに対応しています。

![20260323100904](https://static-docs.nocobase.com/20260323100904.png)

MCP サービスを追加する際は、`名前`、`タイトル`、`説明`を入力する必要があります。`名前`は MCP サービスの一意識別子です。`タイトル`はシステム内で MCP サービスを表示するための名称です。`説明`はオプションで、MCP サービスが提供する機能を簡単に説明するために使用します。

![20260323101635](https://static-docs.nocobase.com/20260323101635.png)

### Stdio 

stdio 転送プロトコルに対応した MCP サービスを追加する場合は、MCP サービスを実行する`コマンド`と`コマンド引数`を入力する必要があります。必要に応じて、MCP サービスのコマンド実行に必要な`環境変数`を追加できます。

:::warning 注意
MCP サービスを実行する node、npx、uvx、go などのコマンドは、NocoBase をデプロイしたサーバー環境でサポートされている必要があります。

NocoBase の Docker イメージは node、npx などの Node.js 環境コマンドのみをサポートしています。
:::

![20260323103511](https://static-docs.nocobase.com/20260323103511.png)

### HTTP

HTTP 転送プロトコルに対応した MCP サービスを追加する場合は、MCP サービスの `URL` アドレスを入力する必要があります。必要に応じて`リクエストヘッダー`を追加できます。

HTTP 転送プロトコルは Streamable と SSE の 2 種類の転送方式に対応しています。Streamable は MCP 標準に新しく追加された転送方式で、SSE 転送方式は廃止予定です。使用する MCP サービスのドキュメントに従って、適切な転送方式を選択してください。

![20260323103906](https://static-docs.nocobase.com/20260323103906.png)

### 可用性テスト

MCP サービスの追加や編集時に、MCP 設定情報を入力した後、MCP サービスの可用性テストを実行できます。MCP 設定情報が正しく、MCP サービスが利用可能な場合、MCP サービスの可用性テスト成功のメッセージが返されます。

![20260323105608](https://static-docs.nocobase.com/20260323105608.png)

## MCP サービスの確認

MCP サービス一覧の`表示`ボタンをクリックすると、MCP サービスが提供するツール一覧を確認できます。

MCP サービスのツール一覧では、AI 従業員がそのツールを使用する際の権限設定も行えます。ツール権限を `Ask` に設定すると、呼び出し前に許可を求める確認が表示されます。`Allow` に設定すると、必要に応じてツールが直接呼び出されます。

![20260323111106](https://static-docs.nocobase.com/20260323111106.png)

## MCP サービスの使用

MCP 設定モジュールで使用したい MCP サービスを有効にすると、AI 従業員との会話中に、AI 従業員が MCP サービスの提供するツールを自動的に使用してタスクを完了します。

![20260323110535](https://static-docs.nocobase.com/20260323110535.png)
