# AI 従業員・Nathan：フロントエンドエンジニア

> フロントエンドコードの記述と最適化、複雑なインタラクションロジックの実装をサポートします。

`Nathan` は NocoBase のフロントエンド開発のエキスパートです。`JSBlock`、`JSField`、`JSItem`、`JSColumn`、`JSAction`、`Event Flow`、`Linkage` など、JavaScript の記述が必要な場面では、コードエディターの右上に `Nathan` のアバターが表示されます。そこから、コードエディター内のコードの記述や修正を依頼できます。

![20251022163405](https://static-docs.nocobase.com/20251022163405.png)

## 操作手順

コードエディターで `Nathan` をクリックすると AI 従業員ダイアログが開きます。コードエディター内のコードは自動的に入力ボックスに添付され、アプリケーションコンテキストとして `Nathan` に送信されます。

![20251022163935](https://static-docs.nocobase.com/20251022163935.png)

コーディングの要件を入力し、`Nathan` に送信して返信を待ちます。

![20251022164041](https://static-docs.nocobase.com/20251022164041.png)

`Nathan` の返信にあるコードブロックの `Apply to editor` ボタンをクリックすると、そのコードがコードエディターに上書きされます。

![20251022164323](https://static-docs.nocobase.com/20251022164323.png)

コードエディターの `Run` ボタンをクリックすると、リアルタイムで効果を確認できます。

![20251022164436](https://static-docs.nocobase.com/20251022164436.png)

## コード履歴

`Nathan` のダイアログボックス右上にある「コマンドライン」アイコンをクリックすると、現在のセッションで送信したコードスニペットと、`Nathan` が返信したコードスニペットを確認できます。
