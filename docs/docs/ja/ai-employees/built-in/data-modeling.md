# AI エージェント・Orin：データモデラー

> コレクションをスマートに設計し、データベース構造を最適化します。

`Orin` はデータモデリングの専門家です。メインのデータソース設定ページで、`Orin` にコレクションの作成や変更を手伝ってもらうことができます。

![20251022160628](https://static-docs.nocobase.com/20251022160628.png)

## 操作手順

データソース管理プラグインにアクセスし、メインのデータソースを設定します。

![20251022161146](https://static-docs.nocobase.com/20251022161146.png)

右上隅にある `Orin` のアバターをクリックして、AI エージェントのダイアログを開きます。

![20251022161641](https://static-docs.nocobase.com/20251022161641.png)

`Orin` にモデリングの要件を伝え、送信したら返信を待ちます。

![20251022161749](https://static-docs.nocobase.com/20251022161749.png)

`Orin` が要件を確認すると、スキルを使用してデータモデリングのプレビューを返信します。

プレビューを確認したら、`Finish review and apply` ボタンをクリックして、`Orin` のモデリングに従ってコレクションを作成します。

![20251022162142](https://static-docs.nocobase.com/20251022162142.png)
