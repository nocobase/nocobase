# 組み込み AI 従業員

NocoBase はさまざまなシナリオ向けに組み込み AI 従業員を提供しています。データ分析、データ整理、メール処理、翻訳、ローカライゼーション、フロントエンド開発などを支援します。

![](https://static-docs.nocobase.com/202605121048555.png)
