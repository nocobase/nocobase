# 組み込みAI従業員

## 概要

NocoBaseには、以下の組み込みAI従業員が用意されています。彼らは完全なスキル、ツール、知識ベースを備えており、LLMを設定するだけで、すぐに作業を開始できます。

- `Orin`: データモデリングエキスパート
- `Avery`: フォーム入力担当
- `Viz`: インサイトアナリスト
- `Lexi`: 翻訳者
- `Nathan`: フロントエンドコードエンジニア
- `Cole`: NocoBaseアシスタント
- `Vera`: リサーチアナリスト
- `Dex`: データオーガナイザー
- `Ellis`: メールエキスパート

## 有効化の方法

AI従業員プラグインの設定ページに移動し、`AI employees`タブをクリックしてAI従業員管理ページを開きます。

システムには複数の組み込みAI従業員が用意されていますが、初期状態ではどれも有効になっていません。このため、アプリケーションページでこれらのAI従業員と連携することはまだできません。

![20251022121248](https://static-docs.nocobase.com/20251022121248.png)

有効にしたい組み込みAI従業員を選択し、`Edit`ボタンをクリックしてAI従業員編集ページに進みます。

まず、`Profile`タブで`Enable`スイッチをオンにします。

![20251022121546](https://static-docs.nocobase.com/20251022121546.png)

次に、`Model settings`タブで、組み込みAI従業員にモデルを設定します。

- LLMサービス管理で作成したLLMサービスを選択します。
- 使用したい大規模モデルの名前を入力します。

![20251022121729](https://static-docs.nocobase.com/20251022121729.png)

### 有効化の完了

組み込みAI従業員にモデルを設定したら、`Submit`ボタンをクリックして変更を保存します。

その後、ページの右下にあるAI従業員クイック起動ボタンに、この組み込みAI従業員が表示されるようになります。

![20251022121951](https://static-docs.nocobase.com/20251022121951.png)

### 注意

組み込みAI従業員の中には、有効にしても右下のAI従業員リストに表示されないものもあります。例えば、Orinはメインデータ設定ページでのみ、NathanはJSエディターでのみ表示されます。
