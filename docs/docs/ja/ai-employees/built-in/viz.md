# AI従業員・Viz：インサイトアナリスト

> ワンクリックでグラフとインサイトを生成し、データに語らせましょう。

## 1. Vizとは

**Viz**は、NocoBaseに組み込まれた**AIインサイトアナリスト**です。
現在のページにあるデータ（リード、商談、アカウントなど）を読み取り、トレンドグラフ、比較グラフ、KPIカード、そして簡潔な結論を自動で生成します。これにより、ビジネス分析が簡単かつ直感的になります。

Vizは単なる冷たいレポートツールではありません。質問を理解し、ストーリーを語れるアナリストなのです。

> 💡 「最近の売上が下がったのはなぜだろう？」と知りたいですか？
> Vizに一言伝えるだけで、売上減少がどの段階で発生したのか、その原因として何が考えられるか、そして次にとるべき行動を教えてくれます。

## 2. Vizでできること

| 能力             | 説明                               | 例                     |
| :--------------- | :--------------------------------- | :--------------------- |
| 📊 **グラフの自動生成** | ワンクリックでデータを可視化。SQLの記述は不要です。 | 今月の売上トレンドを生成     |
| 🔍 **変化と異常の発見** | 上昇または下降の原因を分析します。         | 今月は先月と比べてどこが良かった？ |
| 🧭 **意思決定の支援** | データに基づいた実行可能な提案を提供します。 | どのチャネルに予算を増やすべきか？ |
| 🧩 **データ視点の集約** | 地域、製品、ソースなど、多角的な視点で比較します。 | 各地域の収益比較を表示       |

毎月の経営状況の確認、チャネルROI、セールスファネルなど、Vizは数秒でグラフと分析結果を生成できます。

## 3. 利用方法

### 3.1 ページからのアクセス方法

*   **右上ボタン（推奨）**
    リード、商談、アカウントなどのページの右上にある**Vizアイコン**をクリックすると、以下のようなプリセットタスクを選択できます。

    *   ステージ変換とトレンド
    *   ソースチャネルの比較
    *   月次レビュー分析

    ![Leadsページ例](https://static-docs.nocobase.com/00_QuickStart_cn-2025-09-29-16-46-00.png)

*   **右下グローバルパネル**
    どのページにいても、グローバルAIパネルを呼び出し、Vizに直接話しかけることができます。

    ```
    Analyze sales changes in the last 90 days
    ```

    Vizは、現在表示しているページのデータコンテキストを自動的に取り込みます。

### 3.2 Vizとの対話方法

Vizは自然言語での質問に対応しており、複数回の追加質問も理解します。
例：

```
Hi Viz，生成本月线索趋势。
```

```
只看第三方渠道的表现。
```

```
那哪个地区的增长最快？
```

追加の質問をするたびに、前回の分析結果に基づいてさらに深く掘り下げてくれます。データの条件を繰り返し入力する必要はありません。

## 4. 一般的な分析シナリオ

| シナリオ           | 知りたいこと                       | Vizの出力                       |
| :--------------- | :--------------------------------- | :------------------------------ |
| **月次レビュー**   | 今月は先月と比べてどこが良かった？         | KPIカード + トレンドグラフ + 3つの改善提案 |
| **成長要因の分解** | 収益増加は量によるものか、価格によるものか？ | 要因分解グラフ + 比較表             |
| **チャネル分析**   | どのチャネルに継続投資すべきか？           | ROIグラフ + リテンションカーブ + 提案 |
| **ファネル分析**   | トラフィックはどの段階で停滞しているか？   | ファネルグラフ + ボトルネックの説明   |
| **顧客維持**       | どの顧客が最も価値があるか？               | RFMセグメンテーショングラフ + リテンションカーブ |
| **プロモーション評価** | 大規模プロモーションの効果は？             | 比較グラフ + 価格弾力性分析         |

> 📈 すべてのグラフは、有効なECharts形式で生成されます。各グラフには1つの主要なポイントがあり、簡潔な結論が添えられています。
> データが不足している場合、Vizは結果を捏造することなく、その旨を直接伝えます。

## 5. Vizとの対話のヒント

| 方法                 | 効果                                   |
| :------------------- | :------------------------------------- |
| ✅ **期間を指定する**     | 「過去30日間」「先月と今月」のように指定すると、より正確になります。 |
| ✅ **ディメンションを指定する** | 「地域別」「チャネル別」「製品別」のように指定すると、視点を合わせやすくなります。 |
| ✅ **詳細ではなくトレンドに注目する** | Vizは変化の方向性や主要な原因を見つけるのが得意です。       |
| ✅ **自然言語で話す**     | 命令形を使う必要はありません。チャットするように質問してください。 |

## 6. Vizはどのような方におすすめか

| 役割                 | 用途                               |
| :------------------- | :--------------------------------- |
| **セールスマネージャー** | ステージ変換率、チャネルパフォーマンス、チームの業績を確認 |
| **マーケター**       | 広告ROI、プロモーション効果、顧客維持を分析 |
| **オペレーションアナリスト** | 迅速なデータ抽出、異常の発見、仮説の検証   |
| **経営層**           | 経営状況を一目で把握し、意思決定のヒントを得る |

## 7. 利用のヒント

1.  **プリセットタスクから始める**
    公式デモには一般的なタスクが組み込まれており、プロンプトなしで直接効果を体験できます。
    例：リードページ → **Viz → ステージ変換とトレンド**をクリック

2.  **出力スタイルを観察する**
    各分析ポイントには、独立したグラフと簡潔な説明が付いています。
    明確なグラフと簡潔なテキストが、Vizの標準的な出力です。

3.  **段階的に質問を深める**
    分析レポートを読んだ後、「なぜ？」「どう改善すればいい？」とさらに質問を続けると、Vizが自動的にフォローアップします。

## 8. まとめ

*   Viz = あなたのデータインサイトアシスタント
*   SQLの記述やグラフの設定は不要です

*   すべての結論は実際のデータに基づいており、明確で信頼できます
