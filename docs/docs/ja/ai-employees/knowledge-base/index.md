# 概要

## はじめに

AIナレッジベースプラグインは、AIエージェントにRAG検索機能を提供します。

RAG検索機能により、AIエージェントはユーザーの質問に回答する際に、より正確で専門的、かつ企業内部との関連性が高い回答を提供できるようになります。

管理者がメンテナンスするナレッジベースから提供される専門分野や企業内部のドキュメントを活用することで、AIエージェントの回答の正確性と追跡可能性を向上させます。

### RAGとは

RAG（Retrieval Augmented Generation）は、「検索拡張生成」を意味します。

- 検索：ユーザーの質問は、Embeddingモデル（例：BERT）によってベクトルに変換され、ベクトルライブラリから密な検索（セマンティック類似度）または疎な検索（キーワードマッチング）を通じて、関連性の高いTop-K個のテキストチャンクが呼び出されます。
- 拡張：検索結果は元の質問と結合され、拡張プロンプト（Prompt）としてLLMのコンテキストウィンドウに注入されます。
- 生成：LLMは拡張プロンプトを組み合わせて最終的な回答を生成し、事実性と追跡可能性を保証します。
