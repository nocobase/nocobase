# RAG 検索

## はじめに

知識ベースを設定したら、AI従業員の設定でRAG機能を有効にできます。

RAGを有効にすると、ユーザーがAI従業員と会話する際に、AI従業員はRAG検索を利用し、ユーザーのメッセージに基づいて知識ベースからドキュメントを取得し、取得したドキュメントに基づいて返信します。

## RAGを有効にする

AI従業員プラグインの設定ページに移動し、`AI employees` タブをクリックして、AI従業員管理ページに入ります。

![20251023010811](https://static-docs.nocobase.com/20251023010811.png)

RAGを有効にしたいAI従業員を選択し、`Edit` ボタンをクリックして、AI従業員編集ページに入ります。

`Knowledge base` タブで、`Enable` スイッチをオンにします。

- `Knowledge Base Prompt` に、知識ベースを参照するためのプロンプトを入力します。`{knowledgeBaseData}` は固定のプレースホルダーですので、変更しないでください。
- `Knowledge Base` で、設定済みの知識ベースを選択します。参照：[知識ベース](/ai-employees/knowledge-base/knowledge-base)
- `Top K` 入力ボックスに、検索するドキュメントの数を入力します。デフォルトは3です。
- `Score` 入力ボックスに、検索時のドキュメント関連性しきい値を入力します。

`Submit` ボタンをクリックして、AI従業員の設定を保存します。
