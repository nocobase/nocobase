# ファイルプレビュー

ファイルフィールド（添付ファイルフィールドを含む）が含まれるインターフェースでは、ファイルのサムネイルやアイコンをクリックすることでファイルをプレビューできます。組み込みのプレビュー機能は、画像、PDF、およびほとんどのブラウザがネイティブにサポートしているファイル形式など、さまざまなファイルタイプに対応しています。

![20251129232307](https://static-docs.nocobase.com/20251129232307.png)

ネイティブプレビューに対応していないファイルタイプについては、対応するファイルプレビュープラグインをインストールまたは拡張することで、プレビュー機能を実現できます。例えば、Officeファイルプレビュープラグインをインストールすると、Word、Excel、PowerPointファイルをプレビューできるようになります。

現在、NocoBaseが提供しているファイルプレビュープラグイン：

* [Officeファイルプレビュープラグイン](../file-preview/ms-office.md)
