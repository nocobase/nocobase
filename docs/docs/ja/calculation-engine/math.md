| **isPrime** | 入力が素数かどうかをチェックします。 | `isPrime(7)` |  | `true` |
| **isZero** | 入力が 0 かどうかをチェックします。 | `isZero(0)` |  | `true` |
| **numeric** | 入力を数値型（number、BigNumber など）に変換します。 | `numeric('123')` |  | `123` |
| **typeOf** | 入力値の型名を返します。 | `typeOf([1, 2, 3])` |  | `Array` |
