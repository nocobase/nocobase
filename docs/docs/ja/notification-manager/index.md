# 通知マネージャー

## はじめに

通知マネージャーは、複数の通知チャネルを統合した一元的なサービスです。チャネル設定、送信管理、ログ記録を統一的に提供し、柔軟な拡張にも対応しています。

![20240928112556](https://static-docs.nocobase.com/20240928112556.png)

- **紫色の部分**: 通知マネージャーは、チャネル設定やログ記録などの機能を含む統合的な管理サービスを提供し、通知チャネルの拡張も可能です。
- **緑色の部分**: アプリ内メッセージ（In-App Message）は、組み込みチャネルとして、ユーザーがアプリケーション内で直接メッセージ通知を受け取れるようにします。
- **赤色の部分**: メール（Email）は、拡張可能なチャネルとして、ユーザーがメールで通知を受け取れるようにします。

## チャネル管理

![20240928181752](https://static-docs.nocobase.com/20240928181752.png)

現在サポートされているチャネルは以下の通りです。

- [アプリ内メッセージ](/notification-manager/notification-in-app-message)
- [メール（Email）](/notification-manager/notification-email)（組み込みのSMTP転送方式を使用）

さらに多くのチャネル通知を拡張することも可能です。[チャネル拡張](/notification-manager/development/extension)ドキュメントをご参照ください。

## 通知ログ

各通知の送信詳細とステータスを詳細に記録することで、分析やトラブルシューティングに役立てることができます。
