# メールエイリアス

## 機能説明

メールエイリアスは、同一のメールアカウントで異なる差出人アイデンティティを使用してメールを送信する機能です。

メール送信時に、差出人セレクターからメインメールアドレスまたは同期済みのエイリアスアドレスを選択できます。返信、転送、下書きの復元時には、元の差出人アイデンティティが保持されます。

![](https://static-docs.nocobase.com/%E5%8F%91%E9%80%81%E9%82%AE%E4%BB%B6-04-02-2026_06_02_PM.png)

> Outlook はこの機能をサポートしていません。

## エイリアスの同期

Gmail アカウントの認証が成功すると、システムはそのメールアカウントで利用可能なエイリアスを自動的に同期します。

その後 Gmail でエイリアスを追加または変更した場合は、メール設定に移動し、「差出人名」設定内の「エイリアスを同期」をクリックして再取得できます。

![](https://static-docs.nocobase.com/Email-settings-04-02-2026_06_04_PM.png)

## 送信時のエイリアス選択

メールエディターで差出人セレクターをクリックすると、メインメールアドレスとそのアカウントに同期済みのエイリアスが表示されます。

同一のエイリアスメールアドレスが複数のアカウントに関連付けられている場合、セレクターには所属するメインメールアドレスが付記され、どのアカウントのコンテキストで使用されるかを区別しやすくなります。

![](https://static-docs.nocobase.com/%E5%8F%91%E9%80%81%E9%82%AE%E4%BB%B6-04-02-2026_06_02_PM.png)
