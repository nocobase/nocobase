# メール管理

## はじめに
メール管理プラグインは、効率的で便利なツールです。Gmail や Outlook のメールボックスへの認証接続に対応しており、ユーザーがメール管理やメールの送受信といった機能を、さまざまなブロックやページに統合できるように支援します。簡単な認証設定を行うことで、複数のアカウントを一元管理し、シームレスなメール通信体験を享受できます。
