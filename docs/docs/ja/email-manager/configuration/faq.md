# FAQ

## Microsoftアカウントで認証ログイン後、メールが受信できません

A: 現在、認証ログインに対応しているのは、OutlookメールアカウントとGmailメールアカウントのみです。MicrosoftアカウントとGoogleアカウントは現在サポートされていません。詳細については、以下をご参照ください：[answers.microsoft.com](https://answers.microsoft.com/zh-hans/outlook_com/forum/all/%E7%8E%B0%E6%9C%89%E5%BE%AE%E8%BD%AF%E8%B4%A6/dba12dda-a7c7-4346-8263-53f4a6d9dc68)

**ヒント**：ご自身が「本物のOutlook.comメール」または「Gmailメール」をお持ちかどうか不明な場合は、ウェブブラウザでOutlook.comまたはGmail.comにアクセスし、直接ログインして通常通りメールを送信できるか試してみてください。もしできない場合は、対応するメールサービスをお持ちでない可能性があり、まず開通するか、別のメールをご利用いただく必要があります。

## 認証の取り消し

メールアカウントがすでに認証済みで、その認証を削除または再認証したい場合は、以下の手順に従ってください。

#### **Gmail**

1. https://myaccount.google.com/u/0/connections を開き、ログインします。

![](https://static-docs.nocobase.com/mail-1735634131347.png)

2. 該当するアプリをクリックし、次に「削除」をクリックします。

![](https://static-docs.nocobase.com/mail-1735634131518.png)

![](https://static-docs.nocobase.com/mail-1735634131697.png)

#### **Microsoftメール**

1. https://account.microsoft.com/ を開き、ログインします。
    
2. 「Apps and services that can access your data」ボタンをクリックします。

![](https://static-docs.nocobase.com/mail-1735634131870.png)

3. 「編集」をクリックし、次に「削除」をクリックします。

![](https://static-docs.nocobase.com/mail-1735634132052.png)
