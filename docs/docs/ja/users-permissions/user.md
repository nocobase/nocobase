# ユーザー

## ユーザー管理

このプラグインは、設定センターでユーザー管理インターフェースを提供しています。アプリケーションの初期化後、削除できないスーパー管理者アカウントが自動的に追加されます。この管理者はRoot権限を持ち、すべてのリソースにアクセスできます。

![](https://static-docs.nocobase.com/44bf40f56b45d4dd96c424fb08082cf6.png)

この他に、ページでユーザーの**コレクション**を使って、ユーザーを管理するためのさまざまなブロック（例えばテーブルブロックなど）を追加することもできます。

![](https://static-docs.nocobase.com/76b5a4652f869541a9e8f18a4568a7c9.png)

## 設定

:::info{title=ヒント}
NocoBase v1.3.34-beta 以降が必要です。
:::

ユーザープロフィールの変更とユーザーパスワードの変更を許可するかどうかを設定します（すべてのユーザーに適用されます）。
