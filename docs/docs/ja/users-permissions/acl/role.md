# ロール

## 管理センター

### ロールの管理

![](https://static-docs.nocobase.com/da7083c67d794e23dc6eb0f85b1de86c.png)

アプリケーションを初めてインストールした際には、「Admin」と「Member」の2つのロールが組み込まれており、それぞれ異なるデフォルトの権限設定がされています。

### ロールの追加、削除、変更

ロール識別子（システム内で一意の識別子）を使用すると、デフォルトロールをカスタマイズできますが、システムに組み込まれているデフォルトロールは削除できません。

![](https://static-docs.nocobase.com/35f323b346db4f9f12f9bee4dea63302.png)

### デフォルトロールの設定

ここでいうデフォルトロールとは、新規ユーザー作成時にロールが割り当てられなかった場合に、自動的に適用されるロールのことです。

![](https://static-docs.nocobase.com/f41bba7ff55ca28715c486dc45bc1708.png)

## 個人センター

### ロールの切り替え

1人のユーザーに複数のロールを割り当てることができます。複数のロールを持つユーザーは、個人センターでロールを切り替えることが可能です。

![](https://static-docs.nocobase.com/e311d11ec1ca3b8b7e0472105b167819.png)

システムにログインする際のデフォルトロールの優先順位は、以下の通りです。最後に切り替えたロール（ロールを切り替えるたびにデフォルトロール値が更新されます）> 最初のロール（システムデフォルトロール）。
