# プラグイン管理

:::tip 前提条件

このページを読む前に、[AI ビルダー クイックスタート](./index.md)に従って NocoBase CLI のインストールと初期化が完了していることを確認してください。

:::

## 概要

プラグイン管理 Skill は、NocoBase プラグインの一覧表示、有効化、無効化を行うためのツールです。ローカル環境かリモート環境かを自動的に識別し、適切な実行バックエンドを選択して、読み戻し検証により操作の成功を確認します。


## 対応機能

- プラグイン一覧と有効化ステータスの確認
- プラグインの有効化
- プラグインの無効化

## プロンプト例

### シナリオ A：プラグインステータスの確認

プロンプトモード
```
現在の環境にはどのプラグインがありますか
```
コマンドラインモード
```
nb plugin list
```

すべてのプラグインとその有効化ステータス、バージョン情報が一覧表示されます。

![プラグインステータスの確認](https://static-docs.nocobase.com/20260417150510.png)

### シナリオ B：プラグインの有効化

プロンプトモード
```
ローカライゼーションプラグインを有効にしてください
```
コマンドラインモード
```
nb plugin enable <ローカライゼーション>
```

Skill が順番に有効化し、各有効化後に読み戻し検証で `enabled=true` を確認します。

![プラグインの有効化](https://static-docs.nocobase.com/20260417153023.png)

### シナリオ C：プラグインの無効化

プロンプトモード
```
ローカライゼーションプラグインを無効にしてください
```
コマンドラインモード
```
nb plugin disable  <ローカライゼーション>
```

![プラグインの無効化](https://static-docs.nocobase.com/20260417173442.png)

## よくある質問

**プラグインを有効にしたのに反映されない場合は？**

一部のプラグインは有効化後にアプリケーションの再起動が必要です。Skill の結果に再起動が必要かどうかが表示されます。
