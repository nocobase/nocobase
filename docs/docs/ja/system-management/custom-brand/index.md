# カスタムブランド

## 概要

システムのデフォルトブランド名は「NocoBase」で、主に以下の3箇所に表示されます。

![20240409111100](https://static-docs.nocobase.com/20240409111100.png)

ログインページの下部：

![20240409111254](https://static-docs.nocobase.com/20240409111254.png)

位置1は、[システム設定](/system-management/system-settings/)の「ロゴ」オプションで設定できます。位置2と3は、このプラグインで設定可能です。

## 使用方法

プラグインを有効化したら、ページ右上の「カスタムブランド」メニューをクリックして設定ページにアクセスします。

![20240409113640](https://static-docs.nocobase.com/20240409113640.png)

- **ブランド**: ログインページ下部のテキスト（位置3）を設定します。
- **About**: ページ右上のメニューコンテンツを設定します（下図参照）。

![20240409113915](https://static-docs.nocobase.com/20240409113915.png)

設定ではHTML形式がサポートされており、コンテンツを柔軟にカスタマイズできます。簡単なテキスト変更のみが必要な場合は、下図のように既存のコンテンツを直接編集できます。
