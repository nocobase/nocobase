# プラグイン管理

## プラグインマネージャー

ページの右上にあるプラグインマネージャーアイコンをクリックすると、プラグインマネージャーページにアクセスできます。このページでは、プラグインの表示、インストール、有効化、または無効化が可能です。

![2024-04-10_08-47-45](https://static-docs.nocobase.com/2024-04-10_08-47-45.png)

## プラグイン設定センター

プラグインには通常、それぞれ独自の設定ページがあります。ページの右上にある設定アイコンをクリックすると、すべてのプラグインの設定メニューが表示されます。該当するメニュー項目をクリックして、プラグインを設定できます。
