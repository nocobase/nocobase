# NocoBase 2.0 入門チュートリアル

このチュートリアルでは、NocoBase 2.0 を使ってゼロから**シンプルな IT チケットシステム（HelpDesk）**を構築していきます。システム全体でわずか **2 つのデータテーブル**だけを使い、コードを一行も書かずに、チケットの提出、分類管理、変更追跡、権限制御、データダッシュボードを実現します。

## チュートリアルの対象

- **対象読者**：業務担当者、技術者、または NocoBase に興味のある方（基本的なコンピュータの知識があると良いでしょう）
- **サンプルプロジェクト**：シンプルな IT チケットシステム（HelpDesk）、テーブルはわずか 2 つ
- **所要時間**：2〜3 時間（非技術者）、1〜1.5 時間（技術者）
- **前提条件**：Docker 環境または[オンライン Demo](https://demo-cn.nocobase.com/new)（有効期限 24 時間、インストール不要）
- **バージョン**：NocoBase 2.0

## データモデルのプレビュー

このチュートリアルはシンプルなデータモデルを中心に進めます。テーブルは **2 つ**だけですが、データモデリング、ページ構築、フォーム設計、権限制御、ワークフロー、ダッシュボードといったコア機能を十分にカバーできます。

| データテーブル | 主なフィールド |
|--------|---------|
| チケット（tickets） | タイトル、説明、ステータス、優先度 |
| チケット分類（categories） | 分類名、色 |

## よくある質問

### NocoBase はどのような場面に適していますか？

社内ツール、データ管理システム、承認ワークフロー、CRM、ERP など、柔軟なカスタマイズが必要な場面に適しており、プライベートデプロイにも対応しています。

### このチュートリアルを完了するにはどのような基礎知識が必要ですか？

プログラミングは不要ですが、基本的なコンピュータの知識があると良いでしょう。チュートリアルではデータテーブル、フィールド、リレーションなどの概念を段階的に解説します。データベースや Excel の使用経験があれば、より理解しやすいでしょう。

### チュートリアルで作ったシステムは拡張できますか？

はい。このチュートリアルではテーブルを 2 つしか使いませんが、NocoBase は複雑な多テーブルリレーション、外部 API 連携、カスタムプラグインなどをサポートしています。

### どのようなデプロイ環境が必要ですか？

Docker（Docker Desktop または Linux サーバー）を推奨します。最低 2 コア 4GB メモリが必要です。Git ソースコードからの実行にも対応しています。学習目的であれば、[オンライン Demo](https://demo-cn.nocobase.com/new) を直接申請できます。インストール不要で、有効期限は 24 時間です。

### 無料版にはどのような制限がありますか？

コア機能はすべて無料のオープンソースです。商用版では追加の高度なプラグインとテクニカルサポートが提供されます。詳細は[商用版の料金](https://www.nocobase.com/cn/commercial)をご覧ください。

## 関連技術スタック

NocoBase 2.0 は以下の技術で構築されています。

- **フロントエンドフレームワーク**: React + [Ant Design](https://ant.design/) 5.0
- **バックエンド**: Node.js + Koa
- **データベース**: PostgreSQL（[MySQL](/get-started/installation/docker)、MariaDB にも対応）
- **デプロイ方式**: [Docker](/get-started/installation/docker)、Kubernetes

## 類似プラットフォームの参考

ノーコード/ローコードプラットフォームを検討中の方に、いくつかの比較参考をご紹介します。

| プラットフォーム | 特徴 | NocoBase との違い |
|------|------|-------------------|
| [Appsmith](https://www.appsmith.com/) | オープンソースのノーコード、フロントエンドのカスタマイズ性が高い | NocoBase はデータモデル駆動を重視 |
| [Retool](https://retool.com/) | 社内ツール向けプラットフォーム | NocoBase は完全オープンソースで利用制限なし |
| [Airtable](https://airtable.com/) | オンラインコラボレーションデータベース | NocoBase はプライベートデプロイ対応、データを自己管理可能 |
| [Budibase](https://budibase.com/) | オープンソースのローコード、セルフホスティング対応 | NocoBase はプラグインアーキテクチャで拡張性が高い |
