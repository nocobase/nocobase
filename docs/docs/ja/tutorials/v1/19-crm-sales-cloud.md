# CRM セールスクラウドシステム

## 1.1 背景

- **背景説明** 企業のデジタルトランスフォーメーションが加速する中、リード、顧客、商談、見積もり、受注、日常活動を一元管理する統合的な営業プラットフォームの構築は不可欠です。本ソリューションは NocoBase をベースに CRM システム（セールスクラウド）を構築し、データの一元管理とインテリジェントなプロセスを実現します。
- > 注：本ソリューションは軽量 CRM システムの第一部分であり、購買、出荷、サプライヤー管理などの内容は含まれていません。機能実装方法の紹介を主目的としており、参考としてご活用ください。
  >

## 1.2 プロジェクトの目標と範囲

- **目標** リード、顧客、連絡先、商談、商品、見積もり、受注、活動管理を統合した CRM システムを構築し、営業効率とデータ精度を向上させます。
- **範囲** 営業チェーン全体のデータ入力、追跡、変換、レポート分析をカバーします。マーケティング活動管理はオプションモジュールで、購買、出荷、サプライヤー管理などは現時点では含まれていません。

## 1.3 ユーザーロールと権限の概要

- **営業担当者**：リードの入力とフォローアップ、顧客・連絡先・商談・見積もり・受注の作成、日常活動の記録を担当します。
- **営業マネージャー**：営業進捗の監視、見積もりと受注の承認、各種データレポートの閲覧を担当します。
- **システム管理者**：ユーザー権限の設定、データメンテナンス、システム設定、インターフェース管理を担当します。

## 1.4 機能モジュールの概要

- システムに含まれるコア機能をすばやく把握できます：


| 機能モジュール                   | サブモジュール/サブ機能                                                      | 説明                                                                             |
| -------------------------------- | ---------------------------------------------------------------------------- | -------------------------------------------------------------------------------- |
| **リード管理**                   | リード入力、フォローアップ＆ステータス管理、リード変換                        | 潜在顧客情報を収集・フォローアップし、条件を満たすリードを顧客・連絡先・商談に変換します。 |
| **顧客＆連絡先管理**             | 顧客管理、連絡先管理                                                         | 顧客ファイルを構築し、会社および連絡先情報を記録します。                         |
| **商談管理**                     | 商談作成、営業ステージ管理、予測分析、商品管理、クローズ理由、回収予定管理     | 営業案件を記録し、営業プロセスの追跡と販売トレンドの予測を行います。             |
| **商品＆価格表管理**             | 商品カタログ管理、価格表管理                                                 | 商品情報、在庫、価格設定を管理します。                                           |
| **見積管理**                     | 見積書生成、承認フロー、見積明細管理                                         | 正式な見積書を作成・管理し、承認と合計金額の自動計算をサポートします。           |
| **受注管理**                     | 受注作成、ステータス追跡、出荷管理                                           | 受注の生成、承認、物流ステータスを記録します。                                   |
| **活動管理**                     | タスク管理、ミーティング記録、通話記録、カレンダー＆リマインダー               | 営業タスク、ミーティング、通話を管理し、スケジュール管理をサポートします。       |
| **データレポート＆分析**         | 営業実績レポート、商談コンバージョン分析、セールスファネル分析                 | 多次元の営業統計レポートを生成し、データ分析をサポートします。                   |
| **マーケティング活動管理（オプション）** | キャンペーン企画、キャンペーン効果追跡                                       | マーケティングキャンペーンを企画・評価します。                                   |
