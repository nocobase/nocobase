# カレンダーブロック

## はじめに

カレンダーブロックは、イベントや日付関連のデータをカレンダー形式で表示し、会議のスケジュール設定、イベントの計画など、様々なシーンで活用できます。

## インストール

組み込みのプラグインですので、追加のインストールは不要です。

## ブロックの追加

![20250403220300](https://static-docs.nocobase.com/20250403220300.png)

1.  タイトルフィールド: カレンダーのバーに表示される情報です。現在、`input`、`select`、`phone`、`email`、`radioGroup`、`sequence`などのフィールドタイプに対応しています。プラグインを使って、カレンダーブロックがサポートするタイトルフィールドのタイプを拡張することも可能です。
2.  開始時間: タスクの開始時刻です。
3.  終了時間: タスクの終了時刻です。

### 背景色フィールド

:::info{title=ヒント}
NocoBase のバージョンは v1.4.0-beta 以降が必要です。
:::

このオプションは、カレンダーイベントの背景色を設定するために使用できます。使用方法は以下の通りです。

1.  カレンダーのデータテーブルには、**ドロップダウン単一選択（Single select）**または**ラジオグループ（Radio group）**タイプのフィールドが必要です。このフィールドには色を設定しておく必要があります。
2.  次に、カレンダーブロックの設定画面に戻り、**背景色フィールド**で、先ほど色を設定したフィールドを選択します。
3.  最後に、カレンダーイベントに色を選択し、「送信」をクリックしてみてください。色が適用されていることを確認できます。

![20240914192017_rec_](https://static-docs.nocobase.com/20240914192017_rec_.gif)

### 週の開始曜日

> v1.7.7 以降のバージョンでサポートされています

カレンダーブロックでは、週の開始曜日を設定できます。週の最初の日として**日曜日**または**月曜日**を選択可能です。
デフォルトの開始曜日は**月曜日**です。これにより、ユーザーは地域ごとの習慣に合わせてカレンダー表示を調整し、より実用的なニーズに合わせることができます。

### 今日

カレンダーブロックの「今日」ボタンは、便利なナビゲーション機能を提供します。これにより、ユーザーは他の日付に移動した後でも、現在の日にちを含むカレンダーページに素早く戻ることができます。

![20240419203514](https://static-docs.nocobase.com/20240419203514.png)
