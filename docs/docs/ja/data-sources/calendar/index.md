# カレンダーブロック

## 概要

カレンダーブロックはカレンダービューでイベントや日付に関連するデータを表示し、会議のスケジュールやイベント計画などのシーンに適しています。

## インストール

組み込みプラグインのため、インストールは不要です。

## ブロックの追加

![20250403220300](https://static-docs.nocobase.com/20250403220300.png)

1. タイトルフィールド: カレンダー上のバーに表示する情報です。現在、`input`、`select`、`phone`、`email`、`radioGroup`、`sequence` などのフィールドタイプに対応しています。プラグインを拡張することで、カレンダーブロックでサポートされるタイトルフィールドの種類を追加できます。
2. 開始時間: タスクの開始時間です。
3. 終了時間: タスクの終了時間です。

### 背景色フィールド

:::info{title=ヒント}
NocoBase のバージョンが v1.4.0-beta 以上である必要があります。
:::

この項目では、カレンダーイベントの背景色を設定できます。使用方法は以下のとおりです。

1. カレンダーデータテーブルに、**ドロップダウン（単一選択）（Single select）**または**ラジオグループ（Radio group）**タイプのフィールドが必要です。このフィールドには色を設定しておく必要があります。
2. 次に、カレンダーブロックの設定画面に戻り、**背景色フィールド**で、先ほど色を設定したフィールドを選択します。
3. 最後に、カレンダーイベントの色を1つ選択して送信すると、色が反映されていることを確認できます。

![20240914192017_rec_](https://static-docs.nocobase.com/20240914192017_rec_.gif)

### 週の開始日

> v1.7.7 以降のバージョンでサポートされています

カレンダーブロックでは、週の開始日を設定できます。**日曜日**または**月曜日**を週の最初の日として選択できます。
デフォルトの開始日は**月曜日**です。地域ごとの習慣に合わせてカレンダー表示を調整でき、実際の利用ニーズにより適合します。

### 今日

カレンダーブロックの「今日」ボタンは便利なナビゲーション機能を提供します。別の日付に移動した後、現在の日付が表示されているカレンダーページにすばやく戻ることができます。
