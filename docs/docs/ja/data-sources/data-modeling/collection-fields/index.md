# コレクションフィールド

## フィールドのインターフェースタイプ

NocoBase では、インターフェースの視点からフィールドを以下のカテゴリに分類しています。

## フィールドのデータ型

各フィールドインターフェースにはデフォルトのデータ型があります。例えば、インターフェースが数値（Number）のフィールドでは、データ型はデフォルトで `double` ですが、`float` や `decimal` なども選択できます。現在サポートされているデータ型は以下のとおりです。

![20240512103733](https://static-docs.nocobase.com/20240512103733.png)

## フィールドタイプのマッピング

メインデータベースに新しいフィールドを追加するプロセスは以下のとおりです。

1.  インターフェースタイプを選択します。
2.  現在のインターフェースで選択可能なデータ型を設定します。

![20240512172416](https://static-docs.nocobase.com/20240512172416.png)

外部データソースのフィールドマッピングのプロセスは以下のとおりです。

1.  外部データベースのフィールドタイプに基づいて、対応するデータ型（Field type）とUIタイプ（Field Interface）を自動的にマッピングします。
2.  必要に応じて、より適切なデータ型とインターフェースタイプに変更します。
