# ローカルストレージ

アップロードされたファイルは、サーバーのローカルディスクディレクトリに保存されます。これは、システムで管理するアップロードファイルの総量が比較的少ない場合や、実験的なシナリオに適しています。

### パス

パスは、ファイルがサーバーに保存される相対パスと、URLアクセスパスの両方を表します。例えば、「`user/avatar`」（先頭と末尾の「`/`」は不要です）は、以下を表します。

1.  アップロードされたファイルがサーバーに保存される相対パス: `/path/to/nocobase-app/storage/uploads/user/avatar`。
2.  ファイルにアクセスする際のURLプレフィックス: `http://localhost:13000/storage/uploads/user/avatar`。
