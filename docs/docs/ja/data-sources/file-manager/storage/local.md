# ローカルストレージ

アップロードしたファイルはサーバーのローカルハードディスク上のディレクトリに保存されます。システムで管理するアップロードファイルの総量が少ない場合や、試験的な用途に適しています。

### パス

サーバー上のファイルの相対保存パスと、URL のアクセスパスを同時に指定します。例：「`user/avatar`」（先頭と末尾に「`/`」を付ける必要はありません）は、次を表します。

1. ファイルのアップロード時にサーバー上で保存される相対パス：`/path/to/nocobase-app/storage/uploads/user/avatar`。
2. アクセス時の URL アドレスのプレフィックス：`http://localhost:13000/storage/uploads/user/avatar`。
