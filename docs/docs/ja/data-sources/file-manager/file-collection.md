# ファイルコレクション

## はじめに

ファイルは、特別な構造を持つデータテーブルのレコードです。この特別な構造を持つデータテーブルは、**ファイルコレクション**と呼ばれ、ファイルのメタ情報を保存するために使用されます。ファイルマネージャーを通じて管理することも可能です。
