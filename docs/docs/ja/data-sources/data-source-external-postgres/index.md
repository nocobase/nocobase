# 外部データソース - PostgreSQL

## はじめに

外部のPostgreSQLデータベースをデータソースとして利用できます。現在サポートされているバージョンは、PostgreSQL 9.5以上です。

## インストール

このプラグインは商用プラグインです。プラグインマネージャーからアップロードして有効化する必要があります。
