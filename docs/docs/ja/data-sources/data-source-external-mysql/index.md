# 外部データソース - MySQL

## はじめに

外部のMySQLデータベースをデータソースとして利用します。現在サポートされているバージョンは MySQL >= 5.7 です。

## インストール

このプラグインは商用プラグインです。プラグインマネージャーからアップロードして有効化する必要があります。
