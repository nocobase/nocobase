# 外部データソース - MySQL

## はじめに

外部のMySQLデータベースをデータソースとして利用します。現在サポートされているバージョンは MySQL >= 5.7 です。

## インストール

このプラグインは商用プラグインです。プラグインマネージャーからアップロードして有効化する必要があります。

![20240323162741](https://static-docs.nocobase.com/20240323162741.png)

## 使用方法

[データソース / 外部データベース](/data-sources/data-source-manager/external-database) セクションを参照してください。
