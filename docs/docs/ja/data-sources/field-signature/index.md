# データテーブルフィールド：手書き署名

## 紹介

手書き署名フィールドは、ユーザーがキャンバス上でマウスやタッチスクリーンを使って署名を書くことを可能にします。保存後、署名画像は選択した**ファイルデータテーブル**に書き込まれ、**ファイルマネージャー**が提供するファイルアップロードとストレージのフローを再利用します。

## インストール

1. 現在の環境が**プロフェッショナル版以上**であり、ライセンスが有効であることを確認してください。
2. **プラグインマネージャー**を開き、**データテーブルフィールド：手書き署名**（`@nocobase/plugin-field-signature`）を見つけて有効にしてください。
3. **ファイルマネージャー**（`@nocobase/plugin-file-manager`）が有効であることを確認してください。手書き署名フィールドは、ファイルデータテーブル、アップロード、ストレージ機能の提供にファイルマネージャーに依存しています。有効でない場合、署名画像を保存できません。

## 使用方法

### フィールドの追加

**データソース** → データテーブルを選択 → **フィールドの設定** → **フィールドの追加** → マルチメディアグループから**手書き署名**を選択します。

### フィールド設定

- **ファイルデータテーブル**：必須。ファイルを保存するためのファイルデータテーブル（例：`attachments`）を選択してください。署名画像はここに保存されます。
- 署名画像が実際に使用するストレージ設定とアップロードルールは、選択したファイルデータテーブル自体によって決まります。

### インターフェース設定

- 手書き署名フィールドをフォームに追加した後、フィールドのインターフェース設定で**署名設定**を調整できます。これには、ペンの色、背景色、署名キャンバスの幅、署名キャンバスの高さ、およびサムネイルの幅と高さが含まれます。
- 読み取り専用の表示シナリオでは、署名サムネイルの幅と高さを調整して、署名画像の表示サイズを制御することもできます。

### インターフェース操作

- フィールドエリアをクリックすると署名キャンバスが開き、署名が完了したら確認することで、対応するファイルレコードにアップロードして関連付けることができます。
- 小さい画面のデバイスでは、横向き/フルスクリーンの署名インターフェースを使用して、書きやすくすることができます。
