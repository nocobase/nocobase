# 外部データソース - MSSQL

## はじめに

外部のMSSQLデータベースをデータソースとして利用できます。現在サポートされているバージョンは、SQL Server 2014-2019です。

## インストール

このプラグインは商用プラグインです。プラグインマネージャーからアップロードして有効化する必要があります。
