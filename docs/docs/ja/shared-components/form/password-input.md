# PasswordInput

`PasswordInput` は、強度表示付きでパスワードを入力するためのコンポーネントです。

## API

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| `value` | `string` | 現在の値 |
| `onChange` | `(event) => void` | 値変更時のコールバック |
| `disabled` | `boolean` | 無効化するかどうか |
| `placeholder` | `string` | プレースホルダー文字列 |
