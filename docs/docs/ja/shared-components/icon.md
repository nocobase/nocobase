# Icon

`Icon` は、Ant Design アイコンを使う、またはカスタムアイコンを登録するためのコンポーネントです。

## 基本的な使い方

```tsx file="./_demos/icon.tsx" preview
```

## API

| パラメータ | 型 | 説明 |
| --- | --- | --- |
| `type` | `string` | アイコン名 |
| `component` | `React.ComponentType` | カスタム SVG コンポーネント |
