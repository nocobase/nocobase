### 数値の書式設定

#### 1. :formatN(precision)

##### 構文説明
数値をロケール設定に基づいて書式設定します。  
パラメーター:
- `precision`: 小数点以下の桁数  
  ODS/XLSX形式の場合、表示される小数点以下の桁数はテキストエディターによって決まります。その他の形式では、このパラメーターが使用されます。

##### 実行例
```
// 実行環境の例: APIオプション { "lang": "en-us" }
'10':formatN()         // 出力 "10.000"
'1000.456':formatN()   // 出力 "1,000.456"
```

##### 結果
指定された精度とロケール形式で数値が出力されます。


#### 2. :round(precision)

##### 構文説明
数値を指定された小数点以下の桁数で四捨五入します。

##### 実行例
```
10.05123:round(2)      // 出力 10.05
1.05:round(1)          // 出力 1.1
```

##### 結果
四捨五入された数値が出力されます。
