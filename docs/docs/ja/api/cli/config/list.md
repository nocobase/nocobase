# nb config list

明示的に設定された CLI 設定を一覧表示します。一度も明示的に設定されていないデフォルト値は表示されません。

## 使い方

```bash
nb config list
```

## 使用例

```bash
nb config list
```

## 補足

設定値が 1 つもない場合、CLI は `No CLI config values are set.` を出力します。
