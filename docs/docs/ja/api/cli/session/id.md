# nb session id

現在有効なセッション ID を表示します。

現在のシェルまたはランタイムに使用可能な `NB_SESSION_ID` がない場合、このコマンドはまず [`nb session setup`](./setup.md) を実行し、その後新しいシェルセッションまたはランタイムを開くよう案内します。

## 使い方


nb session id

## 使用例


nb session id

## 関連コマンド

- [`nb session setup`](./setup.md)
- [`nb env current`](../env/current.md)
