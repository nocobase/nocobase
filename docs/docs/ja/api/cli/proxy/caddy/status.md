# nb proxy caddy status

現在の Caddy proxy の実行状態を表示します。

## 使い方

```bash
nb proxy caddy status
```

## 出力

通常、次の情報が含まれます。

- `driver`
- `status`
- `config`
- `caddy` または `container`
