# nb proxy caddy current

現在の Caddy provider driver を出力します。

## 使い方

```bash
nb proxy caddy current
```

## 出力

このコマンドは次のいずれか 1 つの値を直接出力します。

- `local`
- `docker`

## 例

```bash
nb proxy caddy current
```
