# nb proxy nginx status

現在の Nginx proxy の実行状態を表示します。

## 使い方

```bash
nb proxy nginx status
```

## 出力

通常、次の情報が含まれます。

- `driver`
- `status`
- `config`
- `nginx` または `container`
