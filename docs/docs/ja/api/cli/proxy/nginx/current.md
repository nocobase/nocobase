# nb proxy nginx current

現在の Nginx provider driver を出力します。

## 使い方

```bash
nb proxy nginx current
```

## 出力

このコマンドは次のいずれか 1 つの値を直接出力します。

- `local`
- `docker`

## 例

```bash
nb proxy nginx current
```
