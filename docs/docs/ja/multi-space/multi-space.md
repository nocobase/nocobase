# マルチスペース

## 紹介

**マルチスペースプラグイン**を使用すると、単一のアプリケーションインスタンス内で論理的な分離を行い、複数の独立したデータスペースを実現できます。

#### 活用シーン
- **多店舗または工場の管理**：在庫管理、生産計画、販売戦略、レポートテンプレートなどのビジネスプロセスやシステム設定は高度に統一されているが、各拠点ごとにデータが干渉しないように独立させる必要がある場合。
- **多組織または子会社の管理**：グループ企業内の複数の組織や子会社が同一のプラットフォームを共有しつつ、ブランドごとに顧客、製品、注文データを独立して管理する場合。

## インストール

プラグインマネージャーで **マルチスペース（Multi-Space）** プラグインを見つけ、有効化します。

![](https://static-docs.nocobase.com/multi-space/Plugin-manager-NocoBase-10-15-2025_09_57_AM.png)

## ユーザーマニュアル

### マルチスペースの管理

プラグインを有効化した後、「ユーザーと権限」設定ページに移動し、**スペース** パネルに切り替えることでスペースを管理できます。

> 初期状態では、組み込みの **未割り当てスペース（Unassigned Space）** が存在します。これは主に、スペースに関連付けられていない既存データを確認するために使用されます。

![](https://static-docs.nocobase.com/multi-space/Users-Permissions-NocoBase-10-15-2025_09_58_AM.png)

#### スペースの作成

「スペースを追加」ボタンをクリックして、新しいスペースを作成します。

![](https://static-docs.nocobase.com/multi-space/Users-Permissions-NocoBase-10-15-2025_09_58_AM%20(1).png)

#### ユーザーの割り当て

作成したスペースを選択した後、右側でそのスペースに所属するユーザーを設定できます。

> **ヒント：** ユーザーをスペースに割り当てた後、右上のスペース切り替えリストを最新の状態に更新するには、**ページを手動でリフレッシュ**する必要があります。

![](https://static-docs.nocobase.com/multi-space/Users-Permissions-NocoBase-10-15-2025_09_59_AM.png)

### マルチスペースの切り替えと表示

右上のエリアで現在のスペースを切り替えることができます。  
右側の **目のアイコン**（ハイライト状態）をクリックすると、複数のスペースのデータを同時に表示できます。

![](https://static-docs.nocobase.com/multi-space/Goods-NocoBase-10-15-2025_10_26_AM.png)

### マルチスペースのデータ管理

プラグインを有効にすると、コレクション（Collection）を作成する際に、システムが自動的に **スペースフィールド** をプリセットします。  
**このフィールドが含まれているテーブルのみが、スペース管理ロジックの対象となります。**

![](https://static-docs.nocobase.com/multi-space/data-source-manager-main-NocoBase-10-15-2025_10_01_AM.png)

既存のコレクションについては、手動でスペースフィールドを追加することでスペース管理を有効にできます。

![](https://static-docs.nocobase.com/multi-space/data-source-manager-main-NocoBase-10-15-2025_10_03_AM.png)

#### デフォルトのロジック

スペースフィールドを含むコレクションでは、システムによって以下のロジックが自動的に適用されます。

1. データの作成時、現在選択されているスペースに自動的に関連付けられます。
2. データのフィルタリング時、現在選択されているスペースのデータのみに自動的に限定されます。

### 既存データのマルチスペース分類

マルチスペースプラグインを有効にする前に存在していたデータについては、以下の手順でスペースの分類を行うことができます。

#### 1. スペースフィールドの追加

既存のテーブルに手動でスペースフィールドを追加します。

![](https://static-docs.nocobase.com/multi-space/data-source-manager-main-NocoBase-10-15-2025_10_18_AM.png)

#### 2. ユーザーを未割り当てスペースに割り当てる

既存データを管理するユーザーを、**未割り当てスペース（Unassigned Space）** を含むすべてのスペースに関連付けます。これにより、まだスペースに属していないデータを確認できるようになります。

![](https://static-docs.nocobase.com/multi-space/Users-Permissions-NocoBase-10-15-2025_10_14_AM.png)

#### 3. すべてのスペースのデータを表示するように切り替える

上部の設定で、すべてのスペースのデータを表示するように選択します。

![](https://static-docs.nocobase.com/multi-space/Goods-NocoBase-10-15-2025_10_20_AM.png)

#### 4. 既存データ割り当て用のページを設定する

既存データの割り当てを行うための新しいページを作成します。**リストブロック** と **編集フォーム** に「スペースフィールド」を表示させ、手動で所属スペースを調整できるようにします。

![](https://static-docs.nocobase.com/multi-space/Goods-10-15-2025_10_21_AM.png)

スペースフィールドを編集可能に設定します。

![](https://static-docs.nocobase.com/multi-space/Goods-10-15-2025_10_22_AM.png)

#### 5. データのスペースを手動で割り当てる

作成したページを通じてデータを手動で編集し、既存データに正しいスペースを順次割り当てます（一括編集を構成して行うことも可能です）。
