# 中国地域フィールド

## はじめに

中国地域フィールドプラグイン（`field-china-region`）は、NocoBaseに「中国地域」フィールドタイプを追加します。

## インストール

NocoBaseのメインインターフェースで「プラグインマネージャー」に移動し、「中国地域」プラグインを見つけて有効にします。

## 使い方

コレクションの設定で新しいフィールドを追加し、フィールドタイプとして「中国地域」を選択します。

![](/images/field-china-region-1.png)

### フィールド設定

フィールド設定では、フィールドの表示モードを設定できます。

- 省
- 省/市
- 省/市/区
- 省/市/区/県
