# ワークフロー出力

## はじめに

「ワークフロー出力」ノードは、呼び出されたワークフローでその出力値を定義するために使用されます。あるワークフローが別のワークフローから呼び出された際、「ワークフロー出力」ノードを使って、呼び出し元に値を渡すことができます。

## ノードの作成

呼び出されたワークフローに、「ワークフロー出力」ノードを追加します。

![20241231002033](https://static-docs.nocobase.com/20241231002033.png)

## ノードの設定

### 出力値

出力値として変数を入力または選択します。出力値は任意の型にすることができ、文字列、数値、論理値、日付、カスタムJSONなどの定数、またはワークフロー内の他の変数も指定できます。

![20241231003059](https://static-docs.nocobase.com/20241231003059.png)

:::info{title=ヒント}
呼び出されたワークフローに複数の「ワークフロー出力」ノードが追加されている場合、そのワークフローが呼び出されると、最後に実行された「ワークフロー出力」ノードの値が出力されます。
:::
