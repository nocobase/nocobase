# 抄送 <Badge>v1.8.2+</Badge>

## 介绍

抄送ノードは、ワークフローの実行プロセスにおける特定のコンテキスト内容を指定されたユーザーに送信し、把握や閲覧に供するために使用されます。例えば、承認やその他のプロセスにおいて、関連情報を他の参加者に抄送することで、彼らが業務の進捗をタイムリーに把握できるようにします。

ワークフロー内に複数の抄送ノードを設定でき、ワークフローがそのノードに到達した際に、関連情報を指定された受信者に送信します。

抄送された内容は待機センターの「自分に抄送」メニューに表示され、ユーザーは自分に抄送されたすべての内容をここで確認できます。また、未読状態に基づいて未確認の抄送内容が通知され、ユーザーは確認後に手動で既読としてマークできます。

## 节点创建

ワークフロー設定画面で、プロセス内のプラス（「+」）ボタンをクリックし、「抄送」ノードを追加します：

![抄送_添加](https://static-docs.nocobase.com/20250710222842.png)

## 节点配置

![节点配置](https://static-docs.nocobase.com/20250710224041.png)

ノード設定画面では、以下のパラメーターを設定できます：

### 接收者

受信者は抄送対象ユーザーの集合であり、1人または複数のユーザーを指定できます。選択ソースは、ユーザーリストから選択する静的な値、変数で指定する動的な値、またはユーザーコレクションへのクエリ結果から選択できます。

![接收者配置](https://static-docs.nocobase.com/20250710224421.png)

### 用户界面

受信者は待機センターの「自分に抄送」メニューで抄送内容を確認する必要があります。ワークフローコンテキスト内のトリガーや任意のノードの結果をコンテンツブロックとして設定できます。

![用户界面](https://static-docs.nocobase.com/20250710225400.png)

### 待办卡片 <Badge>2.0+</Badge>

待機センターの「自分に抄送」リスト内のタスクカードを設定するために使用されます。

![20260213010947](https://static-docs.nocobase.com/20260213010947.png)

カード内には、表示したい業務フィールド（関係フィールドを除く）を自由に設定できます。

ワークフローの抄送タスクが作成されると、待機センターのリストにカスタマイズされたタスクカードが表示されます：

![20260214124325](https://static-docs.nocobase.com/20260214124325.png)

### 任务标题

タスクタイトルは待機センターに表示されるタイトルです。ワークフローコンテキスト内の変数を使用して、タイトルを動的に生成できます。

![任务标题](https://static-docs.nocobase.com/20250710225603.png)

## 待办中心

ユーザーは待機センターで自分に抄送されたすべての内容を確認・管理でき、既読状態に基づいてフィルタリングや閲覧が可能です。

![20250710232932](https://static-docs.nocobase.com/20250710232932.png)

![20250710233032](https://static-docs.nocobase.com/20250710233032.png)

閲覧後に既読としてマークでき、未読数がそれに応じて減少します。
