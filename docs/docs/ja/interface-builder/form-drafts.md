# フォームの下書き

フォームの下書き機能を有効にすると、入力中の内容がローカルに自動保存されます。誤ってフォームを閉じたり、ページを更新したりしても、入力済みのデータが失われることはありません。

![20251213093717](https://static-docs.nocobase.com/20251213093717.png)

下書きが存在する場合、フォームの上部に通知メッセージが表示されます。送信が成功すると下書きは自動的に削除されます。不要になった場合は、いつでも手動で削除することも可能です。

![20251213092734](https://static-docs.nocobase.com/20251213092734.png)

::: tip ヒント

現在、フォームの下書きはローカルキャッシュのみをサポートしています。今後のバージョンではサーバー側のキャッシュに対応し、複数デバイス間での同期が可能になる予定です。

:::
