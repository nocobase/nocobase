# テーブルフィールド

## はじめに

テーブルフィールドでは、列幅の調整、フィールドタイトルの変更、並べ替えなどの基本的な設定に加えて、日付フィールド、関連フィールド、数値フィールドといった特定のフィールドに対して、より詳細な表示設定が可能です。

![20251024174558](https://static-docs.nocobase.com/20251024174558.png)

## フィールド設定オプション

### 日付フィールドの書式設定

![20251024175303](https://static-docs.nocobase.com/20251024175303.png)

詳細については、[日付の書式設定](/interface-builder/fields/specific/date-picker) を参照してください。

### 数値フィールドの書式設定

簡単な単位変換、桁区切り（千分位）、接頭辞・接尾辞、精度、指数表記に対応しています。

![20251024175445](https://static-docs.nocobase.com/20251024175445.png)

詳細については、[数値の書式設定](/interface-builder/fields/field-settings/number-format) を参照してください。

### クイック編集を有効にする

クイック編集を有効にすると、列にマウスカーソルを合わせると編集ボタンが表示されます。このボタンをクリックすると、ポップアップウィンドウでデータを素早く編集・保存できます。

![20251025171158](https://static-docs.nocobase.com/20251025171158.gif)

### クリックで開くを有効にする

関連フィールドがクリックでポップアップを開く機能に対応しているだけでなく、通常のフィールドでも「クリックで開く」を有効にすることで、ポップアップを開くための入り口として利用できます。

![20251025172308](https://static-docs.nocobase.com/20251025172308.gif)

### 内容のオーバーフロー表示モード

列の内容がテーブルの列幅を超過する場合、オーバーフローの表示方法を設定できます。

- 省略表示（デフォルト）
- 折り返し

![20251025172549](https://static-docs.nocobase.com/20251025172549.png)

### フィールドコンポーネント

一部のフィールドは複数の表示形式に対応しており、ユーザーはフィールドコンポーネントを切り替えることで、さまざまな表示効果を実現し、異なるシナリオのニーズを満たすことができます。例えば、**URL** タイプフィールドは **Preview** コンポーネントに切り替えることで、リンクコンテンツや画像プレビューをより効果的に表示できます。

![20251025171658](https://static-docs.nocobase.com/20251025171658.png)

### 並べ替え可能

現在、ほとんどのフィールドタイプがサーバーサイドでの並べ替えに対応しています。並べ替えを有効にすると、対象フィールドに基づいてデータを昇順または降順で並べ替えることができます。
