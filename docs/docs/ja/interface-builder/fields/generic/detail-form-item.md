# 詳細フィールド

## はじめに

詳細ブロック、リストブロック、グリッドブロックなどのフィールド設定は基本的に同じで、主に読み取り専用モードでのフィールドの表示方法を制御します。

![20251025172851](https://static-docs.nocobase.com/20251025172851.png)

## フィールド設定オプション

### 日付フィールドの書式設定

![20251025173005](https://static-docs.nocobase.com/20251025173005.png)

詳細については、[日付の書式設定](/interface-builder/fields/specific/date-picker) を参照してください。

### 数値フィールドの書式設定

![20251025173242](https://static-docs.nocobase.com/20251025173242.png)

簡単な単位変換、桁区切り（千分位）、接頭辞と接尾辞、精度、科学的表記法に対応しています。

詳細については、[数値の書式設定](/interface-builder/fields/field-settings/number-format) を参照してください。

### クリックして開くを有効にする

関連フィールドがクリックでポップアップを開く機能に対応しているだけでなく、通常のフィールドでもクリックして開く機能を有効にすることで、ポップアップを開くための入り口として利用できます。また、ポップアップの開き方（ドロワー、ダイアログ、サブページ）も設定可能です。

![20251025173549](https://static-docs.nocobase.com/20251025173549.gif)

### コンテンツのオーバーフロー表示モード

フィールドのコンテンツが幅を超えて表示される場合、オーバーフローの表示方法を設定できます。

- 省略表示（デフォルト）
- 折り返し

![20251025173917](https://static-docs.nocobase.com/20251025173917.png)

### フィールドコンポーネント

一部のフィールドは複数の表示形式に対応しており、フィールドコンポーネントを切り替えることで実現できます。

例：`URL` コンポーネントを `Preview` コンポーネントに切り替えることができます。

![20251025174042](https://static-docs.nocobase.com/20251025174042.png)

例：関連フィールドは、タイトルフィールドコンポーネントから `子詳細` に切り替えることで、関連フィールドのコンテンツをより多く表示するなど、異なる表示方法に切り替えることができます。

![20251025174311](https://static-docs.nocobase.com/20251025174311.gif)
