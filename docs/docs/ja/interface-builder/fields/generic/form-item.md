# フォームフィールド

## はじめに
フォーム内のフィールドは種類によって設定オプションが異なります。ユーザーはプラグインを使ってフィールドの設定オプションを拡張し、よりパーソナルなニーズに対応できます。

![20251025174555](https://static-docs.nocobase.com/20251025174555.png)

## フィールド設定オプション

### デフォルト値
フィールドに定数や変数をデフォルト値として設定できます（新規フォームでのみ利用可能です）。

詳細については、[デフォルト値の設定](/interface-builder/fields/field-settings/default-value)をご覧ください。
