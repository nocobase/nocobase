# フィルターフォームのフィールド

## はじめに

フィルターフォームでは、現在のコレクションのフィールドはもちろん、関連するコレクションのフィールド（さらにネストされた関連のフィールドも含む）をフィルターフィールドとして選択できます。

### 演算子

フィールドタイプに応じて適切な演算子を選択し、フィルターの精度と効率を高めましょう。文字列タイプのフィールドの場合、デフォルトでは曖昧一致が適用されます。
