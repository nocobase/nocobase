# 関連フィールドコンポーネント

## はじめに
NocoBase の関連フィールドコンポーネントは、関連データの表示と処理をより簡単に行えるように設計されています。これらのコンポーネントは、関係のタイプに関わらず柔軟性と汎用性があり、ユーザーは特定のニーズに合わせて選択・設定できます。

### ドロップダウン
ターゲットとなる**コレクション**がファイル**コレクション**である場合を除くすべての関連フィールドにおいて、編集モードでのデフォルトコンポーネントはドロップダウンです。ドロップダウンの選択肢にはタイトルフィールドの値が表示され、主要なフィールド情報を表示するだけで関連データを素早く選択したい場合に便利です。

![20240429205659](https://static-docs.nocobase.com/20240429205659.png)

詳細については、[ドロップダウン](/interface-builder/fields/specific/select)をご覧ください。

### ピッカー
ピッカーは、ポップアップモーダル形式でデータを表示します。ユーザーはピッカー内で表示するフィールド（ネストされた関連のフィールドを含む）を設定でき、これにより関連データをより正確に選択できます。

![20240429210824](https://static-docs.nocobase.com/20240429210824.png)

詳細については、[ピッカー](/interface-builder/fields/specific/picker)をご覧ください。

### サブフォーム
より複雑な関連データを扱う場合、ドロップダウンやピッカーでの操作は不便に感じることがあります。このようなケースでは、ユーザーは頻繁にポップアップを開く必要があります。このシナリオに対応するため、サブフォームを利用できます。サブフォームを使えば、新しいポップアップを繰り返し開くことなく、現在のページまたは現在のポップアップブロック上で関連**コレクション**のフィールドを直接管理できるため、操作フローがよりスムーズになります。多階層の関連は、ネストされたフォームとして表示されます。

![20251029122948](https://static-docs.nocobase.com/20251029122948.png)

詳細については、[サブフォーム](/interface-builder/fields/specific/sub-form)をご覧ください。

### サブテーブル
サブテーブルは、一対多または多対多の関連レコードをテーブル形式で表示します。関連データを明確かつ構造化された方法で表示・管理するための機能を提供し、データの新規一括作成や既存データの関連付けをサポートしています。

![20251029123042](https://static-docs.nocobase.com/20251029123042.png)

詳細については、[サブテーブル](/interface-builder/fields/specific/sub-table)をご覧ください。

### サブ詳細
サブ詳細とは、サブフォームの閲覧モードにおける対応コンポーネントです。多階層にネストされた関連データを表示できます。

![20251030213050](https://static-docs.nocobase.com/20251030213050.png)

詳細については、[サブ詳細](/interface-builder/fields/specific/sub-detail)をご覧ください。

### ファイルマネージャー
ファイルマネージャーは、関連のターゲット**コレクション**がファイル**コレクション**である場合に特化した関連フィールドコンポーネントです。

![20240429222753](https://static-docs.nocobase.com/20240429222753.png)

詳細については、[ファイルマネージャー](/interface-builder/fields/specific/file-manager)をご覧ください。

### タイトル
タイトルフィールドコンポーネントは、閲覧モードで使用される関連フィールドコンポーネントです。タイトルフィールドを設定することで、対応するフィールドコンポーネントを構成できます。

![20251030213327](https://static-docs.nocobase.com/20251030213327.png)

詳細については、[タイトル](/interface-builder/fields/specific/title)をご覧ください。
