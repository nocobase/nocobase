# ラベルの表示

## はじめに

ブロック内のフィールドには、デフォルトでフィールドラベルが表示されます。これは各フィールドの意味を識別し、ユーザーがデータ内容を読みやすく、理解しやすくするためです。

特定のシナリオ（例えば、表示専用ページやフィールドの意味がすでに非常に明確な場合など）では、インターフェースをよりシンプルにするために、ラベルを非表示にすることができます。ラベルを非表示にすると、フィールド値のみが表示され、フィールド名は表示されなくなります。
