# 表示モード

## はじめに

ブロックとは異なり、フィールドコンポーネントには3つの表示モードがあります。これらのモードはフォーム内のフィールドでのみ利用可能です。表示モードを切り替えることで、それぞれ異なるフィールド設定オプションが適用されます。

- 編集可能
- 読み取り専用（編集不可）
- 読み取り専用（閲覧モード）

### 関連フィールドの表示モード

**関連フィールドの表示モード**は、そのフィールドがインターフェース上でどのように表示されるかを決定し、選択可能なフィールドコンポーネントのタイプも決定します。

**編集可能状態**では、関連フィールドは様々なコンポーネント形式をサポートしています。ユーザーはビジネス要件に応じて、関連データを表示したり選択したりするために、異なる関連フィールドコンポーネントを選択できます。

#### 編集可能状態での関連フィールドコンポーネント

この状態では、ユーザーはデータをより効率的に処理するために、適切な表示方法を柔軟に選択できます。

#### 閲覧状態での関連フィールドコンポーネント

**閲覧状態**に切り替えると、システムは自動的にデフォルトで**タイトルフィールドコンポーネント**を使用して関連データを表示します。これは、データを変更せずにただ閲覧したいだけのシナリオに適しています。
