# フィールドコンポーネント

## はじめに

フィールドコンポーネントは、フィールドの内容を表示するためのものです。
一部のフィールドには複数の表示コンポーネントが組み込まれています。ユーザーは実際のニーズに合わせて適切なコンポーネントを選択し、フィールドの値をさまざまな方法で表示できます。

例：閲覧モードのURLフィールドは、プレビューコンポーネントまたはURLコンポーネント（デフォルト）に切り替えることができます。

![20251028214628](https://static-docs.nocobase.com/20251028214628.png)
