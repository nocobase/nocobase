# 数値の書式設定

## 概要

数値の書式設定とは、閲覧時の数値型フィールド（数式フィールドを含む）に対して、視覚的な見栄えの向上と標準化を行うことです。これには、桁区切り、小数点以下の精度、単位換算、指数表記などの設定が含まれます。数値の書式設定を構成することで、地域や業界ごとの数値表現形式に対応できます。

![20251028215059](https://static-docs.nocobase.com/20251028215059.png)

## 使用方法

- 簡単な単位換算（乗算/除算）に対応しています。
- 桁区切り
- プレフィックスとサフィックス
- 精度
- 指数表記

**例:** 注文金額をフォーマットする際、通貨記号（人民元記号「￥」など）、桁区切り（カンマやピリオドなど）、小数点以下の精度を追加することで、金額を明確で読みやすく表示できます。

- 指数表記を使用します。
