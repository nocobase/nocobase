# カスケード選択

## はじめに

カスケード選択は、ターゲットとなるコレクションがツリー構造のテーブルである関連フィールドに利用できます。ユーザーはツリー構造の階層に従ってデータを選択でき、データのあいまい検索にも対応しています。

## 使用方法

-   関連が「対一」の場合、カスケード選択は単一選択になります。

![20251125214656](https://static-docs.nocobase.com/20251125214656.png)

-   関連が「対多」の場合、カスケード選択は複数選択になります。

![20251125215318](https://static-docs.nocobase.com/20251125215318.png)

## フィールド設定オプション

### タイトルフィールド

タイトルフィールドは、各オプションに表示されるラベルフィールドです。

![20251125214923](https://static-docs.nocobase.com/20251125214923.gif)

> タイトルフィールドに基づいて高速検索が可能です。

![20251125215026](https://static-docs.nocobase.com/20251125215026.gif)

詳細については、[タイトルフィールド](/interface-builder/fields/field-settings/title-field) を参照してください。

### データスコープ

ツリーリストのデータ範囲を制御します（子レコードが条件に一致する場合、その親レコードも含まれます）。

![20251125215111](https://static-docs.nocobase.com/20251125215111.png)

詳細については、[データスコープ](/interface-builder/fields/field-settings/data-scope) を参照してください。
