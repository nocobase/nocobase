# データ選択器

## はじめに

データ選択器は、ポップアップ形式のテーブル選択器として機能し、ターゲットコレクションの既存データを選択して関連付けたり、ターゲットコレクションにデータを追加してから関連付けたりすることが可能です。

![20251027203343](https://static-docs.nocobase.com/20251027203343.png)

すでに選択済みのデータは、テーブル選択器には表示されません。

![20251027203717](https://static-docs.nocobase.com/20251027203717.gif)

テーブル選択器内で、関連付け先のコレクションをさらに管理（新規追加、削除、インポート/エクスポートなど）することも可能です。

![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
