# サブフォーム

## 概要

サブフォームは、関連データを新規作成してから関連付けを行うようなシナリオに適しています。多階層の関連データも、ネストされたフォームとして分かりやすく表示されます。データセレクターやドロップダウン選択肢と比較して、サブフォームは現在のページブロック内で関連 コレクション のフィールドを直接管理でき、関連データはメインフォームと一緒に送信されます。
