# サブテーブル (行内編集)

## 紹介

サブテーブルは対多のリレーションフィールドの処理に適しており、ターゲットコレクションのデータを一括で新規作成して関連付けたり、既存のデータから選択して関連付けたりすることをサポートしています。

## 使用説明

![20251027223350](https://static-docs.nocobase.com/20251027223350.png)

サブテーブル内の異なるタイプのフィールドは異なるフィールドコンポーネントを表示し、大きなフィールド（リッチテキスト、JSON、複数行テキストなど）はホバーポップアップで編集します。

![20251027223426](https://static-docs.nocobase.com/20251027223426.png)

サブテーブル内のリレーションフィールド。

注文（一対多） > Order Products（一対一） > Opportunity

![20251027223530](https://static-docs.nocobase.com/20251027223530.png)

リレーションフィールドコンポーネントのデフォルトはプルダウン選択（プルダウン選択/データセレクターをサポート）です。

![20251027223754](https://static-docs.nocobase.com/20251027223754.png)

## フィールド設定項目

### 既存データの選択を許可（デフォルトで有効）

既存のデータから選択して関連付けることができます。
![20251027224008](https://static-docs.nocobase.com/20251027224008.png)

![20251027224023](https://static-docs.nocobase.com/20251027224023.gif)

### フィールドコンポーネント

[フィールドコンポーネント](/interface-builder/fields/association-field)：プルダウン選択、データセレクターなど、他のリレーションフィールドコンポーネントに切り替えます。

### 既存データの関連付け解除を許可

> 編集フォーム内のリレーションフィールドデータにおいて、既存データの関連付け解除を許可するかどうか。

![20251028153425](https://static-docs.nocobase.com/20251028153425.gif)
