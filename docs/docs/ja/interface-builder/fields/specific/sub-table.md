# サブテーブル

## 概要

サブテーブルは、対多のリレーションフィールドを扱うのに適しています。ターゲットのコレクションに新しいデータを一括で作成して関連付けたり、既存のデータを選択して関連付けたりすることが可能です。

## 使用方法

![20251027223350](https://static-docs.nocobase.com/20251027223350.png)

サブテーブルでは、フィールドの種類に応じて異なるフィールドコンポーネントが表示されます。リッチテキスト、JSON、複数行テキストなどの大きなフィールドは、ポップアップモーダルで編集します。

![20251027223426](https://static-docs.nocobase.com/20251027223426.png)

リレーションフィールドのデフォルトコンポーネントはドロップダウンです。（ドロップダウン/データピッカーをサポートしています。）

## フィールド設定オプション

### 既存データの選択を許可（デフォルトでは無効）

既存のデータを選択して関連付けることができます。
![20251027224008](https://static-docs.nocobase.com/20251027224008.png)

![20251027224023](https://static-docs.nocobase.com/20251027224023.gif)

### フィールドコンポーネント

[フィールドコンポーネント](/interface-builder/fields/association-field)：ドロップダウンやデータピッカーなど、他のリレーションフィールドコンポーネントに切り替えます。

### 既存データの関連付け解除を許可

> 編集フォーム内のリレーションフィールドデータについて、既存データの関連付け解除を許可するかどうかを設定します。
