# サブ詳細

## 概要

サブ詳細とは、サブフォームの読み取りモードに対応するコンポーネントです。ラベルコンポーネントやタイトルコンポーネントと比較して、サブ詳細では現在のコレクションのデータをより多く表示できるだけでなく、関連するコレクションのデータ情報も表示するように設定できます。複数レベルのリレーションデータは、ネストされた詳細として明確に表示されます。
