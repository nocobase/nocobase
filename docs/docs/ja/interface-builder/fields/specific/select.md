# ドロップダウン

## はじめに

ドロップダウンは、ターゲットコレクションの既存データから選択してデータを関連付けたり、新しいデータを追加してから関連付けたりすることができます。ドロップダウンのオプションはあいまい検索に対応しています。

![20251029205901](https://static-docs.nocobase.com/20251029205901.png)

## フィールド設定

### データ範囲の設定

ドロップダウンリストのデータ範囲を制御します。

![20251029210025](https://static-docs.nocobase.com/20251029210025.png)

詳細については、[データ範囲の設定](/interface-builder/fields/field-settings/data-scope) を参照してください。

### ソート順の設定

ドロップダウンのデータのソート順を制御します。

例：サービス日を降順でソートします。

![20251029210105](https://static-docs.nocobase.com/20251029210105.png)

### 複数レコードの追加/関連付けを許可

多対多のリレーションシップにおいて、関連付けられるレコードを1つに制限します。

![20251029210145](https://static-docs.nocobase.com/20251029210145.png)

### タイトルフィールド

タイトルフィールドは、オプションとして表示されるラベルフィールドです。

![20251029210507](https://static-docs.nocobase.com/20251029210507.gif)

> タイトルフィールドに基づいて高速検索をサポートしています。

詳細については、[タイトルフィールド](/interface-builder/fields/field-settings/title-field) を参照してください。

### クイック作成：データを追加してから選択

![20251125220046](https://static-docs.nocobase.com/20251125220046.png)

#### ドロップダウンメニューからの追加

ターゲットコレクションに新しいレコードを作成した後、システムが自動的にそのレコードを選択し、フォーム送信時に関連付けます。

例として、`Orders` コレクションには多対一のリレーションシップフィールド「Account」があります。

![20251125220447](https://static-docs.nocobase.com/20251125220447.gif)

#### モーダルからの追加

モーダルからの追加は、より複雑なデータ入力シナリオに適しており、新規レコード作成用のフォームをカスタマイズして設定できます。
