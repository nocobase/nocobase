# 日付と時刻

## はじめに

日付フィールドは、カスタムの日付と時刻のフォーマットを設定できます。

## 日付フォーマットの設定場所

### コレクションフィールド

![20240417114311](https://static-docs.nocobase.com/20240417114311.png)

> コレクションフィールドで設定された日付フォーマットは、そのフィールドのデフォルトフォーマットとして使用されます。これは、ブロック設定内のフィールド設定で指定された日付フォーマットによって上書きできます。

### フィールド設定

![20251027191824](https://static-docs.nocobase.com/20251027191824.png)

あらかじめ用意されている様々な日付と時刻のフォーマットに加えて、個別のニーズに合わせてカスタムフォーマットもサポートしています。
