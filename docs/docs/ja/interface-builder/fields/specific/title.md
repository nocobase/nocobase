# タイトル

## はじめに

閲覧モードでは、関連フィールドはデフォルトで**タイトルフィールドコンポーネント**を使用して表示されます。これにより、タイトルフィールドの値を表示することで、現在関連付けられているレコードを識別します。
システムは、異なる種類のタイトルフィールドに応じて、対応するタイトルフィールドコンポーネントを自動的にマッチングし、レンダリングします。
