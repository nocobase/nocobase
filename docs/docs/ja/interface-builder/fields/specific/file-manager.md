# ファイルマネージャー

## 概要

ファイルマネージャーは、ターゲットのコレクションがファイルコレクションであるリレーションシップを扱うための関連フィールドコンポーネントです。

![20251027194124](https://static-docs.nocobase.com/20251027194124.png)

## フィールド設定

- [フィールドコンポーネント](/interface-builder/fields/association-field)：ドロップダウンやデータセレクターなど、他の関連フィールドコンポーネントに切り替えることができます。
- 既存ファイルの選択を許可：デフォルトで有効になっています。アップロード済みのファイルから選択し、関連付けることができます。
