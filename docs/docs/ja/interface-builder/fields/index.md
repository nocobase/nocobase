# フィールド

UI上では、フィールドはフィールドコンポーネントであり、個々のデータを保持する役割を担っています。データの種類に応じて、異なるフィールドコンポーネントで表示されます。フィールドはブロックにのみ紐付けて使用でき、単独で使うことはできません。

## ブロック内のフィールド

フィールドは通常、単独では使用されず、データ型ブロックの子要素として存在します。データ型ブロックには通常「フィールドを設定」というオプションがあり、フィールドリストは現在のコレクションから提供されます。

![20251029204240](https://static-docs.nocobase.com/20251029204240.png)

## フィールドのデザイナー（ツールバー）

ブロックと同様に、フィールドコンポーネントの右上にも3つのアイコンがあります。それぞれ、次の機能を表しています。

- ドラッグしてレイアウト
- [イベントフロー](/interface-builder/event-flow)
- 設定

![20251025165432](https://static-docs.nocobase.com/20251025165432.png)

## フィールドのレイアウト

ドラッグ＆ドロップで、ブロック内のフィールドのレイアウトをカスタマイズして調整できます。

![20251025165605](https://static-docs.nocobase.com/20251025165605.gif)

## フィールドの設定オプション

一般的なフィールド設定オプションに加えて、フィールドの種類によって、特定のオプションがサポートされています。
