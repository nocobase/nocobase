# フィールドの割り当て

## はじめに

フィールドの割り当てとは、操作を実行する前にフィールドの値を事前に設定することです。これにより、レコードの更新や作成時にデータが自動的に入力され、手動で入力する手間が省けます。例えば、注文ステータスの一括更新や、注文が配達された際に配達時間を現在時刻に設定する、といったシナリオで活用できます。
