# 二次確認

## はじめに

二次確認設定は、操作の実行を確認するために使用されます。操作がクリックされると確認ダイアログが表示されます。削除ボタンには、誤操作によるデータ損失や取り返しのつかない結果を防ぐため、デフォルトで二次確認が有効になっています。

![20251029125021](https://static-docs.nocobase.com/20251029125021.png)

二次確認のコンテンツとタイトルをカスタマイズでき、変数を取得することも可能です。
