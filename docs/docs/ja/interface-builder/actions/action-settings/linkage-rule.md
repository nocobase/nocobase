# アクション連動ルール

## はじめに

アクション連動ルールを使用すると、特定の条件に基づいてアクションの状態（表示、有効化、非表示、無効化など）を動的に制御できます。これらのルールを設定することで、アクションボタンの動作を現在のレコード、ユーザーロール、またはコンテキストデータと連動させることができます。

![20251029150224](https://static-docs.nocobase.com/20251029150224.png)

## 使用方法

条件が満たされた場合（条件が設定されていない場合はデフォルトで通過します）、プロパティ設定の実行またはJavaScriptの実行がトリガーされます。条件判定では定数や変数が使用可能です。

![20251030224601](https://static-docs.nocobase.com/20251030224601.png)

ボタンのプロパティを変更することもできます。

![20251029150452](https://static-docs.nocobase.com/20251029150452.png)
