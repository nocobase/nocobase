# エクスポート

## はじめに

エクスポート機能を使うと、フィルターされたレコードを**Excel**形式でエクスポートできます。エクスポートするフィールドも自由に設定可能です。ユーザーは、その後のデータ分析、処理、アーカイブのために、必要に応じてエクスポートするフィールドを選択できます。この機能はデータ操作の柔軟性を高め、特にデータを他のプラットフォームへ転送したり、さらに処理したりする必要がある場合に非常に役立ちます。

### 主な機能：
- **フィールド選択**：ユーザーはエクスポートするフィールドを設定・選択できます。これにより、エクスポートされるデータは正確かつ簡潔になります。
- **Excel形式に対応**：エクスポートされたデータは標準のExcelファイルとして保存されるため、他のデータとの統合分析が容易になります。

この機能を使えば、業務上の重要なデータを簡単にエクスポートして外部で利用できるため、作業効率の向上につながります。

## アクション設定

### エクスポート可能なフィールド

- 第一階層：現在のコレクションのすべてのフィールド
- 第二階層：関連フィールドの場合、関連するコレクションのフィールドを選択する必要があります。
- 第三階層：3つの階層のみが処理され、最終階層の関連フィールドは表示されません。

![20251029171557](https://static-docs.nocobase.com/20251029171557.png)

- [連動ルール](/interface-builder/actions/action-settings/linkage-rule)：ボタンを動的に表示/非表示にします。
- [ボタンの編集](/interface-builder/actions/action-settings/edit-button)：ボタンのタイトル、種類、アイコンを編集します。
