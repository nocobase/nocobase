# 送信

## はじめに

送信アクションは、フォームデータを保存するために使用します（フォームブロックに特有の機能です）。また、ワークフローと組み合わせることで、データの自動化プロセスを実現できます。

### ワークフローをバインド

データが正常に送信された後にのみ、バインドされたワークフローがトリガーされます。
