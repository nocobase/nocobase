# 添付ファイルのエクスポート

## 概要

添付ファイルのエクスポート機能では、添付ファイル関連のフィールドを圧縮ファイル形式でエクスポートできます。

#### 添付ファイルのエクスポート設定

![20251029173251](https://static-docs.nocobase.com/20251029173251.png)

![20251029173425](https://static-docs.nocobase.com/20251029173425.png)

![20251029173345](https://static-docs.nocobase.com/20251029173345.png)

- エクスポートする添付ファイルフィールドを設定します。複数選択が可能です。
- 各レコードごとにフォルダを生成するかどうかを選択できます。

ファイル名のルール：

- 各レコードごとにフォルダを生成することを選択した場合、ファイル名のルールは次のとおりです： `{レコードのタイトルフィールド値}/{添付ファイルフィールド名}[-{ファイル連番}].{ファイル拡張子}`。
- フォルダを生成しないことを選択した場合、ファイル名のルールは次のとおりです： `{レコードのタイトルフィールド値}-{添付ファイルフィールド名}[-{ファイル連番}].{ファイル拡張子}`。

添付ファイルフィールドに複数の添付ファイルが存在する場合、ファイル連番は自動的に生成されます。

- [連携ルール](/interface-builder/actions/action-settings/linkage-rule)：ボタンを動的に表示/非表示にします。
- [ボタンの編集](/interface-builder/actions/action-settings/edit-button)：ボタンのタイトル、タイプ、アイコンを編集します。
