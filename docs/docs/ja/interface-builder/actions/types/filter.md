# フィルター

## はじめに

フィルター操作は通常、データブロックに表示され、様々な条件を設定してデータを絞り込みます。注意点として、もしブロックにデータスコープがすでに設定されている場合、フィルター操作はデータスコープの条件と結合され、最終的な絞り込み条件として適用されます。

## 操作設定

![20251029173708](https://static-docs.nocobase.com/20251029173708.png)

### フィルター可能なフィールド

デフォルトでは、このコレクションのすべてのフィールドが有効になっています。有効なフィールドのみが条件フィールドリストで選択可能です。

リレーションテーブル（関連コレクション）のフィールドもフィルターフィールドとして使用できます。
