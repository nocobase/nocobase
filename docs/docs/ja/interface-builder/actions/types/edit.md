# 編集

## 概要

「編集」アクションは、現在のレコードをコンテキストとして持つ行ボタンです。ポップアップ形式で表示され、ユーザーはポップアップ内でブロックを自由にカスタマイズ・設定できます。

![20251029165442](https://static-docs.nocobase.com/20251029165442.png)

必要に応じてブロックを設定します。

![20251029165514](https://static-docs.nocobase.com/20251029165514.png)

## アクション設定

![20251029165620](https://static-docs.nocobase.com/20251029165620.png)

- [連携ルール](/interface-builder/actions/action-settings/linkage-rule)：ボタンの表示/非表示を動的に切り替えます。
- [編集ボタン](/interface-builder/actions/action-settings/edit-button)：ボタンのタイトル、タイプ、アイコンを編集します。
- [編集ポップアップ](/interface-builder/actions/action-settings/edit-popup)：開く方法とサイズを設定します。
