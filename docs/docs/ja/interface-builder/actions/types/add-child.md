# 子レコードを追加

## はじめに

この操作は、**ツリーテーブルブロック**専用に設計されており、子レコードを素早く追加するために利用できます。子レコードを追加する際、選択された行が親ノードとして扱われるため、ユーザーは任意の親ノードの下に簡単に子レコードを作成できます。この機能により、階層データの管理プロセスが大幅に簡素化され、データ構造の拡張と整理がより効率的かつ直感的に行えるようになります。
