# アクション

UIにおいて、アクションとは、一連の命令をトリガーするためのボタンです。ページ、ダイアログボックス、ドロワーに直接配置できるほか、ブロックと組み合わせて使用することも可能です。現在、動的に設定可能なアクションは、主にブロックと組み合わせて使用されます。
