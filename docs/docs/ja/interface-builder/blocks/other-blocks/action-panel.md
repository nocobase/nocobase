# ブロック：ワークベンチ

## はじめに

様々なクイック操作を配置できるブロックです。現在、以下の操作に対応しています。

- [リンク](/interface-builder/actions/types/link)
- [QRコードスキャン]
- [ポップアップ操作](/interface-builder/actions/types/pop-up)

![20251023225754](https://static-docs.nocobase.com/20251023225754.png)

> 💡 今後のバージョンでは、より多くの操作タイプが継続的に追加される予定です。

## 操作タイプ

- **リンク**：指定されたページまたは外部URLへ移動します。

![20251023230643](https://static-docs.nocobase.com/20251023230643.png)

- **QRコードスキャン**：QRコードをスキャンして、対応する操作をトリガーします。

- **ポップアップを開く**：ブロックコンテンツを表示するためにポップアップを開きます。
  ![20251023230522](https://static-docs.nocobase.com/20251023230522.png)
