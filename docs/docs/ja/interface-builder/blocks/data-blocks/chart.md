# チャート

## はじめに

NocoBase のデータビジュアライゼーションプラグインは、視覚的なデータクエリと豊富なチャートコンポーネントを提供します。ユーザーは簡単な設定で、迅速に可視化ダッシュボードを構築し、データインサイトを表示できます。また、多次元データ分析と表示にも対応しています。
