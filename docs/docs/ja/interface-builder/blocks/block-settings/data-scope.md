# データ範囲の設定

## はじめに

データ範囲の設定とは、データブロックにデフォルトのフィルター条件を定義することです。ユーザーはビジネス要件に応じてデータ範囲を柔軟に調整できますが、どのようなフィルター操作を行っても、システムはこのデフォルトのフィルター条件を自動的に適用し、データが常に指定された範囲の制限内に収まるようにします。

## 利用ガイド

![20251027110053](https://static-docs.nocobase.com/20251027110053.png)

フィルターフィールドでは、現在のコレクションのフィールドや、関連するコレクションのフィールドを選択できます。

![20251027110140](https://static-docs.nocobase.com/20251027110140.png)

### オペレーター

フィールドのタイプによって、サポートされるオペレーターは異なります。例えば、テキストフィールドでは「等しい」「等しくない」「含む」などのオペレーターをサポートし、数値フィールドでは「より大きい」「より小さい」などのオペレーターをサポートします。また、日付フィールドでは「範囲内」「特定の日付より前」などのオペレーターをサポートしています。

![20251027111124](https://static-docs.nocobase.com/20251027111124.png)
