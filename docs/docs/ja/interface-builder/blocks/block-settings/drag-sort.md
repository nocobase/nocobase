# ドラッグ＆ドロップ並べ替え

## 紹介

ドラッグ＆ドロップ並べ替えは、並べ替えフィールドを使用して、ブロック内のレコードを手動で並べ替える機能です。


:::info{title=ヒント}
* 同じ並べ替えフィールドを複数のブロックでドラッグ＆ドロップ並べ替えに使用すると、既存の順序が崩れる可能性があります。
* テーブルのドラッグ＆ドロップ並べ替えでは、並べ替えフィールドにグルーピングルールを設定することはできません。
* ツリーテーブルは、同じ階層内のノードの並べ替えのみをサポートしています。

:::


## 並べ替えの設定

「並べ替え」タイプのフィールドを追加します。コレクション作成時に並べ替えフィールドは自動生成されなくなったため、手動で作成する必要があります。

## 並べ替えルールの説明

現在の順序が以下のようであると仮定します。

```
[1,2,3,4,5,6,7,8,9]
```

ある要素（例：5）を 3 の位置まで前方に移動させると、3、4、5 の並べ替え値のみが変化します。5 が 3 の位置を占め、3 と 4 はそれぞれ 1 つずつ後方に移動します。

```
[1,2,5,3,4,6,7,8,9]
```

次に 6 を 8 の位置まで後方に移動させると、6 が 8 の位置を占め、7 と 8 はそれぞれ 1 つずつ前方に移動します。

```
[1,2,5,3,4,7,8,6,9]
```
