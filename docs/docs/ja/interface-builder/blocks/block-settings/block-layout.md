# レイアウト

## はじめに

フォームブロック、詳細ブロック、リストブロックなどのフォーム系ブロックでは、レイアウト設定が可能です。レイアウトは、垂直レイアウト（ラベルと入力フィールドが1行ずつ配置される）と水平レイアウト（ラベルと入力フィールドが横並びに配置される）の2種類があります。ユーザーは、情報入力や表示が明確かつ効率的になるよう、具体的な要件に合わせてブロックのフィールドレイアウトを調整できます。

### 垂直（デフォルト）

デフォルトのレイアウトは垂直です。

![20251024120143](https://static-docs.nocobase.com/20251024120143.png)

### 水平

水平レイアウトでは、ラベルと入力フィールドが横並びに表示され、スペースを効果的に節約できます。タイトルが長いフィールドの場合、インターフェースの整理と美しさを保つために、省略または改行を設定できます。

![20251024120237](https://static-docs.nocobase.com/20251024120237.png)

#### フィールドタイトルの配置

- **左揃え**（デフォルト）
- **右揃え**

#### フィールドタイトルの幅

- デフォルトの幅は **120px**、最小幅は **50px** です。

#### フィールドタイトルが幅を超える場合の処理

- **改行**
- **省略**

#### コロンの表示

フィールドタイトルの後にコロンを表示するかどうかを選択できます。
