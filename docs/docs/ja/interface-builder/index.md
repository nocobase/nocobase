# UIビルダー

## WYSIWYG（ウィジウィグ）

NocoBaseでは、WYSIWYG（ウィジウィグ）方式のUI構築が可能です。UIビルダーボタンをクリックすると、設定モードと使用モードを切り替えられます。

## UI レイアウト

NocoBase には、デスクトップレイアウトとモバイルレイアウトがあります。デスクトップレイアウトは一般的な管理操作に適しており、スマートフォンではナビゲーションとページコンテンツが自動的に調整されます。モバイルレイアウトでは、モバイル端末向けの独立したナビゲーションとページを設定できます。詳しくは [UI レイアウト](./ui-layout/index.md)を参照してください。

## 設定オプション
