# UIビルダー

## WYSIWYG（ウィジウィグ）

NocoBaseでは、WYSIWYG（ウィジウィグ）方式のUI構築が可能です。UIビルダーボタンをクリックすると、設定モードと使用モードを切り替えられます。

## レイアウトテンプレート

NocoBaseには、デフォルトのレイアウトテンプレートが組み込まれています。上部と左側がナビゲーションエリア、右側がコンテンツエリアとなっています。

![未命名.002](https://static-docs.nocobase.com/未命名.002.jpeg)

## 設定オプション

UIビルダーモードに入ると、設定可能な要素がオレンジ色で表示されます。通常、各要素の設定オプションへの入り口は、その要素の右上隅にあります。ほとんどの要素は、UI上でリアルタイムに設定し、その効果をすぐに確認できます。いくつか例を挙げます。
