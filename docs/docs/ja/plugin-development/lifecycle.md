# ライフサイクル

このセクションでは、プラグインのサーバーサイドとクライアントサイドにおけるライフサイクルフックを整理し、開発者がリソースを適切に登録および解放できるよう支援します。

FlowModelのライフサイクルと比較することで、共通の概念を強調できます。

## 含まれる内容

- プラグインのインストール、有効化、無効化、アンインストール時にトリガーされるコールバック。
- クライアントサイドコンポーネントのマウント、更新、および破棄のタイミング。
- ライフサイクル内での非同期タスクとエラー処理に関する推奨事項。
