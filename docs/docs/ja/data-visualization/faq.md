# よくある質問

## チャート選択に関する質問
### 適切なチャートはどのように選べばよいですか？
答：データと目的に合わせて選択してください。
- 傾向や変化を示す場合：折れ線グラフまたは面グラフ
- 数値の比較：棒グラフまたは横棒グラフ
- 構成比率：円グラフまたはドーナツグラフ
- 相関性や分布：散布図
- 階層構造や進捗状況：ファネルグラフ

その他のチャートタイプについては、[ECharts の例](https://echarts.apache.org/examples)をご参照ください。

### NocoBase はどのようなチャートタイプをサポートしていますか？
答：ビジュアル設定モードでは、一般的なチャート（折れ線グラフ、面グラフ、棒グラフ、横棒グラフ、円グラフ、ドーナツグラフ、ファネルグラフ、散布図など）が組み込まれています。カスタム設定モードでは、ECharts のすべてのチャートタイプをご利用いただけます。

## データクエリに関する質問
### ビジュアル設定モードとSQL設定モードは互換性がありますか？
答：互換性はありません。設定はそれぞれ独立して保存されます。最後に保存された設定モードが有効になります。

## チャートオプションに関する質問
### チャートのフィールドはどのように設定すればよいですか？
答：ビジュアル設定モードでは、チャートタイプに応じて対応するデータフィールドを選択します。例えば、折れ線グラフや棒グラフではX軸とY軸のフィールドを設定する必要があり、円グラフではカテゴリフィールドと値フィールドを設定する必要があります。
まず「クエリを実行」して、データが期待どおりか確認することをお勧めします。デフォルトでは、チャートフィールドは自動的にマッチングされます。

## プレビューと保存に関する質問
### 設定変更後、手動でプレビューする必要がありますか？
答：ビジュアル設定モードでは、設定を変更すると自動的にプレビューされます。SQL設定モードやカスタム設定モードでは、頻繁な更新を避けるため、編集が完了した後に手動で「プレビュー」をクリックしてください。

### ポップアップを閉じるとチャートのプレビュー効果が失われますか？
答：プレビューは一時的な確認のためのものです。設定を変更した場合は、閉じる前に必ず保存してください。保存されていない変更は保持されません。
