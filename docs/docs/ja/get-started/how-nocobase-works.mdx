1. AI Agent を使って NocoBase をインストールし、接続して、そのまま構築を始めたいなら、[クイックスタート](../ai/quick-start.md) に進んでください
2. 先にインストール方法とバージョンチャネルを比べてから決めたいなら、[インストール方法とバージョンの比較](/ai/quickstart) に進んでください
