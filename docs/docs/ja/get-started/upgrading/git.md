# Git ソースインストール版のアップグレード

:::warning アップグレード前の準備

- 必ずデータベースをバックアップしてください。
- 実行中のNocoBaseを停止してください（`Ctrl + C`）。

:::

## 1. NocoBaseプロジェクトディレクトリに移動します

```bash
cd my-nocobase-app
```

## 2. 最新のコードをプルします

```bash
git pull
```

## 3. キャッシュと古い依存関係を削除します（オプション）

通常のアップグレードプロセスが失敗した場合は、キャッシュと依存関係をクリアしてから再ダウンロードを試すことができます。

```bash
# NocoBaseキャッシュをクリア
yarn nocobase clean
# 依存関係を削除
yarn rimraf -rf node_modules # rm -rf node_modules と同等
```

## 4. 依存関係を更新します

📢 ネットワーク環境やシステム設定などの要因により、このステップには10分以上かかる場合があります。

```bash
yarn install
```

## 5. アップグレードコマンドを実行します

```bash
yarn nocobase upgrade
```

## 6. NocoBaseを起動します

```bash
yarn dev
```

:::tip 本番環境に関するヒント

ソースコードからインストールしたNocoBaseを本番環境に直接デプロイすることは推奨されません（本番環境については、[本番環境へのデプロイ](../deployment/production.md) を参照してください）。

:::
