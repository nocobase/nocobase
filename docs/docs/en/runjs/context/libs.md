# ctx.libs

Built-in library collection in RunJS. Use it for JSX rendering (`ctx.render`) or custom rendering with `ctx.element`, rather than native HTML elements.

Common sub-properties:

- `ctx.libs.antd`: Ant Design component library (Button, Card, Table, Form, Input, Modal, Space / Row / Col, etc.)
- Other built-in libraries (e.g. dayjs, lodash, echarts) depend on the runtime environment

See the docs for each sub-module.
