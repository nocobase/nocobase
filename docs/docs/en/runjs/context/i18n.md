# ctx.i18n

The i18n instance for the current context. Use `ctx.t()` for translations.

## Type definition

```typescript
interface i18n: {
  language: string;
  changeLanguage(lng: string): Promise<any>;
}
```

## Properties

| Property | Type | Description |
|------|------|------|
| `language` | string | Current language code (e.g. `zh-CN`, `en-US`) |

## Methods

### changeLanguage(lng)

Switch the current language.

| Parameter | Type | Description |
|------|------|------|
| `lng` | string | Target language code (e.g. `'en-US'`, `'zh-CN'`) |

**Return**: Promise resolved after the language has switched.

## Example

```javascript
// Read current language
const current = ctx.i18n.language;
console.log(current); // e.g. 'zh-CN'

// Switch language
await ctx.i18n.changeLanguage('en-US');
await ctx.i18n.changeLanguage('zh-CN');
```
