# Internationalization

## Internationalization Files

In a plugin, both frontend and backend locale files are stored in the `src/locale` folder. Click here to view [all languages](/development/others/languages) supported by NocoBase.

```bash
|- /plugin-i18n
  |- /src
    |- /locale      # locale folder
      |- en-US.ts   # English language
      |- zh-CN.ts   # Chinese language
```

Just add the translation keys to the corresponding locale file (`/src/locale/${lang}.ts`). If it's the first time adding a locale file, you need to restart the application for it to take effect. You can check the `app:getLang` endpoint to verify if the translation keys have been added successfully.

Default URL: http://localhost:13000/api/app:getLang?locale=zh-CN

## Server

### app.i18n <Badge>server</Badge>

`app.i18n` is the global i18n instance, generally used in the CLI. For example, it can be used with inquirer to implement command-line interactions.

```ts
import select from '@inquirer/select';
import input from '@inquirer/input';

export class PluginSampleI18nServer extends Plugin {
  load() {
    this.app.command('test-i18n').action(async () => {
      const answer1 = await select({
        message: 'Select a language',
        choices: [
          {
            name: '中文',
            value: 'zh-CN',
          },
          {
            name: 'English',
            value: 'en-US',
          },
        ],
      });
      await this.app.changeLanguage(answer1);
      const answer2 = await input({
        message: app.i18n.t('Enter your name'),
      });
      console.log(app.i18n.t(`Your name is {{name}}`, { name: answer2 }));
    });
  }
}
```

### ctx.i18n <Badge>server</Badge>

`ctx.i18n` is a cloneInstance of the global `app.i18n`. The `ctx` for each request is completely independent, responding with multilingual information based on the client's language.

Client request parameters can be placed in the query string

```bash
GET /?locale=en-US HTTP/1.1
Host: localhost:13000
```

They can also be placed in the request headers (recommended)

```bash
GET / HTTP/1.1
Host: localhost:13000
X-Locale: en-US
```

Example

```ts
export class PluginSampleI18nServer extends Plugin {
  load() {
    this.app.use(async (ctx, next) => {
      if (ctx.path === '/api/test-i18n') {
        ctx.body = `${ctx.i18n.t('Hello')} ${ctx.i18n.t('World')}`;
      }
      await next();
    });
  }
}
```

Check http://localhost:13000/api/test-i18n?locale=zh-CN
