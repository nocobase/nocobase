# Lifecycle

Outlines the server-side and client-side lifecycle hooks for plugins, helping developers correctly register and release resources.

It can be compared to the FlowModel lifecycle to highlight common concepts.

## Suggested Content

- Callbacks triggered during plugin installation, enabling, disabling, and uninstallation.
- The timing of client-side component mounting, updating, and destruction.
- Recommendations for handling asynchronous tasks and errors within the lifecycle.
