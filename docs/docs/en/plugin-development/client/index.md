# Overview

NocoBase client-side plugin development provides various functionalities and capabilities to help developers customize and extend NocoBase's frontend features. The following are the main capabilities and related chapters of NocoBase client-side plugin development:

| Module              | Description                                      | Related Chapter                              |
| ------------------- | ------------------------------------------------ | -------------------------------------------- |
| **Plugin Class**    | Create and manage client-side plugins, extend frontend functionality | [plugin.md](plugin.md)                      |
| **Router**          | Customize frontend routing, implement page navigation and redirects | [router.md](router.md)                      |
| **Resource**        | Manage frontend resources, handle data fetching and operations | [resource.md](resource.md)                   |
| **Request**         | Customize HTTP requests, handle API calls and data transmission | [request.md](request.md)                     |
| **Context**         | Get and use application context, access global state and services | [context.md](context.md)                    |
| **ACL**             | Implement frontend access control, control page and feature access permissions | [acl.md](acl.md)                            |
| **DataSourceManager** | Manage and use multiple data sources, implement data source switching and access | [data-source-manager.md](data-source-manager.md) |
| **Styles & Themes** | Customize styles and themes, implement UI customization and beautification | [styles-themes.md](styles-themes.md)         |
| **I18n**            | Integrate multi-language support, implement internationalization and localization | [i18n.md](i18n.md)                           |
| **Logger**          | Customize log formats and output methods, enhance debugging and monitoring capabilities | [logger.md](logger.md)                      |
| **Test**            | Write and run test cases, ensure plugin stability and functional accuracy | [test.md](test.md)                           |
