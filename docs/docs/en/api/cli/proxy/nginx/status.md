# nb proxy nginx status

Show the current Nginx proxy runtime status.

## Usage

```bash
nb proxy nginx status
```

## Output

The output usually includes:

- `driver`
- `status`
- `config`
- `nginx` or `container`

## Examples

```bash
nb proxy nginx status
```
