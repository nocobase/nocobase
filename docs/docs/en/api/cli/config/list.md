# nb config list

List explicitly configured CLI settings. Default values that were never set explicitly are not displayed.

## Usage

```bash
nb config list
```

## Examples

```bash
nb config list
```

## Notes

If no configuration values are set, the CLI prints `No CLI config values are set.`.
