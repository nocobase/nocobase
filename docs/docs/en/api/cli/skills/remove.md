# nb skills remove

Remove global NocoBase AI coding skills managed by `nb`.

## Usage

```bash
nb skills remove [flags]
```

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `--yes`, `-y` | boolean | Skip removal confirmation |
| `--json` | boolean | Output JSON |
| `--verbose` | boolean | Show detailed removal output |

## Examples

```bash
nb skills remove
nb skills remove --yes
nb skills remove --json
```
