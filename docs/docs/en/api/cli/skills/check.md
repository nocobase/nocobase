# nb skills check

Check global NocoBase AI coding skills and report whether they are managed by the CLI and whether updates are available.

## Usage

```bash
nb skills check [flags]
```

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `--json` | boolean | Output JSON |

## Examples

```bash
nb skills check
nb skills check --json
```
