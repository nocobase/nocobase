# CRM Solution

> **Version Note**: The current version runs on NocoBase 2.0 beta, but the business logic and data table design still follow version 1.x. Only partial frontend UI improvements have been made. It is not fully complete and is **for preview only**. In future versions, we will completely rebuild the CRM solution, providing more complete functionality and a better user experience.

## Overview

NocoBase CRM is a no-code, flexible, and powerful customer relationship management platform. It's ready to use out of the box, offers ultimate customization capabilities, and ensures complete data sovereignty. You can extend fields, blocks, features, and pages to fully meet your business needs.

## Key Features

### Completely No-Code

No programming skills required to customize your CRM.

### Comprehensive CRM Functions

- **Customer Management**: Customer information, contacts, customer grading
- **Sales Pipeline**: Lead management, opportunity tracking, sales funnel
- **Order Management**: Quotations, orders, contracts
- **Communication & Collaboration**: Email communication, comments, approval workflows
- **Data Analysis**: Dashboards, reports, data visualization

### Fully Customizable and Extensible

Support for custom fields, workflows, and business logic. Flexible architecture makes "secondary development" simple.

### Open Source, Full Data Control

Complete control over system code and data, ensuring enterprise data security.

## Core Modules

| Module | Description |
|--------|-------------|
| Lead Management | Lead entry, distribution, conversion |
| Customer Management | Customer profiles, contact management, customer grading |
| Opportunity Management | Opportunity tracking, sales stages, won/lost |
| Order Management | Quotations, order creation, contract management |
| Dashboard | Sales funnel, performance statistics, trend analysis |

```

```
