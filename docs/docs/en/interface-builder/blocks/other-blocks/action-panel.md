# Block: Workbench

## Introduction

Used to place various quick actions. Currently supports:

- [Link](/interface-builder/actions/types/link)
- [Scan QR Code]
- [Popup Action](/interface-builder/actions/types/pop-up)

## Action Types

- **Link**: Navigates to a specified page or external URL.


![20251023230643](https://static-docs.nocobase.com/20251023230643.png)


- **Scan QR Code**: Triggers a corresponding action by scanning a QR code.

- **Open Popup**: Opens a popup to display block content.
