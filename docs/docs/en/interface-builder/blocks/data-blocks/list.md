# List Block

## Introduction

The List block displays data in a list format, suitable for data display scenarios such as task lists, news, and product information.

## Block Configuration


![20251023202835](https://static-docs.nocobase.com/20251023202835.png)


### Set Data Scope

As shown: Filter for orders with a "Cancelled" status.


![20251023202927](https://static-docs.nocobase.com/20251023202927.png)


For more details, see [Set Data Scope](/interface-builder/blocks/block-settings/data-scope)

### Set Sorting Rules

As shown: Sort by order amount in descending order.


![20251023203022](https://static-docs.nocobase.com/20251023203022.png)


For more details, see [Set Sorting Rules](/interface-builder/blocks/block-settings/sorting-rule)

## Configure Fields

### Fields from this Collection

> **Note**: Fields from inherited collections (i.e., parent collections) are automatically merged and displayed in the current field list.


![20251023203103](https://static-docs.nocobase.com/20251023203103.png)


### Fields from Associated Collections

> **Note**: Fields from associated collections can be displayed (currently only to-one relationships are supported).
