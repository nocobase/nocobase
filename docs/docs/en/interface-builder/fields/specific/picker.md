# Data Selector

## Introduction

The Data Selector, in the form of a popup table selector, supports selecting existing data from the target collection for association or adding data to the target collection and then associating it.


![20251027203343](https://static-docs.nocobase.com/20251027203343.png)


Selected data will not be displayed in the table selector.


![20251027203717](https://static-docs.nocobase.com/20251027203717.gif)


In the table selector, you can also further manage the target association collection (add, delete, import/export, etc.).


![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
