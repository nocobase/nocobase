# File Manager

## Introduction

The File Manager is an association field component for handling relationships where the target collection is the files collection.


![20251027194124](https://static-docs.nocobase.com/20251027194124.png)


## Field Configuration

- [Field Component](/interface-builder/fields/association-field): Switch to other association field components, such as Dropdown, Data Selector, etc.
- Allow selecting existing files: Enabled by default, allows selecting and associating from uploaded files.
