# Details Fields

## Introduction

Field configurations for Details, List, and Grid blocks are basically the same, primarily controlling how fields are displayed in read-only mode.


![20251025172851](https://static-docs.nocobase.com/20251025172851.png)


## Field Configuration Options

### Date Field Formatting


![20251025173005](https://static-docs.nocobase.com/20251025173005.png)


For more details, see [Date Formatting](/interface-builder/fields/specific/date-picker)

### Number Field Formatting


![20251025173242](https://static-docs.nocobase.com/20251025173242.png)


Supports simple unit conversion, thousand separators, prefixes and suffixes, precision, and scientific notation.

For more details, see [Number Formatting](/interface-builder/fields/field-settings/number-format)

### Enable Click to Open

Besides association fields, regular fields can also be configured to open a pop-up on click. You can also set how the pop-up opens (Drawer, Dialog, or Sub-page).


![20251025173549](https://static-docs.nocobase.com/20251025173549.gif)


### Content Overflow Display Mode

When the field content overflows its width, you can set the overflow mode.

- Ellipsis (default)
- Wrap


![20251025173917](https://static-docs.nocobase.com/20251025173917.png)


### Field Component

Some fields support multiple display formats, which can be achieved by switching the field component.

For example: The `URL` component can be switched to the `Preview` component.


![20251025174042](https://static-docs.nocobase.com/20251025174042.png)


For example, association fields can be displayed differently by switching from the title field component to `Sub-details` to show more content from the relationship.


![20251025174311](https://static-docs.nocobase.com/20251025174311.gif)
