# Table Fields

## Introduction

In addition to adjusting column width, field title, sorting, etc., table fields also support more personalized display configurations for specific fields (such as date fields, association fields, and number fields).


![20251024174558](https://static-docs.nocobase.com/20251024174558.png)


## Field Configuration Options

### Date Field Formatting


![20251024175303](https://static-docs.nocobase.com/20251024175303.png)


For more details, see [Date Formatting](/interface-builder/fields/specific/date-picker)

### Number Field Formatting

Supports simple unit conversion, thousand separators, prefixes and suffixes, precision, and scientific notation.


![20251024175445](https://static-docs.nocobase.com/20251024175445.png)


For more details, see [Number Formatting](/interface-builder/fields/field-settings/number-format)

### Enable Quick Edit

When quick edit is enabled, an edit button will appear on mouse hover over the column. Click it to quickly edit and save data in a pop-up window.


![20251025171158](https://static-docs.nocobase.com/20251025171158.gif)


### Enable Click to Open

In addition to association fields that support clicking to open a pop-up, regular fields can also be enabled with "Click to Open" to serve as an entry point for opening a pop-up.


![20251025172308](https://static-docs.nocobase.com/20251025172308.gif)


### Content Overflow Display Mode

When the column content overflows the table column width, you can set the overflow mode:

-   Ellipsis (default)
-   Wrap


![20251025172549](https://static-docs.nocobase.com/20251025172549.png)


### Fixed Column


![20251025170858](https://static-docs.nocobase.com/20251025170858.gif)


### Field Component

Some fields support multiple display formats. Users can switch field components to achieve different display effects to meet the needs of different scenarios. For example, a **URL** type field can be switched to a **Preview** component to better display link content or image previews.


![20251025171658](https://static-docs.nocobase.com/20251025171658.png)


### Display Style

-   Tag
-   Text


![20251025172723](https://static-docs.nocobase.com/20251025172723.png)

### Sortable

Most field types currently support server-side sorting. When sorting is enabled, data can be sorted in ascending or descending order based on the target field.
