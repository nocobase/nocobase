# Table Fields

## Introduction

In addition to adjusting column width, field titles, and sorting, table fields support more personalized display options for specific field types, such as date, association, and number fields.


![20251024174558](https://static-docs.nocobase.com/20251024174558.png)


## Field Configuration Options

### Date Field Formatting


![20251024175303](https://static-docs.nocobase.com/20251024175303.png)


For more details, see [Date Formatting](/interface-builder/fields/specific/date-picker)

### Number Field Formatting

Supports simple unit conversion, thousands separators, prefixes and suffixes, precision, and scientific notation.


![20251024175445](https://static-docs.nocobase.com/20251024175445.png)


For more details, see [Number Formatting](/interface-builder/fields/field-settings/number-format)

### Enable Quick Edit

When Quick Edit is enabled, an edit button appears when you hover over the column. Clicking it allows you to quickly edit and save data in a pop-up window.


![20251025171158](https://static-docs.nocobase.com/20251025171158.gif)


### Enable Click to Open

In addition to association fields that support 'Click to Open' for pop-ups, regular fields can also be configured with this option to serve as entry points for opening pop-ups.


![20251025172308](https://static-docs.nocobase.com/20251025172308.gif)


### Content Overflow Display Mode

When the column content exceeds the table column width, you can configure the overflow mode:

-   Ellipsis (default)
-   Wrap


![20251025172549](https://static-docs.nocobase.com/20251025172549.png)

### Field Component

Some fields support multiple display formats. Users can switch field components to achieve different display styles to meet various use cases. For example, a **URL** type field can be switched to a **Preview** component to better display link content or image previews.


![20251025171658](https://static-docs.nocobase.com/20251025171658.png)
