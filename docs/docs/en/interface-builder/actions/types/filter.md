# Filter

## Introduction

The Filter action usually appears in data blocks to filter data by configuring various conditions. Note that if a data scope is already configured for the block, the filter action will be combined with the data scope conditions as the final filtering condition.

### Filterable Fields

By default, all fields of this collection are enabled. Only enabled fields can be selected in the condition field list.

Fields from associated collections can also be used as filter fields.
