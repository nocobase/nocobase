# Bulk edit

## Introduction

Bulk edit is designed for flexible batch data updates. After clicking the bulk edit button, configure the bulk edit form in the modal and set update strategies for each field.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM.png)


## Operation settings

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM%20(1).png)


## Guide

### Bulk edit form setup

1. Add the bulk edit button.

2. Set the scope for bulk edit: selected/all, default is selected.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_14_AM.png)

3. Add the bulk edit form.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_14_AM.png)

4. Configure the fields to edit and add a submit button.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM%20(1).png)

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM.png)

### Submit the form

1. Select the rows to edit.

2. Choose an edit mode for each field and enter the values to submit.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_10_33_AM.png)

:::info{title=Available edit modes}
* Remains the same: keep the field unchanged
* Changed to: update the field to the submitted value
* Clear: clear the field value

:::


3. Submit the form.
