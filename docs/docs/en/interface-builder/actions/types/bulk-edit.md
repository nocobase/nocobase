# Bulk Edit

## Introduction

Bulk Edit is suitable for scenarios requiring flexible batch updates of data. After clicking the Bulk Edit button, you can configure the bulk edit form in a pop-up window and set different update strategies for each field.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM.png)

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM%20(1).png)


## User Guide

### Bulk Edit Form Configuration

1. Add a Bulk Edit button.

2. Set the scope for bulk editing: Selected / All, defaults to Selected.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_14_AM.png)

3. Add a bulk edit form.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_14_AM.png)

4. Configure the fields to be edited and add a submit button.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM%20(1).png)

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM.png)

### Form Submission

1. Select the rows of data to be edited.

2. Select the edit mode for the fields and fill in the values to be submitted.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_10_33_AM.png)

:::info{title=Available Edit Modes}
* **Do not update**: The field remains unchanged.
* **Change to**: Update the field to the submitted value.
* **Clear**: Clear the data in the field.

:::

3. Submit the form.
