# Confirmation

## Introduction

The confirmation configuration is used to confirm whether an action is executed. A confirmation dialog pops up when the action is clicked. The delete button has confirmation enabled by default to prevent data loss or irreversible results due to accidental operations.


![20251029125021](https://static-docs.nocobase.com/20251029125021.png)


It supports customizing the confirmation content and title, and accessing variables.
