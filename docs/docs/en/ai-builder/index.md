# AI Builder

AI Builder is NocoBase's AI-assisted building capability — you can describe your requirements in natural language, and the AI will automatically handle data modeling, page configuration, permission setup, and more.

For example, tell the AI "Create a customer management page with name, phone number, and follow-up status," and it will set up the data table, list block, and form all at once. No need to manually drag and configure each element.

:::tip

The configuration generated by AI Builder is identical to what you'd create manually. You can adjust and modify it anytime through the visual interface.

:::

## Agent Skills

Agent Skills are domain knowledge packages that can be installed into AI Agents. Once installed, AI tools (such as Claude Code, Codex, Cursor, Windsurf, etc.) can understand NocoBase's configuration system and help you complete building tasks.

### Install all skills at once

```sh
npx skills add nocobase/skills -y
```

### Install individual Skills as needed

If you only need a specific functional domain, you can install it separately:

| Skill | Install Command | Description |
|-------|----------------|-------------|
| [Installation & Upgrade](./env-bootstrap) | `npx skills add nocobase/skills --skill nocobase-env-bootstrap -y` | Environment checks, installation, deployment, upgrades, and troubleshooting |
| [Data Modeling](./data-modeling) | `npx skills add nocobase/skills --skill nocobase-data-modeling -y` | Create and manage data tables, fields, and relationships |
| [UI Configuration](./ui-builder) | `npx skills add nocobase/skills --skill nocobase-ui-builder -y` | Create and edit pages, blocks, popups, and interactive linkages |
| [Workflow Management](./workflow) | `npx skills add nocobase/skills --skill nocobase-workflow-manage -y` | Create, edit, enable, and diagnose workflows |
| [ACL Configuration](./acl) | `npx skills add nocobase/skills --skill nocobase-acl-manage -y` | Manage roles, permission policies, user bindings, and risk assessments |
| [Solutions](./dsl-reconciler) | `npx skills add nocobase/skills --skill nocobase-dsl-reconciler -y` | Batch-build entire business systems from YAML |
| [Plugin Management](./plugin-manage) | `npx skills add nocobase/skills --skill nocobase-plugin-manage -y` | View, install, enable, and disable plugins |
| [Release Management](./publish) | `npx skills add nocobase/skills --skill nocobase-publish-manage -y` | Cross-environment releases, backup & restore, and migration |
