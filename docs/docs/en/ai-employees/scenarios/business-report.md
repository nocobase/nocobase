# Generate Business Analysis Report

Using the online DEMO CRM system as an example, let AI Employees analyze customer and order data in the system and generate an analysis report.

## Load Skills

First, the AI Employee will load the relevant skills, then start working.

![](https://static-docs.nocobase.com/202604230924035.png)

## Analyze Data Models

Retrieve data sources and collection metadata.

![](https://static-docs.nocobase.com/202604230926297.png)

## Query Business Data

Start querying business data. Depending on the content being analyzed, this process usually takes from tens of seconds to a few minutes.

![](https://static-docs.nocobase.com/202604230927861.png)

## Generate Report

After collecting enough business data, start generating the analysis report.

![](https://static-docs.nocobase.com/202604230920113.png)

The final result is a richly illustrated business analysis report that can be downloaded in Markdown, HTML, or PDF format.

![](https://static-docs.nocobase.com/202604230922030.png)
