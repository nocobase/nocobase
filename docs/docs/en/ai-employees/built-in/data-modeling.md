> Intelligently design collections and optimize database structure.

`Orin` is the built-in data model expert. On the main data source configuration page, you can have `Orin` help you create or modify collections.


![20251022160628](https://static-docs.nocobase.com/20251022160628.png)


## Steps

Go to the Data Source Manager plugin and select to configure the main data source.


![20251022161146](https://static-docs.nocobase.com/20251022161146.png)


Click the `Orin` avatar in the upper right corner to open the AI Agent dialog box.


![20251022161641](https://static-docs.nocobase.com/20251022161641.png)


Describe your modeling requirements to `Orin`, send it, and wait for a response.


![20251022161749](https://static-docs.nocobase.com/20251022161749.png)


After `Orin` confirms your requirements, it will use its skills and reply with a preview of the data model.

After reviewing the preview, click the `Finish review and apply` button to create collections according to `Orin`'s model.


![20251022162142](https://static-docs.nocobase.com/20251022162142.png)
