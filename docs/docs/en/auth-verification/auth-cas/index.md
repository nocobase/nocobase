# Authentication: CAS

<PluginInfo commercial="true" name="auth-cas"></PluginInfo>

## Introduction

The Authentication: CAS plugin follows the CAS (Central Authentication Service) protocol standard, allowing users to log in to NocoBase using accounts provided by a third-party Identity Provider (IdP).
