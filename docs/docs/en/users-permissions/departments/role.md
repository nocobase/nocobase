# Department Role Management

By associating one or more roles with a department, members of that department can have these roles.

There are two ways to associate roles with a department.

The first way is to select roles in the Role field of the department's edit form.


![](https://static-docs.nocobase.com/70f77bb89aa1fb415c152a51a51cc23b.png)


The second way is to add departments to the corresponding role in Role Management.


![](https://static-docs.nocobase.com/f2a7bec937cf2f179ce868a92b98416d.png)


Departments that already have this role cannot be selected.


![](https://static-docs.nocobase.com/be10299893581e1f97a4e01ddd5c7e59.png)
