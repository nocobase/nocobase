# Email Manager

## Introduction
The Email Manager plugin is an efficient and convenient tool that supports authorization and connection for Gmail and Outlook mailboxes. It helps users integrate email management capabilities, such as sending and receiving emails, into various blocks and pages. Through simple authorization configuration, users can achieve unified management of multiple accounts and enjoy a seamless email communication experience.
