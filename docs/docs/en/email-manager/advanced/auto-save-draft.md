# Auto-save Draft

In the **Save Draft** button's menu, you can enable **Auto-save draft**.  

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_29_PM.png)


During the email editing process, the system saves the draft in real-time, ensuring that no content is lost.  

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_31_PM.png)


When you open the email editor again (for the same sender and recipient), the system will ask if you want to use the saved draft.  

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_08_35_PM.png)
