# Bulk Sending

## Bulk Sending Overview
You can send emails to multiple people in bulk (each recipient will receive a separate email).  

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_06_10_PM.png)


## Bulk Sending via Table
You can also send emails in bulk via a table:

1. Add a send email button to the table
   
![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_14_PM.png)


2. Configure the email field in the table (used to parse email addresses after selecting table rows)  
   
![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_13_PM.png)


3. Check the table data to send emails
   
![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_15_PM.png)


If you need to send emails to all addresses in the entire table, you can enable the **Select all records in the current table as recipients by default** feature:  

![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_16_PM.png)


After editing the email content, click the bulk send button to send:  

![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_17_PM.png)
