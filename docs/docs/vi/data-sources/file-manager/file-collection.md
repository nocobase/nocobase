# Bộ sưu tập Tệp

## Giới thiệu

Tệp là các bản ghi trong một bộ sưu tập có cấu trúc đặc biệt, được gọi là bộ sưu tập tệp. Bộ sưu tập này dùng để lưu trữ siêu dữ liệu của tệp và có thể được quản lý thông qua Trình quản lý Tệp.
