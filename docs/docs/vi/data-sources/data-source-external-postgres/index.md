# Nguồn Dữ Liệu Ngoài - PostgreSQL

## Giới Thiệu

Sử dụng cơ sở dữ liệu PostgreSQL bên ngoài làm nguồn dữ liệu. Các phiên bản hiện được hỗ trợ: PostgreSQL >= 9.5

## Cài Đặt

Plugin này là một plugin thương mại, cần được tải lên và kích hoạt thông qua trình quản lý plugin.
