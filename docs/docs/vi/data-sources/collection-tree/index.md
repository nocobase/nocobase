# Bảng cây

## Giới thiệu

Bảng cấu trúc cây là một mẫu thiết kế Collection dùng để lưu trữ dữ liệu có quan hệ phân cấp. Cấu trúc bảng này giống như cấu trúc cây, trong đó mỗi mục dữ liệu có thể có một hoặc nhiều mục con, và các mục con này lại có thể có các mục con của riêng chúng.
