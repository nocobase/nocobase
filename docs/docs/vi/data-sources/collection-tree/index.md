# Bộ sưu tập dạng cây

## Giới thiệu

Bộ sưu tập cấu trúc cây là một mẫu thiết kế bộ sưu tập dữ liệu được sử dụng để tổ chức dữ liệu có quan hệ phân cấp. Cấu trúc bộ sưu tập này mô phỏng một cây, trong đó mỗi mục dữ liệu có thể có một hoặc nhiều mục con, và các mục con này lại có thể có các mục con của riêng chúng.
