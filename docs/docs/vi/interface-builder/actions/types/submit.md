# Gửi

## Giới thiệu

Thao tác gửi dùng để lưu dữ liệu biểu mẫu (đặc trưng của khối biểu mẫu). Ngoài ra, thao tác này có thể kết hợp với các luồng công việc để triển khai các quy trình dữ liệu tự động.
