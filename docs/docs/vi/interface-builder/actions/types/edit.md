# Chỉnh sửa

## Giới thiệu

Action chỉnh sửa thuộc nút hàng, có bản ghi hiện tại làm ngữ cảnh, được hiển thị dưới dạng Popup, bạn có thể tùy chỉnh cấu hình Block trong Popup.
