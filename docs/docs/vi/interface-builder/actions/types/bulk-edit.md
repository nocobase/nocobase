# Chỉnh sửa hàng loạt

## Giới thiệu

Chỉnh sửa hàng loạt phù hợp cho các tình huống cần cập nhật dữ liệu hàng loạt một cách linh hoạt. Sau khi nhấp vào nút Chỉnh sửa hàng loạt, bạn có thể cấu hình biểu mẫu chỉnh sửa hàng loạt trong một cửa sổ bật lên và thiết lập các chiến lược cập nhật khác nhau cho từng trường.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM.png)


## Cấu hình thao tác

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM%20(1).png)


## Hướng dẫn sử dụng

### Cấu hình biểu mẫu chỉnh sửa hàng loạt

1. Thêm nút Chỉnh sửa hàng loạt.

2. Thiết lập phạm vi chỉnh sửa hàng loạt: Đã chọn / Tất cả, mặc định là Đã chọn.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_14_AM.png)

3. Thêm biểu mẫu chỉnh sửa hàng loạt.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_14_AM.png)

4. Cấu hình các trường cần chỉnh sửa và thêm nút gửi.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM%20(1).png)

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM.png)

### Gửi biểu mẫu

1. Chọn các hàng dữ liệu cần chỉnh sửa.

2. Chọn chế độ chỉnh sửa cho các trường và điền giá trị cần gửi.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_10_33_AM.png)

:::info{title=Các chế độ chỉnh sửa có sẵn}
* Không cập nhật: Trường này được giữ nguyên.
* Thay đổi thành: Cập nhật trường này thành giá trị đã gửi.
* Xóa trống: Xóa dữ liệu trong trường này.

:::

3. Gửi biểu mẫu.
