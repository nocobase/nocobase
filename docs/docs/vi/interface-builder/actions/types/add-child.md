# Thêm bản ghi con

## Giới thiệu

Action này được thiết kế chuyên cho **Block Table cây**, dùng để nhanh chóng thêm bản ghi con. Khi thêm bản ghi con, hàng được chọn sẽ được coi là node cha, bạn có thể dễ dàng tạo bản ghi con dưới bất kỳ node cha nào. Tính năng này đơn giản hóa đáng kể quy trình quản lý dữ liệu phân cấp, làm cho việc mở rộng và tổ chức cấu trúc dữ liệu hiệu quả và trực quan hơn.
