# Thêm bản ghi con

## Giới thiệu

Thao tác này được thiết kế đặc biệt cho **khối bảng dạng cây** để nhanh chóng thêm các bản ghi con. Khi thêm bản ghi con, hàng được chọn sẽ được coi là nút cha, cho phép người dùng dễ dàng tạo bản ghi con dưới bất kỳ nút cha nào. Tính năng này giúp đơn giản hóa đáng kể quy trình quản lý dữ liệu phân cấp, làm cho việc mở rộng và tổ chức cấu trúc dữ liệu trở nên hiệu quả và trực quan hơn.
