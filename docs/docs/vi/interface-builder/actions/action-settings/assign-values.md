# Gán giá trị Field

## Giới thiệu

Gán giá trị Field nghĩa là đặt trước giá trị Field trước khi thực hiện Action, tự động điền dữ liệu này khi cập nhật hoặc tạo bản ghi, không cần điền thủ công, ví dụ các trường hợp như cập nhật trạng thái đơn hàng hàng loạt, đặt thời gian giao hàng là thời gian hiện tại khi đơn hàng được giao, v.v.
