# File Manager

## Giới thiệu

File Manager là component Field quan hệ dùng để xử lý các Field quan hệ có Table mục tiêu là Table tập tin.

![20251027194124](https://static-docs.nocobase.com/20251027194124.png)

## Tùy chọn cấu hình Field

- [Component Field](/interface-builder/fields/association-field): chuyển sang component Field quan hệ khác, như Select dropdown, Data Picker, v.v.;
- Cho phép chọn tập tin hiện có: mặc định bật, chọn liên kết từ các tập tin đã tải lên;
