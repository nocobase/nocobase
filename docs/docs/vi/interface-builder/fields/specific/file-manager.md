# Trình quản lý tệp

## Giới thiệu

Trình quản lý tệp là một thành phần trường liên kết được sử dụng để xử lý các mối quan hệ mà bộ sưu tập đích là bộ sưu tập tệp.

![20251027194124](https://static-docs.nocobase.com/20251027194124.png)

## Cấu hình trường

- [Thành phần trường](/interface-builder/fields/association-field): Chuyển sang các thành phần trường liên kết khác, ví dụ như Dropdown, Data Selector, v.v.
- Cho phép chọn tệp đã có: Mặc định được bật, cho phép chọn và liên kết từ các tệp đã tải lên.
