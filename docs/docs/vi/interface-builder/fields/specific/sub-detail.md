# Sub-Detail

## Giới thiệu

Sub-Detail là component tương ứng của Sub-Form ở chế độ đọc, so với component Tag và Tiêu đề, Sub-Detail không chỉ có thể hiển thị nhiều dữ liệu của Table này hơn, mà còn có thể cấu hình hiển thị thông tin dữ liệu Table quan hệ, dữ liệu quan hệ nhiều cấp được hiển thị rõ ràng dưới dạng Chi tiết lồng nhau.
