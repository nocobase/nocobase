# Ngày giờ

## Giới thiệu

Field ngày tháng hỗ trợ tùy chỉnh định dạng ngày và giờ.

## Có thể cài đặt định dạng ngày tháng ở đâu

### Field Table dữ liệu


![20240417114311](https://static-docs.nocobase.com/20240417114311.png)

> Định dạng ngày tháng được cài đặt trong Field Table dữ liệu sẽ là định dạng mặc định của Field, có thể bị ghi đè bởi định dạng ngày tháng được cài đặt trong cấu hình Block.
### Cấu hình Field

![20251027191824](https://static-docs.nocobase.com/20251027191824.png)
