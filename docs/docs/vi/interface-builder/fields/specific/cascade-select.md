# Cascade Select

## Giới thiệu

Cascade Select picker phù hợp với Field quan hệ có Table mục tiêu là Table cây, bạn có thể chọn dữ liệu theo cấu trúc phân cấp của Table cây, hỗ trợ tìm kiếm mờ dữ liệu.

## Hướng dẫn sử dụng

- Quan hệ là đến một, cascade là chọn đơn.

![20251125214656](https://static-docs.nocobase.com/20251125214656.png)

- Quan hệ là đến nhiều, cascade là chọn đa.

![20251125215318](https://static-docs.nocobase.com/20251125215318.png)
