# Data Picker

## Giới thiệu

Data Picker dưới dạng Popup picker Table, hỗ trợ chọn dữ liệu hiện có của Table mục tiêu để liên kết hoặc thêm dữ liệu cho Table mục tiêu rồi liên kết.

![20251027203343](https://static-docs.nocobase.com/20251027203343.png)

Dữ liệu đã chọn sẽ không hiển thị trong picker Table.

![20251027203717](https://static-docs.nocobase.com/20251027203717.gif)

Trong picker Table, còn có thể quản lý thêm Table mục tiêu quan hệ (thêm, xóa, nhập xuất, v.v.).

![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
