# Bộ chọn dữ liệu

## Giới thiệu

Bộ chọn dữ liệu, dưới dạng bảng chọn pop-up, hỗ trợ chọn dữ liệu hiện có từ **bộ sưu tập** mục tiêu để liên kết, hoặc thêm dữ liệu mới vào **bộ sưu tập** mục tiêu rồi liên kết.

![20251027203343](https://static-docs.nocobase.com/20251027203343.png)

Dữ liệu đã chọn sẽ không hiển thị trong bảng chọn.

![20251027203717](https://static-docs.nocobase.com/20251027203717.gif)

Trong bảng chọn, bạn cũng có thể quản lý sâu hơn **bộ sưu tập** liên kết mục tiêu (thêm, xóa, nhập/xuất, v.v.).

![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
