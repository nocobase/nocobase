# Chỉnh sửa Mô tả

## Giới thiệu

Mô tả trường là một đoạn văn bản ngắn gọn, hiển thị bằng chữ màu xám nhạt bên dưới thành phần trường. Đoạn mô tả này giúp người dùng hiểu rõ mục đích và các yêu cầu nhập liệu của trường.
