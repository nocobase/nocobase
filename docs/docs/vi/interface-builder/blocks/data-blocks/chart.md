# Biểu đồ

## Giới thiệu

Plugin trực quan hóa dữ liệu của NocoBase cung cấp khả năng truy vấn dữ liệu trực quan và các thành phần biểu đồ phong phú. Người dùng có thể thông qua cấu hình đơn giản để nhanh chóng xây dựng bảng điều khiển trực quan, hiển thị thông tin chi tiết về dữ liệu, đồng thời hỗ trợ phân tích và trình bày dữ liệu đa chiều.
