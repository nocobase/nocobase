# Chart

## Giới thiệu

Plugin trực quan hóa dữ liệu của NocoBase cung cấp truy vấn dữ liệu trực quan và component biểu đồ phong phú. Bạn có thể nhanh chóng thiết lập bảng điều khiển trực quan thông qua cấu hình đơn giản, hiển thị thông tin chi tiết về dữ liệu, đồng thời hỗ trợ phân tích và hiển thị dữ liệu đa chiều.
