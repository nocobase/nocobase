# Grid card

## Giới thiệu

Block Grid card hiển thị thông tin tóm tắt của bản ghi dữ liệu dưới dạng thẻ, hỗ trợ cấu hình số cột theo các kích thước màn hình khác nhau, đảm bảo hiển thị thân thiện trên các thiết bị có kích thước khác nhau.
