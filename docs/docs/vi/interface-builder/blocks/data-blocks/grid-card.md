# Thẻ lưới

## Giới thiệu

Khối Thẻ lưới hiển thị thông tin tóm tắt của các bản ghi dữ liệu dưới dạng thẻ. Khối này hỗ trợ cấu hình số lượng cột cho các kích thước màn hình khác nhau, nhằm đảm bảo hiển thị thân thiện với người dùng trên nhiều thiết bị.
