# Chế độ tải dữ liệu

## Giới thiệu

Chế độ tải dữ liệu mặc định là "Tải tất cả dữ liệu khi điều kiện lọc trống", ở đây "điều kiện lọc" đề cập đến các giá trị trong "Khối bộ lọc". Một tùy chọn khác là "Không tải dữ liệu khi điều kiện lọc trống".
