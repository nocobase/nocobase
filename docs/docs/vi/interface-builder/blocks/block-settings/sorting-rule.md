# Cấu hình Quy tắc Sắp xếp

## Giới thiệu

Cấu hình quy tắc sắp xếp là việc định nghĩa các quy tắc sắp xếp mặc định cho một khối dữ liệu, để dữ liệu trong khối đó sẽ được hiển thị theo một thứ tự cụ thể.
