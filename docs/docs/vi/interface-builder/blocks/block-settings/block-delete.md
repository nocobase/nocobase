# Xóa Khối

## Giới thiệu

Việc xóa một khối sẽ loại bỏ khối đó khỏi trang hiện tại và đây là một thao tác không thể hoàn tác. Nếu có bản sao lưu được tạo trước khi xóa, khối có thể được khôi phục từ bản sao lưu đó.
