# Quản lý Email

## Giới thiệu
Plugin Quản lý Email là một công cụ tiện lợi và hiệu quả, hỗ trợ tích hợp ủy quyền cho hộp thư Gmail và Outlook, giúp người dùng tích hợp các khả năng quản lý Email, gửi nhận Email vào các block và trang. Thông qua cấu hình ủy quyền đơn giản, bạn có thể thực hiện quản lý thống nhất nhiều tài khoản, tận hưởng trải nghiệm giao tiếp Email liền mạch.
