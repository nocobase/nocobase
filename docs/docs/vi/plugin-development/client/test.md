# Test - Kiểm thử

:::tip

Chương này sẽ giới thiệu cách viết và chạy kiểm thử cho phần frontend của Plugin. Nội dung sẽ bao gồm:
- Cấu hình môi trường kiểm thử và các phụ thuộc;
- Sử dụng Vitest và Testing Library để viết kiểm thử đơn vị;
- Cách mô phỏng môi trường runtime frontend của NocoBase;
- Tự động thực thi kiểm thử trong CI.

Chúng tôi sẽ bổ sung các ví dụ chi tiết và best practices trong các phiên bản tiếp theo.

:::
