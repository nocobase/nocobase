# i18n

## File i18n

Trong plugin, các file đa ngôn ngữ của cả phía client và server đều được đặt trong thư mục `src/locale`.

```bash
|- /plugin-i18n
  |- /src
    |- /locale      # Thư mục đa ngôn ngữ
      |- en-US.ts   # Ngôn ngữ tiếng Anh
      |- zh-CN.ts   # Ngôn ngữ tiếng Trung
```

Bạn chỉ cần thêm các mục dịch vào file ngôn ngữ tương ứng (`/src/locale/${lang}.ts`). Nếu là lần đầu thêm file đa ngôn ngữ, bạn cần khởi động lại ứng dụng để có hiệu lực. Bạn có thể xem API `app:getLang` để kiểm tra xem các mục dịch đã được thêm thành công hay chưa.

Địa chỉ mặc định: http://localhost:13000/api/app:getLang?locale=zh-CN

## Server

### app.i18n <Badge>server</Badge>

app.i18n là instance i18n toàn cục, thường được dùng trong CLI. Ví dụ kết hợp với inquirer để triển khai tương tác dòng lệnh.

```ts
import select from '@inquirer/select';
import input from '@inquirer/input';

export class PluginSampleI18nServer extends Plugin {
  load() {
    this.app.command('test-i18n').action(async () => {
      const answer1 = await select({
        message: 'Select a language',
        choices: [
          {
            name: '中文',
            value: 'zh-CN',
          },
          {
            name: 'English',
            value: 'en-US',
          },
        ],
      });
      await this.app.changeLanguage(answer1);
      const answer2 = await input({
        message: app.i18n.t('Enter your name'),
      });
      console.log(app.i18n.t(`Your name is {{name}}`, { name: answer2 }));
    });
  }
}
```

### ctx.i18n <Badge>server</Badge>

ctx.i18n là cloneInstance của app.i18n toàn cục. ctx của mỗi request hoàn toàn độc lập, phản hồi thông tin đa ngôn ngữ theo ngôn ngữ của client.

Tham số request từ client có thể đặt trong query string

```bash
GET /?locale=en-US HTTP/1.1
Host: localhost:13000
```

Hoặc đặt trong request headers (khuyến nghị)

```bash
GET / HTTP/1.1
Host: localhost:13000
X-Locale: en-US
```

Ví dụ

```ts
export class PluginSampleI18nServer extends Plugin {
  load() {
    this.app.use(async (ctx, next) => {
      if (ctx.path === '/api/test-i18n') {
        ctx.body = `${ctx.i18n.t('Hello')} ${ctx.i18n.t('World')}`;
      }
      await next();
    });
  }
}
```

Xem http://localhost:13000/api/test-i18n?locale=zh-CN
