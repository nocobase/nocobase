# Xem trước tệp

Trong các giao diện chứa trường tệp (bao gồm cả trường tệp đính kèm), bạn có thể xem trước tệp bằng cách nhấp vào hình thu nhỏ hoặc biểu tượng của tệp. Chức năng xem trước tích hợp hỗ trợ nhiều loại tệp khác nhau, bao gồm hình ảnh, PDF và hầu hết các loại tệp được trình duyệt hỗ trợ nguyên bản.

![20251129232307](https://static-docs.nocobase.com/20251129232307.png)

Đối với các loại tệp không hỗ trợ xem trước nguyên bản, bạn có thể kích hoạt chức năng xem trước bằng cách cài đặt hoặc mở rộng các plugin xem trước tệp tương ứng. Ví dụ, sau khi cài đặt plugin Xem trước tệp Office, bạn có thể xem trước các tệp Word, Excel và PowerPoint.

Hiện tại, NocoBase cung cấp các plugin xem trước tệp sau:

* [Plugin Xem trước tệp Office](../file-preview/ms-office.md)
