## BelongsToRepository

Interface giống [HasOneRepository](./has-one-repository.md).
`BelongsToRepository` là `Repository` dùng để xử lý quan hệ `BelongsTo`, cung cấp một số phương thức tiện lợi để xử lý quan hệ `BelongsTo`.
