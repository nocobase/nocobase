# nb skills remove

Gỡ NocoBase AI coding skills toàn cục do `nb` quản lý.

## Cách dùng

```bash
nb skills remove [flags]
```

## Tham số

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| `--yes`, `-y` | boolean | Bỏ qua xác nhận gỡ |
| `--json` | boolean | Output dạng JSON |
| `--verbose` | boolean | Hiển thị output gỡ chi tiết |

## Ví dụ

```bash
nb skills remove
nb skills remove --yes
nb skills remove --json
```
