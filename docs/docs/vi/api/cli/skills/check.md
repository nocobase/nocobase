# nb skills check

Kiểm tra NocoBase AI coding skills toàn cục, báo cáo skills có do CLI quản lý hay không và có bản cập nhật nào khả dụng không.

## Cách dùng

```bash
nb skills check [flags]
```

## Tham số

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| `--json` | boolean | Output dạng JSON |

## Ví dụ

```bash
nb skills check
nb skills check --json
```
