# nb skills

Kiểm tra, cài đặt, cập nhật hoặc gỡ NocoBase AI coding skills toàn cục.

## Cách dùng

```bash
nb skills <command>
```

## Lệnh con

| Lệnh | Mô tả |
| --- | --- |
| [`nb skills check`](./check.md) | Kiểm tra NocoBase AI coding skills toàn cục |
| [`nb skills install`](./install.md) | Cài đặt NocoBase AI coding skills toàn cục |
| [`nb skills update`](./update.md) | Cập nhật NocoBase AI coding skills đã cài |
| [`nb skills remove`](./remove.md) | Gỡ NocoBase AI coding skills do `nb` quản lý |

## Ví dụ

```bash
nb skills check
nb skills install --yes
nb skills update --json
nb skills remove --yes
```
