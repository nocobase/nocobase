# nb config list

Liệt kê các thiết lập CLI đã được cấu hình rõ ràng. Những giá trị mặc định chưa từng được đặt rõ ràng sẽ không được hiển thị.

## Cách dùng

```bash
nb config list
```

## Ví dụ

```bash
nb config list
```

## Ghi chú

Nếu chưa có giá trị cấu hình nào được thiết lập, CLI sẽ in ra `No CLI config values are set.`.
