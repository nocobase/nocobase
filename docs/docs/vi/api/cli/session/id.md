# nb session id

Hiển thị id phiên đang có hiệu lực.

Nếu không có `NB_SESSION_ID` khả dụng trong shell hoặc runtime hiện tại, lệnh sẽ hướng dẫn bạn chạy [`nb session setup`](./setup.md) trước, rồi mở shell hoặc runtime mới.

## Cách dùng


nb session id

## Ví dụ


nb session id

## Lệnh liên quan

- [`nb session setup`](./setup.md)
- [`nb env current`](../env/current.md)
