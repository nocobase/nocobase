# nb proxy nginx current

Hiển thị driver hiện tại của provider Nginx.

## Cách dùng

```bash
nb proxy nginx current
```

## Đầu ra

Lệnh sẽ in trực tiếp một trong hai giá trị sau:

- `local`
- `docker`

## Ví dụ

