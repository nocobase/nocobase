# Tùy chỉnh Thương hiệu

## Giới thiệu

Tên thương hiệu mặc định của hệ thống là "NocoBase", chủ yếu hiển thị ở ba vị trí sau:

![20240409111100](https://static-docs.nocobase.com/20240409111100.png)

Dưới cùng của trang đăng nhập:

![20240409111254](https://static-docs.nocobase.com/20240409111254.png)

Vị trí 1 có thể cấu hình thông qua tùy chọn Logo trong [Cài đặt Hệ thống](/system-management/system-settings/); còn vị trí 2 và 3 có thể cấu hình bằng plugin này.

## Hướng dẫn Sử dụng

Sau khi kích hoạt plugin, nhấp vào menu "Tùy chỉnh Thương hiệu" ở góc trên bên phải trang để truy cập trang cấu hình.

![20240409113640](https://static-docs.nocobase.com/20240409113640.png)

- **Thương hiệu**: Dùng để thiết lập văn bản ở cuối trang đăng nhập (vị trí 3)
- **Giới thiệu**: Dùng để cấu hình nội dung menu ở góc trên bên phải trang (như hình bên dưới)

![20240409113915](https://static-docs.nocobase.com/20240409113915.png)

Cấu hình hỗ trợ định dạng HTML để dễ dàng tùy chỉnh nội dung. Nếu bạn chỉ cần thay đổi văn bản đơn giản, có thể trực tiếp chỉnh sửa dựa trên nội dung hiện có, như hình bên dưới:

![20240409114740](https://static-docs.nocobase.com/20240409114740.png)

Thiết lập biểu tượng trang web (favicon)

![20250601143958_rec_](https://static-docs.nocobase.com/20250601143958_rec_.gif)
