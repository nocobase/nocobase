# Quản lý plugin

## Trình quản lý plugin

Click icon Trình quản lý plugin ở góc trên bên phải trang để vào trang Trình quản lý plugin. Tại trang này có thể xem, cài đặt, kích hoạt hoặc vô hiệu hóa plugin.

![2024-04-10_08-47-45](https://static-docs.nocobase.com/2024-04-10_08-47-45.png)

## Trung tâm cài đặt plugin

Plugin thường có trang cài đặt riêng. Click icon cài đặt ở góc trên bên phải trang, có thể xem menu cài đặt của tất cả plugin, click vào mục menu tương ứng để cấu hình plugin.
