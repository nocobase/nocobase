# Quản lý plugin

## Trình quản lý plugin

Nhấp vào biểu tượng trình quản lý plugin ở góc trên bên phải trang để truy cập trang trình quản lý plugin. Tại trang này, quý vị có thể xem, cài đặt, bật hoặc tắt các plugin.

![2024-04-10_08-47-45](https://static-docs.nocobase.com/2024-04-10_08-47-45.png)

## Trung tâm cài đặt plugin

Các plugin thường đi kèm với các trang cài đặt riêng. Nhấp vào biểu tượng cài đặt ở góc trên bên phải trang để xem các menu cài đặt của tất cả các plugin. Nhấp vào mục menu tương ứng để cấu hình plugin.
