## API

| Tham số | Kiểu | Mô tả |
| --- | --- | --- |
| `value` | `string` | Giá trị hiện tại |
| `onChange` | `(event) => void` | Callback khi thay đổi |
| `disabled` | `boolean` | Có bị vô hiệu hóa hay không |
| `placeholder` | `string` | Placeholder text |
