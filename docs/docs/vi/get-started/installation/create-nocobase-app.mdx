# Cài đặt create-nocobase-app

:::tip Điều kiện tiên quyết

- Đã cài đặt Node.js 20+, Yarn 1.22.x
- Cấu hình và khởi động một trong các cơ sở dữ liệu yêu cầu: MySQL 8.0.17+, MariaDB 10.9+, PostgreSQL 10+
- Do điều kiện môi trường mạng nội địa, khuyến nghị bạn thay đổi mirror nội địa (không bắt buộc)

```bash
$ yarn config set disable-self-update-check true
$ yarn config set registry https://registry.npmmirror.com/
```

:::

:::warning Giải thích biến môi trường

- `TZ` dùng để thiết lập múi giờ của ứng dụng, mặc định là múi giờ của hệ điều hành
- `APP_KEY` là khóa bí mật của ứng dụng, dùng để tạo token người dùng, v.v. (Nếu `APP_KEY` bị thay đổi, các token cũ cũng sẽ mất hiệu lực). Nó có thể là bất kỳ chuỗi ngẫu nhiên nào. **Vui lòng sửa đổi thành khóa bí mật của riêng bạn và đảm bảo không tiết lộ ra ngoài**
- `DB_*` là các cấu hình liên quan đến cơ sở dữ liệu, vui lòng sửa đổi theo thông tin kết nối cơ sở dữ liệu thực tế của bạn

:::

## 2. Chuyển thư mục

```bash
cd my-nocobase-app
```

## 3. Cài đặt các gói phụ thuộc

📢 Do ảnh hưởng của môi trường mạng, cấu hình hệ thống và các yếu tố khác, bước này có thể mất khoảng mười mấy phút.

```bash
yarn install
```

:::tip Gợi ý cho môi trường sản xuất

Khi triển khai môi trường sản xuất, để giảm dung lượng, có thể chỉ cài đặt các gói phụ thuộc cần thiết:

```bash
yarn install --production
```

:::

## 4. Cài đặt NocoBase

```bash
yarn nocobase install --lang=zh-CN
```

Quá trình cài đặt sẽ tự động tạo cấu trúc bảng cơ sở dữ liệu và khởi tạo dữ liệu.

## 5. Khởi động NocoBase

**Môi trường phát triển**

```bash
yarn dev
```

**Môi trường sản xuất**

```bash
yarn start
```

## 6. Đăng nhập NocoBase

Sử dụng trình duyệt mở [http://localhost:13000](http://localhost:13000), tài khoản và mật khẩu khởi tạo là `admin@nocobase.com` và `admin123`.

:::warning Lưu ý bảo mật tài khoản

Sau khi đăng nhập lần đầu, vui lòng thay đổi mật khẩu mặc định kịp thời để đảm bảo an toàn hệ thống.

:::
