# Triển khai cluster

NocoBase hỗ trợ triển khai ứng dụng theo chế độ cluster để nâng cao hiệu năng và tính sẵn sàng của ứng dụng. Triển khai cluster có thể chạy các instance khác nhau của ứng dụng trên nhiều server, hoặc chạy nhiều tiến trình ở chế độ đa nhân trên một server.

Xem chi tiết tại tài liệu liên quan: [Chế độ cluster](/cluster-mode/index.md).
