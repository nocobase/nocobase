# Triển khai Cụm

NocoBase hỗ trợ triển khai ứng dụng ở chế độ cụm nhằm nâng cao hiệu suất và tính sẵn sàng của ứng dụng. Việc triển khai cụm cho phép bạn chạy các phiên bản khác nhau của ứng dụng trên nhiều máy chủ, hoặc chạy nhiều tiến trình ở chế độ đa lõi trên một máy chủ duy nhất.

Để biết thêm chi tiết, vui lòng tham khảo tài liệu [Chế độ Cụm](/cluster-mode/).
