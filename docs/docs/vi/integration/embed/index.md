# Nhúng NocoBase

## Giới thiệu

Plugin này dùng để nhúng các trang của NocoBase vào các trang web hoặc ứng dụng khác.

## Cài đặt

Đây là một plugin thương mại, bạn cần tải lên và kích hoạt thông qua trình quản lý plugin.

![20240323162741](https://static-docs.nocobase.com/20240323162741.png)

## Hướng dẫn sử dụng

Sau khi kích hoạt cấu hình giao diện, bạn hãy nhấp vào "Sao chép liên kết nhúng" trong trình cài đặt trang (góc trên bên phải).

![](https://static-docs.nocobase.com/f11bd6d5e88d38731d7cd3cb149022c8.png)

Sau khi nhấp, bạn sẽ nhận được một liên kết có thể mở riêng, ví dụ: `https://xxx.com/embed/qs087rz4o2b`.

![](https://static-docs.nocobase.com/9d847805e88d38731d7cd3cb149022c8.png)

Nếu bạn muốn nhúng các trang của NocoBase vào các trang web hoặc ứng dụng khác, cần phải thực hiện xác thực người dùng và nối thêm `token` vào liên kết, ví dụ: `https://xxx.com/embed/qs087rz4o2b?token=xxx`.
