# Nhúng NocoBase

<PluginInfo commercial="true" name="embed"></PluginInfo>

## Giới thiệu

Plugin này dùng để nhúng các trang của NocoBase vào website hoặc ứng dụng khác.

## Cài đặt

Plugin này là plugin thương mại. Để biết chi tiết về cách kích hoạt, vui lòng tham khảo: [Hướng dẫn kích hoạt plugin thương mại](https://www.nocobase.com/cn/blog/nocobase-commercial-license-activation-guide)

## Hướng dẫn sử dụng

Sau khi kích hoạt cấu hình giao diện, trong settings (góc trên bên phải) của trang, click "Sao chép liên kết nhúng".

![](https://static-docs.nocobase.com/f11bd6d5e88d38731d7cd3cb149022c8.png)

Liên kết lấy được sau khi click có thể mở riêng, ví dụ `https://example.com/embed/qs087rz4o2b`.

![](https://static-docs.nocobase.com/9d847805a00fd07372a474665ec0e968.png)

Nếu muốn nhúng trang NocoBase vào website hoặc ứng dụng khác, cần thực hiện kết nối người dùng và thêm `token` vào liên kết, ví dụ `https://example.com/embed/qs087rz4o2b?token=xxx`.
