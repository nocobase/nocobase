# Quản lý Tệp

## Giới thiệu

Cấu hình phương thức lưu trữ cho các tệp được tải lên trong quá trình trò chuyện của nhân viên AI.

## Cài đặt

Truy cập trang cấu hình plugin nhân viên AI, nhấp vào tab `Settings` để vào trang cài đặt phương thức lưu trữ.

![20251022002648](https://static-docs.nocobase.com/20251022002648.png)

Các tùy chọn lưu trữ có sẵn cho `Files storage` được cấu hình trong plugin Quản lý Tệp.

![20251022002853](https://static-docs.nocobase.com/20251022002853.png)
