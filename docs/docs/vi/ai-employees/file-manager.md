# Quản lý tệp

## Giới thiệu

Cấu hình phương thức lưu trữ tệp được tải lên trong quá trình hội thoại với Nhân viên AI.

## Thiết lập

Vào trang cấu hình Plugin Nhân viên AI, nhấp vào tab `Settings` để vào trang cài đặt phương thức lưu trữ.

![20251022002648](https://static-docs.nocobase.com/20251022002648.png)

Các tùy chọn lưu trữ khả dụng cho `Files storage` được thiết lập trong Plugin quản lý tệp.

![20251022002853](https://static-docs.nocobase.com/20251022002853.png)
