# Trường Khu vực Trung Quốc

## Giới thiệu

Plugin Trường Khu vực Trung Quốc (`field-china-region`) bổ sung loại trường "Khu vực Trung Quốc" vào NocoBase.

## Cài đặt

Trong giao diện chính của NocoBase, truy cập vào "Trình quản lý Plugin", tìm plugin "Khu vực Trung Quốc" và kích hoạt nó.

## Sử dụng

Trong cấu hình bộ sưu tập, thêm một trường mới và chọn "Khu vực Trung Quốc" làm loại trường.

![](/images/field-china-region-1.png)

### Cấu hình Trường

Trong cấu hình trường, bạn có thể thiết lập chế độ hiển thị cho trường:

- Tỉnh
- Tỉnh/Thành phố
- Tỉnh/Thành phố/Quận
- Tỉnh/Thành phố/Quận/Huyện

![](/images/field-china-region-2.png)
