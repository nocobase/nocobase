# Trình xuất Telemetry: HTTP

## Cấu hình biến môi trường

### TELEMETRY_METRIC_READER

Loại trình xuất chỉ số telemetry.

```bash
TELEMETRY_METRIC_READER=http
```

### TELEMETRY_HTTP_URL

URL HTTP để xuất dữ liệu telemetry.

```bash
TELEMETRY_HTTP_URL=
```
