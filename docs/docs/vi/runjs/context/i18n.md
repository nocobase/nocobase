# ctx.i18n

Instance i18n của ngữ cảnh hiện tại, được sử dụng để đọc hoặc chuyển đổi ngôn ngữ. **Sử dụng thống nhất [ctx.t()](./t.md) để dịch văn bản**, không sử dụng `ctx.i18n.t`.

## Phạm vi áp dụng

Tất cả môi trường thực thi RunJS đều có thể sử dụng `ctx.i18n` (ví dụ: JSBlock, JSField, JSItem, JSColumn, luồng công việc, quy tắc liên kết, v.v.).

## Định nghĩa kiểu

```typescript
interface i18n: {
  language: string;
  changeLanguage(lng: string): Promise<any>;
}
```

## Thuộc tính thường dùng

| Thuộc tính | Kiểu | Mô tả |
|------|------|------|
| `language` | `string` | Mã ngôn ngữ hiện đang kích hoạt (ví dụ: `zh-CN`, `en-US`) |

## Phương thức thường dùng

### changeLanguage(lng)

Chuyển đổi ngôn ngữ hiện tại.

| Tham số | Kiểu | Mô tả |
|------|------|------|
| `lng` | `string` | Mã ngôn ngữ đích (ví dụ: `'en-US'`, `'zh-CN'`) |

**Giá trị trả về**: `Promise<any>`, resolve sau khi hoàn tất việc chuyển đổi ngôn ngữ.

## Ví dụ

### Đọc ngôn ngữ hiện tại

```ts
const lang = ctx.i18n.language;
// 'zh-CN' | 'en-US' | ...
if (lang.startsWith('zh')) {
  ctx.render(ctx.t('Giao diện tiếng Trung'));
} else {
  ctx.render(ctx.t('English UI'));
}
```

### Chuyển đổi ngôn ngữ

```ts
// Chuyển sang tiếng Anh
await ctx.i18n.changeLanguage('en-US');

// Chuyển sang tiếng Trung
await ctx.i18n.changeLanguage('zh-CN');
```

### Nút chuyển đổi ngôn ngữ

```ts
const { Button } = ctx.libs.antd;
const isZh = ctx.i18n.language.startsWith('zh');
ctx.render(
  <Button onClick={async () => {
    await ctx.i18n.changeLanguage(isZh ? 'en-US' : 'zh-CN');
  }}>
    {ctx.t(isZh ? 'Switch to English' : 'Chuyển sang tiếng Trung')}
  </Button>,
);
```

## Lưu ý

- **Văn bản dịch**: Sử dụng thống nhất `ctx.t()`, không sử dụng `ctx.i18n.t`.
