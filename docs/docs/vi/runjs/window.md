# window

Các thuộc tính sau có thể được truy cập trực tiếp thông qua `window`:

* `setTimeout` / `clearTimeout`
* `setInterval` / `clearInterval`
* `console`
* `Math`
* `Date`
* `FormData`
* `addEventListener`
* `open` (Chỉ cho phép `http:`, `https:`, hoặc `about:blank`)
* `location` (Chỉ đọc, hỗ trợ điều hướng an toàn)
* `navigator`

Chỉ hỗ trợ các khả năng truy vấn và tạo DOM cơ bản và an toàn:

* `createElement(tagName)`
* `querySelector(selectors)`
* `querySelectorAll(selectors)`
