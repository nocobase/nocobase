# Поле "Регіон Китаю"

## Вступ

Плагін поля "Регіон Китаю" (`field-china-region`) додає тип поля "Регіон Китаю" до NocoBase.

## Встановлення

В основному інтерфейсі NocoBase перейдіть до "Менеджера плагінів", знайдіть плагін "Регіон Китаю" та увімкніть його.

## Використання

У конфігурації колекції додайте нове поле та оберіть "Регіон Китаю" як тип поля.

![](/images/field-china-region-1.png)

### Конфігурація поля

У конфігурації поля ви можете налаштувати режим відображення для поля:

- Провінція
- Провінція/Місто
- Провінція/Місто/Район
- Провінція/Місто/Район/Повіт

![](/images/field-china-region-2.png)
