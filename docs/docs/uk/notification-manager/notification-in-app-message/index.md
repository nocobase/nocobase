# Сповіщення: Повідомлення в застосунку

## Вступ

Дозволяє користувачам отримувати сповіщення в реальному часі безпосередньо в застосунку NocoBase.

## Встановлення

Цей плагін є вбудованим, тому додаткове встановлення не потрібне.

## Додавання каналу повідомлень у застосунку

Перейдіть до Керування сповіщеннями, натисніть кнопку Додати та оберіть Повідомлення в застосунку.
![2024-11-08-08-33-26-20241108083326](https://static-docs.nocobase.com/2024-11-08-08-33-26-20241108083326.png)

Введіть назву та опис каналу, потім натисніть Надіслати.
![2024-11-08-08-34-32-20241108083431](https://static-docs.nocobase.com/2024-11-08-08-34-32-20241108083431.png)

Новий канал з'явиться у списку.

![2024-11-08-08-34-52-20241108083452](https://static-docs.nocobase.com/2024-11-08-08-34-52-20241108083452.png)

## Приклад використання

Щоб допомогти вам краще зрозуміти, як використовувати повідомлення в застосунку, ось приклад "Відстеження маркетингових лідів".

Уявіть, що ваша команда проводить велику маркетингову кампанію, спрямовану на відстеження відгуків та потреб потенційних клієнтів. Використовуючи повідомлення в застосунку, ви можете:

**Створіть канал сповіщень**: Спершу, у Керуванні сповіщеннями, налаштуйте канал повідомлень у застосунку з назвою "Marketing Clue", щоб члени команди могли чітко ідентифікувати його призначення.

![2024-11-08-08-34-32-20241108083431](https://static-docs.nocobase.com/2024-11-08-08-34-32-20241108083431.png)

**Налаштуйте робочий процес**: Створіть робочий процес, який автоматично запускатиме сповіщення при появі нового маркетингового ліда. Ви можете додати вузол сповіщення до робочого процесу, оберіть створений вами канал "Marketing Clue" та налаштуйте вміст повідомлення за потреби. Наприклад:

![image-1-2024-10-27-14-07-17](https://static-docs.nocobase.com/image-1-2024-10-27-14-07-17.png)

**Отримуйте сповіщення в реальному часі**: Щойно робочий процес буде запущено, весь відповідний персонал отримає сповіщення в реальному часі, що гарантує швидке реагування та дії команди.

![image-2-2024-10-27-14-07-22](https://static-docs.nocobase.com/image-2-2024-10-27-14-07-22.png)

**Керування повідомленнями та відстеження**: Повідомлення в застосунку групуються за назвою каналу. Ви можете фільтрувати повідомлення за статусом "прочитано" та "не прочитано" для швидкого перегляду важливої інформації. Натискання кнопки "Переглянути" перенаправить вас на налаштовану сторінку за посиланням для подальших дій.

![20241027140648-2024-10-27-14-06-51-2024-10-29-13-26-41](https://static-docs.nocobase.com/20241027140648-2024-10-27-14-06-51-2024-10-29-13-26-41.png)
