# Менеджер сповіщень

## Вступ

Менеджер сповіщень — це централізований сервіс, що об'єднує різні канали сповіщень. Він пропонує єдиний інтерфейс для налаштування каналів, керування їхньою доставкою та ведення журналів, а також підтримує гнучке розширення для додавання нових каналів.

![20240928112556](https://static-docs.nocobase.com/20240928112556.png)

- **Фіолетова частина**: Менеджер сповіщень надає комплексний сервіс, що включає налаштування каналів та ведення журналів, з можливістю розширення для додаткових каналів сповіщень.
- **Зелена частина**: Повідомлення в застосунку (In-App Message) — це вбудований канал, який дозволяє користувачам отримувати сповіщення безпосередньо в застосунку.
- **Червона частина**: Електронна пошта (Email) — це розширюваний канал, що дозволяє користувачам отримувати сповіщення електронною поштою.

## Керування каналами

![20240928181752](https://static-docs.nocobase.com/20240928181752.png)

Наразі підтримуються такі канали:

- [Повідомлення в застосунку](/notification-manager/notification-in-app-message)
- [Електронна пошта](/notification-manager/notification-email) (використовує вбудований SMTP-транспорт)

Ви також можете розширити функціонал, додавши більше каналів сповіщень. Зверніться до документації [Розширення каналів](/notification-manager/development/extension).
