# Сповіщення: WeCom

## Вступ

**Плагін WeCom** дозволяє застосунку надсилати сповіщення користувачам WeCom.

## Додавання та налаштування автентифікатора WeCom

Спершу вам потрібно додати та налаштувати автентифікатор WeCom у NocoBase. Зверніться до розділу [Автентифікація користувачів - WeCom](/auth-verification/auth-wecom). Лише системні користувачі, які увійшли в систему через WeCom, можуть отримувати системні сповіщення через WeCom.

## Налаштування каналу сповіщень WeCom

Оберіть автентифікатор, який ви щойно налаштували.

![](https://static-docs.nocobase.com/202412041525284.png)

## Налаштування вузла сповіщень робочого процесу

Оберіть налаштований канал сповіщень WeCom. Він підтримує три типи повідомлень: текстова картка, Markdown та картка-шаблон.
