# Рушій сховища: Aliyun OSS

Це рушій сховища, що базується на Aliyun OSS. Перед використанням вам потрібно підготувати відповідний обліковий запис та дозволи.

## Параметри конфігурації

![Приклад конфігурації рушія сховища Aliyun OSS](https://static-docs.nocobase.com/20240712220011.png)

:::info{title=Примітка}
Тут ми розглянемо лише специфічні параметри рушія сховища Aliyun OSS. Загальні параметри дивіться у розділі [Загальні параметри рушія](./index#引擎通用参数).
:::

### Регіон

Вкажіть регіон сховища OSS, наприклад: `oss-cn-hangzhou`.

:::info{title=Примітка}
Ви можете переглянути інформацію про регіон вашого сховища в [консолі Aliyun OSS](https://oss.console.aliyun.com/). Вам потрібно використовувати лише префікс регіону (без повного доменного імені).
:::

### AccessKey ID

Введіть ID вашого ключа доступу Aliyun.

### AccessKey Secret

Введіть Secret вашого ключа доступу Aliyun.

### Бакет

Вкажіть назву бакета OSS.

### Час очікування

Вкажіть час очікування для завантаження в Aliyun OSS, у мілісекундах. За замовчуванням встановлено `60000` мілісекунд (тобто 60 секунд).
