# Tencent Cloud COS

Рушій зберігання даних, що базується на Tencent Cloud COS. Перед використанням необхідно підготувати відповідний обліковий запис та дозволи.

## Параметри конфігурації

![Приклад конфігурації рушія зберігання Tencent COS](https://static-docs.nocobase.com/20240712222125.png)

:::info{title=Примітка}
Тут представлені лише специфічні параметри для рушія зберігання Tencent Cloud COS. Загальні параметри дивіться у розділі [Загальні параметри рушія](./index.md#general-engine-parameters).
:::

### Регіон

Введіть регіон для зберігання COS, наприклад: `ap-chengdu`.

:::info{title=Примітка}
Ви можете переглянути інформацію про регіон вашого сховища (bucket) у [консолі Tencent Cloud COS](https://console.cloud.tencent.com/cos). Вам потрібно використовувати лише префікс регіону (без повного доменного імені).
:::

### SecretId

Введіть ID вашого ключа доступу Tencent Cloud.

### SecretKey

Введіть Secret вашого ключа доступу Tencent Cloud.

### Сховище (Bucket)

Введіть назву сховища COS (bucket), наприклад: `qing-cdn-1234189398`.
