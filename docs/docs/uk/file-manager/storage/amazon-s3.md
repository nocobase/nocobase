# Рушій зберігання: Amazon S3

Рушій зберігання, що базується на Amazon S3. Перед використанням необхідно підготувати відповідний обліковий запис та дозволи.

## Параметри конфігурації

![Приклад конфігурації рушія зберігання Amazon S3](https://static-docs.nocobase.com/20251031092524.png)

:::info{title=Примітка}
Цей розділ представляє лише спеціальні параметри рушія зберігання Amazon S3. Щодо загальних параметрів, будь ласка, зверніться до [Загальні параметри рушія](./index#引擎通用参数).
:::

### Регіон

Введіть регіон зберігання S3, наприклад: `us-west-1`.

:::info{title=Примітка}
Ви можете переглянути інформацію про регіон вашого кошика в [консолі Amazon S3](https://console.aws.amazon.com/s3/). Вам потрібно лише використати префікс регіону (без повного доменного імені).
:::

### AccessKey ID

Введіть AccessKey ID для авторизованого доступу до Amazon S3.

### AccessKey Secret

Введіть AccessKey Secret для авторизованого доступу до Amazon S3.

### Кошик

Введіть назву кошика S3.
