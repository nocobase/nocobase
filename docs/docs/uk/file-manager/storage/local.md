# Рушій зберігання: Локальне сховище

Завантажені файли зберігатимуться в локальному каталозі на жорсткому диску сервера. Це підходить для сценаріїв, де загальний обсяг завантажених файлів, керованих системою, невеликий, або для експериментальних цілей.

## Параметри конфігурації

![Приклад конфігурації рушія зберігання файлів](https://static-docs.nocobase.com/20240529115151.png)

:::info{title=Примітка}
У цьому розділі представлено лише параметри, специфічні для локального рушія зберігання. Загальні параметри дивіться у розділі [Загальні параметри рушія](./index.md#引擎通用参数).
:::

### Шлях

Цей параметр одночасно визначає як відносний шлях для зберігання файлів на сервері, так і шлях доступу через URL. Наприклад, "`user/avatar`" (без початкових і кінцевих скісних рисок) означає:

1. Відносний шлях на сервері, де зберігаються завантажені файли: `/path/to/nocobase-app/storage/uploads/user/avatar`.
2. Префікс URL-адреси для доступу до файлів: `http://localhost:13000/storage/uploads/user/avatar`.
