# Поле «Вкладення»

## Вступ

Система має вбудований тип поля «Вкладення», який дозволяє користувачам завантажувати файли до користувацьких колекцій.

Поле «Вкладення» по суті є полем зв'язку «багато до багатьох», яке вказує на вбудовану системну колекцію файлів «Вкладення» (`attachments`). Коли ви створюєте поле «Вкладення» в будь-якій колекції, автоматично генерується проміжна таблиця для зв'язку «багато до багатьох». Метадані завантажених файлів зберігаються в колекції «Вкладення», а інформація про файли, на які посилається ваша колекція, пов'язується через цю проміжну таблицю.

### Обмеження за типом MIME

Використовується для обмеження типів файлів, дозволених для завантаження, за допомогою синтаксису [MIME](https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/MIME_types). Наприклад: `image/*` позначає файли зображень. Кілька типів можна розділити комою, наприклад: `image/*,application/pdf` дозволяє як зображення, так і файли PDF.

### Рушій зберігання

Виберіть рушій зберігання для завантажених файлів. Якщо залишити це поле порожнім, буде використано рушій зберігання за замовчуванням.
