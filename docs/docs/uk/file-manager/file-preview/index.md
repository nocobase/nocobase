# Попередній перегляд файлів

В інтерфейсах, що містять поля файлів (включаючи поля вкладень), ви можете переглядати файли, натиснувши на мініатюру або іконку файлу. Вбудована функція попереднього перегляду підтримує різні типи файлів, зокрема зображення, PDF та більшість типів файлів, які підтримуються браузерами нативно.

![20251129232307](https://static-docs.nocobase.com/20251129232307.png)

Для типів файлів, які не підтримують нативний перегляд, ви можете увімкнути функцію попереднього перегляду, встановивши або розширивши відповідні плагіни для попереднього перегляду файлів. Наприклад, після встановлення плагіна для попереднього перегляду файлів Office ви зможете переглядати файли Word, Excel та PowerPoint.
