## BelongsToRepository

Його інтерфейс ідентичний [HasOneRepository](./has-one-repository.md).
`BelongsToRepository` — це `Repository` для обробки зв'язків `BelongsTo`. Він надає кілька зручних методів для роботи зі зв'язками `BelongsTo`.
