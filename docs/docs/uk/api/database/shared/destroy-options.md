**Тип**

```typescript
interface DestroyOptions extends SequelizeDestroyOptions {
  filter?: Filter;
  filterByTk?: TargetKey | TargetKey[];
  truncate?: boolean;
  context?: any;
}
```

**Деталі**

- `filter`: Визначає умови фільтрації для записів, які потрібно видалити. Детальне використання фільтра дивіться у методі [`find()`](#find).
- `filterByTk`: Визначає умови фільтрації для записів, які потрібно видалити за TargetKey.
- `truncate`: Чи потрібно очистити дані колекції. Цей параметр діє лише тоді, коли не передано параметри `filter` або `filterByTk`.
- `transaction`: Об'єкт транзакції. Якщо параметр транзакції не передано, метод автоматично створить внутрішню транзакцію.
