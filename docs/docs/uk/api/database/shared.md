## Параметри

| Назва параметра         | Тип          | Значення за замовчуванням | Опис                                                              |
| :--------------------- | :----------- | :------------------------ | :---------------------------------------------------------------- |
| `options.values`       | `M`          | `{}`                      | Об'єкт даних для вставки                                          |
| `options.whitelist?`   | `string[]`   | -                         | Білий список полів для `values`. Будуть збережені лише поля, що входять до цього списку. |
| `options.blacklist?`   | `string[]`   | -                         | Чорний список полів для `values`. Поля, що входять до цього списку, не будуть збережені. |
| `options.transaction?` | `Transaction` | -                         | Транзакція                                                        |
