# IRepository

Інтерфейс `Repository` визначає низку методів для роботи з моделями, що дозволяють адаптувати CRUD-операції джерела даних.

## API

### find()

Повертає список моделей, що відповідають параметрам запиту.

#### Сигнатура

- `find(options?: any): Promise<IModel[]>`

### findOne()

Повертає модель, що відповідає параметрам запиту. Якщо є кілька відповідних моделей, повертається лише перша.

#### Сигнатура

- `findOne(options?: any): Promise<IModel>`

### count()

Повертає кількість моделей, що відповідають параметрам запиту.

#### Сигнатура

- `count(options?: any): Promise<Number>`

### findAndCount()

Повертає список та кількість моделей, що відповідають параметрам запиту.

#### Сигнатура

- `findAndCount(options?: any): Promise<[IModel[], Number]>`

### create()

Створює об'єкт даних моделі.

#### Сигнатура

- `create(options: any): void`

### update()

Оновлює об'єкт даних моделі на основі умов запиту.

#### Сигнатура

- `update(options: any): void`

### destroy()

Видаляє об'єкт даних моделі на основі умов запиту.

#### Сигнатура

- `destroy(options: any): void`
