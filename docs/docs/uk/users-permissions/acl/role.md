# Ролі

## Панель керування

### Керування ролями

![](https://static-docs.nocobase.com/da7083c67d794e23dc6eb0f85b1de86c.png)

Після інсталяції застосунок за замовчуванням містить дві ролі: "Admin" та "Member". Кожна з них має різні налаштування дозволів за замовчуванням.

### Додавання, видалення та зміна ролей

Ідентифікатор ролі є унікальним системним ідентифікатором. Ви можете налаштовувати ролі за замовчуванням, але системні ролі, визначені заздалегідь, видалити неможливо.

![](https://static-docs.nocobase.com/35f323b346db4f9f12f9bee4dea63302.png)

### Встановлення ролі за замовчуванням

Роль за замовчуванням – це та, що автоматично призначається новим користувачам, якщо під час їх створення не було вказано жодної конкретної ролі.

![](https://static-docs.nocobase.com/f41bba7ff55ca28715c486dc45bc1708.png)

## Особистий кабінет

### Перемикання ролей

Користувачу можна призначити кілька ролей. Якщо користувач має кілька ролей, він може перемикатися між ними в Особистому кабінеті.

![](https://static-docs.nocobase.com/e331d11ec1ca3b8b7e0472105b167819.png)

Роль за замовчуванням, яка призначається користувачу при вході в систему, визначається за таким пріоритетом: спочатку використовується роль, на яку було переключено востаннє (це значення оновлюється при кожному перемиканні), а потім – перша роль (системна роль за замовчуванням).
