# Користувачі

## Керування користувачами

Цей плагін надає інтерфейс керування користувачами в центрі конфігурації. Після ініціалізації застосунку автоматично створюється обліковий запис супер-адміністратора, який неможливо видалити. Супер-адміністратор має права Root і може отримати доступ до всіх ресурсів.

![](https://static-docs.nocobase.com/44bf40f56b45d4dd96c424fb08082cf6.png)

Крім того, ви також можете додавати різні блоки з колекції Користувачі для керування користувачами, наприклад, табличні блоки:

![](https://static-docs.nocobase.com/76b59541a9e8f18a4568a7c9.png)
