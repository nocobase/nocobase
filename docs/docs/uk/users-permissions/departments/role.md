# Керування ролями відділів

Прив'язавши одну або кілька ролей до відділу, ви можете надати ці ролі співробітникам цього відділу.

Існує два способи прив'язати ролі до відділу.

Перший спосіб — це обрати роль у полі "Роль" у формі редагування відділу.

![](https://static-docs.nocobase.com/70f77bb89aa1fb415c152a51a51cc23b.png)

Другий спосіб — це додати відділи до відповідної ролі в розділі керування ролями.

![](https://static-docs.nocobase.com/f2a77bb89aa1fb415c152a51a51cc23b.png)

Відділи, яким вже призначено цю роль, недоступні для вибору.

![](https://static-docs.nocobase.com/be10299893581e1f97a4e01ddd5c7e59.png)
