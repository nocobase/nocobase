# Швидкий старт

## Вступ

Перед використанням AI-співробітника вам потрібно спочатку підключити онлайн-сервіс LLM. NocoBase наразі підтримує основні онлайн-сервіси LLM, такі як OpenAI, Gemini, Claude, DepSeek, Qwen тощо.
Окрім онлайн-сервісів LLM, NocoBase також підтримує підключення локальних моделей Ollama.

## Налаштування сервісу LLM

Перейдіть на сторінку налаштувань плагіна AI-співробітника, натисніть вкладку `LLM service`, щоб перейти на сторінку керування сервісами LLM.

![20251021213122](https://static-docs.nocobase.com/20251021213122.png)

Наведіть курсор миші на кнопку `Add New` у верхньому правому куті списку сервісів LLM і виберіть сервіс LLM, який ви хочете використовувати.

![20251021213358](https://static-docs.nocobase.com/20251021213358.png)

На прикладі OpenAI, у спливаючому вікні введіть легко запам'ятовуваний `title`, потім введіть `API key`, отриманий від OpenAI, і натисніть `Submit`, щоб зберегти. Це завершить налаштування сервісу LLM.

`Base URL` зазвичай можна залишити порожнім. Якщо ви використовуєте сторонній сервіс LLM, сумісний з API OpenAI, будь ласка, вкажіть відповідний Base URL.

![20251021214549](https://static-docs.nocobase.com/20251021214549.png)

## Тестування доступності

На сторінці налаштувань сервісу LLM натисніть кнопку `Test flight`, введіть назву моделі, яку ви хочете використовувати, і натисніть кнопку `Run`, щоб перевірити, чи доступний сервіс LLM та модель.

![20251021214903](https://static-docs.nocobase.com/20251021214903.png)
