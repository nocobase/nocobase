# Швидкий старт

Налаштуймо мінімальну робочу конфігурацію AI-співробітника за 5 хвилин.

## Встановлення плагіна

AI-співробітники є вбудованим плагіном NocoBase (`@nocobase/plugin-ai`), тому окреме встановлення не потрібне.

## Конфігурація моделей

Ви можете налаштувати сервіси LLM через будь-який із цих входів:

1. Панель керування: `Налаштування системи -> AI-співробітники -> Сервіс LLM`.
2. Швидкий доступ на фронтенді: у панелі чату AI скористайтеся `Model Switcher`, щоб вибрати модель, а потім натисніть на ярлик «Додати сервіс LLM» для прямого переходу.

![quick-start-model-switcher-add-llm-service.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/quick-start-model-switcher-add-llm-service.png)

Зазвичай вам потрібно підтвердити наступне:
1. Вибрати провайдера (Provider).
2. Ввести API-ключ.
3. Налаштувати `Enabled Models` (Увімкнені моделі); за замовчуванням можна просто використовувати Recommend.

## Увімкнення вбудованих співробітників

Вбудовані AI-співробітники за замовчуванням увімкнені всі, тому зазвичай немає потреби активувати їх вручну по одному.

Якщо вам потрібно змінити доступність (увімкнути/вимкнути конкретного співробітника), ви можете змінити перемикач `Enabled` (Увімкнено) на сторінці списку `Налаштування системи -> AI-співробітники`.

![ai-employee-list-enable-switch.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employee-list-enable-switch.png)

## Початок співпраці

На сторінці додатка наведіть курсор на ярлик швидкого доступу в нижньому правому куті та виберіть AI-співробітника.
![ai-employees-entry-bottom-right.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employees-entry-bottom-right.png)

Натисніть, щоб відкрити діалогове вікно чату AI:

![chat-footer-employee-switcher-and-model-switcher.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/chat-footer-employee-switcher-and-model-switcher.png)

Ви також можете:  
* Додавати блоки
* Додавати вкладення
* Увімкнути вебпошук
* Змінювати AI-співробітників
* Вибирати моделі

Вони також можуть автоматично отримувати структуру сторінки як контекст. Наприклад, Dex у блоці форми автоматично зчитує структуру полів форми та викликає відповідні навички для роботи зі сторінкою.

## Швидкі завдання

## Огляд вбудованих співробітників

NocoBase пропонує кілька вбудованих AI-співробітників для різних сценаріїв.

Вам потрібно лише:

1. Налаштувати сервіси LLM.
2. За потреби налаштувати статус активації співробітників (за замовчуванням увімкнено).
3. Вибрати модель у чаті та почати співпрацю.

| Ім'я співробітника | Роль | Ключові можливості |
| :--- | :--- | :--- |
| **Cole** | Помічник NocoBase | Питання та відповіді щодо використання продукту, пошук у документації |
| **Ellis** | Експерт з електронної пошти | Написання листів, створення резюме, пропозиції відповідей |
| **Dex** | Експерт з упорядкування даних | Переклад полів, форматування, вилучення інформації |
| **Viz** | Аналітик інсайтів | Аналіз даних, аналіз трендів, інтерпретація ключових показників |
| **Lexi** | Асистент перекладу | Багатомовний переклад, допомога в комунікації |
| **Vera** | Аналітик-дослідник | Пошук у мережі, агрегація інформації, глибокі дослідження |
| **Dara** | Експерт з візуалізації даних | Налаштування діаграм, створення візуальних звітів |
| **Orin** | Експерт з моделювання даних | Допомога в розробці структури колекцій, пропозиції щодо полів |
| **Nathan** | Frontend-інженер | Допомога в написанні фрагментів коду фронтенду, налаштування стилів |

**Примітки**

Деякі вбудовані AI-співробітники не з’являються у списку в нижньому правому куті, оскільки вони мають спеціальні сценарії роботи:

- Orin: сторінки моделювання даних.
- Dara: блоки налаштування діаграм.
- Nathan: JS-блок та подібні редактори коду.
