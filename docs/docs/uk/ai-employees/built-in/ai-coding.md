# Штучний інтелект-помічник · Натан: Фронтенд-інженер

> Допомагає писати та оптимізувати фронтенд-код для реалізації складної інтерактивної логіки.

`Натан` — це експерт з фронтенд-розробки в NocoBase. У сценаріях, що вимагають написання JavaScript, таких як `JSBlock`, `JSField`, `JSItem`, `JSColumn`, `JSAction`, `Event Flow` та `Linkage`, ви побачите аватар `Натана` у верхньому правому куті редактора коду. Ви можете попросити його допомогти вам написати або змінити код у редакторі.

![20251022163405](https://static-docs.nocobase.com/20251022163405.png)

## Кроки

У редакторі коду натисніть `Натан`, щоб відкрити діалогове вікно AI-помічника. Код з редактора буде автоматично додано до поля введення та надіслано `Натану` як контекст застосунку.

![20251022163935](https://static-docs.nocobase.com/20251022163935.png)

Введіть ваші вимоги до коду та надішліть їх `Натану`, потім дочекайтеся його відповіді.

![20251022164041](https://static-docs.nocobase.com/20251022164041.png)

Натисніть кнопку `Apply to editor` на блоці коду у відповіді `Натана`, щоб перезаписати код у редакторі.

![20251022164323](https://static-docs.nocobase.com/20251022164323.png)

Натисніть кнопку `Run` у редакторі коду, щоб переглянути результат у реальному часі.

![20251022164436](https://static-docs.nocobase.com/20251022164436.png)

## Історія коду

Натисніть піктограму «командний рядок» у верхньому правому куті діалогового вікна `Натана`, щоб переглянути фрагменти коду, які ви надсилали, та фрагменти, на які `Натан` відповідав у поточній сесії.
