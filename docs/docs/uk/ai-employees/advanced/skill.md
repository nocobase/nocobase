# Розширені можливості

## Вступ

Сучасні великі мовні моделі (ВММ) мають здатність використовувати інструменти. Плагін "AI співробітник" містить вбудовані поширені інструменти, які можуть використовувати ВММ.

Навички, які ви налаштовуєте на сторінці параметрів AI співробітника, є саме тими інструментами, що доступні для використання великими мовними моделями.

![20251022142348](https://static-docs.nocobase.com/20251022142348.png)

## Налаштування навичок

Перейдіть на сторінку конфігурації плагіна "AI співробітник", натисніть вкладку `AI employees`, щоб потрапити на сторінку керування AI співробітниками.

Оберіть AI співробітника, для якого ви хочете налаштувати навички, і натисніть кнопку `Edit`, щоб перейти на сторінку редагування AI співробітника.

На вкладці `Skills` натисніть кнопку `Add Skill`, щоб додати навичку для поточного AI співробітника.

![20251022145748](https://static-docs.nocobase.com/20251022145748.png)

## Опис навичок

### Frontend

Група Frontend дозволяє AI співробітнику взаємодіяти з фронтенд-компонентами.

- Навичка `Form filler` дозволяє AI співробітнику заповнювати згенеровані дані форми у вказану користувачем форму.

![20251022145954](https://static-docs.nocobase.com/20251022145954.png)

### Моделювання даних

Група навичок "Моделювання даних" надає AI співробітнику можливість викликати внутрішні API NocoBase для моделювання даних.

- `Intent Router` (Маршрутизатор намірів) визначає, чи хоче користувач змінити структуру *колекції* або створити нову.
- `Get collection names` отримує назви всіх існуючих *колекцій* у системі.
- `Get collection metadata` отримує інформацію про структуру вказаної *колекції*.
- `Define collections` дозволяє AI співробітнику створювати *колекції* в системі.

![20251022150441](https://static-docs.nocobase.com/20251022150441.png)

### Виклик робочого процесу

`Workflow caller` надає AI співробітнику можливість виконувати *робочі процеси*. *Робочі процеси*, налаштовані в *плагіні* *робочих процесів* з `Trigger type` як `AI employee event`, будуть доступні тут як навички для використання AI співробітником.

![20251022153320](https://static-docs.nocobase.com/20251022153320.png)

### Редактор коду

Навички в групі "Редактор коду" в основному дозволяють AI співробітнику взаємодіяти з редактором коду.

- `Get code snippet list` отримує список попередньо встановлених фрагментів коду.
- `Get code snippet content` отримує вміст вказаного фрагмента коду.

![20251022153811](https://static-docs.nocobase.com/20251022153811.png)

### Інші

- `Chart generator` надає AI співробітнику можливість генерувати діаграми та виводити їх безпосередньо в розмові.

![20251022154141](https://static-docs.nocobase.com/20251022154141.png)
