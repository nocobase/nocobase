# Багатомодальний діалог

## Зображення

За умови підтримки моделлю, вузол LLM може надсилати зображення моделі. Для цього вам потрібно вибрати поле вкладення або пов'язаний запис колекції файлів за допомогою змінної. При виборі запису колекції файлів ви можете вказати його на рівні об'єкта або вибрати поле URL.

![](https://static-docs.nocobase.com/202503041034858.png)

Існує два варіанти формату надсилання зображень:

- Надсилати через URL – Усі зображення, крім тих, що зберігаються локально, будуть надсилатися у вигляді URL-адрес. Локально збережені зображення будуть конвертовані у формат base64 перед надсиланням.
- Надсилати через base64 – Усі зображення, незалежно від того, чи зберігаються вони локально, чи в хмарі, будуть надсилатися у форматі base64. Цей варіант підходить для випадків, коли URL-адреса зображення не може бути безпосередньо доступна онлайн-сервісом LLM.

![](https://static-docs.nocobase.com/202503041200638.png)
