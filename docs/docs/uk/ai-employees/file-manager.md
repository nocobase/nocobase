# Керування файлами

## Вступ

Налаштуйте спосіб зберігання файлів, які завантажуються під час розмов зі співробітниками ШІ.

## Налаштування

Перейдіть на сторінку конфігурації плагіна співробітників ШІ, натисніть вкладку `Settings`, щоб перейти на сторінку налаштувань способу зберігання.

![20251022002648](https://static-docs.nocobase.com/20251022002648.png)

Доступні параметри зберігання для `Files storage` налаштовуються в плагіні керування файлами.

![20251022002853](https://static-docs.nocobase.com/20251022002853.png)
