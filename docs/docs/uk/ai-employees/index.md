# ШІ Співробітники · Початок роботи

У NocoBase ШІ Співробітники не є чат-ботами чи ізольованими агентами — це інтелектуальні колеги, бездоганно інтегровані у вашу бізнес-систему, здатні розуміти бізнес-контекст та безпосередньо виконувати завдання.

Вони з'являються там, де ви їх потребуєте — поруч з таблицями та формами — щоб ви могли легко їх викликати.

![20251102121036-2025-11-02-12-10-38](https://static-docs.nocobase.com/20251102121036-2025-11-02-12-10-38.png)

Ви також можете викликати їх зі списку співробітників у нижньому правому куті сторінки.

![20251102121159-2025-11-02-12-12-01](https://static-docs.nocobase.com/20251102121159-2025-11-02-12-12-01.png)

Вони автоматично отримують дані як контекст — наприклад, **Viz** поруч з блоком таблиці автоматично отримує дані з таблиці та викликає відповідні інструменти для їх обробки. Це означає, що вам більше не потрібно копіювати дані в чат-бот.

<video controls class="rounded shadow"><source src="https://static-docs.nocobase.com/z-2025-11-02-12.13.35-2025-11-02-12-13-46.mp4" type="video/mp4"></video>

Вони також можуть автоматично розуміти структури сторінок як контекст — наприклад, **Dex** поруч з блоком форми автоматично отримує структуру полів форми та викликає потрібні інструменти для роботи зі сторінкою. Це означає, що вам не потрібно копіювати дані назад з чат-бота.

<video controls class="rounded shadow"><source src="https://static-docs.nocobase.com/z-2025-11-02-12.16.07-2025-11-02-12-16-21.mp4" type="video/mp4"></video>

Ви також можете безпосередньо вибирати елементи на сторінці та надсилати їх ШІ Співробітникам, дозволяючи їм автоматично витягувати відповідні дані та структуру.

<video controls class="rounded shadow"><source src="https://static-docs.nocobase.com/z-2025-11-02-12.17.25-2025-11-02-12-17-44.mp4" type="video/mp4"></video>
