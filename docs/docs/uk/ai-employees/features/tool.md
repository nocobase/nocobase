# Використання навичок

Навички (Інструменти) визначають, що може робити ШІ-співробітник.

## Структура навичок

Сторінка навичок розділена на три категорії:

1. `General skills`: спільні для всіх ШІ-співробітників, лише для читання.
2. `Employee-specific skills`: доступні лише для поточного співробітника, зазвичай лише для читання.
3. `Custom skills`: власні навички, які можна додавати/видаляти та налаштовувати для них дозволи за замовчуванням.

![skills-three-sections-general-specific-custom.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/skills-three-sections-general-specific-custom.png)

## Дозволи навичок

Дозволи для навичок уніфіковані:

- `Ask`: запитувати підтвердження перед викликом.
- `Allow`: дозволити прямий виклик.

Рекомендація: використовуйте `Ask` за замовчуванням для навичок, що передбачають зміну даних.

![skills-permission-ask-allow-segmented.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/skills-permission-ask-allow-segmented.png)

## Додавання та обслуговування

Натисніть `Add skill` у розділі `Custom skills`, щоб додати навички та налаштувати дозволи відповідно до бізнес-ризиків.
