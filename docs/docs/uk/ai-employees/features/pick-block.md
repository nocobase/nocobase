# Додавання контексту — Блок

Під час спілкування з AI-співробітником, окрім надсилання файлів, ви також можете надсилати контекстну інформацію додатка. Це дозволяє AI-співробітнику надавати відповіді на основі контексту додатка.

### Надсилання інформації про блок

У вікні чату з AI-співробітником натисніть кнопку `Додати робочий контекст` у нижньому лівому куті та виберіть `Вибрати блок`. Сторінка додатка перейде в режим вибору блоку.

![20251022010148](https://static-docs.nocobase.com/20251022010148.png)

При наведенні курсору на блок, колір блоку, який можна надіслати як контекст, зміниться.

![20251022010422](https://static-docs.nocobase.com/20251022010422.png)

Після вибору потрібного блоку інформація про його контекст буде додана до вікна чату.

![20251022010543](https://static-docs.nocobase.com/20251022010543.png)

Після введення запитання натисніть кнопку надсилання (так само як при звичайному повідомленні), і AI-співробітник надасть відповідь на основі інформації з блоку.

Інформація про блок також з'явиться у списку повідомлень.

![20251022010758](https://static-docs.nocobase.com/20251022010758.png)

### Завершення вибору блоку

Якщо сторінка додатка перейшла в режим вибору блоку, але ви не хочете вибирати жоден блок, натисніть іконку «Зупинити» внизу, щоб вийти з режиму вибору.

![20251022011043](https://static-docs.nocobase.com/20251022011043.png)
