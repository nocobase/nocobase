# Швидкі завдання

Щоб зробити роботу ШІ-співробітників ефективнішою, ви можете прив'язати їх до блоків сценаріїв та заздалегідь налаштувати кілька типових завдань.

Це дозволяє користувачам запускати обробку завдань одним кліком, уникаючи необхідності щоразу **вибирати блок** та **вводити інструкції**.

## Прив'язка ШІ-співробітника до блоку

Після переходу в режим редагування інтерфейсу (UI), для блоків, що підтримують `Actions`, виберіть меню `AI employees` у розділі `Actions`, а потім виберіть ШІ-співробітника. Цей ШІ-співробітник буде прив'язаний до поточного блоку.

![20251022135306](https://static-docs.nocobase.com/20251022135306.png)

Після завершення прив'язки при кожному вході на сторінку в області `Actions` блоку відображатиметься ШІ-співробітник, закріплений за цим блоком.

![20251022135438](https://static-docs.nocobase.com/20251022135438.png)

## Налаштування завдань

У режимі редагування інтерфейсу наведіть курсор на іконку ШІ-співробітника, прив'язаного до блоку. З'явиться кнопка меню; виберіть `Edit tasks`, щоб перейти на сторінку налаштування завдань.

На сторінці налаштування завдань ви можете додати кілька завдань для поточного ШІ-співробітника.

Кожна вкладка представляє окреме завдання; натисніть «+» поруч, щоб додати нове завдання.

![clipboard-image-1771913187](https://static-docs.nocobase.com/clipboard-image-1771913187.png)

Форма налаштування завдання:

- У полі `Title` введіть назву завдання. Це короткий опис, який з'явиться у списку завдань ШІ-співробітника.
- У полі `Background` введіть основний зміст завдання. Цей текст використовуватиметься як системна підказка (system prompt) під час діалогу з ШІ-співробітником.
- У полі `Default user message` введіть повідомлення користувача за замовчуванням. Воно автоматично заповнить поле введення після вибору завдання.
- У розділі `Work context` виберіть контекстну інформацію додатка, яка за замовчуванням надсилатиметься ШІ-співробітнику. Це працює так само, як і в панелі чату.
- У списку `Skills` відображаються навички, доступні поточному ШІ-співробітнику. Ви можете вимкнути певну навичку, щоб ШІ-співробітник ігнорував її під час виконання цього завдання.
- Прапорець `Send default user message automatically` визначає, чи буде повідомлення користувача за замовчуванням надсилатися автоматично відразу після натискання на завдання.


## Список завдань

Після налаштування завдань для ШІ-співробітника вони відображатимуться у спливаючому вікні профілю ШІ-співробітника та у вітальному повідомленні перед початком бесіди. Натисніть на завдання, щоб почати його виконання.
