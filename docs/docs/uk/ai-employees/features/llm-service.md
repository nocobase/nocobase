# Налаштування сервісу LLM

Перш ніж використовувати AI-співробітників, необхідно налаштувати доступні сервіси LLM.

Наразі підтримуються OpenAI, Gemini, Claude, DeepSeek, Qwen, Kimi, а також локальні моделі Ollama.

## Створення сервісу

Перейдіть до `Налаштування системи -> AI-співробітники -> LLM service`.

1. Натисніть `Add New`, щоб відкрити вікно створення.
2. Виберіть `Provider`.
3. Заповніть `Title`, `API Key` та `Base URL` (необов'язково).
4. Налаштуйте `Enabled Models`:
   - `Recommended models`: використання офіційно рекомендованих моделей.
   - `Select models`: вибір зі списку, наданого провайдером.
   - `Manual input`: ручне введення ID моделі та назви для відображення.
5. Натисніть `Submit`, щоб зберегти.

![llm-service-create-provider-enabled-models.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/llm-service-create-provider-enabled-models.png)

## Увімкнення та сортування сервісів

У списку сервісів LLM ви можете безпосередньо:

- Використовувати перемикач `Enabled` для ввімкнення або вимкнення сервісу.
- Перетягувати сервіси для зміни їхнього порядку (це впливає на черговість відображення моделей).

![llm-service-list-enabled-and-sort.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/llm-service-list-enabled-and-sort.png)

## Тестування доступності

Використовуйте `Test flight` у нижній частині вікна конфігурації сервісу, щоб перевірити доступність сервісу та моделей.

Рекомендується спочатку провести тестування, а потім використовувати сервіс у робочих процесах.
