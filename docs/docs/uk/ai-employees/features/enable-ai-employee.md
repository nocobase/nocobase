# Увімкнення AI-співробітників

NocoBase пропонує кілька вбудованих AI-співробітників для різних сценаріїв.

## Як увімкнути

Перейдіть на сторінку керування `AI employees` та активуйте перемикач `Enabled` безпосередньо у списку.

![ai-employee-list-enable-switch.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employee-list-enable-switch.png)

## Використання після увімкнення

Після увімкнення співробітник з'явиться у доступних точках входу (наприклад, у меню швидкого доступу в нижньому правому куті або в точках входу для конкретних сценаріїв).
