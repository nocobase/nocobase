# Пошук в інтернеті

Пошук в інтернеті використовується для доповнення знань моделі актуальною зовнішньою інформацією, що виходить за межі її навчальних даних.

## Як це працює

Доступність пошуку в інтернеті залежить від того, чи підтримує вибраний сервіс моделі функцію Web Search у поточному сеансі.

- **Підтримується:** відображається перемикач пошуку в інтернеті, який можна вмикати або вимикати за потреби.
- **Не підтримується:** перемикач приховано, а функція пошуку автоматично вимикається.

## Використання в чаті

Використовуйте перемикач пошуку в інтернеті в області введення повідомлення:

- **Увімкнено:** ШІ може спочатку виконати пошук в мережі, а потім надати відповідь.

![web-search-switch-visible-on-supported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-visible-on-supported-model.png)

- **Вимкнено:** ШІ відповідає лише на основі наявного контексту.

![web-search-switch-hidden-on-unsupported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-hidden-on-unsupported-model.png)

## Відмінності між платформами

Можливості пошуку в інтернеті відрізняються залежно від моделі. Будь ласка, використовуйте цю функцію відповідно до фактичних можливостей вибраного вами сервісу та моделі.
