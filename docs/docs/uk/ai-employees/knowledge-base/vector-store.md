# Векторне сховище

## Вступ

У базі знань, як при збереженні документів, так і при пошуку (для векторизації пошукових термінів), необхідно використовувати `Embedding model` для перетворення вихідного тексту у вектори.

У плагіні AI Knowledge Base векторне сховище — це зв'язка `Embedding model` та векторної бази даних.

## Керування векторними сховищами

Перейдіть на сторінку конфігурації плагіна AI Employees, натисніть вкладку `Vector store` і виберіть `Vector store`, щоб перейти на сторінку керування векторними сховищами.

![20251023003023](https://static-docs.nocobase.com/20251023003023.png)

Натисніть кнопку `Add new` у верхньому правому куті, щоб додати нове векторне сховище:

- У полі `Name` введіть назву векторного сховища;
- У `Vector store` виберіть уже налаштовану векторну базу даних. Дивіться: [Векторна база даних](/ai-employees/knowledge-base/vector-database);
- У `LLM service` виберіть уже налаштований LLM сервіс. Дивіться: [Керування LLM сервісами](/ai-employees/quick-start/llm-service);
- У полі `Embedding model` введіть назву `Embedding` моделі, яку потрібно використовувати;

Натисніть кнопку `Submit`, щоб зберегти інформацію про векторне сховище.
