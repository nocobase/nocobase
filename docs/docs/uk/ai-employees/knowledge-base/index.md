# Огляд

## Вступ

Плагін AI: База знань надає AI-агентам можливості RAG-пошуку.

Можливості RAG-пошуку дозволяють AI-агентам надавати більш точні, професійні та релевантні для підприємства відповіді на запитання користувачів.

Використання професійних галузевих та внутрішніх корпоративних документів з бази знань, яку підтримує адміністратор, підвищує точність та простежуваність відповідей AI-агентів.

### Що таке RAG

RAG (Retrieval Augmented Generation) означає "Пошук-Доповнення-Генерація".

- **Пошук**: Запитання користувача перетворюється на вектор за допомогою моделі вбудовування (Embedding model) (наприклад, BERT). З векторної бібліотеки вибираються Top-K релевантних текстових фрагментів за допомогою щільного пошуку (семантична схожість) або розрідженого пошуку (відповідність ключових слів).
- **Доповнення**: Результати пошуку об'єднуються з початковим запитанням, утворюючи доповнений запит (Prompt), який потім вводиться в контекстне вікно LLM.
- **Генерація**: LLM використовує доповнений запит для генерації остаточної відповіді, забезпечуючи її фактологічність та простежуваність.
