# База знань

## Вступ

База знань є основою для RAG-пошуку. Вона організовує документи за категоріями та створює індекси. Коли AI-співробітник відповідає на запитання, він насамперед шукатиме відповіді в базі знань.

## Керування базою знань

Перейдіть на сторінку конфігурації плагіна AI-співробітників, натисніть вкладку `Knowledge base`, щоб перейти на сторінку керування базою знань.

![20251023095649](https://static-docs.nocobase.com/20251023095649.png)

Натисніть кнопку `Add new` у верхньому правому куті, щоб додати локальну (`Local`) базу знань.

![20251023095826](https://static-docs.nocobase.com/20251023095826.png)

Введіть необхідну інформацію для нової бази знань:

- У полі `Name` введіть назву бази знань;
- У `File storage` виберіть місце зберігання файлів;
- У `Vector store` виберіть векторне сховище, див. [Векторне сховище](/ai-employees/knowledge-base/vector-store);
- У полі `Description` введіть опис бази знань;

Натисніть кнопку `Submit`, щоб створити базу знань.

![20251023095909](https://static-docs.nocobase.com/20251023095909.png)

## Керування документами бази знань

Після створення бази знань, на сторінці списку баз знань, натисніть на щойно створену базу знань, щоб перейти на сторінку керування документами бази знань.

![20251023100458](https://static-docs.nocobase.com/20251023100458.png)

![20251023100527](https://static-docs.nocobase.com/20251023100527.png)

Натисніть кнопку `Upload`, щоб завантажити документи. Після завантаження документів автоматично розпочнеться векторизація. Дочекайтеся, поки `Status` зміниться зі стану `Pending` на `Success`.

Наразі база знань підтримує такі типи документів: txt, pdf, doc, docx, ppt, pptx; PDF підтримує лише звичайний текст.

![20251023100901](https://static-docs.nocobase.com/20251023100901.png)

## Типи баз знань

### Локальна база знань (`Local`)

Локальна (`Local`) база знань — це база знань, що зберігається локально в NocoBase. Документи та їхні векторні дані зберігаються локально NocoBase.

![20251023101620](https://static-docs.nocobase.com/20251023101620.png)

### База знань лише для читання (`Readonly`)

База знань лише для читання (`Readonly`) — це база знань, що доступна тільки для читання. Документи та векторні дані підтримуються ззовні. У NocoBase створюється лише підключення до векторної бази даних (наразі підтримується лише PGVector).

![20251023101743](https://static-docs.nocobase.com/20251023101743.png)

### Зовнішня база знань (`External`)

Зовнішня (`External`) база знань — це зовнішня база знань, де документи та векторні дані підтримуються ззовні. Пошук у векторній базі даних потребує розширення розробниками, що дозволяє використовувати векторні бази даних, які наразі не підтримуються NocoBase.

![20251023101949](https://static-docs.nocobase.com/20251023101949.png)
