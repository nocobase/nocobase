# RAG-пошук

## Вступ

Після налаштування бази знань ви можете увімкнути функцію RAG у налаштуваннях AI-співробітника.

Коли RAG увімкнено, під час розмови користувача з AI-співробітником, AI-співробітник використовуватиме RAG-пошук, щоб отримати документи з бази знань на основі повідомлення користувача та відповісти, спираючись на знайдені документи.

## Увімкнення RAG

Перейдіть на сторінку конфігурації плагіна AI-співробітників, натисніть вкладку `AI employees`, щоб перейти на сторінку керування AI-співробітниками.

![20251023010811](https://static-docs.nocobase.com/20251023010811.png)

Виберіть AI-співробітника, для якого ви хочете увімкнути RAG, натисніть кнопку `Edit`, щоб перейти на сторінку редагування AI-співробітника.

На вкладці `Knowledge base` увімкніть перемикач `Enable`.

- У полі `Knowledge Base Prompt` введіть підказку для посилання на базу знань. `{knowledgeBaseData}` є фіксованим заповнювачем, його не слід змінювати;
- У полі `Knowledge Base` виберіть налаштовану базу знань. Дивіться: [База знань](/ai-employees/knowledge-base/knowledge-base);
- У полі `Top K` введіть кількість документів для пошуку, за замовчуванням – 3;
- У полі `Score` введіть поріг релевантності документів для пошуку;

Натисніть кнопку `Submit`, щоб зберегти налаштування AI-співробітника.
