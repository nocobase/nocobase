# Життєвий цикл

Цей розділ систематизує хуки життєвого циклу для плагінів як на стороні сервера, так і на стороні клієнта, допомагаючи розробникам правильно реєструвати та звільняти ресурси.

Ви можете порівняти його з життєвим циклом FlowModel, щоб виділити спільні концепції.

## Рекомендований зміст

- Колбеки, що спрацьовують, коли плагіни встановлюються, вмикаються, вимикаються або видаляються.
- Моменти монтування, оновлення та знищення клієнтських компонентів.
- Рекомендації щодо обробки асинхронних завдань та помилок в межах життєвого циклу.
