# ACL: Контроль доступу

:::tip

У цьому розділі ми розглянемо, як використовувати ACL (список контролю доступу) для керування дозволами у фронтенд-плагінах. Зміст включатиме:

- Перевірка дозволів;
- Як динамічно відображати контент на основі дозволів;

Ми доповнимо цей розділ детальними прикладами та найкращими практиками у майбутніх версіях.

:::
