# Керування мультизастосунками

## Огляд

NocoBase дозволяє керувати кількома ізольованими застосунками через єдину точку входу за допомогою AppSupervisor.


На старті зазвичай достатньо одного застосунку. Із ростом збільшується складність, вартість і потреба в ізоляції.


У цьому режимі кілька застосунків працюють в одному інстансі NocoBase: БД можуть бути окремими, але процес і пам’ять спільні.

![](https://static-docs.nocobase.com/202512231055907.png)


Для великого масштабу використовуйте гібрид: Supervisor + кілька Worker-середовищ.

![](https://static-docs.nocobase.com/202512231215186.png)
