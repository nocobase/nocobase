# Вбудовування NocoBase

## Вступ

Цей плагін використовується для вбудовування сторінок NocoBase в інші вебсайти або застосунки.

## Встановлення

Цей плагін є комерційним, і його потрібно завантажити та активувати через менеджер плагінів.

![20240323162741](https://static-docs.nocobase.com/20240323162741.png)

## Інструкція з використання

Після активації конфігурації інтерфейсу натисніть «Копіювати посилання для вбудовування» у налаштуваннях сторінки (у верхньому правому куті).

![](https://static-docs.nocobase.com/f11bd6d5e88d38731d7cd3cb149022c8.png)

Після натискання ви отримаєте посилання, яке можна відкрити окремо, наприклад, `https://xxx.com/embed/qs087rz4o2b`.

![](https://static-docs.nocobase.com/9d847805e00fd07372a474665ec0e968.png)

Якщо ви бажаєте вбудувати сторінки NocoBase в інші вебсайти або застосунки, потрібна автентифікація користувача, а `token` необхідно додати до посилання, наприклад, `https://xxx.com/embed/qs087rz4o2b?token=xxx`.
