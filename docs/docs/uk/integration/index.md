# Інтеграція

## Огляд

NocoBase пропонує широкі можливості інтеграції, що дозволяють безперешкодно підключатися до зовнішніх систем, сторонніх сервісів та різноманітних джерел даних. Завдяки гнучким методам інтеграції ви можете розширити функціональність NocoBase для задоволення різноманітних бізнес-потреб.

## Методи інтеграції

### Інтеграція через API

NocoBase надає потужні можливості API для інтеграції із зовнішніми застосунками та сервісами:

![20250301003106](https://static-docs.nocobase.com/20250301003106.png)

- **[Ключі API](/integration/api-keys/)**: Використовуйте ключі API для безпечної автентифікації та програмного доступу до ресурсів NocoBase.
- **[Документація API](/integration/api-doc/)**: Вбудована документація API для дослідження та тестування кінцевих точок.

### Єдиний вхід (SSO)

Інтегруйтеся з корпоративними системами ідентифікації для єдиної автентифікації:

![auth_sso-2025-11-24-08-26-46](https://static-docs.nocobase.com/auth_sso-2025-11-24-08-26-46.png)

- **[Інтеграція SSO](/integration/sso/)**: Підтримка автентифікації через SAML, OIDC, CAS, LDAP та сторонні платформи.
- Централізоване керування користувачами та контроль доступу.
- Безперешкодний досвід автентифікації в різних системах.

### Інтеграція робочих процесів

Підключайте робочі процеси NocoBase до зовнішніх систем:

![auth_sso-2025-11-24-12-02-01](https://static-docs.nocobase.com/auth_sso-2025-11-24-12-02-01.png)

- **[Вебхуки робочих процесів](/integration/workflow-webhook/)**: Отримуйте події від зовнішніх систем для запуску робочих процесів.
- **[HTTP-запити робочих процесів](/integration/workflow-http-request/)**: Надсилайте HTTP-запити до зовнішніх API з робочих процесів.
- Автоматизуйте бізнес-процеси на різних платформах.

### Зовнішні джерела даних

Підключайтеся до зовнішніх баз даних та систем даних:

![20240512085558](https://static-docs.nocobase.com/20240512085558.png)

- **[Зовнішні бази даних](/data-sources/)**: Безпосередньо підключайтеся до баз даних MySQL, PostgreSQL, MariaDB, MSSQL, Oracle та KingbaseES.
- Розпізнавайте структури таблиць зовнішніх баз даних та виконуйте операції CRUD (створення, читання, оновлення, видалення) із зовнішніми даними безпосередньо в NocoBase.
- Єдиний інтерфейс керування даними.

### Вбудований вміст

Вбудовуйте зовнішній вміст у NocoBase:

![20240408220322](https://static-docs.nocobase.com/20240408220322.png)

- **[Блок Iframe](/integration/block-iframe/)**: Вбудовуйте зовнішні веб-сторінки та застосунки.
- **Блоки JS**: Виконуйте власний JavaScript-код для розширених інтеграцій.

## Поширені сценарії інтеграції

### Інтеграція з корпоративними системами

- Підключайте NocoBase до ERP, CRM або інших корпоративних систем.
- Синхронізуйте дані в обох напрямках.
- Автоматизуйте міжсистемні робочі процеси.

### Інтеграція зі сторонніми сервісами

- Запитуйте статус платежів у платіжних шлюзів, інтегруйте служби обміну повідомленнями або хмарні платформи.
- Використовуйте зовнішні API для розширення функціональності.
- Створюйте власні інтеграції за допомогою вебхуків та HTTP-запитів.

### Інтеграція даних

- Підключайтеся до кількох джерел даних.
- Агрегуйте дані з різних систем.
- Створюйте єдині інформаційні панелі та звіти.

## Міркування безпеки

Під час інтеграції NocoBase із зовнішніми системами враховуйте наступні найкращі практики безпеки:

1.  **Використовуйте HTTPS**: Завжди використовуйте зашифровані з'єднання для передачі даних.
2.  **Захищайте ключі API**: Безпечно зберігайте ключі API та регулярно їх змінюйте.
3.  **Принцип найменших привілеїв**: Надавайте лише необхідні дозволи для інтеграцій.
4.  **Журнали аудиту**: Моніторте та реєструйте діяльність інтеграції.
5.  **Валідація даних**: Перевіряйте всі дані, що надходять із зовнішніх джерел.
