# Блокування користувачів

## Вступ

Цей розділ дозволяє керувати заблокованими користувачами та активно блокувати їх.

![](https://static-docs.nocobase.com/202412281450083.png)

### Розблокування користувачів

Користувачів, які були заблоковані через перевищення ліміту невдалих спроб входу з невірним паролем, можна розблокувати, видаливши відповідний запис.

### Блокування користувачів

Ви можете активно заблокувати користувача, додавши відповідний запис. Після блокування користувачеві буде заборонено доступ до системи будь-яким методом автентифікації, включно з API-ключами.

![](https://static-docs.nocobase.com/202412281450512.png)
