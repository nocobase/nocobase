# Журнал аудиту

## Вступ

Журнал аудиту використовується для запису та відстеження активності користувачів та історії операцій з ресурсами в системі.

![](https://static-docs.nocobase.com/202501031627719.png)

![](https://static-docs.nocobase.com/202501031627922.png)

## Опис параметрів

| Параметр                  | Опис                                                                                                 |
| :------------------------ | :--------------------------------------------------------------------------------------------------- |
| **Resource**              | Тип цільового ресурсу операції                                                                       |
| **Action**                | Тип виконаної операції                                                                               |
| **User**                  | Користувач, який виконав операцію                                                                    |
| **Role**                  | Роль користувача під час операції                                                                    |
| **Data source**           | Джерело даних                                                                                        |
| **Target collection**     | Цільова колекція                                                                                     |
| **Target record UK**      | Унікальний ідентифікатор цільової колекції                                                           |
| **Source collection**     | Вихідна колекція поля зв'язку                                                                        |
| **Source record UK**      | Унікальний ідентифікатор вихідної колекції                                                           |
| **Status**                | HTTP-код стану відповіді на запит операції                                                           |
| **Created at**            | Час операції                                                                                         |
| **UUID**                  | Унікальний ідентифікатор операції, що відповідає Request ID запиту операції; може використовуватися для пошуку логів застосунку. |
| **IP**                    | IP-адреса користувача                                                                                |
| **UA**                    | Інформація UA користувача                                                                            |
| **Metadata**              | Метадані, такі як параметри, тіло запиту та вміст відповіді операції                                 |

## Опис аудиторських ресурсів

Наразі до журналу аудиту записуються такі операції з ресурсами:

### Основний застосунок

| Операція         | Опис                  |
| :--------------- | :-------------------- |
| `app:resart`     | Перезапуск застосунку |
| `app:clearCache` | Очищення кешу застосунку |

### Менеджер плагінів

| Операція     | Опис            |
| :----------- | :-------------- |
| `pm:add`     | Додати плагін   |
| `pm:update`  | Оновити плагін  |
| `pm:enable`  | Увімкнути плагін |
| `pm:disable` | Вимкнути плагін |
| `pm:remove`  | Видалити плагін |

### Автентифікація користувача

| Операція              | Опис          |
| :-------------------- | :------------ |
| `auth:signIn`         | Вхід          |
| `auth:signUp`         | Реєстрація    |
| `auth:signOut`        | Вихід         |
| `auth:changePassword` | Зміна пароля  |

### Користувач

| Операція              | Опис             |
| :-------------------- | :--------------- |
| `users:updateProfile` | Оновлення профілю |

### Конфігурація UI

| Операція                   | Опис             |
| :------------------------- | :--------------- |
| `uiSchemas:insertAdjacent` | Вставити UI Schema |
| `uiSchemas:patch`          | Змінити UI Schema |
| `uiSchemas:remove`         | Видалити UI Schema |

### Операції з колекціями

| Операція         | Опис                       |
| :--------------- | :------------------------- |
| `create`         | Створити запис             |
| `update`         | Оновити запис              |
| `destroy`        | Видалити запис             |
| `updateOrCreate` | Оновити або створити запис |
| `firstOrCreate`  | Знайти або створити запис  |
| `move`           | Перемістити запис          |
| `set`            | Встановити запис поля зв'язку |
| `add`            | Додати запис поля зв'язку  |
| `remove`         | Видалити запис поля зв'язку |
| `export`         | Експортувати записи        |
| `import`         | Імпортувати записи         |

## Додавання інших аудиторських ресурсів

Якщо ви розширили операції з іншими ресурсами за допомогою плагінів і бажаєте записувати ці дії до журналу аудиту, будь ласка, зверніться до [API](/api/server/audit-manager.md).
