# Колекції

Колекція — це одне з базових та ключових понять у NocoBase. Можна уявити колекцію як таблицю в базі даних або аркуш в Excel.

## Типи колекцій

У NocoBase існує два типи колекцій:

- **Звичайна колекція**: найпоширеніший тип, що відповідає фізичній таблиці в базі даних. Ви можете вільно додавати, видаляти та змінювати поля.
- **SQL-колекція**: дані для цього типу колекції надходять із SQL-запиту. Вона доступна лише для читання, і ви не можете безпосередньо змінювати її структуру чи дані. В основному використовується для відображення результатів складних запитів.

## Створення колекції

Щоб створити нову колекцію, виконайте наступні кроки:

1.  У меню ліворуч натисніть «Джерела даних».
2.  Виберіть джерело даних, до якого ви хочете додати колекцію.
3.  Натисніть кнопку «Додати колекцію».
4.  Виберіть тип колекції (звичайна або SQL) та заповніть відповідну інформацію.

### Приклад звичайної колекції

Коли ви створюєте «Звичайну колекцію», вам потрібно вказати:

- **Назва для відображення**: наприклад, «Замовлення».
- **Назва колекції**: наприклад, `orders`, буде використовуватися як назва таблиці.

### Приклад SQL-колекції

Коли ви створюєте «SQL-колекцію», вам потрібно надати SQL-запит:

```sql
SELECT
  o.id,
  o.order_date,
  c.name AS customer_name,
  SUM(p.price) AS total_amount
FROM
  orders o
JOIN
  customers c ON o.customer_id = c.id
JOIN
  order_items oi ON o.id = oi.order_id
JOIN
  products p ON oi.product_id = p.id
GROUP BY
  o.id, c.name;
```

Цей запит створить колекцію «Підсумок замовлень» (лише для читання), що відображатиме загальну суму та ім'я клієнта для кожного замовлення.

## Керування колекціями

Після створення колекції ви можете:

- **Додавати поля**: визначати нові стовпці для колекції.
- **Налаштовувати представлення**: створювати різні способи відображення даних, наприклад, таблиці, канбан-дошки тощо.
- **Встановлювати дозволи**: контролювати, які ролі можуть отримувати доступ до даних у колекції або змінювати їх.

Для отримання додаткової інформації дивіться розділи [Керування полями](#) та [Налаштування дозволів](#).
