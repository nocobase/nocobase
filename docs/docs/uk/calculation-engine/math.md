# Mathjs

[Math.js](https://mathjs.org/) — це багатофункціональна математична бібліотека для JavaScript та Node.js.

## Довідник функцій

### Вирази

| Функція | Визначення | Приклад виклику | Параметри | Очікуваний результат |

| **evaluate** | Обчислює вираз та повертає результат. | `evaluate('2 + 3')` | вираз (рядок), область видимості (необов'язково) | `5` |
