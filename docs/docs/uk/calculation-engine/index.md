# Огляд механізмів обчислення

Механізми обчислення відіграють ключову роль у полях формул, правилах зв'язку та робочих процесах NocoBase, надаючи користувачам універсальні інструменти для розрахунків.

## Formula.js

**Formula.js** — це легка бібліотека JavaScript, розроблена спеціально для обчислення формул. Вона відтворює багато функцій Microsoft Excel, що дозволяє користувачам легко виконувати складні розрахунки. Основні особливості включають:

- **Широкий набір функцій**: охоплює велику кількість функцій для різних математичних операцій, включаючи статистичні та фінансові розрахунки.
- **Простота у використанні**: пропонує простий API, що дозволяє як розробникам, так і звичайним користувачам створювати формули без написання великої кількості коду.
- **Обчислення в реальному часі**: оптимізовано для обробки даних у реальному часі, що підходить для додатків, які потребують миттєвого результату обчислень.

## Math.js

**Math.js** — це потужна математична бібліотека для JavaScript та Node.js. Вона підтримує широкий спектр операцій: від базової арифметики до просунутої алгебри, статистики та лінійної алгебри. Основні особливості включають:

- **Комплексна функціональність**: надає величезну кількість функцій для всіх типів математичних операцій.
- **Гнучкий розбір виразів**: здатна розбирати та обчислювати математичні вирази у форматі рядків.
- **Складні обчислення**: підтримує операції з матрицями, символьні обчислення, статистичний аналіз та інші просунуті функції.
- **Розширюваність**: бібліотеку можна доповнювати власними функціями та константами для задоволення специфічних потреб.

## Застосування в NocoBase

Завдяки поєднанню Formula.js та Math.js, користувачі NocoBase можуть:

- **Виконувати широкий спектр обчислень**: від простих арифметичних дій до складних математичних операцій, розширюючи обчислювальні можливості своїх додатків.
- **Створювати надійні та ефективні додатки**: розробляти рішення, які потребують точних і потужних математичних обчислень, забезпечуючи надійність та продуктивність.
- **Підвищувати продуктивність**: використовувати готові функції, щоб уникнути написання великої кількості власного коду, тим самим прискорюючи процес розробки.

Ці дві бібліотеки відіграють ключову роль у no-code платформі NocoBase, дозволяючи створювати гнучку та настроювану логіку обчислень без необхідності глибокого програмування. Це не лише спрощує процес розробки, але й розширює функціональність та адаптивність додатків.
