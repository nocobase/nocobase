# Вступ

NocoBase — це відкрита no-code платформа для розробки з широкими можливостями розширення. Вона дозволяє створювати власні платформи для співпраці та системи управління без програмування, задовольняючи різноманітні бізнес-потреби.

На відміну від таких інструментів, як Airtable та Notion, NocoBase зберігає дані у вашій базі даних. Це надає вам повний контроль над даними та необмежені можливості для зовнішніх інтеграцій і власної розробки.

## Чому варто обрати NocoBase?

### 1. No-code

NocoBase має зручний та приємний інтерфейс, що дозволяє легко розпочати роботу навіть нетехнічним користувачам.

- **Візуальне налаштування форм та таблиць**: легко створюйте та керуйте моделями даних за допомогою перетягування компонентів.
- **Гнучкі представлення**: таблиця, форма, канбан-дошка, календар, деталі тощо.
- **Детальний контроль доступу**: керування доступом на основі ролей з точністю до окремого поля.
- **Потужний рушій робочих процесів**: автоматизуйте бізнес-процеси та підвищуйте ефективність.

### 2. Потужні можливості розширення

NocoBase використовує архітектуру на основі мікроядра та плагінів, що робить його гнучким для розширення.

- **Архітектура на основі плагінів**: усі функції реалізовані як плагіни, які можна встановлювати та видаляти за потреби.
- **Фрагменти коду**: реалізуйте складні функції за допомогою простих фрагментів коду.
- **Власна розробка**: розробляйте власні плагіни для задоволення індивідуальних потреб.

### 3. Автономність даних

NocoBase підтримує різноманітні популярні бази даних, тож ви можете зберігати дані на власних серверах.

- **Підтримка різних баз даних**: MySQL, PostgreSQL, SQL Server, SQLite тощо.
- **Безпека даних**: дані зберігаються на ваших серверах, що забезпечує їх безпеку та контроль.
- **Легка міграція**: ви можете будь-коли перенести свої дані до інших систем.
