# Чернетки форм

Коли ввімкнено чернетки форм, вміст, що заповнюється, автоматично зберігається локально. Навіть якщо форму випадково закрити або оновити сторінку, заповнені дані не будуть втрачені.

![20251213093717](https://static-docs.nocobase.com/20251213093717.png)

Якщо чернетка існує, у верхній частині форми з'явиться сповіщення. Після успішного надсилання форми чернетка буде видалена автоматично; якщо вона більше не потрібна, її також можна видалити вручну в будь-який час.

![20251213092734](https://static-docs.nocobase.com/20251213092734.png)

::: tip Порада

Наразі чернетки форм підтримують лише локальне кешування. У майбутніх версіях буде реалізовано кешування на стороні сервера для синхронізації між декількома пристроями.

:::
