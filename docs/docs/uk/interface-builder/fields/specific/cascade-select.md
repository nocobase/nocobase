# Каскадний вибір

## Вступ

Каскадний селектор призначений для полів зв'язку, цільова колекція яких є деревоподібною таблицею. Він дозволяє користувачам вибирати дані відповідно до ієрархічної структури деревоподібної колекції та підтримує нечіткий пошук для швидкої фільтрації.

## Інструкції з використання

- Для зв'язку **"один до одного"** каскадний селектор передбачає **одиночний вибір**.

![20251125214656](https://static-docs.nocobase.com/20251125214656.png)

- Для зв'язку **"один до багатьох"** каскадний селектор передбачає **множинний вибір**.

![20251125215318](https://static-docs.nocobase.com/20251125215318.png)
