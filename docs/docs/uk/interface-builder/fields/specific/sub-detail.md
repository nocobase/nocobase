# Піддеталі

## Вступ

Піддеталі — це компонент, який відповідає підформі в режимі перегляду. На відміну від компонентів «Мітка» та «Заголовок», Піддеталі можуть не лише відображати більше даних з поточної колекції, але й дозволяють налаштовувати відображення даних з пов'язаних колекцій. Багаторівневі пов'язані дані чітко відображаються у вигляді вкладених деталей.
