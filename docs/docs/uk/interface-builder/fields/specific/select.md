# Випадаючий список

## Вступ

Випадаючий список дозволяє пов'язувати дані, обираючи їх з існуючих записів у цільовій колекції, або додаючи нові дані до неї для подальшого зв'язування. Опції випадаючого списку підтримують нечіткий пошук.

![20251029205901](https://static-docs.nocobase.com/20251029205901.png)

## Налаштування поля

### Налаштування області даних

Визначає область даних, доступних у випадаючому списку.

![20251029210025](https://static-docs.nocobase.com/20251029210025.png)

Детальніше дивіться у розділі [Налаштування області даних](/interface-builder/fields/field-settings/data-scope)

### Налаштування правил сортування

Контролює порядок сортування даних у випадаючому списку.

Приклад: Сортування за датою послуги у зворотному порядку.

![20251029210105](https://static-docs.nocobase.com/20251029210105.png)

### Дозволити додавання/пов'язування кількох записів

Обмежує зв'язок "один-до-багатьох", дозволяючи пов'язувати лише один запис.

![20251029210145](https://static-docs.nocobase.com/20251029210145.png)

### Поле заголовка

Поле заголовка — це поле-мітка, що відображається в опціях випадаючого списку.

![20251029210507](https://static-docs.nocobase.com/20251029210507.gif)

> Підтримує швидкий пошук за полем заголовка

Детальніше дивіться у розділі [Поле заголовка](/interface-builder/fields/field-settings/title-field)

### Швидке створення: Спочатку створити, потім обрати

![20251125220046](https://static-docs.nocobase.com/20251125220046.png)

#### Додавання через випадаючий список

Після створення нового запису в цільовій колекції система автоматично обирає його та пов'язує після надсилання форми.

У прикладі нижче колекція Замовлень має поле зв'язку "багато-до-одного" **"Обліковий запис"**.

![20251125220447](https://static-docs.nocobase.com/20251125220447.gif)

#### Додавання через модальне вікно

Додавання через модальне вікно підходить для складніших сценаріїв введення даних і дозволяє налаштовувати власну форму для створення нових записів.

![20251125220607](https://static-docs.nocobase.com/20251125220607.gif)
