# Підформа

## Вступ

Підформи ідеально підходять для сценаріїв, коли ви спочатку створюєте пов'язані дані, а потім їх пов'язуєте. Багаторівневі пов'язані дані чітко відображаються у вигляді вкладених форм. На відміну від компонентів «Селектор даних» та «Випадаючий список», підформа дозволяє безпосередньо керувати полями пов'язаної колекції в поточному блоці сторінки, і пов'язані дані надсилаються разом з основною формою.
