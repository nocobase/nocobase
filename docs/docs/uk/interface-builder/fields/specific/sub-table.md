# Підтаблиця (вбудоване редагування)

## Вступ

Підтаблиця ідеально підходить для роботи з полями Association field типу One-to-Many (O2M). Вона дозволяє як пакетно створювати та пов'язувати нові дані в цільовій таблиці (Collection), так і вибирати та пов'язувати вже існуючі дані.

## Інструкції з використання

![20251027223350](https://static-docs.nocobase.com/20251027223350.png)

Різні типи полів у підтаблиці відображаються за допомогою різних компонентів. Великі поля (наприклад, Rich Text, JSON, Long Text) редагуються у спливаючому модальному вікні.

![20251027223426](https://static-docs.nocobase.com/20251027223426.png)

Association fields у підтаблиці.

Замовлення (One-to-Many (O2M)) > Товари замовлення (One-to-One (O2O)) > Можливість

![20251027223530](https://static-docs.nocobase.com/20251027223530.png)

Компонент поля зв'язку за замовчуванням — це Single select (підтримує Single select / Collection selector).
