# Підтаблиця

## Вступ

Підтаблиця ідеально підходить для роботи з полями зв'язку типу "один-до-багатьох". Вона дозволяє як пакетно створювати та пов'язувати нові дані в цільовій колекції, так і вибирати та пов'язувати вже існуючі дані.

## Інструкції з використання

![20251027223350](https://static-docs.nocobase.com/20251027223350.png)

Різні типи полів у підтаблиці відображаються за допомогою різних компонентів. Великі поля (наприклад, розширений текст, JSON, багаторядковий текст) редагуються у спливаючому модальному вікні.

![20251027223426](https://static-docs.nocobase.com/20251027223426.png)

Поля зв'язку в підтаблиці.

Замовлення (Один-до-багатьох) > Товари замовлення (Один-до-одного) > Можливість

![20251027223530](https://static-docs.nocobase.com/20251027223530.png)

Компонент поля зв'язку за замовчуванням — це випадаючий список (підтримує випадаючий список / селектор даних).

![20251027223754](https://static-docs.nocobase.com/20251027223754.png)
