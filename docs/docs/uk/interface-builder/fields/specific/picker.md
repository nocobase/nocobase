# Селектор даних

## Вступ

Селектор даних, що реалізований у вигляді спливаючого табличного селектора, дозволяє обирати наявні дані з цільової колекції для зв'язування або додавати нові дані до цільової колекції з подальшим зв'язуванням.

![20251027203343](https://static-docs.nocobase.com/20251027203343.png)

Обрані дані не відображатимуться у табличному селекторі.

![20251027203717](https://static-docs.nocobase.com/20251027203717.gif)

У табличному селекторі ви також можете додатково керувати цільовою колекцією для зв'язування (додавати, видаляти, імпортувати/експортувати тощо).

![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
