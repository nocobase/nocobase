# Заголовок

## Вступ

У режимі перегляду поля зв'язку за замовчуванням відображаються за допомогою **компонента поля заголовка**. Він ідентифікує поточний пов'язаний запис, показуючи значення поля заголовка.
Система автоматично підбирає та відображає відповідний компонент поля заголовка залежно від його типу.
