# Дата та час

## Вступ

Поле "Дата" підтримує власні формати дати та часу.

## Де можна налаштувати формат дати

### Поля колекції

![20240417114311](https://static-docs.nocobase.com/20240417114311.png)

> Формат дати, встановлений у полі колекції, буде використовуватися як формат за замовчуванням для цього поля. Його можна змінити за допомогою налаштувань формату дати в конфігурації блоку.

### Налаштування поля

![20251027191824](https://static-docs.nocobase.com/20251027191824.png)

Окрім різноманітних попередньо встановлених форматів дати та часу, також підтримуються власні формати для задоволення індивідуальних потреб.
