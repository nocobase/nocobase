# Файловий менеджер

## Вступ

Файловий менеджер — це компонент поля зв'язку, призначений для роботи зі зв'язками, де цільовою колекцією є колекція файлів.

![20251027194124](https://static-docs.nocobase.com/20251027194124.png)

## Конфігурація поля

- [Компонент поля](/interface-builder/fields/association-field): Переключіться на інші компоненти поля зв'язку, такі як спадний список, селектор даних тощо.
- Дозволити вибір існуючих файлів: Увімкнено за замовчуванням, дозволяє вибирати та зв'язувати файли з уже завантажених.
