# Поле
В інтерфейсі користувача (UI) поле — це компонент поля, який є носієм даних. Різні типи даних відображаються за допомогою різних компонентів поля. Поля можуть бути прикріплені лише до блоків і не можуть використовуватися самостійно.

## Поля в блоці
Поля зазвичай не використовуються окремо, а існують як дочірні елементи блоків типу даних. Блоки типу даних зазвичай мають опцію «Налаштувати поля», а список полів надається поточною колекцією.

![20251029204240](https://static-docs.nocobase.com/20251029204240.png)

## Дизайнер полів (Панель інструментів)
Як і блоки, компонент поля також має три іконки у верхньому правому куті, а саме:

- Перетягування для компонування
- [Потік подій](/interface-builder/event-flow)
- Налаштування

![20251025165432](https://static-docs.nocobase.com/20251025165432.png)

## Компонування полів
Ви можете налаштувати компонування полів у блоці за допомогою перетягування.

![20251025165605](https://static-docs.nocobase.com/20251025165605.gif)

## Опції налаштування полів
Окрім загальних опцій налаштування полів, різні типи полів підтримують специфічні опції налаштування.
