# Компонент поля

## Вступ

Компоненти поля використовуються для відображення вмісту поля.
Деякі поля мають кілька вбудованих компонентів відображення. Користувачі можуть вибрати відповідний компонент відповідно до своїх фактичних потреб, щоб представити значення поля різними способами.

Приклад: У режимі перегляду поле URL може бути переключено на компонент попереднього перегляду або компонент URL (за замовчуванням).
