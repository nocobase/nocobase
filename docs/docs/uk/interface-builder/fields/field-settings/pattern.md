# Режим відображення

## Вступ

На відміну від блоків, компоненти полів мають три режими відображення (підтримується лише для полів у формі). При перемиканні між різними режимами відображення доступні різні параметри конфігурації поля.

- Редагований;
- Лише для читання (нередагований);
- Лише для читання (режим перегляду);

### Режими відображення для полів зв'язків

**Режим відображення поля зв'язку** визначає, як поле відображається в інтерфейсі, а також які типи компонентів поля доступні.

У **редагованому стані** поля зв'язків підтримують кілька типів компонентів, що дозволяє користувачам вибирати різні компоненти поля зв'язків для відображення або вибору пов'язаних даних відповідно до бізнес-потреб.

У цьому стані користувачі можуть гнучко вибирати відповідний метод відображення для більш ефективної обробки даних.

#### Компоненти полів зв'язків у стані лише для читання

При перемиканні в **стан лише для читання** система автоматично за замовчуванням використовує **компонент поля заголовка** для відображення пов'язаних даних. Це підходить для сценаріїв, коли вам потрібно лише переглядати дані без внесення змін.
