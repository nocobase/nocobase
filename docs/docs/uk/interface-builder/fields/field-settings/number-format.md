# Форматування чисел

## Вступ

Форматування чисел – це візуальне покращення та стандартизація числових полів (включно з полями формул) у режимі перегляду. Воно охоплює такі налаштування, як роздільники тисяч, точність десяткових знаків, перетворення одиниць вимірювання та наукова нотація. Налаштувавши форматування чисел, ви можете адаптувати відображення числових значень до звичок різних регіонів та галузей.

## Як використовувати

- Підтримує просте перетворення одиниць вимірювання (множення/ділення)
- Роздільник тисяч
- Префікс і суфікс
- Точність
- Наукова нотація.

Приклад: Відформатуйте суму замовлення, додавши символ валюти (наприклад, знак юаня ￥), роздільник тисяч (наприклад, кома або крапка) та точність десяткових знаків, щоб сума була чіткою та легко читалася.
