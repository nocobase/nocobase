# Відображення міток

## Вступ

Поля в блоці за замовчуванням відображають мітки, які вказують на значення кожного поля, що робить читання та розуміння даних зручнішим для користувачів.

У деяких сценаріях (наприклад, на сторінках лише для відображення або коли значення полів вже дуже зрозумілі) ви можете приховати мітки, щоб зробити інтерфейс більш лаконічним. Після приховування міток відображаються лише значення полів, а назви полів більше не показуються.
