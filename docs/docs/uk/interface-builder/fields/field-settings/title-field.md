# Поле заголовка

## Вступ

**Поле заголовка** зазвичай використовується в компонентах полів зв'язку як ідентифікатор для пов'язаних даних, що відображаються в інтерфейсі. Воно визначає, як подається основна інформація про дані, зазвичай допомагаючи користувачам швидко ідентифікувати та розуміти дані.

Різні типи полів заголовка відповідають різним компонентам полів і підтримують відповідні параметри конфігурації. Наприклад:

- Коли поле заголовка є **полем дати**, компонент поля використовує **компонент дати** для відображення даних.
- Коли поле заголовка є **текстовим полем**, компонент поля використовує **текстовий компонент** для відображення даних.
- Коли поле заголовка є **полем вибору**, компонент поля використовує **компонент вибору** для відображення даних.
