# Поля форми

## Вступ

Різні типи полів у формі мають різні параметри конфігурації. Це дозволяє користувачам розширювати ці параметри за допомогою плагінів, щоб задовольнити більше індивідуальних потреб.

![20251025174555](https://static-docs.nocobase.com/20251025174555.png)
