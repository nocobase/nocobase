# Поля таблиці

## Вступ

Поля таблиці дозволяють не лише налаштовувати ширину стовпців, заголовки полів та сортування, але й пропонують розширені можливості персоналізації відображення для певних типів полів, таких як поля дати, поля зв'язків та числові поля.

![20251024174558](https://static-docs.nocobase.com/20251024174558.png)

## Налаштування полів

### Форматування поля дати

![20251024175303](https://static-docs.nocobase.com/20251024175303.png)

Докладніше дивіться у розділі [Форматування дати](/interface-builder/fields/specific/date-picker)

### Форматування числового поля

Підтримує просте перетворення одиниць виміру, роздільники тисяч, префікси та суфікси, точність та наукову нотацію.

![20251024175445](https://static-docs.nocobase.com/20251024175445.png)

Докладніше дивіться у розділі [Форматування чисел](/interface-builder/fields/field-settings/number-format)

### Увімкнути швидке редагування

Якщо увімкнено швидке редагування, при наведенні курсора миші на стовпець з'явиться кнопка редагування. Натисніть її, щоб швидко відредагувати та зберегти дані у спливаючому вікні.

![20251025171158](https://static-docs.nocobase.com/20251025171158.gif)

### Увімкнути "Натисніть, щоб відкрити"

Окрім полів зв'язків, які підтримують відкриття спливаючого вікна за кліком, звичайні поля також можуть бути налаштовані на "Натисніть, щоб відкрити" як точку входу для відкриття спливаючого вікна.

![20251025172308](https://static-docs.nocobase.com/20251025172308.gif)

### Режим відображення переповнення вмісту

Якщо вміст стовпця перевищує його ширину, ви можете налаштувати режим відображення переповнення:

-   Багатокрапка (за замовчуванням)
-   Перенесення рядка

![20251025172549](https://static-docs.nocobase.com/20251025172549.png)

### Компонент поля

Деякі поля підтримують кілька форматів відображення. Користувачі можуть перемикати компоненти полів, щоб досягти різних візуальних ефектів та задовольнити потреби різних сценаріїв. Наприклад, поле типу **URL** можна переключити на компонент **Preview**, щоб краще відображати вміст посилання або попередній перегляд зображення.

![20251025171658](https://static-docs.nocobase.com/20251025171658.png)

### Стиль відображення

-   Мітка
-   Текст

### Сортування

Наразі більшість типів полів підтримують сортування на стороні сервера. Якщо сортування увімкнено, дані можна сортувати за цільовим полем у порядку зростання або спадання.
