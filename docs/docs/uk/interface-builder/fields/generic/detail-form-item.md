# Поля деталей

## Вступ

Налаштування полів для блоків деталей, списків та сіток здебільшого ідентичні. Вони переважно контролюють відображення полів у режимі лише для читання.

![20251025172851](https://static-docs.nocobase.com/20251025172851.png)

## Параметри конфігурації полів

### Форматування поля дати

![20251025173005](https://static-docs.nocobase.com/20251025173005.png)

Докладніше дивіться у розділі [Форматування дати](/interface-builder/fields/specific/date-picker)

### Форматування числового поля

![20251025173242](https://static-docs.nocobase.com/20251025173242.png)

Підтримує просте перетворення одиниць виміру, роздільники тисяч, префікси та суфікси, точність та наукову нотацію.

Докладніше дивіться у розділі [Форматування чисел](/interface-builder/fields/field-settings/number-format)

### Увімкнути відкриття за кліком

Окрім полів зв'язків, які підтримують відкриття спливаючих вікон за кліком, звичайні поля також можна налаштувати для відкриття вікна за кліком. Ви також можете визначити спосіб відкриття спливаючого вікна (як бічну панель, діалогове вікно або підсторінку).

![20251025173549](https://static-docs.nocobase.com/20251025173549.gif)

### Спосіб відображення при переповненні вмісту

Якщо вміст поля перевищує його ширину, ви можете налаштувати спосіб відображення переповнення.

- Скорочене відображення (за замовчуванням)
- Перенос рядка

![20251025173917](https://static-docs.nocobase.com/20251025173917.png)

### Компонент поля

Деякі поля підтримують різні формати відображення, що можна реалізувати, перемикаючи компонент поля.

Наприклад: компонент `URL` можна перемкнути на компонент `Preview`.

![20251025174042](https://static-docs.nocobase.com/20251025174042.png)

Наприклад, поля зв'язків можна відображати по-різному, перемикаючись з компонента поля заголовка на `Піддеталі`, щоб показати більше вмісту з цього зв'язку.

![20251025174311](https://static-docs.nocobase.com/20251025174311.gif)
