# Редагувати

## Вступ

Дія "Редагувати" – це кнопка рядка, яка використовує поточний запис як контекст. Вона відображається у вигляді спливаючого вікна, де ви можете налаштовувати та конфігурувати блоки.

![20251029165442](https://static-docs.nocobase.com/20251029165442.png)

Налаштовуйте блоки за потреби.
