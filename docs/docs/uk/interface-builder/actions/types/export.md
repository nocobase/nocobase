# Експорт

## Вступ

Функція експорту дозволяє експортувати відфільтровані записи у форматі **Excel**, а також підтримує налаштування полів для експорту. Користувачі можуть обирати потрібні поля для подальшого аналізу, обробки або архівування даних. Ця функція підвищує гнучкість роботи з даними, особливо в сценаріях, коли дані потрібно перенести на інші платформи або додатково обробити.

### Основні можливості:
- **Вибір полів**: Користувачі можуть налаштовувати та обирати поля для експорту, забезпечуючи точність та лаконічність експортованих даних.
- **Підтримка формату Excel**: Експортовані дані будуть збережені як стандартний файл Excel, що полегшує інтеграцію та аналіз з іншими даними.

Завдяки цій функції користувачі можуть легко експортувати ключові дані зі своєї роботи для зовнішнього використання, підвищуючи ефективність роботи.

### Поля для експорту

- Перший рівень: Усі поля поточної колекції;
- Другий рівень: Якщо це поле зв'язку, потрібно обрати поля з пов'язаної колекції;
- Третій рівень: Обробляються лише три рівні; поля зв'язку на останньому рівні не відображаються;
