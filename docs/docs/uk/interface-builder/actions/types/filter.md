# Фільтр

## Вступ

Операція фільтрації зазвичай використовується в блоках даних для фільтрації інформації за допомогою різних умов. Важливо пам'ятати, що якщо для блоку вже налаштовано діапазон даних, умови фільтрації будуть об'єднані з умовами цього діапазону, утворюючи кінцевий критерій фільтрації.

### Поля для фільтрації

За замовчуванням, усі поля цієї колекції увімкнені. Лише увімкнені поля можна вибрати у списку полів умов.

Поля з пов'язаних колекцій також можуть бути використані як поля для фільтрації.
