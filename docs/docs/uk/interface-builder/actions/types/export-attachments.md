# Експорт вкладень

## Вступ

Експорт вкладень підтримує експорт полів, пов'язаних з вкладеннями, у форматі стисненого архіву.

#### Конфігурація експорту вкладень

![20251029173251](https://static-docs.nocobase.com/20251029173251.png)

![20251029173425](https://static-docs.nocobase.com/20251029173425.png)

![20251029173345](https://static-docs.nocobase.com/20251029173345.png)

- Налаштуйте поля вкладень для експорту; підтримується множинний вибір.
- Ви можете вибрати, чи створювати папку для кожного запису.

Правила іменування файлів:

- Якщо ви вирішите створити папку для кожного запису, правило іменування файлів таке: `{значення поля заголовка запису}/{назва поля вкладення}[-{порядковий номер файлу}].{розширення файлу}`.
- Якщо ви вирішите не створювати папку, правило іменування файлів таке: `{значення поля заголовка запису}-{назва поля вкладення}[-{порядковий номер файлу}].{розширення файлу}`.

Порядковий номер файлу генерується автоматично, якщо поле вкладення містить кілька вкладень.

- [Правило зв'язування](/interface-builder/actions/action-settings/linkage-rule): Динамічне відображення/приховування кнопки;
- [Редагувати кнопку](/interface-builder/actions/action-settings/edit-button): Редагування заголовка, типу та іконки кнопки;
