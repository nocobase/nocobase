# Масове редагування

## Вступ

Масове редагування підходить для сценаріїв, що потребують гнучкого групового оновлення даних. Після натискання кнопки «Масове редагування» ви можете налаштувати форму масового редагування у спливаючому вікні та встановити різні стратегії оновлення для кожного поля.

## Посібник користувача

### Конфігурація форми масового редагування

1. Додайте кнопку «Масове редагування».

2. Встановіть область масового редагування: Вибрані / Усі (за замовчуванням — Вибрані).

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_14_AM.png)

3. Додайте форму масового редагування.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_14_AM.png)

4. Налаштуйте поля, які потрібно редагувати, і додайте кнопку підтвердження.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM%20(1).png)

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM.png)

### Надсилання форми

1. Виберіть рядки даних, які потрібно відредагувати.

2. Виберіть режим редагування для полів і введіть значення для надсилання.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_10_33_AM.png)

:::info{title=Доступні режими редагування}
* **Не оновлювати**: поле залишається без змін.
* **Змінити на**: оновити поле до вказаного значення.
* **Очистити**: видалити дані з поля.

:::

3. Надішліть форму.
