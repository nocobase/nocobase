# Відкрити модальне вікно

## Вступ

Дія **Відкрити модальне вікно** схожа на дії "Перегляд" та "Редагування". Ви можете налаштувати кнопку **Відкрити модальне вікно** у рядках або заголовку таблиці. Після натискання кнопки відкриється відповідне модальне вікно, а блоки в ньому відображатимуть дані на основі поточного запису контексту.

![20251029182351](https://static-docs.nocobase.com/20251029182351.png)

Налаштуйте блоки в модальному вікні за потреби.

![20251029190208](https://static-docs.nocobase.com/20251029190208.png)

## Налаштування дії

![20251029190352](https://static-docs.nocobase.com/20251029190352.png)

- [Редагувати кнопку](/interface-builder/actions/action-settings/edit-button): Змініть заголовок, тип та іконку кнопки.
- [Правило зв'язування](/interface-builder/linkage-rule): Динамічно керуйте станом кнопки (видима/вимкнена).
- [Редагувати модальне вікно](/interface-builder/actions/action-settings/edit-popup): Налаштуйте спосіб відкриття (модальне вікно або висувна панель) та розмір.
