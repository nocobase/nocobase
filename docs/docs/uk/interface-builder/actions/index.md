# Дія

В інтерфейсі користувача дія — це кнопка, яка використовується для запуску набору інструкцій. Її можна розмістити безпосередньо на сторінці, у діалоговому вікні або в бічній панелі, а також використовувати в поєднанні з блоками. Наразі дії, які можна динамічно налаштовувати, переважно використовуються в поєднанні з блоками.
