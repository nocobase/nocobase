# Підтвердження

## Вступ

Налаштування підтвердження дозволяє перевірити, чи дійсно ви хочете виконати певну дію. При натисканні на дію з'являється вікно підтвердження. Для кнопки видалення підтвердження увімкнено за замовчуванням, щоб запобігти випадковій втраті даних або незворотним наслідкам.

![20251029125021](https://static-docs.nocobase.com/20251029125021.png)

Підтримує налаштування вмісту та заголовка підтвердження, а також доступ до змінних.
