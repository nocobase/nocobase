# Редагування кнопки

## Вступ

Система дозволяє налаштовувати назву, іконку, колір фону та тип кнопки, пропонуючи гнучкі можливості для персоналізації інтерфейсу. Це допомагає користувачам адаптувати стиль відображення та функціональність кнопки відповідно до їхніх конкретних потреб.

- **Назва кнопки**: Ви можете налаштувати текст кнопки відповідно до бізнес-потреб, щоб напис на кнопці був чітким та інтуїтивно зрозумілим.
- **Іконка кнопки**: Підтримується встановлення іконки для кнопки, що підвищує візуальну впізнаваність та дозволяє користувачам швидко ідентифікувати її функцію.
- **Небезпечна дія**: Коли кнопка представляє **небезпечну дію** (наприклад, видалення, скидання тощо), її можна встановити як тип **Danger Action**.
- **Тип кнопки (Button Type)**: Кнопки підтримують різні налаштування типів. Поширені типи кнопок включають:
  - **За замовчуванням (Default)**: Стандартна кнопка, що підходить для звичайних операцій.
  - **Основний (Primary)**: Використовується для важливих або основних операцій, зазвичай має помітний візуальний ефект.
  - **Пунктирний (Dashed)**: Кнопка з пунктирною рамкою, що підходить для другорядних операцій або кнопок-підказок.
  - **Текстовий (Text)**: Кнопка, що містить лише текст без фонового кольору, зазвичай використовується для легких операцій або посилань.
