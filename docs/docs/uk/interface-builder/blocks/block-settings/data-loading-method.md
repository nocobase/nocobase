# Режим завантаження даних

## Вступ

Типовим режимом завантаження даних є «Завантажувати всі дані, якщо умови фільтрації порожні». Тут під «умовами фільтрації» маються на увазі значення у «Блоці фільтрації». Іншим варіантом є «Не завантажувати дані, якщо умови фільтрації порожні».
