# Налаштування області даних

## Вступ

Налаштування області даних означає визначення умов фільтрації за замовчуванням для блоку даних. Користувачі можуть гнучко змінювати область даних відповідно до своїх бізнес-потреб, але незалежно від виконаних операцій фільтрації, система автоматично застосовуватиме ці умови фільтрації за замовчуванням, гарантуючи, що дані завжди відповідатимуть встановленим обмеженням.

## Посібник користувача

![20251027110053](https://static-docs.nocobase.com/20251027110053.png)

Поля фільтрації підтримують вибір полів з поточної колекції та пов'язаних колекцій.

![20251027110140](https://static-docs.nocobase.com/20251027110140.png)

### Оператори

Різні типи полів підтримують різні оператори. Наприклад, текстові поля підтримують оператори "дорівнює", "не дорівнює", "містить" тощо; числові поля підтримують оператори "більше", "менше" тощо; а поля дати підтримують оператори "в межах", "до певної дати" тощо.

![20251027111124](https://static-docs.nocobase.com/20251027111124.png)
