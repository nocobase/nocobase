# Сортування перетягуванням

## Вступ

Сортування перетягуванням базується на полі сортування та використовується для ручного впорядкування записів у блоці.


:::info{title=Порада}
* Якщо одне й те саме поле сортування використовується для сортування перетягуванням у кількох блоках одночасно, це може порушити існуючий порядок.
* Під час сортування перетягуванням у таблиці для поля сортування не можна встановлювати правила групування.
* Деревоподібні таблиці підтримують сортування вузлів лише в межах одного рівня.

:::


## Конфігурація

Додайте поле типу «Сортування». Поля сортування більше не створюються автоматично під час створення колекції; їх потрібно створювати вручну.

## Пояснення правил сортування

Припустимо, поточний порядок такий:

```
[1,2,3,4,5,6,7,8,9]
```

Коли певний елемент (наприклад, 5) переміщується вперед на позицію 3, зміняться значення сортування лише для 3, 4 та 5: 5 займає позицію 3, а 3 та 4 зміщуються на одну позицію назад.

```
[1,2,5,3,4,6,7,8,9]
```

Якщо після цього перемістити 6 назад на позицію 8, то 6 займе позицію 8, а 7 та 8 змістяться на одну позицію вперед.

```
[1,2,5,3,4,7,8,6,9]
```
