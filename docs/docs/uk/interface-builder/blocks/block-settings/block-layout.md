# Макет

## Вступ

Блоки типу форми, такі як блоки форми, блоки деталей та блоки списку, підтримують налаштування макета. Варіанти макета поділяються на вертикальний макет (де мітки та поля вводу розташовуються рядок за рядком) та горизонтальний макет (де мітки та поля вводу розміщуються поруч). Ви можете налаштовувати макет полів блоку відповідно до своїх конкретних потреб, щоб введення або перегляд інформації був чітким та ефективним.

### Вертикальний (за замовчуванням)

Макетом за замовчуванням є вертикальний.

![20251024120143](https://static-docs.nocobase.com/20251024120143.png)

### Горизонтальний

Горизонтальний макет відображає мітки та поля вводу поруч, що ефективно економить простір. Для полів з довгими заголовками ви можете обрати перенесення тексту або відображення багатокрапки, щоб забезпечити охайний та естетичний вигляд інтерфейсу.

![20251024120237](https://static-docs.nocobase.com/20251024120237.png)

#### Вирівнювання заголовка поля

- **Вирівнювання по лівому краю** (за замовчуванням)
- **Вирівнювання по правому краю**

#### Ширина заголовка поля

- Ширина за замовчуванням становить **120px**, мінімальна ширина — **50px**.

#### Обробка заголовків полів, що перевищують ширину

- **Перенесення тексту**
- **Багатокрапка**

#### Показувати двокрапку

Ви можете обрати, чи відображати двокрапку після заголовка поля.
