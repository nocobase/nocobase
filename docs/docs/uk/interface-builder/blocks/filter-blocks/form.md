# Форма фільтрації

## Вступ

Форма фільтрації дозволяє користувачам фільтрувати дані, заповнюючи поля форми. Її можна використовувати для фільтрації блоків таблиць, блоків діаграм, блоків списків тощо.

## Як використовувати

Давайте розглянемо простий приклад, щоб швидко зрозуміти, як використовувати форму фільтрації. Припустімо, у нас є блок таблиці з інформацією про користувачів, і ми хочемо фільтрувати ці дані за допомогою форми фільтрації, як показано нижче:

![20251031163036_rec_](https://static-docs.nocobase.com/20251031163036_rec_.gif)

Кроки налаштування:

1. Увімкніть режим редагування та додайте на сторінку блок "Форма фільтрації" та блок "Таблиця".
![20251031163525_rec_](https://static-docs.nocobase.com/20251031163525_rec_.gif)
2. Додайте поле "Псевдонім" як до блоку таблиці, так і до блоку форми фільтрації.
![20251031163932_rec_](https://static-docs.nocobase.com/20251031163932_rec_.gif)
3. Тепер ви можете почати використовувати її.
![20251031163036_rec_](https://static-docs.nocobase.com/20251031163036_rec_.gif)

## Розширене використання

Блок форми фільтрації підтримує більше розширених налаштувань. Нижче наведено деякі поширені випадки використання.

### Зв'язування кількох блоків

Одне поле форми може одночасно фільтрувати дані в кількох блоках. Ось як це зробити:

1. Натисніть опцію налаштування "Connect fields" для поля.
![20251031170300](https://static-docs.nocobase.com/20251031170300.png)
2. Додайте цільові блоки, які ви хочете зв'язати. У цьому прикладі ми виберемо блок списку на сторінці.
![20251031170718](https://static-docs.nocobase.com/20251031170718.png)
3. Виберіть одне або кілька полів із блоку списку для зв'язування. Тут ми вибираємо поле "Псевдонім".
![20251031171039](https://static-docs.nocobase.com/20251031171039.png)
4. Натисніть кнопку збереження, щоб завершити налаштування. Результат виглядає так:
![20251031171209_rec_](https://static-docs.nocobase.com/20251031171209_rec_.gif)

### Користувацькі поля

Окрім вибору полів із колекцій, ви також можете створювати поля форми за допомогою "Користувацьких полів". Наприклад, можна створити поле випадаючого списку з власними опціями. Ось як це зробити:

1. Натисніть опцію "Користувацькі поля", щоб відкрити панель налаштувань.
![20251031173833](https://static-docs.nocobase.com/20251031173833.png)
2. Заповніть заголовок поля, виберіть "Select" як модель поля та налаштуйте опції.
![20251031174857_rec_](https://static-docs.nocobase.com/20251031174857_rec_.gif)
3. Новостворені користувацькі поля потрібно вручну зв'язати з полями в цільових блоках. Ось як це зробити:
![20251031181957_rec_](https://static-docs.nocobase.com/20251031181957_rec_.gif)
4. Налаштування завершено. Результат виглядає так:
![20251031182235_rec_](https://static-docs.nocobase.com/20251031182235_rec_.gif)

Наразі підтримуються такі моделі полів:

- Input: Однорядкове текстове поле
- Number: Поле для введення чисел
- Date: Вибір дати
- Select: Випадаючий список (можна налаштувати для одиночного або множинного вибору)
- Radio group: Група радіокнопок
- Checkbox group: Група прапорців

### Згортання

Додайте кнопку згортання, щоб згортати та розгортати вміст форми фільтрації, заощаджуючи місце на сторінці.

![20251031182743](https://static-docs.nocobase.com/20251031182743.png)

Підтримуються такі налаштування:

![20251031182849](https://static-docs.nocobase.com/20251031182849.png)

- **Кількість рядків у згорнутому стані**: Встановлює, скільки рядків полів форми відображається у згорнутому стані.
- **Згортати за замовчуванням**: Якщо увімкнено, форма фільтрації за замовчуванням відображається у згорнутому стані.
