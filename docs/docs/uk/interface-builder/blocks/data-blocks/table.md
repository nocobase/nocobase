# Табличний блок

## Вступ

Табличний блок є одним з основних вбудованих блоків даних у **NocoBase**, призначений для відображення та керування структурованими даними у табличному форматі. Він пропонує гнучкі параметри конфігурації, завдяки яким користувачі можуть налаштовувати стовпці таблиці, їх ширину, правила сортування та діапазон даних відповідно до своїх потреб, щоб забезпечити відповідність відображуваних даних конкретним бізнес-вимогам.

#### Основні функції:
- **Гнучке налаштування стовпців**: Ви можете налаштовувати стовпці таблиці та їх ширину відповідно до різних вимог відображення даних.
- **Правила сортування**: Підтримує сортування даних у таблиці. Користувачі можуть впорядковувати дані за різними полями за зростанням або спаданням.
- **Налаштування діапазону даних**: Встановивши діапазон даних, ви можете контролювати обсяг відображуваних даних, уникаючи втручання нерелевантних даних.
- **Налаштування операцій**: Табличний блок має вбудовані різні опції операцій. Користувачі можуть легко налаштовувати такі операції, як фільтрація, створення, редагування, видалення тощо, для швидкого керування даними.
- **Швидке редагування**: Підтримує пряме редагування даних у таблиці, спрощуючи робочий процес та підвищуючи ефективність.

## Налаштування блоку

![20251023150819](https://static-docs.nocobase.com/20251023150819.png)

### Правила зв'язування блоків

Керуйте поведінкою блоку (наприклад, чи відображати його або виконувати JavaScript) за допомогою правил зв'язування.

![20251023194550](https://static-docs.nocobase.com/20251023194550.png)

Більше деталей дивіться у [Правилах зв'язування](/interface-builder/linkage-rule)

### Встановлення діапазону даних

Приклад: За замовчуванням фільтрувати замовлення зі «Статусом» «Оплачено».

![20251023150936](https://static-docs.nocobase.com/20251023150936.png)

Більше деталей дивіться у [Встановленні діапазону даних](/interface-builder/blocks/block-settings/data-scope)

### Встановлення правил сортування

Приклад: Відображати замовлення за датою у зворотному порядку.

![20251023155114](https://static-docs.nocobase.com/20251023155114.png)

Більше деталей дивіться у [Встановленні правил сортування](/interface-builder/blocks/block-settings/sorting-rule)

### Увімкнення швидкого редагування

Активуйте «Увімкнути швидке редагування» в налаштуваннях блоку та налаштуваннях стовпців таблиці, щоб налаштувати, які стовпці можна швидко редагувати.

![20251023190149](https://static-docs.nocobase.com/20251023190149.png)

![20251023190519](https://static-docs.nocobase.com/20251023190519.gif)

### Увімкнення таблиці-дерева

Коли таблиця даних є ієрархічною (деревовидною) таблицею, табличний блок може увімкнути функцію «Увімкнути таблицю-дерево». За замовчуванням ця опція вимкнена. Після увімкнення блок відображатиме дані у деревовидній структурі та підтримуватиме відповідні параметри конфігурації та операції.

![20251125205918](https://static-docs.nocobase.com/20251125205918.png)

### Розгортати всі рядки за замовчуванням

Коли таблиця-дерево увімкнена, блок підтримує розгортання всіх дочірніх даних за замовчуванням під час завантаження.
