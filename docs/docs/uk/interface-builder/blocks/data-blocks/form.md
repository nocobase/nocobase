# Блок форми

## Вступ

Блок форми — це важливий блок для створення інтерфейсів введення та редагування даних. Він має високий ступінь налаштовуваності, використовуючи відповідні компоненти для відображення потрібних полів на основі моделі даних. Завдяки потокам подій, таким як правила зв'язування, блок форми може динамічно відображати поля. Крім того, його можна поєднувати з **робочими процесами** для запуску автоматизованих процесів та обробки даних, що додатково підвищує ефективність роботи або дозволяє реалізувати логічну оркестрацію.

## Додавання блоку форми

- **Редагувати форму**: Використовується для зміни наявних даних.
- **Додати форму**: Використовується для створення нових записів даних.

### Макет

Блок форми підтримує два режими макета, які можна встановити за допомогою атрибута `layout`:

- **horizontal** (горизонтальний макет): Цей макет відображає мітку та вміст в одному рядку, заощаджуючи вертикальний простір. Він підходить для простих форм або випадків з невеликою кількістю інформації.
- **vertical** (вертикальний макет) (за замовчуванням): Мітка розташована над полем. Цей макет робить форму легшою для читання та заповнення, особливо для форм з кількома полями або складними елементами введення.
