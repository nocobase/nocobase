# Діаграма

## Вступ

Плагін візуалізації даних NocoBase надає можливості візуального запиту даних та багатий набір компонентів діаграм. Користувачі можуть швидко створювати візуалізаційні панелі (дашборди) за допомогою простої конфігурації, щоб відображати аналітичні висновки та підтримувати багатовимірний аналіз і представлення даних.
