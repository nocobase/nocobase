# Сіткові картки

## Вступ

Блок сіткових карток відображає зведену інформацію про записи даних у форматі карток. Він дозволяє налаштовувати кількість стовпців для різних розмірів екрана, щоб забезпечити зручне відображення на різних пристроях.
