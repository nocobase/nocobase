# Блок

Блоки є носіями даних та вмісту. Їх можна розміщувати на сторінці (Page), у модальному вікні (Modal) або на висувній панелі (Drawer), а кілька блоків можна вільно перетягувати та впорядковувати.

## Типи блоків

![Типи блоків](https://static-docs.nocobase.com/f71af45b5cd914ea0558f760ddbbba58.png)

- **Блок даних**: Використовується для відображення даних з джерела даних на інтерфейсі.
- **Блок фільтрації**: Використовує дані з джерела даних як умови фільтрації для фільтрації інших блоків даних.
- **Інші блоки**: Містить специфічний або незалежний вміст, такий як завдання робочого процесу, журнали аудиту, Markdown тощо.

## Додавання блоків

Блоки можна розміщувати на сторінці (Page), у модальному вікні (Modal) або на висувній панелі (Drawer).

### Блоки на сторінці

Наразі типи блоків на сторінці включають: блок даних, блок фільтрації та інші блоки.

![20251023222441](https://static-docs.nocobase.com/20251023222441.png)

### Блоки у спливаючих вікнах (модальні вікна або висувні панелі)

Спливаючі вікна бувають двох типів: модальні вікна та висувні панелі. Як і на сторінках, до них також можна додавати блоки. Відмінність полягає в тому, що блокові форми у спливаючих вікнах зазвичай використовуються для додавання, редагування або перегляду одного запису. Типи блоків включають блок даних та інші блоки.

![20251023222613](https://static-docs.nocobase.com/20251023222613.png)

## Конструктор блоків

Кожен блок має три невеликі іконки у верхньому правому куті. Зліва направо це:

1. Перетягування для зміни макета
2. [Потік подій](/interface-builder/event-flow)
3. Налаштування блоку

![20251023224032](https://static-docs.nocobase.com/20251023224032.png)

Для простих блоків усі параметри конфігурації зосереджені в розділі «Налаштування блоку», наприклад, JS Block.

![20251023224903](https://static-docs.nocobase.com/20251023224903.png)

Складні блоки даних також надають окремі вбудовані «Налаштування полів» та «Налаштування дій».

![20251023225141](https://static-docs.nocobase.com/20251023225141.png)

Крім того, ви можете вільно експериментувати та надавати більше можливостей для вкладеності, як, наприклад, у блоці діаграм.

![](https://static-docs.nocobase.com/07588190b3f41ae3060e71d8b76b4447.png)
