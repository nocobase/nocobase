# Конструктор інтерфейсу

## Що бачите, те й отримуєте (WYSIWYG)

NocoBase пропонує режим побудови інтерфейсу за принципом «Що бачите, те й отримуєте» (WYSIWYG). Натисніть кнопку «Конструктор інтерфейсу», щоб перемикатися між режимами редагування та перегляду.

## Шаблон макета

NocoBase має вбудований шаблон макета: навігація розташована зверху та ліворуч, а область вмісту — праворуч.

![未命名.002](https://static-docs.nocobase.com/未命名.002.jpeg)

## Параметри конфігурації

Увійшовши в режим конструктора інтерфейсу, ви побачите помаранчеві маркери, що вказують на елементи, які можна налаштувати. Зазвичай, точка входу для налаштувань елемента знаходиться у його верхньому правому куті. Майже всі елементи можна налаштовувати в реальному часі безпосередньо в інтерфейсі та одразу бачити результат. Ось кілька прикладів:
