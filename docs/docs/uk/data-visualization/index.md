# Огляд

Плагін візуалізації даних NocoBase пропонує візуальні запити даних та широкий набір компонентів діаграм. Завдяки простій конфігурації ви можете швидко створювати інформаційні панелі, представляти аналітичні дані та підтримувати багатовимірний аналіз і відображення.

![clipboard-image-1761749573](https://static-docs.nocobase.com/clipboard-image-1761749573.png)

## Основні поняття
- Блок діаграми: Конфігурований компонент діаграми на сторінці, який підтримує запити даних, параметри діаграм та інтерактивні події.
- Запит даних (Builder / SQL): Налаштовуйте візуально за допомогою Builder або пишіть SQL для отримання даних.
- Показники (Measures) та Виміри (Dimensions): Показники використовуються для числової агрегації; Виміри групують дані (наприклад, дата, категорія, регіон).
- Мапування полів: Зіставляйте стовпці результатів запиту з основними полями діаграми, такими як `xField`, `yField`, `seriesField` або `Category / Value`.
- Параметри діаграми (Basic / Custom): Basic налаштовує загальні властивості візуально; Custom повертає повний об'єкт `option` ECharts за допомогою JS.
- Виконання запиту: Виконайте запит і отримайте дані на панелі конфігурації; перемикайтеся між Table / JSON для перегляду отриманих даних.
- Попередній перегляд та Збереження: Попередній перегляд є тимчасовим; натискання кнопки «Зберегти» записує конфігурацію в базу даних і застосовує її.
- Контекстні змінні: Повторно використовуйте контекстну інформацію сторінки, користувача та фільтра (наприклад, `{{ ctx.user.id }}`) у запитах та конфігурації діаграм.
- Фільтри та зв'язування: Блоки фільтрів на рівні сторінки збирають уніфіковані умови, автоматично об'єднуються в запити діаграм та оновлюють пов'язані діаграми.
- Інтерактивні події: Реєструйте події за допомогою `chart.on`, щоб увімкнути виділення, навігацію та деталізацію (drill-down).

## Встановлення
Візуалізація даних є вбудованим плагіном NocoBase; вона працює одразу після встановлення, без необхідності окремої інсталяції.
