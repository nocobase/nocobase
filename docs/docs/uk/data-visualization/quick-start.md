# Швидкий старт

Давайте з нуля налаштуємо діаграму, використовуючи основні необхідні функції. Додаткові можливості будуть розглянуті в наступних розділах.

Попередні вимоги:
- Налаштовано **джерело даних** та **колекцію** (таблицю), і ви маєте права на читання.

## Додавання блоку діаграми

У конструкторі сторінок натисніть «Додати блок», виберіть «Діаграма» та додайте блок діаграми.

![clipboard-image-1761554593](https://static-docs.nocobase.com/clipboard-image-1761554593.png)

Після додавання натисніть «Налаштувати» у верхньому правому куті блоку.

![clipboard-image-1761554709](https://static-docs.nocobase.com/clipboard-image-1761554709.png)

Праворуч відкриється панель налаштувань діаграми. Вона містить три розділи: Запит даних, Опції діаграми та Події.

![clipboard-image-1761554848](https://static-docs.nocobase.com/clipboard-image-1761554848.png)

## Налаштування запиту даних

На панелі «Запит даних» можна налаштувати **джерело даних**, умови фільтрації запиту тощо.

- Спочатку виберіть **джерело даних** та **колекцію**
  - На панелі «Запит даних» виберіть **джерело даних** та **колекцію** як основу для запиту.
  - Якщо **колекція** недоступна для вибору або порожня, спершу перевірте, чи вона створена та чи має поточний користувач відповідні дозволи.

- Налаштування показників (Measures)
  - Виберіть одне або кілька числових полів як показники.
  - Для кожного показника встановіть агрегацію: Сума (Sum) / Кількість (Count) / Середнє (Avg) / Максимум (Max) / Мінімум (Min).

- Налаштування вимірів (Dimensions)
  - Виберіть одне або кілька полів як виміри для групування (дата, категорія, регіон тощо).
  - Для полів дати/часу можна встановити формат (наприклад, `YYYY-MM`, `YYYY-MM-DD`), щоб забезпечити єдине відображення.

![clipboard-image-1761555060](https://static-docs.nocobase.com/clipboard-image-1761555060.png)

Інші умови: фільтрація, сортування, пагінація є необов'язковими.

## Виконання запиту та перегляд даних

- Натисніть «Виконати запит», щоб отримати дані та відтворити попередній перегляд діаграми (прямо на лівій частині сторінки).
- Ви можете натиснути «Переглянути дані», щоб переглянути отримані результати даних; підтримується перемикання між форматами Таблиці та JSON. Натисніть ще раз, щоб згорнути попередній перегляд даних.
- Якщо результат даних порожній або не відповідає очікуванням, поверніться до панелі запитів і перевірте дозволи **колекції**, відповідність полів для показників/вимірів та типи даних.

![clipboard-image-1761555228](https://static-docs.nocobase.com/clipboard-image-1761555228.png)

## Налаштування опцій діаграми

На панелі «Опції діаграми» можна вибрати тип діаграми та налаштувати її параметри.

- Спочатку виберіть тип діаграми (лінійна/області, стовпчаста/гістограма, кругова/кільцева, точкова тощо).
- Завершіть основні відповідності полів:
  - Лінійна/області/стовпчаста/гістограма: `xField` (вимір), `yField` (показник), `seriesField` (серія, необов'язково)
  - Кругова/кільцева: `Category` (категоріальний вимір), `Value` (показник)
  - Точкова: `xField`, `yField` (два показники або виміри)
  - Для отримання додаткових налаштувань діаграм зверніться до документації Echarts: [Axis](https://echarts.apache.org/handbook/en/concepts/axis)
- Після натискання «Виконати запит» відповідності полів за замовчуванням заповнюються автоматично; якщо ви зміните виміри/показники, будь ласка, перевірте відповідності ще раз.

![clipboard-image-1761555586](https://static-docs.nocobase.com/clipboard-image-1761555586.png)

## Попередній перегляд та збереження

Зміни в налаштуваннях автоматично оновлюють попередній перегляд у реальному часі, і ви можете бачити діаграму на лівій частині сторінки. Однак, будь ласка, зверніть увагу, що всі зміни не будуть фактично збережені, доки ви не натиснете кнопку «Зберегти».

Ви також можете скористатися кнопками внизу:

- Попередній перегляд: Зміни в налаштуваннях автоматично оновлюють попередній перегляд у реальному часі, або ви можете натиснути кнопку «Попередній перегляд» внизу, щоб вручну запустити оновлення.
- Скасувати: Якщо ви не бажаєте зберігати поточні зміни в налаштуваннях, ви можете натиснути кнопку «Скасувати» внизу або оновити сторінку, щоб скасувати ці зміни та повернутися до останнього збереженого стану.
- Зберегти: Натисніть «Зберегти», щоб фактично зберегти всі поточні налаштування запиту та діаграми в базі даних; вони стануть чинними для всіх користувачів.

![clipboard-image-1761555803](https://static-docs.nocobase.com/clipboard-image-1761555803.png)

## Поширені поради

- Мінімальна робоча конфігурація: Виберіть **колекцію** + принаймні один показник; рекомендується додати виміри для зручнішого групованого відображення.
- Для вимірів дати рекомендується встановлювати відповідний формат (наприклад, `YYYY-MM` для місячної статистики), щоб уникнути розривів або безладу на осі X.
- Якщо запит порожній або діаграма не відображається:
  - Перевірте **колекцію**/дозволи та відповідності полів;
  - У розділі «Переглянути дані» переконайтеся, що назви стовпців та їхні типи відповідають відповідностям діаграми.
- Попередній перегляд є тимчасовим станом: Він використовується лише для перевірки та коригування. Офіційно зміни набудуть чинності лише після натискання «Зберегти».
