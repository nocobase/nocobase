# Колекція-дерево

## Вступ

Колекція-дерево — це шаблон проєктування колекції даних, який використовується для організації ієрархічних даних. Ця структура колекції нагадує дерево, де кожен елемент даних може мати один або кілька дочірніх елементів, а ці дочірні елементи, своєю чергою, можуть мати власних нащадків.
