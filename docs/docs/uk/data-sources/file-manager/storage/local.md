# Локальне сховище

Завантажені файли зберігатимуться в локальному каталозі на сервері. Це підходить для невеликих або експериментальних сценаріїв, де загальний обсяг файлів, якими керує система, є відносно невеликим.

### Шлях

Шлях позначає як відносний шлях до файлу, що зберігається на сервері, так і URL-шлях доступу. Наприклад, "`user/avatar`" (без початкового та кінцевого символів "`/`") означає:

1.  Відносний шлях до завантаженого файлу, що зберігається на сервері: `/path/to/nocobase-app/storage/uploads/user/avatar`.
2.  Префікс URL-адреси для доступу до файлу: `http://localhost:13000/storage/uploads/user/avatar`.
