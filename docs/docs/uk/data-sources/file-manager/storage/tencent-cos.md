# Tencent COS

Рушій зберігання даних, що базується на Tencent Cloud COS. Перед використанням необхідно підготувати відповідний обліковий запис та дозволи.

## Параметри конфігурації

![Приклад налаштування рушія зберігання Tencent COS](https://static-docs.nocobase.com/20240712222125.png)

:::info{title=Підказка}
Цей розділ охоплює лише специфічні параметри рушія зберігання Tencent Cloud COS. Загальні параметри дивіться у розділі [Загальні параметри рушія](./index.md#引擎通用参数).
:::

### Регіон

Вкажіть регіон зберігання COS, наприклад: `ap-chengdu`.

:::info{title=Підказка}
Ви можете переглянути інформацію про регіон сховища у [консолі Tencent Cloud COS](https://console.cloud.tencent.com/cos). Достатньо вказати лише префікс регіону (без повного доменного імені).
:::

### SecretId

Вкажіть ID авторизованого ключа доступу Tencent Cloud.

### SecretKey

Вкажіть Secret авторизованого ключа доступу Tencent Cloud.

### Кошик

Вкажіть назву кошика зберігання COS, наприклад: `qing-cdn-1234189398`.
