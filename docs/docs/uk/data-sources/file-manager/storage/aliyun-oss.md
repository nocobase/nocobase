# Aliyun OSS

Рушій зберігання даних, що базується на Aliyun OSS. Перед використанням необхідно підготувати відповідні облікові записи та дозволи.

## Налаштування

![Приклад налаштування Aliyun OSS](https://static-docs.nocobase.com/20240712220011.png)

:::info{title=Примітка}
Тут представлені лише специфічні параметри для рушія зберігання даних Aliyun OSS. Щодо загальних параметрів, будь ласка, зверніться до розділу [Загальні параметри рушія](./index.md#common-engine-parameters).
:::

### Регіон

Вкажіть регіон зберігання OSS, наприклад: `oss-cn-hangzhou`.

:::info{title=Примітка}
Ви можете переглянути інформацію про регіон сховища (bucket) в [консолі Aliyun OSS](https://oss.console.aliyun.com/). Вам потрібно використовувати лише префікс регіону (без повного доменного імені).
:::

### AccessKey ID

Введіть ID авторизованого ключа доступу Alibaba Cloud.

### AccessKey Secret

Введіть Secret авторизованого ключа доступу Alibaba Cloud.

### Бакет

Введіть назву бакету OSS.
