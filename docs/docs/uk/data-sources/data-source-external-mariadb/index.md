# Зовнішнє джерело даних - MariaDB

## Вступ

Використовуйте зовнішню базу даних MariaDB як джерело даних. Наразі підтримуються версії: MariaDB >= 10.3

## Встановлення

Цей плагін є комерційним плагіном, який потрібно завантажити та активувати через менеджер плагінів.
