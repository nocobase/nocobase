# Блок Календар

## Вступ

Блок Календар дозволяє зручно переглядати та керувати подіями та даними, пов'язаними з датами, у форматі календаря. Він ідеально підходить для планування зустрічей, заходів та ефективної організації вашого часу.

## Інсталяція

Цей плагін є вбудованим, тому додаткова інсталяція не потрібна.

## Додавання блоків

![20250403220300](https://static-docs.nocobase.com/20250403220300.png)

1.  Поле заголовка: Використовується для відображення інформації на смугах календаря. Наразі підтримуються такі типи полів, як `Single Line Text` (однорядковий текст), `Single select` (одиночний вибір), `Phone` (телефон), `Email` (електронна пошта), `Radio Group` (група перемикачів) та `Sequence` (послідовність). Підтримувані типи полів заголовка можна розширити за допомогою плагінів.
2.  Час початку: Вказує, коли починається завдання.
3.  Час закінчення: Вказує, коли завдання завершується.

### Поле кольору фону

:::info{title=Порада}
Версія NocoBase має бути v1.4.0-beta або вище.
:::

Ця опція дозволяє налаштувати колір фону подій календаря. Ось як її використовувати:

1.  Таблиця даних календаря повинна містити поле типу **Одиночний вибір (Single select)** або **Група перемикачів (Radio group)**, і це поле має бути налаштоване з кольорами.
2.  Потім поверніться до інтерфейсу налаштування блоку календаря та виберіть поле, яке ви щойно налаштували з кольорами, у розділі **Поле кольору фону**.
3.  Нарешті, ви можете спробувати вибрати колір для події календаря та натиснути «Надіслати». Ви побачите, що колір застосовано.

![20240914192017_rec_](https://static-docs.nocobase.com/20240914192017_rec_.gif)

### День початку тижня

> Підтримується у версії v1.7.7 та вище

Блок календаря підтримує встановлення дня початку тижня, дозволяючи вам вибрати **Неділю** або **Понеділок** як перший день тижня. За замовчуванням день початку тижня — **Понеділок**, що полегшує користувачам налаштування відображення календаря відповідно до регіональних звичок для кращого досвіду використання.
