# Колекція коментарів

## Вступ

Колекція коментарів — це спеціалізований шаблон таблиці даних, призначений для зберігання коментарів та відгуків користувачів. Завдяки функції коментування ви можете додати можливість коментувати будь-яку таблицю даних, дозволяючи користувачам обговорювати, надавати відгуки або робити примітки до певних записів. Колекція коментарів підтримує редагування форматованого тексту, забезпечуючи гнучкі можливості для створення контенту.

![comment-collection-2025-11-01-00-39-01](https://static-docs.nocobase.com/comment-collection-2025-11-01-00-39-01.png)

## Особливості

- **Редагування форматованого тексту**: За замовчуванням містить редактор Markdown (vditor), що підтримує створення контенту з форматованим текстом.
- **Зв'язок з будь-якою таблицею даних**: Можна пов'язувати коментарі із записами в будь-якій таблиці даних за допомогою полів зв'язку.
- **Багаторівневі коментарі**: Підтримує відповіді на коментарі, створюючи деревоподібну структуру коментарів.
- **Відстеження користувачів**: Автоматично записує автора коментаря та час створення.

## Посібник користувача

### Створення колекції коментарів

![comment-collection-2025-11-01-00-37-10](https://static-docs.nocobase.com/comment-collection-2025-11-01-00-37-10.png)

1. Перейдіть на сторінку керування таблицями даних.
2. Натисніть кнопку «Створити колекцію».
3. Виберіть шаблон «Колекція коментарів».
4. Введіть назву таблиці (наприклад: «Коментарі до завдань», «Коментарі до статей» тощо).
5. Система автоматично створить таблицю коментарів з такими полями за замовчуванням:
   - Вміст коментаря (тип Markdown vditor)
   - Створено (пов'язано з таблицею користувачів)
   - Час створення (тип дата і час)

### Налаштування зв'язків

Щоб пов'язати коментарі з цільовою таблицею даних, вам потрібно налаштувати поля зв'язку:

![](https://static-docs.nocobase.com/Solution/demoE3v1-19N.gif)

1. Додайте поле зв'язку «Багато до одного» в таблиці коментарів.
2. Виберіть цільову таблицю даних для зв'язування (наприклад: таблиця завдань, таблиця статей тощо).
3. Встановіть назву поля (наприклад: «Належить до завдання», «Належить до статті» тощо).

### Використання блоків коментарів на сторінках

![Enable Comments Collection](https://static-docs.nocobase.com/Solution/demoE3v1-20.gif)

1. Перейдіть на сторінку, куди ви хочете додати функцію коментування.
2. Додайте блок у деталі або спливаюче вікно цільового запису.
3. Виберіть тип блоку «Коментарі».
4. Виберіть колекцію коментарів, яку ви щойно створили.

### Типові сценарії використання

- **Системи управління завданнями**: Члени команди обговорюють завдання та надають відгуки.
- **Системи управління контентом**: Читачі коментують статті та взаємодіють з ними.
- **Робочі процеси затвердження**: Затверджувачі роблять примітки та висловлюють думки щодо заявок.
- **Відгуки клієнтів**: Збір відгуків клієнтів про продукти чи послуги.

## Примітки

- Колекція коментарів є функцією комерційного плагіна і вимагає активації плагіна коментарів для використання.
- Рекомендується встановити відповідні дозволи для таблиці коментарів, щоб контролювати, хто може переглядати, створювати та видаляти коментарі.
- Для сценаріїв з великою кількістю коментарів рекомендується увімкнути пагінацію для підвищення продуктивності.
