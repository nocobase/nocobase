# Поля колекції

## Типи інтерфейсів полів

NocoBase класифікує поля за інтерфейсом на такі категорії:

![20240512110352](https://static-docs.nocobase.com/20240512110352.png)

## Типи даних полів

Кожен інтерфейс поля має тип даних за замовчуванням. Наприклад, для полів з інтерфейсом «Число» (Number) тип даних за замовчуванням — `double`, але він також може бути `float`, `decimal` тощо. Наразі підтримуються такі типи даних:

![20240512103733](https://static-docs.nocobase.com/20240512103733.png)

## Мапування типів полів

Процес додавання нових полів до основної бази даних виглядає так:

1. Виберіть тип інтерфейсу.
2. Налаштуйте доступний тип даних для поточного інтерфейсу.

![20240512172416](https://static-docs.nocobase.com/20240512172416.png)

Процес мапування полів із зовнішніх джерел даних:

1. Автоматично мапуйте відповідний тип даних (Field type) та тип інтерфейсу (Field Interface) на основі типу поля зовнішньої бази даних.
2. За потреби змініть на більш відповідний тип даних та тип інтерфейсу.

![20240512172759](https://static-docs.nocobase.com/20240512172759.png)
