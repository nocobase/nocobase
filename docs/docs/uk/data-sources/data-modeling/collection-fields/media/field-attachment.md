# Поле вкладень

## Вступ

Система має вбудований тип поля "Вкладення", який дозволяє користувачам завантажувати файли до власних **колекцій**.

По суті, поле вкладення — це поле зв'язку "багато до багатьох", яке вказує на вбудовану системну **колекцію** файлів "Вкладення" (`attachments`). Коли ви створюєте поле вкладення в будь-якій **колекції**, автоматично генерується проміжна таблиця зв'язку "багато до багатьох". Метадані завантажених файлів зберігаються в **колекції** "Вкладення", а інформація про файли, на які посилається ваша **колекція**, пов'язується через цю проміжну таблицю.

### Обмеження за типом MIME

Використовується для обмеження типів файлів, дозволених для завантаження, за допомогою синтаксису [MIME](https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/MIME_types). Наприклад, `image/*` позначає файли зображень. Кілька типів можна розділяти комами, наприклад, `image/*,application/pdf` дозволяє завантажувати як зображення, так і файли PDF.

### Механізм сховища

Виберіть механізм сховища для завантажених файлів. Якщо залишити поле порожнім, буде використано механізм сховища за замовчуванням.
