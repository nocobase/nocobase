# Менеджер маршрутів

## Вступ

Менеджер маршрутів — це інструмент для керування маршрутами головної сторінки системи, що підтримує `десктопні` та `мобільні` пристрої. Маршрути, створені за допомогою менеджера маршрутів, автоматично відображаються в меню (можна налаштувати, щоб вони не відображалися). І навпаки, пункти меню, додані в меню сторінки, також синхронізуються зі списком менеджера маршрутів.

## Посібник користувача

### Типи маршрутів

Система підтримує чотири типи маршрутів:

- Група (`group`): Використовується для групування та керування маршрутами, може містити дочірні маршрути.
- Сторінка (`page`): Внутрішня сторінка системи.
- Вкладка (`tab`): Тип маршруту, що використовується для перемикання між вкладками на сторінці.
- Посилання (`link`): Внутрішнє або зовнішнє посилання, яке дозволяє безпосередньо перейти за вказаною адресою.

### Додавання маршруту

Щоб створити новий маршрут, натисніть кнопку «Add new» у верхньому правому куті:

1. Виберіть тип маршруту (`Type`).
2. Введіть заголовок маршруту (`Title`).
3. Виберіть іконку маршруту (`Icon`).
4. Налаштуйте відображення в меню (`Show in menu`).
5. Увімкніть або вимкніть вкладки сторінки (`Enable page tabs`).
6. Для типу «Сторінка» система автоматично згенерує унікальний шлях маршруту (`Path`).

![20250124131803](https://static-docs.nocobase.com/20250124131803.png)

### Дії з маршрутами

Кожен запис маршруту підтримує такі дії:

- Add child: Додати дочірній маршрут.
- Edit: Редагувати конфігурацію маршруту.
- View: Переглянути сторінку маршруту.
- Delete: Видалити маршрут.

### Масові дії

Верхня панель інструментів надає такі функції масових дій:

- Refresh: Оновити список маршрутів.
- Delete: Видалити вибрані маршрути.
- Hide in menu: Приховати вибрані маршрути в меню.
- Show in menu: Показати вибрані маршрути в меню.

### Фільтрація маршрутів

Використовуйте функцію «Filter» у верхній частині, щоб відфільтрувати список маршрутів за потреби.

:::info{title=Примітка}
Зміни в конфігурації маршрутів безпосередньо впливають на структуру навігаційного меню системи. Будь ласка, дійте обережно та переконайтеся в правильності налаштувань маршрутів.
:::
