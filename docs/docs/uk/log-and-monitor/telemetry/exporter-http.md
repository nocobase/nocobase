# Експортер телеметрії: HTTP

## Змінні оточення

### TELEMETRY_METRIC_READER

Тип експортера метрик телеметрії.

```bash
TELEMETRY_METRIC_READER=http
```

### TELEMETRY_HTTP_URL

HTTP URL для експорту даних телеметрії.

```bash
TELEMETRY_HTTP_URL=
```
