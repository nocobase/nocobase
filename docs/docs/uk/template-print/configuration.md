## Інструкції з налаштування

### Активація функції друку за шаблоном
Друк за шаблоном наразі підтримується для блоків деталей та табличних блоків. Нижче ми розглянемо способи налаштування для обох типів блоків.

#### Блоки деталей

1. **Відкрийте блок деталей**:
- В застосунку перейдіть до блоку деталей, де ви плануєте використовувати функцію друку за шаблоном.

2. **Відкрийте меню "Налаштування операцій"**:
- У верхній частині інтерфейсу натисніть меню "Налаштування операцій".

3. **Оберіть "Друк за шаблоном"**:
- У випадаючому меню натисніть опцію "Друк за шаблоном", щоб активувати функціонал плагіна.

![Активація друку за шаблоном](https://static-docs.nocobase.com/20241212150539-2024-12-12-15-05-43.png)

### Налаштування шаблонів

1. **Перейдіть на сторінку налаштування шаблонів**:
- У меню налаштувань кнопки "Друк за шаблоном" оберіть опцію "Налаштування шаблонів".

![Опція налаштування шаблонів](https://static-docs.nocobase.com/20241212151858-2024-12-12-15-19-01.png)

2. **Додайте новий шаблон**:
- Натисніть кнопку "Додати шаблон", щоб перейти на сторінку додавання шаблону.

![Кнопка "Додати шаблон"](https://static-docs.nocobase.com/20241212151243-2024-12-12-15-12-46.png)

3. **Заповніть інформацію про шаблон**:
- У формі шаблону вкажіть назву шаблону та оберіть його тип (Word, Excel, PowerPoint).
- Завантажте відповідний файл шаблону (підтримуються формати `.docx`, `.xlsx`, `.pptx`).

![Налаштування назви та файлу шаблону](https://static-docs.nocobase.com/20241212151518-2024-12-12-15-15-21.png)

4. **Редагування та збереження шаблону**:
- Перейдіть на сторінку "Список полів", скопіюйте поля та вставте їх у шаблон.
  ![Список полів](https://static-docs.nocobase.com/20250107141010.png)
  ![20241212152743-2024-12-12-15-27-45](https://static-docs.nocobase.com/20241212152743-2024-12-12-15-27-45.png)
- Після заповнення всіх деталей натисніть кнопку "Зберегти", щоб завершити додавання шаблону.

5. **Керування шаблонами**:
- Натисніть кнопку "Використати" праворуч від списку шаблонів, щоб активувати шаблон.
- Натисніть кнопку "Редагувати", щоб змінити назву шаблону або замінити файл шаблону.
- Натисніть кнопку "Завантажити", щоб завантажити вже налаштований файл шаблону.
- Натисніть кнопку "Видалити", щоб видалити шаблони, які більше не потрібні. Система запросить підтвердження, щоб уникнути випадкового видалення.
  ![Керування шаблонами](https://static-docs.nocobase.com/20250107140436.png)

#### Табличні блоки

Використання табличних блоків в основному ідентичне блокам деталей, за винятком наступних відмінностей:
1. Підтримка друку кількох записів: Спершу необхідно позначити записи, які ви хочете надрукувати. Одночасно можна надрукувати до 100 записів.
   
![20250416215633-2025-04-16-21-56-35](https://static-docs.nocobase.com/20250416215633-2025-04-16-21-56-35.png)

2. Ізольоване керування шаблонами: Шаблони для табличних блоків та блоків деталей не є взаємозамінними — це пов'язано з різницею у структурах даних (один є об'єктом, інший — масивом).
