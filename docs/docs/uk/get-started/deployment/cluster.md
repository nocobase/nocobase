# Кластерне розгортання

NocoBase підтримує розгортання застосунків у кластерному режимі для підвищення їхньої продуктивності та доступності. Кластерне розгортання дозволяє запускати різні екземпляри застосунку на кількох серверах або запускати кілька процесів у багатоядерному режимі на одному сервері.

Детальніше дивіться в документації про [Кластерний режим](/cluster-mode/).
