# Встановлення create-nocobase-app

:::tip Передумови

- Встановлено Node.js 20+ та Yarn 1.22.x
- Налаштовано та запущено одну з необхідних баз даних: MySQL 8.0.17+, MariaDB 10.9+ або PostgreSQL 10+


:::

## 1. Створення проєкту NocoBase

Оберіть версію NocoBase ([Порівняння версій](../quickstart.md)) та тип бази даних, а потім виконайте відповідну команду.

<Tabs>

```bash

   -e DB_PORT=5432 \

   -e DB_USER=nocobase \
```

:::warning Змінні середовища

- Змінна `TZ` використовується для встановлення часового поясу застосунку, за замовчуванням використовується часовий пояс операційної системи.
- Змінна `APP_KEY` — це секретний ключ застосунку, що використовується для генерації токенів користувачів тощо (якщо `APP_KEY` буде змінено, старі токени стануть недійсними). Це може бути будь-який випадковий рядок. **Будь ласка, змініть його на власний секретний ключ і переконайтеся, що він не буде розголошений.**
- Змінні `DB_*` — це конфігурації, пов'язані з базою даних. Будь ласка, змініть їх відповідно до ваших фактичних налаштувань підключення до бази даних.

:::

## 2. Зміна каталогу

```bash
cd my-nocobase-app
```

## 3. Встановлення залежностей

📢 Залежно від вашого мережевого середовища, конфігурації системи та інших факторів, цей крок може зайняти понад десять хвилин.

```bash
yarn install
```

:::tip Порада для виробничого середовища

Під час розгортання у виробничому середовищі, щоб зменшити розмір, можна встановити лише необхідні залежності:

```bash
yarn install --production
```

:::

## 4. Встановлення NocoBase

```bash
yarn nocobase install --lang=en-US
```

Процес встановлення автоматично створить структуру таблиць бази даних та ініціалізує дані.

## 5. Запуск NocoBase

**Середовище розробки**

```bash
yarn dev
```

**Виробниче середовище**

```bash
yarn start
```

## 6. Вхід у NocoBase

Відкрийте [http://localhost:13000](http://localhost:13000) у своєму браузері. Початковий обліковий запис та пароль: `admin@nocobase.com` та `admin123`.

:::warning Порада щодо безпеки облікового запису

Після першого входу, будь ласка, негайно змініть пароль за замовчуванням, щоб забезпечити безпеку системи.

:::
