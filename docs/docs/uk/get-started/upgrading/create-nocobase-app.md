# Оновлення інсталяції `create-nocobase-app`

:::warning Підготовка перед оновленням

- Обов'язково створіть резервну копію бази даних.
- Зупиніть запущений екземпляр NocoBase.

:::

## 1. Зупиніть запущений екземпляр NocoBase

Якщо процес не працює у фоновому режимі, зупиніть його за допомогою `Ctrl + C`. У виробничому середовищі для зупинки виконайте команду `pm2-stop`.

```bash
yarn nocobase pm2-stop
```

## 2. Виконайте команду оновлення

Просто виконайте команду оновлення `yarn nocobase upgrade`.

```bash
# Перейдіть до відповідної директорії
cd my-nocobase-app
# Виконайте команду оновлення
yarn nocobase upgrade
# Запустіть
yarn dev
```

### Оновлення до певної версії

Змініть файл `package.json` у кореневій директорії проєкту та оновіть номери версій для `@nocobase/cli` та `@nocobase/devtools` (можна лише оновлювати, а не понижувати версію). Наприклад:

```diff
{
  "dependencies": {
-   "@nocobase/cli": "1.5.11"
+   "@nocobase/cli": "1.6.0-beta.8"
  },
  "devDependencies": {
-   "@nocobase/devtools": "1.5.11"
+   "@nocobase/devtools": "1.6.0-beta.8"
  }
}
```

Потім виконайте команду оновлення

```bash
yarn install
yarn nocobase upgrade --skip-code-update
```
