# Часті запитання

## Не вдається отримувати листи після авторизації за допомогою облікового запису Microsoft

Наразі для авторизованого входу підтримуються лише облікові записи електронної пошти Outlook та Gmail. Облікові записи Microsoft та Google наразі не підтримуються. Додаткову інформацію дивіться тут: [answers.microsoft.com](https://answers.microsoft.com/zh-hans/outlook_com/forum/all/%E7%8E%B0%E6%9C%89%E5%BE%AE%E8%BD%AF%E8%B4%A6/dba12dda-a7c7-4346-8263-53f4a6d9dc68)

**Порада**: Якщо ви не впевнені, чи маєте ви "справжню електронну пошту Outlook.com" або "Gmail", спробуйте відвідати Outlook.com або Gmail.com у своєму веббраузері. Перевірте, чи можете ви увійти безпосередньо та надсилати листи без проблем. Якщо ні, це означає, що ви, можливо, не маєте відповідної поштової служби, і вам потрібно спочатку її активувати або скористатися іншою електронною поштою.

## Скасування авторизації

Якщо ваш обліковий запис електронної пошти вже авторизовано, але ви хочете його видалити або повторно авторизувати, виконайте такі дії:

#### **Gmail**

1. Відкрийте https://myaccount.google.com/u/0/connections та увійдіть.

![](https://static-docs.nocobase.com/mail-1735634131347.png)

2. Натисніть на відповідний застосунок, а потім натисніть "Видалити".

![](https://static-docs.nocobase.com/mail-1735634131518.png)

![](https://static-docs.nocobase.com/mail-1735634131697.png)

#### **Електронна пошта Microsoft**

1. Відкрийте https://account.microsoft.com/ та увійдіть.
    
2. Натисніть кнопку "Програми та служби, які можуть отримати доступ до ваших даних".

![](https://static-docs.nocobase.com/mail-1735634131870.png)

3. Натисніть "Редагувати", а потім "Видалити".

![](https://static-docs.nocobase.com/mail-1735634132052.png)
