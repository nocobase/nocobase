# Конфігурація блоків

## Блок повідомлень електронної пошти

### Додавання блоку

На сторінці конфігурації натисніть кнопку **Створити блок** та оберіть блок **Повідомлення електронної пошти (Усі)** або **Повідомлення електронної пошти (Особисті)**, щоб додати блок повідомлень електронної пошти.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_03_58_PM.png)

### Конфігурація полів

Натисніть кнопку **Поля** блоку, щоб обрати поля, які потрібно відобразити. Для детальніших інструкцій зверніться до методу конфігурації полів для таблиць.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_37_PM.png)

### Конфігурація фільтрації даних

Натисніть на іконку конфігурації праворуч від таблиці та оберіть **Діапазон даних**, щоб встановити діапазон даних для фільтрації електронних листів.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_39_PM.png)

Ви можете фільтрувати електронні листи з однаковим суфіксом за допомогою змінних:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_41_PM.png)

## Блок деталей електронного листа

Спершу увімкніть функцію **Увімкнути відкриття за кліком** у полі блоку повідомлень електронної пошти:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_45_PM.png)

У спливаючому вікні додайте блок **Деталі електронного листа**:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_46_PM.png)

Ви можете переглянути детальний вміст електронного листа:
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_04_49_PM.png)

У нижній частині ви можете самостійно налаштувати потрібні кнопки.

## Блок надсилання електронного листа

Існує два способи створити форму надсилання електронного листа:

1. Додайте кнопку **Надіслати електронний лист** у верхній частині таблиці:
   ![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_52_PM.png)

2. Додайте блок **Надіслати електронний лист**:
   ![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_53_PM.png)

Обидва способи дозволяють створити повну форму надсилання електронного листа:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_53_PM%20(1).png)

Кожне поле у формі електронного листа відповідає звичайній формі і може бути налаштоване з **Значенням за замовчуванням** або **Правилами зв'язування** тощо.

> Форми відповіді та пересилання електронного листа, вбудовані в нижню частину деталей електронного листа, за замовчуванням містять певну обробку даних, яку можна змінити за допомогою **FlowEngine**.
