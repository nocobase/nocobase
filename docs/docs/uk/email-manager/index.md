# Менеджер електронної пошти

## Вступ
Плагін Менеджер електронної пошти — це ефективний та зручний інструмент, який підтримує авторизацію та підключення поштових скриньок Gmail та Outlook. Він допомагає користувачам інтегрувати можливості керування електронною поштою, такі як надсилання та отримання листів, у різні блоки та сторінки. Завдяки простій конфігурації авторизації ви зможете централізовано керувати кількома обліковими записами та насолоджуватися безперебійним досвідом електронного спілкування.
