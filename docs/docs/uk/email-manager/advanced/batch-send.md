# Масова розсилка

## Огляд масової розсилки
Ви можете надсилати електронні листи багатьом людям одночасно (кожен отримувач отримає окремий лист).
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_06_10_PM.png)

## Масова розсилка через таблицю
Ви також можете надсилати електронні листи масово за допомогою таблиці:

1. Додайте кнопку надсилання електронного листа до таблиці
   ![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_14_PM.png)

2. Налаштуйте поле електронної пошти в таблиці (використовується для аналізу адрес електронної пошти після вибору рядків таблиці)
   ![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_13_PM.png)

3. Виберіть дані таблиці для надсилання електронних листів
   ![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_15_PM.png)

Якщо вам потрібно надіслати листи на всі адреси в усій таблиці, ви можете увімкнути функцію **За замовчуванням вибрати всі записи в поточній таблиці як отримувачів**:
![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_16_PM.png)

Після редагування вмісту електронного листа натисніть кнопку масової розсилки, щоб надіслати його:
![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_06_17_PM.png)

## Відстеження надсилання електронних листів
Ви можете переглядати та відстежувати статус надсилання конкретних електронних листів у Центрі керування електронною поштою:
![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_08_24_PM%20(1).png)
