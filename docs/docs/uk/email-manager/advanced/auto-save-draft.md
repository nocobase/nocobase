# Автозбереження чернетки

У меню кнопки **Зберегти чернетку** ви можете увімкнути **Автозбереження чернетки**.
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_29_PM.png)

Під час редагування електронного листа система зберігатиме чернетку в реальному часі, гарантуючи, що вміст не буде втрачено.
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_31_PM.png)

Коли ви знову відкриєте вікно редагування листа (для того ж відправника та отримувача), система запитає, чи бажаєте ви використати збережену чернетку.
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_08_35_PM.png)
