# Керування плагінами

## Менеджер плагінів

Натисніть на іконку менеджера плагінів у верхньому правому куті сторінки, щоб перейти на сторінку менеджера плагінів. Тут ви можете переглядати, встановлювати, вмикати або вимикати плагіни.

![2024-04-10_08-47-45](https://static-docs.nocobase.com/2024-04-10_08-47-45.png)

## Центр налаштувань плагінів

Плагіни зазвичай мають власні сторінки налаштувань. Натисніть на іконку налаштувань у верхньому правому куті сторінки, щоб переглянути меню налаштувань для всіх плагінів. Натисніть на відповідний пункт меню, щоб налаштувати плагін.
