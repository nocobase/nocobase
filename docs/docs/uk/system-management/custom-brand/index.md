# Налаштування бренду

## Вступ

Назва бренду за замовчуванням – "NocoBase". Вона відображається переважно у трьох місцях:

![20240409111100](https://static-docs.nocobase.com/20240409111100.png)

У нижній частині сторінки входу:

![20240409111254](https://static-docs.nocobase.com/20240409111254.png)

Позицію 1 можна налаштувати за допомогою опції «Логотип» у [Системних налаштуваннях](/system-management/system-settings/). Позиції 2 та 3 можна налаштувати за допомогою цього плагіна.

## Інструкції з використання

Після активації плагіна натисніть меню «Налаштування бренду» у верхньому правому куті сторінки, щоб перейти на сторінку конфігурації.

![20240409113640](https://static-docs.nocobase.com/20240409113640.png)

- **Бренд**: Використовується для встановлення тексту в нижній частині сторінки входу (позиція 3)
- **Про нас**: Використовується для налаштування вмісту меню у верхньому правому куті сторінки (як показано на зображенні нижче)

![20240409113915](https://static-docs.nocobase.com/20240409113915.png)

Конфігурація підтримує формат HTML для гнучкого налаштування вмісту. Якщо вам потрібно лише внести прості текстові зміни, ви можете безпосередньо змінити наявний вміст, як показано нижче:

![20240409114740](https://static-docs.nocobase.com/20240409114740.png)
