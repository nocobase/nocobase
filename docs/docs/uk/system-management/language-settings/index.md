# Налаштування мови

У розділі «Системні налаштування» ви можете налаштувати глобальне мовне середовище системи, вибравши «Увімкнені мови». Перша мова у списку буде мовою за замовчуванням.

![20240324192005](https://static-docs.nocobase.com/20240324192005.png)

Коли увімкнено кілька мов, користувачі можуть встановити свої особисті мовні налаштування в Особистому кабінеті (у верхньому правому куті сторінки).

![20240324192509](https://static-docs.nocobase.com/20240324192509.png)

Якщо поточному мовному середовищу бракує перекладів, ви можете скористатися плагіном «[Керування локалізацією](/system-management/localization/)», щоб додати або відредагувати переклади.

![20240324193055](https://static-docs.nocobase.com/20240324193055.png)
