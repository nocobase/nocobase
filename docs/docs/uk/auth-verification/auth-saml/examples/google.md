# Google Workspace

## Налаштуйте Google як IdP

[Консоль адміністратора Google](https://admin.google.com/) - Додатки - Веб- та мобільні додатки

![](https://static-docs.nocobase.com/0812780b990a97a63c14ea8991959827.png)

Після налаштування додатка скопіюйте **URL-адресу SSO**, **Ідентифікатор сутності** та **Сертифікат**.

![](https://static-docs.nocobase.com/aafd20a794730e85411c0c8f368637e0.png)

## Додайте новий автентифікатор у NocoBase

Налаштування плагінів - Автентифікація користувачів - Додати - SAML

![](https://static-docs.nocobase.com/5bc18c7952b8f15828e26bb07251a335.png)

Введіть скопійовану інформацію:

- SSO URL: URL-адреса SSO
- Public Certificate: Сертифікат
- idP Issuer: Ідентифікатор сутності
- http: Поставте прапорець, якщо тестуєте локально через HTTP.

Потім скопіюйте SP Issuer/EntityID та ACS URL з розділу Usage.

## Заповніть інформацію про SP у Google

Поверніться до консолі Google, на сторінці **Деталі постачальника послуг** введіть раніше скопійовані ACS URL та Ідентифікатор сутності, а також встановіть прапорець **Підписана відповідь**.

![](https://static-docs.nocobase.com/1536268bf8df4a5ebc72384317172191.png)

![](https://static-docs.nocobase.com/c7de1f8b84c1335de110e5a7c96255c4.png)

У розділі **Мапування атрибутів** додайте мапування для відповідних атрибутів.

![](https://static-docs.nocobase.com/27180f2f46480c3fee3016df86d6fdb8.png)
