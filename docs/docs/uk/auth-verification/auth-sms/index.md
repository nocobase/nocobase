# SMS-автентифікація

## Вступ

Плагін SMS-автентифікації дозволяє користувачам реєструватися та входити в NocoBase за допомогою SMS.

> Для роботи цього плагіна потрібна функція SMS-коду підтвердження, яку надає [`плагін @nocobase/plugin-verification`](/auth-verification/verification/).

## Додавання SMS-автентифікації

Перейдіть на сторінку керування плагінами автентифікації користувачів.

![](https://static-docs.nocobase.com/202502282112517.png)

## Конфігурація нової версії

:::info{title=Примітка}
Нову конфігурацію було впроваджено у версії `1.6.0-alpha.30`, а стабільна підтримка планується з версії `1.7.0`.
:::

![](https://static-docs.nocobase.com/202502282114821.png)

**Верифікатор:** Прив'яжіть SMS-верифікатор для надсилання SMS-кодів підтвердження. Якщо доступних верифікаторів немає, спочатку потрібно перейти на сторінку керування верифікацією та створити SMS-верифікатор.
Дивіться також:

- [Верифікація](../verification/index.md)
- [Верифікація: SMS](../verification/sms/index.md)

**Автоматична реєстрація, якщо користувач не існує:** Якщо цю опцію увімкнено, і номер телефону користувача не знайдено, буде зареєстровано нового користувача, використовуючи номер телефону як ім'я користувача (нікнейм).

## Конфігурація старої версії

![](https://static-docs.nocobase.com/a4d35ec63ba22ae2ea9e3e8e1cbb783d.png)

Функція SMS-автентифікації для входу використовуватиме налаштований і встановлений за замовчуванням провайдер SMS-кодів підтвердження для надсилання текстових повідомлень.

**Автоматична реєстрація, якщо користувач не існує:** Якщо цю опцію увімкнено, і номер телефону користувача не знайдено, буде зареєстровано нового користувача, використовуючи номер телефону як ім'я користувача (нікнейм).
