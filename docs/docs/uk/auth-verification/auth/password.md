## Дозволити реєстрацію

Якщо реєстрацію дозволено, на сторінці входу відображатиметься посилання для створення облікового запису, за яким можна перейти на сторінку реєстрації.

![](https://static-docs.nocobase.com/78903930d4b47aaf75cf94c55dd3596e.png)

Сторінка реєстрації

![](https://static-docs.nocobase.com/ac3c3ab42df28cb7c6dc70b24e99e7f7.png)

Якщо реєстрацію не дозволено, на сторінці входу не відображатиметься посилання для створення облікового запису.

![](https://static-docs.nocobase.com/8d5e3b6df9991bfc1c2e095a93745121.png)

Якщо реєстрацію не дозволено, доступ до сторінки реєстрації буде неможливий.

![](https://static-docs.nocobase.com/09325c4b07e09f88f80a14dff8430556.png)

## Налаштування форми реєстрації<Badge>v1.4.0-beta.7+</Badge>

Ви можете налаштувати, які поля в колекції користувачів мають відображатися у формі реєстрації, а також чи є вони обов'язковими. Принаймні одне з полів (ім'я користувача або електронна пошта) має бути налаштоване як видиме та обов'язкове.

## Забули пароль<Badge>v1.8.0+</Badge>

Функція відновлення пароля дозволяє користувачам скинути свій пароль за допомогою верифікації електронною поштою, якщо вони його забули.

### Конфігурація адміністратора

1.  **Увімкнути функцію відновлення пароля**

    На вкладці "Налаштування" > "Автентифікація" > "Забули пароль" встановіть прапорець "Увімкнути функцію відновлення пароля".

    ![20250423071957_rec_](https://static-docs.nocobase.com/20250423071957_rec_.gif)

2.  **Налаштувати канал сповіщень**

    Виберіть канал сповіщень електронною поштою (наразі підтримується лише електронна пошта). Якщо доступних каналів сповіщень немає, спочатку потрібно додати новий.

    ![20250423072225_rec_](https://static-docs.nocobase.com/20250423072225_rec_.gif)

3.  **Налаштувати лист для скидання пароля**

    Налаштуйте тему та вміст листа, підтримуються формати HTML або звичайний текст. Ви можете використовувати такі змінні:
    - Поточний користувач
    - Системні налаштування
    - Посилання для скидання пароля
    - Термін дії посилання для скидання (хвилини)

    ![20250427170047](https://static-docs.nocobase.com/20250427170047.png)

4.  **Встановити термін дії посилання для скидання**

    Встановіть термін дії посилання для скидання (у хвилинах). За замовчуванням – 120 хвилин.

    ![20250423073557](https://static-docs.nocobase.com/20250423073557.png)

### Порядок використання для користувача

1.  **Ініціювати запит на скидання пароля**

    На сторінці входу натисніть посилання "Забули пароль" (адміністратор має попередньо увімкнути цю функцію), щоб перейти на сторінку відновлення пароля.

    ![20250421103458_rec_](https://static-docs.nocobase.com/20250421103458_rec_.gif)

    Введіть зареєстровану адресу електронної пошти та натисніть кнопку "Надіслати лист для скидання".

    ![20250421113442_rec_](https://static-docs.nocobase.com/20250421113442_rec_.gif)

2.  **Скинути пароль**

    Користувач отримає лист, що містить посилання для скидання пароля. Натисніть на посилання, щоб відкрити сторінку, де ви зможете встановити новий пароль.

    ![20250421113748](https://static-docs.nocobase.com/20250421113748.png)

    Після завершення налаштування користувач зможе увійти в систему з новим паролем.

### Примітки

- Посилання для скидання має обмежений термін дії, за замовчуванням воно дійсне протягом 120 хвилин після генерації (може бути налаштовано адміністратором).
- Посилання можна використати лише один раз; після використання воно негайно стає недійсним.
- Якщо користувач не отримав лист для скидання пароля, будь ласка, перевірте правильність адреси електронної пошти або папку зі спамом.
- Адміністратор повинен переконатися, що конфігурація поштового сервера є правильною, щоб гарантувати успішне надсилання листів для скидання пароля.
