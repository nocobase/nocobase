# Верифікація: TOTP-автентифікатор

## Вступ

TOTP-автентифікатор дозволяє користувачам прив'язувати будь-який автентифікатор, що відповідає специфікації TOTP (Time-based One-Time Password) (<a href="https://www.rfc-editor.org/rfc/rfc6238" target="_blank">RFC-6238</a>), та виконувати верифікацію особи за допомогою одноразового пароля, що базується на часі (TOTP).

## Конфігурація для адміністратора

Перейдіть на сторінку керування верифікацією.

![](https://static-docs.nocobase.com/202502271726791.png)

Додати - TOTP-автентифікатор

![](https://static-docs.nocobase.com/202502271745028.png)

Окрім унікального ідентифікатора та заголовка, TOTP-автентифікатор не потребує додаткових налаштувань.

![](https://static-docs.nocobase.com/202502271746034.png)

## Прив'язка для користувача

Після додавання автентифікатора користувачі можуть прив'язати TOTP-автентифікатор у розділі керування верифікацією свого особистого кабінету.

![](https://static-docs.nocobase.com/202502272252324.png)

:::warning
Наразі плагін не надає механізму кодів відновлення. Після прив'язки TOTP-автентифікатора, будь ласка, зберігайте його в безпеці. Якщо ви випадково втратили автентифікатор, ви можете використати альтернативний метод верифікації для підтвердження своєї особи, відв'язати автентифікатор, а потім прив'язати його знову.
:::
