# ctx.i18n

Екземпляр i18n поточного контексту, що використовується для читання або перемикання мов. **Для перекладу тексту використовуйте [ctx.t()](./t.md)**; не використовуйте `ctx.i18n.t`.

## Сценарії використання

`ctx.i18n` можна використовувати у всіх середовищах виконання RunJS (наприклад, JSBlock, JSField, JSItem, JSColumn, робочий процес, правила зв'язку тощо).

## Визначення типу

```typescript
interface i18n: {
  language: string;
  changeLanguage(lng: string): Promise<any>;
}
```

## Загальні властивості

| Властивість | Тип | Опис |
|------|------|------|
| `language` | `string` | Код поточної активної мови (наприклад, `en-US`, `zh-CN`) |

## Загальні методи

### changeLanguage(lng)

Перемикає поточну мову.

| Параметр | Тип | Опис |
|------|------|------|
| `lng` | `string` | Код цільової мови (наприклад, `'en-US'`, `'zh-CN'`) |

**Повертає**: `Promise<any>`, який виконується (resolves) після завершення перемикання мови.

## Приклади

### Читання поточної мови

```ts
const lang = ctx.i18n.language;
// 'en-US' | 'zh-CN' | ...
if (lang.startsWith('zh')) {
  ctx.render(ctx.t('Chinese UI'));
} else {
  ctx.render(ctx.t('English UI'));
}
```

### Перемикання мови

```ts
// Перемкнути на англійську
await ctx.i18n.changeLanguage('en-US');

// Перемкнути на китайську
await ctx.i18n.changeLanguage('zh-CN');
```

### Кнопка перемикання мови

```ts
const { Button } = ctx.libs.antd;
const isZh = ctx.i18n.language.startsWith('zh');
ctx.render(
  <Button onClick={async () => {
    await ctx.i18n.changeLanguage(isZh ? 'en-US' : 'zh-CN');
  }}>
    {ctx.t(isZh ? 'Switch to English' : 'Switch to Chinese')}
  </Button>,
);
```

## Примітки

- **Переклад тексту**: Послідовно використовуйте `ctx.t()`, не використовуйте `ctx.i18n.t`.
