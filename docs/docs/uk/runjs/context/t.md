# ctx.t()

Швидка функція i18n у RunJS для перекладу тексту на основі мовних налаштувань поточного контексту. Підходить для інтернаціоналізації вбудованого тексту, такого як кнопки, заголовки та підказки.

## Сценарії використання

`ctx.t()` можна використовувати у всіх середовищах виконання RunJS.

## Визначення типів

```ts
t(key: string, options?: Record<string, any>): string
```

## Параметри

| Параметр | Тип | Опис |
|----------|------|-------------|
| `key` | `string` | Ключ перекладу або шаблон із заповнювачами (наприклад, `Hello {{name}}`, `{{count}} rows`). |
| `options` | `object` | Необов'язково. Змінні інтерполяції (наприклад, `{ name: 'Іван', count: 5 }`) або параметри i18n (наприклад, `defaultValue`, `ns`). |

## Значення, що повертається

- Повертає перекладений рядок. Якщо переклад для ключа відсутній і не вказано `defaultValue`, може повернути сам ключ або рядок після інтерполяції.

## Простір імен (ns)

**Типовим простором імен для середовища RunJS є `runjs`**. Якщо `ns` не вказано, `ctx.t(key)` шукатиме ключ у просторі імен `runjs`.

```ts
// За замовчуванням шукає ключ у просторі імен 'runjs'
ctx.t('Submit'); // Еквівалентно ctx.t('Submit', { ns: 'runjs' })

// Пошук ключа у конкретному просторі імен
ctx.t('Submit', { ns: 'myModule' });

// Послідовний пошук у кількох просторах імен (спочатку 'runjs', потім 'common')
ctx.t('Save', { ns: ['runjs', 'common'] });
```

## Приклади

### Простий ключ

```ts
ctx.t('Submit');
ctx.t('No data');
```

### Зі змінними інтерполяції

```ts
const text = ctx.t('Hello {{name}}', { name: ctx.user?.nickname || 'Guest' });
ctx.render(`<div>${text}</div>`);
```

```ts
ctx.message.success(ctx.t('Processed {{count}} rows', { count: rows.length }));
```

### Динамічний текст (наприклад, відносний час)

```ts
if (minutes < 60) return ctx.t('{{count}} minutes ago', { count: minutes });
if (hours < 24) return ctx.t('{{count}} hours ago', { count: hours });
```

### Вказання простору імен

```ts
ctx.t('Hello {{name}}', { name: 'Guest', ns: 'myModule' });
```

## Примітки

- **Плагін локалізації**: Щоб перекладати текст, необхідно активувати плагін локалізації. Відсутні ключі перекладу будуть автоматично додані до списку керування локалізацією для зручного обслуговування та перекладу.
- Підтримує інтерполяцію у стилі i18next: використовуйте `{{variableName}}` у ключі та передайте відповідну змінну в `options` для заміни.
- Мова визначається поточним контекстом (наприклад, `ctx.i18n.language`, локаль користувача).
