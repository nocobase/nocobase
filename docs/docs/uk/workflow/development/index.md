# Огляд

Вбудовані функції робочого процесу не можуть охопити всі сценарії. Наприклад, вбудовані типи вузлів не можуть вичерпати кожну можливу операцію у всіх бізнес-сценаріях. Тому ми також пропонуємо можливість розширення робочого процесу, включно з розширенням типів тригерів та вузлів. У бізнес-сценаріях, де вбудованих функцій недостатньо, ви можете розширити їх, використовуючи low-code методи.

Розширення переважно поділяються на дві частини:

- [Розширення типів тригерів](./trigger.md)
- [Розширення типів вузлів](./node.md)
