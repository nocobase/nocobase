# Вивід робочого процесу

## Вступ

Вузол "Вивід робочого процесу" використовується у викликаному робочому процесі для визначення його вихідного значення. Коли один робочий процес викликає інший, вузол "Вивід робочого процесу" дозволяє передати значення назад до того, хто його викликав.

## Створення вузла

У викликаному робочому процесі додайте вузол "Вивід робочого процесу":

![20241231002033](https://static-docs.nocobase.com/20241231002033.png)

## Налаштування вузла

### Вихідне значення

Введіть або виберіть змінну як вихідне значення. Вихідне значення може бути будь-якого типу: константа (наприклад, рядок, число, логічне значення, дата або власний JSON) або інша змінна з робочого процесу.

![20241231003059](https://static-docs.nocobase.com/20241231003059.png)

:::info{title=Порада}
Якщо до викликаного робочого процесу додано кілька вузлів "Вивід робочого процесу", то під час виклику цього робочого процесу буде виведено значення останнього виконаного вузла "Вивід робочого процесу".
:::
