# Додати запис

Використовується для додавання нового запису до колекції.

Значення полів для нового запису можуть використовувати змінні з контексту робочого процесу. Для присвоєння значень полям зв'язків можна безпосередньо посилатися на відповідні змінні даних у контексті, які можуть бути об'єктом або значенням зовнішнього ключа. Якщо ви не використовуєте змінні, вам потрібно вручну ввести значення зовнішніх ключів. Для кількох значень зовнішніх ключів у зв'язку «один до багатьох» їх слід розділяти комами.

## Створити вузол

В інтерфейсі конфігурації робочого процесу натисніть кнопку плюса («+») у потоці, щоб додати вузол «Додати запис»:

![Додати вузол «Додати запис»](https://static-docs.nocobase.com/386c8c01c89b1eeab848510e77f4841a.png)

## Конфігурація вузла

![Вузол «Додати запис»_Приклад_Конфігурація вузла](https://static-docs.nocobase.com/5f7b97a51b64a1741cf82a4d4455b610.png)

### Колекція

Виберіть колекцію, до якої ви хочете додати новий запис.

### Значення полів

Призначте значення полям колекції. Ви можете використовувати змінні з контексту робочого процесу або вручну вводити статичні значення.

:::info{title="Примітка"}
Дані, створені вузлом «Додати запис» у робочому процесі, не обробляють автоматично дані користувачів, такі як «Створено» та «Востаннє змінено». Вам потрібно самостійно налаштувати значення для цих полів за потреби.
:::

### Попереднє завантаження даних зв'язків

Якщо поля нового запису містять поля зв'язків і ви хочете використовувати відповідні дані зв'язків у подальших кроках робочого процесу, ви можете позначити відповідні поля зв'язків у конфігурації попереднього завантаження. Таким чином, після створення нового запису відповідні дані зв'язків будуть автоматично завантажені та збережені разом у результуючих даних вузла.

## Приклад

Наприклад, коли запис у колекції «Статті» створюється або оновлюється, потрібно автоматично створити запис «Версії статті», щоб зафіксувати історію змін статті. Для цього можна використати вузол «Додати запис»:

![Вузол «Додати запис»_Приклад_Конфігурація робочого процесу](https://static-docs.nocobase.com/dfd4820d49c145fa331883fc09c9161f.png)

![Вузол «Додати запис»_Приклад_Конфігурація вузла](https://static-docs.nocobase.com/1a0992e66170be12a068da6503298868.png)

Після активації робочого процесу з цією конфігурацією, при зміні запису в колекції «Статті» автоматично буде створено запис «Версії статті», який фіксуватиме історію змін статті.
