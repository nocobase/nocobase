# HTTP Відповідь

## Вступ

Цей вузол підтримується лише в синхронних робочих процесах Webhook і використовується для повернення відповіді сторонній системі. Наприклад, під час обробки зворотного виклику платежу, якщо бізнес-процес має неочікуваний результат (наприклад, помилка або збій), ви можете використати вузол відповіді, щоб повернути повідомлення про помилку сторонній системі, щоб деякі сторонні системи могли повторити спробу пізніше, ґрунтуючись на статусі.

Крім того, виконання вузла відповіді припинить виконання робочого процесу, і наступні вузли не будуть виконуватися. Якщо в усьому робочому процесі не налаштовано вузол відповіді, система автоматично відповість на основі статусу виконання процесу: повертаючи `200` у разі успішного виконання, і `500` у разі невдалого виконання.
