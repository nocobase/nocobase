# מבוא ל-NocoBase



## מהי NocoBase?

NocoBase היא מערכת ניהול תוכן Headless ופלטפורמת Low-Code המיועדת לבניית יישומים מותאמים אישית במהירות. היא מספקת מודל נתונים גמיש, תהליכי עבודה עוצמתיים, ומערכת תוספים ניתנת להרחבה.

## מושגי ליבה

### אוספים

ב-NocoBase, **אוסף** הוא אבן בניין יסודית. הוא מייצג סוג נתונים, בדומה לטבלה במסד נתונים. אתם יכולים להגדיר שדות עבור האוספים שלכם, כגון טקסט, מספרים, תאריכים, וקשרים לאוספים אחרים.

לדוגמה, ייתכן שיהיה לכם אוסף `Posts` עם שדות כמו `title`, `content`, `author`, ו-`publishDate`.

### תהליכי עבודה

**תהליכי עבודה** מאפשרים לכם להפוך תהליכים לאוטומטיים ולהגדיר לוגיקה עסקית. אתם יכולים ליצור תהליכי עבודה שמופעלים באירועים ספציפיים, כגון כאשר פריט חדש נוצר באוסף, או כאשר שדה מתעדכן.

תהליכי עבודה יכולים לבצע פעולות כמו שליחת מיילים, עדכון אוספים אחרים, או קריאה ל-APIs חיצוניים.

### מקורות נתונים

NocoBase יכולה להתחבר למגוון **מקורות נתונים**. בעוד שהיא מגיעה עם מסד נתונים מובנה, אתם יכולים גם להשתלב עם מסדי נתונים חיצוניים או APIs כדי לאחזר ולנהל את הנתונים שלכם.

גמישות זו מאפשרת לכם לאחד נתונים ממערכות שונות בתוך יישום ה-NocoBase שלכם.

### תוספים

הפונקציונליות של NocoBase ניתנת להרחבה באמצעות **תוספים**. תוספים מוסיפים תכונות חדשות, אינטגרציות, או משנים התנהגויות קיימות.

אתם יכולים להתקין תוספים משוק התוספים של NocoBase או לפתח תוספים מותאמים אישית משלכם כדי לענות על צרכים ספציפיים.
