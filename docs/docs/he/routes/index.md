# מנהל נתיבים

## מבוא

מנהל הנתיבים הוא כלי לניהול הנתיבים של הדף הראשי במערכת, התומך ב`מחשבים שולחניים` וב`מכשירים ניידים`. נתיבים שנוצרים באמצעות מנהל הנתיבים יוצגו באופן אוטומטי בתפריט (ניתן להגדיר שלא יוצגו בתפריט). באופן דומה, תפריטים שנוספים בתפריט הדף יופיעו גם הם ברשימת מנהל הנתיבים.

![20250107115449](https://static-docs.nocobase.com/20250107115449.png)

## מדריך למשתמש

### סוגי נתיבים

המערכת תומכת בארבעה סוגי נתיבים:

-   קבוצה (group): מאפשר לקבץ נתיבים לניהול נוח יותר, ויכול להכיל נתיבי משנה.
-   דף (page): דף פנימי של המערכת.
-   לשונית (tab): סוג נתיב המשמש למעבר בין לשוניות בתוך דף.
-   קישור (link): קישור פנימי או חיצוני, המאפשר קפיצה ישירה לכתובת הקישור שהוגדרה.

### הוספת נתיב

כדי ליצור נתיב חדש, לחצו על כפתור "Add new" בפינה הימנית העליונה:

1.  בחרו את סוג הנתיב (Type)
2.  מלאו את כותרת הנתיב (Title)
3.  בחרו את אייקון הנתיב (Icon)
4.  הגדירו אם להציג בתפריט (Show in menu)
5.  הגדירו אם להפעיל לשוניות דף (Enable page tabs)
6.  עבור סוג נתיב מסוג 'דף', המערכת תיצור באופן אוטומטי נתיב ייחודי (Path).

![20250124131803](https://static-docs.nocobase.com/20250124131803.png)

### פעולות נתיב

כל פריט נתיב תומך בפעולות הבאות:

-   הוספת ילד (Add child): הוספת נתיב משנה.
-   עריכה (Edit): עריכת תצורת הנתיב.
-   צפייה (View): צפייה בדף הנתיב.
-   מחיקה (Delete): מחיקת הנתיב.

### פעולות אצווה

סרגל הכלים העליון מציע את פעולות האצווה הבאות:

-   רענון (Refresh): רענון רשימת הנתיבים.
-   מחיקה (Delete): מחיקת הנתיבים שנבחרו.
-   הסתר בתפריט (Hide in menu): הסתרת הנתיבים שנבחרו מהתפריט.
-   הצג בתפריט (Show in menu): הצגת הנתיבים שנבחרו בתפריט.

### סינון נתיבים

השתמשו בפונקציית "Filter" בחלק העליון כדי לסנן את רשימת הנתיבים לפי הצורך.

:::info{title=הערה}
שינוי תצורות הנתיבים ישפיע ישירות על מבנה תפריט הניווט של המערכת. אנא פעלו בזהירות וודאו את נכונות תצורות הנתיבים.
:::
