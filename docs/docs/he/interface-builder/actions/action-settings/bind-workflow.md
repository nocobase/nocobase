## קישור מספר תהליכי עבודה

כפתור פעולה יכול להיות מקושר למספר תהליכי עבודה. כאשר מקושרים מספר תהליכי עבודה, סדר הביצוע שלהם פועל לפי הכללים הבאים:

1.  עבור תהליכי עבודה מאותו סוג טריגר, תהליכי עבודה סינכרוניים יבוצעו תחילה, ואחריהם תהליכי עבודה אסינכרוניים.
2.  תהליכי עבודה מאותו סוג טריגר יבוצעו לפי סדר ההגדרה.
3.  בין תהליכי עבודה מסוגי טריגר שונים:
    1.  אירועי לפני פעולה תמיד יבוצעו לפני אירועי אחרי פעולה ואישור.
    2.  לאירועי אחרי פעולה ואישור אין סדר ספציפי, ולוגיקה עסקית לא צריכה להיות תלויה בסדר ההגדרה.
