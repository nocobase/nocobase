# עריכת לחצן

## מבוא

המערכת תומכת בהתאמה אישית של שם הלחצן, האייקון שלו, צבע הרקע וסוג הלחצן. היא מציעה אפשרויות גמישות להתאמת ממשק המשתמש, כדי לעזור לכם להתאים את סגנון התצוגה והפונקציונליות של הלחצן לצרכים הספציפיים שלכם.

- **שם הלחצן**: באפשרותכם להתאים אישית את הטקסט של הלחצן בהתאם לצרכים העסקיים, כדי להבטיח שהתווית שלו תהיה ברורה ואינטואיטיבית.
- **אייקון הלחצן**: ניתן להגדיר אייקון ללחצן כדי לשפר את הזיהוי הוויזואלי ולאפשר למשתמשים לזהות במהירות את פונקציונליות הלחצן.
- **פעולה מסוכנת**: כאשר לחצן מייצג **פעולה מסוכנת** (כגון מחיקה, איפוס וכדומה), ניתן להגדיר אותו כסוג `Danger Action`.
- **סוג לחצן (`Button Type`)**: לחצנים תומכים בהגדרות סוג שונות. סוגי הלחצנים הנפוצים כוללים:
  - **`Default`**: לחצן סטנדרטי, מתאים לפעולות רגילות.
  - **`Primary`**: משמש לפעולות חשובות או ראשיות, ובדרך כלל מגיע עם אפקט ויזואלי בולט.
  - **`Dashed`**: לחצן עם מסגרת מקווקוות, מתאים לפעולות משניות או ללחצני רמז.
  - **`Text`**: לחצן הכולל טקסט בלבד וללא צבע רקע, משמש בדרך כלל לפעולות קלות משקל או לקישורים.
