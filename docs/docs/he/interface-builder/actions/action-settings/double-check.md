# אישור פעולה

## מבוא

הגדרת אישור הפעולה משמשת לוודא האם פעולה מסוימת תבוצע. כאשר לוחצים על פעולה, נפתחת תיבת דו-שיח לאישור. כפתור המחיקה מגיע עם אישור פעולה מופעל כברירת מחדל, וזאת כדי למנוע אובדן נתונים או תוצאות בלתי הפיכות עקב פעולות שגויות.
