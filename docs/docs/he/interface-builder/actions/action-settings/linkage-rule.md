# כללי קישוריות פעולה

## מבוא

כללי קישוריות פעולה מאפשרים לכם לשלוט באופן דינמי במצב של פעולה (כגון הצגה, הפעלה, הסתרה, השבתה וכו') בהתבסס על תנאים ספציפיים. על ידי הגדרת כללים אלה, תוכלו לקשר את ההתנהגות של כפתורי פעולה לרשומה הנוכחית, לתפקיד המשתמש או לנתוני הקשר.

![20251029150224](https://static-docs.nocobase.com/20251029150224.png)

## אופן השימוש

כאשר התנאי מתקיים (ברירת המחדל היא מעבר אם לא הוגדר תנאי), מופעלת הגדרת מאפיינים או ביצוע JavaScript. קבועים ומשתנים נתמכים בבדיקת התנאי.

![20251030224601](https://static-docs.nocobase.com/20251030224601.png)

הכלל תומך בשינוי מאפייני כפתורים.

![20251029150452](https://static-docs.nocobase.com/20251029150452.png)

## קבועים

דוגמה: הזמנות ששולמו אינן ניתנות לעריכה.

![20251029150638](https://static-docs.nocobase.com/20251029150638.png)

## משתנים

### משתני מערכת

![20251029150014](https://static-docs.nocobase.com/20251029150014.png)

דוגמה 1: שליטה בהצגת כפתור בהתבסס על סוג המכשיר הנוכחי.

![20251029151057](https://static-docs.nocobase.com/20251029151057.png)

דוגמה 2: כפתור העדכון בכמות גדולה (Bulk Update) בכותרת טבלת בלוק ההזמנות זמין רק לתפקיד מנהל המערכת; תפקידים אחרים אינם יכולים לבצע פעולה זו.

![20251029151209](https://static-docs.nocobase.com/20251029151209.png)

### משתני הקשר

דוגמה: כפתור הוספה בהזדמנויות הזמנה (בלוק קשר) מופעל רק כאשר סטטוס ההזמנה הוא "ממתין לתשלום" או "טיוטה". בסטטוסים אחרים, הכפתור יהיה מושבת.

![20251029151520](https://static-docs.nocobase.com/20251029151520.png)

![20251029152200](https://static-docs.nocobase.com/20251029152200.png)
