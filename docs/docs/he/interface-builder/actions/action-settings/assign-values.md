# הקצאת שדה

## מבוא

הקצאת שדה מתייחסת להגדרה מראש של ערך שדה לפני ביצוע פעולה. בעת עדכון או יצירת רשומה, נתונים אלו מתמלאים אוטומטית ללא צורך בהזנה ידנית, לדוגמה, בעדכון סטטוס הזמנות בכמות גדולה, או הגדרת זמן אספקה לזמן הנוכחי כאשר הזמנה נמסרת.
