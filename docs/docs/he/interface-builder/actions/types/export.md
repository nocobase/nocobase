# ייצוא

## מבוא

תכונת הייצוא מאפשרת לייצא רשומות מסוננות בפורמט **Excel**, ותומכת בהגדרת השדות לייצוא. משתמשים יכולים לבחור את השדות שהם צריכים לייצא לצורך ניתוח נתונים, עיבוד או ארכיון עתידי. תכונה זו משפרת את הגמישות בפעולות נתונים, במיוחד בתרחישים שבהם יש צורך להעביר נתונים לפלטפורמות אחרות או לעבד אותם הלאה.

### דגשים עיקריים:
- **בחירת שדות**: משתמשים יכולים להגדיר ולבחור את השדות לייצוא, כדי להבטיח שהנתונים המיוצאים יהיו מדויקים ותמציתיים.
- **תמיכה בפורמט Excel**: הנתונים המיוצאים יישמרו כקובץ Excel סטנדרטי, מה שמקל על שילובם וניתוחם עם נתונים אחרים.

באמצעות תכונה זו, משתמשים יכולים לייצא בקלות נתוני מפתח מעבודתם לשימוש חיצוני, ובכך לשפר את יעילות העבודה.

### שדות לייצוא

- רמה ראשונה: כל השדות של ה**אוסף** הנוכחי;
- רמה שנייה: אם זהו שדה קישור, יש לבחור שדות מה**אוסף** המקושר;
- רמה שלישית: מעובדות שלוש רמות בלבד; שדות הקישור ברמה האחרונה אינם מוצגים;

![20251029171557](https://static-docs.nocobase.com/20251029171557.png)

- [כלל קישוריות](/interface-builder/actions/action-settings/linkage-rule): הצגה/הסתרה דינמית של הכפתור;
- [עריכת כפתור](/interface-builder/actions/action-settings/edit-button): עריכת הכותרת, הצבע והאייקון של הכפתור;
