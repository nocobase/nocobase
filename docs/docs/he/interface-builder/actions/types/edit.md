# עריכה

## מבוא

פעולת העריכה היא כפתור שורה המשתמש ברשומה הנוכחית כהקשר. היא מוצגת כחלון קופץ שבו תוכלו להתאים אישית ולהגדיר בלוקים.

![20251029165442](https://static-docs.nocobase.com/20251029165442.png)

הגדירו בלוקים לפי הצורך.

![20251029165514](https://static-docs.nocobase.com/20251029165514.png)
