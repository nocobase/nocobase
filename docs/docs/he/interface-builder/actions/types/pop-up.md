# פתיחת חלון קופץ

## מבוא

**פעולת פתיחת חלון קופץ** דומה לפעולות הצפייה והעריכה. משתמשים יכולים להגדיר כפתור **פתיחת חלון קופץ** בשורות או בכותרת הטבלה. לחיצה על הכפתור תפתח חלון קופץ מתאים, והבלוקים בתוך חלון קופץ זה יציגו נתונים בהתאם לרשומת ההקשר הנוכחית.

![20251029182351](https://static-docs.nocobase.com/20251029182351.png)

הגדירו בלוקים בחלון הקופץ לפי הצורך.

![20251029190208](https://static-docs.nocobase.com/20251029190208.png)

## הגדרות פעולה

![20251029190352](https://static-docs.nocobase.com/20251029190352.png)

- [עריכת כפתור](/interface-builder/actions/action-settings/edit-button): עריכת הכותרת, הסוג והאייקון של הכפתור.
- [כלל קישוריות](/interface-builder/linkage-rule): שליטה דינמית על מצב הכפתור (גלוי/מושבת).
- [עריכת חלון קופץ](/interface-builder/actions/action-settings/edit-popup): הגדרת אופן הפתיחה (חלון קופץ או מגירה) והגודל.
