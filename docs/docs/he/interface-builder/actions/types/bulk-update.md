# עדכון מרובה

## מבוא

פעולת העדכון המרובה משמשת כאשר אתם צריכים לבצע את אותו עדכון על קבוצה של רשומות. לפני ביצוע עדכון מרובה, עליכם להגדיר מראש את לוגיקת הקצאת השדות לעדכון. לוגיקה זו תוחל על כל הרשומות שנבחרו כאשר המשתמש ילחץ על כפתור העדכון.

![20251029195320](https://static-docs.nocobase.com/20251029195320.png)

## הגדרות פעולה

![20251029195729](https://static-docs.nocobase.com/20251029195729.png)

### נתונים לעדכון

נבחרים/הכל, ברירת המחדל היא נבחרים.

![20251029200034](https://static-docs.nocobase.com/20251029200034.png)

### הקצאת שדות

הגדירו את השדות לעדכון מרובה. רק השדות שהוגדרו יעודכנו.

כפי שמוצג בתמונה, הגדירו את פעולת העדכון המרובה בטבלת ההזמנות כדי לעדכן באופן מרובה את הנתונים הנבחרים ל"ממתין לאישור".

![20251029200109](https://static-docs.nocobase.com/20251029200109.png)

- [עריכת כפתור](/interface-builder/actions/action-settings/edit-button): עריכת הכותרת, הסוג והאייקון של הכפתור;
- [כלל קישוריות](/interface-builder/actions/action-settings/linkage-rule): הצגה/הסתרה דינמית של הכפתור;
- [אישור כפול](/interface-builder/actions/action-settings/double-check)
