# הוספת רשומת בן

## מבוא

פעולה זו תוכננה במיוחד עבור **בלוקי טבלת עץ** כדי להוסיף במהירות רשומות בנים. בעת הוספת רשומת בן, השורה הנבחרת נחשבת לצומת האב, מה שמאפשר למשתמשים ליצור בקלות רשומות בנים תחת כל אב. תכונה זו מפשטת באופן ניכר את ניהול הנתונים ההיררכיים, והופכת את הרחבת וארגון מבני הנתונים ליעילים ואינטואיטיביים יותר.
