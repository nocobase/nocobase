# עריכה המונית

## מבוא

עריכה המונית מתאימה לתרחישים הדורשים עדכון גמיש של נתונים בקבוצות. לאחר לחיצה על כפתור העריכה ההמונית, ניתן להגדיר את טופס העריכה ההמונית בחלון קופץ ולקבוע אסטרטגיות עדכון שונות לכל שדה.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM.png)


## הגדרת פעולה

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM%20(1).png)


## מדריך למשתמש

### הגדרת טופס עריכה המונית

1. הוסיפו כפתור עריכה המונית.

2. הגדירו את טווח העריכה ההמונית: נבחרים (Selected) / הכל (All), ברירת המחדל היא נבחרים.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_14_AM.png)

3. הוסיפו טופס עריכה המונית.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_14_AM.png)

4. הגדירו את השדות לעריכה והוסיפו כפתור שליחה.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM%20(1).png)

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM.png)

### שליחת הטופס

1. סמנו את שורות הנתונים לעריכה.

2. בחרו את מצב העריכה עבור השדות והזינו את הערכים לשליחה.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_10_33_AM.png)

:::info{title="מצבי עריכה זמינים"}
* **ללא עדכון**: השדה יישאר ללא שינוי.
* **שינוי ל-**: עדכון השדה לערך שהוזן.
* **ניקוי**: מחיקת הנתונים בשדה.

:::

3. שלחו את הטופס.
