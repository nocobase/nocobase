# קישור

## מבוא

פעולת הקישור משתמשת בניווט נתיב (route navigation), תומכת בהעברת משתני URL, ומאפשרת לעמוד היעד להתאים באופן דינמי את תוכנו בהתבסס על הנתונים שהועברו. ניתן להגדיר אותה לשימוש בבלוקי נתונים.
