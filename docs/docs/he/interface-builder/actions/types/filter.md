# סינון

## מבוא
פעולת הסינון מופיעה בדרך כלל בבלוקי נתונים, ומאפשרת לסנן נתונים על ידי הגדרת תנאים שונים. חשוב לציין שאם כבר הוגדר היקף נתונים (data scope) עבור הבלוק, פעולת הסינון תשלב את התנאים של היקף הנתונים כתנאי הסינון הסופי.

### שדות ניתנים לסינון
כברירת מחדל, כל השדות של אוסף זה מופעלים. רק שדות מופעלים ניתנים לבחירה ברשימת שדות התנאי.
שדות מאוספים מקושרים יכולים לשמש גם כשדות סינון.
