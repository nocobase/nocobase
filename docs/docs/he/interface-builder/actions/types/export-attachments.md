# ייצוא קבצים מצורפים

## מבוא

ייצוא קבצים מצורפים תומך בייצוא שדות קשורים לקבצים מצורפים בפורמט חבילה דחוסה.

#### הגדרות ייצוא קבצים מצורפים

![20251029173251](https://static-docs.nocobase.com/20251029173251.png)

![20251029173425](https://static-docs.nocobase.com/20251029173425.png)

![20251029173345](https://static-docs.nocobase.com/20251029173345.png)

- הגדירו את שדות הקבצים המצורפים לייצוא; קיימת תמיכה בבחירה מרובה.
- באפשרותכם לבחור האם ליצור תיקייה עבור כל רשומה.

כללי מתן שמות לקבצים:

- אם תבחרו ליצור תיקייה עבור כל רשומה, כלל מתן השמות לקובץ יהיה: `{ערך שדה הכותרת של הרשומה}/{שם שדה הקובץ המצורף}[-{מספר סידורי של הקובץ}].{סיומת הקובץ}`.
- אם תבחרו שלא ליצור תיקייה, כלל מתן השמות לקובץ יהיה: `{ערך שדה הכותרת של הרשומה}-{שם שדה הקובץ המצורף}[-{מספר סידורי של הקובץ}].{סיומת הקובץ}`.

המספר הסידורי של הקובץ נוצר אוטומטית כאשר שדה קובץ מצורף מכיל מספר קבצים מצורפים.

- [כלל קישוריות](/interface-builder/actions/action-settings/linkage-rule): הצגה/הסתרה דינמית של הכפתור;
- [עריכת כפתור](/interface-builder/actions/action-settings/edit-button): עריכת כותרת הכפתור, סוגו והאייקון שלו;
