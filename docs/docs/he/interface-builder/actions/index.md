# פעולה

בממשק המשתמש (UI), פעולה היא כפתור המשמש להפעלת סט הוראות. ניתן למקם אותה ישירות בדף, בתיבת דו-שיח או במגירה, וגם לשלב אותה עם בלוקים. נכון לעכשיו, פעולות הניתנות להגדרה דינמית משמשות בעיקר בשילוב עם בלוקים.
