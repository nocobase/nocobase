# שדה

בממשק המשתמש (UI), שדה הוא רכיב שדה המשמש כנשא לנתוני יחידה. סוגי נתונים שונים מוצגים באמצעות רכיבי שדה שונים. שדות יכולים להיות משויכים רק לבלוקים, ואינם ניתנים לשימוש באופן עצמאי.

## שדות בבלוק

שדות בדרך כלל אינם משמשים לבדם, אלא קיימים כתת-אלמנטים של בלוקים מסוג נתונים. לבלוקים מסוג נתונים יש בדרך כלל אפשרות "הגדרת שדות", ורשימת השדות מסופקת על ידי האוסף הנוכחי.

![20251029204240](https://static-docs.nocobase.com/20251029204240.png)

## מעצב השדות (סרגל כלים)

בדומה לבלוקים, גם בפינה הימנית העליונה של רכיב שדה ישנם שלושה אייקונים, והם:

- גרירה לסידור
- [זרימת אירועים](/interface-builder/event-flow)
- תצורה

![20251025165432](https://static-docs.nocobase.com/20251025165432.png)

## סידור שדות

ניתן להתאים אישית את סידור השדות בתוך בלוק באמצעות גרירה ושחרור.

![20251025165605](https://static-docs.nocobase.com/20251025165605.gif)

## אפשרויות תצורת שדה

בנוסף לאפשרויות תצורת השדה הכלליות, סוגים שונים של שדות תומכים באפשרויות תצורה ספציפיות.
