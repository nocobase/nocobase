# שדות פרטים

## מבוא

הגדרות השדות עבור בלוקי פרטים, רשימה ורשת זהות במהותן, והן שולטות בעיקר באופן שבו השדות מוצגים במצב קריאה בלבד.

![20251025172851](https://static-docs.nocobase.com/20251025172851.png)

## אפשרויות הגדרת שדה

### עיצוב שדה תאריך

![20251025173005](https://static-docs.nocobase.com/20251025173005.png)

לפרטים נוספים, עיינו ב[עיצוב תאריך](/interface-builder/fields/specific/date-picker)

### עיצוב שדה מספרי

![20251025173242](https://static-docs.nocobase.com/20251025173242.png)

תומך בהמרת יחידות פשוטה, מפרידי אלפים, קידומות וסיומות, דיוק וכתיב מדעי.

לפרטים נוספים, עיינו ב[עיצוב מספרי](/interface-builder/fields/field-settings/number-format)

### הפעלת לחיצה לפתיחה

מלבד שדות יחסים התומכים בפתיחת חלון קופץ בלחיצה, ניתן להגדיר גם שדות רגילים להיפתח בלחיצה כנקודת כניסה לחלון קופץ. בנוסף, אפשר לקבוע את אופן פתיחת החלון הקופץ (מגירה, תיבת דו-שיח או דף משנה).

![20251025173549](https://static-docs.nocobase.com/20251025173549.gif)

### מצב תצוגה לגלישת תוכן

כאשר תוכן השדה גולש מעבר לרוחבו, ניתן להגדיר את מצב הגלישה.

- קיטוע (ברירת מחדל)
- גלישת שורה

![20251025173917](https://static-docs.nocobase.com/20251025173917.png)

### רכיב שדה

חלק מהשדות תומכים בפורמטים שונים של תצוגה, אותם ניתן להשיג באמצעות החלפת רכיב השדה.

לדוגמה: ניתן להחליף את רכיב ה-`URL` לרכיב `Preview`.

![20251025174042](https://static-docs.nocobase.com/20251025174042.png)

לדוגמה, שדות יחסים יכולים להיות מוצגים באופן שונה על ידי מעבר מרכיב שדה הכותרת ל-`Sub-details` (פרטי משנה) כדי להציג תוכן נוסף משדה היחסים.

![20251025174311](https://static-docs.nocobase.com/20251025174311.gif)
