# שדות טבלה

## מבוא

שדות טבלה, בנוסף להתאמת רוחב עמודה, כותרת שדה, מיון ועוד, תומכים גם באפשרויות תצוגה מותאמות אישית נוספות עבור שדות ספציפיים (כמו שדות תאריך, שדות יחסים ושדות מספריים).

![20251024174558](https://static-docs.nocobase.com/20251024174558.png)

## הגדרות שדה

### עיצוב שדה תאריך

![20251024175303](https://static-docs.nocobase.com/20251024175303.png)

לפרטים נוספים, ראו [עיצוב תאריך](/interface-builder/fields/specific/date-picker)

### עיצוב שדה מספרי

תומך בהמרת יחידות פשוטה, מפרידי אלפים, קידומות וסיומות, דיוק, וסימון מדעי.

![20251024175445](https://static-docs.nocobase.com/20251024175445.png)

לפרטים נוספים, ראו [עיצוב מספרי](/interface-builder/fields/field-settings/number-format)

### הפעלת עריכה מהירה

כאשר עריכה מהירה מופעלת, כפתור עריכה יופיע בעת ריחוף העכבר מעל העמודה. לחצו עליו כדי לערוך ולשמור נתונים במהירות בחלון קופץ.

![20251025171158](https://static-docs.nocobase.com/20251025171158.gif)

### הפעלת לחיצה לפתיחה

בנוסף לשדות יחסים התומכים בלחיצה לפתיחת חלון קופץ, ניתן להפעיל "לחיצה לפתיחה" גם עבור שדות רגילים כדי שישמשו כנקודת כניסה לפתיחת חלון קופץ.

![20251025172308](https://static-docs.nocobase.com/20251025172308.gif)

### אופן הצגת גלישת תוכן

כאשר תוכן העמודה חורג מרוחב עמודת הטבלה, ניתן להגדיר את אופן הטיפול בגלישה:

- קיטוע עם שלוש נקודות (ברירת מחדל)
- גלישת שורה

![20251025172549](https://static-docs.nocobase.com/20251025172549.png)

### עמודה מקובעת

![20251025170858](https://static-docs.nocobase.com/20251025170858.gif)

### רכיב שדה

חלק מהשדות תומכים במספר פורמטים של תצוגה. משתמשים יכולים להחליף רכיבי שדה כדי להשיג אפקטי תצוגה שונים ולענות על צרכים של תרחישים שונים. לדוגמה, שדה מסוג **URL** יכול לעבור לרכיב **Preview** כדי להציג טוב יותר תוכן קישור או תצוגה מקדימה של תמונה.

![20251025171658](https://static-docs.nocobase.com/20251025171658.png)

### ניתן למיון

רוב סוגי השדות תומכים כיום במיון בצד השרת. כאשר המיון מופעל, ניתן למיין נתונים בסדר עולה או יורד בהתבסס על שדה היעד.
