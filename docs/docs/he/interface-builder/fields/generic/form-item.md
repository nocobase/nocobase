# שדות טופס

## מבוא

לסוגי שדות שונים בטופס יש אפשרויות תצורה שונות. ניתן להרחיב את אפשרויות התצורה של השדות באמצעות תוספים (Plugin) כדי לענות על צרכים מותאמים אישית נוספים.

![20251025174555](https://static-docs.nocobase.com/20251025174555.png)

## אפשרויות תצורה של שדות
