# הצגת תווית

## מבוא

כברירת מחדל, שדות בבלוק מציגים תוויות שדה. תוויות אלו מזהות את המשמעות של כל שדה ומקלות על המשתמשים לקרוא ולהבין את תוכן הנתונים.

בתרחישים מסוימים (לדוגמה, בדפי תצוגה בלבד, או כשהמשמעות של השדות ברורה מאוד), תוכלו לבחור להסתיר את התוויות כדי ליצור ממשק נקי ומינימליסטי יותר. לאחר הסתרת התוויות, יוצגו רק ערכי השדות, ושמות השדות לא יופיעו.
