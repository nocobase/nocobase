# עריכת הסבר מרחף

## מבוא

הסבר מרחף של שדה הוא הטקסט המופיע כאשר אתם מרחפים עם העכבר מעל שדה, שנועד לספק הסבר נוסף או הנחיה לגבי השדה הזה. הסברים מרחפים אלה כוללים בדרך כלל מידע כגון מטרת השדה, פורמט הקלט ואילוצים, כדי לעזור למשתמשים להבין ולהשתמש בשדה טוב יותר.
