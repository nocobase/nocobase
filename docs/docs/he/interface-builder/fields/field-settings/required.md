# שדה חובה

## מבוא

"שדה חובה" הוא כלל נפוץ לאימות טפסים. באפשרותכם להפעיל אותו ישירות בהגדרות השדה, או להגדיר שדה כחובה באופן דינמי באמצעות כללי הקישוריות (linkage rules) של הטופס.

## היכן ניתן להגדיר שדה כחובה

### הגדרות שדה באוסף

כאשר שדה באוסף מוגדר כחובה, הדבר מפעיל אימות בצד השרת (backend validation), והשדה מוצג כחובה גם בממשק המשתמש (frontend) כברירת מחדל (לא ניתן לשינוי).
![20251025175418](https://static-docs.nocobase.com/20251025175418.png)

### הגדרות שדה

הגדירו שדה כחובה באופן ישיר. אפשרות זו מתאימה לשדות שתמיד דורשים מילוי על ידי המשתמש, כגון שם משתמש, סיסמה וכדומה.

![20251028222818](https://static-docs.nocobase.com/20251028222818.png)

### כללי קישוריות

הגדירו שדה כחובה בהתאם לתנאים, באמצעות כללי הקישוריות של שדות בבלוק הטופס.

דוגמה: מספר ההזמנה הוא שדה חובה כאשר תאריך ההזמנה אינו ריק.

![20251028223004](https://static-docs.nocobase.com/20251028223004.png)
