# עיצוב מספרים

## מבוא

עיצוב מספרים מתייחס לשיפור ויזואלי ולסטנדרטיזציה של שדות מספריים (כולל שדות נוסחה) במצב קריאה. הוא כולל הגדרות כמו מפרידי אלפים, דיוק עשרוני, המרת יחידות וסימון מדעי. על ידי הגדרת עיצוב מספרים, ניתן להתאים את אופן הצגת המספרים להרגלי הביטוי המספריים באזורים ובתעשיות שונות.

## אופן השימוש

- תומך בהמרת יחידות פשוטה (כפל/חילוק)
- מפריד אלפים
- קידומת וסיומת
- דיוק
- סימון מדעי.

דוגמה: עצבו סכום הזמנה על ידי הוספת סמל מטבע (לדוגמה, $), מפריד אלפים (כגון פסיק או נקודה) ודיוק עשרוני, כדי להפוך את הסכום לברור וקל לקריאה.
