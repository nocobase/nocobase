# מצב תצוגה

## מבוא

בניגוד לבלוקים, לרכיבי שדה יש שלושה מצבי תצוגה (הנתמכים רק עבור שדות בתוך טופס). מעבר בין מצבי תצוגה שונים מתאים לאפשרויות תצורה שונות של השדה.

- ניתן לעריכה;
- לקריאה בלבד (לא ניתן לעריכה);
- לקריאה בלבד (מצב קריאה);

### מצבי תצוגה לשדות קשר

**מצב התצוגה של שדה קשר** קובע כיצד השדה יוצג בממשק וכן אילו סוגי רכיבי שדה זמינים.

ב**מצב עריכה**, שדות קשר תומכים במספר סוגי רכיבים, מה שמאפשר למשתמשים לבחור רכיבי שדה קשר שונים כדי להציג או לבחור נתונים מקושרים בהתאם לצרכים העסקיים.

#### רכיבי שדה קשר במצב עריכה

![20251028220447](https://static-docs.nocobase.com/20251028220447.png)

במצב זה, משתמשים יכולים לבחור באופן גמיש את שיטת התצוגה המתאימה כדי לטפל בנתונים בצורה יעילה יותר.

#### רכיבי שדה קשר במצב קריאה בלבד

כאשר עוברים ל**מצב קריאה בלבד**, המערכת משתמשת אוטומטית כברירת מחדל ב**רכיב שדה הכותרת** כדי להציג את הנתונים המקושרים. מצב זה מתאים לתרחישים שבהם אתם רק צריכים לצפות בנתונים מבלי לבצע שינויים.
