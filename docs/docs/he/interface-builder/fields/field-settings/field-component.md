# רכיב שדה

## מבוא

רכיבי שדה משמשים להצגת התוכן של שדה.
לחלק מהשדות ישנם מספר רכיבי תצוגה מובנים. משתמשים יכולים לבחור את הרכיב המתאים בהתאם לצרכים בפועל, כדי להציג את ערך השדה בדרכים שונות.

דוגמה: במצב צפייה, שדה URL ניתן להחליף לרכיב תצוגה מקדימה, או לרכיב URL (ברירת מחדל).
