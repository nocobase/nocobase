# הגדרת טווח נתונים

## מבוא

הגדרת טווח הנתונים עבור שדה יחס דומה להגדרת טווח הנתונים עבור בלוק. היא קובעת תנאי סינון ברירת מחדל עבור הנתונים המקושרים.

## הוראות שימוש

![20251028211328](https://static-docs.nocobase.com/20251028211328.png)

### ערך סטטי

דוגמה: רק מוצרים שלא נמחקו ניתנים לבחירה לקישור.

> רשימת השדות מכילה שדות מתוך אוסף היעד של שדה היחס.

![20251028211434](https://static-docs.nocobase.com/20251028211434.png)

### ערך משתנה

דוגמה: רק מוצרים שתאריך השירות שלהם מאוחר מתאריך ההזמנה ניתנים לבחירה לקישור.

![20251028211727](https://static-docs.nocobase.com/20251028211727.png)

למידע נוסף על משתנים, עיינו ב-[משתנים](/interface-builder/variables)

### קישוריות שדות יחס

קישוריות בין שדות יחס מושגת באמצעות הגדרת טווח הנתונים.

דוגמה: באוסף ההזמנות קיים שדה יחס מסוג "אחד לרבים" בשם "מוצר הזדמנות" ושדה יחס מסוג "רבים לאחד" בשם "הזדמנות". לאוסף "מוצר הזדמנות" יש שדה יחס מסוג "רבים לאחד" בשם "הזדמנות". בבלוק טופס ההזמנה, הנתונים הניתנים לבחירה עבור "מוצר הזדמנות" מסוננים כדי להציג רק את מוצרי ההזדמנות המקושרים ל"הזדמנות" שנבחרה בטופס הנוכחי.
