# שדה כותרת

## מבוא

**שדה הכותרת** משמש בדרך כלל ברכיבי שדות יחסים כמזהה לנתונים המקושרים המוצגים בממשק. הוא קובע את אופן הצגת המידע העיקרי של הנתונים, ובדרך כלל נועד לעזור למשתמשים לזהות ולהבין את הנתונים במהירות.

סוגים שונים של שדות כותרת מתאימים לרכיבי שדות שונים ותומכים באפשרויות תצורה מתאימות. לדוגמה:

- כאשר שדה הכותרת הוא **שדה תאריך**, רכיב השדה משתמש ב**רכיב תאריך** כדי להציג את הנתונים.
- כאשר שדה הכותרת הוא **שדה טקסט**, רכיב השדה משתמש ב**רכיב טקסט** כדי להציג את הנתונים.
- כאשר שדה הכותרת הוא **שדה אפשרויות**, רכיב השדה משתמש ב**רכיב אפשרויות** כדי להציג את הנתונים.
