# בורר נפתח (Dropdown)

## מבוא

בורר נפתח מאפשר לקשר נתונים על ידי בחירה מתוך נתונים קיימים באוסף היעד, או על ידי הוספת נתונים חדשים לאוסף ולאחר מכן קישורם. אפשרויות הבחירה הנפתחת תומכות בחיפוש חלקי.

![20251029205901](https://static-docs.nocobase.com/20251029205901.png)

## הגדרות שדה

### הגדרת טווח נתונים

קובע את טווח הנתונים של הרשימה הנפתחת.

![20251029210025](https://static-docs.nocobase.com/20251029210025.png)

למידע נוסף, עיינו ב[הגדרת טווח נתונים](/interface-builder/fields/field-settings/data-scope)

### הגדרת כללי מיון

קובע את מיון הנתונים בבורר הנפתח.

דוגמה: מיון לפי תאריך שירות בסדר יורד.

![20251029210105](https://static-docs.nocobase.com/20251029210105.png)

### אפשרות הוספה/קישור של מספר רשומות

מגביל את הקישור של נתונים ביחסי 'רבים' (to-many) לרשומה אחת בלבד.

![20251029210145](https://static-docs.nocobase.com/20251029210145.png)

### שדה כותרת

שדה הכותרת הוא שדה התווית המוצג באפשרויות.

![20251029210507](https://static-docs.nocobase.com/20251029210507.gif)

> תומך בחיפוש מהיר המבוסס על שדה הכותרת

למידע נוסף, עיינו ב[שדה כותרת](/interface-builder/fields/field-settings/title-field)

### יצירה מהירה: הוסיפו תחילה, ואז בחרו

![20251125220046](https://static-docs.nocobase.com/20251125220046.png)

#### הוספה דרך תפריט נפתח

לאחר יצירת רשומה חדשה באוסף היעד, המערכת בוחרת אותה אוטומטית ומקשרת אותה בעת שליחת הטופס.

באוסף ההזמנות קיים שדה יחס "רבים לאחד" (many-to-one) בשם `Account`.

![20251125220447](https://static-docs.nocobase.com/20251125220447.gif)

#### הוספה דרך חלון קופץ (Modal)

הוספה דרך חלון קופץ מתאימה לתרחישי הזנת נתונים מורכבים יותר, ומאפשרת להגדיר טופס מותאם אישית ליצירת רשומות חדשות.

![20251125220607](https://static-docs.nocobase.com/20251125220607.gif)
