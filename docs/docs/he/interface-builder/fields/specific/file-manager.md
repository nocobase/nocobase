# מנהל קבצים

## מבוא

מנהל הקבצים הוא רכיב שדה קישור המיועד לטיפול בקשרים שבהם אוסף היעד הוא אוסף הקבצים.

![20251027194124](https://static-docs.nocobase.com/20251027194124.png)

## הגדרות שדה

- [רכיב שדה](/interface-builder/fields/association-field): החליפו לרכיבי שדה קישור אחרים, כמו רשימה נפתחת, בורר נתונים וכדומה.
- אפשרו בחירת קבצים קיימים: מופעל כברירת מחדל, מאפשר בחירה וקישור מתוך קבצים שהועלו.
