# טופס משנה

## מבוא

טופסי משנה מתאימים לתרחישים שבהם אתם יוצרים נתונים מקושרים חדשים לפני קישורם. נתונים מקושרים מרובי רמות מוצגים בצורה ברורה כטופס מקונן. בהשוואה לרכיבי בורר הנתונים (Data Selector) ובורר הרשימה הנפתחת (Dropdown), טופס משנה מאפשר לכם לנהל ישירות את השדות של ה**אוסף** המקושר בתוך בלוק העמוד הנוכחי, והנתונים המקושרים נשלחים יחד עם הטופס הראשי.
