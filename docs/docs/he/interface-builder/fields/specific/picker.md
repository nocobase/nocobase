# בורר נתונים

## מבוא

בורר הנתונים, המופיע כבורר טבלאי בחלון קופץ, מאפשר לכם לבחור נתונים קיימים מתוך אוסף היעד לצורך קישור, או להוסיף נתונים חדשים לאוסף היעד ולקשר אותם.

![20251027203343](https://static-docs.nocobase.com/20251027203343.png)

נתונים שכבר נבחרו לא יוצגו בבורר הטבלאי.

![20251027203717](https://static-docs.nocobase.com/20251027203717.gif)

בבורר הטבלאי, תוכלו גם לנהל באופן מתקדם את אוסף היעד המקושר (לדוגמה: הוספה, מחיקה, ייבוא/ייצוא ועוד).

![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
