# בחירה מדורגת

## מבוא

הבורר המדורג מיועד לשדות יחס שאוסף היעד שלהם הוא טבלת עץ. הוא מאפשר למשתמשים לבחור נתונים בהתאם למבנה ההיררכי של טבלת העץ, ותומך בחיפוש חלקי (fuzzy search) לסינון מהיר של הנתונים.

## הוראות שימוש

- כאשר היחס הוא 'אחד לאחד', הבחירה המדורגת היא בחירה יחידה.

![20251125214656](https://static-docs.nocobase.com/20251125214656.png)

- כאשר היחס הוא 'אחד לרבים', הבחירה המדורגת היא בחירה מרובה.

![20251125215318](https://static-docs.nocobase.com/20251125215318.png)

## אפשרויות תצורה של שדה

### שדה כותרת

שדה הכותרת קובע את התווית המוצגת עבור כל אפשרות.

![20251125214923](https://static-docs.nocobase.com/20251125214923.gif)

> תומך בחיפוש מהיר על בסיס שדה הכותרת

![20251125215026](https://static-docs.nocobase.com/20251125215026.gif)

למידע נוסף, ראו: [שדה כותרת](/interface-builder/fields/field-settings/title-field)

### הגדרת טווח נתונים

שולט בטווח הנתונים של רשימת העץ (כאשר רשומת בן עומדת בתנאים, רשומת האב שלה תיכלל גם היא).

![20251125215111](https://static-docs.nocobase.com/20251125215111.png)

למידע נוסף, ראו: [טווח נתונים](/interface-builder/fields/field-settings/data-scope)
