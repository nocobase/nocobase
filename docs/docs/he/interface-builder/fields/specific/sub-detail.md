# פרטי משנה

## מבוא
פרטי משנה הם הרכיב המקביל לטופס משנה במצב קריאה בלבד. בהשוואה לרכיבי תווית וכותרת, פרטי משנה מאפשרים לא רק להציג יותר נתונים מהאוסף הנוכחי, אלא גם ניתן להגדיר אותם להצגת מידע מאוספים קשורים. נתוני יחסים מרובי רמות מוצגים בצורה ברורה כפרטים מקוננים.
