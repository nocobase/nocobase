# תת-טבלה

## מבוא

תת-הטבלה מתאימה לטיפול בשדות יחס מסוג 'רבים'. היא תומכת ביצירה וקישור של נתונים חדשים באוסף היעד באופן אצווה, או בבחירה וקישור של נתונים קיימים.

## הוראות שימוש

![20251027223350](https://static-docs.nocobase.com/20251027223350.png)

סוגי שדות שונים בתת-הטבלה מציגים רכיבי שדה שונים. שדות גדולים (כגון טקסט עשיר, JSON, טקסט רב-שורתי וכדומה) נערכים באמצעות חלון קופץ.

![20251027223426](https://static-docs.nocobase.com/20251027223426.png)

שדות יחס בתת-הטבלה.

הזמנות (אחד לרבים) > מוצרי הזמנה (אחד לאחד) > הזדמנות

![20251027223530](https://static-docs.nocobase.com/20251027223530.png)

רכיב שדה היחס המוגדר כברירת מחדל הוא בורר נגלל (Dropdown) (תומך בבורר נגלל / בורר נתונים).

![20251027223754](https://static-docs.nocobase.com/20251027223754.png)

## אפשרויות תצורת שדה

### אפשר בחירת נתונים קיימים (מושבת כברירת מחדל)

תומך בבחירה וקישור של נתונים קיימים.
![20251027224008](https://static-docs.nocobase.com/20251027224008.png)

![20251027224023](https://static-docs.nocobase.com/20251027224023.gif)

### רכיב שדה

[רכיב שדה](/interface-builder/fields/association-field): החליפו לרכיבי שדה יחס אחרים, כגון בורר נגלל, בורר נתונים וכדומה;

### אפשר ניתוק קישור נתונים קיימים

> האם לאפשר ניתוק קישור של נתונים קיימים עבור שדה היחס בטופס העריכה.
