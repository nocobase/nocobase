# כותרת

## מבוא

במצב קריאה בלבד, שדות קישור מוצגים כברירת מחדל באמצעות **רכיב שדה הכותרת**, המזהה את הרשומה המקושרת הנוכחית על ידי הצגת ערך שדה הכותרת.
המערכת מתאימה ומציגה באופן אוטומטי את רכיב שדה הכותרת המתאים, בהתאם לסוג שדה הכותרת.
