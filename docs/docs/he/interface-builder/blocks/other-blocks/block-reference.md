# בלוק הפניה

## מבוא
בלוק ההפניה מאפשר להציג בלוק קיים ישירות בעמוד הנוכחי, על ידי ציון ה-UID של בלוק היעד. אין צורך להגדיר מחדש את בלוק היעד.

## הפעלת התוסף
תוסף זה מובנה אך אינו מופעל כברירת מחדל.
פתחו את "ניהול תוספים" → מצאו את "בלוק: הפניה" → לחצו על "הפעלה".

![הפעלת בלוק הפניה במנהל התוספים](https://static-docs.nocobase.com/block-reference-enable-20251102.png)

## איך מוסיפים את הבלוק
1) הוסיפו בלוק → קבוצת "בלוקים אחרים" → בחרו "בלוק הפניה".  
2) בהגדרות בלוק ההפניה, הגדירו:
   - `Block UID`: ה-UID של בלוק היעד
   - `Reference mode`: בחרו `Reference` או `Copy`

![הדגמת הוספה והגדרה של בלוק הפניה](https://static-docs.nocobase.com/20251102193949_rec_.gif)

### איך משיגים את ה-UID של הבלוק
- פתחו את תפריט ההגדרות של בלוק היעד, ולחצו על `Copy UID` כדי להעתיק את ה-UID של הבלוק.  

![העתקת UID מהגדרות הבלוק](https://static-docs.nocobase.com/block-reference-copy-uid-20251102.png)

## מצבים והתנהגות
- `Reference` (ברירת מחדל)
  - חולק את אותה תצורה כמו הבלוק המקורי; שינויים בבלוק המקורי או בכל מקום שבו הוא מופנה, יעדכנו את כל ההפניות באופן סינכרוני.

- `Copy`
  - יוצר בלוק עצמאי זהה למקורי באותו רגע; שינויים מאוחרים יותר לא יסונכרנו ביניהם ולא ישפיעו זה על זה.

## הסבר על התצורה
- בלוק הפניה:
  - "הגדרות הפניה": משמשות לציון ה-UID של בלוק היעד ובחירת מצב "Reference/Copy";
  - כמו כן, יוצגו ההגדרות המלאות של הבלוק המופנה עצמו (שווה ערך להגדרת הבלוק המקורי ישירות).

![הגדרות בלוק ההפניה](https://static-docs.nocobase.com/block-reference-settings-20251102.png)

- בלוק שהועתק:
  - הבלוק שמתקבל לאחר ההעתקה הוא מאותו סוג כמו הבלוק המקורי, והוא כולל רק את ההגדרות שלו עצמו;
  - הוא אינו כולל עוד את "הגדרות הפניה".

## מצבי שגיאה וגיבוי
- כאשר היעד חסר/לא חוקי: תוצג הודעת שגיאה. ניתן להגדיר מחדש את ה-UID של הבלוק בהגדרות בלוק ההפניה (הגדרות הפניה → Block UID), ולאחר השמירה התצוגה תשוחזר.  

![מצב שגיאה כאשר בלוק היעד אינו חוקי](https://static-docs.nocobase.com/block-reference-error-20251102.png)

## הערות ומגבלות
- זוהי תכונה ניסיונית – השתמשו בזהירות בסביבת ייצור.
- בעת העתקת בלוק, ייתכן שיהיה צורך להגדיר מחדש חלק מהתצורות התלויות ב-UID של היעד.
- כל התצורות של בלוק מופנה מסונכרנות אוטומטית, כולל תצורות כמו "טווח נתונים". עם זאת, לבלוק מופנה יכולה להיות [תצורת זרימת אירועים](/interface-builder/event-flow/) משלו, ולכן ניתן להשיג טווחי נתונים שונים או תצורות קשורות אחרות באופן עקיף באמצעות זרימות אירועים ופעולות JavaScript מותאמות אישית.
