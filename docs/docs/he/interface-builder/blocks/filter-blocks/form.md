# טופס סינון

## מבוא

טופס הסינון מאפשר למשתמשים לסנן נתונים על ידי מילוי שדות טופס. ניתן להשתמש בו לסינון בלוקי טבלה, בלוקי תרשים, בלוקי רשימה ועוד.

## כיצד להשתמש

בואו נתחיל עם דוגמה פשוטה כדי להבין במהירות כיצד להשתמש בטופס הסינון. נניח שיש לנו בלוק טבלה המכיל מידע על משתמשים, ואנחנו רוצים לסנן את הנתונים באמצעות טופס סינון, כך:

![20251031163036_rec_](https://static-docs.nocobase.com/20251031163036_rec_.gif)

שלבי ההגדרה הם:

1. הפעילו את מצב העריכה והוסיפו בלוק "טופס סינון" ובלוק "טבלה" לדף.
![20251031163525_rec_](https://static-docs.nocobase.com/20251031163525_rec_.gif)
2. הוסיפו את השדה "כינוי" גם לבלוק הטבלה וגם לבלוק טופס הסינון.
![20251031163932_rec_](https://static-docs.nocobase.com/20251031163932_rec_.gif)
3. כעת תוכלו להתחיל להשתמש בו.
![20251031163036_rec_](https://static-docs.nocobase.com/20251031163036_rec_.gif)

## שימוש מתקדם

בלוק טופס הסינון תומך בהגדרות מתקדמות יותר. הנה כמה מקרי שימוש נפוצים.

### חיבור מספר בלוקים

שדה טופס יחיד יכול לסנן נתונים במספר בלוקים בו-זמנית. כך תעשו זאת:

1. לחצו על אפשרות ההגדרה "Connect fields" עבור השדה.
![20251031170300](https://static-docs.nocobase.com/20251031170300.png)
2. הוסיפו את בלוקי היעד שברצונכם לחבר. בדוגמה זו, נבחר את בלוק הרשימה בדף.
![20251031170718](https://static-docs.nocobase.com/20251031170718.png)
3. בחרו שדה אחד או יותר מבלוק הרשימה לחיבור. כאן אנו בוחרים את השדה "כינוי".
![20251031171039](https://static-docs.nocobase.com/20251031171039.png)
4. לחצו על כפתור השמירה כדי להשלים את ההגדרה. התוצאה נראית כך:
![20251031171209_rec_](https://static-docs.nocobase.com/20251031171209_rec_.gif)

### חיבור בלוקי תרשים

ראו: [מסנני עמוד וקישוריות](../../../data-visualization/guide/filters-and-linkage.md)

### שדות מותאמים אישית

בנוסף לבחירת שדות מתוך אוספים (collections), תוכלו גם ליצור שדות טופס באמצעות "שדות מותאמים אישית". לדוגמה, תוכלו ליצור שדה בחירה נפתחת עם אפשרויות מותאמות אישית. כך תעשו זאת:

1. לחצו על האפשרות "שדות מותאמים אישית" כדי לפתוח את חלונית ההגדרות.
![20251031173833](https://static-docs.nocobase.com/20251031173833.png)
2. מלאו את כותרת השדה, בחרו "Select" כמודל השדה, והגדירו את האפשרויות.
![20251031174857_rec_](https://static-docs.nocobase.com/20251031174857_rec_.gif)
3. שדות מותאמים אישית שנוספו לאחרונה צריכים להיות מקושרים ידנית לשדות בבלוקי היעד. כך תעשו זאת:
![20251031181957_rec_](https://static-docs.nocobase.com/20251031181957_rec_.gif)
4. ההגדרה הושלמה. התוצאה נראית כך:
![20251031182235_rec_](https://static-docs.nocobase.com/20251031182235_rec_.gif)

מודלי השדות הנתמכים כעת הם:

- Input: שדה קלט טקסט חד-שורי
- Number: שדה קלט מספרי
- Date: בורר תאריכים
- Select: רשימה נפתחת (ניתן להגדיר לבחירה יחידה או מרובה)
- Radio group: כפתורי רדיו
- Checkbox group: תיבות סימון

### כיווץ

הוסיפו כפתור כיווץ כדי לכווץ ולפרוס את תוכן טופס הסינון, ובכך לחסוך מקום בדף.

![20251031182743](https://static-docs.nocobase.com/20251031182743.png)

הגדרות נתמכות:

![20251031182849](https://static-docs.nocobase.com/20251031182849.png)

- **שורות מכווצות**: קובע כמה שורות של שדות טופס יוצגו במצב מכווץ.
- **מכווץ כברירת מחדל**: כאשר מופעל, טופס הסינון יוצג במצב מכווץ כברירת מחדל.
