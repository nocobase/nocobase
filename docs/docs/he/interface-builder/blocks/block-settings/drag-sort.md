# מיון בגרירה

## היכרות

מיון בגרירה (Drag-and-drop sorting) מתבסס על שדה מיון ומשמש לסידור ידני של רשומות בתוך בלוק.


:::info{title=טיפ}
* כאשר משתמשים באותו שדה מיון למיון בגרירה במספר בלוקים שונים, הדבר עלול לשבש את הסדר הקיים.
* בעת שימוש במיון בגרירה בטבלה, לא ניתן להגדיר כללי הקבצה (grouping) עבור שדה המיון.
* טבלאות עץ תומכות במיון צמתים באותה רמה בלבד.
:::

## הסבר על כללי המיון

נניח שהסדר הנוכחי הוא:

```
[1,2,3,4,5,6,7,8,9]
```

כאשר אלמנט מסוים (למשל 5) מועבר קדימה למיקום של 3, רק ערכי המיון של 3, 4 ו-5 ישתנו: 5 תופס את המיקום של 3, ו-3 ו-4 זזים כל אחד מיקום אחד אחורה.

```
[1,2,5,3,4,6,7,8,9]
```

אם לאחר מכן תזיזו את 6 אחורה למיקום של 8, 6 יתפוס את המיקום של 8, ו-7 ו-8 יזוזו כל אחד מיקום אחד קדימה.

```
[1,2,5,3,4,7,8,6,9]
```
