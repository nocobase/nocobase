# פריסה

## מבוא

בלוקים מסוג טופס, כמו בלוקי טופס, בלוקי פרטים ובלוקי רשימה, תומכים בהגדרות פריסה. אפשרויות הפריסה כוללות פריסה אנכית (בה תוויות ושדות קלט מסודרים שורה אחר שורה) ופריסה אופקית (בה תוויות ושדות קלט ממוקמים זה לצד זה). משתמשים יכולים להתאים את פריסת שדות הבלוק בהתאם לצרכים הספציפיים שלהם, כדי להבטיח שהזנת או הצגת המידע תהיה ברורה ויעילה כאחד.

### אנכית (ברירת מחדל)

פריסת ברירת המחדל היא אנכית.

### אופקית

פריסה אופקית מציגה תוויות ושדות קלט זה לצד זה, מה שחוסך מקום ביעילות. עבור שדות עם כותרות ארוכות, ניתן לבחור להגדיר אותן לגלישת טקסט או להצגה עם שלוש נקודות (אליפסיס), כדי להבטיח ממשק נקי ואסתטי.

![20251024120237](https://static-docs.nocobase.com/20251024120237.png)
