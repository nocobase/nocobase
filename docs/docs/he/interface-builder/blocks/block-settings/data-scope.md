# הגדרת טווח נתונים

## מבוא
הגדרת טווח נתונים פירושה הגדרת תנאי סינון ברירת מחדל עבור בלוק נתונים. משתמשים יכולים להתאים באופן גמיש את טווח הנתונים בהתאם לצרכים העסקיים, אך ללא קשר לפעולות הסינון שיבוצעו, המערכת תחיל באופן אוטומטי את תנאי הסינון המוגדר כברירת מחדל, ובכך מבטיחה שהנתונים תמיד יישארו בגבולות הטווח שהוגדר.

## מדריך למשתמש

![20251027110053](https://static-docs.nocobase.com/20251027110053.png)

שדה הסינון תומך בבחירת שדות מהאוסף הנוכחי ואוספים קשורים.

![20251027110140](https://static-docs.nocobase.com/20251027110140.png)

### אופרטורים
סוגי שדות שונים תומכים באופרטורים שונים. לדוגמה, שדות טקסט תומכים באופרטורים כמו 'שווה ל-', 'לא שווה ל-', ו'מכיל'; שדות מספריים תומכים באופרטורים כמו 'גדול מ-' ו'קטן מ-'; ואילו שדות תאריך תומכים באופרטורים כמו 'בטווח' ו'לפני תאריך ספציפי'.

![20251027111124](https://static-docs.nocobase.com/20251027111124.png)

### ערך סטטי
דוגמה: סינון נתונים לפי 'סטטוס' ההזמנה.

![20251027111229](https://static-docs.nocobase.com/20251027111229.png)

### ערך משתנה
דוגמה: סינון נתוני הזמנות עבור המשתמש הנוכחי.

![20251027113349](https://static-docs.nocobase.com/20251027113349.png)
