# בלוק

בלוקים מכילים נתונים ותוכן. ניתן למקם אותם בדף (Page), בחלון קופץ (Modal) או במגירה (Drawer), וניתן לגרור ולסדר מספר בלוקים בחופשיות.

## סוגי בלוקים

![סוגי בלוקים](https://static-docs.nocobase.com/f71af45b5cd914ea0558f760ddbbba58.png)

- בלוק נתונים: משמש להצגת נתונים ממקור נתונים בממשק.
- בלוק סינון: משמש לשימוש בנתונים ממקור נתונים כתנאי סינון, כדי לסנן בלוקי נתונים אחרים.
- בלוק אחר: משמש להצגת תוכן ספציפי או עצמאי, כגון משימות תהליך עבודה, יומני ביקורת (audit logs), Markdown ועוד.

## הוספת בלוק

ניתן למקם בלוקים בדף (Page), בחלון קופץ (Modal) או במגירה (Drawer).

### בלוקים בדף

נכון לעכשיו, סוגי הבלוקים הזמינים בדף כוללים: בלוק נתונים, בלוק סינון ובלוק אחר.

![20251023222441](https://static-docs.nocobase.com/20251023222441.png)

### בלוקים בחלון קופץ (Modal או Drawer)

חלונות קופצים מגיעים בשני סוגים: חלונות קופצים (Modals) ומגירות (Drawers). בדומה לדפים, גם אליהם ניתן להוסיף בלוקים. ההבדל הוא שבלוקי טופס בחלונות קופצים משמשים בדרך כלל להוספה, עריכה או צפייה ברשומה בודדת. סוגי הבלוקים הזמינים כוללים בלוק נתונים ובלוק אחר.

![20251023222613](https://static-docs.nocobase.com/20251023222613.png)

## מעצב הבלוקים

לכל בלוק יש שלושה אייקונים קטנים בפינה הימנית העליונה. משמאל לימין, הם:

1. פריסת גרירה ושחרור
2. [זרימת אירועים](/interface-builder/event-flow)
3. הגדרות בלוק

![20251023224032](https://static-docs.nocobase.com/20251023224032.png)

עבור בלוקים פשוטים, כל אפשרויות התצורה מרוכזות ב'הגדרות בלוק', כמו למשל בלוק JS.

![20251023224903](https://static-docs.nocobase.com/20251023224903.png)

בלוקי נתונים מורכבים מציעים גם 'הגדרת שדות' ו'הגדרת פעולות' נפרדות ומוטמעות.

![20251023225141](https://static-docs.nocobase.com/20251023225141.png)

בנוסף, קיימת גמישות לאפשרויות קינון נוספות, כפי שניתן לראות בבלוק תרשים.

![](https://static-docs.nocobase.com/07588190b3f41ae3060e71d8b76b4447.png)
