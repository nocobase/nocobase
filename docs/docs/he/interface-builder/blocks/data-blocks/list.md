# בלוק רשימה

## מבוא

בלוק הרשימה מציג נתונים בפורמט של רשימה, ומתאים לתרחישי הצגת נתונים כמו רשימות משימות, חדשות ומידע על מוצרים.

## הגדרות בלוק

![20251023202835](https://static-docs.nocobase.com/20251023202835.png)

### הגדרת טווח נתונים

כפי שמוצג: סננו הזמנות עם סטטוס "מבוטל".

![20251023202927](https://static-docs.nocobase.com/20251023202927.png)

לפרטים נוספים, עיינו ב-[הגדרת טווח נתונים](/interface-builder/blocks/block-settings/data-scope)

### הגדרת כללי מיון

כפי שמוצג: מיין לפי סכום ההזמנה בסדר יורד.

![20251023203022](https://static-docs.nocobase.com/20251023203022.png)

לפרטים נוספים, עיינו ב-[הגדרת כללי מיון](/interface-builder/blocks/block-settings/sorting-rule)

## הגדרת שדות

### שדות מהאוסף הנוכחי

> **שימו לב**: שדות מאוספים בירושה (כלומר, שדות מאוספי אב) מוצגים באופן אוטומטי ומשולבים ברשימת השדות הנוכחית.

![20251023203103](https://static-docs.nocobase.com/20251023203103.png)

### שדות מאוספים קשורים

> **שימו לב**: ניתן להציג שדות מאוספים קשורים (נכון לעכשיו, נתמכות רק יחסי "אחד לאחד").

![20251023203611](https://static-docs.nocobase.com/20251023203611.png)

להגדרת שדות רשימה, עיינו ב-[שדה פרטים](/interface-builder/fields/generic/detail-form-item)
