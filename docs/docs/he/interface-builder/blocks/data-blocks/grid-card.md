# כרטיס רשת

## מבוא

בלוק כרטיס רשת מציג מידע מסכם של רשומות נתונים בפורמט כרטיסים. הוא תומך בהגדרת מספר העמודות עבור גדלי מסך שונים, כדי להבטיח תצוגה ידידותית למשתמש במגוון מכשירים.
