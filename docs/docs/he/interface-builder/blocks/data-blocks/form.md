# בלוק טופס

## מבוא

בלוק הטופס הוא בלוק חשוב לבניית ממשקי הזנת ועריכת נתונים. הוא ניתן להתאמה אישית ברמה גבוהה, ומשתמש ברכיבים מתאימים כדי להציג את השדות הנדרשים בהתבסס על מודל הנתונים. באמצעות זרימות אירועים כמו כללי קישוריות, בלוק הטופס יכול להציג שדות באופן דינמי. בנוסף, ניתן לשלב אותו עם תהליכי עבודה (Workflow) כדי להפעיל תהליכים אוטומטיים ולטפל בנתונים, ובכך לשפר עוד יותר את יעילות העבודה או ליישם תזמור לוגי.

## הוספת בלוק טופס

- **עריכת טופס**: משמש לעדכון נתונים קיימים.
- **הוספת טופס**: משמש ליצירת רשומות נתונים חדשות.

### פריסה

בלוק הטופס תומך בשני מצבי פריסה, אותם ניתן להגדיר באמצעות המאפיין `layout`:

- **horizontal** (פריסה אופקית): פריסה זו מציגה את התווית והתוכן באותה שורה, חוסכת מקום אנכי ומתאימה לטפסים פשוטים או למקרים עם מעט מידע.
- **vertical** (פריסה אנכית) (ברירת מחדל): התווית ממוקמת מעל השדה. פריסה זו הופכת את הטופס לקל יותר לקריאה ולמילוי, במיוחד עבור טפסים המכילים שדות מרובים או פריטי קלט מורכבים.
