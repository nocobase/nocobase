# תרשים

## מבוא

תוסף הדמיית הנתונים של NocoBase מספק שאילתות נתונים ויזואליות ומגוון עשיר של רכיבי תרשימים. משתמשים יכולים לבנות במהירות לוחות מחוונים ויזואליים באמצעות הגדרה פשוטה, כדי להציג תובנות נתונים ולתמוך בניתוח והצגה רב-ממדיים של נתונים.
