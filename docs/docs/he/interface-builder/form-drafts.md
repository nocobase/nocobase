# טיוטות טפסים

כאשר טיוטות טפסים מופעלות, התוכן המוזן נשמר באופן אוטומטי מקומית. גם אם הטופס נסגר בטעות או שהדף רוענן, הנתונים שהוזנו לא יאבדו.

![20251213093717](https://static-docs.nocobase.com/20251213093717.png)

כאשר קיימת טיוטה, תוצג הודעה בחלקו העליון של הטופס. הטיוטה תימחק אוטומטית לאחר שליחה מוצלחת; אם אין בה צורך יותר, ניתן גם למחוק אותה ידנית בכל עת.

![20251213092734](https://static-docs.nocobase.com/20251213092734.png)

::: tip טיפ

כרגע, טיוטות טפסים תומכות רק באחסון במטמון מקומי. גרסאות עתידיות יתמכו באחסון במטמון בצד השרת, מה שיאפשר סנכרון בין מכשירים מרובים.

:::
