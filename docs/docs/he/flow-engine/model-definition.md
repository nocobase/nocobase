# שדה אזור סין

## מבוא

תוסף שדה אזור סין (`field-china-region`) מוסיף את סוג השדה "אזור סין" ל-NocoBase.

## התקנה

בממשק הראשי של NocoBase, עברו ל"מנהל התוספים", מצאו את התוסף "אזור סין" והפעילו אותו.

## שימוש

בתצורת האוסף, הוסיפו שדה חדש ובחרו ב"אזור סין" כסוג השדה.

![](/images/field-china-region-1.png)

### תצורת השדה

בתצורת השדה, באפשרותכם להגדיר את מצב התצוגה עבור השדה:

- מחוז
- מחוז/עיר
- מחוז/עיר/נפה
- מחוז/עיר/נפה/מחוז משנה

![](/images/field-china-region-2.png)

בתצורת הממשק, באפשרותכם להגדיר טפסי ממשק שונים עבור שדה "אזור סין".
