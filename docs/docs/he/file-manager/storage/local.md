# מנוע אחסון: אחסון מקומי

קבצים שיועלו יישמרו בספרייה מקומית על הדיסק הקשיח של השרת. פתרון זה מתאים לתרחישים שבהם נפח הקבצים הכולל המנוהל על ידי המערכת קטן, או למטרות ניסיוניות.

## פרמטרי תצורה

![דוגמת תצורה של מנוע אחסון קבצים](https://static-docs.nocobase.com/20240529115151.png)

:::info{title=הערה}
סעיף זה מציג רק פרמטרים ספציפיים למנוע האחסון המקומי. לפרמטרים כלליים, אנא עיינו ב[פרמטרים כלליים של המנוע](./index.md#general-engine-parameters).
:::

### נתיב

מייצג גם את הנתיב היחסי לאחסון קבצים בשרת וגם את נתיב הגישה ב-URL. לדוגמה, "`user/avatar`" (ללא לוכסנים מובילים או סופיים) מייצג:

1. הנתיב היחסי בשרת שבו קבצים מועלים נשמרים: `/path/to/nocobase-app/storage/uploads/user/avatar`.
2. קידומת ה-URL לגישה לקבצים: `http://localhost:13000/storage/uploads/user/avatar`.
