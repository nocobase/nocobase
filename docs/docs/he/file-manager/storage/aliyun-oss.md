# מנוע אחסון: Aliyun OSS

מנוע אחסון המבוסס על Aliyun OSS. לפני השימוש, עליכם להכין את החשבון וההרשאות הנדרשים.

## פרמטרי תצורה

![דוגמת תצורה של מנוע אחסון Aliyun OSS](https://static-docs.nocobase.com/20240712220011.png)

:::info{title=הערה}
סעיף זה מציג רק את הפרמטרים הספציפיים למנוע האחסון של Aliyun OSS. לפרמטרים כלליים, אנא עיינו ב[פרמטרים כלליים של המנוע](./index#引擎通用参数).
:::

### אזור

הזינו את אזור האחסון של OSS, לדוגמה: `oss-cn-hangzhou`.

:::info{title=הערה}
תוכלו לצפות בפרטי האזור של ה-Bucket ב[קונסולת Aliyun OSS](https://oss.console.aliyun.com/). עליכם להשתמש רק בקידומת האזור (ללא שם הדומיין המלא).
:::

### AccessKey ID

הזינו את ה-ID של מפתח הגישה המורשה שלכם ב-Aliyun.

### AccessKey Secret

הזינו את ה-Secret של מפתח הגישה המורשה שלכם ב-Aliyun.

### Bucket

הזינו את שם ה-Bucket של אחסון ה-OSS.

### זמן קצוב

הזינו את זמן הקצוב להעלאה ל-Aliyun OSS, ביחידות של אלפיות השנייה. ברירת המחדל היא `60000` אלפיות השנייה (כלומר 60 שניות).
