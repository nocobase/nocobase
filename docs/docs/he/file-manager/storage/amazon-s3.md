# מנוע אחסון: Amazon S3

מנוע אחסון המבוסס על Amazon S3. לפני השימוש, יש להכין את החשבון וההרשאות הנדרשות.

## פרמטרי תצורה

![דוגמת תצורה למנוע אחסון Amazon S3](https://static-docs.nocobase.com/20251031092524.png)

:::info{title=הערה}
סעיף זה מציג רק את הפרמטרים הספציפיים למנוע האחסון של Amazon S3. לפרמטרים כלליים, אנא עיינו ב[פרמטרים כלליים למנוע](./index#引擎通用参数).
:::

### אזור

הזינו את אזור האחסון של S3, לדוגמה: `us-west-1`.

:::info{title=הערה}
ניתן לצפות בפרטי האזור של הדלי (bucket) ב[קונסולת Amazon S3](https://console.aws.amazon.com/s3/). יש להשתמש רק בקידומת האזור (אין צורך בשם הדומיין המלא).
:::

### AccessKey ID

הזינו את ה-ID של מפתח הגישה המורשה של Amazon S3.

### AccessKey Secret

הזינו את ה-Secret של מפתח הגישה המורשה של Amazon S3.

### דלי (Bucket)

הזינו את שם הדלי של אחסון S3.
