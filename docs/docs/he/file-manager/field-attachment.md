# שדה קובץ מצורף

## מבוא

המערכת כוללת שדה מובנה מסוג "קובץ מצורף" (Attachment), המאפשר למשתמשים להעלות קבצים לתוך אוספים מותאמים אישית.

שדה קובץ מצורף הוא למעשה שדה יחס מסוג רבים-לרבים (Many-to-Many), המצביע על אוסף הקבצים המובנה של המערכת, הנקרא "קבצים מצורפים" (`attachments`). כאשר יוצרים שדה קובץ מצורף בכל אוסף, נוצר באופן אוטומטי טבלת קישור (junction table) עבור יחס רבים-לרבים זה. מטא-הנתונים של הקבצים שהועלו נשמרים באוסף "קבצים מצורפים", ופרטי הקבצים המקושרים באוסף המקורי מקושרים דרך טבלת קישור זו.

### הגבלות סוגי MIME

משמש להגבלת סוגי הקבצים המותרים להעלאה, תוך שימוש בפורמט תיאור התחביר של [MIME](https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/MIME_types). לדוגמה: `image/*` מייצג קובצי תמונה. ניתן להפריד בין מספר סוגים באמצעות פסיק באנגלית, למשל: `image/*,application/pdf` מאפשר קובצי תמונה וקובצי PDF.

### מנוע אחסון

בחרו את מנוע האחסון שישמש לשמירת הקבצים שהועלו. אם השדה נשאר ריק, ייעשה שימוש במנוע האחסון ברירת המחדל של המערכת.
