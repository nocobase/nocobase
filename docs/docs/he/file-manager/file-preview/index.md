# תצוגה מקדימה של קבצים

בממשקים המכילים שדות קבצים (כולל שדות קבצים מצורפים), ניתן להציג קבצים בתצוגה מקדימה על ידי לחיצה על התמונה הממוזערת או על אייקון הקובץ. פונקציית התצוגה המקדימה המובנית תומכת במגוון סוגי קבצים, כולל תמונות, קובצי PDF ורוב סוגי הקבצים הנתמכים באופן טבעי על ידי דפדפנים.

![20251129232307](https://static-docs.nocobase.com/20251129232307.png)

עבור סוגי קבצים שאינם תומכים בתצוגה מקדימה מובנית, ניתן לאפשר פונקציונליות תצוגה מקדימה על ידי התקנה או הרחבה של תוספי תצוגה מקדימה מתאימים. לדוגמה, לאחר התקנת תוסף תצוגה מקדימה לקבצי Office, תוכלו להציג קובצי Word, Excel ו-PowerPoint.
