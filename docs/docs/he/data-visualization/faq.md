# שאלות נפוצות

## בחירת תרשימים
### כיצד עלי לבחור את התרשים המתאים?
תשובה: בחרו בהתאם לנתונים ולמטרות שלכם:
- מגמה ושינוי: תרשים קו או תרשים שטח
- השוואת ערכים: תרשים עמודות או תרשים פסים
- מבנה יחסי: תרשים עוגה או תרשים טבעת
- מתאם והתפלגות: תרשים פיזור
- מבנה היררכי והתקדמות שלבים: תרשים משפך

לסוגי תרשימים נוספים, עיינו ב[דוגמאות של ECharts](https://echarts.apache.org/examples).

### אילו סוגי תרשימים NocoBase תומכת בהם?
תשובה: מצב התצורה הוויזואלית כולל תרשימים נפוצים (תרשים קו, תרשים שטח, תרשים עמודות, תרשים פסים, תרשים עוגה, תרשים טבעת, תרשים משפך, תרשים פיזור ועוד); מצב התצורה המותאמת אישית תומך בכל סוגי התרשימים של ECharts.

## בעיות בשאילתות נתונים
### האם מצבי התצורה הוויזואלית ו-SQL חולקים הגדרות?
תשובה: לא, ההגדרות שלהם נשמרות בנפרד. מצב התצורה שבוצע בו השמירה האחרונה הוא זה שנכנס לתוקף.

## אפשרויות תרשים
### כיצד עלי להגדיר שדות תרשים?
תשובה: במצב התצורה הוויזואלית, בחרו שדות בהתאם לסוג התרשים. לדוגמה, תרשימי קו או עמודות דורשים שדות ציר X וציר Y; תרשימי עוגה דורשים שדה קטגוריה ושדה ערך.
מומלץ להריץ תחילה את "הפעל שאילתה" (`Run query`) כדי לוודא שהנתונים תואמים לציפיות; מיפוי השדות מתבצע אוטומטית כברירת מחדל.

## תצוגה מקדימה ושמירה
### האם עלי להציג שינויים בתצוגה מקדימה באופן ידני?
תשובה: במצב התצורה הוויזואלית, שינויים מוצגים אוטומטית בתצוגה מקדימה. במצבי SQL ותצורה מותאמת אישית, כדי למנוע רענונים תכופים, סיימו את העריכה ולחצו ידנית על "תצוגה מקדימה" (`Preview`).

### מדוע התצוגה המקדימה נעלמת לאחר סגירת הדיאלוג?
תשובה: התצוגה המקדימה מיועדת לצפייה זמנית בלבד. שמרו את השינויים לפני הסגירה; שינויים שלא נשמרו לא יישמרו.
