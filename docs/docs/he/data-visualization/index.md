# סקירה כללית

תוסף ויזואליזציית הנתונים של NocoBase מציע שאילתות נתונים ויזואליות ומגוון עשיר של רכיבי גרפים. באמצעות הגדרה פשוטה, תוכלו לבנות במהירות לוחות מחוונים, להציג תובנות נתונים, ולתמוך בניתוח והצגה רב-ממדיים של נתונים.

![clipboard-image-1761749573](https://static-docs.nocobase.com/clipboard-image-1761749573.png)

## מושגי יסוד
- בלוק גרף: רכיב גרף הניתן להגדרה בעמוד, התומך בשאילתות נתונים, אפשרויות גרף ואירועי אינטראקציה.
- שאילתת נתונים (Builder / SQL): הגדירו ויזואלית באמצעות ה-Builder או כתבו SQL כדי לאחזר נתונים.
- מדדים (Measures) וממדים (Dimensions): מדדים משמשים לצבירה מספרית; ממדים מקבצים נתונים (לדוגמה, תאריך, קטגוריה, אזור).
- מיפוי שדות: מפו עמודות של תוצאות שאילתה לשדות גרף מרכזיים כמו `xField`, `yField`, `seriesField` או `Category / Value`.
- אפשרויות גרף (Basic / Custom): Basic מגדיר תכונות נפוצות באופן ויזואלי; Custom מחזיר אובייקט `option` מלא של ECharts באמצעות JS.
- הפעלת שאילתה: הריצו את השאילתה ואחזרו נתונים בלוח ההגדרות; ניתן לעבור בין Table / JSON כדי לבדוק את הנתונים שהוחזרו.
- תצוגה מקדימה ושמירה: תצוגה מקדימה היא זמנית; לחיצה על 'שמור' כותבת את ההגדרה למסד הנתונים ומחילה אותה.
- משתני הקשר: שימוש חוזר במידע הקשרי של עמוד, משתמש ומסנן (לדוגמה, `{{ ctx.user.id }}`) בשאילתות ובהגדרות גרף.
- מסננים וקישוריות: בלוקי מסננים ברמת העמוד אוספים תנאים מאוחדים, מתמזגים אוטומטית לשאילתות גרף ומרעננים גרפים מקושרים.
- אירועי אינטראקציה: רישום אירועים באמצעות `chart.on` כדי לאפשר הדגשה, ניווט ו-drill-down.

## התקנה
ויזואליזציית נתונים היא תוסף מובנה ב-NocoBase; הוא עובד ישר מהקופסה ולא דורש התקנה נפרדת.
