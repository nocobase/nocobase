# פלט תהליך עבודה

## מבוא

צומת "פלט תהליך עבודה" משמש בתהליך עבודה נקרא כדי להגדיר את ערך הפלט שלו. כאשר תהליך עבודה אחד נקרא על ידי תהליך עבודה אחר, ניתן להשתמש בצומת "פלט תהליך עבודה" כדי להעביר ערך בחזרה לקורא.

## יצירת צומת

בתהליך העבודה הנקרא, הוסיפו צומת "פלט תהליך עבודה":

![20241231002033](https://static-docs.nocobase.com/20241231002033.png)

## הגדרת הצומת

### ערך פלט

הזינו או בחרו משתנה כערך הפלט. ערך הפלט יכול להיות מכל סוג, כגון קבוע (מחרוזת, מספר, ערך בוליאני, תאריך או JSON מותאם אישית), או משתנה אחר מתוך תהליך העבודה.

![20241231003059](https://static-docs.nocobase.com/20241231003059.png)

:::info{title=טיפ}
אם תהליך עבודה נקרא מכיל מספר צומתי "פלט תהליך עבודה", בעת קריאתו, הפלט יהיה הערך של הצומת האחרון שבוצע.
:::
