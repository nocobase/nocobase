# מחיקת נתונים

משמש למחיקת נתונים מתוך אוסף העונים על תנאים מסוימים.

השימוש הבסיסי בצומת המחיקה דומה לצומת העדכון, אלא שצומת המחיקה אינו דורש הקצאת שדות. עליכם רק לבחור את האוסף ואת תנאי הסינון. התוצאה של צומת המחיקה מחזירה את מספר השורות שנמחקו בהצלחה, שניתן לצפות בהן רק בהיסטוריית הביצוע ולא ניתן להשתמש בהן כמשתנה בצמתים הבאים.

:::info{title=שימו לב}
נכון לעכשיו, צומת המחיקה אינו תומך במחיקה פריט-אחר-פריט; הוא מבצע מחיקה בצובר. לכן, הוא לא יפעיל אירועים אחרים עבור כל מחיקת נתונים בנפרד.
:::

### אוסף

בחרו את האוסף שממנו ברצונכם למחוק נתונים.

### תנאי סינון

בדומה לתנאי הסינון בשאילתת אוסף רגילה, ניתן להשתמש במשתני ההקשר של תהליך העבודה.

## דוגמה

לדוגמה, כדי לנקות באופן תקופתי נתוני הזמנות היסטוריים שבוטלו ואינם תקפים, ניתן להשתמש בצומת המחיקה:

![צומת מחיקה_דוגמה_הגדרות צומת](https://static-docs.nocobase.com/b94b725f8523c3f13ce5f320.png)

תהליך העבודה יופעל באופן תקופתי ויבצע את מחיקת כל נתוני ההזמנות ההיסטוריים שבוטלו ואינם תקפים.
