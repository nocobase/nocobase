# הוספת רשומה

משמש להוספת רשומה חדשה לאוסף.

ערכי השדות של הרשומה החדשה יכולים להשתמש במשתנים מתוך הקונטקסט של תהליך העבודה. כדי להקצות ערכים לשדות יחסים, ניתן להתייחס ישירות למשתני הנתונים המתאימים בקונטקסט, שיכולים להיות אובייקט או ערך מפתח זר. אם אינכם משתמשים במשתנים, עליכם להזין ידנית את ערכי המפתח הזר. עבור מספר ערכי מפתח זר ביחסים מסוג 'רבים לרבים', יש להפריד ביניהם באמצעות פסיקים באנגלית.

## יצירת צומת

בממשק התצורה של תהליך העבודה, לחצו על כפתור הפלוס ("+") בזרימה כדי להוסיף צומת "הוספת רשומה":

![הוספת צומת 'הוספת רשומה'](https://static-docs.nocobase.com/386c8c01c89b1eeab848510e77f4841a.png)

## תצורת צומת

![צומת הוספת רשומה_דוגמה_תצורת צומת](https://static-docs.nocobase.com/5f7b97a51b64a1741cf82a4d4455b610.png)

### אוסף

בחרו את האוסף שאליו ברצונכם להוסיף רשומה חדשה.

### ערכי שדות

הקצו ערכים לשדות האוסף. באפשרותכם להשתמש במשתנים מתוך הקונטקסט של תהליך העבודה או להזין ערכים סטטיים ידנית.

:::info{title="הערה"}
נתונים שנוצרו על ידי צומת "הוספת רשומה" בתהליך עבודה אינם מטפלים אוטומטית בנתוני משתמש כמו "נוצר על ידי" ו"שונה לאחרונה על ידי". עליכם להגדיר את הערכים עבור שדות אלו בעצמכם, לפי הצורך.
:::

### טעינה מוקדמת של נתוני יחסים

אם שדות הרשומה החדשה כוללים שדות יחסים ואתם מעוניינים להשתמש בנתוני היחסים המתאימים בשלבים הבאים של תהליך העבודה, באפשרותכם לסמן את שדות היחסים המתאימים בתצורת הטעינה המוקדמת. בדרך זו, לאחר יצירת הרשומה החדשה, נתוני היחסים המתאימים ייטענו אוטומטית ויישמרו יחד בנתוני התוצאה של הצומת.

## דוגמה

לדוגמה, כאשר רשומה באוסף "מאמרים" נוצרת או מתעדכנת, יש ליצור אוטומטית רשומה מסוג "גרסאות מאמר" כדי לתעד היסטוריית שינויים עבור המאמר. ניתן להשתמש בצומת "הוספת רשומה" כדי להשיג זאת:
