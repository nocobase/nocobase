# סקירת מנועי חישוב

ב-NocoBase, מנועי החישוב מהווים חלק בלתי נפרד משדות נוסחה, כללי קישור (linkage rules) ותהליכי עבודה, ומספקים למשתמשים כלי חישוב ורסטיליים.

## Formula.js

**Formula.js** היא ספריית JavaScript קלת משקל המשחזרת רבות מפונקציות הנוסחה של Microsoft Excel. היא מאפשרת למשתמשים לבצע חישובים מורכבים ללא מאמץ. המאפיינים העיקריים כוללים:

- **ספריית פונקציות עשירה**: מציעה מגוון רחב של פונקציות לפעולות מתמטיות שונות, כולל חישובים סטטיסטיים ופיננסיים.
- **ממשק ידידותי למשתמש (API)**: תוכנן לנוחות שימוש, ומאפשר למפתחים ולמשתמשים שאינם מפתחים ליצור נוסחאות ללא צורך בכתיבת קוד נרחבת.
- **חישובים בזמן אמת**: מותאם לעיבוד נתונים בזמן אמת, מה שהופך אותו למתאים ליישומים הדורשים משוב חישובי מיידי.

## Math.js

**Math.js** היא ספריית מתמטיקה מקיפה עבור JavaScript ו-Node.js. היא תומכת בקשת רחבה של פעולות מתמטיות, מאריתמטיקה בסיסית ועד אלגברה מתקדמת, סטטיסטיקה ואלגברה ליניארית. המאפיינים העיקריים כוללים:

- **פונקציונליות מקיפה**: מספקת מערך נרחב של פונקציות לכל סוגי הפעולות המתמטיות.
- **ניתוח ביטויים גמיש**: מסוגלת לנתח ולהעריך ביטויים מתמטיים בפורמט מחרוזת (string).
- **חישובים מתקדמים**: תומכת בפעולות מטריצה, חישובים סימבוליים וניתוח סטטיסטי, לצד פונקציות מתמטיות מתקדמות אחרות.
- **יכולת הרחבה**: ניתן להרחיב אותה באמצעות פונקציות וקבועים מותאמים אישית כדי לענות על צרכים ספציפיים.

## יישום ב-NocoBase

באמצעות שילוב של Formula.js ו-Math.js, משתמשי NocoBase יכולים:

- **לבצע מגוון רחב של חישובים**: מאריתמטיקה פשוטה ועד פעולות מתמטיות מורכבות, תוך שיפור יכולות החישוב של היישומים שלהם.
- **לפתח יישומים חזקים ויעילים**: לבנות יישומים הדורשים חישובים מתמטיים מדויקים ועוצמתיים, המבטיחים אמינות וביצועים.
- **להגביר את הפרודוקטיביות**: להשתמש בפונקציות מובנות מראש כדי לצמצם את הצורך בכתיבת קוד מותאם אישית נרחב, ובכך לשפר את יעילות הפיתוח.

ספריות אלו הן מרכזיות בפלטפורמת ה-no-code של NocoBase, ומאפשרות למשתמשים ליצור לוגיקת חישוב גמישה וניתנת להתאמה אישית מבלי לצלול לתכנות מורכב. גישה זו מפשטת את תהליך הפיתוח תוך שיפור הפונקציונליות והגמישות של היישומים.
