# IRepository

ממשק ה-`Repository` מגדיר סדרה של שיטות פעולה למודלים, שמטרתן להתאים את פעולות ה-CRUD של מקור הנתונים.

## API

### find()

מחזירה רשימה של מודלים התואמים לפרמטרי השאילתה.

#### חתימה

- `find(options?: any): Promise<IModel[]>`

### findOne()

מחזירה מודל התואם לפרמטרי השאילתה. אם קיימים מספר מודלים תואמים, יוחזר רק הראשון.

#### חתימה

- `findOne(options?: any): Promise<IModel>`

### count()

מחזירה את מספר המודלים התואמים לפרמטרי השאילתה.

#### חתימה

- `count(options?: any): Promise<Number>`

### findAndCount()

מחזירה את רשימת המודלים ומספרם, התואמים לפרמטרי השאילתה.

#### חתימה

- `findAndCount(options?: any): Promise<[IModel[], Number]>`

### create()

יוצרת אובייקט נתונים של מודל.

#### חתימה

- `create(options: any): void`

### update()

מעדכנת אובייקט נתונים של מודל בהתבסס על תנאי השאילתה.

#### חתימה

- `update(options: any): void`

### destroy()

מוחקת אובייקט נתונים של מודל בהתבסס על תנאי השאילתה.

#### חתימה

- `destroy(options: any): void`
