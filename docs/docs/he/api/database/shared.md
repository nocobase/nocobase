## פרמטרים

| שם פרמטר                 | סוג          | ערך ברירת מחדל | תיאור                                            |
| ---------------------- | ------------- | ------ | ----------------------------------------------- |
| `options.values`       | `M`           | `{}`   | אובייקט הנתונים להוספה                           |
| `options.whitelist?`   | `string[]`    | -      | רשימת היתר (whitelist) של שדות עבור `values`. רק השדות ברשימה יישמרו. |
| `options.blacklist?`   | `string[]`    | -      | רשימת חסימה (blacklist) של שדות עבור `values`. השדות ברשימה לא יישמרו.   |
| `options.transaction?` | `Transaction` | -      | טרנזקציה (transaction)                                            |
