## BelongsToRepository

הממשק שלו זהה ל-[HasOneRepository](./has-one-repository.md).
`BelongsToRepository` הוא `Repository` המיועד לטיפול במערכות יחסים מסוג `BelongsTo`. הוא מספק כמה שיטות נוחות לטיפול במערכות יחסים אלו.
