# מבוא ל-NocoBase

NocoBase היא פלטפורמת no-code בקוד פתוח המתמקדת ביכולת הרחבה (scalability). היא מאפשרת לבנות מערכות ניהול נתונים וכלים פנימיים במהירות וביעילות, ללא צורך בכתיבת קוד מורכב.

## מושגי מפתח

*   **אוסף (Collection):** משמש להגדרת מבנה הנתונים, השדות והקשרים ביניהם.
*   **תהליך עבודה (Workflow):** מנוע אוטומציה המאפשר להגדיר ולהפעיל לוגיקה עסקית באופן אוטומטי.
*   **מקור נתונים (Data Source):** מאפשר להתחבר למסדי נתונים קיימים או לנהל מקורות נתונים חדשים בתוך המערכת.
*   **תוסף (Plugin):** ארכיטקטורת המערכת מבוססת על תוספים, המאפשרים להרחיב ולהתאים אישית כל פונקציונליות ב-NocoBase.
