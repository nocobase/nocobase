## פרמטרים

מרבית הפרמטרים זהים לאלו של `find()`. ההבדל הוא ש-`findOne()` מחזירה רשומה בודדת בלבד, ולכן אין צורך בפרמטר `limit`, והוא תמיד `1` בעת השאילתה.
