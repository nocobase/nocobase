# מחזור חיים

כאן נארגן את ווים מחזור החיים של תוספים, הן בצד השרת והן בצד הלקוח, כדי לעזור למפתחים לרשום ולשחרר משאבים בצורה נכונה.

ניתן להשוות זאת למחזור החיים של FlowModel כדי להדגיש מושגים משותפים.

## תוכן מוצע

- קריאות חוזרות המופעלות כאשר תוספים מותקנים, מופעלים, מושבתים או מוסרים.
- עיתוי ההרכבה (mounting), העדכון וההשמדה (destruction) של רכיבים בצד הלקוח.
- המלצות לטיפול במשימות אסינכרוניות ובשגיאות בתוך מחזור החיים.
