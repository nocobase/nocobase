# ACL - בקרת הרשאות

:::tip

בפרק זה נלמד כיצד להשתמש ב-ACL (רשימת בקרת גישה) לניהול הרשאות בתוספי צד-לקוח. הנושאים שיכוסו כוללים:

- בדיקת הרשאות;
- כיצד לרנדר תוכן באופן דינמי בהתאם להרשאות;

בגרסאות עתידיות נוסיף דוגמאות מפורטות ושיטות עבודה מומלצות.

:::
