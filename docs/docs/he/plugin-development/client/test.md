# בדיקות

:::tip

פרק זה יציג כיצד לכתוב ולהריץ בדיקות עבור צד הלקוח (frontend) של תוספים. התוכן יכלול:
- הגדרת סביבת בדיקה ותלויותיה;
- כתיבת בדיקות יחידה (unit tests) באמצעות Vitest ו-Testing Library;
- כיצד לדמות את סביבת הריצה של NocoBase בצד הלקוח;
- הרצת בדיקות אוטומטית ב-CI.

בגרסאות עתידיות נוסיף דוגמאות מפורטות ושיטות עבודה מומלצות.

:::
