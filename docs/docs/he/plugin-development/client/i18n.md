# I18n בינאום

תוספים ב-NocoBase תומכים בבינאום (i18n) רב-לשוני גם בצד הלקוח (frontend) וגם בצד השרת (backend). באמצעות מנגנון אחיד, תוכלו ליישם בקלות תוכן רב-לשוני בתוספים.

## ניהול קבצי ריבוי שפות

קבצי ריבוי השפות של התוספים נשמרים באופן אחיד בתיקיית `src/locale`. מומלץ לתת להם שמות לפי השפה, לדוגמה:

```bash
|- /plugin-hello
  |- /src
    |- /locale
      |- en-US.json   # שפה אנגלית
      |- zh-CN.json   # שפה סינית
```

כל קובץ שפה מייצא אובייקט JSON המכיל את כל ערכי התרגום עבור אותה שפה, לדוגמה:

```json
// zh-CN.json
{
  "Hello": "你好",
  "World": "世界",
  "Enter your name": "请输入你的名字",
  "Your name is {{name}}": "你的名字是 {{name}}"
}
```

```json
// en-US.json
{
  "Hello": "Hello",
  "World": "World",
  "Enter your name": "Enter your name",
  "Your name is {{name}}": "Your name is {{name}}"
}
```

> כאשר מוסיפים קבצי שפה בפעם הראשונה, יש להפעיל מחדש את היישום כדי שהם ייכנסו לתוקף. ניתן לוודא שערכי התרגום נכנסו לתוקף באמצעות ה-API:
> http://localhost:13000/api/app:getLang?locale=zh-CN
