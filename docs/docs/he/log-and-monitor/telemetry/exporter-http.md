# יצואן טלמטריה: HTTP

## משתני סביבה

### TELEMETRY_METRIC_READER

סוג יצואן מדדי הטלמטריה.

```bash
TELEMETRY_METRIC_READER=http
```

### TELEMETRY_HTTP_URL

כתובת URL של HTTP לייצוא נתוני טלמטריה.

```bash
TELEMETRY_HTTP_URL=
```
