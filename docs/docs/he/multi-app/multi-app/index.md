# ניהול ריבוי יישומים

## סקירה כללית

NocoBase מאפשרת לנהל כמה יישומים מבודדים דרך נקודת כניסה אחת באמצעות AppSupervisor.


בתחילת הדרך מספיק יישום אחד. עם הגדילה עולים עומס, מורכבות ודרישת בידוד.


במצב זה כמה יישומים רצים באותה מופע NocoBase, עם DB נפרד לכל יישום אך תהליך וזיכרון משותפים.
