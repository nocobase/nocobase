# הטמעת NocoBase

## מבוא

תוסף זה משמש להטמעת דפי NocoBase באתרי אינטרנט או יישומים אחרים.

## התקנה

תוסף זה הוא תוסף מסחרי, ויש להעלות אותו ולהפעיל אותו דרך מנהל התוספים.

![20240323162741](https://static-docs.nocobase.com/20240323162741.png)

## מדריך שימוש

לאחר הפעלת תצורת הממשק, לחצו על "העתק קישור הטמעה" בתפריט ההגדרות של הדף (בפינה הימנית העליונה).

![](https://static-docs.nocobase.com/f11bd6d5e88d38731d7cd3cb149022c8.png)

לאחר הלחיצה, תקבלו קישור שניתן לפתוח בנפרד, לדוגמה, `https://xxx.com/embed/qs087rz4o2b`.

![](https://static-docs.nocobase.com/9d847805a00fd07372a474665ec0e968.png)

אם ברצונכם להטמיע דפי NocoBase באתרי אינטרנט או יישומים אחרים, נדרשת אימות משתמש, ויש לצרף את ה-`token` לקישור, לדוגמה, `https://xxx.com/embed/qs087rz4o2b?token=xxx`.
