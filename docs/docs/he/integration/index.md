# אינטגרציה

## סקירה כללית
NocoBase מציעה יכולות אינטגרציה מקיפות, המאפשרות חיבור חלק עם מערכות חיצוניות, שירותי צד שלישי ומגוון מקורות נתונים. באמצעות שיטות אינטגרציה גמישות, תוכלו להרחיב את הפונקציונליות של NocoBase כדי לענות על צרכים עסקיים מגוונים.

## שיטות אינטגרציה

### אינטגרציית API
NocoBase מספקת יכולות API עוצמתיות לאינטגרציה עם יישומים ושירותים חיצוניים:

![20250301003106](https://static-docs.nocobase.com/20250301003106.png)

- **[מפתחות API](/integration/api-keys/)**: השתמשו במפתחות API לאימות מאובטח ולגישה תכנותית למשאבי NocoBase.
- **[תיעוד API](/integration/api-doc/)**: תיעוד API מובנה לחקירה ובדיקה של נקודות קצה (endpoints).

### כניסה יחידה (SSO)
שלבו את NocoBase עם מערכות זהות ארגוניות לאימות מאוחד:

![auth_sso-2025-11-24-08-26-46](https://static-docs.nocobase.com/auth_sso-2025-11-24-08-26-46.png)

- **[אינטגרציית SSO](/integration/sso/)**: תמיכה באימות SAML, OIDC, CAS, LDAP ופלטפורמות צד שלישי.
- ניהול משתמשים ובקרת גישה מרכזיים.
- חווית אימות חלקה בין מערכות.

### אינטגרציית תהליך עבודה
חברו תהליכי עבודה של NocoBase למערכות חיצוניות:

![auth_sso-2025-11-24-12-02-01](https://static-docs.nocobase.com/auth_sso-2025-11-24-12-02-01.png)

- **[ווב-הוק של תהליך עבודה](/integration/workflow-webhook/)**: קבלו אירועים ממערכות חיצוניות כדי להפעיל תהליכי עבודה.
- **[בקשת HTTP של תהליך עבודה](/integration/workflow-http-request/)**: שלחו בקשות HTTP לממשקי API חיצוניים מתוך תהליכי עבודה.
- אוטומציה של תהליכים עסקיים בין פלטפורמות.

### מקורות נתונים חיצוניים
התחברו למסדי נתונים ומערכות נתונים חיצוניים:

![20240512085558](https://static-docs.nocobase.com/20240512085558.png)

- **[מסדי נתונים חיצוניים](/data-sources/)**: התחברו ישירות למסדי נתונים כמו MySQL, PostgreSQL, MariaDB, MSSQL, Oracle ו-KingbaseES.

- ממשק ניהול נתונים אחיד.

### תוכן מוטמע

## תרחישי אינטגרציה נפוצים

### אינטגרציה עם מערכות ארגוניות
- חברו את NocoBase למערכות ERP, CRM או מערכות ארגוניות אחרות.
- סנכרון נתונים דו-כיווני.
- אוטומציה של תהליכי עבודה בין מערכות.

### אינטגרציה עם שירותי צד שלישי
- שאילתת סטטוס תשלום משערי תשלום, שילוב שירותי הודעות או פלטפורמות ענן.
- ניצול ממשקי API חיצוניים להרחבת פונקציונליות.
- בניית אינטגרציות מותאמות אישית באמצעות ווב-הוקס ובקשות HTTP.

### אינטגרציית נתונים
- התחברות למספר מקורות נתונים.
- איגום נתונים ממערכות שונות.
- יצירת לוחות מחוונים ודוחות אחידים.

## שיקולי אבטחה
בעת שילוב NocoBase עם מערכות חיצוניות, קחו בחשבון את שיטות העבודה המומלצות הבאות לאבטחה:

1. **השתמשו ב-HTTPS**: השתמשו תמיד בחיבורים מוצפנים להעברת נתונים.
2. **אבטחו מפתחות API**: אחסנו מפתחות API באופן מאובטח וסובבו אותם באופן קבוע.
3. **עקרון ההרשאה המינימלית**: העניקו רק את ההרשאות הנדרשות לאינטגרציות.
4. **רישום ביקורת (Audit Logging)**: עקבו ותעדו פעילויות אינטגרציה.
5. **אימות נתונים**: ודאו את כל הנתונים המגיעים ממקורות חיצוניים.
