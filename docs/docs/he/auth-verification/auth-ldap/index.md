# אימות: LDAP

## מבוא

תוסף אימות: LDAP פועל לפי תקן פרוטוקול LDAP (Lightweight Directory Access Protocol), ומאפשר למשתמשים להתחבר ל-NocoBase באמצעות פרטי ההתחברות שלהם משרת LDAP.

## הגדרות

### הגדרות בסיסיות

<img src="https://static-docs.nocobase.com/202405101605728.png"/>

- הרשמה אוטומטית כאשר המשתמש אינו קיים - האם ליצור משתמש חדש באופן אוטומטי כאשר לא נמצא משתמש קיים תואם.
- LDAP URL - כתובת URL של שרת LDAP
- Bind DN - DN המשמש לבדיקת קישוריות השרת וחיפוש משתמשים
- Bind password - הסיסמה של Bind DN
- בדיקת חיבור - לחצו על הכפתור כדי לבדוק את קישוריות השרת ולאמת את ה-Bind DN.

### הגדרות חיפוש

<img src="https://static-docs.nocobase.com/202405101609984.png"/>

- Search DN - DN המשמש לחיפוש משתמשים
- מסנן חיפוש - תנאי סינון לחיפוש משתמשים, כאשר `{{account}}` מייצג את חשבון המשתמש המשמש להתחברות.
- טווח - `Base`, `One level`, `Subtree`, ברירת מחדל `Subtree`
- מגבלת גודל - גודל עמוד חיפוש

### מיפוי תכונות

<img src="https://static-docs.nocobase.com/202405101612814.png"/>

- השתמשו בשדה זה כדי לקשר את המשתמש - שדה המשמש לקישור למשתמשים קיימים. בחרו 'שם משתמש' אם חשבון ההתחברות הוא שם משתמש, או 'אימייל' אם הוא כתובת אימייל. ברירת המחדל היא שם משתמש.
- מיפוי תכונות - מיפוי של תכונות משתמש לשדות בטבלת המשתמשים של NocoBase.
