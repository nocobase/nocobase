## שכחתי סיסמה<Badge>v1.8.0+</Badge>

תכונת "שכחתי סיסמה" מאפשרת למשתמשים לאפס את סיסמתם באמצעות אימות דוא"ל, במקרה ששכחו אותה.

### הגדרות מנהל מערכת

1.  **הפעלת תכונת "שכחתי סיסמה"**

    בכרטיסייה "הגדרות" > "אימות" > "שכחתי סיסמה", סמנו את תיבת הסימון "הפעלת תכונת 'שכחתי סיסמה'".

    ![20250423071957_rec_](https://static-docs.nocobase.com/20250423071957_rec_.gif)

2.  **הגדרת ערוץ התראות**

    בחרו ערוץ התראות בדוא"ל (כרגע נתמך רק דוא"ל). אם אין ערוצי התראות זמינים, עליכם להוסיף אחד תחילה.

    ![20250423072225_rec_](https://static-docs.nocobase.com/20250423072225_rec_.gif)

3.  **הגדרת דוא"ל איפוס סיסמה**

    התאימו אישית את נושא ותוכן הדוא"ל, עם תמיכה בפורמט HTML או טקסט רגיל. ניתן להשתמש במשתנים הבאים:
    - משתמש נוכחי
    - הגדרות מערכת
    - קישור לאיפוס סיסמה
    - תוקף קישור איפוס (דקות)

    ![20250427170047](https://static-docs.nocobase.com/20250427170047.png)

4.  **הגדרת תוקף קישור האיפוס**

    הגדירו את משך התוקף (בדקות) של קישור האיפוס. ברירת המחדל היא 120 דקות.

    ![20250423073557](https://static-docs.nocobase.com/20250423073557.png)

### תהליך שימוש למשתמש

1.  **יצירת בקשה לאיפוס סיסמה**

    בעמוד ההתחברות, לחצו על הקישור "שכחתי סיסמה" (מנהל המערכת צריך לאפשר את התכונה תחילה) כדי לעבור לעמוד "שכחתי סיסמה".

    ![20250421103458_rec_](https://static-docs.nocobase.com/20250421103458_rec_.gif)

    הזינו את כתובת הדוא"ל הרשומה ולחצו על הכפתור "שלח דוא"ל איפוס".

    ![20250421113442_rec_](https://static-docs.nocobase.com/20250421113442_rec_.gif)

2.  **איפוס סיסמה**

    המשתמש יקבל דוא"ל המכיל קישור לאיפוס. לחצו על הקישור כדי לפתוח עמוד שבו תוכלו להגדיר סיסמה חדשה.

    ![20250421113748](https://static-docs.nocobase.com/20250421113748.png)

    לאחר ההגדרה, המשתמש יוכל להתחבר למערכת עם הסיסמה החדשה.

### הערות

-   לקישור האיפוס יש מגבלת זמן; כברירת מחדל הוא תקף למשך 120 דקות מרגע יצירתו (ניתן להגדרה על ידי מנהל המערכת).
-   ניתן להשתמש בקישור פעם אחת בלבד, ולאחר השימוש הוא הופך ללא תקף באופן מיידי.
-   אם המשתמש לא קיבל את דוא"ל האיפוס, אנא ודאו שכתובת הדוא"ל נכונה, או בדקו את תיקיית דואר הזבל.
-   מנהל המערכת צריך לוודא שתצורת שרת הדוא"ל נכונה, כדי להבטיח שדוא"ל האיפוס יישלח בהצלחה.
