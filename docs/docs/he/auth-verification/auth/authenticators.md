# מנהלי אימות

## ניהול אימות משתמשים

כאשר תוסף אימות המשתמשים מותקן, הוא יאתחל שיטת אימות `סיסמה` המבוססת על שם המשתמש והדוא"ל שלו.

![](https://static-docs.nocobase.com/66eaa9d5421c9cb713b117366bd8a5d5.png)

## הפעלת מנהלי אימות

![](https://static-docs.nocobase.com/7f1fb8f8ca5de67ffc68eff0a65848f5.png)

רק סוגי אימות מופעלים יוצגו בדף ההתחברות.

![](https://static-docs.nocobase.com/8375a36ef98417af0f0977f1e07345dd.png)

## סוגי אימות משתמשים

![](https://static-docs.nocobase.com/da42343ebe470cbf7be4b12e4.png)

על ידי הוספת סוגים שונים של מנהלי אימות, תוכלו להוסיף למערכת שיטות אימות מתאימות.

בנוסף לסוגי האימות המסופקים על ידי תוספים קיימים, מפתחים יכולים גם להרחיב סוגי אימות משתמשים. לפרטים נוספים, עיינו ב[מדריך למפתחים](./dev/).
