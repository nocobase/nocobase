# אימות: מאמת TOTP

## מבוא

מאמת ה-TOTP מאפשר למשתמשים לחבר כל מאמת התואם למפרט TOTP (סיסמה חד-פעמית מבוססת-זמן) (<a href="https://www.rfc-editor.org/rfc/rfc6238" target="_blank">RFC-6238</a>), ולבצע אימות זהות באמצעות סיסמה חד-פעמית מבוססת-זמן (TOTP).

## הגדרות מנהל מערכת

עברו לדף ניהול האימות.

![](https://static-docs.nocobase.com/202502271726791.png)

הוספה - מאמת TOTP

![](https://static-docs.nocobase.com/202502271745028.png)

מלבד מזהה ייחודי וכותרת, מאמת ה-TOTP אינו דורש הגדרות נוספות.

![](https://static-docs.nocobase.com/202502271746034.png)

## חיבור משתמש

לאחר הוספת המאמת, משתמשים יכולים לחבר את מאמת ה-TOTP באזור ניהול האימות האישי שלהם.

![](https://static-docs.nocobase.com/202502272252324.png)

:::warning
התוסף אינו מספק כרגע מנגנון קודי שחזור. לאחר חיבור מאמת ה-TOTP, מומלץ למשתמשים לשמור עליו היטב. במקרה של אובדן מקרי של המאמת, ניתן להשתמש בשיטת אימות חלופית כדי לאמת את הזהות, ולאחר מכן לנתק אותו ולחבר מחדש באמצעות שיטת אימות אחרת.
:::
