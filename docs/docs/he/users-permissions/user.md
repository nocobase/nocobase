# משתמשים

## ניהול משתמשים

תוסף זה מספק ממשק לניהול משתמשים במרכז התצורה. לאחר אתחול היישום, נוצר חשבון סופר-אדמין שאינו ניתן למחיקה. למנהל המערכת יש הרשאות Root והוא יכול לגשת לכל המשאבים.

![](https://static-docs.nocobase.com/44bf40f56b45d4dd96c424fb08082cf6.png)

בנוסף, ניתן גם להוסיף בלוקים שונים מתוך אוסף המשתמשים כדי לנהל משתמשים, לדוגמה, בלוק טבלה:
