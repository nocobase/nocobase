# סנכרון נתוני משתמשים

## מבוא

תכונה זו מאפשרת לכם לרשום ולנהל מקורות סנכרון נתוני משתמשים. כברירת מחדל, מסופק API HTTP, אך ניתן לתמוך במקורות נתונים נוספים באמצעות **תוספים**. היא תומכת בסנכרון נתונים ל**אוספי** ה**משתמשים** וה**מחלקות** כברירת מחדל, עם אפשרות להרחיב את הסנכרון למשאבי יעד אחרים באמצעות **תוספים**.

## ניהול מקורות נתונים וסנכרון

![](https://static-docs.nocobase.com/202412041043465.png)

:::info
אם לא מותקנים **תוספים** המספקים מקורות סנכרון נתוני משתמשים, ניתן לסנכרן נתוני משתמשים באמצעות API HTTP. עיינו ב[מקור נתונים - API HTTP](./sources/api.md).
:::

## הוספת מקור נתונים

לאחר שתתקינו **תוסף** המספק מקור סנכרון נתוני משתמשים, תוכלו להוסיף את **מקור הנתונים** המתאים. רק **מקורות נתונים** מופעלים יציגו את כפתורי "סנכרון" ו"משימה".

> דוגמה: WeCom

![](https://static-docs.nocobase.com/202412041053785.png)

## סנכרון נתונים

לחצו על כפתור "סנכרון" כדי להתחיל בסנכרון נתונים.

![](https://static-docs.nocobase.com/202412041055022.png)

לחצו על כפתור "משימה" כדי לצפות בסטטוס הסנכרון. לאחר סנכרון מוצלח, תוכלו לצפות בנתונים ברשימות המשתמשים והמחלקות.

![](https://static-docs.nocobase.com/202412041202337.png)

עבור משימות סנכרון שנכשלו, תוכלו ללחוץ על "נסה שוב".

![](https://static-docs.nocobase.com/202412041058337.png)

במקרה של כשלים בסנכרון, תוכלו לאתר את הסיבה באמצעות יומני המערכת. בנוסף, רשומות סנכרון גולמיות נשמרות בספריית `user-data-sync` תחת תיקיית יומני היישום.

![](https://static-docs.nocobase.com/202412041205655.png)
