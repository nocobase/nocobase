# ניהול תפקידי מחלקות

ניתן לשייך תפקיד אחד או יותר למחלקה, וכך חברי המחלקה יקבלו את התפקידים הללו.

ישנן שתי דרכים לשייך תפקידים למחלקה:

הדרך הראשונה היא לבחור תפקיד בשדה התפקיד בטופס עריכת המחלקה.

![](https://static-docs.nocobase.com/70f77bb89aa1fb415c152a51a51cc23b.png)

הדרך השנייה היא להוסיף מחלקות לתפקיד המתאים בניהול התפקידים.

![](https://static-docs.nocobase.com/f2a7bec937cf2f179ce868a92b98416d.png)

לא ניתן לבחור מחלקות שכבר משויכות לתפקיד זה.

![](https://static-docs.nocobase.com/be10299893581e1f97a4e01ddd5c7e59.png)
