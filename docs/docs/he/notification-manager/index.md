# מנהל התראות

## מבוא

מנהל ההתראות הוא שירות מרכזי המשלב מספר ערוצי התראה. הוא מספק ממשק אחיד להגדרת ערוצים, ניהול שליחה ותיעוד יומנים, ותומך בהרחבה גמישה לערוצים נוספים.

![20240928112556](https://static-docs.nocobase.com/20240928112556.png)

- חלק סגול: מנהל ההתראות מספק שירות מקיף הכולל הגדרת ערוצים ותיעוד יומנים, עם אפשרות להרחבה לערוצי התראה נוספים.
- חלק ירוק: הודעות בתוך האפליקציה (In-App Message), ערוץ מובנה המאפשר למשתמשים לקבל התראות ישירות בתוך האפליקציה.
- חלק אדום: דוא"ל (Email), ערוץ הניתן להרחבה המאפשר למשתמשים לקבל התראות באמצעות דוא"ל.

## ניהול ערוצים

![20240928181752](https://static-docs.nocobase.com/20240928181752.png)

הערוצים הנתמכים כרגע הם:

- [הודעות בתוך האפליקציה](/notification-manager/notification-in-app-message)
- [דוא"ל](/notification-manager/notification-email) (באמצעות העברת SMTP מובנית)

ניתן גם להרחיב לערוצים נוספים, עיינו בתיעוד [הרחבת ערוצים](/notification-manager/development/extension).
