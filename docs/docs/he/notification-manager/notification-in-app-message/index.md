# התראה: הודעה פנימית

## מבוא

מאפשר למשתמשים לקבל התראות הודעה בזמן אמת ישירות בתוך אפליקציית NocoBase.

## התקנה

תוסף זה הוא תוסף מובנה, ולכן אין צורך בהתקנה נוספת.

## הוספת ערוץ הודעות פנימיות

עברו לניהול התראות, לחצו על כפתור 'הוספה', ובחרו 'הודעה פנימית'.

![2024-11-08-08-34-32-20241108083431](https://static-docs.nocobase.com/2024-11-08-08-34-32-20241108083431.png)

הערוץ החדש יופיע כעת ברשימה.

![2024-11-08-08-34-52-20241108083452](https://static-docs.nocobase.com/2024-11-08-08-34-52-20241108083452.png)

## דוגמה לתרחיש שימוש

כדי לעזור לכם להבין טוב יותר כיצד להשתמש בהודעות פנימיות, הנה דוגמה ל"מעקב אחר לידים שיווקיים".

דמיינו שצוותכם מנהל קמפיין שיווקי חשוב, שמטרתו לעקוב אחר תגובות וצרכים של לקוחות פוטנציאליים. באמצעות הודעות פנימיות, תוכלו:

**יצירת ערוץ התראות**: ראשית, בניהול התראות, הגדירו ערוץ הודעות פנימיות בשם "Marketing Clue" כדי לוודא שחברי הצוות יוכלו לזהות בבירור את מטרתו של ערוץ זה.

![2024-11-08-08-34-32-20241108083431](https://static-docs.nocobase.com/2024-11-08-08-34-32-20241108083431.png)

**הגדרת תהליך עבודה**: צרו **תהליך עבודה** שיפעיל אוטומטית התראה כאשר נוצר ליד שיווקי חדש. תוכלו להוסיף צומת התראה ל**תהליך העבודה**, לבחור את ערוץ "Marketing Clue" שיצרתם, ולהגדיר את תוכן ההודעה לפי הצורך. לדוגמה:

![image-1-2024-10-27-14-07-17](https://static-docs.nocobase.com/image-1-2024-10-27-14-07-17.png)

**קבלת התראות בזמן אמת**: ברגע ש**תהליך העבודה** מופעל, כל הצוותים הרלוונטיים יקבלו התראות בזמן אמת, מה שיבטיח שהצוות יוכל להגיב ולפעול במהירות.

![image-2-2024-10-27-14-07-22](https://static-docs.nocobase.com/image-2-2024-10-27-14-07-22.png)

**ניהול ומעקב אחר הודעות**: הודעות פנימיות מקובצות לפי שם הערוץ שלהן. תוכלו לסנן הודעות לפי מצב 'נקרא' ו'לא נקרא' כדי לצפות במהירות במידע חשוב. לחיצה על כפתור 'הצג' תעביר אתכם לדף הקישור שהוגדר לצורך טיפול נוסף בעניין.

![20241027140648-2024-10-27-14-06-51-2024-10-29-13-26-41](https://static-docs.nocobase.com/20241027140648-2024-10-27-14-06-51-2024-10-29-13-26-41.png)
