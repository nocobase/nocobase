# תחילת עבודה מהירה

## מבוא

לפני השימוש ב"עובד AI" (AI Employee), עליכם לחבר שירות LLM מקוון. NocoBase תומכת כיום בשירותי LLM מקוונים מובילים כמו OpenAI, Gemini, Claude, DepSeek, Qwen ועוד.
בנוסף לשירותי LLM מקוונים, NocoBase תומכת גם בחיבור למודלים מקומיים של Ollama.

## הגדרת שירות LLM

עברו לדף הגדרות ה**תוסף** של AI Employee, לחצו על לשונית `LLM service` כדי להיכנס לדף ניהול שירותי LLM.

![20251021213122](https://static-docs.nocobase.com/20251021213122.png)

רחפו עם העכבר מעל כפתור `Add New` בפינה הימנית העליונה של רשימת שירותי LLM, ובחרו את שירות ה-LLM שבו תרצו להשתמש.

![20251021213358](https://static-docs.nocobase.com/20251021213358.png)

ניקח את OpenAI כדוגמה: בחלון הקופץ, הזינו `title` (כותרת) שקל לזכור, לאחר מכן הזינו את ה-`API key` שקיבלתם מ-OpenAI, ולחצו על `Submit` (שלח) כדי לשמור. בכך תושלם הגדרת שירות ה-LLM.

את השדה `Base URL` ניתן להשאיר ריק בדרך כלל. אם אתם משתמשים בשירות LLM של צד שלישי התואם לממשק ה-API של OpenAI, אנא מלאו את ה-`Base URL` המתאים.

![20251021214549](https://static-docs.nocobase.com/20251021214549.png)

## בדיקת זמינות

בדף הגדרות שירות ה-LLM, לחצו על כפתור `Test flight`, הזינו את שם המודל שבו תרצו להשתמש, ולחצו על כפתור `Run` (הפעל) כדי לבדוק אם שירות ה-LLM והמודל זמינים.

![20251021214903](https://static-docs.nocobase.com/20251021214903.png)
