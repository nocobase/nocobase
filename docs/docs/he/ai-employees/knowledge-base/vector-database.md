# מסד נתונים וקטורי

## מבוא

במאגר ידע, מסד הנתונים הווקטורי שומר את מסמכי מאגר הידע לאחר שעברו וקטוריזציה. מסמכים וקטוריים אלו משמשים כמעין אינדקס למסמכים.

כאשר אחזור RAG מופעל בשיחה עם סוכן AI, הודעת המשתמש עוברת וקטוריזציה. לאחר מכן, מסד הנתונים הווקטורי מאחזר קטעים ממסמכי מאגר הידע כדי למצוא התאמות לפסקאות רלוונטיות ולטקסט המקורי של המסמך.

נכון לעכשיו, התוסף של מאגר הידע ב-AI תומך באופן מובנה רק ב-PGVector, שהוא תוסף למסד נתונים של PostgreSQL.

## ניהול מסד נתונים וקטורי

עברו לדף התצורה של תוסף סוכן ה-AI, לחצו על לשונית `Vector store`, ובחרו ב-`Vector database` כדי להיכנס לדף ניהול מסד הנתונים הווקטורי.

![20251022233704](https://static-docs.nocobase.com/20251022233704.png)

לחצו על כפתור `Add new` בפינה הימנית העליונה כדי להוסיף חיבור חדש למסד נתונים וקטורי מסוג `PGVector`:

- בשדה הקלט `Name`, הזינו את שם החיבור.
- בשדה הקלט `Host`, הזינו את כתובת ה-IP של מסד הנתונים הווקטורי.
- בשדה הקלט `Port`, הזינו את מספר הפורט של מסד הנתונים הווקטורי.
- בשדה הקלט `Username`, הזינו את שם המשתמש של מסד הנתונים הווקטורי.
- בשדה הקלט `Password`, הזינו את הסיסמה של מסד הנתונים הווקטורי.
- בשדה הקלט `Database`, הזינו את שם מסד הנתונים.
- בשדה הקלט `Table name`, הזינו את שם הטבלה, שבו יש להשתמש בעת יצירת טבלה חדשה לאחסון נתונים וקטוריים.

לאחר הזנת כל המידע הנדרש, לחצו על כפתור `Test` כדי לבדוק אם שירות מסד הנתונים הווקטורי זמין, ולחצו על כפתור `Submit` כדי לשמור את פרטי החיבור.
