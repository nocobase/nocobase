# עובד AI · Viz: אנליסט תובנות

> צרו גרפים ותובנות בלחיצת כפתור, ותנו לנתונים לדבר בעד עצמם.

## 1. מי זה Viz

**Viz** הוא **אנליסט תובנות AI** מובנה.
הוא יודע לקרוא את הנתונים בעמוד הנוכחי שלכם (כמו לידים, הזדמנויות, חשבונות), ולייצר באופן אוטומטי גרפי מגמה, גרפי השוואה, כרטיסי KPI ומסקנות תמציתיות, מה שהופך את ניתוח הנתונים העסקיים לקל ואינטואיטיבי.

הוא אינו רק כלי דיווח קר, אלא אנליסט שמסוגל להבין שאלות ולספר סיפורים.

> 💡 רוצים לדעת "למה המכירות ירדו לאחרונה"?
> פשוט אמרו ל-Viz משפט אחד, והוא יוכל לומר לכם היכן התרחשה הירידה, מהן הסיבות האפשריות, ומה תוכלו לעשות הלאה.

## 2. מה אתם יכולים לעשות עם Viz

| יכולת             | תיאור              | דוגמה            |
| -------------- | --------------- | ------------- |
| 📊 **יצירת גרפים אוטומטית**  | הדמיית נתונים בלחיצת כפתור, ללא צורך בכתיבת SQL | "צרו מגמת מכירות לחודש זה"    |
| 🔍 **גילוי שינויים וחריגות** | ניתוח הסיבות לעליות או ירידות      | "במה חודש זה טוב יותר מהחודש הקודם?"  |
| 🧭 **סיוע בקבלת החלטות**    | מתן המלצות מעשיות מבוססות נתונים    | "באיזה ערוץ כדאי להגדיל את התקציב?" |
| 🧩 **צבירת נקודות מבט על נתונים**  | השוואה בין ממדים מרובים כמו אזור, מוצר, מקור ועוד | "הציגו השוואת הכנסות לפי אזור"   |

בין אם מדובר בסקירות עסקיות חודשיות, ROI של ערוצים, או משפכי מכירה, Viz יכול לייצר גרפים ופרשנויות בתוך שניות.

## 3. אופן השימוש

### 3.1 נקודות כניסה בעמוד

*   **כפתור בפינה הימנית העליונה (מומלץ)**
    בפינה הימנית העליונה של עמודים כמו לידים, הזדמנויות וחשבונות, לחצו על **אייקון Viz** כדי לבחור משימות מוגדרות מראש, כגון:

    *   המרת שלבים ומגמות
    *   השוואת ערוצי מקור
    *   ניתוח סקירה חודשית

    ![דוגמה בעמוד לידים](https://static-docs.nocobase.com/00_QuickStart_cn-2025-09-29-16-46-00.png)

*   **פאנל גלובלי בפינה הימנית התחתונה**
    בכל עמוד שבו אתם נמצאים, תוכלו להעלות את פאנל ה-AI הגלובלי ולדבר ישירות עם Viz:

    ```
    Analyze sales changes in the last 90 days
    ```

    Viz ישתמש אוטומטית בהקשר הנתונים של העמוד שבו אתם נמצאים.

### 3.2 שיטת אינטראקציה

Viz תומך בשאלות בשפה טבעית, ומסוגל להבין גם שאלות המשך מרובות.
דוגמה:

```
Hi Viz, generate this month's lead trend.
```

```
Show me the performance of third-party channels only.
```

```
Then which region has the fastest growth?
```

כל שאלת המשך תעמיק את הניתוח בהתבסס על תוצאות הניתוח הקודם, ללא צורך בהזנה חוזרת של תנאי נתונים.

## 4. תרחישי ניתוח נפוצים

| תרחיש       | מה אתם רוצים לדעת      | פלט של Viz             |
| -------- | ------------ | ------------------- |
| **סקירה חודשית** | במה חודש זה טוב יותר מהחודש הקודם?   | כרטיס KPI + גרף מגמה + שלוש הצעות לשיפור |
| **פירוק צמיחה** | האם עליית ההכנסות נובעת משינוי בכמות או במחיר? | גרף פירוק גורמים + טבלת השוואה         |
| **ניתוח ערוצים** | באיזה ערוץ כדאי להמשיך להשקיע? | גרף ROI + עקומת שימור + המלצות    |
| **ניתוח משפך** | היכן התנועה נתקעת?     | גרף משפך + הסבר על צווארי בקבוק          |
| **שימור לקוחות** | אילו לקוחות הם בעלי הערך הרב ביותר?    | גרף פילוח RFM + עקומת שימור       |
| **הערכת מבצעים** | מה הייתה יעילות המבצע הגדול?     | גרף השוואה + ניתוח גמישות מחיר        |

> 📈 כל הגרפים נוצרים בפורמט ECharts תקף, עם נקודת מפתח אחת לכל גרף, ומלווים במסקנה תמציתית.
> אם הנתונים אינם מספיקים, Viz יציין זאת ישירות, ולא ימציא תוצאות.

## 5. טיפים לשיחה עם Viz

| פעולה         | השפעה                  |
| ---------- | ------------------- |
| ✅ ציינו טווח זמן   | "30 הימים האחרונים", "חודש קודם לעומת חודש זה" לדיוק רב יותר |
| ✅ ציינו ממדים     | "לפי אזור/ערוץ/מוצר" עוזר ליישר נקודות מבט  |
| ✅ התמקדו במגמות, לא בפרטים | Viz מצטיין בזיהוי כיוון השינוי והסיבות המרכזיות   |
| ✅ השתמשו בשפה טבעית   | אין צורך בתחביר פקודות, פשוט שאלו כאילו אתם משוחחים  |

## 6. למי Viz מתאים במיוחד?

| תפקיד        | שימוש                |
| --------- | ----------------- |
| **מנהל מכירות**  | צפייה בשיעורי המרה של שלבים, ביצועי ערוצים, וביצועי צוות |
| **איש שיווק**  | ניתוח ROI של קמפיינים, יעילות מבצעים, ושימור לקוחות |
| **אנליסט תפעול** | שליפת נתונים מהירה, גילוי חריגות, אימות השערות    |
| **הנהלה**   | הבנת מצב העסק במבט חטוף, קבלת אותות להחלטות   |

## 7. הצעות לשימוש

1.  **התחילו עם משימות מוגדרות מראש**
    הדמו הרשמי כולל משימות נפוצות מובנות, כך שתוכלו לחוות את התוצאות ישירות ללא צורך בהנחיות.
    לדוגמה: עמוד לידים → לחצו על **Viz → המרת שלבים ומגמות**

2.  **צפו בסגנון הפלט**
    לכל נקודת ניתוח יש גרף נפרד ותיאור קצר.
    גרפים ברורים וטקסט תמציתי הם הפלט הסטנדרטי של Viz.

3.  **שאלו שאלות המשך בהדרגה**
    לאחר קריאת דוח הניתוח, המשיכו לשאול "למה" ו"איך לשפר", ו-Viz ימשיך באופן אוטומטי.

## 8. סיכום

*   Viz = עוזר התובנות שלכם מבוסס נתונים
*   אין צורך לכתוב SQL, וגם לא להגדיר גרפים
*   משפט אחד בשפה טבעית, ותקבלו דוח ניתוח
*   כל המסקנות מבוססות על נתונים אמיתיים, ברורות ואמינות

> התחילו עם **לידים → Viz → המרת שלבים ומגמות**,
> לראות את הגרף הראשון זו נקודת ההתחלה הטובה ביותר להבנת עובד ה-AI הזה.
