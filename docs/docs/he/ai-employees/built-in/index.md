# עובדי AI מובנים

## סקירה כללית

NocoBase כולל את עובדי ה-AI המובנים הבאים. הם מצוידים במערך שלם של מיומנויות, כלים ובסיסי ידע. כל שעליכם לעשות הוא להגדיר עבורם LLM, והם יהיו מוכנים להתחיל לעבוד.

- `Orin`: מומחה למידול נתונים
- `Avery`: ממלא טפסים
- `Viz`: אנליסט תובנות
- `Lexi`: מתרגם
- `Nathan`: מהנדס קוד פרונטאנד
- `Cole`: עוזר NocoBase
- `Vera`: אנליסט מחקר
- `Dex`: מארגן נתונים
- `Ellis`: מומחה לדוא"ל

## כיצד להפעיל

עברו לדף הגדרות ה-`תוסף` של עובדי ה-AI, לחצו על לשונית `AI employees` כדי להיכנס לדף ניהול עובדי ה-AI.

תוכלו לראות שהמערכת כוללת מספר עובדי AI מובנים, אך אף אחד מהם אינו מופעל. בשלב זה, לא תוכלו לשתף פעולה עם עובדי AI אלה בדף היישום.

![20251022121248](https://static-docs.nocobase.com/20251022121248.png)

בחרו את עובד ה-AI המובנה שברצונכם להפעיל, לחצו על כפתור `Edit` כדי להיכנס לדף עריכת עובד ה-AI.

ראשית, בלשונית `Profile`, הפעילו את מתג `Enable`.

![20251022121546](https://static-docs.nocobase.com/20251022121546.png)

לאחר מכן, בלשונית `Model settings`, הגדירו את המודל עבור עובד ה-AI המובנה:

- בחרו את שירות ה-LLM שיצרתם בניהול שירותי LLM;
- הזינו את שם המודל הגדול שבו ברצונכם להשתמש

![20251022121729](https://static-docs.nocobase.com/20251022121729.png)

### השלמת ההפעלה

לאחר הגדרת המודל עבור עובד ה-AI המובנה, לחצו על כפתור `Submit` כדי לשמור את השינויים.

לאחר מכן, תוכלו לראות את עובד ה-AI המובנה הזה בכפתור ההפעלה המהירה של עובדי ה-AI, בפינה הימנית התחתונה של הדף.

![20251022121951](https://static-docs.nocobase.com/20251022121951.png)

### הערה

חלק מעובדי ה-AI המובנים לא יופיעו ברשימת עובדי ה-AI שבפינה הימנית התחתונה לאחר הפעלתם. לדוגמה, Orin יופיע רק בדף הגדרות הנתונים הראשי; Nathan יופיע רק בעורך ה-JS.
