# סוכן AI · Dex: מומחה לארגון נתונים

> חלץ ומלא טפסים במהירות, והפוך מידע מבולגן למובנה.

`Dex` הוא מומחה לארגון נתונים. הוא מחלץ מידע נדרש מנתונים או קבצים לא מובנים, מארגן אותו למידע מובנה, ויכול להשתמש בכלים כדי למלא טפסים.

![20251022155210](https://static-docs.nocobase.com/20251022155210.png)

## שלבים

בעמוד הטופס, הפעילו את `Dex` כדי לפתוח את חלון השיחה.

![20251022155316](https://static-docs.nocobase.com/20251022155316.png)

לחצו על `Add work context` בתיבת הקלט ובחרו ב-`Pick block`. העמוד ייכנס למצב בחירת בלוק.

![20251022155414](https://static-docs.nocobase.com/20251022155414.png)

בחרו את בלוק הטופס בעמוד.

![20251022155523](https://static-docs.nocobase.com/20251022155523.png)

הזינו בתיבת הדו-שיח את הנתונים שברצונכם ש-`Dex` ימלא.

![20251022155555](https://static-docs.nocobase.com/20251022155555.png)

לאחר השליחה, `Dex` יבנה את הנתונים וישתמש בכישוריו כדי לעדכן את הנתונים בטופס הנבחר.

![20251022155733](https://static-docs.nocobase.com/20251022155733.png)
