# התחלה מהירה

בואו נשלים הגדרה בסיסית של עובד AI ב-5 דקות.

## התקנת התוסף

עובדי AI הם חלק מובנה ב-NocoBase (התוסף `@nocobase/plugin-ai`), ולכן אין צורך בהתקנה נפרדת.

## הגדרת מודלים

ניתן להגדיר שירותי LLM דרך אחת משתי נקודות הכניסה הבאות:

1. כניסה דרך הניהול: `הגדרות מערכת -> עובדי AI -> שירות LLM` (System Settings -> AI Employees -> LLM service).
2. קיצור דרך בממשק המשתמש: בחלונית הצ'אט של ה-AI, השתמשו ב-`Model Switcher` כדי לבחור מודל, ולאחר מכן לחצו על קיצור הדרך "הוסף שירות LLM" כדי לעבור ישירות להגדרות.

![quick-start-model-switcher-add-llm-service.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/quick-start-model-switcher-add-llm-service.png)

בדרך כלל עליכם לאשר:
1. בחירת ספק (Provider).
2. הזנת מפתח API (API Key).
3. הגדרת `Enabled Models`; ניתן פשוט להשתמש ב-Recommend כברירת מחדל.

## הפעלת עובדים מובנים

עובדי ה-AI המובנים מופעלים כברירת מחדל, ובדרך כלל אין צורך להפעיל אותם אחד אחד באופן ידני.

אם עליכם לשנות את זמינותם (הפעלה/השבתה של עובד מסוים), ניתן לעדכן את מתג ה-`Enabled` בדף הרשימה תחת `הגדרות מערכת -> עובדי AI`.

![ai-employee-list-enable-switch.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employee-list-enable-switch.png)

## תחילת עבודה משותפת

בדף האפליקציה, רחפו עם העכבר מעל קיצור הדרך בפינה הימנית התחתונה ובחרו עובד AI.
![ai-employees-entry-bottom-right.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employees-entry-bottom-right.png)

לחצו כדי לפתוח את תיבת הדו-שיח של ה-AI:

![chat-footer-employee-switcher-and-model-switcher.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/chat-footer-employee-switcher-and-model-switcher.png)

תוכלו גם:  
* להוסיף בלוקים
* להוסיף קבצים מצורפים
* להפעיל חיפוש באינטרנט
* להחליף עובד AI
* לבחור מודלים

העובדים יכולים גם לקבל באופן אוטומטי את מבנה הדף כהקשר (context). לדוגמה, Dex בבלוק של טופס יכול לקרוא את מבנה השדות בטופס ולהשתמש במיומנויות המתאימות כדי לבצע פעולות בדף.

## משימות מהירות 

ניתן להגדיר מראש משימות נפוצות עבור כל עובד AI במיקום הנוכחי, כך שתוכלו להתחיל לעבוד בלחיצה אחת, בצורה מהירה ונוחה.

## סקירת עובדים מובנים

NocoBase מספקת מספר עובדי AI מובנים המיועדים לתרחישים שונים.

כל מה שעליכם לעשות הוא:

1. להגדיר שירותי LLM.
2. להתאים את סטטוס ההפעלה של העובדים לפי הצורך (מופעלים כברירת מחדל).
3. לבחור מודל בשיחה ולהתחיל לשתף פעולה.

| שם העובד | הגדרת תפקיד | יכולות ליבה |
| :--- | :--- | :--- |
| **Cole** | עוזר NocoBase | שאלות ותשובות על השימוש במוצר, אחזור מסמכים |
| **Ellis** | מומחה דואר אלקטרוני | כתיבת מיילים, יצירת סיכומים, הצעות לתגובה |
| **Dex** | מומחה לארגון נתונים | תרגום שדות, עיצוב (formatting), חילוץ מידע |
| **Viz** | אנליסט תובנות | תובנות נתונים, ניתוח מגמות, פרשנות של מדדי מפתח |
| **Lexi** | עוזר תרגום | תרגום רב-לשוני, סיוע בתקשורת |
| **Vera** | אנליסט מחקר | חיפוש באינטרנט, ריכוז מידע, מחקר מעמיק |
| **Dara** | מומחה לוויזואליזציה של נתונים | הגדרת תרשימים, יצירת דוחות ויזואליים |
| **Orin** | מומחה למידול נתונים | סיוע בעיצוב מבנה אוספים, הצעות לשדות |
| **Nathan** | מהנדס פרונט-אנד | סיוע בכתיבת קטעי קוד פרונט-אנד, התאמות עיצוב |

**הערות**

חלק מעובדי ה-AI המובנים לא יופיעו ברשימה בפינה הימנית התחתונה, מכיוון שיש להם תרחישי עבודה ייעודיים:

- Orin: דפי מידול נתונים.
- Dara: בלוקים של הגדרת תרשימים.
- Nathan: בלוק JS ועורכי קוד דומים.
