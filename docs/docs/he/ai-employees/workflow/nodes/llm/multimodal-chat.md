# שיחה מולטימודלית

## תמונות

אם המודל תומך בכך, צומת ה-LLM יכול לשלוח תמונות למודל. בעת השימוש, עליכם לבחור שדה קובץ מצורף או רשומה מקושרת מתוך אוסף קבצים באמצעות משתנה. כשבוחרים רשומת אוסף קבצים, ניתן לבחור אותה ברמת האובייקט או לבחור את שדה ה-URL.

![](https://static-docs.nocobase.com/202503041034858.png)

קיימות שתי אפשרויות לבחירה עבור פורמט שליחת התמונות:

- שליחה באמצעות URL - כל התמונות, למעט אלו המאוחסנות מקומית, יישלחו ככתובות URL. תמונות המאוחסנות מקומית יומרו לפורמט base64 לפני השליחה.
- שליחה באמצעות base64 - כל התמונות, בין אם מאוחסנות מקומית ובין אם בענן, יישלחו בפורמט base64. אפשרות זו מתאימה למקרים שבהם שירות ה-LLM המקוון אינו יכול לגשת ישירות לכתובת ה-URL של התמונה.
