# שימוש במיומנויות

מיומנויות (Tools) מגדירות מה עובד AI "יכול לעשות".

## מבנה המיומנויות

דף המיומנויות מחולק לשלוש קטגוריות:

1. `General skills`: משותפות לכל עובדי ה-AI, לקריאה בלבד.
2. `Employee-specific skills`: ייחודיות לעובד הנוכחי, בדרך כלל לקריאה בלבד.
3. `Custom skills`: מיומנויות מותאמות אישית שניתן להוסיף או להסיר ולהגדיר עבורן הרשאות ברירת מחדל.

![skills-three-sections-general-specific-custom.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/skills-three-sections-general-specific-custom.png)

## הרשאות מיומנות

הרשאות המיומנות מאוחדות ל:

- `Ask`: בקשת אישור לפני ההפעלה.
- `Allow`: אישור הפעלה ישירה.

המלצה: עבור מיומנויות הכרוכות בשינוי נתונים, מומלץ להשתמש ב-`Ask` כברירת מחדל.

![skills-permission-ask-allow-segmented.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/skills-permission-ask-allow-segmented.png)

## הוספה ותחזוקה

לחצו על `Add skill` תחת `Custom skills` כדי להוסיף מיומנויות ולהגדיר הרשאות בהתאם לסיכונים העסקיים.
