# חיפוש באינטרנט

חיפוש באינטרנט משמש להשלמת מידע עדכני מעבר לנתוני האימון של המודל.

## איך זה עובד

הזמינות של חיפוש באינטרנט תלויה בשאלה האם שירות המודל שנבחר בשיחה הנוכחית תומך ב-Web Search.

- **נתמך:** מתג החיפוש באינטרנט מוצג וניתן להפעלה או כיבוי לפי הצורך.
- **לא נתמך:** המתג אינו מוצג, ומצב החיפוש יכובה באופן אוטומטי.

## שימוש בשיחה

השתמשו במתג החיפוש באינטרנט באזור הזנת ההודעה:

- **מופעל:** ה-AI יכול לבצע חיפוש ברשת לפני מתן התשובה.

![web-search-switch-visible-on-supported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-visible-on-supported-model.png)

- **כבוי:** ה-AI עונה על סמך ההקשר הקיים בלבד.

![web-search-switch-hidden-on-unsupported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-hidden-on-unsupported-model.png)

## הבדלים בין ספקים

היכולת לחיפוש באינטרנט משתנה בין מודלים שונים. יש להשתמש בהתאם למצב בפועל של שירות המודל והמודל שנבחר.
