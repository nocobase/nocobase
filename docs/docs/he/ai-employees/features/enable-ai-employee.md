# הפעלת עובדי AI

NocoBase מספקת מספר עובדי AI מובנים המיועדים לתרחישים שונים.

## כיצד להפעיל

היכנסו לדף הניהול `AI employees` והפעילו את המתג `Enabled` ישירות ברשימה.

![ai-employee-list-enable-switch.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employee-list-enable-switch.png)

## שימוש לאחר ההפעלה

לאחר ההפעלה, העובד יופיע בנקודות הגישה הזמינות (כגון קיצור הדרך בפינה הימנית התחתונה או בנקודות גישה ספציפיות לתרחיש).
