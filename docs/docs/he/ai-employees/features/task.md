# משימות קיצור דרך

כדי לאפשר לעובדי AI להתחיל לעבוד בצורה יעילה יותר, ניתן לקשור עובדי AI לבלוקים של תרחישים ולהגדיר מראש מספר משימות נפוצות.

הדבר מאפשר למשתמשים להתחיל בעיבוד משימות בלחיצה אחת, ללא צורך ב**בחירת בלוק** ו**הזנת פקודה** בכל פעם מחדש.

## קישור עובד AI לבלוק

לאחר הכניסה למצב עריכת ממשק משתמש (UI), עבור בלוקים התומכים ב-`Actions`, בחרו בתפריט `AI employees` תחת `Actions`, ולאחר מכן בחרו עובד AI. עובד ה-AI הזה ייקשר לבלוק הנוכחי.

![20251022135306](https://static-docs.nocobase.com/20251022135306.png)

לאחר השלמת הקישור, בכל פעם שתיכנסו לדף, אזור ה-Actions של הבלוק יציג את עובד ה-AI הקשור לבלוק הנוכחי.

![20251022135438](https://static-docs.nocobase.com/20251022135438.png)

## הגדרת משימות

לאחר הכניסה למצב עריכת ממשק משתמש, רחפו עם העכבר מעל סמל עובד ה-AI הקשור לבלוק. יופיע כפתור תפריט, בחרו ב-`Edit tasks` כדי להיכנס לדף הגדרות המשימות.

בדף הגדרות המשימות, ניתן להוסיף מספר משימות עבור עובד ה-AI הנוכחי.

כל לשונית מייצגת משימה עצמאית. לחצו על סימן ה-"+" שלידן כדי להוסיף משימה חדשה.

![clipboard-image-1771913187](https://static-docs.nocobase.com/clipboard-image-1771913187.png)

טופס הגדרות המשימה:

- בתיבת הקלט `Title`, הזינו את כותרת המשימה. תיאור קצר של המשימה שיופיע ברשימת המשימות של עובד ה-AI.
- בתיבת הקלט `Background`, הזינו את התוכן העיקרי של המשימה. תוכן זה ישמש כהנחיית מערכת (System Prompt) בעת שיחה עם עובד ה-AI.
- בתיבת הקלט `Default user message`, הזינו הודעת משתמש ברירת מחדל. לאחר בחירת המשימה, היא תמולא אוטומטית בתיבת הקלט של המשתמש.
- ב-`Work context`, בחרו את מידע הקשר האפליקציה (App Context) שיישלח כברירת מחדל לעובד ה-AI. פעולה זו זהה לפעולה בחלונית הצ'אט.
- בבורר ה-`Skills`, מוצגות המיומנויות הזמינות לעובד ה-AI הנוכחי. ניתן לבטל מיומנות מסוימת כדי שעובד ה-AI יתעלם ממנה בעת ביצוע משימה זו.
- תיבת הסימון `Send default user message automatically` קובעת האם לשלוח אוטומטית את הודעת המשתמש המוגדרת כברירת מחדל מיד לאחר הלחיצה על ביצוע המשימה.


## רשימת משימות

לאחר הגדרת המשימות עבור עובד ה-AI, הן יופיעו בחלון הצף של פרופיל עובד ה-AI ובהודעת הפתיחה לפני תחילת השיחה. לחצו עליהן כדי לבצע את המשימה.
