# עובדי AI מובנים

NocoBase מגיע עם מספר עובדי AI מובנים המותאמים לתרחישים ספציפיים.

כל שעליכם לעשות הוא להגדיר את שירות ה-LLM ולהפעיל את העובד המתאים כדי להתחיל לעבוד; ניתן להחליף מודלים לפי דרישה במהלך השיחה.


## מבוא

![clipboard-image-1766653060](https://static-docs.nocobase.com/clipboard-image-1766653060.png)

| שם העובד | הגדרת תפקיד | יכולות ליבה |
| :--- | :--- | :--- |
| **Cole** | עוזר NocoBase | שאלות ותשובות על המוצר, אחזור מסמכים |
| **Ellis** | מומחה דואר אלקטרוני | ניסוח מיילים, יצירת סיכומים, הצעות לתגובה |
| **Dex** | מומחה לארגון נתונים | תרגום שדות, עיצוב, חילוץ מידע |
| **Viz** | אנליסט תובנות | תובנות נתונים, ניתוח מגמות, פרשנות מדדי מפתח |
| **Lexi** | עוזר תרגום | תרגום רב-לשוני, סיוע בתקשורת |
| **Vera** | אנליסט מחקר | חיפוש ברשת, ריכוז מידע, מחקר מעמיק |

באפשרותכם ללחוץ על **בועת ה-AI הצפה** בפינה הימנית התחתונה של ממשק האפליקציה ולבחור את העובד הדרוש לכם כדי להתחיל בשיתוף הפעולה.


## עובדי AI לתרחישים ייעודיים

חלק מעובדי ה-AI המובנים (מסוג בוני מערכת) אינם מופיעים ברשימת עובדי ה-AI בפינה הימנית התחתונה; יש להם סביבות עבודה ייעודיות, לדוגמה:

* Orin מופיע רק בדף הגדרת מקור הנתונים;
* Dara מופיעה רק בדף הגדרת התרשימים;
* Nathan מופיע רק בעורך ה-JS.



---

להלן מספר תרחישי יישום טיפוסיים עבור עובדי AI כדי לספק לכם השראה. פוטנציאל נוסף ממתין לחקירה שלכם בתרחישים עסקיים אמיתיים.


## Viz: אנליסט תובנות

### מבוא

> יצירת תרשימים ותובנות בלחיצה אחת, תנו לנתונים לדבר בעד עצמם.

**Viz** הוא **אנליסט תובנות ה-AI** המובנה.
הוא יודע לקרוא את הנתונים בדף הנוכחי שלכם (כגון לידים, הזדמנויות, חשבונות), ויוצר באופן אוטומטי תרשימי מגמות, תרשימי השוואה, כרטיסי KPI ומסקנות תמציתיות, מה שהופך את הניתוח העסקי לקל ואינטואיטיבי.

> רוצים לדעת "למה המכירות ירדו לאחרונה"?
> פשוט אמרו מילה ל-Viz, והוא יוכל לומר לכם איפה התרחשה הירידה, מהן הסיבות האפשריות ומה יכולים להיות הצעדים הבאים.

### תרחישי שימוש

בין אם מדובר בסקירות עסקיות חודשיות, ROI של ערוצי שיווק או משפכי מכירות, תוכלו לתת ל-Viz לנתח, ליצור תרשימים ולפרש תוצאות.

| תרחיש | מה תרצו לדעת | התוצר של Viz |
| -------- | ------------ | ------------------- |
| **סקירה חודשית** | במה החודש הזה טוב יותר מהקודם? | כרטיס KPI + תרשים מגמות + שלוש הצעות לשיפור |
| **פירוט צמיחה** | האם הצמיחה בהכנסות נובעת מנפח או ממחיר? | תרשים פירוק גורמים + טבלת השוואה |
| **ניתוח ערוצים** | באיזה ערוץ הכי כדאי להמשיך להשקיע? | תרשים ROI + עקומת שימור + המלצות |
| **ניתוח משפך** | איפה התנועה נתקעת? | תרשים משפך + הסבר על צווארי בקבוק |
| **שימור לקוחות** | אילו לקוחות הם בעלי הערך הרב ביותר? | תרשים סגמנטציית RFM + עקומת שימור |
| **הערכת מבצעים** | עד כמה המבצע הגדול היה אפקטיבי? | תרשים השוואה + ניתוח גמישות מחירים |

### אופן השימוש

**נקודות כניסה בדף**

* **כפתור בפינה הימנית העליונה (מומלץ)**
  
  בדפים כמו לידים (Leads), הזדמנויות (Opportunities) וחשבונות (Accounts), לחצו על **אייקון Viz** בפינה הימנית העליונה כדי לבחור משימות מוגדרות מראש, כגון:

  * המרות שלבים ומגמות
  * השוואת ערוצי מקור
  * ניתוח סקירה חודשית

  ![clipboard-image-1771913319](https://static-docs.nocobase.com/clipboard-image-1771913319.png)

* **פאנל גלובלי בפינה הימנית התחתונה**
  
  בכל דף, תוכלו להפעיל את פאנל ה-AI הגלובלי ולדבר ישירות עם Viz:

  ```
  Analyze sales changes over the last 90 days
  ```

  Viz יביא באופן אוטומטי את הקשר הנתונים (Context) של הדף הנוכחי שלכם.

**אינטראקציה**

Viz תומך בשאלות בשפה טבעית ומבין שאלות המשך מרובות שלבים.
דוגמה:

```
Hi Viz, generate lead trends for this month.
```

```
Only show performance from third-party channels.
```

```
Which region is growing the fastest?
```

כל שאלת המשך תמשיך להעמיק על בסיס תוצאות הניתוח הקודמות, ללא צורך בהזנה מחדש של תנאי הנתונים.

### טיפים לשיחה עם Viz

| שיטה | השפעה |
| ---------- | ------------------- |
| ציינו טווח זמן | "30 הימים האחרונים" או "חודש שעבר מול חודש נוכחי" מדויק יותר |
| ציינו ממדים | "הצג לפי אזור/ערוץ/מוצר" עוזר ליישר מבט |
| התמקדו במגמות ולא בפרטים | Viz מצטיין בזיהוי כיוון השינוי וסיבות המפתח |
| השתמשו בשפה טבעית | אין צורך בתחביר פקודות, פשוט שאלו שאלות כאילו אתם בשיחה |

## Dex: מומחה לארגון נתונים

### מבוא

> חילוץ ומילוי טפסים במהירות, הפיכת מידע מבולגן לנתונים מובנים.

`Dex` הוא מומחה לארגון נתונים המחלץ מידע נדרש מנתונים או קבצים לא מובנים ומארגן אותו למידע מובנה. הוא יכול גם להשתמש בכלים כדי למלא את המידע בטפסים.

### אופן השימוש

הפעילו את `Dex` בדף הטופס כדי לפתוח את חלון השיחה.

![20251022155316](https://static-docs.nocobase.com/20251022155316.png)

לחצו על **Add work context** בתיבת הקלט ובחרו ב-**Pick block**; הדף ייכנס למצב בחירת בלוקים.

![20251022155414](https://static-docs.nocobase.com/20251022155414.png)

בחרו את בלוק הטופס בדף.

![20251022155523](https://static-docs.nocobase.com/20251022155523.png)

הזינו בתיבת הדו-שיח את הנתונים שברצונכם ש-`Dex` יארגן.

![20251022155555](https://static-docs.nocobase.com/20251022155555.png)

לאחר השליחה, `Dex` יבנה את הנתונים וישתמש בכישוריו כדי לעדכן את הנתונים בטופס שנבחר.

![20251022155733](https://static-docs.nocobase.com/20251022155733.png)

## Orin: מומחה למידול נתונים

### מבוא

> עיצוב אוספים בצורה חכמה ואופטימיזציה של מבני מסד הנתונים.

`Orin` הוא מומחה למידול נתונים. בדף הגדרת מקור הנתונים הראשי, תוכלו לתת ל-`Orin` לעזור לכם ליצור או לשנות אוספים.

![20251022160628](https://static-docs.nocobase.com/20251022160628.png)

### אופן השימוש

היכנסו לתוסף מנהל מקור הנתונים (Data Source Manager) ובחרו להגדיר את מקור הנתונים הראשי.

![20251022161146](https://static-docs.nocobase.com/20251022161146.png)

לחצו על האוואטר של `Orin` בפינה הימנית העליונה כדי לפתוח את תיבת הדו-שיח של עובד ה-AI.

![20251022161641](https://static-docs.nocobase.com/20251022161641.png)

תארו את דרישות המידול שלכם ל-`Orin`, שלחו והמתינו לתגובה. 

ברגע ש-`Orin` יאשר את הדרישות שלכם, הוא ישתמש בכישוריו וישיב עם תצוגה מקדימה של מידול הנתונים.

לאחר סקירת התצוגה המקדימה, לחצו על כפתור **Finish review and apply** כדי ליצור אוספים בהתאם למידול של `Orin`.

![20251022162142](https://static-docs.nocobase.com/20251022162142.png)

## Nathan: מהנדס פרונט-אנד

### מבוא

> עוזר לכם לכתוב ולבצע אופטימיזציה לקוד פרונט-אנד למימוש לוגיקת אינטראקציה מורכבת.

`Nathan` הוא המומחה לפיתוח פרונט-אנד ב-NocoBase. בתרחישים בהם נדרש JavaScript, כגון `JSBlock`, `JSField`, `JSItem`, `JSColumn`, `JSAction`, `Event Flow` ו-`Linkage`, האוואטר של Nathan יופיע בפינה הימנית העליונה של עורך הקוד, מה שיאפשר לכם לבקש ממנו עזרה בכתיבה או בשינוי הקוד בעורך.

![20251022163405](https://static-docs.nocobase.com/20251022163405.png)

### אופן השימוש

בעורך הקוד, לחצו על `Nathan` כדי לפתוח את תיבת הדו-שיח של עובד ה-AI; הקוד בעורך יצורף אוטומטית לתיבת הקלט ויישלח ל-`Nathan` כהקשר של האפליקציה.

![20251022163935](https://static-docs.nocobase.com/20251022163935.png)

הזינו את דרישות הקידוד שלכם, שלחו אותן ל-`Nathan` והמתינו לתגובתו.

![20251022164041](https://static-docs.nocobase.com/20251022164041.png)

לחצו על כפתור **Apply to editor** בבלוק הקוד ש-`Nathan` השיב איתו כדי להחליף את הקוד בעורך בקוד שלו.

![20251022164323](https://static-docs.nocobase.com/20251022164323.png)

לחצו על כפתור **Run** בעורך הקוד כדי לצפות בתוצאות בזמן אמת.

![20251022164436](https://static-docs.nocobase.com/20251022164436.png)

### היסטוריית קוד

לחצו על אייקון ה-**Command Line** בפינה הימנית העליונה של תיבת הדו-שיח של `Nathan` כדי לצפות בקטעי הקוד ששלחתם ובקטעי הקוד ש-`Nathan` השיב איתם בשיחה הנוכחית.

![20251022164644](https://static-docs.nocobase.com/20251022164644.png)
