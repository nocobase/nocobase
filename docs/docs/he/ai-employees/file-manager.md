# ניהול קבצים

## מבוא

הגדירו את שיטת האחסון לקבצים המועלים במהלך שיחות עם עובדי AI.

## הגדרות

עברו לדף הגדרות התוסף של עובדי AI, לחצו על לשונית `Settings` כדי להיכנס לדף הגדרות שיטת האחסון.

![20251022002648](https://static-docs.nocobase.com/20251022002648.png)

אפשרויות האחסון הזמינות עבור `Files storage` מוגדרות בתוסף ניהול הקבצים.
