# מתקדם

## מבוא

ניתן לשייך עובדי AI לעמודים או לבלוקים. לאחר השיוך, תוכלו להגדיר משימות המותאמות לעסק שלכם, מה שיאפשר למשתמשים להשתמש במהירות בעובד ה-AI לטיפול במשימות ישירות מהעמוד או הבלוק.

## שיוך עובד AI לעמוד

לאחר שהעמוד נכנס למצב עריכת ממשק משתמש, יופיע סימן "+" ליד כפתור הגישה המהירה של עובד ה-AI בפינה הימנית התחתונה. רחפו עם העכבר מעל סימן ה-"+", ותופיע רשימה של עובדי AI. בחרו עובד AI כדי לשייך אותו לעמוד הנוכחי.

![20251022134656](https://static-docs.nocobase.com/20251022134656.png)

לאחר השלמת השיוך, בכל פעם שתכנסו לעמוד, יוצג עובד ה-AI המשויך לעמוד הנוכחי בפינה הימנית התחתונה.

![20251022134903](https://static-docs.nocobase.com/20251022134903.png)

![20251022135306](https://static-docs.nocobase.com/20251022135306.png)

לאחר השלמת השיוך, בכל פעם שתכנסו לעמוד, יוצג עובד ה-AI המשויך לבלוק הנוכחי באזור ה-`Actions` של הבלוק.

![20251022135438](https://static-docs.nocobase.com/20251022135438.png)

## הגדרת משימות

לאחר שהעמוד נכנס למצב עריכת ממשק משתמש, רחפו עם העכבר מעל אייקון עובד ה-AI המשויך לעמוד או לבלוק. יופיע כפתור תפריט. בחרו ב-`Edit tasks` כדי להיכנס לדף הגדרת המשימות.

![20251022135710](https://static-docs.nocobase.com/20251022135710.png)

לאחר הכניסה לדף הגדרת המשימות, תוכלו להוסיף מספר משימות לעובד ה-AI הנוכחי.

כל לשונית מייצגת משימה נפרדת. לחצו על סימן ה-"+ " שליד כדי להוסיף משימה חדשה.

![20251022140058](https://static-docs.nocobase.com/20251022140058.png)

טופס הגדרת משימה:

- בשדה הקלט `Title`, הזינו את כותרת המשימה. תארו בקצרה את תוכן המשימה. כותרת זו תופיע ברשימת המשימות של עובד ה-AI.
- בשדה הקלט `Background`, הזינו את התוכן העיקרי של המשימה. תוכן זה ישמש כהנחיה (prompt) למערכת בעת שיחה עם עובד ה-AI.
- בשדה הקלט `Default user message`, הזינו את הודעת המשתמש שתשלח כברירת מחדל. הודעה זו תאוכלס אוטומטית בשדה הקלט של המשתמש לאחר בחירת המשימה.
- ב-`Work context`, בחרו את מידע הקשר היישומי (application context) שיישלח כברירת מחדל לעובד ה-AI. פעולה זו זהה לפעולה בתיבת הדו-שיח.
- תיבת הבחירה `Skills` מציגה את הכישורים הזמינים לעובד ה-AI הנוכחי. באפשרותכם לבטל בחירה של כישור מסוים כדי שעובד ה-AI יתעלם ממנו ולא ישתמש בו בעת ביצוע משימה זו.
- תיבת הסימון `Send default user message automatically` קובעת האם לשלוח אוטומטית את הודעת המשתמש המוגדרת כברירת מחדל לאחר לחיצה על ביצוע המשימה.

![20251022140805](https://static-docs.nocobase.com/20251022140805.png)

## רשימת משימות

לאחר שתגדירו משימות עבור עובד AI, משימות אלו יוצגו בחלון הקופץ של פרופיל עובד ה-AI ובהודעת הפתיחה לפני תחילת שיחה. לחצו על משימה כדי לבצע אותה.
