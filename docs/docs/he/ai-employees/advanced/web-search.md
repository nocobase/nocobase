# מתקדם

## מבוא

בדרך כלל, למודלי שפה גדולים (LLMs) יש נתונים שאינם עדכניים מספיק, והם חסרים מידע חדש. לכן, פלטפורמות שירותי LLM מקוונות מציעות בדרך כלל תכונת חיפוש אינטרנט, המאפשרת ל-AI לחפש מידע באמצעות כלים לפני מתן תשובה, ולאחר מכן להגיב בהתבסס על תוצאות החיפוש.

עובדי AI הותאמו לתכונת חיפוש האינטרנט של פלטפורמות שירותי LLM מקוונות שונות. באפשרותכם להפעיל את תכונת חיפוש האינטרנט בהגדרות המודל של עובד ה-AI וגם במהלך שיחות.

## הפעלת תכונת חיפוש אינטרנט

עברו לדף הגדרות ה`תוסף` של עובד ה-AI, לחצו על לשונית `AI employees` כדי להיכנס לדף ניהול עובדי AI.

![20251021225643](https://static-docs.nocobase.com/20251021225643.png)

בחרו את עובד ה-AI שעבורו אתם רוצים להפעיל את תכונת חיפוש האינטרנט, לחצו על כפתור `Edit` כדי להיכנס לדף עריכת עובד ה-AI.

![20251022114043](https://static-docs.nocobase.com/20251022114043.png)

בלשונית `Model settings`, הפעילו את המתג `Web Search`, ולחצו על כפתור `Submit` כדי לשמור את השינויים.

![20251022114300](https://static-docs.nocobase.com/20251022114300.png)

## שימוש בתכונת חיפוש אינטרנט בשיחות

לאחר שעובד AI הפעיל את תכונת חיפוש האינטרנט, יופיע אייקון "אינטרנט" בתיבת קלט השיחה. חיפוש אינטרנט מופעל כברירת מחדל, ובאפשרותכם ללחוץ עליו כדי לבטל את ההפעלה.

![20251022115110](https://static-docs.nocobase.com/20251022115110.png)

כאשר חיפוש אינטרנט מופעל, תגובת עובד ה-AI תציג את תוצאות חיפוש האינטרנט.

![20251022115502](https://static-docs.nocobase.com/20251022115502.png)
