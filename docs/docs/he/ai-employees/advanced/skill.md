# מתקדם

## מבוא

למודלי שפה גדולים (LLMs) מובילים יש את היכולת להשתמש בכלים. תוסף עובד ה-AI כולל כלים נפוצים מובנים לשימושם של מודלי שפה גדולים.

הכישורים המוגדרים בדף ההגדרות של עובד ה-AI הם הכלים הזמינים למודל השפה הגדול.

![20251022142348](https://static-docs.nocobase.com/20251022142348.png)

## הגדרת כישורים

עברו לדף התצורה של תוסף עובד ה-AI, לחצו על לשונית `AI employees` כדי להיכנס לדף ניהול עובדי ה-AI.

בחרו את עובד ה-AI שעבורו תרצו להגדיר כישורים, לחצו על כפתור `Edit` כדי להיכנס לדף עריכת עובד ה-AI.

בלשונית `Skills`, לחצו על כפתור `Add Skill` כדי להוסיף כישור לעובד ה-AI הנוכחי.

![20251022145748](https://static-docs.nocobase.com/20251022145748.png)

## היכרות עם הכישורים

### Frontend

קבוצת Frontend מאפשרת לעובד ה-AI ליצור אינטראקציה עם רכיבי צד-לקוח (frontend).

- כישור `Form filler` מאפשר לעובד ה-AI למלא נתוני טופס שנוצרו בחזרה לטופס שהוגדר על ידי המשתמש.

![20251022145954](https://static-docs.nocobase.com/20251022145954.png)

### Data modeling

קבוצת הכישורים Data modeling מעניקה לעובד ה-AI את היכולת לקרוא לממשקי API פנימיים של NocoBase לצורך מידול נתונים.

- `Intent Router` (נתב כוונות) – מנתב כוונות, וקובע אם המשתמש רוצה לשנות מבנה של אוסף קיים או ליצור אוסף חדש.
- `Get collection names` – מקבל את השמות של כל האוספים הקיימים במערכת.
- `Get collection metadata` – מקבל את פרטי המבנה של אוסף ספציפי.
- `Define collections` – מאפשר לעובד ה-AI ליצור אוספים במערכת.

![20251022150441](https://static-docs.nocobase.com/20251022150441.png)

### Workflow caller

`Workflow caller` מעניק לעובד ה-AI את היכולת לבצע תהליכי עבודה. תהליכי עבודה המוגדרים עם `Trigger type` מסוג `AI employee event` בתוסף תהליכי העבודה, יהיו זמינים כאן ככישורים לשימוש עובד ה-AI.

![20251022153320](https://static-docs.nocobase.com/20251022153320.png)

### Code Editor

הכישורים בקבוצת Code Editor מאפשרים בעיקר לעובד ה-AI ליצור אינטראקציה עם עורך הקוד.

- `Get code snippet list` – מקבל את רשימת קטעי הקוד המוגדרים מראש.
- `Get code snippet content` – מקבל את התוכן של קטע קוד ספציפי.

![20251022153811](https://static-docs.nocobase.com/20251022153811.png)

### אחרים

- `Chart generator` מעניק לעובד ה-AI את היכולת ליצור תרשימים ולהציג אותם ישירות בשיחה.

![20251022154141](https://static-docs.nocobase.com/20251022154141.png)
