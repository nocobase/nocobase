### עיצוב מטבע

#### 1. :formatC(precisionOrFormat, targetCurrency)

##### הסבר תחביר
מעצב מספר מטבע, ניתן לציין את מספר הספרות העשרוניות או פורמט פלט ספציפי.  
פרמטרים:
- precisionOrFormat: פרמטר אופציונלי, יכול להיות מספר (המציין את מספר הספרות העשרוניות) או מזהה פורמט ספציפי:
  - מספר שלם: שינוי דיוק ברירת המחדל של הספרות העשרוניות
  - `'M'`: פלט של שם המטבע הראשי בלבד
  - `'L'`: פלט של המספר בצירוף סימן המטבע (ברירת מחדל)
  - `'LL'`: פלט של המספר בצירוף שם המטבע הראשי
- targetCurrency: אופציונלי, קוד מטבע היעד (באותיות גדולות, כגון USD, EUR), יעקוף את ההגדרות הגלובליות

##### דוגמה
```
'1000.456':formatC()      // פלט "$2,000.91"
'1000.456':formatC('M')    // פלט "dollars"
'1':formatC('M')           // פלט "dollar"
'1000':formatC('L')        // פלט "$2,000.00"
'1000':formatC('LL')       // פלט "2,000.00 dollars"
```

##### תוצאה
התוצאה תלויה באפשרויות ה-API ובהגדרות שער החליפין.


#### 2. :convCurr(target, source)

##### הסבר תחביר
המרת מספר ממטבע אחד למטבע אחר. ניתן להעביר את שער החליפין דרך אפשרויות ה-API או להגדירו גלובלית.  
אם לא צוינו פרמטרים, ההמרה מתבצעת אוטומטית מ-`options.currencySource` ל-`options.currencyTarget`.  
פרמטרים:
- target: אופציונלי, קוד מטבע היעד (ברירת המחדל היא `options.currencyTarget`)
- source: אופציונלי, קוד מטבע המקור (ברירת המחדל היא `options.currencySource`)

##### דוגמה
```
10:convCurr()              // פלט 20
1000:convCurr()            // פלט 2000
1000:convCurr('EUR')        // פלט 1000
1000:convCurr('USD')        // פלט 2000
1000:convCurr('USD', 'USD') // פלט 1000
```

##### תוצאה
הפלט הוא ערך המטבע המומר.
