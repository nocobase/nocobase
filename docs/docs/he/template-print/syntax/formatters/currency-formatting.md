### עיצוב מטבע

#### 1. :formatC(precisionOrFormat, targetCurrency)

##### הסבר תחביר
מעצב מספר מטבע ומאפשר לציין את מספר הספרות העשרוניות או פורמט פלט ספציפי.
פרמטרים:
- `precisionOrFormat`: פרמטר אופציונלי שיכול להיות מספר (המציין את מספר הספרות העשרוניות) או מזהה פורמט ספציפי:
  - מספר שלם: משנה את דיוק ברירת המחדל של הספרות העשרוניות.
  - `'M'`: מציג רק את שם המטבע הראשי.
  - `'L'`: מציג את המספר יחד עם סימן המטבע (ברירת מחדל).
  - `'LL'`: מציג את המספר יחד עם שם המטבע הראשי.
- `targetCurrency`: אופציונלי; קוד המטבע היעד (באותיות גדולות, לדוגמה USD, EUR) שידרוס את ההגדרות הגלובליות.

##### דוגמה
```
// סביבת דוגמה: אפשרויות API { "lang": "en-us", "currency": { "source": "EUR", "target": "USD", "rates": { "EUR": 1, "USD": 2 } } }
'1000.456':formatC()      // פלט: "$2,000.91"
'1000.456':formatC('M')    // פלט: "dollars"
'1':formatC('M')           // פלט: "dollar"
'1000':formatC('L')        // פלט: "$2,000.00"
'1000':formatC('LL')       // פלט: "2,000.00 dollars"

// דוגמה בצרפתית (כאשר הגדרות הסביבה שונות):
'1000.456':formatC()      // פלט: "2 000,91 ..."  
'1000.456':formatC()      // כאשר מטבע המקור ומטבע היעד זהים, הפלט הוא "1 000,46 €"
```

##### תוצאה
הפלט תלוי באפשרויות ה-API ובהגדרות שער החליפין.


#### 2. :convCurr(target, source)

##### הסבר תחביר
ממיר מספר ממטבע אחד למטבע אחר. שער החליפין יכול לעבור דרך אפשרויות ה-API או להיות מוגדר גלובלית.
אם לא צוינו פרמטרים, ההמרה מתבצעת אוטומטית מ-`options.currencySource` ל-`options.currencyTarget`.
פרמטרים:
- `target`: אופציונלי; קוד מטבע היעד (ברירת המחדל היא `options.currencyTarget`).
- `source`: אופציונלי; קוד מטבע המקור (ברירת המחדל היא `options.currencySource`).

##### דוגמה
```
// סביבת דוגמה: אפשרויות API { "currency": { "source": "EUR", "target": "USD", "rates": { "EUR": 1, "USD": 2 } } }
10:convCurr()              // פלט: 20
1000:convCurr()            // פלט: 2000
1000:convCurr('EUR')        // פלט: 1000
1000:convCurr('USD')        // פלט: 2000
1000:convCurr('USD', 'USD') // פלט: 1000
```

##### תוצאה
הפלט הוא ערך המטבע המומר.
