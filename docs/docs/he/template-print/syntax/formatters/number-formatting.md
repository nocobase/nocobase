### עיצוב מספרים

#### 1. :formatN(precision)

##### הסבר תחבירי
מעצב מספר בהתאם להגדרות לוקליזציה.  
פרמטר:
- precision: מספר הספרות העשרוניות  
  עבור פורמטים של ODS/XLSX, מספר הספרות העשרוניות המוצגות נקבע על ידי עורך הטקסט; פורמטים אחרים תלויים בפרמטר זה.

##### דוגמה
```
'10':formatN()         // פלט "10.000"
'1000.456':formatN()   // פלט "1,000.456"
```

##### תוצאה
המספר יוצא לפי הדיוק שצוין ופורמט הלוקליזציה.


#### 2. :round(precision)

##### הסבר תחבירי
מבצע עיגול של המספר, הפרמטר מציין את מספר הספרות העשרוניות.

##### דוגמה
```
10.05123:round(2)      // פלט 10.05
1.05:round(1)          // פלט 1.1
```

##### תוצאה
הפלט הוא הערך לאחר עיגול.


#### 3. :add(value)

##### הסבר תחבירי
מחבר את המספר הנוכחי עם הערך שצוין.  
פרמטר:
- value: המספר לחיבור

##### דוגמה
```
1000.4:add(2)         // פלט 1002.4
'1000.4':add('2')      // פלט 1002.4
```

##### תוצאה
הפלט הוא הערך לאחר החיבור.


#### 4. :sub(value)

##### הסבר תחבירי
מחסיר את הערך שצוין מהמספר הנוכחי.  
פרמטר:
- value: המחוסר

##### דוגמה
```
1000.4:sub(2)         // פלט 998.4
'1000.4':sub('2')      // פלט 998.4
```

##### תוצאה
הפלט הוא הערך לאחר החיסור.


#### 5. :mul(value)

##### הסבר תחבירי
מכפיל את המספר הנוכחי בערך שצוין.  
פרמטר:
- value: המכפיל

##### דוגמה
```
1000.4:mul(2)         // פלט 2000.8
'1000.4':mul('2')      // פלט 2000.8
```

##### תוצאה
הפלט הוא הערך לאחר הכפל.


#### 6. :div(value)

##### הסבר תחבירי
מחלק את המספר הנוכחי בערך שצוין.  
פרמטר:
- value: המחלק

##### דוגמה
```
1000.4:div(2)         // פלט 500.2
'1000.4':div('2')      // פלט 500.2
```

##### תוצאה
הפלט הוא הערך לאחר החילוק.


#### 7. :mod(value)

##### הסבר תחבירי
מחשב את המודולו (שארית) של המספר הנוכחי מול הערך שצוין.  
פרמטר:
- value: המודולוס

##### דוגמה
```
4:mod(2)              // פלט 0
3:mod(2)              // פלט 1
```

##### תוצאה
הפלט הוא תוצאת פעולת המודולו.


#### 8. :abs

##### הסבר תחבירי
מחזיר את הערך המוחלט של המספר.

##### דוגמה
```
-10:abs()             // פלט 10
-10.54:abs()          // פלט 10.54
10.54:abs()           // פלט 10.54
'-200':abs()          // פלט 200
```

##### תוצאה
הפלט הוא הערך המוחלט.


#### 9. :ceil

##### הסבר תחבירי
עיגול כלפי מעלה, כלומר מחזיר את המספר השלם הקטן ביותר שגדול או שווה למספר הנוכחי.

##### דוגמה
```
10.05123:ceil()       // פלט 11
1.05:ceil()           // פלט 2
-1.05:ceil()          // פלט -1
```

##### תוצאה
הפלט הוא המספר השלם לאחר העיגול.


#### 10. :floor

##### הסבר תחבירי
עיגול כלפי מטה, כלומר מחזיר את המספר השלם הגדול ביותר שקטן או שווה למספר הנוכחי.

##### דוגמה
```
10.05123:floor()      // פלט 10
1.05:floor()          // פלט 1
-1.05:floor()         // פלט -2
```

##### תוצאה
הפלט הוא המספר השלם לאחר העיגול.


#### 11. :int

##### הסבר תחבירי
המרת המספר למספר שלם (לא מומלץ לשימוש).

##### דוגמה ותוצאה
בהתאם למקרה ההמרה הספציפי.


#### 12. :toEN

##### הסבר תחבירי
המרת המספר לפורמט אנגלי (נקודה עשרונית היא '.'), לא מומלץ לשימוש.

##### דוגמה ותוצאה
בהתאם למקרה ההמרה הספציפי.


#### 13. :toFixed

##### הסבר תחבירי
המרת המספר למחרוזת, תוך שמירה על מספר הספרות העשרוניות שצוין בלבד, לא מומלץ לשימוש.

##### דוגמה ותוצאה
בהתאם למקרה ההמרה הספציפי.


#### 14. :toFR

##### הסבר תחבירי
המרת המספר לפורמט צרפתי (נקודה עשרונית היא ','), לא מומלץ לשימוש.

##### דוגמה ותוצאה
בהתאם למקרה ההמרה הספציפי.
