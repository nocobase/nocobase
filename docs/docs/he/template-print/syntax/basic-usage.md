## שימוש בסיסי

תוסף הדפסת התבניות מציע מגוון תחבירים שמאפשרים להכניס נתונים דינמיים ומבנים לוגיים לתבניות בצורה גמישה. להלן הסברים מפורטים על התחביר ודוגמאות שימוש.

### החלפה בסיסית

השתמשו במצייני מיקום בפורמט `{d.xxx}` להחלפת נתונים. לדוגמה:

- `{d.title}`: קורא את השדה `title` ממערך הנתונים.
- `{d.date}`: קורא את השדה `date` ממערך הנתונים.

**דוגמה**:

תוכן התבנית:
```
לקוח/ה יקר/ה,

תודה שרכשתם את המוצר שלנו: {d.productName}.
מספר הזמנה: {d.orderId}
תאריך הזמנה: {d.orderDate}

אנו מאחלים לכם שימוש מהנה!
```

מערך נתונים:
```json
{
  "productName": "שעון חכם",
  "orderId": "A123456789",
  "orderDate": "2025-01-01"
}
```

תוצאה מעובדת:
```
לקוח/ה יקר/ה,

תודה שרכשתם את המוצר שלנו: שעון חכם.
מספר הזמנה: A123456789
תאריך הזמנה: 2025-01-01

אנו מאחלים לכם שימוש מהנה!
```

### גישה לאובייקטי משנה

אם מערך הנתונים מכיל אובייקטי משנה, תוכלו לגשת למאפיינים שלהם באמצעות סימון נקודה.

**תחביר**: `{d.parent.child}`

**דוגמה**:

מערך נתונים:
```json
{
  "customer": {
    "name": "לי לי",
    "contact": {
      "email": "lilei@example.com",
      "phone": "13800138000"
    }
  }
}
```

תוכן התבנית:
```
שם לקוח: {d.customer.name}
כתובת אימייל: {d.customer.contact.email}
מספר טלפון: {d.customer.contact.phone}
```

תוצאה מעובדת:
```
שם לקוח: לי לי
כתובת אימייל: lilei@example.com
מספר טלפון: 13800138000
```

### גישה למערכים

אם מערך הנתונים מכיל מערכים, תוכלו להשתמש במילת המפתח השמורה `i` כדי לגשת לאלמנטים במערך.

**תחביר**: `{d.arrayName[i].field}`

**דוגמה**:

מערך נתונים:
```json
{
  "staffs": [
    { "firstname": "James", "lastname": "Anderson" },
    { "firstname": "Emily", "lastname": "Roberts" },
    { "firstname": "Michael", "lastname": "Johnson" }
  ]
}
```

תוכן התבנית:
```
שם המשפחה של העובד הראשון הוא {d.staffs[i=0].lastname}, ושמו הפרטי הוא {d.staffs[i=0].firstname}
```

תוצאה מעובדת:
```
שם המשפחה של העובד הראשון הוא Anderson, ושמו הפרטי הוא James
```
