## הוראות הגדרה

### הפעלת הדפסת תבניות
הדפסת תבניות תומכת כרגע בבלוקי פרטים ובבלוקי טבלה. להלן נפרט את דרכי ההגדרה עבור שני סוגי הבלוקים הללו.

#### בלוקי פרטים

1. **פתיחת בלוק הפרטים**:
- באפליקציה, עברו לבלוק הפרטים שבו תרצו להשתמש בתכונת הדפסת התבניות.

2. **כניסה לתפריט פעולות ההגדרה**:
- בחלק העליון של הממשק, לחצו על תפריט "פעולות הגדרה".

3. **בחירת "הדפסת תבניות"**:
- בתפריט הנפתח, לחצו על האפשרות "הדפסת תבניות" כדי להפעיל את פונקציונליות ה<תוסף>.

![הפעלת הדפסת תבניות](https://static-docs.nocobase.com/20241212150539-2024-12-12-15-05-43.png)

### הגדרת תבניות

1. **כניסה לדף הגדרת התבניות**:
- בתפריט ההגדרה של כפתור "הדפסת תבניות", בחרו באפשרות "הגדרת תבניות".

![אפשרות הגדרת תבניות](https://static-docs.nocobase.com/20241212151858-2024-12-12-15-19-01.png)

2. **הוספת תבנית חדשה**:
- לחצו על כפתור "הוסף תבנית" כדי לעבור לדף הוספת התבנית.

![כפתור הוסף תבנית](https://static-docs.nocobase.com/20241212151243-2024-12-12-15-12-46.png)

3. **מילוי פרטי התבנית**:
- בטופס התבנית, מלאו את שם התבנית ובחרו את סוג התבנית (Word, Excel, PowerPoint).
- העלו את קובץ התבנית המתאים (תומך בפורמטים `.docx`, `.xlsx`, `.pptx`).

![הגדרת שם וקובץ התבנית](https://static-docs.nocobase.com/20241212151518-2024-12-12-15-15-21.png)

4. **עריכה ושמירת התבנית**:
- עברו לדף "רשימת שדות", העתיקו את השדות ומלאו אותם בתבנית.
  ![רשימת שדות](https://static-docs.nocobase.com/20250107141010.png)
  ![20241212152743-2024-12-12-15-27-45](https://static-docs.nocobase.com/20241212152743-2024-12-12-15-27-45.png)
- לאחר מילוי הפרטים, לחצו על כפתור "שמור" כדי להשלים את הוספת התבנית.

5. **ניהול תבניות**:
- לחצו על כפתור "השתמש" בצד ימין של רשימת התבניות כדי להפעיל את התבנית.
- לחצו על כפתור "ערוך" כדי לשנות את שם התבנית או להחליף את קובץ התבנית.
- לחצו על כפתור "הורד" כדי להוריד את קובץ התבנית שהוגדר.
- לחצו על כפתור "מחק" כדי להסיר תבניות שאינן נחוצות יותר. המערכת תבקש אישור לפעולה כדי למנוע מחיקה בשוגג.
  ![ניהול תבניות](https://static-docs.nocobase.com/20250107140436.png)

#### בלוקי טבלה

השימוש בבלוקי טבלה זהה בעיקרו לשימוש בבלוקי פרטים, אך קיימים ההבדלים הבאים:
1. **תמיכה בהדפסת מספר רשומות**: עליכם לסמן תחילה את הרשומות שברצונכם להדפיס. ניתן להדפיס עד 100 רשומות בו-זמנית.
   
![20250416215633-2025-04-16-21-56-35](https://static-docs.nocobase.com/20250416215633-2025-04-16-21-56-35.png)

2. **ניהול תבניות מבודד**: תבניות עבור בלוקי טבלה ובלוקי פרטים אינן ניתנות להחלפה ביניהן – מכיוון שמבני הנתונים שונים (אחד הוא אובייקט, והשני הוא מערך).
