# יומן ביקורת

## מבוא

יומן הביקורת משמש לתיעוד ולמעקב אחר פעילויות משתמשים והיסטוריית פעולות על משאבים בתוך המערכת.

![](https://static-docs.nocobase.com/202501031627719.png)

![](https://static-docs.nocobase.com/202501031627922.png)

## תיאור פרמטרים

| פרמטר                 | תיאור                                                                                             |
| :-------------------- | :------------------------------------------------------------------------------------------------ |
| **Resource**          | סוג המשאב שעליו בוצעה הפעולה                                                                      |
| **Action**            | סוג הפעולה שבוצעה                                                                                 |
| **User**              | המשתמש שביצע את הפעולה                                                                            |
| **Role**              | תפקיד המשתמש בעת ביצוע הפעולה                                                                     |
| **Data source**       | מקור הנתונים                                                                                      |
| **Target collection** | אוסף היעד                                                                                         |
| **Target record UK**  | מזהה ייחודי של אוסף היעד                                                                          |
| **Source collection** | אוסף המקור של שדה הקישור                                                                          |
| **Source record UK**  | מזהה ייחודי של אוסף המקור                                                                         |
| **Status**            | קוד סטטוס HTTP של תגובת בקשת הפעולה                                                               |
| **Created at**        | זמן הפעולה                                                                                        |
| **UUID**              | מזהה ייחודי של הפעולה, תואם ל-Request ID של בקשת הפעולה, וניתן להשתמש בו לאחזור יומני יישום        |
| **IP**                | כתובת ה-IP של המשתמש                                                                              |
| **UA**                | פרטי ה-UA של המשתמש                                                                               |
| **Metadata**          | מטא-נתונים כגון פרמטרים, גוף בקשה ותוכן תגובה של בקשת הפעולה                                      |

## תיאור משאבי ביקורת

נכון לעכשיו, פעולות המשאבים הבאות יתועדו ביומן הביקורת:

### יישום ראשי

| פעולה             | תיאור                 |
| :--------------- | :-------------------- |
| `app:resart`     | הפעלה מחדש של היישום  |
| `app:clearCache` | ניקוי מטמון היישום    |

### מנהל תוספים

| פעולה         | תיאור        |
| :----------- | :----------- |
| `pm:add`     | הוספת תוסף   |
| `pm:update`  | עדכון תוסף   |
| `pm:enable`  | הפעלת תוסף   |
| `pm:disable` | השבתת תוסף   |
| `pm:remove`  | הסרת תוסף    |

### אימות משתמשים

| פעולה                  | תיאור        |
| :-------------------- | :----------- |
| `auth:signIn`         | כניסה        |
| `auth:signUp`         | הרשמה        |
| `auth:signOut`        | יציאה        |
| `auth:changePassword` | שינוי סיסמה  |

### משתמש

| פעולה                  | תיאור          |
| :-------------------- | :------------- |
| `users:updateProfile` | עדכון פרופיל   |

### תצורת ממשק משתמש

| פעולה                       | תיאור                      |
| :------------------------- | :------------------------- |
| `uiSchemas:insertAdjacent` | הוספת סכימת ממשק משתמש    |
| `uiSchemas:patch`          | שינוי סכימת ממשק משתמש    |
| `uiSchemas:remove`         | הסרת סכימת ממשק משתמש     |

### פעולות אוסף

| פעולה             | תיאור                     |
| :--------------- | :------------------------ |
| `create`         | יצירת רשומה                |
| `update`         | עדכון רשומה               |
| `destroy`        | מחיקת רשומה               |
| `updateOrCreate` | עדכון או יצירת רשומה      |
| `firstOrCreate`  | שאילתה או יצירת רשומה     |
| `move`           | העברת רשומה               |
| `set`            | הגדרת רשומת שדה קישור     |
| `add`            | הוספת רשומת שדה קישור     |
| `remove`         | הסרת רשומת שדה קישור      |
| `export`         | ייצוא רשומה               |
| `import`         | ייבוא רשומה               |

## הוספת משאבי ביקורת נוספים

אם הרחבתם פעולות משאבים נוספות באמצעות תוספים וברצונכם לתעד פעולות משאבים אלו ביומן הביקורת, תוכלו לעיין ב- [API](/api/server/audit-manager.md).
