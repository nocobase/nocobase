# נעילת משתמשים

## מבוא

נהלו משתמשים נעולים ונעלו משתמשים באופן יזום.

![](https://static-docs.nocobase.com/202412281450083.png)

### שחרור נעילת משתמש

משתמשים שננעלו עקב חריגה ממגבלת ניסיונות התחברות עם סיסמה שגויה, ניתנים לשחרור על ידי מחיקת הרשומה שלהם.

### נעילת משתמש

הוספת רשומת משתמש מאפשרת לכם לנעול משתמש באופן יזום. לאחר שמשתמש ננעל, המערכת תמנע ממנו גישה בכל שיטות האימות, כולל מפתחות API.

![](https://static-docs.nocobase.com/202412281450512.png)
