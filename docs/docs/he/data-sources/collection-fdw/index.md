# חיבור טבלאות נתונים חיצוניות (FDW)

## מבוא

זהו תוסף המאפשר חיבור לטבלאות נתונים מרוחקות, המבוסס על מימוש Foreign Data Wrapper (FDW) של מסד הנתונים. נכון לעכשיו, הוא תומך במסדי נתונים MySQL ו-PostgreSQL.

:::info{title="חיבור מקורות נתונים מול חיבור טבלאות נתונים חיצוניות"}
- **חיבור מקורות נתונים** מתייחס ליצירת חיבור למסד נתונים ספציפי או לשירות API, ומאפשר שימוש מלא בתכונות מסד הנתונים או בשירותים המסופקים על ידי ה-API;
- **חיבור טבלאות נתונים חיצוניות** מתייחס לקבלת נתונים ממקור חיצוני ומיפויים לשימוש מקומי. במסד הנתונים, זה נקרא FDW (Foreign Data Wrapper), זוהי טכנולוגיית מסדי נתונים המתמקדת בשימוש בטבלאות מרוחקות כטבלאות מקומיות, ומאפשרת חיבור טבלה אחר טבלה בלבד. מכיוון שמדובר בגישה מרוחקת, יהיו מגבלות ואילוצים שונים בשימוש בה.

ניתן גם להשתמש בשני המושגים בשילוב. הראשון משמש ליצירת חיבור למקור הנתונים, והשני משמש לגישה בין מקורות נתונים שונים. לדוגמה, אם מחובר מקור נתונים מסוג PostgreSQL, וטבלה מסוימת במקור נתונים זה היא טבלת נתונים חיצונית שנוצרה באמצעות FDW.
:::

### MySQL

MySQL משתמש במנוע ה-`federated`, שדורש הפעלה, ותומך בחיבור למסדי נתונים מרוחקים של MySQL ומסדי נתונים תואמי פרוטוקול, כגון MariaDB. לפרטים נוספים, עיין בתיעוד [Federated Storage Engine](https://dev.mysql.com/doc/refman/8.0/en/federated-storage-engine.html).

### PostgreSQL

ב-PostgreSQL, ניתן להשתמש בסוגים שונים של הרחבות `fdw` כדי לתמוך בסוגי נתונים מרוחקים שונים. ההרחבות הנתמכות כרגע כוללות:

- [postgres_fdw](https://www.postgresql.org/docs/current/postgres-fdw.html): חיבור למסד נתונים מרוחק של PostgreSQL מתוך PostgreSQL.
- [mysql_fdw(בפיתוח)](https://github.com/EnterpriseDB/mysql_fdw): חיבור למסד נתונים מרוחק של MySQL מתוך PostgreSQL.
- עבור סוגים אחרים של הרחבות fdw, עיין ב-[PostgreSQL Foreign Data Wrappers](https://wiki.postgresql.org/wiki/Foreign_data_wrappers). חיבור ל-NocoBase דורש מימוש ממשק ההתאמה המתאים בקוד.

## התקנה

דרישות קדם

- אם מסד הנתונים הראשי של NocoBase הוא MySQL, יש להפעיל את `federated`. עיין ב-[כיצד להפעיל את מנוע ה-federated ב-MySQL](./enable-federated.md)

לאחר מכן, התקן והפעל את התוסף באמצעות מנהל התוספים

## מדריך למשתמש

תחת "ניהול אוספים > יצירת אוסף" בתפריט הנפתח, בחר "חיבור נתונים חיצוניים"

![חיבור נתונים חיצוניים](https://static-docs.nocobase.com/029d946a6d067d1c35a39755219d623c.png)

בתפריט הנפתח "שירות מסד נתונים", בחר שירות מסד נתונים קיים, או "צור שירות מסד נתונים"

![שירות מסד נתונים](https://static-docs.nocobase.com/766271708a911950a5599d60d6be4a4d.png)

![צור שירות מסד נתונים](https://static-docs.nocobase.com/1e357216e04cc4f200bd6212827281c8.png)
