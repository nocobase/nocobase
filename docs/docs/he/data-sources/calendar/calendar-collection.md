# אוסף לוח שנה

## מבוא

אוסף לוח שנה הוא טבלת נתונים ייעודית שנועדה לאחסן תאריכים ומידע הקשור לתאריכים. הוא משמש בדרך כלל לניהול ומעקב אחר זמן בתוך יישומים או מערכות. המטרה העיקרית של אוסף לוח השנה היא לספק גישה גמישה ויעילה למידע תאריכים, כדי לאפשר שליפה מהירה של נתונים רלוונטיים בעת הצורך.
