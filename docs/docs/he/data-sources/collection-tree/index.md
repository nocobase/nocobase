# אוסף עץ

## מבוא

אוסף עץ הוא תבנית עיצוב של אוסף נתונים המשמש לארגון נתונים היררכיים. מבנה אוסף זה משקף עץ, שבו כל פריט נתונים יכול להכיל פריט/ים צאצא/ים אחד/ים או יותר, ופריטי צאצא אלו יכולים, בתורם, להכיל צאצאים משלהם.
