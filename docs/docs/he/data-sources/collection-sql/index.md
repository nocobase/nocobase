# אוסף SQL

## מבוא

אוסף SQL מספק שיטה עוצמתית לאחזור נתונים באמצעות שאילתות SQL. על ידי חילוץ שדות נתונים דרך שאילתות SQL וקביעת תצורה של מטא-נתונים קשורים לשדות, משתמשים יכולים להשתמש בשדות אלה כאילו הם עובדים עם טבלה רגילה. תכונה זו מועילה במיוחד לתרחישים הכוללים שאילתות צירוף מורכבות, ניתוח סטטיסטי ועוד.

## מדריך למשתמש

### יצירת אוסף SQL חדש

<img src="https://static-docs.nocobase.com/202405191452918.png"/>

1.  הזינו את שאילתת ה-SQL שלכם בתיבת הקלט המיועדת ולחצו על 'בצע' (Execute). המערכת תנתח את השאילתה כדי לקבוע אילו טבלאות ושדות מעורבים, ותחלץ אוטומטית את מטא-הנתונים הרלוונטיים של השדות מטבלאות המקור.

<img src="https://static-docs.nocobase.com/202405191453556.png"/>

2.  אם ניתוח המערכת של טבלאות ושדות המקור אינו מדויק, באפשרותכם לבחור ידנית את הטבלאות והשדות המתאימים כדי להבטיח שימוש במטא-הנתונים הנכונים. התחילו בבחירת טבלת המקור, ולאחר מכן בחרו את השדות המתאימים בקטע 'מקור שדה' למטה.

<img src="https://static-docs.nocobase.com/202405191453579.png"/>

3.  עבור שדות שאין להם מקור ישיר, המערכת תסיק את סוג השדה בהתבסס על סוג הנתונים. אם הסקה זו אינה נכונה, באפשרותכם לבחור ידנית את סוג השדה המתאים.

<img src="https://static-docs.nocobase.com/202405191454703.png"/>

4.  בזמן שאתם מגדירים כל שדה, תוכלו לצפות בתצוגה המקדימה שלו באזור התצוגה המקדימה, מה שיאפשר לכם לראות את ההשפעה המיידית של ההגדרות שלכם.

<img src="https://static-docs.nocobase.com/202405191455439.png"/>

5.  לאחר שסיימתם את ההגדרה ואישרתם שהכל תקין, לחצו על כפתור 'אשר' (Confirm) מתחת לתיבת קלט ה-SQL כדי לסיים את השליחה.

<img src="https://static-docs.nocobase.com/202405191455302.png"/>

### עריכה

1.  אם עליכם לשנות את שאילתת ה-SQL, לחצו על כפתור 'ערוך' (Edit) כדי לשנות ישירות את הצהרת ה-SQL ולהגדיר מחדש את השדות לפי הצורך.

2.  כדי להתאים את מטא-הנתונים של השדות, השתמשו באפשרות 'הגדר שדות' (Configure fields), המאפשרת לכם לעדכן את הגדרות השדה בדיוק כפי שהייתם עושים עבור טבלה רגילה.

### סנכרון

אם שאילתת ה-SQL נשארת ללא שינוי אך מבנה טבלת מסד הנתונים הבסיסי השתנה, באפשרותכם לסנכרן ולהגדיר מחדש את השדות על ידי בחירה ב'הגדר שדות' (Configure fields) - 'סנכרן ממסד נתונים' (Sync from database).

<img src="https://static-docs.nocobase.com/202405191456216.png"/>

| :--- | :--- | :--- | :--- |
| SQL | מודלים פשוטים, מקרי שימוש קלים<br />אינטראקציה מוגבלת עם מסד הנתונים<br />הימנעות מתחזוקת תצוגות<br />העדפה לפעולות מונחות ממשק משתמש | שאילתת SQL משנית | לא נתמך |

בעת שימוש באוסף SQL, הקפידו לבחור טבלאות הניתנות לניהול בתוך NocoBase. שימוש בטבלאות מאותו מסד נתונים שאינן מחוברות ל-NocoBase עלול להוביל לניתוח שגוי של שאילתות SQL. אם זו דאגה, שקלו ליצור ולקשר לתצוגה.
:::
