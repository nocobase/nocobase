# סקירת אוספים

NocoBase מספקת DSL ייחודי לתיאור מבנה הנתונים, המכונה אוסף, המאחד את מבני הנתונים ממקורות שונים ומספק בסיס אמין לניהול נתונים, ניתוח ויישומים.

![20240512161522](https://static-docs.nocobase.com/20240512161522.png)

כדי להשתמש בנוחות במודלי נתונים שונים, NocoBase תומכת ביצירת סוגים שונים של אוספים:

- [אוסף רגיל](/data-sources/data-source-main/general-collection): כולל שדות מערכת נפוצים מובנים;
- [אוסף ירושה](/data-sources/data-source-main/inheritance-collection): מאפשר ליצור אוסף אב, שממנו ניתן לגזור אוספי בן. אוסף הבן יירש את מבנה אוסף האב ובנוסף יוכל להגדיר עמודות משלו.
- [אוסף עץ](/data-sources/collection-tree): אוסף בעל מבנה עץ, התומך כרגע רק בעיצוב רשימת סמיכות;
- [אוסף לוח שנה](/data-sources/calendar/calendar-collection): משמש ליצירת אוספי אירועים הקשורים ללוח שנה;
- [אוסף קבצים](/data-sources/file-manager/file-collection): משמש לניהול אחסון קבצים;
- : משמש לתרחישי ביטויים דינמיים בתהליכי עבודה;
- [אוסף SQL](/data-sources/collection-sql): אינו אוסף מסד נתונים ממשי, אלא מציג שאילתות SQL בצורה מובנית ומהירה;
- [אוסף תצוגה](/data-sources/collection-view): מתחבר לתצוגות מסד נתונים קיימות;
- [אוסף חיצוני](/data-sources/collection-fdw): מאפשר למערכת מסד הנתונים לגשת ולשאול נתונים ישירות ממקורות נתונים חיצוניים, בהתבסס על טכנולוגיית FDW.
