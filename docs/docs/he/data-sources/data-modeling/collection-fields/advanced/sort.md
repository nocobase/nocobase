# בורר אוספים

שדה בורר האוספים הוא סוג שדה מיוחד המאפשר למשתמשים לבחור אוסף אחד או יותר מתוך רשימה מוגדרת מראש. סוג שדה זה שימושי מאוד בתרחישים שבהם יש צורך להתייחס באופן דינמי לאוספים שונים, כגון בניית דוחות הניתנים להגדרה, לוחות מחוונים (dashboards), או יישומים מורכבים שצריכים לשנות מקורות נתונים בהתבסס על בחירת המשתמש.

## תרחישי שימוש

### יצירת דוחות דינמית

במודול יצירת דוחות, משתמשים עשויים להזדקק לבחור אוסף מתוך מספר אוספים כמקור הנתונים עבור הדוח. באמצעות בורר האוספים, משתמשים יכולים לבחור בקלות את האוסף הרצוי, והמערכת תיצור את הדוח באופן דינמי בהתבסס על בחירת המשתמש.

### לוחות מחוונים הניתנים להגדרה

ביישום לוח מחוונים, משתמשים יכולים להתאים אישית את הנתונים המוצגים בלוח המחוונים. בורר האוספים מאפשר למשתמשים לבחור את האוספים שבהם הם מעוניינים, ובכך להציג תרשימים ונתונים סטטיסטיים ממקורות נתונים שונים בלוח המחוונים.

### כלי ייבוא/ייצוא נתונים גמישים

בעת פיתוח כלי כללי לייבוא או ייצוא נתונים, ניתן להשתמש בבורר האוספים כדי לאפשר למשתמשים לציין את האוסף שעליו יש לבצע את הפעולה. בדרך זו, ניתן להשתמש באותו כלי כדי לעבד נתונים מכל אוסף במערכת, מה שמשפר מאוד את יכולת השימוש החוזר של הכלי.
