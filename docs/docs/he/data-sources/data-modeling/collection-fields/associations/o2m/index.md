## תיאור פרמטרים

### אוסף מקור

אוסף המקור, כלומר האוסף שבו נמצא השדה הנוכחי.

### אוסף יעד

אוסף היעד, כלומר האוסף שאליו מקשרים.

### מפתח מקור

השדה באוסף המקור שאליו מתייחס מפתח החוץ. עליו להיות ייחודי (Unique).

### מפתח חוץ

השדה באוסף היעד המשמש ליצירת הקשר בין שני האוספים.

### מפתח יעד

השדה באוסף היעד המשמש לצפייה בכל רשומת שורה בבלוק הקשרים, ובדרך כלל מדובר בשדה ייחודי.

### ON DELETE

`ON DELETE` מתייחס לכללים המופעלים על הפניות של מפתחות חוץ באוספי צאצא קשורים, כאשר רשומות באוסף האב נמחקות. זוהי אפשרות המשמשת בעת הגדרת אילוץ מפתח חוץ. אפשרויות `ON DELETE` נפוצות כוללות:

- **CASCADE**: כאשר רשומה באוסף האב נמחקת, כל הרשומות הקשורות באוסף הצאצא נמחקות באופן אוטומטי.
- **SET NULL**: כאשר רשומה באוסף האב נמחקת, ערכי מפתח החוץ ברשומות הקשורות באוסף הצאצא מוגדרים ל-`NULL`.
- **RESTRICT**: אפשרות ברירת המחדל. היא מונעת מחיקה של רשומה באוסף האב אם קיימות רשומות קשורות באוסף הצאצא.
- **NO ACTION**: בדומה ל-`RESTRICT`, היא מונעת מחיקה של רשומה באוסף האב אם קיימות רשומות קשורות באוסף הצאצא.
