# שדה קובץ מצורף

## מבוא

המערכת כוללת שדה מובנה מסוג "קובץ מצורף", המאפשר למשתמשים להעלות קבצים לאוספים מותאמים אישית.

מתחת למכסה המנוע, שדה הקובץ המצורף הוא שדה יחס מסוג רבים-לרבים, המצביע אל אוסף הקבצים המצורפים (`attachments`) המובנה של המערכת. כאשר אתם יוצרים שדה קובץ מצורף בכל אוסף, טבלת צימוד מסוג רבים-לרבים נוצרת באופן אוטומטי. מטא-הנתונים של הקבצים שהועלו נשמרים באוסף "קבצים מצורפים", ופרטי הקבצים שאליהם מתייחסים באוסף שלכם מקושרים דרך טבלת צימוד זו.

### הגבלת סוגי MIME

משמש להגבלת סוגי הקבצים המותרים להעלאה, באמצעות תחביר [MIME](https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/MIME_types). לדוגמה, `image/*` מייצג קובצי תמונה. ניתן להפריד בין מספר סוגים באמצעות פסיקים, כגון `image/*,application/pdf`, המאפשר קובצי תמונה וקובצי PDF.

### מנוע אחסון

בחרו את מנוע האחסון עבור הקבצים שהועלו. אם השדה נשאר ריק, ייעשה שימוש במנוע האחסון ברירת המחדל של המערכת.
