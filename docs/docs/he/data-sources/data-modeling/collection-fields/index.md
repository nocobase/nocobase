# שדות אוסף

## סוגי ממשק של שדות

NocoBase מסווגת שדות לקטגוריות הבאות מנקודת מבט הממשק:

![20240512110352](https://static-docs.nocobase.com/20240512110352.png)

## סוגי נתונים של שדות

לכל סוג ממשק (Field Interface) יש סוג נתונים ברירת מחדל. לדוגמה, עבור שדות עם ממשק מסוג מספר (Number), סוג הנתונים המוגדר כברירת מחדל הוא double, אך הוא יכול להיות גם float, decimal וכו'. סוגי הנתונים הנתמכים כרגע הם:

![20240512103733](https://static-docs.nocobase.com/20240512103733.png)

## מיפוי סוגי שדות

התהליך להוספת שדות חדשים למסד הנתונים הראשי הוא כדלקמן:

1.  בחרו את סוג הממשק (Interface type)
2.  הגדירו את סוג הנתונים האפשרי עבור הממשק הנוכחי

![20240512172416](https://static-docs.nocobase.com/20240512172416.png)

תהליך מיפוי השדות ממקורות נתונים חיצוניים הוא:

1.  מפו באופן אוטומטי את סוג הנתונים (Field type) וסוג ממשק המשתמש (Field Interface) המתאימים, בהתבסס על סוג השדה של מסד הנתונים החיצוני.
2.  שנו לסוג נתונים וסוג ממשק מתאימים יותר, לפי הצורך.

![20240512172759](https://static-docs.nocobase.com/20240512172759.png)
