# אוסף קבצים

## מבוא

קבצים הם רשומות באוסף בעל מבנה מיוחד, המכונה אוסף קבצים. הוא משמש לאחסון מטא-נתונים של קבצים וניתן לנהל אותו באמצעות מנהל הקבצים (File Manager).
