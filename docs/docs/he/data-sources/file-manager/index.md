# מנהל קבצים

## מבוא

תוסף מנהל הקבצים מספק אוסף קבצים, שדה קבצים מצורפים ומנועי אחסון קבצים, לניהול יעיל של קבצים. קבצים הם רשומות בתוך אוסף מסוג מיוחד, המכונה אוסף קבצים, אשר מאחסן מטא-נתונים של קבצים וניתן לנהל אותו באמצעות מנהל הקבצים. שדות קבצים מצורפים הם שדות קישור ספציפיים המקושרים לאוסף הקבצים. התוסף תומך בשיטות אחסון מרובות, כולל אחסון מקומי, Alibaba Cloud OSS, Amazon S3 ו-Tencent Cloud COS.

## מדריך למשתמש

### אוסף קבצים

אוסף attachments מובנה קיים לאחסון כל הקבצים המקושרים לשדות קבצים מצורפים. בנוסף, ניתן ליצור אוספי קבצים חדשים לאחסון קבצים ספציפיים.

[קראו עוד בתיעוד אוסף הקבצים](/data-sources/file-manager/file-collection)

### שדה קבצים מצורפים

שדות קבצים מצורפים הם שדות קישור ספציפיים המקושרים לאוסף הקבצים, וניתן ליצור אותם באמצעות סוג השדה "קובץ מצורף" או להגדיר אותם באמצעות שדה "קישור".

[קראו עוד בתיעוד שדה הקבצים המצורפים](/data-sources/file-manager/field-attachment)

### מנוע אחסון קבצים

מנוע אחסון הקבצים משמש לשמירת קבצים בשירותים ספציפיים, כולל אחסון מקומי (שמירה לדיסק הקשיח של השרת), אחסון בענן, ועוד.

[קראו עוד בתיעוד מנוע אחסון הקבצים](./storage/index.md)

### API HTTP

העלאות קבצים ניתנות לטיפול באמצעות API HTTP, ראו [API HTTP](./http-api.md).
