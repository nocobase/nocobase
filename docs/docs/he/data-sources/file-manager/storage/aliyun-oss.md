# Aliyun OSS

מנוע אחסון המבוסס על Aliyun OSS. לפני השימוש, עליכם להכין את החשבונות וההרשאות הנדרשים.

## הגדרות

![דוגמה להגדרות מנוע אחסון Aliyun OSS](https://static-docs.nocobase.com/20240712220011.png)

:::info{title=הערה}
כאן מוצגים רק הפרמטרים הספציפיים למנוע אחסון Aliyun OSS. לפרמטרים כלליים, אנא עיינו ב[פרמטרים כלליים של המנוע](./index.md#common-engine-parameters).
:::

### אזור

ציינו את אזור האחסון של OSS, לדוגמה: `oss-cn-hangzhou`.

:::info{title=הערה}
תוכלו לצפות בפרטי האזור של דלי האחסון ב[קונסולת Aliyun OSS](https://oss.console.aliyun.com/). עליכם להשתמש רק בחלק הקידומת של האזור (ללא שם הדומיין המלא).
:::

### AccessKey ID

מלאו את ה-ID של מפתח הגישה המורשה של Alibaba Cloud.

### AccessKey Secret

מלאו את ה-Secret של מפתח הגישה המורשה של Alibaba Cloud.

### דלי

מלאו את שם הדלי של אחסון OSS.
