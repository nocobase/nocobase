# אחסון מקומי

קבצים שמועלים יישמרו בספרייה מקומית על השרת. זה מתאים לתרחישים ניסיוניים או בקנה מידה קטן, שבהם כמות הקבצים הכוללת המנוהלת על ידי המערכת קטנה יחסית.

## אפשרויות

![דוגמה לאפשרויות מנוע אחסון קבצים](https://static-docs.nocobase.com/20240529115151.png)

:::info{title=הערה}
סעיף זה מתאר רק את האפשרויות הספציפיות למנוע האחסון המקומי. עבור פרמטרים כלליים, אנא עיינו ב[פרמטרים כלליים של המנוע](./index.md#general-engine-parameters).
:::

### נתיב

הנתיב מייצג הן את הנתיב היחסי של הקובץ המאוחסן בשרת והן את נתיב הגישה ב-URL. לדוגמה, "`user/avatar`" (ללא ה-"` / `" בהתחלה ובסוף) מייצג:

1. הנתיב היחסי של הקובץ שהועלה המאוחסן בשרת: `/path/to/nocobase-app/storage/uploads/user/avatar`.
2. קידומת ה-URL לגישה לקובץ: `http://localhost:13000/storage/uploads/user/avatar`.
