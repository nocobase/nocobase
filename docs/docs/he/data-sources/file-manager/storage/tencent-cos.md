# Tencent COS

מנוע אחסון המבוסס על Tencent Cloud COS. לפני השימוש, יש להכין את החשבון וההרשאות הנדרשים.

## אפשרויות תצורה

![דוגמה לאפשרויות מנוע האחסון של Tencent COS](https://static-docs.nocobase.com/20240712222125.png)

:::info{title=טיפ}
חלק זה מציג רק את האפשרויות הספציפיות למנוע האחסון של Tencent Cloud COS. לפרמטרים כלליים, אנא עיין/י ב[פרמטרים כלליים של המנוע](./index.md#common-engine-parameters).
:::

### אזור

יש למלא את אזור האחסון של COS, לדוגמה: `ap-chengdu`.

:::info{title=טיפ}
ניתן לצפות בפרטי האזור של דלי האחסון ב[קונסולת Tencent Cloud COS](https://console.cloud.tencent.com/cos). יש להשתמש רק בחלק הקידומת של האזור (אין צורך בשם הדומיין המלא).
:::

### SecretId

יש למלא את ה-ID של מפתח הגישה המורשה של Tencent Cloud.

### SecretKey

יש למלא את ה-Secret של מפתח הגישה המורשה של Tencent Cloud.
