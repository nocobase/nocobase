# אוסף תגובות

## מבוא

אוסף תגובות הוא תבנית אוסף נתונים ייעודית, המיועדת לאחסון תגובות ומשוב ממשתמשים. באמצעות תכונת התגובות, תוכלו להוסיף יכולות תגובה לכל אוסף נתונים, ולאפשר למשתמשים לדון, לספק משוב או להוסיף הערות לרשומות ספציפיות. אוסף התגובות תומך בעריכת טקסט עשיר, ומספק יכולות יצירת תוכן גמישות.

![comment-collection-2025-11-01-00-39-01](https://static-docs.nocobase.com/comment-collection-2025-11-01-00-39-01.png)

## תכונות

- **עריכת טקסט עשיר**: כולל עורך Markdown (vditor) כברירת מחדל, ותומך ביצירת תוכן טקסט עשיר.
- **קישור לכל אוסף נתונים**: ניתן לשייך תגובות לרשומות בכל אוסף נתונים באמצעות שדות יחסים.
- **תגובות מרובות רמות**: תומך במענה לתגובות, ובניית מבנה עץ תגובות.
- **מעקב משתמשים**: מתעד אוטומטית את יוצר התגובה ואת זמן יצירתה.

## מדריך למשתמש

### יצירת אוסף תגובות

![comment-collection-2025-11-01-00-37-10](https://static-docs.nocobase.com/comment-collection-2025-11-01-00-37-10.png)

1. עברו לדף ניהול אוספי הנתונים.
2. לחצו על כפתור "צור אוסף".
3. בחרו את תבנית "אוסף תגובות".
4. הזינו את שם האוסף (לדוגמה: "תגובות משימה", "תגובות מאמר" וכו').
5. המערכת תיצור אוטומטית אוסף תגובות עם השדות הבאים כברירת מחדל:
   - תוכן תגובה (סוג Markdown vditor)
   - נוצר על ידי (מקשר לאוסף משתמשים)
   - נוצר בתאריך (סוג תאריך ושעה)

### הגדרת יחסי גומלין

כדי לאפשר לתגובות להיות מקושרות לאוסף נתונים יעד, עליכם להגדיר שדות יחסים:

![](https://static-docs.nocobase.com/Solution/demoE3v1-19N.gif)

1. הוסיפו שדה יחסים מסוג "רבים לאחד" באוסף התגובות.
2. בחרו את אוסף נתוני היעד שאליו תרצו לקשר (לדוגמה: אוסף משימות, אוסף מאמרים וכו').
3. הגדירו את שם השדה (לדוגמה: "שייך למשימה", "שייך למאמר" וכו').

### שימוש בבלוקי תגובות בדפים

![Enable Comments Collection](https://static-docs.nocobase.com/Solution/demoE3v1-20.gif)

1. עברו לדף שבו תרצו להוסיף את פונקציונליות התגובות.
2. הוסיפו בלוק בפרטי הרשומה או בחלון הקופץ שלה.
3. בחרו את סוג הבלוק "תגובות".
4. בחרו את אוסף התגובות שיצרתם זה עתה.

### תרחישי שימוש נפוצים

- **מערכות לניהול משימות**: חברי צוות דנים ומספקים משוב על משימות.
- **מערכות לניהול תוכן**: קוראים מגיבים ויוצרים אינטראקציה עם מאמרים.
- **תהליכי עבודה של אישורים**: מאשרים מוסיפים הערות וחוות דעת על טפסי בקשה.
- **משוב לקוחות**: איסוף ביקורות לקוחות על מוצרים או שירותים.

## הערות

- אוסף תגובות הוא תכונה של תוסף מסחרי, ודורש הפעלה של תוסף התגובות כדי להשתמש בו.
- מומלץ להגדיר הרשאות מתאימות לאוסף התגובות, כדי לשלוט מי יכול לצפות, ליצור ולמחוק תגובות.
- בתרחישים עם מספר רב של תגובות, מומלץ להפעיל טעינה מדורגת (pagination) לשיפור הביצועים.
