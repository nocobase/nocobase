# תהליך הגדרה

## סקירה כללית
לאחר הפעלת תוסף המייל, על מנהלי המערכת להשלים תחילה את ההגדרות הנדרשות, ורק אז יוכלו משתמשים רגילים לחבר את חשבונות המייל שלהם ל-NocoBase. (נכון לעכשיו, נתמכת רק התחברות מאושרת (Authorization) עבור חשבונות Outlook ו-Gmail; התחברות ישירה באמצעות חשבונות Microsoft ו-Google אינה זמינה עדיין).

ליבת ההגדרה טמונה בהגדרות האימות (Authentication) עבור קריאות ה-API של ספק שירותי המייל. מנהלי המערכת צריכים להשלים את השלבים הבאים כדי להבטיח שהתוסף יפעל כראוי:

1.  **קבלת פרטי אימות מספק השירות**
    -   היכנסו לקונסולת המפתחים של ספק שירותי המייל (לדוגמה, Google Cloud Console או Microsoft Azure Portal).
    -   צרו יישום או פרויקט חדש והפעילו את שירות ה-API של Gmail או Outlook.
    -   קבלו את מזהה הלקוח (Client ID) והסוד הלקוח (Client Secret) המתאימים.
    -   הגדירו את ה-URI להפניה מחדש (Redirect URI) וודאו שהוא תואם לכתובת הקריאה החוזרת (callback address) של תוסף NocoBase.

2.  **הגדרת ספק שירותי המייל**
    -   עברו לדף ההגדרות של תוסף המייל.
    -   ספקו את פרטי אימות ה-API הנדרשים, כולל מזהה הלקוח (Client ID) והסוד הלקוח (Client Secret), כדי להבטיח אימות תקין מול ספק שירותי המייל.

3.  **כניסה מאושרת**
    -   משתמשים מתחברים לחשבונות המייל שלהם באמצעות פרוטוקול OAuth.
    -   התוסף ייצור ויאחסן באופן אוטומטי את אסימון האימות (authorization token) של המשתמש, המשמש לקריאות API ופעולות מייל עתידיות.

4.  **חיבור חשבונות מייל**
    -   לאחר אימות מוצלח, חשבון המייל של המשתמש יחובר ל-NocoBase.
    -   התוסף יסנכרן את נתוני המייל של המשתמש ויספק פונקציונליות לניהול, שליחה וקבלה של מיילים.

5.  **שימוש בתכונות המייל**
    -   משתמשים יכולים לצפות, לנהל ולשלוח מיילים ישירות מתוך הפלטפורמה.
    -   כל הפעולות מתבצעות באמצעות קריאות API של ספק שירותי המייל, מה שמבטיח סנכרון בזמן אמת והעברה יעילה.

באמצעות התהליך המתואר לעיל, תוסף המייל של NocoBase מספק למשתמשים שירותי ניהול מייל יעילים ומאובטחים. אם נתקלתם בבעיות כלשהן במהלך ההגדרה, אנא עיינו בתיעוד הרלוונטי או צרו קשר עם צוות התמיכה הטכנית לקבלת עזרה.

## הגדרות התוסף

### הפעלת תוסף המייל

1.  עברו לדף ניהול התוספים
2.  מצאו את התוסף "Email manager" והפעילו אותו

### הגדרת ספק שירותי המייל

לאחר הפעלת תוסף המייל, תוכלו להגדיר את ספקי שירותי המייל. נכון לעכשיו, נתמכים שירותי המייל של גוגל ומיקרוסופט. לחצו על "הגדרות" -> "הגדרות מייל" בסרגל העליון כדי לעבור לדף ההגדרות.

![](https://static-docs.nocobase.com/mail-1733818617187.png)

![](https://static-docs.nocobase.com/mail-1733818617514.png)

עבור כל ספק שירות, עליכם למלא את מזהה הלקוח (Client ID) והסוד הלקוח (Client Secret). הסעיפים הבאים יפרטו כיצד להשיג שני פרמטרים אלו.
