# הגדרות בלוקים

## בלוק הודעות דוא"ל

### הוספת בלוק

בדף ההגדרות, לחצו על כפתור **יצירת בלוק**, ובחרו בבלוק **הודעות דוא"ל (הכל)** או **הודעות דוא"ל (אישי)** כדי להוסיף בלוק הודעות דוא"ל.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_03_58_PM.png)

### הגדרות שדות

לחצו על כפתור **שדות** של הבלוק כדי לבחור את השדות שיוצגו. לפעולות מפורטות, עיינו בשיטת הגדרת השדות עבור טבלאות.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_37_PM.png)

### הגדרות סינון נתונים

לחצו על אייקון ההגדרות בצד ימין של הטבלה ובחרו **טווח נתונים** כדי להגדיר את טווח הנתונים לסינון הודעות דוא"ל.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_39_PM.png)

באפשרותכם לסנן הודעות דוא"ל עם אותה סיומת באמצעות משתנים:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_41_PM.png)

## בלוק פרטי דוא"ל

ראשית, הפעילו את התכונה **אפשרו לחיצה לפתיחה** על שדה בבלוק הודעות הדוא"ל:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_45_PM.png)

הוסיפו את בלוק **פרטי דוא"ל** בחלון הקופץ:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_46_PM.png)

תוכלו לצפות בתוכן המפורט של הודעת הדוא"ל:
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_04_49_PM.png)

בתחתית המסך תוכלו להגדיר את הכפתורים הנדרשים.

## בלוק שליחת דוא"ל

קיימות שתי דרכים ליצור טופס שליחת דוא"ל:

1.  הוסיפו כפתור **שליחת דוא"ל** בחלק העליון של הטבלה:
    ![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_52_PM.png)

2.  הוסיפו בלוק **שליחת דוא"ל**:
    ![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_53_PM.png)

שתי הדרכים מאפשרות ליצור טופס שליחת דוא"ל מלא:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_53_PM%20(1).png)

כל שדה בטופס הדוא"ל תואם לטופס רגיל וניתן להגדיר עבורו **ערך ברירת מחדל** או **כללי קישוריות**, ועוד.

> טופסי התשובה וההעברה המובנים בתחתית פרטי הדוא"ל כוללים עיבוד נתונים מסוים כברירת מחדל, וניתן לשנות אותם באמצעות **מנוע זרימה** (FlowEngine).
