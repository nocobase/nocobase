# שליחה בכמות גדולה

## סקירה כללית של שליחה בכמות גדולה
באפשרותכם לשלוח מיילים למספר רב של אנשים בבת אחת (כל נמען יקבל מייל נפרד).

![](https://static-docs.nocobase.com/email-manager/Loading--12-31-2025_10_15_PM.png)

## שליחה בכמות גדולה באמצעות טבלה
ניתן גם לשלוח מיילים בכמות גדולה דרך טבלה:

1. הוסיפו כפתור שליחת מייל לטבלה.
   ![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_04_PM.png)

2. הגדירו את שדה המייל בטבלה (משמש לזיהוי כתובות מייל לאחר בחירת שורות בטבלה).
   ![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_17_PM%20(1)

3. סמנו את נתוני הטבלה הרצויים לשליחת מיילים.
   ![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_19_PM.png)

אם אתם צריכים לשלוח מיילים לכל הכתובות בטבלה כולה, תוכלו להפעיל את התכונה **בחירת כל הרשומות בטבלה הנוכחית כנמענים כברירת מחדל**:
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_18_PM%20(1)

לאחר עריכת תוכן המייל, לחצו על כפתור השליחה בכמות גדולה כדי לשלוח.
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_19_PM.png)

## מעקב אחר שליחת מיילים
במרכז ניהול המיילים, תוכלו לצפות ולעקוב אחר סטטוס שליחת המיילים הספציפי.
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_20_PM.png).png)
![](https://static-docs.nocobase.com/email-manager/Users-10-31-2025_08_24_PM.png)
