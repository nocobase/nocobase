# שמירה אוטומטית של טיוטה

בתפריט של כפתור **שמור טיוטה**, תוכלו להפעיל את **שמירה אוטומטית של טיוטה**.
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_29_PM.png)

במהלך עריכת המייל, המערכת שומרת את הטיוטה בזמן אמת, כדי להבטיח שאף תוכן לא יאבד.
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_31_PM.png)

כאשר תפתחו שוב את עורך המייל (עבור אותו שולח ונמען), המערכת תשאל אם תרצו להשתמש בטיוטה השמורה.
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_08_35_PM.png)
