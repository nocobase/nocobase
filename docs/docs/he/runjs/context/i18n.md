# ctx.i18n

מופע ה-i18n של ההקשר הנוכחי, המשמש לקריאה או החלפה של שפות. **יש להשתמש ב-[ctx.t()](./t.md) עבור תרגום טקסטים**; אין להשתמש ב-`ctx.i18n.t`.

## תרחישי שימוש

ניתן להשתמש ב-`ctx.i18n` בכל סביבות ההרצה של RunJS (כגון JSBlock, JSField, JSItem, JSColumn, תהליך עבודה, כללי קישוריות וכו').

## הגדרת סוג

```typescript
interface i18n: {
  language: string;
  changeLanguage(lng: string): Promise<any>;
}
```

## מאפיינים נפוצים

| מאפיין | סוג | תיאור |
|------|------|------|
| `language` | `string` | קוד השפה הפעילה כעת (למשל `en-US`, `zh-CN`) |

## מתודות נפוצות

### changeLanguage(lng)

מחליף את השפה הנוכחית.

| פרמטר | סוג | תיאור |
|------|------|------|
| `lng` | `string` | קוד שפת היעד (למשל `'en-US'`, `'zh-CN'`) |

**ערך חזרה**: `Promise<any>`, מתבצע (resolves) לאחר סיום החלפת השפה.

## דוגמאות

### קריאת השפה הנוכחית

```ts
const lang = ctx.i18n.language;
// 'en-US' | 'zh-CN' | ...
if (lang.startsWith('zh')) {
  ctx.render(ctx.t('Chinese UI'));
} else {
  ctx.render(ctx.t('English UI'));
}
```

### החלפת שפה

```ts
// החלפה לאנגלית
await ctx.i18n.changeLanguage('en-US');

// החלפה לסינית
await ctx.i18n.changeLanguage('zh-CN');
```

### כפתור החלפת שפה

```ts
const { Button } = ctx.libs.antd;
const isZh = ctx.i18n.language.startsWith('zh');
ctx.render(
  <Button onClick={async () => {
    await ctx.i18n.changeLanguage(isZh ? 'en-US' : 'zh-CN');
  }}>
    {ctx.t(isZh ? 'Switch to English' : 'Switch to Chinese')}
  </Button>,
);
```

## הערות

- **טקסט תרגום**: יש להשתמש ב-`ctx.t()` באופן עקבי; אין להשתמש ב-`ctx.i18n.t`.

## נושאים קשורים

- [ctx.t()](./t.md): תרגום טקסט, השתמשו במתודה זו באופן עקבי.
