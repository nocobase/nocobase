# פריסת אשכול

NocoBase תומכת בפריסת יישומים במצב אשכול כדי לשפר את ביצועי היישום וזמינותו. פריסת אשכול מאפשרת להריץ מופעים שונים של היישום על מספר שרתים, או להריץ מספר תהליכים במצב מרובה ליבות על שרת יחיד.

לפרטים נוספים, עיין/י בתיעוד [מצב אשכול](/cluster-mode/) הרלוונטי.
