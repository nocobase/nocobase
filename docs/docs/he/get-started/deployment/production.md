# פריסה בסביבת ייצור

כשפורסים את NocoBase בסביבת ייצור, התקנת התלויות עלולה להיות מורכבת בשל הבדלים בשיטות הבנייה בין מערכות וסביבות שונות. כדי ליהנות מחוויה פונקציונלית מלאה, אנו ממליצים לפרוס באמצעות **Docker**. אם סביבת המערכת שלכם אינה תומכת ב-Docker, תוכלו לפרוס גם באמצעות **create-nocobase-app**.

:::warning

לא מומלץ לפרוס ישירות מקוד המקור בסביבת ייצור. לקוד המקור יש תלויות רבות, הוא גדול בנפחו, וקומפילציה מלאה דורשת משאבי CPU וזיכרון גבוהים. אם אתם חייבים לפרוס מקוד המקור, מומלץ לבנות תחילה אימג' Docker מותאם אישית ולאחר מכן לפרוס אותו.

:::
