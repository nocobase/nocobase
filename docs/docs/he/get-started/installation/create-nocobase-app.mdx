# התקנת create-nocobase-app

:::tip דרישות קדם

- Node.js 20+ ו-Yarn 1.22.x מותקנים.
- אחד ממסדי הנתונים הנדרשים מוגדר ופועל: MySQL 8.0.17+, MariaDB 10.9+, או PostgreSQL 10+.

:::

:::warning הסבר על משתני סביבה

- `TZ` משמש להגדרת אזור הזמן של היישום, וברירת המחדל שלו היא אזור הזמן של מערכת ההפעלה.
- `APP_KEY` הוא מפתח הסוד של היישום, המשמש ליצירת אסימוני משתמש (tokens) וכדומה (אם ה-`APP_KEY` משתנה, אסימונים ישנים יהפכו ללא תקפים). הוא יכול להיות כל מחרוזת אקראית. **אנא שנו אותו למפתח סוד משלכם וודאו שהוא אינו נחשף כלפי חוץ.**
- `DB_*` הם הגדרות הקשורות למסד הנתונים. אנא עדכנו אותן כך שיתאימו לפרטי החיבור של מסד הנתונים שלכם בהתאם להגדרות בפועל.

:::

## 2. שינוי ספרייה

```bash
cd my-nocobase-app
```

## 3. התקנת תלויות

📢 בהתאם לסביבת הרשת שלכם, תצורת המערכת וגורמים נוספים, שלב זה עשוי לארוך למעלה מעשר דקות.

```bash
yarn install
```

:::tip טיפ לסביבת ייצור

בעת פריסה בסביבת ייצור, כדי להקטין את הנפח, ניתן להתקין רק את התלויות הנדרשות:

```bash
yarn install --production
```

:::

## 4. התקנת NocoBase

```bash
yarn nocobase install --lang=he-IL
```

תהליך ההתקנה ייצור באופן אוטומטי את מבנה טבלאות מסד הנתונים ויאתחל את הנתונים.

## 5. הפעלת NocoBase

**סביבת פיתוח**

```bash
yarn dev
```

**סביבת ייצור**

```bash
yarn start
```

## 6. כניסה ל-NocoBase

פתחו את [http://localhost:13000](http://localhost:13000) בדפדפן שלכם. שם המשתמש והסיסמה הראשוניים הם `admin@nocobase.com` ו-`admin123`.

:::warning טיפ לאבטחת חשבון

לאחר הכניסה הראשונה, אנא שנו מיד את סיסמת ברירת המחדל כדי להבטיח את אבטחת המערכת.

:::
