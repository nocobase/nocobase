# הגדרות שפה

ב"הגדרות מערכת", תוכלו להתאים את סביבת השפה הגלובלית של המערכת על ידי הגדרת "שפות מופעלות". השפה הראשונה ברשימה תשמש כשפת ברירת המחדל.

![20240324192005](https://static-docs.nocobase.com/20240324192005.png)

כאשר מופעלות מספר שפות, משתמשים יכולים להגדיר את העדפת השפה האישית שלהם במרכז האישי (בפינה הימנית העליונה של העמוד).

![20240324192509](https://static-docs.nocobase.com/20240324192509.png)

אם סביבת השפה שבה אתם משתמשים חסרה תרגומים, תוכלו להשתמש בתוסף "[ניהול לוקליזציה](/system-management/localization/)" כדי להוסיף או לערוך תרגומים.

![20240324193055](https://static-docs.nocobase.com/20240324193055.png)
