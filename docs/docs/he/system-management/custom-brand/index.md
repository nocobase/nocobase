# התאמה אישית של המותג

## מבוא

שם המותג שמוגדר כברירת מחדל הוא "NocoBase", והוא מופיע בעיקר בשלושה מקומות:

![20240409111100](https://static-docs.nocobase.com/20240409111100.png)

בתחתית עמוד ההתחברות:

![20240409111254](https://static-docs.nocobase.com/20240409111254.png)

מיקום 1 ניתן להגדרה דרך האפשרות Logo ב[הגדרות מערכת](/system-management/system-settings/). מיקומים 2 ו-3 ניתנים להגדרה באמצעות ה**תוסף** הזה.

## הוראות שימוש

לאחר הפעלת ה**תוסף**, לחצו על תפריט "התאמה אישית של המותג" בפינה הימנית העליונה של העמוד כדי לגשת לדף ההגדרות.

![20240409113640](https://static-docs.nocobase.com/20240409113640.png)

- **מותג**: משמש להגדרת הטקסט בתחתית עמוד ההתחברות (מיקום 3)
- **אודות**: משמש להגדרת תוכן התפריט בפינה הימנית העליונה של העמוד (כפי שמוצג בתמונה למטה)

![20240409113915](https://static-docs.nocobase.com/20240409113915.png)

ההגדרות תומכות בפורמט HTML להתאמה אישית גמישה של התוכן. אם אתם צריכים רק לבצע שינויי טקסט פשוטים, תוכלו לערוך ישירות את התוכן הקיים, כפי שמוצג להלן:

![20240409114740](https://static-docs.nocobase.com/20240409114740.png)

הגדרת אייקון אתר (favicon)
