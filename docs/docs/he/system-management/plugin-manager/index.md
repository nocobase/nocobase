# ניהול תוספים

## מנהל התוספים

לחצו על אייקון מנהל התוספים שבפינה הימנית העליונה של העמוד כדי לעבור אל עמוד מנהל התוספים. בעמוד זה, תוכלו להציג, להתקין, להפעיל או להשבית תוספים.

![2024-04-10_08-47-45](https://static-docs.nocobase.com/2024-04-10_08-47-45.png)

## מרכז הגדרות התוספים

תוספים מגיעים בדרך כלל עם עמודי הגדרות משלהם. לחצו על אייקון ההגדרות שבפינה הימנית העליונה כדי להציג את תפריטי ההגדרות עבור כל התוספים. לחצו על פריט התפריט המתאים כדי להגדיר את התוסף.
