# ניהול לוקליזציה

## מבוא

תוסף ניהול הלוקליזציה משמש לניהול ויישום משאבי הלוקליזציה של NocoBase, ומאפשר לתרגם את תפריטי המערכת, האוספים, השדות וכל התוספים, כדי להתאים לשפה ולתרבות של אזורים ספציפיים.

## התקנה

תוסף זה הוא תוסף מובנה ואינו דורש התקנה נוספת.

### סנכרון רשומות תרגום

<img src="https://static-docs.nocobase.com/202404202134850.png"/>

נכון לעכשיו, ניתן לסנכרן את התכנים הבאים:

- חבילות שפה מקומיות של המערכת והתוספים
- כותרות אוספים, כותרות שדות ותוויות אפשרויות שדה
- כותרות תפריטים

לאחר השלמת הסנכרון, המערכת תציג את כל רשומות התרגום הזמינות עבור השפה הנוכחית.

<img src="https://static-docs.nocobase.com/202404202136567.png"/>

:::info{title=טיפ}
מודולים שונים עשויים להכיל רשומות מקור זהות, ויש לתרגם אותן בנפרד.
:::

### יצירה אוטומטית של רשומות

בעת עריכת דף, טקסטים מותאמים אישית בכל בלוק ייצרו באופן אוטומטי את הרשומות המתאימות, ויפיקו בו-זמנית את תוכן התרגום עבור השפה הנוכחית.

![](https://static-docs.nocobase.com/Localization-02-12-2026_08_39_AM.png)

![](https://static-docs.nocobase.com/Localization-NocoBase-02-12-2026_08_39_AM.png)

:::info{title=טיפ}
בעת הגדרת טקסט בקוד, יש לציין ידנית את ה-ns (מרחב שמות), לדוגמה: `${ctx.i18n.t('My custom js block', { ns: 'lm-flow-engine' })}`
:::
