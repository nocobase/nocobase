# ניהול לוקליזציה

## מבוא

תוסף ניהול הלוקליזציה משמש לניהול ויישום משאבי הלוקליזציה של NocoBase. הוא מאפשר לתרגם תפריטי מערכת, אוספים, שדות, וכל התוספים, כדי להתאים לשפה ולתרבות של אזורים ספציפיים.

## התקנה

תוסף זה מובנה ואינו דורש התקנה נוספת.

### סנכרון רשומות תרגום

<img src="https://static-docs.nocobase.com/202404202134850.png"/>

נכון לעכשיו, ניתן לסנכרן את התכנים הבאים:

- חבילות שפה מקומיות למערכת ולתוספים
- כותרות אוספים, כותרות שדות ותוויות אפשרויות שדה
- כותרות תפריטים

לאחר הסנכרון, המערכת תציג את כל רשומות התרגום הזמינות עבור השפה הנוכחית.

<img src="https://static-docs.nocobase.com/202404202136567.png"/>

:::info{title=הערה}
מודולים שונים עשויים להכיל רשומות טקסט מקוריות זהות, אותן יש לתרגם בנפרד.
:::
