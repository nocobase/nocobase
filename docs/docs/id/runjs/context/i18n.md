# ctx.i18n

Instans i18n dari konteks saat ini, digunakan untuk membaca atau mengganti bahasa. **Gunakan [ctx.t()](./t.md) untuk menerjemahkan teks**; jangan gunakan `ctx.i18n.t`.

## Skenario Penggunaan

Semua lingkungan eksekusi RunJS dapat menggunakan `ctx.i18n` (misalnya, JSBlock, JSField, JSItem, JSColumn, alur kerja, aturan keterkaitan, dll.).

## Definisi Tipe

```typescript
interface i18n: {
  language: string;
  changeLanguage(lng: string): Promise<any>;
}
```

## Properti Umum

| Properti | Tipe | Deskripsi |
|------|------|------|
| `language` | `string` | Kode bahasa yang aktif saat ini (misalnya, `zh-CN`, `en-US`) |

## Metode Umum

### changeLanguage(lng)

Mengganti bahasa saat ini.

| Parameter | Tipe | Deskripsi |
|------|------|------|
| `lng` | `string` | Kode bahasa target (misalnya, `'en-US'`, `'zh-CN'`) |

**Nilai balik**: `Promise<any>`, diselesaikan (resolve) setelah penggantian bahasa selesai.

## Contoh

### Membaca bahasa saat ini

```ts
const lang = ctx.i18n.language;
// 'zh-CN' | 'en-US' | ...
if (lang.startsWith('zh')) {
  ctx.render(ctx.t('Antarmuka Bahasa Mandarin'));
} else {
  ctx.render(ctx.t('English UI'));
}
```

### Mengganti bahasa

```ts
// Ganti ke Bahasa Inggris
await ctx.i18n.changeLanguage('en-US');

// Ganti ke Bahasa Mandarin
await ctx.i18n.changeLanguage('zh-CN');
```

### Tombol pengganti bahasa

```ts
const { Button } = ctx.libs.antd;
const isZh = ctx.i18n.language.startsWith('zh');
ctx.render(
  <Button onClick={async () => {
    await ctx.i18n.changeLanguage(isZh ? 'en-US' : 'zh-CN');
  }}>
    {ctx.t(isZh ? 'Switch to English' : 'Ganti ke Bahasa Mandarin')}
  </Button>,
);
```

## Catatan

- **Teks terjemahan**: Gunakan `ctx.t()` secara konsisten; jangan gunakan `ctx.i18n.t`.
