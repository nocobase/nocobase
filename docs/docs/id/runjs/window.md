# window

Properti berikut dapat diakses secara langsung melalui `window`:

* `setTimeout` / `clearTimeout`
* `setInterval` / `clearInterval`
* `console`
* `Math`
* `Date`
* `FormData`
* `addEventListener`
* `open` (Hanya `http:`, `https:`, atau `about:blank` yang diizinkan)
* `location` (Hanya baca, mendukung navigasi aman)
* `navigator`

Hanya kemampuan kueri dan pembuatan DOM dasar serta aman yang didukung:

* `createElement(tagName)`
* `querySelector(selectors)`
* `querySelectorAll(selectors)`
