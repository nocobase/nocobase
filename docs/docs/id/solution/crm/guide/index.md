# Panduan Penggunaan

> Panduan ini akan membawa Anda melalui alur penjualan inti CRM dari nol. Disarankan membaca secara berurutan, setiap bab akan terhubung dengan hasil operasi bab sebelumnya.

## Urutan Membaca

Rantai bisnis inti CRM adalah **Lead → Pelanggan/Kontak → Peluang → Penawaran → Pesanan**. Panduan ini disusun mengikuti alur utama ini:

| Bab | Konten | Yang Akan Anda Pelajari |
|------|------|---------|
| [Pengenalan Sistem dan Dashboard](./guide-overview) | Ikhtisar sistem, Analytics analisis data, workspace Overview | Struktur menu, drill-down KPI, interaksi chart |
| [Manajemen Lead](./guide-leads) | Pembuatan Lead, filter, skoring AI, follow-up dan konversi | Cara menyaring Lead berkualitas tinggi secara efisien, satu klik konversi ke Pelanggan + Peluang |
| [Manajemen Peluang](./guide-opportunities) | Kanban Peluang, peningkatan tahap, pembuatan Penawaran dan persetujuan | Cara mendorong sales funnel, melalui persetujuan Penawaran |
| [Produk, Penawaran, dan Pesanan](./guide-products-orders) | Katalog Produk, harga bertingkat, detail Penawaran, pengiriman Pesanan, dan pembayaran | Dari pemeliharaan Produk hingga closed-loop Pesanan |
| [Pelanggan, Kontak, dan Email](./guide-customers-emails) | Tampilan 360 Pelanggan, skor kesehatan, penggabungan, Kontak, email, bantuan AI | Siklus hidup Pelanggan + manajemen komunikasi |
| [Role dan Izin](./guide-roles) | Sistem Role, visibilitas menu, ruang lingkup operasi data | Apa yang dapat dilihat dan dioperasikan setiap posisi |
| AI Employee | Penggunaan tombol AI, contoh skenario | Cara membiarkan AI membantu pekerjaan penjualan harian |

## Prasyarat

- Sudah menginstal dan menjalankan CRM 2.0 (lihat [Cara Instalasi](../installation))
- Login ke sistem dengan akun admin
