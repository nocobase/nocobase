# User Lockout

<PluginInfo licenseBundled="professional" name="password-policy"></PluginInfo>

## Pengantar

Mengelola user yang di-lock, lock user secara aktif.

![](https://static-docs.nocobase.com/202412281450083.png)

### Unlock User

User yang di-lock karena melebihi batas login password invalid, dapat di-unlock dengan menghapus record.

### Lock User

Menambahkan record user dapat lock user secara aktif. Setelah user di-lock, akan dilarang mengakses sistem dengan metode autentikasi apa pun, termasuk API keys.

![](https://static-docs.nocobase.com/202412281450512.png)
