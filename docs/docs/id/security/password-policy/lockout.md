# Penguncian Pengguna

## Pendahuluan

Mengelola pengguna yang terkunci dan mengunci pengguna secara proaktif.

![](https://static-docs.nocobase.com/202412281450083.png)

### Membuka Kunci Pengguna

Pengguna yang terkunci karena melebihi batas percobaan login dengan kata sandi yang salah dapat dibuka kuncinya dengan menghapus catatan.

### Mengunci Pengguna

Menambahkan catatan pengguna memungkinkan Anda untuk mengunci pengguna secara proaktif. Setelah pengguna terkunci, mereka akan dilarang mengakses sistem melalui metode autentikasi apa pun, termasuk API keys.

![](https://static-docs.nocobase.com/202412281450512.png)
