# Pratinjau File

Pada antarmuka yang berisi bidang file (termasuk bidang lampiran), Anda dapat melihat pratinjau file dengan mengeklik pratinjau gambar (thumbnail) atau ikon file. Fungsi pratinjau bawaan mendukung berbagai jenis file, termasuk gambar, PDF, dan sebagian besar jenis file yang didukung secara bawaan oleh peramban (browser).

![20251129232307](https://static-docs.nocobase.com/20251129232307.png)

Untuk jenis file yang tidak mendukung pratinjau bawaan, Anda dapat mengaktifkan fungsi pratinjau dengan menginstal atau memperluas plugin pratinjau file yang sesuai. Sebagai contoh, setelah menginstal plugin Pratinjau File Office, Anda dapat melihat pratinjau file Word, Excel, dan PowerPoint.
