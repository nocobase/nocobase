## 1.2 Antarmuka Dasar NocoBase

Selamat datang di NocoBase! Saat pertama kali menggunakan, antarmuka mungkin terasa asing, tidak tahu harus mulai dari mana. Jangan khawatir, mari kita kenali area fitur utama langkah demi langkah, membantu Anda memulai dengan cepat.

### 1.2.1 **Konfigurasi Antarmuka**

Saat Anda pertama kali masuk ke NocoBase, Anda akan melihat antarmuka utama yang sederhana dan intuitif. Di pojok kanan atas adalah tombol [**Konfigurasi Antarmuka**](https://docs-cn.nocobase.com/handbook/ui/ui-editor), klik dan sistem akan beralih ke mode konfigurasi antarmuka. Ini adalah area kerja utama Anda untuk membangun halaman sistem.

![Mode Konfigurasi Antarmuka](https://static-docs.nocobase.com/nocobase_tutorials_zh/202412152031029.png)

**Langkah Operasi:**

1. **Masuk ke Mode Konfigurasi**: Klik tombol "Konfigurasi Antarmuka" di pojok kanan atas, masuk ke mode konfigurasi.
2. **Tambah Halaman [Menu](https://docs-cn.nocobase.com/handbook/ui/menus)**:
   - Klik "Tambah menu".
   - Masukkan nama menu (contohnya "Halaman Uji"), lalu klik konfirmasi.
   - Sistem akan otomatis membuat dan beralih ke halaman uji yang baru dibuat.

![demov4-001.gif](https://static-docs.nocobase.com/nocobase_tutorials_zh/202412152032346.gif)

3. **Buat [Block](https://docs-cn.nocobase.com/handbook/ui/blocks)**:
   - Pada halaman uji, klik tombol "Buat Block".
   - Pilih satu Block data dari tipe Block, contohnya "Block Tabel".
   - Hubungkan satu tabel data, seperti tabel "Pengguna" bawaan sistem.
   - Pilih field yang ingin Anda tampilkan, klik konfirmasi.
4. Dengan demikian, satu Block tabel yang menampilkan daftar pengguna selesai!

![Buat Block](https://static-docs.nocobase.com/nocobase_tutorials_zh/202412152032964.gif)

Bukankah sangat sederhana? Inspirasi desain Block NocoBase berasal dari Notion, tetapi fiturnya lebih kuat, dapat mendukung pembangunan sistem yang lebih kompleks. Pada tutorial selanjutnya, kita akan secara mendalam menjelajahi fitur berbagai Block, nantikan!

### 1.2.2 **Plugin Manager**

Plugin adalah tool penting untuk memperluas fitur NocoBase. Pada [**Plugin Manager**](https://docs-cn.nocobase.com/handbook/plugin-manager), Anda dapat melihat, menginstal, mengaktifkan, atau menonaktifkan berbagai Plugin untuk memenuhi kebutuhan bisnis yang berbeda.

Melalui ekstensi Plugin, Anda dapat memperluas dan mengintegrasikan fitur yang lebih nyaman atau tidak terduga, lebih memudahkan kreasi dan pengembangan Anda.

![Plugin Manager](https://static-docs.nocobase.com/nocobase_tutorials_zh/202412152034703.png)

**Langkah Operasi:**

1. **Lihat Plugin yang Sudah Diinstal**: Klik "Plugin Manager", Anda akan melihat list semua Plugin yang saat ini sudah diinstal.
2. **Aktifkan Plugin**:
   - Temukan Plugin yang Anda butuhkan, contohnya Plugin "Theme Editor".
   - Klik tombol "Aktifkan" untuk mengaktifkan Plugin.
3. **Uji Fitur Plugin**:
   - Setelah mengaktifkan "Theme Editor", di personal center pojok kanan atas, Anda dapat dengan cepat mengubah tema sistem.
     ![Mengubah Tema Sistem](https://static-docs.nocobase.com/nocobase_tutorials_zh/202412152035380.gif)
   - Di pusat pengaturan, Anda akan melihat antarmuka Theme Editor, di sini dapat mempersonalisasi tema sistem, seperti mengubah warna, font, dll.
     ![Antarmuka Theme Editor](https://static-docs.nocobase.com/nocobase_tutorials_zh/202412152035889.png)

### 1.2.3 **Halaman Pengaturan**

**Halaman Pengaturan** mengintegrasikan berbagai opsi pengaturan sistem dan beberapa Plugin, membantu Anda mengelola berbagai aspek NocoBase secara komprehensif.

![Halaman Pengaturan](https://static-docs.nocobase.com/nocobase_tutorials_zh/202412152036847.png)

**Beberapa item konfigurasi Plugin yang umum digunakan, seperti:**

- [**Manajemen Data Source**](https://docs-cn.nocobase.com/handbook/data-source-manager): Mengelola semua tabel data, mengonfigurasi database utama atau database eksternal.
- [**Pengaturan Sistem**](https://docs-cn.nocobase.com/handbook/system-settings): Memodifikasi nama sistem, Logo, bahasa, dan informasi dasar lainnya.
- [**Pengguna dan Izin**](https://docs-cn.nocobase.com/handbook/users): Mengelola akun pengguna, mengonfigurasi izin role yang berbeda.
- [**Pengaturan Plugin**](https://docs-cn.nocobase.com/handbook/plugin-manager): Konfigurasi dan manajemen detail untuk Plugin yang sudah diinstal.

## 1.3 Instalasi NocoBase

Setelah memutuskan untuk menggunakan NocoBase secara mendalam, kita perlu menginstalnya di komputer atau server Anda. NocoBase menyediakan berbagai metode instalasi, pilih metode yang sesuai dengan Anda untuk dengan mudah memulai perjalanan pengembangan no-code.

### 1.3.1 **Metode Instalasi**

1. **Instalasi Docker (Direkomendasikan)**

   - **Keunggulan**: Cepat, sederhana, cocok untuk pengguna yang familiar dengan Docker.
   - **Pemilihan Versi**:
     - **Versi main & latest**: Ini adalah versi paling stabil saat ini, cocok untuk sebagian besar pengguna.
     - **Versi next**: Versi internal beta, cocok untuk pengguna yang ingin mencoba fitur baru. Perhatikan, versi ini mungkin belum sepenuhnya stabil, disarankan menggunakan setelah backup data penting.
   - **Langkah Operasi**:
     - Lihat [panduan instalasi resmi](https://docs-cn.nocobase.com/welcome/getting-started/installation/docker-compose), ikuti langkah-langkah untuk men-deploy NocoBase menggunakan Docker Compose.
2. **Instalasi Create-NocoBase-App**

   - **Cocok Untuk**: Developer frontend atau pengguna yang familiar dengan npm.
   - **Langkah Operasi**:
     - Lihat [panduan instalasi](https://docs-cn.nocobase.com/welcome/getting-started/installation/create-nocobase-app), instal melalui paket npm.
3. **Instalasi Source Code**

   - **Cocok Untuk**: Developer yang perlu kustomisasi mendalam terhadap NocoBase.
   - **Langkah Operasi**:
     - Lihat [panduan instalasi](https://docs-cn.nocobase.com/welcome/getting-started/installation/git-clone), clone source code dari GitHub, dan instal sesuai kebutuhan kustom.

### 1.3.2 **Panduan Instalasi Detail (Contoh dengan Docker)**

Apa pun metode instalasi yang dipilih, Anda dapat menemukan langkah dan penjelasan detail di **Dokumentasi Instalasi NocoBase**. Berikut adalah langkah singkat instalasi Docker, membantu Anda memulai dengan cepat:

1. **Instal Docker**: Pastikan sistem Anda sudah menginstal Docker. Jika belum, Anda dapat mengunjungi [situs resmi Docker](https://www.docker.com/) untuk download dan menginstal.
2. **Dapatkan File Docker Compose**:

   - Buka terminal atau alat command line.
   - Buat direktori nocobase, buat konfigurasi Docker Compose.

```bash
mkdir nocobase
cd nocobase
vim docker-compose.yml
```

3. Setelah masuk ke `docker-compose.yml`, tempel konfigurasi di bawah, sesuaikan dan simpan file sesuai kebutuhan

```bash
version: "3"

networks:
  nocobase:
        driver: bridge

services:
  app:
        image: registry.cn-shanghai.aliyuncs.com/nocobase/nocobase:latest
        networks:
          - nocobase
        depends_on:
          - postgres
        environment:
          # Kunci aplikasi, digunakan untuk menghasilkan Token pengguna, dll.
          # Jika APP_KEY dimodifikasi, Token lama akan menjadi tidak valid
          # Bisa berupa string acak apa saja, dan pastikan tidak bocor ke pihak luar
          - APP_KEY=your-secret-key
          # Tipe database, mendukung postgres, mysql, mariadb, sqlite
          - DB_DIALECT=postgres
          # Host database, dapat diganti dengan IP server database yang sudah ada
          - DB_HOST=postgres
          # Nama database
          - DB_DATABASE=nocobase
          # Pengguna database
          - DB_USER=nocobase
          # Password database
          - DB_PASSWORD=nocobase
          # Zona waktu
          - TZ=Asia/Shanghai
        volumes:
          - ./storage:/app/nocobase/storage
        ports:
          - "13000:80"
        # init: true

  # Jika menggunakan layanan database yang sudah ada, postgres tidak perlu dijalankan
  postgres:
        image: registry.cn-shanghai.aliyuncs.com/nocobase/postgres:16
        restart: always
        command: postgres -c wal_level=logical
        environment:
          POSTGRES_USER: nocobase
          POSTGRES_DB: nocobase
          POSTGRES_PASSWORD: nocobase
        volumes:
          - ./storage/db/postgres:/var/lib/postgresql/data
        networks:
          - nocobase
```

4. **Jalankan NocoBase**:
   - Pada direktori nocobase jalankan perintah berikut untuk memulai layanan:

```bash
docker-compose up -d
```

- Ini akan mendownload image yang diperlukan dan menjalankan layanan NocoBase.

5. **Akses NocoBase**:
   - Buka browser, akses `http://localhost:13000` (mungkin berbeda berdasarkan konfigurasi), Anda akan melihat antarmuka login NocoBase.

Setelah menyelesaikan langkah-langkah di atas, Anda telah berhasil menginstal dan menjalankan NocoBase! Selanjutnya, Anda dapat mengikuti panduan dalam tutorial untuk mulai membangun sistem aplikasi Anda sendiri.
