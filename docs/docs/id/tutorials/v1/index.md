# Ikhtisar Sistem Manajemen Tugas

## Pengantar

Selamat datang di dunia **NocoBase**! Dalam lingkungan bisnis yang berubah cepat saat ini, perusahaan dan tim development sering menghadapi tantangan berikut:

- **Kebutuhan bisnis sering berubah**, development tradisional sulit merespons dengan cepat.
- **Waktu pengiriman development yang ketat**, alur yang rumit dan efisiensi rendah.
- **Kemampuan platform no-code terbatas**, sulit memenuhi kebutuhan kompleks.
- **Privasi data dan stabilitas sistem** sulit dijamin.
- **Integrasi dengan sistem yang ada sulit**, mempengaruhi efisiensi keseluruhan.
- **Tarif berdasarkan pengguna atau aplikasi**, biaya sulit dikontrol.

**NocoBase** lahir untuk menyelesaikan masalah-masalah ini. Sebagai **platform pengembangan no-code yang sangat mudah diperluas**, NocoBase memiliki keunggulan unik berikut:

- **Gratis dan Open Source, Fleksibel dan Cepat**: Source code terbuka, dukungan komunitas yang aktif. Instalasi dan online dalam beberapa menit, instant development dan deployment.
- **Sangat Dapat Diperluas**: Arsitektur microkernel, desain modular, semua fitur disediakan dalam bentuk Plugin.
- **Konsep Inti Unik**: Membangun sistem melalui kombinasi data source, Block, Action, pengalaman lancar dan natural.
- **WYSIWYG**: UI editor yang intuitif, mendesain antarmuka dengan mudah.
- **Data Driven**: Mendukung berbagai data source, struktur data terpisah dari antarmuka.

## Tujuan Desain NocoBase

NocoBase menemukan keseimbangan yang lebih baik antara **kemudahan penggunaan**, **fitur yang kuat**, dan **kompleksitas rendah**. Tidak hanya menyediakan modul fitur yang kaya untuk memenuhi berbagai kebutuhan kompleks, tetapi juga mempertahankan antarmuka pengguna yang sederhana dan intuitif, sehingga Anda dapat dengan mudah memulai. Selain itu, **mekanisme Plugin** memungkinkan pengguna untuk menerobos batasan di atas dasar platform, merealisasikan ekstensi kustom yang sangat tinggi, memastikan fleksibilitas dan pengembangan berkelanjutan sistem.

## Mengapa Memilih Sistem Manajemen Tugas?

Sistem manajemen tugas adalah proyek pemula yang sangat cocok untuk pemula:

- Di satu sisi, ia berhubungan erat dengan kebutuhan harian kita;
- Di sisi lain, strukturnya sederhana tetapi memiliki ekstensibilitas yang kuat, dari manajemen tugas dasar dapat secara bertahap diperluas menjadi sistem manajemen proyek lengkap.

Tutorial ini akan dimulai dari fitur dasar, mencakup modul inti dan operasi NocoBase, termasuk pembuatan tugas, interaksi komentar, manajemen izin, pengaturan notifikasi, dan lainnya, membantu Anda memahami fitur dasar NocoBase secara komprehensif.

### Konsep Inti dan Manajemen Tugas

Di setiap bab, kita akan secara mendalam memahami beberapa konsep inti NocoBase melalui praktik, termasuk tetapi tidak terbatas pada:

- **Tabel Data**: Struktur data dasar sistem, seperti tabel data tugas, pengguna, komentar memberikan dasar informasi untuk sistem.
- **Block**: Menampilkan data di halaman, mendukung berbagai gaya tampilan. Melalui Block, dapat menampilkan dinamika data dalam skenario pembuatan tugas, edit, lihat, dan kelola, juga dapat memperluas lebih banyak fitur melalui Plugin (seperti Block komentar).
- **Action**: CRUD dan kontrol manajemen data, pengguna dapat mengeksekusi tambah, filter, update, hapus pada data tugas dan komentar untuk memenuhi berbagai kebutuhan penggunaan.
- **Ekstensi Plugin**: Semua fitur NocoBase terintegrasi melalui Plugin, memiliki ekstensibilitas yang tinggi. Tutorial ini akan memperkenalkan Plugin Markdown dan Komentar, menambahkan fitur praktis untuk deskripsi tugas dan kolaborasi tim.
- **Workflow**: Salah satu sorotan NocoBase. Tutorial ini akan membawa Anda mempraktikkan Workflow otomasi dasar, seperti pengingat penanggung jawab tugas, agar Anda dapat awal merasakan kekuatan Workflow.
