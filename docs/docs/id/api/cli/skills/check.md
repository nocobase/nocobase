# nb skills check

Memeriksa NocoBase AI coding skills global, melaporkan apakah dikelola oleh CLI dan apakah ada update yang tersedia.

## Penggunaan

```bash
nb skills check [flags]
```

## Parameter

| Parameter | Tipe | Penjelasan |
| --- | --- | --- |
| `--json` | boolean | Output JSON |

## Contoh

```bash
nb skills check
nb skills check --json
```
