# nb skills remove

Menghapus NocoBase AI coding skills global yang dikelola oleh `nb`.

## Penggunaan

```bash
nb skills remove [flags]
```

## Parameter

| Parameter | Tipe | Penjelasan |
| --- | --- | --- |
| `--yes`, `-y` | boolean | Lewati konfirmasi penghapusan |
| `--json` | boolean | Output JSON |
| `--verbose` | boolean | Menampilkan output penghapusan detail |

## Contoh

```bash
nb skills remove
nb skills remove --yes
nb skills remove --json
```
