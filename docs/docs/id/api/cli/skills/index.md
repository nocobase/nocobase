# nb skills

Memeriksa, menginstal, memperbarui, atau menghapus NocoBase AI coding skills global.

## Penggunaan

```bash
nb skills <command>
```

## Subcommand

| Perintah | Penjelasan |
| --- | --- |
| [`nb skills check`](./check.md) | Memeriksa NocoBase AI coding skills global |
| [`nb skills install`](./install.md) | Menginstal NocoBase AI coding skills secara global |
| [`nb skills update`](./update.md) | Memperbarui NocoBase AI coding skills yang terinstal |
| [`nb skills remove`](./remove.md) | Menghapus NocoBase AI coding skills yang dikelola oleh `nb` |

## Contoh

```bash
nb skills check
nb skills install --yes
nb skills update --json
nb skills remove --yes
```
