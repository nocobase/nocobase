# nb session id

Menampilkan id sesi efektif saat ini.

Jika tidak ada `NB_SESSION_ID` yang bisa digunakan pada shell atau runtime saat ini, perintah akan memberi tahu Anda untuk menjalankan [`nb session setup`](./setup.md) terlebih dahulu, lalu membuka shell atau runtime baru.

## Penggunaan


nb session id

## Contoh


nb session id

## Perintah Terkait

- [`nb session setup`](./setup.md)
- [`nb env current`](../env/current.md)
