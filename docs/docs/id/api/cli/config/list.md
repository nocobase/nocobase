# nb config list

Menampilkan daftar pengaturan CLI yang dikonfigurasi secara eksplisit. Nilai default yang tidak pernah disetel secara eksplisit tidak akan ditampilkan.

## Penggunaan

```bash
nb config list
```

## Contoh

```bash
nb config list
```

## Catatan

Jika tidak ada nilai konfigurasi yang disetel, CLI akan mencetak `No CLI config values are set.`.

## Perintah Terkait

- [`nb config get`](./get.md)
- [`nb config set`](./set.md)
