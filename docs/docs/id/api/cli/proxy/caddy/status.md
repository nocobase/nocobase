# nb proxy caddy status

Menampilkan status runtime proxy Caddy saat ini.

## Penggunaan

```bash
nb proxy caddy status
```

## Output

Output biasanya mencakup:

- `driver`
- `status`
- `config`
- `caddy` atau `container`
