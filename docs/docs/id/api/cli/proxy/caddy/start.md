# nb proxy caddy start

Menjalankan proxy Caddy dengan driver saat ini.

## Penggunaan

```bash
nb proxy caddy start
```

## Contoh

```bash
nb proxy caddy start
```

## Catatan

- Dengan driver `local`, perintah ini menjalankan proses Caddy lokal
- Dengan driver `docker`, perintah ini menjalankan atau membuat container Docker
- Jika proxy sudah berjalan, perintah akan memberi tahu bahwa proxy sudah aktif
