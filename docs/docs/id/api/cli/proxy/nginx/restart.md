# nb proxy nginx restart

Menjalankan ulang proxy Nginx dengan driver saat ini.

## Penggunaan

```bash
nb proxy nginx restart
```

## Contoh

```bash
nb proxy nginx restart
```

## Catatan

- Perintah ini akan menghentikan proxy terlebih dahulu, lalu menjalankannya lagi
- Dengan `local` atau `docker`, perintah ini bekerja pada proses lokal atau container Docker sesuai driver saat ini
