# nb proxy nginx current

Menampilkan driver provider Nginx yang sedang digunakan.

## Penggunaan

```bash
nb proxy nginx current
```

## Output

Perintah ini langsung menampilkan satu nilai:

- `local`
- `docker`

## Contoh

```bash
nb proxy nginx current
```
