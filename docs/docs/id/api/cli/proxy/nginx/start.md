# nb proxy nginx start

Menjalankan proxy Nginx dengan driver saat ini.

## Penggunaan

```bash
nb proxy nginx start
```

## Contoh

```bash
nb proxy nginx start
```

## Catatan

- Dengan driver `local`, perintah ini menjalankan proses Nginx lokal
- Dengan driver `docker`, perintah ini menjalankan atau membuat container Docker
- Jika proxy sudah berjalan, perintah akan memberi tahu bahwa proxy sudah aktif
