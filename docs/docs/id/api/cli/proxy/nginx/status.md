# nb proxy nginx status

Menampilkan status runtime proxy Nginx saat ini.

## Penggunaan

```bash
nb proxy nginx status
```

## Output

Output biasanya mencakup:

- `driver`
- `status`
- `config`
- `nginx` atau `container`

## Contoh

```bash
nb proxy nginx status
```
