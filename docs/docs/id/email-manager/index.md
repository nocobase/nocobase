# Email Manager

## Pengantar
Plugin Email Manager adalah tool yang efisien dan praktis, mendukung integrasi otorisasi akun email Gmail dan Outlook, membantu user mengintegrasikan kemampuan manajemen email, pengiriman dan penerimaan email, dan lainnya ke setiap block dan halaman. Melalui konfigurasi otorisasi yang sederhana, user dapat mencapai manajemen multi-akun terpadu, menikmati pengalaman komunikasi email yang seamless.
