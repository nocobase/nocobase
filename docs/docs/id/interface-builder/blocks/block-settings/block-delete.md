# Hapus Blok

## Pendahuluan

Menghapus blok akan menghapusnya dari halaman saat ini dan merupakan tindakan yang tidak dapat diurungkan. Jika ada cadangan yang dibuat sebelum penghapusan, blok tersebut dapat dipulihkan.
