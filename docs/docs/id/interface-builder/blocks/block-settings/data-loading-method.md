# Metode Pemuatan Data

## Pengantar

Metode pemuatan data default adalah "Muat semua data saat kondisi filter kosong". "Kondisi filter" di sini merujuk pada nilai di "Filter Block". Opsi alternatif lainnya adalah "Tidak muat data saat kondisi filter kosong".
