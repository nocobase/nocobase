# Grid Card

## Pengantar

Block Grid Card menampilkan informasi ringkasan record data dalam bentuk card, mendukung konfigurasi jumlah kolom berdasarkan ukuran layar yang berbeda untuk memastikan tampilan yang ramah di perangkat dengan ukuran berbeda.
