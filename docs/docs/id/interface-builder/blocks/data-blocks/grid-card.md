# Kartu Grid

## Pendahuluan

Blok Kartu Grid menampilkan informasi ringkasan catatan data dalam format kartu. Blok ini mendukung konfigurasi jumlah kolom untuk ukuran layar yang berbeda, memastikan tampilan yang ramah pengguna di berbagai perangkat.
