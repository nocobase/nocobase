# Tautan

## Pendahuluan

Aksi tautan menggunakan navigasi rute, mendukung penerusan variabel URL, dan memungkinkan halaman tujuan untuk menyesuaikan kontennya secara dinamis berdasarkan data yang diteruskan. Ini dapat dikonfigurasi di blok data.
