# Edit Tombol

## Pengantar

Sistem mendukung kustomisasi nama, ikon, warna latar belakang, dan tipe tombol, menyediakan opsi kustomisasi UI yang fleksibel, membantu pengguna menyesuaikan style tampilan dan fungsi tombol sesuai kebutuhan tertentu.

- **Nama Tombol**: Pengguna dapat mengkustomisasi teks tombol sesuai kebutuhan bisnis, memastikan label tombol jelas dan intuitif.
- **Ikon Tombol**: Mendukung pengaturan ikon untuk tombol, meningkatkan pengenalan visual, memungkinkan pengguna untuk dengan cepat mengidentifikasi fungsi tombol.
- **Action Berbahaya**: Ketika tombol mewakili **Action berbahaya** (seperti hapus, reset, dll.), dapat diatur sebagai tipe **Danger Action**.
- **Tipe Tombol (Button Type)**: Tombol mendukung pengaturan tipe yang berbeda. Tipe tombol umum meliputi:
  - **Tipe Default (Default)**: Tombol standar, cocok untuk Action biasa.
  - **Tipe Primary (Primary)**: Digunakan untuk Action penting atau utama, biasanya dengan efek visual yang mencolok.
  - **Tipe Dashed (Dashed)**: Tombol dengan border garis putus-putus, cocok untuk Action sekunder atau tombol prompt.
  - **Tipe Text (Text)**: Tombol dengan hanya teks tanpa warna latar belakang, biasanya digunakan untuk Action ringan atau link.
