# Edit Tombol

## Pendahuluan

Sistem mendukung penyesuaian nama, ikon, warna latar belakang, dan tipe tombol. Ini memberikan opsi kustomisasi UI yang fleksibel untuk membantu pengguna menyesuaikan gaya tampilan dan fungsionalitas tombol sesuai kebutuhan spesifik.

-   **Nama Tombol**: Pengguna dapat menyesuaikan teks tombol sesuai kebutuhan bisnis untuk memastikan label tombol jelas dan intuitif.
-   **Ikon Tombol**: Mendukung pengaturan ikon untuk tombol guna meningkatkan pengenalan visual, memungkinkan pengguna mengidentifikasi fungsi tombol dengan cepat.
-   **Aksi Berbahaya (Danger Action)**: Ketika tombol merepresentasikan **aksi berbahaya** (seperti menghapus, mereset, dll.), tombol dapat diatur ke tipe **Danger Action**.
-   **Tipe Tombol (Button Type)**: Tombol mendukung berbagai pengaturan tipe. Tipe tombol yang umum meliputi:
    -   **Default**: Tombol standar, cocok untuk operasi normal.
    -   **Primary**: Digunakan untuk operasi penting atau utama, biasanya dengan efek visual yang menonjol.
    -   **Dashed**: Tombol dengan batas putus-putus, cocok untuk operasi sekunder atau tombol seperti petunjuk.
    -   **Text**: Tombol yang hanya memiliki teks tanpa warna latar belakang, biasanya digunakan untuk operasi ringan atau tautan.
