# Tampilkan Label

## Pendahuluan

Bidang-bidang dalam blok secara default menampilkan label bidang untuk mengidentifikasi makna setiap bidang, memudahkan pengguna membaca dan memahami konten data.

Dalam beberapa skenario (misalnya, pada halaman yang hanya menampilkan data atau ketika makna bidang sudah sangat jelas), Anda dapat memilih untuk menyembunyikan label agar antarmuka lebih ringkas. Setelah menyembunyikan label, hanya nilai bidang yang ditampilkan, dan nama bidang tidak lagi muncul.
