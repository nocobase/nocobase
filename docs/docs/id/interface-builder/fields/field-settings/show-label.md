# Tampilkan Label

## Pengantar

Field di Block secara default menampilkan label Field, digunakan untuk mengidentifikasi arti setiap Field, memudahkan pengguna untuk membaca dan memahami konten data.

Dalam beberapa skenario (misalnya Page tipe tampilan atau saat arti Field sudah sangat jelas), Anda dapat memilih untuk menyembunyikan label, membuat antarmuka lebih ringkas. Setelah label disembunyikan, hanya nilai Field yang ditampilkan, nama Field tidak lagi ditampilkan.
