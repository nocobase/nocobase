# Edit Deskripsi

## Pengantar

Deskripsi Field adalah teks penjelasan singkat yang ditampilkan dalam huruf abu-abu muda di bawah komponen Field, dapat digunakan untuk membantu pengguna memahami tujuan Field dan persyaratan input.
