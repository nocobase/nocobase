# Tanggal Waktu

## Pengantar

Field tanggal mendukung kustomisasi format tanggal dan waktu.

## Di Mana Dapat Mengatur Format Tanggal

### Field Collection


![20240417114311](https://static-docs.nocobase.com/20240417114311.png)

> Format tanggal yang diatur di Field Collection akan digunakan sebagai format default Field, dapat ditimpa oleh format tanggal yang diatur di Field di konfigurasi Block.
### Pengaturan Field

![20251027191824](https://static-docs.nocobase.com/20251027191824.png)

Selain berbagai format tanggal-waktu yang sudah ada, juga mendukung format kustom untuk memenuhi kebutuhan personal.
