# Bidang Tabel

## Pendahuluan
Selain mendukung penyesuaian lebar kolom, judul bidang, pengurutan, dan lain-lain pada bidang, bidang tabel juga mendukung konfigurasi tampilan yang lebih personal untuk bidang-bidang tertentu (seperti bidang tanggal, bidang relasi, dan bidang angka).

![20251024174558](https://static-docs.nocobase.com/20251024174558.png)

## Opsi Konfigurasi Bidang

### Pemformatan Bidang Tanggal
![20251024175303](https://static-docs.nocobase.com/20251024175303.png)
Untuk detail lebih lanjut, lihat [Pemformatan Tanggal](/interface-builder/fields/specific/date-picker)

### Pemformatan Bidang Angka
Mendukung konversi unit sederhana, pemisah ribuan, awalan dan akhiran, presisi, serta notasi ilmiah.
![20251024175445](https://static-docs.nocobase.com/20251024175445.png)
Untuk detail lebih lanjut, lihat [Pemformatan Angka](/interface-builder/fields/field-settings/number-format)

### Aktifkan Edit Cepat
Ketika edit cepat diaktifkan, tombol edit akan muncul saat kursor diarahkan ke kolom tersebut. Klik tombol tersebut untuk mengedit dan menyimpan data dengan cepat di jendela pop-up.
![20251025171158](https://static-docs.nocobase.com/20251025171158.gif)

### Aktifkan Klik untuk Membuka
Selain bidang relasi yang mendukung klik untuk membuka pop-up, bidang biasa juga dapat diaktifkan dengan "Klik untuk Membuka" sebagai titik masuk untuk membuka pop-up.
![20251025172308](https://static-docs.nocobase.com/20251025172308.gif)

### Mode Tampilan Overflow Konten
Ketika konten kolom melebihi lebar kolom tabel, Anda dapat mengatur mode overflow:
- Elipsis (default)
- Bungkus Teks
![20251025172549](https://static-docs.nocobase.com/20251025172549.png)

### Komponen Bidang
Beberapa bidang mendukung berbagai format tampilan. Pengguna dapat mengganti komponen bidang untuk mencapai efek tampilan yang berbeda, memenuhi kebutuhan di berbagai skenario. Misalnya, bidang tipe **URL** dapat diganti menjadi komponen **Preview** untuk menampilkan konten tautan atau pratinjau gambar dengan lebih baik.
![20251025171658](https://static-docs.nocobase.com/20251025171658.png)

### Dapat Diurutkan
Saat ini, sebagian besar tipe bidang mendukung pengurutan sisi server. Ketika pengurutan diaktifkan, data dapat diurutkan secara menaik atau menurun berdasarkan bidang target.
