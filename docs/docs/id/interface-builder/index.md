# Pembangun UI

## WYSIWYG (What You See Is What You Get)

NocoBase menggunakan mode pembangunan antarmuka WYSIWYG (What You See Is What You Get). Klik tombol Pembangun UI untuk beralih antara mode Konfigurasi dan mode Tampilan.

## Templat Tata Letak

NocoBase dilengkapi dengan templat tata letak bawaan: area navigasi di bagian atas dan kiri, serta area konten di sebelah kanan.

![未命名.002](https://static-docs.nocobase.com/未命名.002.jpeg)

## Opsi Konfigurasi

Setelah masuk ke mode Pembangun UI, penanda berwarna oranye akan muncul di antarmuka untuk menunjukkan elemen yang dapat dikonfigurasi. Biasanya, titik masuk untuk pengaturan elemen terletak di sudut kanan atas elemen tersebut. Hampir semua elemen mendukung konfigurasi waktu nyata dengan pratinjau instan.
