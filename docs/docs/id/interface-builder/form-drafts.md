# Draft Form

Setelah mengaktifkan draft Form, konten yang sedang diisi akan otomatis disimpan secara lokal. Bahkan jika Form ditutup secara tidak sengaja atau halaman direfresh, data yang telah diisi tidak akan hilang.

![20251213093717](https://static-docs.nocobase.com/20251213093717.png)

Ketika ada draft, informasi peringatan akan ditampilkan di bagian atas Form. Setelah submit berhasil, draft akan dihapus secara otomatis; jika tidak diperlukan lagi, Anda juga dapat menghapusnya secara manual kapan saja.

![20251213092734](https://static-docs.nocobase.com/20251213092734.png)

::: tip Tips

Draft Form saat ini hanya mendukung cache lokal. Versi mendatang akan mendukung cache di server untuk sinkronisasi multi-perangkat.

:::
