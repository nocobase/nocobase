# Draf Formulir

Saat draf formulir diaktifkan, konten yang sedang diisi akan disimpan secara otomatis ke penyimpanan lokal. Meskipun formulir ditutup secara tidak sengaja atau halaman dimuat ulang, data yang telah diisi tidak akan hilang.

![20251213093717](https://static-docs.nocobase.com/20251213093717.png)

Ketika draf tersedia, pesan pemberitahuan akan ditampilkan di bagian atas formulir. Draf akan dihapus secara otomatis setelah pengiriman berhasil; jika tidak lagi diperlukan, draf juga dapat dihapus secara manual kapan saja.

![20251213092734](https://static-docs.nocobase.com/20251213092734.png)

::: tip Tip

Saat ini, draf formulir hanya mendukung penyimpanan cache lokal. Versi mendatang akan mendukung penyimpanan cache di sisi server untuk memungkinkan sinkronisasi antar perangkat.

:::
