# Kolom Wilayah Tiongkok

## Pendahuluan

Plugin Kolom Wilayah Tiongkok (`field-china-region`) menambahkan tipe kolom "Wilayah Tiongkok" ke NocoBase.

## Instalasi

Di antarmuka utama NocoBase, buka "Pengelola Plugin", cari plugin "Wilayah Tiongkok", lalu aktifkan.

## Penggunaan

Dalam konfigurasi koleksi, tambahkan kolom baru dan pilih "Wilayah Tiongkok" sebagai tipe kolom.

![](/images/field-china-region-1.png)

### Konfigurasi Kolom

Dalam konfigurasi kolom, Anda dapat mengatur mode tampilan untuk kolom tersebut:

- Provinsi
- Provinsi/Kota
- Provinsi/Kota/Kabupaten
- Provinsi/Kota/Kabupaten/Kecamatan

![](/images/field-china-region-2.png)
