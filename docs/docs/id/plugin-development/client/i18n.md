# I18n Internasionalisasi

Di plugin NocoBase, internasionalisasi multibahasa frontend-backend (i18n) didukung, melalui mekanisme terpadu Anda dapat dengan mudah mengimplementasikan konten multibahasa di plugin.

## Manajemen File Multibahasa

File multibahasa plugin disimpan secara terpadu di direktori `src/locale`, disarankan dinamai berdasarkan file bahasa, contoh:

```bash
|- /plugin-hello
  |- /src
    |- /locale
      |- en-US.json   # Bahasa Inggris
      |- zh-CN.json   # Bahasa Tionghoa
```

Setiap file bahasa mengekspor sebuah object JSON yang berisi semua entri terjemahan untuk bahasa tersebut, contoh:

```json
// zh-CN.json
{
  "Hello": "你好",
  "World": "世界",
  "Enter your name": "请输入你的名字",
  "Your name is {{name}}": "你的名字是 {{name}}"
}
```

```json
// en-US.json
{
  "Hello": "Hello",
  "World": "World",
  "Enter your name": "Enter your name",
  "Your name is {{name}}": "Your name is {{name}}"
}
```

> Pertama kali menambahkan file bahasa perlu restart aplikasi agar berlaku. Anda dapat memvalidasi apakah entri terjemahan berlaku melalui interface:  
> http://localhost:13000/api/app:getLang?locale=zh-CN
