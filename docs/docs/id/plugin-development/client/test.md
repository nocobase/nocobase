# Tes

:::tip

Bab ini akan menjelaskan cara menulis dan menjalankan tes untuk bagian frontend dari plugin. Isinya akan mencakup:
- Konfigurasi dan dependensi lingkungan tes;
- Menulis tes unit menggunakan Vitest dan Testing Library;
- Cara mensimulasikan lingkungan runtime frontend NocoBase;
- Menjalankan tes secara otomatis di CI.

Kami akan melengkapi contoh detail dan praktik terbaik di versi mendatang.
