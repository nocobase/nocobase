# Pengaturan Bahasa

Di "Pengaturan Sistem", dengan mengonfigurasi "Bahasa yang Diaktifkan", Anda dapat menyesuaikan environment bahasa global sistem. Bahasa pertama dalam daftar akan menjadi bahasa default.

![20240324192005](https://static-docs.nocobase.com/20240324192005.png)

Saat beberapa bahasa diaktifkan, user dapat mengatur preferensi bahasa pribadi user saat ini di pusat pribadi (pojok kanan atas halaman).

![20240324192509](https://static-docs.nocobase.com/20240324192509.png)

Jika environment bahasa yang sedang digunakan kekurangan terjemahan, Anda dapat menggunakan plugin "[Manajemen Lokalisasi](/system-management/localization/index.md)" untuk menerjemahkan dan melengkapi.

![20240324193055](https://static-docs.nocobase.com/20240324193055.png)
