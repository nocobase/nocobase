# Membuat Koleksi



## Membuat Koleksi

Koleksi adalah unit dasar untuk menyimpan dan mengelola data di NocoBase. Anda dapat menganggapnya seperti sebuah tabel dalam basis data.

### Langkah-Langkah Pembuatan

1.  Pada menu di sisi kiri, klik `Sumber Data`.
2.  Pilih sumber data tempat Anda ingin menambahkan koleksi.
3.  Klik tombol `Buat koleksi`.
4.  Pada kotak dialog yang muncul, konfigurasikan opsi berikut:
    *   **Nama tampilan koleksi**: `Pesanan`
    *   **Nama koleksi**: `orders`
5.  Klik `OK` untuk menyimpan.

Setelah berhasil dibuat, Anda dapat mulai [menambahkan field](#).
