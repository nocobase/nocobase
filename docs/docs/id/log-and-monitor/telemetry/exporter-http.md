# Eksportir Telemetri: HTTP

## Variabel Lingkungan

### TELEMETRY_METRIC_READER

Tipe eksportir metrik telemetri.

```bash
TELEMETRY_METRIC_READER=http
```

### TELEMETRY_HTTP_URL

URL HTTP untuk mengekspor data telemetri.

```bash
TELEMETRY_HTTP_URL=
```
