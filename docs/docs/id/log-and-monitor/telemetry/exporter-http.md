# Telemetry Exporter: HTTP

## Konfigurasi Variabel Lingkungan

### TELEMETRY_METRIC_READER

Tipe telemetry metric exporter.

```bash
TELEMETRY_METRIC_READER=http
```

### TELEMETRY_HTTP_URL

URL HTTP untuk ekspor data telemetri.

```bash
TELEMETRY_HTTP_URL=
```
