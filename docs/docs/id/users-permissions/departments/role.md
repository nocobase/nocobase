# Manajemen Peran Departemen

Dengan mengaitkan satu atau lebih peran ke sebuah departemen, anggota departemen tersebut dapat memiliki peran-peran ini.

Ada dua cara untuk mengaitkan peran dengan departemen.

Pertama, pilih peran pada kolom peran di formulir pengeditan departemen.

![](https://static-docs.nocobase.com/70f77bb89aa1fb415c152a51a51cc23b.png)

Kedua, tambahkan departemen ke peran yang sesuai di manajemen peran.

![](https://static-docs.nocobase.com/f2a7bec937cf2f179ce868a92b98416d.png)

Departemen yang sudah memiliki peran ini tidak dapat dipilih.

![](https://static-docs.nocobase.com/be10299893581e1f97a4e01ddd5c7e59.png)
