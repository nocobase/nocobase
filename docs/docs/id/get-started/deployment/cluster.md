# Deployment Klaster

NocoBase mendukung deployment aplikasi dalam mode klaster untuk meningkatkan performa dan ketersediaan aplikasi. Deployment klaster memungkinkan Anda menjalankan berbagai instans aplikasi di beberapa server, atau menjalankan beberapa proses dalam mode multi-core pada satu server.

Untuk detail lebih lanjut, silakan lihat dokumentasi [Mode Klaster](/cluster-mode/).
