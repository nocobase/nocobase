# Deployment Cluster

NocoBase mendukung deployment aplikasi dalam mode cluster untuk meningkatkan performa dan ketersediaan aplikasi. Deployment cluster dapat menjalankan instans aplikasi yang berbeda pada beberapa server, atau menjalankan beberapa proses dalam mode multi-core pada satu server.

Untuk konten detail, silakan lihat: dokumentasi terkait [Mode Cluster](/cluster-mode/index.md).
