# Persetujuan Node

## Notifikasi

Setelah Node Karyawan AI dalam Workflow dipicu, reviewer yang dikonfigurasi dalam Node dapat melihat indikator merah di entry Karyawan AI di halaman utama.

![20260420171416](https://static-docs.nocobase.com/20260420171416.png)

Masuk ke daftar sesi Karyawan AI dapat melihat tugas Workflow yang menunggu persetujuan.

![20260420171642](https://static-docs.nocobase.com/20260420171642.png)

## Persetujuan

Klik tugas Workflow untuk membuka jendela sesi, menampilkan pesan history Karyawan AI. Kartu yang dikirim Karyawan AI terakhir adalah konten yang perlu disetujui Node saat ini.

![20260420180710](https://static-docs.nocobase.com/20260420180710.png)

Untuk konten yang menunggu persetujuan ada tiga operasi: Approve (menyetujui), Revise (memodifikasi), Reject (menolak).

Klik Approve, menyetujui penggunaan konten yang dihasilkan AI, Workflow melanjutkan alur. Sesi menjadi status read-only, tidak dapat melanjutkan dialog dengan Karyawan AI.

![20260420180632](https://static-docs.nocobase.com/20260420180632.png)

Klik Revise, mengajukan saran modifikasi ke AI, setelah dikirim AI akan membalas kartu persetujuan baru.

![20260420180908](https://static-docs.nocobase.com/20260420180908.png)

Klik Reject, menolak eksekusi, Workflow berhenti mengalir. Sesi menjadi status read-only, tidak dapat melanjutkan dialog dengan Karyawan AI.

![20260420181003](https://static-docs.nocobase.com/20260420181003.png)
