# Agen AI Â· Dex: Asisten Penataan Data

> Ekstrak dan isi formulir dengan cepat, mengubah informasi yang berantakan menjadi terstruktur.

`Dex` adalah pakar pengorganisasian data. Ia mengekstrak informasi yang diperlukan dari data atau file tidak terstruktur, mengaturnya menjadi informasi terstruktur, dan dapat menggunakan alat untuk mengisi formulir.

![20251022155210](https://static-docs.nocobase.com/20251022155210.png)

## Langkah-langkah

Di halaman formulir, panggil `Dex` untuk membuka jendela dialog.

![20251022155316](https://static-docs.nocobase.com/20251022155316.png)

Klik `Add work context` di kotak input dan pilih `Pick block`. Halaman akan masuk ke mode pemilihan blok.

![20251022155414](https://static-docs.nocobase.com/20251022155414.png)

Pilih blok formulir di halaman.

![20251022155523](https://static-docs.nocobase.com/20251022155523.png)

Masukkan data yang ingin Anda atur dengan `Dex` di kotak dialog.

![20251022155555](https://static-docs.nocobase.com/20251022155555.png)

Setelah dikirim, `Dex` akan menstrukturkan data dan menggunakan kemampuannya untuk memperbarui data di formulir yang dipilih.

![20251022155733](https://static-docs.nocobase.com/20251022155733.png)
