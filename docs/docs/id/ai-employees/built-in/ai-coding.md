# Karyawan AI Â· Nathan: Teknisi Frontend

> Membantu Anda menulis dan mengoptimalkan kode frontend untuk mengimplementasikan logika interaktif yang kompleks.

`Nathan` adalah ahli pengembangan frontend di NocoBase. Dalam skenario yang memerlukan penulisan JavaScript, seperti `JSBlock`, `JSField`, `JSItem`, `JSColumn`, `JSAction`, `Event Flow`, dan `Linkage`, Anda akan menemukan avatar `Nathan` di sudut kanan atas editor kode. Anda dapat memintanya untuk membantu menulis atau memodifikasi kode di editor tersebut.

![20251022163405](https://static-docs.nocobase.com/20251022163405.png)

## Langkah-langkah

Di editor kode, klik `Nathan` untuk membuka kotak dialog Karyawan AI. Kode dari editor akan secara otomatis dilampirkan ke kotak input dan dikirim ke `Nathan` sebagai konteks aplikasi.

![20251022163935](https://static-docs.nocobase.com/20251022163935.png)

Masukkan kebutuhan pengodean Anda dan kirimkan ke `Nathan`, lalu tunggu balasannya.

![20251022164041](https://static-docs.nocobase.com/20251022164041.png)

Klik tombol `Apply to editor` pada blok kode di balasan `Nathan` untuk menimpa kode di editor kode.

![20251022164323](https://static-docs.nocobase.com/20251022164323.png)

Klik tombol `Run` di editor kode untuk melihat hasilnya secara real-time.

![20251022164436](https://static-docs.nocobase.com/20251022164436.png)

## Riwayat Kode

Klik ikon 'baris perintah' di sudut kanan atas kotak dialog `Nathan` untuk melihat potongan kode yang telah Anda kirim dan yang telah dibalas oleh `Nathan` dalam sesi saat ini.
