# Agen AI Â· Orin: Pemodel Data

> Merancang koleksi secara cerdas dan mengoptimalkan struktur basis data.

`Orin` adalah pakar pemodelan data. Pada halaman konfigurasi sumber data utama, Anda dapat meminta `Orin` membantu Anda membuat atau memodifikasi koleksi.

![20251022160628](https://static-docs.nocobase.com/20251022160628.png)

## Langkah-langkah

Buka plugin Pengelola Sumber Data dan pilih untuk mengonfigurasi sumber data utama.

![20251022161146](https://static-docs.nocobase.com/20251022161146.png)

Klik avatar `Orin` di pojok kanan atas untuk membuka kotak dialog Agen AI.

![20251022161641](https://static-docs.nocobase.com/20251022161641.png)

Jelaskan kebutuhan pemodelan Anda kepada `Orin`, kirim, dan tunggu responsnya.

![20251022161749](https://static-docs.nocobase.com/20251022161749.png)

Setelah `Orin` mengonfirmasi kebutuhan Anda, ia akan menggunakan kemampuannya dan membalas dengan pratinjau model data.

Setelah meninjau pratinjau, klik tombol `Finish review and apply` untuk membuat koleksi sesuai dengan model `Orin`.

![20251022162142](https://static-docs.nocobase.com/20251022162142.png)
