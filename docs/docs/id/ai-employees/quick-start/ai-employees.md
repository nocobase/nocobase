# Mulai Cepat

## Pendahuluan
Setelah menyelesaikan konfigurasi layanan LLM, kita dapat mulai membuat karyawan AI.

## Karyawan AI Baru

### Konfigurasi Informasi Dasar

Buka halaman konfigurasi **plugin** karyawan AI, lalu klik tab `AI employees` untuk masuk ke halaman manajemen karyawan AI.

![20251021225643](https://static-docs.nocobase.com/20251021225643.png)

Klik tombol `New AI employee` untuk masuk ke halaman pembuatan karyawan AI.

Pertama, di tab `Profile`, kita dapat mengonfigurasi informasi dasar karyawan AI:

- Di kolom input `Username`, masukkan identifikasi unik untuk karyawan AI;
- Di kolom input `Nickname`, masukkan nama panggilan untuk karyawan AI;
- Di kolom input `Position`, masukkan posisi karyawan AI, yang digunakan untuk menjelaskan secara singkat tanggung jawab mereka;
- Di `Avatar`, pilih avatar untuk karyawan AI.

![20251021232246](https://static-docs.nocobase.com/20251021232246.png)

- Di kolom input `Bio`, masukkan teks pengantar untuk karyawan AI, yang menjelaskan keterampilan dan fungsinya. Teks ini akan ditampilkan di profil karyawan AI;
- Di kolom input `About me`, masukkan deskripsi rinci fungsi karyawan AI. Teks ini akan digunakan sebagai *prompt* sistem saat berinteraksi dengan karyawan AI.
- Di kolom input `Greeting message`, masukkan pesan sapaan untuk karyawan AI. Setiap kali pengguna memulai percakapan baru dengan karyawan AI, sistem akan secara otomatis mengirimkan pesan ini kepada pengguna.

![20251021232712](https://static-docs.nocobase.com/20251021232712.png)

### Konfigurasi Model Besar

Di tab `Model settings`, kita dapat mengonfigurasi pengaturan model untuk karyawan AI:

- Pilih layanan LLM yang telah kita buat di manajemen layanan LLM;
- Masukkan nama model besar yang ingin kita gunakan.

![20251021231727](https://static-docs.nocobase.com/20251021231727.png)

### Selesaikan Pembuatan

Setelah memasukkan semua informasi yang diperlukan untuk membuat karyawan AI, klik tombol `Submit` untuk menyelesaikan pembuatan.

Kemudian, kita dapat melihat karyawan AI yang baru dibuat di daftar karyawan AI. Selain itu, karyawan AI yang baru saja kita buat juga akan terlihat di tombol panggil cepat karyawan AI di kanan bawah halaman.

![20251021232836](https://static-docs.nocobase.com/20251021232836.png)
