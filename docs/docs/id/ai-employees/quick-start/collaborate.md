# Mulai Cepat

## Pendahuluan

Setelah membuat Karyawan AI, Anda dapat berkolaborasi dengannya di halaman mana pun dalam NocoBase.

Anda dapat mengeklik avatar karyawan di daftar karyawan pojok kanan bawah untuk memanggil dan mulai berkolaborasi dengannya.

![20251102121159-2025-11-02-12-12-01](https://static-docs.nocobase.com/20251102121159-2025-11-02-12-12-01.png)

Anda juga dapat menempatkan Karyawan AI di mana pun Anda membutuhkannya — seperti pada tabel, formulir, atau komponen lainnya — dan mengaktifkannya langsung dengan sekali klik.

![20251102121036-2025-11-02-12-10-38](https://static-docs.nocobase.com/20251102121036-2025-11-02-12-10-38.png)

## Operasi Dasar di Kotak Obrolan

Setelah memanggil Karyawan AI, masukkan pesan Anda dan klik tombol kirim untuk memulai percakapan.

![20251021234541](https://static-docs.nocobase.com/20251021234541.png)

Saat Anda mengarahkan kursor ke pesan pengguna, ikon tindakan cepat akan muncul:
- Edit dan kirim ulang pesan
- Salin konten pesan pengguna

![20251021235715](https://static-docs.nocobase.com/20251021235715.png)

Saat Anda mengarahkan kursor ke pesan AI, ikon tindakan cepat akan muncul:
- Minta AI untuk membalas lagi
- Salin konten pesan AI

![20251021235847](https://static-docs.nocobase.com/20251021235847.png)

Klik ikon "Unggah" untuk mengunggah dan mengirim berkas ke Karyawan AI.

![20251022000327](https://static-docs.nocobase.com/20251022000327.png)

Di pojok kiri atas kotak obrolan terdapat daftar riwayat percakapan. Klik untuk memperluas daftar dan melihat percakapan sebelumnya.

![20251021234640](https://static-docs.nocobase.com/20251021234640.png)

Klik ikon "pensil" di pojok kanan atas untuk memulai percakapan baru.

![20251021234841](https://static-docs.nocobase.com/20251021234841.png)

Klik ikon "i" di pojok kanan atas untuk mengedit prompt sistem kustom.

![20251021235314](https://static-docs.nocobase.com/20251021235314.png)

Klik ikon "perluas" di pojok kanan atas untuk menampilkan kotak obrolan Karyawan AI dalam mode layar penuh.

![20251021235427](https://static-docs.nocobase.com/20251021235427.png)
