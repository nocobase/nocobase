# Mengaktifkan Karyawan AI

NocoBase menyediakan beberapa Karyawan AI bawaan untuk berbagai skenario.

## Cara Mengaktifkan

Buka halaman manajemen `AI employees`, lalu aktifkan sakelar `Enabled` secara langsung pada daftar untuk mengaktifkan karyawan.

![ai-employee-list-enable-switch.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employee-list-enable-switch.png)

## Penggunaan Setelah Diaktifkan

Setelah diaktifkan, karyawan tersebut akan muncul di titik akses yang tersedia (seperti pintasan di pojok kanan bawah atau titik akses skenario tertentu).
