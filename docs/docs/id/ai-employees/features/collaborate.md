# Berkolaborasi dengan Karyawan AI

Setelah membuat dan mengaktifkan Karyawan AI, Anda dapat berkolaborasi dengan mereka di dalam halaman.

## Titik Akses

Terdapat dua jenis titik akses umum:

1. **Titik akses utama di pojok kanan bawah**: Membuka panel obrolan AI dari pojok kanan bawah halaman bisnis. Cocok untuk tanya jawab umum dan kolaborasi lintas blok.
2. **Titik akses Action blok**: Pada blok yang mendukung `Actions`, akses melalui `Actions -> AI employees`. Cocok untuk menjalankan tugas pada blok saat ini (misalnya skenario JSBlock).

## Operasi Dasar Obrolan

Panel obrolan mendukung operasi umum seperti mengirim pesan, mengunggah lampiran, melihat riwayat percakapan, membuat percakapan baru, dan mengedit perintah sistem (*system prompts*).

## Peralihan di Dalam Obrolan

Di bagian bawah area input, Anda dapat beralih secara langsung:

- `AI Employee Switcher`: Beralih antar karyawan saat ini.
- `Model Switcher`: Beralih model sesi saat ini.

Preferensi model disimpan berdasarkan dimensi karyawan dan akan dipulihkan secara otomatis saat akses berikutnya.
