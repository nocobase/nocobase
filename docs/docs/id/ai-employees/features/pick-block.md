# Tambah Konteks - Blok

Saat berinteraksi dengan AI Employee, selain mengirimkan berkas, Anda juga dapat mengirimkan informasi konteks aplikasi kepada AI Employee, sehingga AI Employee dapat memberikan balasan berdasarkan informasi konteks aplikasi tersebut.

### Mengirimkan Informasi Blok

Di dalam kotak obrolan AI Employee, klik tombol `Tambah konteks` di sudut kiri bawah dan pilih `Pilih Blok`. Halaman aplikasi akan masuk ke mode pemilihan blok.

![20251022010148](https://static-docs.nocobase.com/20251022010148.png)

Saat mengarahkan kursor ke atas sebuah blok, warna blok yang dapat dikirim sebagai konteks akan berubah.

![20251022010422](https://static-docs.nocobase.com/20251022010422.png)

Setelah memilih blok yang ingin dikirim, informasi konteks blok tersebut akan dilampirkan ke dalam kotak obrolan.

![20251022010543](https://static-docs.nocobase.com/20251022010543.png)

Setelah memasukkan pesan pertanyaan, klik tombol kirim seperti biasa, dan AI Employee akan memberikan balasan berdasarkan informasi blok tersebut.

Informasi blok juga akan muncul dalam daftar pesan.

![20251022010758](https://static-docs.nocobase.com/20251022010758.png)

### Mengakhiri Pemilihan Blok

Ketika halaman aplikasi berada dalam mode pemilihan blok namun Anda tidak ingin memilih blok apa pun, klik ikon "Berhenti" di bagian bawah untuk keluar dari mode pemilihan blok.

![20251022011043](https://static-docs.nocobase.com/20251022011043.png)
