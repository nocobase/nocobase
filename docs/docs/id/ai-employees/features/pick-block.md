# Tambahkan Konteks - Block

Saat berdialog dengan Karyawan AI, selain mengirim file, juga dapat mengirim informasi konteks aplikasi ke Karyawan AI, biarkan Karyawan AI membalas berdasarkan informasi konteks aplikasi.

### Mengirim Informasi Block

Pada kotak dialog Karyawan AI, klik tombol `Add work context` di pojok kiri bawah, pilih `Pick block`, halaman aplikasi akan masuk ke status pemilihan Block.

![20251022010148](https://static-docs.nocobase.com/20251022010148.png)

Saat mouse hover pada Block, warna Block yang dapat dikirim sebagai konteks akan berubah.

![20251022010422](https://static-docs.nocobase.com/20251022010422.png)

Setelah memilih Block yang ingin dikirim, informasi konteks Block akan terlampir di kotak dialog.

![20251022010543](https://static-docs.nocobase.com/20251022010543.png)

Setelah memasukkan pesan pertanyaan dan sama seperti mengirim pesan, klik tombol kirim, Karyawan AI akan membalas berdasarkan informasi Block.

Informasi Block juga akan muncul di daftar pesan.

![20251022010758](https://static-docs.nocobase.com/20251022010758.png)

### Mengakhiri Pemilihan Block

Saat halaman aplikasi masuk ke status pemilihan Block, namun tidak ingin memilih Block apa pun, klik ikon "stop" di bagian bawah untuk keluar dari status pemilihan Block.

![20251022011043](https://static-docs.nocobase.com/20251022011043.png)
