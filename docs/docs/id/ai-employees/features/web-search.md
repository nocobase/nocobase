# Pencarian Web

Pencarian Web digunakan untuk melengkapi informasi terbaru di luar data pelatihan model.

## Cara Kerja

Ketersediaan Pencarian Web bergantung pada apakah layanan model yang dipilih dalam sesi saat ini mendukung Pencarian Web.

- **Mendukung**: Sakelar Pencarian Web akan ditampilkan dan dapat diaktifkan/dinonaktifkan sesuai kebutuhan.
- **Tidak mendukung**: Sakelar tidak akan ditampilkan dan status pencarian akan dinonaktifkan secara otomatis.

## Penggunaan dalam Sesi Percakapan

Gunakan sakelar Pencarian Web di area input percakapan:

- **Aktif**: AI dapat melakukan pencarian terlebih dahulu sebelum memberikan jawaban.

![web-search-switch-visible-on-supported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-visible-on-supported-model.png)

- **Nonaktif**: AI hanya akan menjawab berdasarkan konteks yang sudah ada.

![web-search-switch-hidden-on-unsupported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-hidden-on-unsupported-model.png)

## Perbedaan Platform

Kemampuan dukungan Pencarian Web bervariasi antar model, silakan gunakan sesuai dengan situasi aktual.
