# Tugas Pintasan

Agar karyawan AI dapat mulai bekerja dengan lebih efisien, kita dapat menautkan karyawan AI ke blok skenario dan mengatur beberapa tugas umum sebelumnya.

Hal ini memungkinkan pengguna untuk memulai pemrosesan tugas dengan satu klik, tanpa harus **Memilih Blok** dan **Memasukkan Perintah** setiap saat.

## Menautkan Karyawan AI ke Blok

Setelah masuk ke mode pengeditan UI, pada blok yang mendukung `Actions`, pilih menu `AI employees` di bawah `Actions`, lalu pilih seorang karyawan AI. Karyawan AI tersebut akan ditautkan ke blok saat ini.

![20251022135306](https://static-docs.nocobase.com/20251022135306.png)

Setelah penautan selesai, setiap kali Anda masuk ke halaman tersebut, area Actions pada blok akan menampilkan karyawan AI yang ditautkan dengan blok tersebut.

![20251022135438](https://static-docs.nocobase.com/20251022135438.png)

## Mengatur Tugas

Setelah masuk ke mode pengeditan UI, arahkan kursor ke ikon karyawan AI yang ditautkan pada blok. Tombol menu akan muncul, pilih `Edit tasks` untuk masuk ke halaman pengaturan tugas.

Di halaman pengaturan tugas, Anda dapat menambahkan beberapa tugas untuk karyawan AI tersebut.

Setiap tab mewakili tugas yang independen, klik tanda "+" di sampingnya untuk menambahkan tugas baru.

![clipboard-image-1771913187](https://static-docs.nocobase.com/clipboard-image-1771913187.png)

Formulir pengaturan tugas:

- Masukkan judul tugas di kotak input `Title`. Judul yang singkat akan muncul dalam daftar tugas karyawan AI.
- Masukkan konten utama tugas di kotak input `Background`. Konten ini akan digunakan sebagai prompt sistem saat berdialog dengan karyawan AI.
- Masukkan pesan pengguna default di kotak input `Default user message`. Pesan ini akan terisi secara otomatis di kotak input pengguna setelah tugas dipilih.
- Di bagian `Work context`, pilih informasi konteks aplikasi default yang akan dikirim ke karyawan AI. Bagian ini beroperasi sama seperti pada panel dialog.
- Kotak pilihan `Skills` menampilkan keterampilan yang dimiliki oleh karyawan AI saat ini. Anda dapat membatalkan pilihan keterampilan tertentu agar karyawan AI mengabaikannya saat menjalankan tugas tersebut.
- Centang kotak `Send default user message automatically` untuk mengonfigurasi apakah pesan pengguna default akan dikirim secara otomatis setelah tugas diklik untuk dijalankan.


## Daftar Tugas

Setelah tugas diatur untuk karyawan AI, tugas-tugas tersebut akan muncul di jendela popover profil karyawan AI dan dalam pesan sapaan sebelum percakapan dimulai. Klik untuk menjalankan tugas.
