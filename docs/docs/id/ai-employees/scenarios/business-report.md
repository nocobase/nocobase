# Menghasilkan Laporan Analisis Bisnis

Mengambil contoh sistem CRM pada DEMO online, biarkan Karyawan AI menganalisis data pelanggan dan order dalam sistem, dan menghasilkan laporan analisis.

## Memuat Skills

Pertama, Karyawan AI akan memuat Skills terkait, kemudian mulai bekerja.

![](https://static-docs.nocobase.com/202604230924035.png)

## Menganalisis Model Data

Mendapatkan data source, informasi meta tabel data.

![](https://static-docs.nocobase.com/202604230926297.png)

## Query Data Bisnis

Mulai query data bisnis, tergantung pada konten yang dianalisis berbeda, proses ini biasanya membutuhkan beberapa puluh detik hingga beberapa menit.

![](https://static-docs.nocobase.com/202604230927861.png)

## Menghasilkan Laporan

Setelah mengumpulkan cukup banyak data bisnis, mulai menghasilkan laporan analisis.

![](https://static-docs.nocobase.com/202604230920113.png)

Akhirnya menghasilkan laporan analisis bisnis dengan teks dan gambar, mendukung download dalam format Markdown, HTML, PDF.
