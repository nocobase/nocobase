# Поле "Регион Китая"

## Введение

Плагин "Поле "Регион Китая"" (`field-china-region`) добавляет в NocoBase тип поля "Регион Китая".

## Установка

В основном интерфейсе NocoBase перейдите в "Менеджер плагинов", найдите плагин "Регион Китая" и включите его.

## Использование

В конфигурации коллекции добавьте новое поле и выберите "Регион Китая" в качестве его типа.

![](/images/field-china-region-1.png)

### Конфигурация поля

В конфигурации поля вы можете установить режим отображения для поля:

- Провинция
- Провинция/Город
- Провинция/Город/Район
- Провинция/Город/Район/Уезд

![](/images/field-china-region-2.png)
