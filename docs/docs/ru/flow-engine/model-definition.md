# Поле "Регион Китая"

## Введение

Плагин "Поле Региона Китая" (`field-china-region`) добавляет в NocoBase тип поля "Регион Китая".

## Установка (Installation)

В основном интерфейсе NocoBase перейдите в "Менеджер плагинов", найдите плагин "Регион Китая" и включите его.

## Использование

В настройке коллекции добавьте новое поле и выберите "Регион Китая" в качестве типа поля.

![](/images/field-china-region-1.png)

### Конфигурация поля

В конфигурации поля можно задать режим отображения:

- Провинция
- Провинция/город
- Провинция/город/район
- Провинция/город/район/округ

![](/images/field-china-region-2.png)
