# Лог аудита

## Введение

Лог аудита используется для записи и отслеживания действий пользователей и истории операций с ресурсами в системе.

![](https://static-docs.nocobase.com/202501031627719.png)

![](https://static-docs.nocobase.com/202501031627922.png)

## Описание параметров

| Параметр | Описание |
| --------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| **Ресурс** | Тип целевого ресурса операции |
| **Действие** | Тип выполняемой операции |
| **Пользователь** | Пользователь, выполняющий операцию |
| **Роль** | Роль пользователя во время выполнения операции |
| **Источник данных** | Источник данных |
| **Целевая коллекция** | Целевая коллекция |
| **Уникальный ключ целевой записи** | Уникальный ключ целевой записи в коллекции |
| **Исходная коллекция** | Исходная коллекция поля связи |
| **Уникальный ключ исходной записи** | Уникальный ключ исходной записи в коллекции |
| **Статус** | Код состояния HTTP в ответе на запрос операции |
| **Время создания** | Время выполнения операции |
| **UUID** | Уникальный идентификатор операции; совпадает с ID запроса операции, может использоваться для получения логов приложения |
| **IP** | IP-адрес пользователя |
| **идентификатор клиента** | Сведения об агенте пользователя (клиентское приложение и браузер) |
| **Метаданные** | Метаданные запроса операции: параметры, тело запроса и содержимое ответа |

## Описание ресурсов аудита

В настоящее время в лог аудита записываются следующие операции с ресурсами:

### Основное приложение

| Операция | Описание |
| ---------------- | ----------------------- |
| `app:resart`     | Перезапуск приложения |
| `app:clearCache` | Очистка кэша приложения |

### Управление плагинами

| Операция | Описание |
| ------------ | -------------- |
| `pm:add`     | Добавить плагин |
| `pm:update`  | Обновить плагин |
| `pm:enable`  | Включить плагин |
| `pm:disable` | Отключить плагин |
| `pm:remove`  | Удалить плагин |

### Аутентификация пользователей

| Операция | Описание |
| --------------------- | --------------- |
| `auth:signIn`         | Вход |
| `auth:signUp`         | Регистрация |
| `auth:signOut`        | Выход |
| `auth:changePassword` | Смена пароля |

### Пользователь

| Операция | Описание |
| --------------------- | -------------- |
| `users:updateProfile` | Обновление профиля |

### Конфигурация UI

| Операция | Описание |
| -------------------------- | ---------------- |
| `uiSchemas:insertAdjacent` | Вставка схемы UI |
| `uiSchemas:patch`          | Изменение схемы UI |
| `uiSchemas:remove`         | Удаление схемы UI |

### Операции с коллекциями

| Операция | Описание |
| ---------------- | --------------------------------- |
| `create`         | Создание записи |
| `update`         | Обновление записи |
| `destroy`        | Удаление записи |
| `updateOrCreate` | Обновление или создание записи |
| `firstOrCreate`  | Поиск или создание записи |
| `move`           | Перемещение записи |
| `set`            | Установка записи поля связи |
| `add`            | Добавление записи поля связи |
| `remove`         | Удаление записи поля связи |
| `export`         | Экспорт записи |
| `import`         | Импорт записи |

## Добавление других ресурсов аудита

Если вы расширили другие операции с ресурсами через плагины и хотите записывать эти действия в логи аудита, см. [API](/api/server/audit-manager.md).
