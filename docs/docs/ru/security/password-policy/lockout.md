# Блокировка пользователей

## Введение

Управляйте заблокированными пользователями и активно блокируйте их.

![](https://static-docs.nocobase.com/202412281450083.png)

### Разблокировать пользователя

Пользователи, заблокированные из-за превышения лимита неверных попыток входа, могут быть разблокированы путем удаления соответствующей записи.

### Заблокировать пользователя

Вы можете активно заблокировать пользователя, добавив соответствующую запись. После блокировки пользователь не сможет получить доступ к системе любым способом аутентификации, включая API-ключи.

![](https://static-docs.nocobase.com/202412281450512.png)
