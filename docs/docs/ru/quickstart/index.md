# Быстрый старт

Если вы впервые используете этот интерфейс командной строки, вам не нужно запоминать все команды вначале. Используйте `nb init --ui`, чтобы сначала установить приложение, а затем продолжить просмотр остальных по сценарию.

## Сначала установите самый важный ум

В NocoBase CLI последующие операции по умолчанию вращаются не вокруг «определенного каталога» или «определенного порта», а вокруг **env**.

Вы можете думать о env как о «наборе информации о подключении приложения и запуске, запоминаемой CLI». Если он был успешно сохранен, многие последующие команды можно использовать напрямую:

- Используйте `nb init`, чтобы установить новое приложение и сохранить его как env.
– Используйте `nb env add` для подключения существующего приложения к CLI.
- Управляйте этим окружением с помощью `nb app start`, `nb app logs`, `nb app upgrade`.
- Сделайте резервную копию и восстановите эту среду с помощью `nb backup`.
– Используйте `nb app autostart`, `nb proxy`, чтобы продолжать расширять возможности производственной среды.

Имейте это в виду в первую очередь, и последующие документы будут намного более гладкими.

## Рекомендуемый путь по умолчанию

Если вы не знаете, с чего начать, обычно проще всего пойти по этому пути:

1. Сначала прочтите [Установка с помощью CLI (рекомендуется)](./installation/cli.md) и один раз выполните `nb init`.
2. После сохранения приложения как среды просмотрите [Управление несколькими средами](./operations/multi-environment.md), чтобы подтвердить текущую среду, переключить среду и проверить статус.
3. Для ежедневного запуска, остановки, регистрации и обновления см. раздел [Управление приложением](./operations/manage-app.md).
4. Прежде чем выполнять обновления, миграцию или важные изменения, см. раздел [Резервное копирование и восстановление](./operations/backup-restore.md).
5. Если вы готовы официально подключиться к Интернету, введите [Обзор развертывания производственной среды] (./production/index.md).

Первые три шага охватывают большинство сценариев использования.

## Когда следует просмотреть ссылку на команду

Этот набор документов для быстрого старта больше похож на вопрос «что я хочу сделать сейчас». Если вы уже знаете, какую команду хотите выполнить, и хотите продолжить просмотр полных параметров, просто перейдите в [Справочник по командам NocoBase CLI](../api/cli/index.md).

Предложения по умолчанию:

- Сначала используйте документ «Быстрый старт», чтобы определить путь.
- Затем проверьте сведения о параметрах на конкретной странице команды.

Это упрощает начало работы, чем чтение полного дерева команд с первого взгляда.
