# Экспорт телеметрии: HTTP

## Переменные окружения

### TELEMETRY_METRIC_READER

Тип экспортера метрик телеметрии.

```bash
TELEMETRY_METRIC_READER=http
```

### TELEMETRY_HTTP_URL

HTTP URL для экспорта данных телеметрии.

```bash
TELEMETRY_HTTP_URL=
```
