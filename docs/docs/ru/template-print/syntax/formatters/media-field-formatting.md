### Форматирование медиа-полей

#### 1. :attachment

##### Синтаксис

Выводит изображение из поля «Вложение». Обычно переменную можно скопировать напрямую из «Списка полей».

##### Пример

```text
{d.contractFiles[0].id:attachment()}
```

##### Результат

Выводит соответствующее изображение из вложения.

#### 2. :signature

##### Синтаксис

Выводит изображение подписи, связанное с полем рукописной подписи. Обычно переменную можно скопировать напрямую из «Списка полей».

##### Пример

```text
{d.customerSignature:signature()}
```

##### Результат

Выводит соответствующее изображение рукописной подписи.

> **Внимание:** для полей вложений и рукописной подписи рекомендуется копировать переменную напрямую из списка полей в «Конфигурации шаблона», чтобы избежать ошибок при ручном вводе.
