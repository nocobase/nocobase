## Инструкции по настройке

### Активация печати шаблонов
Сейчас печать шаблонов поддерживает блоки деталей и табличные блоки. Ниже приведены способы настройки для этих двух типов блоков.

#### Блоки деталей

1. **Откройте блок деталей**:
- Перейдите к блоку деталей в приложении, где нужно использовать функцию печати шаблонов.

2. **Откройте меню операций настройки**:
- Нажмите меню «Операция настройки» в верхней части интерфейса.

3. **Выберите «Печать шаблонов»**:
- Нажмите пункт «Печать шаблонов» в выпадающем меню, чтобы активировать плагин.

![Активация печати шаблонов](https://static-docs.nocobase.com/20241212150539-2024-12-12-15-05-43.png)

### Настройка шаблонов

1. **Откройте страницу настройки шаблонов**:
- В меню настройки кнопки «Печать шаблонов» выберите пункт «Конфигурация шаблона».

![Пункт «Конфигурация шаблона»](https://static-docs.nocobase.com/20241212151858-2024-12-12-15-19-01.png)

2. **Добавьте новый шаблон**:
- Нажмите кнопку «Добавить шаблон», чтобы перейти на страницу добавления шаблона.

![Кнопка «Добавить шаблон»](https://static-docs.nocobase.com/20241212151243-2024-12-12-15-12-46.png)

3. **Заполните информацию о шаблоне**:
- В форме шаблона укажите имя шаблона и выберите тип шаблона (Word, Excel, PowerPoint).
- Загрузите соответствующий файл шаблона (поддерживаются форматы `.docx`, `.xlsx`, `.pptx`).

![Настройка имени и файла шаблона](https://static-docs.nocobase.com/20241212151518-2024-12-12-15-15-21.png)

4. **Отредактируйте и сохраните шаблон**:
- Перейдите на страницу «Список полей», скопируйте поля и вставьте их в шаблон.
![Список полей](https://static-docs.nocobase.com/20250107141010.png)
![20241212152743-2024-12-12-15-27-45](https://static-docs.nocobase.com/20241212152743-2024-12-12-15-27-45.png)
- После заполнения деталей нажмите кнопку «Сохранить», чтобы завершить добавление шаблона.

5. **Управление шаблонами**:
- Нажмите кнопку «Использовать» справа в списке шаблонов, чтобы активировать шаблон.
- Нажмите кнопку «Редактировать», чтобы изменить имя шаблона или заменить файл шаблона.
- Нажмите кнопку «Скачать», чтобы скачать настроенный файл шаблона.
- Нажмите кнопку «Удалить», чтобы убрать ненужные шаблоны. Система запросит подтверждение, чтобы избежать случайного удаления.
![Управление шаблонами](https://static-docs.nocobase.com/20250107140436.png)

#### Табличные блоки

Использование табличных блоков в целом такое же, как у блоков деталей, со следующими отличиями:

1. **Поддержка печати нескольких записей**: сначала нужно отметить записи для печати. За один раз можно распечатать до 100 записей.

![20250416215633-2025-04-16-21-56-35](https://static-docs.nocobase.com/20250416215633-2025-04-16-21-56-35.png)

2. **Изолированное управление шаблонами**: шаблоны для табличных блоков и блоков деталей не взаимозаменяемы, потому что структуры данных различаются (в одном случае объект, в другом — массив).
