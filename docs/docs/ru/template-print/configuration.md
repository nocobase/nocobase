## Настройка

### Активация печати по шаблону
Печать по шаблону в настоящее время поддерживается для блоков с деталями и табличных блоков. Ниже мы рассмотрим способы настройки для каждого из этих типов блоков.

#### Блоки с деталями

1.  **Откройте блок с деталями**:
    -   В приложении перейдите к блоку с деталями, где вы хотите использовать функцию печати по шаблону.

2.  **Откройте меню «Операции настройки»**:
    -   В верхней части интерфейса нажмите меню «Операции настройки».

3.  **Выберите «Печать по шаблону»**:
    -   В выпадающем меню нажмите опцию «Печать по шаблону», чтобы активировать функциональность плагина.

![Активация печати по шаблону](https://static-docs.nocobase.com/20241212150539-2024-12-12-15-05-43.png)

### Настройка шаблонов

1.  **Перейдите на страницу настройки шаблонов**:
    -   В меню настройки кнопки «Печать по шаблону» выберите опцию «Настройка шаблонов».

![Опция настройки шаблонов](https://static-docs.nocobase.com/20241212151858-2024-12-12-15-19-01.png)

2.  **Добавьте новый шаблон**:
    -   Нажмите кнопку «Добавить шаблон», чтобы перейти на страницу добавления шаблона.

![Кнопка «Добавить шаблон»](https://static-docs.nocobase.com/20241212151243-2024-12-12-15-12-46.png)

3.  **Заполните информацию о шаблоне**:
    -   В форме шаблона укажите название шаблона и выберите его тип (Word, Excel, PowerPoint).
    -   Загрузите соответствующий файл шаблона (поддерживаются форматы `.docx`, `.xlsx`, `.pptx`).

![Настройка названия и файла шаблона](https://static-docs.nocobase.com/20241212151518-2024-12-12-15-15-21.png)

4.  **Отредактируйте и сохраните шаблон**:
    -   Перейдите на страницу «Список полей», скопируйте поля и вставьте их в шаблон.
    ![Список полей](https://static-docs.nocobase.com/20250107141010.png)
    ![Изображение](https://static-docs.nocobase.com/20241212152743-2024-12-12-15-27-45.png)
    -   После заполнения всех данных нажмите кнопку «Сохранить», чтобы завершить добавление шаблона.

5.  **Управление шаблонами**:
    -   Нажмите кнопку «Использовать» справа от списка шаблонов, чтобы активировать шаблон.
    -   Нажмите кнопку «Редактировать», чтобы изменить название шаблона или заменить файл шаблона.
    -   Нажмите кнопку «Скачать», чтобы загрузить уже настроенный файл шаблона.
    -   Нажмите кнопку «Удалить», чтобы удалить ненужный шаблон. Система запросит подтверждение действия во избежание случайного удаления.
    ![Управление шаблонами](https://static-docs.nocobase.com/20250107140436.png)

#### Табличные блоки

Использование табличных блоков в основном аналогично блокам с деталями, но есть следующие отличия:

1.  Поддержка печати нескольких записей: Сначала вам нужно выбрать записи для печати, установив флажки. Одновременно можно распечатать до 100 записей.
    
![Выбор нескольких записей для печати](https://static-docs.nocobase.com/20250416215633-2025-04-16-21-56-35.png)

2.  Изолированное управление шаблонами: Шаблоны для табличных блоков и блоков с деталями не взаимозаменяемы, поскольку их структуры данных различаются (один представляет собой объект, другой — массив).
