# HTTP-ответ

## Введение

Этот узел поддерживается только в синхронных рабочих процессах Webhook и используется для отправки ответа сторонней системе. Например, при обработке обратного вызова платежа, если бизнес-процесс завершается с непредвиденным результатом (например, ошибкой или сбоем), вы можете использовать узел ответа, чтобы вернуть сообщение об ошибке сторонней системе. Это позволит некоторым сторонним системам повторить попытку позже, основываясь на полученном статусе.

Кроме того, выполнение узла ответа завершает выполнение рабочего процесса, и последующие узлы не будут выполняться. Если в рабочем процессе не настроен узел ответа, система автоматически отправит ответ, основываясь на статусе выполнения процесса: вернёт `200` при успешном выполнении и `500` при сбое.

## Создание узла ответа

В интерфейсе настройки рабочего процесса нажмите кнопку «плюс» («+») в схеме, чтобы добавить узел «Ответ»:
