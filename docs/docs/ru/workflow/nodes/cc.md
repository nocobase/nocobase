# Копия <Badge>v1.8.2+</Badge>

## Введение

Узел Копия используется для отправки определенного контекстного содержимого из процесса выполнения рабочего процесса указанным пользователям для их ознакомления и просмотра. Например, в процессе утверждения или другом рабочем процессе соответствующая информация может быть отправлена в копии другим участникам, чтобы они могли своевременно отслеживать ход работы.

Вы можете настроить несколько узлов Копия в рабочем процессе. Когда выполнение рабочего процесса достигнет такого узла, соответствующая информация будет отправлена указанным получателям.

Содержимое, отправленное в копии, будет отображаться в меню «Копия мне» в Центре задач, где пользователи могут просмотреть весь контент, отправленный им в копии. Также будет отображаться индикатор непрочитанных элементов, и пользователи смогут вручную отметить их как прочитанные после просмотра.

## Создание узла

В интерфейсе настройки рабочего процесса нажмите кнопку плюс («+») в потоке, чтобы добавить узел «Копия»:

![Добавить узел Копия](https://static-docs.nocobase.com/20250710222842.png)

## Настройка узла

![Настройка узла](https://static-docs.nocobase.com/20250710224041.png)

В интерфейсе настройки узла вы можете задать следующие параметры:

### Получатели

Получатели — это коллекция целевых пользователей для копии, которая может состоять из одного или нескольких пользователей. Источник может быть статическим значением, выбранным из списка пользователей, динамическим значением, указанным переменной, или результатом запроса к коллекции пользователей.

![Настройка получателей](https://static-docs.nocobase.com/20250710224421.png)

### Пользовательский интерфейс

Получателям необходимо просматривать содержимое, отправленное в копии, в меню «Копия мне» в Центре задач. Вы можете настроить результаты триггера и любого узла в контексте рабочего процесса в качестве блоков содержимого.

![Пользовательский интерфейс](https://static-docs.nocobase.com/20250710225400.png)

### Заголовок задачи

Заголовок задачи — это заголовок, отображаемый в Центре задач. Вы можете использовать переменные из контекста рабочего процесса для динамического формирования заголовка.

![Заголовок задачи](https://static-docs.nocobase.com/20250710225603.png)

## Центр задач

Пользователи могут просматривать и управлять всем содержимым, отправленным им в копии, в Центре задач, а также фильтровать и просматривать его по статусу прочтения.

![20250710232932](https://static-docs.nocobase.com/20250710232932.png)

![20250710233032](https://static-docs.nocobase.com/20250710233032.png)

После просмотра вы можете отметить его как прочитанное, и количество непрочитанных элементов соответственно уменьшится.

![20250710233102](https://static-docs.nocobase.com/20250710233102.png)
