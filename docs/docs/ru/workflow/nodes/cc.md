# Копия <Badge>v1.8.2+</Badge>

## Введение

Узел «Копия» используется для отправки определенного контекстного содержимого из процесса выполнения рабочего процесса указанным пользователям для их ознакомления и просмотра. Например, в процессе утверждения или других процессах соответствующая информация может быть отправлена в копии другим участникам, чтобы они могли своевременно узнавать о ходе работы.

В рабочем процессе можно настроить несколько узлов «Копия», чтобы при достижении процессом этого узла соответствующая информация отправлялась указанным получателям.

Содержимое копии будет отображаться в меню «Копия мне» Центра задач, где пользователи могут просматривать все отправленные им материалы. Система будет уведомлять о непрочитанном содержимом, которое пользователи могут самостоятельно отметить как прочитанное после ознакомления.

## Создание узла

В интерфейсе конфигурации рабочего процесса нажмите кнопку «плюс» («+») в потоке, чтобы добавить узел «Копия»:

![抄送_添加](https://static-docs.nocobase.com/20250710222842.png)

## Настройка узла

![节点配置](https://static-docs.nocobase.com/20250710224041.png)

В интерфейсе конфигурации узла можно настроить следующие параметры:

### Получатели

Получатели — это набор целевых пользователей для копии, который может включать одного или нескольких пользователей. Источником выбора может быть статическое значение из списка пользователей, динамическое значение, заданное переменной, или результат запроса к таблице пользователей.

![接收者配置](https://static-docs.nocobase.com/20250710224421.png)

### Пользовательский интерфейс

Получатели просматривают содержимое копии в меню «Копия мне» Центра задач. Вы можете настроить результаты триггера и любого узла из контекста рабочего процесса в качестве блоков содержимого.

![用户界面](https://static-docs.nocobase.com/20250710225400.png)

### Карточка задачи <Badge>2.0+</Badge>

Используется для настройки карточек задач в списке «Копия мне» Центра задач.

![20260213010947](https://static-docs.nocobase.com/20260213010947.png)

В карточке можно свободно настраивать отображение необходимых бизнес-полей (за исключением полей отношений).

После создания задачи копирования в рабочем процессе в списке Центра задач появится настроенная карточка задачи:

![20260214124325](https://static-docs.nocobase.com/20260214124325.png)

### Заголовок задачи

Заголовок задачи — это заголовок, отображаемый в Центре задач. Вы можете использовать переменные из контекста рабочего процесса для динамического формирования заголовка.

![任务标题](https://static-docs.nocobase.com/20250710225603.png)

## Центр задач

Пользователи могут просматривать и управлять всем содержимым, отправленным им в копии, в Центре задач, а также фильтровать и просматривать его в зависимости от статуса прочтения.

![20250710232932](https://static-docs.nocobase.com/20250710232932.png)

![20250710233032](https://static-docs.nocobase.com/20250710233032.png)

После просмотра содержимое можно пометить как прочитанное, при этом количество непрочитанных элементов уменьшится.

![20250710233102](https://static-docs.nocobase.com/20250710233102.png)
