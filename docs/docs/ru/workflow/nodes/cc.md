# Копия <Badge>v1.8.2+</Badge>

## Введение

Узел «Копия» используется для отправки определенного контекстного содержимого из процесса выполнения рабочего процесса указанным пользователям для ознакомления и контроля. Например, в процессе согласования или другом процессе релевантную информацию можно отправить копией другим участникам, чтобы они были в курсе прогресса.

В одном рабочем процессе можно настроить несколько узлов «Копия». Когда выполнение рабочего процесса доходит до такого узла, соответствующая информация отправляется указанным получателям.

Содержимое, отправленное копией, отображается в меню «Копия мне» Центра задач (список задач), где пользователи могут просматривать все материалы, отправленные им копией. Там также показываются непрочитанные элементы, и после просмотра их можно вручную пометить как прочитанные.

## Создание узла

В интерфейсе настройки рабочего процесса нажмите кнопку плюса ("+") в потоке, чтобы добавить узел «Копия»:

![Добавление узла копии](https://static-docs.nocobase.com/20250710222842.png)

## Настройка узла

![Настройка узла](https://static-docs.nocobase.com/20250710224041.png)

В интерфейсе конфигурации узла можно настроить следующие параметры:

### Получатели

Получатели — это набор целевых пользователей для отправки копии; это может быть один или несколько пользователей. Источником может быть статическое значение из списка пользователей, динамическое значение через переменную или результат запроса к коллекции пользователей.

![Настройка получателей](https://static-docs.nocobase.com/20250710224421.png)

### Пользовательский интерфейс

Получатели просматривают содержимое копии в меню «Копия мне» Центра задач (список задач). В качестве блоков содержимого можно настроить результаты триггера и любого узла в контексте рабочего процесса.

![Пользовательский интерфейс](https://static-docs.nocobase.com/20250710225400.png)

### Карточка задачи <Badge>2.0+</Badge>

Используется для настройки карточки задачи в списке «Копия мне» Центра задач (список задач).

![20260213010947](https://static-docs.nocobase.com/20260213010947.png)

Карточку можно свободно настроить для отображения нужных бизнес-полей (кроме полей связей).

После создания задачи копии в рабочем процессе в списке Центра задач будет отображаться настроенная карточка задачи:

![20260214124325](https://static-docs.nocobase.com/20260214124325.png)

### Заголовок задачи

Заголовок задачи — это заголовок, отображаемый в Центре задач (список задач). Для динамической генерации заголовка можно использовать переменные контекста рабочего процесса.

![Заголовок задачи](https://static-docs.nocobase.com/20250710225603.png)

## Центр задач (список задач)

Пользователи могут просматривать и управлять всем содержимым, отправленным им копией, в Центре задач (список задач), а также фильтровать и просматривать элементы по статусу прочтения.

![20250710232932](https://static-docs.nocobase.com/20250710232932.png)

![20250710233032](https://static-docs.nocobase.com/20250710233032.png)

После просмотра элемент можно пометить как прочитанный, и счетчик непрочитанных уменьшится соответствующим образом.

![20250710233102](https://static-docs.nocobase.com/20250710233102.png)
