# Вывод

## Введение

Узел «Вывод» используется во вложенном (вызываемом) рабочем процессе для определения его выходного значения. Когда один рабочий процесс вызывается из другого, узел «Вывод» может передать значение обратно в вызывающий рабочий процесс.

## Создание узла

В вызываемом рабочем процессе добавьте узел «Вывод»:

![Создание узла «Вывод»](https://static-docs.nocobase.com/20241231002033.png)

## Настройка узла

### Значение вывода

Введите или выберите переменную как выходное значение. Выходное значение может быть любого типа: константа (строка, число, логическое значение, дата или пользовательский JSON) либо другая переменная из рабочего процесса.

![Настройка значения вывода](https://static-docs.nocobase.com/20241231003059.png)

:::info{title=Совет}
Если в вызываемом рабочем процессе добавлено несколько узлов «Вывод», при вызове рабочего процесса будет выведено значение последнего выполненного узла «Вывод».
:::
