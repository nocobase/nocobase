# Вывод рабочего процесса

## Введение

Узел «Вывод рабочего процесса» используется в вызываемом рабочем процессе для определения его выходного значения. Когда один рабочий процесс вызывается другим, узел «Вывод рабочего процесса» позволяет передать значение обратно вызывающей стороне.

## Создание узла

В вызываемом рабочем процессе добавьте узел «Вывод рабочего процесса»:

![20241231002033](https://static-docs.nocobase.com/20241231002033.png)

## Настройка узла

### Выходное значение

Введите или выберите переменную в качестве выходного значения. Выходное значение может быть любого типа: константой (например, строкой, числом, логическим значением, датой или пользовательским JSON) или другой переменной из рабочего процесса.

![20241231003059](https://static-docs.nocobase.com/20241231003059.png)

:::info{title=Совет}
Если в вызываемый рабочий процесс добавлено несколько узлов «Вывод рабочего процесса», то при вызове этого рабочего процесса будет выведено значение последнего выполненного узла «Вывод рабочего процесса».
:::
