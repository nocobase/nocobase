# Отправка электронной почты

## Введение

Этот узел используется для отправки электронных писем. Он поддерживает текстовый и HTML-формат содержимого.

## Создание узла

В интерфейсе настройки рабочего процесса нажмите кнопку с плюсом («+»), чтобы добавить узел «Отправка электронной почты»:

![20251031131125](https://static-docs.nocobase.com/20251031131125.png)

## Часто задаваемые вопросы

### Ограничение частоты отправки писем через Gmail

При отправке некоторых писем вы можете столкнуться со следующей ошибкой:

```json
{
  "code": "ECONNECTION",
  "response": "421-4.7.0 Try again later, closing connection. (EHLO)\n421-4.7.0 For more information, go to\n421 4.7.0 About SMTP error messages - Google Workspace Admin Help 3f1490d57ef6-e7f7352f44csm831688276.30 - gsmtp",
  "responseCode": 421,
  "command": "EHLO"
}
```

Это происходит потому, что Gmail ограничивает частоту отправки запросов с неопределённых доменов. При развёртывании приложения вам необходимо настроить имя хоста сервера так, чтобы оно соответствовало домену, который вы привязали в Gmail. Например, при развёртывании в Docker:

```yaml
services:
  app:
    image: nocobase/nocobase
    hostname: <your-custom-hostname> # Укажите ваш привязанный домен для отправки
```

Ссылка: [Google SMTP Relay - Intermittent problems](https://forum.nocobase.com/t/google-smtp-relay-intermittent-problems/5483/6)
