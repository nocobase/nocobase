# Обзор

Встроенные возможности рабочего процесса не могут покрыть все сценарии. Например, встроенные типы узлов не охватывают все возможные операции во всех бизнес-сценариях. Поэтому мы также предоставляем механизм расширения рабочего процесса, включая расширение типов триггеров и узлов. Там, где встроенных возможностей недостаточно, вы можете расширять их методами с минимумом кода.

Расширения в основном делятся на две части:

- [Расширение типов триггеров](./trigger.md)
- [Расширение типов узлов](./node.md)
