# Обзор

Встроенные функции **рабочего процесса** не всегда могут охватить все возможные сценарии. Например, стандартные типы узлов не могут исчерпывающе представить каждую операцию, необходимую в различных бизнес-процессах. Поэтому мы предусмотрели возможность расширения **рабочего процесса**, включая расширение типов триггеров и узлов. В тех случаях, когда встроенных функций недостаточно, вы можете использовать low-code методы для реализации необходимых расширений.

Расширения в основном делятся на две части:

- [Расширение типов триггеров](./trigger.md)
- [Расширение типов узлов](./node.md)
