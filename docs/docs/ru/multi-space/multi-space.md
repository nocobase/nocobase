# Мультипространство

## Введение

**Плагин Мультипространство (Multi-space)** позволяет создавать несколько независимых областей данных внутри одного экземпляра приложения с помощью логической изоляции.

#### Сценарии использования
- **Несколько магазинов или заводов**: Бизнес-процессы и конфигурации системы идентичны (например, единое управление запасами, планирование производства, стратегии продаж и шаблоны отчетов), но необходимо обеспечить независимость данных для каждого подразделения.
- **Управление холдингом или дочерними компаниями**: Несколько организаций или филиалов в составе группы компаний используют одну платформу, но каждый бренд имеет свои независимые данные о клиентах, продуктах и заказах.

## Установка

Найдите плагин **Multi-Space** в менеджере плагинов и включите его.

![](https://static-docs.nocobase.com/multi-space/Plugin-manager-NocoBase-10-15-2025_09_57_AM.png)

## Руководство пользователя

### Управление пространствами

После включения плагина перейдите на страницу настроек **«Пользователи и права» (Users & Permissions)** и переключитесь на панель **Пространства (Workspaces)** для управления ими.

> По умолчанию существует встроенное **Нераспределенное пространство (Unassigned Space)**, которое в основном используется для просмотра старых данных, еще не связанных ни с одним пространством.

![](https://static-docs.nocobase.com/multi-space/Users-Permissions-NocoBase-10-15-2025_09_58_AM.png)

#### Создание пространства

Нажмите кнопку «Добавить пространство», чтобы создать новое пространство:

![](https://static-docs.nocobase.com/multi-space/Users-Permissions-NocoBase-10-15-2025_09_58_AM%20(1).png)

#### Назначение пользователей

Выбрав созданное пространство, вы можете назначить пользователей, принадлежащих этому пространству, в правой части экрана:

> **Подсказка:** После назначения пользователей пространству необходимо **вручную обновить страницу**, чтобы список переключения пространств в правом верхнем углу обновился и отобразил актуальные данные.

![](https://static-docs.nocobase.com/multi-space/Users-Permissions-NocoBase-10-15-2025_09_59_AM.png)

### Переключение и просмотр пространств

Вы можете переключать текущее пространство в правом верхнем углу.  
При нажатии на **иконку глаза** справа (в активном состоянии) вы можете просматривать данные из нескольких пространств одновременно.

![](https://static-docs.nocobase.com/multi-space/Goods-NocoBase-10-15-2025_10_26_AM.png)

### Управление данными в мультипространстве

Когда плагин включен, система автоматически добавляет предустановленное **поле пространства** при создании новой коллекции (Collection).  
**Только те коллекции, которые содержат это поле, будут включены в логику управления пространствами.**

![](https://static-docs.nocobase.com/multi-space/data-source-manager-main-NocoBase-10-15-2025_10_01_AM.png)

Для существующих коллекций вы можете вручную добавить поле пространства, чтобы включить управление ими:

![](https://static-docs.nocobase.com/multi-space/data-source-manager-main-NocoBase-10-15-2025_10_03_AM.png)

#### Логика по умолчанию

В коллекциях, содержащих поле пространства, система автоматически применяет следующую логику:

1. При создании данных они автоматически связываются с текущим выбранным пространством;
2. При фильтрации данных отображаются только те записи, которые относятся к текущему выбранному пространству.

### Классификация существующих данных по пространствам

Для данных, которые существовали до включения плагина «Мультипространство», вы можете выполнить распределение по пространствам следующим образом:

#### 1. Добавление поля пространства

Вручную добавьте поле пространства в существующую коллекцию:

![](https://static-docs.nocobase.com/multi-space/data-source-manager-main-NocoBase-10-15-2025_10_18_AM.png)

#### 2. Назначение пользователей в нераспределенное пространство

Свяжите пользователей, управляющих старыми данными, со всеми пространствами, включая **Нераспределенное пространство (Unassigned Space)**, чтобы они могли видеть данные, еще не закрепленные за конкретным пространством:

![](https://static-docs.nocobase.com/multi-space/Users-Permissions-NocoBase-10-15-2025_10_14_AM.png)

#### 3. Переключение на просмотр данных всех пространств

Выберите опцию в верхней части экрана для просмотра данных из всех пространств:

![](https://static-docs.nocobase.com/multi-space/Goods-NocoBase-10-15-2025_10_20_AM.png)

#### 4. Настройка страницы распределения старых данных

Создайте новую страницу для распределения старых данных. Добавьте «Поле пространства» как в **блок списка**, так и в **форму редактирования**, чтобы иметь возможность вручную изменять принадлежность к пространству.

![](https://static-docs.nocobase.com/multi-space/Goods-10-15-2025_10_21_AM.png)

Установите поле пространства в режим редактирования:

![](https://static-docs.nocobase.com/multi-space/Goods-10-15-2025_10_22_AM.png)

#### 5. Ручное распределение данных по пространствам

Используя созданную страницу, вручную отредактируйте записи, чтобы постепенно назначить правильное пространство для старых данных (вы также можете настроить массовое редактирование самостоятельно).
