# nb db stop

Останавливает встроенный контейнер базы данных для указанного env. Эта команда применяется только к env, в которых включена встроенная база данных, управляемая CLI.

## Использование

```bash
nb db stop [flags]
```

## Параметры

| Параметр      | Тип     | Описание                                                                                                 |
| ------------- | ------- | -------------------------------------------------------------------------------------------------------- |
| `--env`, `-e` | string  | Имя CLI env, встроенную базу данных которого нужно остановить; если не указано, используется текущий env |
| `--verbose`   | boolean | Показывает вывод базовой команды Docker                                                                  |

## Примеры

```bash
nb db stop
nb db stop --env app1
nb db stop --env app1 --verbose
```
