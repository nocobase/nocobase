# nb db stop

Останавливает контейнер встроенной базы данных для указанного окружения. Эта команда применяется только к окружениям со встроенной базой данных, управляемой CLI.

## Использование

```bash
nb db stop [flags]
```

## Параметры

| Параметр      | Тип     | Описание                                                                                                 |
| ------------- | ------- | -------------------------------------------------------------------------------------------------------- |
| `--env`, `-e` | string  | Имя CLI-окружения, встроенную базу данных которого нужно остановить; если не указано, используется текущее окружение |
| `--verbose`   | boolean | Показывает вывод низкоуровневых Docker-команд                                                            |

## Примеры

```bash
nb db stop
nb db stop --env app1
nb db stop --env app1 --verbose
```
