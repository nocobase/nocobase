# nb db start

Запуск контейнера встроенной базы данных выбранного окружения. Эта команда применима только к окружениям, созданным со встроенной базой данных, управляемой CLI.

## Использование

```bash
nb db start [flags]
```

## Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| `--env`, `-e` | string | Имя CLI-окружения, для которого запускается встроенная база данных; если не указано, используется текущее окружение |
| `--verbose` | boolean | Показать вывод низкоуровневых Docker-команд |

## Примеры

```bash
nb db start
nb db start --env app1
nb db start --env app1 --verbose
```
