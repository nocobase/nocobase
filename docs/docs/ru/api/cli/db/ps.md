# nb db ps

Просмотр состояния среды выполнения встроенной базы данных без запуска или остановки каких-либо ресурсов. Если параметр `--env` не указан, отображается состояние базы данных всех настроенных окружений.

## Использование

```bash
nb db ps [flags]
```

## Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| `--env`, `-e` | string | Имя CLI-окружения для просмотра; если не указано, отображаются все окружения |

## Примеры

```bash
nb db ps
nb db ps --env app1
```
