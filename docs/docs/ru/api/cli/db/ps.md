# nb db ps

Просмотр состояния встроенной базы данных без запуска или остановки каких-либо ресурсов. Если параметр `--env` не указан, отображается состояние базы данных всех настроенных env.

## Использование

```bash
nb db ps [flags]
```

## Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| `--env`, `-e` | string | Имя CLI env для просмотра; если не указано, отображаются все env |

## Примеры

```bash
nb db ps
nb db ps --env app1
```
