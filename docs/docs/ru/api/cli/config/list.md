# nb config list

Показывает явно заданные настройки CLI. Значения по умолчанию, которые никогда не задавались явно, не отображаются.

## Использование

```bash
nb config list
```

## Примеры

```bash
nb config list
```

## Примечания

Если значения конфигурации не заданы, CLI выведет `No CLI config values are set.`.
