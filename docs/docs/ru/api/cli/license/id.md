# nb license id

Показывает идентификатор экземпляра (`Instance ID`) коммерческой лицензии для выбранного окружения. Если сохранённого идентификатора экземпляра ещё нет, CLI автоматически создаст и сохранит его.

## Использование

```bash
nb license id [flags]
```

## Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| `--env`, `-e` | string | Имя CLI-окружения; если не указано, используется текущее окружение |
| `--yes`, `-y` | boolean | Если явно переданный `--env` указывает на окружение, отличное от текущего, пропускает интерактивное подтверждение |
| `--force` | boolean | Принудительно пересоздать идентификатор экземпляра, даже если он уже сохранён |
| `--json` | boolean | Вывод JSON |

## Примеры

```bash
nb license id
nb license id --env app1
nb license id --env app1 --yes
nb license id --env app1 --force
nb license id --env app1 --json
```

`--force` только принудительно пересоздаёт идентификатор экземпляра. Он не заменяет подтверждение операции с другим окружением: если явно переданный `--env` указывает на окружение, отличное от текущего, всё равно потребуется подтверждение или `--yes`.
