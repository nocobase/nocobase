# nb proxy nginx status

Показывает текущее состояние среды выполнения прокси Nginx.

## Использование

```bash
nb proxy nginx status
```

## Вывод

Вывод обычно включает:

- `driver`
- `status`
- `config`
- `nginx` или `container`

## Примеры

```bash
nb proxy nginx status
```
