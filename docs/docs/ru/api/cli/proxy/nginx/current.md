# nb proxy nginx current

Показывает текущий driver provider Nginx.

## Использование

```bash
nb proxy nginx current
```

## Вывод

Команда выводит одно из двух значений:

- `local`
- `docker`

## Примеры

```bash
nb proxy nginx current
```
