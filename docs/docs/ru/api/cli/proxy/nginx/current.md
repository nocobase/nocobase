# nb proxy nginx current

Выводит текущий драйвер провайдера Nginx.

## Использование

```bash
nb proxy nginx current
```

## Вывод

Команда напрямую выводит одно значение:

- `local`
- `docker`

## Примеры

```bash
nb proxy nginx current
```
