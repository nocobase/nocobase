# nb proxy caddy current

Показывает текущий driver provider Caddy.

## Использование

```bash
nb proxy caddy current
```

## Вывод

Команда выводит одно из двух значений:

- `local`
- `docker`

## Примеры

```bash
nb proxy caddy current
```
