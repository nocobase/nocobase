# nb proxy caddy current

Выводит текущий драйвер провайдера Caddy.

## Использование

```bash
nb proxy caddy current
```

## Вывод

Команда напрямую выводит одно значение:

- `local`
- `docker`

## Примеры

```bash
nb proxy caddy current
```
