# nb proxy caddy status

Показывает текущее состояние среды выполнения прокси Caddy.

## Использование

```bash
nb proxy caddy status
```

## Вывод

Вывод обычно включает:

- `driver`
- `status`
- `config`
- `caddy` или `container`

## Примеры

```bash
nb proxy caddy status
```
