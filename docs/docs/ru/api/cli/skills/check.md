# nb skills check

Проверяет глобальные навыки ИИ-разработки NocoBase и сообщает, управляет ли ими CLI и доступны ли обновления.

## Использование

```bash
nb skills check [flags]
```

## Параметры

| Параметр | Тип | Описание |
| --- | --- | --- |
| `--json` | boolean | Вывести в формате JSON |

## Примеры

```bash
nb skills check
nb skills check --json
```
