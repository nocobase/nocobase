**Тип**

type FindOneOptions = Omit<FindOptions, 'limit'>;

**Параметры**

Большинство параметров совпадают с параметрами `find()`. Разница в том, что `findOne()` возвращает только одну запись, поэтому параметр `limit` не требуется и при запросе всегда равен `1`.
