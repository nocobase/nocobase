## Параметры

| Имя параметра | Тип | Значение по умолчанию | Описание |
| ---------------------- | ------------- | ------------- | --------------------------------------------------------------------------- |
| `options.values` | `M` | `{}` | Объект данных, который необходимо вставить |
| `options.whitelist?` | `string[]` | - | Белый список полей для `values`. Будут сохранены только поля в списке.   |
| `options.blacklist?` | `string[]` | - | Черный список полей для `values`. Поля в списке не будут сохранены.    |
| `options.transaction?` | `Transaction` | - | Транзакция |
