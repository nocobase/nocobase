## Параметры

| Имя параметра         | Тип          | Значение по умолчанию | Описание                                                               |
| :--------------------- | :----------- | :-------------------- | :--------------------------------------------------------------------- |
| `options.values`       | `M`          | `{}`                  | Объект данных для вставки                                              |
| `options.whitelist?`   | `string[]`   | -                     | Белый список полей для `values`. Сохраняются только поля из этого списка. |
| `options.blacklist?`   | `string[]`   | -                     | Черный список полей для `values`. Поля из этого списка не сохраняются.   |
| `options.transaction?` | `Transaction` | -                     | Транзакция                                                             |
