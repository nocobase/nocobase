# Auth - Аутентификация

## Обзор

Класс `Auth` в основном используется на стороне клиента для доступа к информации о пользователе и запросов к API, связанным с аутентификацией пользователя.

## Свойства экземпляра

### `locale`

Язык, используемый текущим пользователем.

### `role`

Роль, используемая текущим пользователем.

### `token`

API-токен.

### `authenticator`

Аутентификатор, используемый для аутентификации текущего пользователя. См. [Аутентификация пользователя](/auth-verification/auth/).

## Методы класса

### `signIn()`

Вход пользователя.

#### Сигнатура

- `async signIn(values: any, authenticator?: string): Promise<AxiosResponse<any>>`

#### Подробности

| Имя параметра   | Тип      | Описание                                                |
| --------------- | -------- | ------------------------------------------------------- |
| `values`        | `any`    | Параметры запроса для API входа                         |
| `authenticator` | `string` | Идентификатор аутентификатора, используемого для входа |

### `signUp()`

Регистрация пользователя.

#### Сигнатура

- `async signUp(values: any, authenticator?: string): Promise<AxiosResponse<any>>`

#### Подробности

| Имя параметра   | Тип      | Описание                                                      |
| --------------- | -------- | ------------------------------------------------------------- |
| `values`        | `any`    | Параметры запроса для API регистрации                         |
| `authenticator` | `string` | Идентификатор аутентификатора, используемого для регистрации |

### `signOut()`

Выход пользователя.

#### Сигнатура

- `async signOut(values: any, authenticator?: string): Promise<AxiosResponse<any>>`

#### Подробности

| Имя параметра   | Тип      | Описание                                                |
| --------------- | -------- | ------------------------------------------------------- |
| `values`        | `any`    | Параметры запроса для API выхода                        |
| `authenticator` | `string` | Идентификатор аутентификатора, используемого для выхода |
