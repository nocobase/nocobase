# Расширенные возможности

## Введение

Современные большие языковые модели (БЯМ) обладают способностью использовать инструменты. Плагин "AI-сотрудник" включает в себя набор встроенных инструментов, которые БЯМ могут использовать.

Навыки, которые вы настраиваете на странице параметров AI-сотрудника, и являются теми инструментами, которые доступны для использования большой языковой моделью.

![20251022142348](https://static-docs.nocobase.com/20251022142348.png)

## Настройка навыков

Перейдите на страницу конфигурации плагина AI-сотрудника, затем нажмите вкладку `AI employees`, чтобы попасть на страницу управления AI-сотрудниками.

Выберите AI-сотрудника, для которого вы хотите настроить навыки, и нажмите кнопку `Edit`, чтобы перейти на страницу редактирования AI-сотрудника.

На вкладке `Skills` нажмите кнопку `Add Skill`, чтобы добавить навык для текущего AI-сотрудника.

![20251022145748](https://static-docs.nocobase.com/20251022145748.png)

## Обзор навыков

### Frontend

Группа Frontend позволяет AI-сотруднику взаимодействовать с компонентами пользовательского интерфейса (фронтенда).

- Навык `Form filler` позволяет AI-сотруднику заполнять сгенерированные данные формы в указанную пользователем форму.

![20251022145954](https://static-docs.nocobase.com/20251022145954.png)

### Моделирование данных

Группа навыков "Моделирование данных" предоставляет AI-сотруднику возможность вызывать внутренние API NocoBase для моделирования данных.

- `Intent Router` (Маршрутизатор намерений) определяет, хочет ли пользователь изменить структуру **коллекции** или создать новую.
- `Get collection names` получает названия всех существующих **коллекций** в системе.
- `Get collection metadata` получает информацию о структуре указанной **коллекции**.
- `Define collections` позволяет AI-сотруднику создавать **коллекции** в системе.

![20251022150441](https://static-docs.nocobase.com/20251022150441.png)

### Workflow caller

`Workflow caller` предоставляет AI-сотруднику возможность выполнять **рабочие процессы**. **Рабочие процессы**, настроенные в **плагине** **рабочих процессов** с `Trigger type` как `AI employee event`, будут доступны здесь в качестве навыков для использования AI-сотрудником.

![20251022153320](https://static-docs.nocobase.com/20251022153320.png)

### Редактор кода

Навыки, входящие в группу "Редактор кода", в основном позволяют AI-сотруднику взаимодействовать с редактором кода.

- `Get code snippet list` получает список предустановленных фрагментов кода.
- `Get code snippet content` получает содержимое указанного фрагмента кода.

![20251022153811](https://static-docs.nocobase.com/20251022153811.png)

### Прочее

- `Chart generator` предоставляет AI-сотруднику возможность генерировать диаграммы и выводить их непосредственно в диалоге.
