# Продвинутые возможности

## Введение

ИИ-сотрудников можно привязывать к страницам или блокам. После привязки вы сможете настроить задачи, специфичные для текущего бизнес-процесса, что позволит пользователям быстро использовать ИИ-сотрудника для их выполнения прямо на странице или в блоке.

## Привязка ИИ-сотрудника к странице

Когда страница переходит в режим редактирования UI, в правом нижнем углу, рядом с кнопкой быстрого вызова ИИ-сотрудника, появится значок «+». Наведите курсор на этот значок, чтобы открыть список ИИ-сотрудников. Выберите нужного ИИ-сотрудника, и он будет привязан к текущей странице.

![20251022134656](https://static-docs.nocobase.com/20251022134656.png)

После завершения привязки, при каждом входе на страницу, в правом нижнем углу будет отображаться ИИ-сотрудник, привязанный к текущей странице.

![20251022134903](https://static-docs.nocobase.com/20251022134903.png)

## Привязка ИИ-сотрудника к блоку

Когда страница переходит в режим редактирования UI, для блока, поддерживающего настройку `Actions`, выберите меню `AI employees` в разделе `Actions`, а затем выберите ИИ-сотрудника, чтобы привязать его к текущему блоку.

![20251022135306](https://static-docs.nocobase.com/20251022135306.png)

После завершения привязки, при каждом входе на страницу, в области `Actions` блока будет отображаться ИИ-сотрудник, привязанный к текущему блоку.

![20251022135438](https://static-docs.nocobase.com/20251022135438.png)

## Настройка задач

Когда страница переходит в режим редактирования UI, наведите курсор на значок ИИ-сотрудника, привязанного к странице или блоку. Появится кнопка меню. Выберите `Edit tasks`, чтобы перейти на страницу настройки задач.

![20251022135710](https://static-docs.nocobase.com/20251022135710.png)

На странице настройки задач вы можете добавить несколько задач для текущего ИИ-сотрудника.

Каждая вкладка представляет собой отдельную задачу. Нажмите значок «+» рядом с ней, чтобы добавить новую задачу.

![20251022140058](https://static-docs.nocobase.com/20251022140058.png)

Форма настройки задач:

- В поле ввода `Title` введите заголовок задачи. Кратко опишите ее содержание. Этот заголовок будет отображаться в списке задач ИИ-сотрудника.
- В поле ввода `Background` введите основное содержание задачи. Это содержание будет использоваться в качестве системного промпта при общении с ИИ-сотрудником.
- В поле ввода `Default user message` введите сообщение пользователя по умолчанию, которое будет автоматически заполняться в поле ввода пользователя после выбора задачи.
- В `Work context` выберите информацию о контексте приложения по умолчанию, которая будет отправлена ИИ-сотруднику. Эта операция аналогична действиям в диалоговом окне.
- В поле выбора `Skills` отображаются доступные навыки текущего ИИ-сотрудника. Вы можете отменить выбор определенного навыка, чтобы ИИ-сотрудник игнорировал его и не использовал при выполнении этой задачи.
- Флажок `Send default user message automatically` определяет, будет ли автоматически отправляться сообщение пользователя по умолчанию после нажатия кнопки выполнения задачи.

![20251022140805](https://static-docs.nocobase.com/20251022140805.png)

## Список задач

После настройки задач для ИИ-сотрудника, эти задачи будут отображаться во всплывающем окне профиля ИИ-сотрудника и в приветственном сообщении перед началом беседы. Нажмите на задачу, чтобы выполнить ее.
