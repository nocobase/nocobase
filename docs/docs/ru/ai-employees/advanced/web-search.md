# Продвинутые возможности

## Введение

Обычно большие языковые модели (БЯМ) имеют низкую актуальность данных и не располагают свежими данными. Поэтому онлайн-платформы сервисов БЯМ обычно предоставляют функцию веб-поиска, позволяя ИИ использовать инструменты для поиска информации перед ответом, а затем отвечать на основе результатов поиска.

ИИ-сотрудники поддерживают функцию веб-поиска различных онлайн-платформ сервисов БЯМ. Вы можете включить эту функцию как в конфигурации модели ИИ-сотрудника, так и непосредственно в диалогах.

## Включение функции веб-поиска

Перейдите на страницу настройки плагина ИИ-сотрудников, нажмите вкладку `AI employees`, чтобы перейти на страницу управления ИИ-сотрудниками.

![20251021225643](https://static-docs.nocobase.com/20251021225643.png)

Выберите ИИ-сотрудника, для которого вы хотите включить функцию веб-поиска, нажмите кнопку `Edit`, чтобы перейти на страницу редактирования ИИ-сотрудника.

![20251022114043](https://static-docs.nocobase.com/20251022114043.png)

На вкладке `Model settings` включите переключатель `Web Search` и нажмите кнопку `Submit`, чтобы сохранить изменения.

![20251022114300](https://static-docs.nocobase.com/20251022114300.png)

## Использование функции веб-поиска в диалогах

После включения функции веб-поиска для ИИ-сотрудника в поле ввода диалога появится значок "Веб". По умолчанию веб-поиск включен, но вы можете нажать на значок, чтобы отключить его.

![20251022115110](https://static-docs.nocobase.com/20251022115110.png)

При включенном веб-поиске в ответе ИИ-сотрудника будут отображаться результаты веб-поиска.

![20251022115502](https://static-docs.nocobase.com/20251022115502.png)
