# Расширенные возможности

## Введение

При общении с AI-агентом вы можете не только отправлять файлы, но и передавать ему контекст приложения. Это позволит AI-агенту отвечать, основываясь на этой контекстной информации.

### Отправка информации о блоках

В диалоговом окне AI-агента нажмите кнопку `Add work context` в левом нижнем углу и выберите `Pick block`. Страница приложения перейдет в режим выбора блоков.

![20251022010148](https://static-docs.nocobase.com/20251022010148.png)

При наведении курсора мыши на блок его цвет изменится, указывая на то, что его можно отправить в качестве контекста.

![20251022010422](https://static-docs.nocobase.com/20251022010422.png)

После выбора нужного блока информация о его контексте будет добавлена в диалоговое окно.

![20251022010543](https://static-docs.nocobase.com/20251022010543.png)

После ввода вопроса, как и при отправке обычного сообщения, нажмите кнопку отправки. AI-агент ответит, основываясь на информации из блока.

Информация о блоке также появится в списке сообщений.

![20251022010758](https://static-docs.nocobase.com/20251022010758.png)

### Завершение выбора блоков

Если страница приложения находится в режиме выбора блоков, но вы не хотите выбирать ни один из них, вы можете нажать значок "Стоп" внизу, чтобы выйти из этого режима.

![20251022011043](https://static-docs.nocobase.com/20251022011043.png)
