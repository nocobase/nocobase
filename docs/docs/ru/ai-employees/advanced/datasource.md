# Расширенные возможности

## Введение

В плагине AI-помощника вы можете настроить источники данных и предварительно задать запросы к коллекциям. Эти запросы отправляются в качестве контекста при общении с AI-помощником, который затем отвечает на основе полученных результатов.

## Настройка источников данных

Перейдите на страницу настроек плагина AI-помощника, нажмите вкладку `Data source`, чтобы попасть на страницу управления источниками данных AI-помощника.

![20251022103526](https://static-docs.nocobase.com/20251022103526.png)

Нажмите кнопку `Add data source`, чтобы перейти на страницу создания источника данных.

Шаг 1: Введите основную информацию о `коллекции`:
- В поле `Title` введите легко запоминающееся название для источника данных;
- В поле `Collection` выберите источник данных и коллекцию, которые вы хотите использовать;
- В поле `Description` введите описание источника данных.
- В поле `Limit` укажите ограничение на количество записей в запросе к источнику данных, чтобы избежать возврата слишком большого объема данных, который может превысить контекст диалога AI.

![20251022103935](https://static-docs.nocobase.com/20251022103935.png)

Шаг 2: Выберите поля для запроса:

В списке `Fields` отметьте поля, которые вы хотите запросить.

![20251022104516](https://static-docs.nocobase.com/20251022104516.png)

Шаг 3: Установите условия запроса:

![20251022104635](https://static-docs.nocobase.com/20251022104635.png)

Шаг 4: Установите условия сортировки:

![20251022104722](https://static-docs.nocobase.com/20251022104722.png)

Наконец, перед сохранением источника данных вы можете предварительно просмотреть результаты запроса к источнику данных.

![20251022105012](https://static-docs.nocobase.com/20251022105012.png)

## Отправка источников данных в диалогах

В диалоговом окне AI-помощника нажмите кнопку `Add work context` в левом нижнем углу, выберите `Data source`, и вы увидите только что добавленный источник данных.

![20251022105240](https://static-docs.nocobase.com/20251022105240.png)

Отметьте источник данных, который вы хотите отправить; выбранный источник данных будет прикреплен к диалоговому окну.

![20251022105401](https://static-docs.nocobase.com/20251022105401.png)

После ввода вопроса, как и при отправке обычного сообщения, нажмите кнопку отправки, и AI-помощник ответит на основе источника данных.

Источник данных также появится в списке сообщений.

![20251022105611](https://static-docs.nocobase.com/20251022105611.png)

## Примечания

Источник данных автоматически отфильтрует информацию на основе прав доступа ACL текущего пользователя, показывая только те данные, к которым у пользователя есть доступ.
