# Мультимодальный диалог

## Изображения

Если модель поддерживает это, узел LLM может отправлять изображения в модель. При использовании вам необходимо выбрать поле вложения или связанную запись коллекции файлов через переменную. При выборе записи коллекции файлов вы можете выбрать её на уровне объекта или указать поле URL.

![](https://static-docs.nocobase.com/202503041034858.png)

Доступны два варианта формата отправки изображений:

- Отправка по URL – Все изображения, кроме хранящихся локально, будут отправлены в виде URL-адресов. Локально хранящиеся изображения будут преобразованы в формат base64 перед отправкой.
- Отправка в формате base64 – Все изображения, будь то локально хранящиеся или в облаке, будут отправлены в формате base64. Этот вариант подходит для случаев, когда URL изображения не может быть напрямую доступен онлайн-сервисом LLM.
