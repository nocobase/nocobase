# Мультимодальный чат

## Изображения

Если модель это поддерживает, узел языковой модели может отправлять модели изображения. При использовании этой возможности вам нужно выбрать поле вложения или связанную запись коллекции файлов через переменную. При выборе записи коллекции файлов вы можете выбрать её целиком как объект либо выбрать поле URL.

![](https://static-docs.nocobase.com/202503041034858.png)

Существует два варианта формата отправки изображений:

- Отправка через URL — все изображения, кроме хранящихся локально, будут отправлены как URL. Локально хранящиеся изображения перед отправкой будут преобразованы в формат base64.
- Отправка в base64 — все изображения, независимо от того, хранятся ли они локально или в облаке, будут отправлены в формате base64. Такой вариант подходит для случаев, когда онлайн‑сервис языковой модели не имеет прямого доступа к URL изображений.
