# Согласование узла

## Уведомление

После срабатывания узла AI-сотрудника в рабочем процессе настроенные в узле согласующие лица увидят красную точку-индикатор у точки входа AI-сотрудников на главной странице.

![20260420171416](https://static-docs.nocobase.com/20260420171416.png)

В списке диалогов с AI-сотрудником можно увидеть задачи рабочего процесса, ожидающие согласования.

![20260420171642](https://static-docs.nocobase.com/20260420171642.png)

## Согласование

Нажмите на задачу рабочего процесса, чтобы открыть окно диалога, где отображается история сообщений AI-сотрудника. Последняя карточка, отправленная AI-сотрудником, и есть содержимое, требующее согласования в данном узле.

![20260420180710](https://static-docs.nocobase.com/20260420180710.png)

Для содержимого, ожидающего согласования, доступны три действия: Approve (одобрить), Revise (изменить), Reject (отклонить).

При нажатии Approve содержимое, сгенерированное ИИ, одобряется, и рабочий процесс продолжается. Диалог становится доступным только для чтения, продолжать общение с AI-сотрудником невозможно.

![20260420180632](https://static-docs.nocobase.com/20260420180632.png)

При нажатии Revise можно отправить ИИ замечания на правку. После отправки ИИ ответит новой карточкой согласования.

![20260420180908](https://static-docs.nocobase.com/20260420180908.png)

При нажатии Reject выполнение отклоняется, и рабочий процесс останавливается. Диалог становится доступным только для чтения, продолжать общение с AI-сотрудником невозможно.

![20260420181003](https://static-docs.nocobase.com/20260420181003.png)
