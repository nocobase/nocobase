# Узел согласования

## Уведомление

После срабатывания узла ИИ-сотрудника в рабочем процессе настроенные в узле согласующие лица увидят красную точку-индикатор у точки входа ИИ-сотрудников на главной странице.

![20260420171416](https://static-docs.nocobase.com/20260420171416.png)

В списке диалогов с ИИ-сотрудником можно увидеть задачи рабочего процесса, ожидающие согласования.

![20260420171642](https://static-docs.nocobase.com/20260420171642.png)

## Согласование

Нажмите на задачу рабочего процесса, чтобы открыть окно диалога, где отображается история сообщений ИИ-сотрудника. Последняя карточка, отправленная ИИ-сотрудником, и есть содержимое, требующее согласования в данном узле.

![20260420180710](https://static-docs.nocobase.com/20260420180710.png)

Для содержимого, ожидающего согласования, доступны три действия: `Одобрить`, `Доработать` и `Отклонить`.

При нажатии `Одобрить` содержимое, сгенерированное ИИ, одобряется, и рабочий процесс продолжается. Диалог становится доступным только для чтения, продолжать общение с ИИ-сотрудником невозможно.

![20260420180632](https://static-docs.nocobase.com/20260420180632.png)

При нажатии `Доработать` можно отправить ИИ замечания на правку. После отправки ИИ ответит новой карточкой согласования.

![20260420180908](https://static-docs.nocobase.com/20260420180908.png)

При нажатии `Отклонить` выполнение отклоняется, и рабочий процесс останавливается. Диалог становится доступным только для чтения, продолжать общение с ИИ-сотрудником невозможно.

![20260420181003](https://static-docs.nocobase.com/20260420181003.png)
