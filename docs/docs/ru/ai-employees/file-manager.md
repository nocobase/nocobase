# Управление файлами

## Введение

Здесь вы можете настроить способ хранения файлов, загружаемых во время диалогов с AI-сотрудниками.

## Настройки

Перейдите на страницу конфигурации плагина AI-сотрудников, затем нажмите вкладку `Settings`, чтобы открыть страницу настроек способа хранения.

![20251022002648](https://static-docs.nocobase.com/20251022002648.png)

Доступные варианты хранения для `Files storage` настраиваются в плагине управления файлами.

![20251022002853](https://static-docs.nocobase.com/20251022002853.png)
