# Управление файлами

## Введение

Настройте метод хранения файлов, которые загружаются во время диалогов с ИИ-сотрудником.

## Параметры

Перейдите на страницу настройки плагина ИИ-сотрудника, нажмите на вкладку Настройки, чтобы открыть страницу настроек метода хранения.

![20251022002648](https://static-docs.nocobase.com/20251022002648.png)

Доступные параметры хранилища файлов настраиваются в плагине Управления файлами.
