# Использование навыков

Навыки — это руководства по экспертным знаниям, предоставляемые ИИ-сотруднику, которые помогают ему использовать несколько инструментов для решения задач в специализированных областях.

В настоящее время навыки не поддерживают пользовательскую настройку и предустановлены системой.

## Структура навыков

Страница навыков делится на две категории:

1. `Общие навыки`: общие для всех ИИ-сотрудников, обычно доступны только для чтения.
2. `Навыки, специфичные для сотрудника`: индивидуальные для текущего сотрудника.

## Описание навыков

### Общие навыки

| Название навыка          | Описание функций                                                                                                                |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------- |
| Метаданные данных            | Получение метаданных модели данных системы, информации о коллекциях и полях, что помогает ИИ-сотруднику понять контекст бизнеса. |
| Запрос данных               | Запросы к данным коллекций с поддержкой фильтрации по условиям, агрегатных запросов и других функций, помогает ИИ-сотруднику получать бизнес-данные. |
| Бизнес-аналитический отчёт | Генерация аналитических отчётов на основе бизнес-данных с поддержкой многомерного анализа и визуализации, помогает ИИ-сотруднику получать бизнес-инсайты. |
| Поиск по документации          | Поиск и чтение предустановленного контента документации, помогает ИИ-сотруднику выполнять работу на основе документации, в настоящее время в основном для написания JS-кода. |

### Навыки, специфичные для сотрудника

| Название навыка    | Описание функций                                              | Принадлежащий сотрудник |
| ------------------ | -------------------------------------------------------------- | ----------------------- |
| Моделирование данных      | Навык моделирования данных, понимание и построение бизнес-моделей данных | Orin                    |
| Разработчик фронтенда | Написание и тестирование JS-кода для фронтенд-блоков          | Nathan                  |
