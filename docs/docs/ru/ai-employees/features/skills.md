# Использование навыков

Skills (навыки) — это руководства по экспертным знаниям, предоставляемые AI-сотруднику, которые помогают ему использовать несколько инструментов для решения задач в специализированных областях.

В настоящее время навыки не поддерживают пользовательскую настройку и предустановлены системой.

## Структура навыков

Страница навыков делится на две категории:

1. `General skills`: общие для всех AI-сотрудников, обычно доступны только для чтения.
2. `Employee-specific skills`: индивидуальные для текущего сотрудника.

## Описание навыков

### Общие навыки

| Название навыка          | Описание функций                                                                                                                |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------- |
| Data metadata            | Получение метаданных модели данных системы, информации о таблицах данных и полях, что помогает AI-сотруднику понять контекст бизнеса. |
| Data query               | Запросы к данным таблиц с поддержкой фильтрации по условиям, агрегатных запросов и других функций, помогает AI-сотруднику получать бизнес-данные. |
| Business analysis report | Генерация аналитических отчётов на основе бизнес-данных с поддержкой многомерного анализа и визуализации, помогает AI-сотруднику получать бизнес-инсайты. |
| Document search          | Поиск и чтение предустановленного контента документации, помогает AI-сотруднику выполнять работу на основе документации, в настоящее время в основном для написания JS-кода. |
