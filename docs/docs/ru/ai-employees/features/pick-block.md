# Добавление контекста — Блок

При общении с AI-сотрудником, помимо отправки файлов, вы также можете передавать контекстную информацию приложения. Это позволяет AI-сотруднику формировать ответы на основе предоставленного контекста.

### Отправка информации о блоке

В окне чата с AI-сотрудником нажмите кнопку `Добавить рабочий контекст` в левом нижнем углу и выберите `Выбрать блок`. Страница приложения перейдет в режим выбора блоков.

![20251022010148](https://static-docs.nocobase.com/20251022010148.png)

При наведении курсора на блок цвет блоков, которые можно отправить в качестве контекста, изменится.

![20251022010422](https://static-docs.nocobase.com/20251022010422.png)

После выбора нужного блока информация о его контексте будет прикреплена к окну чата.

![20251022010543](https://static-docs.nocobase.com/20251022010543.png)

Введите свой вопрос и, как при отправке обычного сообщения, нажмите кнопку отправки. AI-сотрудник ответит, основываясь на информации из выбранного блока.

Информация о блоке также появится в списке сообщений.

![20251022010758](https://static-docs.nocobase.com/20251022010758.png)

### Завершение выбора блока

Если страница приложения перешла в режим выбора блоков, но вы не хотите выбирать ни один из них, нажмите на иконку «Стоп» в нижней части экрана, чтобы выйти из режима выбора.

![20251022011043](https://static-docs.nocobase.com/20251022011043.png)
