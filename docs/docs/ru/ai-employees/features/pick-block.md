# Добавить контекст - блок

Во время общения с ИИ-сотрудником помимо отправки файлов вы также можете отправлять ИИ-сотруднику сведения о контексте приложения, чтобы ИИ-сотрудник отвечал, опираясь на него.

### Отправка информации о блоке

В окне чата ИИ-сотрудника нажмите кнопку `Добавить рабочий контекст` в левом нижнем углу и выберите `Выбрать блок`. Страница приложения перейдет в режим выбора блока.

![20251022010148](https://static-docs.nocobase.com/20251022010148.png)

Когда вы наводите курсор на блок, цвет блока, который можно отправить как контекст, изменится.

![20251022010422](https://static-docs.nocobase.com/20251022010422.png)

После выбора блока для отправки сведения контекста блока будут прикреплены к окну чата.

![20251022010543](https://static-docs.nocobase.com/20251022010543.png)

После ввода сообщения с вопросом, так же как при отправке сообщения, нажмите кнопку отправки — и ИИ-сотрудник ответит, опираясь на информацию о блоке.

Информация о блоке также отобразится в списке сообщений.

![20251022010758](https://static-docs.nocobase.com/20251022010758.png)

### Завершение выбора блока

Когда страница приложения перейдет в режим выбора блока, но вы не хотите выбирать ни один блок, нажмите на значок «Остановить» в нижней части, чтобы выйти из этого режима.

![20251022011043](https://static-docs.nocobase.com/20251022011043.png)
