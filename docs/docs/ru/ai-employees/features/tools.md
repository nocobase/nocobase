# Использование инструментов

Tools (инструменты) определяют, «что может делать» AI-сотрудник.

## Структура инструментов

Страница инструментов делится на три категории:

1. `General tools`: общие для всех AI-сотрудников, обычно доступны только для чтения.
2. `Employee-specific tools`: индивидуальные для текущего сотрудника.
3. `Custom tools`: пользовательские инструменты, создаваемые через триггер «AI employee event» в рабочем процессе. Можно добавлять, удалять и настраивать права по умолчанию.

![20260331182248](https://static-docs.nocobase.com/20260331182248.png)

## Права инструментов

Права инструментов унифицированы:

- `Ask`: перед вызовом запрашивается подтверждение.
- `Allow`: разрешён прямой вызов.

Рекомендация: для инструментов, изменяющих данные, по умолчанию используйте `Ask`.

![20260331182832](https://static-docs.nocobase.com/20260331182832.png)

## Описание инструментов

### Общие инструменты

| Название инструмента | Описание функций                                                              |
| -------------------- | ----------------------------------------------------------------------------- |
| Form filler          | Заполнение данных в указанную форму                                            |
| Chart generator      | Генерация JSON-конфигурации диаграммы ECharts                                  |
| Load specific SKILLS | Загрузка навыков и необходимых для них инструментов                             |
| Suggestions          | Предложение следующих действий на основе текущего диалога и контекста           |

### Специализированные инструменты

| Название инструмента         | Описание функций                                                          | Принадлежащий сотрудник |
| ---------------------------- | ------------------------------------------------------------------------- | ----------------------- |
| AI employee task dispatching | Инструмент диспетчеризации работы, распределяет задачи по типу задачи и возможностям сотрудника | Atlas                   |
| List AI employees            | Перечисление всех доступных сотрудников                                    | Atlas                   |
| Get AI employee              | Получение подробной информации о указанном сотруднике, включая навыки и инструменты | Atlas                   |

### Пользовательские инструменты

В модуле рабочих процессов создайте рабочий процесс с типом триггера `AI employee event`.

![20260331185556](https://static-docs.nocobase.com/20260331185556.png)

В разделе `Custom tools` нажмите `Add tool`, чтобы добавить рабочий процесс в качестве инструмента, и настройте права в соответствии с бизнес-рисками.

![20260331185711](https://static-docs.nocobase.com/20260331185711.png)
