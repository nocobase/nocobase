# Включение AI-сотрудников

NocoBase содержит встроенных AI-сотрудников для разных сценариев. Полное описание см. в [Встроенные AI-сотрудники](/ai-employees/built-in/).

## Как включить

Откройте страницу управления `AI employees` и включите переключатель `Enabled` прямо в списке.

![](https://static-docs.nocobase.com/202605121048555.png)

## Использование после включения

После включения сотрудник появится в доступных точках входа, например в быстром входе справа внизу или в специальном сценарии.
