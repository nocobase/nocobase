# Включение AI-сотрудников

NocoBase предлагает несколько встроенных AI-сотрудников для различных сценариев.

## Как включить

Перейдите на страницу управления `AI employees` и активируйте переключатель `Enabled` непосредственно в списке.

![ai-employee-list-enable-switch.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employee-list-enable-switch.png)

## Использование после включения

После включения сотрудник появится в доступных точках входа (например, в ярлыке быстрого доступа в правом нижнем углу или в точках входа для конкретных сценариев).
