# Включить ИИ-сотрудников

NocoBase предоставляет несколько встроенных ИИ-сотрудников для разных сценариев. Полное описание см. в [Встроенные ИИ-сотрудники](/ai-employees/built-in/).

## Как включить

Откройте страницу управления ИИ-сотрудниками и переключите `Включено` прямо в списке.

![ai-employee-list-enable-switch.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employee-list-enable-switch.png)

## После включения

После включения сотрудник появляется в доступных точках входа (например, во всплывающем входе или в сценарно-специфичном входе).
