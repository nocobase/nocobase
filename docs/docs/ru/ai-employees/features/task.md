# Быстрые задачи

Чтобы повысить эффективность работы AI-сотрудников, вы можете привязать их к блокам сценариев и заранее настроить несколько часто используемых задач.

Это позволяет пользователям запускать обработку задач одним кликом, избавляя от необходимости каждый раз **выбирать блок** и **вводить команды**.

## Привязка AI-сотрудника к блоку

После перехода в режим редактирования интерфейса (UI editing mode) для блоков, поддерживающих действия (`Actions`), выберите пункт `AI employees` в меню `Actions`, а затем выберите нужного AI-сотрудника. Этот сотрудник будет привязан к текущему блоку.

![20251022135306](https://static-docs.nocobase.com/20251022135306.png)

После завершения привязки при каждом входе на страницу в области действий (`Actions`) блока будет отображаться привязанный к нему AI-сотрудник.

![20251022135438](https://static-docs.nocobase.com/20251022135438.png)

## Настройка задач

В режиме редактирования интерфейса наведите курсор на иконку AI-сотрудника, привязанного к блоку. Появится кнопка меню; выберите `Edit tasks`, чтобы перейти на страницу настроек задач.

На странице настроек вы можете добавить несколько задач для текущего AI-сотрудника.

Каждая вкладка представляет собой отдельную задачу. Нажмите «+» рядом с вкладками, чтобы добавить новую задачу.

![clipboard-image-1771913187](https://static-docs.nocobase.com/clipboard-image-1771913187.png)

Форма настройки задачи:

- Введите заголовок задачи в поле `Title`. Это краткое описание задачи, которое будет отображаться в списке задач AI-сотрудника.
- Введите основное содержание задачи в поле `Background`. Этот текст будет использоваться в качестве системного промпта (system prompt) при диалоге с AI-сотрудником.
- Введите сообщение пользователя по умолчанию в поле `Default user message`. Оно будет автоматически подставляться в поле ввода после выбора задачи.
- В разделе `Work context` выберите контекст приложения, который будет отправляться AI-сотруднику по умолчанию. Это работает так же, как и в панели чата.
- В списке `Skills` отображаются навыки, доступные текущему AI-сотруднику. Вы можете отключить определенный навык, чтобы AI-сотрудник не использовал его при выполнении данной задачи.
- Флажок `Send default user message automatically` определяет, будет ли сообщение пользователя по умолчанию отправляться автоматически сразу после выбора задачи.


## Список задач

После настройки задач они будут отображаться во всплывающем окне профиля AI-сотрудника и в приветственном сообщении перед началом диалога. Нажмите на задачу, чтобы запустить её выполнение.
