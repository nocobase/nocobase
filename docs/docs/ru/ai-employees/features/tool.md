# Использование навыков

Навыки (инструменты) определяют, что «может делать» AI-сотрудник.

## Структура навыков

Страница навыков разделена на три категории:

1. `General skills` (Общие навыки): доступны всем AI-сотрудникам, только для чтения.
2. `Employee-specific skills` (Специфические навыки сотрудника): относятся только к текущему сотруднику, обычно только для чтения.
3. `Custom skills` (Пользовательские навыки): настраиваемые навыки, которые можно добавлять, удалять и для которых можно настраивать разрешения по умолчанию.

![skills-three-sections-general-specific-custom.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/skills-three-sections-general-specific-custom.png)

## Права доступа к навыкам

Права доступа к навыкам унифицированы:

- `Ask` (Спрашивать): запрашивать подтверждение перед вызовом.
- `Allow` (Разрешить): разрешить прямой вызов.

Рекомендация: используйте `Ask` по умолчанию для навыков, связанных с изменением данных.

![skills-permission-ask-allow-segmented.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/skills-permission-ask-allow-segmented.png)

## Добавление и обслуживание

Нажмите `Add skill` в разделе `Custom skills`, чтобы добавить навыки и настроить права доступа в соответствии с бизнес-рисками.
