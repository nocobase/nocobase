# Взаимодействие с AI-сотрудниками

После создания и включения AI-сотрудников вы сможете взаимодействовать с ними на страницах.

## Точки входа

Существует два основных способа доступа:

1. **Главный вход в правом нижнем углу**: открывает панель чата AI в правом нижнем углу рабочих страниц. Подходит для общих вопросов и ответов, а также для взаимодействия между блоками.
2. **Вход через действия блока**: в блоках, поддерживающих `Actions`, доступ осуществляется через `Actions -> AI employees`. Подходит для выполнения задач, связанных с текущим блоком (например, в сценариях с JSBlock).

## Основные операции в чате

Панель чата поддерживает стандартные операции, такие как отправка сообщений, загрузка вложений, просмотр истории диалогов, создание новых чатов и редактирование системных промптов.

## Переключение внутри чата

В нижней части области ввода можно напрямую переключать:

- `AI Employee Switcher`: переключение текущего сотрудника.
- `Model Switcher`: переключение модели текущей сессии.

Предпочтения по моделям сохраняются для каждого сотрудника и автоматически восстанавливаются при следующем входе.
