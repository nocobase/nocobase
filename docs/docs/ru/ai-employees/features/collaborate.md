# Совместная работа с AI-сотрудниками

После создания и включения AI-сотрудников вы можете работать с ними на страницах.

## Точки входа

1. **Основной вход в правом нижнем углу**: открывает AI-чат в правом нижнем углу бизнес-страниц. Подходит для общих вопросов и совместной работы между блоками.
2. **Вход через Action блока**: в блоках с поддержкой `Actions` добавьте AI-сотрудников через `Actions -> AI employees`. Подходит для задач в текущем блоке, например заполнения форм.
3. **Специальный вход**: сотрудники для разработки, такие как Nathan, Lina и Dara, предоставляют отдельные входы в специальных сценариях, например JS-блоках, блоках диаграмм или Localization Management.

## Базовые операции чата

Панель чата поддерживает обычные операции: отправку сообщений, загрузку вложений, просмотр истории, создание новых чатов и редактирование системного prompt.

## Переключение внутри диалога

В большинстве случаев можно просто общаться с Atlas; он подберет подходящего AI-сотрудника для помощи с задачей.

Чтобы использовать конкретного AI-сотрудника, нажмите выпадающий список AI-сотрудников в поле отправки и выберите его.

![20260331174320](https://static-docs.nocobase.com/20260331174320.png)

Предпочтения модели сохраняются для каждого сотрудника и восстанавливаются при следующем входе.
