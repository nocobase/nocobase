# Встроенные ИИ-сотрудники

NocoBase предоставляет несколько встроенных ИИ-сотрудников для конкретных сценариев.

Вам достаточно настроить службы языковой модели и включить нужных сотрудников; модели можно переключать в чате при необходимости.


## Введение

![clipboard-image-1766653060](https://static-docs.nocobase.com/clipboard-image-1766653060.png)

| Имя сотрудника | Позиционирование роли | Ключевые возможности |
| :--- | :--- | :--- |
| **Cole** | Ассистент NocoBase | Вопросы и ответы по использованию продукта, поиск документации |
| **Ellis** | Эксперт по email | Написание писем, генерация сводок, подсказки по ответам |
| **Dex** | Организатор данных | Перевод полей, форматирование, извлечение информации |
| **Viz** | Аналитик данных | Анализ по данным, анализ тенденций, интерпретация ключевых показателей |
| **Lexi** | Ассистент по переводу | Многоязычный перевод, помощь в коммуникации |
| **Vera** | Аналитик исследований | Веб-поиск, агрегация информации, углубленные исследования |
| **Dara** | Эксперт по визуализации данных | Настройка диаграмм, генерация визуальных отчетов |
| **Nathan** | Инженер фронтенда | Помощь в написании фрагментов кода для фронтенда, корректировка стилей |


Вы можете нажать на **плавающую кнопку ИИ** в правом нижнем углу интерфейса приложения, выбрать нужного сотрудника и начать сотрудничество.


## ИИ-сотрудники для конкретных сценариев

Некоторые встроенные ИИ-сотрудники не отображаются в списке ИИ-сотрудников в правом нижнем углу. У них есть эксклюзивные сценарии работы, например:

* Dara будет отображаться только на странице настройки диаграмм;
* Nathan будет отображаться только в JS-редакторе.


---

Ниже приведены несколько типовых сценариев применения ИИ-сотрудников — надеемся, что это даст вам вдохновение. Нам интересно, какие возможности вы обнаружите в реальном бизнесе.


## Viz: Аналитик данных

### Введение

> Генерируйте диаграммы и выводы одним кликом, позволяя данным говорить самим за себя.

**Viz** — встроенный **ИИ-аналитик данных**.
Он умеет читать данные с текущей страницы (например, Лиды), автоматически создавая диаграммы тенденций, диаграммы сравнения, карточки KPI и лаконичные выводы, чтобы бизнес-анализ был простым и интуитивным.

> Хотите узнать «Почему продажи недавно снизились»?
> Просто скажите Viz одну фразу — и он подскажет, какое звено просело, какие могут быть причины, и что делать дальше.

### Сценарии использования

Нужно ли вам разбирать ежемесячный склад/запасы, ROI по каналам или воронку продаж — вы можете поручить Viz выполнить анализ, сгенерировать диаграммы и интерпретировать результаты.

| Сценарий | Что вы хотите узнать | Результат Viz |
| :--- | :--- | :--- |
| **Ежемесячный обзор** | Что стало лучше в этом месяце по сравнению с прошлым? | Карточка KPI + диаграмма тенденций + три предложения по улучшению |
| **Разбор роста** | Рост выручки обусловлен объемом или ценой? | Диаграмма разложения факторов + таблица сравнения |
| **Анализ каналов** | Какой канал наиболее выгодно продолжать инвестировать? | Диаграмма ROI + кривая удержания + предложения |
| **Анализ воронки** | Где «застревает» трафик? | Диаграмма воронки + объяснение недочетов |
| **Удержание клиентов** | Какие клиенты наиболее ценны? | Диаграмма сегментации RFM + кривая удержания |
| **Оценка промо** | Каков был эффект крупной промоакции? | Диаграмма сравнения + анализ ценовой эластичности |

### Как использовать

**Вход на страницу**

* **Кнопка в правом верхнем углу (рекомендуется)**

  На страницах вроде Лиды и т.д. нажмите **иконку Viz** в правом верхнем углу, чтобы выбрать предустановленные задачи, например:

  * Конверсия по этапам и тенденции
  * Сравнение каналов-источников
  * Анализ ежемесячного обзора

  ![clipboard-image-1771913319](https://static-docs.nocobase.com/clipboard-image-1771913319.png)

* **Глобальная панель в правом нижнем углу**

  Вы можете вызвать глобальную панель ИИ на любой странице и напрямую обращаться к Viz:

  ```
  Проанализируй изменения в продажах за последние 90 дней
  ```

Viz автоматически подтянет контекст данных текущей страницы.

**Режим взаимодействия**

Viz поддерживает вопросы на естественном языке и также понимает продолжения в несколько раундов.
Пример:

```
Привет, Viz, построй тренд лидов за этот месяц.
```

```
Рассматривай только эффективность сторонних каналов.
```

```
Тогда в каком регионе самый быстрый рост?
```

Каждое последующее уточнение будет углубляться на основе предыдущих результатов анализа, без необходимости повторять условия данных.

### Советы по общению с Viz

| Практика | Эффект |
| :--- | :--- |
| Укажите период | «Последние 30 дней» и «Прошлый месяц vs этот месяц» дают более точный результат |
| Укажите измерение | «Показать по региону/каналу/продукту» помогает согласовать перспективы |
| Фокус на тенденциях, а не на деталях | Viz умеет находить направления изменений и ключевые причины |
| Используйте естественный язык | Не нужен императивный синтаксис — просто задавайте вопрос в стиле общения |

## Dex: Организатор данных

### Введение

> Быстро извлекайте и заполняйте формы, превращая «кашу» информации в структурированные данные.

`Dex` — эксперт по Организации данных. Он извлекает нужную информацию из неструктурированных данных или файлов в структурированный вид и может вызывать инструменты, чтобы заполнять информацию в формы.

### Как использовать

Вызовите `Dex` на странице формы, чтобы открыть окно чата.

![20251022155316](https://static-docs.nocobase.com/20251022155316.png)

Нажмите `Добавить рабочий контекст` в поле ввода и выберите `Выбрать блок`. Страница перейдет в режим выбора блока.

![20251022155414](https://static-docs.nocobase.com/20251022155414.png)

Выберите Блок формы на странице.

![20251022155523](https://static-docs.nocobase.com/20251022155523.png)

Введите данные, которые нужно организовать с помощью `Dex`, в поле чата.

![20251022155555](https://static-docs.nocobase.com/20251022155555.png)

После отправки `Dex` структурирует данные и использует навыки, чтобы обновить данные в выбранной форме.

![20251022155733](https://static-docs.nocobase.com/20251022155733.png)

## Nathan: Инженер фронтенда

### Введение

> Помогает вам писать и оптимизировать код фронтенда для реализации сложной логики взаимодействий.

`Nathan` — эксперт по разработке фронтенда в NocoBase. В сценариях, где требуется JavaScript, таких как `JS-блок`, `Поле JS`, `Элемент JS`, `Колонка JS `, `Действие JS`, `Поток событий`, `Правила связывания` и т.д., аватар `Nathan` появится в правом верхнем углу редактора кода. Вы можете попросить его помочь вам написать или изменить код в редакторе.

![20251022163405](https://static-docs.nocobase.com/20251022163405.png)

### Как использовать

В редакторе кода нажмите `Nathan`, чтобы открыть чат ИИ-сотрудника. Код в редакторе будет автоматически прикреплен к полю ввода как контекст приложения и отправлен `Nathan`.

![20251022163935](https://static-docs.nocobase.com/20251022163935.png)

Укажите свои потребности по коду, отправьте их `Nathan` и дождитесь ответа.

![20251022164041](https://static-docs.nocobase.com/20251022164041.png)

Нажмите кнопку `Применить в редактор` в блоке кода, на который ответил `Nathan`, чтобы перезаписать его код в редакторе.

![20251022164323](https://static-docs.nocobase.com/20251022164323.png)

Нажмите кнопку `Запустить` в редакторе кода, чтобы увидеть результат в реальном времени.

![20251022164436](https://static-docs.nocobase.com/20251022164436.png)

### История кода

Нажмите на значок «командная строка» в правом верхнем углу окна чата `Nathan`, чтобы просмотреть фрагменты кода, которые вы отправили, и фрагменты кода, на которые ответил `Nathan` в текущей сессии.

![20251022164644](https://static-docs.nocobase.com/20251022164644.png)

![20251022164713](https://static-docs.nocobase.com/20251022164713.png)
