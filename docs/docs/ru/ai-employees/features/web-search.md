# Веб-поиск

Веб-поиск дополняет знания модели актуальной внешней информацией.

## Как это работает

Доступность веб-поиска зависит от того, поддерживает ли выбранная в данный момент служба модели веб-поиск.

- Поддерживается: переключатель поиска виден и его можно включать/отключать.
- Не поддерживается: переключатель скрыт, а состояние поиска сбрасывается автоматически.

## Использование в чате

Используйте переключатель поиска в нижней панели отправителя:

- Включено: ИИ сначала выполняет поиск, затем формирует ответ.

![web-search-switch-visible-on-supported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-visible-on-supported-model.png)

- Выключено: ИИ формирует ответ только на основе уже имеющегося контекста.

![web-search-switch-hidden-on-unsupported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-hidden-on-unsupported-model.png)

## Отличия провайдеров

Возможности веб-поиска отличаются у разных моделей. Проверьте поведение, используя выбранную службу модели и выбранную модель.
