# Поиск в интернете

Поиск в интернете используется для дополнения данных обучения модели актуальной информацией из внешних источников.

## Как это работает

Доступность поиска в интернете зависит от того, поддерживает ли выбранный в текущем сеансе сервис модели функцию Web Search.

- **Поддерживается:** отображается переключатель поиска в интернете, который можно включить или выключить по необходимости.
- **Не поддерживается:** переключатель не отображается, а функция поиска автоматически отключается.

## Использование в чате

Используйте переключатель поиска в интернете в области ввода сообщения:

- **Включено:** ИИ может сначала выполнить поиск в сети, а затем подготовить ответ.

![web-search-switch-visible-on-supported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-visible-on-supported-model.png)

- **Выключено:** ИИ отвечает только на основе имеющегося контекста.

![web-search-switch-hidden-on-unsupported-model.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/web-search-switch-hidden-on-unsupported-model.png)

## Различия платформ

Возможности поиска в интернете различаются в зависимости от провайдера. Пожалуйста, используйте функцию с учетом особенностей выбранного вами сервиса и модели.
