# Настройка службы языковой модели

Перед использованием ИИ-сотрудников сначала настройте доступные службы языковой модели.

Поддерживаемые провайдеры включают OpenAI, Gemini, Claude, DeepSeek, Qwen, Kimi, GigaChat(требуется плагин) и локальные модели Ollama.

## Создать службу

Перейдите в `Настройки системы -> ИИ-сотрудники -> Служба языковой модели`.

1. Нажмите `Добавить`, чтобы открыть диалог создания.
2. Выберите `Провайдер`.
3. Заполните `Заголовок`, `Ключ API` и `Базовый URL` (необязательно).
4. Настройте `Включенные модели`:
   - `Рекомендуемые модели`: используйте официально рекомендованные модели.
   - `Выбрать модели`: выберите из списка моделей провайдера.
   - `Ручной ввод`: вручную введите ID модели и отображаемое имя.
5. Нажмите `Отправить`.

## Включить и упорядочить службы

В списке служб языковой модели вы можете:

- Переключать статус службы с помощью `Включено`.
- Перетаскивать, чтобы изменить порядок служб (это влияет на порядок отображения моделей).

![llm-service-list-enabled-and-sort.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/llm-service-list-enabled-and-sort.png)

## Тест доступности

Используйте `Тест запуска` внизу диалога службы, чтобы проверить доступность службы и моделей.

Рекомендуется запускать этот тест перед использованием в продакшене.
