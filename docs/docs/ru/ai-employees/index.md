# AI-сотрудники · Начало работы

AI-сотрудники в NocoBase — это не чат-боты и не изолированные агенты. Это интеллектуальные коллеги, которые бесшовно интегрированы в вашу бизнес-систему, понимают контекст вашей работы и напрямую выполняют задачи.

Они появляются там, где вам это нужно — рядом с таблицами и формами, чтобы вы могли легко их вызвать.

![20251102121036-2025-11-02-12-10-38](https://static-docs.nocobase.com/20251102121036-2025-11-02-12-10-38.png)

Вы также можете вызвать их из списка сотрудников в правом нижнем углу страницы.

![20251102121159-2025-11-02-12-12-01](https://static-docs.nocobase.com/20251102121159-2025-11-02-12-12-01.png)

Они автоматически получают данные в качестве контекста. Например, **Viz** рядом с блоком таблицы автоматически извлекает данные из таблицы и вызывает соответствующие инструменты для их обработки. Это означает, что вам больше не нужно копировать данные в чат-бот.

<video controls class="rounded shadow"><source src="https://static-docs.nocobase.com/z-2025-11-02-12.13.35-2025-11-02-12-13-46.mp4" type="video/mp4"></video>

Они также могут автоматически понимать структуру страницы как контекст. Например, **Dex** рядом с блоком формы автоматически получает структуру полей формы и вызывает нужные инструменты для работы со страницей. Это означает, что вам не нужно копировать данные обратно из чат-бота.

<video controls class="rounded shadow"><source src="https://static-docs.nocobase.com/z-2025-11-02-12.16.07-2025-11-02-12-16-21.mp4" type="video/mp4"></video>

Вы также можете напрямую выбирать элементы на странице и отправлять их AI-сотруднику, что позволит ему автоматически извлечь соответствующие данные и структуру.

<video controls class="rounded shadow"><source src="https://static-docs.nocobase.com/z-2025-11-02-12.17.25-2025-11-02-12-17-44.mp4" type="video/mp4"></video>

Вы можете заранее настроить задачи для каждого AI-сотрудника в текущем местоположении, чтобы пользователи могли запускать их одним кликом — даже не вводя ни слова.
