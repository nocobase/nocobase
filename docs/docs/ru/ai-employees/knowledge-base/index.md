# Обзор

## Введение

Плагин База знаний ИИ предоставляет возможности извлечения для Генерации с расширенными возможностями поиска (RAG) для ИИ-агентов.

Возможности Генерации с расширенными возможностями поиска (RAG) позволяют ИИ-агентам отвечать на вопросы пользователей более точно, профессионально и с учетом требований предприятия.

Использование профессиональных доменных и внутренних корпоративных документов из базы знаний, поддерживаемой администратором, повышает точность и прослеживаемость ответов ИИ-агентов.

### Что такое Генерация с расширенными возможностями поиска (RAG)

Генерация с расширенными возможностями поиска (RAG) расшифровывается как «Retrieval-Augmented-Generation».

- Извлечение (Retrieval): вопрос пользователя преобразуется в вектор моделью эмбеддингов (Embedding model, например, BERT). Затем из векторной библиотеки извлекаются наиболее релевантные фрагменты текста Top-K с помощью плотного извлечения (dense retrieval, семантическое сходство) или разреженного извлечения (sparse retrieval, совпадение ключевых слов).
- Дополнение (Augmentation): результаты извлечения объединяются с исходным вопросом, формируя дополненную подсказку (augmented prompt). Затем эта подсказка внедряется в контекстное окно языковой модели.
- Генерация (Generation): языковая модель объединяет дополненную подсказку, чтобы сгенерировать окончательный ответ, обеспечивая фактическую корректность и прослеживаемость.
