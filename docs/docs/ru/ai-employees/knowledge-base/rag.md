# RAG-поиск

## Введение

После настройки базы знаний вы сможете включить функцию RAG в настройках AI-сотрудника.

Когда RAG включен, при общении пользователя с AI-сотрудником, последний будет использовать RAG-поиск для извлечения документов из базы знаний на основе сообщения пользователя и отвечать, опираясь на найденные документы.

## Включение RAG

Перейдите на страницу настройки плагина AI-сотрудников, затем нажмите вкладку `AI employees`, чтобы попасть на страницу управления AI-сотрудниками.

![20251023010811](https://static-docs.nocobase.com/20251023010811.png)

Выберите AI-сотрудника, для которого вы хотите включить RAG, и нажмите кнопку `Edit`, чтобы перейти на страницу редактирования AI-сотрудника.

На вкладке `Knowledge base` включите переключатель `Enable`.

- В поле `Knowledge Base Prompt` введите подсказку для ссылки на базу знаний. `{knowledgeBaseData}` является фиксированным плейсхолдером и не должен изменяться.
- В разделе `Knowledge Base` выберите настроенную базу знаний. См.: [База знаний](/ai-employees/knowledge-base/knowledge-base).
- В поле `Top K` укажите количество документов для извлечения (по умолчанию — 3).
- В поле `Score` введите пороговое значение релевантности документов для поиска.

Нажмите кнопку `Submit`, чтобы сохранить настройки AI-сотрудника.
