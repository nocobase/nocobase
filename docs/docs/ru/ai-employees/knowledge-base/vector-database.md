# Векторная база данных

## Введение

В базе знаний векторная база данных хранит векторизованные документы базы знаний. Векторизованные документы выступают в качестве индекса для документов.

Когда включенf Генерациz с расширенными возможностями поиска (RAG) в диалоге с агентом ИИ, сообщение пользователя векторизуется, а фрагменты документов базы знаний извлекаются из векторной базы данных для сопоставления релевантных абзацев документов и исходного текста.

На данный момент плагин База знаний ИИ поддерживает из коробки только PGVector — плагин базы данных PostgreSQL.

## Управление векторной базой данных

Перейдите на страницу конфигурации плагина агента ИИ, нажмите вкладку `Векторное хранилище` и выберите `Векторная база данных`, чтобы открыть страницу управления векторной базой данных.

![20251022233704](https://static-docs.nocobase.com/20251022233704.png)

Нажмите кнопку `Добавить` в правом верхнем углу, чтобы добавить новое подключение `PGVector` к векторной базе данных:

- В поле ввода `Название` укажите имя подключения.
- В поле ввода `Хост` укажите IP-адрес векторной базы данных.
- В поле ввода `Порт` укажите номер порта векторной базы данных.
- В поле ввода `Имя пользователя` укажите имя пользователя векторной базы данных.
- В поле ввода `Пароль` укажите пароль векторной базы данных.
- В поле ввода `База данных` укажите имя базы данных.
- В поле ввода `Имя таблицы` укажите имя таблицы, которое используется при создании новой таблицы для хранения векторных данных.

После ввода всех необходимых сведений нажмите кнопку `Тест`, чтобы проверить доступность сервиса векторной базы данных, а затем нажмите кнопку `Отправить`, чтобы сохранить сведения о подключении.
