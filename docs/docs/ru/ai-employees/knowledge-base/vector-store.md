# Векторное хранилище

## Введение

В базе знаний при сохранении документов они векторизуются. При извлечении документов векторизуются поисковые термины. Для обоих процессов требуется `Модель эмбеддингов` для векторизации исходного текста.

В плагине База знаний ИИ векторное хранилище — это привязка `Модели эмбеддингов` и векторной базы данных.

## Управление векторным хранилищем

Перейдите на страницу настроек плагина ИИ-сотрудников, нажмите вкладку `Векторное хранилище` и выберите `Векторное хранилище`, чтобы открыть страницу управления векторным хранилищем.

![20251023003023](https://static-docs.nocobase.com/20251023003023.png)

Нажмите кнопку `Добавить` в правом верхнем углу, чтобы добавить новое векторное хранилище:

- В поле ввода `Название` укажите имя векторного хранилища;
- В `Векторное хранилище` выберите уже настроенную векторную базу данных. См.: [Векторная база данных](/ai-employees/knowledge-base/vector-database);
- В `Служба языковой модели` выберите уже настроенную службу языковой модели. См.: [Управление службой языковой модели](/ai-employees/features/llm-service);
- В поле ввода `Модель эмбеддингов` укажите название модели `Эмбеддинг`, которая будет использоваться;

Нажмите кнопку `Отправить`, чтобы сохранить сведения о векторном хранилище.
