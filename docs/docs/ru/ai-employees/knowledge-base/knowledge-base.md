# База знаний

## Введение
База знаний является основой Генерации с расширенными возможностями поиска (RAG). Она организует документы по категориям и строит индекс. Когда ИИ-сотрудник отвечает на вопрос, он в первую очередь ищет ответы в базе знаний.

## Управление базой знаний
Перейдите на страницу конфигурации плагина ИИ-сотрудников, нажмите вкладку `База знаний`, чтобы открыть страницу управления базой знаний.

![20251023095649](https://static-docs.nocobase.com/20251023095649.png)

Нажмите кнопку `Добавить` в правом верхнем углу, чтобы добавить базу знаний `Локальная`.

![20251023095826](https://static-docs.nocobase.com/20251023095826.png)

Введите необходимые сведения для новой базы знаний:

- В поле ввода `Название` укажите имя базы знаний;
- В `Хранилище файлов` выберите расположение хранилища файлов;
- В `Векторное хранилище` выберите векторное хранилище, см.: [Векторное хранилище](/ai-employees/knowledge-base/vector-store);
- В поле ввода `Описание` укажите описание базы знаний;

Нажмите кнопку `Отправить`, чтобы создать базу знаний.

![20251023095909](https://static-docs.nocobase.com/20251023095909.png)

## Управление документами базы знаний
После создания базы знаний на странице списка баз знаний нажмите на только что созданную базу, чтобы открыть страницу управления документами базы знаний.

![20251023100458](https://static-docs.nocobase.com/20251023100458.png)

![20251023100527](https://static-docs.nocobase.com/20251023100527.png)

Нажмите кнопку `Загрузить`, чтобы загрузить документы. После загрузки документов векторизация начнется автоматически. Дождитесь, когда `Статус` изменится с `Ожидание` на `Успешно`.

В настоящее время база знаний поддерживает следующие типы документов: txt, pdf, doc, docx, ppt, pptx; для pdf поддерживается только обычный текст.

![20251023100901](https://static-docs.nocobase.com/20251023100901.png)

## Типы баз знаний

### Локальная база знаний
Локальная база знаний это база знаний, хранящаяся локально в NocoBase. Документы и их векторные данные целиком хранятся локально силами NocoBase.

![20251023101620](https://static-docs.nocobase.com/20251023101620.png)

### База знаний только для чтения
Поддерживаются внешние документы и векторные данные. В NocoBase создается только подключение к векторной базе данных (в настоящее время поддерживается только PGVector).

![20251023101743](https://static-docs.nocobase.com/20251023101743.png)

### Внешняя база знаний
Извлечение из векторной базы данных должно быть реализовано разработчиками, что позволит использовать векторные базы данных, которые в данный момент не поддерживаются NocoBase.

![20251023101949](https://static-docs.nocobase.com/20251023101949.png)
