# Быстрый старт

За 5 минут завершите минимально достаточную настройку ИИ-сотрудника.

## Установка плагина

ИИ-сотрудники встроены в NocoBase (`@nocobase/plugin-ai`), поэтому отдельная установка не требуется.

## Настройка моделей

Службы языковой модели можно настроить в одном из двух мест:

1. В админ-разделе: `Настройки системы -> ИИ-сотрудники -> служба языковой модели`.
2. В фронтенде: на панели чата ИИ введите выбор модели с помощью Переключателя модели, затем нажмите на соответствующий ярлык, чтобы добавить службу языковой модели.

![quick-start-model-switcher-add-llm-service.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/quick-start-model-switcher-add-llm-service.png)

Обычно требуется указать:
1. Выберите Провайдера.
2. Укажите ключ API.
3. Настройте Включенные модели; обычно достаточно значения Рекомендованные модели по умолчанию.

## Включение встроенных сотрудников

Встроенные ИИ-сотрудники включены по умолчанию и обычно не требуется включать их по одному.

Если нужно настроить доступность (включить/отключить конкретного сотрудника), изспольуйте переключатель Включено в списке Настроек системы  -> ИИ-сотрудники.

![ai-employee-list-enable-switch.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employee-list-enable-switch.png)

## Начало работы

На странице приложения наведите курсор на элемент быстрого доступа в правом нижнем углу и выберите ИИ-сотрудника.
![ai-employees-entry-bottom-right.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employees-entry-bottom-right.png)

Нажмите, чтобы открыть диалог чата ИИ:

![chat-footer-employee-switcher-and-model-switcher.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/chat-footer-employee-switcher-and-model-switcher.png)

Также можно:
- Добавить блоки
- Добавить вложения
- Включить веб-поиск
- Переключить ИИ-сотрудников
- Выбрать модели

ИИ-сотрудники также могут автоматически получать структуру страницы в качестве контекста. Например, Dex на блоке формы может считывать структуры полей формы и вызывать подходящие навыки и инструменты, чтобы выполнять действия на странице.

## Быстрые задачи

Вы можете заранее задать быстрые задачи для каждого ИИ-сотрудника в текущем месте, чтобы работу можно было начать одним щелчком.

## Обзор встроенных сотрудников

NocoBase предоставляет несколько встроенных ИИ-сотрудников для разных сценариев.

Достаточно:

1. Настроить службы языковой модели.
2. При необходимости настроить статус включения сотрудников (по умолчанию включены).
3. Выбрать модель в чате и начать сотрудничество.

| Имя сотрудника | Позиционирование роли | Ключевые возможности |
| :--- | :--- | :--- |
| **Cole** | Ассистент NocoBase | Вопросы и ответы по использованию продукта, поиск в документации |
| **Ellis** | Эксперт по email | Написание писем, генерация кратких сводок, подсказки по ответам |
| **Dex** | Организатор данных | Перевод полей, форматирование, извлечение информации |
| **Viz** | Аналитик данных | Анализ данных, анализ тенденций, интерпретация ключевых показателей |
| **Lexi** | Ассистент по переводу | Многоязычный перевод, помощь в коммуникации |
| **Vera** | Аналитик исследований | Веб-поиск, агрегация информации, глубокие исследования |
| **Dara** | Эксперт по визуализации данных | Настройка диаграмм, генерация визуальных отчетов |
| **Orin** | Эксперт по моделированию данных | Помощь в проектировании структур таблиц данных, подсказки по полям |
| **Nathan** | Инженер фронтенда | Помощь в написании фрагментов кода для фронтенда, корректировка стилей |

**Примечания**

Некоторые встроенные ИИ-сотрудники не отображаются в списке в правом нижнем углу, потому что у них есть специализированные сценарии:

- Orin: страницы для моделирования данных.
- Dara: блоки для настройки диаграмм.
- Nathan: JS-блок и похожие редакторы кода.
