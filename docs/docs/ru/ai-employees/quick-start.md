# Быстрый старт

Давайте настроим минимальную рабочую конфигурацию AI-сотрудника за 5 минут.

## Установка плагина

AI-сотрудники являются встроенным плагином NocoBase (`@nocobase/plugin-ai`), поэтому отдельная установка не требуется.

## Настройка моделей

Вы можете настроить LLM-сервисы через любой из следующих входов:

1. Через панель управления: `Системные настройки -> AI-сотрудники -> LLM-сервис`.
2. Через быстрый доступ в интерфейсе: в панели AI-чата при выборе модели с помощью `Model Switcher` нажмите на ярлык «Добавить LLM-сервис» для прямого перехода к настройкам.

![quick-start-model-switcher-add-llm-service.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/quick-start-model-switcher-add-llm-service.png)

Обычно вам необходимо:
1. Выбрать провайдера (Provider).
2. Ввести API-ключ.
3. Настроить `Включенные модели` (Enabled Models); по умолчанию можно просто использовать вариант Recommend.

## Включение встроенных сотрудников

Встроенные AI-сотрудники по умолчанию включены все, поэтому обычно их не нужно активировать по отдельности.

Если вам нужно изменить область доступности (включить или выключить конкретного сотрудника), вы можете изменить переключатель `Enabled` на странице списка в разделе `Системные настройки -> AI-сотрудники`.

![ai-employee-list-enable-switch.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employee-list-enable-switch.png)

## Начало совместной работы

На странице приложения наведите курсор на ярлык быстрого доступа в правом нижнем углу и выберите AI-сотрудника.
![ai-employees-entry-bottom-right.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/ai-employees-entry-bottom-right.png)

Нажмите, чтобы открыть диалоговое окно AI-чата:

![chat-footer-employee-switcher-and-model-switcher.png](https://static-docs.nocobase.com/ai-employees/2026-02-14/chat-footer-employee-switcher-and-model-switcher.png)

Вы также можете:  
* Добавлять блоки
* Добавлять вложения
* Включать веб-поиск
* Переключать AI-сотрудников
* Выбирать модели

Они также могут автоматически получать структуру страницы в качестве контекста. Например, Dex в блоке формы автоматически считывает структуру полей формы и вызывает подходящие навыки для работы со страницей.

## Быстрые задачи 

Вы можете задать предустановленные часто используемые задачи для каждого AI-сотрудника в текущем расположении. Это позволит начинать работу одним кликом — быстро и удобно.

## Обзор встроенных сотрудников

В NocoBase предустановлено несколько AI-сотрудников, ориентированных на различные сценарии.

Вам нужно только:

1. Настроить LLM-сервис.
2. Отрегулировать статус активации сотрудников (по умолчанию включены).
3. Выбрать модель в чате и начать совместную работу.

| Имя сотрудника | Роль | Ключевые навыки |
| :--- | :--- | :--- |
| **Cole** | Помощник NocoBase | Ответы на вопросы по использованию продукта, поиск по документации |
| **Ellis** | Эксперт по электронной почте | Написание писем, создание резюме, предложения по ответам |
| **Dex** | Специалист по упорядочиванию данных | Перевод полей, форматирование, извлечение информации |
| **Viz** | Аналитик инсайтов | Анализ данных, анализ трендов, интерпретация ключевых показателей |
| **Lexi** | Ассистент-переводчик | Многоязычный перевод, помощь в общении |
| **Vera** | Исследователь-аналитик | Поиск в сети, агрегация информации, глубокие исследования |
| **Dara** | Эксперт по визуализации данных | Настройка диаграмм, создание визуальных отчетов |
| **Orin** | Эксперт по моделированию данных | Помощь в проектировании структуры коллекций, рекомендации по полям |
| **Nathan** | Фронтенд-инженер | Помощь в написании фрагментов фронтенд-кода, настройка стилей |

**Примечание**

Некоторые встроенные AI-сотрудники не отображаются в списке в правом нижнем углу, так как у них есть свои специфические рабочие сценарии:

- Orin: страницы моделирования данных.
- Dara: блоки настройки диаграмм.
- Nathan: JS-блоки и другие редакторы кода.
