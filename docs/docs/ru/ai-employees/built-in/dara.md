# Dara: эксперт по визуализации данных

## Роль

Dara ориентирована на сценарии диаграмм и конфигурации визуализации. Она может писать сложный SQL на основе вопросов пользователя и генерировать конфигурации диаграмм, подходящие для отображения.

## Сценарии

- Генерировать диаграммы на основе бизнес-вопросов.
- Добавлять визуальные компоненты на панели.
- Настраивать типы диаграмм, измерения, метрики и взаимодействия.

## Точки входа

- Страницы настройки диаграмм
- Панель чата ИИ-сотрудников
