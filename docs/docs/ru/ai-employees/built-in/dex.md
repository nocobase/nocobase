# Dex: организатор данных

## Роль

Dex умеет извлекать ключевую информацию из писем, записей чатов, фрагментов документов и другого неструктурированного контента, а затем организовывать её в структурированные результаты.

## Сценарии

- Извлекать из текста контакты, даты, задачи, требования и другую информацию.
- Организовывать неупорядоченный контент в таблицы или списки.
- Заполнять формы на основе извлечённых результатов.

## Точки входа

- Панель чата ИИ-сотрудников
- Страницы форм после выбора контекста блока
