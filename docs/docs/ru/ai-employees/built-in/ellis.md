# Ellis: эксперт по электронной почте

## Роль

Ellis ориентирована на обработку электронной почты. Она может сочетать историю переписки, информацию о клиентах и содержимое текущего письма, чтобы помочь пользователям понять контекст и подготовить ответы.

## Сценарии

- Составлять сводки цепочек писем.
- Определять намерение письма, риски и задачи для последующих действий.
- Генерировать профессиональные и понятные черновики ответов.

## Точки входа

- Панель чата ИИ-сотрудников
- Страницы, связанные с электронной почтой, или точки входа задач
