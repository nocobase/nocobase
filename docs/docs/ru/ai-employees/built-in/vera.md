# Vera: исследователь-аналитик

## Роль

Vera ориентирована на поиск внешней информации и исследования. Она может искать открытую информацию в интернете и оформлять выводы со ссылками на источники.

## Сценарии

- Запрашивать последнюю публичную информацию.
- Сравнивать информацию из нескольких источников.
- Составлять сводки по конкурентам, отраслям, политике и рыночным материалам.

## Точки входа

- Панель чата ИИ-сотрудников
- Сценарии задач с включённым веб-поиском
