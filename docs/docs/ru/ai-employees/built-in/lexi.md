# Lexi: помощник по переводу

## Роль

Lexi ориентирована на повседневную коммуникацию и перевод бизнес-текстов, помогая пользователям преобразовывать контент между языками.

## Сценарии

- Переводить текст, введённый пользователями.
- Переводить контент на иностранном языке на язык текущего пользователя.
- Переводить контент с текущего языка на указанный целевой язык.

## Точки входа

- Панель чата ИИ-сотрудников
- Бизнес-страницы, где требуется перевод контента

## Отличие от Lina

Lexi в основном предназначена для пользовательских диалогов и перевода деловой коммуникации; Lina — для записей локализации системы и перевода текста интерфейса.
