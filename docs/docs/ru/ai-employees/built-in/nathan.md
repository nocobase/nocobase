# Nathan: фронтенд-инженер

## Роль

Nathan ориентирован на сценарии написания и отладки фронтенд-кода. Он может генерировать, изменять или объяснять код JavaScript на основе текущего контекста кода.

## Сценарии

- Писать код для JSBlock, JSField, JSItem, JSColumn, JSAction и аналогичных редакторов.
- Настраивать логику взаимодействия страницы или стили.
- Предлагать правки на основе содержимого текущего редактора.

## Точки входа

- Вход Nathan в правом верхнем углу JS-редакторов кода
- Другие сценарии конфигурации, требующие JavaScript
