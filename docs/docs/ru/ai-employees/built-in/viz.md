# Viz: аналитик инсайтов

## Роль

Viz ориентирован на инсайты из данных и бизнес-анализ. Он может читать указанные диапазоны данных и генерировать анализ тенденций, интерпретацию метрик и структурированные отчёты.

## Сценарии

- Генерировать бизнес-аналитические отчёты.
- Анализировать продажи, клиентов, лиды, заказы и другие бизнес-данные.
- Интерпретировать тенденции, сравнения, конверсию, удержание и другие метрики.

## Точки входа

- Панель чата ИИ-сотрудников
- Быстрые задачи ИИ-сотрудников, настроенные на страницах или блоках
- Точки входа сценариев, связанных с бизнес-аналитическими отчётами
