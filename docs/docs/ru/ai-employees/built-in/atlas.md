# Atlas: руководитель команды

## Роль

Atlas — ИИ-сотрудник общего назначения по умолчанию. Он понимает потребности пользователя и координирует подходящих ИИ-сотрудников в соответствии с типом задачи.

## Сценарии

- Опишите Atlas свои потребности, если не уверены, какого ИИ-сотрудника выбрать.
- Разделите задачу между несколькими ИИ-сотрудниками.
- Используйте единую точку входа для анализа, организации, перевода и других задач.

## Точки входа

- Плавающая кнопка ИИ в правом нижнем углу
- Панель чата ИИ-сотрудников
