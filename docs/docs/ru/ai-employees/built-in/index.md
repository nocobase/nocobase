# Встроенные ИИ-сотрудники

NocoBase предоставляет набор встроенных ИИ-сотрудников для разных сценариев. Они помогают с анализом данных, организацией данных, обработкой электронной почты, переводом, локализацией, фронтенд-разработкой и многим другим.

![](https://static-docs.nocobase.com/202605121048555.png)

## Список сотрудников

| Сотрудник | Роль | Типичный сценарий | Тип |
| --- | --- | --- | --- |
| [Atlas](./atlas.md) | Руководитель команды | Понимание намерений пользователя и координация подходящих ИИ-сотрудников | Главный агент |
| [Viz](./viz.md) | Аналитик инсайтов | Запрос бизнес-данных и генерация аналитических выводов или отчётов | Бизнес-сотрудник |
| [Dex](./dex.md) | Организатор данных | Извлечение структурированной информации из текста и помощь в заполнении форм | Бизнес-сотрудник |
| [Ellis](./ellis.md) | Эксперт по электронной почте | Сводка контекста писем и подготовка ответов | Бизнес-сотрудник |
| [Lexi](./lexi.md) | Помощник по переводу | Диалоговый многоязычный перевод | Бизнес-сотрудник |
| [Vera](./vera.md) | Исследователь-аналитик | Веб-поиск, проверка информации и составление исследовательских сводок | Бизнес-сотрудник |
| [Nathan](./nathan.md) | Фронтенд-инженер | Помощь в написании или изменении фронтенд-кода в редакторах кода | Сотрудник-разработчик |
| [Lina](./lina.md) | Инженер локализации | Перевод записей локализации системы и текста интерфейса плагинов | Сотрудник-разработчик |
| [Dara](./dara.md) | Эксперт по визуализации данных | Генерация диаграмм и конфигураций визуализации на основе данных | Сотрудник-разработчик |
