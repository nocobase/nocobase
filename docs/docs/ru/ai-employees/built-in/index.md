# Встроенные AI-сотрудники

## Обзор

NocoBase включает в себя следующих встроенных AI-сотрудников. Они обладают полным набором навыков, инструментов и баз знаний. Вам нужно лишь настроить для них LLM, чтобы они могли приступить к работе.

- `Orin`: эксперт по моделированию данных
- `Avery`: заполнитель форм
- `Viz`: аналитик инсайтов
- `Lexi`: переводчик
- `Nathan`: инженер по фронтенд-коду
- `Cole`: помощник NocoBase
- `Vera`: аналитик-исследователь
- `Dex`: организатор данных
- `Ellis`: эксперт по электронной почте

## Как включить

Перейдите на страницу конфигурации плагина AI-сотрудников, нажмите вкладку `AI employees`, чтобы попасть на страницу управления AI-сотрудниками.

Вы увидите, что в системе уже есть несколько встроенных AI-сотрудников, но ни один из них не активирован. Пока вы не сможете взаимодействовать с ними на странице приложения.

![20251022121248](https://static-docs.nocobase.com/20251022121248.png)

Выберите встроенного AI-сотрудника, которого хотите включить, нажмите кнопку `Edit`, чтобы перейти на страницу редактирования AI-сотрудника.

Сначала на вкладке `Profile` включите переключатель `Enable`.

![20251022121546](https://static-docs.nocobase.com/20251022121546.png)

Затем на вкладке `Model settings` настройте модель для встроенного AI-сотрудника:

- Выберите LLM-сервис, который вы создали в разделе управления LLM-сервисами;
- Введите название большой модели, которую вы хотите использовать.

![20251022121729](https://static-docs.nocobase.com/20251022121729.png)

### Завершение активации

После настройки модели для встроенного AI-сотрудника нажмите кнопку `Submit`, чтобы сохранить изменения.

Затем вы сможете увидеть этого встроенного AI-сотрудника в кнопке быстрого запуска AI-сотрудников в правом нижнем углу страницы.

![20251022121951](https://static-docs.nocobase.com/20251022121951.png)

### Примечание

Некоторые встроенные AI-сотрудники после активации не будут отображаться в списке AI-сотрудников в правом нижнем углу. Например, Orin появится только на странице конфигурации основных данных; Nathan — только в JS-редакторе.
