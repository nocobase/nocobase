# AI Агент · Dex: Специалист по организации данных

> Быстро извлекайте данные и заполняйте формы, превращая неструктурированную информацию в структурированную.

`Dex` — это специалист по организации данных. Он извлекает необходимую информацию из неструктурированных данных или файлов, организует ее в структурированный вид и может использовать инструменты для заполнения форм.

![20251022155210](https://static-docs.nocobase.com/20251022155210.png)

## Шаги

На странице формы вызовите `Dex`, чтобы открыть диалоговое окно.

![20251022155316](https://static-docs.nocobase.com/20251022155316.png)

Нажмите `Add work context` в поле ввода и выберите `Pick block`. Страница перейдет в режим выбора блока.

![20251022155414](https://static-docs.nocobase.com/20251022155414.png)

Выберите блок формы на странице.

![20251022155523](https://static-docs.nocobase.com/20251022155523.png)

В диалоговом окне введите данные, которые вы хотите, чтобы `Dex` организовал.

![20251022155555](https://static-docs.nocobase.com/20251022155555.png)

После отправки `Dex` структурирует данные и использует свои навыки для обновления данных в выбранной форме.

![20251022155733](https://static-docs.nocobase.com/20251022155733.png)
