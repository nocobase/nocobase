# Быстрый старт

## Введение
После завершения настройки сервиса LLM вы можете приступить к созданию AI-сотрудников.

## Новый AI-сотрудник

### Настройка основной информации

Перейдите на страницу настройки **плагина** AI-сотрудников, затем нажмите вкладку `AI employees`, чтобы попасть на страницу управления AI-сотрудниками.

![20251021225643](https://static-docs.nocobase.com/20251021225643.png)

Нажмите кнопку `New AI employee`, чтобы перейти на страницу создания AI-сотрудника.

Сначала на вкладке `Profile` вы можете настроить основную информацию об AI-сотруднике:

- В поле ввода `Username` введите уникальный идентификатор AI-сотрудника;
- В поле ввода `Nickname` введите псевдоним AI-сотрудника;
- В поле ввода `Position` введите должность AI-сотрудника, чтобы кратко описать его обязанности;
- В разделе `Avatar` выберите аватар для AI-сотрудника.

![20251021232246](https://static-docs.nocobase.com/20251021232246.png)

- В поле ввода `Bio` введите вводный текст для AI-сотрудника, описывающий его навыки и функции. Этот текст будет отображаться в профиле AI-сотрудника;
- В поле ввода `About me` введите подробное описание функций AI-сотрудника. Этот текст будет использоваться в качестве системного запроса при общении с AI-сотрудником.
- В поле ввода `Greeting message` введите приветственное сообщение для AI-сотрудника. Всякий раз, когда пользователь начинает новый разговор с AI-сотрудником, система автоматически отправит это сообщение пользователю.

![20251021232712](https://static-docs.nocobase.com/20251021232712.png)

### Настройка большой модели

На вкладке `Model settings` вы можете настроить параметры модели для AI-сотрудника:

- Выберите сервис LLM, который вы создали в разделе управления сервисами LLM;
- Введите название большой модели, которую вы хотите использовать.

![20251021231727](https://static-docs.nocobase.com/20251021231727.png)

### Завершение создания

После ввода всей необходимой информации для создания AI-сотрудника нажмите кнопку `Submit`, чтобы завершить процесс.

Затем вы сможете увидеть только что созданного AI-сотрудника в списке AI-сотрудников. Кроме того, он будет отображаться в кнопке быстрого вызова AI-сотрудников в правом нижнем углу страницы.

![20251021232836](https://static-docs.nocobase.com/20251021232836.png)
