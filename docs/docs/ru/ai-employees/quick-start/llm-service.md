# Быстрый старт

## Введение

Прежде чем начать использовать AI-сотрудника, вам потребуется подключиться к онлайн-сервису LLM. NocoBase на данный момент поддерживает основные онлайн-сервисы LLM, такие как OpenAI, Gemini, Claude, DepSeek, Qwen и другие.
Помимо онлайн-сервисов LLM, NocoBase также поддерживает подключение к локальным моделям Ollama.

## Настройка сервиса LLM

Перейдите на страницу конфигурации плагина AI-сотрудника, нажмите вкладку `LLM service`, чтобы открыть страницу управления сервисами LLM.

![20251021213122](https://static-docs.nocobase.com/20251021213122.png)

Наведите курсор на кнопку `Add New` в правом верхнем углу списка сервисов LLM и выберите сервис LLM, который вы хотите использовать.

![20251021213358](https://static-docs.nocobase.com/20251021213358.png)

В качестве примера возьмем OpenAI: во всплывающем окне введите легко запоминающийся `title`, затем введите `API key`, полученный от OpenAI, и нажмите `Submit` для сохранения. Таким образом вы завершите настройку сервиса LLM.

Поле `Base URL` обычно можно оставить пустым. Если вы используете сторонний сервис LLM, совместимый с API OpenAI, пожалуйста, укажите соответствующий Base URL.

![20251021214549](https://static-docs.nocobase.com/20251021214549.png)

## Проверка доступности

На странице конфигурации сервиса LLM нажмите кнопку `Test flight`, введите название модели, которую вы хотите использовать, и затем нажмите кнопку `Run`, чтобы проверить доступность сервиса LLM и модели.

![20251021214903](https://static-docs.nocobase.com/20251021214903.png)
