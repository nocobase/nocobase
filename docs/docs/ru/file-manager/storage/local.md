# Движок хранения: локальное хранилище

Загруженные файлы будут сохранены в локальном каталоге на жёстком диске сервера. Это подходит для сценариев с небольшим суммарным объёмом загруженных файлов, которыми управляет система, или для экспериментальных целей.

### Путь

Задаёт одновременно относительный путь хранения файлов на сервере и URL-путь доступа. Например, "`user/avatar`" (без начальных и конечных слэшей) означает:

1. Относительный путь на сервере, где будут храниться загруженные файлы: `/path/to/nocobase-app/storage/uploads/user/avatar`.
2. URL-префикс для доступа к файлам: `http://localhost:13000/storage/uploads/user/avatar`.
