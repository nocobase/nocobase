### Регион

Укажите регион хранения S3, например: `us-west-1`.

:::info{title=Примечание}
Вы можете просмотреть информацию о регионе вашего хранилища в [консоли Amazon S3](https://console.aws.amazon.com/s3/). При этом достаточно указать только префикс региона (полное доменное имя не требуется).
:::

### AccessKey ID

Укажите ID ключа доступа Amazon S3.

### AccessKey Secret

Укажите Secret ключа доступа Amazon S3.

### Бакет

Укажите имя бакета S3.
