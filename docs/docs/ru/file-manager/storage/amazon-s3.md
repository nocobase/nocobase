# Движок хранения: Amazon S3

Движок хранения на базе Amazon S3. Перед использованием необходимо подготовить соответствующую учётную запись и права доступа.

## Параметры конфигурации

![Amazon S3 Storage Engine привер конфигурации](https://static-docs.nocobase.com/20251031092524.png)

:::info{title=Примечание}
Этот раздел описывает только специфические параметры движка хранения Amazon S3. Общие параметры см. в разделе [Общие параметры движков](./index).
:::

### Регион

Введите регион хранилища S3, например: `us-west-1`.

:::info{title=Примечание}
Вы можете посмотреть информацию о регионе для вашего бакета в [консоли Amazon S3](https://console.aws.amazon.com/s3/). Используйте только префикс региона (не полное доменное имя).
:::

### ID ключа доступа

Введите AccessKey ID Amazon S3.

### Секрет ключа доступа

Введите AccessKey Secret Amazon S3.

### Бакет

Введите имя бакета S3.
