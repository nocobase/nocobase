# Механизм хранения: Aliyun OSS

Это механизм хранения, основанный на Aliyun OSS. Перед использованием необходимо подготовить соответствующую учетную запись и предоставить необходимые разрешения.

## Параметры конфигурации

![Пример конфигурации механизма хранения Aliyun OSS](https://static-docs.nocobase.com/20240712220011.png)

:::info{title=Примечание}
Здесь описаны только специфические параметры механизма хранения Aliyun OSS. Общие параметры вы можете найти в разделе [Общие параметры механизма](./index#引擎通用参数).
:::

### Регион

Укажите регион для хранения OSS, например: `oss-cn-hangzhou`.

:::info{title=Примечание}
Информацию о регионе вашего хранилища вы можете посмотреть в [консоли Aliyun OSS](https://oss.console.aliyun.com/). Вам нужно использовать только префикс региона, без полного доменного имени.
:::

### AccessKey ID

Укажите ID ключа доступа Aliyun.

### AccessKey Secret

Укажите Secret ключа доступа Aliyun.

### Корзина

Укажите имя корзины OSS.

### Время ожидания

Укажите время ожидания для загрузки в Aliyun OSS в миллисекундах. По умолчанию установлено значение `60000` миллисекунд (то есть 60 секунд).
