# Tencent Cloud COS

Механизм хранения, основанный на Tencent Cloud COS. Перед использованием вам необходимо подготовить соответствующую учетную запись и разрешения.

## Параметры конфигурации

![Пример настройки механизма хранения Tencent COS](https://static-docs.nocobase.com/20240712222125.png)

:::info{title=Примечание}
Здесь описаны только специфические параметры для механизма хранения Tencent Cloud COS. Общие параметры вы можете найти в разделе [Общие параметры механизма](./index.md#general-engine-parameters).
:::

### Регион

Укажите регион для хранения COS, например: `ap-chengdu`.

:::info{title=Примечание}
Вы можете просмотреть информацию о регионе вашего хранилища (бакета) в [консоли Tencent Cloud COS](https://console.cloud.tencent.com/cos). Вам нужно использовать только префикс региона (без полного доменного имени).
:::

### SecretId

Укажите ID ключа доступа Tencent Cloud.

### SecretKey

Укажите Secret ключа доступа Tencent Cloud.

### Бакет

Укажите имя бакета для хранения COS, например: `qing-cdn-1234189398`.
