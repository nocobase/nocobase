# Движок хранения: Tencent Cloud COS

Движок хранения на базе Tencent Cloud COS. Перед использованием необходимо подготовить соответствующую учётную запись и права доступа.

## Параметры конфигурации

![Tencent COS Storage Engine пример конфигурации](https://static-docs.nocobase.com/20240712222125.png)

:::info{title=Примечание}
Этот раздел описывает только специфические параметры движка хранения Tencent Cloud COS. Общие параметры см. в разделе [Общие параметры движков](./index.md#general-engine-parameters).
:::

### Регион

Введите регион COS-хранилища, например: `ap-chengdu`.

::::info{title=Note}
Информацию о регионе вашего бакета можно посмотреть в [консоли Tencent Cloud COS](https://console.cloud.tencent.com/cos). Используйте только префикс региона (не полное доменное имя).
::::

### Идентификатор секрета

Введите ID вашего ключа доступа Tencent Cloud.

### Секретный ключ

Введите secret вашего ключа доступа Tencent Cloud.

### Бакет

Введите имя бакета COS, например: `qing-cdn-1234189398`.
