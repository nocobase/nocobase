# Поле "Вложение"

## Введение

В системе предусмотрен встроенный тип поля "Вложение", который позволяет пользователям загружать файлы в пользовательские коллекции.

Поле "Вложение" по своей сути является полем связи "многие ко многим", которое ссылается на встроенную системную коллекцию "Вложения" (`attachments`). При создании поля "Вложение" в любой коллекции автоматически генерируется промежуточная таблица для связи "многие ко многим". Метаданные загруженных файлов хранятся в коллекции "Вложения", а информация о файлах, на которые ссылаются в коллекции, связывается через эту промежуточную таблицу.

### Ограничения по типу MIME

Используется для ограничения типов файлов, разрешенных к загрузке, с использованием синтаксиса [MIME](https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/MIME_types). Например: `image/*` обозначает файлы изображений. Несколько типов можно разделить запятой, например: `image/*,application/pdf` разрешает загрузку файлов изображений и PDF-файлов.

### Движок хранения

Выберите движок хранения для загружаемых файлов. Если поле оставить пустым, будет использоваться движок хранения по умолчанию.
