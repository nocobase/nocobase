# Вложение

## Введение

В системе есть встроенный тип поля «Вложение» для поддержки загрузки файлов в пользовательских коллекциях.

Поле вложения по сути является полем связи многие-ко-многим, которое указывает на встроенную коллекцию «Вложения» (`attachments`). Когда поле вложения создаётся в любой коллекции, автоматически создаётся таблица связей для отношения многие-ко-многим. Метаданные загруженных файлов хранятся в коллекции «Вложения», а информация о файлах, на которые ссылается коллекция, связывается через эту таблицу связей.

### Ограничения по MIME-типам

Используется для ограничения типов файлов, разрешённых к загрузке, в формате синтаксиса [MIME](https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/MIME_types). Например, `image/*` означает файлы изображений. Несколько типов можно разделять запятой, например: `image/*,application/pdf` разрешает и изображения, и PDF.

### Движок хранения

Выберите движок хранения для сохранения загруженных файлов. Если оставить пустым, будет использован движок хранения по умолчанию в системе.
