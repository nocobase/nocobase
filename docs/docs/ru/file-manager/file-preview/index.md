# Предварительный просмотр файлов

В интерфейсах, содержащих поля файлов (включая поля вложений), вы можете просматривать файлы, нажимая на миниатюру или иконку файла. Встроенная функция предварительного просмотра поддерживает различные типы файлов, включая изображения, PDF и большинство типов файлов, поддерживаемых браузерами нативно.

![20251129232307](https://static-docs.nocobase.com/20251129232307.png)

Для типов файлов, которые не поддерживают нативный просмотр, вы можете включить функцию предварительного просмотра, установив или расширив соответствующие плагины для просмотра файлов. Например, после установки плагина для предварительного просмотра файлов Office вы сможете просматривать файлы Word, Excel и PowerPoint.
