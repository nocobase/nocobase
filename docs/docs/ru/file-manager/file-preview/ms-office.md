# Предпросмотр Office-файлов <Badge>v1.8.11+</Badge>

Плагин предпросмотра Office-файлов позволяет пользователям просматривать файлы форматов Office (например, Word, Excel и PowerPoint) прямо внутри приложения NocoBase.  
Он использует публичный онлайн-сервис Microsoft для встраивания содержимого файла в интерфейс предпросмотра, позволяя просматривать документы в браузере без скачивания и без использования приложений Office.

## Руководство пользователя

По умолчанию этот плагин **выключен**. Его можно включить в менеджере плагинов; дополнительная конфигурация не требуется.

![Активация плагина](https://static-docs.nocobase.com/20250731140048.png)

После успешной загрузки Office-файла (Word / Excel / PowerPoint) в файловое поле таблицы данных вы можете нажать на значок файла или ссылку, чтобы просмотреть его содержимое во всплывающем окне или встроенном интерфейсе.

![Пример](https://static-docs.nocobase.com/20250731143231.png)

## Как это работает

Этот плагин использует публичный онлайн-сервис Microsoft (Office Web Viewer) для рендеринга предпросмотра. Базовый рабочий процесс:

- фронтенд формирует публично доступный URL для загруженного файла (включая подписанные URL, например, из плагина S3-Pro);
- плагин встраивает предпросмотр в iframe в следующем формате:

  ```
  https://view.officeapps.live.com/op/embed.aspx?src=<public file URL>
  ```

- сервис Microsoft получает файл по URL, рендерит его и возвращает страницу, пригодную для предпросмотра.

## Примечания

- Поскольку плагин зависит от онлайн-сервиса Microsoft, убедитесь, что сеть работает стабильно и сервисы Microsoft доступны.
- Microsoft будет обращаться к URL файла, который вы предоставляете, и содержимое файла может временно кэшироваться на их серверах для рендеринга. Это создаёт потенциальные риски приватности — если это критично, рекомендуется не использовать этот плагин для чувствительных документов[^1].
- Чтобы предпросмотр работал, файлы должны быть доступны по публичному URL. Обычно для файлов, загруженных в NocoBase, автоматически генерируются публичные ссылки (включая подписанные URL из плагина S3-Pro). Однако если права доступа ограничены или файл хранится во внутренней сети, предпросмотр работать не будет[^2].
- Этот сервис не поддерживает аутентификацию или приватное хранилище. Например, файлы, доступ к которым требует входа в систему или возможен только из внутренней сети, не будут работать с предпросмотром.
- После того как Microsoft получит файл, он может оставаться в кэше некоторое время. Даже если исходный файл удалён, предпросмотр может быть доступен ещё какое-то время.
- Рекомендуемые ограничения размера для стабильного предпросмотра: файлы Word и PowerPoint — не более 10MB, файлы Excel — не более 5MB[^3].
- Официального публичного заявления о коммерческом использовании этого сервиса нет. Оцените риски перед использованием в коммерческих приложениях[^4].

## Поддерживаемые типы файлов

Плагин поддерживает предпросмотр только для следующих форматов Office, на основе MIME-типа или расширения файла:

- Документ Word:
  `application/vnd.openxmlformats-officedocument.wordprocessingml.document` (`.docx`) или `application/msword` (`.doc`)
- Таблица Excel:
  `application/vnd.openxmlformats-officedocument.spreadsheetml.sheet` (`.xlsx`) или `application/vnd.ms-excel` (`.xls`)
- Презентация PowerPoint:
  `application/vnd.openxmlformats-officedocument.presentationml.presentation` (`.pptx`) или `application/vnd.ms-powerpoint` (`.ppt`)
- Текст OpenDocument:
  `application/vnd.oasis.opendocument.text` (`.odt`)

Другие форматы файлов этот плагин не будет предварительно просматривать.
