# Предварительный просмотр файлов Office <Badge>v1.8.11+</Badge>

Плагин "Предварительный просмотр файлов Office" предназначен для просмотра файлов форматов Office (таких как Word, Excel, PowerPoint и др.) непосредственно в приложениях NocoBase.  
Он основан на общедоступном онлайн-сервисе Microsoft, который позволяет встраивать файлы, доступные по публичному URL, в интерфейс просмотра. Пользователи могут просматривать эти файлы в браузере без необходимости скачивания или использования приложений Office.

## Руководство пользователя

По умолчанию этот плагин находится в состоянии **отключен**. Его можно использовать после активации в менеджере плагинов, дополнительная настройка не требуется.

![Интерфейс активации плагина](https://static-docs.nocobase.com/20250731140048.png)

После успешной загрузки файла Office (Word / Excel / PowerPoint) в поле файла коллекции, нажмите на соответствующий значок файла или ссылку, чтобы просмотреть содержимое во всплывающем или встроенном интерфейсе предварительного просмотра.

![Пример операции предварительного просмотра](https://static-docs.nocobase.com/20250731143231.png)

## Принцип реализации

Предварительный просмотр, встроенный в этот плагин, опирается на публичный онлайн-сервис Microsoft (Office Web Viewer). Основной процесс выглядит следующим образом:

- Фронтенд генерирует публично доступный URL для файла, загруженного пользователем (включая подписанные URL-адреса S3);
- Плагин загружает предварительный просмотр файла в iframe, используя следующий адрес:

  ```
  https://view.officeapps.live.com/op/embed.aspx?src=<Публичный URL файла>
  ```

- Сервис Microsoft запрашивает содержимое файла по этому URL, выполняет рендеринг и возвращает страницу для просмотра.

## Примечания

- Поскольку этот плагин зависит от онлайн-сервиса Microsoft, убедитесь, что сетевое соединение стабильно и есть доступ к соответствующим сервисам Microsoft.
- Microsoft получит доступ к предоставленному вами URL-адресу файла, и содержимое файла будет кратковременно кэшировано на их сервере для рендеринга страницы просмотра. Это влечет за собой определенные риски конфиденциальности. Если у вас есть опасения по этому поводу, рекомендуем не использовать функцию предварительного просмотра, предоставляемую этим плагином[^1].
- Файл для предварительного просмотра должен быть доступен по публичному URL. Обычно файлы, загруженные в NocoBase, автоматически генерируют доступные публичные ссылки (включая подписанные URL, созданные плагином S3-Pro), но если для файла установлены права доступа или он хранится во внутренней сети, предварительный просмотр будет невозможен[^2].
- Сервис не поддерживает аутентификацию или ресурсы в частных хранилищах. Например, файлы, доступные только во внутренней сети или требующие входа в систему, не могут использовать эту функцию предварительного просмотра.
- После того как содержимое файла будет получено сервисом Microsoft, оно может быть кэшировано на короткое время. Даже если исходный файл будет удален, содержимое предварительного просмотра может оставаться доступным в течение некоторого времени.
- Существуют рекомендуемые ограничения на размер файлов: для Word и PowerPoint — не более 10 МБ, для Excel — не более 5 МБ для обеспечения стабильности просмотра[^3].
- На данный момент официальное четкое описание лицензии на коммерческое использование этого сервиса отсутствует. Пожалуйста, самостоятельно оцените риски при его использовании[^4].

## Поддерживаемые форматы файлов

Плагин поддерживает предварительный просмотр только следующих форматов файлов Office на основании их MIME-типа или расширения:

- Документы Word:
  `application/vnd.openxmlformats-officedocument.wordprocessingml.document` (`.docx`) или `application/msword` (`.doc`)
- Таблицы Excel:
  `application/vnd.openxmlformats-officedocument.spreadsheetml.sheet` (`.xlsx`) или `application/vnd.ms-excel` (`.xls`)
- Презентации PowerPoint:
  `application/vnd.openxmlformats-officedocument.presentationml.presentation` (`.pptx`) или `application/vnd.ms-powerpoint` (`.ppt`)
- Текст OpenDocument: `application/vnd.oasis.opendocument.text` (`.odt`)

Для файлов других форматов функция предварительного просмотра этого плагина не будет активирована.
