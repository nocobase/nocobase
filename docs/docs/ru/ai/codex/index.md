# Используйте Codex для ИИ-разработки и разработки плагинов в NocoBase

[Codex](https://github.com/openai/codex) — официальный ИИ-ассистент от OpenAI: работает в терминале, читает и пишет код, выполняет команды и помогает — от написания кода до построения систем. После подключения к NocoBase вы можете на естественном языке создавать таблицы данных, собирать страницы, настраивать рабочие процессы и с помощью моделей серии GPT быстро строить бизнес-системы.

## Что такое Codex

Codex — ИИ агент на базе CLI от OpenAI, работающий на моделях серии GPT (включая o3, o4-mini и др.). Запускается в локальной песочнице и безопасно выполняет код и команды. Основные возможности:

- **На базе серии GPT** — на основе последних моделей OpenAI, отлично справляется с генерацией кода и планированием задач
- **Выполнение в песочнице** — команды выполняются в изолированной песочнице, безопасно и под контролем
- **Мультимодальное понимание** — поддерживает текст, изображения и другие типы ввода; может понимать макеты интерфейса по скриншотам
- **Гибкие уровни автономности** — от режима подсказок до полной автоматизации; вы сами выбираете степень автономности ИИ

## Почему выбрать Codex

Если вы выбираете, какой ИИ агент использовать для управления NocoBase, вот сценарии, где Codex подходит лучше всего:

- **Уже используете экосистему OpenAI** — если у вас подписка ChatGPT Plus/Pro или ключ OpenAI API, Codex — самый естественный выбор
- **Цените безопасность** — механизм выполнения в песочнице гарантирует, что операции ИИ случайно не повлияют на вашу систему
- **Нужен гибкий контроль** — переключайте уровни автономности в зависимости от сложности задачи: полная автоматизация для простых задач, подтверждение для чувствительных операций
- **Предпочитаете стиль моделей OpenAI** — серия GPT сильна в планировании задач и пошаговом выполнении

## Как происходит подключение

Codex работает с NocoBase следующим образом:

```
Вы (Terminal / ...)
  |
  └─→ Codex
        |
        ├── NocoBase Skills (позволяет агенту понимать систему конфигурации NocoBase)
        |
        └── NocoBase CLI (выполняет создание, изменение, развёртывание и другие операции)
              |
              └─→ NocoBase Service (ваша бизнес-система)
```

- **NocoBase Skills** — пакеты предметных знаний, которые объясняют Codex, как управлять NocoBase
- **NocoBase CLI** — инструмент командной строки, который фактически выполняет моделирование данных, создание страниц и другие операции
- **NocoBase Service** — ваш работающий экземпляр NocoBase

## Предварительные требования

Перед началом убедитесь, что подготовлено следующее окружение:

- Установлен Codex (`npm install -g @openai/codex`)
- Node.js >= 22 (для работы NocoBase CLI и Skills)
- Если у Вас уже есть экземпляр NocoBase — **из-за быстрого развития ИИ-возможностей минимальная требуемая версия — NocoBase 2.1.0. Рекомендуется обновиться до последней версии.**

## Быстрый старт

### Установка в один клик с помощью ИИ

Скопируйте приведённый ниже запрос в Codex — он автоматически выполнит установку NocoBase CLI, инициализацию и настройку окружения:

```
Помоги мне установить NocoBase CLI и завершить инициализацию: https://docs.nocobase.com/ai/ai-quick-start.md (пожалуйста, прочитай содержимое по ссылке напрямую)
```

### Ручная установка

```bash
npm install -g @nocobase/cli
nb init --ui
```

Браузер автоматически откроет страницу визуальной настройки и проведёт вас через установку NocoBase Skills, настройку базы данных и запуск приложения. Подробные шаги см. в [Быстрый старт](../quick-start.md).

После завершения установки выполните `nb env list`, чтобы подтвердить состояние окружения:

```bash
nb env list
```

Убедитесь, что в выводе отображаются настроенное окружение и его статус выполнения.
