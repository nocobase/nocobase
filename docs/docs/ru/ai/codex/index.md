# Управляйте NocoBase через Codex — стройте и разрабатывайте без компромиссов

[Codex](https://github.com/openai/codex) — официальный ИИ-помощник по программированию от OpenAI. Он работает в терминале, умеет читать и писать код, выполнять команды и помогает справляться с разными задачами от написания кода до построения целых систем. После подключения к NocoBase Вы сможете на естественном языке создавать таблицы данных, строить страницы и настраивать workflow, быстро собирая бизнес-систему благодаря возможностям моделей серии GPT.

## Что такое Codex

Codex — это AI Agent в формате CLI от OpenAI, основанный на моделях семейства GPT (включая o3, o4-mini и др.). Он работает в локальной песочнице и умеет безопасно выполнять код и команды. Ключевые особенности:

- **Поддержка моделей GPT** — на базе самых свежих моделей OpenAI, отлично подходит для генерации кода и планирования задач
- **Исполнение в песочнице** — выполняет команды в изолированной песочнице, безопасно и контролируемо
- **Мультимодальное понимание** — поддерживает разные типы ввода (текст, изображения и т. д.) и понимает разметку UI на скриншотах
- **Гибкие уровни автономности** — от режима подсказок до полностью автоматического режима, степень автономности выбираете Вы

## Почему стоит выбрать Codex

Если Вы выбираете AI Agent для работы с NocoBase, Codex лучше всего подойдёт в следующих случаях:

- **Уже работаете в экосистеме OpenAI** — если у Вас есть подписка ChatGPT Plus/Pro или ключ OpenAI API, Codex — самый естественный выбор
- **Цените безопасность** — механизм песочницы гарантирует, что действия ИИ не повлияют на Вашу систему случайным образом
- **Нужен гибкий контроль** — уровень автономности можно менять в зависимости от сложности задачи: простое выполняется автоматически, чувствительные операции требуют подтверждения
- **Нравится стиль моделей OpenAI** — серия GPT имеет свои сильные стороны в планировании задач и пошаговом выполнении

## Принцип подключения

Codex работает с NocoBase следующим образом:

```
Вы (терминал / ...)
  │
  └─→ Codex
        │
        ├── NocoBase Skills (помогают Agent понимать систему конфигурации NocoBase)
        │
        └── NocoBase CLI (выполняет создание, изменение, развёртывание и т. п.)
              │
              └─→ Сервис NocoBase (Ваша бизнес-система)
```

- **NocoBase Skills** — пакет доменных знаний, благодаря которому Codex знает, как работать с NocoBase
- **NocoBase CLI** — инструмент командной строки, фактически выполняющий моделирование данных, построение страниц и т. д.
- **Сервис NocoBase** — Ваш запущенный экземпляр NocoBase

## Предварительные требования

Перед началом убедитесь, что Ваше окружение готово:

- Установлен Codex (`npm install -g @openai/codex`)
- Node.js >= 22 (для работы NocoBase CLI и Skills)
- Если у Вас уже есть экземпляр NocoBase — **из-за быстрого развития ИИ-возможностей полноценный опыт сейчас доступен только в последней beta-версии. Минимальная требуемая версия >= 2.1.0-beta.20, настоятельно рекомендуется обновиться до самой свежей версии.**

## Быстрый старт

### Установка одной фразой через ИИ

Скопируйте промпт ниже и отправьте его Codex. Он автоматически выполнит установку NocoBase CLI, инициализацию и настройку окружения:

```
Помоги мне установить NocoBase CLI и выполнить инициализацию: https://docs.nocobase.com/cn/ai/ai-quick-start.md (открой ссылку и следуй её содержимому)
```

### Ручная установка

```bash
npm install -g @nocobase/cli@beta
nb init --ui
```

В браузере автоматически откроется страница визуальной настройки, которая проведёт Вас через установку NocoBase Skills, настройку базы данных и запуск приложения. Подробные шаги см. в [Быстром старте](../quick-start.md).

После установки выполните `nb env list`, чтобы проверить состояние окружения:

```bash
nb env list
```

Убедитесь, что в выводе присутствует настроенное окружение и его статус выполнения.
