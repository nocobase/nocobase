# Освободите руки — управляйте NocoBase с помощью WorkBuddy

[WorkBuddy](https://www.codebuddy.cn) — универсальный офисный ИИ-агент от Tencent: одна фраза с описанием задачи — и он сам спланирует шаги и выполнит их. После подключения к NocoBase Вы сможете удалённо управлять Вашей бизнес-системой через WeCom, Lark, DingTalk и другие платформы, выполняя повседневные задачи без открытия браузера.

## Что такое WorkBuddy

WorkBuddy — это «универсальный десктопный рабочий стол офисного ИИ-агента» от Tencent. В отличие от обычных чат-инструментов, WorkBuddy получает задачу, автоматически разбивает её на шаги, планирует, выполняет и в итоге передаёт пригодный для проверки результат — Вам не нужно вести его шаг за шагом.

Ключевые особенности:

- **Автономное планирование и выполнение** — после получения задачи сам разбивает её на шаги, выполняет их по очереди и сдаёт готовый результат
- **Подключение к разным платформам** — поддерживает основные офисные платформы: WeCom, Lark, DingTalk, QQ и другие
- **20+ встроенных навыков** — генерация документов, анализ данных, создание презентаций, редактирование писем и др. — всё из коробки
- **Работа с локальными файлами** — может читать и обрабатывать локальные файлы, на которые Вы дали разрешение

Проще говоря: традиционные ИИ-инструменты дают советы, а действовать приходится Вам, тогда как WorkBuddy всё делает за Вас.

| Традиционный ИИ-чат                | WorkBuddy                                          |
| ---------------------------------- | -------------------------------------------------- |
| Только разговор и советы           | Реально выполняет задачи                           |
| Нужно вручную работать с файлами   | Автоматически работает с локальными файлами        |
| Простые одношаговые задачи         | Сложные многошаговые задачи разбиваются автоматически |
| Текстовый ответ                    | Сдаёт результат, пригодный для проверки            |

## Почему стоит выбрать WorkBuddy

Если Вы выбираете AI Agent для работы с NocoBase, WorkBuddy лучше всего подойдёт в следующих случаях:

- **Команда использует WeCom / Lark / DingTalk** — WorkBuddy поддерживает самый широкий набор офисных платформ
- **Нужно мобильное управление NocoBase** — управляйте системой в любой момент в дороге, без привязки к устройству
- **Хотите решение «из коробки»** — продукт от Tencent с 20+ встроенными навыками и низким порогом настройки
- **Делаете акцент на автоматизации задач** — WorkBuddy хорошо справляется с автоматическим разбиением и выполнением многошаговых задач, что подходит для повседневного администрирования и сопровождения

## Принцип подключения

WorkBuddy работает с NocoBase следующим образом:

```
Вы (WeCom / Lark / DingTalk / ...)
  │
  └─→ WorkBuddy
        │
        ├── NocoBase Skills (помогают Agent понимать систему конфигурации NocoBase)
        │
        └── NocoBase CLI (выполняет создание, изменение, развёртывание и т. п.)
              │
              └─→ Сервис NocoBase (Ваша бизнес-система)
```

Вы отправляете команду в любой из поддерживаемых платформ, WorkBuddy на бэкенде через Skills и CLI выполняет операции с NocoBase, а результат в реальном времени возвращается в Ваше окно диалога.

## Предварительные требования

Перед началом убедитесь, что Ваше окружение готово:

- Учётная запись WorkBuddy ([регистрация](https://www.codebuddy.cn))
- Node.js >= 22 (для работы NocoBase CLI и Skills)
- Если у Вас уже есть экземпляр NocoBase — **из-за быстрого развития ИИ-возможностей полноценный опыт сейчас доступен только в последней beta-версии. Минимальная требуемая версия >= 2.1.0-beta.20, настоятельно рекомендуется обновиться до самой свежей версии.**

:::warning Внимание

WorkBuddy сейчас активно развивается, отдельные функции могут измениться. Рекомендуется следить за актуальной информацией на [официальной странице документации WorkBuddy](https://www.codebuddy.cn/docs/workbuddy/Overview).

:::

## Быстрый старт

### Установка одной фразой через ИИ

Скопируйте промпт ниже и отправьте его WorkBuddy. Он автоматически выполнит установку NocoBase CLI, инициализацию и настройку окружения:

```
Помоги мне установить NocoBase CLI и выполнить инициализацию: https://docs.nocobase.com/cn/ai/ai-quick-start.md (открой ссылку и следуй её содержимому)
```

### Ручная установка

```bash
npm install -g @nocobase/cli@beta
nb init --ui
```

В браузере автоматически откроется страница визуальной настройки, которая проведёт Вас через установку NocoBase Skills, настройку базы данных и запуск приложения. Подробные шаги см. в [Быстром старте](../quick-start.md).

После установки выполните `nb env list`, чтобы проверить состояние окружения:

```bash
nb env list
```

Убедитесь, что в выводе присутствует настроенное окружение и его статус выполнения.
