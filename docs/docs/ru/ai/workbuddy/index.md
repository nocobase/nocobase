# Освободите руки: WorkBuddy управляет NocoBase

[WorkBuddy](https://www.codebuddy.cn) — ИИ агент для рабочего места на все сценарии от Tencent: опишите задачу одной фразой — и он самостоятельно спланирует шаги и выполнит их. После подключения к NocoBase вы можете удалённо управлять бизнес-системой через WeCom, Lark, DingTalk и другие платформы, выполняя повседневные задачи управления без открытия браузера.

## Что такое WorkBuddy

WorkBuddy — «настольная рабочая станция ИИ агента для всех сценариев рабочего места» от Tencent. В отличие от обычных диалоговых ИИ-инструментов, WorkBuddy при получении задачи автоматически разбивает её, планирует и выполняет, в итоге выдавая проверяемый результат — без пошагового руководства с вашей стороны.

Основные возможности:

- **Автономное планирование и выполнение** — автоматически разбивает задачи на шаги, выполняет их по очереди и выдаёт готовый результат
- **Доступ с нескольких платформ** — поддерживает WeCom, Lark, DingTalk, QQ и другие популярные корпоративные платформы Китая
- **Более 20 встроенных навыков** — генерация документов, анализ данных, создание PPT, редактирование писем и многое другое — готово к использованию из коробки
- **Работа с локальными файлами** — может читать и обрабатывать локальные файлы, к которым вы дали доступ

Короче говоря, традиционные ИИ-инструменты дают рекомендации, а работу выполняете вы; WorkBuddy выполняет работу за вас.

| Традиционный ИИ-чат              | WorkBuddy                                 |
| -------------------------------- | ----------------------------------------- |
| Только общается, даёт рекомендации | Реально выполняет задачи                |
| Требует ручной работы с файлами  | Автоматически работает с локальными файлами        |
| Простые одношаговые задачи         | Многошаговые сложные задачи, автоматическая декомпозиция |
| Выдаёт текстовые ответы             | Выдаёт проверяемые результаты               |

## Почему выбрать WorkBuddy

Если вы выбираете, какой ИИ агент использовать для управления NocoBase, вот сценарии, где WorkBuddy подходит лучше всего:

- **Команда использует WeCom / Lark / DingTalk** — WorkBuddy поддерживает самый широкий набор китайских корпоративных платформ
- **Нужно управлять NocoBase с мобильного** — управляйте системой в любое время и из любого места, без привязки к устройству
- **Нужен готовый опыт из коробки** — продукт Tencent, более 20 встроенных навыков и низкий порог настройки
- **Фокус на автоматизации задач** — WorkBuddy силён в автоматической декомпозиции и выполнении многошаговых задач, подходит для повседневной эксплуатации и управления

## Как происходит подключение

WorkBuddy работает с NocoBase следующим образом:

```
Вы (WeCom / Lark / DingTalk / ...)
  |
  └─→ WorkBuddy
        |
        ├── NocoBase Skills (позволяет агенту понимать систему конфигурации NocoBase)
        |
        └── NocoBase CLI (выполняет создание, изменение, развёртывание и другие операции)
              |
              └─→ NocoBase Service (ваша бизнес-система)
```

Вы отправляете инструкции с любой поддерживаемой платформы, WorkBuddy на бэкенде выполняет операции NocoBase через Skills и CLI, а результаты в реальном времени приходят в окно чата.

## Предварительные требования

Перед началом убедитесь, что подготовлено следующее окружение:

- Учётная запись WorkBuddy ([регистрация](https://www.codebuddy.cn))
- Node.js >= 22 (для работы NocoBase CLI и Skills)
- Если у Вас уже есть экземпляр NocoBase — **из-за быстрого развития ИИ-возможностей минимальная требуемая версия — NocoBase 2.1.0. Рекомендуется обновиться до последней версии.**

:::warning Примечание

WorkBuddy сейчас активно развивается, некоторые возможности могут меняться. Рекомендуем следить за [официальной документацией WorkBuddy](https://www.codebuddy.cn/docs/workbuddy/Overview) для актуальной информации.

:::

## Быстрый старт

### Установка в один клик с помощью ИИ

Скопируйте приведённый ниже запрос в WorkBuddy — он автоматически выполнит установку NocoBase CLI, инициализацию и настройку окружения:

```
Помоги мне установить NocoBase CLI и завершить инициализацию: https://docs.nocobase.com/ai/ai-quick-start.md (пожалуйста, прочитай содержимое по ссылке напрямую)
```

### Ручная установка

```bash
npm install -g @nocobase/cli
nb init --ui
```

Браузер автоматически откроет страницу визуальной настройки и проведёт вас через установку NocoBase Skills, настройку базы данных и запуск приложения. Подробные шаги см. в [Быстрый старт](../quick-start.md).

После завершения установки выполните `nb env list`, чтобы подтвердить состояние окружения:

```bash
nb env list
```

Убедитесь, что в выводе отображаются настроенное окружение и его статус выполнения.
