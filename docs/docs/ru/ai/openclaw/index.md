# OpenClaw + NocoBase: самый популярный ИИ агент работает на вас

[OpenClaw](https://github.com/openclaw/openclaw) — самый популярный в мире фреймворк ИИ агентов с открытым исходным кодом: он не просто общается, а реально выполняет задачи. После подключения к NocoBase вы можете на естественном языке создавать таблицы данных, собирать страницы, настраивать рабочие процессы и даже позволить ему работать автономно 24/7, непрерывно сопровождая вашу бизнес-систему.

## Что такое OpenClaw

OpenClaw — фреймворк ИИ агентов с открытым исходным кодом, созданный разработчиком Peter Steinberger, и один из самых популярных проектов ИИ агентов в мире (более 300 тыс. звёзд на GitHub). Его позиционирование — «ИИ-ассистент, который доводит дела до конца». В отличие от диалоговых инструментов вроде ChatGPT и Claude, у OpenClaw четыре ключевые возможности:

- **Способность к выполнению** — после получения инструкций на естественном языке автоматически завершает задачи, а не только даёт рекомендации
- **Межсессионная память** — запоминает ваши предпочтения и привычки использования, становясь лучше по мере применения
- **Экосистема навыков** — расширяйте возможности установкой навыков, словно «обучая ассистента новым навыкам»
- **Работа 24/7** — поддерживает запланированные задачи и проактивную отчётность, не нужно постоянно следить самому

OpenClaw поддерживает более 26 платформ, включая Lark, Telegram и Discord, поэтому можно общаться с ним прямо в инструментах повседневной работы. Пользователи Lark могут развернуть его в один клик, без технической подготовки.

## Почему выбрать OpenClaw

Если вы выбираете, какой ИИ агент использовать для управления NocoBase, вот сценарии, где OpenClaw подходит лучше всего:

- **Нужен вход без барьеров** — пользователи Lark могут развернуть в один клик, без настройки собственного сервера
- **Команда работает в Lark** — OpenClaw глубоко интегрирован с Lark, с удобными возможностями вроде потоковых сообщений и @бота в групповых чатах
- **Нужна доступность 24/7** — OpenClaw развёрнут в облаке, не зависит от состояния вашего локального компьютера
- **Цените экосистему сообщества** — у OpenClaw крупнейшее сообщество Skills, доступно много навыков помимо NocoBase

## Как происходит подключение

OpenClaw работает с NocoBase следующим образом:

```
Вы (Lark / Telegram / ...)
  |
  └─→ OpenClaw Agent
        |
        ├── NocoBase Skills (позволяет агенту понимать систему конфигурации NocoBase)
        |
        └── NocoBase CLI (выполняет создание, изменение, развёртывание и другие операции)
              |
              └─→ NocoBase Service (ваша бизнес-система)
```

- **NocoBase Skills** — пакеты предметных знаний, которые объясняют OpenClaw, как управлять NocoBase
- **NocoBase CLI** — инструмент командной строки, который фактически выполняет моделирование данных, создание страниц и другие операции
- **NocoBase Service** — ваш работающий экземпляр NocoBase

## Предварительные требования

Перед началом убедитесь, что подготовлено следующее окружение:

- Развёрнутый OpenClaw Agent ([развёртывание в Lark в один клик](https://openclaw.feishu.cn) или локальное развёртывание)
- Node.js >= 22 (для работы NocoBase CLI и Skills)
- Если у Вас уже есть экземпляр NocoBase — **из-за быстрого развития ИИ-возможностей минимальная требуемая версия — NocoBase 2.1.0. Рекомендуется обновиться до последней версии.**

:::warning Примечание

При установке сторонних навыков обращайте внимание на безопасность — отдавайте приоритет официальным или проверенным источникам навыков и избегайте непроверенных навыков сообщества.

:::

## Быстрый старт

### Установка в один клик с помощью ИИ

Скопируйте приведённый ниже запрос в OpenClaw — он автоматически выполнит установку NocoBase CLI, инициализацию и настройку окружения:

```
Помоги мне установить NocoBase CLI и завершить инициализацию: https://docs.nocobase.com/ai/ai-quick-start.md (пожалуйста, прочитай содержимое по ссылке напрямую)
```

### Ручная установка

```bash
npm install -g @nocobase/cli
nb init --ui
```

Браузер автоматически откроет страницу визуальной настройки и проведёт вас через установку NocoBase Skills, настройку базы данных и запуск приложения. Подробные шаги см. в [Быстрый старт](../quick-start.md).

После завершения установки выполните `nb env list`, чтобы подтвердить состояние окружения:

```bash
nb env list
```

Убедитесь, что в выводе отображаются настроенное окружение и его статус выполнения.
