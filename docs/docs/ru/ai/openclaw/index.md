# OpenClaw + NocoBase: самый популярный в мире AI Agent работает за Вас

[OpenClaw](https://github.com/openclaw/openclaw) — самый популярный в мире open source фреймворк AI Agent. Это не просто чат, он умеет реально выполнять задачи. После подключения к NocoBase Вы сможете на естественном языке создавать таблицы данных, строить страницы, настраивать workflow и даже доверить ему круглосуточно (7×24) автономно сопровождать Вашу бизнес-систему.

## Что такое OpenClaw

OpenClaw — open source фреймворк AI Agent, созданный разработчиком Питером Штайнбергером (Peter Steinberger). На сегодняшний день — один из самых популярных в мире проектов AI Agent (300k+ звёзд на GitHub). Его позиционирование — «ИИ-помощник, который умеет реально что-то делать». В отличие от чат-инструментов вроде ChatGPT и Claude, у OpenClaw четыре ключевые особенности:

- **Способность выполнять** — получив инструкцию на естественном языке, автоматически выполняет задачу, а не просто даёт советы
- **Межсессионная память** — запоминает Ваши предпочтения и привычки, так что работа становится всё удобнее
- **Экосистема Skills** — расширяет возможности через установку Skills, словно Вы «учите помощника новым навыкам»
- **Работа 7×24** — поддерживает запланированные задачи и проактивные отчёты, не требуя постоянного контроля

OpenClaw поддерживает 26+ платформ — Lark, Telegram, Discord и другие. Вы можете общаться с ним прямо в привычных рабочих инструментах. Пользователи Lark могут развернуть его в один клик без каких-либо технических навыков.

## Почему стоит выбрать OpenClaw

Если Вы выбираете AI Agent для работы с NocoBase, OpenClaw лучше всего подойдёт в следующих случаях:

- **Нужен порог входа близкий к нулю** — пользователи Lark могут развернуть его в один клик без своего сервера
- **Команда работает в Lark** — OpenClaw глубоко интегрирован с Lark: потоковая генерация сообщений, упоминание @bot в групповых чатах и другие сценарии работают плавно
- **Нужна работа 7×24** — OpenClaw развёрнут в облаке, его работа не зависит от состояния Вашего локального компьютера
- **Цените сообщество** — у OpenClaw самое большое сообщество Skills, кроме NocoBase доступно множество других навыков

## Принцип подключения

OpenClaw работает с NocoBase следующим образом:

```
Вы (Lark / Telegram / ...)
  │
  └─→ OpenClaw Agent
        │
        ├── NocoBase Skills (помогают Agent понимать систему конфигурации NocoBase)
        │
        └── NocoBase CLI (выполняет создание, изменение, развёртывание и т. п.)
              │
              └─→ Сервис NocoBase (Ваша бизнес-система)
```

- **NocoBase Skills** — пакет доменных знаний, благодаря которому OpenClaw знает, как работать с NocoBase
- **NocoBase CLI** — инструмент командной строки, фактически выполняющий моделирование данных, построение страниц и т. д.
- **Сервис NocoBase** — Ваш запущенный экземпляр NocoBase

## Предварительные требования

Перед началом убедитесь, что Ваше окружение готово:

- Развёрнутый OpenClaw Agent ([развёртывание в Lark в один клик](https://openclaw.feishu.cn) или локальное развёртывание)
- Node.js >= 22 (для работы NocoBase CLI и Skills)
- Если у Вас уже есть экземпляр NocoBase — **из-за быстрого развития ИИ-возможностей полноценный опыт сейчас доступен только в последней beta-версии. Минимальная требуемая версия >= 2.1.0-beta.20, настоятельно рекомендуется обновиться до самой свежей версии.**

:::warning Внимание

При установке сторонних Skills соблюдайте осторожность — отдавайте предпочтение Skills из официальных или доверенных источников и избегайте непроверенных навыков из сообщества.

:::

## Быстрый старт

### Установка одной фразой через ИИ

Скопируйте промпт ниже и отправьте его OpenClaw. Он автоматически выполнит установку NocoBase CLI, инициализацию и настройку окружения:

```
Помоги мне установить NocoBase CLI и выполнить инициализацию: https://docs.nocobase.com/cn/ai/ai-quick-start.md (открой ссылку и следуй её содержимому)
```

### Ручная установка

```bash
npm install -g @nocobase/cli@beta
nb init --ui
```

В браузере автоматически откроется страница визуальной настройки, которая проведёт Вас через установку NocoBase Skills, настройку базы данных и запуск приложения. Подробные шаги см. в [Быстром старте](../quick-start.md).

После установки выполните `nb env list`, чтобы проверить состояние окружения:

```bash
nb env list
```

Убедитесь, что в выводе присутствует настроенное окружение и его статус выполнения.
