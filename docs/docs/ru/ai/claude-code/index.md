# Claude Code + NocoBase: самый мощный ИИ-мозг — ваш главный архитектор NocoBase

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) — официальный ИИ-ассистент от Anthropic: работает прямо в терминале, понимает всю кодовую базу и помогает — от написания кода до построения систем. После подключения к NocoBase вы можете на естественном языке создавать таблицы данных, собирать страницы, настраивать рабочие процессы и пользоваться возможностями самых мощных ИИ-моделей.

## Что такое Claude Code

Claude Code — ИИ агент на базе CLI от Anthropic, работающий на моделях серии Claude. Запускается прямо в терминале, понимает контекст проекта и выполняет операции. Основные возможности:

- **Модели высшего уровня** — на базе Claude Opus / Sonnet, лидирует в понимании и генерации кода
- **Нативная работа в терминале** — запуск прямо в терминале, бесшовная интеграция с рабочим процессом разработчика
- **Осведомлённость о проекте** — автоматически понимает структуру проекта, зависимости и соглашения кода
- **Совместная работа с множеством инструментов** — чтение и запись файлов, выполнение команд, поиск по коду и многое другое

Claude Code также поддерживает интеграцию с VS Code, JetBrains и другими IDE, доступен как отдельное настольное и веб-приложение.

## Почему выбрать Claude Code

Если вы выбираете, какой ИИ агент использовать для управления NocoBase, вот сценарии, где Claude Code подходит лучше всего:

- **Нужны самые мощные модели** — серия моделей Claude отлично справляется со сложными рассуждениями и генерацией кода
- **Повседневный рабочий процесс разработчика** — нативная работа в терминале, бесшовная интеграция с IDE, Git, npm и другими инструментами
- **Нужно глубокое понимание проекта** — Claude Code автоматически анализирует структуру проекта и даёт контекстно-зависимые рекомендации
- **И создание, и разработка** — помогает и собирать приложения NocoBase, и разрабатывать пользовательские плагины

## Как происходит подключение

Claude Code работает с NocoBase следующим образом:

```
Вы (Terminal / VS Code / JetBrains / ...)
  |
  └─→ Claude Code
        |
        ├── NocoBase Skills (позволяет агенту понимать систему конфигурации NocoBase)
        |
        └── NocoBase CLI (выполняет создание, изменение, развёртывание и другие операции)
              |
              └─→ NocoBase Service (ваша бизнес-система)
```

- **NocoBase Skills** — пакеты предметных знаний, которые объясняют Claude Code, как управлять NocoBase
- **NocoBase CLI** — инструмент командной строки, который фактически выполняет моделирование данных, создание страниц и другие операции
- **NocoBase Service** — ваш работающий экземпляр NocoBase

## Предварительные требования

Перед началом убедитесь, что подготовлено следующее окружение:

- Установлен Claude Code (`npm install -g @anthropic-ai/claude-code`)
- Node.js >= 22 (для работы NocoBase CLI и Skills)
- Если у Вас уже есть экземпляр NocoBase — **из-за быстрого развития ИИ-возможностей минимальная требуемая версия — NocoBase 2.1.0. Рекомендуется обновиться до последней версии.**

## Быстрый старт

### Установка в один клик с помощью ИИ

Скопируйте приведённый ниже запрос в Claude Code — он автоматически выполнит установку NocoBase CLI, инициализацию и настройку окружения:

```
Помоги мне установить NocoBase CLI и завершить инициализацию: https://docs.nocobase.com/ai/ai-quick-start.md (пожалуйста, прочитай содержимое по ссылке напрямую)
```

### Ручная установка

```bash
npm install -g @nocobase/cli
nb init --ui
```

Браузер автоматически откроет страницу визуальной настройки и проведёт вас через установку NocoBase Skills, настройку базы данных и запуск приложения. Подробные шаги см. в [Быстрый старт](../quick-start.md).

После завершения установки выполните `nb env list`, чтобы подтвердить состояние окружения:

```bash
nb env list
```

Убедитесь, что в выводе отображаются настроенное окружение и его статус выполнения.
