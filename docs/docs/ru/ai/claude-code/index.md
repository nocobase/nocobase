# Claude Code + NocoBase: мощнейший ИИ-мозг и Ваш главный архитектор NocoBase

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) — официальный ИИ-помощник по программированию от Anthropic. Он работает прямо в терминале, понимает весь Ваш кодовый репозиторий и помогает решать задачи от написания кода до построения целых систем. После подключения к NocoBase Вы сможете на естественном языке создавать таблицы данных, строить страницы и настраивать workflow, получая опыт построения, который обеспечивают самые мощные ИИ-модели.

## Что такое Claude Code

Claude Code — это AI Agent в формате CLI от Anthropic, основанный на моделях семейства Claude. Он работает прямо в терминале, понимает контекст проекта и выполняет действия. Ключевые особенности:

- **Лучшие модели** — на базе Claude Opus / Sonnet, которые лидируют в понимании и генерации кода
- **Нативный для терминала** — работает прямо в терминале и бесшовно встраивается в рабочий процесс разработчика
- **Понимание проекта** — автоматически разбирает структуру проекта, зависимости и стандарты кода
- **Совместная работа инструментов** — поддерживает чтение и запись файлов, выполнение команд, поиск по коду и другие операции

Claude Code также интегрируется с VS Code, JetBrains и другими IDE и доступен как самостоятельное приложение для рабочего стола и веб.

## Почему стоит выбрать Claude Code

Если Вы выбираете AI Agent для работы с NocoBase, Claude Code лучше всего подойдёт в следующих случаях:

- **Стремление к максимальной мощи модели** — модели семейства Claude отлично справляются со сложными рассуждениями и генерацией кода
- **Повседневный рабочий процесс разработчика** — нативный для терминала, прекрасно сочетается с Вашими IDE, Git, npm и другими инструментами
- **Нужно глубокое понимание проекта** — Claude Code сам анализирует структуру проекта и даёт рекомендации с учётом контекста
- **Одновременно построение и разработка** — помогает и собирать приложение NocoBase, и разрабатывать собственные плагины

## Принцип подключения

Claude Code работает с NocoBase следующим образом:

```
Вы (терминал / VS Code / JetBrains / ...)
  │
  └─→ Claude Code
        │
        ├── NocoBase Skills (помогают Agent понимать систему конфигурации NocoBase)
        │
        └── NocoBase CLI (выполняет создание, изменение, развёртывание и т. п.)
              │
              └─→ Сервис NocoBase (Ваша бизнес-система)
```

- **NocoBase Skills** — пакет доменных знаний, благодаря которому Claude Code знает, как работать с NocoBase
- **NocoBase CLI** — инструмент командной строки, фактически выполняющий моделирование данных, построение страниц и т. д.
- **Сервис NocoBase** — Ваш запущенный экземпляр NocoBase

## Предварительные требования

Перед началом убедитесь, что Ваше окружение готово:

- Установлен Claude Code (`npm install -g @anthropic-ai/claude-code`)
- Node.js >= 22 (для работы NocoBase CLI и Skills)
- Если у Вас уже есть экземпляр NocoBase — **из-за быстрого развития ИИ-возможностей полноценный опыт сейчас доступен только в последней beta-версии. Минимальная требуемая версия >= 2.1.0-beta.20, настоятельно рекомендуется обновиться до самой свежей версии.**

## Быстрый старт

### Установка одной фразой через ИИ

Скопируйте промпт ниже и отправьте его Claude Code. Он автоматически выполнит установку NocoBase CLI, инициализацию и настройку окружения:

```
Помоги мне установить NocoBase CLI и выполнить инициализацию: https://docs.nocobase.com/cn/ai/ai-quick-start.md (открой ссылку и следуй её содержимому)
```

### Ручная установка

```bash
npm install -g @nocobase/cli@beta
nb init --ui
```

В браузере автоматически откроется страница визуальной настройки, которая проведёт Вас через установку NocoBase Skills, настройку базы данных и запуск приложения. Подробные шаги см. в [Быстром старте](../quick-start.md).

После установки выполните `nb env list`, чтобы проверить состояние окружения:

```bash
nb env list
```

Убедитесь, что в выводе присутствует настроенное окружение и его статус выполнения.
