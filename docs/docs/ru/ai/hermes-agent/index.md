# Hermes Agent: помощник NocoBase, который учится по ходу работы

[Hermes Agent](https://github.com/nousresearch/hermes-agent) — ИИ агент с открытым исходным кодом для самостоятельного хостинга: автоматически превращает каждую успешную операцию в переиспользуемые документы навыков и становится умнее по мере использования. После подключения к NocoBase вы можете не только создавать и управлять системами на естественном языке, но и постепенно обучать его вашим бизнес-соглашениям и предпочтениям.

## Что такое Hermes Agent

Hermes Agent разработан Nous Research (35,7 тыс. звёзд на GitHub); ключевая философия — «чем дольше используешь, тем умнее становится». В отличие от других ИИ агентов, у Hermes есть полный замкнутый цикл обучения:

- **Межсессионная память** — на основе полнотекстового поиска и суммаризации языковой моделью; способен вспоминать контекст разговоров недельной давности
- **Автоматическая дистилляция навыков** — после каждого успешного выполнения сложной задачи автоматически создаёт переиспользуемые документы навыков
- **Непрерывное самосовершенствование** — навыки уточняются при повторном использовании и со временем становятся точнее
- **Поддержка более 400 моделей** — совместим с основными поставщиками языковых моделей, без привязки к одной модели

Hermes поддерживает 8 платформ, включая Lark, Telegram, Discord и Slack, а также может использоваться прямо в терминале.

:::tip Подсказка

Hermes Agent работает на вашем сервере; все данные и память хранятся локально — подходит для сценариев с требованиями к безопасности данных.

:::

## Почему выбрать Hermes Agent

Если вы выбираете, какой ИИ агент использовать для управления NocoBase, вот сценарии, где Hermes подходит лучше всего:

- **Долгосрочное сопровождение одной системы** — механизм памяти Hermes означает, что со временем он всё лучше понимает ваш бизнес; не нужно каждый раз заново объяснять контекст
- **Команда требует самостоятельного хостинга** — данные остаются полностью локальными, не проходят через сторонние облачные сервисы
- **Нужны стандартизированные процедуры работы** — автоматически дистиллированные документы навыков Hermes могут служить руководством по эксплуатации для вашей команды
- **Предпочитаете работу в терминале** — Hermes нативно поддерживает взаимодействие через CLI, удобно для повседневного использования техническими командами

## Как происходит подключение

Hermes Agent работает с NocoBase следующим образом:

```
Вы (Lark / Telegram / Terminal / ...)
  |
  └─→ Hermes Agent
        |
        ├── NocoBase Skills (позволяет агенту понимать систему конфигурации NocoBase)
        |
        ├── NocoBase CLI (выполняет создание, изменение, развёртывание и другие операции)
        |
        └── Memory & Skill Documents (автоматически дистиллируются, непрерывно переиспользуются)
              |
              └─→ NocoBase Service (ваша бизнес-система)
```

В отличие от других агентов, Hermes обновляет память и документы навыков после каждой операции. Эта информация хранится локально и автоматически переиспользуется в последующих операциях.

## Предварительные требования

Перед началом убедитесь, что подготовлено следующее окружение:

- Сервер для запуска Hermes Agent (Linux / macOS, Python 3.10+)
- Node.js >= 22 (для работы NocoBase CLI и Skills)
- Если у Вас уже есть экземпляр NocoBase — **из-за быстрого развития ИИ-возможностей минимальная требуемая версия — NocoBase 2.1.0. Рекомендуется обновиться до последней версии.**

Установка Hermes требует всего одной команды:

```bash
curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash
```

:::warning Примечание

Hermes Agent требует самостоятельного развёртывания и сопровождения. Если предпочитаете готовый опыт без настройки, рассмотрите [OpenClaw](../openclaw/index.md) (развёртывание в Lark в один клик) или [WorkBuddy](../workbuddy/index.md) (хостинг от Tencent).

:::

## Быстрый старт

### Установка в один клик с помощью ИИ

Скопируйте приведённый ниже запрос в Hermes Agent — он автоматически выполнит установку NocoBase CLI, инициализацию и настройку окружения:

```
Помоги мне установить NocoBase CLI и завершить инициализацию: https://docs.nocobase.com/ai/ai-quick-start.md (пожалуйста, прочитай содержимое по ссылке напрямую)
```

### Ручная установка

```bash
npm install -g @nocobase/cli
nb init --ui
```

Браузер автоматически откроет страницу визуальной настройки и проведёт вас через установку NocoBase Skills, настройку базы данных и запуск приложения. Подробные шаги см. в [Быстрый старт](../quick-start.md).

После завершения установки выполните `nb env list`, чтобы подтвердить состояние окружения:

```bash
nb env list
```

Убедитесь, что в выводе отображаются настроенное окружение и его статус выполнения.
