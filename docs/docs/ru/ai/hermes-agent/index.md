# Hermes Agent: ассистент NocoBase, который понимает Вас всё лучше

[Hermes Agent](https://github.com/nousresearch/hermes-agent) — это self-hosted open source AI Agent, который автоматически превращает каждую успешную операцию в переиспользуемый навык — и со временем всё лучше понимает Вашу систему. После подключения к NocoBase Вы сможете не только строить и сопровождать систему на естественном языке, но и научите его постепенно следовать Вашим бизнес-привычкам и предпочтениям.

## Что такое Hermes Agent

Hermes Agent разработан Nous Research (35.7k звёзд на GitHub). Его основная идея — «чем дольше используешь, тем умнее становится». В отличие от других AI Agent, у Hermes есть полный замкнутый цикл обучения:

- **Межсессионная память** — на основе полнотекстового поиска и LLM-сводок может возвращаться к контексту диалогов многонедельной давности
- **Автоматическое накопление навыков** — после успешного выполнения сложной задачи автоматически создаёт документ переиспользуемого навыка
- **Постоянное самосовершенствование** — навыки оптимизируются при многократном использовании, становясь всё точнее
- **Поддержка 400+ моделей** — совместим с основными провайдерами LLM, не привязан к конкретной модели

Hermes поддерживает 8 платформ — Lark, Telegram, Discord, Slack и др., а также может использоваться прямо в терминале.

:::tip Подсказка

Hermes Agent работает на Вашем собственном сервере. Все данные и память хранятся локально, что подходит для сценариев с повышенными требованиями к безопасности.

:::

## Почему стоит выбрать Hermes Agent

Если Вы выбираете AI Agent для работы с NocoBase, Hermes лучше всего подойдёт в следующих случаях:

- **Долгосрочное сопровождение одной системы** — механизм памяти Hermes позволяет ему всё лучше понимать Ваш бизнес, и Вам не приходится каждый раз заново объяснять контекст
- **Команде нужно self-hosted решение** — данные полностью локальные и не проходят через сторонние облачные сервисы
- **Нужна стандартизация процессов** — автоматически накапливаемые документы навыков могут служить руководством для команды
- **Предпочитаете работу в терминале** — Hermes изначально поддерживает CLI-взаимодействие, что подходит техническим командам в повседневной работе

## Принцип подключения

Hermes Agent работает с NocoBase следующим образом:

```
Вы (Lark / Telegram / терминал / ...)
  │
  └─→ Hermes Agent
        │
        ├── NocoBase Skills (помогают Agent понимать систему конфигурации NocoBase)
        │
        ├── NocoBase CLI (выполняет создание, изменение, развёртывание и т. п.)
        │
        └── Память и документы навыков (автоматически накапливаются и переиспользуются)
              │
              └─→ Сервис NocoBase (Ваша бизнес-система)
```

В отличие от других Agent, после каждой операции Hermes обновляет свою память и документы навыков. Эта информация хранится локально и автоматически используется в последующих операциях.

## Предварительные требования

Перед началом убедитесь, что Ваше окружение готово:

- Сервер для запуска Hermes Agent (Linux / macOS, Python 3.10+)
- Node.js >= 22 (для работы NocoBase CLI и Skills)
- Если у Вас уже есть экземпляр NocoBase — **из-за быстрого развития ИИ-возможностей полноценный опыт сейчас доступен только в последней beta-версии. Минимальная требуемая версия >= 2.1.0-beta.20, настоятельно рекомендуется обновиться до самой свежей версии.**

Установка Hermes выполняется одной командой:

```bash
curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash
```

:::warning Внимание

Hermes Agent требует самостоятельного развёртывания и сопровождения. Если Вам нужно решение «из коробки» без настройки, рассмотрите [OpenClaw](../openclaw/index.md) (развёртывание в Lark в один клик) или [WorkBuddy](../workbuddy/index.md) (хостинг от Tencent).

:::

## Быстрый старт

### Установка одной фразой через ИИ

Скопируйте промпт ниже и отправьте его Hermes Agent. Он автоматически выполнит установку NocoBase CLI, инициализацию и настройку окружения:

```
Помоги мне установить NocoBase CLI и выполнить инициализацию: https://docs.nocobase.com/cn/ai/ai-quick-start.md (открой ссылку и следуй её содержимому)
```

### Ручная установка

```bash
npm install -g @nocobase/cli@beta
nb init --ui
```

В браузере автоматически откроется страница визуальной настройки, которая проведёт Вас через установку NocoBase Skills, настройку базы данных и запуск приложения. Подробные шаги см. в [Быстром старте](../quick-start.md).

После установки выполните `nb env list`, чтобы проверить состояние окружения:

```bash
nb env list
```

Убедитесь, что в выводе присутствует настроенное окружение и его статус выполнения.
