# Облачная CRM-система продаж

## 1.1 Контекст

- **Контекст**: с ускорением цифровой трансформации компаниям необходима единая платформа продаж для управления лидами, клиентами, сделками, КП, заказами и ежедневной активностью. На базе NocoBase мы строим облачную CRM-систему продаж, которая обеспечивает централизованное управление данными и автоматизацию процессов.
- > Примечание: это первая часть лёгкого CRM-решения, она не охватывает закупки, отгрузку, управление поставщиками и т. п. Цель — продемонстрировать подходы к реализации функций. Документ носит ознакомительный характер.

## 1.2 Цели и границы проекта

- **Цель**: построить интегрированную CRM-систему, охватывающую лиды, клиентов, контакты, сделки, продукты, КП, заказы и активности, для повышения эффективности продаж и точности данных.
- **Границы**: охватывает ввод, отслеживание, конвертацию данных и отчётность по всему циклу продаж. Управление маркетинговыми активностями — опциональный модуль; закупки, отгрузки, поставщики не включены.

## 1.3 Роли пользователей и права

- **Менеджер по продажам**: вводит и работает с лидами, создаёт клиентов, контакты, сделки, КП, заказы, ведёт ежедневные активности.
- **Руководитель продаж**: контролирует ход продаж, утверждает КП и заказы, смотрит отчёты.
- **Системный администратор**: настраивает права пользователей, обслуживает данные, конфигурирует систему и интерфейсы.

## 1.4 Обзор функциональных модулей

- Краткий обзор основных функций системы:


| Модуль                                  | Подмодули / функции                                                       | Описание                                                                                          |
| --------------------------------------- | ------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------- |
| **Управление лидами**                   | Ввод лидов, работа со статусами, конвертация лидов                        | Сбор и обработка лидов, конвертация подходящих в клиентов, контакты и сделки.                       |
| **Управление клиентами и контактами**   | Управление клиентами, управление контактами                               | Создание карточек клиентов, ведение информации о компаниях и контактах.                              |
| **Управление сделками**                 | Создание сделок, стадии продаж, прогнозы, продукты, причины проигрыша, ожидаемая оплата | Учёт продажных возможностей, отслеживание процесса и прогноз тенденций.                              |
| **Продукты и прайс-листы**              | Каталог продуктов, прайс-листы                                            | Управление продуктами, остатками и ценами.                                                        |
| **Управление КП**                        | Создание КП, согласование, позиции КП                                     | Создание и управление официальными КП с поддержкой согласований и автоматическим расчётом сумм.   |
| **Управление заказами**                  | Создание заказа, отслеживание статуса, отгрузка                           | Учёт создания, согласования и логистики заказов.                                                  |
| **Управление активностями**              | Задачи, встречи, звонки, календарь и напоминания                          | Управление задачами, встречами и звонками с поддержкой расписания.                                |
| **Отчёты и аналитика**                   | Отчёты по продажам, анализ конверсии, воронка продаж                      | Многомерные отчёты по продажам и аналитика данных.                                                |
| **Маркетинговые активности (опционально)** | Планирование, отслеживание эффективности                                  | Планирование и оценка маркетинговых активностей.                                                  |
