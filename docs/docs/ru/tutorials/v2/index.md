# Учебные материалы NocoBase 2.0

В этих учебных материалах мы шаг за шагом построим **минимальную IT-систему обработки заявок** на NocoBase 2.0. Системе достаточно **2 коллекций** и не нужно ни строчки кода — Вы получите подачу заявок, управление категориями, отслеживание изменений, контроль доступа и дашборд с данными.

## Об учебных материалах

- **Целевая аудитория**: бизнес-пользователи, технические специалисты или все, кому интересен NocoBase (желательно базовое понимание компьютеров)
- **Учебный проект**: минимальная IT-система заявок, всего 2 коллекции
- **Время выполнения**: 2–3 часа (для нетехнических пользователей), 1–1,5 часа (для технических)
- **Требования**: окружение Docker или [онлайн-демо](https://demo.nocobase.com/new) (срок действия 24 часа, без установки)
- **Версия**: NocoBase 2.0

## FAQ

### Для каких сценариев подходит NocoBase?

Подходит для внутренних корпоративных инструментов, систем управления данными, согласований, CRM, ERP — всего, где нужна гибкая кастомизация. Поддерживает самостоятельное развёртывание.

### Какая база нужна для прохождения учебных материалов?

Программирование не требуется, но желательны базовые знания о компьютерах. Учебные материалы постепенно объясняют понятия коллекций, полей, связей. Опыт работы с базами данных или Excel облегчит освоение.

### Можно ли расширить систему из учебных материалов?

Да. В учебных материалах используются всего 2 коллекции, но NocoBase поддерживает сложные связи между несколькими коллекциями, интеграцию с внешним API, кастомные плагины и многое другое.

### Какое окружение нужно для развёртывания?

Рекомендуется Docker (Docker Desktop или Linux-сервер), минимум 2 ядра и 4 ГБ ОЗУ. Также поддерживается запуск из исходного кода через Git. Для ознакомления можно сразу запросить [онлайн-демо](https://demo.nocobase.com/new) — устанавливать ничего не нужно, срок действия 24 часа.

### Какие ограничения у бесплатной версии?

Базовая функциональность полностью бесплатна и с открытым исходным кодом. Коммерческая версия даёт расширенные плагины и техподдержку — см. [тарифы коммерческой версии](https://nocobase.ru/commercial).

## Технологический стек

NocoBase 2.0 построен на:

- **Клиентская часть**: React + [Ant Design](https://ant.design/) 5.0
- **Серверная часть**: Node.js + Koa
- **БД**: PostgreSQL (также поддерживаются [MySQL](/get-started/installation/docker) и MariaDB)
- **Развёртывание**: [Docker](/get-started/installation/docker), Kubernetes

## Похожие платформы

Если Вы оцениваете no-code/low-code платформы, вот сравнение для ориентира:

| Платформа | Особенности | Отличия от NocoBase |
|------|------|-------------------|
| [Appsmith](https://www.appsmith.com/) | no-code с открытым исходным кодом, сильная кастомизация UI | NocoBase делает упор на модель данных |
| [Retool](https://retool.com/) | Платформа внутренних инструментов | NocoBase полностью с открытым исходным кодом, без ограничений |
| [Airtable](https://airtable.com/) | Онлайн совместная БД | NocoBase поддерживает самостоятельное развёртывание, Ваши данные у Вас |
| [Budibase](https://budibase.com/) | low-code с открытым исходным кодом, самостоятельное развёртывание | Плагин-архитектура NocoBase лучше расширяется |
