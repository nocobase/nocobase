# Менеджер маршрутов

## Введение

Менеджер маршрутов — это инструмент для управления маршрутами главной страницы системы, поддерживающий `десктопные` и `мобильные` устройства. Маршруты, созданные с помощью менеджера маршрутов, автоматически отображаются в меню (их можно настроить так, чтобы они не отображались в меню). И наоборот, пункты меню, добавленные через меню страницы, также синхронизируются со списком менеджера маршрутов.

## Руководство пользователя

### Типы маршрутов

Система поддерживает четыре типа маршрутов:

- Группа (`group`): Используется для группировки маршрутов и может содержать дочерние маршруты.
- Страница (`page`): Внутренняя страница системы.
- Вкладка (`tab`): Тип маршрута, используемый для переключения между вкладками внутри страницы.
- Ссылка (`link`): Внутренняя или внешняя ссылка, которая позволяет напрямую перейти по указанному адресу.

### Добавление маршрута

Чтобы создать новый маршрут, нажмите кнопку «Add new» в правом верхнем углу:

1. Выберите тип маршрута (`Type`).
2. Введите заголовок маршрута (`Title`).
3. Выберите иконку маршрута (`Icon`).
4. Укажите, отображать ли маршрут в меню (`Show in menu`).
5. Настройте включение вкладок страницы (`Enable page tabs`).
6. Для типа «Страница» система автоматически сгенерирует уникальный путь маршрута (`Path`).

### Действия с маршрутами

Каждый элемент маршрута поддерживает следующие действия:

- Add child: Добавить дочерний маршрут.
- Edit: Редактировать конфигурацию маршрута.
- View: Просмотреть страницу маршрута.
- Delete: Удалить маршрут.

### Массовые действия

Верхняя панель инструментов предоставляет следующие функции массовых действий:

- Refresh: Обновить список маршрутов.
- Delete: Удалить выбранные маршруты.
- Hide in menu: Скрыть выбранные маршруты в меню.
- Show in menu: Показать выбранные маршруты в меню.

### Фильтрация маршрутов

Используйте функцию «Filter» в верхней части, чтобы отфильтровать список маршрутов по вашим потребностям.

:::info{title=Примечание}
Изменение конфигурации маршрутов напрямую влияет на структуру навигационного меню системы. Пожалуйста, действуйте осторожно и убедитесь в правильности настроек маршрутов.
:::
