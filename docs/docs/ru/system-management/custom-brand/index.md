# Настройка бренда

## Введение

По умолчанию название бренда — "NocoBase". Оно отображается в трёх основных местах:

![20240409111100](https://static-docs.nocobase.com/20240409111100.png)

В нижней части страницы входа:

![20240409111254](https://static-docs.nocobase.com/20240409111254.png)

Позицию 1 можно настроить с помощью опции "Логотип" в разделе [Системные настройки](/system-management/system-settings/). Позиции 2 и 3 настраиваются с помощью этого плагина.

## Инструкция по использованию

После активации плагина нажмите на меню "Настройка бренда" в правом верхнем углу страницы, чтобы перейти на страницу конфигурации.

![20240409113640](https://static-docs.nocobase.com/20240409113640.png)

- **Бренд**: Используется для настройки текста в нижней части страницы входа (позиция 3).
- **О нас**: Используется для настройки содержимого меню в правом верхнем углу (как показано на изображении ниже).

![20240409113915](https://static-docs.nocobase.com/20240409113915.png)

Конфигурация поддерживает формат HTML для гибкой настройки содержимого. Если вам нужно внести простые текстовые изменения, вы можете отредактировать существующее содержимое напрямую, как показано ниже:

![20240409114740](https://static-docs.nocobase.com/20240409114740.png)
