# Пользовательский бренд

## Введение

Бренд по умолчанию — "NocoBase". В основном он отображается в следующих трёх местах:

![20240409111100](https://static-docs.nocobase.com/20240409111100.png)

Внизу страницы входа:

![20240409111254](https://static-docs.nocobase.com/20240409111254.png)

Позицию 1 можно настроить через параметр «Логотип» в [Настройках системы](/system-management/system-settings/). Позиции 2 и 3 настраиваются этим плагином.

## Инструкции по использованию

После включения плагина нажмите пункт меню «Пользовательский бренд» в правом верхнем углу страницы, чтобы открыть страницу настройки.

![20240409113640](https://static-docs.nocobase.com/20240409113640.png)

- **Бренд**: используется для настройки текста внизу страницы входа (позиция 3)
- **О системе**: используется для настройки содержимого меню в правом верхнем углу (как показано на изображении ниже)

![20240409113915](https://static-docs.nocobase.com/20240409113915.png)

Настройка поддерживает формат HTML для гибкой настройки содержимого. Если нужны только простые текстовые изменения, можно напрямую изменить существующее содержимое, как показано ниже:

![20240409114740](https://static-docs.nocobase.com/20240409114740.png)
