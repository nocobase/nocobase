# Настройки языка

В разделе «Настройки системы» можно настроить глобальную языковую среду системы через параметр «Включённые языки». Первый язык в списке будет использоваться как язык по умолчанию.

![20240324192005](https://static-docs.nocobase.com/20240324192005.png)

Если включено несколько языков, пользователи могут задать предпочитаемый язык в «Личном кабинете» в правом верхнем углу страницы.

![20240324192509](https://static-docs.nocobase.com/20240324192509.png)

Если для используемой языковой среды недостаточно переводов, можно использовать плагин [Управление локализацией](/system-management/localization/) для добавления или редактирования переводов.

![20240324193055](https://static-docs.nocobase.com/20240324193055.png)
