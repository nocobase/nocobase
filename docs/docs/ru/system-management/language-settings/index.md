# Настройки языка

В «Системных настройках» вы можете настроить глобальную языковую среду системы, выбрав «Включенные языки». Первый язык в списке будет использоваться по умолчанию.

![20240324192005](https://static-docs.nocobase.com/20240324192005.png)

Если включено несколько языков, пользователи могут установить свои личные языковые предпочтения в Личном кабинете (в правом верхнем углу страницы).

![20240324192509](https://static-docs.nocobase.com/20240324192509.png)

Если в используемой языковой среде отсутствуют переводы, вы можете использовать плагин «[Управление локализацией](/system-management/localization/)» для добавления или редактирования переводов.

![20240324193055](https://static-docs.nocobase.com/20240324193055.png)
