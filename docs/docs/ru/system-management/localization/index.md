# Управление локализацией

## Введение

Плагин «Управление локализацией» предназначен для управления и реализации ресурсов локализации NocoBase. С его помощью вы можете переводить системные меню, коллекции, поля и все плагины, чтобы адаптировать систему к языку и культуре конкретных регионов.

Если вы хотите внести стандартные переводы системы и официальных плагинов в NocoBase, см. [Вклад в переводы](/get-started/translations).

## Установка

Этот плагин является встроенным и не требует дополнительной установки.

### Синхронизация записей перевода

<img src="https://static-docs.nocobase.com/202404202134850.png"/>

В настоящее время можно синхронизировать следующее содержимое:

- Локальные языковые пакеты для системы и плагинов
- Заголовки коллекций, заголовки полей и метки опций полей
- Заголовки меню

После синхронизации система отобразит все доступные для перевода записи для текущего языка.

<img src="https://static-docs.nocobase.com/202404202136567.png"/>

:::info{title=Примечание}
В разных модулях могут встречаться одинаковые исходные записи, которые необходимо переводить по отдельности.
:::

Если переводы встроенных записей системы или плагинов были изменены вручную или перезаписаны AI-переводом, при синхронизации выберите `Сбросить переводы встроенных системных записей`. После синхронизации система перезапишет существующие переводы встроенных записей текущего языка переводами из встроенного языкового пакета, чтобы восстановить значения по умолчанию.

### Автоматическое создание записей

При редактировании страницы пользовательский текст в различных блоках будет автоматически создавать соответствующие записи и синхронно генерировать перевод для текущего языка.

![](https://static-docs.nocobase.com/Localization-NocoBase-02-12-2026_08_39_AM.png)

:::info{title=Примечание}
При определении текста в коде необходимо вручную указать ns (namespace), например: ${ctx.i18n.t('My custom js block', { ns: 'lm-flow-engine' })}

### Редактирование содержимого перевода

<img src="https://static-docs.nocobase.com/202404202142836.png"/>

Столбец перевода поддерживает быстрое редактирование. Можно нажать ячейку перевода в таблице и изменить ее напрямую, нажать Enter или убрать фокус с поля для сохранения, а `Esc` — для отмены изменения. Чтобы просмотреть исходный текст, модуль или длинный перевод, можно по-прежнему использовать кнопку редактирования в действиях строки и открыть редактор в выдвижной панели.

### Использование AI-перевода

Управление локализацией поддерживает перевод записей с помощью AI-сотрудника Lina. После включения AI-сотрудников и настройки сервиса модели вы можете использовать AI-перевод на странице управления локализацией для пакетного создания переводов для текущего языка.

![](https://static-docs.nocobase.com/202605121152196.png)

Поддерживаемые области перевода:

- **Полный перевод**: переводит все записи текущего языка и перезаписывает существующие переводы.
- **Инкрементальный перевод**: переводит только записи, у которых еще нет перевода в текущем языке. Для встроенных записей перевод также считается существующим, если он уже есть в системном или плагинном языковом пакете целевого языка.
- **Перевод выбранных элементов**: выберите записи в таблице и переведите только выбранное содержимое.

![](https://static-docs.nocobase.com/202605191341968.png)

При создании задачи полного или инкрементального перевода можно выбрать область перевода в диалоге подтверждения:

- **Все**: обработать все записи, соответствующие условиям текущей задачи.
- **Встроенные записи**: системные и плагинные записи.
- **Пользовательские записи**: названия маршрутов, коллекций и полей, а также UI-контент.

Диалог подтверждения также поддерживает настройку языков справочного перевода. Для полного и инкрементального перевода отдельно настраиваются язык по умолчанию и резервный язык для встроенных и пользовательских записей. Перевод выбранных элементов показывает только одну общую настройку справочных языков.

AI-перевод создает фоновую задачу. Во время выполнения можно отслеживать прогресс. После завершения переводы записываются в соответствующий язык, но их все равно следует проверить и исправить с учетом фактического контекста.

Полное руководство см. в разделе [AI-сотрудник - Lina](/ai-employees/built-in/lina).

:::warning{title=Примечание}
Переводы, созданные AI, могут содержать смысловые отклонения, несогласованную терминологию или недостаточное понимание контекста. Перед публикацией вручную проверьте важные страницы, бизнес-термины и тексты, видимые пользователям.
:::
