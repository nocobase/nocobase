# Управление плагинами

## Менеджер плагинов

Нажмите на иконку менеджера плагинов в правом верхнем углу страницы, чтобы перейти на страницу менеджера плагинов. На этой странице вы можете просматривать, устанавливать, включать или отключать плагины.

![2024-04-10_08-47-45](https://static-docs.nocobase.com/2024-04-10_08-47-45.png)

## Центр настроек плагинов

Плагины обычно имеют свои собственные страницы настроек. Нажмите на иконку настроек в правом верхнем углу страницы, чтобы просмотреть меню настроек для всех плагинов. Нажмите на соответствующий пункт меню, чтобы настроить плагин.
