# Управление плагинами

## Менеджер плагинов

Нажмите на значок менеджера плагинов в правом верхнем углу страницы, чтобы открыть страницу менеджера плагинов. На этой странице можно просматривать, устанавливать, включать и отключать плагины.

![2024-04-10_08-47-45](https://static-docs.nocobase.com/2024-04-10_08-47-45.png)

## Центр настроек плагинов

У плагинов обычно есть собственные страницы настроек. Нажмите значок настроек в правом верхнем углу, чтобы увидеть меню настроек всех плагинов. Нажмите соответствующий пункт меню, чтобы настроить плагин.
