# Редактор тем

> Текущая функция тем реализована на базе Ant Design 5.x. Перед прочтением этого документа рекомендуем ознакомиться с концепцией [настройки тем](https://ant.design/docs/react/customize-theme).

## Введение

Плагин "Редактор тем" используется для изменения стилей всего интерфейса. В настоящее время он позволяет редактировать глобальные [SeedToken](https://ant.design/docs/react/customize-theme#seedtoken), [MapToken](https://ant.design/docs/react/customize-theme#maptoken) и [AliasToken](https://ant.design/docs/react/customize-theme#aliastoken), а также переключаться на `тёмный режим` (Dark Mode) и `компактный режим` (Compact Mode). В будущем может быть добавлена поддержка настройки тем на [уровне компонентов](https://ant.design/docs/react/customize-theme#component-level-customization).

## Инструкции по использованию

### Включение плагина "Редактор тем"

Сначала обновите NocoBase до последней версии (v0.11.1 или выше). Затем на странице управления плагинами найдите карточку `Редактор тем`, нажмите кнопку `Включить` в правом нижнем углу карточки и дождитесь обновления страницы.

![20240409132838](https://static-docs.nocobase.com/20240409132838.png)

### Переход на страницу настройки тем

После включения плагина нажмите кнопку настроек в левом нижнем углу карточки, чтобы перейти на страницу редактирования тем. По умолчанию доступны четыре варианта тем: `Тема по умолчанию`, `Тёмная тема`, `Компактная тема` и `Компактная тёмная тема`.

![20240409133020](https://static-docs.nocobase.com/20240409133020.png)

### Добавление новой темы

Нажмите кнопку `Добавить новую тему` и выберите `Создать совершенно новую тему`. Справа на странице появится редактор тем, где вы сможете изменить `Цвета`, `Размеры`, `Стили` и другие параметры. После завершения редактирования введите название темы и нажмите "Сохранить", чтобы создать новую тему.

![20240409133147](https://static-docs.nocobase.com/20240409133147.png)

### Применение новой темы

Наведите курсор мыши на правый верхний угол страницы, чтобы увидеть переключатель тем. Нажмите на него, чтобы выбрать другую тему, например, только что созданную вами.

![20240409133247](https://static-docs.nocobase.com/20240409133247.png)

### Редактирование существующей темы

Нажмите кнопку `Редактировать` в левом нижнем углу карточки. Справа на странице появится редактор тем (аналогичный тому, что используется при добавлении новой темы). После завершения редактирования нажмите "Сохранить", чтобы применить изменения к теме.

![20240409134413](https://static-docs.nocobase.com/20240409134413.png)

### Настройка тем, доступных для выбора пользователем

Новые темы по умолчанию доступны для выбора пользователями. Если вы не хотите, чтобы пользователи могли переключаться на определённую тему, отключите переключатель `Доступно для выбора пользователем` в правом нижнем углу карточки темы. В этом случае пользователи не смогут выбрать эту тему.

![20240409133331](https://static-docs.nocobase.com/20240409133331.png)

### Установка темы по умолчанию

Изначально темой по умолчанию является `Тема по умолчанию`. Чтобы установить определённую тему как тему по умолчанию, включите переключатель `Тема по умолчанию` в правом нижнем углу карточки этой темы. Таким образом, пользователи будут видеть эту тему при первом открытии страницы. Обратите внимание: тему по умолчанию удалить нельзя.

![20240409133409](https://static-docs.nocobase.com/20240409133409.png)

### Удаление темы

Нажмите кнопку `Удалить` под карточкой, затем подтвердите действие во всплывающем диалоговом окне, чтобы удалить тему.

![20240409133435](https://static-docs.nocobase.com/20240409133435.png)
