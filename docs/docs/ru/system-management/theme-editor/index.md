# Редактор тем

> Текущая функция тем реализована на базе Ant Design 5.x. Перед чтением этого документа рекомендуется ознакомиться с концепцией [настройки темы](https://ant.design/docs/react/customize-theme).

## Введение

Плагин «Редактор тем» используется для изменения стилей всей страницы интерфейса. Сейчас поддерживается редактирование глобальных [SeedToken](https://ant.design/docs/react/customize-theme#seedtoken), [MapToken](https://ant.design/docs/react/customize-theme#maptoken) и [AliasToken](https://ant.design/docs/react/customize-theme#aliastoken), а также включение [переключателя](https://ant.design/docs/react/customize-theme#use-preset-algorithms) тёмного и компактного режимов. В будущем возможно появится поддержка [настройки темы на уровне компонентов](https://ant.design/docs/react/customize-theme#component-level-customization).

## Инструкции по использованию

### Включение плагина «Редактор тем»

Сначала обновите NocoBase до последней версии (v0.11.1 или выше). Затем найдите карточку «Редактор тем» на странице менеджера плагинов. Нажмите кнопку «Включить» в правом нижнем углу карточки и дождитесь обновления страницы.

![20240409132838](https://static-docs.nocobase.com/20240409132838.png)

### Переход на страницу настройки темы

После включения плагина нажмите кнопку настроек в левом нижнем углу карточки, чтобы перейти на страницу редактирования тем. По умолчанию есть четыре темы: «Тема по умолчанию», «Тёмная тема», «Компактная тема» и «Компактная тёмная тема».

![20240409133020](https://static-docs.nocobase.com/20240409133020.png)

### Добавление новой темы

Нажмите кнопку «Добавить новую тему» и выберите «Создать совершенно новую тему». Справа откроется редактор темы, где можно редактировать цвета, размеры, стили и другие параметры. После редактирования введите название темы и нажмите «Сохранить», чтобы завершить создание темы.

![20240409133147](https://static-docs.nocobase.com/20240409133147.png)

### Применение новой темы

Переместите курсор в правый верхний угол страницы, чтобы увидеть переключатель тем. Нажмите его, чтобы переключиться на другую тему, например на недавно добавленную.

![20240409133247](https://static-docs.nocobase.com/20240409133247.png)

### Редактирование существующей темы

Нажмите кнопку «Редактировать» в левом нижнем углу карточки. Справа откроется редактор темы (как и при создании новой темы). После редактирования нажмите «Сохранить», чтобы завершить изменение темы.

![20240409134413](https://static-docs.nocobase.com/20240409134413.png)

### Настройка тем, доступных пользователю

Новые темы по умолчанию доступны для переключения пользователями. Если вы не хотите, чтобы пользователи переключались на конкретную тему, выключите переключатель «Доступна пользователю» в правом нижнем углу карточки темы — тогда эта тема не будет доступна для выбора.

![20240409133331](https://static-docs.nocobase.com/20240409133331.png)

### Установка темы по умолчанию

Изначально тема по умолчанию — «Тема по умолчанию». Чтобы сделать другую тему темой по умолчанию, включите переключатель «Тема по умолчанию» в правом нижнем углу карточки. Тогда при первом открытии страницы пользователь увидит именно эту тему. Примечание: тему по умолчанию нельзя удалить.

![20240409133409](https://static-docs.nocobase.com/20240409133409.png)

### Удаление темы

Нажмите кнопку «Удалить» под карточкой, затем подтвердите действие во всплывающем окне.

![20240409133435](https://static-docs.nocobase.com/20240409133435.png)
