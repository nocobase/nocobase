# Управление источниками данных

## Введение

NocoBase предлагает плагин для управления источниками данных, который позволяет управлять источниками данных и их коллекциями. Плагин управления источниками данных предоставляет лишь интерфейс для их администрирования, но сам по себе не обеспечивает подключение к ним. Его необходимо использовать в сочетании с различными плагинами источников данных. В настоящее время поддерживаются следующие источники данных:

- [Основная база данных](/data-sources/data-source-main): Главная база данных NocoBase, поддерживающая реляционные базы данных, такие как MySQL, PostgreSQL и MariaDB.
- [Внешний MySQL](/data-sources/data-source-external-mysql): Использование внешней базы данных MySQL в качестве источника данных.
- [Внешний MariaDB](/data-sources/data-source-external-mariadb): Использование внешней базы данных MariaDB в качестве источника данных.
- [Внешний PostgreSQL](/data-sources/data-source-external-postgres): Использование внешней базы данных PostgreSQL в качестве источника данных.
- [Внешний MSSQL](/data-sources/data-source-external-mssql): Использование внешней базы данных MSSQL (SQL Server) в качестве источника данных.
- [Внешний Oracle](/data-sources/data-source-external-oracle): Использование внешней базы данных Oracle в качестве источника данных.

Кроме того, с помощью плагинов можно расширить поддержку других типов источников данных, включая распространённые виды баз данных или платформы, предоставляющие API (SDK).

## Установка

Это встроенный плагин, отдельная установка не требуется.

## Инструкции по использованию

При инициализации и установке приложения по умолчанию предоставляется источник данных для хранения данных NocoBase, известный как основная база данных. Подробнее см. в документации по [Основной базе данных](/data-sources/data-source-main/).

### Внешние источники данных
