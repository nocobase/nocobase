# Управление источниками данных

## Введение

NocoBase предоставляет плагин управления источниками данных для управления источниками данных и их таблицами. Плагин управления источниками данных предоставляет только интерфейс управления всеми источниками данных и не обеспечивает возможность подключения к ним. Для работы он должен использоваться вместе с различными плагинами источников данных. В настоящее время поддерживаются следующие источники данных:

- [Основная база данных](/data-sources/data-source-main/): основная база данных NocoBase, поддерживаются MySQL, PostgreSQL, MariaDB, KingbaseES и OceanBase.
- [Внешний PostgreSQL](/data-sources/data-source-external-postgres/): внешняя база данных PostgreSQL используется в качестве источника данных.
- [Внешний MySQL](/data-sources/data-source-external-mysql/): внешняя база данных MySQL используется в качестве источника данных.
- [Внешний MariaDB](/data-sources/data-source-external-mariadb/): внешняя база данных MariaDB используется в качестве источника данных.
- [Внешний MSSQL](/data-sources/data-source-external-mssql/): внешняя база данных MSSQL (SQL Server) используется в качестве источника данных.
- [Внешний KingbaseES](/data-sources/data-source-kingbase/): внешняя база данных KingbaseES используется в качестве источника данных.
- [Внешний OceanBase](/data-sources/external/oceanbase): внешняя база данных OceanBase используется в качестве источника данных.
- [Внешний Oracle](/data-sources/data-source-external-oracle/): внешняя база данных Oracle используется в качестве источника данных.
- [Внешний ClickHouse](/data-sources/external/clickhouse): внешняя база данных ClickHouse используется в качестве источника данных, обычно для запросов, статистики и отображения отчётов.
- [Внешний Doris](/data-sources/external/doris): внешняя база данных Doris используется в качестве источника данных, обычно для запросов, статистики и отображения отчётов.
- [Источник данных REST API](/data-sources/data-source-rest-api/): подключение данных из источников REST API к NocoBase.
- [Внешний NocoBase](/data-sources/data-source-external-nocobase/): другое приложение NocoBase используется в качестве внешнего источника данных через удалённый API NocoBase.

Кроме того, с помощью плагинов можно расширить набор поддерживаемых типов. Это могут быть распространённые базы данных различных типов, а также платформы, предоставляющие API (SDK).

## Установка

Встроенный плагин, отдельная установка не требуется.

## Инструкция по использованию

При первоначальной установке приложения по умолчанию предоставляется источник данных для хранения данных NocoBase, который называется основной базой данных. Подробнее см. в документации [Основная база данных](/data-sources/data-source-main/index.md).

### Внешние источники данных

Поддерживается использование внешних баз данных в качестве источников данных. Подробнее см. в документации [Внешняя база данных / Введение](/data-sources/data-source-manager/external-database.md).
