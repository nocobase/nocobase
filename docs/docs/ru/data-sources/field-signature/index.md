# Поле таблицы данных: рукописная подпись

## Введение

Поле «Рукописная подпись» позволяет пользователям рисовать подпись на холсте мышью или на сенсорном экране. После сохранения изображение подписи записывается в выбранную **таблицу файлов**, при этом повторно используется процесс загрузки и хранения файлов из **Менеджера файлов**.

## Установка

1. Убедитесь, что текущая среда — **Pro и выше** и лицензия активна.
2. Откройте **Менеджер плагинов**, найдите **Поле таблицы данных: рукописная подпись** (`@nocobase/plugin-field-signature`) и включите его.
3. Убедитесь, что **Менеджер файлов** (`@nocobase/plugin-file-manager`) включён. Поле «Рукописная подпись» зависит от него: оно использует таблицу файлов, загрузку и хранилище. Если плагин выключен, изображение подписи не будет сохранено.

## Инструкция по использованию

### Добавление поля

В **Источник данных** → выберите таблицу данных → **Настроить поля** → **Добавить поле** → в группе мультимедиа выберите **Рукописная подпись**.

### Конфигурация поля

- **Таблица файлов**: обязательно. Выберите таблицу файлов для хранения файлов (например, `attachments`), туда будет сохраняться изображение подписи.
- Конкретные настройки хранилища и правила загрузки изображения подписи определяются выбранной таблицей файлов.

### Конфигурация интерфейса

- После добавления поля «Рукописная подпись» в форму в настройках интерфейса поля можно настроить **параметры подписи**: цвет штриха, цвет фона, ширину и высоту холста подписи, а также ширину и высоту миниатюры.
- В сценариях отображения только для чтения дополнительно можно регулировать ширину и высоту миниатюры подписи, чтобы управлять размером её отображения.

### Действия в интерфейсе

- Нажмите по области поля, чтобы открыть холст подписи; после рисования подтвердите ввод — изображение будет загружено и привязано к соответствующей записи файла.
- На устройствах с маленьким экраном можно использовать горизонтальный/полноэкранный интерфейс подписи, удобный для рисования.
