# Блок календаря

## Введение

Блок календаря отображает события и данные, связанные с датами, в виде календаря и подходит для таких сценариев, как планирование встреч и мероприятий.

## Установка

Встроенный плагин, установка не требуется.

## Добавление блока

![20250403220300](https://static-docs.nocobase.com/20250403220300.png)

1. Поле заголовка: информация, отображаемая на полосе календаря; в настоящее время поддерживаются поля типов `input`, `select`, `phone`, `email`, `radioGroup`,`sequence` и другие; поддерживаемые блоком календаря типы полей можно расширить с помощью плагинов.
2. Время начала: время начала задачи;
3. Время окончания: время окончания задачи;

### Поле цвета фона

:::info{title=Подсказка}
Требуется версия NocoBase v1.4.0-beta или выше.
:::

Этот параметр можно использовать для настройки цвета фона событий календаря. Используйте его следующим образом:

1. В таблице данных календаря должно быть поле типа **Раскрывающийся список с одиночным выбором (Single select)** или **Группа переключателей (Radio group)**, для которого необходимо настроить цвет.
2. Затем вернитесь к интерфейсу настройки блока календаря и в поле **Поле цвета фона** выберите поле, для которого только что был настроен цвет.
3. Наконец, попробуйте выбрать цвет для события календаря и нажмите кнопку отправки — вы увидите, что цвет применился.

![20240914192017_rec_](https://static-docs.nocobase.com/20240914192017_rec_.gif)

### Первый день недели

> Поддерживается в версии v1.7.7 и выше

Блок календаря поддерживает настройку первого дня недели: в качестве первого дня можно выбрать **воскресенье** или **понедельник**.
По умолчанию первым днем недели является **понедельник**, что позволяет пользователям настраивать отображение календаря с учетом региональных привычек и лучше адаптировать его к реальным потребностям.

### Сегодня

Кнопка «Сегодня» в блоке календаря предоставляет удобную функцию навигации и позволяет пользователям быстро вернуться на страницу календаря с текущей датой после перехода к другим датам.
