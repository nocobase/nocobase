# Блок «Календарь»

## Введение

Блок «Календарь» позволяет отображать события и данные, связанные с датами, в удобном календарном формате. Он идеально подходит для планирования встреч, мероприятий и эффективной организации вашего времени.

## Установка

Этот плагин является встроенным, поэтому дополнительная установка не требуется.

## Добавление блока

![20250403220300](https://static-docs.nocobase.com/20250403220300.png)

1. Поле заголовка: Используется для отображения информации на полосах календаря. В настоящее время поддерживаются поля типов `input`, `select`, `phone`, `email`, `radioGroup`, `sequence` и другие. Вы можете расширить список поддерживаемых типов полей заголовка для блока «Календарь» с помощью плагинов.
2. Время начала: Указывает время начала задачи.
3. Время окончания: Указывает время окончания задачи.

### Поле для цвета фона

:::info{title=Подсказка}
Требуется версия NocoBase v1.4.0-beta или выше.
:::

Эта опция позволяет настроить цвет фона для событий календаря. Вот как её использовать:

1. В таблице данных календаря должно быть поле типа **«Выпадающий список с одним выбором» (Single select)** или **«Группа переключателей» (Radio group)**, и для этого поля должны быть настроены цвета.
2. Затем вернитесь к интерфейсу настройки блока «Календарь» и в поле **«Цвет фона»** выберите то поле, для которого вы только что настроили цвета.
3. Наконец, вы можете попробовать выбрать цвет для события календаря, затем нажать «Отправить» и убедиться, что цвет применился.

![20240914192017_rec_](https://static-docs.nocobase.com/20240914192017_rec_.gif)

### Начало недели

> Поддерживается в версии v1.7.7 и выше

Блок «Календарь» позволяет настроить день начала недели, выбрав **воскресенье** или **понедельник** в качестве первого дня недели.
По умолчанию неделя начинается с **понедельника**, что позволяет пользователям настраивать отображение календаря в соответствии с региональными привычками для более удобного использования.
