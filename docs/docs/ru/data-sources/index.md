# Обзор

Моделирование данных — это ключевой этап проектирования базы данных. Он включает глубокий анализ и абстрагирование различных типов данных реального мира и взаимосвязей между ними. В ходе этого процесса мы стремимся выявить внутренние связи между данными и формализованно описать их с помощью модели данных, заложив основу для структуры базы данных информационной системы. NocoBase — это платформа, управляемая моделью данных, со следующими особенностями:

## Поддержка подключения данных из различных источников

Источниками данных NocoBase могут быть распространённые базы данных, платформы API (SDK) и файлы.

![20240512085558](https://static-docs.nocobase.com/20240512085558.png)

NocoBase предоставляет [плагин управления источниками данных](./data-source-manager/index.md), предназначенный для управления источниками данных и их таблицами. Плагин управления источниками данных предоставляет только интерфейс управления всеми источниками данных и не обеспечивает возможность подключения к ним. Для работы его необходимо использовать совместно с плагинами различных источников данных. В настоящее время поддерживаются следующие источники данных:

- [Основной источник данных](./data-source-main/index.md): основная база данных NocoBase; поддерживаются PostgreSQL, MySQL, MariaDB, KingbaseES и OceanBase.
- [Внешний PostgreSQL](./data-source-external-postgres/index.md): подключение к существующей базе данных PostgreSQL.
- [Внешний MySQL](./data-source-external-mysql/index.md): подключение к существующей базе данных MySQL.
- [Внешний MariaDB](./data-source-external-mariadb/index.md): подключение к существующей базе данных MariaDB.
- [Внешний MSSQL](./data-source-external-mssql/index.md): подключение к существующей базе данных SQL Server.
- [Внешний KingbaseES](./data-source-kingbase/index.md): подключение к существующей базе данных KingbaseES.
- [Внешний OceanBase](./external/oceanbase.md): подключение к существующей базе данных OceanBase.
- [Внешний Oracle](./data-source-external-oracle/index.md): подключение к существующей базе данных Oracle.
- [Внешний ClickHouse](./external/clickhouse.md): подключение к существующей базе данных ClickHouse.
- [Внешний Doris](./external/doris.md): подключение к существующей базе данных Doris.
- [Источник данных REST API](./data-source-rest-api/index.md): отображение REST API сторонней системы в качестве источника данных.
- [Внешний источник данных NocoBase](./data-source-external-nocobase/index.md): подключение к таблицам данных в другом приложении NocoBase.

## Предоставляет разнообразные инструменты моделирования данных

**Простой интерфейс управления таблицами данных**: используется для создания различных моделей (таблиц данных) или подключения к существующим моделям (таблицам данных).

![20240512090751](https://static-docs.nocobase.com/20240512090751.png)

**Визуальный интерфейс, аналогичный ER-диаграмме**: используется для извлечения сущностей и связей между ними из пользовательских и бизнес-требований. Он предоставляет наглядный и понятный способ описания модели данных. ER-диаграмма помогает лучше понять основные сущности данных в системе и связи между ними.

![20240512091042](https://static-docs.nocobase.com/20240410075906.png)
