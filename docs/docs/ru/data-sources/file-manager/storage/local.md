# Локальное хранилище

Загруженные файлы будут сохраняться в локальной директории на сервере. Это подходит для сценариев с небольшим объемом загружаемых файлов, управляемых системой, или для экспериментальных целей.

### Путь

Путь представляет собой как относительный путь к файлу на сервере, так и URL-адрес для доступа к нему. Например, значение "`user/avatar`" (без начального и конечного символов "`/`") означает следующее:

1. Относительный путь к загруженному файлу на сервере: `/path/to/nocobase-app/storage/uploads/user/avatar`.
2. Префикс URL-адреса для доступа к файлу: `http://localhost:13000/storage/uploads/user/avatar`.
