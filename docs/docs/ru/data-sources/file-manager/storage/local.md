# Локальное хранилище

Загруженные файлы будут сохраняться в локальном каталоге на жёстком диске сервера. Это подходит для случаев, когда общее количество загружаемых файлов невелико или требуется пробное использование системы.

### Путь

Одновременно задаёт относительный путь хранения файлов на сервере и путь доступа к ним через URL. Например: «`user/avatar`» (без начального и конечного «`/`») означает:

1. Относительный путь хранения файлов на сервере при загрузке: `/path/to/nocobase-app/storage/uploads/user/avatar`.
2. Префикс URL-адреса для доступа: `http://localhost:13000/storage/uploads/user/avatar`.
