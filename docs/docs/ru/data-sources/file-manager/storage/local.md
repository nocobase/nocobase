# Локальное хранилище

Загруженные файлы будут сохраняться в локальном каталоге на сервере. Это подходит для небольших или экспериментальных сценариев, где общее количество файлов, управляемых системой, относительно невелико.

### Путь

Путь определяет и относительный путь файла, хранящегося на сервере, и путь доступа по URL. Например, "`user/avatar`" (без начального и конечного "`/`") обозначает:

1. Относительный путь загруженного файла, хранящегося на сервере: `/path/to/nocobase-app/storage/uploads/user/avatar`.
2. Префикс URL для доступа к файлу: `http://localhost:13000/storage/uploads/user/avatar`.
