# Tencent COS

Движок хранения на основе Tencent Cloud COS. Перед использованием подготовьте соответствующую учетную запись и права доступа.

## Параметры

![Пример параметров Tencent COS](https://static-docs.nocobase.com/20240712222125.png)

:::info{title=Hint}
В этом разделе рассматриваются только специфические параметры для движка хранения Tencent Cloud COS. Для общих параметров см. [Общие параметры движка](./index.md#common-engine-parameters).
:::

### Регион

Укажите регион для хранения COS, например: `ap-chengdu`.

:::info{title=Hint}
Информацию о регионе для бакета хранения можно посмотреть в [консоли Tencent Cloud COS](https://console.cloud.tencent.com/cos). Достаточно использовать префикс части региона (без полного доменного имени).
:::

### Идентификатор SecretId

Укажите ID авторизованного ключа доступа Tencent Cloud.

### Секретный ключ SecretKey

Укажите секрет авторизованного ключа доступа Tencent Cloud.

### Бакет

Укажите имя бакета COS, например: `qing-cdn-1234189398`.
