# Aliyun OSS

Движок хранения на основе Aliyun OSS. Заранее подготовьте соответствующие учетные записи и права доступа.

## Конфигурация

![Прмер конфгурации Aliyun OSS](https://static-docs.nocobase.com/20240712220011.png)

:::info{title=Note}
Здесь рассматриваются только специфические параметры для движка хранения Aliyun OSS. Для общих параметров см. [Общие параметры движка](./index.md#common-engine-parameters).
:::

### Регион

Укажите регион для хранения OSS, например: `oss-cn-hangzhou`.

:::info{title=Note}
Информацию о регионе для бакета хранения можно посмотреть в [консоли Aliyun OSS](https://oss.console.aliyun.com/). Достаточно использовать префикс части региона (без полного доменного имени).
:::

### ID ключа доступа (AccessKey ID)

Укажите ID авторизованного ключа доступа Alibaba Cloud.

### Секрет ключа доступа (AccessKey Secret)

Укажите секрет авторизованного ключа доступа Alibaba Cloud.

### Бакет (Bucket)

Укажите имя бакета OSS.
