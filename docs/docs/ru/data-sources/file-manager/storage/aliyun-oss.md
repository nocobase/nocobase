# Alibaba Cloud OSS

Перед использованием движка хранилища на базе Alibaba Cloud OSS необходимо подготовить соответствующую учётную запись и права доступа.

## Параметры конфигурации

![Пример конфигурации движка хранилища Alibaba Cloud OSS](https://static-docs.nocobase.com/20240712220011.png)

:::info{title=Подсказка}
Здесь описаны только специальные параметры движка хранилища Alibaba Cloud OSS. Общие параметры см. в разделе [Общие параметры движка](./index.md#引擎通用参数).
:::

### Регион

Укажите регион хранилища OSS, например: `oss-cn-hangzhou`.

:::info{title=Подсказка}
Информацию о регионе хранилища можно посмотреть в [консоли Alibaba Cloud OSS](https://oss.console.aliyun.com/). Достаточно указать только префикс региона (полное доменное имя не требуется).
:::

### Идентификатор AccessKey

Укажите идентификатор ключа авторизованного доступа Alibaba Cloud.

### Секрет AccessKey

Укажите секрет ключа авторизованного доступа Alibaba Cloud.

### Бакет

Укажите имя бакета хранилища OSS.
