### Регион

Укажите регион для хранилища OSS, например: `oss-cn-hangzhou`.

:::info{title=Примечание}
Вы можете просмотреть информацию о регионе вашего хранилища в [консоли Aliyun OSS](https://oss.console.aliyun.com/). Достаточно указать только префикс региона (полное доменное имя не требуется).
:::

### AccessKey ID

Введите ID авторизованного ключа доступа Alibaba Cloud.

### AccessKey Secret

Введите Secret авторизованного ключа доступа Alibaba Cloud.

### Корзина

Введите название корзины OSS.
