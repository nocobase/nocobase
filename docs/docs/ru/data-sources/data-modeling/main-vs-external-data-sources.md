# Сравнение основной и внешней баз данных

Различия между основной и внешней базами данных в NocoBase в основном заключаются в следующих четырёх аспектах: поддержка типов баз данных, типов таблиц и типов полей, а также резервное копирование, восстановление и миграция.

## 1. Поддержка типов баз данных

Подробнее см. в разделе [Управление источниками данных](https://docs.nocobase.com/data-sources/data-source-manager)

### Тип базы данных

| Тип базы данных | Поддержка основной базой данных | Поддержка внешней базой данных |
|-----------|-------------|--------------|
| PostgreSQL | ✅ | ✅ |
| MySQL | ✅ | ✅ |
| MariaDB | ✅ | ✅ |
| KingbaseES | ✅ | ✅ |
| MSSQL | ❌ | ✅ |
| Oracle | ❌ | ✅ |

### Управление таблицами данных

| Управление таблицами данных | Поддержка основной базой данных | Поддержка внешней базой данных |
|-----------|-------------|--------------|
| Базовое управление | ✅ | ✅ |
| Визуальное управление | ✅ | ❌ |

## 2. Поддержка типов таблиц данных

Подробнее см. в разделе [Таблицы данных](https://docs.nocobase.com/data-sources/data-modeling/collection)

|-----------|---------|-----------|------|
| Обычная таблица | ✅ | ✅ | Базовая таблица данных |
| Таблица-представление | ✅ | ✅ | Представление источника данных |
| Унаследованная таблица | ✅ | ❌ | Поддерживает наследование моделей данных; доступно только для основного источника данных |
| Таблица файлов | ✅ | ❌ | Поддерживает загрузку файлов; доступно только для основного источника данных |
| Таблица комментариев | ✅ | ❌ | Встроенная система комментариев; доступно только для основного источника данных |
| Таблица календаря | ✅ | ❌ | Таблица данных для представления календаря |
| Таблица выражений | ✅ | ❌ | Поддерживает вычисление формул |
| Древовидная таблица | ✅ | ❌ | Используется для моделирования данных древовидной структуры |

| Таблица подключения к внешним данным | ✅ | ❌ | Таблица подключения к внешнему источнику данных с ограниченной функциональностью |

:::

## 4. Сравнение поддержки резервного копирования и миграции

| Функция | Основная база данных | Внешняя база данных |
|-----|---------|-----------|
| Резервное копирование и восстановление | ✅ | ❌ (требуется самостоятельная обработка) |
| Управление миграциями | ✅ | ❌ (требуется самостоятельная обработка) |

:::info
NocoBase предоставляет возможности резервного копирования, восстановления и миграции структуры основной базы данных. Для внешних баз данных эти операции пользователь должен выполнять самостоятельно с учётом особенностей своей среды базы данных; NocoBase не предоставляет встроенную поддержку.
:::

## Итоговое сравнение

| Параметр сравнения | Основная база данных | Внешняя база данных |
|-------|---------|-----------|
| Типы баз данных | PostgreSQL、MySQL、MariaDB、KingbaseES | PostgreSQL、MySQL、MariaDB、MSSQL、Oracle、KingbaseES |
| Поддержка типов таблиц | Все типы таблиц | Только обычные таблицы и таблицы-представления |
| Поддержка типов полей | Все типы полей | Все типы полей, кроме полей вложений |
| Резервное копирование и миграция | Встроенная поддержка | Требуется самостоятельная обработка |

## Рекомендации

- **Если вы используете NocoBase для создания новой бизнес-системы с нуля**, используйте **основную базу данных** — это позволит использовать все функции NocoBase.
- **Если вы используете NocoBase для подключения к базе данных другой системы и выполнения базовых операций создания, чтения, обновления и удаления данных**, используйте **внешнюю базу данных**.
