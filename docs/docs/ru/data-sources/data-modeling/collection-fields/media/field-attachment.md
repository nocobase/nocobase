# Поле "Вложение"

## Введение

Система включает встроенное поле типа "Вложение", которое позволяет пользователям загружать файлы в пользовательские коллекции.

По сути, поле "Вложение" — это поле связи "многие ко многим", которое указывает на встроенную системную коллекцию файлов "Вложения" (`attachments`). При создании поля "Вложение" в любой коллекции автоматически генерируется промежуточная таблица для связи "многие ко многим". Метаданные загруженных файлов хранятся в коллекции "Вложения", а информация о файлах, на которые ссылается ваша коллекция, связывается через эту промежуточную таблицу.

### Ограничение по MIME-типам

Используется для ограничения типов файлов, разрешенных к загрузке, с помощью синтаксиса [MIME](https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/MIME_types). Например, `image/*` обозначает файлы изображений. Несколько типов можно разделить запятыми, например: `image/*,application/pdf` разрешает загрузку файлов изображений и PDF-файлов.

### Движок хранения

Выберите движок хранения для загружаемых файлов. Если это поле оставить пустым, будет использоваться движок хранения по умолчанию.
