# Поле вложений

## Введение

В системе есть встроенный тип поля «Вложение», который поддерживает загрузку файлов в пользовательских коллекциях.

Под капотом поле вложений — это поле связи Многие-комногим, которое указывает на встроенную системную коллекцию «Вложения» (`attachments`). Когда вы создаете поле вложений в любой коллекции, автоматически генерируется таблица-соединитель Многие-комногим. Метаданные загруженных файлов хранятся в коллекции «Вложения», а информация о файлах, на которую ссылается ваша коллекция, связывается через эту таблицу-соединитель.

### Белый список MIME-типов

Используется, чтобы ограничить типы файлов, разрешенные для загрузки, с помощью синтаксиса [MIME](https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/MIME_types). Например, `image/*` обозначает файлы изображений. Несколько типов можно перечислять через запятые, например `image/*,application/pdf`, что позволяет загружать и изображения, и PDF-файлы.

### Хранилище

Выберите механизм хранения загруженных файлов. Если оставить поле пустым, будет использоваться механизм хранения по умолчанию в системе.
