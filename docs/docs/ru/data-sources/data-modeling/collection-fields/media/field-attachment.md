# Поле вложений

## Описание

В систему встроен тип поля «Вложения», который позволяет пользователям загружать файлы в настраиваемые таблицы данных.

В основе поля вложений лежит поле связи «многие ко многим», ссылающееся на встроенную в систему таблицу файлов «Вложения» (`attachments`). После создания поля вложений в любой таблице данных автоматически создаётся промежуточная таблица связи «многие ко многим» с таблицей вложений. Метаданные загруженных файлов сохраняются в таблице «Вложения», а информация о файлах, используемых в таблице данных, связывается через эту промежуточную таблицу.

### Ограничение типов MIME

Используется для ограничения типов файлов, разрешённых к загрузке. Формат описывается с помощью синтаксиса [MIME](https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/MIME_types). Например: `image/*` обозначает файлы изображений. Несколько типов можно разделить запятыми, например: `image/*,application/pdf` означает, что разрешена загрузка изображений и PDF-файлов.

### Механизм хранения

Выберите механизм хранения для загружаемых файлов. Если оставить поле пустым, будет использоваться механизм хранения системы по умолчанию.
