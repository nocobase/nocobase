# Селектор коллекций

Поле "Селектор коллекций" — это специальный тип поля, который позволяет пользователям выбирать одну или несколько коллекций из предопределенного списка. Этот тип поля очень полезен в сценариях, где вам необходимо динамически ссылаться на различные коллекции, например, при создании настраиваемых отчетов, дашбордов или сложных приложений, которым нужно изменять источники данных в зависимости от выбора пользователя.

## Сценарии использования

### Динамическое формирование отчетов

В модуле формирования отчетов пользователям может потребоваться выбрать коллекцию из нескольких коллекций в качестве источника данных для отчета. Используя Селектор коллекций, пользователи могут легко выбрать нужную коллекцию, и система динамически сформирует отчет на основе выбора пользователя.

### Настраиваемые дашборды

В приложении дашбордов пользователи могут настраивать данные, отображаемые на дашборде. Селектор коллекций позволяет пользователям выбирать интересующие их коллекции, тем самым отображая графики и статистические данные из разных источников данных на дашборде.

### Гибкие инструменты импорта/экспорта данных

При разработке универсального инструмента импорта или экспорта данных вы можете использовать Селектор коллекций, чтобы пользователи могли указать коллекцию для работы. Таким образом, один и тот же инструмент может использоваться для обработки данных из любой коллекции в системе, значительно повышая повторное использование инструмента.
