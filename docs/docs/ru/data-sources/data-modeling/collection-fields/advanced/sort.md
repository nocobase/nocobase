# Селектор коллекций

Поле «Селектор коллекций» — это специальный тип поля, который позволяет пользователям выбирать одну или несколько коллекций из заранее определенного списка. Этот тип поля особенно полезен в сценариях, где нужно динамически ссылаться на разные коллекции, например при создании настраиваемых отчетов, панелей мониторинга или сложных приложений, которым требуется менять источники данных в зависимости от выбора пользователя.

## Сценарии использования

### Генерация динамических отчетов

В модуле генерации отчетов пользователям может понадобиться выбрать коллекцию из нескольких коллекций в качестве источника данных для отчета. Используя селектор коллекций, пользователи могут легко выбрать нужную коллекцию, а система динамически сгенерирует отчет на основе выбора пользователя.

### Настраиваемые панели

В приложении панелей мониторинга пользователи могут настраивать данные, отображаемые на панели. Селектор коллекций позволяет пользователям выбирать интересующие их коллекции, тем самым показывая диаграммы и статистические данные из разных источников данных на панели мониторинга.

### Гибкие инструменты импорта/экспорта данных

При разработке универсального инструмента импорта или экспорта вы можете использовать селектор коллекций, чтобы позволить пользователям указывать коллекцию, с которой нужно работать. Таким образом, тот же инструмент можно использовать для обработки данных из любой коллекции в системе, что значительно повышает повторное использование этого инструмента.
