# Поля коллекции

## Типы интерфейсов полей

NocoBase разделяет поля на следующие категории с точки зрения интерфейса:

![20240512110352](https://static-docs.nocobase.com/20240512110352.png)

## Типы данных полей

Каждый тип интерфейса поля имеет тип данных по умолчанию. Например, для полей с интерфейсом «Число» (Number) тип данных по умолчанию — `double`, но он также может быть `float`, `decimal` и т.д. В настоящее время поддерживаются следующие типы данных:

![20240512103733](https://static-docs.nocobase.com/20240512103733.png)

## Сопоставление типов полей

Процесс добавления новых полей в основную базу данных выглядит так:

1. Выберите тип интерфейса.
2. Настройте необязательный тип данных для текущего интерфейса.

![20240512172416](https://static-docs.nocobase.com/20240512172416.png)

Процесс сопоставления полей для внешних источников данных следующий:

1. Автоматически сопоставьте соответствующий тип данных (`Field type`) и тип пользовательского интерфейса (`Field Interface`) на основе типа поля внешней базы данных.
2. При необходимости измените на более подходящий тип данных и тип интерфейса.

![20240512172759](https://static-docs.nocobase.com/20240512172759.png)
