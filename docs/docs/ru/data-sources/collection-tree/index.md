# Древовидная коллекция

## Введение

Коллекция с древовидной структурой — это шаблон проектирования коллекции данных, используемый для организации иерархических данных. Такая структура повторяет дерево: у каждого элемента данных может быть один или несколько дочерних элементов, а у дочерних элементов, в свою очередь, могут быть свои потомки.
