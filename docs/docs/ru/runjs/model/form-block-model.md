# Модель FormBlockModel

Модель блока формы: тип модели блока для блоков формы (`Form`, `EditForm` и т. д.). В RunJS, когда текущий JS выполняется внутри блока формы, `ctx.blockModel` часто является `FormBlockModel` (или `EditFormModel`). См. [ctx.blockModel](../context/block-model.md) и [ctx.form](../context/form.md) для практического использования.
