# ctx.t()

Вспомогательная функция i18n в RunJS для перевода текстов; использует язык текущего контекста. Применяется для кнопок, заголовков, подсказок и других строк интерфейса.

## Сценарии использования

Доступна во всех окружениях RunJS.

## Тип

```ts
t(key: string, options?: Record<string, any>): string
```

## Параметры

| Параметр | Тип | Описание |
|----------|-----|----------|
| `key` | `string` | Ключ перевода или шаблон с плейсхолдерами (например, `Привет, {{name}}`, `{{count}} строк`) |
| `options` | `object` | Необязательно. Интерполяция (например, `{ name: 'John', count: 5 }`) или параметры локализации (`defaultValue`, `ns`) |

## Возвращаемое значение

- Переведённая строка; если перевода для ключа нет и не задан `defaultValue`, может вернуться сам ключ или интерполированная строка.

## Пространство имён (ns)

В RunJS пространство имён по умолчанию — **`runjs`**. Без `ns` вызов `ctx.t(key)` ищет ключ в пространстве имён `runjs`.

```ts
// По умолчанию ищет ключ в пространстве имен 'runjs'
ctx.t('Submit');  // то же, что ctx.t('Submit', { ns: 'runjs' })

// Поиск ключа в конкретном пространстве имен
ctx.t('Submit', { ns: 'myModule' });

// Последовательный поиск в нескольких пространствах имен (сначала 'runjs', затем 'common')
ctx.t('Save', { ns: ['runjs', 'common'] });
```

## Примеры

### Простой ключ

```ts
ctx.t('Submit');
ctx.t('No data');
```

### С интерполяцией

```ts
const text = ctx.t('Hello {{name}}', { name: ctx.user?.nickname || 'Guest' });
ctx.render(`<div>${text}</div>`);
```

```ts
ctx.message.success(ctx.t('Processed {{count}} rows', { count: rows.length }));
```

### Относительное время и т. п.

```ts
if (minutes < 60) return ctx.t('{{count}} minutes ago', { count: minutes });
if (hours < 24) return ctx.t('{{count}} hours ago', { count: hours });
```

### Явное указание пространства имён

```ts
ctx.t('Hello {{name}}', { name: 'Guest', ns: 'myModule' });
```

## Примечания

- **Плагин локализации**: включите плагин локализации для управления переводами. Отсутствующие ключи можно собирать и добавлять в словарь.
- Интерполяция в стиле i18next: используйте `{{varName}}` в ключе и передавайте значение с тем же именем в `options`.
- Язык берётся из текущего контекста (например, `ctx.i18n.language`, пользовательская локаль).
