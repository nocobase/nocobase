# ctx.setValue()

В контекстах редактируемых полей — **Поле JS**, **Элемент JS** и др. — устанавливает текущее значение поля. Используйте совместно с `ctx.getValue()` для двусторонней привязки к форме.

## Сценарии использования

| Сценарий | Описание |
|----------|----------|
| **Поле JS** | Запись выбора пользователя или вычисленного значения в редактируемые пользовательские поля |
| **Элемент JS** | Обновление текущего значения ячейки в редактируемых элементах таблицы или подтаблицы |
| **JS-столбец таблицы** | Обновление значения поля строки при рендере колонки |

> `ctx.setValue(v)` доступен только в контекстах RunJS с привязкой к форме; в потоке событий, при связывании, JS-блоке и т. п. (без привязки к полю) метод может отсутствовать — используйте опциональную цепочку: `ctx.setValue?.(value)`.

## Тип

```ts
setValue<T = any>(value: T): void;
```

- **Параметр**: `value` — значение, которое нужно записать; тип зависит от типа элемента формы для поля.

## Поведение

- `ctx.setValue(v)` обновляет текущее значение поля в Ant Design Form и запускает связывание и валидацию.
- Если форма ещё не отрендерена или поле не зарегистрировано, вызов может не дать эффекта; используйте `ctx.getValue()` для проверки.

## Примеры

### Двусторонняя привязка с getValue

```tsx
const { Input } = ctx.libs.antd;

const defaultValue = ctx.getValue() ?? '';

ctx.render(
  <Input
    defaultValue={defaultValue}
    onChange={(e) => ctx.setValue(e.target.value)}
  />
);
```

### Задание значения по условию

```ts
const status = ctx.getValue();
if (status == null || status === '') {
  ctx.setValue?.('draft');
}
```

### Запись текущего поля при связывании с другими

```ts
// Синхронное обновление текущего поля при изменении другого поля
const otherValue = ctx.form?.getFieldValue('type');
if (otherValue === 'custom') {
  ctx.setValue?.({ label: 'Пользовательский', value: 'custom' });
}
```

## Примечания

- В контекстах, где редактирование не поддерживается (например, режим деталей поля JS, JS-блок), `ctx.setValue` может быть `undefined`; используйте `ctx.setValue?.(value)`.
- Для полей ассоциаций (m2o, o2m и т. д.) передавайте значение, соответствующее типу поля (например `{ id, [titleField]: label }`); см. конфигурацию поля.
