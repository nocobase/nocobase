# ctx.i18n

Экземпляр i18n текущего контекста, используемый для чтения или переключения языков. **Для перевода текста используйте [ctx.t()](./t.md)**; не используйте `ctx.i18n.t`.

## Области применения

Все среды выполнения RunJS могут использовать `ctx.i18n` (например, JSBlock, JSField, JSItem, JSColumn, рабочий процесс, правила связей и т. д.).

## Определение типов

```typescript
interface i18n: {
  language: string;
  changeLanguage(lng: string): Promise<any>;
}
```

## Основные свойства

| Свойство | Тип | Описание |
|------|------|------|
| `language` | `string` | Код текущего активного языка (например, `zh-CN`, `en-US`) |

## Основные методы

### changeLanguage(lng)

Переключает текущий язык.

| Параметр | Тип | Описание |
|------|------|------|
| `lng` | `string` | Код целевого языка (например, `'en-US'`, `'zh-CN'`) |

**Возвращаемое значение**: `Promise<any>`, разрешается (resolves) после завершения смены языка.

## Примеры

### Чтение текущего языка

```ts
const lang = ctx.i18n.language;
// 'zh-CN' | 'en-US' | ...
if (lang.startsWith('zh')) {
  ctx.render(ctx.t('中文界面'));
} else {
  ctx.render(ctx.t('English UI'));
}
```

### Переключение языка

```ts
// Переключить на английский
await ctx.i18n.changeLanguage('en-US');

// Переключить на китайский
await ctx.i18n.changeLanguage('zh-CN');
```

### Кнопка переключения языка

```ts
const { Button } = ctx.libs.antd;
const isZh = ctx.i18n.language.startsWith('zh');
ctx.render(
  <Button onClick={async () => {
    await ctx.i18n.changeLanguage(isZh ? 'en-US' : 'zh-CN');
  }}>
    {ctx.t(isZh ? 'Switch to English' : '切换到中文')}
  </Button>,
);
```

## Примечания

- **Перевод текста**: Всегда используйте `ctx.t()`, не используйте `ctx.i18n.t`.
