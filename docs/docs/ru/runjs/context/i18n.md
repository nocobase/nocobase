# ctx.i18n

Экземпляр i18n для текущего контекста; используется для чтения и смены языка. **Для любого переводимого текста используйте [ctx.t()](./t.md)**; не используйте `ctx.i18n.t`.

## Сценарии использования

Доступен во всех окружениях RunJS (**JS-блок**, **Поле JS**, **Элемент JS**, **JS-столбец таблицы**, поток событий, связывание и т. д.).

## Тип

```typescript
interface i18n: {
  language: string;
  changeLanguage(lng: string): Promise<any>;
}
```

## Основные свойства

| Свойство | Тип | Описание |
|----------|-----|----------|
| `language` | `string` | Текущий код языка (например, `zh-CN`, `en-US`) |

## Основные методы

### changeLanguage(lng)

Изменяет текущий язык.

| Параметр | Тип | Описание |
|----------|-----|----------|
| `lng` | `string` | Код целевого языка (например, `en-US`, `zh-CN`) |

**Возвращает**: `Promise<any>`, который завершается после смены языка.

## Примеры

### Чтение текущего языка

```ts
const lang = ctx.i18n.language;
// 'zh-CN' | 'en-US' | ...
if (lang.startsWith('zh')) {
  ctx.render(ctx.t('Chinese UI'));
} else {
  ctx.render(ctx.t('English UI'));
}
```

### Смена языка

```ts
// Переключить на английский
await ctx.i18n.changeLanguage('en-US');

// Переключить на китайский
await ctx.i18n.changeLanguage('zh-CN');
```

### Кнопка переключения языка

```ts
const { Button } = ctx.libs.antd;
const isZh = ctx.i18n.language.startsWith('zh');
ctx.render(
  <Button onClick={async () => {
    await ctx.i18n.changeLanguage(isZh ? 'en-US' : 'zh-CN');
  }}>
    {ctx.t(isZh ? 'Switch to English' : 'Switch to Chinese')}
  </Button>,
);
```

## Примечания

- **Переводимый текст**: используйте только `ctx.t()`, а не `ctx.i18n.t`.
