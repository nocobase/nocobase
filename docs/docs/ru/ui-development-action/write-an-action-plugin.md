# Плагин: Рабочий процесс


## Рабочий процесс

Плагин «Рабочий процесс» (Workflow) — это мощный инструмент для автоматизации бизнес-процессов. Он позволяет создавать пользовательские рабочие процессы, которые запускают действия на основе определённых событий, таких как создание, обновление или удаление данных.

### Ключевые особенности

*   **Управление по событиям**: Рабочие процессы запускаются определёнными событиями, например, созданием новой записи в коллекции «Заказы».
*   **Гибкие узлы**: Рабочие процессы состоят из последовательности узлов, каждый из которых выполняет определённую задачу, например, условное ветвление, обработку данных или отправку уведомлений.
*   **Расширяемость**: Новые триггеры и узлы действий можно добавлять через плагины для удовлетворения конкретных бизнес-требований.

### Примеры использования

*   **Обработка заказов**: При создании нового заказа автоматически отправлять клиенту письмо с подтверждением и уведомлять складской отдел о подготовке к отгрузке.
*   **Регистрация пользователей**: После регистрации нового пользователя автоматически отправлять приветственное письмо и добавлять его в список для email-рассылок.
*   **Утверждение контента**: Когда автор отправляет новую статью, автоматически уведомлять редактора для утверждения. После утверждения статья публикуется автоматически.

### Как использовать

1.  **Установите и включите плагин «Рабочий процесс»**: Найдите и установите плагин «Рабочий процесс» в менеджере плагинов NocoBase.
2.  **Создайте новый рабочий процесс**: В меню слева нажмите «Рабочий процесс», а затем — кнопку «Добавить рабочий процесс».
3.  **Настройте триггер**: Выберите триггер для запуска рабочего процесса, например, «Событие коллекции > После создания записи».
4.  **Добавьте и настройте узлы**: Перетащите узлы из списка на холст и настройте их по своему усмотрению. Например, добавьте узел «Отправить email».
5.  **Соедините узлы**: С помощью мыши соедините узлы, чтобы определить последовательность выполнения.
6.  **Сохраните и включите рабочий процесс**: После завершения настройки нажмите кнопку «Сохранить» и включите рабочий процесс.
