# История изменений

На этой странице фиксируется содержимое обновлений каждой версии решения CRM. При каждом выпуске новой резервной копии в верхней части страницы добавляется новая запись.

---

## 2026-04-05

**Файлы резервной копии**:
- [nocobase_crm_v2_backup_260406.nbdata](https://static-docs.nocobase.com/nocobase_crm_v2_backup_260406.nbdata)
- [nocobase_crm_v2_sql_260406.zip](https://static-docs.nocobase.com/nocobase_crm_v2_sql_260406.zip)

**Содержимое обновления**:
- Исправлена ошибка некорректного отображения сводки Summary в таблице
- Завершена настройка прав доступа для ролей
- Добавлена страница глобального поиска (Lookup), позволяющая одновременно искать записи в нескольких таблицах на одной странице
- Исправлена ошибка `plugin-disable-pm-add` not found, возникающая при установке
- Добавлена кнопка назначения Лидов
- Исправлена ошибка, возникающая при клике на фильтр Сделок

---

## 2026-03-27

**Файлы резервной копии**:
- [nocobase_crm_v2_backup_260327.nbdata](https://static-docs.nocobase.com/nocobase_crm_v2_backup_260327.nbdata)
- [nocobase_crm_v2_sql_260327.zip](https://static-docs.nocobase.com/nocobase_crm_v2_sql_260327.zip)

**Содержимое обновления**:

**Оптимизация панелей мониторинга**
- Добавлены панели мониторинга в стиле V1
- Улучшен внешний вид прежних панелей мониторинга
- В графиках и KPI-карточках добавлены события клика: поддержка перехода с параметрами и вложенных всплывающих окон

**Исправления функциональности**
- Исправлена ошибка, при которой два JS Block во вкладке Pipeline раздела Opportunities не могли совместно фильтровать данные

**Доработка коммерческих предложений**
- Добавлен процесс согласования коммерческих предложений
- Дополнены тестовые данные для согласований

**Управление клиентами**
- Добавлена функция объединения Клиентов и соответствующие страницы

**Базовые данные**
- Дополнены данные подразделений

## 2026-02-23

**Файлы резервной копии**:
- nocobase_crm_v2_backup_260223.nbdata
- nocobase_crm_v2_sql_260223.zip

**Содержимое обновления**:
- Первый выпуск CRM 2.0
- 7 базовых модулей: Лиды, Контакты, Клиенты, Сделки, Предложения, Заказы, Активности
- 5 AI-сотрудников: Scout (оценка Лидов), Viz (анализ данных), Ellis (помощник по email), Dex (здоровье клиентов), Lexi (перевод)
- Поддержка нескольких валют и управление обменными курсами
- 15 панелей мониторинга (три ракурса: менеджер по продажам / торговый представитель / руководство)
