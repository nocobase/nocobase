# Управление Лидами

> Лид (Lead) — это начало процесса продаж: первый контакт с каждым потенциальным Клиентом начинается отсюда. В этой главе Вы пройдёте полный жизненный цикл Лида: создание, оценка, фильтрация, ведение, конвертация.

![cn_01-leads](https://yifan2-500g.oss-cn-hangzhou.aliyuncs.com/crm-guide-mermaid/cn_01-leads.png)

## Обзор страницы Лидов

В верхнем меню кликните **Sales → Leads**, чтобы перейти на страницу управления Лидами.

![01-leads-2026-04-02-00-04-18](https://static-docs.nocobase.com/01-leads-2026-04-02-00-04-18.png)

В верхней части страницы расположен ряд **кнопок умной фильтрации**, помогающих быстро переключаться между представлениями:

Первая группа:

| Кнопка | Описание |
|------|------|
| All | Показать все Лиды |
| New | Новый Лид, работа ещё не начата |
| Working | В работе |
| Qualified | Подтверждён как квалифицированный Лид |
| Unqualified | Помечен как неквалифицированный |

Вторая группа:

| Метка | Значение |
|------|------|
| 🔥 Hot | Лиды с AI-оценкой ≥ 75 |
| Сегодня | Лиды, созданные сегодня |
| Текущая неделя | Лиды, созданные на текущей неделе |
| Текущий месяц | Лиды, созданные в текущем месяце |
| Не назначено | Лиды без указанного Owner |
| Крупные предприятия | Лиды от крупных корпоративных Клиентов |


![01-leads-2026-04-02-00-06-19](https://static-docs.nocobase.com/01-leads-2026-04-02-00-06-19.gif)


Таблица позволяет с одного взгляда получить ключевую информацию и содержит составные столбцы:

- **AI-индикатор оценки**: круговой индикатор от 0 до 100, красный (низкий) → жёлтый (средний) → зелёный (высокий) — наглядно отражает качество Лида
- **Составной столбец «Имя + компания»**: имя и название компании отображаются вместе, экономя место
- **Составной столбец «Email + телефон»**: контактные данные видны сразу
- **Столбец относительного времени**: показывает «3 часа назад», «2 дня назад» и т. п.; просроченные Лиды подсвечиваются красным, чтобы Вы своевременно с ними связались

![01-leads-2026-04-02-00-07-04](https://static-docs.nocobase.com/01-leads-2026-04-02-00-07-04.gif)

## Создание Лида

Нажмите кнопку **Add new** над таблицей, чтобы открыть форму создания Лида.

![01-leads-2026-04-02-00-08-08](https://static-docs.nocobase.com/01-leads-2026-04-02-00-08-08.png)

Заполните следующие сведения:

| Поле | Описание | Обязательно |
|------|------|---------|
| Name | Имя Лида | ✅ |
| Company | Компания | Желательно заполнить |
| Email | Адрес email | Желательно заполнить |
| Phone | Номер телефона | Желательно заполнить |
| Source | Источник Лида (например, форма сайта, выставка, рекомендация и т. п.) | Желательно заполнить |
...

### Проверка дубликатов в реальном времени

В процессе заполнения формы система выполняет проверку дубликатов в реальном времени по полям имени, компании, email, телефона, мобильного. Если при вводе обнаружена совпадающая запись:

- **Жёлтое предупреждение**: найдена похожая запись, рекомендуется проверить
- **Красное предупреждение**: найдено полное совпадение, настоятельно рекомендуется сначала проверить существующую запись

![01-leads-2026-04-02-00-11-05](https://static-docs.nocobase.com/01-leads-2026-04-02-00-11-05.png)


Это эффективно предотвращает повторный ввод одного и того же человека.

### AI-заполнение формы

Если у Вас есть текст визитки или запись разговора, не нужно заполнять поля вручную — нажмите кнопку AI, выберите «Заполнение формы», вставьте текст, и AI автоматически извлечёт имя, компанию, email, телефон и заполнит форму одним кликом.

После заполнения нажмите **Submit** для сохранения.

![01-leads-2026-04-02-00-15-14](https://static-docs.nocobase.com/01-leads-2026-04-02-00-15-14.png)

### Автоматическая AI-оценка

После сохранения система автоматически запускает **workflow AI-оценки**. AI комплексно анализирует всю информацию о Лиде и формирует следующие результаты:

| Вывод AI | Описание |
|---------|------|
| Score | Сводная оценка от 0 до 100 |
| Conversion Probability | Прогноз вероятности конвертации |
| NBA (Next Best Action — рекомендованное следующее действие) | Рекомендация AI по сопровождению, например «Рекомендуется позвонить в течение 24 часов» |
| Tags | Автоматически сгенерированные теги, например «Высокий интерес», «Принимающий решения» и т. п. |

![01-leads-2026-04-02-00-15-53](https://static-docs.nocobase.com/01-leads-2026-04-02-00-15-53.png)

> 💡 **Подсказка**: Чем выше AI-оценка, тем качественнее Лид. Рекомендуется в первую очередь работать с Hot-Лидами (≥ 75 баллов), сосредотачивая усилия на Клиентах с наибольшей вероятностью сделки.

## Фильтрация и поиск

Кнопки умной фильтрации в верхней части поддерживают **фильтрацию в реальном времени** — клик срабатывает мгновенно, без перезагрузки страницы.

Несколько типичных сценариев:

- **Утреннее начало работы**: нажмите «Сегодня», чтобы посмотреть Лиды, поступившие сегодня, затем нажмите «Hot» — есть ли среди них высокооценённые Лиды, требующие немедленного внимания
- **Распределение Лидов**: нажмите «Не назначено», чтобы найти Лиды без Owner, и распределите их между коллегами по продажам
- **Анализ результатов**: нажмите «Unqualified», чтобы пересмотреть Лиды, помеченные как неквалифицированные, и проверить, нет ли ошибок в оценке

> 💡 **Подсказка**: Система поддерживает фильтрацию через параметры URL. Например, при переходе на страницу Лидов с параметром `?status=new` страница автоматически выберет фильтр «New». Это удобно при переходе с других страниц.

## Подробности Лида

Кликните любую строку в таблице — откроется всплывающее окно с подробностями. Оно содержит **3 вкладки**:

### Вкладка «Подробности»

Это самая информативная вкладка. Сверху вниз она содержит:

![01-leads-2026-04-02-00-17-36](https://static-docs.nocobase.com/01-leads-2026-04-02-00-17-36.png)

**Перевод этапов и кнопки операций**

В верхней части находятся полоса этапов и кнопки операций (Edit / Convert / Lost / Assign). Полоса этапов:

```
New → Working → Converted / Lost
```

Вы можете **кликнуть по нужному этапу** для перевода статуса Лида. Например, начиная работу, нажмите «Working»; после подтверждения квалификации — «Converted», что запустит процесс конвертации.

![01-leads-2026-04-02-00-23-03](https://static-docs.nocobase.com/01-leads-2026-04-02-00-23-03.png)

Если целевой объект (Клиент, контакт, Сделка) уже существует, найдите его поиском и выберите. Если нет — нажмите кнопку создания справа от поля ввода: откроется форма создания, автоматически заполненная данными Лида.
![01-leads-2026-04-07-00-14-21](https://static-docs.nocobase.com/01-leads-2026-04-07-00-14-21.gif)


При нажатии «Lost» открывается диалог с просьбой указать причину потери — это полезно для последующего анализа.

![01-leads-2026-04-02-00-23-25](https://static-docs.nocobase.com/01-leads-2026-04-02-00-23-25.png)


**Карточка AI-оценки**

Отображает подробности AI-оценки, включая:
- Индикатор AI Score (0–100)
- Conversion Probability (вероятность конвертации)
- Pipeline Days (дни в воронке)
- NBA (рекомендованное следующее действие)

**Область цветных меток**

Цветные метки отображают ключевые свойства: Rating (рейтинг), Status (статус), Source (источник) и т. д.

**Базовая информация и кнопки быстрых активностей**

Информация о компании, контактные данные и другие базовые поля. В этой области также есть набор кнопок быстрых активностей: Log Call (записать звонок), Send Email (отправить email), Schedule (создать встречу) — выполненные действия автоматически связываются с текущим Лидом.

**AI Insights**

Аналитические выводы и рекомендации по сопровождению, сгенерированные AI.

**Область комментариев**

Здесь члены команды могут оставлять сообщения для обсуждения. Все комментарии после конвертации Лида автоматически переносятся в созданную запись Клиента.

![01-leads-2026-04-02-00-24-10](https://static-docs.nocobase.com/01-leads-2026-04-02-00-24-10.png)

### Вкладка «Email»

Отображает все письма, связанные с этим Лидом, что удобно для просмотра истории общения. Поддерживается отправка email прямо отсюда, есть кнопки AI-помощи.

![01-leads-2026-04-02-00-17-57](https://static-docs.nocobase.com/01-leads-2026-04-02-00-17-57.png)

### Вкладка «История изменений»

Фиксирует все изменения полей этого Лида с точностью до «кто и когда изменил какое поле с A на B». Используется для отслеживания и анализа.

## Конвертация Лида

Это **самая важная операция** в управлении Лидами — превращение квалифицированного Лида в Клиента, контакт и Сделку одним кликом.

### Как конвертировать

В окне подробностей Лида кликните по этапу **Converted** на компоненте смены этапов.

![01-leads-2026-04-02-00-26-01](https://static-docs.nocobase.com/01-leads-2026-04-02-00-26-01.png)

### Процесс конвертации

Система автоматически запускает **workflow конвертации Лида**, выполняя за один раз следующие операции:

1. **Создание Клиента (Customer)** — создаётся новая запись Клиента по названию компании из Лида; имя/отрасль/размер/адрес автоматически заполняются из Лида, выполняется проверка дубликатов
2. **Создание контакта (Contact)** — создаётся контакт по имени, email, телефону, должности из Лида и связывается с Клиентом
3. **Создание Сделки (Opportunity)** — создаётся новая запись Сделки; имя/источник/сумма/этап автоматически заполняются из Лида и связываются с Клиентом
4. **Перенос комментариев** — все комментарии Лида автоматически копируются в новые записи
5. **Обновление статуса Лида** — статус Лида помечается как Qualified

### Эффект после конвертации

После конвертации, вернувшись в список Лидов, Вы увидите, что **составной столбец «Имя + компания»** этого Лида превратился в кликабельные ссылки:

- Клик по имени → переход к подробностям контакта
- Клик по названию компании → переход к подробностям Клиента

![01-leads-2026-04-02-00-26-36](https://static-docs.nocobase.com/01-leads-2026-04-02-00-26-36.png)

> 💡 **Подсказка**: Конвертация — необратимая операция. Перед конвертацией убедитесь в точности и полноте данных Лида, особенно названия компании и контактных данных — они напрямую станут начальными данными Клиента и контакта.

## Автоматическое назначение

Когда у Лида не указан Owner, система автоматически запускает **workflow назначения Лида**.

Логика назначения проста: **автоматически назначить тому из менеджеров по продажам, у кого сейчас меньше всего Лидов**, что обеспечивает равномерную нагрузку команды.

Этот workflow срабатывает как при создании, так и при обновлении Лида: если поле Owner пустое, выполняется автоматическое назначение.

> 💡 **Подсказка**: Если Вы хотите назначить Owner вручную, просто отредактируйте поле Owner в подробностях. Ручное назначение перекроет результат автоматического.
