# Тестирование

:::tip

В этой главе будет рассказано, как писать и запускать тесты для фронтенд-части плагинов. Содержание включает:
- Настройку тестового окружения и зависимостей;
- Написание модульных тестов с использованием Vitest и Testing Library;
- Способы симуляции фронтенд-окружения выполнения NocoBase;
- Автоматический запуск тестов в CI.

В следующих версиях мы добавим подробные примеры и лучшие практики.

:::
