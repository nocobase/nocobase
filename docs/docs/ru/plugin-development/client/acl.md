:::tip

В этой главе рассказывается, как использовать ACL (Access Control List- Список контроля доступа) для управления правами во фронтенд-плагинах. Содержание включает:

- Проверку прав;
- Динамический рендеринг контента на основе прав доступа;

В следующих версиях мы добавим подробные примеры и лучшие практики.
