# Обзор

Разработка клиентских плагинов NocoBase предоставляет множество функций и возможностей, которые помогают разработчикам настраивать и расширять фронтенд NocoBase. Ниже представлены основные возможности разработки клиентских плагинов NocoBase и соответствующие разделы:

| :--------------------------- | :------------------------------------------------------------------------ | :---------------------------------------------------------- |
| **Класс плагина**            | Создание и управление клиентскими плагинами, расширение функциональности фронтенда | [плагин](plugin.md)                                         |
| **Маршрутизация**            | Настройка фронтенд-маршрутизации, реализация навигации по страницам и перенаправлений | [маршрутизация](router.md)                                  |

| **Контекст**                 | Получение и использование контекста приложения, доступ к глобальному состоянию и сервисам | [контекст](context.md)                                      |
| **ACL**                      | Реализация контроля доступа на фронтенде, управление разрешениями на доступ к страницам и функциям | [ACL](acl.md)                                               |
