# Обзор

Разработка серверных плагинов NocoBase предоставляет широкий набор возможностей, которые помогают разработчикам настраивать и расширять основные функции NocoBase. Ниже перечислены ключевые возможности и соответствующие главы документации:

| Модуль | Описание | Связанная глава |
| --------------------------- | ------------------------------------------------ | -------------------------------------------- |
| **Класс плагинов** | Создавайте серверные плагины и управляйте ими, расширяйте основные функциональные возможности | [Плагин](plugin.md) |
| **Операции с базой данных** | Предоставляет интерфейсы для операций с базами данных, поддерживая CRUD и управление транзакциями | [База данных](database.md) |
| **Индивидуальные коллекции** | Настраивайте структуры сбора данных в соответствии с потребностями бизнеса для гибкого управления моделями данных | [Коллекции](collections.md) |
| **Совместимость данных обновления плагина** | Гарантирует, что обновления плагинов не влияют на существующие данные, выполняя миграцию данных и обработку совместимости | [Миграция](migration.md) |
| **Управление внешними источниками данных** | Интегрируйте внешние источники данных и управляйте ими для обеспечения взаимодействия данных | [Менеджер источников данных](data-source-manager.md) |
| **Пользовательские API** | Расширьте управление ресурсами API, написав собственные интерфейсы | [Менеджер ресурсов](resource-manager.md) |
| **Управление разрешениями API** | Настройте разрешения API для детального контроля доступа | [Список управления доступом (ACL)](acl.md) |
| **Перехват и фильтрация запросов/ответов** | Добавьте перехватчики запросов и ответов или промежуточное программное обеспечение для выполнения таких задач, как ведение журнала, аутентификация и т. д. | [Контекст](context.md) и [Промежуточное ПО](middleware.md) |
| **Прослушивание событий** | Прослушивать системные события (например, из приложения или базы данных) и запускать соответствующие обработчики | [События](event.md) |
| **Управление кэшем** | Управляйте кешем, чтобы повысить производительность приложений и скорость ответа | [Кэш](cache.md) |
| **Запланированные задачи** | Создавайте запланированные задачи и управляйте ими, например периодическую очистку, синхронизацию данных и т. д. | [Менеджер cron-задач](cron-job-manager.md) |
| **Многоязычная поддержка** | Интеграция многоязычной поддержки для реализации интернационализации и локализации | [Интернационализация (i18n)](i18n.md) |
| **Вывод журнала** | Настройте форматы журналов и методы вывода для расширения возможностей отладки и мониторинга | [Логер](logger.md) |
| **Пользовательские команды** | Расширьте NocoBase CLI, добавив пользовательские команды | [Команда](command.md) |
| **Написание тестовых примеров** | Напишите и запустите тестовые примеры, чтобы обеспечить стабильность и функциональную точность плагина | [Тест](test.md) |
