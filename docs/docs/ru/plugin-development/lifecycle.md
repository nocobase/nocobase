# Жизненный цикл

В этом разделе мы рассмотрим хуки жизненного цикла для плагинов как на стороне сервера, так и на стороне клиента. Это поможет вам правильно регистрировать и освобождать ресурсы.

Вы можете сравнить их с жизненным циклом FlowModel, чтобы выделить общие концепции.

## Рекомендуемое содержание

- Колбэки, которые срабатывают при установке, включении, отключении или удалении плагинов.
- Моменты монтирования, обновления и уничтожения клиентских компонентов.
- Рекомендации по обработке асинхронных задач и ошибок в рамках жизненного цикла.
