# Обзор NocoBase CLI

Если Вы впервые используете этот CLI, не нужно сразу запоминать все команды. Сначала установите приложение через `nb init --ui`, а затем переходите к остальным разделам по сценарию.

## Сначала усвойте главную идею

В NocoBase CLI последующие операции по умолчанию строятся не вокруг «определённого каталога» или «определённого порта», а вокруг **окружения**.

**Окружение** — это набор сведений о подключении и запуске приложения, который CLI запоминает. После успешного сохранения многие последующие команды можно использовать напрямую:

- `nb init` — установить новое приложение и сохранить его как окружение
- `nb env add` — подключить существующее приложение к CLI
- `nb app start`, `nb app logs`, `nb app upgrade` — управлять этим окружением
- `nb backup` — создавать резервные копии и восстанавливать это окружение
- `nb app autostart`, `nb proxy` — дополнить возможности производственного окружения

Сначала запомните эту идею — дальше документация будет читаться намного проще.

## Рекомендуемый путь по умолчанию

Если непонятно, с чего начать, обычно проще всего пройти такой путь:

1. Прочитайте [Установить с помощью CLI (рекомендуется)](./installation/cli.md) и один раз выполните `nb init`.
2. После сохранения приложения как окружения откройте [Управление несколькими окружениями](./operations/multi-environment.md), чтобы проверить текущее окружение, переключить окружение и посмотреть статус.
3. Для ежедневного запуска, остановки, просмотра логов и обновления см. [Управление приложением](./operations/manage-app.md).
4. Перед обновлениями, миграциями или важными изменениями см. [Резервное копирование и восстановление](./operations/backup-restore.md).
5. Когда будете готовы официально вывести систему в сеть, переходите к [Обзору развёртывания производственного окружения](./production/index.md).

Первые три шага покрывают большинство сценариев.

## Когда смотреть справочник команд

Этот набор документов для быстрого старта отвечает на вопрос «что мне сейчас нужно сделать». Если Вы уже знаете, какую команду хотите выполнить, и нужны только полные параметры, переходите сразу в [Справочник команд NocoBase CLI](../api/cli/index.md).

Рекомендуемый порядок:

- Сначала используйте этот обзор и документы быстрого старта, чтобы понять маршрут.
- Затем уточняйте параметры на странице конкретной команды.

Так начать проще, чем сразу читать полное дерево команд.
