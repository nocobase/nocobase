# Часто задаваемые вопросы

## Выбор диаграмм
### Как выбрать подходящую диаграмму?
Ответ: Выбирайте, исходя из ваших данных и целей:
- Для отображения трендов и изменений: линейная диаграмма или диаграмма с областями.
- Для сравнения значений: столбчатая или гистограмма.
- Для отображения структуры и соотношения частей: круговая или кольцевая диаграмма.
- Для корреляции и распределения: точечная диаграмма.
- Для иерархической структуры и прогресса этапов: воронкообразная диаграмма.

Дополнительные типы диаграмм вы можете найти в [примерах ECharts](https://echarts.apache.org/examples).

### Какие типы диаграмм поддерживает NocoBase?
Ответ: В режиме визуальной настройки доступны распространённые типы диаграмм (линейные, с областями, столбчатые, гистограммы, круговые, кольцевые, воронкообразные, точечные и т.д.). В режиме пользовательской настройки вы можете использовать все типы диаграмм ECharts.

## Вопросы по запросам данных
### Совместимы ли режимы визуальной настройки и настройки SQL?
Ответ: Нет, они несовместимы, и их настройки хранятся независимо. Активным будет тот режим настройки, который использовался при последнем сохранении.

## Вопросы по параметрам диаграмм
### Как настроить поля диаграммы?
Ответ: В режиме визуальной настройки выберите соответствующие поля данных в зависимости от типа диаграммы. Например, для линейных или столбчатых диаграмм необходимо настроить поля для осей X и Y, а для круговых диаграмм — поле категории и поле значения.
Рекомендуется сначала выполнить «Запустить запрос», чтобы убедиться, что данные соответствуют вашим ожиданиям. По умолчанию поля диаграммы будут сопоставлены автоматически.

## Вопросы по предварительному просмотру и сохранению
### Нужно ли вручную просматривать изменения после настройки?
Ответ: В режиме визуальной настройки изменения автоматически отображаются в предварительном просмотре. В режимах SQL и пользовательской настройки, чтобы избежать частых обновлений, завершите редактирование и нажмите «Предварительный просмотр» вручную.

### Почему предварительный просмотр диаграммы исчезает после закрытия всплывающего окна?
Ответ: Предварительный просмотр предназначен только для временного ознакомления. После внесения изменений, пожалуйста, сначала сохраните их, а затем закройте окно; несохранённые изменения не будут сохранены.
