# Частые вопросы (FAQ)

## Выбор графика
### Какой график использовать?
Ответ: выбирайте исходя из данных и цели:
- Тренд или изменение: линейный или областной
- Сравнение значений: столбчатый или линейчатый
- Состав: круговой или кольцевой
- Корреляция или распределение: точечный
- Иерархия или прогресс по этапам: воронка

Дополнительные типы графиков- см. [примеры ECharts](https://echarts.apache.org/examples).

### Какие типы графиков поддерживает NocoBase?
Ответ: в визуальном режиме доступны распространённые графики (линейный, областной, столбчатый, линейчатый, круговой, кольцевой, воронка, точечный и т.д.). В пользовательском режиме поддерживаются все типы графиков ECharts.

## Вопросы по запросам данных
### Общие ли конфигурации у визуального и SQL режимов?
Ответ: нет. Их конфигурации хранятся отдельно. Применяется режим, использованный при последнем сохранении.

## Параметры графика
### Как настроить поля графика?
Ответ: в настройках графика выбирайте поля в соответствии с типом графика. Например, для линейного или столбчатого графиков требуются поля X и Y; для кругового — поле категории и поле значения.
Сначала нажмите `Выполнить запрос`, чтобы проверить данные на соответствие ожиданиям; сопоставление полей по умолчанию подбирается автоматически.

## Предпросмотр и сохранение
### Нужно ли вручную запускать предпросмотр изменений?
Ответ: в визуальном режиме изменения предпросматриваются автоматически. В SQL и пользовательском, чтобы избежать частых обновлений, завершите редактирование и нажмите `Предпросмотр` вручную.

### Почему предпросмотр теряется после закрытия диалога?
Ответ: предпросмотр предназначен для временного просмотра. Сохраните изменения перед закрытием; несохранённые изменения не сохраняются.
