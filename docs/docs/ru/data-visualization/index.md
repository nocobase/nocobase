# Обзор

Плагин для визуализации данных NocoBase предоставляет возможности визуального запроса данных и богатый набор компонентов для построения диаграмм. С помощью простой настройки вы можете быстро создавать дашборды, представлять аналитические выводы, а также поддерживать многомерный анализ и отображение данных.

![clipboard-image-1761749573](https://static-docs.nocobase.com/clipboard-image-1761749573.png)

## Основные понятия
- Блок диаграммы: настраиваемый компонент диаграммы на странице, который поддерживает запросы данных, параметры диаграммы и интерактивные события.
- Запрос данных (Конструктор / SQL): Настраивайте визуально с помощью Конструктора или пишите SQL для получения данных.
- Метрики (Measures) и Измерения (Dimensions): Метрики используются для агрегации числовых значений; Измерения группируют данные (например, по дате, категории, региону).
- Сопоставление полей: Сопоставляйте столбцы результатов запроса с основными полями диаграммы, такими как `xField`, `yField`, `seriesField` или `Category / Value`.
- Параметры диаграммы (Базовые / Пользовательские): Базовые параметры настраивают общие свойства визуально; Пользовательские возвращают полный объект `option` ECharts через JS.
- Выполнение запроса: Выполняйте запрос и получайте данные на панели конфигурации; переключайтесь между режимами Table / JSON для просмотра полученных данных.
- Предварительный просмотр и сохранение: Предварительный просмотр является временным; нажатие кнопки «Сохранить» записывает конфигурацию в базу данных и применяет её.
- Контекстные переменные: Повторно используйте контекст страницы, пользователя и фильтров (например, `{{ ctx.user.id }}`) в запросах и конфигурации диаграмм.
- Фильтры страницы и связывание: Блоки фильтров уровня страницы собирают унифицированные условия, автоматически объединяют их в запросы диаграмм и обновляют связанные диаграммы.
- Интерактивные события: Регистрируйте события через `chart.on` для включения подсветки, навигации и детализации (drill-down).

## Установка
Визуализация данных — это встроенный плагин NocoBase, который работает «из коробки» и не требует отдельной установки.
