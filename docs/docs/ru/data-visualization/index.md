# Обзор

Плагин визуализации данных NocoBase предоставляет визуальные запросы данных и большой набор компонентов графиков. С помощью простой конфигурации вы можете быстро создавать дашборды, представлять инсайты и поддерживать многомерный анализ и отображение.

![clipboard-image-1761749573](https://static-docs.nocobase.com/clipboard-image-1761749573.png)

## Базовые понятия
- Блок графика: настраиваемый компонент графика на странице, поддерживающий запрос данных, параметры графика и события взаимодействия.
- Запрос данных (Конструктор/SQL): визуальная настройка через Конструктор или написание SQL для получения данных.
- Показатели и измерения: показатели используются для числовой агрегации; измерения группируют данные (например, дата, категория, регион).
- Сопоставление полей: сопоставление столбцов результата запроса с базовыми полями графика, такими как `Поле X`, `Поле Y`, `Поле серии` или `Категория / Значение`.
- Параметры графика (Базовые/Пользовательские): в Базовых часто используемые свойства настраиваются визуально; в Пользовательских через JS возвращается полный `option` ECharts.
- Запустить запрос: запуск запроса и получение данных в панели конфигурации; режимы Таблица / JSON для просмотра возвращаемых данных.
- Предпросмотр и сохранение: предвариательный просмотр; нажатие Сохранить записывает конфигурацию в базу данных и применяет её.
- Контекстные переменные: переиспользование контекста страницы, пользователя и фильтра (например, `{{ ctx.user.id }}`) в запросах и конфигурации графика.
- Фильтры и связывание: блоки фильтра на уровне страницы собирают единые условия, автоматически объединяют их в запросы графиков и обновляют связанные графики.
- События взаимодействия: регистрируйте события через `chart.on`, чтобы включать выделение, навигацию и детализацию.

## Установка
Визуализация данных — это встроенный плагин NocoBase; он работает «из коробки», отдельная установка не требуется.
