# Быстрый старт
Это руководство показывает, как с нуля настроить график, используя основные возможности. Опциональные возможности рассматриваются в следующих разделах.

Предварительные условия:
- Источник данных и коллекция (таблица) настроены, и у вас есть право на чтение.

## Добавить блок графика
В дизайнере страницы нажмите «Добавить блок», выберите «Графики» и добавьте блок графика.

![clipboard-image-1761554593](https://static-docs.nocobase.com/clipboard-image-1761554593.png)

После добавления нажмите «Настроить» в правом верхнем углу блока.

![clipboard-image-1761554709](https://static-docs.nocobase.com/clipboard-image-1761554709.png)

Справа откроется панель настройки. Она включает три секции: запрос данных, параметры графика и события.

![clipboard-image-1761554848](https://static-docs.nocobase.com/clipboard-image-1761554848.png)

## Настроить запрос данных
В панели запроса данных настройте источник данных, фильтры запроса и связанные параметры.

- Выбор источника данных и коллекции
  - Выберите источник данных и коллекцию как базу для запроса.
  - Если коллекция недоступна для выбора или пуста, сначала проверьте, существует ли она и есть ли у вашего пользователя права доступа.

- Настройка показателей
  - Выберите одно или несколько числовых полей в качестве measures.
  - Установите агрегацию для каждого покаазателя: сумма / количество / среднее / максимум / минимум.

- Настройка измерений
  - Выберите одно или несколько полей в качестве измерения для группировки (дата, категория, регион и т.д.).
  - Для полей даты/времени задайте формат (например, `YYYY-MM`, `YYYY-MM-DD`), чтобы сохранить консистентное отображение.

![clipboard-image-1761555060](https://static-docs.nocobase.com/clipboard-image-1761555060.png)

Другие параметры, такие как Фильтр, Сортировка и Пагинация, являются опциональными.

## Запустить запрос и просмотреть данные

- Нажмите «Запустить запрос», чтобы получить данные и сразу отрисовать график-предпросмотр слева.
- Нажмите «Просмотр данных», чтобы просмотреть возвращённые данные; можно переключаться между форматами Таблициа и JSON. Нажмите снова, чтобы свернуть предпросмотр.
- Если результат пустой или неожиданный, вернитесь в панель запроса и проверьте права доступа к коллекции, сопоставление полей Показатели/Измерения и типы данных.

![clipboard-image-1761555228](https://static-docs.nocobase.com/clipboard-image-1761555228.png)

## Настроить параметры графика

В панели параметров графика выберите тип графика и настройте его параметры.

- Сначала выберите тип графика (линейный/областной, столбчатый/линейчатый, круговой/кольцевой, точечный и т.д.).
- Затем заполните базовые сопоставления полей:
  - Линейная/областная/столбчатая/линейчатая: `Поле X` (измерение), `Поле Y` (показатель), `Поле серии` (серия, необязательно)
  - Круговая/кольцевая: `Категория` (категориальное измерение), `Занчение` (показатель)
  - Точечная: `xField`, `yField` (два показателя или два измерения)
  - Для дополнительных параметров графиков см. документацию ECharts: [Ось (Axis)](https://echarts.apache.org/handbook/en/concepts/axis)
- После «Выполнить запрос» сопоставления полей по умолчанию заполняются автоматически. Если вы измените измерения/показатели, повторно проверьте сопоставления.

![clipboard-image-1761555586](https://static-docs.nocobase.com/clipboard-image-1761555586.png)

## Предпросмотр и сохранение
Изменения обновляют предпросмотр в реальном времени слева, но не сохраняются, пока вы не нажмёте «Сохранить».

Также можно использовать кнопки внизу:

- Предпросмотр: изменения конфигурации автоматически обновляют предпросмотр, или нажмите «Предпросмотр» для ручного обновления.
- Отмена: если текущие изменения не нужны, нажмите «Отмена» внизу или обновите страницу, чтобы вернуться к последнему сохранённому состоянию.
- Сохранить: нажмите «Сохранить», чтобы сохранить текущую конфигурацию запроса и графика в базу данных; после этого она становится активной для всех пользователей.

![clipboard-image-1761555803](https://static-docs.nocobase.com/clipboard-image-1761555803.png)

## Полезные советы

- Минимально рабочая настройка: выберите коллекцию и хотя бы однин показатель; для группированного отображения рекомендуется добавить измерения.
- Для измерений даты задайте подходящий формат (например, помесячный `YYYY-MM`), чтобы избежать разрывов или «шумной» оси X.
- Если запрос пустой или график не отображается:
  - Проверьте коллекцию/права и сопоставления полей.
  - Используйте «Просмотр данных», чтобы убедиться, что имена столбцов и типы данных соответствуют сопоставлению графика.
- Предпросмотр временный: он предназначен для проверки и корректировок. Эффект применится только после нажатия «Сохранить».
