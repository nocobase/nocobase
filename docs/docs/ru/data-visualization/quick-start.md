# Быстрый старт
В этом руководстве мы настроим диаграмму с нуля, используя основные необходимые функции. Дополнительные возможности будут рассмотрены в следующих главах.

Предварительные требования:
- У вас настроен источник данных и коллекция (таблица), и вы имеете права на чтение.

## Добавление блока диаграммы
В конструкторе страниц нажмите «Добавить блок», выберите «Диаграмма» и добавьте блок диаграммы.

![clipboard-image-1761554593](https://static-docs.nocobase.com/clipboard-image-1761554593.png)

После добавления нажмите «Настроить» в правом верхнем углу блока.

![clipboard-image-1761554709](https://static-docs.nocobase.com/clipboard-image-1761554709.png)

Справа откроется панель настроек диаграммы. Она состоит из трех разделов: «Запрос данных», «Параметры диаграммы» и «События».

![clipboard-image-1761554848](https://static-docs.nocobase.com/clipboard-image-1761554848.png)

## Настройка запроса данных
На панели «Запрос данных» вы можете настроить источник данных, условия фильтрации запроса и другие параметры.

- Сначала выберите источник данных и коллекцию
  - На панели «Запрос данных» выберите источник данных и коллекцию в качестве основы для запроса.
  - Если коллекция недоступна для выбора или пуста, сначала проверьте, создана ли она и имеет ли ваш пользователь необходимые права.

- Настройка показателей (Measures)
  - Выберите одно или несколько числовых полей в качестве показателей.
  - Для каждого показателя установите агрегацию: Sum (Сумма) / Count (Количество) / Avg (Среднее) / Max (Максимум) / Min (Минимум).

- Настройка измерений (Dimensions)
  - Выберите одно или несколько полей в качестве измерений для группировки (дата, категория, регион и т. д.).
  - Для полей даты/времени можно установить формат (например, `YYYY-MM`, `YYYY-MM-DD`) для единообразного отображения.

![clipboard-image-1761555060](https://static-docs.nocobase.com/clipboard-image-1761555060.png)

Другие условия: фильтрация, сортировка, пагинация — являются необязательными.

## Выполнение запроса и просмотр данных

- Нажмите «Выполнить запрос», чтобы получить данные и отобразить предварительный просмотр диаграммы (прямо на левой стороне страницы).
- Вы можете нажать «Просмотреть данные», чтобы увидеть предварительный результат возвращенных данных; поддерживается переключение между форматами Table (Таблица) и JSON. Повторное нажатие сворачивает предварительный просмотр данных.
- Если результат пуст или не соответствует ожиданиям, вернитесь на панель запросов и проверьте права доступа к коллекции, сопоставление полей для показателей/измерений и типы данных.

![clipboard-image-1761555228](https://static-docs.nocobase.com/clipboard-image-1761555228.png)

## Настройка параметров диаграммы

На панели «Параметры диаграммы» вы можете выбрать тип диаграммы и настроить ее параметры.

- Сначала выберите тип диаграммы (линейная/областная, столбчатая/гистограмма, круговая/кольцевая, точечная и т. д.).
- Завершите основные сопоставления полей:
  - Линейная/областная/столбчатая/гистограмма: `xField` (измерение), `yField` (показатель), `seriesField` (серия, необязательно)
  - Круговая/кольцевая: `Category` (измерение категории), `Value` (показатель)
  - Точечная: `xField`, `yField` (два показателя или измерения)
  - Для получения дополнительной информации о других типах графиков и настройках обратитесь к документации Echarts: [Axis](https://echarts.apache.org/handbook/en/concepts/axis)
- После нажатия «Выполнить запрос» сопоставления полей автоматически заполняются по умолчанию; если вы измените измерения/показатели, пожалуйста, перепроверьте сопоставления.

![clipboard-image-1761555586](https://static-docs.nocobase.com/clipboard-image-1761555586.png)

## Предварительный просмотр и сохранение
Изменения конфигурации автоматически обновляются в режиме реального времени в предварительном просмотре, и вы можете видеть диаграмму на левой стороне страницы. Однако обратите внимание, что все изменения не будут фактически сохранены до тех пор, пока вы не нажмете кнопку «Сохранить».

Вы также можете использовать кнопки внизу:

- Предварительный просмотр: Изменения конфигурации автоматически обновляют предварительный просмотр, но вы также можете нажать кнопку «Предварительный просмотр» внизу, чтобы вручную обновить его.
- Отмена: Если вы не хотите сохранять текущие изменения конфигурации, вы можете нажать кнопку «Отмена» внизу или обновить страницу, чтобы отменить текущие изменения и вернуться к последнему сохраненному состоянию.
- Сохранить: Нажмите «Сохранить», чтобы фактически сохранить все текущие запросы и настройки диаграммы в базе данных, после чего они станут доступны всем пользователям.

## Распространенные советы

- Минимальная рабочая конфигурация: выберите коллекцию + как минимум один показатель; рекомендуется добавить измерения для группированного отображения.
- Для измерений типа «дата» рекомендуется установить подходящий формат (например, `YYYY-MM` для ежемесячной статистики), чтобы избежать прерывистой или некорректной оси X.
- Если запрос пуст или диаграмма не отображается:
  - Проверьте коллекцию/права доступа и сопоставления полей;
  - В разделе «Просмотреть данные» убедитесь, что имена столбцов и их типы соответствуют сопоставлению диаграммы.
- Предварительный просмотр является временным: он используется только для проверки и корректировки. Изменения вступают в силу только после нажатия кнопки «Сохранить».
