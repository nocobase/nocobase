# Мультипространство

## Введение

**Плагин «Мультипространство»** позволяет создавать несколько независимых пространств данных посредством логической изоляции в рамках одного экземпляра приложения.

#### Сценарии использования
- **Несколько магазинов или заводов**: Бизнес-процессы и системные конфигурации очень схожи, например, унифицированное управление запасами, планирование производства, стратегии продаж и шаблоны отчетов. При этом необходимо обеспечить, чтобы данные каждого бизнес-подразделения не пересекались.
- **Управление несколькими организациями или дочерними компаниями**: Несколько организаций или дочерних компаний в рамках одной группы используют общую платформу, но при этом каждый бренд имеет независимые данные о клиентах, продуктах и заказах.

## Установка

В менеджере плагинов найдите плагин **«Мультипространство»** и активируйте его.

![](https://static-docs.nocobase.com/multi-space/Plugin-manager-NocoBase-10-15-2025_09_57_AM.png)

## Руководство пользователя

### Управление мультипространством

После активации плагина перейдите на страницу настроек **«Пользователи и разрешения»** и переключитесь на панель **«Пространства»**, чтобы управлять пространствами.

> Изначально существует встроенное **Неназначенное пространство (Unassigned Space)**, которое в основном используется для просмотра старых данных, не связанных ни с одним пространством.

![](https://static-docs.nocobase.com/multi-space/Users-Permissions-NocoBase-10-15-2025_09_58_AM.png)

#### Создание пространства

Нажмите кнопку «Добавить пространство», чтобы создать новое пространство:

![](https://static-docs.nocobase.com/multi-space/Users-Permissions-NocoBase-10-15-2025_09_58_AM%20(1).png)

#### Назначение пользователей

После выбора созданного пространства вы можете назначить пользователей, принадлежащих этому пространству, в правой части экрана:

> **Совет:** После назначения пользователей пространству необходимо **вручную обновить страницу**, чтобы список переключения пространств в правом верхнем углу обновился и отобразил актуальное пространство.

![](https://static-docs.nocobase.com/multi-space/Users-Permissions-NocoBase-10-15-2025_09_59_AM.png)

### Переключение и просмотр мультипространства

Вы можете переключить текущее пространство в правом верхнем углу.
При нажатии на **иконку глаза** справа (в выделенном состоянии) вы можете одновременно просматривать данные из нескольких пространств.

![](https://static-docs.nocobase.com/multi-space/Goods-NocoBase-10-15-2025_10_26_AM.png)

### Управление данными мультипространства

После активации плагина система автоматически добавит **поле «Пространство»** при создании коллекции.
**Только коллекции, содержащие это поле, будут включены в логику управления пространствами.**

![](https://static-docs.nocobase.com/multi-space/data-source-manager-main-NocoBase-10-15-2025_10_01_AM.png)

Для существующих коллекций вы можете вручную добавить поле «Пространство», чтобы включить управление пространствами:

![](https://static-docs.nocobase.com/multi-space/data-source-manager-main-NocoBase-10-15-2025_10_03_AM.png)

#### Логика по умолчанию

В коллекциях, содержащих поле «Пространство», система автоматически применяет следующую логику:

1. При создании данных они автоматически связываются с текущим выбранным пространством;
2. При фильтрации данных они автоматически ограничиваются данными текущего выбранного пространства.

### Классификация старых данных по пространствам

Для данных, существовавших до активации плагина «Мультипространство», вы можете выполнить классификацию по пространствам, следуя этим шагам:

#### 1. Добавление поля «Пространство»

Вручную добавьте поле «Пространство» в старую коллекцию:

![](https://static-docs.nocobase.com/multi-space/data-source-manager-main-NocoBase-10-15-2025_10_18_AM.png)

#### 2. Назначение пользователей Неназначенному пространству

Свяжите пользователя, управляющего старыми данными, со всеми пространствами, включая **Неназначенное пространство (Unassigned Space)**, чтобы просматривать данные, которые еще не были отнесены к какому-либо пространству:

![](https://static-docs.nocobase.com/multi-space/Users-Permissions-NocoBase-10-15-2025_10_14_AM.png)

#### 3. Переключение для просмотра данных всех пространств

Вверху выберите просмотр данных из всех пространств:

![](https://static-docs.nocobase.com/multi-space/Goods-NocoBase-10-15-2025_10_20_AM.png)

#### 4. Настройка страницы для распределения старых данных

Создайте новую страницу для распределения старых данных. Отобразите «поле Пространство» на **странице списка** и **странице редактирования**, чтобы вручную настроить принадлежность к пространству.

![](https://static-docs.nocobase.com/multi-space/Goods-10-15-2025_10_21_AM.png)

Сделайте поле «Пространство» редактируемым

![](https://static-docs.nocobase.com/multi-space/Goods-10-15-2025_10_22_AM.png)

#### 5. Ручное назначение данных пространствам

Через созданную выше страницу вручную отредактируйте данные, чтобы постепенно назначить старым данным правильное пространство (вы также можете самостоятельно настроить массовое редактирование).
