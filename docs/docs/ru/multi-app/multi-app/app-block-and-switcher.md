# Блок приложений и переключатель

Помимо управления дочерними приложениями в админке, Multi-app может предоставлять входы в приложения на фронтенде. Обычно используются:

- Блок "Applications" на странице для отображения доступных дочерних приложений
- Переключатель приложений в левом верхнем углу для перехода между основным и дочерними приложениями

## Блок приложений

![](https://static-docs.nocobase.com/202605271350840.png)

Блок "Applications" показывает список дочерних приложений на фронтенд-странице. Он подходит для простого портала приложений, из которого пользователи переходят в разные бизнес-приложения.

Каждое приложение показывает:

- Иконку приложения
- Название приложения
- Вход для доступа

При клике открывается соответствующее дочернее приложение.

### Настройка иконки

При создании или редактировании приложения в App Supervisor можно загрузить иконку в разделе "Display configuration".

Если иконка не загружена, система создает иконку по первой букве названия приложения.

![](https://static-docs.nocobase.com/202605271402603.png)

### Скрытие приложения

Если приложение не должно отображаться в блоке "Applications", включите "Hide in Applications block" в настройках приложения.

После скрытия:

- Приложение все еще можно администрировать
- Приложение все еще доступно по прямому адресу
- Оно просто не отображается в блоке "Applications"

![](https://static-docs.nocobase.com/202605271403980.png)

## Переключатель приложений

![](https://static-docs.nocobase.com/202605271403304.png)

Переключатель приложений отображается в левом верхнем углу и используется для быстрого перехода к другим приложениям.

Чтобы приложение отображалось в переключателе, включите "Show in app switcher" в его настройках.

После включения пользователи увидят переключатель в основном или дочернем приложении и смогут открыть другие приложения из списка.

![](https://static-docs.nocobase.com/202605271404322.png)

### Как открываются приложения

Переключатель открывает приложения так:

- Из основного приложения в дочернее: в новой вкладке
- Из одного дочернего приложения в другое: в текущей вкладке

Так работа в основном приложении не прерывается, а переходы между дочерними приложениями остаются удобными.
