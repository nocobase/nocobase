# Черновики форм

При включении черновиков форм вводимый контент автоматически сохраняется локально. Даже если вы случайно закроете форму или обновите страницу, заполненные данные не будут потеряны.

![20251213093717](https://static-docs.nocobase.com/20251213093717.png)

Если черновик существует, в верхней части формы отобразится уведомление. После успешной отправки формы черновик будет удален автоматически; если он больше не нужен, его также можно удалить вручную в любое время.

![20251213092734](https://static-docs.nocobase.com/20251213092734.png)

::: tip Подсказка

В настоящее время черновики форм поддерживают только локальное кэширование. В будущих версиях будет добавлена поддержка кэширования на стороне сервера для синхронизации между несколькими устройствами.

:::
