# Черновики форм

Когда черновики форм включены, содержимое, которое вы заполняете, автоматически сохраняется локально. Даже если форма случайно закрыта или страница перезагружена, введенные данные не будут потеряны.

![20251213093717](https://static-docs.nocobase.com/20251213093717.png)

Если черновик существует, сверху формы будет отображено уведомление. Черновик будет автоматически удален после успешной отправки; если он больше не нужен, вы также можете удалить его вручную в любой момент.

![20251213092734](https://static-docs.nocobase.com/20251213092734.png)

::: tip Совет

На данный момент черновики форм поддерживают только локальное кэширование. В будущих версиях появится серверное кэширование, чтобы обеспечить синхронизацию между несколькими устройствами.

:::
