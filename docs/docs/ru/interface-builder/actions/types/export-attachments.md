# Экспорт вложений

:::: tip Совет
Эта функция предоставляется коммерческим плагином `plugin-action-export-pro`. Для деталей ознакомьтесь с коммерческой лицензией.
::::

## Введение

Экспорт вложений поддерживает экспорт полей, связанных с вложениями, в виде сжатого пакета.

#### Конфигурация экспорта вложений

![20251029173251](https://static-docs.nocobase.com/20251029173251.png)

![20251029173425](https://static-docs.nocobase.com/20251029173425.png)

![20251029173345](https://static-docs.nocobase.com/20251029173345.png)

- Настройте поля вложений для экспорта (поддерживается несколько выборов).
- Можно выбрать, нужно ли создавать отдельную папку для каждой записи.

Правила именования файлов:

- Если вы выбираете создание папки для каждой записи, правило именования файла: `{Значение поля заголовка записи}/{Название поля вложения}[-{Порядковый номер файла}].{Расширение файла}`.
- Если вы не создаете папку, правило именования файла: `{Значение поля заголовка записи}-{Название поля вложения}[-{Порядковый номер файла}].{Расширение файла}`.

Номер последовательности файла автоматически формируется, когда поле вложений содержит несколько вложений.

- [Правило связывания](/interface-builder/actions/action-settings/linkage-rule): динамически показывать/скрывать кнопку;
- [Кнопка редактирования](/interface-builder/actions/action-settings/edit-button): редактировать заголовок, тип и значок кнопки;
