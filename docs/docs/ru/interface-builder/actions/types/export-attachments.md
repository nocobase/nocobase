# Экспорт вложений

:::tip Совет
Эта функция предоставляется коммерческим плагином `plugin-action-export-pro`. Подробности см. в коммерческой лицензии.
:::

## Введение

Экспорт вложений позволяет экспортировать поля, связанные с вложениями, в виде сжатого архива.

#### Настройка экспорта вложений

![20251029173251](https://static-docs.nocobase.com/20251029173251.png)

![20251029173425](https://static-docs.nocobase.com/20251029173425.png)

![20251029173345](https://static-docs.nocobase.com/20251029173345.png)

- Настройте поля вложений для экспорта; поддерживается множественный выбор.
- Вы можете выбрать, создавать ли отдельную папку для каждой записи.

Правила именования файлов:

- Если вы выбрали создание папки для каждой записи, правило именования файлов будет следующим: `{Значение поля заголовка записи}/{Имя поля вложения}[-{Порядковый номер файла}].{Расширение файла}`.
- Если вы выбрали не создавать папки, правило именования файлов будет следующим: `{Значение поля заголовка записи}-{Имя поля вложения}[-{Порядковый номер файла}].{Расширение файла}`.

Порядковый номер файла генерируется автоматически, если поле вложения содержит несколько файлов.

- [Правило связывания](/interface-builder/actions/action-settings/linkage-rule): динамическое отображение/скрытие кнопки;
- [Редактирование кнопки](/interface-builder/actions/action-settings/edit-button): редактирование заголовка, типа и значка кнопки;
