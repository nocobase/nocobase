# Массовое редактирование

## Введение

Массовое редактирование предназначено для гибких пакетных обновлений данных. После нажатия на кнопку массового редактирования настройте форму массового редактирования во всплывающем окне и задайте стратегии обновления для каждого поля.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM.png)


## Настройки операции

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM%20(1).png)


## Руководство

### Настройка формы массового редактирования

1. Добавьте кнопку массового редактирования.

2. Задайте область массового редактирования: «Выбранные/Все»; по умолчанию — «Выбранные».

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_14_AM.png)

3. Добавьте форму массового редактирования.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_14_AM.png)

4. Настройте поля для редактирования и добавьте кнопку отправки.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM%20(1).png)

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM.png)

### Отправить форму

1. Выберите строки для редактирования.

2. Для каждого поля выберите режим редактирования и укажите значения для отправки.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_10_33_AM.png)

::::info{title=Доступные режимы редактирования}
* Остаётся без изменений: сохранить поле как есть
* Изменить на: обновить поле указанным значением
* Очистить: очистить значение поля

:::

3. Отправьте форму.
