# Массовое редактирование

## Введение

Массовое редактирование подходит для сценариев, требующих гибкого пакетного обновления данных. После нажатия кнопки «Массовое редактирование» вы можете настроить форму во всплывающем окне и задать различные стратегии обновления для каждого поля.

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM.png)


## Настройка действия

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM%20(1).png)


## Руководство пользователя

### Настройка формы массового редактирования

1. Добавьте кнопку «Массовое редактирование».

2. Установите область массового редактирования: Выбранные / Все (по умолчанию — Выбранные).

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_14_AM.png)

3. Добавьте форму массового редактирования.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_14_AM.png)

4. Настройте поля, которые необходимо отредактировать, и добавьте кнопку отправки.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM%20(1).png)

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM.png)

### Отправка формы

1. Выберите строки данных, которые нужно отредактировать.

2. Выберите режим редактирования для полей и введите значения для отправки.

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_10_33_AM.png)

:::info{title="Доступные режимы редактирования"}
* **Не обновлять**: поле остается без изменений.
* **Изменить на**: обновить значение поля на указанное.
* **Очистить**: удалить данные из поля.

:::

3. Отправьте форму.
