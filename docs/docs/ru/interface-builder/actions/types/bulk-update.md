# Массовое обновление

## Введение

Операция массового обновления используется, когда вам нужно применить одно и то же обновление к группе записей. Перед выполнением массового обновления пользователю необходимо заранее определить логику присвоения значений полям для обновления. Эта логика будет применена ко всем выбранным записям, когда пользователь нажмет кнопку «Обновить».

![20251029195320](https://static-docs.nocobase.com/20251029195320.png)

## Настройка операции

![20251029195729](https://static-docs.nocobase.com/20251029195729.png)

### Данные для обновления

Выбранные/Все, по умолчанию — Выбранные.

![20251029200034](https://static-docs.nocobase.com/20251029200034.png)

### Присвоение значений полям

Укажите поля для массового обновления. Обновлены будут только те поля, которые вы укажете.

Как показано на рисунке, настройте операцию массового обновления в таблице заказов, чтобы массово обновить выбранные данные до статуса «Ожидает утверждения».

![20251029200109](https://static-docs.nocobase.com/20251029200109.png)
