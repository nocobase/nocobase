# Редактировать

## Введение

Действие Редактировать — это кнопка строки, в которой текущая запись используется как контекст. Оно отображается во всплывающем окне, где пользователи могут настраивать и конфигурировать блоки.

![20251029165442](https://static-docs.nocobase.com/20251029165442.png)

Настраивайте блоки по мере необходимости.

![20251029165514](https://static-docs.nocobase.com/20251029165514.png)

## Настройки действия

![20251029165620](https://static-docs.nocobase.com/20251029165620.png)

- [Правила связывания](/interface-builder/actions/action-settings/linkage-rule): динамически показывать/скрывать кнопку;
- [Кнопка редактирования](/interface-builder/actions/action-settings/edit-button): редактировать заголовок кнопки, тип и значок;
- [Редактирование всплывающего окна](/interface-builder/actions/action-settings/edit-popup): задавать режим открытия и размер.
