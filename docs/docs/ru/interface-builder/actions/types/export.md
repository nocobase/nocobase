# Экспорт

## Введение

Функция экспорта позволяет экспортировать отфильтрованные записи в формате **Excel**, а также настраивать поля, которые будут экспортироваться. Пользователи могут выбрать нужные поля для последующего анализа, обработки или архивирования данных. Эта функция повышает гибкость операций с данными, особенно в сценариях, когда данные нужно передать в другие платформы или выполнить дальнейшую обработку.

### Основные возможности
- **Выбор полей**: пользователи могут настроить и выбрать поля для экспорта, чтобы экспортируемые данные были точными и при этом лаконичными.
- **Поддержка формата Excel**: экспортируемые данные будут сохранены в виде стандартного файла Excel — это упрощает интеграцию и анализ в других инструментах.

С помощью этой функции пользователи могут легко экспортировать ключевые данные из своей работы для внешнего использования, повышая эффективность работы.

### Поля, доступные для экспорта

- Первый уровень: все поля текущей коллекции;
- Второй уровень: если это поле связи, нужно выбрать поля из связанной коллекции;
- Третий уровень: обрабатываются только три уровня; поля связей на последнем уровне не отображаются;
