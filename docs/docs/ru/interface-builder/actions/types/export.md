# Экспорт

## Введение

Функция экспорта позволяет экспортировать отфильтрованные записи в формате **Excel**, а также настраивать поля для экспорта. Вы можете выбрать необходимые поля для последующего анализа, обработки или архивирования данных. Эта функция повышает гибкость работы с данными, особенно в случаях, когда данные необходимо перенести на другие платформы или подвергнуть дальнейшей обработке.

### Ключевые особенности:
- **Выбор полей**: Вы можете настроить и выбрать поля для экспорта, обеспечивая точность и лаконичность экспортируемых данных.
- **Поддержка формата Excel**: Экспортированные данные будут сохранены в стандартном файле Excel, что упрощает их интеграцию и анализ с другими данными.

С помощью этой функции вы можете легко экспортировать ключевые рабочие данные для внешнего использования, повышая эффективность вашей работы.

### Экспортируемые поля

- Первый уровень: Все поля текущей **коллекции**;
- Второй уровень: Если это поле связи, необходимо выбрать поля из связанной **коллекции**;
- Третий уровень: Обрабатываются только три уровня; поля связи на последнем уровне не отображаются;
