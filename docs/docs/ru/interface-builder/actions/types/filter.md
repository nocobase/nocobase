# Фильтр

## Введение

Действие Фильтр обычно появляется в блоках данных для фильтрации данных путем настройки различных условий. Обратите внимание: если для блока уже настроена область данных, действие Фильтр будет объединено с условиями области данных, и итоговым условием фильтрации станет совокупность обоих.

## Настройки действия

![20251029173708](https://static-docs.nocobase.com/20251029173708.png)

### Поля, доступные для фильтрации

По умолчанию включены все поля этой коллекции. В списке условий можно выбирать только включенные поля.

Поля из связанных коллекций также могут использоваться в качестве полей фильтра.
