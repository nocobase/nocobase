# Фильтр

## Введение

Операция фильтрации обычно используется в блоках данных для фильтрации данных путем настройки различных условий. Обратите внимание, что если для блока уже настроена область данных, операция фильтрации будет объединена с условиями области данных в качестве окончательного условия фильтрации.

### Поля для фильтрации

По умолчанию включены все поля этой коллекции. В списке полей условий можно выбрать только включенные поля.

Поля из связанных коллекций также могут использоваться в качестве полей для фильтрации.
