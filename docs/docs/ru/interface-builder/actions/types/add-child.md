# Добавление дочерней записи

## Введение

Эта операция разработана специально для **блоков древовидных таблиц** для быстрого добавления дочерних записей. При добавлении дочерней записи выбранная строка будет рассматриваться как родительский узел, что позволяет пользователям легко создавать дочерние записи под любым родительским узлом. Эта функция значительно упрощает процесс управления иерархическими данными, делая расширение и организацию структур данных более эффективными и интуитивно понятными.
