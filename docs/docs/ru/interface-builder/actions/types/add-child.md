# Добавить дочерний элемент

## Введение

Эта операция специально предназначена для **древовидных таблиц**, чтобы быстро добавлять дочерние записи. При добавлении дочерней записи текущая строка рассматривается как родительский узел, что позволяет пользователям легко создавать дочерние записи под любым родителем. Эта возможность значительно упрощает управление и организацию иерархических данных, делая процесс более эффективным и интуитивным.
