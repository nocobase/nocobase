# Ссылка

## Введение

Действие Ссылка использует навигацию по маршрутам, поддерживает передачу переменных URL и позволяет целевой странице динамически адаптировать контент на основе переданных данных. Доступно для настройки в блоках данных.

### Открывать в новом окне

После того как вы включите опцию «Открывать в новом окне», ссылка будет открываться в новом окне.

![20251029175327](https://static-docs.nocobase.com/20251029175327.png)

- [Кнопка редактирования](/interface-builder/actions/action-settings/edit-button): редактировать заголовок кнопки, тип и значок;
- [Правила связывания](/interface-builder/linkage-rule): динамически управлять состоянием кнопки.
