# Открыть модальное окно

## Введение

Действие **Открыть модальное окно** похоже на действия "Просмотр" и "Редактирование". Вы можете настроить кнопку **Открыть модальное окно** в строках или заголовке таблицы. При нажатии на кнопку открывается соответствующее модальное окно, а блоки внутри этого модального окна отображают данные на основе текущей контекстной записи.

![20251029182351](https://static-docs.nocobase.com/20251029182351.png)

Настройте блоки в модальном окне по мере необходимости.

![20251029190208](https://static-docs.nocobase.com/20251029190208.png)

## Настройки действия

![20251029190352](https://static-docs.nocobase.com/20251029190352.png)

- [Редактировать кнопку](/interface-builder/actions/action-settings/edit-button): Измените заголовок, тип и значок кнопки.
- [Правило связывания](/interface-builder/linkage-rule): Динамически управляйте состоянием кнопки (отображение/отключение).
- [Редактировать модальное окно](/interface-builder/actions/action-settings/edit-popup): Установите режим открытия (модальное окно или выдвижная панель) и размер.
