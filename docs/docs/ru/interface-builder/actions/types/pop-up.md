# Открыть модальное окно

## Введение

Действие **Открыть модальное окно** похоже на действия Просмотр и Редактирование. Пользователи могут настроить кнопку **Открыть модальное окно** в строках или в заголовке таблицы. При нажатии на кнопку открывается соответствующее модальное окно, а блоки внутри этого модального окна отображают данные на основе текущей записи контекста.

![20251029182351](https://static-docs.nocobase.com/20251029182351.png)

Настраивайте блоки внутри модального окна по мере необходимости.

![20251029190208](https://static-docs.nocobase.com/20251029190208.png)

## Настройки действия

![20251029190352](https://static-docs.nocobase.com/20251029190352.png)

- [Кнопка редактирования](/interface-builder/actions/action-settings/edit-button): редактировать заголовок кнопки, тип и значок;
- [Правила связывания](/interface-builder/linkage-rule): динамически контролировать состояние кнопки (видимость/доступность);
- [Редактирование модального окна](/interface-builder/actions/action-settings/edit-popup): задаёт режим открытия (модальное окно или выдвижной блок) и размер.
