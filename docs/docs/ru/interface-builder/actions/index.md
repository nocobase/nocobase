# Действие

В UI действие — это кнопка, которая запускает набор инструкций. Его можно разместить непосредственно на странице, в диалоговом окне или в выдвижной панели, а также использовать в сочетании с блоками. В настоящее время динамически настраиваемые действия в основном применяются в комбинации с блоками.
