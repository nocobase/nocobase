# Действие

В интерфейсе действие — это кнопка, которая используется для запуска набора инструкций. Действие можно размещать непосредственно на странице, в диалоговом окне или в выдвижном блоке, а также использовать совместно с блоками. В настоящее время динамически настраиваемые действия в основном используются вместе с блоками.
