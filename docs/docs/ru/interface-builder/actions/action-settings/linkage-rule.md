# Правила взаимосвязи действий

## Введение

Правила взаимосвязи действий позволяют динамически управлять состоянием действия (например, отображать, включать, скрывать, отключать и т.д.) в зависимости от заданных условий. Настроив эти правила, вы можете связать поведение кнопок действий с текущей записью, ролью пользователя или контекстными данными.

![20251029150224](https://static-docs.nocobase.com/20251029150224.png)

## Как использовать

Когда условие выполняется (если условие не задано, оно считается выполненным по умолчанию), запускается выполнение настроек свойств или JavaScript-кода. В условиях можно использовать константы и переменные.

![20251030224601](https://static-docs.nocobase.com/20251030224601.png)

Правило позволяет изменять свойства кнопок.

![20251029150452](https://static-docs.nocobase.com/20251029150452.png)

## Константы

Пример: Оплаченные заказы нельзя редактировать.
