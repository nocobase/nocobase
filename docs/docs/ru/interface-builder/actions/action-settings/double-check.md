# Подтверждение

## Введение

Конфигурация подтверждения используется, чтобы подтвердить, выполняется ли действие. При клике по действию появляется диалог подтверждения. Кнопка удаления имеет подтверждение включённым по умолчанию, чтобы предотвратить потерю данных или необратимые результаты из‑за случайных операций.

![20251029125021](https://static-docs.nocobase.com/20251029125021.png)

Поддерживается настройка содержимого и заголовка подтверждения, а также доступ к переменным.
