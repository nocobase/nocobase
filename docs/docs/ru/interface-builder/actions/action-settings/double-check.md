# Подтверждение

## Введение

Настройка подтверждения используется для проверки выполнения действия. При нажатии на действие появляется диалоговое окно подтверждения. Для кнопки удаления подтверждение включено по умолчанию, чтобы предотвратить потерю данных или необратимые последствия из-за случайных операций.

![20251029125021](https://static-docs.nocobase.com/20251029125021.png)

Вы можете настроить содержимое и заголовок подтверждения, а также получить доступ к переменным.
