## UID модального окна

«UID модального окна» — это уникальный идентификатор компонента, который открывает это модальное окно. Он также соответствует сегменту `viewUid` в URL-адресе `view/:viewUid`. Вы можете быстро получить его, нажав «Копировать UID модального окна» в меню настроек поля или кнопки, которая вызывает модальное окно.

![popup-copy-uid-20251102](https://static-docs.nocobase.com/popup-copy-uid-20251102.png)

Настройка UID модального окна позволяет повторно использовать его.

### Внутреннее модальное окно (по умолчанию)
- «UID модального окна» равен UID текущей кнопки действия (по умолчанию используется UID этой кнопки).

### Внешнее модальное окно (повторное использование)
- В поле «UID модального окна» укажите UID кнопки-триггера из другого места (то есть UID модального окна), чтобы повторно использовать это модальное окно.
- Типичное применение: несколько страниц или блоков используют один и тот же интерфейс и логику модального окна, что позволяет избежать дублирования настроек.
- При использовании внешнего модального окна некоторые параметры становятся недоступными для изменения (см. ниже).

## Другие связанные параметры

- `Data source / Collection`: Только для чтения. Показывает **источник данных** и **коллекцию**, к которым привязано модальное окно; по умолчанию используется **коллекция** текущего блока. В режиме внешнего модального окна используются настройки целевого модального окна, и этот параметр изменить нельзя.
- `Association name`: Необязательный параметр. Используется для открытия модального окна из поля ассоциации; отображается только при наличии значения по умолчанию. В режиме внешнего модального окна используются настройки целевого модального окна, и этот параметр изменить нельзя.
- `Source ID`: Отображается только при установке `Association name`; по умолчанию используется `sourceId` текущего контекста; может быть переменной или фиксированным значением в зависимости от потребностей.
- `filterByTk`: Может быть пустым, переменной или фиксированным значением, используется для ограничения записей данных, загружаемых модальным окном.
