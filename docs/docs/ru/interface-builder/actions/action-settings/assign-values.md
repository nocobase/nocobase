# Присвоение значений полям

## Введение

Присвоение значений полям — это предварительная установка значений для полей перед выполнением операции. При обновлении или создании записи эти данные заполняются автоматически, избавляя вас от необходимости вводить их вручную. Примеры таких сценариев включают массовое обновление статусов заказов или автоматическую установку текущего времени в качестве времени доставки при получении заказа.
