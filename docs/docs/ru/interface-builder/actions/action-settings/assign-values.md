# Присвоение значений

## Введение

Присвоение значений полю — это предварительная настройка значения поля перед выполнением действия. При обновлении или создании записи эти данные автоматически подставляются без ручного ввода, например при массовом обновлении статусов заказов или при установке времени доставки на текущее время, когда заказ доставлен.
