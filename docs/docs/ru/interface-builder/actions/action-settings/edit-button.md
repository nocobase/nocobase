# Редактирование кнопки

## Введение

Система позволяет настраивать название, иконку, цвет фона и тип кнопки, предоставляя гибкие возможности кастомизации пользовательского интерфейса. Это помогает пользователям адаптировать внешний вид и функциональность кнопок под свои конкретные нужды.

- **Название кнопки**: Вы можете настроить текст кнопки в соответствии с вашими бизнес-требованиями, чтобы надпись была понятной и интуитивной.
- **Иконка кнопки**: Поддерживается установка иконки для кнопки, что улучшает визуальное распознавание и позволяет пользователям быстро определить её функцию.
- **Опасное действие (Danger Action)**: Если кнопка выполняет **опасное действие** (например, удаление, сброс и т.п.), её можно установить как тип **Danger Action**.
- **Тип кнопки (Button Type)**: Кнопки поддерживают различные настройки типа. Распространенные типы кнопок включают:
  - **По умолчанию (Default)**: Стандартная кнопка, подходящая для обычных операций.
  - **Основной (Primary)**: Используется для важных или основных действий, обычно имеет заметный визуальный эффект.
  - **Пунктирный (Dashed)**: Кнопка с пунктирной рамкой, подходит для второстепенных действий или кнопок-подсказок.
  - **Текст (Text)**: Кнопка только с текстом и без фонового цвета, обычно используется для легких операций или ссылок.
