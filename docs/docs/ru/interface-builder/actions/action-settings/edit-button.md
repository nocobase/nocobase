# Настройки кнопки

## Введение

Система поддерживает настройку имени кнопки, значка, цвета фона и типа кнопки, предоставляя гибкие возможности для визуальной кастомизации, чтобы пользователи могли подстраивать отображение и функциональность кнопки под конкретные потребности.

- **Имя кнопки**: пользователи могут настраивать текст кнопки под бизнес-требования, чтобы метка была понятной и интуитивной.
- **Иконка кнопки**: поддерживает настройку значка для кнопки, чтобы улучшить визуальное распознавание и помочь быстро понять назначение.
- **Опасное действие**: когда кнопка представляет **опасное действие** (например, удаление, сброс и т. п.), ее можно настроить как тип **Опасное действие**.
- **Тип кнопки**: кнопки поддерживают разные типы. Распространенные типы включают:
  - **По умолчанию**: стандартная кнопка для обычных действий.
  - **Основная**: используется для важных или первичных действий, обычно с более заметным визуальным акцентом.
  - **Пунктирная**: кнопка с пунктирной рамкой, подходит для второстепенных действий или подсказок.
  - **Текст**: кнопка только с текстом без цвета фона, обычно для легких действий или ссылок.
