# Подробности

## Введение

Подробности — это компонент, соответствующий подформе в режиме только для чтения. По сравнению с отображением в виде метки поля или заголовка подробности позволяют показывать больше данных из текущей коллекции и при необходимости — из связанных коллекций. Многоуровневые данные связей наглядно показываются в виде вложенных блоков подробностей в форме.
