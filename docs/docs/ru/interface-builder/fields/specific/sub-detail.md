# Дочерние детали

## Введение

Дочерние детали — это компонент, соответствующий подформе в режиме чтения. В отличие от компонентов «Метка» и «Заголовок», дочерние детали позволяют не только отображать больше данных из текущей коллекции, но и настраивать отображение информации из связанных коллекций. Многоуровневые связанные данные чётко представляются в виде вложенных деталей.
