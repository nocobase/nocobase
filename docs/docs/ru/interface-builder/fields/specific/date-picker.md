# Дата и время

## Введение

Поле даты поддерживает настраиваемые форматы даты и времени.

## Где задаётся формат даты

### Поля коллекции

![20240417114311](https://static-docs.nocobase.com/20240417114311.png)

> Формат даты, заданный в поле **коллекции**, используется как формат по умолчанию для поля. Его можно переопределить форматом даты в настройках поля в конфигурации блока.

### Настройки поля

![20251027191824](https://static-docs.nocobase.com/20251027191824.png)
