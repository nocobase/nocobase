# Селектор данных

## Введение

Селектор данных, в виде таблицы во всплывающем окне, поддерживает выбор существующих данных из целевой коллекции для создания связи или добавление данных в целевую коллекцию с последующим связыванием.

![20251027203343](https://static-docs.nocobase.com/20251027203343.png)

Выбранные данные не будут отображаться в селекторе таблицы.

![20251027203717](https://static-docs.nocobase.com/20251027203717.gif)

В таблице селектора можно дополнительно управлять целевой коллекцией связи: добавлять и удалять записи, импортировать и экспортировать данные и т. д.

![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
