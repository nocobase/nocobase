# Селектор данных

## Введение

Селектор данных, представленный в виде всплывающего табличного селектора, позволяет выбирать существующие данные из целевой коллекции для связывания или добавлять новые данные в целевую коллекцию с последующим связыванием.

![20251027203343](https://static-docs.nocobase.com/20251027203343.png)

Выбранные данные не будут отображаться в табличном селекторе.

![20251027203717](https://static-docs.nocobase.com/20251027203717.gif)

В табличном селекторе вы также можете дополнительно управлять целевой коллекцией для связывания (добавлять, удалять, импортировать/экспортировать и т.д.).

![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
