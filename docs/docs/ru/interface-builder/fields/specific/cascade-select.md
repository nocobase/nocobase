# Каскадный выбор

## Введение

Каскадный селектор используется для полей связей, где целевая коллекция представлена в виде древовидной таблицы. Он позволяет выбирать данные в соответствии с иерархической структурой древовидной коллекции и поддерживает нечёткий поиск для быстрого отбора.

## Инструкции по использованию

- При связи **один к одному** каскадный селектор поддерживает **одиночный выбор**.

![20251125214656](https://static-docs.nocobase.com/20251125214656.png)

- При связи **один ко многим** каскадный селектор поддерживает **множественный выбор**.

![20251125215318](https://static-docs.nocobase.com/20251125215318.png)

## Параметры настройки поля

### Поле заголовка

Поле заголовка определяет, какая метка будет отображаться для каждого варианта.

![20251125214923](https://static-docs.nocobase.com/20251125214923.gif)

> Поддерживает быстрый поиск по полю заголовка.

![20251125215026](https://static-docs.nocobase.com/20251125215026.gif)

Подробнее см. [Поле заголовка](/interface-builder/fields/field-settings/title-field)

### Диапазон данных

Управляет диапазоном данных древовидного списка. Если дочерняя запись соответствует условиям, ее родительская запись также будет включена.

![20251125215111](https://static-docs.nocobase.com/20251125215111.png)

Подробнее см. [Диапазон данных](/interface-builder/fields/field-settings/data-scope)
