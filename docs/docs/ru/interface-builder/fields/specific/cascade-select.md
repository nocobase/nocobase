# Каскадный выбор

## Введение

Каскадный выбор предназначен для полей связи, целевой коллекцией которых является древовидная таблица. Он позволяет выбирать данные в соответствии с иерархической структурой дерева и поддерживает нечёткий поиск для быстрой фильтрации.

## Инструкции по использованию

- Для связей «один-к-одному» каскадный выбор работает в режиме одиночного выбора.

![20251125214656](https://static-docs.nocobase.com/20251125214656.png)

- Для связей «один-ко-многим» каскадный выбор работает в режиме множественного выбора.

![20251125215318](https://static-docs.nocobase.com/20251125215318.png)

## Параметры конфигурации поля

### Заголовок поля

Указывается поле, значение которого отображается в качестве подписи каждого варианта.

![20251125214923](https://static-docs.nocobase.com/20251125214923.gif)

> Поддерживается быстрый поиск по полю заголовка

![20251125215026](https://static-docs.nocobase.com/20251125215026.gif)

Подробнее см. [Заголовок поля](/interface-builder/fields/field-settings/title-field)

### Область данных

Управляет областью данных в древовидном списке (если дочерняя запись соответствует условиям, она будет включена вместе с родительской записью).

![20251125215111](https://static-docs.nocobase.com/20251125215111.png)

Подробнее см. [Область данных](/interface-builder/fields/field-settings/data-scope)

Остальные варианты оформления полей связи описаны в разделе [Компоненты поля связи](/interface-builder/fields/association-field).
