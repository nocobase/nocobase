# Заголовок

## Введение

В режиме просмотра поля связей по умолчанию отображаются с помощью **компонента поля заголовка**, который идентифицирует текущую связанную запись, отображая значение поля заголовка. Система автоматически подбирает и отрисовывает соответствующий компонент поля заголовка в зависимости от его типа.
