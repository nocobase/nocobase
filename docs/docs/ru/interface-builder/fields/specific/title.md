# Заголовок

## Введение

В режиме только для чтения поля связи по умолчанию отображаются с помощью **компонента поля заголовка**. Он указывает на связанную запись, показывая значение поля, выбранного в качестве заголовка. Система автоматически сопоставляет и отображает соответствующий компонент поля заголовка в зависимости от типа поля заголовка.
