# Выпадающий список

## Введение

Выпадающий список поддерживает связывание данных: можно выбрать существующие записи из целевой коллекции либо добавить в неё новые записи для последующего связывания. Варианты в списке поддерживают нечёткий поиск.

![20251029205901](https://static-docs.nocobase.com/20251029205901.png)

## Настройка поля

### Область данных

Управляет областью данных списка в выпадающем списке.

![20251029210025](https://static-docs.nocobase.com/20251029210025.png)

Подробнее см. [Область данных](/interface-builder/fields/field-settings/data-scope)

### Правила сортировки

Управляет сортировкой данных в выпадающем списке.

Пример: сортировка по времени создания записи по убыванию.

![20251029210105](https://static-docs.nocobase.com/20251029210105.png)

### Разрешить добавление и связывание нескольких записей

Ограничивает связь «один-ко-многим», чтобы можно было связать только одну запись.

![20251029210145](https://static-docs.nocobase.com/20251029210145.png)

### Заголовок поля

Указывается поле, значение которого отображается в качестве подписи каждого варианта.

![20251029210507](https://static-docs.nocobase.com/20251029210507.gif)

> Поддерживается быстрый поиск по полю заголовка

Подробнее см. [Заголовок поля](/interface-builder/fields/field-settings/title-field)

### Быстрое создание: добавить, затем выбрать

![20251125220046](https://static-docs.nocobase.com/20251125220046.png)

#### Добавить через выпадающий список

После создания новой записи в целевой таблице система автоматически выбирает её и связывает при отправке формы.

В примере ниже в таблице «Заказы» поле связи «многие-к-одному» — «Учётная запись».

![20251125220447](https://static-docs.nocobase.com/20251125220447.gif)

#### Добавить через модальное окно

Создание через модальное окно подходит для более сложных сценариев ввода данных и позволяет настроить пользовательскую форму для создания новых записей.

![20251125220607](https://static-docs.nocobase.com/20251125220607.gif)

Подробнее см. [Компоненты поля связи](/interface-builder/fields/association-field)
