# Вложенная форма

## Введение

Вложенные формы подходят для сценариев, когда вам нужно сначала создать связанные данные, а затем связать их. Многоуровневые связанные данные четко отображаются в виде вложенной формы. По сравнению с компонентами «Выбор данных» и «Выпадающий список», вложенная форма позволяет напрямую управлять полями связанной коллекции в текущем блоке страницы, а связанные данные отправляются вместе с основной формой.
