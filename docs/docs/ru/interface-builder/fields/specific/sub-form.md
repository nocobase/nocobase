# Подформа

## Введение

Подформы подходят для сценариев, когда перед созданием связи вы формируете новые связанные данные. Многоуровневые связанные данные наглядно отображаются в виде вложенной формы. По сравнению с селектором данных и выпадающим списком подформа позволяет непосредственно управлять полями связанной коллекции в рамках текущего блока страницы, а связанные данные отправляются вместе с основной формой.
