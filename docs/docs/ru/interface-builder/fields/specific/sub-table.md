# Подтаблица (строчное редактирование)

## Введение

Подтаблица подходит для работы с полями отношений типа «один-ко-многим». Она позволяет как пакетно создавать и связывать новые данные в целевой коллекции, так и выбирать и связывать уже существующие данные.

## Инструкции по использованию

![20251027223350](https://static-docs.nocobase.com/20251027223350.png)

Поля различных типов в подтаблице отображаются с помощью разных компонентов. Крупные поля (например, Rich Text, JSON, Long Text) редактируются во всплывающем окне.

![20251027223426](https://static-docs.nocobase.com/20251027223426.png)

Поля ассоциации в подтаблице.

Заказы (One-to-Many) > Продукты заказа (One-to-One) > Возможность

![20251027223530](https://static-docs.nocobase.com/20251027223530.png)

Компонент поля отношений по умолчанию — это Single select (поддерживает Single select / Collection selector).

![20251027223754](https://static-docs.nocobase.com/20251027223754.png)
