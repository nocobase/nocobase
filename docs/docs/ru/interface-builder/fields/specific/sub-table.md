# Подтаблица (строчное редактирование)

## Введение

Подтаблица подходит для полей связи «один-ко-многим». Она поддерживает пакетное создание и связывание новых записей в целевой коллекции либо выбор и связывание уже существующих записей.

## Инструкции по использованию

![20251027223350](https://static-docs.nocobase.com/20251027223350.png)

Разные типы полей в подтаблице отображаются разными компонентами полей. Объёмные поля (например, форматированный текст, JSON или длинный текст) редактируются через модальное окно.

![20251027223426](https://static-docs.nocobase.com/20251027223426.png)

Пример вложенных полей связи в подтаблице:

Заказы (один-ко-многим) → позиции заказа (один-к-одному) → сделка

![20251027223530](https://static-docs.nocobase.com/20251027223530.png)

По умолчанию для поля связи используется выпадающий список; при необходимости можно переключиться на селектор данных.

![20251027223754](https://static-docs.nocobase.com/20251027223754.png)
