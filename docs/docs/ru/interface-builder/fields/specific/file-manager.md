# Файловый менеджер

## Введение

Файловый менеджер — это компонент поля связи для связей, в которых целевая коллекция — коллекция файлов.

![20251027194124](https://static-docs.nocobase.com/20251027194124.png)

## Настройка поля

- [Компонент поля](/interface-builder/fields/association-field): переключение на другие компоненты полей связи, например выпадающий список, селектор данных и т. д.;
- Разрешить выбор существующих файлов: по умолчанию включено — можно выбирать и связывать уже загруженные пользователем файлы.
