# Файловый менеджер

## Введение

Файловый менеджер — это компонент поля связи, предназначенный для работы со связями, где целевой коллекцией является коллекция файлов.

![20251027194124](https://static-docs.nocobase.com/20251027194124.png)

## Настройки поля

- [Компонент поля](/interface-builder/fields/association-field): Переключение на другие компоненты поля связи, такие как Выпадающий список, Селектор данных и т. д.
- Разрешить выбор существующих файлов: Включено по умолчанию, позволяет выбирать из уже загруженных файлов для связи.
