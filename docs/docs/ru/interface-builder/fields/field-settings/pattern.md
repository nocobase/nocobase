# Режимы отображения

## Введение

В отличие от блоков, компоненты полей имеют три режима отображения (поддерживается только для полей внутри формы). Переключение между этими режимами отображения изменяет доступные параметры конфигурации поля.

- Редактируемый;
- Только для чтения (нередактируемый);
- Только для чтения (режим просмотра);

### Режимы отображения для полей связей

**Режим отображения поля связи** определяет, как это поле будет представлено в интерфейсе, а также какие типы компонентов поля будут доступны.

В **редактируемом состоянии** поля связей поддерживают различные типы компонентов. Пользователи могут выбирать разные компоненты поля связи для отображения или выбора связанных данных в соответствии со своими бизнес-требованиями.

#### Компоненты поля связи в редактируемом состоянии

![20251028220447](https://static-docs.nocobase.com/20251028220447.png)

В этом состоянии пользователи могут гибко выбирать подходящий метод отображения для более эффективной работы с данными.

#### Компоненты поля связи в состоянии "Только для чтения"

При переключении в **состояние "Только для чтения"** система автоматически по умолчанию использует **компонент поля заголовка** для отображения связанных данных. Это идеально подходит для сценариев, когда вам нужно только просмотреть данные без внесения изменений.
