# Режим отображения

## Введение

В отличие от блоков, компоненты полей имеют три режима отображения (поддерживается только для полей внутри формы). Переключение между режимами отображения соответствует разным параметрам конфигурации поля.

- Редактируемый;
- Только чтение (не редактируется);
- Только чтение (режим просмотра).

### Режимы отображения для полей связи

**Режим отображения поля связи** определяет, как поле отображается в интерфейсе и какие типы компонентов поля доступны.

В **редактируемом состоянии** поля связи поддерживают несколько типов компонентов, что позволяет пользователям выбирать разные компоненты поля связи для отображения или выбора связанных данных в зависимости от бизнес-потребностей.

В этом состоянии пользователи могут гибко выбирать подходящий способ отображения для более эффективной работы с данными.

#### Компоненты поля связи в состоянии только чтение

При переключении в **состояние только чтение** система автоматически по умолчанию использует **компонент поля заголовка** для отображения связанных данных. Это подходит для сценариев, где данные нужно только просматривать без внесения изменений.
