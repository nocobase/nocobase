# Показать метку

## Введение

Поля в блоке по умолчанию отображают метки полей, чтобы обозначить смысл каждого поля и упростить пользователям чтение и понимание содержимого данных.

В некоторых сценариях (например, на страницах только для отображения или когда смысл полей и так уже очень понятен) вы можете скрыть метки ради более чистого интерфейса. После скрытия меток отображаются только значения полей, а названия больше не показываются.
