# Отображение меток

## Введение

Поля в блоке по умолчанию отображают метки полей, что помогает идентифицировать значение каждого поля и облегчает пользователям чтение и понимание содержимого данных.

В некоторых сценариях (например, на страницах только для просмотра или когда значение полей уже очевидно) вы можете скрыть метки для более чистого интерфейса. После скрытия меток отображаются только значения полей, а названия полей больше не показываются.
