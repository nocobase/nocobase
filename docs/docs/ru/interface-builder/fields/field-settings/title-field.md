# Заголовок поля

## Введение

**Заголовок поля** обычно используется в компонентах полей связи как идентификатор связанных данных, отображаемых в интерфейсе. Он определяет, как будет представлена основная информация данных, чтобы пользователям было проще быстро распознавать и понимать записи.

Разные типы заголовка поля соответствуют разным компонентам полей и поддерживают соответствующие параметры конфигурации. Например:

- если заголовок поля — это **поле даты**, компонент поля использует **компонент даты** для отображения данных;
- если заголовок поля — это **текстовое поле**, компонент поля использует **текстовый компонент** для отображения данных;
- если заголовок поля — это **поле выбора**, компонент поля использует **компонент выбора** для отображения данных.
