# Поле заголовка

## Введение

**Поле заголовка** обычно используется в компонентах полей связей в качестве идентификатора для связанных данных, отображаемых в интерфейсе. Оно определяет, как представляется основная информация о данных, и обычно помогает пользователям быстро идентифицировать и понимать данные.

Различные типы полей заголовка соответствуют различным компонентам полей и поддерживают соответствующие параметры конфигурации. Например:

- Когда поле заголовка является **полем даты**, компонент поля использует **компонент даты** для отображения данных.
- Когда поле заголовка является **текстовым полем**, компонент поля использует **текстовый компонент** для отображения данных.
- Когда поле заголовка является **полем выбора**, компонент поля использует **компонент выбора** для отображения данных.
