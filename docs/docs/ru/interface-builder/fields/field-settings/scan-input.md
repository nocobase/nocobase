# Ввод через сканирование

## Описание

Ввод через сканирование используется для текстовых полей в редактируемых формах. После включения справа от поля ввода появляется кнопка сканирования. Пользователь может сканировать QR-код или штрихкод, либо выбрать изображение из альбома для распознавания, а затем записать распознанный результат в текущее поле.

Обычно это подходит для ввода номеров оборудования, кодов активов, номеров заказов, номеров отслеживания и других значений, которые неудобно вводить вручную.

## Поддерживаемые поля

Ввод через сканирование в основном используется для текстовых полей, например:

- Однострочный текст
- Номер мобильного телефона
- Email
- URL
- UUID
- Nano ID

Если поле доступно только для чтения, находится в режиме просмотра или само не поддерживает редактируемый ввод, настройка ввода через сканирование не отображается.

## Настройка

Выберите соответствующее поле в блоке формы, откройте меню настройки поля и найдите `Настройки ввода через сканирование`.

Параметры включают:

- `Включить ввод через сканирование`: после включения справа от поля ввода будет показана кнопка сканирования
- `Запретить ручной ввод`: после включения пользователь сможет записывать значение поля только через сканирование и не сможет редактировать поле вручную

После отключения `Включить ввод через сканирование` параметр `Запретить ручной ввод` также становится неактивным.

## Использование

После нажатия пользователем кнопки сканирования справа от поля он может использовать камеру для распознавания QR-кода или штрихкода. Для сканирования в браузере требуется разрешение на доступ к камере. В мобильной среде с поддержкой встроенного сканирования сначала будет использоваться встроенная возможность устройства.

Если использовать камеру напрямую неудобно, пользователь также может нажать `Альбом`, чтобы выбрать изображение для распознавания.
