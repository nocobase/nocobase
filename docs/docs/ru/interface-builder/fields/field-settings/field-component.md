# Компонент поля

## Введение

Компоненты поля используются для отображения содержимого поля.
У некоторых полей есть несколько встроенных компонентов отображения. Пользователи могут выбрать подходящий компонент в зависимости от реальных потребностей, чтобы представлять значение поля разными способами.

Пример: в режиме просмотра поле URL можно переключить на компонент превью или компонент URL (по умолчанию).
