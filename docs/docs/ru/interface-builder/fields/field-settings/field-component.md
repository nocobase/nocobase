# Компонент поля

## Введение

Компоненты поля используются для отображения содержимого поля.
Некоторые поля имеют несколько встроенных компонентов отображения. Вы можете выбрать подходящий компонент в соответствии с вашими потребностями, чтобы представить значение поля различными способами.

Пример: В режиме просмотра поле URL можно переключить на компонент «Предварительный просмотр» или компонент «URL» (по умолчанию).

![20251028214628](https://static-docs.nocobase.com/20251028214628.png)

Пример: Поле связи можно переключить на подформу, подтаблицу (только для связей «один-ко-многим» или «многие-ко-многим»), селектор данных, выпадающий селектор (по умолчанию) или файловый менеджер (только если целевая **коллекция** связи является файловой **коллекцией**).
