# Форматирование чисел

## Введение

Форматирование чисел — это визуальное улучшение и стандартизация числовых полей (включая поля формул) в режиме только для чтения. Оно включает настройки разделителей тысяч, точности десятичных знаков, преобразования единиц измерения и научной нотации. Настроив форматирование чисел, можно учитывать привычные способы представления чисел в разных регионах и отраслях.

## Как использовать

- Поддерживается простое преобразование единиц (умножение/деление)
- Разделитель тысяч
- Префикс и суффикс
- Точность
- Научная нотация

Пример: отформатируйте сумму заказа, добавив символ валюты (например, $), разделитель тысяч (например, запятую или точку) и точность десятичных знаков, чтобы сумма была понятной и удобной для чтения.
