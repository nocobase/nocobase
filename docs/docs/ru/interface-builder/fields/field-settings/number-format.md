# Форматирование чисел

## Введение

Форматирование чисел — это визуальное улучшение и стандартизация числовых полей (включая поля формул) в режиме просмотра. Это включает в себя настройки разделителей тысяч, точности десятичных знаков, преобразования единиц измерения и научного представления чисел. Настроив форматирование чисел, вы можете адаптировать их представление под привычки разных регионов и отраслей.

## Как использовать

- Поддерживает простое преобразование единиц измерения (умножение/деление)
- Разделитель тысяч
- Префиксы и суффиксы
- Точность
- Научное представление чисел.

Пример: Отформатируйте сумму заказа, добавив символ валюты (например, символ юаня ￥), разделитель тысяч (например, запятую или точку) и точность десятичных знаков, чтобы сумма была понятной и легко читаемой.
