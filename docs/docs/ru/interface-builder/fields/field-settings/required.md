# Обязательное поле

## Введение

Обязательное поле — это распространённое правило для валидации форм. Вы можете включить его непосредственно в настройках поля или динамически установить поле как обязательное с помощью правил связывания формы.

## Где можно установить поле как обязательное

### Настройки поля коллекции

Когда поле коллекции установлено как обязательное, это запускает валидацию на бэкенде, и на фронтенде оно также по умолчанию отображается как обязательное (не может быть изменено).
![20251025175418](https://static-docs.nocobase.com/20251025175418.png)

### Настройки поля

Вы можете напрямую установить поле как обязательное. Этот подход подходит для полей, которые всегда должны быть заполнены пользователем, например, имя пользователя, пароль и т.д.

![20251028222818](https://static-docs.nocobase.com/20251028222818.png)

### Правила связывания

Установите поле как обязательное на основе условий, используя правила связывания полей в блоке формы.

Пример: Номер заказа является обязательным, если дата заказа не пуста.
