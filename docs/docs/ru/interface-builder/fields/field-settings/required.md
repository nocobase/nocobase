# Обязательное

## Введение

Обязательное — это общее правило для валидации в форме. Вы можете включить его напрямую в настройках поля или динамически задавать поле как обязательное через правила связывания формы.

## Где установить, что поле обязательно

### Настройки поля коллекции

Когда поле коллекции установлено как обязательное, это запускает серверную валидацию, а фронтенд по умолчанию также отображает поле как обязательное (его нельзя изменить).
![20251025175418](https://static-docs.nocobase.com/20251025175418.png)

### Настройки поля

Непосредственно задайте поле как обязательное. Это подходит для полей, которые всегда должны быть заполнены пользователем, например для имени пользователя, пароля и т. п.

![20251028222818](https://static-docs.nocobase.com/20251028222818.png)

### Правила связывания

Установите поле как обязательное на основе условий через правила связывания полей формы.

Например: номер заказа обязателен, когда дата заказа не пуста.
