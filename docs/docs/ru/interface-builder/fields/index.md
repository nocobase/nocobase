# Поле

В UI поле — это компонент, который служит носителем для отдельных единиц данных. Различные типы данных отображаются с помощью разных компонентов полей. Поля могут быть привязаны только к блокам и не могут использоваться независимо.

## Поля в блоке

Поля обычно не используются отдельно, а существуют как дочерние элементы блоков, работающих с данными. Блоки, работающие с данными, обычно имеют опцию «Настроить поля», а список полей предоставляется текущей коллекцией.

![20251029204240](https://static-docs.nocobase.com/20251029204240.png)

## Дизайнер полей (Панель инструментов)

Как и у блоков, в правом верхнем углу компонента поля также есть три значка:

- Перетаскивание для изменения макета
- [Поток событий](/interface-builder/event-flow)
- Конфигурация

![20251025165432](https://static-docs.nocobase.com/20251025165432.png)

## Макет поля

Вы можете настроить макет полей внутри блока, просто перетаскивая их.
