# Поле

В интерфейсе поле — это компонент поля, который служит контейнером для единичных данных. Различные типы данных отображаются с помощью разных компонентов полей. Поля можно прикреплять только к блокам, и использовать их независимо нельзя.

## Поля в блоке

Обычно поля не используются сами по себе — они существуют как дочерние элементы блоков с типом данных. Блоки типа данных обычно имеют опцию `Настроить поля`, а список полей предоставляется текущей коллекцией.

![20251029204240](https://static-docs.nocobase.com/20251029204240.png)

## Конструктор поля (панель инструментов)

Как и у блоков, в верхнем правом углу компонента поля также есть три значка:

- Перетащить для компоновки
- [Поток событий](/interface-builder/event-flow)
- Конфигурация
