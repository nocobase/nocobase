# Поля формы фильтра

## Введение

Форма фильтра поддерживает выбор полей из текущей коллекции и связанных коллекций (поля вложенных связей) в качестве полей фильтра.

### Оператор

Выберите подходящий оператор в зависимости от типа поля, чтобы повысить точность и эффективность фильтрации. Для строковых полей по умолчанию используется нечёткое совпадение.
