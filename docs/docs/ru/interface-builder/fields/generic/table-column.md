# Поля таблицы

## Введение

Поля таблицы позволяют не только настраивать ширину столбцов, заголовки полей и сортировку, но и предлагают более гибкие параметры отображения для специфических типов полей, таких как поля даты, поля связей и числовые поля.

![20251024174558](https://static-docs.nocobase.com/20251024174558.png)

## Параметры настройки полей

### Форматирование полей даты

![20251024175303](https://static-docs.nocobase.com/20251024175303.png)

Подробнее см. в разделе [Форматирование даты](/interface-builder/fields/specific/date-picker)

### Форматирование числовых полей

Поддерживает простое преобразование единиц измерения, разделители тысяч, префиксы и суффиксы, точность и научную нотацию.

![20251024175445](https://static-docs.nocobase.com/20251024175445.png)

Подробнее см. в разделе [Форматирование чисел](/interface-builder/fields/field-settings/number-format)

### Включение быстрого редактирования

Когда быстрое редактирование включено, при наведении курсора мыши на столбец появится кнопка редактирования. Нажмите на неё, чтобы быстро отредактировать и сохранить данные во всплывающем окне.

![20251025171158](https://static-docs.nocobase.com/20251025171158.gif)

### Включение открытия по клику

Помимо полей связей, которые поддерживают открытие всплывающего окна по клику, для обычных полей также можно включить опцию «Открыть по клику», чтобы они служили точкой входа для открытия всплывающего окна.

![20251025172308](https://static-docs.nocobase.com/20251025172308.gif)

### Режим отображения переполнения содержимого

Если содержимое столбца превышает его ширину, вы можете выбрать режим отображения переполнения:

-   Сокращение (по умолчанию)
-   Перенос строки

![20251025172549](https://static-docs.nocobase.com/20251025172549.png)

### Компонент поля

Некоторые поля поддерживают несколько форматов отображения. Пользователи могут переключать компоненты полей для достижения различных визуальных эффектов, что позволяет удовлетворить потребности разных сценариев. Например, поле типа **URL** можно переключить на компонент **Preview**, чтобы лучше отображать содержимое ссылки или предварительный просмотр изображения.

![20251025171658](https://static-docs.nocobase.com/20251025171658.png)
