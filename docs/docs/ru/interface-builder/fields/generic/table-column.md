# Поля таблицы

## Введение

Помимо настройки ширины столбца, заголовка поля, сортировки и т. д., поля таблицы также поддерживают более индивидуальные параметры отображения для отдельных полей (например, поля даты, поля связи и числовые поля).

![20251024174558](https://static-docs.nocobase.com/20251024174558.png)

## Параметры конфигурации поля

### Форматирование поля даты

![20251024175303](https://static-docs.nocobase.com/20251024175303.png)

Подробнее см. [Форматирование даты](/interface-builder/fields/specific/date-picker)

### Форматирование числового поля

Поддерживаются простое преобразование единиц, разделители тысяч, префиксы и суффиксы, точность и научная нотация.

![20251024175445](https://static-docs.nocobase.com/20251024175445.png)

Подробнее см. [Форматирование чисел](/interface-builder/fields/field-settings/number-format)

### Включить быстрое редактирование

Когда включено быстрое редактирование, при наведении мыши на столбец появляется кнопка редактирования. Нажмите её, чтобы быстро изменить и сохранить данные во всплывающем окне.

![20251025171158](https://static-docs.nocobase.com/20251025171158.gif)

### Включить открытие по клику

Помимо полей связи, которые поддерживают открытие всплывающего окна по клику, для обычных полей также можно включить открытие по клику и использовать их как точку входа для открытия всплывающего окна.

![20251025172308](https://static-docs.nocobase.com/20251025172308.gif)

### Режим отображения при переполнении содержимого

Когда содержимое столбца выходит за ширину столбца таблицы, можно задать режим переполнения:

- Многоточие (по умолчанию)
- Перенос строки

![20251025172549](https://static-docs.nocobase.com/20251025172549.png)

### Компонент поля

Некоторые поля поддерживают несколько форматов отображения. Пользователи могут переключать компонент поля для получения разных визуальных эффектов в разных сценариях. Например, поле типа «URL-адрес» можно переключить на компонент «Предпросмотр», чтобы лучше отображать содержимое ссылок или предпросмотр изображений.

![20251025171658](https://static-docs.nocobase.com/20251025171658.png)
