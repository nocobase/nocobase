# Поля раздела «Подробности»

## Введение

Настройки полей для блоков «Подробности», «Список» и «Сетка» практически идентичны и в первую очередь определяют, как поля отображаются в режиме просмотра (только для чтения).

![20251025172851](https://static-docs.nocobase.com/20251025172851.png)

## Параметры настройки полей

### Форматирование полей даты

![20251025173005](https://static-docs.nocobase.com/20251025173005.png)

Подробнее см. [Форматирование даты](/interface-builder/fields/specific/date-picker)

### Форматирование числовых полей

![20251025173242](https://static-docs.nocobase.com/20251025173242.png)

Поддерживается простое преобразование единиц измерения, разделители тысяч, префиксы и суффиксы, точность и научная нотация.

Подробнее см. [Форматирование чисел](/interface-builder/fields/field-settings/number-format)

### Разрешить открытие по клику

Помимо полей связей, обычные поля также можно настроить для открытия всплывающего окна по клику. Вы также можете выбрать способ открытия всплывающего окна (в виде выдвижной панели, диалогового окна или на дочерней странице).

![20251025173549](https://static-docs.nocobase.com/20251025173549.gif)

### Способ отображения переполненного содержимого

Когда содержимое поля превышает его ширину, вы можете настроить режим отображения переполнения.

- Многоточие (по умолчанию)
- Перенос строки

![20251025173917](https://static-docs.nocobase.com/20251025173917.png)

### Компонент поля

Некоторые поля поддерживают несколько форматов отображения, что достигается переключением компонента поля.

Например: компонент `URL` можно переключить на компонент `Preview`.

![20251025174042](https://static-docs.nocobase.com/20251025174042.png)

Например, поля связей можно отображать по-разному, переключив компонент поля заголовка на «Дочерние подробности», чтобы показать больше содержимого из связи.

![20251025174311](https://static-docs.nocobase.com/20251025174311.gif)
