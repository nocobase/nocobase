# Поля деталей

## Введение

Конфигурации полей для блоков «Детали», «Список» и «Сетка карточек» в целом одинаковы и в основном управляют тем, как поля отображаются в режиме только для чтения.

![20251025172851](https://static-docs.nocobase.com/20251025172851.png)

## Параметры конфигурации поля

### Форматирование поля даты

![20251025173005](https://static-docs.nocobase.com/20251025173005.png)

Подробнее см. [Форматирование даты](/interface-builder/fields/specific/date-picker)

### Форматирование числового поля

![20251025173242](https://static-docs.nocobase.com/20251025173242.png)

Поддерживаются простое преобразование единиц, разделители тысяч, префиксы и суффиксы, точность и научная нотация.

Подробнее см. [Форматирование чисел](/interface-builder/fields/field-settings/number-format)

### Включить открытие по клику

Помимо полей связи, обычные поля также можно настроить на открытие всплывающего окна по клику. Также можно задать способ открытия всплывающего окна (выдвижной блок, диалоговое окно или подстраница).

![20251025173549](https://static-docs.nocobase.com/20251025173549.gif)

### Режим отображения при переполнении содержимого

Когда содержимое поля превышает его ширину, можно задать режим переполнения.

- Многоточие (по умолчанию)
- Перенос строки

![20251025173917](https://static-docs.nocobase.com/20251025173917.png)

### Компонент поля

Некоторые поля поддерживают несколько форматов отображения, что можно реализовать переключением компонента поля.

Например: компонент «URL-адрес» можно переключить на компонент «Предпросмотр».

![20251025174042](https://static-docs.nocobase.com/20251025174042.png)

Например, поля связи можно отображать по-разному, переключаясь с компонента поля заголовка на «Подробности», чтобы показывать больше содержимого связи.

![20251025174311](https://static-docs.nocobase.com/20251025174311.gif)
