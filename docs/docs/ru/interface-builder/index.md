# UI Редактор

## WYSIWYG (What You See Is What You Get — «что видишь, то и получаешь»)

NocoBase предоставляет возможности WYSIWYG для создания интерфейсов. Нажмите кнопку UI Редактор (UI Builder), чтобы переключаться между режимом редактирования и режимом просмотра.

## Шаблон макета

По умолчанию NocoBase поставляется с шаблоном макета: навигация сверху и слева, а область контента — справа.

![未命名.002](https://static-docs.nocobase.com/未命名.002.jpeg)

## Параметры конфигурации

В режиме редактирования оранжевые маркеры указывают на элементы, которые можно настраивать. Точка входа для настроек элемента обычно находится в его правом верхнем углу. Большинство элементов поддерживают конфигурацию в реальном времени с мгновенным предварительным просмотром. Примеры:
