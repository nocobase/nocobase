# Конструктор интерфейса

## Что вы видите, то и получаете

NocoBase позволяет создавать интерфейсы по принципу «Что вы видите, то и получаете». Нажмите кнопку «Конструктор интерфейса», чтобы переключаться между режимами редактирования и просмотра.

## Макеты UI

NocoBase предоставляет макет для компьютера и мобильный макет. Макет для компьютера подходит для обычных задач управления и адаптирует навигацию и содержимое страницы к узкому экрану. Мобильный макет предоставляет отдельные мобильные навигацию и страницы. Подробности приведены в разделе [«Макеты UI»](./ui-layout/index.md).

## Параметры конфигурации

В режиме редактирования на экране появляются оранжевые маркеры, указывающие на настраиваемые элементы. Точка входа для настроек элемента обычно находится в его правом верхнем углу.
