# Высота блока

## Введение

Высота блока поддерживает три режима: **Высота по умолчанию**, **Заданная высота** и **Полная высота**. Большинство блоков поддерживают настройку высоты.

## Режимы высоты

### Высота по умолчанию

Для разных типов блоков используются разные стратегии высоты по умолчанию. Например, блоки Таблица и Форма автоматически подстраивают высоту под содержимое, и внутри блока не появляются полосы прокрутки.

### Заданная высота

Вы можете вручную задать общую высоту внешнего каркаса блока. Внутренняя область блока автоматически рассчитает и выделит доступную высоту.

![20260211154043](https://static-docs.nocobase.com/20260211154043.gif)

### Полная высота

Режим полной высоты похож на режим заданной высоты, за исключением того, что высота блока рассчитывается на основе текущего размера **области просмотра** текущего браузера, чтобы заполнить максимальную высоту экрана. На странице браузера полос прокрутки нет; они появляются только внутри блока.

Поведение внутренней прокрутки в режиме полной высоты немного отличается в зависимости от блока:

- **Таблица**: внутренняя прокрутка в `tbody`;
- **Форма / Детали**: прокрутка внутри сетки (контент прокручивается, кроме области действий);
- **Список / Сетка карточек**: прокрутка внутри сетки (контент прокручивается, кроме области действий и панели пагинации);
- **Карта / Календарь**: общая адаптивная высота, полос прокрутки нет;
- **Встроенный фрейм / Markdown**: ограничение высоты внешнего каркаса блока, полосы прокрутки появляются внутри блока.
