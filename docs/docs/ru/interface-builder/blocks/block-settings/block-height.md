# Высота блока

## Введение

Высота блока поддерживает три режима: **Высота по умолчанию**, **Заданная высота** и **Полная высота**. Большинство блоков поддерживают настройку высоты.

![20260211154020](https://static-docs.nocobase.com/20260211154020.png)

## Режимы высоты

### Высота по умолчанию

Стратегия высоты по умолчанию различается для разных типов блоков. Например, блоки «Таблица» и «Форма» автоматически адаптируют свою высоту в зависимости от содержимого, и внутри блока не появляются полосы прокрутки.

### Заданная высота

Вы можете вручную указать общую высоту внешней рамки блока. Доступная внутренняя высота будет рассчитана и распределена автоматически.

![20260211154043](https://static-docs.nocobase.com/20260211154043.gif)

### Полная высота

Режим полной высоты аналогичен заданной высоте, но высота блока рассчитывается на основе текущей **области просмотра** (viewport) браузера для достижения максимальной высоты на весь экран. Полосы прокрутки не появляются на самой странице браузера; они отображаются только внутри блока.

Обработка внутренней прокрутки в режиме полной высоты немного отличается в зависимости от типа блока:

- **Таблица**: внутренняя прокрутка внутри `tbody`;
- **Форма / Подробности**: прокрутка внутри сетки (Grid) (прокрутка содержимого, за исключением области действий);
- **Список / Сетка карточек**: прокрутка внутри сетки (Grid) (прокрутка содержимого, за исключением области действий и панели пагинации);
- **Карта / Календарь**: общая адаптивная высота, без полос прокрутки;
- **Iframe / Markdown**: ограничивает общую высоту рамки блока, полосы прокрутки появляются внутри блока.
