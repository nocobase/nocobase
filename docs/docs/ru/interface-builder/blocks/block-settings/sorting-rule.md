# Настройка правил сортировки

## Введение

Настройка правил сортировки означает определение правил сортировки по умолчанию для блока данных, чтобы данные в блоке отображались в заданном порядке.

### Несколько правил сортировки

Подходит для сложных сценариев сортировки: например, сначала сортировать по статусу заказа, а затем — по времени заказа в рамках одного статуса.
