# Макет

## Введение

Блочные элементы форм, такие как блоки форм, блоки деталей и блоки списков, поддерживают настройки макета. Варианты макета включают вертикальный (метки и поля ввода располагаются построчно) и горизонтальный (метки и поля ввода размещаются рядом). Вы можете настроить макет полей блока в соответствии с вашими потребностями, чтобы обеспечить ясность и эффективность ввода или просмотра информации.

### Вертикальный (по умолчанию)

Макет по умолчанию — вертикальный.

![20251024120143](https://static-docs.nocobase.com/20251024120143.png)

### Горизонтальный

Горизонтальный макет отображает метки и поля ввода рядом, что эффективно экономит пространство. Для полей с длинными заголовками вы можете выбрать перенос текста или сокращение (многоточие), чтобы обеспечить аккуратный и эстетичный вид интерфейса.

![20251024120237](https://static-docs.nocobase.com/20251024120237.png)

#### Выравнивание заголовка поля

- **По левому краю** (по умолчанию)
- **По правому краю**

#### Ширина заголовка поля

- Ширина по умолчанию составляет **120px**, минимальная ширина — **50px**.

#### Обработка заголовков полей, превышающих ширину

- **Перенос строки**
- **Многоточие**

#### Показывать двоеточие

Вы можете выбрать, отображать ли двоеточие после заголовка поля.
