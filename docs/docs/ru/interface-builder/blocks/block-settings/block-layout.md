# Макет

## Введение

Блоки, похожие на формы (например, блоки Форма, Детали и Список), поддерживают настройки макета. Варианты макета включают вертикальную компоновку (метки и поля ввода располагаются построчно) и горизонтальную компоновку (метки и поля ввода располагаются рядом). Пользователи могут настроить компоновку полей блока в соответствии с конкретными потребностями, чтобы ввод или просмотр информации был одновременно понятным и эффективным.

### Вертикальный (по умолчанию)

По умолчанию используется вертикальный макет.

### Горизонтальный

Горизонтальная компоновка отображает метки и поля ввода рядом, что эффективно экономит пространство. Для полей с длинными названиями можно выбрать вариант переноса текста или показывать многоточие, чтобы интерфейс оставался аккуратным и визуально приятным.

![20251024120237](https://static-docs.nocobase.com/20251024120237.png)

#### Выравнивание заголовка поля 

- Выровнять по левому краю (по умолчанию)
- Выровнять по правому краю

#### Ширина заголовка поля

- По умолчанию ширина — **120px**, минимальная ширина — **50px**.

#### Обработка для заголовков полей, которые превышают ширину

- **Перенос**
- **Многоточие**

#### Отображать двоеточие

Вы можете выбрать, отображать ли двоеточие после заголовка поля.
