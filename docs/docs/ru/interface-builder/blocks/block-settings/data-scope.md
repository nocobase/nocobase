# Область данных

## Введение

Установка области данных означает задание условий фильтрации по умолчанию для блока. Пользователи могут гибко менять область данных в соответствии с бизнес-требованиями, но независимо от того, какие операции фильтрации выполнены, система автоматически применит заданное условие по умолчанию, гарантируя, что данные всегда будут находиться в пределах заданной области.

## Руководство пользователя

Поле фильтра позволяет выбирать поля из текущей коллекции и связанных коллекций.

![20251027110140](https://static-docs.nocobase.com/20251027110140.png)

### Операторы

Разные типы полей поддерживают разные операторы. Например, для текстовых полей доступны операторы вроде «равно», «не равно» и «содержит»; для числовых полей — «больше чем» и «меньше чем»; а для полей дат — «находится в периоде» и «до определенной даты».

![20251027111124](https://static-docs.nocobase.com/20251027111124.png)
