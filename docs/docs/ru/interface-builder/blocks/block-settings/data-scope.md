# Установка области данных

## Введение

Установка области данных означает определение условий фильтрации по умолчанию для блока данных. Пользователи могут гибко корректировать область данных в соответствии с бизнес-потребностями, но независимо от любых операций фильтрации, система автоматически применит эти условия по умолчанию, гарантируя, что данные всегда будут оставаться в пределах указанной области.

## Руководство пользователя

Поле фильтрации позволяет выбирать поля из текущей коллекции и связанных коллекций.

![20251027110140](https://static-docs.nocobase.com/20251027110140.png)

### Операторы

Различные типы полей поддерживают разные операторы. Например, текстовые поля поддерживают операторы «равно», «не равно» и «содержит»; числовые поля — операторы «больше» и «меньше»; а поля даты — операторы «в диапазоне» и «до определенной даты».

![20251027111124](https://static-docs.nocobase.com/20251027111124.png)
