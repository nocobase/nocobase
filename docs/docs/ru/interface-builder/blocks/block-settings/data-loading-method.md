# Метод загрузки данных

## Введение

По умолчанию используется метод загрузки данных "Загружать все данные, когда фильтр пуст", где "фильтр" относится к значениям в блоке **Фильтр**. Альтернативная опция: "Не загружать данные, когда фильтр пуст".
