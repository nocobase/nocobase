# Режим загрузки данных

## Введение

Режим загрузки данных по умолчанию — «Загружать все данные, если условия фильтрации пусты». Под «условиями фильтрации» здесь понимаются значения в «Блоке фильтрации». Другой доступный вариант — «Не загружать данные, если условия фильтрации пусты».
