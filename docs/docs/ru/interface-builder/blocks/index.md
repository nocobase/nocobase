# Блок

Блоки — это контейнеры для данных и контента. Они могут размещаться на странице, в модальном окне или в выдвижном блоке, а несколько блоков можно свободно перетаскивать и упорядочивать.

## Типы блоков

![Типы блоков](https://static-docs.nocobase.com/f71af45b5cd914ea0558f760ddbbba58.png)

- Блок данных: используется для отображения данных из источника данных в интерфейсе.
- Блок фильтра: используется для выбора данных из источника данных в качестве условий фильтрации других блоков данных.
- Прочие блоки: используются для размещения специфического или независимого контента, такого как задачи рабочих процессов, журналы аудита, Markdown и т.д.

## Добавить блок

Блоки могут размещаться на странице, в модальном окне или в выдвижном блоке.

### Блоки на странице

Сейчас типы блоков на странице включают: блок данных, блок фильтра и прочие блоки.

![20251023222441](https://static-docs.nocobase.com/20251023222441.png)

### Блоки во всплывающем окне

Всплывающие окна бывают двух типов: модальные окна и выдвижные блоки. Как и на страницах, в них тоже можно добавлять блоки. Разница в том, что блоки формы во всплывающих окнах обычно используются для добавления, редактирования или просмотра одной записи. Типы блоков включают блок данных и прочие блоки.

![20251023222613](https://static-docs.nocobase.com/20251023222613.png)

## Конструктор блока

У каждого блока в правом верхнем углу есть три маленькие иконки. Слева направо:

1. Макет перетаскиванием
2. [Поток событий](/interface-builder/event-flow)
3. Настройки блока

![20251023224032](https://static-docs.nocobase.com/20251023224032.png)

Для простых блоков все параметры конфигурации сосредоточены в «Настройках блока», например для JS-блока.

![20251023224903](https://static-docs.nocobase.com/20251023224903.png)

Сложные блоки данных также предоставляют отдельные встроенные опции «Настроить поля» и «Настроить действия».

![20251023225141](https://static-docs.nocobase.com/20251023225141.png)

Кроме того, есть гибкость для более глубоких вариантов вложенности, как в блоке диаграммы.

![](https://static-docs.nocobase.com/07588190b3f41ae3060e71d8b76b4447.png)
