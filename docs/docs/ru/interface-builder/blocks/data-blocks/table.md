# Блок «Таблица»

## Введение

Блок «Таблица» — это один из основных встроенных блоков данных в **NocoBase**, который предназначен для отображения и управления структурированными данными в табличном формате. Он предлагает гибкие настройки, позволяющие вам адаптировать столбцы таблицы, их ширину, правила сортировки и область данных в соответствии с вашими потребностями, чтобы отображаемые данные соответствовали конкретным бизнес-требованиям.

#### Основные возможности:
- **Гибкая настройка столбцов**: Вы можете настроить столбцы таблицы и их ширину для различных требований к отображению данных.
- **Правила сортировки**: Поддерживается сортировка данных в таблице. Вы можете упорядочивать данные по различным полям в порядке возрастания или убывания.
- **Настройка области данных**: Установив область данных, вы можете контролировать диапазон отображаемых данных, избегая ненужной информации.
- **Настройка операций**: Блок «Таблица» включает в себя различные встроенные операции. Вы можете легко настроить такие действия, как фильтрация, создание, редактирование и удаление для быстрого управления данными.
- **Быстрое редактирование**: Поддерживается прямое редактирование данных в таблице, что упрощает рабочий процесс и повышает эффективность.

![20251023190519](https://static-docs.nocobase.com/20251023190519.gif)

### Включение древовидной таблицы

Если таблица данных является иерархической (древовидной), блок «Таблица» может использовать функцию **«Включить древовидную таблицу»**. По умолчанию эта опция отключена. После включения блок будет отображать данные в древовидной структуре и поддерживать соответствующие параметры конфигурации и операции.

![20251125205918](https://static-docs.nocobase.com/20251125205918.png)

### Разворачивать все строки по умолчанию

Если древовидная таблица включена, блок поддерживает автоматическое разворачивание всех дочерних строк при загрузке.
