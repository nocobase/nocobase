# Блок деталей

## Введение

Блок деталей используется для отображения значений полей каждой записи данных. Он поддерживает гибкие макеты полей и имеет встроенные функции действий с данными, что удобно для просмотра и управления информацией.

## Настройки блока

![20251029202614](https://static-docs.nocobase.com/20251029202614.png)

### Правила связывания блока

Управляйте поведением блока (например, отображать ли его или выполнять JavaScript) через правила связывания.

![20251023195004](https://static-docs.nocobase.com/20251023195004.png)

Для подробностей см. [Правила связывания](/interface-builder/linkage-rule)

### Установить область данных

Пример: отображать только оплаченные заказы

![20251023195051](https://static-docs.nocobase.com/20251023195051.png)

Для подробностей см. [Установить область данных](/interface-builder/blocks/block-settings/data-scope)

### Правила связывания полей

Правила связывания в блоке деталей поддерживают динамическую настройку показа/скрытия полей.

Пример: не отображать сумму, когда статус заказа — «Отменён».

![20251023200739](https://static-docs.nocobase.com/20251023200739.png)

Для подробностей см. [Правила связывания](/interface-builder/linkage-rule)

## Настройка полей

### Поля из этой коллекции

> **Примечание**: поля из наследуемых коллекций (то есть полей родительской коллекции) автоматически объединяются и отображаются в текущем списке полей.

![20251023201012](https://static-docs.nocobase.com/20251023201012.png)

### Поля из связанных коллекций

> **Примечание**: поддерживается отображение полей из связанных коллекций (сейчас только для связей «один-к-одному»).

![20251023201258](https://static-docs.nocobase.com/20251023201258.png)

### Другие поля

- Поле JS
- Элемент JS
- Разделитель
- Markdown

![20251023201514](https://static-docs.nocobase.com/20251023201514.png)

> **Совет**: можно писать JavaScript для реализации пользовательского отображения контента, что позволяет показывать более сложную информацию.  
> Например, можно рендерить разные эффекты отображения в зависимости от типа данных, условий или логики.

![20251023202017](https://static-docs.nocobase.com/20251023202017.png)

### Шаблон поля

Шаблоны полей позволяют переиспользовать конфигурацию секции полей (выбранные поля, макет и настройки полей) в блоках деталей. См. [Шаблон поля](../../ui-templates.md).

![field-template-menu-20251228](https://static-docs.nocobase.com/field-template-menu-20251228.png)
