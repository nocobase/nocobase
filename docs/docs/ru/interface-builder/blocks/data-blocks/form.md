# Блок формы

## Введение

Блок формы — это важный элемент для создания интерфейсов ввода и редактирования данных. Он обладает высокой степенью настраиваемости: на основе модели данных используются соответствующие компоненты для отображения необходимых полей. С помощью потоков событий, таких как правила связывания, блок формы может динамически отображать поля. Кроме того, его можно интегрировать с рабочими процессами для запуска автоматизированных операций и обработки данных, что ещё больше повышает эффективность работы или позволяет реализовать сложную логику.

## Добавление блока формы

- **Редактировать форму**: Используется для изменения существующих данных.
- **Добавить форму**: Используется для создания новых записей данных.

### Макет

Блок формы поддерживает два режима макета, которые можно настроить с помощью атрибута `layout`:

- **horizontal** (горизонтальный): Этот макет отображает метку и содержимое в одной строке, экономя вертикальное пространство. Он подходит для простых форм или случаев с небольшим объемом информации.
- **vertical** (вертикальный) (по умолчанию): Метка располагается над полем. Этот макет делает форму более удобной для чтения и заполнения, особенно для форм с несколькими полями или сложными элементами ввода.
