# Блок календаря

## Введение

Блок календаря наглядно отображает события и связанные с датами данные в календарном виде; подходит для типичных сценариев — планирования встреч и мероприятий.

![](https://static-docs.nocobase.com/Calendar-04-13-2026_03_27_PM.png)

## Установка

Это встроенный плагин, дополнительная установка не требуется.

## Добавление блока

![](https://static-docs.nocobase.com/Calendar-04-13-2026_03_39_PM.png)

Выберите блок «Календарь» и укажите соответствующую таблицу данных, чтобы завершить создание.

## Параметры блока

![](https://static-docs.nocobase.com/Calendar-04-23-2026_10_38_AM.png)

### Поле заголовка

Используется для отображения заголовка на полосе календарного события.

В настоящее время поддерживаются следующие типы полей: `input`, `select`, `phone`, `email`, `radioGroup`, `sequence`. Дополнительные типы можно подключить через плагины.

### Поле даты начала

Определяет время начала события.

### Поле даты окончания

Определяет время окончания события.

### Быстрое создание события

При нажатии на пустую область с датой в календаре открывается всплывающее окно для быстрого создания события.

![](https://static-docs.nocobase.com/Add-new-04-23-2026_10_39_AM.png)

При нажатии на существующее событие:
- текущее событие подсвечивается;
- одновременно открывается окно с подробностями для просмотра или редактирования.

### Отображать лунный календарь

После включения в календаре будет отображаться соответствующая информация по китайскому лунному календарю.

![](https://static-docs.nocobase.com/Calendar-04-23-2026_10_41_AM.png)

### Область данных

Используется для ограничения данных, отображаемых в блоке календаря.

![](https://static-docs.nocobase.com/Calendar-04-13-2026_03_30_PM.png)

Подробнее см. [Настройка области данных](/interface-builder/blocks/block-settings/data-scope).

### Высота блока

Высоту блока календаря можно задавать вручную, чтобы избежать появления внутренней полосы прокрутки и улучшить общий вид компоновки.

![](https://static-docs.nocobase.com/Calendar-04-13-2026_03_30_PM%20(1).png)

Подробнее см. [Высота блока](/interface-builder/blocks/block-settings/block-height).

### Поле цвета

Используется для настройки фонового цвета событий в календаре, что улучшает визуальное различение.

Шаги настройки:

1. В таблице данных создайте поле **«Выпадающий список (Single select)»** или **«Радиогруппа (Radio group)»** и задайте цвет для каждой опции.
2. В настройках блока календаря назначьте это поле как «Поле цвета».
3. При создании или редактировании события выберите цвет — он применится к событию в календаре.

![](https://static-docs.nocobase.com/Calendar-04-23-2026_10_41_AM%20(1).png)

### Начало недели

Поддерживается выбор дня начала недели:
- воскресенье;
- понедельник (по умолчанию).

Можно настроить в соответствии с привычками региона, чтобы календарь лучше соответствовал реальным потребностям.


## Настройка операций

![](https://static-docs.nocobase.com/Calendar-04-13-2026_03_32_PM.png)

### Сегодня

Нажатие на кнопку «Сегодня» быстро возвращает к виду календаря, содержащему текущую дату.

### Переключение страниц

Перемещение по времени зависит от текущего режима просмотра:
- Режим месяца — предыдущий/следующий месяц;
- Режим недели — предыдущая/следующая неделя;
- Режим дня — вчера/завтра.

### Выбор представления

Поддерживается переключение между следующими видами:
- Месячный (по умолчанию);
- Недельный;
- Дневной.

### Заголовок

Отображает дату, соответствующую текущему виду.
