# Диаграмма

## Введение

Плагин визуализации данных NocoBase предоставляет визуальные запросы к данным и богатый набор компонентов диаграмм. Пользователи могут быстро собирать дашборды визуализации с помощью простой конфигурации, чтобы отображать инсайты и поддерживать многомерный анализ и представление данных.
