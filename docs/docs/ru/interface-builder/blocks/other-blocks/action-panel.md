# Блок: Рабочее место

## Введение

Используется для размещения различных быстрых действий. В настоящее время поддерживаются:

- [Ссылка](/interface-builder/actions/types/link)
- [Сканирование QR-кода]
- [Действие со всплывающим окном](/interface-builder/actions/types/pop-up)

![20251023225754](https://static-docs.nocobase.com/20251023225754.png)

> 💡 В будущих версиях будут постоянно добавляться другие типы действий.
## Типы действий

- **Ссылка**: переход на указанную страницу или внешний URL.

![20251023230643](https://static-docs.nocobase.com/20251023230643.png)

- **Сканирование QR-кода**: запускает соответствующее действие путем сканирования QR-кода.

- **Открыть всплывающее окно**: открывает всплывающее окно для отображения содержимого блока.
![20251023230522](https://static-docs.nocobase.com/20251023230522.png)
