# Макеты UI

NocoBase предоставляет макет для компьютера и мобильный макет. В обоих макетах можно создавать страницы с помощью конструктора интерфейса, а затем настраивать на них блоки, поля и действия.

Макет для компьютера используется по умолчанию и подходит для повседневного управления и обработки данных на компьютере. Если нужны отдельная мобильная навигация и мобильные страницы, можно дополнительно настроить мобильный макет.

## Макет для компьютера

[Макет для компьютера](./desktop.md) по умолчанию доступен по адресу `/admin`. Он состоит из верхней навигации, боковой навигации и области содержимого страницы и подходит для распространённых рабочих сценариев — управления таблицами, ввода данных через формы и просмотра данных.

Макет для компьютера также адаптируется к узким экранам. Когда страница отображается на узком экране, навигация, отступы и часто используемые компоненты принимают более удобный для небольшого экрана вид. При этом используются те же меню и страницы макета для компьютера.

## Мобильный макет

[Мобильный макет](./mobile.md) по умолчанию доступен по адресу `/mobile`. Для навигации первого уровня он использует нижнюю панель вкладок и предоставляет отдельные мобильные страницы, ссылки и вкладки страниц.

Мобильный макет подходит для задач, которые часто выполняются с телефона, — например, для ввода данных на месте, согласования, обработки задач и поиска данных. Страницы можно создавать и предварительно просматривать в браузере на компьютере, а затем проверять на телефоне с помощью QR-кода.

## Какой макет выбрать

По умолчанию достаточно макета для компьютера.

| Я хочу… | Рекомендуемый вариант |
| --- | --- |
| Работать в основном на компьютере и лишь иногда открывать приложение с телефона | [Макет для компьютера](./desktop.md) |
| Создать для телефона отдельные навигацию, страницы и процессы | [Мобильный макет](./mobile.md) |
| Обеспечить полноценную работу и на компьютере, и на мобильных устройствах | Настроить макет для компьютера и мобильный макет отдельно |

## Как связаны настройки

Макет для компьютера и мобильный макет используют одни и те же источники данных, коллекции и бизнес-данные. На основе одной коллекции можно создать отдельные страницы, подходящие для разных устройств.

Меню, маршруты и страницы настраиваются отдельно. Изменения страницы для компьютера не обновляют мобильную страницу автоматически, а изменения мобильной навигации не влияют на навигацию для компьютера. [Права доступа к маршрутам](../../users-permissions/acl/permissions.md) для каждого макета также настраиваются отдельно.
