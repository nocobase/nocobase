# Установка create-nocobase-app

:::tip Предварительные требования

- Установлены Node.js 20+, Yarn 1.22.x
- Настроена и запущена одна из баз данных: MySQL 8.0.17+, MariaDB 10.9+, PostgreSQL 10+
- Из-за особенностей сетевого окружения в Китае рекомендуется сменить зеркало (необязательно)

```bash
$ yarn config set disable-self-update-check true
$ yarn config set registry https://registry.npmmirror.com/
```

:::

:::warning Описание переменных окружения

- `TZ` используется для настройки часового пояса приложения, по умолчанию используется часовой пояс операционной системы
- `APP_KEY` — это секретный ключ приложения, используемый для генерации токенов пользователей и т. д. (если `APP_KEY` будет изменен, старые токены также станут недействительными). Это может быть любая случайная строка. **Пожалуйста, измените его на свой собственный ключ и убедитесь, что он не разглашается**
- `DB_*` — конфигурации базы данных, пожалуйста, измените их в соответствии с вашими фактическими данными подключения

:::

## 2. Переход в каталог

```bash
cd my-nocobase-app
```

## 3. Установка зависимостей

📢 Из-за сетевого окружения, конфигурации системы и других факторов этот шаг может занять более десяти минут.

```bash
yarn install
```

:::tip Совет для рабочей среды

При развертывании в рабочей среде для уменьшения размера можно установить только необходимые зависимости:

```bash
yarn install --production
```

:::

## 4. Установка NocoBase

```bash
yarn nocobase install --lang=ru-RU
```

В процессе установки автоматически будет создана структура таблиц базы данных и инициализированы данные.

## 5. Запуск NocoBase

**Среда разработки**

```bash
yarn dev
```

**Рабочая среда**

```bash
yarn start
```

## 6. Вход в NocoBase

Откройте [http://localhost:13000](http://localhost:13000) в браузере, логин и пароль по умолчанию: `admin@nocobase.com` и `admin123`.

:::warning Совет по безопасности аккаунта

После первого входа, пожалуйста, своевременно измените пароль по умолчанию для обеспечения безопасности системы.

:::
