# Кластерное развертывание

NocoBase поддерживает развертывание приложений в кластерном режиме для повышения их производительности и доступности. Кластерное развертывание позволяет запускать различные экземпляры приложения на нескольких серверах или несколько процессов в многоядерном режиме на одном сервере.

Более подробную информацию смотрите в документации по [Кластерному режиму](/cluster-mode/).
