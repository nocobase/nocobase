# Кластерное развёртывание

NocoBase поддерживает развёртывание приложений в кластерном режиме для повышения производительности и доступности. Кластерное развёртывание позволяет запускать разные экземпляры приложения на нескольких серверах либо запускать несколько процессов в режиме multi-core на одном сервере.

Для подробностей см. документацию по [Кластерному режиму](/cluster-mode/).
