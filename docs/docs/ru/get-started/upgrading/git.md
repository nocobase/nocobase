# Обновление установки из исходного кода Git

:::warning Подготовка перед обновлением

- Сначала обязательно сделайте резервную копию базы данных
- Остановите запущенный экземпляр NocoBase (`Ctrl + C`)

:::

## 1. Перейдите в каталог проекта NocoBase

```bash
cd my-nocobase-app
```

## 2. Получите последний код

```bash
git pull
```

## 3. Удалите кэш и старые зависимости

Если стандартный процесс обновления завершился с ошибкой, можно попробовать очистить кэш и зависимости, а затем скачать их заново.

```bash
# Очистите кэш nocobase
yarn nocobase clean
# Удалите зависимости
yarn rimraf -rf node_modules # эквивалентно rm -rf node_modules
```

## 4. Обновите зависимости

📢 Из-за таких факторов, как сетевая среда и конфигурация системы, этот следующий шаг может занять более десяти минут.

```bash
yarn install
```

## 5. Выполните команду обновления

```bash
yarn nocobase upgrade
```

## 6. Запустите NocoBase

```bash
yarn dev
```

:::tip Подсказка для производственной среды

Не рекомендуется развёртывать установку NocoBase из исходного кода напрямую в производственной среде (для производственной среды см. [Развёртывание в производственной среде](../deployment/production.md)).

:::
