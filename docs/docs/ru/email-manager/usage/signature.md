# Подпись электронной почты

## Настройка подписи

Перейдите в Центр управления почтой и нажмите кнопку настроек:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_05_34_PM.png)

Переключитесь на панель подписей:
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_05_51_PM.png)

Здесь вы можете управлять подписями. Установите флажок перед подписью, чтобы сделать ее подписью по умолчанию:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_05_37_PM.png)

## Использование подписей

В теле письма нажмите на значок подписи, чтобы выбрать нужную подпись:
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_05_40_PM.png)
