# Подпись почты

## Настройка подписи

Перейдите в Центр управления почтой и нажмите кнопку Настройки.
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_08_PM.png)

Переключитесь на панель Подпись.
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_08_PM%20(1).png)

Здесь вы можете управлять подписями. Выбериите подпись, чтобы установить её как подпись по умолчанию.
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_09_PM.png)

## Использование подписи

В окне создания письма нажмите значок подписи, чтобы выбрать нужную подпись.
![](https://static-docs.nocobase.com/email-manager/Loading--12-31-2025_10_11_PM.png)
