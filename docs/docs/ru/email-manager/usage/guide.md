# Центр электронной почты

## Введение
После активации плагина электронной почты по умолчанию становится доступен Центр электронной почты. Он предназначен для подключения аккаунтов, управления почтой, настройки функций и многого другого.

Чтобы перейти на страницу управления почтой, нажмите на значок сообщения электронной почты в правом верхнем углу.

![](https://static-docs.nocobase.com/mail-1733816161753.png)
## Привязка аккаунта

### Привязать аккаунт
Нажмите кнопку "Account setting" (Настройки аккаунта), затем в открывшемся всплывающем окне нажмите кнопку "Link account" (Привязать аккаунт) и выберите тип почтового аккаунта, который вы хотите привязать.

![](https://static-docs.nocobase.com/mail-1733816162279.png)

Браузер автоматически откроет соответствующую страницу входа в почтовый сервис. Войдите в свой аккаунт и предоставьте необходимые разрешения.

![](https://static-docs.nocobase.com/mail-1733816162534.png)

После завершения авторизации вы будете перенаправлены обратно на страницу NocoBase для привязки аккаунта и синхронизации данных (первая синхронизация может занять некоторое время, пожалуйста, подождите).

![](https://static-docs.nocobase.com/mail-1733816162794.png)

Как только синхронизация данных завершится, текущая страница автоматически закроется, и вы вернетесь на исходную страницу сообщений электронной почты, где увидите, что аккаунт успешно привязан.

![](https://static-docs.nocobase.com/mail-1733816163177.png)

Нажмите на область затемнения, чтобы закрыть всплывающее окно, и вы увидите список писем.

![](https://static-docs.nocobase.com/mail-1733816163503.png)

### Удалить аккаунт
Вы можете нажать "Delete" (Удалить), чтобы удалить аккаунт и связанные с ним письма.

![](https://static-docs.nocobase.com/mail-1733816163758.png)


## Управление почтой

### Фильтрация писем
На странице управления почтой левая часть отведена под область фильтров, а правая — под список писем. По умолчанию при открытии страницы отображается папка "Входящие".

![](https://static-docs.nocobase.com/mail-1733816165536.png)

Письма с одинаковой темой группируются, а рядом с полем темы указывается общее количество писем в переписке.
Если некоторые письма в рамках одной темы соответствуют условиям фильтрации, будет отображено корневое письмо этой переписки, а рядом с полем темы будет указан тип текущего корневого письма.

![](https://static-docs.nocobase.com/mail-1733816165797.png)

Заголовки непрочитанных писем отображаются жирным шрифтом, а рядом со значком почты вверху будет указано количество непрочитанных писем.

![](https://static-docs.nocobase.com/mail-1733816166067.png)

### Ручная синхронизация писем
Текущий интервал синхронизации писем составляет 5 минут. Если вы хотите принудительно синхронизировать письма, нажмите кнопку "Refresh" (Обновить).

![](https://static-docs.nocobase.com/mail-1733816166364.png)

### Изменение статуса прочтения
Кнопки "Mark as read" (Пометить как прочитанное) и "Mark as unread" (Пометить как непрочитанное) позволяют массово изменять статус прочтения писем.

![](https://static-docs.nocobase.com/mail-1733816166621.png)

### Отправка письма
Нажмите кнопку "Write email" (Написать письмо) в верхней части страницы, чтобы открыть панель создания письма.

![](https://static-docs.nocobase.com/mail-1733816166970.png)

После заполнения необходимой информации вы можете отправить письмо. В настоящее время размер вложений ограничен 3 МБ.

![](https://static-docs.nocobase.com/mail-1733816167214.png)

### Просмотр письма
Нажмите кнопку "View" (Просмотреть) в строке, чтобы увидеть детали письма. В настоящее время доступны два формата: для одиночного письма вы можете сразу увидеть подробную информацию.

![](https://static-docs.nocobase.com/mail-1733816167456.png)

Другой формат — для нескольких писем с одинаковой темой. Они по умолчанию отображаются в виде списка, который можно разворачивать и сворачивать.

![](https://static-docs.nocobase.com/mail-1733816167750.png)

После просмотра деталей письма его статус по умолчанию изменится на "прочитано". Вы можете нажать кнопку "..." справа и выбрать "Mark as unread" (Пометить как непрочитанное), чтобы вернуть письму статус "непрочитано".

### Ответить на письмо
После перехода к деталям письма внизу вы найдете кнопку "Reply" (Ответить), с помощью которой можно ответить на письмо. Если в переписке участвует несколько человек, вы можете нажать "Reply all" (Ответить всем), чтобы отправить ответ всем участникам.

![](https://static-docs.nocobase.com/mail-1733816167998.png)

### Переслать письмо
Вы можете нажать кнопку "Forward" (Переслать) внизу, чтобы переслать письмо другим получателям.

![](https://static-docs.nocobase.com/mail-1733816168241.png)
