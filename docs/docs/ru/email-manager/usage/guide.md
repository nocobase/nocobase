# Центр почты

## Введение
После включения плагина **Менеджер почты** система по умолчанию предоставляет центр управления почтой для интеграции аккаунтов, управления письмами и настройки функций.

Нажмите на значок почтовых сообщений в правом верхнем углу, чтобы перейти на страницу управления почтой.

![](https://static-docs.nocobase.com/email-manager/Loading--12-31-2025_09_02_PM.png)
## Привязка аккаунта

### Привязать аккаунт
Нажмите кнопку **Настройки** и после открытия всплывающего окна нажмите кнопку **Привязать аккаунт**, чтобы выбрать тип почты, который вы хотите привязать.

![](https://static-docs.nocobase.com/mail-1733816162279.png)

Браузер автоматически откроет страницу входа соответствующего почтового сервиса. Войдите в свой аккаунт и выполните авторизацию (процесс авторизации зависит от провайдера).

![](https://static-docs.nocobase.com/mail-1733816162534.png)

После завершения авторизации вы будете перенаправлены обратно в NocoBase. Выберите время начала синхронизации, чтобы привязать аккаунт и синхронизировать данные (первая синхронизация может занять много времени — пожалуйста, подождите).

![](https://static-docs.nocobase.com/mail-1733816162794.png)

После завершения синхронизации данных текущая страница автоматически закроется и вернёт вас на исходную страницу почтовых сообщений. В этот момент вы увидите, что аккаунт привязан.

![](https://static-docs.nocobase.com/mail-1733816163177.png)

Нажмите на область затемнения, чтобы закрыть всплывающее окно, и вы увидите список писем.

![](https://static-docs.nocobase.com/mail-1733816163503.png)

### Удалить аккаунт
Нажмите **Удалить**, чтобы удалить аккаунт и связанные письма.

![](https://static-docs.nocobase.com/mail-1733816163758.png)


## Управление почтой

### Фильтрация писем
Левая часть страницы управления почтой — это область фильтрации, правая — область списка писем.

![](https://static-docs.nocobase.com/mail-1733816165536.png)

Письма с одинаковой темой группируются, а рядом с полем темы указывается общее количество писем в переписке.
Если некоторые письма в рамках одной темы соответствуют условиям фильтрации, будет отображено корневое письмо этой переписки, а рядом с полем темы будет указан тип текущего корневого письма.

![](https://static-docs.nocobase.com/mail-1733816165797.png)

Заголовки непрочитанных писем отображаются жирным шрифтом, а рядом со значком почты вверху будет указано количество непрочитанных писем.

![](https://static-docs.nocobase.com/mail-1733816166067.png)

### Ручная синхронизация писем
Текущий интервал синхронизации писем составляет 5 минут. Если вы хотите принудительно синхронизировать письма, нажмите кнопку "Обновить".

![](https://static-docs.nocobase.com/mail-1733816166364.png)

### Изменение статуса прочтения
Кнопки "Пометить как прочитанное" и "Пометить как непрочитанное" позволяют массово изменять статус прочтения писем.

![](https://static-docs.nocobase.com/mail-1733816166621.png)

### Отправка почты
Нажмите кнопку **Отправить письмо** сверху, чтобы открыть панель отправки.

![](https://static-docs.nocobase.com/mail-1733816166970.png)

Заполните нужную информацию и отправьте письмо. Вложения поддерживаются только размером до 3 MБ.

![](https://static-docs.nocobase.com/mail-1733816167214.png)

### Просмотр письма
Нажмите кнопку "Просмотреть" в строке, чтобы увидеть детали письма. В настоящее время доступны два формата: для одиночного письма вы можете сразу увидеть подробную информацию.

![](https://static-docs.nocobase.com/mail-1733816167456.png)

Несколько писем с одинаковой темой по умолчанию отображаются списком — вы можете нажать, чтобы развернуть или свернуть.

![](https://static-docs.nocobase.com/mail-1733816167750.png)

После просмотра деталей письма статус письма по умолчанию становится «прочитано». Вы можете нажать кнопку **...** справа и выбрать **Пометить как непрочитанное**, чтобы сделать его непрочитанным.

### Ответ и пересылка

### Ответить на письмо
После перехода к деталям письма внизу вы найдете кнопку "Ответить", с помощью которой можно ответить на письмо. Если в переписке участвует несколько человек, вы можете нажать "Ответить всем", чтобы отправить ответ всем участникам.

![](https://static-docs.nocobase.com/mail-1733816167998.png)

### Переслать письмо
Вы можете нажать кнопку "Переслать" внизу, чтобы переслать письмо другим получателям.

![](https://static-docs.nocobase.com/mail-1733816168241.png)
