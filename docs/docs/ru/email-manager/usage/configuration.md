# Настройка блока

## Блок почтовых сообщений

### Добавить блок

На странице настройки нажмите кнопку **Создать блок** и выберите блок **Таблица почты**, чтобы добавить блок почтовых сообщений.

![](https://static-docs.nocobase.com/email-manager/Email-12-31-2025_09_56_PM.png)

### Настройка полей

Нажмите кнопку **Поля** у блока, чтобы выбрать поля, которые нужно отображать. Подробные операции см. в настройке полей таблицы.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_37_PM.png)

### Установить область данных

В правой панели настроек блока можно выбрать область данных: все письма или письма текущего вошедшего пользователя.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_39_PM.png)

Вы можете отфильтровать электронные письма с одинаковым суффиксом, используя переменные:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_41_PM.png)

## Блок "Детали электронной почты"

Сначала активируйте функцию **Включить открытие по клику** для поля в блоке сообщений электронной почты:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_45_PM.png)

Добавьте блок **Детали почты** во всплывающее окно.
![](https://static-docs.nocobase.com/email-manager/Email-12-31-2025_10_02_PM.png)

Теперь вы можете просматривать подробное содержимое письма.
![](https://static-docs.nocobase.com/email-manager/Email-12-31-2025_10_03_PM.png)

В нижней части вы можете самостоятельно настроить необходимые кнопки.

## Блок отправки почты

Есть два способа создать форму отправки почты:

1. Добавьте кнопку **Отправить письмо** в верхней части таблицы:
   ![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_52_PM.png)

2. Добавьте блок **Отправка почты**:
   ![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_53_PM.png)

Оба способа создают полноценную форму отправки почты.
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_53_PM%20(1).png)

Каждое поле формы почты соответствует обычным формам и может настраиваться, например, с **значениями по умолчанию** или **правилами связывания** и т. п.

> Формы ответа и пересылки, расположенные в нижней части блока деталей электронной почты, по умолчанию содержат некоторую обработку данных. Вы можете изменить ее с помощью **Движка потоков**.
