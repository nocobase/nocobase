# Конфигурация блоков

## Блок "Сообщения электронной почты"

### Добавление блока

Чтобы добавить блок сообщений электронной почты, на странице конфигурации нажмите кнопку **Создать блок** и выберите **Сообщения электронной почты (Все)** или **Сообщения электронной почты (Личные)**.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_03_58_PM.png)

### Настройка полей

Нажмите кнопку **Поля** блока, чтобы выбрать отображаемые поля. Подробную информацию о настройке полей вы найдете в разделе, посвященном таблицам.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_37_PM.png)

### Настройка фильтрации данных

Нажмите на значок настроек справа от таблицы и выберите **Диапазон данных**, чтобы задать диапазон данных для фильтрации электронных писем.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_39_PM.png)

Вы можете отфильтровать электронные письма с одинаковым суффиксом, используя переменные:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_41_PM.png)

## Блок "Детали электронной почты"

Сначала активируйте функцию **Включить открытие по клику** для поля в блоке сообщений электронной почты:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_45_PM.png)

Во всплывающем окне добавьте блок **Детали электронной почты**:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_46_PM.png)

Вы сможете просмотреть подробное содержимое электронного письма:
![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_04_49_PM.png)

В нижней части вы можете самостоятельно настроить необходимые кнопки.

## Блок "Отправка электронной почты"

Существует два способа создания формы отправки электронной почты:

1. Добавьте кнопку **Отправить письмо** в верхней части таблицы:
   ![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_52_PM.png)

2. Добавьте блок **Отправка электронной почты**:
   ![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_53_PM.png)

В итоге оба этих способа позволяют создать полноценную форму отправки электронных писем:
![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_04_53_PM%20(1).png)

Каждое поле в форме отправки письма аналогично полям обычной формы, и вы можете настроить для него **Значение по умолчанию** или **Правила связывания** и т.д.

> Формы ответа и пересылки, расположенные в нижней части блока деталей электронной почты, по умолчанию содержат некоторую обработку данных. Вы можете изменить ее с помощью **FlowEngine**.
