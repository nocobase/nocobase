# Массовая отправка

## Обзор массовой отправки
Вы можете отправлять электронные письма нескольким получателям одновременно (каждый получатель получит отдельное письмо).

![](https://static-docs.nocobase.com/email-manager/Loading--12-31-2025_10_15_PM.png)

## Массовая отправка с помощью таблицы
Вы также можете выполнить массовую отправку писем с помощью таблицы:

1. Добавьте кнопку отправки письма в таблицу.

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_04_PM.png)

2. Настройте поле для электронной почты в таблице (оно будет использоваться для извлечения адресов после выбора строк таблицы).

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_17_PM%20(1)

3. Выберите данные в таблице для отправки писем.

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_19_PM.png)

Если вам нужно отправить письма на все адреса в таблице, вы можете включить функцию **По умолчанию выбирать все записи текущей таблицы в качестве получателей**.

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_18_PM%20(1)

После редактирования содержимого письма нажмите кнопку массовой отправки, чтобы отправить его.

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_19_PM.png)

## Отслеживание отправки писем
Вы можете просматривать и отслеживать статус отправки писем в Центре управления электронной почтой.

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_20_PM.png).png)
