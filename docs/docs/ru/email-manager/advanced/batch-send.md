# Массовая отправка

## Обзор массовой отправки
Поддерживается пакетная отправка писем сразу нескольким получателям, при этом каждый получатель получит отдельное письмо.

![](https://static-docs.nocobase.com/email-manager/Loading--12-31-2025_10_15_PM.png)

## Массовая отправка через таблицу

Вы можете настроить в таблице массовый выбор и отправку адреса электронной почты.

1. Добавьте в таблицу кнопку отправки почты.
![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_04_PM.png)

2. Настройте поле получателя (система извлечёт почтовые адреса из данных выбранных строк таблицы).

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_17_PM%20(1).png)

3. Отметьте строки таблицы, которым нужно отправить.

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_19_PM.png)

Если нужно отправить всем записям таблицы, вы можете установить область получателей в **Все записи**.

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_18_PM%20(1).png)

После редактирования содержимого письма нажмите кнопку **Массовая отправка**, чтобы завершить отправку.

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_19_PM.png)

## Отслеживание отправки почты
После отправки вы можете посмотреть статус отправки и информацию о письмах в Центре управления почтой.

![](https://static-docs.nocobase.com/email-manager/User-12-31-2025_10_20_PM.png)
