# Автосохранение черновика

Вы можете включить функцию **Автосохранение черновика** на кнопке **Сохранить черновик**.

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_29_PM.png)

После включения система будет автоматически сохранять черновики в реальном времени в процессе редактирования письма, чтобы никакой контент не был потерян.

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_31_PM.png)

Когда вы снова откроете редактор письма с тем же отправителем и получателем, система спросит, нужно ли использовать сохранённый черновик.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_08_35_PM.png)
