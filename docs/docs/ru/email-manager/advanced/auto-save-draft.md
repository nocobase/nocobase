# Автосохранение черновика

В меню кнопки **Сохранить черновик** вы можете включить **Автосохранение черновика**.

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_29_PM.png)

В процессе редактирования письма система сохраняет черновик в реальном времени, гарантируя сохранность содержимого.

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_31_PM.png)

Когда вы снова откроете редактор писем (для того же отправителя и получателя), система предложит использовать сохраненный черновик.

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_08_35_PM.png)
