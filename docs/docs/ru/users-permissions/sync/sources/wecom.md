# Синхронизация данных пользователей из WeChat Work

## Введение

Плагин **WeChat Work** позволяет синхронизировать данные пользователей и отделов из WeChat Work.

## Создайте и настройте собственное приложение WeChat Work

Сначала вам нужно создать собственное приложение в административной консоли WeChat Work и получить **Corp ID**, **AgentId** и **Secret**.

См. также [Аутентификация пользователей — WeChat Work](/auth-verification/auth-wecom/).

## Добавление источника данных для синхронизации в NocoBase

Перейдите в Пользователи и разрешения - Синхронизация - Добавить и заполните полученную информацию.

![](https://static-docs.nocobase.com/202412041251867.png)

## Настройка синхронизации контактов

Перейдите в административную консоль WeChat Work - Безопасность и управление - Инструменты управления и нажмите на Синхронизация контактов.

![](https://static-docs.nocobase.com/202412041249958.png)

Настройте, как показано на рисунке, и укажите доверенный IP-адрес предприятия.

![](https://static-docs.nocobase.com/202412041250776.png)

Теперь вы можете приступить к синхронизации пользовательских данных.

## Настройка сервера для получения событий

Если вы хотите, чтобы изменения данных пользователей и отделов в WeChat Work оперативно синхронизировались с приложением NocoBase, вы можете выполнить дополнительные настройки.

После заполнения предыдущих настроек вы можете скопировать URL-адрес обратного вызова для уведомлений о контактах.

![](https://static-docs.nocobase.com/202412041256547.png)

Вставьте его в настройки WeChat Work, получите Token и EncodingAESKey и завершите настройку источника данных для синхронизации пользователей NocoBase.

![](https://static-docs.nocobase.com/202412041257947.png)
