# Роли

## Центр управления

### Управление ролями

![](https://static-docs.nocobase.com/da7083c67d794e23dc6eb0f85b1de86c.png)

В приложении есть две предопределённые роли: «Admin» и «Member», каждая с отличающимися настройками прав по умолчанию, адаптированными под свои функции.

### Добавление, удаление и изменение ролей

Идентификатор роли — уникальный системный идентификатор. Он позволяет настраивать роли по умолчанию, но предопределённые роли системы удалить нельзя.

![](https://static-docs.nocobase.com/35f323b346db4f9f12f9bee4dea63302.png)

### Установка роли по умолчанию

Роль по умолчанию автоматически назначается новым пользователям, если при их создании не указана конкретная роль.

![](https://static-docs.nocobase.com/f41bba7ff55ca28715c486dc45bc1708.png)

## Личный кабинет

### Переключение ролей

Пользователю может быть назначено несколько ролей, и он может переключаться между ними в личном кабинете.

![](https://static-docs.nocobase.com/e331d11ec1ca3b8b7e0472105b167819.png)

Роль по умолчанию при входе определяется последней выбранной ролью (это значение обновляется при каждом переключении) или, если такой нет, первой ролью (системной ролью по умолчанию).
