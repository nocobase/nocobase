# Роли

## Центр управления

### Управление ролями

![](https://static-docs.nocobase.com/da7083c67d794e23dc6eb0f85b1de86c.png)

Приложение поставляется с двумя встроенными ролями: "Admin" и "Member". Каждая из них имеет свои уникальные настройки разрешений по умолчанию.

### Добавление, удаление и изменение ролей

Идентификатор роли — это уникальный системный идентификатор. Вы можете настраивать роли по умолчанию, но системные роли удалить нельзя.

![](https://static-docs.nocobase.com/35f346db4f9f12f9bee4dea63302.png)

### Настройка роли по умолчанию

Здесь под ролью по умолчанию подразумевается та роль, которая автоматически назначается новым пользователям, если при их создании не была указана конкретная роль.

![](https://static-docs.nocobase.com/f41bba7ff55ca28715c486dc45bc1708.png)

## Личный кабинет

### Переключение ролей

Одному пользователю можно назначить несколько ролей. Если у пользователя есть несколько ролей, он может переключаться между ними в личном кабинете.

![](https://static-docs.nocobase.com/e331d11ec1ca3b8b7e0472105b167819.png)

При входе в систему роль по умолчанию для пользователя определяется по следующему приоритету: сначала используется роль, на которую пользователь переключился в последний раз (это значение обновляется при каждом переключении), затем — первая роль (системная роль по умолчанию).
