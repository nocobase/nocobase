# Пользователи

## Управление пользователями

Этот плагин предоставляет интерфейс управления пользователями в центре конфигурации. После инициализации приложения автоматически создается учетная запись суперпользователя, которую нельзя удалить. Суперпользователь имеет права Root и может получить доступ ко всем ресурсам.

![](https://static-docs.nocobase.com/44bf40f56b45d4dd96c424fb08082cf6.png)

Кроме того, вы также можете использовать коллекцию Пользователи на странице для добавления различных блоков для управления пользователями, например, табличные блоки:

![](https://static-docs.nocobase.com/76b5a4652f869541a9e8f18a4568a7c9.png)
