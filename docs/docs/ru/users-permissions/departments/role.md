# Управление ролями отделов

Привязка одной или нескольких ролей к отделу позволяет сотрудникам этого отдела получить эти роли.

Вы можете привязать роли к отделу двумя способами.

Первый способ — выбрать роль в поле «Роль» формы редактирования отдела.

![](https://static-docs.nocobase.com/70f77bb89aa1fb415c152a51a51cc23b.png)

Второй способ — добавить отделы к соответствующей роли в разделе управления ролями.

![](https://static-docs.nocobase.com/f2a7bec8937cf2f179ce868a92b98416d.png)

Отделы, которым уже назначена эта роль, недоступны для выбора.

![](https://static-docs.nocobase.com/be10299893581e1f97a4e01ddd5c7e59.png)
