## Управление сотрудниками отдела

### Просмотр списка сотрудников отдела

![](https://static-docs.nocobase.com/2aaf4d9bf55da105b5fca4e9f7e23ca7.png)

### Добавление сотрудников в отдел

Пользователь может одновременно входить в несколько отделов. Первый отдел, в который он вступает, по умолчанию становится основным. У пользователя может быть только один основной отдел.

![](https://static-docs.nocobase.com/60afd282f33b555e6fe0662b9da544cc.png)

Пользователи, которые уже являются сотрудниками отдела, не будут отображаться в списке выбора пользователей.

![](https://static-docs.nocobase.com/6bcd93173c169973f970de35d2657993.png)

### Назначение руководителя отдела

В форме редактирования отдела нажмите на поле «Руководитель», чтобы выбрать сотрудников отдела в качестве руководителей. Поддерживается множественный выбор. Сотрудники, которые уже являются руководителями отдела, не будут отображаться в списке выбора сотрудников.

![](https://static-docs.nocobase.com/92970546cbd0aeb5a8b6a36da87583bd.png)

### Настройка отделов для пользователей

Помимо добавления сотрудников в отдел, вы также можете настроить отделы для пользователей с точки зрения пользователя.

![](https://static-docs.nocobase.com/ca82a802012572e225570e8be93a4094.png)

Отделы, в которые пользователь уже входит, недоступны для выбора.

![](https://static-docs.nocobase.com/70e16d17ee9c4b5d43f8a5e1c633b177.png)
