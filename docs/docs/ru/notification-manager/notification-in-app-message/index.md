# Уведомления: Внутрисистемные сообщения

## Введение

Этот плагин позволяет пользователям получать уведомления в реальном времени прямо внутри приложения NocoBase.

## Установка

Этот плагин является встроенным, поэтому дополнительная установка не требуется.

## Добавление канала внутрисистемных сообщений

Перейдите в раздел «Управление уведомлениями», нажмите кнопку «Добавить» и выберите «Внутрисистемные сообщения».
![2024-11-08-08-33-26-20241108083326](https://static-docs.nocobase.com/2024-11-08-08-33-26-20241108083326.png)

Введите название канала и описание, затем нажмите «Отправить».
![2024-11-08-08-34-32-20241108083431](https://static-docs.nocobase.com/2024-11-08-08-34-32-20241108083431.png)

Новый канал появится в списке.

![2024-11-08-08-34-52-20241108083452](https://static-docs.nocobase.com/2024-11-08-08-34-52-20241108083452.png)

## Пример использования

Чтобы помочь вам лучше понять, как использовать внутрисистемные сообщения, рассмотрим пример «Отслеживание маркетинговых лидов».

Представьте, что ваша команда проводит крупную маркетинговую кампанию, направленную на отслеживание откликов и потребностей потенциальных клиентов. Используя внутрисистемные сообщения, вы можете:

**Создать канал уведомлений**: Сначала в разделе «Управление каналами уведомлений» настройте канал внутрисистемных сообщений с именем «Marketing Clue», чтобы члены команды могли четко определить его назначение.

![2024-11-08-08-34-32-20241108083431](https://static-docs.nocobase.com/2024-11-08-08-34-32-20241108083431.png)

**Настроить рабочий процесс**: Создайте **рабочий процесс**, который автоматически запускает уведомление при появлении нового маркетингового лида. Вы можете добавить узел уведомления в **рабочий процесс**, выбрать созданный ранее канал «Marketing Clue» и настроить содержимое сообщения по мере необходимости. Например:

![image-1-2024-10-27-14-07-17](https://static-docs.nocobase.com/image-1-2024-10-27-14-07-17.png)

**Получать уведомления в реальном времени**: Как только **рабочий процесс** будет запущен, весь соответствующий персонал получит уведомления в реальном времени, что позволит команде быстро реагировать и действовать.

![image-2-2024-10-27-14-07-22](https://static-docs.nocobase.com/image-2-2024-10-27-14-07-22.png)

**Управление сообщениями и отслеживание**: Внутрисистемные сообщения группируются по названию канала. Вы можете фильтровать сообщения по статусу «прочитано» и «не прочитано», чтобы быстро просматривать важную информацию. Нажатие кнопки «Просмотреть» перенаправит вас на настроенную страницу по ссылке для дальнейших действий.

![20241027140648-2024-10-27-14-06-51-2024-10-29-13-26-41](https://static-docs.nocobase.com/20241027140648-2024-10-27-14-06-51-2024-10-29-13-26-41.png)
