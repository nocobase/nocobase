# Уведомления: встроенные сообщения

## Введение

Позволяет пользователям получать уведомления о сообщениях в реальном времени прямо внутри приложения NocoBase.

## Установка

Этот плагин является встроенным, поэтому дополнительная установка не требуется.

## Добавление канала встроенных сообщений

Перейдите в управление уведомлениями, нажмите кнопку «Добавить» и выберите «Встроенные сообщения».
![2024-11-08-08-33-26-20241108083326](https://static-docs.nocobase.com/2024-11-08-08-33-26-20241108083326.png)

Введите имя канала и описание, затем нажмите «Отправить».
![2024-11-08-08-34-32-20241108083431](https://static-docs.nocobase.com/2024-11-08-08-34-32-20241108083431.png)

Новый канал появится в списке.

![2024-11-08-08-34-52-20241108083452](https://static-docs.nocobase.com/2024-11-08-08-34-52-20241108083452.png)

## Пример сценария использования

Чтобы вам было проще понять, как использовать встроенные сообщения, приведём пример «Сопровождение маркетингового лида».

Представьте, что ваша команда проводит крупную маркетинговую кампанию, цель которой — отслеживать отклики и потребности потенциальных клиентов. Используя встроенные сообщения, вы можете:

**Создать канал уведомлений**: сначала в управлении уведомлениями настройте канал встроенных сообщений с названием «Маркетинговая зацепка», чтобы участники команды могли однозначно понимать его назначение.

![2024-11-08-08-34-32-20241108083431](https://static-docs.nocobase.com/2024-11-08-08-34-32-20241108083431.png)

**Настроить рабочий процесс**: создайте рабочий процесс, который будет автоматически запускать уведомление, когда создаётся новый маркетинговый лид. Вы можете добавить в рабочий процесс узел уведомления, выбрать созданный вами канал «Маркетинговая зацепка» и при необходимости настроить содержимое сообщения. Например:

![image-1-2024-10-27-14-07-17](https://static-docs.nocobase.com/image-1-2024-10-27-14-07-17.png)

**Получать уведомления в реальном времени**: после срабатывания рабочего процесса все соответствующие участники получат уведомления в реальном времени, что позволит команде быстро реагировать и действовать.

![image-2-2024-10-27-14-07-22](https://static-docs.nocobase.com/image-2-2024-10-27-14-07-22.png)

**Управлять сообщениями и отслеживанием**: встроенные сообщения группируются по имени канала. Вы можете фильтровать сообщения по статусу «прочитано/непрочитано», чтобы быстро находить важную информацию. Нажатие кнопки «Просмотр» перенаправит вас на настроенную страницу ссылки для дальнейших действий.

![20241027140648-2024-10-27-14-06-51-2024-10-29-13-26-41](https://static-docs.nocobase.com/20241027140648-2024-10-27-14-06-51-2024-10-29-13-26-41.png)
