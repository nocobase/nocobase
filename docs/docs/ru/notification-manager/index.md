# Управление уведомлениями

## Введение

Управление уведомлениями — это централизованный сервис, который объединяет несколько каналов уведомлений. Он предоставляет единый интерфейс для настройки каналов, управления отправкой и ведения журналов, а также поддерживает гибкое расширение для добавления новых каналов.

![20240928112556](https://static-docs.nocobase.com/20240928112556.png)

- **Фиолетовая часть**: Управление уведомлениями предоставляет комплексный сервис, включающий настройку каналов и ведение журналов, с возможностью расширения для добавления новых каналов уведомлений.
- **Зеленая часть**: Внутрисистемные сообщения (In-App Message) — это встроенный канал, который позволяет пользователям получать уведомления непосредственно в приложении.
- **Красная часть**: Электронная почта (Email) — это расширяемый канал, позволяющий пользователям получать уведомления по электронной почте.

## Управление каналами

![20240928181752](https://static-docs.nocobase.com/20240928181752.png)

В настоящее время поддерживаются следующие каналы:

- [Внутрисистемные сообщения](/notification-manager/notification-in-app-message)
- [Электронная почта](/notification-manager/notification-email) (с использованием встроенного SMTP-транспорта)

Вы также можете расширить список каналов уведомлений. Подробности смотрите в документации по [расширению каналов](/notification-manager/development/extension).
