# Менеджер уведомлений

## Введение

Менеджер уведомлений — это централизованный сервис, который объединяет несколько каналов уведомлений. Он предоставляет единый интерфейс для настройки каналов, управления доставкой и ведения журналов, а также поддерживает гибкое расширение дополнительными каналами.

![20240928112556](https://static-docs.nocobase.com/20240928112556.png)

- **Фиолетовая область**: Менеджер уведомлений предоставляет комплексный сервис, включающий настройку каналов и ведение журналов, с возможностью расширения дополнительными каналами уведомлений.
- **Зелёная область**: Встроенные сообщения — встроенный канал, позволяющий пользователям получать уведомления прямо в приложении.
- **Красная область**: Электронная почта — расширяемый канал, позволяющий пользователям получать уведомления по электронной почте.

## Управление каналами

![20240928181752](https://static-docs.nocobase.com/20240928181752.png)

Поддерживаемые каналы:

- [Встроенные сообщения](/notification-manager/notification-in-app-message)
- [Электронная почта](/notification-manager/notification-email) (используется встроенный SMTP-транспорт)

Вы также можете расширить список каналов. См. документацию [Расширение каналов](/notification-manager/development/extension).

## Журналы уведомлений

Система записывает подробную информацию и статус для каждого уведомления, что упрощает анализ и устранение неполадок.
