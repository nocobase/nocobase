# Уведомление: WeCom

## Введение

Плагин **WeCom** позволяет приложению отправлять уведомления пользователям WeCom.

## Добавление и настройка аутентификатора WeCom

Сначала вам нужно добавить и настроить аутентификатор WeCom в NocoBase. См. [Аутентификация пользователей - WeCom](/auth-verification/auth-wecom). Только системные пользователи, вошедшие в систему через WeCom, могут получать системные уведомления через WeCom.

## Добавление канала уведомлений WeCom

![](https://static-docs.nocobase.com/202412041522365.png)

## Настройка канала уведомлений WeCom

Выберите аутентификатор, который вы только что настроили.

![](https://static-docs.nocobase.com/202412041525284.png)

## Настройка узла уведомлений рабочего процесса

Выберите настроенный канал уведомлений WeCom. Он поддерживает три типа сообщений: Текстовая карточка, Markdown и Карточка-шаблон.
