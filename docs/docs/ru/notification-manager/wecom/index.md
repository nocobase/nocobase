# Уведомления: WeCom

## Введение

Плагин **WeCom** позволяет приложению отправлять уведомления пользователям WeCom.

## Добавление и настройка аутентификатора WeCom

Сначала необходимо добавить и настроить аутентификатор WeCom в NocoBase. См. [Аутентификация пользователей — WeCom](/auth-verification/auth-wecom). Получать системные уведомления через WeCom могут только системные пользователи, которые входили в систему через WeCom.

## Добавление канала уведомлений WeCom

![](https://static-docs.nocobase.com/202412041522365.png)

## Настройка канала уведомлений WeCom

Выберите аутентификатор, который вы только что настроили.

![](https://static-docs.nocobase.com/202412041525284.png)

## Настройка узла уведомления рабочего процесса

Выберите настроенный канал уведомлений WeCom. Поддерживаются три типа сообщений: текстовая карточка, Markdown и шаблонная карточка.

![](https://static-docs.nocobase.com/202412041529319.png)
