# Верификация: TOTP-аутентификатор

## Введение

TOTP-аутентификатор позволяет пользователям привязывать любой аутентификатор, соответствующий спецификации TOTP (Time-based One-Time Password) (<a href="https://www.rfc-editor.org/rfc/rfc6238" target="_blank">RFC-6238</a>), и проходить аутентификацию с помощью одноразового пароля, основанного на времени (TOTP).

## Настройка администратора

Перейдите на страницу управления верификацией.

![](https://static-docs.nocobase.com/202502271726791.png)

Добавить - TOTP-аутентификатор

![](https://static-docs.nocobase.com/202502271745028.png)

Кроме уникального идентификатора и заголовка, TOTP-аутентификатор не требует дополнительной настройки.

![](https://static-docs.nocobase.com/202502271746034.png)

## Привязка пользователем

После добавления аутентификатора пользователи могут привязать TOTP-аутентификатор в разделе управления верификацией своего личного кабинета.

![](https://static-docs.nocobase.com/202502272252324.png)

:::warning
Плагин пока не предоставляет механизм кодов восстановления. После привязки TOTP-аутентификатора, пожалуйста, храните его в безопасности. Если вы случайно потеряли аутентификатор, вы можете использовать другой способ верификации для подтверждения своей личности, а затем отвязать его и привязать заново.
:::

## Отвязка аутентификатора

Для отвязки аутентификатора требуется верификация с использованием уже привязанного метода.
