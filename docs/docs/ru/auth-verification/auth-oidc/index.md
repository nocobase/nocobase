# Аутентификация: OIDC

## Введение

Плагин «Аутентификация: OIDC» разработан в соответствии со стандартом протокола OIDC (Open ConnectID). Он использует поток кода авторизации (Authorization Code Flow), чтобы пользователи могли входить в NocoBase, используя учетные записи, предоставляемые сторонними поставщиками услуг идентификации (IdP).
