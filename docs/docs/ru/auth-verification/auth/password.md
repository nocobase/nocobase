## Разрешить регистрацию

Если регистрация разрешена, на странице входа отображается ссылка для создания учетной записи, по которой можно перейти на страницу регистрации.

![](https://static-docs.nocobase.com/78903930d4b47aaf75cf94c55dd3596e.png)

Страница регистрации

![](https://static-docs.nocobase.com/ac3c3ab42df28cb7c6dc70b24e99e7f7.png)

Если регистрация не разрешена, на странице входа ссылка для создания учетной записи не отображается.

![](https://static-docs.nocobase.com/8d5e3b6df9991bfc1c2e095a93745121.png)

## Восстановление пароля<Badge>v1.8.0+</Badge>

Функция восстановления пароля позволяет пользователям сбросить свой пароль с помощью подтверждения по электронной почте, если они его забыли.

### Настройка для администратора

1.  **Включите функцию восстановления пароля**

    На вкладке "Настройки" > "Аутентификация пользователей" > "Восстановление пароля" установите флажок "Включить функцию восстановления пароля".

    ![20250423071957_rec_](https://static-docs.nocobase.com/20250423071957_rec_.gif)

2.  **Настройте канал уведомлений**

    Выберите канал уведомлений по электронной почте (в настоящее время поддерживается только электронная почта). Если доступных каналов уведомлений нет, вам необходимо сначала добавить новый.

    ![20250423072225_rec_](https://static-docs.nocobase.com/20250423072225_rec_.gif)

3.  **Настройте письмо для сброса пароля**

    Настройте тему и содержимое письма, поддерживаются форматы HTML или обычный текст. Вы можете использовать следующие переменные:
    - Текущий пользователь
    - Системные настройки
    - Ссылка для сброса пароля
    - Срок действия ссылки для сброса (минут)

    ![20250427170047](https://static-docs.nocobase.com/20250427170047.png)

4.  **Установите срок действия ссылки для сброса**

    Установите срок действия ссылки для сброса (в минутах); по умолчанию он составляет 120 минут.

    ![20250423073557](https://static-docs.nocobase.com/20250423073557.png)

### Процесс для пользователя

1.  **Инициируйте запрос на сброс пароля**

    На странице входа нажмите ссылку "Забыли пароль" (администратор должен предварительно включить функцию восстановления пароля), чтобы перейти на страницу восстановления пароля.

    ![20250421103458_rec_](https://static-docs.nocobase.com/20250421103458_rec_.gif)

    Введите зарегистрированный адрес электронной почты и нажмите кнопку "Отправить письмо для сброса".

    ![20250421113442_rec_](https://static-docs.nocobase.com/20250421113442_rec_.gif)

2.  **Сбросьте пароль**

    Пользователь получит письмо, содержащее ссылку для сброса. После перехода по ссылке на открывшейся странице установите новый пароль.

    ![20250421113748](https://static-docs.nocobase.com/20250421113748.png)

    После завершения настройки пользователь сможет войти в систему с новым паролем.

### Примечания

- Ссылка для сброса имеет ограничение по времени: по умолчанию она действительна в течение 120 минут после создания (может быть настроено администратором).
- Ссылка может быть использована только один раз и становится недействительной сразу после использования.
- Если пользователь не получил письмо для сброса, пожалуйста, проверьте правильность адреса электронной почты или папку со спамом.
- Администратор должен убедиться, что конфигурация почтового сервера верна, чтобы гарантировать успешную отправку письма для сброса.
