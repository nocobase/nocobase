## Разрешить регистрацию

Когда регистрация разрешена, на странице входа отображается ссылка для создания учетной записи, и вы можете перейти на страницу регистрации.

![](https://static-docs.nocobase.com/78903930d4b47aaf75cf94c55dd3596e.png)

Страница регистрации

![](https://static-docs.nocobase.com/ac3c3ab42df28cb7c6dc70b24e99e7f7.png)

Когда регистрация не разрешена, на странице входа не отображается ссылка для создания учетной записи.

![](https://static-docs.nocobase.com/8d5e3b6df9991bfc1c2e095a93745121.png)

Если регистрация не разрешена, страницу регистрации невозможно открыть.

![](https://static-docs.nocobase.com/09325c4b07e09f88f80a14dff8430556.png)

## Настройки формы регистрации <Badge>v1.4.0-beta.7+</Badge>

## Забыли пароль<Badge>v1.8.0+</Badge>

Функция «Забыли пароль» позволяет пользователям сбросить пароль через подтверждение по email.

### Конфигурация администратора

1. **Включить функцию «Забыли пароль»**

   В «Настройки» > «Аутентификация» перейдите на вкладку «Забыли пароль» и установите флажок **Включить функцию «Забыли пароль»**.

   ![](https://static-docs.nocobase.com/20250423071957_rec_.gif)

2. **Настроить канал уведомлений**

   Выберите канал уведомлений по email (сейчас поддерживается только email). Если ни один канал уведомлений не доступен, сначала нужно добавить его.

   ![](https://static-docs.nocobase.com/20250423072225_rec_.gif)

3. **Настроить письмо для сброса пароля**

   Настройте тему и содержимое письма. Поддерживается формат HTML или обычного текста. Можно использовать следующие переменные:
   - Текущий пользователь
   - Системные настройки
   - Ссылка для сброса пароля
   - Истечение срока ссылки для сброса (минуты)

   ![](https://static-docs.nocobase.com/20250427170047.png)

4. **Задать срок действия ссылки для сброса**

   Задайте период действия (в минутах) для ссылки сброса. По умолчанию 120 минут.

   ![](https://static-docs.nocobase.com/20250423073557.png)

### Пользовательский рабочий процесс

1. **Инициировать запрос на сброс пароля**

   Нажмите на кнопку-ссылку «Забыли пароль» на странице входа, чтобы перейти на страницу сброса пароля.

   ![](https://static-docs.nocobase.com/20250421103458_rec_.gif)

   Введите зарегистрированный email и нажмите кнопку «Отправить письмо для сброса пароля».

   ![](https://static-docs.nocobase.com/20250421113442_rec_.gif)

2. **Сбросить пароль**

   Пользователь получит email со ссылкой для сброса пароля. Откройте ссылку, чтобы перейти на страницу, где можно установить новый пароль.

   ![](https://static-docs.nocobase.com/20250421113748.png)

   После настройки пользователь сможет войти в систему по новому паролю.

### Примечания

- Ссылка для сброса имеет ограничение по времени: по умолчанию она действительна в течение 120 минут после генерации (настраивается администратором).
- Ссылка может быть использована только один раз и сразу становится недействительной после использования.
- Если пользователь не получил письмо со ссылкой для сброса, проверьте корректность email-адреса или папку со спамом.
- Администратор должен убедиться, что конфигурация почтового сервера корректна, чтобы письмо для сброса отправлялось успешно.
