# Аутентификация по SMS

## Введение

Плагин SMS-аутентификации позволяет пользователям регистрироваться и входить в NocoBase с помощью SMS.

> Его необходимо использовать в сочетании с функцией SMS-кода подтверждения, предоставляемой [`плагином @nocobase/plugin-verification`](/auth-verification/verification/).

## Добавление SMS-аутентификации

Перейдите на страницу управления плагинами пользовательской аутентификации.

![](https://static-docs.nocobase.com/202502282112517.png)

Добавить - SMS

## Настройка новой версии

:::info{title=Примечание}
Новая конфигурация была представлена в версии `1.6.0-alpha.30` и планируется к стабильной поддержке начиная с `1.7.0`.
:::

![](https://static-docs.nocobase.com/202502282114821.png)

**Верификатор:** Привяжите SMS-верификатор для отправки SMS-кодов подтверждения. Если доступных верификаторов нет, вам сначала нужно перейти на страницу управления верификацией и создать SMS-верификатор.
См. также:

- [Верификация](../verification/index.md)
- [Верификация: SMS](../verification/sms/index.md)

**Автоматическая регистрация, если пользователь не существует:** Если эта опция выбрана, то при отсутствии номера телефона пользователя будет зарегистрирован новый пользователь, используя этот номер телефона в качестве ника.

## Настройка старой версии

![](https://static-docs.nocobase.com/a4d35ec63ba22ae2ea9e3e8e1cbb783d.png)

Функция SMS-аутентификации для входа будет использовать настроенный и установленный по умолчанию провайдер SMS-кодов подтверждения для отправки текстовых сообщений.

**Автоматическая регистрация, если пользователь не существует:** Если эта опция выбрана, то при отсутствии номера телефона пользователя будет зарегистрирован новый пользователь, используя этот номер телефона в качестве ника.
