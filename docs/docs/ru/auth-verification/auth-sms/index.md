# Аутентификация: SMS

## Введение

Плагин аутентификации по SMS позволяет пользователям регистрироваться через SMS и входить в NocoBase.

> Нужно использовать вместе с функцией кода верификации по SMS, предоставляемой плагином [верификации `@nocobase/plugin-verification`](../verification/index.md)

## Добавить SMS-аутентификацию

Перейдите на страницу управления плагинами аутентификации пользователей.

![](https://static-docs.nocobase.com/202502282112517.png)

Добавить — SMS

![](https://static-docs.nocobase.com/202502282113553.png)

## Конфигурация новой версии

:::info{title=Примечание}
Новая конфигурация была представлена в `1.6.0-alpha.30` и планируется к стабильной поддержке, начиная с `1.7.0`.
:::

![](https://static-docs.nocobase.com/202502282114821.png)

**Верификатор:** привяжите SMS-вeрификатор, чтобы отправлять коды верификации по SMS. Если доступного верификатора нет, сначала перейдите на страницу управления верификацией и создайте SMS-вeрификатор.
См. также:

- [Верификация (Verification)](../verification/index.md)
- [Верификация: SMS (Verification: SMS)](../verification/sms/index.md)

**Автоматическая регистрация, если пользователя нет:** Когда этот параметр включен, если номер телефона пользователя не существует, будет зарегистрирован новый пользователь, а номер телефона будет использован как имя пользователя.

## Конфигурация старой версии

![](https://static-docs.nocobase.com/a4d35ec63ba22ae2ea9e3e8e1cbb783d.png)

Функция входа по SMS будет использовать настроенный и/или встроенный провайдер кода верификации по SMS, чтобы отправлять текстовые сообщения.

**Автоматическая регистрация, если пользователя нет:** Когда этот параметр включен, если номер телефона пользователя не существует, будет зарегистрирован новый пользователь, а номер телефона будет использован как nickname.
