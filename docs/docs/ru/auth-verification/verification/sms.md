# Верификация: SMS

## Введение

Код верификации по SMS — встроенный тип верификации, используемый для генерации одноразового динамического пароля (OTP) и отправки его пользователю через SMS.

## Добавить SMS-аутентификатор

Перейдите на страницу управления верификацией.

![](https://static-docs.nocobase.com/202502271726791.png)

Добавить — SMS OTP

![](https://static-docs.nocobase.com/202502271726056.png)

## Конфигурация администратора

![](https://static-docs.nocobase.com/202502271727711.png)

На данный момент поддерживаются следующие провайдеры SMS-сервиса:

- <a href="https://www.aliyun.com/product/sms" target="_blank">Aliyun SMS</a>
- <a href="https://cloud.tencent.com/product/sms" target="_blank">Tencent Cloud SMS</a>

При настройке SMS-шаблона в админ-панели провайдера необходимо зарезервировать параметр под код верификации.

- Пример конфигурации Aliyun: `Ваш код верификации: ${code}`

- Пример конфигурации Tencent Cloud: `Ваш код верификации: {1}`

Разработчики также могут расширять поддержку других провайдеров SMS в виде плагинов. См. раздел [Расширение провайдеров SMS](./dev/sms-type).

## Привязка пользователем

После добавления верификатора пользователи могут привязать номер телефона в своем личном разделе управления верификацией.

![](https://static-docs.nocobase.com/202502271737016.png)

![](https://static-docs.nocobase.com/202502271737769.png)

![](https://static-docs.nocobase.com/202502271738515.png)

После успешной привязки верификацию личности можно выполнять в любых сценариях, которые используют этот верификатор.

![](https://static-docs.nocobase.com/202502271739607.png)
