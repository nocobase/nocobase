## पैरामीटर

| पैरामीटर का नाम        | प्रकार          | डिफ़ॉल्ट मान | विवरण                                                              |
| :--------------------- | :-------------- | :----------- | :------------------------------------------------------------------ |
| `options.values`       | `M`             | `{}`         | वह डेटा ऑब्जेक्ट जिसे इन्सर्ट किया जाना है।                         |
| `options.whitelist?`   | `string[]`      | -            | `values` फ़ील्ड्स के लिए सफ़ेद सूची। इस सूची में शामिल फ़ील्ड्स को ही स्टोर किया जाएगा। |
| `options.blacklist?`   | `string[]`      | -            | `values` फ़ील्ड्स के लिए काली सूची। इस सूची में शामिल फ़ील्ड्स को स्टोर नहीं किया जाएगा। |
| `options.transaction?` | `Transaction`   | -            | ट्रांज़ैक्शन                                                        |
