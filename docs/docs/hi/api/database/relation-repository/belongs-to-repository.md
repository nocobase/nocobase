## BelongsToRepository

इसका इंटरफ़ेस [HasOneRepository](./has-one-repository.md) के समान है।
`BelongsToRepository` एक `Repository` है जो `BelongsTo` संबंधों को संभालने के लिए है। यह `BelongsTo` संबंधों को संभालने के लिए कुछ सुविधाजनक तरीके प्रदान करता है।
