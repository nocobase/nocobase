**प्रकार**

```typescript
interface DestroyOptions extends SequelizeDestroyOptions {
  filter?: Filter;
  filterByTk?: TargetKey | TargetKey[];
  truncate?: boolean;
  context?: any;
}
```

**विवरण**

- ``filter``: डिलीट किए जाने वाले रिकॉर्ड के लिए फ़िल्टर शर्तें निर्दिष्ट करता है। फ़िल्टर के विस्तृत उपयोग के लिए, कृपया [`find()`](#find) विधि देखें।
- ``filterByTk``: TargetKey द्वारा डिलीट किए जाने वाले रिकॉर्ड के लिए फ़िल्टर शर्तें निर्दिष्ट करता है।
- ``truncate``: क्या संग्रह डेटा को ट्रंकेट करना है। यह केवल तभी प्रभावी होता है जब ``filter`` या ``filterByTk`` पैरामीटर प्रदान नहीं किए जाते हैं।
- ``transaction``: ट्रांज़ैक्शन ऑब्जेक्ट। यदि कोई ट्रांज़ैक्शन पैरामीटर पास नहीं किया जाता है, तो यह विधि स्वचालित रूप से एक आंतरिक ट्रांज़ैक्शन बनाएगी।
