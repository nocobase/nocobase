**प्रकार**

```typescript
type FindOneOptions = Omit<FindOptions, 'limit'>;
```

**पैरामीटर**

अधिकांश पैरामीटर `find()` के समान हैं। अंतर यह है कि `findOne()` केवल एक ही रिकॉर्ड लौटाता है, इसलिए `limit` पैरामीटर की आवश्यकता नहीं होती है, और क्वेरी के दौरान यह हमेशा `1` होता है।
