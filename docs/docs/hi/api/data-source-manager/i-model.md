# IModel

`IModel` इंटरफ़ेस एक मॉडल ऑब्जेक्ट के मूल गुणों और विधियों को परिभाषित करता है।

```typescript
export interface IModel {
  toJSON: () => any;
}
```

## API

### toJSON()

मॉडल ऑब्जेक्ट को JSON फ़ॉर्मेट में बदलता है।
