# IField

`IField` उस इंटरफ़ेस को परिभाषित करता है जिसे एक फ़ील्ड को लागू करना होता है।

```typescript
export type FieldOptions = {
  name: string;
  field: string;
  rawType: string;
  type: string;
  description?: string;
  interface?: string;
  uiSchema?: any;
  possibleTypes?: string[];
  defaultValue?: any;
  primaryKey: boolean;
  unique: boolean;
  allowNull?: boolean;
  autoIncrement?: boolean;
  [key: string]: any;
};

export interface IField {
  options: FieldOptions;
}
```

### options

- **प्रकार**: `FieldOptions`
