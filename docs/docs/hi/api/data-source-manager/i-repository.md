# IRepository

`Repository` इंटरफ़ेस मॉडल ऑपरेशन के तरीकों की एक श्रृंखला को परिभाषित करता है, जो डेटा स्रोत के CRUD (बनाना, पढ़ना, अपडेट करना और हटाना) ऑपरेशनों के अनुकूल होने के लिए उपयोग किए जाते हैं।

## API

### find()

क्वेरी पैरामीटर से मेल खाने वाले मॉडलों की एक सूची देता है।

#### सिग्नेचर

- `find(options?: any): Promise<IModel[]>`

### findOne()

क्वेरी पैरामीटर से मेल खाने वाला एक मॉडल देता है। यदि कई मॉडल मेल खाते हैं, तो केवल पहला मॉडल ही वापस किया जाता है।

#### सिग्नेचर

- `findOne(options?: any): Promise<IModel>`

### count()

क्वेरी पैरामीटर से मेल खाने वाले मॉडलों की संख्या देता है।

#### सिग्नेचर

- `count(options?: any): Promise<Number>`

### findAndCount()

क्वेरी पैरामीटर से मेल खाने वाले मॉडलों की सूची और उनकी संख्या देता है।

#### सिग्नेचर

- `findAndCount(options?: any): Promise<[IModel[], Number]>`

### create()

एक मॉडल डेटा ऑब्जेक्ट बनाता है।

#### सिग्नेचर

- `create(options: any): void`

### update()

क्वेरी शर्तों के आधार पर एक मॉडल डेटा ऑब्जेक्ट को अपडेट करता है।

#### सिग्नेचर

- `update(options: any): void`

### destroy()

क्वेरी शर्तों के आधार पर एक मॉडल डेटा ऑब्जेक्ट को हटाता है।

#### सिग्नेचर

- `destroy(options: any): void`
