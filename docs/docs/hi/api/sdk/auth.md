# Auth

## अवलोकन

`Auth` क्लास का उपयोग मुख्य रूप से क्लाइंट साइड पर उपयोगकर्ता जानकारी तक पहुँचने और उपयोगकर्ता प्रमाणीकरण (authentication) से संबंधित API का अनुरोध करने के लिए किया जाता है।

## इंस्टेंस प्रॉपर्टीज़

### `locale`

वर्तमान उपयोगकर्ता द्वारा उपयोग की जाने वाली भाषा।

### `role`

वर्तमान उपयोगकर्ता द्वारा उपयोग की जाने वाली भूमिका।

### `token`

API टोकन।

### `authenticator`

वर्तमान उपयोगकर्ता के प्रमाणीकरण के लिए उपयोग किया जाने वाला ऑथेंटिकेटर। [उपयोगकर्ता प्रमाणीकरण](/auth-verification/auth/) देखें।

## क्लास मेथड्स

### `signIn()`

उपयोगकर्ता साइन इन।

#### सिग्नेचर

- `async signIn(values: any, authenticator?: string): Promise<AxiosResponse<any>>`

#### विवरण

| पैरामीटर का नाम | टाइप     | विवरण                                          |
| --------------- | -------- | ---------------------------------------------- |
| `values`        | `any`    | साइन-इन API के लिए अनुरोध पैरामीटर            |
| `authenticator` | `string` | साइन-इन के लिए उपयोग किए जाने वाले ऑथेंटिकेटर का पहचानकर्ता |

### `signUp()`

उपयोगकर्ता साइन अप।

#### सिग्नेचर

- `async signUp(values: any, authenticator?: string): Promise<AxiosResponse<any>>`

#### विवरण

| पैरामीटर का नाम | टाइप     | विवरण                                          |
| --------------- | -------- | ---------------------------------------------- |
| `values`        | `any`    | साइन-अप API के लिए अनुरोध पैरामीटर            |
| `authenticator` | `string` | साइन-अप के लिए उपयोग किए जाने वाले ऑथेंटिकेटर का पहचानकर्ता |

### `signOut()`

साइन आउट।

#### सिग्नेचर

- `async signOut(values: any, authenticator?: string): Promise<AxiosResponse<any>>`

#### विवरण

| पैरामीटर का नाम | टाइप     | विवरण                                          |
| --------------- | -------- | ---------------------------------------------- |
| `values`        | `any`    | साइन-आउट API के लिए अनुरोध पैरामीटर           |
| `authenticator` | `string` | साइन-आउट के लिए उपयोग किए जाने वाले ऑथेंटिकेटर का पहचानकर्ता |
