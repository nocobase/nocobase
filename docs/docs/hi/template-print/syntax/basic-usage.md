## बुनियादी उपयोग

टेम्पलेट प्रिंटिंग प्लगइन टेम्पलेट में डायनामिक डेटा और लॉजिकल स्ट्रक्चर को आसानी से डालने के लिए कई तरह के सिंटैक्स (syntax) प्रदान करता है। नीचे सिंटैक्स की विस्तृत व्याख्या और उपयोग के उदाहरण दिए गए हैं।

### बुनियादी प्रतिस्थापन (Basic Replacement)

डेटा बदलने के लिए `{d.xxx}` फॉर्मेट के प्लेसहोल्डर (placeholder) का उपयोग करें। उदाहरण के लिए:

- `{d.title}`: यह डेटासेट (dataset) से `title` फ़ील्ड को पढ़ता है।
- `{d.date}`: यह डेटासेट से `date` फ़ील्ड को पढ़ता है।

**उदाहरण**:

टेम्पलेट सामग्री:
```
प्रिय ग्राहक,

हमारे उत्पाद खरीदने के लिए धन्यवाद: {d.productName}।
ऑर्डर आईडी: {d.orderId}
ऑर्डर दिनांक: {d.orderDate}

आशा है आपको इसका उपयोग करके अच्छा लगेगा!
```

डेटासेट:
```json
{
  "productName": "स्मार्ट वॉच",
  "orderId": "A123456789",
  "orderDate": "2025-01-01"
}
```

रेंडर किया गया परिणाम:
```
प्रिय ग्राहक,

हमारे उत्पाद खरीदने के लिए धन्यवाद: स्मार्ट वॉच।
ऑर्डर आईडी: A123456789
ऑर्डर दिनांक: 2025-01-01

आशा है आपको इसका उपयोग करके अच्छा लगेगा!
```

### सब-ऑब्जेक्ट (Sub-objects) तक पहुँचना

यदि डेटासेट में सब-ऑब्जेक्ट शामिल हैं, तो आप डॉट नोटेशन (dot notation) का उपयोग करके सब-ऑब्जेक्ट की प्रॉपर्टी (property) तक पहुँच सकते हैं।

**सिंटैक्स**: `{d.parent.child}`

**उदाहरण**:

डेटासेट:
```json
{
  "customer": {
    "name": "ली रे",
    "contact": {
      "email": "lilei@example.com",
      "phone": "13800138000"
    }
  }
}
```

टेम्पलेट सामग्री:
```
ग्राहक का नाम: {d.customer.name}
ईमेल पता: {d.customer.contact.email}
फ़ोन नंबर: {d.customer.contact.phone}
```

रेंडर किया गया परिणाम:
```
ग्राहक का नाम: ली रे
ईमेल पता: lilei@example.com
फ़ोन नंबर: 13800138000
```

### एरे (Arrays) तक पहुँचना

यदि डेटासेट में एरे शामिल हैं, तो आप एरे के तत्वों (elements) तक पहुँचने के लिए आरक्षित कीवर्ड (reserved keyword) `i` का उपयोग कर सकते हैं।

**सिंटैक्स**: `{d.arrayName[i].field}`

**उदाहरण**:

डेटासेट:
```json
{
  "staffs": [
    { "firstname": "James", "lastname": "Anderson" },
    { "firstname": "Emily", "lastname": "Roberts" },
    { "firstname": "Michael", "lastname": "Johnson" }
  ]
}
```

टेम्पलेट सामग्री:
```
पहले कर्मचारी का अंतिम नाम {d.staffs[i=0].lastname} है, और पहला नाम {d.staffs[i=0].firstname} है।
```

रेंडर किया गया परिणाम:
```
पहले कर्मचारी का अंतिम नाम Anderson है, और पहला नाम James है।
```
