### मुद्रा फ़ॉर्मेटिंग

#### 1. :formatC(precisionOrFormat, targetCurrency)

##### सिंटैक्स स्पष्टीकरण
मुद्रा संख्या को फ़ॉर्मेट करें, इसमें दशमलव स्थानों या विशिष्ट आउटपुट फ़ॉर्मेट को निर्दिष्ट किया जा सकता है।  
पैरामीटर:
- precisionOrFormat: वैकल्पिक पैरामीटर, यह या तो एक संख्या (दशमलव स्थानों को निर्दिष्ट करने के लिए) हो सकता है, या एक विशिष्ट फ़ॉर्मेट पहचानकर्ता:
  - पूर्णांक: डिफ़ॉल्ट दशमलव सटीकता को बदलता है
  - `'M'`: केवल मुख्य मुद्रा का नाम आउटपुट करता है
  - `'L'`: संख्या और मुद्रा प्रतीक के साथ आउटपुट करता है (डिफ़ॉल्ट)
  - `'LL'`: संख्या और मुख्य मुद्रा के नाम के साथ आउटपुट करता है
- targetCurrency: वैकल्पिक, लक्ष्य मुद्रा कोड (बड़े अक्षरों में, जैसे USD, EUR), यह वैश्विक सेटिंग्स को ओवरराइड करेगा

##### उदाहरण
```
'1000.456':formatC()      // आउटपुट "$2,000.91"
'1000.456':formatC('M')    // आउटपुट "dollars"
'1':formatC('M')           // आउटपुट "dollar"
'1000':formatC('L')        // आउटपुट "$2,000.00"
'1000':formatC('LL')       // आउटपुट "2,000.00 dollars"
```

##### परिणाम
आउटपुट परिणाम API विकल्पों और विनिमय दर सेटिंग्स पर आधारित होता है।


#### 2. :convCurr(target, source)

##### सिंटैक्स स्पष्टीकरण
किसी संख्या को एक मुद्रा से दूसरी मुद्रा में बदलें। विनिमय दर को API विकल्पों के माध्यम से पास किया जा सकता है या वैश्विक स्तर पर सेट किया जा सकता है।  
यदि कोई पैरामीटर निर्दिष्ट नहीं है, तो यह स्वचालित रूप से `options.currencySource` से `options.currencyTarget` में परिवर्तित हो जाता है।  
पैरामीटर:
- target: वैकल्पिक, लक्ष्य मुद्रा कोड (डिफ़ॉल्ट रूप से `options.currencyTarget` के बराबर)
- source: वैकल्पिक, स्रोत मुद्रा कोड (डिफ़ॉल्ट रूप से `options.currencySource` के बराबर)

##### उदाहरण
```
10:convCurr()              // आउटपुट 20
1000:convCurr()            // आउटपुट 2000
1000:convCurr('EUR')        // आउटपुट 1000
1000:convCurr('USD')        // आउटपुट 2000
1000:convCurr('USD', 'USD') // आउटपुट 1000
```

##### परिणाम
आउटपुट परिवर्तित मुद्रा मान है।
