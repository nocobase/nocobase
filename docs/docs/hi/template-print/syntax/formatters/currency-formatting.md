### मुद्रा फ़ॉर्मेटिंग

#### 1. :formatC(precisionOrFormat, targetCurrency)

##### सिंटैक्स स्पष्टीकरण
यह एक मुद्रा संख्या को फ़ॉर्मेट करता है और आपको दशमलव स्थानों की संख्या या एक विशेष आउटपुट फ़ॉर्मेट निर्दिष्ट करने की सुविधा देता है।  
पैरामीटर्स:
- **precisionOrFormat:** यह एक वैकल्पिक पैरामीटर है जो या तो एक संख्या हो सकता है (दशमलव स्थानों की संख्या बताने के लिए) या एक फ़ॉर्मेट स्पेसिफ़ायर हो सकता है:
  - एक पूर्णांक: डिफ़ॉल्ट दशमलव सटीकता को बदलता है।
  - `'M'`: केवल मुख्य मुद्रा का नाम आउटपुट करता है।
  - `'L'`: संख्या के साथ मुद्रा प्रतीक आउटपुट करता है (यह डिफ़ॉल्ट है)।
  - `'LL'`: संख्या के साथ मुख्य मुद्रा का नाम आउटपुट करता है।
- **targetCurrency:** वैकल्पिक; यह लक्ष्य मुद्रा कोड (बड़े अक्षरों में, जैसे USD, EUR) है जो वैश्विक सेटिंग्स को ओवरराइड करता है।

##### उदाहरण
```
// उदाहरण वातावरण: API विकल्प { "lang": "en-us", "currency": { "source": "EUR", "target": "USD", "rates": { "EUR": 1, "USD": 2 } } }
'1000.456':formatC()      // आउटपुट "$2,000.91"
'1000.456':formatC('M')    // आउटपुट "dollars"
'1':formatC('M')           // आउटपुट "dollar"
'1000':formatC('L')        // आउटपुट "$2,000.00"
'1000':formatC('LL')       // आउटपुट "2,000.00 dollars"

// फ़्रेंच उदाहरण (जब वातावरण सेटिंग्स भिन्न हों):
'1000.456':formatC()      // आउटपुट "2 000,91 ..."  
'1000.456':formatC()      // जब स्रोत और लक्ष्य मुद्राएँ समान हों, तो आउटपुट "1 000,46 €"
```

##### परिणाम
आउटपुट API विकल्पों और विनिमय दर सेटिंग्स पर निर्भर करता है।

#### 2. :convCurr(target, source)

##### सिंटैक्स स्पष्टीकरण
यह एक संख्या को एक मुद्रा से दूसरी मुद्रा में परिवर्तित करता है। विनिमय दर को API विकल्पों के माध्यम से पास किया जा सकता है या विश्व स्तर पर सेट किया जा सकता है।  
यदि कोई पैरामीटर निर्दिष्ट नहीं किया जाता है, तो रूपांतरण स्वचालित रूप से `options.currencySource` से `options.currencyTarget` तक किया जाता है।  
पैरामीटर्स:
- **target:** वैकल्पिक; यह लक्ष्य मुद्रा कोड है (डिफ़ॉल्ट रूप से `options.currencyTarget` होता है)।
- **source:** वैकल्पिक; यह स्रोत मुद्रा कोड है (डिफ़ॉल्ट रूप से `options.currencySource` होता है)।

##### उदाहरण
```
// उदाहरण वातावरण: API विकल्प { "currency": { "source": "EUR", "target": "USD", "rates": { "EUR": 1, "USD": 2 } } }
10:convCurr()              // आउटपुट 20
1000:convCurr()            // आउटपुट 2000
1000:convCurr('EUR')        // आउटपुट 1000
1000:convCurr('USD')        // आउटपुट 2000
1000:convCurr('USD', 'USD') // आउटपुट 1000
```

##### परिणाम
आउटपुट परिवर्तित मुद्रा मूल्य है।
