### संख्या फ़ॉर्मेटिंग

#### 1. :formatN(precision)

##### सिंटैक्स स्पष्टीकरण
यह फ़ंक्शन संख्या को स्थानीयकरण सेटिंग्स के अनुसार फ़ॉर्मेट करता है।  
पैरामीटर:
- `precision`: दशमलव स्थानों की संख्या।  
  ODS/XLSX फ़ॉर्मेट के लिए, प्रदर्शित दशमलव स्थानों की संख्या टेक्स्ट एडिटर द्वारा निर्धारित की जाती है; अन्य फ़ॉर्मेट के लिए, इस पैरामीटर का उपयोग किया जाता है।

##### उदाहरण
```
// उदाहरण वातावरण: API विकल्प { "lang": "en-us" }
'10':formatN()         // आउटपुट "10.000"
'1000.456':formatN()   // आउटपुट "1,000.456"
```

##### परिणाम
संख्या को निर्दिष्ट `precision` और स्थानीयकरण फ़ॉर्मेट के अनुसार आउटपुट किया जाता है।

#### 2. :round(precision)

##### सिंटैक्स स्पष्टीकरण
यह फ़ंक्शन संख्या को निर्दिष्ट दशमलव स्थानों तक पूर्णांकित करता है।

##### उदाहरण
```
10.05123:round(2)      // आउटपुट 10.05
1.05:round(1)          // आउटपुट 1.1
```

##### परिणाम
आउटपुट दी गई `precision` तक पूर्णांकित संख्या है।

#### 3. :add(value)

##### सिंटैक्स स्पष्टीकरण
यह फ़ंक्शन वर्तमान संख्या में निर्दिष्ट मान जोड़ता है।  
पैरामीटर:
- `value`: जोड़ने वाली संख्या।

##### उदाहरण
```
1000.4:add(2)         // आउटपुट 1002.4
'1000.4':add('2')      // आउटपुट 1002.4
```

##### परिणाम
आउटपुट वर्तमान संख्या और निर्दिष्ट मान का योग है।

#### 4. :sub(value)

##### सिंटैक्स स्पष्टीकरण
यह फ़ंक्शन वर्तमान संख्या से निर्दिष्ट मान घटाता है।  
पैरामीटर:
- `value`: घटाने वाली संख्या।

##### उदाहरण
```
1000.4:sub(2)         // आउटपुट 998.4
'1000.4':sub('2')      // आउटपुट 998.4
```

##### परिणाम
आउटपुट वर्तमान संख्या में से निर्दिष्ट मान घटाने के बाद की संख्या है।

#### 5. :mul(value)

##### सिंटैक्स स्पष्टीकरण
यह फ़ंक्शन वर्तमान संख्या को निर्दिष्ट मान से गुणा करता है।  
पैरामीटर:
- `value`: गुणा करने वाली संख्या।

##### उदाहरण
```
1000.4:mul(2)         // आउटपुट 2000.8
'1000.4':mul('2')      // आउटपुट 2000.8
```

##### परिणाम
आउटपुट वर्तमान संख्या और निर्दिष्ट मान का गुणनफल है।

#### 6. :div(value)

##### सिंटैक्स स्पष्टीकरण
यह फ़ंक्शन वर्तमान संख्या को निर्दिष्ट मान से विभाजित करता है।  
पैरामीटर:
- `value`: भाजक।

##### उदाहरण
```
1000.4:div(2)         // आउटपुट 500.2
'1000.4':div('2')      // आउटपुट 500.2
```

##### परिणाम
आउटपुट विभाजन का परिणाम है।

#### 7. :mod(value)

##### सिंटैक्स स्पष्टीकरण
यह फ़ंक्शन वर्तमान संख्या को निर्दिष्ट मान से विभाजित करने पर शेषफल (मॉड्यूलस) की गणना करता है।  
पैरामीटर:
- `value`: मॉड्यूलस भाजक।

##### उदाहरण
```
4:mod(2)              // आउटपुट 0
3:mod(2)              // आउटपुट 1
```

##### परिणाम
आउटपुट मॉड्यूलस ऑपरेशन का शेषफल है।

#### 8. :abs

##### सिंटैक्स स्पष्टीकरण
यह फ़ंक्शन संख्या का निरपेक्ष मान लौटाता है।

##### उदाहरण
```
-10:abs()             // आउटपुट 10
-10.54:abs()          // आउटपुट 10.54
10.54:abs()           // आउटपुट 10.54
'-200':abs()          // आउटपुट 200
```

##### परिणाम
आउटपुट इनपुट संख्या का निरपेक्ष मान है।

#### 9. :ceil

##### सिंटैक्स स्पष्टीकरण
यह फ़ंक्शन संख्या को ऊपर की ओर पूर्णांकित करता है, यानी वर्तमान संख्या से बड़ी या उसके बराबर सबसे छोटी पूर्णांक संख्या लौटाता है।

##### उदाहरण
```
10.05123:ceil()       // आउटपुट 11
1.05:ceil()           // आउटपुट 2
-1.05:ceil()          // आउटपुट -1
```

##### परिणाम
आउटपुट निकटतम पूर्णांक तक ऊपर की ओर पूर्णांकित संख्या है।

#### 10. :floor

##### सिंटैक्स स्पष्टीकरण
यह फ़ंक्शन संख्या को नीचे की ओर पूर्णांकित करता है, यानी वर्तमान संख्या से छोटी या उसके बराबर सबसे बड़ी पूर्णांक संख्या लौटाता है।

##### उदाहरण
```
10.05123:floor()      // आउटपुट 10
1.05:floor()          // आउटपुट 1
-1.05:floor()         // आउटपुट -2
```

##### परिणाम
आउटपुट निकटतम पूर्णांक तक नीचे की ओर पूर्णांकित संख्या है।

#### 11. :int

##### सिंटैक्स स्पष्टीकरण
यह फ़ंक्शन संख्या को पूर्णांक में बदलता है (उपयोग के लिए अनुशंसित नहीं है)।

##### उदाहरण और परिणाम
यह विशिष्ट रूपांतरण स्थिति पर निर्भर करता है।

#### 12. :toEN

##### सिंटैक्स स्पष्टीकरण
यह फ़ंक्शन संख्या को अंग्रेज़ी फ़ॉर्मेट में बदलता है (दशमलव बिंदु के रूप में `.` का उपयोग करके)। उपयोग के लिए अनुशंसित नहीं है।

##### उदाहरण और परिणाम
यह विशिष्ट रूपांतरण स्थिति पर निर्भर करता है।

#### 13. :toFixed

##### सिंटैक्स स्पष्टीकरण
यह फ़ंक्शन संख्या को एक स्ट्रिंग में बदलता है, जिसमें केवल निर्दिष्ट दशमलव स्थानों की संख्या रखी जाती है। उपयोग के लिए अनुशंसित नहीं है।

##### उदाहरण और परिणाम
यह विशिष्ट रूपांतरण स्थिति पर निर्भर करता है।

#### 14. :toFR

##### सिंटैक्स स्पष्टीकरण
यह फ़ंक्शन संख्या को फ्रेंच फ़ॉर्मेट में बदलता है (दशमलव विभाजक के रूप में `,` का उपयोग करके)। उपयोग के लिए अनुशंसित नहीं है।

##### उदाहरण और परिणाम
यह विशिष्ट रूपांतरण स्थिति पर निर्भर करता है।
