# फ़ाइल प्रबंधन

## परिचय

AI कर्मचारी संवादों के दौरान अपलोड की गई फ़ाइलों के लिए स्टोरेज विधि कॉन्फ़िगर करें।

## सेटिंग्स

AI कर्मचारी प्लगइन कॉन्फ़िगरेशन पेज पर जाएँ, `Settings` टैब पर क्लिक करें, और स्टोरेज विधि सेटिंग्स पेज पर पहुँचें।

![20251022002648](https://static-docs.nocobase.com/20251022002648.png)

`Files storage` के लिए उपलब्ध स्टोरेज विकल्प फ़ाइल प्रबंधन प्लगइन में कॉन्फ़िगर किए जाते हैं।
