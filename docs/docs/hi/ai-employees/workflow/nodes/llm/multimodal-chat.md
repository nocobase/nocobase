# मल्टीमोडल बातचीत

## छवियाँ

यदि मॉडल इसे सपोर्ट करता है, तो LLM नोड मॉडल को छवियाँ भेज सकता है। इसका उपयोग करते समय, आपको एक वेरिएबल के माध्यम से अटैचमेंट फ़ील्ड या संबंधित फ़ाइल संग्रह रिकॉर्ड का चयन करना होगा। फ़ाइल संग्रह रिकॉर्ड का चयन करते समय, आप इसे ऑब्जेक्ट स्तर पर चुन सकते हैं या URL फ़ील्ड का चयन कर सकते हैं।

![](https://static-docs.nocobase.com/202503041034858.png)

छवि भेजने के प्रारूप के लिए दो विकल्प उपलब्ध हैं:

- URL के माध्यम से भेजें - स्थानीय रूप से संग्रहीत छवियों को छोड़कर, सभी छवियाँ URL के रूप में भेजी जाएँगी। स्थानीय रूप से संग्रहीत छवियों को भेजने से पहले base64 प्रारूप में परिवर्तित किया जाएगा।
- base64 के माध्यम से भेजें - चाहे छवियाँ स्थानीय रूप से संग्रहीत हों या क्लाउड में, सभी को base64 प्रारूप में भेजा जाएगा। यह उन स्थितियों के लिए उपयुक्त है जहाँ छवि URL को ऑनलाइन LLM सेवा द्वारा सीधे एक्सेस नहीं किया जा सकता है।
