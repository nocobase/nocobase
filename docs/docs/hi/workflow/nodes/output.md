# वर्कफ़्लो आउटपुट

## परिचय

`वर्कफ़्लो आउटपुट` नोड का उपयोग कॉल किए गए वर्कफ़्लो में उसके आउटपुट मान को परिभाषित करने के लिए किया जाता है। जब आप एक वर्कफ़्लो को दूसरे वर्कफ़्लो से कॉल करते हैं, तो `वर्कफ़्लो आउटपुट` नोड का उपयोग करके आप मान को कॉलर को वापस भेज सकते हैं।

## नोड बनाएँ

कॉल किए गए वर्कफ़्लो में, एक `वर्कफ़्लो आउटपुट` नोड जोड़ें:

![20241231002033](https://static-docs.nocobase.com/20241231002033.png)

## नोड कॉन्फ़िगर करें

### आउटपुट मान

आउटपुट मान के रूप में एक वेरिएबल दर्ज करें या चुनें। आउटपुट मान किसी भी प्रकार का हो सकता है, जैसे कि एक स्थिरांक (स्ट्रिंग, संख्या, बूलियन, दिनांक, या कस्टम JSON), या वर्कफ़्लो से कोई अन्य वेरिएबल।

![20241231003059](https://static-docs.nocobase.com/20241231003059.png)

:::info{title=सुझाव}
यदि आप कॉल किए गए वर्कफ़्लो में एक से अधिक `वर्कफ़्लो आउटपुट` नोड जोड़ते हैं, तो वर्कफ़्लो को कॉल करने पर, अंतिम निष्पादित `वर्कफ़्लो आउटपुट` नोड का मान ही आउटपुट के रूप में मिलेगा।
:::
