# अवलोकन

वर्कफ़्लो की अंतर्निहित सुविधाएँ सभी परिदृश्यों को कवर नहीं कर सकतीं। उदाहरण के लिए, अंतर्निहित नोड प्रकार सभी व्यावसायिक परिदृश्यों में हर संभव ऑपरेशन को शामिल नहीं कर सकते। इसलिए, हमने वर्कफ़्लो के विस्तार के लिए एक डिज़ाइन भी प्रदान किया है, जिसमें ट्रिगर और नोड प्रकारों का विस्तार करना शामिल है। जिन व्यावसायिक परिदृश्यों में अंतर्निहित सुविधाएँ पर्याप्त नहीं हैं, आप उन्हें लो-कोड तरीकों का उपयोग करके विस्तारित कर सकते हैं।
