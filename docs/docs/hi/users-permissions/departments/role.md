# विभाग भूमिका प्रबंधन

किसी विभाग से एक या एक से अधिक भूमिकाएँ (roles) जोड़कर, आप उस विभाग के सदस्यों को वे भूमिकाएँ दे सकते हैं।

किसी विभाग से भूमिकाएँ जोड़ने के दो तरीके हैं।

पहला तरीका यह है कि विभाग संपादन फ़ॉर्म (editing form) के भूमिका फ़ील्ड (role field) में भूमिका चुनें।

![](https://static-docs.nocobase.com/70f77bb89aa1fb415c152a51a51cc23b.png)

दूसरा तरीका यह है कि भूमिका प्रबंधन (role management) में संबंधित भूमिका (corresponding role) में विभाग जोड़ें।

![](https://static-docs.nocobase.com/f2a7bec89aa1fb415c152a51a51cc23b.png)

जिन विभागों के पास पहले से यह भूमिका है, उन्हें चुना नहीं जा सकता।

![](https://static-docs.nocobase.com/be10299893581e1f97a4e01ddd5c7e59.png)
