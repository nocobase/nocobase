# क्लस्टर डिप्लॉयमेंट

NocoBase एप्लिकेशन के प्रदर्शन और उपलब्धता को बेहतर बनाने के लिए उन्हें क्लस्टर मोड में डिप्लॉय करने का समर्थन करता है। क्लस्टर डिप्लॉयमेंट से आप एप्लिकेशन के अलग-अलग इंस्टेंस को कई सर्वर पर चला सकते हैं, या एक ही सर्वर पर मल्टी-कोर मोड में कई प्रोसेस चला सकते हैं।

विस्तृत जानकारी के लिए, कृपया [क्लस्टर मोड](/cluster-mode/) दस्तावेज़ देखें।
