# Git स्रोत से इंस्टॉल करें

:::tip पूर्वापेक्षाएँ

- Git, Node.js 20+ और Yarn 1.22.x इंस्टॉल होने चाहिए।
- आवश्यक डेटाबेस (MySQL 8.0.17+, MariaDB 10.9+, या PostgreSQL 10+ में से कोई एक) कॉन्फ़िगर और चालू होना चाहिए।

:::

## 1. NocoBase को अपने स्थानीय मशीन पर डाउनलोड करें

NocoBase का एक संस्करण चुनें ([संस्करण तुलना](../quickstart.md)) और संबंधित कमांड चलाएँ।

<Tabs>
<Tab label="नवीनतम संस्करण">

```bash
git clone https://github.com/nocobase/nocobase.git -b main --depth=1 my-nocobase
```

</Tab>
<Tab label="बीटा संस्करण">

```bash
git clone https://github.com/nocobase/nocobase.git -b next --depth=1 my-nocobase
```

</Tab>
<Tab label="अल्फा संस्करण">

```bash
git clone https://github.com/nocobase/nocobase.git -b develop --depth=1 my-nocobase
```

</Tab>
</Tabs>

## 2. डायरेक्टरी बदलें

```bash
cd my-nocobase
```

## 3. डिपेंडेंसीज़ इंस्टॉल करें

📢 नेटवर्क वातावरण, सिस्टम कॉन्फ़िगरेशन और अन्य कारकों के आधार पर, इस चरण में दस मिनट से अधिक का समय लग सकता है।

```bash
yarn install --frozen-lockfile
```

## 4. एनवायरनमेंट वैरिएबल सेट करें

NocoBase के लिए आवश्यक एनवायरनमेंट वैरिएबल रूट डायरेक्टरी में `.env` फ़ाइल में संग्रहीत होते हैं। अपनी वास्तविक स्थिति के अनुसार एनवायरनमेंट वैरिएबल को संशोधित करें।

```bash
TZ=UTC
APP_KEY=your-secret-key
DB_HOST=localhost
DB_PORT=5432
DB_DATABASE=postgres
DB_USER=nocobase
DB_PASSWORD=nocobase
```

:::warning एनवायरनमेंट वैरिएबल विवरण

- `TZ` का उपयोग एप्लिकेशन के टाइम ज़ोन को सेट करने के लिए किया जाता है, जो डिफ़ॉल्ट रूप से ऑपरेटिंग सिस्टम के टाइम ज़ोन पर सेट होता है।
- `APP_KEY` एप्लिकेशन की गुप्त कुंजी है, जिसका उपयोग यूज़र टोकन आदि बनाने के लिए किया जाता है (यदि `APP_KEY` बदल दिया जाता है, तो पुराने टोकन अमान्य हो जाएंगे)। यह कोई भी रैंडम स्ट्रिंग हो सकती है। कृपया इसे अपनी गुप्त कुंजी में बदलें और सुनिश्चित करें कि यह उजागर न हो।
- `DB_*` डेटाबेस से संबंधित हैं। यदि आप उदाहरण में दिए गए डिफ़ॉल्ट डेटाबेस सेवा का उपयोग नहीं कर रहे हैं, तो कृपया अपनी वास्तविक स्थिति के अनुसार उन्हें संशोधित करें।

:::

## 5. NocoBase इंस्टॉल करें

```bash
yarn nocobase install --lang=en-US
```

## 6. NocoBase शुरू करें

**डेवलपमेंट एनवायरनमेंट**

```bash
yarn dev
```

**प्रोडक्शन एनवायरनमेंट (अनुशंसित नहीं)**

प्रोडक्शन एनवायरनमेंट में स्रोत से सीधे डिप्लॉय करना अनुशंसित नहीं है। प्रोडक्शन के लिए, कृपया [प्रोडक्शन डिप्लॉयमेंट](../deployment/production.md) देखें।

```bash
yarn build
yarn start
```

## 7. NocoBase में लॉग इन करें

अपने ब्राउज़र में [http://localhost:13000](http://localhost:13000) खोलें। प्रारंभिक अकाउंट और पासवर्ड `admin@nocobase.com` और `admin123` हैं।

:::warning अकाउंट सुरक्षा चेतावनी

पहली बार लॉग इन करने के बाद, सिस्टम सुरक्षा सुनिश्चित करने के लिए कृपया डिफ़ॉल्ट पासवर्ड तुरंत बदलें।

:::
