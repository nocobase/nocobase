# उपयोगकर्ता अवरोधन

## परिचय

आप लॉक किए गए उपयोगकर्ताओं को प्रबंधित कर सकते हैं और सक्रिय रूप से उपयोगकर्ताओं को लॉक कर सकते हैं।

![](https://static-docs.nocobase.com/202412281450083.png)

### उपयोगकर्ता को अनलॉक करना

जो उपयोगकर्ता अमान्य पासवर्ड लॉगिन सीमा से अधिक होने के कारण लॉक हो गए हैं, उन्हें रिकॉर्ड हटाकर अनलॉक किया जा सकता है।

### उपयोगकर्ता को लॉक करना

उपयोगकर्ता रिकॉर्ड जोड़कर आप सक्रिय रूप से किसी उपयोगकर्ता को लॉक कर सकते हैं। एक बार जब कोई उपयोगकर्ता लॉक हो जाता है, तो उसे किसी भी प्रमाणीकरण विधि, जिसमें API keys भी शामिल हैं, के माध्यम से सिस्टम तक पहुँचने से रोक दिया जाएगा।
