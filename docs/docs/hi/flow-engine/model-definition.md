# चीन क्षेत्र फ़ील्ड

## परिचय

चीन क्षेत्र फ़ील्ड प्लगइन (`field-china-region`) NocoBase में "चीन क्षेत्र" फ़ील्ड प्रकार जोड़ता है।

## इंस्टॉलेशन

NocoBase के मुख्य इंटरफ़ेस में, "प्लगइन मैनेजर" पर जाएँ, "चीन क्षेत्र" प्लगइन खोजें और इसे सक्षम करें।

## उपयोग

संग्रह कॉन्फ़िगरेशन में, एक नया फ़ील्ड जोड़ें और फ़ील्ड प्रकार के रूप में "चीन क्षेत्र" चुनें।

![](/images/field-china-region-1.png)

### फ़ील्ड कॉन्फ़िगरेशन

फ़ील्ड कॉन्फ़िगरेशन में, आप फ़ील्ड के लिए डिस्प्ले मोड सेट कर सकते हैं:

- प्रांत
- प्रांत/शहर
- प्रांत/शहर/ज़िला
- प्रांत/शहर/ज़िला/काउंटी

![](/images/field-china-region-2.png)

### इंटरफ़ेस कॉन्फ़िगरेशन

इंटरफ़ेस कॉन्फ़िगरेशन में, आप "चीन क्षेत्र" फ़ील्ड के लिए विभिन्न इंटरफ़ेस फ़ॉर्म कॉन्फ़िगर कर सकते हैं।
