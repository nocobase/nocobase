# ctx.t()

RunJS में टेक्स्ट का अनुवाद करने के लिए एक i18n शॉर्टकट फ़ंक्शन, जो वर्तमान संदर्भ (context) की भाषा सेटिंग्स पर आधारित है। यह बटन, शीर्षक और प्रॉम्प्ट जैसे इनलाइन टेक्स्ट के अंतर्राष्ट्रीयकरण (internationalization) के लिए उपयुक्त है।

##适用场景 (उपयोग के मामले)

सभी RunJS निष्पादन वातावरणों (execution environments) में `ctx.t()` का उपयोग किया जा सकता है।

## 类型定义 (टाइप परिभाषा)

```ts
t(key: string, options?: Record<string, any>): string
```

## 参数 (पैरामीटर)

| पैरामीटर | प्रकार | विवरण |
|------|------|------|
| `key` | `string` | अनुवाद कुंजी (key) या प्लेसहोल्डर वाला टेम्पलेट (जैसे `Hello {{name}}`, `{{count}} rows`) |
| `options` | `object` | वैकल्पिक। इंटरपोलेशन वेरिएबल्स (जैसे `{ name: 'John', count: 5 }`), या i18n विकल्प (जैसे `defaultValue`, `ns`) |

## 返回值 (रिटर्न वैल्यू)

- अनुवादित स्ट्रिंग लौटाता है; यदि कुंजी के लिए कोई अनुवाद मौजूद नहीं है और `defaultValue` प्रदान नहीं किया गया है, तो यह स्वयं कुंजी या इंटरपोलेशन के बाद वाली स्ट्रिंग लौटा सकता है।

## 命名空间 (ns)

RunJS वातावरण के लिए **डिफ़ॉल्ट नेमस्पेस (namespace) `runjs` है**। जब `ns` निर्दिष्ट नहीं होता है, तो `ctx.t(key)` `runjs` नेमस्पेस में कुंजी की तलाश करेगा।

```ts
// डिफ़ॉल्ट रूप से 'runjs' नेमस्पेस से कुंजी खोजी जाती है
ctx.t('Submit'); // ctx.t('Submit', { ns: 'runjs' }) के बराबर

// किसी विशिष्ट नेमस्पेस से कुंजी खोजी जाती है
ctx.t('Submit', { ns: 'myModule' });

// क्रमिक रूप से कई नेमस्पेस में खोजता है (पहले runjs, फिर common)
ctx.t('Save', { ns: ['runjs', 'common'] });
```

## 示例 (उदाहरण)

### सरल कुंजी (Simple key)

```ts
ctx.t('Submit');
ctx.t('No data');
```

### इंटरपोलेशन वेरिएबल्स के साथ (With interpolation variables)

```ts
const text = ctx.t('Hello {{name}}', { name: ctx.user?.nickname || 'Guest' });
ctx.render(`<div>${text}</div>`);
```

```ts
ctx.message.success(ctx.t('Processed {{count}} rows', { count: rows.length }));
```

### डायनेमिक टेक्स्ट जैसे सापेक्ष समय (Dynamic copy)

```ts
if (minutes < 60) return ctx.t('{{count}} minutes ago', { count: minutes });
if (hours < 24) return ctx.t('{{count}} hours ago', { count: hours });
```

### नेमस्पेस निर्दिष्ट करना (Specifying a namespace)

```ts
ctx.t('Hello {{name}}', { name: 'Guest', ns: 'myModule' });
```

## 注意事项 (महत्वपूर्ण बातें)

- **लोकलाइज़ेशन प्लगइन**: टेक्स्ट का अनुवाद करने के लिए, लोकलाइज़ेशन प्लगइन सक्रिय होना चाहिए। अनुपलब्ध अनुवाद कुंजियाँ स्वचालित रूप से लोकलाइज़ेशन प्रबंधन सूची में आ जाएँगी, जिससे उनका एकीकृत रखरखाव और अनुवाद करना आसान हो जाता है।
- i18next-स्टाइल इंटरपोलेशन का समर्थन करता है: कुंजी में `{{variableName}}` का उपयोग करें और उसे बदलने के लिए `options` में संबंधित वेरिएबल पास करें।
- भाषा वर्तमान संदर्भ (जैसे `ctx.i18n.language`, उपयोगकर्ता लोकेल) द्वारा निर्धारित की जाती है।
