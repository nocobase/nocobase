# ctx.i18n

वर्तमान संदर्भ (context) का i18n इंस्टेंस, जिसका उपयोग भाषा पढ़ने या बदलने के लिए किया जाता है। **टेक्स्ट अनुवाद के लिए [ctx.t()](./t.md) का उपयोग करें**; `ctx.i18n.t` का उपयोग न करें।

## उपयुक्त परिदृश्य

सभी RunJS निष्पादन वातावरण (execution environments) `ctx.i18n` का उपयोग कर सकते हैं (जैसे JSBlock, JSField, JSItem, JSColumn, वर्कफ़्लो, लिंकेज नियम आदि)।

## टाइप परिभाषा (Type Definition)

```typescript
interface i18n: {
  language: string;
  changeLanguage(lng: string): Promise<any>;
}
```

## सामान्य गुण (Properties)

| गुण | टाइप | विवरण |
|------|------|------|
| `language` | `string` | वर्तमान में सक्रिय भाषा कोड (जैसे `en-US`, `zh-CN`) |

## सामान्य मेथड

### changeLanguage(lng)

वर्तमान भाषा बदलता है।

| पैरामीटर | टाइप | विवरण |
|------|------|------|
| `lng` | `string` | लक्ष्य भाषा कोड (जैसे `'en-US'`, `'zh-CN'`) |

**रिटर्न वैल्यू**: `Promise<any>`, भाषा बदलने की प्रक्रिया पूरी होने के बाद रिज़ॉल्व (resolve) होता है।

## उदाहरण

### वर्तमान भाषा पढ़ना

```ts
const lang = ctx.i18n.language;
// 'en-US' | 'zh-CN' | ...
if (lang.startsWith('zh')) {
  ctx.render(ctx.t('Chinese UI'));
} else {
  ctx.render(ctx.t('English UI'));
}
```

### भाषा बदलना

```ts
// अंग्रेज़ी में बदलें
await ctx.i18n.changeLanguage('en-US');

// चीनी में बदलें
await ctx.i18n.changeLanguage('zh-CN');
```

### भाषा बदलने का बटन

```ts
const { Button } = ctx.libs.antd;
const isZh = ctx.i18n.language.startsWith('zh');
ctx.render(
  <Button onClick={async () => {
    await ctx.i18n.changeLanguage(isZh ? 'en-US' : 'zh-CN');
  }}>
    {ctx.t(isZh ? 'Switch to English' : 'Switch to Chinese')}
  </Button>,
);
```

## महत्वपूर्ण बातें

- **अनुवाद टेक्स्ट**: `ctx.t()` का निरंतर उपयोग करें; `ctx.i18n.t` का उपयोग न करें।
