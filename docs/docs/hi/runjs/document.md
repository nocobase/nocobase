:::tip{title="AI अनुवाद सूचना"}
यह दस्तावेज़ AI द्वारा अनुवादित है। सटीक जानकारी के लिए कृपया [अंग्रेज़ी संस्करण](/runjs/document) देखें।
:::

कृपया वह चीनी स्रोत टेक्स्ट (source text) प्रदान करें जिसका आप अनुवाद करना चाहते हैं। मैं आपके निर्देशों में निर्दिष्ट सभी नियमों और शब्दावली को लागू करने के लिए तैयार हूँ।
