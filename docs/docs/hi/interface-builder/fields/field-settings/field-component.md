# फ़ील्ड घटक

## परिचय

फ़ील्ड घटक, फ़ील्ड की सामग्री को प्रदर्शित करने के लिए उपयोग किए जाते हैं।
कुछ फ़ील्ड में कई अंतर्निहित डिस्प्ले घटक होते हैं। उपयोगकर्ता अपनी वास्तविक ज़रूरतों के अनुसार उपयुक्त घटक चुन सकते हैं, ताकि फ़ील्ड के मान को अलग-अलग तरीकों से प्रस्तुत किया जा सके।

उदाहरण: देखने के मोड में, एक URL फ़ील्ड को पूर्वावलोकन घटक या URL घटक (डिफ़ॉल्ट) में स्विच किया जा सकता है।

![20251028214628](https://static-docs.nocobase.com/20251028214628.png)

उदाहरण: एक संबंध फ़ील्ड को उप-फ़ॉर्म, उप-तालिका (केवल कई-से-कई संबंधों के लिए), डेटा चयनकर्ता, ड्रॉपडाउन चयनकर्ता (डिफ़ॉल्ट), या फ़ाइल प्रबंधक (केवल तभी जब संबंध का लक्ष्य संग्रह एक फ़ाइल संग्रह हो) में स्विच किया जा सकता है।
