# सत्यापन नियम सेट करें

## परिचय

सत्यापन नियमों का उपयोग यह सुनिश्चित करने के लिए किया जाता है कि उपयोगकर्ता द्वारा दर्ज किया गया डेटा अपेक्षाओं के अनुरूप हो।

## फ़ील्ड सत्यापन नियम कहाँ सेट करें

### संग्रह फ़ील्ड के लिए सत्यापन नियम कॉन्फ़िगर करें

अधिकांश फ़ील्ड सत्यापन नियमों को कॉन्फ़िगर करने का समर्थन करते हैं। एक बार जब किसी फ़ील्ड में सत्यापन नियम कॉन्फ़िगर हो जाते हैं, तो डेटा सबमिट करते समय बैकएंड सत्यापन ट्रिगर होता है। विभिन्न प्रकार के फ़ील्ड अलग-अलग सत्यापन नियमों का समर्थन करते हैं।

### फ़ील्ड कॉन्फ़िगरेशन में फ़्रंटएंड सत्यापन

फ़ील्ड कॉन्फ़िगरेशन में सेट किए गए सत्यापन नियम फ़्रंटएंड सत्यापन को ट्रिगर करेंगे, ताकि यह सुनिश्चित हो सके कि उपयोगकर्ता इनपुट नियमों का पालन करता है।
