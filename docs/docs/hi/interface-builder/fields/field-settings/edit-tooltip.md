# टूलटिप संपादित करें

## परिचय

फ़ील्ड टूलटिप वह टेक्स्ट है जो किसी फ़ील्ड पर माउस ले जाने पर आपको दिखाई देता है। इसका मुख्य उद्देश्य उस फ़ील्ड के बारे में अतिरिक्त जानकारी या मार्गदर्शन देना है। इन टूलटिप में अक्सर फ़ील्ड का क्या उपयोग है, इसमें किस प्रारूप में डेटा डालना है, और इसकी क्या सीमाएँ हैं, जैसी बातें शामिल होती हैं। यह उपयोगकर्ताओं को फ़ील्ड को बेहतर ढंग से समझने और उसका सही उपयोग करने में मदद करता है।
