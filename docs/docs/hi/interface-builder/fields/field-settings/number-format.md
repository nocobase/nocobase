# संख्या स्वरूपण

## परिचय

संख्या स्वरूपण का अर्थ है, केवल पढ़ने की स्थिति में संख्यात्मक प्रकार के फ़ील्ड (फ़ॉर्मूला फ़ील्ड सहित) को दृश्य रूप से बेहतर और मानकीकृत करना। इसमें हज़ार विभाजक, दशमलव सटीकता, इकाई रूपांतरण और वैज्ञानिक संकेतन जैसी सेटिंग्स शामिल हैं। संख्या स्वरूपण को कॉन्फ़िगर करके, आप विभिन्न क्षेत्रों और उद्योगों की संख्यात्मक अभिव्यक्ति की ज़रूरतों को पूरा कर सकते हैं।

## उपयोग कैसे करें

- सरल इकाई रूपांतरण (गुणा/भाग) का समर्थन करता है
- हज़ार विभाजक
- उपसर्ग और प्रत्यय
- सटीकता
- वैज्ञानिक संकेतन।

उदाहरण: किसी ऑर्डर की राशि को फ़ॉर्मेट करें, जिसमें मुद्रा प्रतीक (जैसे, ₹), हज़ार विभाजक (जैसे कॉमा या बिंदु), और दशमलव सटीकता जोड़ना शामिल है, ताकि राशि स्पष्ट और पढ़ने में आसान हो।
