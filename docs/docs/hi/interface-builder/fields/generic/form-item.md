# फ़ॉर्म फ़ील्ड

## परिचय

फ़ॉर्म में अलग-अलग तरह के फ़ील्ड के लिए अलग-अलग कॉन्फ़िगरेशन विकल्प होते हैं। आप प्लगइन के ज़रिए फ़ील्ड के कॉन्फ़िगरेशन विकल्पों को बढ़ा सकते हैं ताकि अपनी खास ज़रूरतों को पूरा कर सकें।

![20251025174555](https://static-docs.nocobase.com/20251025174555.png)

## फ़ील्ड कॉन्फ़िगरेशन विकल्प

### डिफ़ॉल्ट मान

फ़ील्ड के लिए कॉन्स्टेंट/वेरिएबल को डिफ़ॉल्ट मान के तौर पर सेट करने की सुविधा मिलती है (यह सुविधा केवल नए फ़ॉर्म के लिए उपलब्ध है)।

![20251025174724](https://static-docs.nocobase.com/20251025174724.png)

ज़्यादा जानकारी के लिए, [डिफ़ॉल्ट मान सेट करें](/interface-builder/fields/field-settings/default-value) देखें।
