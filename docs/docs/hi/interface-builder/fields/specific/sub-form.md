# सब-फ़ॉर्म

## परिचय

सब-फ़ॉर्म उन स्थितियों के लिए उपयुक्त हैं जहाँ आप डेटा को लिंक करने से पहले नया संबंधित डेटा बनाते हैं। बहु-स्तरीय संबंधित डेटा को नेस्टेड फ़ॉर्म के रूप में स्पष्ट रूप से प्रदर्शित किया जाता है। डेटा सेलेक्टर और ड्रॉपडाउन घटकों की तुलना में, सब-फ़ॉर्म आपको वर्तमान पेज ब्लॉक के भीतर संबंधित **संग्रह** के फ़ील्ड को सीधे प्रबंधित करने की सुविधा देते हैं, और संबंधित डेटा मुख्य फ़ॉर्म के साथ ही सबमिट किया जाता है।
