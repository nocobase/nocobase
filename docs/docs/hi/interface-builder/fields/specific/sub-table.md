# सब-टेबल

## परिचय

सब-टेबल एक-से-कई संबंध फ़ील्ड को संभालने के लिए उपयुक्त है। यह लक्ष्य **संग्रह** में नया डेटा एक साथ बनाने और उसे संबंधित करने, या मौजूदा डेटा को चुनकर संबंधित करने की सुविधा देता है।

## उपयोग के निर्देश

![20251027223350](https://static-docs.nocobase.com/20251027223350.png)

सब-टेबल में विभिन्न प्रकार के फ़ील्ड अलग-अलग फ़ील्ड घटक दिखाते हैं। बड़े फ़ील्ड (जैसे रिच टेक्स्ट, JSON, बहु-पंक्ति टेक्स्ट) को एक पॉप-अप मोडल के माध्यम से संपादित किया जाता है।

![20251027223426](https://static-docs.nocobase.com/20251027223426.png)

सब-टेबल में संबंध फ़ील्ड।

ऑर्डर (एक-से-कई) > ऑर्डर प्रोडक्ट (एक-से-एक) > अपॉर्चुनिटी

![20251027223530](https://static-docs.nocobase.com/20251027223530.png)

संबंध फ़ील्ड के लिए डिफ़ॉल्ट घटक ड्रॉपडाउन है (यह ड्रॉपडाउन/डेटा पिकर को सपोर्ट करता है)।

![20251027223754](https://static-docs.nocobase.com/20251027223754.png)
