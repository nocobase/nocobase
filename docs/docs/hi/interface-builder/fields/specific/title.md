# शीर्षक

## परिचय

केवल पढ़ने की स्थिति में, संबंध फ़ील्ड डिफ़ॉल्ट रूप से **शीर्षक फ़ील्ड घटक** का उपयोग करके प्रदर्शित किए जाते हैं। यह शीर्षक फ़ील्ड के मान को दिखाकर वर्तमान में संबंधित रिकॉर्ड की पहचान करता है।
सिस्टम, शीर्षक फ़ील्ड के प्रकार के आधार पर, स्वचालित रूप से संबंधित शीर्षक फ़ील्ड घटक का मिलान और उसे रेंडर करता है।
