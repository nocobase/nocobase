# उप-विवरण

## परिचय

उप-विवरण, केवल-पठन मोड में उप-फ़ॉर्म का संगत घटक है। लेबल और शीर्षक घटकों की तुलना में, उप-विवरण न केवल वर्तमान संग्रह से अधिक डेटा प्रदर्शित कर सकता है, बल्कि संबंधित संग्रहों से डेटा प्रदर्शित करने के लिए भी कॉन्फ़िगर किया जा सकता है। बहु-स्तरीय संबंध डेटा को नेस्टेड विवरणों के रूप में स्पष्ट रूप से प्रदर्शित किया जाता है।
