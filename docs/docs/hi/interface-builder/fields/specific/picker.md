# डेटा सेलेक्टर

## परिचय

डेटा सेलेक्टर, एक पॉपअप टेबल सेलेक्टर के रूप में, आपको लक्ष्य संग्रह से मौजूदा डेटा चुनने या लक्ष्य संग्रह में डेटा जोड़ने के बाद उसे संबद्ध करने की सुविधा देता है।

![20251027203343](https://static-docs.nocobase.com/20251027203343.png)

चयनित डेटा टेबल सेलेक्टर में प्रदर्शित नहीं होगा।

![20251027203717](https://static-docs.nocobase.com/20251027203717.gif)

टेबल सेलेक्टर में, आप लक्ष्य संग्रह को आगे भी प्रबंधित कर सकते हैं (जैसे जोड़ना, हटाना, इम्पोर्ट/एक्सपोर्ट करना आदि)।

![20251027204028](https://static-docs.nocobase.com/20251027204028.png)
