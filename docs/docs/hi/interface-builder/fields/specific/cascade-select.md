# कैस्केडिंग चयन

## परिचय

कैस्केडिंग चयनकर्ता उन संबंध फ़ील्ड्स के लिए डिज़ाइन किया गया है जिनका लक्ष्य संग्रह एक ट्री (वृक्ष) तालिका है। यह उपयोगकर्ताओं को ट्री संग्रह की पदानुक्रमित संरचना के अनुसार डेटा चुनने की सुविधा देता है, और त्वरित फ़िल्टरिंग के लिए फ़ज़ी सर्च का भी समर्थन करता है।

## उपयोग निर्देश

- **एक-से-एक** संबंध के लिए, कैस्केडिंग चयनकर्ता **एकल-चयन** होता है।

![20251125214656](https://static-docs.nocobase.com/20251125214656.png)

- **एक-से-अनेक** संबंध के लिए, कैस्केडिंग चयनकर्ता **बहु-चयन** होता है।

![20251125215318](https://static-docs.nocobase.com/20251125215318.png)
