# फ़ाइल मैनेजर

## परिचय

फ़ाइल मैनेजर एक संबंध फ़ील्ड घटक है जिसका उपयोग उन संबंधों को संभालने के लिए किया जाता है जहाँ लक्ष्य संग्रह फ़ाइलों का संग्रह होता है।

![20251027194124](https://static-docs.nocobase.com/20251027194124.png)

## फ़ील्ड कॉन्फ़िगरेशन

- [फ़ील्ड घटक](/interface-builder/fields/association-field): अन्य संबंध फ़ील्ड घटकों पर स्विच करें, जैसे ड्रॉपडाउन, डेटा सेलेक्टर आदि।
- मौजूदा फ़ाइलों को चुनने की अनुमति दें: यह डिफ़ॉल्ट रूप से सक्षम है, जिससे आप अपलोड की गई फ़ाइलों में से चुनकर उन्हें संबद्ध कर सकते हैं।
