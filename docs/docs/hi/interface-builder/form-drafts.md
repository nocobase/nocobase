# फ़ॉर्म ड्राफ्ट (Form Drafts)

जब फ़ॉर्म ड्राफ्ट सक्षम होते हैं, तो भरी जा रही सामग्री स्वचालित रूप से स्थानीय रूप से (locally) सहेजी जाती है। भले ही फ़ॉर्म गलती से बंद हो जाए या पेज रिफ्रेश हो जाए, भरा हुआ डेटा नहीं खोएगा।

![20251213093717](https://static-docs.nocobase.com/20251213093717.png)

जब कोई ड्राफ्ट मौजूद होता है, तो फ़ॉर्म के शीर्ष पर एक सूचना दिखाई देगी। सफलतापूर्वक सबमिट करने के बाद, ड्राफ्ट स्वचालित रूप से हटा दिया जाएगा; यदि अब इसकी आवश्यकता नहीं है, तो इसे किसी भी समय मैन्युअल रूप से भी हटाया जा सकता है।

![20251213092734](https://static-docs.nocobase.com/20251213092734.png)

::: tip सुझाव

वर्तमान में, फ़ॉर्म ड्राफ्ट केवल स्थानीय कैशिंग (local caching) का समर्थन करते हैं। भविष्य के संस्करणों में मल्टी-डिवाइस सिंक्रोनाइज़ेशन को सक्षम करने के लिए सर्वर-साइड कैशिंग का समर्थन किया जाएगा।

:::
