# चार्ट

## परिचय

NocoBase का डेटा विज़ुअलाइज़ेशन प्लगइन आपको विज़ुअल डेटा क्वेरी और कई तरह के चार्ट घटक प्रदान करता है। आप साधारण कॉन्फ़िगरेशन के ज़रिए तेज़ी से विज़ुअलाइज़ेशन डैशबोर्ड बना सकते हैं, जिससे आप डेटा अंतर्दृष्टि प्रदर्शित कर सकें और बहु-आयामी डेटा विश्लेषण व प्रस्तुति का समर्थन कर सकें।
