# ग्रिड कार्ड

## परिचय

ग्रिड कार्ड ब्लॉक डेटा रिकॉर्ड की संक्षिप्त जानकारी को कार्ड के रूप में दिखाता है। यह अलग-अलग स्क्रीन साइज़ के लिए कॉलम की संख्या को कॉन्फ़िगर करने की सुविधा देता है, ताकि यह सुनिश्चित हो सके कि यह विभिन्न डिवाइस पर उपयोगकर्ता के अनुकूल दिखे।
