# लेआउट

## परिचय

फॉर्म ब्लॉक, डिटेल्स ब्लॉक और लिस्ट ब्लॉक जैसे फॉर्म-जैसे ब्लॉक लेआउट सेटिंग्स को सपोर्ट करते हैं। लेआउट के विकल्प वर्टिकल लेआउट (जहाँ लेबल और इनपुट फ़ील्ड एक के बाद एक पंक्तियों में व्यवस्थित होते हैं) और हॉरिजॉन्टल लेआउट (जहाँ लेबल और इनपुट फ़ील्ड अगल-बगल रखे जाते हैं) हैं। उपयोगकर्ता अपनी विशिष्ट आवश्यकताओं के अनुसार ब्लॉक फ़ील्ड लेआउट को एडजस्ट कर सकते हैं, ताकि जानकारी दर्ज करना या देखना स्पष्ट और कुशल दोनों हो।

### वर्टिकल (डिफ़ॉल्ट)

डिफ़ॉल्ट लेआउट वर्टिकल होता है।

### हॉरिजॉन्टल

हॉरिजॉन्टल लेआउट लेबल और इनपुट फ़ील्ड को अगल-बगल दिखाता है, जिससे जगह की प्रभावी बचत होती है। लंबे शीर्षक वाले फ़ील्ड के लिए, आप उन्हें रैप करने या एलिप्सिस (संक्षेप) दिखाने का विकल्प चुन सकते हैं, ताकि इंटरफ़ेस साफ-सुथरा और आकर्षक बना रहे।

![20251024120237](https://static-docs.nocobase.com/20251024120237.png)

#### फ़ील्ड शीर्षक अलाइनमेंट (संरेखण)

- **बाएँ अलाइन करें** (डिफ़ॉल्ट)
- **दाएँ अलाइन करें**

#### फ़ील्ड शीर्षक की चौड़ाई

- डिफ़ॉल्ट चौड़ाई **120px** है, और न्यूनतम चौड़ाई **50px** तक सीमित है।

#### चौड़ाई से अधिक होने पर फ़ील्ड शीर्षकों को कैसे हैंडल करें

- **रैप करें**
- **एलिप्सिस**

#### क्या कॉलन दिखाएँ

आप चुन सकते हैं कि फ़ील्ड शीर्षक के बाद कॉलन (:) दिखाना है या नहीं।
