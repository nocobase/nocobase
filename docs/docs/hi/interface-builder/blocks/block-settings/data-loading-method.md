# डेटा लोडिंग मोड

## परिचय

डिफ़ॉल्ट डेटा लोडिंग मोड "फ़िल्टर शर्तें खाली होने पर सभी डेटा लोड करें" है। यहाँ, "फ़िल्टर शर्तें" का अर्थ "फ़िल्टर ब्लॉक" (Filter block) के मानों से है। दूसरा विकल्प "फ़िल्टर शर्तें खाली होने पर डेटा लोड न करें" है।
