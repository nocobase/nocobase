# एक्शन

UI में, एक्शन एक बटन होता है जिसका उपयोग निर्देशों के एक सेट को ट्रिगर करने के लिए किया जाता है। इसे सीधे किसी पेज पर, डायलॉग बॉक्स में या ड्रॉअर में रखा जा सकता है, और ब्लॉक्स के साथ मिलाकर भी इस्तेमाल किया जा सकता है। वर्तमान में, गतिशील रूप से कॉन्फ़िगर किए जा सकने वाले एक्शन मुख्य रूप से ब्लॉक्स के साथ ही उपयोग किए जाते हैं।
