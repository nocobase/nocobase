# मॉडल खोलें

## परिचय

**मॉडल खोलें** क्रिया, "देखें" और "संपादित करें" क्रियाओं के समान है। आप किसी तालिका की पंक्तियों या हेडर में एक **मॉडल खोलें** बटन कॉन्फ़िगर कर सकते हैं। इस बटन पर क्लिक करने पर एक संबंधित मॉडल खुलता है, और इस मॉडल के भीतर के ब्लॉक वर्तमान संदर्भ रिकॉर्ड के आधार पर डेटा दिखाते हैं।

मॉडल में अपनी आवश्यकतानुसार ब्लॉक कॉन्फ़िगर करें।

## क्रिया सेटिंग्स

![20251029190352](https://static-docs.nocobase.com/20251029190352.png)

- [बटन संपादित करें](/interface-builder/actions/action-settings/edit-button): बटन का शीर्षक, प्रकार और आइकन संपादित करें।
- [लिंकेज नियम](/interface-builder/linkage-rule): बटन की स्थिति (दिखाई दे रहा/अक्षम) को गतिशील रूप से नियंत्रित करें।
- [मॉडल संपादित करें](/interface-builder/actions/action-settings/edit-popup): खोलने का तरीका (मॉडल या ड्रॉअर) और आकार सेट करें।
