# बल्क एडिट (Bulk Edit)

## परिचय

बल्क एडिट उन स्थितियों के लिए उपयुक्त है जहाँ डेटा के लचीले बैच अपडेट की आवश्यकता होती है। बल्क एडिट बटन पर क्लिक करने के बाद, आप एक पॉप-अप विंडो में बल्क एडिट फ़ॉर्म को कॉन्फ़िगर कर सकते हैं और प्रत्येक फ़ील्ड के लिए अलग-अलग अपडेट रणनीतियाँ (strategies) सेट कर सकते हैं।

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM.png)


## एक्शन कॉन्फ़िगरेशन

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_13_AM%20(1).png)


## उपयोगकर्ता मार्गदर्शिका

### बल्क एडिट फ़ॉर्म कॉन्फ़िगरेशन

1. एक बल्क एडिट बटन जोड़ें।

2. बल्क एडिटिंग का दायरा (scope) सेट करें: चयनित (Selected) / सभी (All), डिफ़ॉल्ट रूप से 'चयनित' पर सेट होता है।

![](https://static-docs.nocobase.com/Orders-02-12-2026_07_14_AM.png)

3. एक बल्क एडिट फ़ॉर्म जोड़ें।

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_14_AM.png)

4. एडिट किए जाने वाले फ़ील्ड्स को कॉन्फ़िगर करें और एक सबमिट बटन जोड़ें।

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM%20(1).png)

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_07_15_AM.png)

### फ़ॉर्म सबमिशन

1. डेटा की उन पंक्तियों (rows) को चुनें जिन्हें एडिट करना है।

2. फ़ील्ड्स के लिए एडिट मोड चुनें और सबमिट किए जाने वाले मान (values) भरें।

![](https://static-docs.nocobase.com/Bulk-edit-02-12-2026_10_33_AM.png)

:::info{title=उपलब्ध_एडिट_मोड}
* **अपडेट न करें (Do not update)**: फ़ील्ड अपरिवर्तित रहता है।
* **इसमें बदलें (Change to)**: फ़ील्ड को सबमिट किए गए मान के साथ अपडेट करें।
* **साफ़ करें (Clear)**: फ़ील्ड के डेटा को साफ़ करें।

:::

3. फ़ॉर्म सबमिट करें।
