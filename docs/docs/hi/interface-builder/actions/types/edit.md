# संपादन

## परिचय

संपादन क्रिया एक पंक्ति बटन है जो वर्तमान रिकॉर्ड को संदर्भ के रूप में लेता है। यह एक पॉपअप के रूप में प्रस्तुत होता है जहाँ आप ब्लॉक को अपनी आवश्यकतानुसार अनुकूलित और कॉन्फ़िगर कर सकते हैं।

## क्रिया सेटिंग्स

![20251029165620](https://static-docs.nocobase.com/20251029165620.png)

- [लिंकेज नियम](/interface-builder/actions/action-settings/linkage-rule): बटन को गतिशील रूप से दिखाएं/छिपाएं;
- [संपादन बटन](/interface-builder/actions/action-settings/edit-button): बटन का शीर्षक, प्रकार और आइकन संपादित करें;
- [संपादन पॉपअप](/interface-builder/actions/action-settings/edit-popup): खोलने का तरीका और आकार सेट करें।
