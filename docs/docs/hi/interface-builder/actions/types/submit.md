# सबमिट करें

## परिचय

सबमिट कार्रवाई का उपयोग फ़ॉर्म डेटा को सेव करने के लिए किया जाता है (यह फ़ॉर्म ब्लॉक के लिए विशिष्ट है)। इसे वर्कफ़्लो के साथ जोड़कर डेटा ऑटोमेशन प्रक्रियाओं को लागू भी किया जा सकता है।
