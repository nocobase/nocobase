# जोड़ें
## परिचय

"जोड़ें" क्रिया वर्तमान ब्लॉक के **संग्रह** में एक रिकॉर्ड जोड़ती है। एक फ़ॉर्म ब्लॉक को कॉन्फ़िगर करके, आप वर्तमान ब्लॉक के **संग्रह** के लिए नए डेटा रिकॉर्ड दर्ज कर सकते हैं।
