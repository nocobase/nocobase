# फ़िल्टर

## परिचय

फ़िल्टर क्रिया आमतौर पर डेटा ब्लॉक में विभिन्न शर्तों के आधार पर डेटा को फ़िल्टर करने के लिए उपयोग की जाती है। कृपया ध्यान दें कि यदि किसी ब्लॉक के लिए पहले से ही डेटा स्कोप कॉन्फ़िगर किया गया है, तो फ़िल्टर क्रिया को डेटा स्कोप की शर्तों के साथ मिलाकर अंतिम फ़िल्टरिंग शर्त के रूप में लागू किया जाएगा।

### फ़िल्टर किए जा सकने वाले फ़ील्ड

डिफ़ॉल्ट रूप से, इस **संग्रह** के सभी फ़ील्ड सक्षम होते हैं। केवल सक्षम फ़ील्ड को ही शर्त फ़ील्ड सूची में चुना जा सकता है।

संबंधित **संग्रहों** के फ़ील्ड को भी फ़िल्टर फ़ील्ड के रूप में उपयोग किया जा सकता है।
