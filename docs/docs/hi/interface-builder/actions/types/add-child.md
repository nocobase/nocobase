# उप-रिकॉर्ड जोड़ें

## परिचय

यह ऑपरेशन खास तौर पर **ट्री टेबल ब्लॉक** के लिए डिज़ाइन किया गया है, ताकि आप तेज़ी से उप-रिकॉर्ड जोड़ सकें। उप-रिकॉर्ड जोड़ते समय, चुनी गई पंक्ति को पैरेंट नोड माना जाता है। इससे यूज़र किसी भी पैरेंट के तहत आसानी से उप-रिकॉर्ड बना सकते हैं। यह सुविधा पदानुक्रमित डेटा (hierarchical data) के प्रबंधन प्रक्रिया को बहुत सरल बनाती है, जिससे डेटा संरचनाओं का विस्तार और संगठन ज़्यादा कुशल और सहज हो जाता है।
