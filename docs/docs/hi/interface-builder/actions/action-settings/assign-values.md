# फ़ील्ड असाइनमेंट

## परिचय

फ़ील्ड असाइनमेंट का अर्थ है किसी ऑपरेशन को निष्पादित करने से पहले फ़ील्ड के मान को पहले से निर्धारित करना। रिकॉर्ड को अपडेट या बनाते समय, यह डेटा स्वचालित रूप से भर जाता है, जिससे आपको मैन्युअल रूप से जानकारी दर्ज करने की आवश्यकता नहीं पड़ती। उदाहरण के लिए, आप ऑर्डर की स्थिति को एक साथ अपडेट कर सकते हैं, या जब कोई ऑर्डर डिलीवर होता है, तो डिलीवरी के समय को वर्तमान समय पर सेट कर सकते हैं।
