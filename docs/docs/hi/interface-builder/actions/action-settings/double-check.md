# पुष्टि

## परिचय

पुष्टि कॉन्फ़िगरेशन का उपयोग यह सुनिश्चित करने के लिए किया जाता है कि कोई क्रिया निष्पादित हुई है या नहीं। जब आप किसी क्रिया पर क्लिक करते हैं, तो एक पुष्टि डायलॉग बॉक्स पॉप अप होता है। डिलीट बटन पर डिफ़ॉल्ट रूप से पुष्टि सक्षम होती है, ताकि आकस्मिक संचालन के कारण डेटा हानि या अपरिवर्तनीय परिणामों को रोका जा सके।
