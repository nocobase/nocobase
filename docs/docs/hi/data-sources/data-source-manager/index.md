# डेटा स्रोत प्रबंधन

## परिचय

NocoBase, डेटा स्रोतों और उनके संग्रहों को प्रबंधित करने के लिए एक डेटा स्रोत प्रबंधन प्लगइन प्रदान करता है। डेटा स्रोत प्रबंधन प्लगइन केवल सभी डेटा स्रोतों के लिए एक प्रबंधन इंटरफ़ेस प्रदान करता है, न कि डेटा स्रोतों तक पहुँचने की क्षमता। इसे विभिन्न डेटा स्रोत प्लगइन्स के साथ मिलकर उपयोग करने की आवश्यकता है। वर्तमान में पहुँच के लिए समर्थित डेटा स्रोत में शामिल हैं:

- [मुख्य डेटाबेस](/data-sources/data-source-main): NocoBase का मुख्य डेटाबेस, MySQL, PostgreSQL, और MariaDB जैसे रिलेशनल डेटाबेस को सपोर्ट करता है।
- [बाहरी MySQL](/data-sources/data-source-external-mysql): बाहरी MySQL डेटाबेस को डेटा स्रोत के रूप में उपयोग करें।
- [बाहरी MariaDB](/data-sources/data-source-external-mariadb): बाहरी MariaDB डेटाबेस को डेटा स्रोत के रूप में उपयोग करें।
- [बाहरी PostgreSQL](/data-sources/data-source-external-postgres): बाहरी PostgreSQL डेटाबेस को डेटा स्रोत के रूप में उपयोग करें।
- [बाहरी MSSQL](/data-sources/data-source-external-mssql): बाहरी MSSQL (SQL Server) डेटाबेस को डेटा स्रोत के रूप में उपयोग करें।
- [बाहरी Oracle](/data-sources/data-source-external-oracle): बाहरी Oracle डेटाबेस को डेटा स्रोत के रूप में उपयोग करें।

इसके अतिरिक्त, प्लगइन्स के माध्यम से और अधिक प्रकारों का विस्तार किया जा सकता है, जो सामान्य प्रकार के डेटाबेस या API (SDK) प्रदान करने वाले प्लेटफ़ॉर्म हो सकते हैं।

## स्थापना

यह एक अंतर्निहित प्लगइन है, इसे अलग से स्थापित करने की आवश्यकता नहीं है।

## उपयोग के निर्देश

जब एप्लिकेशन को इनिशियलाइज़ और इंस्टॉल किया जाता है, तो NocoBase डेटा को स्टोर करने के लिए डिफ़ॉल्ट रूप से एक डेटा स्रोत प्रदान किया जाता है, जिसे मुख्य डेटाबेस के रूप में जाना जाता है। अधिक जानकारी के लिए, [मुख्य डेटाबेस](/data-sources/data-source-main/) दस्तावेज़ देखें।

### बाहरी डेटा स्रोत

बाहरी डेटाबेस को डेटा स्रोतों के रूप में सपोर्ट किया जाता है। अधिक जानकारी के लिए, [बाहरी डेटाबेस / परिचय](/data-sources/data-source-manager/external-database) दस्तावेज़ देखें।
