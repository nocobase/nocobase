# Tencent Cloud COS

Tencent Cloud COS पर आधारित एक स्टोरेज इंजन। इसका उपयोग करने से पहले, आपको संबंधित अकाउंट और अनुमतियाँ तैयार करनी होंगी।

## कॉन्फ़िगरेशन पैरामीटर

![Tencent COS स्टोरेज इंजन कॉन्फ़िगरेशन का उदाहरण](https://static-docs.nocobase.com/20240712222125.png)

:::info{title=संकेत}
यह सेक्शन केवल Tencent Cloud COS स्टोरेज इंजन के विशिष्ट पैरामीटर को कवर करता है। सामान्य पैरामीटर के लिए, कृपया [इंजन के सामान्य पैरामीटर](./index.md#common-engine-parameters) देखें।
:::

### क्षेत्र

COS स्टोरेज का क्षेत्र भरें, उदाहरण के लिए: `ap-chengdu`।

:::info{title=संकेत}
आप [Tencent Cloud COS कंसोल](https://console.cloud.tencent.com/cos) में स्टोरेज बकेट की क्षेत्र जानकारी देख सकते हैं, और केवल क्षेत्र के प्रीफिक्स हिस्से को लेना होगा (पूरे डोमेन नाम की आवश्यकता नहीं है)।
:::

### SecretId

Tencent Cloud अधिकृत एक्सेस कुंजी का ID भरें।

### SecretKey

Tencent Cloud अधिकृत एक्सेस कुंजी का Secret भरें।

### बकेट

COS बकेट का नाम भरें, उदाहरण के लिए: `qing-cdn-1234189398`।
