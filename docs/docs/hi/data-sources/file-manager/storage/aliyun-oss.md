# अलीबाबा क्लाउड OSS
यह अलीबाबा क्लाउड OSS पर आधारित एक स्टोरेज इंजन है। इसका उपयोग करने से पहले, आपको संबंधित अकाउंट और अनुमतियाँ तैयार करनी होंगी।

## कॉन्फ़िगरेशन

![अलीबाबा क्लाउड OSS कॉन्फ़िगरेशन का उदाहरण](https://static-docs.nocobase.com/20240712220011.png)

:::info{title=ध्यान दें}
यहाँ केवल अलीबाबा क्लाउड OSS स्टोरेज इंजन के विशिष्ट पैरामीटर बताए गए हैं। सामान्य पैरामीटर के लिए, कृपया [इंजन के सामान्य पैरामीटर](./index.md#common-engine-parameters) देखें।
:::

### क्षेत्र
OSS स्टोरेज का क्षेत्र दर्ज करें, उदाहरण के लिए: `oss-cn-hangzhou`।

:::info{title=ध्यान दें}
आप [अलीबाबा क्लाउड OSS कंसोल](https://oss.console.aliyun.com/) में स्टोरेज बकेट की क्षेत्र जानकारी देख सकते हैं। आपको केवल क्षेत्र के प्रीफ़िक्स भाग का उपयोग करना होगा (पूरे डोमेन नाम की आवश्यकता नहीं है)।
:::

### AccessKey ID
अलीबाबा क्लाउड के अधिकृत एक्सेस की (Access Key) का ID दर्ज करें।

### AccessKey Secret
अलीबाबा क्लाउड के अधिकृत एक्सेस की (Access Key) का Secret दर्ज करें।

### बकेट
OSS बकेट का नाम दर्ज करें।
