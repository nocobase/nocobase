# स्थानीय स्टोरेज

अपलोड की गई फ़ाइलें सर्वर की स्थानीय डायरेक्टरी में सहेजी जाएँगी। यह उन स्थितियों के लिए उपयुक्त है जहाँ सिस्टम द्वारा प्रबंधित फ़ाइलों की कुल संख्या कम हो या फिर प्रायोगिक उपयोग के लिए।

## कॉन्फ़िगरेशन विकल्प

![फ़ाइल स्टोरेज इंजन विकल्पों का उदाहरण](https://static-docs.nocobase.com/20240529115151.png)

:::info{title=ध्यान दें}
यह खंड केवल स्थानीय स्टोरेज इंजन के विशिष्ट विकल्पों को कवर करता है। सामान्य पैरामीटर के लिए, कृपया [इंजन के सामान्य पैरामीटर](./index.md#इंजन-के-सामान्य-पैरामीटर) देखें।
:::

### पाथ

पाथ सर्वर पर सहेजी गई फ़ाइल के सापेक्ष पाथ और URL एक्सेस पाथ दोनों को दर्शाता है। उदाहरण के लिए, "`user/avatar`" (शुरुआत और अंत में "`/`" के बिना) का अर्थ है:

1. अपलोड की गई फ़ाइल का सर्वर पर सहेजा गया सापेक्ष पाथ: `/path/to/nocobase-app/storage/uploads/user/avatar`।
2. फ़ाइल को एक्सेस करने के लिए URL प्रीफ़िक्स: `http://localhost:13000/storage/uploads/user/avatar`।
