# फ़ाइल संग्रह

## परिचय

फ़ाइलें एक विशेष संरचना वाले संग्रह में रिकॉर्ड होती हैं, जिसे फ़ाइल संग्रह कहा जाता है। इसका उपयोग फ़ाइलों के मेटाडेटा को संग्रहीत करने और फ़ाइल मैनेजर के माध्यम से प्रबंधित करने के लिए किया जाता है।
