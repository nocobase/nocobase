# वृक्ष संग्रह

## परिचय

वृक्ष संरचना संग्रह एक डेटा संग्रह डिज़ाइन पैटर्न है जिसका उपयोग पदानुक्रमित डेटा को व्यवस्थित करने के लिए किया जाता है। यह संग्रह संरचना एक वृक्ष के समान होती है, जहाँ प्रत्येक डेटा आइटम के एक या अधिक चाइल्ड आइटम हो सकते हैं, और उन चाइल्ड आइटम के बदले में अपने स्वयं के वंशज हो सकते हैं।
