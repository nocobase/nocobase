# बाहरी डेटा तालिकाएँ (FDW) जोड़ें

## परिचय

यह एक प्लगइन है जो डेटाबेस के फॉरेन डेटा रैपर (FDW) का उपयोग करके रिमोट डेटा तालिकाएँ जोड़ता है। वर्तमान में, यह MySQL और PostgreSQL डेटाबेस को सपोर्ट करता है।

:::info{title="डेटा स्रोत जोड़ना बनाम बाहरी डेटा तालिकाएँ जोड़ना"}
- **डेटा स्रोत जोड़ना** किसी विशिष्ट डेटाबेस या API सेवा के साथ कनेक्शन स्थापित करने को संदर्भित करता है, जिससे आप डेटाबेस की सभी सुविधाओं या API द्वारा प्रदान की गई सेवाओं का पूरी तरह से उपयोग कर सकते हैं;
- **बाहरी डेटा तालिकाएँ जोड़ना** बाहर से डेटा प्राप्त करने और उसे स्थानीय उपयोग के लिए मैप करने को संदर्भित करता है। डेटाबेस में, इसे FDW (फॉरेन डेटा रैपर) कहा जाता है, जो एक डेटाबेस तकनीक है जो रिमोट तालिकाओं को स्थानीय तालिकाओं के रूप में उपयोग करने पर केंद्रित है और केवल एक-एक करके जोड़ा जा सकता है। चूंकि यह रिमोट एक्सेस है, इसलिए इसका उपयोग करते समय विभिन्न बाधाएँ और सीमाएँ होंगी।

इन दोनों का उपयोग एक साथ भी किया जा सकता है। पहला डेटा स्रोत के साथ कनेक्शन स्थापित करने के लिए उपयोग किया जाता है, और दूसरा क्रॉस-डेटा स्रोत एक्सेस के लिए उपयोग किया जाता है। उदाहरण के लिए, यदि कोई PostgreSQL डेटा स्रोत जुड़ा हुआ है, तो उस डेटा स्रोत में एक तालिका FDW के आधार पर बनाई गई एक बाहरी डेटा तालिका हो सकती है।
:::

### MySQL

MySQL `federated` इंजन का उपयोग करता है, जिसे सक्रिय करने की आवश्यकता है, और यह रिमोट MySQL और प्रोटोकॉल-संगत डेटाबेस, जैसे MariaDB, को जोड़ने का समर्थन करता है। अधिक जानकारी के लिए, [Federated Storage Engine](https://dev.mysql.com/doc/refman/8.0/en/federated-storage-engine.html) दस्तावेज़ देखें।

### PostgreSQL

PostgreSQL में, विभिन्न प्रकार के `fdw` एक्सटेंशन का उपयोग विभिन्न प्रकार के रिमोट डेटा को सपोर्ट करने के लिए किया जा सकता है। वर्तमान में समर्थित एक्सटेंशन में शामिल हैं:

- [postgres_fdw](https://www.postgresql.org/docs/current/postgres-fdw.html): PostgreSQL में एक रिमोट PostgreSQL डेटाबेस से कनेक्ट करें।
- [mysql_fdw(विकास में)](https://github.com/EnterpriseDB/mysql_fdw): PostgreSQL में एक रिमोट MySQL डेटाबेस से कनेक्ट करें।
- अन्य प्रकार के fdw एक्सटेंशन के लिए, [PostgreSQL Foreign Data Wrappers](https://wiki.postgresql.org/wiki/Foreign_data_wrappers) देखें। NocoBase में एकीकृत करने के लिए आपको कोड में संबंधित अनुकूलन इंटरफ़ेस को लागू करना होगा।
