# संग्रह फ़ील्ड

## फ़ील्ड के इंटरफ़ेस प्रकार

NocoBase इंटरफ़ेस के दृष्टिकोण से फ़ील्ड को निम्नलिखित श्रेणियों में वर्गीकृत करता है:

![20240512110352](https://static-docs.nocobase.com/20240512110352.png)

## फ़ील्ड डेटा प्रकार

प्रत्येक फ़ील्ड इंटरफ़ेस का एक डिफ़ॉल्ट डेटा प्रकार होता है। उदाहरण के लिए, 'संख्या' (Number) इंटरफ़ेस वाले फ़ील्ड के लिए, डिफ़ॉल्ट डेटा प्रकार 'double' होता है, लेकिन यह 'float', 'decimal' आदि भी हो सकता है। वर्तमान में समर्थित डेटा प्रकार इस प्रकार हैं:

![20240512103733](https://static-docs.nocobase.com/20240512103733.png)

## फ़ील्ड प्रकार मैपिंग

मुख्य डेटाबेस में नए फ़ील्ड जोड़ने की प्रक्रिया इस प्रकार है:

1. इंटरफ़ेस प्रकार चुनें
2. वर्तमान इंटरफ़ेस के लिए वैकल्पिक डेटा प्रकार कॉन्फ़िगर करें

![20240512172416](https://static-docs.nocobase.com/20240512172416.png)

बाहरी डेटा स्रोत से फ़ील्ड मैपिंग की प्रक्रिया इस प्रकार है:

1. बाहरी डेटाबेस के फ़ील्ड प्रकार के आधार पर, संबंधित डेटा प्रकार (फ़ील्ड प्रकार) और UI प्रकार (फ़ील्ड इंटरफ़ेस) को स्वचालित रूप से मैप करें।
2. आवश्यकतानुसार अधिक उपयुक्त डेटा प्रकार और इंटरफ़ेस प्रकार में संशोधित करें।

![20240512172759](https://static-docs.nocobase.com/20240512172759.png)
