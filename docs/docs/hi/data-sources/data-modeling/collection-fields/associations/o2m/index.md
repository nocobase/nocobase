# एक-से-अनेक

## पैरामीटर विवरण

### स्रोत संग्रह

यह वह स्रोत संग्रह है, जहाँ वर्तमान फ़ील्ड मौजूद है।

### लक्ष्य संग्रह

यह वह लक्ष्य संग्रह है, जिसके साथ संबंध स्थापित करना है।

### स्रोत कुंजी

स्रोत संग्रह में वह फ़ील्ड जिसे फ़ॉरेन कुंजी द्वारा संदर्भित किया जाता है। यह अद्वितीय होना चाहिए।

### फ़ॉरेन कुंजी

लक्ष्य संग्रह में वह फ़ील्ड जिसका उपयोग दो संग्रहों के बीच संबंध स्थापित करने के लिए किया जाता है।

### लक्ष्य कुंजी

लक्ष्य संग्रह में वह फ़ील्ड जिसका उपयोग संबंध ब्लॉक में प्रत्येक पंक्ति रिकॉर्ड को देखने के लिए किया जाता है, आमतौर पर यह एक अद्वितीय फ़ील्ड होता है।

### ON DELETE

ON DELETE उन नियमों को संदर्भित करता है जो पैरेंट संग्रह में रिकॉर्ड हटाए जाने पर संबंधित चाइल्ड संग्रहों में फ़ॉरेन कुंजी संदर्भों पर लागू होते हैं। यह फ़ॉरेन कुंजी बाधा (constraint) को परिभाषित करते समय उपयोग किया जाने वाला एक विकल्प है। सामान्य ON DELETE विकल्प इस प्रकार हैं:

- **CASCADE**: जब पैरेंट संग्रह में कोई रिकॉर्ड हटाया जाता है, तो चाइल्ड संग्रह में उससे संबंधित सभी रिकॉर्ड स्वचालित रूप से हटा दिए जाते हैं।
- **SET NULL**: जब पैरेंट संग्रह में कोई रिकॉर्ड हटाया जाता है, तो संबंधित चाइल्ड संग्रह रिकॉर्ड में फ़ॉरेन कुंजी मान NULL पर सेट कर दिए जाते हैं।
- **RESTRICT**: यह डिफ़ॉल्ट विकल्प है, यदि चाइल्ड संग्रह में संबंधित रिकॉर्ड मौजूद हैं, तो यह पैरेंट संग्रह रिकॉर्ड को हटाने से रोकता है।
- **NO ACTION**: RESTRICT के समान, यदि चाइल्ड संग्रह में संबंधित रिकॉर्ड मौजूद हैं, तो यह पैरेंट संग्रह रिकॉर्ड को हटाने से रोकता है।
