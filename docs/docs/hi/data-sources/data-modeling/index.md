# अवलोकन

डेटा मॉडलिंग डेटाबेस डिज़ाइन करते समय एक महत्वपूर्ण कदम है, जिसमें वास्तविक दुनिया के विभिन्न प्रकार के डेटा और उनके आपसी संबंधों का गहराई से विश्लेषण और अमूर्त रूप में प्रस्तुत करना शामिल है। इस प्रक्रिया में, हम डेटा के बीच आंतरिक संबंधों को उजागर करने और उन्हें डेटा मॉडल के रूप में औपचारिक रूप देने का प्रयास करते हैं, जो सूचना प्रणाली के डेटाबेस संरचना की नींव रखता है। NocoBase एक डेटा मॉडल-संचालित प्लेटफ़ॉर्म है, जिसमें निम्नलिखित विशेषताएँ हैं:

## विभिन्न स्रोतों से डेटा तक पहुँच का समर्थन

NocoBase विभिन्न स्रोतों से डेटा स्रोत का समर्थन करता है, जिनमें सामान्य डेटाबेस, API/SDK प्लेटफ़ॉर्म और फ़ाइलें शामिल हैं।

![20240512085558](https://static-docs.nocobase.com/20240512085558.png)

NocoBase विभिन्न डेटा स्रोतों और उनके **संग्रहों** को प्रबंधित करने के लिए एक [डेटा स्रोत प्रबंधक](/data-sources/data-source-manager) **प्लगइन** प्रदान करता है। डेटा स्रोत प्रबंधक **प्लगइन** केवल सभी डेटा स्रोतों के लिए एक प्रबंधन इंटरफ़ेस प्रदान करता है और सीधे डेटा स्रोतों तक पहुँचने की क्षमता प्रदान नहीं करता है। इसे विभिन्न डेटा स्रोत **प्लगइनों** के साथ मिलकर उपयोग करने की आवश्यकता है। वर्तमान में समर्थित डेटा स्रोतों में शामिल हैं:

- [मुख्य डेटाबेस](/data-sources/data-source-main): NocoBase का मुख्य डेटाबेस, जो MySQL, PostgreSQL, और MariaDB जैसे रिलेशनल डेटाबेस का समर्थन करता है।
- [KingbaseES](/data-sources/data-source-kingbase): KingbaseES डेटाबेस को एक **डेटा स्रोत** के रूप में उपयोग करें, जिसे मुख्य डेटाबेस और बाहरी डेटाबेस दोनों के रूप में इस्तेमाल किया जा सकता है।
- [बाहरी MySQL](/data-sources/data-source-external-mysql): बाहरी MySQL डेटाबेस को एक **डेटा स्रोत** के रूप में उपयोग करें।
- [बाहरी MariaDB](/data-sources/data-source-external-mariadb): बाहरी MariaDB डेटाबेस को एक **डेटा स्रोत** के रूप में उपयोग करें।
- [बाहरी PostgreSQL](/data-sources/data-source-external-postgres): बाहरी PostgreSQL डेटाबेस को एक **डेटा स्रोत** के रूप में उपयोग करें।
- [बाहरी MSSQL](/data-sources/data-source-external-mssql): बाहरी MSSQL (SQL Server) डेटाबेस को एक **डेटा स्रोत** के रूप में उपयोग करें।
- [बाहरी Oracle](/data-sources/data-source-external-oracle): बाहरी Oracle डेटाबेस को एक **डेटा स्रोत** के रूप में उपयोग करें।

![20240512083651](https://static-docs.nocobase.com/20240512083651.png)

## विभिन्न प्रकार के डेटा मॉडलिंग उपकरण प्रदान करता है

**सरल संग्रह प्रबंधन इंटरफ़ेस**: इसका उपयोग विभिन्न मॉडल (**संग्रह**) बनाने या मौजूदा **संग्रहों** से जुड़ने के लिए किया जाता है।

![20240512090751](https://static-docs.nocobase.com/20240512090751.png)

**ER-शैली का विज़ुअल इंटरफ़ेस**: इसका उपयोग उपयोगकर्ता और व्यावसायिक आवश्यकताओं से संस्थाओं (entities) और उनके संबंधों को निकालने के लिए किया जाता है। यह डेटा मॉडल का वर्णन करने का एक सहज और समझने में आसान तरीका प्रदान करता है। ER आरेखों के माध्यम से, आप सिस्टम में मुख्य डेटा संस्थाओं और उनके संबंधों को अधिक स्पष्ट रूप से समझ सकते हैं।

![20240512091042](https://static-docs.nocobase.com/20240410075906.png)

## विभिन्न प्रकार के **संग्रहों** का समर्थन करता है

| **संग्रह** | विवरण |
| - | - |
| [सामान्य **संग्रह**](/data-sources/data-source-main/general-collection) | इसमें सामान्य सिस्टम फ़ील्ड अंतर्निहित होते हैं; |
| [कैलेंडर **संग्रह**](/data-sources/calendar/calendar-collection) | कैलेंडर-संबंधी इवेंट **संग्रह** बनाने के लिए उपयोग किया जाता है; |
| टिप्पणी **संग्रह** | डेटा पर टिप्पणियाँ या प्रतिक्रिया संग्रहीत करने के लिए उपयोग किया जाता है; |
| [ट्री **संग्रह**](/data-sources/collection-tree) | ट्री-संरचित **संग्रह**, वर्तमान में केवल आसन्न सूची मॉडल (adjacency list model) का समर्थन करता है; |
| [फ़ाइल **संग्रह**](/data-sources/file-manager/file-collection) | फ़ाइल स्टोरेज प्रबंधन के लिए उपयोग किया जाता है; |
| [SQL **संग्रह**](/data-sources/collection-sql) | यह कोई वास्तविक डेटाबेस **संग्रह** नहीं है, बल्कि SQL क्वेरी को संरचित तरीके से प्रदर्शित करता है; |
| [डेटाबेस व्यू से कनेक्ट करें](/data-sources/collection-view) | मौजूदा डेटाबेस व्यू से जुड़ता है; |
| एक्सप्रेशन **संग्रह** | **वर्कफ़्लो** में डायनामिक एक्सप्रेशन परिदृश्यों के लिए उपयोग किया जाता है; |
| [विदेशी डेटा से कनेक्ट करें](/data-sources/collection-fdw) | FDW तकनीक के आधार पर डेटाबेस सिस्टम को बाहरी डेटा स्रोतों में डेटा को सीधे एक्सेस और क्वेरी करने की अनुमति देता है। |

![20240512102212](https://static-docs.nocobase.com/20240512102212.png)

अधिक जानकारी के लिए, "[**संग्रह** / अवलोकन](/data-sources/data-modeling/collection)" अनुभाग देखें।
