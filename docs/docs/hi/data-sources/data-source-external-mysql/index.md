# बाहरी डेटा स्रोत - MySQL

## परिचय

आप बाहरी MySQL डेटाबेस को डेटा स्रोत के रूप में उपयोग कर सकते हैं। वर्तमान में समर्थित संस्करण: MySQL >= 5.7

## स्थापना

यह प्लगइन एक व्यावसायिक प्लगइन है, जिसे प्लगइन मैनेजर के माध्यम से अपलोड और सक्रिय करना होगा।
