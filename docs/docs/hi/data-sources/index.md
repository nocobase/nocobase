# अवलोकन

डेटा मॉडलिंग डेटाबेस डिज़ाइन करते समय एक महत्वपूर्ण कदम है, इसमें वास्तविक दुनिया के विभिन्न प्रकार के डेटा और उनके आपसी संबंधों का गहन विश्लेषण और अमूर्त (abstraction) प्रक्रिया शामिल है। इस प्रक्रिया में, हम डेटा के बीच आंतरिक संबंधों को उजागर करने और उन्हें डेटा मॉडल के रूप में औपचारिक रूप देने का प्रयास करते हैं, जो सूचना प्रणाली की डेटाबेस संरचना के लिए आधार तैयार करता है। NocoBase एक डेटा मॉडल-संचालित प्लेटफ़ॉर्म है, जिसकी निम्नलिखित विशेषताएँ हैं:

## विभिन्न स्रोतों से डेटा तक पहुँचने में सहायता

NocoBase विभिन्न स्रोतों से **डेटा स्रोत** का समर्थन करता है, जिनमें सामान्य डेटाबेस, API (SDK) प्लेटफ़ॉर्म और फ़ाइलें शामिल हैं।

![20240512085558](https://static-docs.nocobase.com/20240512085558.png)

NocoBase विभिन्न **डेटा स्रोत** और उनके **संग्रह** को प्रबंधित करने के लिए एक [डेटा स्रोत प्रबंधक](/data-sources/data-source-manager) **प्लगइन** प्रदान करता है। डेटा स्रोत प्रबंधक **प्लगइन** केवल सभी **डेटा स्रोत** के लिए एक प्रबंधन इंटरफ़ेस प्रदान करता है और सीधे **डेटा स्रोत** तक पहुँचने की क्षमता प्रदान नहीं करता है। इसे विभिन्न **डेटा स्रोत** **प्लगइन** के साथ मिलकर उपयोग करने की आवश्यकता है। वर्तमान में समर्थित **डेटा स्रोत** में शामिल हैं:

- [मुख्य डेटाबेस](/data-sources/data-source-main): NocoBase का मुख्य डेटाबेस, जो MySQL, PostgreSQL, MariaDB जैसे रिलेशनल डेटाबेस का समर्थन करता है।
- [KingbaseES](/data-sources/data-source-kingbase): KingbaseES डेटाबेस को एक **डेटा स्रोत** के रूप में उपयोग करें, जिसे मुख्य डेटाबेस और बाहरी डेटाबेस दोनों के रूप में उपयोग किया जा सकता है।
- [बाहरी MySQL](/data-sources/data-source-external-mysql): एक बाहरी MySQL डेटाबेस को **डेटा स्रोत** के रूप में उपयोग करें।
- [बाहरी MariaDB](/data-sources/data-source-external-mariadb): एक बाहरी MariaDB डेटाबेस को **डेटा स्रोत** के रूप में उपयोग करें।
- [बाहरी PostgreSQL](/data-sources/data-source-external-postgres): एक बाहरी PostgreSQL डेटाबेस को **डेटा स्रोत** के रूप में उपयोग करें।
- [बाहरी MSSQL](/data-sources/data-source-external-mssql): एक बाहरी MSSQL (SQL Server) डेटाबेस को **डेटा स्रोत** के रूप में उपयोग करें।
- [बाहरी Oracle](/data-sources/data-source-external-oracle): एक बाहरी Oracle डेटाबेस को **डेटा स्रोत** के रूप में उपयोग करें।

![20240512083651](https://static-docs.nocobase.com/20240512083651.png)

## विभिन्न प्रकार के डेटा मॉडलिंग उपकरण प्रदान करता है

**सरल संग्रह प्रबंधन इंटरफ़ेस**: विभिन्न मॉडल (**संग्रह**) बनाने या मौजूदा मॉडल (**संग्रह**) से जुड़ने के लिए उपयोग किया जाता है।

![20240512090751](https://static-docs.nocobase.com/20240512090751.png)

**ER-शैली का विज़ुअल इंटरफ़ेस**: उपयोगकर्ता और व्यावसायिक आवश्यकताओं से संस्थाओं (entities) और उनके संबंधों को निकालने के लिए उपयोग किया जाता है। यह डेटा मॉडल का वर्णन करने का एक सहज और समझने में आसान तरीका प्रदान करता है। ER आरेखों के माध्यम से, आप सिस्टम में मुख्य डेटा संस्थाओं और उनके संबंधों को अधिक स्पष्ट रूप से समझ सकते हैं।

![20240512091042](https://static-docs.nocobase.com/20240410075906.png)

## विभिन्न प्रकार के संग्रह बनाने में सहायता

| संग्रह | विवरण |
| - | - |
| [सामान्य संग्रह](/data-sources/data-source-main/general-collection) | इसमें सामान्य सिस्टम फ़ील्ड अंतर्निहित हैं। |
| [कैलेंडर संग्रह](/data-sources/calendar/calendar-collection) | कैलेंडर-संबंधित इवेंट संग्रह बनाने के लिए उपयोग किया जाता है। |
| टिप्पणी संग्रह | डेटा पर टिप्पणियाँ या प्रतिक्रिया संग्रहीत करने के लिए उपयोग किया जाता है। |
| [ट्री संग्रह](/data-sources/collection-tree) | ट्री-संरचित संग्रह, वर्तमान में केवल आसन्न सूची (adjacency list) मॉडल का समर्थन करता है। |
| [फ़ाइल संग्रह](/data-sources/file-manager/file-collection) | फ़ाइल भंडारण प्रबंधन के लिए उपयोग किया जाता है। |
| [SQL संग्रह](/data-sources/collection-sql) | यह एक वास्तविक डेटाबेस संग्रह नहीं है, बल्कि SQL क्वेरी को संरचित तरीके से प्रदर्शित करता है। |
| [डेटाबेस व्यू से कनेक्ट करें](/data-sources/collection-view) | मौजूदा डेटाबेस व्यू से कनेक्ट करता है। |
| अभिव्यक्ति संग्रह | **वर्कफ़्लो** में गतिशील अभिव्यक्ति परिदृश्यों के लिए उपयोग किया जाता है। |
| [बाहरी डेटा से कनेक्ट करें](/data-sources/collection-fdw) | FDW तकनीक के आधार पर डेटाबेस सिस्टम को बाहरी **डेटा स्रोत** में डेटा को सीधे एक्सेस और क्वेरी करने की अनुमति देता है। |

![20240512102212](https://static-docs.nocobase.com/20240512102212.png)
