# बहु-एप्लिकेशन प्रबंधन

## अवलोकन

NocoBase में AppSupervisor के जरिए एक एंट्री से कई अलग-थलग एप्लिकेशन बनाए और प्रबंधित किए जा सकते हैं।


शुरुआत में एक ऐप पर्याप्त होता है, लेकिन स्केल बढ़ने पर मेंटेनेंस लागत और आइसोलेशन की जरूरत बढ़ती है।


इस मोड में कई ऐप एक ही NocoBase इंस्टेंस में चलते हैं; DB अलग हो सकता है, पर प्रोसेस/मेमोरी साझा होती है।

![](https://static-docs.nocobase.com/202512231055907.png)


बड़े स्केल के लिए Supervisor + कई Worker वातावरण वाला हाइब्रिड मॉडल उपयुक्त है।

![](https://static-docs.nocobase.com/202512231215186.png)
