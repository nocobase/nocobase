# प्लगइन प्रबंधन

## प्लगइन प्रबंधक

पेज के ऊपरी-दाएँ कोने में दिए गए प्लगइन प्रबंधक आइकन पर क्लिक करके आप प्लगइन प्रबंधक पेज पर जा सकते हैं। इस पेज पर, आप प्लगइन को देख सकते हैं, इंस्टॉल कर सकते हैं, सक्षम कर सकते हैं या अक्षम कर सकते हैं।

## प्लगइन सेटिंग केंद्र

प्लगइन आमतौर पर अपने खुद के सेटिंग पेज के साथ आते हैं। पेज के ऊपरी-दाएँ कोने में दिए गए सेटिंग आइकन पर क्लिक करके आप सभी प्लगइन के सेटिंग मेनू देख सकते हैं। प्लगइन को कॉन्फ़िगर करने के लिए संबंधित मेनू आइटम पर क्लिक करें।
