# भाषा सेटिंग्स

"सिस्टम सेटिंग्स" में, आप "सक्षम भाषाएँ" को कॉन्फ़िगर करके सिस्टम के वैश्विक भाषा परिवेश को समायोजित कर सकते हैं। सूची में पहली भाषा डिफ़ॉल्ट भाषा होगी।

![20240324192005](https://static-docs.nocobase.com/20240324192005.png)

जब कई भाषाएँ सक्षम की जाती हैं, तो उपयोगकर्ता व्यक्तिगत केंद्र (पेज के ऊपरी दाएं कोने में) में अपनी व्यक्तिगत भाषा वरीयता सेट कर सकते हैं।

![20240324192509](https://static-docs.nocobase.com/20240324192509.png)

यदि आपके द्वारा उपयोग किए जा रहे भाषा परिवेश में अनुवाद की कमी है, तो आप "[स्थानीयकरण प्रबंधन](/system-management/localization/)" प्लगइन का उपयोग अनुवाद करने और पूरक करने के लिए कर सकते हैं।

![20240324193055](https://static-docs.nocobase.com/20240324193055.png)
