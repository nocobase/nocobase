# अवलोकन

NocoBase का डेटा विज़ुअलाइज़ेशन प्लगइन विज़ुअल डेटा क्वेरी और चार्ट घटकों का एक समृद्ध सेट प्रदान करता है। सरल कॉन्फ़िगरेशन के साथ, आप तेज़ी से विज़ुअलाइज़ेशन पैनल बना सकते हैं, डेटा अंतर्दृष्टि प्रस्तुत कर सकते हैं, और बहु-आयामी विश्लेषण और प्रदर्शन का समर्थन कर सकते हैं।

![clipboard-image-1761749573](https://static-docs.nocobase.com/clipboard-image-1761749573.png)

## मूल अवधारणाएँ
- चार्ट ब्लॉक: पेज पर एक कॉन्फ़िगर करने योग्य चार्ट घटक जो डेटा क्वेरी, चार्ट विकल्पों और इंटरैक्शन इवेंट्स का समर्थन करता है।
- डेटा क्वेरी (बिल्डर / SQL): बिल्डर के साथ विज़ुअली कॉन्फ़िगर करें या डेटा प्राप्त करने के लिए SQL लिखें।
- मेज़र (Measures) और डाइमेंशन (Dimensions): मेज़र का उपयोग संख्यात्मक एकत्रीकरण के लिए किया जाता है; डाइमेंशन डेटा को समूहित करते हैं (उदाहरण के लिए, दिनांक, श्रेणी, क्षेत्र)।
- फ़ील्ड मैपिंग: क्वेरी परिणाम कॉलम को मुख्य चार्ट फ़ील्ड्स जैसे `xField`, `yField`, `seriesField`, या `Category / Value` पर मैप करें।
- चार्ट विकल्प (बेसिक / कस्टम): बेसिक सामान्य गुणों को विज़ुअली कॉन्फ़िगर करता है; कस्टम JS के माध्यम से एक पूर्ण ECharts `option` लौटाता है।
- क्वेरी चलाएँ: कॉन्फ़िगरेशन पैनल में क्वेरी चलाएँ और डेटा प्राप्त करें; लौटाए गए डेटा का निरीक्षण करने के लिए टेबल / JSON पर स्विच करें।
- पूर्वावलोकन और सहेजें: पूर्वावलोकन अस्थायी होता है; सहेजें पर क्लिक करने से कॉन्फ़िगरेशन डेटाबेस में लिखा जाता है और लागू होता है।
- संदर्भ वैरिएबल: क्वेरी और चार्ट कॉन्फ़िगरेशन में पेज, उपयोगकर्ता और फ़िल्टर संदर्भ (उदाहरण के लिए, `{{ ctx.user.id }}`) जैसी संदर्भ जानकारी का पुन: उपयोग करें।
- फ़िल्टर और लिंकेज: पेज-लेवल फ़िल्टर ब्लॉक एकीकृत शर्तों को एकत्र करते हैं, स्वचालित रूप से चार्ट क्वेरी में मर्ज हो जाते हैं, और लिंक्ड चार्ट को रीफ़्रेश करते हैं।
- इंटरैक्शन इवेंट्स: हाईलाइट, नेविगेशन और ड्रिल-डाउन जैसी क्रियाओं को सक्षम करने के लिए `chart.on` के माध्यम से इवेंट्स रजिस्टर करें।

## स्थापना
डेटा विज़ुअलाइज़ेशन NocoBase का एक बिल्ट-इन प्लगइन है; यह तुरंत उपयोग के लिए तैयार है, इसे अलग से स्थापित करने की आवश्यकता नहीं है।
