# संग्रह क्या है


संग्रह (Collection) NocoBase में डेटा भंडारण और प्रबंधन की एक मुख्य अवधारणा है। यह एक रिलेशनल डेटाबेस में तालिका के समान है, लेकिन अधिक लचीली और शक्तिशाली सुविधाएँ प्रदान करता है।

- **डेटा संरचना परिभाषा**: संग्रह आपको अपने डेटा के फ़ील्ड, प्रकार और सत्यापन नियम परिभाषित करने की अनुमति देता है।
- **डेटा भंडारण**: वास्तविक डेटा रिकॉर्ड संग्रह में संग्रहीत होते हैं।
- **API इंटरफ़ेस**: NocoBase प्रत्येक संग्रह के लिए स्वचालित रूप से RESTful API का एक सेट जेनरेट करता है, जिससे आपको डेटा संचालन में सुविधा होती है।
