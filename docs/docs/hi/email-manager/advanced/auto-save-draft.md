# ऑटो-सेव ड्राफ़्ट

"सेव ड्राफ़्ट" बटन के मेन्यू में, आप "ऑटो-सेव ड्राफ़्ट" सुविधा को चालू कर सकते हैं।

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_29_PM.png)

ईमेल संपादन प्रक्रिया के दौरान, सिस्टम ड्राफ़्ट को वास्तविक समय में सहेजता है, जिससे यह सुनिश्चित होता है कि आपकी कोई भी सामग्री गुम न हो।

![](https://static-docs.nocobase.com/email-manager/Loading--10-31-2025_08_31_PM.png)

जब आप ईमेल एडिटर को फिर से खोलते हैं (उसी भेजने वाले और प्राप्तकर्ता के लिए), तो सिस्टम आपसे पूछेगा कि क्या आप सहेजे गए ड्राफ़्ट का उपयोग करना चाहते हैं।

![](https://static-docs.nocobase.com/email-manager/Email-10-31-2025_08_35_PM.png)
