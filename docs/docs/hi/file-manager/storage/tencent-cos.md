# टेनसेंट क्लाउड COS

टेनसेंट क्लाउड COS पर आधारित एक स्टोरेज इंजन। इसका उपयोग करने से पहले आपको संबंधित अकाउंट और अनुमतियाँ तैयार करनी होंगी।

## कॉन्फ़िगरेशन पैरामीटर

![टेनसेंट COS स्टोरेज इंजन कॉन्फ़िगरेशन उदाहरण](https://static-docs.nocobase.com/20240712222125.png)

:::info{title=ध्यान दें}
यह अनुभाग केवल टेनसेंट क्लाउड COS स्टोरेज इंजन के विशिष्ट पैरामीटरों का परिचय देता है। सामान्य पैरामीटरों के लिए, कृपया [सामान्य इंजन पैरामीटर](./index.md#general-engine-parameters) देखें।
:::

### क्षेत्र

COS स्टोरेज के लिए क्षेत्र दर्ज करें, उदाहरण के लिए: `ap-chengdu`।

:::info{title=ध्यान दें}
आप [टेनसेंट क्लाउड COS कंसोल](https://console.cloud.tencent.com/cos) में अपने बकेट की क्षेत्र जानकारी देख सकते हैं। आपको केवल क्षेत्र उपसर्ग का उपयोग करना होगा (पूरे डोमेन नाम की आवश्यकता नहीं है)।
:::

### SecretId

अपने टेनसेंट क्लाउड एक्सेस कुंजी का ID दर्ज करें।

### SecretKey

अपने टेनसेंट क्लाउड एक्सेस कुंजी का Secret दर्ज करें।

### बकेट

COS बकेट का नाम दर्ज करें, उदाहरण के लिए: `qing-cdn-1234189398`।
