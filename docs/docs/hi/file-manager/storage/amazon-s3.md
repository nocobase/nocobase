# स्टोरेज इंजन: Amazon S3

यह Amazon S3 पर आधारित एक स्टोरेज इंजन है। इसका उपयोग करने से पहले आपको संबंधित अकाउंट और अनुमतियाँ तैयार करनी होंगी।

## कॉन्फ़िगरेशन पैरामीटर

![Amazon S3 स्टोरेज इंजन कॉन्फ़िगरेशन का उदाहरण](https://static-docs.nocobase.com/20251031092524.png)

:::info{title=नोट}
यह सेक्शन केवल Amazon S3 स्टोरेज इंजन के विशिष्ट पैरामीटरों का परिचय देता है। सामान्य पैरामीटरों के लिए, कृपया [इंजन के सामान्य पैरामीटर](./index#引擎通用参数) देखें।
:::

### रीजन

S3 स्टोरेज का रीजन दर्ज करें, उदाहरण के लिए: `us-west-1`।

:::info{title=नोट}
आप [Amazon S3 कंसोल](https://console.aws.amazon.com/s3/) में अपने बकेट की रीजन जानकारी देख सकते हैं, और आपको केवल रीजन प्रीफ़िक्स का उपयोग करना होगा (पूरे डोमेन नाम की आवश्यकता नहीं है)।
:::

### AccessKey ID

Amazon S3 AccessKey ID दर्ज करें।

### AccessKey Secret

Amazon S3 AccessKey Secret दर्ज करें।

### बकेट

S3 बकेट का नाम दर्ज करें।
