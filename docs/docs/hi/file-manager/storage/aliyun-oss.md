# स्टोरेज इंजन: अलीबाबा क्लाउड OSS

अलीबाबा क्लाउड OSS पर आधारित एक स्टोरेज इंजन है। इसका उपयोग करने से पहले, आपको संबंधित अकाउंट और अनुमतियाँ तैयार करनी होंगी।

## कॉन्फ़िगरेशन पैरामीटर

![अलीबाबा क्लाउड OSS स्टोरेज इंजन कॉन्फ़िगरेशन का उदाहरण](https://static-docs.nocobase.com/20240712220011.png)

:::info{title=जानकारी}
यह अनुभाग केवल अलीबाबा क्लाउड OSS स्टोरेज इंजन के विशिष्ट पैरामीटर बताता है। सामान्य पैरामीटर के लिए, कृपया [इंजन सामान्य पैरामीटर](./index#引擎通用参数) देखें।
:::

### क्षेत्र

OSS स्टोरेज का क्षेत्र दर्ज करें, उदाहरण के लिए: `oss-cn-hangzhou`।

:::info{title=जानकारी}
आप [अलीबाबा क्लाउड OSS कंसोल](https://oss.console.aliyun.com/) में अपने बकेट की क्षेत्र जानकारी देख सकते हैं, और आपको केवल क्षेत्र उपसर्ग का उपयोग करना होगा (पूरे डोमेन नाम की आवश्यकता नहीं है)।
:::

### AccessKey ID

अपने अलीबाबा क्लाउड एक्सेस कुंजी का ID दर्ज करें।

### AccessKey Secret

अपने अलीबाबा क्लाउड एक्सेस कुंजी का Secret दर्ज करें।

### बकेट

OSS बकेट का नाम दर्ज करें।

### टाइमआउट

अलीबाबा क्लाउड OSS पर अपलोड करने के लिए टाइमआउट दर्ज करें, मिलीसेकंड में। डिफ़ॉल्ट `60000` मिलीसेकंड (यानी 60 सेकंड) है।
