# फ़ाइल पूर्वावलोकन

फ़ाइल फ़ील्ड (अटैचमेंट फ़ील्ड सहित) वाले इंटरफ़ेस में, आप फ़ाइल थंबनेल या आइकन पर क्लिक करके फ़ाइल का पूर्वावलोकन कर सकते हैं। अंतर्निहित पूर्वावलोकन सुविधा कई प्रकार की फ़ाइलें सपोर्ट करती है, जिनमें इमेज, PDF और ज़्यादातर ब्राउज़र द्वारा मूल रूप से समर्थित फ़ाइल प्रकार शामिल हैं।

![20251129232307](https://static-docs.nocobase.com/20251129232307.png)

जिन फ़ाइल प्रकारों के लिए मूल पूर्वावलोकन समर्थित नहीं है, आप संबंधित फ़ाइल पूर्वावलोकन प्लगइन को इंस्टॉल या एक्सटेंड करके पूर्वावलोकन सुविधा को सक्षम कर सकते हैं। उदाहरण के लिए, Office फ़ाइल पूर्वावलोकन प्लगइन इंस्टॉल करने के बाद, आप Word, Excel और PowerPoint फ़ाइलों का पूर्वावलोकन कर सकते हैं।
