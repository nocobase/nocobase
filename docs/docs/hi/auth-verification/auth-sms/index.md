# एसएमएस प्रमाणीकरण

## परिचय

एसएमएस प्रमाणीकरण प्लगइन उपयोगकर्ताओं को एसएमएस के माध्यम से रजिस्टर करने और NocoBase में लॉग इन करने में सहायता करता है।

> इसे [`@nocobase/plugin-verification` प्लगइन](/auth-verification/verification/) द्वारा प्रदान की गई एसएमएस सत्यापन कोड सुविधा के साथ उपयोग करने की आवश्यकता है।

## एसएमएस प्रमाणीकरण जोड़ें

उपयोगकर्ता प्रमाणीकरण प्लगइन प्रबंधन पृष्ठ पर जाएँ।

![](https://static-docs.nocobase.com/202502282112517.png)

जोड़ें - एसएमएस (SMS)

![](https://static-docs.nocobase.com/202502282113553.png)

## नए संस्करण का कॉन्फ़िगरेशन

:::info{title=नोट}
नया कॉन्फ़िगरेशन `1.6.0-alpha.30` में पेश किया गया था और `1.7.0` से स्थिर समर्थन के लिए योजना बनाई गई है।
:::

![](https://static-docs.nocobase.com/202502282114821.png)

**वेरिफ़िकेटर (Verificator):** एसएमएस सत्यापन कोड भेजने के लिए एक एसएमएस वेरिफ़िकेटर को बाइंड करें। यदि कोई वेरिफ़िकेटर उपलब्ध नहीं है, तो आपको पहले सत्यापन प्रबंधन पृष्ठ पर जाकर एक एसएमएस वेरिफ़िकेटर बनाना होगा।
यह भी देखें:

- [सत्यापन](../verification/index.md)
- [सत्यापन: एसएमएस](../verification/sms/index.md)

**उपयोगकर्ता के मौजूद न होने पर स्वचालित रूप से साइन अप करें (Sign up automatically when the user does not exist):** जब यह विकल्प चुना जाता है, यदि उपयोगकर्ता का फ़ोन नंबर मौजूद नहीं है, तो फ़ोन नंबर को उपनाम के रूप में उपयोग करके एक नया उपयोगकर्ता पंजीकृत किया जाएगा।

## पुराने संस्करण का कॉन्फ़िगरेशन

![](https://static-docs.nocobase.com/a4d35ec63ba22ae2ea9e3e8e1cbb783d.png)

एसएमएस लॉगिन प्रमाणीकरण सुविधा कॉन्फ़िगर किए गए और डिफ़ॉल्ट एसएमएस सत्यापन कोड प्रोवाइडर का उपयोग करके टेक्स्ट संदेश भेजेगी।

**उपयोगकर्ता के मौजूद न होने पर स्वचालित रूप से साइन अप करें (Sign up automatically when the user does not exist):** जब यह विकल्प चुना जाता है, यदि उपयोगकर्ता का फ़ोन नंबर मौजूद नहीं है, तो फ़ोन नंबर को उपनाम के रूप में उपयोग करके एक नया उपयोगकर्ता पंजीकृत किया जाएगा।
