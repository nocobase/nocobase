# बाहरी डेटा तालिकाओं को जोड़ना (FDW)

## परिचय

यह सुविधा डेटाबेस के Foreign Data Wrapper (FDW) पर आधारित रिमोट डेटा तालिकाओं (remote data tables) से जुड़ने की क्षमता प्रदान करती है। वर्तमान में, यह MySQL और PostgreSQL डेटाबेस का समर्थन करती है।

:::info{title="डेटा स्रोत को जोड़ना बनाम बाहरी डेटा तालिकाओं को जोड़ना"}
- **डेटा स्रोत को जोड़ना** का अर्थ है किसी विशिष्ट डेटाबेस या API सेवा के साथ कनेक्शन स्थापित करना, जिससे आप डेटाबेस की विशेषताओं या API द्वारा प्रदान की गई सेवाओं का पूरी तरह से उपयोग कर सकते हैं;
- **बाहरी डेटा तालिकाओं को जोड़ना** का अर्थ है बाहर से डेटा प्राप्त करना और उसे स्थानीय उपयोग के लिए मैप करना। डेटाबेस में इसे FDW (Foreign Data Wrapper) कहा जाता है, जो एक डेटाबेस तकनीक है जो रिमोट तालिकाओं को स्थानीय तालिकाओं के रूप में उपयोग करने पर केंद्रित है और एक समय में केवल एक तालिका को जोड़ सकती है। चूंकि यह रिमोट एक्सेस है, इसलिए इसका उपयोग करते समय विभिन्न बाधाएं और सीमाएं होंगी।

इन दोनों का उपयोग संयोजन में भी किया जा सकता है। पहले का उपयोग डेटा स्रोत के साथ कनेक्शन स्थापित करने के लिए किया जाता है, और दूसरे का उपयोग क्रॉस डेटा-सोर्स एक्सेस के लिए किया जाता है। उदाहरण के लिए, यदि कोई PostgreSQL डेटा स्रोत जुड़ा हुआ है, और उस डेटा स्रोत में कोई तालिका FDW के आधार पर बनाई गई बाहरी डेटा तालिका है।
:::

### MySQL

MySQL `federated` इंजन का उपयोग करता है, जिसे सक्रिय करने की आवश्यकता होती है। यह रिमोट MySQL और MariaDB जैसे प्रोटोकॉल-संगत डेटाबेस से जुड़ने का समर्थन करता है। विस्तृत विवरण के लिए [Federated Storage Engine](https://dev.mysql.com/doc/refman/8.0/en/federated-storage-engine.html) दस्तावेज़ देखें।

### PostgreSQL

PostgreSQL में, विभिन्न प्रकार के रिमोट डेटा का समर्थन करने के लिए अलग-अलग प्रकार के `fdw` एक्सटेंशन का उपयोग किया जा सकता है। वर्तमान में समर्थित एक्सटेंशन में शामिल हैं:

- [postgres_fdw](https://www.postgresql.org/docs/current/postgres-fdw.html): PostgreSQL में रिमोट PostgreSQL डेटाबेस से जुड़ें।
- [mysql_fdw](https://github.com/EnterpriseDB/mysql_fdw): PostgreSQL में रिमोट MySQL डेटाबेस से जुड़ें।
- अन्य प्रकार के fdw एक्सटेंशन के लिए, [PostgreSQL Foreign Data Wrappers](https://wiki.postgresql.org/wiki/Foreign_data_wrappers) देखें। आपको कोड में संबंधित अनुकूलन इंटरफ़ेस (adaptation interface) लागू करने की आवश्यकता होगी।

## पूर्व-आवश्यकताएँ

- यदि NocoBase का मुख्य डेटाबेस MySQL है, तो आपको `federated` को सक्रिय करना होगा। [MySQL में federated इंजन को कैसे सक्षम करें](./enable-federated) देखें।
