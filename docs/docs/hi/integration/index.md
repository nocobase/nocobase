# एकीकरण

## अवलोकन

NocoBase व्यापक एकीकरण क्षमताएँ प्रदान करता है, जिससे आप बाहरी सिस्टम, तीसरे पक्ष की सेवाओं और विभिन्न **डेटा स्रोत** से सहजता से जुड़ सकते हैं। एकीकरण के लचीले तरीकों के माध्यम से, आप अपनी विविध व्यावसायिक आवश्यकताओं को पूरा करने के लिए NocoBase की कार्यक्षमता का विस्तार कर सकते हैं।

## एकीकरण के तरीके

### API एकीकरण

NocoBase बाहरी एप्लिकेशन और सेवाओं के साथ एकीकरण के लिए शक्तिशाली API क्षमताएँ प्रदान करता है:

![20250301003106](https://static-docs.nocobase.com/20250301003106.png)

- **[API कुंजियाँ](/integration/api-keys/)**: NocoBase संसाधनों तक प्रोग्रामेटिक रूप से पहुँचने के लिए API कुंजियों का उपयोग करके सुरक्षित प्रमाणीकरण करें।
- **[API दस्तावेज़](/integration/api-doc/)**: एंडपॉइंट्स को एक्सप्लोर करने और उनका परीक्षण करने के लिए अंतर्निहित API दस्तावेज़।

### सिंगल साइन-ऑन (SSO)

एकीकृत प्रमाणीकरण के लिए एंटरप्राइज़ पहचान प्रणालियों के साथ एकीकृत करें:

![auth_sso-2025-11-24-08-26-46](https://static-docs.nocobase.com/auth_sso-2025-11-24-08-26-46.png)

- **[SSO एकीकरण](/integration/sso/)**: SAML, OIDC, CAS, LDAP और तीसरे पक्ष के प्लेटफ़ॉर्म प्रमाणीकरण के लिए समर्थन।
- केंद्रीकृत उपयोगकर्ता प्रबंधन और पहुँच नियंत्रण।
- सिस्टमों में सहज प्रमाणीकरण अनुभव।

### वर्कफ़्लो एकीकरण

NocoBase **वर्कफ़्लो** को बाहरी सिस्टम से कनेक्ट करें:

![auth_sso-2025-11-24-12-02-01](https://static-docs.nocobase.com/auth_sso-2025-11-24-12-02-01.png)

- **[वर्कफ़्लो वेबहुक](/integration/workflow-webhook/)**: **वर्कफ़्लो** को ट्रिगर करने के लिए बाहरी सिस्टम से इवेंट प्राप्त करें।
- **[वर्कफ़्लो HTTP अनुरोध](/integration/workflow-http-request/)**: **वर्कफ़्लो** से बाहरी API को HTTP अनुरोध भेजें।
- प्लेटफ़ॉर्मों पर व्यावसायिक प्रक्रियाओं को स्वचालित करें।

### बाहरी डेटा स्रोत

बाहरी डेटाबेस और डेटा सिस्टम से कनेक्ट करें:

![20240512085558](https://static-docs.nocobase.com/20240512085558.png)

- **[बाहरी डेटाबेस](/data-sources/)**: MySQL, PostgreSQL, MariaDB, MSSQL, Oracle और KingbaseES डेटाबेस से सीधे कनेक्ट करें।
- बाहरी डेटाबेस तालिका संरचनाओं को पहचानें और NocoBase के भीतर सीधे बाहरी डेटा पर CRUD (बनाना, पढ़ना, अपडेट करना, हटाना) ऑपरेशन करें।
- एकीकृत डेटा प्रबंधन इंटरफ़ेस।

### एम्बेडेड सामग्री

NocoBase के भीतर बाहरी सामग्री एम्बेड करें:

![20240408220322](https://static-docs.nocobase.com/20240408220322.png)

- **[Iframe ब्लॉक](/integration/block-iframe/)**: बाहरी वेब पेज और एप्लिकेशन एम्बेड करें।
- **JS ब्लॉक**: उन्नत एकीकरण के लिए कस्टम JavaScript कोड निष्पादित करें।

## सामान्य एकीकरण परिदृश्य

### एंटरप्राइज़ सिस्टम एकीकरण

- NocoBase को ERP, CRM या अन्य एंटरप्राइज़ सिस्टम से कनेक्ट करें।
- डेटा को द्विदिश रूप से सिंक्रनाइज़ करें।
- क्रॉस-सिस्टम **वर्कफ़्लो** को स्वचालित करें।

### तीसरे पक्ष की सेवा का एकीकरण

- पेमेंट गेटवे से भुगतान की स्थिति क्वेरी करें, मैसेजिंग सेवाओं या क्लाउड प्लेटफ़ॉर्म को एकीकृत करें।
- कार्यक्षमता का विस्तार करने के लिए बाहरी API का लाभ उठाएँ।
- वेबहुक और HTTP अनुरोधों का उपयोग करके कस्टम एकीकरण बनाएँ।

### डेटा एकीकरण

- कई **डेटा स्रोत** से कनेक्ट करें।
- विभिन्न सिस्टमों से डेटा एकत्रित करें।
- एकीकृत डैशबोर्ड और रिपोर्ट बनाएँ।

## सुरक्षा संबंधी विचार

NocoBase को बाहरी सिस्टम के साथ एकीकृत करते समय, कृपया निम्नलिखित सुरक्षा सर्वोत्तम प्रथाओं पर विचार करें:

1.  **HTTPS का उपयोग करें**: डेटा ट्रांसमिशन के लिए हमेशा एन्क्रिप्टेड कनेक्शन का उपयोग करें।
2.  **API कुंजियों को सुरक्षित रखें**: API कुंजियों को सुरक्षित रूप से संग्रहीत करें और उन्हें नियमित रूप से बदलें।
3.  **न्यूनतम विशेषाधिकार का सिद्धांत**: एकीकरण के लिए केवल आवश्यक अनुमतियाँ प्रदान करें।
4.  **ऑडिट लॉगिंग**: एकीकरण गतिविधियों की निगरानी और उन्हें लॉग करें।
5.  **डेटा सत्यापन**: बाहरी स्रोतों से प्राप्त सभी डेटा को सत्यापित करें।
