# अवलोकन

NocoBase सर्वर-साइड प्लगइन विकास डेवलपर्स को NocoBase की मुख्य कार्यक्षमताओं को अनुकूलित (customize) और विस्तारित (extend) करने के लिए विभिन्न सुविधाएँ और क्षमताएँ प्रदान करता है। NocoBase सर्वर-साइड प्लगइन विकास की प्रमुख क्षमताएँ और उनसे संबंधित अध्याय नीचे दिए गए हैं:

| कार्य मॉड्यूल                  | विवरण                                           | संबंधित अध्याय                                      |
|---------------------------|------------------------------------------------|-----------------------------------------------|
| **प्लगइन क्लास**               | सर्वर-साइड प्लगइन बनाएँ और प्रबंधित करें, मुख्य कार्यक्षमता का विस्तार करें             | [plugin.md](plugin.md)                       |
| **डेटाबेस संचालन**             | डेटाबेस संचालन के लिए इंटरफ़ेस प्रदान करता है, CRUD और ट्रांज़ैक्शन प्रबंधन का समर्थन करता है | [database.md](database.md)                    |
| **कस्टम संग्रह**           | व्यावसायिक आवश्यकताओं के आधार पर संग्रह (collection) संरचनाओं को अनुकूलित करें, लचीले डेटा मॉडल प्रबंधन के लिए | [collections.md](collections.md)              |
| **प्लगइन अपग्रेड डेटा संगतता** | सुनिश्चित करता है कि प्लगइन अपग्रेड मौजूदा डेटा को प्रभावित न करें, डेटा माइग्रेशन और संगतता हैंडलिंग करके | [migration.md](migration.md)                  |
| **बाहरी डेटा स्रोत प्रबंधन**         | डेटा इंटरैक्शन को सक्षम करने के लिए बाहरी डेटा स्रोतों को एकीकृत और प्रबंधित करें             | [data-source-manager.md](data-source-manager.md) |
| **कस्टम API**             | कस्टम इंटरफ़ेस लिखकर API संसाधन प्रबंधन का विस्तार करें               | [resource-manager.md](resource-manager.md)    |
| **API अनुमति प्रबंधन** | बारीक पहुँच नियंत्रण (fine-grained access control) के लिए API अनुमतियों को अनुकूलित करें             | [acl.md](acl.md)                              |
| **अनुरोध/प्रतिक्रिया अवरोधन और फ़िल्टरिंग** | लॉगिंग, प्रमाणीकरण (authentication) आदि जैसे कार्यों को संभालने के लिए अनुरोध और प्रतिक्रिया इंटरसेप्टर या मिडलवेयर जोड़ें | [context.md](context.md) और [middleware.md](middleware.md) |
| **इवेंट सुनना**               | सिस्टम इवेंट (जैसे एप्लिकेशन या डेटाबेस से) सुनें और संबंधित हैंडलर को ट्रिगर करें       | [event.md](event.md)                          |
| **कैश प्रबंधन**               | एप्लिकेशन प्रदर्शन और प्रतिक्रिया गति में सुधार के लिए कैश का प्रबंधन करें               | [cache.md](cache.md)                          |
| **निर्धारित कार्य**               | निर्धारित कार्य बनाएँ और प्रबंधित करें, जैसे आवधिक सफाई, डेटा सिंक्रनाइज़ेशन आदि     | [cron-job-manager.md](cron-job-manager.md)    |
| **बहु-भाषा समर्थन**             | अंतर्राष्ट्रीयकरण (internationalization) और स्थानीयकरण (localization) को लागू करने के लिए बहु-भाषा समर्थन को एकीकृत करें             | [i18n.md](i18n.md)                            |
| **लॉग आउटपुट**               | डिबगिंग और निगरानी क्षमताओं को बढ़ाने के लिए लॉग प्रारूपों और आउटपुट विधियों को अनुकूलित करें   | [logger.md](logger.md)                        |
| **कस्टम कमांड**             | कस्टम कमांड जोड़कर NocoBase CLI का विस्तार करें               | [command.md](command.md)                      |
| **टेस्ट केस लिखना**           | प्लगइन की स्थिरता और कार्यात्मक सटीकता सुनिश्चित करने के लिए टेस्ट केस लिखें और चलाएँ | [test.md](test.md)                            |
