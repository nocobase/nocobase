# ACL अनुमति नियंत्रण

:::tip

यह अध्याय बताएगा कि फ़्रंटएंड प्लगइन में अनुमति प्रबंधन के लिए ACL (एक्सेस कंट्रोल लिस्ट) का उपयोग कैसे करें। इसमें शामिल होगा:

- अनुमति का पता लगाना;
- अनुमतियों के आधार पर सामग्री को गतिशील रूप से कैसे प्रस्तुत करें;

हम भविष्य के संस्करणों में विस्तृत उदाहरण और सर्वोत्तम अभ्यास जोड़ेंगे।

:::
