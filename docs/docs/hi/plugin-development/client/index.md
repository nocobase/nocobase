# अवलोकन
NocoBase क्लाइंट-साइड प्लगइन डेवलपमेंट कई तरह की कार्यक्षमताएँ और क्षमताएँ प्रदान करता है, जो डेवलपर्स को NocoBase की फ्रंटएंड सुविधाओं को कस्टमाइज़ और एक्सटेंड करने में मदद करती हैं। NocoBase क्लाइंट-साइड प्लगइन डेवलपमेंट की मुख्य क्षमताएँ और उनसे संबंधित अध्याय नीचे दिए गए हैं:

|-----------------------|----------------------------------------------------------------------------------|-----------------------------------------------|

| **स्टाइल और थीम**     | स्टाइल और थीम को कस्टमाइज़ करें, UI कस्टमाइज़ेशन और सौंदर्यीकरण लागू करें          | [styles-themes.md](styles-themes.md)         |
