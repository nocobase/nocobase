# परीक्षण

:::tip

यह अध्याय बताएगा कि प्लगइन के फ़्रंटएंड भाग के लिए परीक्षण कैसे लिखें और चलाएं। इसमें शामिल होगा:
- परीक्षण वातावरण का कॉन्फ़िगरेशन और निर्भरताएँ;
- Vitest और Testing Library का उपयोग करके यूनिट परीक्षण लिखना;
- NocoBase फ़्रंटएंड रनटाइम वातावरण का अनुकरण कैसे करें;
- CI में परीक्षणों को स्वचालित रूप से निष्पादित करना।

हम भविष्य के संस्करणों में विस्तृत उदाहरण और सर्वोत्तम अभ्यास जोड़ेंगे।

:::
