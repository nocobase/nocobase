# जीवनचक्र

यह अनुभाग सर्वर-साइड और क्लाइंट-साइड दोनों पर प्लगइन के जीवनचक्र हुक को व्यवस्थित करता है, जिससे डेवलपर्स को संसाधनों को सही ढंग से रजिस्टर और रिलीज़ करने में मदद मिलती है।

आप सामान्य अवधारणाओं को उजागर करने के लिए FlowModel जीवनचक्र से इसकी तुलना कर सकते हैं।

## सुझाए गए विषय

- जब प्लगइन इंस्टॉल, सक्षम, अक्षम या हटाए जाते हैं, तब ट्रिगर होने वाले कॉलबैक।
- क्लाइंट-साइड कंपोनेंट के माउंट होने, अपडेट होने और नष्ट होने का समय।
- जीवनचक्र के भीतर एसिंक्रोनस कार्यों और त्रुटियों को संभालने के लिए सुझाव।
