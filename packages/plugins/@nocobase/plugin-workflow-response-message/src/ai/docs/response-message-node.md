# 响应消息

## 节点类型

`response-message`
请使用以上 `type` 值创建节点，不要使用文档文件名作为 type。

## 节点描述
配置请求结束时返回给客户端的消息内容（仅拦截类同步流程可用）。

## 业务场景举例
在拦截类流程中返回“操作成功/失败”的提示。

## 配置项列表
| 字段 | 类型 | 默认值 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
| message | string | 无 | 否 | 响应消息内容，支持变量模板。 |

## 分支说明
不支持分支。

## 示例配置
```json
{
  "message": "操作成功：{{ $context.data.title }}"
}
```
