# 并行分支

## 节点类型

`parallel`
请使用以上 `type` 值创建节点，不要使用文档文件名作为 type。

## 节点描述
同时执行多个分支，并按模式汇总分支结果。

## 业务场景举例
同时调用多个第三方接口后汇总结果，可类比 fork/join。

## 配置项列表
| 字段 | 类型 | 默认值 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
| mode | string | all | 否 | 汇总模式：`all`(全部成功才继续)、`any`(任一成功即可继续)、`race`(任一成功继续，任一失败则失败)、`allSettled`(忽略失败，全部结束后继续)。 |

## 分支说明
并行节点开启多个分支：
- `branchIndex` 为正整数（从 1 开始递增），每个分支对应一个并行流程。

## 示例配置
```json
{
  "mode": "all"
}
```
