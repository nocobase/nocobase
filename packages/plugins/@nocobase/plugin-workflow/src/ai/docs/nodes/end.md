# 结束流程

## 节点类型

`end`
请使用以上 `type` 值创建节点，不要使用文档文件名作为 type。

## 节点描述
立即结束当前工作流执行，并以指定状态退出。

## 业务场景举例
在校验失败时直接结束流程，可类比编程语言中的 return。

## 配置项列表
| 字段 | 类型 | 默认值 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
| endStatus | number | 1 | 是 | 结束状态：`1` 表示成功（RESOLVED），`-1` 表示失败（FAILED）。 |

## 分支说明
不支持分支。

## 示例配置
```json
{
  "endStatus": -1
}
```
