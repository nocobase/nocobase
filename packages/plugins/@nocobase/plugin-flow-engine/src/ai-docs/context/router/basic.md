# Basic

## Navigate To

Use this snippet to navigate to.

```ts
ctx.router.navigate(pathname);
```

## Replace Route

Use this snippet to replace route.

```ts
ctx.router.replace(pathname);
```

## Current Location

Use this snippet to current location.

```ts
return ctx.router.location;
```
