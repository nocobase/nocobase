# @onetwosmall/plugin-auth-email

NocoBase 邮箱验证码认证插件，支持 OTP 登录与用户自动注册。

## 功能特性

- **邮箱验证码认证** — 用户通过邮箱 + SMTP 发送的验证码登录
- **自动注册** — 首次登录时自动创建不存在的用户账号
- **SMTP 服务管理** — 可配置 SMTP 设置（主机、端口、SSL、凭证、邮件模板）
- **验证码设置** — 自定义验证码长度、类型（数字/字母/混合）、有效期和重发间隔
- **认证器集成** — 在 NocoBase 认证插件中注册为认证器类型
- **V1 与 V2 客户端支持** — 兼容旧版 SchemaComponent 页面和现代 FlowEngine V2 运行时

## 安装

```bash
yarn add @onetwosmall/plugin-auth-email
```

## 配置

### 启用插件

在 NocoBase 管理后台通过插件管理器启用本插件。

### 配置 SMTP

1. 进入 **认证** > **认证器** > **新建** > **邮箱**
2. 选择**验证器**（需先在验证插件中创建）
3. 可选启用**"用户不存在时自动注册"**
4. 在所选的验证器中配置 SMTP 设置

### 验证码设置

- **验证码长度** — 验证码的位数
- **验证码类型** — 数字、字母或数字字母混合
- **有效期（秒）** — 验证码的有效时长
- **重发间隔（秒）** — 两次请求验证码的最小间隔

## 依赖

- `@nocobase/plugin-auth` — 核心认证插件
- `@nocobase/plugin-verification` — 验证码基础设施
