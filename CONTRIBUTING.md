# Contributing to NocoBase

Thank you for your interest in contributing to NocoBase!  
We welcome all kinds of contributions, including bug fixes, documentation improvements, tests, and small enhancements.

## Ways to contribute

You can help by:

- Fixing bugs
- Improving documentation
- Adding translations
- Writing tests
- Reproducing reported issues
- Suggesting improvements

If you are new to open source, good starting points include documentation fixes and small bug fixes.

## Getting started

1. Fork the repository
2. Create a new branch for your change
3. Make your changes
4. Submit a pull request

Example:

```bash
git checkout -b fix/short-description
